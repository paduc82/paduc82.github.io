tryx3 say:

I think you have the power settings set up because it stayed on all night.  I cannot see that it is an overheating problem as that would crash the system rather than put it to sleep.

I assume you have checked the BIOS to make sure there is nothing in there controlling the sleep event but this is unlikely or you would have spotted the pattern.  Similarly, I asssume you are using Balanced or High performance power plans but that is not going to be the explanation as you'd also have spotted the pattern.

You could recheck your drivers in Control panel, Device manager but I'm only saying that because of the lack of any pattern to the sleeping.

If the laptop came with Win7 installed that that bowls out Win7 compatibility issues for the system itself.

I would suggest booting into Safe mode & doing clean boot checks [see http://support.microsoft.com/kb/929135] but I appreciate that this would be very time-consuming as you'd have to leave the system on for over a day at each check to see how it behaved so it would be impractical.

I don't know what extra devices you have connected ["some devices that don't come back on after sleep"] & I assume you tried testing without them but it would be worth checking their drivers & their compatibility as such behaviour is not the norm.  Even with these devices disconnected they could still be giving a problem as they could still be affecting the system [components are likely to be loaded when Windows starts as you'll see in the safe mode / clean boot article above].

There might be useful info tucked away in Event viewer but I am not familiar with that tool [see Windows help & there are also some forum threads that explain it]

Have you suspected any pattern at all to the sleeping [either in time or related to what devices are connected or what tasks you have been doing]?  If you do deliberately set it to blank the screen then sleep after particular delays does the system respond correctly to these settings?

I assume you have searched the forum for "sleep problem" already.  Anyway, most people reporting such problems are reporting that the system will not sleep.

You might get some more useful assistance in this forum if you do a fresh post with the title "Asus <state model name etc, Win7 version> laptop sleeps despite power plan settings not to do so" and do state in the new post what the extra devices are.

If Asus has forums then they might have useful info unless it is related to these extra devices of course.

Actually libraries (like pictures library) are where you go to manage your documents, music, pictures, and other files. You can browse your files the same way you would in a folder or you can view your files arranged by properties like date, type, and author.

In some ways, a library is similar to a folder. For example, when you open a library, you'll see one or more files. However, unlike a folder, a library gathers files that are stored in several locations. This is a subtle, but important, difference.Libraries don't actually store your items. They monitor folders that contain your items, and let you access and arrange the items in different ways. For instance, if you have document files in folders on your hard disk and on an external drive, you can access all of your document files at once using the Document library.
