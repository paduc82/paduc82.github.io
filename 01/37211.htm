Looks like you didn't release the Win- key soon enough.   I was giving two steps.   Win+C   (See the Charms bar.)   Then, press Enter.   The Start button in the Charms bar has keyboard focus so pressing Enter then switches to the Start Screen.   Normally just pressing the Win- key would switch from the Desktop to the Start Screen but I could not assume that you were at the Desktop, so just asking you to press Win- if you were already at the Start Screen could have caused confusion.  (Win-R will work anywhere.)


Once you have the Start Screen showing press Win+Z  (See the Start Screen's option bar.   Then, press Enter.   Again, the option bar has useful default keyboard focus, this time on the All Apps button.   (There is no Option bar in the Desktop which is why doing Win-Z in its context would have done nothing.)
