bunnyfoot say:

Surface Pro 3 Windows 10 - Power, volume and start buttons not working

Hi there, I have Surface Pro 3 (256 GB / 8GB / i5). Since upgrading to Windows 10 64bit the hardware buttons for power, volume and windows/start menu no longer work. Pressing the Windows button will sometimes give a little bit of haptic feedback. 

Looking in the Device Manager I can that the 'Surface Home Button' is showing as:

This device cannot start. (Code 10)

{Operation Failed}
The requested operation was unsuccessful.

Updating the drivers using the downloads from Microsoft.com doesn't help and same results when attempting to update the drivers via Windows Update: "The best driver software for your device is already installed". 

I've tried uninstalling the driver and reinstalling but still no joy. I've also tried the hard-reset of holding down the power button and volume up button at the same time but this has not helped. Performing a clean install of Windows 10 also produced the same result. 

Any ideas?

Thanks.

EDIT [later 6th October 2015]

I have now fixed this issue.

To resolve:

1. Open Device Manager and expand the System Devices list

2. Right-click on Surface Home Button and choose Update Driver Software...

3. Click 'Browse my computer for driver software' and choose 'Let me pick from a list of device drivers on my computer'

4. Tick the 'Show compatible hardware' button then choose the driver 'Surface Home Button Version 2.0.1179.0 [25/11/2014] and click Next. You will be prompted to restart your Surface - and all the buttons work!

The problem is that the older driver 'Surface Home Button Version 1.1.66.0 [20/02/2015]' has been signed more recently than the newer driver so Windows appears to prioritise it over the more up-to-date one.

I hope this helps someone else.
