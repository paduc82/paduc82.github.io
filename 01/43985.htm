  1. View a list of available networks by swiping in from the right edge of the screen, tapping Settings (or if you're using a mouse, pointing to the upper-right corner of the screen, moving the mouse pointer down, and then clicking Settings), and then tapping or clicking the network icon (or ).

  2. Press and hold or right-click the network you're connected to, and then tap or click Turn sharing on or off.
    Right-clicking on the name of the network only shows: "Show estimated data usage," "Set as metered connection," and "Forget this network." 

    Therefore, right-click on the network icon and select "Open Network and Sharing Center."  Click "Change advanced sharing settings" in the upper left. 

    In the current profile, which presumably has been set to "Private," make sure that the radio buttons are selected to "Turn on network discovery" and "Turn on file and printer sharing."  The check box to "Turn on automatic setup of network connected devices" should also be checked.

    In the "HomeGroup connections" section, select "Allow HomeGroup to manage ..."

    Expand the "All networks" section and check the options there.  Normally, I have the first option selected for each of the 3 features, but make a note of the 3rd one:  Password protected sharing

  3. Choose Yes, turn on sharing and connect to devices for home or small office networks, or when you know and trust the people and devices on the network. This setting allows your PC to connect to devices on the network, such as printers.  N/A

Make sure to click "Save changes" at the bottom,

If "File and printer sharing" has been turned on (see above), you should be able to leave the Windows Firewall on.  In any case, if it was a firewall issue, you would likely be seeing a different error message.

Right-click a file or folder on the 8.1 computer and select "Share with" ... you should see a context menu like this:

Click the "Specific people" link.  You should see a list of users that you have shared the object with.  At the very least, it should show your user account with the "Permission level" of "Owner."

When you open Windows Explorer on your Windows 7 computer, you should see the name of your Windows 8.1 computer in the "Network" category.  I'm going from memory, because I set this up a while ago, but if you attempt to open that computer, you should get a dialog box asking for a user name and password.  Enter the user name and password for an account on the 8.1 computer, preferably a "computer administrator" account.  You then should have access to all of the files and folders normally available to that account.
