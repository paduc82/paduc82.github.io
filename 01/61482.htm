lemp say:

You don't download files "to" Word or WordPad.  You download the file and then open it using whatever program(s) you have installed on your computer that are capable of opening that type of file.

Word is not included with Windows.  Some "brand name" computers may come with Word (and other Microsoft Office application programs) installed -- but only if you pay extra.

WordPad is a rudimentary word processor that IS included (for free) with Windows.  It is capable of opening documents created by Word, but as you have seen, WordPad does not have the capability to display all of the features that may be in a Word document.

Thus, the first question is:  do you have Microsoft Word installed on your computer, and if so, what version?

Assuming that you DO have some version of Word, find one of the *.doc or *.docx files that you downloaded and right-click on it.

Select "Open with" (and if you get a popup menu, click on "Choose Program") to get something like the following:

If you see "Microsoft Word" listed, either under "Recommended Programs" or "Other Programs," select it, check the box to "Always use the selected program to open this kind of file" and click OK. 

From now on, your downloaded *.doc or *.docx should open with Word.

If you don't see Word listed, you probably don't have it, but you can check by clicking the "Browse" button. 

After you click "Browse," you should be in the C:\Program Files directory, like this:

At this point, look for a folder named "Microsoft Office" -- it isn't in the above picture because Word is not installed on the system that I used to capture that screenshot.  If you find the Microsoft Office folder, double click to open it and you will see one or more folders with names such as:

   Office10

   Office11

   Office12

Make a note of the number and then double click on the OfficeNN folder.  In the opened folder, select Winword.exe and then click the "Open" button.

You'll go back to the "Open with" window where you should check the box to "Always use the selected program to open this kind of file" and click OK. 

If you don't have Word installed, post back and I'll give you instructions for how to view the files anyway.

Windows Registry Editor Version 5.00

"Content Type"="text/plain"

@="txtfile"

[HKEY_CLASSES_ROOT\.txt\shell\open]

[HKEY_CLASSES_ROOT\.txt\shell\open\command]

@="notepad.exe %1"

[HKEY_CLASSES_ROOT\.txt\ShellNew]

Method 1: Use the hardware troubleshooter to fix the issue

The Hardware Troubleshooter is an automated tool which will check the hardware connected to the computer for any known issues with it and provides the details and on how to fix them. Follow these steps and run the troubleshooter.

Follow these steps:

  1. Press Windows key to get the Start menu.
  2. Type Troubleshooting in the search box right up corner and press Enter key.
  3. Click on Troubleshooting tab.
  4. Click on view all option on the upper left corner.
  5. Select the Hardware and Device options from the list.
  6. Click Next to run the Hardware and Device Troubleshooter.

Method 2: Update the Bluetooth drivers.

Step 1: Check for the latest driver update through Windows Updates.

  1. At the Start screen, type Windows Update.
  2. Click or tap Settings on the right side of the screen, and then select Install optional updates from the left results.
  3. When the Windows Update control panel opens, click on “Check for Updates” on the top left corner to receive a fresh list of updates.

Incompatible or not properly installed graphics card drivers are the most common cause of this issue.

Step 2: Follow the steps below to update the sound drivers.

  1. Press Windows + X key on the keyboard.
  2. Go to device manager.
  3. Right-click your Bluetooth devices.
  4. Click Update Driver Software and then follow the steps in the wizard that appears.

Restart the computer and check.

Step 3: Check in compatibility mode.

Download the Bluetooth drivers from the manufacture's website for Windows 10, if not available.

Follow these steps to install the drivers in compatibility mode and check if that helps.

  1. Download the driver from the manufacturer’s website.
  2. Right-click on the driver setup file and click on ‘properties’.
  3. Click on the ‘compatibility’ tab and check the box ‘Run this program in compatibility mode for’ and select operating system from the drop down menu and proceed with the installation.
  4. Once this is done, restart the computer and check if the issue persists.
