a. user say:

No problem - this is easy.

Start by locating the executable that launches TOR. Make a shortcut to that executable on the desktop.

Right-click on the desktop shortcut and select Properties from the context menu.

On the Shortcut tab, in the 'Target' box move your cursor to the end of the line. Add a single blank space to the end of the line.

After the blank space, type the desired URL. (Or copy and paste if you're as lousy a typist as I.)

You might want to change what's in the Comment box.

Press Apply, then OK. (You don't really need to press Apply before OK.)

You are done. From now on, every time you double-click on the shortcut, TOR will take you to that website.
