Overview - Microsoft Community

xbr dionysus say:

Overview

Actiontec V1000H

Login Information

  • Username - admin
  • Password - telus
  • Gateway - 192.168.1.254 

How to Find SSID (Network Name)

  1. Open a web browser.
  2. Enter http://192.168.1.254 into the address bar. (Or click the link) Enter the username and password if the gateway asks. (admin/telus)
  3. Select Wireless Setup from the top menu.
  4. Select Basic Settings on the left.
  5. It will be located on this page labeled SSID Name.

 

How To Find Wireless Passphrase (WPA/WEP)

  1. Open a web browser.
  2. Enter http://192.168.1.254 into the address bar. (Or click the link) Enter the username and password if the gateway asks. (admin/telus)
  3. Select Wireless Setup from the top menu.
  4. Select Basic Settings on the left.
  5. The passphrase will be displayed at the bottom of the page in the box that is located in the Security Type section.

 

Changing Wireless Security Type/ Disabling Wireless Security

  1. Open a web browser.
  2. Enter http://192.168.1.254 into the address bar. (Or click the link) Enter the username and password if the gateway asks. (admin/telus)
  3. Select Wireless Setup from the top menu.
  4. Select Basic Settings on the left.
  5. In the drop down menu labeled Security, select the desired security type. (To turn off the security, meaning there is no password needed to connect to the wireless, select Off)
  6. If WEP security is selected the password will need to be 10 charecters in length (if 64-bit) or 26 characters in length (if 128-bit) and can only use 0-9 and A-F.
  7. If WPA or WPA2 was selected the password can be between 8-63 characters, is case sensitive and can use any character on your keyboard.
  8. Enter the desired key in the box labeled Security Key type.
  9. Select Apply.

 

MAC Filtering

  1. Open a web browser.
  2. Enter http://192.168.1.254 into the address bar. (Or click the link) Enter the username and password if the gateway asks. (admin/telus)
  3. Select Wireless Setup from the top menu.
  4. Select MAC address control on the left.
  5. Select the Disable radio button
  6. Select Apply.

 

Change Wireless Channel

  1. Open web browser.
  2. Enter http://192.168.1.254 into the address bar. (Or click the link) Enter the username and password if the gateway asks. (admin/telus)
  3. Select Wireless Setup from the top menu.
  4. Select Basic settings from the menu on the left.
  5. In the drop down box labeled Channel select the desired channel. (Channels 1,3,9,11)
  6. Select Apply.

 

Change Wireless Mode

  1. Open web browser.
  2. Enter http://192.168.1.254 into the address bar. (Or click the link) Enter the username and password if the gateway asks. (admin/telus)
  3. Select Wireless Setup from the top menu.
  4. Select Advanced settings from the menu on the left.
  5. Select the drop down box for compatibility mode.
  6. Set the mode to 802.11 G only.
  7. Select Apply.

 

Change Wireless Band (Width)

*This device only supports the 20Mhz Band*

 

 

 

DHCP Reservation

  1. Open web browser.
  2. Enter http://192.168.1.254 into the address bar. (Or click the link) Enter the username and password if the gateway asks. (admin/telus)
  3. From the menu at the top, click on Advanced Setup.
  4. From the menu on the left, locate the section labeled IP addressing.
  5. Find the xbox's MAC address or IP address from the corresponding drop down boxes.
  6. Click Apply to reserve the xbox in the gateway.

 

UPnP

  1. Open web browser.
  2. Enter http://192.168.1.254 into the address bar. (Or click the link) Enter the username and password if the gateway asks. (admin/telus)
  3. Click on Firewall from the menu at the top.
  4. Click on UPnP from the menu on the left.
  5. Click Enable.

 

DMZ 

  1. Open web browser.
  2. Enter http://192.168.1.254 into the address bar. (Or click the link) Enter the username and password if the gateway asks. (admin/telus)
  3. Click on Firewall at the top.
  4. Click on DMZ hosting from the left.
  5. Click Enable where is says to "select a DMZ state".
  6. Select the Xbox from the device drop down box or type in the Xbox' IP address.
  7. Click Apply.

 

Disable DMZ

  1. Open web browser.
  2. Enter http://192.168.1.254 into the address bar. (Or click the link) Enter the username and password if the gateway asks. (admin/telus)
  3. Click on Firewall at the top.
  4. Click on DMZ hosting from the menu on the left.
  5. Click the radio button next to Disable where is says to "select a DMZ state".
  6. Click Apply.

 

Port Forwarding

  1. Open web browser.
  2. Enter http://192.168.1.254 into the address bar. (Or click the link) Enter the username and password if the gateway asks. (admin/telus)
  3. Click on Port Forwarding in the menu on left.
  4. Enter "3074" in both the Start and End boxes.
  5. For Protocol select "Both" from the dropdown. (If you don't have both, create two of the exact same rules, where you select UDP and TCP independently).
  6. Enter the Xbox‚Äôs IP address into the LAN IP Address area.
  7. Click Apply.

 

Disable Port Forwarding

  1. Open web browser.
  2. Enter http://192.168.1.254 into the address bar. (Or click the link) Enter the username and password if the gateway asks. (admin/telus)
  3. Select Port Forwarding on the left.
  4. Select the Edit button on the Applied Rule you would like to remove.
  5. Delete the rule.
  6. Select Apply.

 

Firewall

  1. Open web browser.
  2. Enter http://192.168.1.254 into the address bar. (Or click the link) Enter the username and password if the gateway asks. (admin/telus)
  3. Select Firewall from the top menu.
  4. Select Firewall from the menu on the left.
  5. Ensure that this option is set to Low or NAT only depending on preference.
  6. Select Apply.

 

How to Find Firmware Version

  1. Open web browser.
  2. Enter http://192.168.1.254 into the address bar. (Or click the link) Enter the username and password if the gateway asks. (admin/telus)
  3. Select Home across the top.
  4. The firmware version is located on this page listed as Firmware Version.

 

Firmware Update

*Firmware updates are installed automatically by the Internet Service Provider.*

 

Restore Factory Defaults

Through Firmware

*Please have PPPoE information available, if you do not have this, please contact your ISP.*

  1. Open web browser.
  2. Enter http://192.168.1.254 into the address bar. (Or click the link) Enter the username and password if the gateway asks. (admin/telus)
  3. Select Advanced Setup from the top.
  4. Select Restore modem to factory defaults.

Through Hardware

  1. Locate the reset button on the back of the device.
  2. Using a toothpick or ballpoint pen press and hold the reset button for 30 seconds.
  3. Wait for the device to restart.

 

 

Bridging Guide

*Bridge mode should only be used if you have a router to plug into the device.*

*Please have PPPoE information available, if you do not have this, please contact your ISP.*

  1. Open web browser.
  2. Enter http://192.168.1.254 into the address bar. (Or click the link) Enter the username and password if the gateway asks. (admin/telus)
  3. Select Advanced Setup across the top.
  4. Select WAN IP Settings on the left.
  5. Select the radio button for RFC 1483 Transparent Bridging.
  6. Select Apply.
  7. Enter the PPPoE information into the router if necessary.

 

Known Issues

*Customers may be unable to connect wirelessly with the 360s using WPA/WPA2 security mode.  Try using No Security or WEP.*

 

Multiple Console Support

We have confirmed that this network device is unable to support more than one console with an Open NAT.
