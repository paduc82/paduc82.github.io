¡firedog say:

It rather depends which password you want to change. 

  

  • When you launch Windows Live Mail, you may be asked to Sign in. If you do so, you will see the calendar and contacts belonging to the Microsoft Account you sign in with. The password for this account can be changed at 
    https://account.live.com/password/change 
      
    If you are automatically signed in in Windows Live Mail (because you elected to Remember me and sign me in automatically last time you signed in), you will be prompted to enter the password next time you launch the program.  
    If this account is also set up as a mail account in Windows Live Mail, you will also have to change the password there following the steps in the next bullet point.
     
    Note that if you enable two-step verification for the account, you will have to generate an app password to use in Windows Live Mail instead of the one you just changed to.
     
      
  • If it's the password for a mail or news account that you want to change, then you must first change it at the Email Service Provider's or news server's website (e.g. att.com or yahoo.com). Having done so, you then have to enter the new password in the account's properties dialogue. Right-click on the account name in the folder pane and select Properties. The password is entered on the Server(s) tab.
      
    Note again that many ESPs now require users to use app passwords in mail programs like Windows Live Mail. This might be called something else by your ESP; AT&T use the term Secure mail key, for example.

Hi bums85,

If you have .ico files inside a folder, you need to click in the Browse button (in the Change Icon dialog) to find and select the desired icon. You can also use icons embedded in some dlls, such as %SystemRoot%\system32\SHELL32.dll.
