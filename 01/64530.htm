considerate_guy say:

It is not possible to have multiple pictures appearing on the screen at once in an animated fashion.  Movie Maker is designed to have one picture at a time.

 

About the only workaround is to create your own "slides" in an image editor where you add multiple pictures together.  But then they would be considered one image and you couldn't really animate the parts separately.

 

Another option, if you have PowerPoint 2010 is to create this in PowerPoint and then use the feature in PP to save it as a video.  You could then make final edits in Movie Maker.
