Hi Ian,

I realize the inconvenience you are experiencing with Wi-Fi connection on Windows 10. I will certainly help you.

I appreciate you for providing details about the issue and your effort towards resolving it.

This issue could be because of one of the following problems.

1. Your PC is too far from the wireless router or access point.

2. Your router and the network adapter in your PC might not work together.

3. There's interference from other devices or due to software conflicts.

4. The router or access point is busy if several PCs or devices using it.

Try the following troubleshooting methods and check if it resolves the issue.

Method 1: Run the Internet Connection troubleshooter

Windows 10 has a built-in troubleshooter to check and fix issues with Internet connection. I would suggest you to run this troubleshooter to check if the issue is with your Internet.

1. Right click on Start button, select Control Panel.
2. Change the view by option on the top right to Large icons.
3. Click on Troubleshooting and click on the view all option on the left panel.
4. Run the Internet Connection troubleshooter.

Method 2: Run Networking commands

If issue persists, try resetting the TCP/IP stack, Release the IP address, Renew the IP address and Flush and

reset the DNS client resolver cache. To do this, try below steps:

1. In the search box on the taskbar, type Command prompt, press and hold (or right-click) Command

    prompt, and then select Run as administrator >Yes.

2. At the command prompt, run the following commands in the listed order, and then check to see if that

    fixes your connection problem:

Type netsh winsock reset and press Enter.

Type netsh int ip reset and press Enter.

Type ipconfig /release and press Enter.

Type ipconfig /renew and press Enter.

Type ipconfig /flushdns and press Enter.

Hope this information helps. If the issue remains unresolved, please get back to us and we would be happy to help.

Hi,

 

If you want to change the way the Folders and Applications are listed in All Programs in the Start Menu, you need to make some changes in the start menu properties. Do this as follows:

 

1.     Right click on the Start orb(Button) and select Properties

2.     Under the Start Menu tab click Customize.

3.     Now find Sort All Programs menu by name and click to clear the check mark.

4.     Click OK and then Apply and OK again.

5.     Go to the Start menu, All Programs and drag and drop the folders as you wish to.
