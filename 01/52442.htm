Log out and go to the sign in screen.
Hold down the shift key on your keyboard while clicking the Power button on the screen.

Continue to hold down the shift key while clicking Restart.
Continue to hold down the shift key until the Advanced Recovery Options menu appears.

Click Troubleshoot
Click Advanced options
Click Startup Settings
Click Restart
Press (4) to Enable Safe Mode
Wait while Windows 10 starts in safe mode.
Log in

Press Windows key + X
Click Device Manager
Expand Display Adapter
Right click your Display adapter and click Uninstall
Exit Device Manager

If you have an older Antivirus utility also uninstall it.
Restart

You might need to go to a working computer, download the latest Video Driver for your computer, save it to a thumbdrive. Copy it to the affected computer and install it. If there is no native Windows 10 driver, download the latest Windows 7 or Windows 8.1 video driver and install in compatibility mode.

Learn more:
How to: Install and Update drivers in Windows 10
http://answers.microsoft.com/en-us/insider/wiki...

At the Sign In screen, hold down the shift key on your keyboard while clicking the Power button on the screen
Continue to hold down the shift key while clicking Restart
Continue to hold down the shift key until the Advanced Recovery Options menu appears

Click Troubleshoot

Click Advanced options

tryx3 say:

Barbara,

1  Yes, it would get your system back up if you had some kind of Windows trouble.  The purpose of a Recovery drive is to "reinstall" Windows 10.  The descriptions are not exactly clear but you can see them in Recovery options in Windows 10 - Windows Help.  There is a fuller explanation of recovery drives in Recover Windows 10 from a Recovery Drive - Windows 10 Forums 

2  According to several experienced users on this forum, there is one drawback to using a Recovery drive and that is that is can only be used successfully on a Windows disk that has I thought it would be misleading not to do so.  You can check whether or not you have partitioned the disk with Windows on it.  One method is:-

  • right-click on the Start button & select Device manager,
  • click on the arrow next to Disk drives to expand it,
  • right-click on your hard disk and select Properties,
  • select the Volumes tab and click on Populate,
  • if you have not partitioned your hard drive you will see a single Volume entry that is likely to be OS10 [C:]
  • but if you have partitioned your hard drive you will see a list of Volume entries that includes something like OS10 [C:] as well as other entries,
  • if you have multiple disks in your computer, you would only expect one of them to have an entry similar to OS10 [C:] so you could check each of them if you wanted.

3  The benefit of making regular system images is that you can restore the exact contents of the Windows drive as they were at the moment you created the image so any corruption that might have occurred can be overwritten. 

  • There is a guide to making them at System Image - Create in Windows 10 - Windows 10 Forums  and advice about restoring them at Windows 10 system image recovery procedure [my own ditty] 
  • Please take note of the warnings about backing up your own datafiles before restoring a system image - you are likely to alter datafiles every day but only to make system images once a week / month so there will always be datafiles that are newer than they were when the image was made.
  • You can also restore an image to a new drive but, to keep things simple, replace a defective drive with one that is of the same brand & type as the previous one.  The replacement disk must be at least as big as the original disk to allow the image to fit on it.  I have successfully restored images onto new drives but I once failed because [I eventually discovered] the replacement drive needed a different set of drivers/controllers.  Had I understood the problem at the time I could probably have solved it easily [I have seen people report the same problem in this forum and they seemed to have been helped to fix things].

4  I used to make System repair disks on CD then transfer them to USB but I do not bother anymore.  Instead, as each new version of Windows 10 is released I use the Media creation tool to make an installation disk on USB and that has all the capabilities that System repair disks do & more besides.  If I wanted to make a System repair disk now I would have to dig out my external CD-DVD drive to make it on then run a particular set of commands on a USB stick to make it bootable and then copy the contents of the System repair disk CD onto the USB - it just does not seem worth the bother.  System repair disks contain several repair tools but they do not have the full capabilities of either a Recovery drive or an installation disk.

5  About that thread that suggests a trick to put Windows system images on USB. 

  • I stumbled across that thread by chance but have not tried it. 
  • The only reason to try it would seem to be if you travelled a lot and wanted the security of having your latest image on a small USB stick rather than a larger external hard drive. 
  • But since you have a Recovery drive on USB anyway that would give you many repair & recovery options. 
  • I always stick within the 10KG cabin baggage allowance when flying yet I never find the additional weight [about 8oz or 230g] of an external hard drive to be a problem.
