andy rathbone say:
Ever since Windows XP, Guest accounts haven't been allowed to initiate an Internet connection. It's a security measure, and I don't think that's changed in Windows 8.

However, if an Internet connection is already present, the Guest account can use it.

So, if you want to give the Guest account access to the Internet, do this:
  1. Log onto the Internet using a traditional account.
  2. Without signing out, let the Guest account sign in.
Since the Internet connection will already be in place, the Guest account will be able to use it.
