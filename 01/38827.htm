A Clean Boot is not the solution, a clean boot does nothing for the install file, which installs just fine on Windows 7 and below (Vista, XP, etc.). 

The problem is that the original file was not originally intended to install on Windows 8 and above, a problem which people are looking for a remedy for. 

The only real remedies are as follows: 

1. Copy all the files from a previous install, including all original registry keys, and copy them to the Windows 8 and above computer, so that the program will run correctly.
