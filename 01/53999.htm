Method 1

 

Windows XP may default to Safe Mode at every startup. Even if you select the option Start Windows Normally, it still boots into Safe Mode. IOW, you cannot get out of Safe mode.

This happens if the /safeboot switch is added in the Boot.ini file. /Safeboot parameter forces a start in safe mode by using the specified parameters. The available parameters are:

a)       minimal

b)       network

c)       safeboot:minimal(alternateshell)

For more information on Boot.ini switches, see Reviewing and Correcting Boot.ini Settings on x86-based Systems

 

RESOLUTION

a)       Click Start, Run and type MSCONFIG

b)       Select the Boot.ini tab

c)       If you have dual-boot setup, select the appropriate entry under [operating systems]

d)       In the Boot Options section at the bottom, uncheck /SAFEBOOT option

e)       Click OK

f)        Restart Windows

The /safeboot entry will now be removed from the Boot.ini file.

ffoote say:

After installing the HP Printer Diver - an unhandled exception ('System.Outofmemoryexception') occurred in HPLaserJetService.exe

Does anyone have a fix for this annoying popup error that comes up every so often.

Windows 7 64bit

lemp say:

I'm not positive, but I suspect that HPLaserJetService.exe monitors the status of your printer's toner supply so it can remind you to buy more HP toner.

Did you install the printer driver by:

  1. Connecting the printer to your computer and allowing Windows 7 to install the driver automatically
  2. Downloading and running the "Full software and Drivers" or "Complete Solution" package from the HP support page
  3. Downloading and running the "Basic driver" package from the HP support page
  4. Running the software from a CD that came with the printer

If you elect to uninstall and reinstall the printer (see last paragraph below), options 2 and 4 are likely to reinstall HPLaserJetService.exe while options 1 and 3 likely will not.

Download (free) Autoruns from https://answers.microsoft.com/en-us/windows/forum/windows_7?sort=lastreplydate&dir=desc&tab=Threads&status=&mod=&modAge=&advFil=&postedAfter=&postedBefore=&threadType=All&tm=1426273705649 (or also just run it by clicking on the other link).

When the Autoruns scan completes, click the "Logon" tab, find the line that includes HPLaserJetService.exe and UNcheck the box.  This will prevent the service from starting and you shouldn't get any more errors from it.

If you really want to have the service run but not throw the occasional error, I suggest that you look in or post to the HP forum:  http://h30434.www3.hp.com/t5/LaserJet-Printing/bd-p/Install

There are quite a few threads there that refer to this service.  As near as I can tell, the two most offered "solutions" are to disable the service or to uninstall and reinstall the printer software and driver.

http://h30434.www3.hp.com/t5/forums/searchpage/tab/message?filter=location&location=forum-board%3AInstall&q=hplaserjetservice.exe
