The RT is a "canned" system and will only run apps available through the Store.  Also, Adobe provides Microsoft with updates to Flashmedia and then feed through Windows Update.  You can no longer download and install Flashmedia player nor can you install JAVA.

 

You cannot uninstall Windows 8 RT and install the normal x86 version.

 

If you are looking for a Surface tablet that runs normal software, that would be the Surface Pro.  If you bought the machine at a local store, check with them to see if you can return/exchange it with the Surface Pro.  That machine will run just about anything that is out there.

 

If you bought it at a store other than a Microsoft Store, chances are, the sales clerk didn't know the difference between them either.

 

Always remember to research something before you buy it.  Basically, the RT is a self-contained machine that can only run apps available for it through the Store.

When it comes to updating your drivers you have the option of using OEM releases or the generic releases.  If you want the OEM release you would obtain them from HP's website or their support.  The problem with OEM releases is that they tend to stop releasing updates after a set period of time.  Unless OEM support can provide you with the latest releases, then you are stuck with the bugs present in the last set of drivers.   While OEM drivers are better suited for your hardware, a more stable, generic release should be preferable over unstable/defective OEM drivers.
