a. user say:

 

See the section entitled Updates from the previous cumulative Windows time zone update on this page: <<http://support.microsoft.com/kb/2633952>>

Do you reside in one of the locations listed? If not, feel free to hide this Optional, non-security update: You don't need it.

 

1. Stop the Automatic Updates service:

Start > Run > (type or copy/paste) services.msc > [OK]
Double-click Automatic Updates > Click on Stop
(Stopping the service will take a moment)

2. Delete the contents of the Download folder:

Start > Run > (type or copy/paste) %windir%\SoftwareDistribution > [OK]
Open the Download folder and delete its contents
Close the window.

3. Open Internet Explorer (only!) to http://windowsupdate.microsoft.com & run a CUSTOM scan | Uncheck/deselect KB2633952 and hide it ("Don't show me this update again").

Tip: If a Root Certificates update is listed in Optional Software updates (in left-hand menu), install it to take full advantage of IE's enhanced security but do NOT install any other Optional Software nor any Optional Hardware updates, just hide them, too.

4. Start the Automatic Updates service:

Start > Run > (type or copy/paste) services.msc > [OK]
Double-click Automatic Updates > Click on Start
(Starting the service will take a moment)
