greg carmack - mvp 2010-2020 say:

Those permissions pictured are only for the shortcut, not the actual folder.  The files themselves must have them.

Those permissions should remain granted once you Allow them under Admin account.  That they don't indicates some level of corruption.  Go over your install with the Windows 10 Performance and Install Integrity Checklist - Microsoft Community to check it's integrity and do any necessary repairs.

To grant permissions the surest method is to install and use the Take Ownership Shortcut.  Do not use it to grant blanket permission to an entire drive, though - especially C.

Be aware also that no permissions are required when loading the image from the booted Rescue disk like Windows installation disk, Recovery Drive or REpair disk which is how it's normally applied in an emergency,

But there are still other shortcomings with Windows Imaging which makes it worthwhile to have a secondary backup or to use instead a more flexible imaging app like popular Macrium Reflect is a Free and Easy To Use Backup Utility.   Often WIndows imaging won't restore to another sized replacement drive, or will only recover partitions to their exact same offset or position on the original drive.  Macrium can restore anywhere, and along with Windows 10 now both are remarkably adaptive to different hardware.
