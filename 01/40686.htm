a. user say:

Welcome to the Microsoft Windows Vista Communities site.

 

 

Understanding backup options in Windows Vista…

 

There are two types of backups available in Windows Vista. However, these two backups operate very differently.

 

File Backup creates a shadow copy (also known as a snapshot) using VSS to make sure that all opened files are flushed from memory to the file system. Once this is done, File Backup reads the files from the snapshot and places them in a zip file. The zip file will contain complete files from both a full and incremental backups, so this is why the zip file is larger than you might expect. Also, File Backup does not make use of snapshots to store incremental block-level changes to files like Complete PC Backup does.

 

Please note that the Complete PC Backup is only available in the Business, Enterprise, and Ultimate editions of Windows Vista.

 

Complete PC Backup

This method of backup is supported in Business, Enterprise, and Ultimate editions of Windows Vista. This method is considered an image backup as it takes a “snapshot” of your programs, system settings, and the files on your computer. You can use this backup if your computer ever stops working. Although this type of backup includes your personal files, we recommend that you backup your files regularly using the Back Up Files Wizard.

Backup Files Wizard

This method of backup is supported in the Home Basic, Home Premium, Business, Enterprise, and Ultimate editions of Windows Vista. This method backs-up the following file categories: Pictures, Music, Videos, E-mail, Documents, TV shows, Compressed files, and Additional files.

 

As you are using Vista Home Premium you only have the Backup Files Wizard available to you. After, a full backup is made all others will be incremental. However, the incremental backups are not done on a “block-level”, but rather file basis.

 

For example, if you had a Microsoft PowerPoint presentation that was 10 MB, the first time you backed it up using Backup Files Wizard, it would take up 10 MB. But if you then added some slides to it and it grew to 11 MB, your next incremental backup will include a complete new copy of it. On the other hand, with Complete PC Backup, the first time it will back up 10 MB, but the second time, it will only capture the block-level changes (within the file), which could be as little as 1 MB.”

Surface with Windows RT uses a completely different processor to standard x86 based PCs so cannot run classic Windows programs of any sort. In order to achieve a good standard of programming for RT Surface will only allow programs to be installed from Windows Store so has to meet their quality requirements. The only exception is that companies that create their own Line of Business apps can "sideload" onto the Surface by the use of a special license.

 

Windows 8 is already available and does allow the installation of existing Windows programs in the same way as Windows 7 etc. The second version of Surface due out next year has an x86 processor in it, runs standard Windows 8 and will allow full applications to be installed.

 

It is because of the platform it is based on Surface with Windows RT cannot be upgraded to Windows 8.
