NOTE: This button will be grayed out if the listed user is connected by a network instead of only being connected locally (EX: Switch User).
8. Click on Log off user to confirm.
WARNING: If the user is working on something when you log them off, they may lose it.
9. Task Manager will now no longer have the user listed since they have been logged off. -Source Meghmala, member

Change Wireless Mode

Login Information

 

  • Gateway: 192.168.2.1
  • Username: admin
  • Password: admin

 

 

 

Change Wireless Mode

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.2.1) into the address bar of the internet browser.
  3. When prompted, enter "admin" for the user name and "admin" for the password.
  4. Select "Device Table" from the menu to the left.
  5. Note down the IP Address of the WAP610N. (It will be in the format of 192.168.2.X)
  6. Open a new browser tab or window.
  7. Log into the router by entering the default gateway (192.168.2.x) into the address bar of the internet browser.
  8. When prompted, enter "admin" for the user name and "admin" for the password.
  9. Select "Wireless" from the menu at the top.
  10. Select "Basic Wireless Settings" from the sub menu, if it is not already selected.
  11. Select the radio button next to "Manual".
  12. Select the desired wireless mode from the "Network Mode" drop down menu.
  13. Select Save Settings.
