miguelonky say:
Hello Donal

Thank you for participating in the Microsoft Community, My name is Miguel T.
For Windows 10 these are the steps to achieve your request:

1. Open Registry Editor (regedit.exe).

2. Navigate to the key below in the left pane of Registry Editor.

HKEY_CURRENT_USER\ControlPanel\Desktop\WindowMetrics

3. In the right pane of the WindowsMetrics key, double click/tap on the CaptionHeight string value to modify it.

4. Enter a value between -120 (smaller) to -1500 (larger) for what you want, and click/tap on OK. (see screenshot below)

-330 is the default value.

The value uses the following formula: -15 x size you want in pixels.

For example: -15 x 22 = -330

Size: 12.0 KB

5. When finished, close Registry Editor.

6. Sign out and sign in to apply.
