a. user say:

They first go into the Recycle Bin of the networked computer.  You may be able to find them there.  If they aren't there, check the Recycle Bin on the computer that issued the delete command (though I don't think they go there - it's still worth a look).

If the Recycle bins have been emptied or you did a shift-delete then you'll need to use the data recovery programs provided by Mr. Murphy - but check the Recycle Bins first since that's much easier - well that's not necessarily entirely true.

If you have Vista Business, Vista Ultimate, or Vista Enterprise you may be in luck because they retain files called Shadow Copies http://windows.microsoft.com/en-AU/windows-vista/Previous-versions-of-files-frequently-asked-questions.  These can be used to retrieve previous versions of files that have been accidentally modified or deleted.

If you have Vista Basic or Vista Premium you do not have Shadow Copies, but there is a free program available for you that does essentially the same thing called Shadow Explorer: http://www.shadowexplorer.com/.  I suggest you download and implement it ASAP to protect you in case you accidentally delete or modify a file.  It won't help you in this instance, but it may save you the next time.

Good luck and I hope this helps!
