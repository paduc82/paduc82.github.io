Hi Daniel,

 

This issue happens due to the corrupted InstallShield or if the Microsoft Installer is not working properly on the computer. I would suggest you to try the following troubleshooting methods and check if that help.

 

 

Method 1: Rename the InstallShield folder

 

a)       Open the File Explorer from the desktop interface of your computer.

b)       Navigate to the following location

 

C: \Program Files\Common Files\

               

c)       Right-click the “InstallShield” folder and then click on “Rename” from its context menu.

d)       Type “InstallShield1” and then press “Enter” key on the keyboard.

e)       Now, try installing the game and check if that resolves the issue.

 

Note: If you are unable to find the “InstallShield” folder in above location, then you may also check if it is available in the location C: \Program Files (x86)\Common Files\. If available, then you may try renaming it and check if that resolves the issue.

 

 If the issue persists, then try the next method.

 

Method 2: Verify if Windows Installer engine is functioning properly.

 

To do this, you may try these steps.

 

a)       Press “Windows Logo” + “X” keys on the keyboard and select “Command Prompt (Admin)” from that menu.

b)       In the “Administrator: Command Prompt” window, type “MSIExec” and then press “Enter”.

c)       If the MSI engine is working properly, there should be no error messages. Instead you would be getting a window showing MSInstaller version and 5 and the details of its various options.

d)       If you get any error message, please do get back to us with the exact error message/code.

 

If it is working fine, then try installing the game. If the issue persists, then go ahead with the next method.

 

Method 3: Reregister the installer engine

 

a)       Press “Windows Logo” + “X” keys on the keyboard and select “Command Prompt (Admin)” from that menu.

b)       In the “Administrator: Command Prompt” window, type the following command and then press “Enter” after each line:

 

MSIExec /unregister

 

MSIExec /regserver

 

c)       Type “Exit” and then press “Enter” to close the “Command Prompt” window.

d)       Now try installing the game and check the issue.

 

If the issue persists, please do get back to us with the troubleshooting result. We’ll be glad to assist you further.

techie bear say:

Hello Michael,

Setting the User Account Control (UAC) to a lower setting (including off) reduces a security feature  in Windows that theoretically prevents an unauthorized user from (remotely) running an application on another person's computer. I call it theoretical because like any security measure if someone just blindly clicks through, the security questions won't have done much good.

 

You adjust the level of control by going to Control Panel, selecting User Account, and when that window opens select Change User Account Controls settings. When that window opens move the slider bar to your desired level, select OK, and you'll need to restart the computer to have the changes take effect.

 

And that's how its done. Let me know if this helps you out.
