¡firedog say:
You should be able to so long as the OS and program versions are identical. Rules are stored in the registry, so you need to copy the appropriate key from the old drive to the new one.
  1. On the old installation, open Registry Editor and navigate to
    HKEY_CURRENT_USER\Software\Microsoft\Windows Live Mail\Rules\Mail
  2. Right-click on Mail and select Export. Give it a name like mailrules
  3. Copy the file you saved to the new installation's desktop.
  4. On the new installation, set a restore point in case anything goes wrong.
    http://windows.microsoft.com/en-US/windows7/Create-a-restore-point 
  5. With Windows Live Mail closed, double-click on the saved .reg file to merge it with the new registry.
  6. Log off Windows, then log on again, launch WLMail and check the rules.

To open Registry Editor, type   regedit   into the start search box and press Enter. Give permission when prompted.

Note that folder names will have to be re-entered, because the new installation will have a different set of virtual folders from the old one. The same may apply to accounts and some other parameters.
