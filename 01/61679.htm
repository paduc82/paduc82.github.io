look for hidden factory install partition if you have one it will reinstall back to when new.

Go to start - computer - right click - manage - disk management .

Your h/d would have 2 partitions 1 with letter C: with a NTFS file system. 

The hidden recovery partition would have no letter and a Fat32 file system with up to 6GB of space.

If there is, Restart the computer and keep tapping each one of these keys F10 or F11.

This should start a PC angel or Ghost program that will erase and format harddisk then re-install OS back to Factory Settings.

When asked by the program do you want to save your data say NO.
