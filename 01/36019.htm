Finally, Clean boot the system and retry Windows Store


  1. Log on to Windows 8 computer with the administrator account.
  2. Click Desktop tile from the Start screen to go to the desktop.
  3. Once on the desktop screen, press Windows + R keys simultaneously to initiate the Run command box.
  4. On the Run command box, type MSCONFIG.EXE command in the available field.
  5. On the opened System Configuration box, make sure that General tab is selected.
  6. On the selected tab, click to select Selective startup radio button.
  7. Once selected, uncheck Load startup items checkbox.
  8. Then go to Services tab.
  9. From the bottom left corner of the selected tab, check Hide all Microsoft services checkbox.
  10. On the same tab, click Disable all button at the bottom right corner to disable all the running services.
  11. Then go to Startup tab, and click Open Task Manager.
  12. On the opened Task Manager window, make sure that Startup tab is selected.
  13. On the selected tab, right-click each startup program one-by-one.
  14. From the displayed context menu, click Disable to disable the selected startup item.
  15. Close Task Manager when done.
  16. Back on System Configuration box, click OK.
  17. On the displayed message box, click Restart to clean boot the Windows 8 computer.
