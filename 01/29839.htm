I had tried each of the following steps, and still could not get Microsoft Updates:

1) Waited 20 mintues then tried again

2) Stopping wuauserv, cryptSvc, bits, msiserver, renamed catroot2 and softwaredistribution, then restarted all the services.

3) Windows Update Troubleshooter

4) Uninstalled ESET Smart Security

5) Turn off Windows Firewall

Right as I was about to give up and ask for help on here, I found a post online that recommended unchecking "Give me recommended updates the same way I receive important updates", while still keeping "Give me updates for other Microsoft products...." checked.  VOILA!

I hope that someone finds this post at the beginning of their troubleshooting, and saves them a lot of time and aggravation.  If you have any other things that you have tried that have worked, please post them as well in this post.  If you are looking for assistance, and none of these suggestions have fixed your issue, please create a new post on answer.microsoft.com as I will not be monitoring this post.
