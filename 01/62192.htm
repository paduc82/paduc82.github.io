tryx3 say:

There are also some utilities that can assist you with part of this but I think the simplest way would also, in this case, be the best.  It involves no printing - it's much better.

 

1  In Windows explorer, get the Documents folder in the right-hand pane.

 

2  Press the Shift key then right-click on the Documents folder & select Open Command window here.

 

3  Enter this in the Command prompt to create a text file listing all your files & folders

 

[substitute your user name for YourUserName & whatever folder you want for the place to create the output file if you don't want Desktop]

 

dir /s /b >"C:\Users\YourUserName\Desktop\List of files in Documents.txt"

 

[you can see other dir options by entering dir /? first if you want]

 

4  Open the file on your Desktop that this command has created [it will open in Notepad].  Then use Edit, Select all and Copy.

 

5 Open Excel or another spreadsheet application, create a new blank file then Paste.

 

6  You can sort this list in Excel to help you be able to see the information clearly. 

 

7  You could also extract various elements of the information such as the file name if you know Excel well enough to do so.  I have some notes somewhere that explain this so, if your file list is so long that it all becomes confusing, post back here & I'll find those notes.
