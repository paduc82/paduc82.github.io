lemp say:

Some keyboard shortcuts are built into Windows.  Others are functions of various software installed in your keyboard.  Probably the best-known standard Windows shortcut is F1 for help.  See https://support.microsoft.com/en-us/help/12445/windows-keyboard-shortcuts for a list of Windows keyboard shortcuts, including F key shortcuts.

Most laptops have software that uses the various F keys to perform things like switching between internal and external display, making the screen brighter or dimmer, turning the wifi radio on and off, etc.  To make things easier for users to remember, these keys often have little icons printed on them.

Some keyboards are sold as "multi-media" keyboards or "gaming" keyboards.  These keyboards may have additional special keys in addition the F keys.   In general, you must install a specific software application (or driver) to enable these functions.  Unfortunately, Acer's website doesn't seem to have drivers for its stand-alone keyboards (although you could try looking it up using the serial number on the bottom of the keyboard:  https://www.acer.com/ac/en/US/content/drivers).  Or ask in the Acer forums: https://community.acer.com/en/  Or ask Acer directly:  https://us.answers.acer.com/app/chat/chat_launch

There are numerous web sites that purport to offer drivers for the Acer SK-9621 keyboard, but I would be very wary of such sites.  If you can't get any help from Acer and still want to try one of these sites, I suggest the following:  (1) Do NOT, under any circumstances, download and install a program that purports to scan your system to find updated drivers; (2) make sure that your antivirus and antimalware applications are up to date to help avoid "drive-by" infections at sketchy web sites; (3) upload any file that you do download to Virus Total before executing the file:  https://www.virustotal.com/; (4) backup your system before installing the driver or at least create a System Restore point.

Xbox 360 games use your account's country to determine which version you get. 

So if your account is from a Spanish speaking country, your downloaded game will be in Spanish, as this specific game only has one language.

In order to get the English language version, you might have to migrate your account to a different country.
