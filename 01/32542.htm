tryx3 say:

Bill,

Answer 1 - No, it is not possible for Windows to create a system image on a USB memory stick. 

  • Windows can only create system images on SSDs / hard drives. 
  • The fact that an external SSD / hard drive might also be connected by USB is not a problem [Windows can still tell the difference between them & a memory stick].

Answer 2 - Yes, it is possible to trick Windows into creating a system image on a USB memory stick despite the fact that it does not want to do so.  How to create Windows 10 system image on a USB flash drive  There is never any guarantee that such a trick will work in the future.

- - - - - - - - - - - - -

I use both Windows' own system imaging facilities and Acronis True image.  I just find it more convenient to use Windows for routine images but I make Acronis images before every significant Windows update [such as a new Windows 10 version].  I have twice replaced hard drives and found that Windows could not restore its image onto the new drive correctly but I was saved by Acronis images which were able to restore correctly. 

Whichever imaging utility you decide on, make sure that you make its necessary rescue disk / boot disk. 

  • For a Windows system image boot disk, make a Windows installation disk [it is a repair tool as well as an installation tool], a "System repair disk" or a "Recovery drive".
  • Whilst Windows can normally restore images using its built-in facilities, if Windows cannot start properly or is not running sufficiently well then you would boot up from one of those to initiate restoring the system image. 
  • Acronis insists on booting from its rescue disk to restore system images.

Whichever imaging utility you decide on, make sure that you test its necessary rescue disk / boot disk to make certain that it can boot up correctly because you do not want to find out that it is faulty when you are trying to use it to get out of a hole.

I don't know anything about Macrium reflect but I have seen many experienced users praise it.  It has a free edition as well as a paid-for one.
