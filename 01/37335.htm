Run the following PowerShell script to re-register the new Store and check if that helps:

  1. In the Search box, type CMD

  2. Right click on Command Prompt and choose Run as administrator

  3. In the command prompt, enter the following command:

PowerShell -ExecutionPolicy Unrestricted -Command "& {$manifest = (Get-AppxPackage Microsoft.WindowsStore).InstallLocation + '\AppxManifest.xml' ; Add-AppxPackage -DisableDevelopmentMode -Register $manifest}"

Hope this helps you. Please do let us know if you need further help.
