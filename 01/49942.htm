The recovery partition is provided for you to use to reinstall Windows should you need to. It is *not* there for you to write anything to. If you do so, you run the risk of screwing up your ability to reinstall Windows. It normally comes full or almost full, so it's no surprise that there is no free space for you to write to.

Also note that your computer presumable ca,e with instructions for you to copy the recovery partition to a DVD. You should do this immediately, since if you have a head crash or a similar problem, you will lose its contents.

Your recovery partition is just that, a partition for recovery there shouldn't be anything else on it but the files used to recover Windows  in the event of a problem. If you are saving other files to this recovery partition it is wise to save them elsewhere.
