tryx3 say:

The Ren command can be manipulated a bit but what you are telling it to do is rename the files it finds as *.bmp i.e. to rename them with whatever their names were in the first place.

 

I have recently been setting up a similar but regular task and I came to the conclusion that only a vbs script would do the job - check the filename length, extract the right-hand characters of that name [i.e. less the first 5 characters ~bob ], then rename the file with what's left over.

 

If you don't want to mess around with vbs then try Bulk Rename Utility

tryx3 say:

Here is a vbs file I've just tested with sample files in "D:\Desktop\TestFolder"

 

Just as you would when creating a batch file, create an empty file called RenameBobsFiles.vbs then right-click on it & select Edit.

 

Paste all this into it then change  "D:\Desktop\TestFolder" to the correct folder name for your needs

 

Dim fso, f, f1, fc, OldName, NewName, ChosenFile, LowerExt, SelectedFolder, NameLen
Set fso = CreateObject("Scripting.FileSystemObject")
SelectedFolder = "D:\Desktop\TestFolder"
' Remember to set this to the correct  folder after all testing is finished
Set f = fso.GetFolder(SelectedFolder)
Set fc = f.Files
For Each f1 in fc
OldName = f1.name
LowerExt = Right(OldName, 3)
LowerExt = LCase(LowerExt)
If LowerExt = "bmp"  Then
                    If Left(OldName,5) = "~bob " Then
                              NameLen = Len(OldName)         
                              NewName = Right (Oldname, NameLen - 5)
                              NewName = SelectedFolder & "\" & NewName
                              Set ChosenFile = fso.GetFile(SelectedFolder & "\" & OldName)
                              ChosenFile.Move (NewName)
                    Else
                    End If
Else
End If
Next

tryx3 say:

Or, if you want to run this in many different folders, it can be easier to get the vbs file to pick up the path to the folder it is in instead of telling it to work on a specified folder

 

Dim fso, f, f1, fc, OldName, NewName, ChosenFile, LowerExt, SelectedFolder, NameLen
Set fso = CreateObject("Scripting.FileSystemObject")
SelectedFolder = CreateObject("Scripting.FileSystemObject").GetAbsolutePathName(".")
Set f = fso.GetFolder(SelectedFolder)
Set fc = f.Files
For Each f1 in fc
OldName = f1.name
LowerExt = Right(OldName, 3)
LowerExt = LCase(LowerExt)
If LowerExt = "bmp"  Then
                    If Left(OldName,5) = "~bob " Then
                                        NameLen = Len(OldName)
                                        NewName = Right (Oldname, NameLen - 5)
                                        NewName = SelectedFolder & "\" & NewName
                                        Set ChosenFile = fso.GetFile(SelectedFolder & "\" & OldName)
                                        ChosenFile.Move (NewName)
                    Else
                    End If
Else
End If
Next
