When a backup is created on a volume on a hard disk, a file called mediaid.bin is added to the root of the volume. Your backup folder is there as well. If you move the backup folder but forget to move the mediaid.bin file, Windows 7 won’t recognize the backup.

restricted access to wallpaper changes - Microsoft Community

yathink103 say:

restricted access to wallpaper changes

I have windows pro and it tells me that the function has been disabled how do I fix it?
daniel martín say: First of all, scan your computer using a current antivirus; many viruses/spyware cause symptoms like yours.

The message "This feature has been disabled. Please contact your administrator for details" appears when someone has applied a policy to disable the possibility of changing the background. There are 3 policies involved:

1. Click Start, type regedit and hit ENTER.

2. Browse to this registry key: HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Policies\System

3. Check if the key contains any of these values (on the right pane). If so, delete them:

NoDispCpl
NoDispBackgroundPage

4. Repeat steps 2-3 but now browse to this registry key: HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Policies\System

5. Browse to this registry key: HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Policies\ActiveDesktop

6. Check if the key contains this value. If so, delete it:

NoChangingWallPaper

7. Repeat steps 5-6 but now browse to this registry key: HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Policies\ActiveDesktop

8. Close Registry Editor, restart your computer, and try to change your background.
