considerate_guy say:

Windows Vista and Windows 7 came with a program called "Windows DVD Maker" that would allow you to create a video DVD (with a menu, scene selection, etc.)

After that (Windows 8 and later) the program was dropped. I think it was about the same time that Apple was dropping their iDVD program from the iLife Suite.

My guess is both companies saw little reason to support burning video DVDs given that it is a legacy format (can't handle HD video, uses the older MPEG-2 codec, etc.).

You can burn the .mp4 files that Movie Maker creates onto a DVD and most *Blu-ray* players will be able to read and play the discs. But you won't have any fancy menus. But you will have HD (high-definition 1080) video rather than SD (standard-definition 480) video.

Or you can look around for DVD authoring programs that will let you do this; they just aren't supplied by Microsoft. Some freeware options are DVD Styler and DVD Flick.

Microsoft Windows is configured to play a loud beeping sound in a variety of different scenarios, such as adjusting the volume, pressing an incorrect key, or even during some normal system operations. jenicel dri say:
Hi Caroline,


"We can’t sign into your account" error occurred along with the release of Windows Anniversary Update. Windows 10 locks you out of your account by delivering a pop-up window, which simply states the fact that Windows can’t sign in. To resolve the issue, we suggest you follow troubleshooting listed below:

Option 1. Create a Temporary Profile

1. Turn on your PC and until the Windows booting screen shows up.
2. As soon as it appears, press Restart button.
3. After that, select Troubleshoot and then click Advanced Options.
4. Opt for Startup and press F4 to open Safe Mode.
5. Then wait until your PC restarts and boots into Safe Mode.
6. During the startup, click Windows key and press R simultaneously.
7. Type regedit in the search box and click Enter to open Registry Editor.
8. Here you should go to the following: HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\ProfileList.
9. Expand ProfileList and you will see a few subfolders named by values starting S-1-5-XX (XX could be any number).
10. You have to select the one that has the ProfileImagePath key set to the system profile path.
11. Click twice on the RefCount key and change its value to 0 from 1 and click OK.
12. Close everything and restart your PC.

Install Windows updates

1. Navigate to Settings App > Select Updates & Security > Windows Updates.
2. Select Check for Updates option and wait until the system scans for available options.
3. As soon as you get the list with all Windows updates that you are recommended installing, click Run the Updates.
4. After all updates are installed, reboot your PC.

Option 2. Launch On-Screen Keyboard

Follow the steps below to launch an on-screen keyboard.

Step 1. Enable On-Screen Keyboard

1. Click on Ease of Access and choose Keyboard.
2. Move the slider to On-Screen Keyboard. In the Options tab, select the method of the information input.
3. With this, you can navigate through the system. Try to implement the following method.

Step 2. Log in as Administrator

With the On-Screen Keyboard on, reboot the device.
When the Windows logo appears, click CTRL+ALT+DEL keys and enter into the login screen.
Type Administrator in the user field and enter the password used during Windows installation.

Option 3. Check if “We can’t sign into your account” is not a scam

1. Press the Power button at the Windows login screen. Now press and hold Shift and click Restart.
2. Opt for Troubleshoot > Advanced options > Startup Settings and finally press Restart.
3. Once your computer becomes active, select Enable Safe Mode or Safe Mode with Networking in Startup Settings window.

Now you will be able to access the computer. This method will not help you remove all possible malicious files but it will help you solve the problem activating other functions.

Option 4. Solve the problem with Command Prompt

1. If you are able to access your operating system via Safe Mode, in the advanced options list, choose Enable Safe Mode with Command Prompt.
2. Follow these commands consecutively. Click Enter after each one.

• net user /add useraccountname mypassword
• net localgroup administrators useraccountname /add
• net share concfg*C:\\/grant:useraccountname,full
• net user useraccountname

Option 5. Use Registry Editor to regain access to your account

1. Click Windows key+R, type regedit, and launch Registry Editor.
2. Go to the following path:

HKEY_USERS\.DEFAULT\Software\Microsoft\IdentityCRL\StoredIdentities

3. In the location, find your account, right-click on it, and delete it.
4. Now close Registry Editor, and head to Settings > Accounts.
5. Add your account again.

Either of these methods will assist you in regaining access to your account. In case these methods did not help you solve the issue, corrupted registry entries might be the cause. Then you might need to install the operating system again.

Let us know if you need further assistance.
