a. user say:

Hello Robert,

We realize your requirement in this regard.

Windows Live Mail is part of Windows Live Essentials, a suite of free programs for photos, movies, instant messaging, and more. The programs are downloaded together, but can be uninstalled separately. To uninstall Windows Live Mail,

  1. Open Programs and Features by right-clicking on Start button and then select Programs and Features.

  2. In the Uninstall or change program list, click Windows Live Essentials, and then click Uninstall/Change.

  3. Click Remove one or more Windows Live programs.

  4. Select the Mail that you want to uninstall, and then click Uninstall.

  5. Follow the on-screen instruction to remove Windows Live Mail from your computer.

Note: To completely remove Windows Live Essentials, select all the Windows Live programs.
