Windows Explorer is the file management software component of Windows Vista (in some prior versions, it actually was called File Manager - bit that was a while ago).  Where you open it determines where in the directory you are placed (and what options show to the left side and in the center) - there's no set location or folder options for all entry points (it differs if you enter through Computer than if you enter through your username).  Here's a good article on what Windows Explorer contains, what is does, and how to do it: http://www.3street.org/windows_explorer/ and here are some related articles to that prior article: http://www.3street.org/windows_explorer/explorer1.html and http://www.3street.org/windows_explorer/explorer2.html.

Here's an article on the history of Windows Explorer and includes most prior versions, XP, Vista, and Windows7 http://www.bing.com/reference/semhtml/Windows_Explorer?fwd=1&qpvt=windows+explorer&src=abop&q=windows+explorer.  This might be useful information in your clasees.  Here's another article on Windows Explorer which explains its features and options (and includes links to other sites with good information): http://www.onecomputerguy.com/app_info/explorer.htm.  Here's an article on Windows Explorer which starts with XP but about half way through gets to Vista - it's a little more advanced than some of the others but it's my no means unreasonable and could provide valuable information: http://www.anvir.com/windows-explorer.htm.  Here's a final article on Windows Explorer with pictures that provides a lot of good information (though somewhat basic but important and including a few tricks): http://www.windowsvistauserguide.com/windows_vista_explorer.htm.


Windows Explorer is NOT an internet browser - that is Internet Exploerer (IE). Spirit gave you some good links on explaining that separate program so I won't add to them since your question was about Windows Explorer  Yes, the names are similar and they are both "Explorer" but they perform completely different functions.  You can download IE onto an XP operating system or a Windows7 operating system and they will work just fine - but Windows Explorer does not have a separate download function and is part of the Vista operating system (and the only way to reload it if something happens is to do a re-install (repair/upgrade or clean) from a Windows Vista Installatio Disk or a recovery disk or recovery partition provided by the manufacturer (which is essentially an image of the system once ready to ship).  The same holds true for the other versions of Windows (you must use THAT version's version of Windows Explorer only and there are no separate downloads).  You cannot search the web with Windows Explorer - that's the function of Internet Exploerer.  These two programs integrate well together and make Vista a much better and more functional program by both being present.  There are many 3rd party alternatives for Internet Explorer (e.g., Mizilla Firefox) but there are only a few no alternatives for Windows Explorer (and you can't uninstall Windows Explorer to use them).  Both programs come bundled with Vista (IE7 which originally came with it has since been upgraded to IE8) and the original programs cannot be re-installed separately or together if someting happens to them - the entire OS must be re-installed (or the problem otherwise fixed).
