canadian tech say:

That SPAM identifier is usually placed there by the receiving email provider.  In other words:  You are using email provider A, you send an email to a recipient that uses email provider B.  Email provider B, using SPAM detection software, decides this particular email is SPAM and then places that label on it.

This could happen even if you both use the same provider.

There are also SPAM fighter programs used by email providers that can label any and all email from a particular sender as SPAM.

So you need to do some detective work to find out who your recipients email provider(s) are.  If you can pin it down to one particular one, then you need to get your recipient(s) to speak to their provider.

Email providers could be Gmail, Yahoo, or a local Internet Service Provider.

I, once wanted to this same task and on a website it was given that using a hex editor it was possible to convert a dynamic disk to a basic one - I tried it but now Disk Management would show my disk as unavailable - something I would, as an advanced user really hate. So I learnt it the harder way - had to reinstall Windows to get my disk back to basic.

Now regarding your question,  SHOULD I CONVERT BACK TO BASIC ONE that totally depends. A downside of dynamic disk is that only the active partition can boot from the hard drive. That means you can use Windows but if you, for example need to access HP System Recovery from F11 at start-up - you cannot as the Recovery partition cannot be booted into; it isn't the active partition.

You must not trust any software allowing you to convert a dynamic disk to a basic one.
