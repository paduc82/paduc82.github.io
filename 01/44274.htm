Since the introduction of Windows Update as part of Windows, Microsoft has delivered updates as way to keep Windows updated and secure. This has always been the way the company keeps commercial versions of Windows on the market current.

frederik long say: Here are the usual methods to resolve this issue:
  • Reboot the machine with your Windows Repair CD or flash disk, then use System Restore to set Windows back by a few days (works only if the problem occurred recently).
  • Log on under an alternative admin account, then reset the password to your usual account (works only if she set up an alternative admin account).
  • Log on as Administrator in Safe Mode, then reset the password to your usual account (works only if she enabled the Administrator account and know its password).
  • Insert your USB flash disk with the password keyfile (works only if she created such a keyfile).
