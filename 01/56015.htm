jeffrey roach say:

The Surface 2 run Windows RT (Windows on ARM) and can install apps from the Windows App Store. It also includes Office 2013 Home and Student (Word, Excel, PowerPoint, OneNote, Outlook).


It would not be able to install applications like Visual Studio or other Win32 x86 Desktop Applications.


If you primary reason for choosing the Surface 2 is a full x86 experience you will be disappointed. If you want to use Office, Browse the Web, Email and Apps on a All Day Battery Life Machine that is more secure than x86 Windows then the Surface 2 will be great.


The Surface Pro 2 is a full laptop in Tablet Form. mcgee man say:

Hi, 

The easiest way to get your files to sync from your Surface Pro to your desktop and vice versa is using OneDrive. To use OneDrive, you need to sign in with a Microsoft account.  Depending on how much files you have you might need an Office 365 subscription because you'll be given 1 TB (1000 Gigabytes) of storage, which is a lot for files. Whereas a Microsoft account offers only 5GB for free (Or you can purchase the OneDrive subscription for 50GB for $2/month).

Here is the easiest way to get started.

Make sure you do these steps for both computers.

1) Open "File Explorer". A window will pop up that will show all your different folder on your computer "Documents, Downloads, Desktop, etc..." ("File Explorer" should be on your "Task bar" and the icon looks like a yellowish folder. Just left click on that).

2) Right click on OneDrive then click "Settings". When you right click on OneDrive (It should have a blue cloud symbol, another little window will open up and you can just left click on "Settings" (Look at my screenshot below).


3) Click on "Auto Save" tab then click on "Update Folders". (Screenshot below).

4) Click on "Start Protection"

What happens now is those three main folders where you store data will start syncing with OneDrive and as long as your signed in to your Surface Pro (or any other Windows 10 PC really) with the same microsoft account, those 3 folders will sync. (This will sync with onedrive.com or the even mobile OneDrive app!)

Obviously there's many other ways to have OneDrive sync you files but i personally think this is the fastest and most accurate steps. I perform these exact steps on all my clients i work with if they request this solution. 

*Note: Programs will not sync, only data will sync.

If you run into an errors or have questions, just reply to my post. I reply quickly since i receive notifications on my personal cell phone. please give me a thumbs up! It truly helps me out.

-Michael.

  2. If youre still having problems, your printer may need a new driver. Right-click (or press and hold) the Start  button, and select Device Manager. Expand Printers, find your printer in the list, right-click (or press and hold) it, then select Update Driver.

If the steps provided did not work, you may need to clear spooler files and restart the spooler service.  

  • In the search box on the taskbar, type Services, then select the search result of the same name.

  • In the Services list, double-tap (or double-click) Print Spooler. Select Stop > OK.

  • In the search box on the taskbar, type %WINDIR%\system32\spool\printers, select the file folder in the search results, and then delete all files in the folder.

  • Search for and open Services again

  • In the Services list, double-tab (or double-click) Print Spooler.

  • Select Start, and in the Startup Type list, make sure that Automatic is selected, and then select OK.

lemp say:

You research errors like this by doing a Google search for the file name.  Usually they turn up no hits.


When a search results in either no hits or suspicious hits, the answer usually is that the file in question is malware-related and your anti-malware application has deleted or quarantined the file but left behind the Windows registry entry that attempts to start the malware when you start up your computer.


In your case, however, there were quite a few hits for hkntdll.dll.  Based only on what I've seen, my best guess is that it is related to a hot-key driver for your keyboard (if you have special keys on your keyboard, try them: if they don't work, that's a good indication that the file is the hot-key driver and was removed from where it was supposed to be.

If you want to get back the operation of your special keys, I suggest that you scan the file with your antivirus application and/or upload it to http://www.virustotal.com/ where it will be scanned by a collection of A/V applications.  Assuming that the file checks out as OK, put a copy of it in C:\Windows\system32 -- that's where most device drivers are located.


In addition, just for safety's sake, download, install, update, and run full scans with both of these free scanning tools:


MalwareBytes AntiMalware
SUPERAntiSpyware

Don't run the scans simultaneously.  They may take a while, so start one up and then go do a non-computer chore for a while.

If you don't care about the functionality of the keyboard (or the file is reported to contain malware) and you want to get rid of the error message, then download and run Autoruns.  Click the "Logon" tab and look in the "Image Path" column for the dll named in your error message.  If it isn't shown in the Logon tab, click the "Everything" tab and use the File > Search tool.  An idiosyncrasy of the Autoruns search tool is that you have to move the selection line below the "found" item before you "Find Next."

Once you find the line that refers to hkntdll.dll, either "Delete" it or (if you want to be safer) uncheck it.  The latter prevents the line from being executed but you can reactivate it later by removing the checkmark.
