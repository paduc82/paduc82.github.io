To add to XP ROCKS! good advice :

The most common problem with a new home build is a loose card and/or connection. Remove
ALL power then reseat the cards, memory and cables (on-both ends when possible) - actually
remove and replace - do not just snug.

Memory tests don't catch as errors such as the tiny speed differences between memory modules
that appear to be identical. So it may be necessary to swap the modules in and out in different
combinations and patterns. Faster memory always has to be installed in front of slower memory.

Be sure to check with MSI Support, their on-line documentation and drivers, and ask in their
forums about any known issues. Update the BIOS, low level chipset drivers, and the major
on-board and separate 3rd party device drivers.
