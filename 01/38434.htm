Escape from Safe mode boot loop using an installation disk

A  To make an installation disk to boot from use http://windows.microsoft.com/en-us/windows-10/media-creation-tool-install  Follow the instructions - Using the tool to create installation media (USB flash drive, DVD or ISO file) to install Windows 10 on a different PC [even though it may not be not a different PC].  Make the installation disk on an 8GB USB [you can also choose to make it on a DVD].  Everything else on the disk will be wiped off so don't use one with any useful files on it.

B  You will need to boot from that disk [it is also a repair tool as well as an installation tool].  Just in case you have never booted from a USB/DVD before -

  • Remove any other USB drives first because they can complicate things,
  • Connect the USB or DVD you made into the installation disk,
  • When the PC starts & whilst the maker's logo is still onscreen [before the Windows symbol comes up],
  • It will probably tell you onscreen which key to press to get to Boot options, press that key
  • [if the screen does not tell you which key to press then you will have to check your computer manual or the computer maker's website to find out because different makers use different keys],
  • In Boot options, select USB if you made the installation disk on USB or DVD if that is what you made it on,

1  That starts it booting from the installation disk,  

2  You'll see a Windows icon at some point as it starts up & it will take longer than your usual boot,

3  You will then see a dialog asking you to choose the language, time-currency & keyboard region,

4  Then click on the button in the bottom right [I think it is labelled Next],

5  Then you'll be taken to the main Install page but do not choose that - click on Repair your computer instead,

6  Then you'll see the Choose an option screen,

7  Click on Troubleshoot to go to the Troubleshoot menu,

8  Click on Advanced options,

9  Click on Command prompt,

10  Enter this     

BCDEdit

11  You will get a response like this -

12  Look at the line that says Windows 10 then look up three lines to the "identifier" where mine says {default} [and yours probably will too],

[If you are worried by the entry here because it is just a string of numbers and letters then that is probably okay but there is nothing to stop you from halting the procedure, asking for help then starting again.  What is important is not entering the wrong thing.  If you do ask any questions about this entry, take a photo of it and paste it in your question so I can see what is going on.],

13.1  If yours does also say {default} then enter this [including its brackets, just as shown] 

bcdedit /deletevalue {default} safeboot

13.2  If yours does not also say {default} then enter whatever it does say inside the curly brackets [including its brackets, just as shown]

15  Close the Command prompt [The menu screen might then close automatically but if not you can just click on the back arrows until you reach the Choose an option screen where you can click on Turn off your PC - you can even just turn it off using the power button],

Invisible Window on Desktop (Windows 10) - Microsoft Community

wynter nyte say:

Invisible Window on Desktop (Windows 10)

I actually wanted to post an answer to my question that I was able to resolve.

The problem: The problem I kept having was this invisible window would show up after a windows update. It was obviously a program window, however, I couldn't move it or resize it or close it. There wasn't a program process showing up in Task Manager. Just this window. I did all of the suggestions of starting in safe mode etc. to try and figure out "What freaking program is running??!??" so I could shut it down. After many attempts and frustrations and searches all over the internet, I finally found out a possibility. For me this was the solution. I will post a screenshot of my desktop with the invisible window for reference. 

The Solution:  For me the solution is my Nvidia GEForce Experience is trying to install or update new drivers. However, it doesn't show up in your Task Manager as the opened invisible window. Just open your GEForce Experience program. Update your drivers and restart your computer. The window should no longer be on the desktop. And you don't need to go insane trying to figure out what program the invisible window is running. Now you know. It's your graphic card just trying to update your drivers. Good Luck for those of you who have had this problem and couldn't find the answer easily. 

As you can see the outline of the window once I clicked on it. It's crazy invisible if not clicked.
