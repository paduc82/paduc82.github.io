 You are going to need an external drive.

Windows Easy transfer to transfer your files and settings to Windows 7

http://www.microsoft.com/en-ca/download/details.aspx?id=7349

This will create a large file on your main drive.  After it is done, you will need to move this file to an external hard drive so you can transfer to the Win7 system.  You can create the file on the external hard drive directly IF AND ONLY IF, there is nothing else on that external drive.

Backup your data too.  Win Easy Xfer almost always works, but you want to be sure.

 

Go to www.support.dell.com put in your Dell Service Tag (a 7 character number/letter string on your computer).  Go to device drivers.  See if you can find drivers for Win7 or Vista.  Vista drivers almost always work for Win7.  Download all the drivers you can find there.  Don't panic too much if you can not find them all.  Windows 7 install is very good at providing drivers and it could do them all for you. 

After Win7 install is complete, your next step should be Windows Update with other Microsoft software.  Sometimes it will offer drivers as well.

 

Please do not even consider ANY of those driver download sites.  They are almost all malware themselves.

 

While your XP system is still up and running, you want to list the detailed descriptions of the devices in your computer so that if you need to search for a driver later, you know what to look for.

 

When you are installing Win7 do NOT put the activation code in.  You want to wait till after the install and you have made certain all the drivers are in place BEFORE you spend your money.

frederik long say:

I suggest you do this:

  1. Modify the first two lines of the script further down to suit your requirements.
  2. Save the script as Filemonitor.vbs in a suitable location.
  3. Invoke the script from a Command Prompt. It will continue running in the background until you press Ctrl+C.
  4. Add a file to the target folder. Within 10 seconds you should see the result in the log file.
  5. Add the script name to your startup or logon script so that it runs automatically.

sTargetFolder = "D:\Test"
sLogfile = "D:\Log.txt"

Set oFSO = CreateObject("Scripting.Filesystemobject")
sFldr = """" & Replace(sTargetFolder, "\", "\\\\") & """"
Set oWMIService = GetObject("winmgmts:{impersonationLevel=impersonate}!\\.\root\cimv2")
Set cMonitoredEvents = oWMIService.ExecNotificationQuery _
    ("SELECT * FROM __InstanceCreationEvent WITHIN 10 WHERE " _
     & "Targetinstance ISA 'CIM_DirectoryContainsFile' and " _
     & "TargetInstance.GroupComponent='Win32_Directory.Name=" & sFldr & "'")
Do
    Set oLatestEvent = cMonitoredEvents.NextEvent
    Set oLog = oFSO.OpenTextFile(sLogfile, 8, True)
    oLog.WriteLine Date() & " " & Time & " " & Replace(oLatestEvent.TargetInstance.PartComponent, "\\", "\")
    oLog.Close
Loop
