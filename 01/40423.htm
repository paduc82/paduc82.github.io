Want to restore data from a monthly backup done to DVD, and must - Microsoft Community

joshc_la say:

Want to restore data from a monthly backup done to DVD, and must rebuild the mediaid.bin. How to do this? UPDATE

I have been restoring individual files from a set of backup DVDs/CDs done over a period of two years with Windows Backup and Restore on a machine running Vista Home Premiun 32-bit.  The backup set consists of one initial full backup and the rest are monthly incrementals.  The Vista computer has gone down, and I am restoring individual files and folders to a new laptop running Windows 7 Professional 64-bit.  It is mostly going well, but one DVD labeled Josh-PC 12/26/2009 2:14 PM Disk 1 will sometimes not allow Restore to complete successfully. 

While I have found many posts dealing with the error message I am getting (apparently the root cause is a corrupted mediaid.bin file on the DVD in question), as well as the posted solution, it seems to me the solution as it is written deals with backups done to a second hard drive or network drive, and I don't know how to apply it to backups done to CDs/DVDs.  If anyone can look at the following error and solution and see a solution for me, I would be genuinely thankful.

 

error that I get:

 

The restore did not complete successfully

Restore encountered a problem while validating the zip file:

\\?\Volume{22b11dfc-da4e-11e0-8b0a-806e6f6e6963}\JOSH-PC\Backup Set 2009-10-26 151232\Backup Files 2010-01-30 155242\Backup files 3.zip

The following file could not be found. Make sure the drive that the backup is saved on is attached or try to restore from a different backup. (0x8100001A)

 

the solution I found (from reading this post  http://social.technet.microsoft.com/Forums/en/w7itprogeneral/thread/b473594a-fea6-47b8-976f-f196489f12b8 ):

 

If you get the following error when restoring your backup to its original HD (see error code bellow)

Error Description:

When you use the Restore Files option to restore a backup file on a Windows Vista-based computer, you may receive the following error message:

The restore did not finish successfully. Error code:
The backup file could not be found. Check your hardware configuration or restore from a different backup (0x8100001A).

and the Mediaid.bin file is corrupted or lost, then use the following instructions for rebuilding it:

1.Unzip the sdutil.zip file and rename it from sdutil.ex_ to sdutil.exe
2.Copy the sdutil.exe to the root of the driver which contains the backup files
3.Open the folder Catalogs located bellow the folder BackupSet “aaaammdd” “xxxxxx”
4.Copy the file GlobalCatalog.wbcat to the root of this HD
5.Go to Start -> Execute and type CMD.exe
6.At this “DOS” window type: sdutil mediaid GlobalCatalog.wbcat [HD letter (as an example D:\)]
7.A new MediaID.bin is created
8.Try to restore the backup as usual using the advanced restore option and next the option restore an older backup created in this machine
9.If done: it’s over!
10.If it fails: Map a network driver at your LAN
11.Copy the MediaId.bin file to the root of the previous mapped LAN driver
12.Copy the whole folder BackupSet “aaaammdd” “xxxxxx” to this same driver
13.Try to restore the backup as usual using the advanced restore option and next the option restore backup created in another machine
14.At the next menu, point the backup folder at your LAN mapped driver and start the restore process
15.Now you shall have your backup proper restored at the right machine
16.Good Luck!

PS: you can find the sdutil file at:
