I'm not sure that this is possible with Surface RT.

 

Ordinarily, from a standard Windows 8 device, you could install the SkyDrive desktop app, and it would create a folder on your PC that would automatically sync its contents into the cloud, whether they be pictures from your camera or otherwise.

 

But Surface RT cannot run the SkyDrive desktop app. It can only run the version of the SkyDrive app that is in the Windows Store, which is (in essence) a front end to the SkyDrive website. You can manually upload contents to SkyDrive with this app, but you won't get the synchronized folders.
