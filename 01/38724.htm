frederik long say:
I want my hotmail to stay in English but I want to be able to type emails in Spanish and have a way of inserting the diacritical marks over a,e,i,o, and n. The keyboard shortcuts I set up to do this in word don't work on hotmail. Is there a way to add those symbols when I type emails? Remember, I want my hotmail language to stay English (till I learn more Spanish.)

Here is a simple method:
  1. Open the Control Panel / Region and Language.
  2. Install a Spanish keyboard.
  3. Ensure that the US keyboard remains your default keyboard.
  4. Ensure that there is a hotkey-combination to switch keyboards, e.g. Left-Alt+Shift.
  5. Create a new shortcut on your desktop, pointing at osk.exe (=On-Screen Keyboard)
When you type something in Spanish, do this:
  1. Double-click the osk-shortcut. The on-screen keyboard will appear.
  2. Type your English text.
  3. When you need a Spanish letter, e.g. ñ, do this:
    - Press left-shift + left-Alt.
    - Briefly click an empty spot on the on-screen keyboard.
    - Move the cursor back to the mail item. The on-screen keyboard will now show the Spanish layout.
    - Click ñ, for example.
The process looks long-winded at first but you get used to it quickly. If you want something faster then I recommend you use the Windows Keyboard Creator. It takes more time to set it up but it lets you create key assignments like these:
Alt+a = à
Alt+e = é
Alt+i = í
Alt+o = ó
Alt+n = ñ
Post again if you need instructions.
