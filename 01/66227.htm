foxfifth say: The Kindle Fire uses an Android operating system and the Homegroup is for networking Windows 7 computers only.  Are you just attempting to connect the Fire to the same wireless router that your Windows 7 uses?  If so, you will need your wireless security key not your homegroup password.  (I don't know whether it is possible to have a Kindle Fire join a Windows network to share files by using a workgroup other than a homegroup, but it is possible to connect to the same router to access the internet.)

There can be confusion over multiple different network & router related passwords (e.g., Homegroup password, router administration password, and the wireless network security key which can also go by other names).

If you are just trying to connect to the wireless router, you can obtain the security key either from the router itself or if you have another Windows 7 computer connected wirelessly, you can get it from that computer by clicking on the wireless icon on the taskbar at the lower right of the screen and then right-click on your wireless name and click on Properties and then on the Security tab click in the box to the left of "Show characters" below the "Network security key"

If you need to get the key from the router, how to do that varies by manufacturer.  In the following link in the section "Set up the router" if gives some information about how you can get to the router's configuration page and from there you might browse around for the wi-fi settings portion and look for the security key (or better yet, look for the manual that came with your router if you think you might have it and follow the directions there about how to display the security key):  http://windows.microsoft.com/en-US/windows7/set-up-a-wireless-router

A site that has the factory default passwords to access the router is: http://www.routerpasswords.com/

I have had a case where a Kindle reader (not a Fire) would not connect to the router even though the correct key was entered and that was resolved by rebooting the router and the Kindle.  Amazon telephone support was helpful in resolving that issue.
