a. user say:

Thank you for the response. There is a possibility that the maximum usage of memory is not selected on your system configuration, and your BIOS don't allow your computer to use the maximum memory installed. To troubleshoot this, please follow the steps below:

Method 1: Enable Maximum memory

  1. Press Windows key + R, and type in msconfig then hit enter.
  2. In the System Configuration window, click Advanced options on the Boot tab.
  3. Click to clear the Maximum memory check box, and then click OK.
  4. Restart the computer.

Method 2: Update the system BIOS

  1. To update the BIOS on your computer, visit the Web site of your computer manufacturer to download the BIOS update. Or, contact your computer manufacturer for help.

Method 3: Memory remapping

  1. Memory remapping will allow your computer to use more than 3GB of physical memory (RAM). Please make sure that you are using Windows 10 X64 because Windows 10 X86 can't support memory more than 4GB. You can enable the memory remapping feature in the BIOS by booting to the system setup. See the User's Guide for your computer for instructions on how to boot to system setup on your computer. The name for the memory remapping feature may be different for different hardware vendors. This can be listed as memory remapping, memory extension, or something similar. Be aware that your computer may not support the memory remapping feature.

Method 4: Change the AGP video aperture size in the BIOS settings

  1. Check the BIOS settings to see how much memory that you have allocated to AGP video aperture. This is the memory that the system is sharing with the video card that is used for texture mapping and rendering. This memory would not be used by the system, because it is locked by the video card. You can adjust the AGP video aperture size in the BIOS. Standard settings are "32MB,""64MB,""128MB,"and "Auto." After you change this setting in the BIOS, restart your computer, and then check the usable memory. You can test each setting to see which offers the best results.

Uninstall the AMD HD audio if you are not using HDMI for your sound.

----------------

Below are several configuration options for when using both speakers and headphones -

- In the Device advanced settings option if you select:

Make front and rear output... simultaneously, the speakers will be disabled when the

headphones are plugged in, and the speakers will function when the headphones are unplugged.
- This option also allows you to set-up the configuration (Sound Effects, etc.).  for the headphones  to be different from the speakers.
(The option  Mute the rear...plugged in  does not offer an option to set-up a different configuration
for the headphones from the speakers, but works the same otherwise).

Make front and rear output... simultaneously the speakers will not work if you unplug the headphones.

operate simultaneously, and when the headphones are unplugged the speakers will still function.

R/click on any of the Devices listed and tick Show Disabled Devices and Show Disconnected Devices.
