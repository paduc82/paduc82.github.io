patricia yu say:

Hi Daniel,

The time out that you're experiencing might be due to Session Time Limits policies on your PC. Try disabling active Session Time Limits policies and check if it helps.

Here's how:

  1. Access Run by pressing the Windows key + R.
  2. Type gpedit.msc, and then click OK.
  3. Navigate to Computer Configuration\Administrative Templates\Windows Components\Remote Desktop Services\Remote Desktop Session Host\Session Time Limits.
  4. If there are any active policies, disable it.
  5. Restart your PC.

Let us know how it turns out.
