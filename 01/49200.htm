You should only obtain the latest Drivers for them from the actual Hardware Manufacturer's website, as Microsoft normally does nor have those latest Drivers available as Microsoft relies of the 3rd Party Hardware Manufacturer to supply them.

"Remove an Update - Windows 7"

shinmila h say:

Hey Ardath Rekha,
 
Here's a little insight on how the two libraries work together.

Windows 7 makes use of Libraries to give a consolidated view of related files in one place. Windows Media Player (WMP) 12 integrates nicely with the new Libraries in Windows 7, so all of the music contents you see in Windows Media Player i.e your "music library" will literally come from your Music library in Windows explorer. And so is the case for Videos and Pictures. When we add any location to Music, Videos, or Pictures library in Windows Explorer, those locations get added to the WMP Library since Windows Media Player 12 shares the same configuration of library location as that of Windows 7 Libraries.

You may however change the way WMP library gathers its contents from the computer.

Here's how:
 
1. Go to Organize > Manage Libraries.
2. Select any one of the library which you don’t want Windows Media Player 12 (WMP 12) to monitor.
3. Right-Click on the library location and select Remove.
   Do this for other libraries.
4. Next, click Organize again and click 'Apply media information changes'.
 
Additionally, configure the following settings.

1. Click the Start button, click All Programs, and then click Windows Media Player.
2. If the Player is currently open and you’re in Now Playing mode, click the Switch to Library button in the upper-right corner of the Player.
3. Go to Organize -> Options -> Library.
4. Uncheck everything except for "Maintain my star ratings as global ratings in files".

Once this configuration is done, see if you notice any change in the way WMP behaves.

For additional information, see:
 
Windows Media Player Library: frequently asked questions
http://windows.microsoft.com/en-us/windows7/Windows-Media-Player-library-frequently-asked-questions

Regards,

Shinmila H - Microsoft Support
