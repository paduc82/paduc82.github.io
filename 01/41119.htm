Hi,

 

Thanks for the update. There is no need to search for calculator each time from start screen. You can try these steps to easily access calculator.

 

Step 1: Pin the calculator shortcut on Windows 8/8.1 start!

 

  1. Please press the key Windows-Logo and type the text "calc", 
  2. Use right click on the Calculator and select "Pin to Start".

 

Step 2: There are times that you need to activate the Windows Calculator quickly. Instead of navigating through the Start menu, you can setup a keyboard shortcut to the application.

 

  1. On an empty area of the Windows desktop, right-click and select New > Shortcut.
  2. When the 'Create Shortcut' dialog box appears, type "calc" (without quotes) and press the Next button.
  3. Press the Finish button to create the new shortcut.
  4. Right-click the new Calculator icon, and select Properties.
  5. When the file properties dialog appears, click in the box next to the 'Shortcut key'.
  6. Type in your desired shortcut. For example, press CTRL-ALT-C.
  7. Press the "OK" button.

 

Now all you have to do is press the shortcut key you created, and the Calculator will appear. If you want to delete the keyboard shortcut, open the file properties dialog box click the box next to 'Shortcut key', and press the Delete key.
