See DaveM121's reply and let me add the following:

You did not delete Edge from the desktop; what you deleted was a *shortcut* to Edge. DaveM121 told you how to put a shortcut to Edge on the desktop, not how to put Edge itself there.

If you are unfamiliar with the term "shortcut," let me explain that it is a "pointer" to where a program is, not the program itself.  Shortcuts are almost always the only things on the desktop, and the only things you should put there. That's because if you delete a shortcut, it's easy to put it back (again, see DaveM121's reply), but if you delete a program, it can be much harder to get it back.

elrumbo say:
Chopper,

We are very close to resolve this problem. With the info that you gave me I can say very confidently that my "guess" was correct. C: is your original drive and you're booting up to that partition . D: was a second partition created and where you probably loaded the restore install of Windows. What you basically need to do now is to delete the second partition D: and reclaim that space for your C: partition. Windows 7 by itself could do that if the D: partition is to the right of the C: partition and if the partitions are not marked as system or bootable which in your case they probably are. Another option is to use a free third-party partition manager like EaseUS. I have used that one and it works like a charm. The only problem that I see with this program is that you have to install it on your computer and you really don't have much space there (2.4 MB). In this case the only viable solution is to make a bootable USB stick with a program to manage partitions (GParted).

- Have a blank USB stick formatted as FAT32 ready.
- Download the ISO of GParted here (This is the actual partitioning program).
- Download TuxBoot here (This is the executable to copy the contents of an ISO to a USB and make it bootable).
- Hook up your USB stick to your computer.
- Run the tuxboot-windows-25.exe that you just downloaded.
- The only change you're going to make is to click on predownloaded ISO, browse for the gparted-live-0.9.0-7.iso (as up to day, that was the latest version).
- Click OK and let the program do its work.
- Boot up with this USB on your daughter's computer.
- Just go with the defaults.
- Identify your partitions (Be extra careful here. You probably have a couple of hidden partitions: one to recover to a factory state and the other the Q: drive that you mentioned which by the way it's a partition that usually is created if you have the free Office version).
- Delete the extra partition (labeled OS).
- Expand the "Recovery" (C:) partition to the unused space (deleted partition).

Note: Pay close attention to the partitions. You don't want to delete or annex the wrong one. Finally, before doing any changes to the partitions backup what you still need or want from the OS partition.

obsessive power say:

Hi bc fox.  There is a known issue with Blood Money and PAL 60 with HDMI Component and VGA connections. To play this game in HD at PAL60 you'll need to do the following fix: 

You'll need a composite video cable - your console should have come with this SD video cable out of the box - Turn your console off, disconnect your current HD cable (what ever that may be), insert the SD composite cable.  Turn on your console, go to System Settings > Display.  Change the setting here to 60Hz.  Turn off your console, connect back up with the HD lead, turn on your console and try your game again.  All being well it should now work.

NOTE: If you have your console connected to a PC monitor, you need to do this fix using a TV first, then you can hook your console back up the PC screen later.

Bear in mind you don't need to do this again, the game should now work straight away from now on without any further issues.

obsessive power say:

UPDATE: If you are connected with a Component HD cable, you don't need disconnect anything. Turn off your console, find your Component cable, there should be a switch on it saying SD/HD.  Flick to 'SD' change the display setting to 60Hz.  Turn off the console, flick the switch back to 'HD' and turn the console back on.  The game should now work,

Yes Windows 10 Pro does support Hyper-V and it is not significantly different from earlier versions.

1st you need to enable Hyper-V in Windows. Go to Control Panel - Programs and Features - Add or Remove Windows features and activate Hyper-V

hairyfool say:

Hyperthreading is not to do with Hardware Virtualisation so recheck the BIOS options for Virtual Machine Extensions being enabled. Hyperthreading is a feature where a single CPU core can execute two threads threads at once yet keep them isolated from each other so appear as two independent cores to the operating system. This function applies to all programs that support multiple threads and is not limited to Hyper-V.

You should have 10 separate Hyper-V services available if set up properly.
