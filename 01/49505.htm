soundersfan13 say:

A bit of background.  USB Video Class (UVC) is a standard USB communications protocol that camera manufacturers use when developing webcams.  If the manufacturer implements the standard, they can have a plug and play experience in all version of Windows since XP SP1 (but it the driver was immature then and didn't support UVC well).  The first real decent implementation of UVC in Windows arrived in XP SP2.  After that, the industry moved toward UVC.  Prior to UVC and during that transition period, many cameras implemented their own custom protocols and implemented custom drivers to convert their custom protocol into Kernel Streaming which DirectShow is built upon.  Most desktop video apps use DirectShow to manage the video streaming.  The first few cameras produced by Microsoft have custom drivers as they were developed during that transition period.  These camera include the VX-1000, VX-3000, and VX-6000.  Windows decided not to allow custom drivers in Store apps.  I'm not sure why but it may be because of the audio/video security added in Windows Store apps.  In Store apps the user must authorize the app to use the camera and microphone before the app can stream from them.  Perhaps that security cannot be applied to custom drivers.  Except for those 3 cameras from Microsoft, I believe all others are UVC.

Hi,

Back Up disc will not help to perform refresh. You need to use the Recovery Media or the installation media.

A PC manufacturer's recovery disk will only return the computer back to how it was when new.

A system (or disk) image is a complete copy of a hard disk partition. That image is contained in a single file. Providing that you imaged your system (usually C:) drive, it contains the boot files, operating system, programs and user data. If you run into problems that you can't fix, or your computer's been rendered unbootable, you can restore the image and your computer will revert back to how it was when the image was created.

A backup is usually a copy of important files such as documents, e-mails, photos, and music etc. saved to an external hard drive, memory stick, or CD etc. An image file can sometimes be classed as a backup because it's usually possible to extract individual files and folders from it.

A system image is likely to be bigger than a recovery disk because it also contains all the extra programs you installed and also your data files.
