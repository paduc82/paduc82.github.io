This is not the greatest fix, but it has worked for me.  I changed the adapter setting to not run on wireless-N anymore, and to only use b/g.  It works now, but it is a little disappointing to suffer a speed loss just to stay connected.

 

To lock Surface to 802.11 b/g band you can use 'Regedit'.
Open Desktop -> Search -> regedit
Open regedit. Select menu Edit -> Find...
Enter "AutoUse40Mhz" -> do search.
There are 2 such entries in registry. We need to change one, located under "HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Class\" branch.
Set "AutoUse40Mhz" to "0" (it has value "1" by default)
Set "Band" to "4" (it has value "6" by default)
Reboot and Surface will connect using 802.11 b/g band

 

Disclaimer: Do not mess with anything else in Regedit other than what is specifically listed in these steps.  Regedit is nothing to play around with, because it is very easy to screw up your whole system if you don't know the values you're changing.

Use Libraries to View OneDrive Folders and Local Folders - Microsoft Community

stanimal21 say:

Use Libraries to View OneDrive Folders and Local Folders

Technical Level : Basic

Summary

Since you cannot choose folders to sync to OneDrive with the desktop clients applications, learn to use Libraries to combine multiple OneDrive and local folders into a combined view. For Windows 8.1 and Windows 8 only. Windows 7 follows the same concept, however the details are a little different.

The scope of this article is for the personal version of OneDrive on Windows 8.1 and earlier versions. Please refer to the table below for additional support for the following products:

Product Support Link
OneDrive for Business Contact Office 365 for Business support
OneDrive on Windows 10 Windows Insider Program


Details

OneDrive is different from other cloud storage solutions because all content is synced through a single folder; you cannot choose which folders to sync like you can with Syncplicity or DropBox. This is a fundamental difference and some people mistakenly move their "My Documents" folder into the OneDrive folder; it breaks other functions in the operating system. To solve this problem you can use Libraries view the contents of multiple directories, regardless of folder locations, under a single view, almost like they're in the same folder. Libraries are not real folders; they're temporary folders, but that also means you can name them whatever you want. It's another way to organize specific directories under common genre's compared to directory structures.

How to Add OneDrive and Local Folders to a Library

As mentioned already, Libraries are designed to combine folders and files from multiple file locations under a single genre. For example, the "Documents" library by default includes the <user>\Documents folder and the Public\Documents folder. The illustration below shows the logical structure of the Documents library:

You'll need to modify the existing library to include the new OneDrive directory that you create (sometimes it's created automatically when you sign up for OneDrive): C:\Users\<username>\OneDrive\Documents. After you include the OneDrive "Documents" folder into the "Documents" library, the structure will look like this:

The first location is located under the OneDrive folder and the other is located somewhere else on the local file system. If you want your files in "My Documents" to sync to OneDrive, you must move them to the "Documents" folder, however, this is not ideal if there are some files you want synced and some files you want to stay local. Use a Library to combine both folders in the same view; we can use the pre-built "Documents" library to include both:

  1. Open File Explorer and then click on "Libraries" in the left-hand sidebar or press the Windows key and then start typing Libraries, then press enter when Libraries appears
  2. Double-click on "Documents" to open the view in File Explorer, click on the "Manage" tab, and then click on "Manage library"; this displays the folders already included in "Documents" library

  1. Click the "Add" button to add a new directory, browse to your OneDrive folder, and then click on the "Documents" folder inside OneDrive
  2. Click "Include Folder" to include the C:\User\<username>\OneDrive\Documents folder in the "Documents" library; a new entry appears in the Library locations list
  3. Right-click on any of those items and then select "Set as default save location" or "Set as public save location" to change which folder receives a file when you Save to a library (since a Library isn't actually a folder, if you save a file to a Library it saves it to the default save location)
  4. Right-click on any of those items and then select "Move up" or "Move down" to change the order in which the folders are displayed in your Library
  5. Click OK to Save your new list

You've successfully added a folder to the "Documents" library. You can customize the view more by clicking the "View" tab and then selecting "Group by" to group by different criteria or don't group at all, "Sort by" to sort the files by different criteria, and "Add columns" to view more information about directories and files. You can include any folder to any library by using this same process; there are no restrictions whether it's music, photo's, or other documents.
