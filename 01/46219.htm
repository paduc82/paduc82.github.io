¡firedog say:

OK. Copy this line, paste it into the start search box and press Enter:


%localappdata%\Microsoft\Windows Live Mail\Proof

Find the file called lang0809.lex, right-click on it and select Open with > Notepad. This is your custom spelling dictionary. Amend it as necessary, then save and close.

If you happen to have a custom dictionary for Office (by default it's called custom.dic), you can also open it with Notepad, copy its contents and paste them into lang0809.lex. This is very helpful for people who've compiled specialist custom dictionaries of, say, scientific or medical terms that aren't found in the standard dictionary. Other words it's handy to have are local place names, proper names and so on.

 

 

¡firedog say:
The feature you're describing isn't 'spell checking'; it's called auto-correct, or Automatically correct common capitalization and spelling mistakes. This will change i to I, teh to the and chicago to Chicago. Unfortunately, for English texts, it only works in US English, not British English (or Australian or Canadian).

There is a way of making it work using auto-replace, but that's very tedious and involves making registry entries. You also have to have a list of common errors with their correct versions.

pa bear - ms mvp say:

First see...

   • HOW TO Hide an Update or Restore a Hidden Update
     http://www.sevenforums.com/tutorials/24376-windows-update-hide-restore-hidden-updates.html

1. Press & hold the CTRL key.

2. Select (highlight) the top-most/first Language Pack.

3. Select (highlight) the bottom-most/last Language Pack then release CTRL key; all of them should be selected (highlighted) now.

4. Hide all of the updates.

canadian tech say:

If you just recently installed SP1, then I presume the operating system install date is a long time ago. If I had to guess, probably more than 7 years ago.  Pros, who use their Windows computers a lot, re-install Windows about once every 2 or 3 years.  You should consider doing this.
