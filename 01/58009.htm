Disable or password protect safemode - Microsoft Community

familybelanger say:

Disable or password protect safemode

I would like to password protect safemode.  Family members circumvent filters by operating under safe mode.  How do I do this?
frederik long say: I am not aware of an inbuilt method to prevent a reboot into Safe Mode but you can probably achieve your aim with a little effort like so:
  1. Create a Restore Point so that you can recover in case you get locked out too.
  2. Burn a Repair CD unless you already have one.
  3. Identify a service that does not run in Safe Mode. I think the Task Scheduler service is one of them.
  4. Place the batch file below into the following folder:
    C:\ProgramData\Microsoft\Windows\Start Menu\Programs\Startup
    Any name will do, e.g. Startup.bat
    You need elevated admin privileges to do this.
    The batch file will shut the PC down at logon time in Safe Mode unless you log on under your own account.
  5. Log into Safe Mode under an account other than your own to test the script.
  6. Make sure none of your users have accounts with admin privileges.
  7. Keep the passwords for all admin accounts secret.

@echo off
net start | find /i "Task Scheduler" 1>nul 2>nul && goto :eof
if /i "%UserName%"=="Administrator" goto :eof
if /i "%UserName%"=="XXX" goto :eof
shutdown  /F /P

Replace XXX with the name of your own account.

Note also that you might start a trench war. Whatever you do to Windows, you cannot prevent someone with the right skills to undo it if he/she has physical access to your machine. Instead of putting up physical or software barriers, you might want to discuss the matter with your family members at a personal level. Playing cat and mouse games won't help anyone.

Post again if you need more detailed instructions.
