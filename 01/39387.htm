If you backed up your data from Windows XP with Backup and Restore files and if you want to restore the file in windows 7 you need to download NTBackup Restore Utility for Windows 7.

 

Description of the Windows NT Backup Restore Utility for Windows 7 and for Windows Server 2008 R2

http://support.microsoft.com/kb/974674/en-us

 

You can restore specific files from a backup that was created on another computer running Windows XP. Follow these steps:

 

1. Download Windows NT Backup - Restore Utility

2. Run the file NTBackup.exe from the folder.

3. NTBackup will indicate that Removable Storage service (removed in Windows Vista & Windows 7). Never mind! This is only required for tape backups. Click OK.

4. In the Welcome to Backup or Restore Wizard page, click Next

5. Select Restore files and settings, and click Next

6. Click Browse. Locate the .BKF file that you want to restore, and click OK.

 

7. Expand the catalog, select the files/folders you want to restore, and click Next.

 

8. Click the Advanced button.

9. Select Alternate location from the dropdown, and mention the destination path:

10. Configure the restore options accordingly.

 

 

11. Click Next, and then click Finish.

 

For more information on Windows Backup refer these Microsoft Article:

 

Backup and restore: frequently asked questions

http://windows.microsoft.com/en-in/windows7/back-up-and-restore-frequently-asked-questions

 

Restore files from a backup
