I think you guys are making this way more difficult than it is. The reason why the entire users folder is shared is because you have the "Sharing Wizard" turned on in the file explorer settings. It basically keeps the folder structure of the shared folders similar to the normal file explorer structure. Otherwise, you'd have to share individual folders one by one such as \\MyProfile\Scanner, \\MyProfile\Music, \\MyProfile\Videos, etc. With the wizard on, it maintains the normal C:\Users\MyProfile\Music, Scanner, Documents, Videos, etc. available. For the most part, the wizard touches anything within the folder structure all the way up to C:\

Obviously this is still a safe method of sharing assuming that you have password protected sharing enabled... Otherwise, yes, everyone on the network can access every folder and file within that wizard enabled structure.

If you prefer the olden style of sharing where you had to enable each and every individual folder without any regard to structure or parent/child relationship, then simply turn the wizard off in the file explorer properties.

If you are the type where you want the public folders to be always available to anyone but not your personal folder, you will have to turn password protected sharing off since it covers the public folder. This means you will have to turn off the Sharing Wizard as well... That's where the "share public folders" or some variant of that depending on which windows you're running comes into play and keeps the folder structure for the public profile since the Sharing Wizard is going to be turned off so that you can maintain the folder hierarchy.

3. Click the Recording tab.
4. To test the microphone, speak into it. To test line-in, send a signal to the line-in port from the source audio device (press play from the audio player). Look at the gray bars next to the device name on the recording tab:
* If the gray bars next to the device name change to green, this device is active and an audio signal is being passed through the microphone or line-in port. The speaker volume for the microphone and line-in may be muted or turned down. Use the following steps to check the speaker volume levels for line-in and microphone.

2. Click Hardware and Sound.
3. Click Sound.
4. Select Speakers, and then click Properties.
5. Click the Levels tab.
6. Click the Muted button next to Microphone and any other line-in that is muted.

8. Click OK.

You may find that a front microphone port does not work, a recording device is missing from the recording tab, a microphone is not recognized when it is plugged in, or other problem. There may be updated audio drivers that can fix these types of issues. Use the following steps to make sure audio drivers are up to date.
1. Connect to the Internet.
2. Click Start, and enter device manager in the Start search field.
3. Click the plus sign, next to Sound, video and game controllers.
4. Right-click the name of the sound hardware and select Update Driver Software....

6. Windows checks a remote Web site to see if updated driver software is available.
* If an update is available, let Windows install the update.
* If an update is not available, skip to the next step.

8. Click OK to make the changes.

Windows 10 Pro and Enterprise editions have access to some special encryption features:

  • BitLocker - allows you to create encrypted volumes on drives.
  • Encrypting File System (EFS) - allows you to encrypt individual folders and files.

To use the Encrypting File System (EFS) feature, you can follow these steps:

  1. Right-click the folder you want to encrypt, then choose Properties.
  2. Click the Advanced button under the General tab.
  3. Check the Encrypt contents to secure data box.
  4. Click OK.

Note: These folders are only encrypted when stored on your hard drive, therefore you can't email them securely without encrypting the folders in a different way.

If you want to use the BitLocker option, you can post your query to the TechNet Forums for a more suitable answer.
