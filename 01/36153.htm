Hi,

Follow the steps mentioned below and try to lock the your homepage.


- Click Start => Run type regedit and click OK.
- Goto HKEY_LOCAL_MACHINE\Software\Policies\Microsoft\Internet Explorer\Control Panel
- Right Click at the empty place of right panel, select New => Dword value
- Name it HomePage
- Double click on the HomePage and enter 1 or 0 (1 = lock homepage, 0 = unlock homepage) as the value

- Next, Go to HKEY_CURRENT_USER\Software\Policies\Microsoft\Internet Explorer\Control Panel
- Right Click at the empty place of right panel, select New => Dword value
- Name it HomePage
- Double click on the HomePage and enter 1 or 0 (1 = lock homepage, 0 = unlock homepage) as the value

Once you finish all the steps above, close all your Internet Explorer and re launch it again. You will see the Home Page will be set according to your setting.

OK, so this must be an IMAP account, since the other types have folders called Inbox, Drafts, Sent items, Junk email and Deleted items. Here are some suggestions for how to configure an IMAP account. You may find on experimentation that different settings work better for you: 
  1. Reset the list of IMAP folders:
    Click on the account name in the folder pane and then press Alt-S F to open the Show/hide IMAP folders dialogue. Click Reset List to refresh the folder list, then use the buttons to show only the folders you want to see. Note the exact names of the 'special' folders.
  2. Map the local special folders to the IMAP folders on the server:

    On the IMAP tab, select Store special folders on IMAP server,  then enter the server's names for the special folders in the boxes.   3. Make Deleted items folders work the same way for all accounts:
    When viewing your message list, press Ctrl-Shift-O for Options.
    On the Advanced tab, select Use the 'Deleted Items' folder for IMAP accounts.
