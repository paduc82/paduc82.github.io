a. user say:

You should be able to get updated drivers for your device that are compatible with your version of Vista from Atheros Support http://support.atheros.com/.  I couldn't get any further than that because it required a username and password to proceed but I'm pretty sure they have them available for download (almost every device manufacturer has downloadable device drivers on their web sites).  I'd give them another shot.  You're very unlikely to find them anywhere else on the web so don't wast time trying to find them - you need to get them from Atheros.

Do you know when this problem began?  Try a System Restore to a point in time BEFORE the problem began.  Here's the procedure: http://www.howtogeek.com/howto/windows-vista/using-windows-vista-system-restore/.  Be sure to check the box to show more than 5 days of restore points.  If the first attempt fails, then try an earlier point or two.  NOTE: You will have to re-install any software and updates you installed between now and the restore point, but you can use Windows Update for the updates.  Use the recovery disk if the system prompt doesnâ€™t work. The recovery disk works a bit different from the above procedures but if you follow the prompts from the System Restore menu option with the above information you should be able to restore with no problems.

Go to Device Manager by going to start / search box and type device manager and enter and then double-click on the program icon that appears.  Check each device for a red x, yellow ! or white ?.  These identify devices with problems (probably drivers, but also conflicts or something else).  Click on each for further details and troubleshooting tips.  If you need to get drivers (and you do for the Atheros device), do so from the computer vendor or device manufacturer (NOT from Microsoft Updates).  In fact, you should turn off automatic driver updates from Windows Update as follows: http://www.addictivetips.com/windows-tips/how-to-disable-automatic-driver-installation-in-windows-vista/. Proceed as follows to get the drivers: http://pcsupport.about.com/od/driverssupport/ht/driverdlmfgr.htm.  Once you have the drivers, you can install them through Device Manager as follows: http://www.vistax64.com/tutorials/193584-device-manager-install-driver.html. Another possible option is to do a driver rollback on the Atheros device to see if a previously-installed driver exists and will get your device up and running again even if ti works differently and needs to be reconfigured  (though this is raking a step backwards - you really want updated drivers not older ones).  I would consider this as a sort-of "last resort" option if nothing else works and we can't find new drivers.

Go to Start / Network and Sharing Center / Setup a connection or Network and try to create a new WIFI connection from scratch (call it some other name).  Perhpas there's some problem with the configuration or system files associated with the current connection and setting up a whole new one may actually solve the problem (though I doubt it because the error says there's a problem with the Atheros device and not with your Vista netowrk setup - but it's worth a try).

Once again, your best bet is to contact Atheros and get updated drivers from them - I feel certain they are available even i if they are harder to get to than on most sites.

I hope this helps.

Good luck!

czirbel say:

I recently fixed updates failing by a simple registry edit.

Click start > run Regedit as admin > find HKLM\Components > rename the line that reads "pendingrequired" D_word to "pendingrequired=1"

On some pc's you will not find HKLM\Components.  The registry entry will be under Hkey\local machine > components
