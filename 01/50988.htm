andre da costa say:

If the Windows 7 license originally came with your Dell, you can't transfer it, because it is an OEM license.

If you had bought and installed the Windows 7 license separately and its retail copy, you can uninstall the key, then install it on the new desktop and use chat support to reactivate.

Click Start, type: CMD
Right click CMD
Click Run as administrator
At the command prompt, type: slmgr.vbs -upk

Hit Enter, this will uninstall the product key from the computer and set it back to trial mode, you are now free to use it on another computer.

After installing Windows 7 on the new computer, you will need to use chat support to reactivate over the Internet:

  • Activate Your Windows License via Microsoft Chat Support
