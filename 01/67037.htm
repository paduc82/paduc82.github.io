¡firedog say:

Synchronization will only happen if the mail account is configured to use DeltaSync over HTTP (i.e. Hotmail-type accounts) or IMAP.

  • For DeltaSync, you cannot decide which folders to synchronize - it's all or none.
  • For IMAP, it depends on the IMAP server. Right-click on the folder in the folder pane and select Synchronization settings. If you can select Don't synchronize, do so.

If you don't know which type of account it is, right-click on the account name and select Properties. The properties dialogue for a DeltaSync account has three tabs, for a POP3 account there are four tabs and for an IMAP account there are five.
