Steps to Disable the Built-in Administrator Account.

a)      Open Elevated Command Prompt from Windows + X Power Users Tasks Menu

b)      Type   net user administrator /active:no

c)       Close the elevated command prompt.

d)      The built-in Administrator will now be available to select to sign into.

e)      Restart the system.

shenan stanley say:

Just like with *nix, if you want a hardware device to communicate with the operating system - you need to install the proper drivers so they can speak to one another.

At this point you are trying to use a firewire (1394) connection to do your networking. ;-)

What you need to do is fairly simple - but will likely require the use of another machine (or a different OS already on the machine that works on the Internet) at least for the first downloads (the motherboard chipset and the network card driver.)

If you visit the Dell support & Downloads web site for that particular desktop computer you will see all the drivers Dell provides for the system using Windows XP.

Dell suggests you download and install the drivers in this order:

  • Chipset
  • Video Adapter
  • Network Interface Card (NIC)
  • Audio Adapter
  • Modem
  • Wireless Network Card
  • Touchpad, Pointer, Trackstick, Mice, and Keyboards
  • Other Devices


In this case, I suggest you download the chipset driver and network card driver first - get them installed in that order with whatever reboots necessary and then you should be able to configure the Internet and get the other drivers using the machine itself.
