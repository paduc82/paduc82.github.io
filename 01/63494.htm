Method 2A: Offline System Restore

Use this method when you have the Windows Vista installation CD or DVD. If you do not have the installation disc, go to method 2B.

To use offline System Restore, you must have a Windows Vista CD or DVD. Additionally, your computer must be configured to start from the CD or DVD drive. To create a new user account, we will need to have administrator privileges. In this situation, we will need to enable the built-in administrator account on your Windows 10. Kindly follow the steps given below:

  1. Click on Start.
  2. Type cmd, and then right-click on Command Prompt.

how do I reduce the memory usage of my computer? - Microsoft Community

phaneendra_01 say:

how do I reduce the memory usage of my computer?

I use a computer of windows 7 which has a 2 gb ram and its of 64 bit
cmdrkeene say:

64-bit systems and operating systems actually allow the use of -- and require more -- RAM than 32-bit.  It would boost your performance tremendously to invest in another 2 or 4 GB of RAM, giving you 4 to 6 GB total.

 

If you cannot do this, you can limit the amount of RAM required at any one time by limiting the number of programs you use at the same time.  Programs that are not running require no RAM at all.  You can also use the MSCONFIG program to disable programs that run in the background at startup.  Some of these programs are not required, such as "Adobe Speed Launch" or quick utilities to adjust your graphics options. If those aren't important to you, keeping them from running in the background will leave more RAM available for programs you ARE using.
