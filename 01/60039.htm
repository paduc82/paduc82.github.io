visageofanangel say:

Sigh... it's not possible to recover your data on the Xbox 360.

The 360 uses it's own type of file structuring called FATX where a PC would use FAT, FAT8, FAT16, FAT32, or NTFS and a Mac would use HFS or HFS+.

  • HFS/HFS+ allows for some great flexibility on how software can access the HDD but is poor for data recovery.
  • FAT/FAT8/FAT16/FAT32/NTFS is more structured and limits how software can run but allows for great data recovery because the marker in the FAT is removed saying what sectors belong to what files but the file is still there until over-written.  That is why some data recovery is possible.
  • FATX is almost a hybrid of the two prior systems and is inclusive to the Xbox 360.  It's proprietary technology owned and patented by Microsoft and has no tools designed to recover data because like HFS/HFS+ when data is deleted it is quite literally deleted.
Hopefully that explains things to you in the most lay way I know possible.
