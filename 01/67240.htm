a. user say:

User accounts can be explained in several ways but most people are only interested in two distinctions:

Administrator vs. Standard User

A user account with Administrator permissions (aka an Administrator account) has the ability ('permission') to make changes that affect the entire computer. Examples include installing software and changing Windows Update settings.

A user account with Standard permissions (aka a Standard User account) is only able to make changes that affect their own user account. Examples include the desktop background and what tiles appear on the desktop.

A PC needs just one Administrator account, for performing tasks that require Administrator permissions. There is no other use for this account.

Each user has their own Standard User account. This is the account you use for your day to day computing and this is the account where you store all your personal files.

Local Account vs. Cloud Account

A local account stores its settings on the computer that created it (the 'local computer.') A  cloud account stores its settings in 'the cloud' which is a fancy word for the internet. A Microsoft Account is Microsoft's version of a cloud account, and iCloud is Apple's version of a cloud account, and a Google Account is ... you get the idea.

Now we're getting to the fun part ...

An Administrator account can be a local account or a Microsoft Account, and so can a Standard User account. Looking at it from the other side, a Microsoft Account can be an Administrator account or a Standard user account. Are you getting it? These are two different categories being used together.

A Family Account is a Microsoft invention that nobody needs to have unless you have young kids. A Family Account is basically a Standard User account that family members can share. But family members can share any user account regardless of whether or not it's a family account. What makes a Family Account different is that you assign maturity levels to the family members, which is how you control your kids' use of the internet. But that's only if you want to use Microsoft's version of parental control. You can also get separate parental control software from third-parties.

Now, c'mon, wasn't that so obvious  ;-)
