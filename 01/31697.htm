greg carmack - mvp 2010-2020 say:

Try unplugging hard drive to test if media on DVD or flash will boot. 

Then try unplugging DVD drive to see if flash media will boot. 

Try removing a RAM stick, then swap RAM sticks, then slots.  You want to see if a stick or slot is blocking it.  You can also create the Memtest86 disk to see if it will boot and test RAM.  RAM - Test with Memtest86+

You essentially want to try every combination of hardware removed and then booting with DVD and flash. 

Try also enabling CSM and setting hard drive first to boot, booting media as a Legacy (not UEFI) device.  If it will boot as Legacy you can go ahead an install after deleting all partitions off the hard drive to reset formatting to MBR.  This can  mean you need a UEFI firmware update once you get to the desktop.

As a last resort you can move the hard drive to another PC to do the Clean Install Windows 10 and then move it back.  Windows 10 is more adaptive than past Windows and should swap out all drivers in a cascade you can monitor from the System Tray animation icon, requesting repeated restarts.

Subst allows you to convert a path into a drive letter

Open a Admin level command prompt

  • Right click the Windows (Start) icon, click on Command Prompt (Admin)
  • Accept the security Warning

subst x: C:\path/to/long/named/files

x is an unused drive letter, the path is to where the files are stored.

now try this to remove all dat files in that path.

del x:\*.dat

The command to remove the drive

subst x: /d
