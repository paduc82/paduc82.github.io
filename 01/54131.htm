Open Mobile hotspot settings (Settings > Network & Internet > Mobile hotspot)
Turn on mobile hotspot
Under "Related Settings", click Change adapter options
Take note of which adapter is connected to the internet and which adapter is your mobile hotspot (typically named "Local Area Connection* [#]")
Right click the adapter that is connected to the internet and click Properties
Go to the Sharing tab
Uncheck "Allow other network users to connect through this computer's Internet connection". If it is already unchecked, skip this step.
Click OK
Again, open Properties > Sharing for the same adapter.
Check "Allow other network users to connect through this computer's Internet connection"

Under "Home networking connection:", select the mobile hotspot adapter
Click OK

Right click Regedit then click Run as administrator
Navigate to the following registry key:

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\WlanSvc\Parameters\HostedNetworkSettings\

Delete the following key:
HostedNetworkSettings

Restart then try again.

Run the Network adapter troubleshooter

Open Start > Settings > Update & security > Troubleshoot
Scroll down, select Network adapters
Click Run the troubleshooter

When complete, restart to see if it works.

Press Windows key + X
Click Device Manager
Expand network adapters
Right click your adapter, click uninstall

Exit Device Manager then restart

This error message indicates that a device driver-almost always a video card driver-is stuck waiting for something (usually a hardware operation) to happen.

Things to Do:


1. Ensure the video drivers are updated to the latest version.

2. The system BIOS is fully up-to-date.
3. If both the video driver and the system BIOS are fully up-to-date, check with the manufacturer for recent driver updates.
4. As a last resort, try exchanging the video card.
5. Check Device Manager for  yellow exclamation (!) mark for all hardware devices. If there is any yellow exclamation mark then install latest drivers for the that particular hardware.

Also, ensure all the pending Windows updates are installed.

a. user say:

Trying to open Control Panel, User Accounts you may see a message like this:

Wrong number of arguments or invalid property assignment

Sounds like one of more of your DLLs and/or CPLs may have become unregistered for some reason.

Since it doesn't hurt to reregister a DLL that is already registered, just register all the suspects.

Click Start, Run and in the box enter (or copy/paste) the following BOLD commands one at a time and click OK after each:

    regsvr32   jscript.dll           (Message:  DllRegisterServer in jscript.dll succeeded)
    regsvr32   nusrmgr.cpl       (Message:  DllRegisterServer in nusrmgr.dll succeeded)
    regsvr32   /i   mshtml.dll    (Depending on your configuration, you may get an error, but that's okay)
    regsvr32   themeui.dll        (Message:  DllRegisterServer in themeui.dll succeeded)
    regsvr32   Vbscript.dll       (Message:  DllRegisterServer in Vbscript.dll succeeded)
    
Try your Control Panel, User Accounts now and report your results.

It never hurts to follow up peculiar system behaviors with this:

No matter what else you are using for malware protection, please do this:

Download, install, update and do a quick scan with these free malware detection programs:

Malwarebytes (MBAM):  http://www.malwarebytes.org/products/malwarebytes_free
SUPERAntiSpyware: (SAS):  http://www.superantispyware.com/

Be sure to select and choose to delete any threats, reboot, then scan again to be sure the threats have really been removed.

SAS will probably report a bunch of tracking cookies and you can just let it delete them.

SAS may install a startup item and launch every time your system reboots so if it is your desire to disable that, from the SAS splash screen, choose Preferences and uncheck the box "Start SUPERAntiSpyware when Windows starts" and the startup item will be removed from your configuration.

SAS may also create a Scheduled Task to run every day which can also be disabled or deleted if that is your desire.  To do that, choose Preferences, Scanning Control tab, Configure Scheduled Scanning and it may remind you to upgrade after 14 days. You don't have to do anything to keep SAS installed to continue using it with updates and run on demand actions.

Do a full scan once in a while when you have more time (perhaps hours).

They can be uninstalled later if desired.
