john582n say:

You can still use Internet Explorer on Windows 10

First make sure Internet Explorer is enabled

Right click on the Start button and left click on Programs and Features

Click on Turn Windows features on or off

If Internet Explorer is ticked, go to the next page, otherwise tick Internet Explorer then OK

 

Now you need to find Internet Explorer and Pin it to the Task bar or Start menu.

Click Start then All apps

Scroll down to Windows Accessories, open the folder and you will see Internet Explorer

Right click on Internet Explorer and left click Pin to Task bar (and/or Pin to Start)

Then the Internet Explorer icon will appear on the Task bar

 

Make Internet Explorer the default browser

Click on Start > Settings

In the Settings window, click System

Click Default Apps, look for Web browser and click on the app underneath that (eg Microsoft Edge) and choose Internet Explorer in the drop down box.  (Or you could set Firefox or Chrome to be the default (you would need to download them and install them first though.)
