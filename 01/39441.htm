rey g. santos say:

It could be your your total vRAM 256 dedicated and 1647 shared.

Your video card and your computer are made to share system RAM, but only
when you need to do so.  For instance, if you are running an application
(such as a graphics intensive game) that requires more memory than that which
is available on your video card, the system kicks in to allow it to use
general computer RAM.  However, when your graphics usage has no need for
system RAM, it's available for other uses.

"For example, if you have a video card that has 256 MB of onboard memory, that memory must be mapped within the first 4 GB of address space. If 4 GB of system memory is already installed, part of that address space must be reserved by the graphics memory mapping. Graphics memory mapping overwrites a part of the system memory."

The system memory that is reported in the System Information dialog box in Windows Vista is less than you expect if 4 GB of RAM is installed
http://support.microsoft.com/kb/929605/en-us

Go into system information on the NVIDIA control panel (Desktop right click or in the Control Panel) and see what is the dedicated vRAM and the shared vRAM information.
