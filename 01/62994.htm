We are sorry to inform you that you cannot rollback after 1 month of Windows 10 installation.

 

When you upgrade to Windows 10, the previous Windows files and folders are automatically stored in C:\Windows.old folder. This folder is kept by Windows only for one month. After one month Windows 10 will automatically delete the folder. So you can restore previous Windows version only within one month after Windows 10 upgrade. Also if you run Disk Cleanup in Windows 10, make sure to uncheck "Previous Windows installation(s)" option. How to use the System File Checker tool to troubleshoot missing or corrupted system files on Windows 8.

otherkenny say:

I have had similar problems with not being able to open any apps.  I found that it was because I was not listed as an administrator.  Try going to the Control Panel, under User Accounts select Change Account Type.  I was listed as having a Standard account.  When I changed it to an Administrator and restarted, all my apps and the Store opened up without any problems.
