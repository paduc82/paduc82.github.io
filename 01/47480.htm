gokul t say:

Hi mmo5,

 

The first time you open Paint and add a text box to the picture, default setting is set to Transparent. There are no options to change the default setting to Opaque.

 

As a workaround, I figured that once you open Paint and change the setting to Opaque after adding a text box, it remains the same for the rest of the time you work on Paint, until you restart the Paint application

 

Once the application is restarted the setting would default to Transparent. This behavior of Paint is by design in Windows 7.

 

If you have any feedbacks, please post the same at Windows 7 Feedback.
