rohn007 say:
This article goes into a lot of detail about services. Worth a read.

Short form, there are 4 service startup status's
Automatic- services is started when Windows is booted
Automatic (delayed) - Starts when Windows is booted, but waits until after boot process ends so that booting is not delayed.
Manual - is not started, but will be started on demand from any Windows request
Disabled - is not started, cannot be started by Windows

So when Installer service is set to Manual that is OK. When you start an installation the service will be started as required. No problem.
