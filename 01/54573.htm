graye say:

How do I recover my favorites which are lost in Windows 10 updates?

Microsoft has a built-in feature called System Restore that makes backup copies of your system before it does an install, update, etc.  This is a good thing, since a lot of people have trouble with missing favorites after a Microsoft update.

So that means that the backup probably has the missing favorites.  And the question now is:  How do I get my favorites from the backup.

There are two approaches... the "broad approach" is to just restore everything on your PC from that backup.  However, that might not be the best approach, since the backup may be several weeks old... and you'd loose everything that you did from that point in time.   The first step is to see when the last backup was...  So, type "create a restore point" at the Cortana prompt to bring up the settings... then click on the "System Restore" button to get to the next prompt

This will show the list of available restore points.  To see what would be affected if you did a restore, there is a button called "Scan for affected programs".   Take a look at the date of the backup, and decide if you want to restore everything from the backup

The other approach is to extract only parts of the data from the restore point.  This would be most helpful if the backup was several weeks old... or perhaps you'd installed a lot of applications, patches, etc. since the last backup (and don't want to do that all over again).

Microsoft doesn't have a "selective restore" option, but there are some free 3rd-party application that can do this.  Personally, I use one called System Restore Explorer.   It will allow you to "mount" the backup as a hard drive... and then copy the specific files that you want to restore.   Afterwards, you unmount the backup.  See the author's website for more information.

If you're talking about Edge, then, you'd be wanting to copy the contents of the \Users\YOUR_NAME_HERE\AppData\Local\Packages\Microsoft.MicrosoftEdge_8wekyb3d8bbwe folder from the backup to the same location on your C:\ drive.  Note: The AppData folder is hidden, so you might have to tweak the File Explorer settings to be able to see that folder.

But, if you're talking about Internet Explorer, then you'd be wanting to copy the \Users\YOUR_NAME_HERE\Favorites folder. 

Congrats to BeefRWijker for figuring this out.
