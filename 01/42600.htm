jetta48 say:

DSCN is part of a digital camera industry standard for the naming of files (& folders). It is a naming convention and does not indicate the image/video file type and therefore does not show how it should be opened.

 

The image/video file type is indicated by the filename suffix (the bit after the . at the end of the file name). For example,

 

DSCN0263.JPG is a JPEG image file.

DSCN0223.MOV is a .MOV movie file format used by many Nikon cameras.

 

(you could, for example, change the name of the above video file from DSCN0223.MOV to holiday.MOV without altering the video in the file or how it can be opened or edited)

 

In the case of .MOV files from the Nikon cameras (if that's what you are trying to open) then Windows Media Player and Apple's Quick Time should play them. They should also be editable in Windows Movie Maker.

 

If you are trying to open a different video file format please tell us the video file format and the camera that produced it.

If what you have is an OEM Windows 10 Pro license, it won't be accepted by the upgrade wizard.

DO the following first:

Upgrade to Windows 10 Pro using the following default product key:

VK7JG-NPHTM-C97JM-9MPGT-3V66T 

Review instructions how to perform an Easy Upgrade using the default key:

How to Perform an Easy Upgrade in Windows 10

Click Start > Settings > Update & security > Activation > Change the product key then enter the new product key then click Next to Activate

darleen pen say:

It could be possible that the volume license key that you're currently using for your computer might have reached it's maximum allowed capacity to activate your computer or it could also be possible that the registration on the devices needs to be manually done on the server. That being said, we would like to know the following information so we can either provide you the applicable troubleshooting or refer you to the correct TechNet forum:

  • Is your computer connected to a domain?
  • Was the product key provided to you already used on other multiple computers?
  • Are you managing a Windows Server or is your computer connected to a Windows Server?
  • If in case, what Windows Server Edition is your company using?

In the meantime, we suggest performing the troubleshooting indicated on one of the articles published by Andre Da Costa here on the Answers Community. This is so you can perform the activation through command prompt as stated on the steps. And if in case that you still aren't able to have it activated, then this would mean that the volume license product key needs certain permissions for it to be activation ready. Thus, the need of it being recognized by the company who owns it. In the event that it needs to be replaced, it would be best to have it raised to our AnswerDesk team so you can be provided with further options with replacing it if you purchased the product key in any local Microsoft Store. In the event that you've purchased your product key from a re-seller, it would be best to ensure that you have the receipt and have your product key changed by them so they'd be able to replace it with a much more suited product key.

If you have further questions, feel free to reply to this post.
