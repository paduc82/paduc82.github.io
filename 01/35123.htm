greatly_frustrated say:

Cannot install applications get an error The Windows INstaller Service culd not be accessed. This can occur if the Windows Installer is not correctly installed

OP: Stranded with Windows Installer issues

Whenever I try to install an application i recieve a Windows Installer error stating "The Windows INstaller Service culd not be accessed. This can occur if the Windows Installer is not correctly installed...." I've tried everything! Please help with a solution.

a. user say:

I would first try and re-register the Windows installer: 

 

1. Quit all Windows programs.

2. Click Start, type Run in the Search box and then press Enter, type msiexec /unregister in the Open box, and then click OK.

3. Click Start, type Run in the Search box and then press Enter, type msiexec /regserver in the Open box, and then click OK.

4. Restart your computer.

 

If the issue still exists, please perform System File Checker by the steps below:

 

1. Open an elevated command prompt. To do this, click Start, click All Programs, click Accessories, right-click Command Prompt, and then click Run as administrator.

If you are prompted for an administrator password or for a confirmation, type the password, or click Allow.

 

2. At the command prompt, type the following command, and then press ENTER:

 

SFC /scannow

 

This will check for any integrity violations and fix any corrupted operating system files.

 

Then reboot your system.

 

Then see if you still get the error message.

 

Also please check if the windows installer service is running or not, to do so, please follow the steps below:

 

1. Click on start, type in services, Hit Enter

2. Under services list, search for the windows installer service.

3. Double click and check whether its running or not.

4. If the service has stopped, then I would suggest you to start the service.

 

See if this resolves the isssue. If not, then-

 

You have to go into your registry.

 

Start, run, type in regedit.

 

Next go to HKEY_LOCAL_MACHINE.
SYSTEM
CurrentControlSet
Services
MSIServer

 

To the right youll see WOW64. Double click on it, or right click on it and click on modify. Change the setting from 1 to 0.

Next, go to the start menu again and type in the search box: services.msc.

Look for and find the Windows Installer service.

Double click on it and tell it to stop. Next, tell it to start.

You should now be able to install your programs.

 

If nothing helps, only a windows 7 repair was able to fix the issue.

 

Hope This Information Helpful.

The reason you don't yet have wi-fi is because you haven't yet installed the proper driver for it.  That driver is one of the following 4:

Intel PROSet/Wireless Software version 14.0.2.0.1

Intel Wireless LAN Driver version 14.1.1.3.1

Atheros Wireless LAN Driver version 9.2.0.206.0

The details for theh Intel version 14.1.1.3.1 says that it's a redirection to a Toshiba site from which you can download the latest version of the driver.

Intel PROSet/Wireless Software incorporates a driver and some control software that replaces the Windows software for controlling/configuring your wireless adapter.  This is optional.  If you have an Intel adapter, you can install either the PROSet software or the Wireless LAN driver.

If entering the serial number of your laptop did not eliminate either Atheros or Intel from the choices for a wireless LAN driver, you will need to determine the make of the wireless adapter that's installed in your laptop.  Open Device Manager (click Start, type Device Manager in the Search box, and press Enter), expand the "Network adapters" section, and list all of the items there. 

If there is a wireless network adapter listed in the "Network adapters" section but it has a warning icon and a "Device Status" indicating that there is no driver for it, you should be able to tell from the item's name whether it is Intel or Atheros.

If there is no wireless network adapter listed in Device Manager, first make sure that it is turned on.  All laptops have some means for enabling/disabling the wireless radio in order to save power when wireless communication is not needed.  On your laptop, pressing the Fn key together with the F8 key turns the wireless devices (wifi, Bluetooth, and broadband wireless if present) on or off.

If the wireless adapter is on, take a look at the "Unknown device" in Device Manager.  Double-click on the item and select the "Details" tab.  In the drop-down list under "Property" select "Hardware Ids".  Right-click on the top line, select "Copy" and then Paste (right-click > Paste or CTRL+V) into your Reply here.

You might also want the Toshiba Wireless LAN Indicator.  This will show you all available Wireless LAN Access Points in the Windows Logon Screen.
