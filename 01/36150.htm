Yes, but you do not need to buy anything other than a USB stick.

You can make an installation disk [USB/DVD] using http://windows.microsoft.com/en-us/windows-10/media-creation-tool-install

You then run a Clean install to overwrite the whole disk -

Clean Install Windows 10 - TenForums

Clean Install Windows 10 - Forum article by Greg Carmack

You will need to go to Lenovo beforehand to download the latest versions of all your hardware drivers.  You cannot trust Windows update to get this right all the time so you will continue to rely on Lenovo for these.

While you are on the Lenovo site, check that you have already done the latest BIOS update that they offer.  The BIOS controls how all your hardware interacts with the system.  Each BIOS update is a one-off task, you do not need to repeat it after reinstallation.

By the way, if you only have one hard drive in your laptop then a reinstallation is the best time to partition your main disk into two separate drives. 

  • The benefit of this is that you can put 80-100GBs in a partition that you install Windows to and keep the rest for your own files. 
  • This keeps system images very small so that you will actually bother making them.
  • System images are the single most effective recovery tool there is - if immediate steps to fix an error in Windows or an application do not work then you restore the most recent image to completely overwrite the C:\ drive to exactly what it was when the image was made.

To better assist you, please answer the following questions:

1. Did you choose the recovery drive to be in the C: drive?

2. When you say D drive, are you referring to other personal files?

A system image is an exact copy of a drive. By default, a system image includes the drives required for Windows to run(C: drive where the operating system is installed). It also includes Windows and your system settings, programs, and files. If the recovery drive is in a different drive (for eg. D drive), when you restore using the system image, everything else will be deleted except the files and settings which are included in the C: drive. If the D drive contains your personal files and data, I would suggest you to create the system image and also create a backup of the D drive and other drive in order to restore your personal files.

Hello Lee,

Thank you for visiting Microsoft Community Forum.

I understand that you want to know if you encrypt your non TPM (Trusted Module Platform) computer with BitLocker and configure it to be decrypted through a USB flash drive. If the PC gets stolen with the flash drive containing the decryption key, does that make all your data on risk?

On computers that do not have a TPM, encrypted Windows operating system drive require the user to insert a USB startup key to start the computer or resume from hibernation, but it does not provide the pre-operating system startup system integrity verification offered by BitLocker with a TPM. 

As all your data are saved locally on your computer and not on cloud server, if your non-TPM computer is stolen with USB flash drive containing decryption key, your computer data can be on risk.
