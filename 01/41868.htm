To create a shortcut to open an FTP site in Windows Explorer

  1. Open Computer by clicking the Start button , and then clicking Computer.

  3. This displays the Add Network Connection wizard. Click Next.

  4. In the wizard, select Choose a custom network location, and then click Next.

  5. Enter the name of the FTP site, with the full FTP:// in front of it, and then click Next.

  6. To use a name and password, clear the Log on anonymously check box. Type a user name, and then click Next.

  7. By default, the name of the shortcut is the same as the FTP address. If you want to give the shortcut a different name, type it in the Type a name for this network location box. Click Next.

  8. If you do not want the FTP site to open after you're done setting up the shortcut, clear the Open this network location when I click Finish check box. Click Finish. A shortcut to the FTP site appears in the Computer folder. You can drag that shortcut to your desktop so that it's easy to find later.

If you have XP or Vista then you have to buy Windows 10 because they do not qualify for a free upgrade.

If you wanted to check whether or not your computer was capable of running Windows 10 then test its compatibility.  If you have XP or Vista then you'll have to run the Windows 8 upgrade checker as the Windows 10 one does not run on them [but you'll still get an accurate answer so it is just as useful].  If you have Windows 7 or 8 then  you can run the Windows 10 upgrade checker.  There are links to all the upgrade checkers and a brief discussion of some key points about compatibility in

Your intended choice of Windows 10 Pro is not an issue as far as compatibility is concerned.  Your choice of 32 bit Windows 10 makes things easier - you might find that the upgrade checker tells you that you have no choice in the matter anyway. 

Even if it is compatible, you'll still need to get drivers for your hardware and they might not all be available in Windows 10-compatible versions.  See

I can also advise you about some of the practicalities of doing the upgrade itself if you decide to proceed, and if you want me to, but it is essential to accept the fact that spending the money & going through all the effort to upgrade to Windows 10 will only be justified if your computer is going to last long enough for the benefits to be worthwhile.  For Windows 7 computers, the choice might seem more straightforward because the upgrade is free.  Nobody can make that decision for you as it really is an act of faith - your computer might fail completely next month and all your money & all the time you spent on the job would be wasted.  I managed to upgrade my 11 year old XP computer to Windows 10 but I also have to accept that it might finally give up the ghost before long.  I also upgraded my 6 year old Windows 7 computer to Windows 10.

Just to be pedantic for a moment.  If you currently have XP or Vista then the "upgrade" will be a complete installation rather than an upgrade as such.  The whole of your hard drive will be overwritten so you will have to backup all your own files to an external first.  If you have Windows 7 then you have the choice of upgrading or doing the job as a full installation but you still ought to back it all up whichever you choose.
