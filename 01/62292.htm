diana d say:

Hi,

 

Step 1: Verify if the bit rate of the file is same as the one supported by MP3 player.

The bitrate is the amount of data used to create the audio stream. If they are different, in some case it would not play.

 

Step 2: Try copying the files through windows explorer and then check if it is playing in the MP3 player.
