When you type reg add /? then you will see the correct syntax to be used for reg.exe. In your case it would be

reg add HKCU\Software\Policies\Microsoft\Windows\RemovableStorageDevices\{53f5630d-b6bf-11d0-94f2-00a0c91efb8b} /v Deny_Read /t Reg_SZ /d 1

In Microsoft's notation, the exclamation mark serves as a delimiter between the registry key and the value name.

Hi Lorrie. I'm Greg, an installation specialist and 9 year Windows MVP, here to help you.

Unplug power and remove battery, press and hold power button for 30 seconds to release all power from circuitry, plug back in and power up to see if any change.

I will give you steps to access the Repair Mode to try all repairs to get Windows started, and if not to do the automated Reset or the superior Clean Install after rescuing your files. These steps don't cover possible hardware failure which might become apparent while performing the steps, so please report back as you go.

If the mouse or keyboard won't work in Step 1 then skip to Step 2 because bootable media uses different drivers so that they always work.

1) Use whichever method works in this link to access Advanced Startup Options: https://www.tenforums.com/tutorials/2294-advanc.... Easiest may be to force shut down the PC (by holding in the power button) three separate times to trigger the Startup Options.

2) If none of the other methods work it may be necessary to Create Windows 10 Installation Media on another PC: http://windows.microsoft.com/en-us/windows-10/m...

Insert media, boot it by powering up PC while pressing the BIOS Boot Menu Key: https://www.sysnative.com/forums/hardware-tutor...
