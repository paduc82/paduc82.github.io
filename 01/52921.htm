You may try deleting power plans by following the steps below:

  1. If you are deleting custom power plans, open a command prompt by right-clicking on the Start button and selecting Command Prompt, then go to step 3.
  2. If you are deleting built-in Balanced, Power Saver, or High Performance power plans, open an elevated command prompt by right-clicking on the Start button and selecting Command Prompt (Admin), then go to step 3.
  3. Copy and paste the following command  into the command prompt, and press Enter. This will give you a list of all existing power plans on the PC along with their GUID. Note the GUIDs that are listed next to the power plans that you want to delete: power cfg /list

      powercfg -delete fcbb086e-4380-47d6-8a79-f4a099526dfa
  6. When finished, you may close the command prompt.

a. user say:

Hi Yves,

The error message that you're getting is particularly referring to the issue on the source partition. The disk space you require to create a backup system image refers to not only the regular partition of your Windows operating system, but also the volume shadow copy of the image. Note that the minimum requirement for volumes less than 500 MB is 50 MB of free space, and for volumes more than 500 MB is 320 MB of free space. Now, to sort out this error message, you can follow the solution below:

Solution: Clear the USN Journal Files.

  1. Press Windows + R, type diskmgmt.msc on the Run dialog box, then press Enter. This will open the Disk Management.
  2. Right-click the system reserved volume and select "Change Drive Letter and Paths" to allocate the drive.
  3. Next, run the Command Prompt as administrator. Press Windows + X, then select Command Prompt (Admin).
  4. Type the command line fsutil usn queryjournal <VolumePath e.g. (E:)>.
  5. Type fsutil usn deletejournal /D /N <VolumePath>
  6. Remove the drive letter from system reserved partition.

Now you can create a system image successfully without receiving the error message in Windows 10.

eddiegct say:

Hey Guys,

In my case I get this error when upgrading to Win10 Redstone1 (aka Anniversary Update ) form TH2.

The installation failed, and Windows automatically restores and reverts back to original build, with the error message 0x8007057 – 0x20007.

Other than error above, the installation may also stuck and hanged at 18% or other progress point indefinitely.

Microsoft has acknowledged the upgrading bug, explaining that it’s due to a bug where too many device INFs are being migrated and hitting an internal limit. However, the suggested workaround to use Disk Cleanup to “clean up system files” and choose “Device driver packages”, “Previous Windows installation(s)” and “Temporary Windows installation files”, and also remove unnecessary peripherals and remove them from Device Manager, does not actually help nor work.

Pieter Wigleven published a solution that resolves the issue and allows Windows 10 to upgrade to build 10122 properly.

  1. Download PsExec.exe from Sysinternals.
  2. Open an elevated Command Prompt by searching for Cmd, then right click on Command Prompt and choose Run as Administrator.
  3. Run the following command: 
    psexec.exe –s –i cmd.exe

    Note that you need to change directory to the folder where you saved the downloaded PsExec, or specify the path to PsExec.exe in the above command.

  4. Another Command Prompt in the system context will be opened with the title of “Administrator: C:\Windows\System32\cmd.exe”.
  5. Execute the following command: 
    rundll32.exe pnpclean.dll,RunDLL_PnpClean /DRIVERS /MAXCLEAN

    Note:

  6. Close the Command Prompts.
  7. Try to upgrade to Windows 10 Pro Build 10122.

Note that you may need to execute the steps above for a few times before the upgrade works.

Hope this one help.

shenan stanley say:

Unfortunately - without a clearer picture of what you did - direct advice will be hard to come by.

  

If you are missing everything anyway *and* you have copies of *your* stuff (files you created, bookmarks, emails, pictures, music, etc.) backed up off the system in question (external drive or media) - then you should first try System Restore (to a time before everything went missing)...

To restore your PC to an earlier point in time

  1. Swipe in from the right edge of the screen, and then tap Search.
    (If you're using a mouse, point to the upper-right corner of the screen, move the mouse pointer down, and then clickSearch.)

  2. Enter Control Panel in the search box, and tap or click Control Panel.

  3. Enter Recovery in the Control Panel search box, and then tap or click Recovery.

  4. Tap or click Open System Restore, and then follow the instructions.

...

If, for whatever reason, a restore point before this all started is unavailable to you OR that doesn't work - you should perform a System Refresh...

To refresh your PC

  1. Swipe in from the right edge of the screen, tap Settings, and then tap Change PC settings.
    (If you're using a mouse, point to the upper-right corner of the screen, move the mouse pointer down, click Settings, and then click Change PC settings.)

  2. Tap or click Update and recovery, and then tap or click Recovery.

  3. Under Refresh your PC without affecting your files, tap or click Get started.

  4. Follow the instructions on the screen.

...

You'll have to update Windows again and install any applications you installed manually (not through the Microsoft store and may even have to re-install some of the applications you installed through the Microsoft Store - but if you installed them the first time, installing them again should be easier.

...

If, for whatever reason, that doesn't work - you'll need to reset the PC (essentially a clean install - everything not backed up will be lost.)

  

To reset your PC

  1. Swipe in from the right edge of the screen, tap Settings, and then tap Change PC settings.
    (If you're using a mouse, point to the upper-right corner of the screen, move the mouse pointer down, click Settings, and then click Change PC settings.)

  2. Tap or click Update and recovery, and then tap or click Recovery.

  3. Under Remove everything and reinstall Windows, tap or click Get started.

  4. Follow the instructions on the screen.

...

Then you will have to setup everything as if the computer was new.
