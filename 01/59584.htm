Method 1:

1. On the Desktop, right-click on an empty space and click Personalize.

2. If you're using an Aero theme, Change it to Windows 7 Basic Under Basic and High Contrast Themes instead.

3. Click Windows Color and Appearance below the Themes selection and check if there's an Enable transparency option. Toggle the check box to disable it.
