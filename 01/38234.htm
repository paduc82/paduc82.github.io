how to reinstall

malkeleah say: WINDOWS COULD NOT FIND START BECAUSE THE FOLLOWING FILE IS MISSING OR CORRUPT
WINDOWS ROOT>\SYSTEM32\hal.dll

That message is rather misleading.  It happens because the boot.ini file that tells the boot where to look for 'Windows' is damaged, so it is looking for files in the wrong place - hal.dll just happens to be the first one it looks for. Set the BIOS to boot CD before Hard Disk.  Boot the XP CD and, instead of Setup, take the immediate R for Repair.  Assume any password requested is blank, and TAB over.

Attrib -H -R -S C:\boot.ini [enter]
DEL C:\boot.ini [enter] [to delete the bad one]
BootCfg  /Rebuild [enter] [to search for Windows installations and make a new one]
MS-MVP - Elephant Boy Computers - Don't Panic!

Step 1: Make sure your Surface is turned off and plugged in (hold down the power button for 30 seconds).

Press and hold the folder that you want to take ownership of, then tap Properties. (If you are using a mouse, right-click the folder, and then click Properties.)

Issues that you experience when you try to access files and folders may be related to permissions. Permissions are rules that determine whether you can access or change files and folders. To check permissions on a file or folder, follow these steps:
Press and hold or right-click the file or folder, and then click Properties.
Tap or click the Security tab.
Under Group or user names, tap or click your name to see the permissions that you have.

Important You must be logged on as an administrator to change permissions on files and folders.

Tap or click the Security tab.
Under Group or user names, tap or click your name to see the permissions that you have.

Encryption can help protect files and folders from unwanted access. You cannot open an encrypted file or folder without the certificate that was used to encrypt it. To determine whether a file or folder is encrypted, follow these steps:
Press and hold or right-click the file or folder, and then tap or click Properties.
Tap or click the General tab, and then tap or click Advanced.
If the Encrypt contents to secure data check box is selected, you have to have the certificate that was used to encrypt the file or folder to be able to open it. Is HP supporting that printer on Windows 10? Let's find out by going to the HP support page for the CP4025 printer series and looking at the Download options tab. Aha - there's a link for Get drivers, software and firmware. We'll click on it.

So there are two CP4025 printers: the "n" and the "dn," where "n" probably standards for networked and "d" probably standards for duplex. And you're in luck because there are Windows 10 drivers for both printers. That means HP does support them on Windows 10, and if HP is providing support that means they will help you to get your printer working on Windows 10.

To get started, click on the Setup and install link on the left side of the page. That gives you detailed setup instructions for physical setup, network setup and installing drivers and accessory software.

If you're still having trouble there's also a Troubleshoot a problem link for you to consult, in addition to other links that could help you solve your problem.

Another great avenue for help is the HP Community, where you can correspond with other people who are using the same printer as yours on Windows 10. If you decide to post there make sure you provide more detail about your problem than "it won't allow me to." People need a more thorough explanation of exactly what happens in order to help you.

If all else fails and you're still within the warranty period you can always contact HP technical support directly.

These are the very same procedures that I follow when I'm having a problem with my hardware. Not everyone provides as much as support for their products as HP, so take advantage of it.
