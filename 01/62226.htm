ronnie vernon say:

I'm seeing the Library Pane option in the Layout section, under Organize on Windows Explorer.  I know turning it on puts a Libraries Header in the main pane.  I don't see any other purpose and I question what its purpose really is.  Reading the explanation in the Working with files and folders description under Windows 7 isn't any help.  It must have some purpose that I'm not seeing.


Hi

When you open any Library, at the top of the library you will see a rectangular section that has the Locations options on the left side and the Arrange By menu on the right side. This is the Library Pane.

When you have a Library open, the Library Pane option will appear on the Organize/Layout menu. If you click this option to remove the check mark the Library Pane will be removed from that particular library only.

The Library Pane is on, by default. When you are finished configuring the Libraries, you can remove that pane to gain more folder space.

Hope this helps.

a. user say:

Hi Terri,

I'm Stefano an Independent Advisor, here to help you.
If you have an OEM license, your activation key (and so the license) is bound to your PC; if you have a RETAIL license, your activation key (and so the license) is bound to your Microsoft Account. To check your license kind:

1. Type "cmd" in windows search bar
2. Right click on "Command Prompt"
3. Select "Run as Administrator"
4. Type "slmgr /dlv" without quote and press ENTER

Check the uppercase word on "Description" line.

mann_manohar say:

Hello,

Thank you for visiting Microsoft Community.

I understand that you are experiencing issue with your Administrator account on your computer and trying to enable hidden admin account to make changes in your admin account. Command prompt fails to run as administrator.

You may also try accessing the hidden admin account from command prompt in Windows Recovery Environment (WinRE). We can try to start the computer in Safe mode with command prompt and run the command to enable hidden admin account.

To start the computer in safe mode with command prompt:

  1. Tap or click the power icon from the sign-in screen, hold Shift, and then tap or click Restart
  2. After restart computer will go to Recovery, select â€œSee Advanced repair optionsâ€
  3. From Choose an option select Troubleshoot

  4. From Troubleshoot, select Advanced options

  5. In Advanced Options, select Windows startup settings

  6. In Windows startup settings select safe mode with command prompt

Enter the following command to enable the hidden administrator account:
net user administrator /active:yes

After the account is enabled, a new user account will appear on the signin page.Note that the account has not assigned a password to it, and that you should consider setting one to improve account security.

Also refer: http://windows.microsoft.com/en-US/windows-8/user-accounts-frequently-asked-questions

After successfully getting access to Default admin account you may try making changes to your admin account. If that does not help you may try creating a new user account with Administrator privilege.

You may refer the article below for help in creating a new admin account.

http://windows.microsoft.com/en-us/windows/create-user-account#create-user-account=windows-8

If you ever want to disable the account follow the same instructions, but run the following command instead:net user administrator /active:no

Hope this helps. Please let us know the results. Feel free to write us back for any further assistance with Windows, we'll be glad to assist you.
