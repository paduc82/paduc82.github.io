Hi Evi,

As per the  issue description, you have installed Ubuntu on your Windows 8 preinstalled laptop to work with Windows 8 as dual boot and then later you disabled the Secure Boot and also changed the UEFI Mode to Legacy from the computer BIOS. Now, when you start your Laptop you get the error: "Operating System not found" on a black screen and you are able to boot into neither Windows 8 nor Ubuntu.

 

I dont think that the Windows 8 is erased from the hard disk. This had happened only because of the changes that you have done in the computer BIOS.     julie ram say:

    Let's make it simple for you, Vickie.

    To add the battery icon to the taskbar, select Start  > Settings  > Personalization > Taskbar, and then scroll down to the notification area. Choose Select which icons appear on the taskbar, and switch the Power toggle to On.

    You can check battery by selecting the battery icon in the taskbar at the bottom right of your screen.

    Notes: You may need to select show hidden icons  on the taskbar before selecting the battery icon. Power is not an option on systems without a battery.
