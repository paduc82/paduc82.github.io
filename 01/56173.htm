frederik long say:

There are a few million Windows 7 machines out there, most of which run perfectly well month after month. If your machine keeps failing persistently then I can think of four reasons:

a) Software . You run something (intentionally or unintentionally) that throws a spanner in the works. If so then you would know by way of timing: One day Windows starts up fine, the next time it takes forever. Obviously something has happened between the two startup events. To fix it you use System Restore which will restore the registry to a previous point. You also review your actions during the past two days.

b) Hardware . You have some hardware or driver incompatibility. If so then it would show up straight away after reloading windows.

c) Registry "Cleaners" . Many people swear by them. Among professionals most are considered totally useless (at best) and damaging (at worst). If a registry "cleaner" were to do a proper job then it would need detailed information for a few ten thousand registry keys from hundreds of software suppliers. You can compare it to tweaking a gene in a human to cure epilepsy.

d) Automatic Updates . This is a very popular cause amonst posters. The reasoning always goes like this: "Yesterday my PC works fine, then there was an automatic update and now it fails. The automatic update is causing a problem." While this is possible, it is rarely the case. Just because B follows A does not necessarily mean that B is caused by A. Automatic Updates are easily excluded from the list of culprits by checking when they happened, then performing a System Restore to a point before the update.

If your problem is recent, use System Restore. If it is old or if you use a registry "cleaner", reload Windows. If it is a hardware problem, check the Sony FAQs.
