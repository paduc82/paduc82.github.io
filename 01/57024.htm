a. user say:

If you are allowing Windows Update to offer hardware driver updates and installing them,  that is never a good idea.

It is never a good ideas to accepting and install hardware driver updates offered by Windows Update.  Microsoft doesn't know what the best drivers are for you system and will often offer out of date and incompatible drivers.  You should only get hardware driver updates from the hardware manufacturers WWW site.

This is one reason there are so many topics that start with "Windows Update broke my computer...".

Just because Windows Update offers you a hardware driver doesn't mean your system needs it or that you need to install it.

You can follow the example in this article to configure your Windows Update to not show you hardware driver updates:

https://support.microsoft.com/en-us/kb/2500967

But I would not even count on that to work as desired.

For Windows Update the best (safest) strategy is to configure your Windows Update settings to Never check for updates and that puts you in charge of what updates get installed on your system instead of Microsoft.  Then when you are in charge you can check periodically (like every Patch Tuesday) to see what updates are available, install the ones you want and chose to Hide any hardware drivers updates or any other that look like Microsoft "snooping" patches and hope you never see them again.

If you accidentally install an afflicted driver from Microsoft and your system malfunctions you might be able to Roll Back the driver to the previous version that worked from Device Manager (but the driver might be offered again if you don't Hide it):

Depending on the model of your Dells (and the Service Tag) if it has Intel components that you think need to be updated check the Intel drivers page and let their analyzing/scanning tool take a look at your system and if offered driver updates consider installing them:

https://downloadcenter.intel.com/

If your system is stable your system might not even need hardware driver updates and only install hardware driver updates if you know it will correct a problem you are having.  A good rule of thumb for hardware drivers is if it ain't broke don't fix it.

A more common problem is that lots of people ignore the iron rule of data processing until they suffer a major data loss. The rule says that important or irreplaceable files must at all times be stored on two independent media. No exceptions.

  6. Open the desktop file Pics.txt.

andre da costa say:

With Windows 8, Microsoft has changed from stickers that have the product key that the user has to type in when installing the operating system to new BIOS embedded product keys. The idea is that by eliminating the sticker, you eliminate one of the easier ways for nefarious users to get a legitimate product key. Eliminating the product key sticker also removes any worry that the sticker might get damaged while at the same time eliminating the long and irritating process of typing in various letters and numbers when installing the operating system.

 

If the user has to reinstall the operating system on a machine that came with Windows 8, the installation process automatically grabs the software product key from the motherboard BIOS with no input from the user. This means that those familiar Windows product key stickers will no longer appear on the Windows 8 computers.
