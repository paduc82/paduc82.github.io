Somehow, the system seems to have applied the plasma screen settings to the laptop (or the laptop is configured to use those settings based on how it was setup or has somehow changed to take on those settings or perhaps even some totally different settings).

Do you know when this problem began (probably when you setup the plasma screen - earlier than just when you plugged it in - unless that occurred long ago and everything was working fine until this recent incident)?  Try a System Restore to a point in timeBEFORE the problem began (and try to give it a couple of days or more of leeway).  Here's the procedure:http://www.howtogeek.com/howto/windows-vista/using-windows-vista-system-restore/.  Be sure to check the box to show more than 5 days of restore points.  If the first attempt fails, then try an earlier point or two.  NOTE: You will have to re-install any software and updates you installed between now and the restore point, but you can use Windows Update for the updates.  Use the recovery disk if the system prompt doesn’t work. The recovery disk works a bit different from the above procedures but if you follow the prompts from the System Restore menu option with the above information you should be able to restore with no problems.

Assuming this fixes the problem with the laptop display, you still need to resetup the plasma screen display (if it was setup between the restore point and the system restore).  Double-check the procedure as it is likely that some setting or configuration caused the problem and we don't want it to occur again.  Re-read the setup instructions carefully.  Be sure you have all the correct drivers and software (double-check and download updated copies if necessary - and do so for the laptop's video card and monitor at the same time).  Triple-check the configuration settings as I suspect the problem is that the laptop doesn't know to still use its default settings on the laptop screen when the plasma screen is attached.
