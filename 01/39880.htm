jweke replied on June 28, 2016
replied on June 28, 2016

I just ran into the same problems after upgrading to Windows 10.
Uninstalling using the hidden Administrator-account helped.
The following steps show you how to activate and deactivate the account.

  1. Opening a command prompt in administrator mode.
  2. At the command prompt, type "net user administrator /active:yes", and then press Enter
  3. Exit the command  prompt.
  4. Logout and login as Administrator.
  5. Uninstall what you need to.
  6. Logout
  7. Login as User
  8. Deactivate the Administrator-account using  "net user administrator /active:no"

Hope this helps you too.
Source: 
http://h30434.www3.hp.com/t5/Notebook-Operating-System-and-Recovery/HP-Power-Manager-amp-Set-Up-Manager-Blocked-Windows-10/td-p/5187800

-----------------------------

But when I was logged in as Administrator, HP Product Detection was not in the list of installed apps.

What helped in my case:

I found out from the error message, that I have to look for C:\Windows\Installer\b66b9ad.msi (Installer directory is a hidden directory, so make hidden directories visible in your file explorer).

In fact in the properties of b66b9ad.msi I saw, that HP had revoked the security certificate.

I opened a cmd.exe window with administrator privileges (right click, run as administrator).

I executed

cd C:\Windows

system32\msiexec.exe /uninstall Installer\b66b9ad.msi

I hope this can help people with similar problems.
