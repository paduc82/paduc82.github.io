ken blake - microsoft mvp say:

Original Title: Recovery Disk problem

 

My mother board failed & after installing a new mother board my restore cd's with windows 7 Home Premium 64 bit were not recognized. I had to install my old

Windows Vista Ultimate 64 Bit program. How can I reinstall Windows 7 Home Premium 64 Bit using my recovery discs?


 

If you have a recovery disk, that means that Windows came preinstalled and you have an OEM copy of of Windows. Two things about OEM copies:

 

1. An OEM copy is usually BIOS-locked to the motherboard, and  won't install on a different motherboard.

 

2. An OEM license ties it to the first computer it's installed on. A different, non-identical motherboard is treated as a different computer, and even if it weren't BIOS-locked and you could install it, you would not have a valid license and you wouldn't be able to activate it.
