lemp say:

As Aileen Alf wrote, the reason that Devices and Printers doesn't show any printers is because the Print Spopler service isn't running.  This is also the reason you can't install a printer.

However, simply telling you to start the spooler isn't likely to help because there is a reason that the spooler stopped in the first place.

Aside from possible malware, the usual reason that the spooler stops is because you either have a corrupt print job stuck in the queue or a corrupt printer driver. 

The procedure for cleaning out the print queue and corrupt printer drivers is below, but considering that you had 5 printers installed and you report that the problem started right after you installed Office 365, you might want to just clean out the queue and then repair your Office 365 installation following the instructions here:  https://www.groovypost.com/howto/repair-microsoft-office-365-installation/

A. Clean out stuck print jobs --

  • Open an Elevated Command Prompt (search for "cmd.exe"; right-click and select "Run as administrator")
  • Type the following in the black Command Prompt window and press Enter after each line

                           net stop spooler
                           del /Q "%windir%\system32\spool\PRINTERS\*.*"
                           net start spooler
                           exit

B. Clean out corrupt printer driver

  • Uninstall printer-related software from Control Panel > Programs and Features
  • Open an Elevated Command Prompt (search for "cmd.exe"; right-click and select "Run as administrator")
  • Type the following in the black Command Prompt window and press Enter
        
            printui /s /t2
            exit

    Be sure to remove the driver AND the package.

C. Reinstall printer using freshly-downloaded software package.
