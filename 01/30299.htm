Hello Rosendo,

To better assist you with your concern, I'd like to know the following:

  • Confirm which build is currently installed on your device. To check, follow the steps below:
  1. Press Windows key + R, then type in winver.
  2. Click OK.
  • What is the make and model of your device?

If you're using a laptop with a trackpad, chances are that your laptop has gestures that triggers Cortana to pop-up. To disable it, refer to the steps below:

1. Press Windows key + I, then select Devices.

2. Go to Mouse. Click on Additional mouse options.

3. Check if there are gestures that enables Cortana and turn it off.

In addition, it's also best to turn off Taskbar tidbits:

1. Press Windows key + S, then click on the gear icon.

2. Under Taskbar tidbits, toggle the switch to disable the said feature.

Upon researching some users had stated that their microphone somehow triggers Cortana even though the Hey Cortana feature is disabled.

If the issue persists, you can disable Cortana using the Local Group Policy Editor:

1. Press Windows key + R, then type in gpedit.msc and press Enter.

2. In the Local Group Policy Editor, navigate to the location below:

Computer Configuration/Administrative Templates/Windows Components/Search

3. Double-click on Allow Cortana and select Disabled. Click Apply and click OK to save changes.

4. Reboot your device.

Note: This method will only work if your computer is running on Windows 10 Pro.

If your device is running on Windows 10 Home, you may need to disable Cortana using Registry Editor.

1. Press Windows key + R, then type in regedit and press Enter.

2. In the Registry window, navigate to this directory:

HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\Windows Search

3. Chang the value of AllowCortana to 0.

4. If Windows Search folder is missing, you'll need to create the said folder.

5. Right-click on the Windows folder, click on New > Key. Name the folder Windows Search.

6. Right-click on the Windows Search folder that you've created, select New > DWORD (32-bit) Value.

7. Name the value AllowCortana. Double-click AllowCortana and set the Value data to 0.

8. Click OK to save changes.

Note: The method to fix the issue will have Registry modification. Before you modify the Registry, make sure to back it up and understand how to restore the registry if a problem occurs. Please refer to this link: How to back up and restore the registry in Windows.

Looking forward to your response.

ramesh.kumar say:

Hi KellCulp,

As per the issue description there is no sound after downgrading from Windows 10 to Windows 7. This could be if the drivers get corrupted or not installed properly. This could also be if the audio service or its dependent services are not running. The issue could also occur if some settings get corrupted while downgrading from Windows 10. 

Let’s run the audio troubleshooter and make internal speaker as the default device by following the steps in the below methods and check the issue.

Method 1: 

If your computer is having problems playing sound, try using the Playing Audio troubleshooter to fix the problem. It checks for common problems with your volume settings, your sound card or driver, and your speakers or headphones.

Open the Playing Audio troubleshooter:

http://windows.microsoft.com/en-us/windows7/Open-the-Playing-Audio-troubleshooter

Method 2: 

Step 1:

a. Click Start, type Sound in the Start Search box.
b. Click Sound in the search results at the top of the Start menu.
c. Click the Playback tab. Notice the check mark next to the default device. For most users, the default device is Internal Speakers. Make sure that your audio settings are configured as expected.

For more information:

Tips for fixing common sound problems:

http://windows.microsoft.com/en-us/windows7/Tips-for-fixing-common-sound-problems

Step 2:  

Follow these steps:

a. Click Start, type Sound in the Start Search box.
b. Click Sound in the search results at the top of the Start menu.

c. On the Playback tab,
d. Select the playback device.
e. Click Properties, click the Advanced tab and make sure that the Default Format is set to the value that you expect.
f. When you finish, close this window to return to the previous window.

Let’s check the configuration settings.

a. On the Playback tab.
b. Select the playback device.
c. Click Configure and then make sure the configurations are set correctly.

If the sound settings are set correct, and your internal speakers are already set as the default device. Select the sound device that you want as the default sound device, and then click Set Default. Click OK to save your settings and to close the Sound dialog box.

Method 3:
If the issue persists, I would suggest you to check if the audio dependency services are started and their Startup Type is set to Automatic.

a. Press Windows Key +R.

b. Type Services.msc and hit Enter.

c. Scroll down through the list and find the following services.

Plug and Play

Multimedia Class Scheduler

Windows Audio Endpoint Builder

d. Double-click on the services one by one and check if the services are started and startup type is automatic.

Hope this would help. If issue still persists post back with current status of your computer and result of proposed suggestion, we will be happy to assist you.
