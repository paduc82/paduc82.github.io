5. Connect the printer and let Windows install the drivers

6. Restart Windows and after a while Windows will have automatically downloaded the proper Device Information plus all its tasks.

I found the problem to be "DeleteUserSettingOnUpgrade". After changing the value to 0, the problem is gone. 

Here's how to fix it"

1) Run regedit.exe

2) Go to \HKEY_LOCAL_MACHINE\SOFTWARE\SynTP\Install

3) Change the value of "DeleteUserSettingOnUpgrade" to 0

4) Save and restart

Your touchpad settings should now stay after reboot!
