Hi Bryce. I'm Greg, an installation specialist and 9 year Windows MVP, here to help you.

Remove any changes you made in msconfig memory or CPU settings as these are for testing only.

Type Resource in Start Search, open Resource Monitor, choose Memory Tab, confirm amount of hardware reserved RAM.

This might be adjusted in the UEFI/BIOS setup, or in the Video Card settings. I would confirm with the maker that this amount of RAM is intended to be reserved as if not then it's some kind of bug or corruption.

Test the RAM overnight to stress it with the best bootable test memtest86, following these steps which test both the sticks and the slots: http://www.sevenforums.com/tutorials/105647-ram...

Confirm the RAM specs and configuration by running Crucial Memory Scanner which is the best authority for this: http://www.crucial.com/systemscanner/

Hey, VideoCurious!

Overall, the NVIDIA GPU should be kicking in automatically depending on the application you're using. But, you can also manually set applications to use a specific GPU in the NVIDIA Control Panel.
