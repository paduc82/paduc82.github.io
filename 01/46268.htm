    DISM.exe /Online /Cleanup-image /Scanhealth

You can't change the type of account; you would have to set up a new POP3 account, then remove the IMAP account. Removing an account removes all of the account folders and permanently deletes their contents, so be sure to save any messages you want to keep before removing the account.

  2. In Windows Live Mail, right-click on the IMAP account name in the folder pane and select Properties. On the General tab, deselect Include this account when receiving mail or synchronizing. Close the properties dialogue.

  4. Complete the wizard using the POP settings on this page: Get started with IMAP and POP3. Note the remarks about recent mode.
  5. Right-click on the new account name in the folder pane and select Properties.
    Give the account a meaningful name, then on the Advanced tab, configure the delivery options you want at the bottom of the page.
