How to: upgrade from previous versions of Windows using Windows 10 ISO file

Click HERE to download official ISO media for Windows 10 from Microsoft.

  • Disable (preferrably uninstall) your Antivirus utility before attempting the upgrade.
  • Restart a few times and try again.
  • Disable General USB Devices (example - Smart Card Reader). 
  • If you are using a SCSI hard disk, make sure you have drivers available for your storage device on a thumdrive and it is connected. During Windows 10 setup, click the Custom Advanced Option and use the Load Driver command to load the appropriate driver for the SCSI drive. If this does not work and setup still fails, consider switching to a IDE based hard disk.
  • Do a clean boot and try again.
  • If you are upgrading using the .ISO file, disconnect from the Internet during setup, if you are connected by LAN (Ethernet) or Wi-Fi, disable both and attempt setup again. 
  • If you are updating through Windows Update, when the download reaches 100% disconnect from the Internet LAN (Ethernet) or Wi-Fi and proceed with the installation.
  • If that does not work, try using the .ISO file to upgrade if possible.
  • If you are connected to a domain, switch to a local account
  • If you have any external devices attached to the machine, disconnect them (example, gaming controllers, USB keys, external hard disc, printers, non-essential devices).

Interesting problem where you get the activated notification yet FSX goes into the 30 minute Demo mode.  You said that you have tried 'all the Help suggestions' though the following is the general one to get FSX activated.   Don't know if you've tried it, but I've given it below (be careful though when amending Registry values, as you would know):

1. Close FSX and run Registry Editor (press Windows key + R and type   regedit    and     Ok)

2. Expand the following registry key:            

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Class\{4d36e967-e325-11ce-bfc1-08002be10318}      

3. After you find the folder, double-click Upperfilters on the right side.

5. Close Registry Editor and reboot Computer.

6. Start FSX and check whether the Activation box comes up.   If so, re-insert your Product Key.  Hope this helps you.

I've suggested this fix even though your problem is a little different to the others who have it, because it could be that the activation notification you received was false.   Hopefully that was the case and this Registry fix will clear things up.

Issue:  H7361-1253-800705AA on the internet

Method: 1

Add the Netflix Web site to the trusted sites under Internet Explorer's Security tab.

a. Open Internet Explorer by clicking the Start button Picture of the Start button, and then clicking Internet Explorer.

b. Navigate to the website that you want to add to a specific security zone.

c. Click the Tools button, and then click Internet Options.

d. Click the Security tab, and then click a security zone (Local intranet, trusted sites, or restricted sites).

e. Click Sites.

f. If you clicked Local intranet in the previous step, click advanced.

g. The website should be shown in the Add this website to the zone field. Click Add.

h. If the site is not a secure site (HTTPS), clear the require server verification (https :) for all sites in this zone check box.

i. Click Close, and then click OK (or click OK twice if you clicked Local intranet in step d).

You might be getting this error code due to the reasons like an outdated or corrupted graphics card driver, Windows updates, Hyper V etc.

Update the Display adapter.

  1. Press Windows + R on the Keyboard.
  2. Type in Devmgmt.msc and click on Ok.
  3. Select the Graphic driver from the list by extending Display adapters.
  4. Right click on the Graphic driver and click on Update driver Software.
