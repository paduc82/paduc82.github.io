      Have you readied a method of performing a FULL SYSTEM IMAGE backup right before you initiate the upgrade? After all the upgrade to Windows 10 from Windows 7/8.1 could be likened to moving a rug while you remain standing on it - so it is fairly complex and has its risks.  A full system image backup to external (and disconnected during the upgrade) media is a VERY good fallback plan in case something goes wrong or - you just don't like it.

    • System Image Backup - Windows 7
    • System Image Backup - Windows 8.1

    You might even want to cleanup that current system some, make sure *it* is as healthy as it could be before trying to upgrade.  A well-functioning and updated system is more likely to result in a smoother experience for an upgrade as extensive as this...  You might look thru this previous conversation where I suggested someone cleanup their system before upgrading:

    http://answers.microsoft.com/en-us/windows/forum/windows8_1-update/scheduling-the-upgrade-to-windows-10-not-working/13e88e85-2919-4e20-847e-94073e48090c

    Some links that would be different than the ones in the aforementioned suggested cleanup link - if you have WIndows 7 and not Windows 8.1 (and want to do the same cleanup) . . .

    Disk Cleanup: http://www.sevenforums.com/tutorials/818-disk-cleanup-open-use.html

    SFC /SCANNOW: http://www.sevenforums.com/tutorials/1538-sfc-scannow-command-system-file-checker.html

    Defragment: http://www.sevenforums.com/tutorials/11733-disk-defragmenter-open-use.html 
      

    When you think you are ready...

    Use the Media Creation Tool web page to kick off your upgrade.

    The big take-aways there are: device drivers, software updates and BACKUP BACKUP BACKUP!

    You could even make a full system image backup of your current system and CLEAN install Windows 10 using your Windows 7/8.1 product key (from - likely - your Product Key sticker on your machine) and start completely clean with Windows 10 - installing just what you want/need and restoring your personal files/etc. from your last personal file/folder backups (not the System Image.)
        
    Then, if things are not working out (device drivers, software, etc.) you could use the system image backup to restore to 'pre-did-anything' state and continue on.  ;-)
