Microsoft's policy prevents us from giving advice on cracking or bypassing passwords with the following exception.

Windows XP includes a built-in account named "Administrator" for which the default password is blank (i.e., no password).  If you have not changed that account (many people don't even know that it exists), you may log on using Administrator and then change the password for your usual account from Control Panel > User Accounts.

Restart the computer in Safe Mode by tapping the F8 key just after the power has come on.  If you see the Windows logo appear, you waited too long to begin tapping and will have to try again.

Once Safe Mode starts, you should see "Administrator" on the Welcome screen along with your usual user account.  Select Administrator and leave the password box blank.

If you happen to have XP Pro, you can access the Administrator account directly from the Welcome screen without having to use Safe Mode.  Press CTRL+ALT+DEL twice while at the Welcome screen to get the "classic" log on dialog.  Select Administrator as the user account and leave the password blank.

If the above doesn't work because you configure a password for Administrator that you now don't recall, you'll have to seek help someplace else.

jdgregson say:

I had this happen on two different Windows 10 computers in the last two days. There must have been a bug with the last auto-update of OneDrive, once again demonstrating how unstable the entire OneDrive product is. I was syncing OneDrive Personal and OneDrive for Business and they both disappeared, with the entire app missing as you described. Ironically, Google Drive was also syncing on both computers just fine the whole time.

At any rate, I solved it using the following process:

  1. go to "Apps & Features" in Windows 10 Settings (or "Add or remove programs" in Windows 7)
  2. uninstall "Microsoft OneDrive"
  3. go here and click the link in the "Need to reinstall? Click here to download." line at the bottom
  4. reinstall using the download
  5. open OneDrive
  6. sign in again
  7. tell it to use your existing OneDrive folder
  8. you're good to go, until Microsoft breaks this very unreliable product again next week
