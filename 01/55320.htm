Alternative methods for opening Computer Management:

 

1) Open the Start menu, in the search box type Computer Management, it should now appear at the top of the list under Programs.

2) Open the Control Panel (Large or Small icons), then open Administrative Tools. In the right pane of the new window you should see a Computer Management shortcut.

3) On your keyboard press the Logo-key+R to open the Run dialog box, type

%windir%\system32\compmgmt.msc /s

and click OK (or press Enter).

 

In the Computer Management window in the left-hand column you'll see Storage, when expanded you'll see Disk Management directly under Storage.

 

As to why the Manage shortcut in the Computer context menu is not working properly, perhaps someone else will know the exact cause.

 

Since the Computer shortcut on the Start menu is an optional shortcut, you might try recreating this shortcut. Right-click an empty spot on the Taskbar and from the context menu select Properties. In the Taskbar and Start Menu Properties dialog box select the Start Menu tab then click the Customize button. In the Customize Start Menu dialog box at the top of the list you should see Computer with the three options. Change the option to Don't display this item then click OK at the bottom, then OK to close the Taskbar and Start Menu Properties dialog box. I would now suggest rebooting your computer and then going back into the dialog box and select the option to Display as a link.
