"Microsoft driver" is far too generic a description.  What specific driver are you trying to install?

The error message you quoted -- "entry point AddDllDirectory could not be located" -- refers to a programming label inside some file.  Again, this doesn't specify which file is missing.

When you restore from a manufacturer's recovery partition, all the necessary drivers are already included.

Now, all of that being said ...

It turns out that the AddDllDirectory function is a new feature that Microsoft added to Windows 8.  It was not present in the release version of Windows 7.

    Hi Pretzel! Have you tried setting your Xbox One audio output to something like Bitstream Out to see what happens? I know from reading a wide variety of other posts on these forums from time to time that the audio settings can be quite fiddly and sometimes require being set to something you wouldn't EXPECT should be the right thing.
