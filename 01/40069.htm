==============================================
You can accomplish this by creating several small projects in Movie Maker
and publishing (saving) each one in the .wmv movie format to a folder on
your hard drive.

I'm thinking the thumbnail (button) for each Scene (chapter) would be
the first frame in that clip.

Then you would import the collection of .wmv clips into DVD Maker and
each one will become a Scene (chapter)...you can have up to 18 Scenes.

FWIW...this is a good method because large/complex Movie Maker projects
are prone to causing resource issues.

In the following article...read the section:
"How are scenes created for my DVD?"

alzs say:

The system you suggest works well after a few tweaks to address the following issues.

 

1.   I am using Windows 7 (64bit).

2.   Adding Music and Transitions has to be done in each clip.

3.   The Pan and Zoom feature of ‘Slide Show’ cannot be used.

 

The procedure I used is listed below. I worked with a set of ~100 photos which I divided into four groups. The results were very good:

The finished show played perfectly either by running it straight through or by jumping to any of the clip starts, now correctly indicated by the first photo of each set being shown in a thumbnail button.  (N.B. the Pan and Zoom feature could not be used).

 

Procedure 

a)   Assemble the sound track music to the required total length of the finished Show (eg by using the excellentNHC WavePad software).

b)  Cut sound track into sections (in my case 4) again usingWavePad and save each with a file name of your choice as a .wav file.

c)   Drag first group of photos intoWindows Live Video Maker

d)  ‘Select All’ and add a transition for this group (there are 66 to choose from including ‘Random’ or if you wish you can specify a different transition for each slide in the group).

e)   Add the .wav music sound track you saved for this group

f)    Choose ‘Fit to Music’

g)  Save the clip using‘Save Movie’ this will ensure that the clip is saved as .wmv file. (NB all other save options give .wlmv files which are NOT recognised by Windows DVD Maker)

h)  Repeat steps c) to g) for each group of photos in turn.

i)    Open Windows DVD Makerand add all the video clips (4 in my case), using the ‘Add items’ button. Use the various features of Windows DVD Maker as required and finally check the finished result using the ‘Preview’ button on the burn page. When happy go ahead and burn the video.

j)    My set up is a 6GB i7 PC with lots of free HDD space and the procedure above ran without problems – I can imagine there could be resource problems with lower specified PCs.

So overall then the procedure works but it is long winded. It would be much better if Microsoft would improve the DVD Maker Slide Show Makerto allow ‘Jump to’ user specified points.

Also it is sad thatDVD Maker can’t handle .wlmv files!

a. user say:

Hello,

Thank you for posting the query on Microsoft Community. I am glad to assist you on this.

In order to change the location where Windows 10 apps are installed, you must first give yourself (or more technically the Administrators of Windows) ownership and editing permissions of the corresponding registry key. To get started, open the Registry Editor:

  • Press Windows + R to open run command.
  • Type regedit.exe and hit enter.
  • In the Registry Editor, navigate to the following location: HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Appx
  • On the left side of the window, right-click the Appx key (or long tap for touch screens) and click Permissions
  • On the Permissions window, click the Advanced button.
  • On the Advanced Security Settings dialog, click the Change link on top
  • Type “administrators” into the box, click Check Names, and click OK
  • Near the top of the Advanced Security Settings dialog, select the Replace owner on subcontainers and objects checkbox and click OK.
  • Back on the Permissions window, select the Administrators entry from the list, for Full Control select the Allow checkbox, and click OK.
  • Back on the main Registry Editor window, double-click the PackageRoot entry on the right side of the window.

Finally, here’s where you define the location where you want Windows 8 app data to reside.

Type the location of the desired drive or folder you created earlier and click OK.
