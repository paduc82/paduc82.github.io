5) Posted by : artologik
The solution came to me when I could not recreate
the problem on a different laptop with Vista Home.
Comparing everything that I could, I noticed that the
%PATH% environment variable was different between
the 2 machines.

I guess WLPG relies on DLLs that must be found
on the system path; when not found, it must revert
back to "basic or incomplete" libraries that it finds
on default or user specific paths. The solution for
me was to ensure the overall system path pointed
to the proper folders in the proper order.

To solve, do this:

A. Right-click the COMPUTER icon on the
desktop and choose PROPERTIES

B. Choose ADVANCED SYSTEM SETTINGS

C. If prompted by UAC to authorize, go ahead
and ALLOW

D. Click ENVIRONMENT VARIABLES....

E. In the bottom half, under SYSTEM VARIABLES,
locate the PATH variable and highlight it

F. Click the EDIT button

G. Make sure that whatever is there starts with
%SystemRoot%\system32;%SystemRoot%;

H. Make sure a semi-colon separates  each entry.

I. Click OK three times

J. Restart computer

K. Et voilà.

Hi Daniel,

 

 

What is the make and model of the computer?

 

 

Secure Boot helps increase the security of Windows 8.1 and Windows RT 8.1 computers by stopping unauthorized software from running. If you see a "SecureBoot isn't configured correctly" message in the lower-right corner of the desktop, it means that Secure Boot has either been turned off or hasn't been set up correctly on the computer.

 

 

 

To turn on or set up Secure Boot, the customer you may need to use the BIOS menu of the computer. Different PCs have different BIOS menus, so you may need to contact the computer manufacturer for instructions.

 

 

Note: Modifying BIOS/ complementary metal oxide semiconductor (CMOS) settings incorrectly can cause serious problems that may prevent your computer from booting properly. Microsoft cannot guarantee that any problems resulting from the configuring of BIOS/CMOS settings can be solved. Modifications of the settings are at your own risk.

 

Hope this information helps. Reply to the post with updated status of the issue to assist you further.
