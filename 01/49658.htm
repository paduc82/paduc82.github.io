Hi Meinklang,

 

Thank you for reaching to us on Microsoft Community forums.

 

The issue got fixed because you have set the security settings to default. Your efforts to resolve this problem is very much appreciated.

 

However, you may change the user permissions for NWN2 folder. To do so folow these steps:

 

Step 1: Taking Ownership

 

a.       In the Properties window go to the Security tab and click on Advanced.

b.      Now go to the Owner tab. Here you will see the current owner of that folder. If your user is not the owner, click on Edit. If you have User Account Control enabled, press on OK.

c.       In the Owner window, click on Other users or groups.

d.      Now you need to type the user name or the group of users that will take ownership of the selected folder. If you type "Administrators" you will give ownership to all the users that are administrators on your PC. If you want to give ownership only to your user, then type your user name. After that, press on Check Names.

e.      If the user or group was found on your computer, you will see that the text you typed was changed to [Your PC Name]\[User] or [Your PC Name]\[Group]. If so, press on OK.

f.        If the user/group was not found you will receive a Name Not Found error window. Make the necessary corrections and try again.

g.       In the Owner window, select the user/group you just added, select Replace owner on sub containers and objects and press OK.

h.      Now you will receive the confirmation that you have taken ownership of the folder. Press on OK and that's it.

 

Step 2: Changing Permissions

 

a.       In the Properties window go to the Security tab and click on Edit.

b.      If you are not on the list of users or groups that have permissions defined, you should click on Add. If your user or group is on that list, select it, click on Allow Full Control and then press OK.

c.       Now you need to type the user name or the group of users for which you want to change the permissions. If you type "Administrators" you will give permissions to all the users that are administrators on your PC. If you want to give permissions only to your user, then type your user name. After that, press on Check Names and then on OK.

d.      If the user/group was not found you will receive a Name Not Found error window. Make the necessary corrections and try again.

e.      In the Security window, select the user/group you just added and then click on Allow Full Control and finally click OK.

Please feel free to reply, in case you face any other issues with Windows in future.
