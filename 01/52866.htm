Missing and corrupt operating system files (like many DLL files) are arguably the biggest cause of major Windows issues. Considering that, plus the fact that SFC /scannnow is completely automatic and very easy to use, the tool should usually be one of your first troubleshooting steps.

Assuming your wireless internet is functioning, with the printer switched off you insert the printer cd and commence the software install, at some point the installation will ask you to specify the type of connection, you select wireless and continue, eventually the install will ask you to switch on the printer (Note some wireless printers have a switch on the printer to activate their wireless connection) the install should then confirm that it has correctly identified the printer. (If you have an Internet Security app, you may get a warning msg requiring you to allow the printer wireless connection)

The exact install procedure might depend on your printer make/model, and you might be advised to uninstall any printer software via add/remove before starting from scratch
