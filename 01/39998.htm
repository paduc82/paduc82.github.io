I had the same problem, and I solved in this way:

Control Panel -> System and Security-> File History -> Turn off

Settings -> Update & Security -> Backup -> More options -> Stop using this drive

Delete the File History folder in your external drive.

Start using File History again.

I did not modify the default settings about how often and how long backup is saved, File History is working ok now. So I guess it is important to click "Stop using this drive" in order to reset File History.

danno mck. say:

The Optional Media Update has nothing to do with which devices mount on the Xbox 360; it is for interpreting file formats/codecs, such as AAC. For the console to see a device, it has to present itself as a "USB mass storage device". None of the iOS devices do so (iPhone, iPod Touch, iPad, 5th gen. Nano or later), so they'll never be usable on a 360.

I'm a little surprised that the Shuffle does not; in iTunes, does it ever show that it has a "disk mode", where you could mount it on the desktop? If not, that's a sign that it will never work on a 360, either. Maybe I shouldn't be too surprised: the older Shuffles we have (2nd gen.), can't play and charge at the same time-- they have to be in one mode or the other, so I doubt they'd work on the 360, either. (They aren't simple flash drives, obviously.) So if the Shuffle were somehow "mounted", I'd be afraid that the charge was draining and the time to use it was too brief.
