benjie_c say:

Hello

Thank you for reaching out to the community!

Just to let you know, by default, when you create a chat group, all users have the permission to remove or add members. You should set your role as Admin first before you can set the permission for the other members within the group. To do this, on the chat of the Skype Group you created, type "/setrole myusername Admin" to make your account to be the administrator. You will be shown a message saying you are now the Administrator.

After doing so, you should be able to set the permission for all the other members.
