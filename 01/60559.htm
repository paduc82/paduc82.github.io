a. user say:

Hello,

The error code 0x80070103 occurs when Windows Update tries to install a driver a second time for an additional piece of identical hardware such as a graphics adapter, and if it tries to update a currently installed driver to an unsuitable version of that driver. For the most recent available drivers for your computer, kindly check with the device manufacturer. This issue can be safely ignored if this applies to a driver.

However, the driver may be offered again by Windows Update in the future. You may hide the update so that it is not offered again. Here's how:

  1. Press the Start button, then select All Programs, then click Windows Update.
  2. Select Check for updates.
  3. After the scan for updates is complete, click the Optional updates are available link.
  4. In the Select the updates you want to install section, right-click the update you want to hide, and then click Hide update.
  5. Click Install Updates to install any additional updates that are selected. If no other updates are select, close Windows Update.

If you need further assistance in updating Windows 7, feel free to ask us.
