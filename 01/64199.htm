For those who found this thread and have Windows 8.1 and even those upgrading from 7:

1) move your files onto OneDrive

2) create the ISO

www.microsoft.com/en-us/software-download/windows10 and use RUFUS to make bootable flash drive

3) test if you are able to boot to your Win10 iso flash drive (some systems need GPT rather than MBR or may need BIOS tweaking)

4) uninstall Office and any other applications that you know how to reinstall after the upgrade

5) Win8 users can create a system recovery USB and boot to it in order to factory reset and clean the disk or just wipe the hard drive and reinstall Win8 if you are able to. (Win7 needs all the updates before it can be upgraded but 8.1 doesn't)

6) Boot the Win10 iso upgrade

7) Login with you Hotmail/Otlook.com/OneDrive credentials and be sure to disable sync all folders. I only mapped two folders with essentials files in order to avoid filling up my hard drive with cached files.

8) If all seems well with Win10, go to System and search "cleanup" to use the tool to free up space; run the tool a second time as administrator in order to remove the old Windows installation.

ken blake - microsoft mvp say:
I have a Ben Q Mirascan V5.01 flatbed scanner but since i upgraded to a windows 7 tower it doesn`t recognise the driver or cant find it.I have reloaded the software disc and still nothing.

Whenever you change to a new version of Windows, you need to install drivers (software that provides the interface between the hardware and Windows) for printers, scanners, video cards, etc., that have been written for that new version of Windows.  And also note that you need different drivers for a 32-bit version of Windows and a 64-bit version.

 

It's up to the manufacturer of the hardware to write and make available for download those drivers. Especially if your hardware is older, it may  well be that the manufacturer has decided that it's not worth his time and effort to develop drivers for hardware that he is no longer selling.

 

 

Check with the manufacturer or on their web site to see if a driver exists. If it doesn't, you're out of luck. Also check their web sites in other countries, if there are any; sometimes you can find the driver you need that way.

 

So in the future, before you do such an upgrade, be sure that you've done the appropriate research to find out what drivers don't exist, and therefore what hardware you will have to replace if you upgrade.

pastorjdh say:

It is very simple to fix. There are several previously installed updates that are blocking the installation. Go to control panel / uninstall a program / view installed updates (top left corner) and then delete these two updates: KB2966826, KB2966827 and KB2966828. Then go back to windows update / view installed updates / enable windows features and check the .net framework 3.5. Then windows update will download it without saying it cannot connect to the internet. Windows update will reinstall the two removed updates later without any ill effect. 

If this does not work look for any other updates related to. NET Framework 3.5 and uninstall them. The problem is that it cannot install the original .NET Framework 3.5 after updates for it have already been installed. For some reason, Microsoft Update thinks that .NET Framework 3.5 is already on Windows 8.1 and we all know it is not.
