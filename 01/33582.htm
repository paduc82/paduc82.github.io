lemp say:
Almost certainly, your installation of Norton 360 Premier has nothing to do with this issue (although you may yet come to regret that decision).

Furthermore, neither of the articles that "will surely help you" will in fact help you at all with this issue.

By default, Windows uses a black font for normal files, a blue font for files that have been compressed to save space (usually by checking the box in Disk Cleanup to "Compress old files to save space"), and a green font for files encrypted with the Windows Encrypting File System (EFS). I recommend that you stay as far as possible away from EFS.

If you have a relatively new computer, you won't notice much ill effect from having the file(s) compressed.  Windows automatically uncompresses the file(s) "on the fly" whenever you open them.  This compression, however, usually doesn't save all that much space and sometimes can cause very long delays in the operation of Disk Cleanup. For that reason, many users disable the option: http://support.microsoft.com/kb/812248

Note the this compression is not the same as ZIPping the file(s).  See http://support.microsoft.com/kb/307987?wa=wsignin1.0

If you want to uncompress a file manually, right-click, select Properties, and click the "Advanced" button on the "General" tab.  Uncheck the box and OK your way out.

You can turn this behavior off or even change the colors associated with these file characteristics.  See http://www.tomshardware.com/forum/36386-45-file-colors

   Information
Data Execution Prevention (DEP) is a security feature that can help prevent damage to your computer from viruses and other security threats. Harmful programs can try to attack Windows by attempting to run (also known as execute) code from system memory (RAM) locations reserved only for Windows and other authorized programs. These types of attacks can harm your programs and files. If DEP notices a program on your computer using memory incorrectly, it closes the program and notifies you with the DEP stop error. DEP is ran in a software mode and in a hardware mode. If your processor supports DEP (NX for AMD, and XD for Intel), then you will have hardware and software DEP. If it doesn't, then Vista will just use software DEP. 64 bit applications will have DEP enabled all the time by default, but 32 bit applications do not and must have DEP manually enabled for them.
