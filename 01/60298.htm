Follow the steps in this section, in order, until the problem is resolved.
Step 1: Disabling the automatic restart option to view error messages
Many hardware driver or operating system errors will cause the computer to display a specific error message before stopping the operation or shutting down the computer. However, if the Automatically restart option is enabled, an error message might not be displayed despite an error taking place. Disable this option to allow the computer to display the error message.
Follow these steps to disable the Automatically restart option:

In Windows, search for and open View advanced system settings.
Click Settings in the Startup and Recovery section.
Figure : System Properties window
System Properties window
Remove the check mark next to Automatically restart under System Failure, then click OK.
Figure : Automatically restart check box
Location of the Automatically restart check box
Restart the computer.

note:
The computer no longer restarts automatically when a problem occurs. Instead, a blue screen appears with more useful information. You can now search for a resolution using the information in the error.

Step 2: Checking for failed hardware
Hardware problems can cause the computer to restart unexpectedly. For more information about checking for hardware problems, see Testing for Hardware Failures (Windows 10, 8) for notebook computers (https://support.hp.com/ee-en/document/c03467259).

note:
Failed hardware is usually caused by increased heat inside the computer. Over time, cooling areas can become clogged with dust buildup causing temperatures inside the case to rise. For more information, see Reducing Heat Inside the PC to Prevent Overheating for notebook computers (https://support.hp.com/ee-en/document/c01657439).

Step 3: Updating the BIOS
A BIOS update can resolve issues with the computer shutting down unexpectedly. Follow the steps in the following sections to use HP Support Assistant, the HP website, or HP SoftPaq Download Manager to check for and download any BIOS updates.

You can use HP Support Assistant to search for a BIOS update. If HP Support Assistant finds an updated BIOS, you can download and install it.
Use the following steps to download and install the latest version of the BIOS.

In Windows, search for and open HP Support Assistant.
On the HP Support Assistant main page, click the Updates button in the My PC box.

Click Check for updates and messages.

HP Support Assistant connects to the HP servers, collects your system information, downloads updates, and displays the results.
If HP Support Assistant finds a BIOS update, click the Install now icon () to download and install the updated BIOS and device drivers.
If HP Support Assistant does not find an update, your computer has the latest BIOS available.

arminatorx say:

[quote user="el n1nj4"]

Make sure your Wireless and LAN adapter on your Laptop are both set to automatic IP adress. Also make sure your console gets the IP address automatically.

Now dis-connect console from the Laptop, and select both WLAN and LAN icons on your PC (if necessary, hold the CTRL key while you click on each network adapter icon). Then right click and choose to bridge the connection.

Now connect the XBox again to the Laptop and make sure the Laptop can access the internet through your WLAN router properly. Check the IP Address of the WLAN adapter. With most routers it should be something like 192.168.xxx.yyy or 10.xxx.yyy.zzz

Now go to the network settings of your console and check the IP address there. The first three octets should match the IP Address of your Laptops WLAN adatper (i.e. if your Laptop is 192.168.1.23 your console should also have 192.168.1."something_other_than_23")

If that is the case, your console should connect.

If not, try to manually enter the IP information, matching EVERYTHING from the Info on your Laptops WLAN (i.e. default Gateway, Network Mask, DNS). However do NOT copy the IP address. You HAVE TO change the last octet to a free IP address in your network.
