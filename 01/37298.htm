a. user say:

Hi Catalin,

Thank you for your interest in Windows 10!

You can certainly upgrade to Windows 10 even if you have downgraded to Windows 7 from Windows Updates.

Alternatively, you can upgrade your current PC to Windows 10 by going to the Windows 10 Software Download page to download the Media Creation Tool. The Media Creation Tool can be used to create a USB or DVD media to upgrade your current PC to Windows 10 or install Windows 10 on a different PC.

Follow the instructions on Windows 10 Software Download page: http://www.microsoft.com/software-download/windows10

Note:

When upgrading to Windows 10 using media, you must run setup.exe from within your Genuine and Qualified version of Windows 7 or Windows 8. If you want to boot from media, you must have upgraded to Windows 10 previously. Boot from Windows 10 media directly without upgrading to Windows 10 previously is not supported and will have activation issues.

Hope this information is helpful. Do let us know if you need any further assistance, we'll be glad to assist you.
