Hi,

 

Thank you for posting on Microsoft Community.

This generally occurs when the file specified in the error message is in the TypeLib table, but the RegisterTypeLibraries action cannot register the file.

 

Follow the suggestion below and check if it helps.

 

Method 1: Reregister the msxml.dll file and check if the issue persists.

  • Type “cmd” on the Start screen.
  • Right-click on it and select “Run as administrator”.
  • Type the following command in the command prompt.
    • regsvr32 msxml4.dll

Method 2: Perform sfc scan and check if you can install the application.

 

This tool will try to replace the missing or corrupted system files (Dll files).

  • Type “cmd” on the Start screen.
  • Right-click on it and select “Run as administrator”.
  • Type the following command and press Enter: sfc /scannow

microsoft mouse stopped working on my Dell laptop How can I get it to - Microsoft Community

carolrowell say:

microsoft mouse stopped working on my Dell laptop How can I get it to respond

Microsoft mouse stopped responding to my Dell computer. How can I get it to work?

greg carmack - mvp 2010-2020 say:
Hi Carol. I'm Greg, an installation specialist and 8 year Windows MVP, here to help you.

Can you use a different mouse to troubleshoot this? If not you may need to work in Safe Mode or Repair Mode which load different drivers that usually work, following whichever method works here to get into Safe Mode https://www.digitalcitizen.life/4-ways-boot-saf... or Repair Mode's Advanced Troubleshoot Options to System Restore: https://www.tenforums.com/tutorials/2294-advanc...

If you can use another mouse or it works in Safe Mode there is a known issue of incorrect drivers being pushed by Windows Update which requires rolling back the driver. Right click Start to open Device Manager, choose mouse device, then Driver tab, then Roll back.

If there is no Roll Back or that doesn't help choose Update Driver on device Driver tab, then Browse Computer, then Let Me Pick to see a list of all previously installed drivers, work backwards trying each to find one that works.

If necessary you can also go to Settings>Update & Security>Windows Update to look for the Windows Update or driver under Installed Updates and Uninstall it form link there. Then you can hide the Update or driver so it isn't pushed again using the Hide Updates tool downloaded from here: http://download.microsoft.com/download/f/2/2/f2...

If none of those work try to System Restore to a point before the problem began: http://home.bt.com/tech-gadgets/computing/how-t.... Make sure System Restore is always enabled for C with plenty of disk space apportioned (5-10%) in its Configuration settings.

As a last resort I'd install Media Creation Tool and choose Upgrade Now to do a Repair Install, which will also advance you to the latest version (which you need anyway) if necessry, by the most stable method: http://windows.microsoft.com/en-us/windows-10/m...


I hope this helps. Feel free to ask back any questions and let us know how it goes. I will keep working with you until it's resolved.
