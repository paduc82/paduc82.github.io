judd - say: Hello RDV25,

Thank you for using the Microsoft Answers Forum

Have you cleaned your fans lately? You should check it at least weekly if not more, to make sure everything is running smooth. An update most likely is not the cause. If the fans are dirty clean it out with a can of compressed air. Compressed air can be found at any local electronics store. 

You may also want to check in task manager for the highest memory processes or highest cpu process(besides System) because typically when large processes are running it can cause one of the fans to run constantly to cool down the computer so it can handle running the process. Safe mode can really help determining if this is the cause of the fan problem; you can do this by running in safe mode, usually by pressing F8 or another key depending on your computer manufacturer, and by seeing if it runs non stop when you run it in safe mode. Another problem could be because of the fan wearing out and not moving air as efficiently as before.

To check your Task Manager for running programs you can perform a control+alt+delete keypress and see if there are programs running that will cause your fan to run excessively.
