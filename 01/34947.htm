Look again:




How do I choose an alias to send from using Windows Live Mail


You can only send from accounts that are set up and show in the folder pane, so you have to set up an account for each alias:

  1. Click Work offline
  2. Press Ctrl-Shift-T to start the Add your email accounts wizard
  3. Enter the alias address and clear the Remember password box
  4. Select Manually configure server settings and click Next
  5. Enter a.b.com for the incoming server address
    Enter smtp.live.com for the outgoing server address
    Change the outgoing server port to 587 and select both Requires a secure connection (SSL) and Requires authentication
  6. Click Next and Finish. If the program tries to connect, click Cancel.
  7. Right-click on the new account name in the folder pane and select Properties.
    Give the account a suitable name and enter the alias address in the Reply address box for safety's sake.
    Deselect Include this account when receiving mail or synchronizing.
  8. On the Servers tab, click Settings
    Click Log on using and enter the primary email address and password, then click OK
  9. Close the properties dialogue


[That takes ten times longer to type than to do.]

With the new account still selected in the folder pane, press Ctrl-N to open a new message compose window. Compose a test message; check that the alias address is showing in the From box. Send the message to the same address.
