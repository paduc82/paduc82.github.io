ecb313131 say:

what's happened to the picture shuffle for desktop in windows 10?

I just installed Windows 10, (up from Windows 8).

 I love having a random picture on my desktop every time I boot up, how can I get that feature back? - pictures from my pictures folder (no, I do not want to access online  pictures for this)

thierry926 say:

You can fix that by using the Registry Editor.

Start the Registry Editor by entering regedit in the search bar at the bottom of your screen and run as a command.

Click Yes if asked whether you want to allow this app to make changes to your PC.

In the Registry Editor window, left column, select HKEY_CURRENT_USER.  Then select Control Panel > Personalization > Desktop Slideshow.

On the right hand side, look at Shuffle.  The binary value should be 1.  If it is set at 0, right click on Shuffle and select Modify and enter 1 in the Value Data field.

You can also change the interval between pictures by editing the field Interval.  The numeric value is in milliseconds.

Hope that helps.  It certainly worked for me.
