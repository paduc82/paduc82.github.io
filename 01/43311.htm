right click the memory card drive letter and from the menu
choose Paste.

the Root Directory of the memory card...you may see

Files generated by Windows and apps the operating system does not consider essential or not created by applications you normally use like Microsoft Word are deleted. 
Windows over time will accumulate data outside of the normal types of data we store. Sometimes we don't need this type of information and removing it can free up precious disk space inaddition to improving overall system performace.

Please becareful though, this folder can actually be useful if you need to go back to your previous version of Windows for whatever reason. So be mindful before deleting it. This is what I do - hopefully it will work for you:

1. Open the action center (click the icon that looks like a word balloon in the task bar next to the date/time in the lower right corner)

2. Click on the "Connect" box (right next to the tablet mode box)

3. An overlay should come up saying it is searching for wireless display and audio devices on the right side of the screen

4. If you haven't already, put your TV into Screen Mirroring mode

5. When the device(s) show up on the overlay on the right, click on the device you want to connect to.
