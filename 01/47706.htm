The System Volume Information folder is where all of the System Restore and Previous Versions shadow copies of files are stored.

Use the following procedure to remove the older System Restore and previous Versions of files.

Go to Start and type  Disk Cleanup  in the Search Box. In the Results, right click Disk Cleanup and select the Run as Administrator option.

Select the Drive you want to clean up and click OK.

After the program scans the drive, check all of the files you want to remove.

Next, click the More Options Tab. In the Sytem Restore and Shadow Copies section, click the Clean up Button.

Click OK.

Click the Delete button in the confirmation popup to delete all but the most recent System Restore point.

Wait for the program to remove all of the selected files. Check the disk space that is currently available.

You should also open Control Panel and select System. On the left side of the window, click System Protection. Select your System drive (C:).

Click the Configure button. This is where you can set the System Restore settings, including the Max Usage for the System Restore files.
