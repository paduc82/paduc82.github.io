SMCD3GN

 

Login Information

  • Username - cusadmin
  • Password - password

 

DHCP Reservation

*This feature is not available.*

 

DMZ

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. Login using the username “cusadmin” and the password “password”.
  4. From the menu on the left select Firewall.
  5. From within the Firewall submenu select DMZ.
  6. Check the enable DMZ Host box.
  7. In the space to the right enter the consoles IP Address.
  8. Click the Apply button at the bottom of the page.

Disable DMZ

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. Login using the username “cusadmin” and the password “password”.
  4. From the menu on the left select Firewall.
  5. From within the Firewall submenu select DMZ.
  6. Uncheck the enable DMZ Host box.
  7. Click the Apply button at the bottom of the page.

 

Port Forwarding

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. Login using the username “cusadmin” and the password “password”.
  4. From the menu on the left select NAT.
  5. From the NAT submenu select Port Forwarding.
  6. Click the Add button under the Customer Defined Service Table.
  7. In the Name field enter XBL or something that indicates Xbox Live.
  8. In the type drop down select TCP/UDP.
  9. In the LAN Server IP enter the IP address of the console.
  10. For the Remote IPs select Any.
  11. From Public IP Ports drop down select Single Port.
  12. In the Start Public Port box enter 3074.
  13. Leave End Public Port blank.
  14. In the Private Ports box enter 3074.
  15. Click the Apply button.

Disable Port Forwarding

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. Login using the username “cusadmin” and the password “password”.
  4. From the menu on the left select NAT.
  5. From the NAT submenu select Port Forwarding.
  6. Select the service you want to remove and click Delete.
  7. Select Apply.

 

UPnP

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. Login using the username “cusadmin” and the password “password”.
  4. Select System from the menu on the left.
  5. Toggle the UPnP on or off depending on the needs of you situation.
  6. Click the Apply button at the bottom.

 

Firewall

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. Login using the username “cusadmin” and the password “password”.
  4. From the menu on the left select Firewall.
  5. On this screen set the Firewall to Disable.
  6. Click the Apply button.

 

Restoring Defaults

Through Hardware

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Locate the Reset button.
  3. Using the tip of a ballpoint pen or toothpick press and hold the reset button for 15 seconds.
  4. Release the reset button.
  5. Wait for the device to restart.

Through Firmware

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open a web browser.
  3. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  4. Login using the username “cusadmin” and the password “password”.
  5. Select Tools from the menu on the left.
  6. Select Configuration Tools from the submenu on the left.
  7. Click the Factory Reset button.
