retired engineer d say:

Hi Viral

1. First check the BIOS version on your computer (via System Information) and update it to version A11 if necessary (download BIOS A11 from dell.com).

2. Next open the computer's Task Scheduler (via Control Panel) and check the run time set for "Application Experience" (expand the Task Schedule Library/Microsoft/Windows).

3. If the BIOS has been updated to A11 recently, either edit the trigger run time of Application Experience to a date/time when the computer will be powered on or leave the computer powered on until after the listed scheduled run time of Application Experience - as Application Experience will not check the BIOS version until it has been run via the Task Scheduler and so may keep reporting that the BIOS does not support Windows 10.

When I followed the above steps my Dell N5110 upgraded from Windows 7 to Windows 10 without problems.

    keith a. miller say:

    The "My..." set of hidden 'folders' aren't really folders. They are junctions, file-system level shortcuts that point to another folder. There are a number of them on a Windows 10 install; they are there for backward-compatibility with older software.

    If you open a command prompt to a folder containing a junction, and use type:

    dir /alh

    it will list the junctions and their targets.

    While you can't browse the junctions, you can traverse them:

    If you paste the path:

    C:\Documents and Settings\<YourUserFolder>\My Documents\My Music\<subfolder of Music>

    into the Address Bar of a File Explorer window, you will see the contents of:

    C:\Users\<YourUserFolder>\Music\<subfolder of Music>

    Bottom line: They were hidden for a reason, you don't need to worry about them.

    Keith
