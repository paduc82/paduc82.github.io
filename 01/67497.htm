Hello Acht27,
I apologize for this inconvenience and think I can definitely shed some light on the situation.

The reason the "Upgrade Assistant" only downloads versions that matches your current version is that it is "Impossible" to run a x64-bit Windows on a x86 or 32-bit processor. The issue is not that it won't let you download the version you want but the version that is compatible and safe to use with your computer. 

A 32 Bit processor does not have the "Spunk" to run a 64 Bit OS, the processor would quickly wear out and literally Cook itself from the inside out destroying your entire computer. If you want to run a 64 Bit OS first you have to check to see if Your Laptops Motherboard supports 64 bit processors and then you will have to order a 64 bit processor. Most likely your motherboard will be incompatible or it would have came pre-installed with a 64 Bit processor.

sc tom say:

If the mouse receiver is unplugged, do you get the same behavior with just the touchpad? If so, then either the controller is going or the driver is corrupt. Reinstalling the touchpad driver would be easier to test.

Another thing you can do (after the driver has been reinstalled) is go into Control Panel and disable the touchpad before plugging the receiver in and test the wireless mouse that way.

If you are having to wiggle the receiver for it to make contact, then the USB port is worn/broken to where it's not holding the receiver tight (try another USB port if you have one), or the receiver itself is worn or damaged. Look at the plug end of the receiver and see if the outer shield is flat and rectangular all the way around. If not, you may be able to straighten it out using a pair of needle-nose pliers. If the inner part of either the receiver or the socket on the laptop is cracked or warped, then it's defective and needs to be replaced.
