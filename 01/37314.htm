grlfrdy say:

I would like to be able to access Outlook from my laptop when the messages are recieved on my main workstation pc, any tips on how to achieve this? thanks for any help.
brian tillman [outlook mvp 2007-2019] say:

There are several ways to achieve this.  One way is to use a desktop connectiopn tool like RDC, VNC, GoToMyPC, or Teamviewer.  WIth tools like these you're actually not using mail on both PCs, but using one PC to access the other PC where Outlook resides.

Another is to install the Outlook on the second PC and configure an account that will access the same mailbox as the one the first PC accesses.  You won't share contacts and calendar, but you can receive and send mail from either or both PCs using the same mail address and it's not too difficult to get contacts and calendar from one PC to the other.

A third way is to put your Outlook data file in a folder that you share on your local LAN and access that data file from either PC.  While it's not a recommended approach, it does work, provided you don't try to access the data from both PCs at the same time, and you'll have all of your mail, contacts, calendar, tasks, etc. available on both PCs.  You'll need to make sure you back up your Outlook data frequently as well, just in case, because Outlook isn't designed to operate in this configuration.

You can also buy third-party tools that will allow the type of sharing you want.  These are safer that sharing a data file on the network, but they're not free.
