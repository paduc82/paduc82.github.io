How To Find Wireless Passphrase (WPA/WEP)

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open a web browser.
  3. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.

  5. Click on Wireless Security from the menu on the left.
  6. The Wireless Passphrase will be displayed next to Security Key/Passphrase.

 

 

Changing the Wireless Security Type/Disabling Wireless Security

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open a web browser.
  3. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  4. From the menu at the top, click on Wireless Setup.
  5. From the menu on the left, click on Wireless Security.
  6. From the drop down menu labeled Security Type, select the desired security type. If no wireless security is desired select Security Off.

The next page shows my harddrive (one hard drive) and has the buttons along the bottom of the screen to select your drive / partition onto which you want to install Windows.

I entered Shift + f10 to go into the command prompt.

And I keyed in:

Select Disk 0 (that is the only disk)

CLEAN

CREATE PARTITION PRIMARY

FORMAT FS=NTFS

Then I go back to the install setup GUI, and I select Disk 0 and when I click on next, the setup wizard gives me the error message "Cannot locate Partition"

Any idea on what commands I need to perform from DISKPART to prep my disk so that Windows can continue with the install?

Thank you

jwconk say:

Thank you  to everyone who replied, but especially to JW Stuart.

Finally my friend Rodney in Charlotte solved the conundrum!!!!

The problem was not the SSD...

My Carbon X1 requires USB EFI boot.

EFI bootable USB needs to have a FAT32 FS (no one seemed to know this...).

Many people told me to use Windows7-USB-DVD-Download-Tool to create the bootable USB, but it does not format the USB to Fat32.

Therefore, I did not need to make any changes on my SSD (format quick fs=ntfs is fine) but I did run the following commands on my USB prior to loading the ISO:

DISKPART

SELECT DISK (usb)

CLEAN

CREATE PARTITION PRIMARY

SELECT PARTITION 1

ACTIVE

FORMAT FS=FAT32 QUICK

ASSIGN

EXIT

Just to be sure, I used the Mount function of Windows 8 to copy the ISO files to the USB to assure that whatever program mounted the ISO did not reformat the USB.

AND IT WORKED!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!

Burning a DVD with movie maker - Microsoft Community

ginger p say:

Burning a DVD with movie maker

I have made a slide show with music in Movie Maker.  I want to burn it to a DVD. When I go to "help" it tells me to click on "burn to DVD" in "save movie" - but THERE IS NO SUCH CHOICE! What can I do?

a. user say:

Hi Ginger,

It appears that you are saving it as a project file. Take note that saving a project file is totally different from saving a movie. Saving a project file will only save the current work and edits you've made to your movie It is more like a blueprint of what you are trying to build. Saving a movie is rendering, applying the edits you've applied to the movie and produces a playable video file making it as the finish product.

 

A project file has a file extension name of .wlmp. It is not a playable and can only be opened using Movie Maker. A finished movie that Movie Maker makes has an extension of .wmv which is the one you can play on Windows Media Player and can burn to your DVD.

 

On the other hand, the Burn a DVD option will only appear if your Windows OS detected a default burning software. If you are using Windows 7, the default burning software is Windows DVD Maker which is preinstalled in the OS. If this does not appear in your programs menu, it's possible that the program is disabled as a Windows Feature. To have Windows DVD Maker enabled in your Windows OS, follow these steps:

 

Note: Make sure Windows Movie Maker is not running while doing this. Please be reminded to save your project before closing Movie Maker.

1. Open the Control Panel from the Start menu.

2. Click Programs.

3. Under Programs and Features, Select Turn Windows features on or off.

4. Look for Media Features folder and click the [+] to expand the folder and then make sure that the box is checked beside Windows DVD Maker folder to enable it on your computer.

5. Click OK to save the changes.

 

Once done, open Movie Maker, re-open your project and then verify if the Burn a DVD option is now available when saving the movie.

 

In case you are using Windows 8, you will have to set save the movie (using recommended, HD, or other option which suits for you) first before burning it using any third-party software. You can find a lot of burning software in the web via Bing which shows a lot of good reviews. Windows DVD Maker apparently is not included anymore to Windows 8 as a preinstalled feature program.
