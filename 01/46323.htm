Bootmgr is missing message or black screen with blinking cursor

Technical Level : Intermediate

Summary

This error message is displayed when either the bootmgr file is corrupt or missing or the BCD (Boot Configuration Data) is corrupt or missing.


Details

If method 1 does not solve the problem and you are still unable to load Windows 7 then see method #2 which completely recreates the BCD store.

Note: If you have black screen with no error message, just a blinking cursor skip method #1 and use method #2.

Be aware that if the corruption was caused by a failing disk drive with the bad disk drive sectors located in the area (normally a small 100MB System partition) used to store the bootmgr file and or BCD then your chances are not great that you can still boot to Windows 7 after completing the steps listed below.

Method 1:

First change your BIOS settings so that the DVD drive is the 1st boot device, followed by the Hard or SDD drive, Save and Exit the BIOS changes and then follow the instructions listed below.

If you do not have access to either a Windows 7 System Repair DVD or installation DVD then download the appropriate Windows ISO (with SP1 included) file and use it to create an installation DVD.

Windows will load the necessary files needed to boot from the DVD.

b) Select the correct Language, Time and Keyboard input method and click ‘Next’.

Note: If you are using the Windows7 Install DVD then at the 'Install Now' prompt (Do not pick the Install Now option) instead choose the 'Repair your computer' option located in the lower left of the display.

c) The System Repair tool will search for a valid operating system and may or may not find / list your Windows 7 Operating System. By default the “Use recovery tools that can help fix problems” is selected, make no changes and click ‘Next’.

d) In the ‘System Recovery Options’ window click on the ‘Startup Repair’ option.

Hi Gill,

If your Surface is unable to find your wireless printer, you can try these solutions: 

Check the power

Make sure that the printer's electrical cord is plugged into an electrical outlet and that the power switch is turned on.

Check wireless connection

  • Make sure the printer’s wireless option is turned on and available. Many printers have a button that displays a blue wireless icon when a wireless connection option for the printer is available. To learn where this button is located on your printer and to find instructions about how to turn it on, see the instructions that came with the printer or check the manufacturer's website for instructions.
  • Run the printer’s wireless connectivity test. Many printers have a menu option to test the printer’s wireless connectivity. Read the instructions that came with the printer or check the printer manufacturer’s website for instructions about how to do this.

Run the printing troubleshooter

Download and run the printing troubleshooter.  It can fix problems with installing and connecting to a printer.

Manually install your printer

To do this, follow these steps:

  1. Type in printer in the search box.
  2. Select Printers & Scanners.
  3. Click Add a printer or scanner.
  4. Choose The printer that I want isn't listed.
  5. Select Add a Bluetooth, wireless or network discoverable printer.
  6. Choose the printer.

Update the printer's drivers

  1. Press and hold (or right-click) the Start  button, then select Device Manager.
  2. Expand Printers, find your printer in the list, and then select Update driver.
  3. Select Search automatically for updated driver software.

If Windows Update can't find a driver for your printer, and the printer didn't come with software that installs a driver, check the manufacturer's website.

Also, you may need to clear spooler files and restart the spooler service. The print spooler is a file that manages the printing process.

To clear and reset the print spooler:

  1. In the search box on the taskbar, type Services, then select the search result of the same name.
  2. In the Services list, double-tap Print Spooler.
  3. Select Stop > OK.
  4. In the search box on the taskbar, type %WINDIR%\system32\spool\printers, select the file folder in the search results, and then delete all files in the folder.
  5. Search for and open Services again.
  6. In the Services list, double-tap Print Spooler.
  7. Select Start, and in the Startup Type list, make sure that Automatic is selected, and then select OK.
