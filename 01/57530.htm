    1. If the line supports a particular speed, say 100mbps, then it depends on the quality of the line (hardware, cables) etc. to achieve the optimum speed.

    Hi

     

    You can Disable programs that are configured to start when you boot the computer, using the Task Manager.

     

    1. Go to the Desktop.

    2. Right click the Taskbar and select Task Manager.

    3. Click the Startup Tab.

     

    This will display a list of the Startup items.

     

    Right click any item and select the Disable option.

     

    Note:

    Make sure that you do not disable any hardware drivers that may be critical to the operation of the computer.

     

    Note 2:

    This does not uninstall or remove any item from the computer. You can always go back and Enable any Disabled item.
