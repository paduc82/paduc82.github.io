techfreak_ say:

There is only 2 years of call history stored in the Skype cloud. This is the reason it only goes back to ~ April 2017. Since Skype's servers have never stored more than 2 years, any history greater than 2 years would rely on the history stored locally on your device in your main.db file (used by classic versions of Skype). If you still have the device you used in the past 4 years and your main.db file exists and is not damaged, you should be able to retrieve 4 years of history. 

To retrieve call history from a main.db file, you have two options:

1. You can use third party software like SkypeLogView(er) [nirsoft.net] to view the data contained in the main.db file.

2. Use the computer that contains your history with the main.db file or copy your Skype name folder onto your new computer. You can use the export feature in Skype for Windows Desktop 8.x to export that older history to a searchable .html file. You might have to recreate a Skype folder in the %appdata% folder on any other computer and copy the Skype name folder, so that the desktop software can detect that older history.

You can export your ~2 years of Skype history from the Skype Export page. You make a request and check the page at a later time. Once the information is collected you can download that export instance for the next 30 days. Extracting this .tar exported file to a folder along with extracted the .zip file of the Skype parser viewer, will allow you to view this ~2 year cloud history. The index.html file of the Skype parser viewer, will allow you to view the messages.json file and its connected content, in your default web browser.

Skype FAQ on this process:

How do I export my Skype files and chat history?
