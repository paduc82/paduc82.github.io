How To Retain Personal Font Settings in WordPad - Microsoft Community

larry - 'flex' say:

How To Retain Personal Font Settings in WordPad

How to preserve personal settings for font type and size, plus line and paragraph spacing in Windows 7 Rich Text Documents, while maintaining customary usage:

Since so many people around the web keep struggling with this issue, I feel compelled to submit this tried-and-proven method, which is simplicity at its best. Briefly, to the Guru's on this forum who devote so much of their time in helping others -- "Thank You". The following post is long-winded out of necessity, but I am sure you will like the short and easy step-by-step instructions at the end...

First -- The Bad News

A few workarounds previously suggested here and elsewhere come close, but not close enough. Users are seriously hampered by confusing and/or incomplete sets of instructions; or having to adopt complex usage patterns; or being forced to open a WordPad template from only a single location (i.e., when pinned to a Taskbar jumplist, or creating and using a shortcut); or having to perform steps that are totally unnecessary like having pre-typed text in their newly created documents; or being told to change the Properties' default path to the WordPad Program itself (a popular PC mag); etc... And all-the-while, each time this topic is revisited an ever-changing solution is proposed.

Next -- The Good News

For over a decade now, I have used WordPad quite extensively for simple everyday documentation purposes (rather than my Office Suite), and obviously millions more can claim the same. I now would like to share with everyone the silver bullet which allows me to open a 'CLEAN' New Rich Text Document, with ALL of my desired font and spacing settings retained, and from any location on my PC except the Start menu.

This method integrates with File drop-down menus for ALL partitions and folders, as well as the right-click context menu in ALL locations including the desktop. The only exception being when opening the WordPad Program directly via the Start menu, where you WILL in fact continue to get the Windows default settings, if needed.

I truely can not remember the last time that I personally went so far out of my way as to open a New Rich Text Document from the Start menu, because work is simply not performed in that location. And there is no need to use a Jumplist or shortcut for WordPad because I am able to simply create a new customized file from within it's intended location as usual, including right-clicking on the desktop as desired.

Now -- The Really Great News

This registry insertion is actually quite elementary and works flawlessly, plus the instructions are easy for most users to understand, and the process takes at most about 5-minutes to implement. New personal settings for font and spacing are in fact preserved in a template document, and settings can be altered in the future at will.

The procedure merely utilizes a template subroutine embedded in Win7, to bypass WordPad's annoyingly entrenched default settings. The location for your personal template is the same place where the default Read-Only Journal Document resides for Windows Journal to access under Tablet PC -- it's a very similar process.

The re-naming functions remain unchanged in behaviour as well, so NO MATTER the name given a new document (including the template's exact title), the template will NEVER be overwritten even if it's not made Read-Only.

This operation is also totally reversible, simply by deleting the template file and the registry String Value that are initially created, without loss of function or stability. The WordPad Program's default settings and Properties are in no way altered, via the registry or otherwise -- they remain wholly intact, as they absolutely should.

         Windows 7 Defaults // My Desired Settings

Font Type:              Calibri // Arial

Font Size:                      11 // 12

Line Spacing:             1.15 // 1.0

Paragraph Spacing: Added // Removed

Step 1 - Creating a Personal Settings Template

a) In C:\Windows\ShellNew\ create a New Rich Text Document named "My WordPad Settings Template" (or whatever you want to name the template, and w/o quotes, just DO NOT add .rtf at the end of this file's name or it will NOT work).

b) Open and edit this file to your desired settings for font type and size, plus line and paragraph spacing. Any other setting changes are at your own discretion.

c) Now, be sure to hit the Enter key a few times (personally I prefer x5) in order to have several blank spaces at the beginning of the template, which WILL retain your font settings (excepting the last space, which when left blank always reverts to Windows' default font, similar to the way Windows XP always did -- remember?).

d) There is absolutely NO need to type any characters into this template! To facilitate making future changes to your settings, the template should NOT be made Read-Only -- after all, the file's location is relatively out-of-the-way. Don't forget to Save before closing this document.

Step 2 - Adding a New String Value in the Registry

Quick Note: If you do not know how to open regedit, I have to say the following -- Consider the Registry much like you would a good guard dog. There's no reason to fear getting bitten, unless you're doing something you're not supposed to be doing. Yet, you MUST show it the respect that it deserves! Now, type regedit into your Start menu's Search box, and click on the program up top...

a) Open regedit, and in the left column navigate to HKEY_CLASSES_ROOT\.rtf\ShellNew. To add a new string value, right-click on ShellNew and select New > String Value, and type "FileName" into the field (w/o quotes or space), then hit Enter.

b) In the window to the right, right-click on the new "FileName" entry and choose Modify..., then set Value data to "My WordPad Settings Template.rtf " (or whatever you named your template, exactly and w/o quotes, plus this entry MUST include the .rtf extention at the end of the file's name). Click OK and exit regedit.

c) Now you will be able to create a New Rich Text Document from any location, just as you always have previously, with several blank spaces that are ready for typing with your favorite settings already in place (except when using the Start menu, which will still open WordPad with all Windows 7 defaults).

I hope this information benefits others as much as it has me in my daily computing. Replies are more than welcome -- however, please try the procedure first, that you may be able to give a comment founded in practical usage. Thank You. :)

Happy Keyboards -- Larry
