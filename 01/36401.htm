rdrewd2 say:

Hi Grandpa Jim,

Yes, you can choose to run your backups once a week with both Backup & Restore(Windows 7) and with File History; for the former, just don't set a backup schedule and run it manually whenever you plug in the drive by clicking: control panel / backup & restore / back up now.  For the latter, set the backups to "daily" and if it starts bugging you to connect your drive, just go to settings / update & security / backup and turn file history off.  To run that one manually, go back to the same spot and turn in back on, then click more options, then back up now.  And yes, it is an excellent idea to keep a backup somewhere off-site like the cloud, a neighbor's house, your kid's house, or all of the above.  Cheers to you.

Ron Carr,

I had the same issue, and I've found the solution.

First off, what you have is partly a known problem many people had, and partly your lack of familiarity with File Explorer, which has superseded Windows Explorer.

Unlike in Windows Explorer, in File Explorer, a double click on 'This PC' is needed to show all drives, so do that. To your relief, the drives under 'This PC' are sorted by drive letter.  :-)

But you will now also notice a problem: your File Explorer' navigation pane is showing every drive twice! Not everyone has that problem, but many do. To solve that, download a registry tweak here.
