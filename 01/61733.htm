Wireless Security - Microsoft Community

xb1r apollo say:

Wireless Security

Login Information

  • Gateway - 192.168.1.1 or 192.168.1.254
  • Username - user | admin | root     | user   |   root    | root
  • Password - pass | admin | 12345 | 12345 | central | ZOOM-DSL
 

Wireless Passphrase

  1. Open a web browser.
  2. Enter http://192.168.1.1 into the address bar. (Or click the link) When prompted enter admin for the user name and admin for the password.
  3. From the menu on the left, select Wireless.
  4. Select Security from the submenu on the left.
  5. The Wireless Passphrase will be listed next to "WPA Pre-Shared Key" for a WPA Key. For a WEP key, click where it says “Set Encryption Keys”. The WEP key will then be listed in one of the boxes 1-4.

 

 

Changing Wireless Security Type/ Disabling

  1. Open a web browser.
  2. Enter http://192.168.1.1 into the address bar. (Or click the link) When prompted enter admin for the user name and admin for the password.
  3. From the menu on the left, select Wireless.
  4. Select Security from the submenu on the left.
  5. In the drop down box labeled “Network Authentication” you can disable or change the wireless security.
  6. If WEP is selected, select the WEP Encryption Strength (either 64-bit or 128-bit).
  7. If WPA-Personal is selected, select the WPA Mode (Auto, WPA or WPA2) and the WPA Encryption (TKIP, AES or Both). WPA2 with AES is recommended.
  8. Once the passphrase has been entered click Save/Apply.

http://answers.microsoft.com/en-us/windows/forum/windows_7-pictures/how-to-turn-on-sony-vaio-built-in-webcam-on/2cba4c07-6b17-465c-baaa-13b806c9574d

Method:

 

a)      Click "Start," "All Programs," then select your webcam from the list that appears.

b)      Click "Settings." Depending on your webcam, this may be under the "Tools" or "Options" menu, or it may be on the main screen, usually using the image of some sort of gear or wrench.

c)       Click "Enable Color" or unclick "Black and White." Again, this depends on your webcam. If there is no color or black and white option, you'll need to modify the saturation

d)      Change the saturation. Usually you can find your image saturation in the same place as your image brightness. If your saturation level is at "0," that means all of the color has been removed from the image. Enter in a higher saturation number (50 is usually the default),
