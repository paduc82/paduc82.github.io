hamilton bia say:

Hi John,

We're here to answer your questions with regards to Windows Update.

To find the total file size of all the updates Windows has downloaded to your machine, you can first check all the installed updates. After that, you can then search its corresponding size in our Microsoft Update Catalog (use Internet Explorer browser).

Follow the steps below to view all the updates installed in your device:

  1.  Click Start, then click Control Panel.
  2.  Under Control Panel, set View by to Small icons and click Windows Update.
  3.  Click Installed Updates.

You can also manually download Windows Update in Microsoft Update Catalog and save it in an external storage device and transfer it to your another device.

The updates can be categorized to either Optional or Important. The Optional updates would usually differ from one device to another. Hence we would suggest that you check the Windows Update list available through our Microsoft Update Catalog.

From the Windows Update Catalog, you may search for Windows Updates rolled out in the past 10 months, or whichever date you prefer, and download those from one machine then install on another. Also, make sure you filter it and download only the updates classified as Critical update, Update, and Update rollup. The other classifications of updates may vary from one machine to another, hence we suggest downloading those directly to your machine.

Port Forwarding - Microsoft Community

xbr dionysus say:

Port Forwarding

Login Information

 

  • Gateway: 192.168.1.1
  • Username: admin
  • Password: admin

Port Forwarding

 

  1. Connect a computer to the gateway with an Ethernet cable.
  2. Open a web browser on a computer.
  3. Type the address 192.168.1.1 into the web address bar.
  4. When prompted to login type admin for both the username and password.
  5. Under the Advanced Menu click on the Application option.
  6. Click on the Port Forwarding option.
  7. Under the Category section click on User.
  8. Click on New
  9. The Rule Management page will open
  10. Enter in a rule name like “Xbox Live”.
  11. For Protocol enter in both.
  12. For Port Start enter in 3074.
  13. For Port End enter in 3074.
  14. For Port Map enter in 3074.
  15. Click the Submit button.

 

 

Disable Port Forwarding

 

  1. Connect a computer to the gateway with an Ethernet cable.
  2. Open a web browser on a computer.
  3. Type the address 192.168.1.1 into the web address bar.
  4. When prompted to login type admin for both the username and password.
  5. Under the Advanced Menu click on the Application option.
  6. Click on the Port Forwarding option.
  7. Under the Applied Rules select the custom port and select the Remove.
  8. Click the Submit button.

You'll like Process Explorer when you get the hang of it.  Process Explorer is the Windows Task Manager on steroids.

Process Explorer installs nothing so it will not slow your system down since it only runs on demand.

Process Explorer may look a little intimidating at first since it presents so much information, but you will start to get to like the way it works when you are looking for performance problems.   You can even tell PE that you want it to be your new default "Task Manager" from now on.  You can still run the original Task Manger too.

Once you get Process Explorer running, expand the columns, drag the corners of the display to make it bigger, etc. so you can see the most information as possible in the window.  Now you can really see everything that is running on the system.

Here is a screenshot of my poor system when I use Process Explorer:

http://img222.imageshack.us/img222/2567/processexplorer.png

CPU
System Commit
Physical Memory
Network
Disk
I/O

If your disk is thrashing, look for a spike in the Disk window or double click it to make it bigger.  When you see a spike, hover you mouse over it and it will tell you the process on your system that is causing it.

When you analyze the volume for fragmentation, look at the Disk activity in Process Explorer and hover your mouse over the spike and it will say dfrgntfs.exe, so now you know who cause the spike.

Close out any applications you do not need for your troubleshooting so they don't clutter things up,  get PE up and running, let things settle down and wait the thrashing and for the spike in the Disk graph, hover your mouse over the spike and find out who is the culprit.

  7. Check the "Enable" check box.
  8. Enter "Xbox" into the "Computer Name" field.
  9. Enter the IP address of the Xbox into the "IP Address" field.
  10. Enter the MAC Address of the Xbox into the "MAC Address" field.
  11. Select Save.
