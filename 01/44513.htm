a. user say:

If your CPU is running at 80% that is abnormal so you should work on figuring out why that is and fixing that.

You can open Task Manager (Ctrl-Alt-Del) and on the Processes tab enabled the check box in the bottom called "Show processes from all users", then click the CPU column header to sort and watch that for a minute.

When things are looking good you should see the System Idle Process at the top.  The System Idle Process is actually how much free time your CPU has so the more the better for that Process.

If you see a svchost.,exe Process at the top and running amok using too much CPU you probably have a known affliction involving Windows Update.   It is normal to see several to many svchost.exe Process running but they should not be doing very much on a normal system.

One symptom of the Windows Update problem is a svchost.exe Process consuming large amounts of RAM and CPU and fans running all the time at high speed trying to keep things from overheating.  Another symptom us a Windows Update that appears to be doing nothing or taking too long (hours+++) to retrieve updates.

And if you just reinstalled Windows for some reason you especially need to fix that known issue with Windows Update ASAP - then troubleshoot any remaining problems.

In that case you should apply the suggestion from Volume Z:

Windows 7 Update Problems? Read this first.

If and after that there is still a process consuming large amounts of CPU you need to tell us what it is or post a screen shot so we can see what it is:

How to include a screenshot in your post
