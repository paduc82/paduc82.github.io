frederik long say:

Assuming that your media server is a Windows workstation rather than a Windows Server with Active Directory, you can do this:

  • Create three folders (Photos, Videos, Music) on the "server".
  • Right-click each folder, then share it as "Photos", "Videos", "Music".
  • While sharing it, grant full access to everyone.
  • Create 3 accounts on the server: One for A, B and C. Use exactly the same names and passwords as these accounts have on the various workstations.
  • Use Windows Explorer to check the security settings of the three folders on the "server". On each folder the accounts A, B and C must have full read/write access.

That's all you need!
