It's an Intel H57 chipset and the drivers ship with Windows on the Windows 7 DVD.  The only "drivers" you get from Intel directly are the Rapid Storage Technology drivers and that covers the SATA controller, it does not install USB drivers.  And all non-mass-storage-class USB devices work just fine - and I have quite a few.  All that being said, I always update my drivers.

Ultimately, I tried my own solution - restoring the \WINDOWS\INF directory from a backup and that solved the problem.  I agree with Chris Windows got corrupted, I just wish I knew how.  You would think SFC /SCANNOW would be helpful... but it seems like most problems SFC finds, it can't fix.

In general there's plenty of examples of weirdisms with USB and storage devices that I would characterize as bugs:

- USB storage devices (BlackBerries are the most notorious) "steal" drive letter assignments voraciously if you plug the same device into a different port - my old Curve would take E: and F: if I plugged it into one USB port, and G: anfd H:if I plugged it into another, which leads to...

- Problems where two devices are plugged in that both want, for example, to be the N: drive.  One just refuses to mount entirely, and your only remedy is to use Storage Manager to reassign the default drive letter for the one that DID mount.  And of course you'd have to do this with each device plugged into a different USB port... which for a machine with 8 ports is 8C2=8!/(8-2!)=56.  That's a lot of permutations.

Obviously this is a long digression but I am doing so in the hopes that it becomes a bug report somewhere inside Microsoft.

If you have bought the retail copy of Windows 8, it should have come with an installation DVD. If you have purchased Windows 8 online and downloaded an ISO, you will need to put that ISO on a USB drive. you can use any image burning program (Windows 8 will even do it natively), such as ImgBurn. Once the USB device has been identified, click the Begin copying button.

 

And that’s it, the next step now is to boot your Tablet from the thumb drive, make sure the thumb drive you will be installing Windows 8 from is plugged into the USB port. start your Tablet and press F12 repeatedly until a screen is displayed giving you the option to boot from a USB Memory device, select that and press Enter. You might need the assistance of a keyboard for these steps.
