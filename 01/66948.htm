How can I repair a corrupted task scheduler in Windows 7 - Microsoft Community

rab126 say:

How can I repair a corrupted task scheduler in Windows 7

After hours with tech support at sosonlinebackup.com, they tell me my task scheduler is corrupted, thus preventing the automatic daily backup.  Can I repair task scheduler by using my Windows 7 Home Premium upgrade disc?
frederik long say: I think it is rather unlikely that the Task Scheduler service is corrupted. A flawed shadow volume service seems a more likely cause. To find out, you can do this:

1. Log on under an admin account
2. Press the Windows key + R
3. Type the three letters  cmd  and press Enter.
4. Type this command and press Enter:
    schtasks /Create /SC Minute /ST 00:00 /TN "Test Task" /TR "cmd /c echo %%date%% %%time%% >> ""%UserProfile%\desktop\test.txt"""
    (make sure to use the correct number of double quotes. It is 1 before 'cmd', two before '%UserProfile%' and three after 'text')

You will see a message that a scheduled task has been created. If it works then it will write a date stamp into the desktop file "test.txt" once every minute. Does it?

To delete the task, do this:
1. Log on under and admin account
2. Press the Windows key + R
3. Type the three letters  cmd  and press Enter.
4. Type this command and press Enter:
    schtasks /delete /tn "Test Task"


frederik long say: Since something flashed by once every minute, you successfully created a task. This means there is nothing wrong with the task scheduler.

What actually is the application you're trying to schedule? What is its command line? Have you tried running it directly rather than scheduling it?
