You cannot change a drive letter if the drive is either a system partition or a boot partition. System partitions and boot partitions are names for partitions (or volumes) on a hard disk that Windows uses when starting, system partition actually contains the files used to boot Windows 7, while the boot partition contains the system files.

To identify the system or boot partition

a)      Open Computer Management by clicking the Start button , clicking Control Panel, clicking System and Security, clicking Administrative Tools, and then double-clicking Computer Management.‚Äå  If you're prompted for an administrator password or confirmation, type the password or provide confirmation.
