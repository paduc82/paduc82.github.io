roke beedell say:

What's the difference between Links & Favourites?

What is the difference between %userprofile%\Links & %userprofile%\Favourites? If I type 'Links' in the directory bar it takes me to the Favourites folder, but Links is certainly a different folder.

I want to know this simply because I want to know what sort of file goes into each (ie. images in the Pictures folder, etc).

a. user say:

Good question.

Favourites (British or Canadian?) refers to something very specific: bookmarks in Internet Explorer.

Netscape, I believe, originally coined the term 'bookmark' to refer to an easy way to remember web pages you want to visit. Microsoft, being Microsoft, wanted Internet Explorer to rule the internet, so they created the term 'Favourite' to mean a bookmark stored in Internet Explorer. The tactic didn't work, so the term Favourite has fallen out of use.

Links is also an older item that has fallen out of use. It was the folder that held shortcuts to frequently accessed files which appeared in the old Windows Explorer. In Windows 10 it has been replaced by Quick Access.
