You can try the following to help with Windows Installer issues:

 

Method 1:  To check whether the Windows Installer service is set to a value of Disable

  1. Click Start, type services.msc in the start search box and press enter.
  2. In the Services (Local) list, right-click Windows Installer, and then click Properties.
  3. If the Startup type drop-down list is set to a value of Disable, select the Manual option from the Startup type drop-down list.
  4. Click on Start button below the heading service status and click OK.

Method 2: Re-register the windows installer

  1. Click Start, open the run box (windows key +R)
  2. Type msiexec.exe /unreg and click ok.
  3. If you receive a confirmation message, click OK again.
  4. Click Start, open the run box (windows key +R)
  5. Type msiexec /regserver and click ok.
  6. If you receive a confirmation message, click OK again.
  7. Try to install the application and see if the issue persists.
  8. If you still get the error then try next method.
