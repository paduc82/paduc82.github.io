robertstaff say: What is a system image?

A system image is an exact copy of a drive. By default, a system image includes the drives required for Windows to run. It also includes Windows and your system settings, programs, and files. You can use a system image to restore the contents of your computer if your hard disk or computer ever stops working. When you restore your computer from a system image, it's a complete restorationâ€”you can't choose individual items to restore, and all of your current programs, system settings, and files are replaced with the contents of the system image.

Although this type of backup includes your personal files, we recommend that you back up your files regularly using Windows Backup so that you can restore individual files and folders as needed. When you set up Windows Backup, you can let Windows choose what to back up, which will include a system image, or you can select the items that you want to back up and whether you want to include a system image. malkeleah say:

You may be right that this is a software (Windows) problem and not hardware. I based my feeling about the culprit being hardware on the fact that you said the recognition of the drive is random. If it were a question of drivers (software) I would expect the behavior to happen all the time instead of just sometimes. However, there is no way for me to know for sure without seeing the machine. You do have an awful lot of partitions, most of which are unnecessary. On systems with a single hard drive, there is some advantage to putting your data on one other separate partition so if you need to reinstall Windows it is preserved. There is no point in having programs on a separate partition since if you need to reinstall Windows you will also need to reinstall all the programs, too. There is also no reason to put different categories of data on separate partitions either. If Windows gets infected or if the hard drive dies, everything will be affected and having all those extra partitions will be no protection. So you might want to use a partitioner to change that. I don't know whether Partition Magic is still compatible with your version of Windows because I haven't used that program in many years. I use Acronis Disk Director but there are others. Gparted is free.

I don't know offhand which registry keys are associated with what you asked about but I rather doubt that registry corruption is the issue. If it were, I'd expect to see consistent behavior, not random issues.

To answer your question about the controller card, I probably wouldn't change the connection for the boot/system drive. As for controller cards, they aren't that expensive. NewEgg has cards starting at $12.99 so it isn't that big of a deal if you want to test one.

But once again I must stress that you should not take what I have written as a definitive diagnosis. I cannot see or test your computer from here. I'm giving you my opinion about your computer based on my professional experience as a computer tech and on what you've written. If the machine were here in the shop I might come to a very different conclusion. I'm sorry not to be more precise but there you have it.
