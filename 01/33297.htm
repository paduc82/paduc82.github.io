    Hi Bitsaron,

    1. An in-place upgrade to Windows Vista (http://support.microsoft.com/kb/968279) is recommended for a different symptom. 
    Is this the only/best way to fix a corrupted registry?

    You’re right that the scenario given in the link is different from yours, but here we’re concentrating on repairing system file corruptions. So, In-place upgrade is the only repair option to fix the corrupted registry. Basically, an In-Place upgrade is actually a repair installation using the upgrade option.


    2. The above article requires that I use "genuine Microsoft software media".  All I have is the Dell Operating System Reinstallation DVD for Windows Vista Home Premium 64BIT SP1, which came with my computer when I purchased it a year ago.
    Will this work?

    Your Dell Operating System DVD Service Pack 1 64 bit will definitely work provided you uninstall Service Pack 2 from your computer.
    If you have installed Service Pack 2 for Windows Vista and your original Windows Vista DVD does not include Service Pack 2, then the Upgrade option will be disabled, thus preventing you from doing an In-place upgrade. The reason for this is that, by installing Service Pack 2 your system is, in effect, newer than that of your original Vista DVD (the only exception to this is if you have a slipstreamed copy of Windows Vista including Service pack 2).

    Therefore, if the original Windows Vista DVD does not contain Service Pack 2 you will need to uninstall Service Pack 1 from your PC prior to attempting the upgrade.

    3. My Windows Vista has SP2 already installed.
    Assuming that the answer to question 2 is "YES", what are the consequences of an "in-place upgrade", or should I say downgrade?

    After performing In-Place Upgrade, your personal data and installed programs will not be removed but we may need to run Windows Update to install all the available updates for your system to update these system files to the current version. You also need to install Service Pack 2. This troubleshooting process is time consuming and may take several hours to complete.

    4. What are the chances that this will fix the problem?
    If your system registry is in a steady condition, then you’ll not encounter any problem installing the software’s on your computer. In-place upgrade will repair your system files and registry so that it will make a way to fix the problem. In-place upgrade is considered as a last resort before performing a clean installation of the operating system.
