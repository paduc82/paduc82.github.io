NOTE : By trying the below two steps Windows Mobility Center will be disabled and you will not be able to use it. To be able to use it again you will have to follow the same steps instead of Enable you will have to select Disable in option e.

If your edition of Windows 7 includes the Group Policy Editor snap-in (gpedit.msc), follow these steps:

e)     select the option Enable.

f)      Click on OK.

g)     Close the Local Group Policy Editor window.

When starting up/rebooting my system, my Microsoft Natural Wireless Ergonomic Kayboard 7000 does not work, thus preventing me the capability of getting into BIOS to make any necessary changes. Can anything be done about this?? Before you ask, yes, legacy USB support is enabled. 
smlunatick say: This problem is generally a BIOS setting issue.  Several BIOSes can have various settings to enable / disable USB keyboard / mouse access.  These tend to be listed as:

Use USB legacy.......

Enable USB keyboard ........

This is a major issue with the motherboards that have the older PS/2 ports for keyboard / mouse.


Also, I have found that if your keyboard is not plugged directly to a rear USB port, you may have this problem.

And depending how many USB devices are connected to your PC, if you "maxed" out the USB device (all ports used,) your keyboard will not work as it should during the Windows "pre-boot" process (including the BIOS "P.O.S.T." screens.)
