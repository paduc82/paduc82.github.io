How can I get the Windows Mail messages from the old hard drive onto my new hard drive?  I see the Export Messages feature in Windows Mail but I didn't do that before my old hard drive was removed!  Can I connect my old hard drive into my computer and export those messages from the old hard drive onto the new one?
steve winograd say: In Windows Mail, click File > Import > Messages.  Select "Microsoft Windows Mail 7" and click Next.  Then browse to the Windows Mail storage folder from the old disk and click Next.  The default storage folder location is C:\Users\<User Name>\AppData\Local\Microsoft\Windows Mail.  You might have to enable viewing of hidden files and folders (in Control Panel > Classic View > Folder Options > View) to see the AppData folder.

The messages will go into a new folder called "Imported Folder".  You can move them from there into the "Local Folders" folder if desired.
