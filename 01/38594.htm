ronnie vernon say:

Hi Barneyboy48

The Libraries are Virtual Folders and contain folders that are physically located in another location on the hard drive. You can add folders from many different locations, such as another hard drive or even a network location.

The Library displays a mirror copy of the files in the physical folder.

In the Documents Library there are 2 folders that are added, by default. These are My Documents and Public Documents. The physical folders are located at:

C:\Users\user name\My Documents   (where user name is the name of your system profile)
C:\Users\Public\Documents

When you delete a file in the library, it should also be deleted from the physical folder.

You can test this by creating a text file in Notepad and placing it in the C:\Users\user name\My Documents folder.

After it is added, open the Documents Library and you should also see the file there. Delete the file in the Library and then check the C:\Users\user name\My Documents folder to see if it was also deleted there.

If you still see the file in the My Documents folder, press the F5 Key to refresh the folder to see if it still remains there.

Let us know the results.

Hi Barneyboy48

The Libraries are Virtual Folders and contain folders that are physically located in another location on the hard drive. You can add folders from many different locations, such as another hard drive or even a network location.

The Library displays a mirror copy of the files in the physical folder.

In the Documents Library there are 2 folders that are added, by default. These are My Documents and Public Documents. The physical folders are located at:

C:\Users\user name\My Documents   (where user name is the name of your system profile)
C:\Users\Public\Documents

When you delete a file in the library, it should also be deleted from the physical folder.

You can test this by creating a text file in Notepad and placing it in the C:\Users\user name\My Documents folder.

After it is added, open the Documents Library and you should also see the file there. Delete the file in the Library and then check the C:\Users\user name\My Documents folder to see if it was also deleted there.

If you still see the file in the My Documents folder, press the F5 Key to refresh the folder to see if it still remains there.

    Hello Ahmet,

    Thank you for posting your question on Microsoft community.

    I appreciate you for providing details about the issue.

    If a driver is not available or if you've updated the driver and still can't select the scanner, find out if your scanner uses TWAIN technology. TWAIN is an application programming interface (API) that is used to import images from scanners and digital cameras to your computer. Windows Photo Gallery and Windows Fax and Scan use a different technology, called the Windows Image Acquisition (WIA) API, and do not work with TWAIN scanners. If your scanner uses TWAIN technology, you might need to use the program that came with your scanner instead of using Windows Fax and Scan or Windows Photo Gallery. To find out if your scanner uses TWAIN technology, check the information that came with the scanner or contact the scanner manufacturer.
