Microsoft Edge actually has two different concepts.

  

First is a "Open with:" where you choose what page/pages the browser will open with EACH TIME.

  

Second is the "Home Page" where you select whether the "Home" icon is shown and where you go when the button is visible and clicked or (button visible or not) you press "ALT+HOME" keys.  (Note: you can only set the HOME page location when the setting for the Home button visibility is ON.  You can turn it back off and the keyboard shortcut still works - you just don't actually have the HOME icon.)
