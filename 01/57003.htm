Important Information If You Have Received a STOP Code

If you have received a blue screen error, or stop code, the computer has shut down abruptly to protect itself from data loss. A hardware device, its driver, or related software might have caused this error. If your copy of Windows came with your computer, call the manufacturer of your computer. The following actions might prevent an error like this from happening again:

Download and install updates and device drivers for your computer from Windows Update.
Scan your computer for computer viruses.
Check your hard disk for errors.

Cause

This error is typically caused by a driver passing a bad memory descriptor list. For example, the driver might have called MmUnlockPages twice with the same list.

Hello Sagar,

Thank you for posting your query in Microsoft community.

I know how this concerns and assist you regarding the issue.

I require more information to help you better.

  1. What is the make and model of your computer?

  2. What is the total storage capacity of your hard disk? How much space is it allocated for the C drive?

  3. Did you check your drives in Disk Management? Are you able to see your personal drives over there?

  4. How many disk partition did you have on the computer?

Follow the below instructions to go Disk Management.

Right click on Start > click on Disk Management .

If you are able to view the missing disk partition in Disk Management, then change the drive letter and check if it helps.

1. Right click on Start.

2. Click Disk Management.

3. Right-click the drive that you want to change, and then click Change Drive Letter and Paths.

  • 4. Do one of the following:
    • To assign a drive letter if one has not already been assigned, click Add, click the letter that you want to use, and then click OK.

    • To change a drive letter, click Change, click Assign the following drive letter, click the letter that you want to use, and then click OK.

    • To remove a drive letter, click Remove, and then click Yes to confirm that you want to remove it.

    Note:Some programs make references to specific drive letters. If you change a drive letter, these programs might not work correctly.

  • If you get an error, the drive you are trying to change might be in use. Close any program or window that is using the drive and try again.

    Reply with necessary information to help you better.

    The BSOD can happen if the following causes are true:

    1- Bad memory

    2- Hard Drive going bad/corrupt

    3- Badly written application

    4- Hardware driver damaged or corrupt

    5- Virus infection

    6- Overheated problem with system

     

    Try with the basic steps;

    Test the memory by running this test and try to reseat the memory and clear the dust from the slot.

    Run the verifier command to see what Driver is not meeting the test, also check Event Viewer for error messages that can shed some light on the causer or give indication that there is a pattern we can track to solve your issue.

    Run a thorough system scan with an up2date security software and make sure all drivers on the system are up2date from the manufacturer website.

    Download this tool, unzip and run the test, bear in mind it will take some time so be ready:
