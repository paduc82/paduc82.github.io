4. If Required enable USB Boot

6. Save BIOS changes, restart your System and it should boot from the Installation Media

frederik long say: A Cyclic Redundancy Check error means that Windows has encountered a serious problem when reading data from your disk. You probably need to replace it.

frederik long say:

I am not the one who asked this question but I am having the same or similar problem. I am trying to re-install my "Dragon 12.5" program and keep getting the "Cyclic Redundancy Check" message as well. I replaced the disk and still get the same message with a brand new program disk.

Saying "I replaced the disk [..] with a brand new program disk" implies that you have several disk. How do you know for sure that the replacement disk generates the CRC error?

If it does then one needs to keep in mind that several components are required to transmit data from the disk to the PC's memory, e.g.

  • the hard disk
  • the connecting cable
  • the motherboard-based disk controller

A failing hard disk is the most frequent cause of a CRC error. To determine if one of the other components might be responsible you need to do this:

  1. Remove the disk.
  2. Put it into a 2.5" USB case.
  3. Connect the case to some other PC.
  4. Run this command:
    chkdsk Q: /r
    (I assume that the disk is visible on drive Q:)
