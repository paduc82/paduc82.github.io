hairyfool say:

Overclocking a laptop is fraught with issues. Laptop design is particularly constrained where cooling is concerned. Overclocking can produce a significant increase in processor temperature and is the primary concern.

 

On a desktop there are various options to compensate for this higher heat output from increased case airflow right up to liquid cooling solutions. These options are not available on a laptop.

 

When you add in the fact that overclocking will have little real effect in terms of visible performance (not benchmarking) I would question the benefits of such action on a laptop.

 

Far better effects can be achieved in other ways. Cleaning and optimising Windows (not a cleanup program, do it manually), more memory and using an SSD.

Owwee.  Where did you get those instructions?

They are flawed (this sometimes happens from not actually recreating the problem and then actually fixing it, or it could just be hasty instructions).

One does not merely copy the compressed winlogon.ex_ from the installation CD to the c:\windows\system32\ folder.

Instead of copying the compressed file, one would expand the compressed file from the source to the destination, such as:

expand d:\i386\winlogon.ex_  c:\windows\system32\winlogon.exe

Files in the i386 folder on a genuine bootable XP installation CD are sometimes compressed...  that is what the underscore character designates when it is the last character of the file name:

winlogon.ex_ = a compressed file
winlogon.exe = an uncompressed file

The winlogon.exe file may indeed be all you need to replace to get your system running, but it doesn't tell you what happened to get your system afflicted in the first place or what to do about it so it doesn't happen again.  Fixing the symptom of the problem and fixing the problem are not the same thing.

Do you have a genuine bootable XP installation CD and are you able to get into the XP Recovery Console?

That is not the same as any System Recovery type CDs that may have come with a store bought system.

If you are not sure you have a genuine bootable XP installation CD, you need to make a bootable XP Recovery Console CD.
