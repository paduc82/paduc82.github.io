Version 6.1 refers to the kernel version of Windows 7. Kernel is a piece of software that communicates with the hardware and is present in all the operating systems, like Windows. The icons and buttons that we use to interact with the computer, known as user interface (or shell), is built on top of kernel. Without kernel, a computer can't perform anything, literally.

Consider an example. You can install Windows in a new PC that doesn't have an operating system, but you can't install Microsoft Office in a new PC, without Windows. This is because, Windows has kernel and Microsoft Office doesn't have kernel. Hence Microsoft Office rely on a platform that has kernel for its working.

So as per the report, the version is correct. Also, you can notice the architecture as AMD64, while you're using an Intel Core i5 processor. This is because, the architecture (a set of instructions that define how a processor works) has been created first by AMD and adopted by Intel.

Further, the report states that Windows is not up to date. Hence install all the updates and then check with the Get Windows 10 App for upgrade.
