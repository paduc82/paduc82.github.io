OEM 'codes' can mean many things - there are OEM SLP Product Keys, OEM COA Product Keys, and OEM PID's, just for starters.
an OEM SLP Key is the Product Key used by a major manufacturer for the install on Windows on multiple Keys - they use one Key for all installs of that model of machine, and it's activated by the BIOS, which has specific 'bits' which validate the install.
an OEM COA Key is the Product Key on the COA Sticker on any machine which has been installed using OEM software - it is specific for use in that one machine, and cannot be used on any other.
an OEM Product ID (PID) is generated whenever a machine using either type of OEM Key - it is identified by the -OEM- characters in the second group of characters in the PID. This code is specific to an install, because it uses details from the machine, as well as the Key used, in its formation.
