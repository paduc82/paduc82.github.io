nexusicon say: Hello again

I fixed it!

I'll post my solution so that anyone else with the same/similar problem doesnt have to launch their machine across the room :)

  • I went into the WoW shortcut properties (right click > properties)
  • Then into the 'Security' tab , where you see the boxes 'Group or user names' and 'Permissions for....'
  • And on this file i had 'Full control' for each user
  • So then I thought I'd check out the other files. So I repeated the steps on the WoW.exe files and suprise suprise the 'Group or user names' box was empty .

  • Then 'Add...'
  • Then in the 'Enter the object names to select' I copied the names from the shortcut Permissions . (type SYSTEM, Administrators , and then your own Username and click the 'Check names' box).
