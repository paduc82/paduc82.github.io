considerate_guy say:

The first question is what version of Windows are you running?  The answer I give depends on the version of Windows you are running.

 

The short answer is, the narration feature was added in Movie Maker 2012 (16.4.3505.0912) which is part of Windows Essentials 2012.  Movie Maker 2012 can be installed on Windows 7 and Windows 8.  While Microsoft previously included "Live" in their branding of consumer apps, they dropped that in 2012, so what used to be Windows Live Movie Maker is just known as Movie Maker.

 

If you are on Windows Vista, however, Microsoft has not made Movie Maker 2012 available.  It requires Windows 7 or later, as I said before.  So if you are on Vista, the latest version you will see is Windows Live Movie Maker 2011 (15.4.3555.0308).  I suspect this might be your situation.

 

All is not lost, however.  You can add your own voice to your movie by using the Sound Recorder application.  Record what you want to say and then add it as your "music" in your project.  If you need to have both music and narration, then you can do that in two passes.  For example, in the first pass, make all your video edits and add your first layer of sound (say the song).  Save your project (File > Save project) and also save the movie (File > Save movie).  Now start a new project and add the movie created in the first pass.  This time when you add "music" use the next layer of sound (e.g. the narration).  Again save your project and movie.
