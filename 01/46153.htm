    a. user say:

    Hi Nischal,

     

    It would be better if you can assist me with a certain information regarding the query:

    Are you referring to charging of a laptop’s battery when the selective suspended mode is disabled for USB devices?

     

    I would like to inform you that the USB selective suspend feature allows the hub driver to suspend an individual port without affecting the operation of the other ports on the hub. Selective suspension of USB devices is especially useful in portable computers, since it helps conserve battery power. This mode for the USB devices also helps in reducing the overall power consumption. Disabling this setting will cause additional battery power consumption when your computer is in Sleep or Hibernation mode and unplugged. Again, if the devices are connected and the laptop is running on a battery, it would take a few more moments to completely charged up as compared the time period which it would normally take if the laptop is in off state or there are no external USB devices connected to it.

    lemp say:
    If you mean that your directory now has:

    File 1
    File 2
    File 3

    The first thing is to make sure that the files don't already have an extension that is just hidden.  In Windows Explorer, go to Tools > Folder Options > View and ensure that "Hide extensions for known file types" is NOT checked.  If the box was checked, uncheck it and click OK.

    If the files really have no extension, then open a Command Prompt (Start > Run > cmd > OK) and navigate to the folder where the files are stored (D:\TextFiles)

       D:  {press Enter}
       cd D:\TextFiles  {press Enter}

    Then type

      ren *. *.txt {press Enter}

    Note that there is a space between the first dot and  the second *.  After the command completes, type

       dir *.* {press Enter}

    just to make sure that the files were all renamed.
