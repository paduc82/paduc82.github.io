If you upgrade from a OEM or retail version of Windows 7 or Windows 8/8.1 to the free Windows 10 upgrade this summer, the license is consumed into it. Because the free upgrade is derived from the base qualifying license, Windows 10 will carry that licensing too.

If you upgrade from a retail version, it carries the rights of a retail version.

If you upgrade from a OEM version, it carries the rights of a OEM version.

Full version (Retail):

- Includes transfer rights to another computer.

- Doesn't require a previous qualifying version of Windows.

- Expensive

Upgrade version (Retail):

- Includes transfer rights to another computer.

- require a previous qualifying version of Windows.

- Expensive, but cheaper than full version

OEM versions of Windows are identical to Full License Retail versions except for the following:

- OEM versions do not offer any free Microsoft direct support from Microsoft support personnel

- OEM licenses are tied to the very first computer you install and activate it on

- OEM versions allow all hardware upgrades except for an upgrade to a different model motherboard

- OEM versions cannot be used to directly upgrade from an older Windows operating system

To display a .doc and a pdf file side-by-side:

1. Open the windows file.

2. Minimize the file to 1/2 screen on the right

3. Open the PDF file

4. Move cursor to the upper left side of your screen and click on the .doc file

5.  Move cursor to the upper left side of screen (a thumbnail of the PDF should be exposed)

6.  Right click on the PDF file

7.  Click insert to right (the PDF and .doc files should be side-by-side)

8.  To close the PDF file press alt >F4

9.  To close the second screen, click >the three dots on the second screed

10. Drag to the left

This works for me. Hope it works for you too!

1. Open reader

2. Move mouse to top of screen until tool bar rolls down that displays the document title and reader logo with the close and minimize buttons.

3. Click and hold this bar.

4. Drag bar down into center of screen. Move it to the left to display document on left or all the way right to display on right. A dark bar with 3 dots in the center should appear. If you don't move left or right reader will remain in full screen.

5. When side is chosen click the other side. it should be gray.

6. Pick the program you want on that side. I usually pick the desktop so I can have full range. It will display your whole desktop on half of your screen. It is much easier.

7. TO EXIT half screen mode drag the bar with 3 dots to the side of the screen and full screen should resume.
