davidmcandy say:
You'll have to try and see what happens. Some programs will automatically
start a service set to manual when it needs it. Other programs only look for
a running service and so don't start the service if it's not running
(Windows Defender is an example).
 
For instance, ITunes has 4 or 5 services. Itunes will autostart all but one
of them (the Bonjour service). So you can set all the services with ITunes
in the name to manual. But the bonjour service needs to be automatic (or you
need to start it manually).
 
DO NOT set everything to manual.
 
Use those sites to guide you.
