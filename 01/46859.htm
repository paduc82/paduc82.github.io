keith a. miller say:

The items you see on your Desktop when all windows are minimized come from several locations --- some are file system objects and some are Namespace items.

In File Explorer, the complete Desktop is seen by clicking the '>' icon found on the far left of the Address bar and selecting "'Desktop'

The Namespace items include User Folders, This PC, One Drive, Libraries, Network, Homegroup, Control Panel, and Recycle Bin.  If you type "Desktop Icons" into your Search Bar, the first choice offered by Cortana should be 'Show or hide common icons on the desktop"

The 'Desktop' shortcut in Quick Access displays the consents of your user profile Desktop folder, which are only the files, folders, and shortcuts that you've created.

The other locations that populate the Desktop are:

"C:\Users\Public\Desktop"

HKCU\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\Desktop\NameSpace

HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\Desktop\NameSpace

Good Luck,
