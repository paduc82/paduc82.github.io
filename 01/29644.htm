mel tar say:

Hi Tony,

You may try deleting Windows.old folder by performing the following method:

Elevated Command Prompt:

  1. Run the Command Prompt as administrator.
  2. Type the following command and press Enter after each one:
    • attrib -r -a -s -h C:\Windows.old /S /D
    • RD /S /Q %SystemDrive%\windows.old
  3. When finished, you can close the command prompt.

Command Prompt on Boot:

  1. Boot the computer and select the Advance options.
  2. Select Command Prompt and type the following command and press Enter after each one:
    • Note: The Windows 10 drive may not always be C: at boot like it is while Windows is started. Verify its drive letter before doing the command.
    • diskpart
    • list volume (Verify the drive letter of your Windows drive. See screenshot below.)
    • exit
  3. To delete the Windows.old folder, type RD /S /D:\Windows.old and press Enter.
    • Note: Be sure to substitute D in the command above with the actual drive letter of your Windows.

Let us know how it goes.
