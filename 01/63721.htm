Disable "Fast Startup in both Windows 8.1 and Windows 10:

Windows 8.1

You can disable Fast Startup by going to:
1) Control Panel/All Control Panel Items/Power Options.

2) In the left pane click: "Chose what the power buttons does"

3) Next click "Change Settings that are currently Unavailable"
(the unavailable settings are the grayed out options located in the bottom half of the window)

4) Next in the "Shutdown settings" grouping, remove the check mark from "Turn on fast startup".

5) Click the 'Save changes' button.

6) Reboot the computer and load Windows 10.

Windows 10:

1) Open the Control Panel (icons view), and click on the 'Power Options' icon.


2) Click on the 'Choose what the power buttons does' link on the left side.


3) Next click on the 'Change settings that are currently Unavailable' link at the top of the list.


4) If prompted by UAC, click on the 'Yes' button.

5) To "Turn Off" Fast Startup, under 'Shutdown settings', uncheck the Turn on fast startup box

(Fast Start is enabled by default) and click on the Save changes button.


6. Exit the System Settings.


7. Reboot the computer.

Now see if you can switch back and forth between Windows 8.1 and Windows 10 without the delay.

Type following (or copy and paste following) one by one and Enter

net stop wuauserv
net stop cryptSvc
net stop bits
net stop msiserver
Ren C:\Windows\SoftwareDistribution SoftwareDistribution.old
Ren C:\Windows\System32\catroot2 Catroot2.old
net start wuauserv
net start cryptSvc
net start bits
net start msiserver

2. Type following in Windows Start Search box and hit Enter %systemroot%\Logs\CBS > Rename CBS.Log to something else.
If it doesn't allow you to rename - Type services in Windows Start Search box > Click Services > Scroll down to find the Windows Modules Installer service and double-click on it > Set its startup type to Manual > Reboot your computer
Go to C:\Windows\Logs\CBS > Rename CBS.Log > Reboot your computer > Then go back into the Services App and set Windows Modules Installer Service back to Automatic Startup (Note CBS log files contain information about windows update, SFC etc)

(Source - https://partnersupport.microsoft.com/en-us/par_...)
