drive letter missing

I have a PC with 2 hard drives running XP.  Something happened and I lost the drive letter for the secondary drive.  I ran disk management and can see both drives are recognized and healthy.  If I right click the main drive (c:), a menu shows with several options, one of which is something like "assign and change the drive letter".  The secondary drive has a blank where the drive letter should be, and if I right click the secondary drive, the menu that shows only has one element "Help".  Is there any way to assign the drive letter to the second HD other than using disk management, which won't work?
greg carmack - mvp 2010-2020 say:

Gene -

I see no secondary data hard drive in the picture, only the C drive you're presently running from and signified by the System label. 

In a Legacy MBR install that C partition should also be marked Active which is what points to the intended System partition.  See if you can right click on C in Disk Management to Set Partition Active. 

If you have a secondary hard drive then power down and make sure it's connected securely, the cables are good and tight, try another port on motherboard, etc.  until it becomes detected.  If not then see if it can be detected on another PC.  The second drive may have failed if it won't show up.

greg carmack - mvp 2010-2020 say:

There appears to be no External hard drive plugged in according to Disk management.  Does it appear in Explorer under My Computer? 

Since it is appearing in Device Manager then it may need to be reformatted.  

Plug it into a PC that accepts it and copy the data off, reconfirm it's copied, then run Diskpart Clean Command to clean it of any interfering code, followed by formatting it in Disk Management as shown in Partition or Volume - Create New .

If these all fail then in a PC that recognizes it run a Disk Check followed by drive diagnostics.  Since that brand doesn't have it's own I'd use  SeaTools for DOS and Windows - How to Use  but if it doesn't detect it try from bootable SeaTools for a USB Drive .

One other thing you can try is to install Partition Wizard to see if it detects it, to assign a letter, run Partition Wizard Check File System, and run it's own diagnostics  Partition surface test with Partition Wizard.

If it fails either test using either method then replace it.
