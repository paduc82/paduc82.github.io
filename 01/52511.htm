Hi,

It's possible that  iTunes folder is set with Read Only permission, which means Windows will allow it to sync with the device you have plugged in. Try the following steps and check if it works.

  1. Open Windows Explorer (Windows + E)
  2. Right click on your iTunes folder
  3. Go to Properties
  4. Uncheck the Read Only option under Atrributes.
  5. Now, check if the issue persist.

If the problem still persist, then make sure that you have full control over the iTunes folder. To do so, follow these steps.

  1. Right click on the iTunes folder.
  2. Select Properties and click on security.
  3. Click edit, check allow full control and Apply

Hope this will help you.
