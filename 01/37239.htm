How do I Reset Gatherer Component of Windows Search - Microsoft Community

curious joe say:

How do I Reset Gatherer Component of Windows Search

Over a year ago, I set the index location of Windows Search (on a computer hosting Windows 7 Home Premium) to "D:\Windows\Search" on a non-system drive in order to improve overall system performance.  Recently, I reset the index location to its original path ("C:\ProgramData\Microsoft").  However, the Gatherer component still uses "D:\Windows\Search".  This means Windows Search uses both the system and non-system drives to maintain its indexes. 

 

Apparently, setting the Index Location option on the Advanced Options dialog (from the Indexing Options dialog) updates the configuration of the Gatherer component only once.  Subsequently changes do not update its configuration as shown by the following registry keys:


Registry Path: Computer\HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows Search\Gathering Manager

Registry Key: DefaultApplicationsPath
Current Value: D:\Windows\Search\Search\Data\Applications\
Original Value: %ProgramData%\Microsoft\Search\Data\Applications

Registry Key: TempPath
Current Value: D:\Windows\Search\Search\Data\Temp\usgthrsvc
Original Value: %ProgramData%\Microsoft\Search\Data\Temp\usgthrsvc

 

Unfortunately, I cannot update these registry keys manually because their access control lists grant Administrators read/query access only.

 

How can I configure Windows Search (including the Gatherer component) to use its original index location?

curious joe say:

I found a solution: Turn off Windows Search and then turn it back on.

 

After turning off Windows Search, I noticed that the aforementioned registry keys were deleted.  Of course, they were recreated after turning it on.  Also, note that Windows requires a computer reboot after enabling or disabling the feature.

 

Here are the steps to enable/disable Windows Search:

 

1. Click Start, Control Panel, Programs, Turn Windows features on or off

2. On the Windows Features dialog, navigate to the Windows Search node, deselect the check mark next to it to disable (or select it to enable if it is already disabled), and then click OK.

 

Hope this helps anyone with a similar issue.

luna wang say: 1601 Fix

1. Click START, and then click RUN.
2. In the OPEN box, Type msiexec /unreg, and then click OK
3. When this process is complete, click START, and then click RUN.
4. In the OPEN box, type msiexec /regserver, and then click OK.
5. Now install  Turbo Tax.



Another Possible Fix for 1601

1. Open a command prompt. (Start -&gt;Run-&gt; type "CMD.EXE" or "command")
2. Type msiexec /unregister and press Enter.
3. Type msiexec /regserver and press Enter.
