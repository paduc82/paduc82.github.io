i dont understand why the command prompt appear briefly then disappear. I couldn't read the message because it happens quickly.
All active windows are minimized during the process. I need help on how i can solve this issue. Please help


Presumably, from a place like WinKey+R, you are typing a command like IPConfig. When you do that IPConfig runs, and when it finishes, the CMD Windows in which it ran shuts down too. Since it runs very quickly, it's all too fast to read.

 

Instead of running IPConfig (or whatever it is you are running) from Winkey+R, run CMD there. That will open a CMD window, and you can now run anything you want there without its shutting down when the command completes.
