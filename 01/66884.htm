fixer upper say:

Hi,

Here is the quickfix. 

Set the laptop to standby instead of hibernate. Laptops can have issues with hibernate function. Here are the instructions to navigate there:

 

1. Go to control panel classic view. (Ctrl-Esc and then type "control" to quickly access control panel)

2. Double click on power options icon

3. Click "Change when the computer sleeps"

4. Click "Change advanced power settings"

 

Go to the sleep menu and

* set "Allow hybrid sleep" to: OFF

* set "Hibernate after" setting to: NEVER

 

Note: You will need to type this text in the field there is no drop down to select.

 

This is very likely to be the issue and the resolution. However, here are some additional troubleshooting steps should you have any issues:
