From what I know is, that NTFS flash devices have always cache enabled. You can't turn it off.

    Port Forwarding

    Login Information

    Gateway - 192.168.1.254

    Username - admin

    Password - admin

     

    Port Forwarding

    1. Login to the router by entering the default gateway IP address.
    2. The router will then prompt you to enter the user name and password.the default user name and password are: "admin", "admin".
    3. From the menu on the left select Advanced.
    4. From the sub-menu select Configuration.
    5. From the sub-menu select Virtual Servers.
    6. From the sub-menu select Port Mapping.
    7. Starting at the Application box enter Xbox Live.
    8. Using the Protocol drop down select all.
    9. Select the xbox from the Internal IP Address drop down.
    10. In the External and Internal Port boxes enter 3074.
    11. Select Add.

     

     

    Disable Port Forwarding

    1. Login to the router by entering the default gateway IP address.
    2. The router will then prompt you to enter the user name and password.the default user name and password are: "admin", "admin".
    3. From the menu on the left select Advanced.
    4. From the sub-menu select Configuration.
    5. From the sub-menu select Virtual Servers.
    6. From the sub-menu select Port Mapping.
    7. Using the radio button on the left of your rule select your port forward rule.
    8. Select Edit/Delete then confirm the deletion.

    Hi,
     

    1. Microsoft ISATAP is a IPv6 related component. The driver for ISTAP can be reinstalled via Device Manager -  Action – Add legacy hardware.
     
    2. Code 10 error description:
    The Code 10 error code is generated when Device Manager has insufficient information or the error that is bubbled up by the device drivers is not recognized by Device Manager. Additionally, this error may occur when one of the drivers that the device needs does not start. If many drivers are being used by a particular device, it can be difficult to determine the one that failed.
     
    3. Cause of this error:
    One of the major cause for this is that network adapter is not currently supporting this feature. You may download the latest driver / update the driver for your network adapter or chipset, and then try to uninstall and reinstall this component.
     
    4. Resolution:
    a. Open Device Manager, in the device Properties dialog box, click the Driver tab, and then click Update Driver to start the Hardware Update Wizard. Follow the instructions to update the driver.
    b. You may be prompted to provide the path of the driver to the location where you have download the latest driver from the hardware vendor’s Web site.
     
    Reference Link:
