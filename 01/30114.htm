ipellant say:

I have had similar with computers upgraded to Windows 10. It happened a lot during the Insider Preview updates.

The problem seems to be that the computer IP address is changed to v6 in Windows 10 which allows a much longer IP address identifier, and that may be different to what the WiFi router has stored in its' table.

The WiFi routers usually have options to refresh the ID's every day, month or never forget. It may be "remembering" a previous IP address for the computer name and that locks it off from Internet access privilege..? Not sure...

Turning the WiFi router off, then rebooting the computer, then turning the WiFI router back on usually fixed the problems I experienced.

Alternatively:

Using the Settings > Network & Internet > Manage Wi-Fi Settings then selecting the vagrant Network and pressing the Forget button will clear any corrupted settings on the computer. Go back to find networks and re-connect with the router. That fixed problems too.
