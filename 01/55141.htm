How do I created a desktop shortcut for Office Home and Student 2010 - Microsoft Community

rbs-jos2500 say:

How do I created a desktop shortcut for Office Home and Student 2010 on my Windows 10 PC?

*Original Title: shortcut for Office in Windows 10

How do I created a desktop shortcut for Office Home and Student 2010 on my Windows 10 PC?

hairyfool say:

Microsoft Office is not one program but a suite of programs. The shortcuts would be for Word, Excel etc.

With the old Start Menu you could right click and drag a menu shortcut to the desktop to copy it, this does not work with Windows 10 from the All Apps list.

Instead click on Start then All Apps, Scroll down to Office and expand it. Right click on any of the programs and select More > Open File location. This opens an Explorer Window with all the program shortcuts. Ctrl + click on each of the ones you want then right click and Copy.

Minimise the Explorer Window, right click an empty part of the desktop and select Paste. An icon for each of the programs will appear on the desktop which you can drag to where you want them.

Bear in mind that the concept is not to place all your programs on the Desktop, Windows 10 allows you to configure the Start menu how you want it. You can Pin any program in the All Apps list to the Start Menu and group or drag around the Tiles into any order you like. So you can create an Office group with the programs you use so that they are at the top of the Start Menu, you don't have to find the Desktop to get to them.
