hairyfool say:

"Preparing your desktop" indicates that the login is being treated as a completely new user with its own user profile. This normally occurs because the user profile has become corrupted and cannot be loaded, it is rarely the individual files but instead is the User hive of the registry.

When this happens Windows will create a temporary user profile which is removed when you log off but if nothing is done to fix the issue creates another temporary profile next time.

Try running a restore point dated prior to the issue arising to see if it can recover a working hive. Alternatively you can try creating another user altogether with Admin privileges and check the C:\Users folder for the "lost" user profile directory. All the missing files should still be there.

Use this as a little reminder that you need to do regular backups, in this case the files should be recoverable. It may not always be the case.

The main reason why the scroll bar is in the middle when you open File Explorer is incorrect folder settings. To resolve this, we suggest changing the File Explorer "Open to" settings. To do so, please follow the steps below:

  1. Open File Explorer.
  2. In the left pane, right-click Quick access, and then click Options.
  3. Under General, set Open File Explorer to: Quick access.
  4. Click Apply.
  5. Click OK.

For more information about Quick access, check this article.
