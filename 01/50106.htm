Hi wizrd8,

 

Did you perform an upgrade installation to Windows 7 or a custom installation of Windows 7?

 

Step 1: Firstly, check if you’ve Windows.old folder in your system partition or C: drive

In some scenarios, you may be unable to perform an upgrade installation from an earlier version of Windows. If you cannot upgrade, you may have to perform a custom installation. When you perform a custom installation of Windows Vista or of Windows 7, Windows creates a Windows.old folder that contains the files from your previous system. You may want to recover your personal files from the Windows.old folder, and move them to the custom installation of Windows.

a.    Click on Start; Go to Computer.

b.    Click on C: drive.

c.    Check if you see Windows.old folder. If yes, then follow step 2.

 

Step 2: How to restore your personal files after you perform a custom installation of Windows Vista or of Windows 7

http://support.microsoft.com/kb/932912

Note 1: Follow the steps under the section:If your previous operating system was windows vista

 

Note 2: Windows.old folder will not contain extracted application setup files so that you can run it directly on Windows 7. In other words, you cannot retrieve application from Windows.old folder. You’ll have to re-install the applications to make it work in Windows 7.

First off, are you using Windows 8 or Windows 8.1.  

To find that out do the following:

  1. Press and hold the Windows key on the keyboard and then press R.  That should open the Run box.  Type winver and click OK.
  2. When that window opens, see if it says: Windows 8 Microsoft Windows Version 6.3 (Build 9600).   If it does then you have Windows 8.1, if it doesn't then do the following:

  1. Open Control Panel and click on Programs and Features
  2. Uninstall iTunes.  If it says to restart after that then restart.

DO THE FOLLOWING IF YOU DO NOT HAVE WINDOWS 8.1 INSTALLED

  1. Make sure you save your work and close out any programs you have opened
  2. Open Control Panel and click on Programs and Features
  3. On the left, click on View installed updates
  4. Make sure KB2871389 and KB2917499.  if not go to http://www.microsoft.com/downloads and find the files, download them and install them.  If the screen says anything about restarting Windows then do so.
  5. Then go to http://www.microsoft.com/downloads and in the search box put in Windows 8.1 Update (KB2919355).  Then either go to the page for x86 (32-bit) or x64 (64-bit), download the file, when finished downloading, close the browser.
  6. Go to the location where the file was downloaded to and double-click on it and follow the onscreen instructions.
  7. IMPORTANT: While the file is installing, DO NOT USE THE SYSTEM
  8. When done installing if it says to restart the system, then click the option to restart.
  9. After you get back to the desktop hold down the Windows key and press R.
  10. In the Run box type in winver and click on OK.  If it says Windows 8 Microsoft Windows Version 6.3 (Build 9600) then you have upgraded to Windows 8.1

HOW TO FIX iTUNES

  1. Go to the Apple website
  2. Download iTunes
  3. After iTunes is done downloading, close out of the web browser and anything else that is opened.
  4. Go to the folder where you downloaded iTunes and install it.
  5. If it says to restart after that then restart your system
  6. After you get back to to the desktop trying using iTunes after that.  There might be a chance you need to add your MP3s to your iTunes library again

See if any of this works
