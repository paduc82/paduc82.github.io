bossdweeb say:

Cpu Turbo mode isn't quite the same as OC'ing the base speed so you should be able to enable that again without issues.

"I've even tried clocking my RAM to slower speeds and that didn't help."

By clocking at lower speed, do you mean you reset BIOS to default ? The highest non-OC speed on this mobo is 2666MHz.

The highest you want to go with your RAM Speed when not OCing the cpu is to use the XMP setting.

If that is not stable stick to the mobo's default speed.

The interesting thing about this issue is your RAM can still pass the test, but you can get seemingly random crashes when

you use OC RAM speeds without OC'ing the cpu.

I believe the reason  for this, it the test stress the cpu and RAM individually, so when gaming the cpu gets overwhelmed trying

to keep up with the too high OC RAM speed.

Generally games that use more RAM than others, will crash sooner (also why the machine does not crash when not gaming).

There is little or no real world gaming benefit to OC RAM these days anyway. Games are far more VRAM intensive than they once were,

and the gpu/VRAM are the most important factors in gaming performance.
