Changing the Wireless Channel - Microsoft Community

xb1r apollo say:

Changing the Wireless Channel

Login Information

  • Gateway - 192.168.1.1
  • Username - admin
  • Password - 1234

Changing the Wireless Channel

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open a web browser.
  3. Log into the router by entering the default gateway, 192.168.1.1 into the address bar of the internet browser.
  4. The router will then prompt you to enter the username and password. The default username is admin and the default password is 1234.
  5. Expand the BASIC directory.
  6. Select Wireless from the submenu.
  7. From the Wireless Channel selection drop down select the desired Channel. (Channels 1,3,9,11)
  8. Click Save Settings at the top of the page.

Xbox one power up issues with instant on?

My xbox one has been running in instant on mode since I first got it and ran great.... Until the June 29 '14 update. After the update I started having an issue with the xbox not wanting to power up using the controller. It would, however, power on with the button on the front or with the kinect.  Also, the controller didn't want to power down when you turned the xbox off. I tried power cycling and checking my batteries multiple times to no avail. The only time it would work was the first time or two after power cycling (unplugging the console). I finally went on the online support page yesterday and was told that I had an issue with my controller and that an exchange would take 2-3 weeks with me having to send my controller back first. Not wanting to be wuthout my xbox for up to 3 weeks I headed down to game stop and bought another  controller only to discover that the new controller had the same issue. I called Microsoft this time and after being sent to about 5 people and then getting disconnected I finally got a competent tech. A good bit of troubleshooting revealed that, when my xbox was in instant on mode  it was not responding to the controller's command to turn on because it thought it was already on and not in standby. It was determined that running my xbox in power saving mode until the next update released was the best approach and then if the next update doesn't fix it then send the xbox in for repairs. If anyone else has the same issue please post here and report it to Microsoft so that hopefully they will fix it as apparently I am the only one who has reported this issue so far.
