john john __ say:

Yes, after you share the CD/DVD drive on the desktop computer it will be available to other users on the network.

You should also be able to do this from the Command Prompt with the NET SHARE command without having to log on to Safe Mode.  The command syntax is:

NET SHARE ShareName=drive:

ShareName is just any relevant name that you want to give to the share, for example CDDesktop.  Drive is the letter of the CD drive, So if the CD/DVD drive is drive D: it would look like this:

NET SHARE CDDeskTop=D:

If the command is successful you will get a message stating that the share was successfully created.  Refresh the Desktop (press the F5 key while you are on the desktop) for Explorer to show the share icon on the newly shared cd drive.

Please not that you will be able to access the files on any inserted CD or DVD but you will not be able to burn to the shared drive from the laptop, as far as the laptop is concerned the CD drive will just be like any other hard drive, it won't know that it is a CD/DVD burner.

John

Please note that if you intend to share the Desktop resources it would be preferable to use a CAT5 Ethernet cable and network the computers via the network card ports (NIC) rather than via USB.  Almost all modern computers have built-in NIC ports, the network will be much faster by Ethernet cable.

If you connect the machines directly to one another via the NIC use a "Crossover" CAT5 Cable.  If you connect them through a router use "Straight" CAT5 patch cables.  These cables are readily available in different lengths at almost all computer supply stores.
