I made a project on movie maker, saved it to a thumb drive but It - Microsoft Community

denisedelavallade say:

I made a project on movie maker, saved it to a thumb drive but It doesn't play on other computers. What do I do?

I made a project on movie maker, saved it to a thumb drive but It doesn't play on other computers. What do I do?
considerate_guy say:

A project file isn't a movie.  A project file is like a recipe for making a movie.  In fact, the project file doesn't contain any photos, videos or music, just references to the files that exist on your computer.

 

Thus if you copy a project file to another computer, it won't play there.  The reason is you don't have any of the source files that were used to provide the preview on your original computer.

 

Instead of saving the project file and trying to use that on another computer, you instead want to create a movie.  This is like following the directions in the recipe to bake a cake.   The cake can be shared and eaten by others; the recipe is less tasty, especially since it is just instructions without ingredients.

 

To save your movie, go to the first blue File menu and choose Save movie (it's lower down in the menu so don't confuse it with Save project).  That will create a .wmv or .mp4 file that can be put on a thumb drive and played in another computer.
