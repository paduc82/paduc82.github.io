I would also suggest using a different DVD drive, or maybe copying the installation files to a USB stick (4gb or bigger) and performing a USB install.

This usually means either a bad DVD, or a read error on the DVD drive.

rick rogers say:

Hi,

If you are able to logon normally in Safe mode, then it means there is an application loading in normal mode interfering with the explorer shell. From Safe mode, click start and run regedit. Expand the branches to reach this key:

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Winlogon

Click on the key, then locate the userinit string in the right pane. Double click it, replace the contents with:

C:\Windows\system32\userinit.exe,

making sure to include the comma at the end. Click ok, close the registry editor and restart the system normally.
