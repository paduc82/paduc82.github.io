The problem comes from the update merrily removing programs it doesn't judge compatible with windows 10.

Some of it was smart (Windows Security Essentials -> rendered obsolete by Windows Defender), some of it was pointless (CPU-Z -> just needed an update), and some of it killed RAID 0 (Intel Rapid Storage Technology "IRST").

Finding the correct version of IRST driver can be a little tricky. Thankfully the setup will cancel and tell you if you are installing a version that doesn't suit your motherboard.

This might be a niche problem, but hopefully it'll help a few people.
