If you're unfamiliar with this tool it can be lethal, so great care is needed. To be safe, you can always physically remove any other good disks - apart from the system drive.

Determine your problem drive's hardware allocation number using:

At the next prompt type clean - please check, re-check & check again that this is the disk with problems as this operation will completey re-initialise the drive & data will be un-recoverable afterwards.

Close the command console.

Go back to the computer management window & the dirve should now show as a single unallocated block. Assuming the drive is working properly, you should be able to initialise it as a basic / raid etc. volume & format it as normal.
