Method 1: Check Wi-Fi drivers and settings in Windows 10

Usually, having the latest driver for your device is the best policy because it has all the latest fixes for your device. To check for the latest drivers automatically:

  1. In Device Manager, select Network adapters, press and hold (or right-click) Update Driver Software.

To find the wireless mode setting:

  1. In Device Manager, select Network adapters > the network adapter name.
