If you brake the raid the data *should* be still intact

Personnally I've seen too many problems with raid implimented by win, or by using the onboard motherboard raid controller, and as such the only way I would imlpliment raid is by using a third party hardware raid card. A decent one will cost a significant sum (more than your motherboard)

bill.p. say: Hi SuperSH, thanks for your question. I'm Graeme, a Microsoft Community expert and fellow user.

There's no way to make it select only the filename other than hiding file extensions.

You can do this in File Explorer by clicking the View tab, then deselecting the checkbox labeled "File name extensions".

Once you've done this, you won't see the file extensions in the filenames, and pressing F2 will only select the filename to change.
