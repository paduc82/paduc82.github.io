lemp say:

Do you still get an "unidentified network" error?  If so, what is the complete text of this error message and under what circumstances does it appear?

It's not clear if the diagnostic info you provided in your initial post is still accurate after you installed a new router and your ISP connection was reset.  However, the APIPA address that you reported (169.254.108.216) is generated when (a) your adapter is set to obtain its IP address automatically (usual default setting) and (b) the adapter cannot obtain an IP address from a DHCP server. 

Except in unusual circumstances (e.g., the router's DHCP server is disabled or broken), when this happens for a wifi adapter the cause is almost always an encryption or other security problem.  The first step in troubleshooting this is to completely disable all encryption and other security features (e.g., IP or MAC filters) on the router and see if the adapter can now obtain an IP address from the router.

You mention that you have tried the A6100 on the laptops.  Do the laptops have the same operating system as your desktop/tower?  If not, the drivers that you installed on the laptops may not be the same as the Windows 7 driver that you installed on your desktop and the comparison may not be completely accurate.  Also, did you use the A6100 in the same wifi mode when installed on the laptops as you use on the desktop/tower?  Even if the A6100 is not actually defective, at least one review characterizes its performance as "underwhelming in 2.4GHz mode and for 802.11n."  See http://www.pcmag.com/article2/0,2817,2426319,00.asp

You can always perform a new install without it invalidating your license.

When you upgraded from a previous version of Windows or receive a new computer preinstalled with Windows 10, what happened is the hardware (your PC) will get a digital entitlement, where a unique signature of the computer will be stored on Microsoft Activation Servers. The Windows 7 or Windows 8 genuine license you were previously running will be exchanged for a diagnostics key.

Anytime you need to reinstall Windows 10 on that machine, just proceed to reinstall Windows 10. It will automatically reactivate.
