greg carmack - mvp 2010-2020 say:
Hi. I'm Greg, a volunteer and 8 year WIndows MVP here to help you.

The place to start with a new hard drive is Disk Management, accessed by right clicking Start button.

Scroll down to find the drive in the map at bottom. If not formatted yet right click on drive to format a partition and if necessary assign a drive letter so it's detected. Then you should be able to access it in File Explorer, set up File History to back up to it, save a System Image backup to use in place of reinstall.

I hope this helps. Feel free to ask back any questions and let us know how it goes. I will keep working with you until it's resolved.
