ganesh achar b say:

Hi Vanamali,

Recovery drive hold a backup for Windows 7.

The 102 MB volume is labeled as System Reserved with NTFS file system, and System, Active, Primary partition attribute with no drive letter in Disk Management. The 102 MB system partition is used primarily as BitLocker partition for BitLocker encryption. Additionally, it also holds the Windows Recovery Environment (WinRE) and boot files with boot manager for booting up the computer for troubleshooting when there is no Windows installation DVD disc on hand. You can just copy the system drive to SSD in order to use SSD as a system disk.

Refer to the following link to know more about Deleting Dell's Recovery partition.

Delete Dell's Recovery partition.http://answers.microsoft.com/en-us/windows/forum/windows_7-system/delete-dells-recovery-partition/984572ea-4482-e011-9b4b-68b599b31bf5

"How to Refresh Windows 8 and Windows 8.1"

  • The PC boots into Windows RE (Recovery Environment).
  • Windows RE scans the hard drive for your data, settings, and apps, and puts them aside (on the same drive).
  • Windows RE installs a fresh copy of Windows.

  • The PC restarts into the newly installed copy of Windows.
