Follow these steps:

1. Revert back to the version of windows that you were using. You should see the option to do so in your Windows Update app. If you have already done this, naturally no action is needed, and you can skip to #2.

2. Go to your Sleep and Power settings. I navigate to things by using search, and typing in "sleep'. It is located within your system settings if that helps. While in these settings you need to set the sleep and screen settings to NEVER time out. This will ensure that Windows doesn't pause while doing your update., and no funny business occurs.

5. There are many drivers that are here, and you should check every last one of them to see if any need updated. You do so by clicking on each category to expand the view of the associated drivers. From there, hover a driver and right click. You will see update driver, and you then click it. Do this for each driver in each of the categories. For me, there were several that needed updated for Windows 10, including the one for my Intel HD Graphics.

6. Now for the most important step, and the one that is the true fix. In the Device Manage window you hopefully still have open(open it again if not), find the category for "Human Interfaces" and expand it to show the drivers. I'm not a computer expert, so you may find you have different drivers listed than I do. If you have the ones listed below, follow these directions. Right click on the driver, and then click on "Properties". Next you need to click on "Power Management", and check the box that says "Allow this device to wake up my computer". Now for the list of drivers that I implemented this  on.

  • HD-Complaint Touch Screen

These two are under the Keyboard category

  • Microsoft USB Dual Receiver Wireless Mouse(Miouse and Keyboard Center)
  • Standard P2/P Keyboard

These two are under Mice and Other Pointing Devices

  • Microsoft USB Dual Receiver Wireless Mouse(Miouse and Keyboard Center)
  • Synaptics SMBus Touchpad <--- You won't see this if not on a laptop.

Hello,

Thank you for posting your query on Microsoft Community.

As per the issue description, I understand that you want to change the images in live tile in Windows 10. I will certainly help you with the query.

 

Please follow the below mentioned steps to change the images in live tile in Windows 10

  1. Press “Windows key”, right click on Photo app
  2. Click “Turn Live tile off”
  3. Now open the “Photo app”, click on “settings” in the left pane
  4. Add photos folder in the sources that you want to view on the tile
  5. In the “photo tile” option, click on “recent photos”
  6. Close the current window, again press “Windows key”
  7. You should be able to see the selected images now

 

Note: It is advised to create a new folder containing your desired images and then pointing the source in step 4 to that folder
