Conditions to use the Extend volume

 

·         You need to have an unallocated space on the hard drive to extend a partition and the unallocated space should show up towards the right of the partition which we want to extend

 

·         You can extend a volume only if it does not have a file system or if it is formatted using the NTFS (New Technology File System) file system. You cannot extend volumes formatted using FAT or FAT32 (File Allocation table).

 

·         You can extend simple or extended volumes that are not system or boot volumes, as long as there is available disk space.

 

In your case, you cannot extend the volume on a partition that is labeled New Volume.
