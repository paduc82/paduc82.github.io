This can happen if the stored password is corrupted.  You can reset it this way:

  1. Go back to the netplwiz options

  2. Turn the "users must use a password" option on, then click Apply.

  3. Turn the "users must use a password" option off, then click Apply again.

  4. Restart the computer and check if it logs in automatically.

Where is Windows Boot Manager (BOOTMGR) Located?

Configuration data required for BOOTMGR can be found in the Boot Configuration Data (BCD) store, a registry-like database that replaced the boot.ini file used in older versions of Windows like Windows XP.

The BOOTMGR file itself is both System Reserved and does not have a drive letter.

If you don't have a System Reserved partition, BOOTMGR is probably located on your primary drive, usually C:.

techfreak_ say:

Skype on Xbox One only allows you to sign in with the account the console is currently signed into. You can sign out of Skype by clicking on the profile icon, then clicking sign out. This stops that account from being signed in, allowing you to remain signed into the console. When you attempt to sign back into Skype, you will only be presented with the option to sign back into the same account the console is currently signed into. The only way for you to sign into a different Skype account is to sign into the Xbox console using that different Skype account. 

Going to your profile:

Signing out of Skype: 

Signing back into Skype with the same account. Signing into another account, is not an option:

Add your Skype/Microsoft account to the console by clicking Add User. Once you sign in and switch to using that account, launching Skype will sign you into the Skype profile associated with that account.
