Let me prefix my journey by stating that I really like the small and powerful form factor of the Mac MINI.  It's an i7 processor that just FLIES, in a footprint no bigger than my desk phone.  OS-X, however, leaves a ton to be desired, which is why I like my Windows operating system.  

WIN 10

I have given up on hoping to get Win10 to function on my Mac MINI Bootcamp partition.  Win10 is incompatible with the original Bootcamp support drivers, which is why you will slowly lose things like sound, Wi-Fi and USB devices which worked perfectly prior to the Win10 upgrade.  Sure, these basic system components might work fine for a couple of months following the Win8.1 to Win10 upgrade, but as Win10 updates get pushed, you will gradually lose them, and there is no way to get them back.  Bootcamp with a WIn10 ISO does not work; I know, I tried. 

SOLUTION

Apple folk -- Stay on Windows 8.1 until Apple updates Bootcamp support to accommodate Win10.  Apple Bootcamp drivers DO support Win 8.1, and Win 8.1 takes up a lot less Bootcamp disk space than Win10!  If your Windows 10 build has you in a rage (like mine did), before you delete your partition, get your Win8.1 ISO build (see steps). 

STEPS TO INSTALL Win8.1 on Apple Mac Mini Bootcamp

  1. Make sure you have your Win8.1 product key and a large USB flash drive
  2. On any PC that has a working Windows partition and network access, download the Windows 8.1 64bit ISO image.  This site has the link to the executable:  http://windows.microsoft.com/en-CA/windows-8/create-reset-refresh-media
  3. Install the image to the USB flash drive (DOES NOT WORK TO USB Hard Drive)
  4. On the Apple bootcamp, select the options to download the Bootcamp drivers and to install "Windows 7 or newer" from the menu.  The Win7 link does not work (discontinued). 
  5. Bootcamp download will take a while (10-30 minutes depending on your network)
  6. Adjust your Windows partition when the window pops up
  7. Select the Bootcamp partition and select the FORMAT icon (this will reformat that space for the Win8.1 OS).
  8. Sit back and let it fly. 
  9. After the install and you are at your Windows desktop screen, go into the USB Flash Drive folder called "Bootcamp" and run the SETUP.EXE file.  This will install the Bootcamp switch and all the underlying Apple drivers onto your PC.
  10. 140+ Win8.1 updates later... 

I am HAPPILY up and running; WIndows 8.1 on my Mac Mini.  Sound, network, USB drives, plus Office 2013 connected to my Outlook.com account.  Oh, and note: Once I authenticated my Outlook.com account, all my contacts, mail and calendar got synced with Win8.1!  

BTW: The OUTLOOK.COM PASSWORD to sync with Outlook 2013 can be found HERE: https://account.live.com/proofs/AppPassword     <--BOOK MARK THIS SITE!!!  You will need if for anything that syncs to Outlook.com!  

Good luck! 

Rade
