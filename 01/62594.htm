You don't format the hard drive. This is what you do.. Note that unless you save all important stuff, it will be lost after the reset.

 

Your computer has been supplied with a recovery partition in place of disks..

This is how the recovery partition is accessed for the most popular makes..

For Dell, press CTRL + F11 directly after powering up the unit

For HP, press F11 directly after powering up the unit

For LG, press F11 directly after powering up the unit

For Toshiba, press and hold '0' BEFORE and during the power up

bossdweeb say:

"HDD which was for all other files and programs. however The SSD is the default drive and is nearly full, the HDD is nearly empty."

- Assuming you are saying you intended for stuff to be installed on the HDD, not the SSD...

When you install any program/game, or download anything, you should be given an option to 'Browse' to the

location you want to install to (often shown as 'advanced' options ).

If you do not manually select your preferred location while installing, all software will install automatically

to C:\Program Files(X86) or C:\Programs Files. They will often also install folders for Data in the Programs Data folder.

(Some programs you install on the HDD will also install folders in C:\ , but they generally contain files that don't

use much space. )

Never move folders that contain installed programs, and never move folders that programs install to keep data in

eg- your User ,  User\AppData  folders, etc. Doing so changes the file path = chaos !

- You can, however move data you have created/downloaded/saved from anywhere in your User folder,

or folder you may have created in C:\ to the HDD  with no problems.

- There would be a few exceptions. You can move games from Steam, but while you don't have to reinstall

the games, there is an involved process getting the Steam client sorted out.

- Moving programs also causes issues when trying to uninstall via Programs and Features.

- You can also have Windows change the location of folders such as your Music or Video to the HDD.

To have Windows move one of your User folders R/click on the folder in question and go to Properties>Location.

This method can cause complications at times, particularly with any folder/sub-folder created by a

program/game you have installed.

I've shied away from the move location option, and just made a new music folder on the HDD, and left the original alone.

How to do a clean install:

Step 1: Use the Upgrade Assistant to qualify the PC and pay the $40 (US)

Step 2: When the "Install Windows 8" message appears, select "Install by creating media" to create a bootable USB flash drive or an .iso (a disk image) file.

Step 3: Wait for the Assistant to download the Windows 8 Pro installation package and create the bootable media or .iso file. The flash drive must be have 3GB or more of free space available.

Step 4: Burn the .iso file, if that's the path you chose, to a blank DVD.

Step 5: Use that USB drive or DVD to boot the PC.

Step 6: When you get to the screen in Setup that asks, "Which type of installation do you want?' choose "Custom."

Step 7: Select a disk partition for the OS in the next screen. At this point, you can create a new partition if there's enough space available, or reformat the drive.

Step 8: Continue with the installation of Windows 8 Pro.

 

You will likely have to do a phone activation after installation is complete.

1.       Press Windows key + X then click Run, then type:   slui.exe  4

2.       Next press the 'ENTER' key

3.       Select your 'Country' from the list.

4.       Choose the 'Phone Activation' option.

5.       Stay on the phone (do not select/press any options) and wait for a person to help you with activation.

6.       Explain your problem clearly to the support person.
