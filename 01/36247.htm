shekhar s say:

Hi Gigishine,

 
The Operating System loads only after it detects Media on your computer either CD ROM or Hard disk.
Windows 7 do not have access to change the boot priority.
The original boot configuration is used to access the boot files that are required to boot the computer and has to be checked in the System Configuration utility so do not try uncheck that option.
 
Your BIOS requires to be updated to the latest version and firmware. I would recommend you to contact the BIOS manufacturer for additional assistance.

Note:
Modifying BIOS/complementary metal oxide semiconductor (CMOS) settings incorrectly can cause serious problems that may prevent your computer from booting properly. Microsoft cannot guarantee that any problems resulting from the configuring of BIOS/CMOS settings can be solved. Modifications of the settings are at your own risk.

amrita m say:

Hi Dartcap,

 

Welcome to Microsoft Answers!

 

 

Linksys wireless adapter wusb300n is not compatible with Windows 7. You may still try to get it work by running the drivers in compatibility mode.

To run the Program Compatibility troubleshooter

  1. Open the Program Compatibility troubleshooter by clicking the Start button, and then clicking Control Panel. In the search box, type troubleshooter, and then click Troubleshooting. Under Programs, click Run programs made for previous versions of Windows.
  2. Follow the instructions in the troubleshooter.

If you cannot install a program, insert the installation disc for the program and, using the troubleshooter, browse to the program's setup file, usually called Setup.exe, Install.exe, or something similar. The troubleshooter is not designed to work on programs that have an .msi file name extension.

For more details on making older versions run on Windows 7

Make older programs run in this version of Windows

http://windows.microsoft.com/en-us/windows7/Make-older-programs-run-in-this-version-of-Windows

 

You may also check the link which talks about different ways you may try to get it work.

http://tinyurl.com/yl4noul

 

You may also try to ‘Add legacy hardware’ mode from the device manager. Follow the steps to add legacy hardware.

  1. Click on Start.
  2. In the Start Search Bar type Device Manager, and hit Enter.
  3. In the Device Manager window select the Wireless Network Adapter.
  4. Select Action button and go to ‘Add Legacy Hardware’.
  5. Follow the steps on the screen and add the hardware, this is a workaround as the drivers are not available for the same.

 

Hope this information is helpful.
