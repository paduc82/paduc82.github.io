Hello Mplan,

Error Code 0x8000FFFF will come up when you're trying to create a new folder on a write protected disk in Windows 10. By default, read and write access is allowed for these type of storage devices for all users. You can modify or set to allow or deny read and write access

Some USB storage devices have a physical switch that enables you to lock or unlock write protection. However, the location of the switch may vary. If your USB storage device has this type of switch, then you need to unlock it.

Now, if your USB storage device doesn't have a physical or visible lock/unlock write-protection switch, you need to unlock or disable the write protection manually, first thing to do are the following:

  1. Insert the USB storage device.
  2. Go to Control Panel.
