brian tillman [outlook mvp 2007-2019] say:
I think it means that your search showed you the contents of "C:\Windows\system.ini".  This file was used back in the days when Windows ran on top of MS-DOS and listed the drivers that needed to be installed at boot time.  In current Windows versions, it controls the driver environment for legacy 16 bit applications.  Usually it's not needed, but it's perfectly safe to have around, particularly when it contains what you describe.  Those are pretty typical system.ini file contents.  The FON files are CGA and EGA display font files, mmdrv.dll is a multimedia driver for a sound card, and timer.drv is a timer driver.
