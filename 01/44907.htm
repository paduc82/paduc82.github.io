Restart directs me to run Setup!? - Microsoft Community

johnpjsmith say:

Restart directs me to run Setup!?

Recently when I attempt to start or restart pc, Windows does not open..

rather I get Press F1 to run Setup

I then click Disable Changes and Exit

and then Windows opens.

Please advise,

Thanks,

hairyfool say:

This has nothing to do with Windows at all as the BIOS is generating the message before loading the operating system.

The BIOS loads a number of parameters held in non-volatile storage (also known as CMOS) each time it starts. Values such as what drives were configured to boot from, how much memory was installed, how the I/O ports are configured etc.

What is happening is when those values are loaded there is a discrepancy with the actual hardware detected and the BIOS wants you to go into setup and confirm that the new settings are correct. Because you are selecting Disable changes then you are not allowing the CMOS memory to be updated so the next time it starts the message comes up again. If you were to "Save changes and exit" then the CMOS would match the hardware and the next time it restarts it should start normally.

Apart from physical hardware changes (install more memory, change drives) the other main reason why this occurs is a flat battery. CMOS may retain the settings if power is disconnected by the use of a separate battery, with desktop computers normally a flat "penny" cell which is most often a CR2032. (laptops are variable). When that battery starts getting flat (3-5 years) the values are corrupted and the BIOS wants to correct them, your selection prevents it.
