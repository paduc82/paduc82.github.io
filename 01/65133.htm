This is a bug in the xbox system software.

The background download feature on xbox 360 is bugged and has been for months. If you don't leave it on the active downloads screen when downloading something or if you let the console go into standby mode while downloading you will get the can't download error and the download will corrupt.

You first need to go into storage management. Find the partially downloaded file then delete it, then clear the cache then turn the console off for a minute.

You then need to start the download again. But you must stay on the active download screen and not do anything else with the console. If you do anything else there is a chance you will get the can't download error again.

Before doing this though you should go into the system settings and turn off the auto power off feature.
