Hello Matthew,

Thank you for posting your concern in Microsoft Community and welcome to the Windows 10 Family.

In some cases, you will experience BAD_POOL_HEADER blue screen errors after you’ve installed new hardware or software. So this error can be a result of incorrectly configured, old, or corrupted device drivers. As well it can be due to corruption in Windows registry from a recent software change.

As an initial recommendation, I suggest you to deploy Driver Verifier to check on any suspect drivers.

Driver Verifier monitors Windows kernel-mode drivers and graphics drivers to detect illegal function calls or actions that might corrupt the system. Driver Verifier can subject the Windows drivers to a variety of stresses and tests to find improper behavior.

You can run Driver Verifier on multiple drivers simultaneously, or on one driver at a time. You can configure which tests to run, which allows you to put a driver through heavy stress loads or through more streamlined testing.

http://msdn.microsoft.com/en-us/library/windows/hardware/dn305113(v=vs.85).aspx

Windows has in built Driver Verifier as well to identify problematic drivers. The Driver Verifier Manager, tests each specified driver at startup. If it detects a problem, it identifies it and then stops it from running. Here's how:

1. Type verifier in Windows Start menu Search box and hit Enter.

2. Navigate to Create Standard Settings > Click Next > Automatically select unsigned drivers > Next.

3. At last you have two options Cancel or Finish.

  • If you click on Cancel, no changes are made to your system configuration.

From the list which is presented, you can manually try to Roll back, Update, Disable or Uninstall the Driver.

  • If you click on Finish, then restart the system, you may be presented with an error message.

This will include the name of the buggy driver along with an error code.

4. Now reboot the system in Safe Mode with Networking to update, disable or uninstall the particular driver.
(Safe mode with Networking starts Windows in safe mode and includes the network drivers, services needed to access the Internet or other computers on your network).

5. Type verifier/reset in Windows Start Menu Search and hit Enter to disable the Driver Verifier Manager.

Additionally, you may also refer to the below Microsoft link on Troubleshoot blue screen errors.

http://windows.microsoft.com/en-us/windows-10/troubleshoot-blue-screen-errors

The Microsoft Account can only have one password at a time, so if you sign in with it on the PC it will be the same as signing into your MS Account email, etc. online.
