With a desktop PC the best place to download the graphics driver would be the Nvidia site, but

with laptops the driver should be from the mfg of the machine.

Do not use Device Manager to install drivers for dedicated graphics cards.

lenovo.com/Laptops-and-netbooks/Lenovo-Edge-Series-Laptops/edge-2

- Note that Win10 wants to force it's preferred driver and will reinstall that driver when you reboot after

uninstalling your current driver (it may not show in Programs and Features when done this way).

You can allow Windows do it's thing and running Windows updates may install another driver as well.

If you are happy with the result stick with the driver Win10 installs, but if you want

a specific driver there are several things that help prevent Win10 barging in.

1) Use the Show/hide tool linked in the article below to block Windows Updates from installing graphics drivers.

How to temporarily prevent a Windows or driver update from reinstalling in Windows 10

2) Uninstall the Nvidia Graphics Drivers using DDU (Display Driver Uninstaller )

( I used to recommend uninstalling via Programs and Features, but this does not get rid of some Registry

entries that seem to trigger Win10 installing it's preferred driver.  DDU does a full clean)

Display Driver Uninstaller Download version 14.1.0.0

3) Disable your internet connection before rebooting as another safety measure.

4) Reboot and install the new driver.

Use the custom install option and do not install components you do not use. eg- if you don't have

Nvidia 3D Vision glasses do not install the 3D Vision driver.

Similarly, if you do not use HDMI for audio do not install the Nvidia HD Audio driver.

- The only required components are the main driver and PhysX.
