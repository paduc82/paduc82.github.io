leatherderby say: I see your posting is several years old so you might have solved it by now but, in case you haven't, here's my experience.

I followed the standard steps from my experience with Windows 98, 2000, and XP. Windows 7 has a few additional items that are well described in;

http://social.technet.microsoft.com/Forums/en-US/w7itpronetworking/thread/033c418b-1096-4df7-bfad-fd3d431f3cd5

But I ended up with 1-way sharing. I could share all files and folders from my XP machines on the Windows 7 but when I tried to access Windows 7 files from the XP machine I got an error message that I didn't have permission.

To make a long story short, what I found out after a long time was that, when you set up sharing, Windows 7 does not change sharing attributes of existing folders. For example, if you have an existing partition that you want to share and you set up sharing for that partition, all existing folders will NOT be shared - new folders will be. Now you have 2 options;
- set up sharing for each folder in the root directory of that partition
- create a new folder, move contents of folder X to the new folder, delete X, rename new folder to X

Microsoft for all their wisdom made this slight change in how Windows 7 works relative to all preceding operating systems and it messed us up.

benjiedog say:

I finally got this sorted for me after trying many permutations, including having the network set as a Workgroup.....but that was not necessary. Now I have my Windows 7 computer set to a homegroup, and under Advanced Sharing options:, network discovery turned on, file and printer sharing on, public folders on, media on, password protection off, and Windows managing Homegroup connections. The key was realising that because a folder says "shared" , it isn't necessarily (it can be given the permission to be shared which has no effect till you say who to share it with).

 

Go to the folder to be shared and right click and select the "Share With" near the top  of the list, and select "Specific People". This opens a window probably only showing the Windows 7 computer administrator. But click the down arrow when others will magically appear to select from, and eg select "Everyone", then click "Share". (It's no use sharing with the Homegroup as only the Windows 7 computer will be a member of the Homegroup in a mixed computer network - although this will give the appearance that the folder is shared, it will not be!).

 

On clicking "Share", the window will close but right click the folder again, hit "Properties" and select the "Sharing" tab. Select "Advanced Sharing", tick for "Share this Folder" and click "Permissions" and then set for each user named. Note that if you set Permissions and even Add people to set them for in this window, it will have no effect unless you have first "Shared With" these people as above!
