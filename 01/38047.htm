Hello,


The Lexmark X4650 should be compatible with Windows 8 and 8.1. It could be that something happened during the upgrade that is causing an issue. Could you please try to download and reinstall the Lexmark driver again.


  1. Unplug the device from the computer (if connected via USB).
  2. Delete any existing print queues from the "Device and Printers" folder by right clicking on the device icon and selecting "Remove Device"
  3. Select the "Microsoft XPS Document Writer" object and then select "Print Server properties" from the folder menu.
  4. Select the Drivers Tab
  5. Select the "Lexmark X4650" (or all Lexmark drivers available) and press the Remove button.
  6. Select "Remove driver and driver package" and press Ok then Yes.
  7. Select "Delete"
  8. Reboot the PC.
  9. Download and run the Lexmark X4650 installer: http://support.lexmark.com/index?segment=SUPPORT&userlocale=EN_US&locale=en&productCode=LEXMARK_X4650&page=product&frompage=null#1

This should hopefully fix the issue with your device.
