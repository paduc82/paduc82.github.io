lemp say:

What do you mean by "desktop files"?

Files with the extension .ini used to be used as "configuration" files back when MSDOS was the Microsoft operating system.  The use of these files continued through Windows Me.  With the introduction of Windows NT (which is the basis for modern versions of Windows), Microsoft switched to the use of the Windows Registry, and encouraged developers to stop using .ini file.  Most have done so, and any .ini files you see today are likely leftovers or used for potential backwards compatibility.

If you want to open an .ini file to see what it is, just right-click and select Edit.  These are plain text files that open with Notepad.

"ink" files -- starting with a lower case I -- are used by a program named MimioStudio.  https://mimio.boxlight.com/mimiostudio-classroom-software/

You are probably looking at .lnk files -- starting with a lower case L.  These are "link" or "shortcut" files.  They are references or pointers to another file.  By default, the icon for a lnk file shows a curved arrow in the lower left corner; the rest of the icon depends on the nature of the file being referenced.

lemp say:

Desktop.ini files are normally hidden.  They are "protected system files" that describe how a folder is displayed.  Are you just curious or is there another reason why you are attempting to open such files?

You should be able to open a Desktop.ini file either by double-clicking it or right-clicking and selecting Edit.
