I would like to inform that in WordPad you can insert Images or Objects to your documents as there is no clipart option, On the contrary, Microsoft Word has the above mentioned feature.

Microsoft Word users can copy an image from any other program and paste that image into the document at the location you want it to appear, or,

  1. Open Microsoft Word
  2. In Microsoft Word click Insert
  3. Click Picture
  4. Click either "Clip Art" or "From File"

If the image you want to place into your document is a file stored on your computer, choose From File and browse to the location of your image. If you do not have a picture in mind and want to see some of the included clip art that came with Microsoft Word, select Clip Art.

Insert an image in Microsoft WordPad:

Microsoft WordPad users can copy an image from any other program and paste that image into the document at the location you want it to appear, or,

  1. Open Microsoft WordPad
  2. In Microsoft WordPad click Insert
  3. Click Object
  4. Select the image file you're inserting and click Ok
