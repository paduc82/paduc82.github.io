If that doesn't work or if you prefer, you can treat the situation like a corrupted user profile.  To fix this use http://windows.microsoft.com/en-AU/windows-vista/Fix-a-corrupted-user-profile.  If that is the only available Administrator Profile (you need to be an administrator to fix this), enable the Hidden Administrator Account (HAA) using http://www.microsoft.com/communities/newsgroups/en-us/default.aspx?&lang=en&cr=US&guid=&sloc=en-us&dg=microsoft.public.windows.vista.administration_accounts_passwords&p=1&tid=d20f9db4-7b2c-48be-a087-7835dc2a9055&mid=d20f9db4-7b2c-48be-a087-7835dc2a9055.  If you can't remember the password, try nothing as that's probably what you (or the vendor) did during installation.  Make sure you have at least two Administrator profiles with strong passwords (that you will remember) along with your standard user profile for daily use.  Once done, be sure to deactivate the HAA to save it in case it's needed again and for security purposes (since people often try to hack into systems using this account).  Do NOT use the HAA as your administrator account because if you lose the only administrator account on the system or it becomes corrupt again then you are hosed.

I hope this helps.

Good luck!
