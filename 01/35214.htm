Hello kippaxblue,

To transfer the song or the album to your MP3 player, you will have to set the Sync setting to Manual.

If you have already set the Sync Setting to Manual, and would like to clear the Sync list so that you can add new albums or tracks, here is what you need to do:

1.      Open Windows Media Player (the program should already be open if the sync list is full, though).

2.      Click on the button labeled "Sync" at the top of the program. The sync list will then be shown on the right side of Windows Media Player.

3.      Click the "Sync List" option at the top left corner of the sync list. Options will drop down.

4.      Select "Clear List." The Windows Media Player sync list will then be cleared.
