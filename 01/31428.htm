                                        SC.exe config trustedinstaller start= auto 

     

    To do this, right click on Windows Start > Windows Powershell (Admin).  Type (or copy and paste) the above command, and hit Enter.

Jamshed Muhammad for the one command line that worked (see Jamshed's succinct comment under this thread â€¦ Others have used the four command lines in E for Excel's post, I just used the above single line per Jamshed's post.  It worked; I stopped.  Above all, thank you AnthonyB2 for the "type SC.exe as opposed to just sc" tip (posted as a comment in this thread), which was the key to a successful update installation for me.  This resolved  the "invalid this, invalid that" text readout that I obtained in the Powershell text response when I used just SC.)

     

    If you see "Success" in the text readout after you hit enter, proceed.  If you don't obtain a successful execution, you might try it without the ".exe" in the above command line.  If you are seeing a bunch of "invalids" in the text readout, as I did before I added the ".exe", then this is probably not the solution for your computer.

     

    2.  Restart your computer.
