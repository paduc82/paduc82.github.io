.

.

Windows 7 - 8 - 8.1 Registry Edits to prevent Windows 10 Upgrade

Note: If your computer has (likely) already downloaded the GWX file, you should first uninstall it, then edit the Task Scheduler to remove the GWX task.  Also, if your computer has already downloaded the Windows 10 update itself (but you have not yet run it), you should first uninstall ("remove") it from Windows Update.

Note: The following information comes from an article originally published at "AskVG.com" and authored by Vishal Gupta, but it is no longer available.  I had printed the instructions in PDF format, which is why I still have the information.

These are the actual Registry Edits that will prevent GWX from being downloaded and will (forever) prevent Windows 7, 8 and 8.1 from upgrading to any version of Windows; (they _can_ be "undone"):

Edit #1):

[HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\GWX]

"DisableGWX"=dword:00000001

[HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\WindowsUpdate]

"DisableOSUpgrade"=dword:00000001

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\WindowsUpdate\OSUpgrade]

"AllowOSUpgrade"=dword:00000000

"ReservationsAllowed"=dword:00000000

Edit #2):

[HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\GWX]

"DisableGWX"=-

[HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\WindowsUpdate]

"DisableOSUpgrade"=-

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\WindowsUpdate\OSUpgrade]

"AllowOSUpgrade"=dword:00000001

"ReservationsAllowed"=dword:00000002

Detailed Instructions:

METHOD 1): Hide "Get Windows 10" Icon in Taskbar:

This is the easiest way to get rid of the icon. You can right-click on the date/time in Taskbar system tray and select "Customize notification icons" option. It'll open a new window. Now look for GWX (Get Windows 10) entry in the list and change its value to "Hide icon and notifications" using the drop-down box.

Apply the changes and it'll hide the White Windows flag icon in Taskbar notification area.

PS: If the options (drop-down boxes) to change behaviors are grayed out or disabled, then uncheck "Always show all icons and notifications on the taskbar" checkbox and it'll allow you to customize icons behaviors.

METHOD 2) Kill "Get Windows 10 (GWX)" Process Using Task Manager:

1. Right-click on Taskbar and select Task Manager option. It'll open Task Manager. You can also directly launch Task Manager by pressing Ctrl+Shift+Esc keys together.

2. Now go to Processes or Details tab and select GWX process and then click on "End Process" button. You can also right-click on GWX process and select "End Process" or "End Task" option. Or you can simply select GWX process and press Delete key.

It'll kill GWX process and will also remove its icon from Taskbar notification area. If the icon is still visible, just hover mouse cursor over it and it'll magically disappear.  If you follow above mentioned method, it'll remove the app and its icon temporarily. After some time or next logon, the app and its icon might come back. The following will disable "Get Windows 10" app and its Taskbar icon permanently:

NEXT) Disable "Get Windows 10" Using Registry Editor:

**Registry Tweak 1:

1. Type regedit in RUN or Start search box and press Enter. It'll open Registry Editor.

2. Now go to following key:

HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows

3. Create a new key under Windows key and set its name as GWX

So the final key path would be:

HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\GWX

4. Now select GWX key and in right-side pane create a new DWORD DisableGWX and set its value to 1

**Registry Tweak 2:

1. Again in Registry Editor, go to following key:

HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows

2. Create a new key under Windows key and set its name as WindowsUpdate

So the final key path would be:

HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\WindowsUpdate

3. Now select WindowsUpdate key and in right-side pane create a new DWORD DisableOSUpgrade and set its value to 1

**Registry Tweak 3:

1. Again in Registry Editor, go to following key:

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\WindowsUpdate\OSUpgrade

2. In right-side pane, look for two DWORDs AllowOSUpgrade and ReservationsAllowed and change their values to 0

After applying all these tweaks, close Registry Editor and restart your computer to take effect. It'll permanently disable "Get Windows 10" app and will also remove its icon from Taskbar. In future, if you decide to re-enable the app and its icon, simply delete the DWORDs created in above steps.

a. user say:

There's no set schedule for delivering updates. There are more than 700 million Windows 10 devices that need updating, so delivery is staggered.

There's nothing wrong with checking for updates manually. The point is: Unless an update has a new feature that you must have now, or fixes a problem that you must fix now, the conventional wisdom has always been to wait at least 2 or 3 months before installing a new version of Windows (now known as a 'feature update.') This is to let early adopters, professional testers and corporate IT departments discover the bugs that are virtually guaranteed in a project as huge and complicated as Windows, and to give Microsoft time to fix them.

It's easy to decide whether a feature update has a new feature that you must have now, or fixes a problem that you must fix now, because Microsoft publishes this information in advance, as well as any bugs that have already been discovered.

No software is 100% bug free. However, the most obvious bugs usually show up within the first few months after release. Thus the conventional wisdom.

The October 2018 Update (v.1809) continues to be plagued with bugs - most recently by a bug that can affect Windows Media Player playlists - so no one knows when it will finally be released for good. Personally, I deferred this update for 120 days.
