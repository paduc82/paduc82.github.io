zedrick del say:

I apologize for the late revert. When you install software that uses the printui.dll dependency, the software should automatically register the file for you. In some cases, your DLL file may not register properly, and as a result, will provide a printui.dll not registered error. Fortunately, you can use a built-in utility called Microsoft Register Server (regsvr32.exe) to re-register your printui.dll file. To do this, follow these steps:

  1. In the Search box, type cmd, right-click on the result and select Run as administrator.
  2. Click Yes when a permission dialog box prompted.
  3. Type the following commands: regsvr32 /u printui.dll
  4. Press Enter. This will un-register your files.
  5. Type the following command: regsvr32 /i printui.dll
  6. Press Enter. This will re-register your file.
  7. Once done, close the Command Prompt window, restart your device to apply the changes and check if the issue persists.
