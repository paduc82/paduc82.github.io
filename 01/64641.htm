To run the MRT just Start - type in Search box -> MRT   find at top of list - Right Click on it Run As Admin.

You can also make a shortcut to it on your desktop and/or start menu.

The Updates go into the "C:\Windows\SoftwareDistribution\Download" where C: is your drive where Windows
resides and could be another letter. However from there they are installed as needed and some endup in the
"C:\Windows\winsxs". However you should not need to go into those at all, and messing around it them can
cause extreme grief.

So basically the updates that are autoinstalled should be transparent and all you need to do is run your programs
as usual.
