Like WinNT, Win2K, WinXP, and Vista before it, Windows 7's NTFS
file security paradigm doesn't rely on, or allow, the crude and
cumbersome method of password protection for individual applications,
ead, it uses the superior method of explicitly
assigning file/folder permissions to individual users and/or groups.

As a crude work-around, you can place the file in a compressed
folder, and set a password to uncompress the folder to view/access its
contents. Also, some applications allow the user to password-protect
their specific data files. Read the specific application's Help files
for further information on this capability.

For your situation, the simplest course of action would be to
create a limited user accounts for anyone else who uses your computer,
and configure it so it doesn't have permissions to access your personal
files.

vanessa yar say:

Hi Mathias,

The error Network Path Not Found (Error 0x80070035) could be triggered by the firewall or antivirus installed on the computer, authentication problem, or due to network security. Upon searching about this error, I found two workarounds that have helped several forumers. Please follow these steps and see if it also works for you.

Enable NetBIOS over TCP/IP:

  1. In Cortana search, type in Control Panel, and click it from search results.
  2. Click Network and Internet, then Network and Sharing Center.
  3. In the left pane, click Change adapter settings.
  4. Right-click on Local Area Connection (if wired) or wireless network connection (if wirelessly connected) and choose Properties. 
  5. Click Internet Protocol (TCP/IP) Version 4 to highlight it, then click Properties.
  6. Click Advanced at the bottom.
  7. In the Advanced TCP/IP Settings windows, click the WINS tab.
  8. Under NetBIOS setting, click Enable NetBIOS over TCP/IP, and then click OK > OK > Close.

Add Windows Credentials

  1. In Cortana search, type in Control Panel, and click it from search results.
  2. If it's on category view, set it to Large or Small icons.
  3. Select Credential Manager from the list.
  4. Under Windows Credentials, click Add a Windows credential, then add the username and password of the laptop.
  5. Click OK.
