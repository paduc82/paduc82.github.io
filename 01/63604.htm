a. user say:

The UltraISO description reads:

 UltraISO  is an ISO CD/DVD image file creating/editing/converting tool and a bootable CD/DVD maker.

It sounds like you want to make a bootable USB flash drive and to do that I would use rufus which you can get here:

https://rufus.akeo.ie/

The error messages like this:

PXE-E61: Media test failure, check cable

PXE-M0F: Exiting boardcom PXE ROM.

No bootable device

PXE stands for "Preboot eXecution Environment" sometimes known as a network boot.

That means that your system of unspecified make and model is trying to boot from the network which in most environments is never going to work.

If the BIOS of your system of unspecified make and model there is probably a section that specifies the boot order of devices it will look at to find an operating system.

The order in which the BIOS looks for devices to boot from might be:

1. Hard Drive

2. CD/DVD Drive

3. USB

4. Floppy (if there is one)

5. Network

Either the boot priority is wrong or the BIOS is trying each device in order, failing to find anything to boot on and the last thing it tries is the network and then it gives up.

When you power up or restart some systems you might briefly see a menu to press some F key (F11 for me) and then you can choose the boot device (HDD, CD/DVD, USB, etc.) or you are going to have to get into the BIOS and adjust the boot order/priority.

Of course first you must create your bootable USB flash drive properly and it doesn't look like UltraISO is the thing to use.

Use rufus and you will be done in just a few clicks.
