henzeberkheij say:

Apparently there is something wrong in the recognition part of the usb devices when they are on the same pinout/internal hub.

every motherboard has one or more pin outs for connecting usb ports, these are internal hubs.  usually one or more usb ports are connected to one of those. for example 2 ports on hub 1, and 2 ports on hub 2.

so If you connect a device to one of those usb ports the computer see's device 1 on usb-port 1 on internal hub 1. => 1-1-1. 

the the device on the second usb port is recognized as device 1 on usb-port 2 on internal hub 1. so: 1-2-1.

Somewhere along the line telling the computer which USB device is the touchinput for which monitor something goes wrong. I think it has something to do with having two touchinput devices on the same internal hub.

This is what I think happening: You tell windows to have monitor A linked to the touchinput device 1 on port 1 on hub 1, and monitor B linked to touchinput device 1 on port 2 on internal hub 1. Somehow windows only remembers device 1 on internal hub 1, linking both touchinput devices to one monitor, monitor A. which is wrong.  

to fix it you should find a usb port that is connected to an other internal hub. so where monitor A is linked to touchinput device 1 on usb port 1 on internal hub 1, monitor B will be recognized as touchinput 1 on usb port 1 and internal hub 2.

if this is too technical, just try out all usb ports you have directly on your computer. try to avoid external usb-hubs. 

if after the switch your inputs are linked to the wrong monitor, you can use the "Tablet-PC" tool in windows to point out which input should link which monitor.

Since firmware updates need to be run just once, it would be inappropriate to include them in the list of Default Programs.

I do not own a Nikon camera but I know that firmware updates usually go like this:

  1. Download the update file to your PC's Downloads folder.
  2. Double-click it to run it.
  3. It will prompt you to connect your camera.
  4. It will do the firmware update.

In other words, it runs from your PC. It does not run from your SD card.

If this is not the way it happened with your upgrade then you need to read the installation instructions. If still unsuccessful, send a note to the Nikon helpdesk.
