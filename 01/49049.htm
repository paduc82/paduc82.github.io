andre da costa say:

OEM versions of Windows 7 are identical to Full License Retail versions except for the following:

- OEM versions do not offer any free Microsoft direct support from Microsoft support personnel

- OEM licenses are tied to the very first computer you install and activate it on

- OEM versions allow all hardware upgrades except for an upgrade to a different model motherboard

- OEM versions cannot be used to directly upgrade from an older Windows operating system

 

 

So I would recommend you install the new hardware first, then proceed to install Windows 7 on the new hardware.

Control panel,

Clear the IE11 checkbox,

OK,

To reload IE11

Control panel,

[View by - Small icons, if necessary to see the next item],

Programs & features,

Turn Windows features on or off [this is  on the left-hand side of the window],

Set the IE11 checkbox,

OK,

OK,

Restart the computer

john_barnett say:

Hi roseann.loasby,

As the display has physically moved to the right the problem is more to do with your monitor than Windows. A monitor is just like a television and relies upon vertical hold and horizontal hold options to keep the picture in place.

There should be options on your monitor to make alterations to the vertical/hortizonal positioning (in your case horizontal). Most monitors have an 'auto' option which automatically adjusts the picture diaply so it is positioned correctly. It is nothing serious, however, you do need to check your monitor user guide to see what options you need to use the re-adjust the horizontal positioning. If there is a 'menu' button on the front of your monitor try pressing that to bring up a display on your monitor. You will then need to either use the 'auto' option or else click the left/right arrow buttons (usually either side of the menu button on your monitor) to adjust the menu and then the position.

If you are not confident to do this yourself ask someone with more knowledge to do it for you. Pressing the wrong button can make the screen more out of alignment, so be careful, otherwise it will be  a case of trial and error until you have reset the monitor correctly.

It is a simple enough job to do but because every monitor is different it is difficult to write specific instruction because I could tell you to press one button and that button may not be there, so take a look at your monitor user guide first to familiarise yourself with what buttons are available.
