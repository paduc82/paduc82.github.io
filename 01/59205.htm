frederik long say:

I'm sure many people have had the same occur to them, and probably would advise me to just leave it as it is.

-> What you perceive as a problem could be considered an irritation but is not a problem as far as Windows is concerned. You need to be aware that every account consists of two entities:

  • The account name, e.g. JBrown
  • The profile folder name, e.g. C:\Users\JBrown

In most cases the two are the same. However, they could be totally different and it would not matter in the least.

While you can change the account name from JBrown to Everybody, the change is only superficial. It's a nickname. The underlying account name remains the same. You can press the Windows key + R and type this command to see for yourself:

cmd.exe  /c  net  user  "%UserName%"  & pause

You cannot change the name of the profile folder. If you did through some sneaky means then you would cripple Windows.

Windows 7 Stuck Checking for Updates - Microsoft Community

ron-ron arago say:

Windows 7 Stuck Checking for Updates

Windows 7 Stuck - Checking for Updates

 

 

1. Reset the Windows Update Components

 

-Hold Windows Key + R > type services.msc > hit Enter

 

-Look for these 4 components - Background Intelligence Transfer Services, Cryptographic Services, Windows Installer and Windows Update.

 

-Make sure that they are running, to check, right click on each > Properties > set it to Automatic (except Windows Installer) and click Stop then Start

 

 

2. Delete the Software Distribution Downloads Folder

 

-Local Drive C > Windows > Software Distribution > Download > Delete everything inside it.

 

 

3. Delete the Temp Files

 

-Hold Windows Key + R > type %temp% > hit Enter > Delete everything inside it.

 

 

4. Delete Program Temp Files

 

-Hold Windows Key + R > type temp > hit Enter > Delete everything inside it.

 

 

5. Temporarily turn off Antivirus and Windows Firewall and disable Windows Defender.

 

 

6. Open the Command Prompt Admin and enter this: ipconfig /flushdns (Leave it open).

 

 

7. Download

 

KB 3135445

32 BIT: https://www.microsoft.com/en-us/download/details.aspx?id=50793

64 BIT: https://www.microsoft.com/en-us/download/details.aspx?id=50797

 

KB 3102810

32 BIT: https://www.microsoft.com/en-us/download/details.aspx?id=49542

64 BIT: https://www.microsoft.com/en-us/download/details.aspx?id=49540

 

KB 3083710

32 BIT: https://www.microsoft.com/en-us/download/details.aspx?id=49172

64 BIT: https://www.microsoft.com/en-us/download/details.aspx?id=49174

 

KB 3020369

32 BIT: https://www.microsoft.com/en-us/download/details.aspx?id=46827

64 BIT: https://www.microsoft.com/en-us/download/details.aspx?id=46817

 

KB 3172605

32 BIT: https://www.microsoft.com/en-us/download/details.aspx?id=53335

64 BIT: https://www.microsoft.com/en-us/download/details.aspx?id=53332

 

 

8. On Command Prompt Admin, enter this code: net stop wuauserv– (DISABLE AUTOMATIC UPDATE)

 

9. Open the Downloaded KB (Update) and install it.

 

10. If the KB  is already installed, let us uninstall it.

 

- Open Command Prompt Admin, type this: wusa /uninstall /KB:Number and hit Enter. After that, go back to Step 7.

 

 

Note: Do this one at a time.

Note: Restart the PC and after that, check for updates again. Leave the device for an hour or two. This process will get all the updates on your PC
