This issue usually occur if the SD card is locked. You can try the methods below and check.

Method 1: Run the Hardware troubleshooter and check.

Follow the steps to run the troubleshooter:

a. Press Windows key + C on your keyboard to show the Charms bar.

b. Type Troubleshooting and click Troubleshooting under Settings.

c. Now type Hardware Troubleshooter in the search option.

d. Click on Hardware troubleshooter to run the troubleshooter.

If the issue persists, follow method 2.

Method 2: Unlock the memory card.
There is a Lock switch on the left side of the SD card. Make sure the Lock switch is slid up (unlock position). You will not be able to modify or delete the contents on the memory card if it is locked.


If the issue persists, follow method 3.


Method 3: Removing the write protection depends on the source of the problem.

The SD card has a write-protection tab on the side that has slid down. This can happen when the card is being inserted into the computer. Try removing the card, push the tab back into place and try inserting it into the computer again. Read the manual that came with the SD card for help.

If the issue persists, follow method 4.

Method 4:

 

a. Insert the memory stick into the computer

b. Go to My Computer and under Devices with Removable Storage look for your card reader

c. Right-click on it and click Properties

d. In the pop-up box, sometimes there's an option to Remove write-protection. Change the status of this option and try again.
