Hi,

 

If the Laptop came with the Windows Operating System preinstalled (Dell, HP, Acer, etc), that Product Key that came with the Laptop is for the preinstalled OEM License and is non-transferable.

 

The OEM License/Product Key (preinstalled Operating Systems on Dell, HP, etc machines) is tied to the first machine it is installed on and dies when that machine dies.

 

Only Retail Operating System License, that you buy and install yourself, are allowed to be transferred to another machine.

And Retail Licenses can also only be installed on one machine at any particular time.

 

Cheers.

takashi yano say:
I have encountered the same problem. After some struggles, I found a solution.
In my case, a third party MPEG2 decoder (ex. Dscaler5) was the culprit. By uninstalling Dscaler5, Windows DVD Player got working.
If you have installed some kind of MPEG2 decoder other than Microsoft product, uninstalling it may help.
