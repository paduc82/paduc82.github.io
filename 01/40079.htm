No Recording Control program in Windows 7 - Microsoft Community

powertracks user say:

No Recording Control program in Windows 7

Why doesn't Windows 7 have the Recording Control that XP had ?    How can Windows 7 replace the functionality of "What U Hear", and converting midi tracks to analog stereo ?    From what I can see so far, Window 7 is a giant step back from XP in that regard.   I read posts everywhere of others having the same problem, but their possible solutions have not worked for me.   Is there a real solutions out there ?
diana d say:

Hi Powertracks User,

Actually in Windows Vista as well as Windows 7, the WaveMix or SteroMix are disabled by default. 

If the sound card/system supports recording streaming audio Input Source selections such as: Stereo Mix, What U Hear, Wave Out Mix, Wave or a similar selection, here's what you need to do to Enable it for recording:

i. Right-Click on your Speaker Icon in your System Tray, (near your clock), select "Recording Devices".(You can also access these settings via Control Panel, Hardware and Sound, Sound).
ii. In the Sound window that opens, near the top, select the "Recording" TAB.Move your cursor down to where devices are shown.
iii. Right-Click in the blank white area and select "Show Disabled Devices".
iv. If supported, new selections will appear such as: Stereo Mix, What U Hear, etc. Right-Click on "Stereo Mix", "What U Hear" or similar and select "Enable".
v. Right-Click on the same item again and select "Set as Default Device". Click OK.
vi. Now in the recorder's Options in the "Input Source" box, you should have the "Stereo Mix", "What U Hear" or similar source available to select for recording streaming, etc. (You can also Enable other hidden selections like Microphone, Line In, etc. if they are not already shown).
vii. If the "Stereo Mix" type device did Not show up in the "Sound" window when right-clicking and selecting Show Disabled Devices, it may indicate that you have Integrated Sound, and recording streaming, etc. is Not supported. In that case (on a Desktop PC), installing a sound card into an available slot should solve the problem.

jeeped say:

You may need to install using Elevated Permissions.

Download the program to the desktop or insert your C/DVD. If an installation initiates, cancel it. Locate the software's installation program in Explorer. Right click the shortcut or installation program you are try to use (e.g. Setup.exe...?) and select Run as administrator. This will start the program as an Elevated Privileges Administrator which is required to perform higher level operation like registry writes and system .DLL replacement.

It may seem silly, but it is a valid security concern. In my opinion Microsoft chose this route because too many people were logging in routinely as an Administrator and naively inviting in viruses and malware through drive-by-downloading or phishing attempts. Once you've invited a bad program in, it assumes your own privilege level. By changing the local administrator level to something more akin to the old Power User account, less damage is possible.

This behavior is by design.
