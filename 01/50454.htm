a. user say:

... I am totally fed up with this attitude towards people like me who wish to be in control of our own destiny...

Windows 10 is not the best choice when you have limited bandwidth. That's because Windows 10 is software-as-a-service.

What that means, practically speaking, is that Windows 10 consists of an initial download followed by continuous updates. The updates are not optional 'extras,' they're part of the operating system. Updates to Windows will be downloaded and installed to your computer on a regular basis, and that's going to use up a limited bandwidth connection in a hurry.

Microsoft didn't invent software-as-a-service but Windows 10 is arguably the first version of Windows that works like that. Microsoft sometimes calls Windows 10 'Windows-as-a-service.'

There are ways that you can manage the use of your bandwidth - like setting your computer as a metered connection or deferring updates - but the same bits will be downloaded anyway, sooner or later.

You can influence your destiny, but you can't control it. You'll realize that in a hurry when you apply for a mortgage, if you haven't already, regardless of what operating system you're using.

allan mej say:

We're currently investigating what's actually causing this issue and as of the moment. In the past few weeks, we did out best to release patch notes to cover the holes made by Meltdown and Spectre:

  • Spectre is a vulnerability that affects all CPUs. It performs branch prediction, a branch prediction is the process of guessing which way a branch (i.e. an if-and-or-there-else program function) will go, which could be used by potential attackers to reveal secret operating system and hardware data.
  • Meltdown is a hardware vulnerability with allow a rouge process to read all memory, including the system memory. To put it simply, Meltdown "melts" the security boundaries of normal applications and kernel applications, allowing potential attackers/data thief/pirate to access hardware's and operating system's secrets. There is a twist, the thinner the boundary, the faster the PC speed gets in exchange of security. The tighter the boundary, the slower the PC gets.

It is possible that the patch released by Intel for your CPU driver or the patch that we released somehow slowed down your PC's performance. As of the moment, there are no reports in which an attacker used Spectre and Meltdown to do some illegal stuff but it is always nice to prevent something bad from happening.

We're doing our best to find what really is causing this issue and if we find that it is caused by the update from us, we'll do our best to fix it as soon as possible.
