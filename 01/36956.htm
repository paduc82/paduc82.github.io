considerate_guy say:

The latest versions of Movie Maker require you to start with a visual element on your storyboard/project area.  So start by adding a single image.  After you have that, the Add music button should appear and you can then pick the .mp3 file.

The next issue you may encounter is that your "movie" will stop when it gets done displaying the image which will default to 7 seconds.  To fix this, go to the Project tab and choose Fit to music.  That will extend your image(s) to the full length of the music.

Now go ahead and create your music segues.  At the end of editing, it sounds like you want an audio-only output.  In Movie Maker 2012, you just have to pick an audio-only output to create a .m4a or .wma file.

A couple things to note.  The newest version of Movie Maker doesn't do a good job of overlapping songs.  It really only lets you fade one song out (completely) and then fade in another song.  This is unlike the classic versions of Movie Maker where you could overlap and get a cross fade.  If this is needed for your project, you might want to load Windows Movie Maker 6.0 (the last classic version).

If you are just creating music files,  you may want to use a dedicated audio-only editor like Audacity, rather than trying to use a movie editor.

1.  Open an Elevated Command Prompt (click Start, type Command Prompt, right-click on the link and select "Run as administrator," and respond to the UAC prompt.  Type the following in the Command Prompt window and press Enter

     regsvr32 /i shell32.dll

  • In HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Advanced\

        HKEY_CURRENT_USER\Software\Microsoft\Windows\Shell\BagMRU
        HKEY_CURRENT_USER\Software\Microsoft\Windows\Shell\Bags

Try this workaround for problems where apps are not starting or are stuck at the splash screen:

  1. Open an Elevated Powershell prompt by pressing the Windows Key, typing Powershell, then pressing Ctrl+Shift+Enter
