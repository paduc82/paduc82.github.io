Using System Image - Microsoft Community

vgatesr say:

Using System Image

You can create a "system image" on the Surface Pro 3 but there seems to be no practical way of using it. What would seem to make sense is to save it to a SD chip then boot from a USB recovery flash drive. However I cannot find a way to save the "image" to my 64gb SD chip. A good system image seems to be a much more practical recovery method since you don't loose your apps.

Can someone please tell me how I can use an image? 

stevenpugliese say:

An image file can be used to restore your entire computer to the earlier state when you made the file. You create the file by going to File History in Control Panel. On the bottom left of that window you will see System Image Backup.

Before you go there, however, you need a place where Windows can create the file. It either has to be an external USB hard drive or an SD card that is formatted in NTFS mode. You cannot save the file to the C:\ drive.

Once created, you can use that file to restore your computer. It works in such a way that when you install the image file (either through Update and Recovery in the Change PC Settings of the Settings charm, or by booting from an external hard drive), it replaces the entire computer (the entire hard drive, actually) with the way the computer was when you made the image file. It restores the entire computer to the way it was, so that means if you added new files in the time between when you made the image file and when you restored it, they will be lost in the restoration. If you can, copy these files to an external hard drive before you perform a system restore. It's best to create image files often. It doesn't take that long and you can do other things while the file is being created (It's a very large file).

You use image file restoring to recover from disasters, either a hard drive crash or a virus takeover or other malware. With a hard drive crash, you will have to boot from an external CD drive which you may need to alter in the bios in order to get the computer to boot, but that's another story.

When you create the image file, Windows will look for an appropriate drive to create the image on. If it cannot find one, it will tell you.
