lemp say:

That's what I was trying to say to you:  the file for Windows Photo Viewer is a DLL file and not an EXE file.  It needs a specific command to run:  

%SystemRoot%\System32\rundll32.exe "%ProgramFiles%\Windows Photo Viewer\PhotoViewer.dll", ImageView_Fullscreen %1

where the last parameter -- %1 -- is the full path to the image file.

Rather than having to use that cumbersome command (from a Command Prompt window), you should be able to get WPV back by right-clicking on an image file and selecting WPV from the list of suggested applications.
