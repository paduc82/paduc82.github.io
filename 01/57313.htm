Hi,

It might be due to corrupt system files. To check, do the following:

  1. Right click the Start button and click Command Prompt (admin).
  2. Type (or copy/paste): sfc /scannow and hit Enter.

SFC is an acronym for System File Checker and the tool will check the integrity of Windows system files for corruption. If any are found, it will attempt to repair the file using the DLL repository.

Sometimes though, SFC cannot repair files and you'll see a message to the effect that file corruption was found, but Windows resource protection was unable to fix some of them.

In that event, use the same command prompt to copy and then right click and paste the following command and hit Enter:

DISM.exe/Online /Cleanup-image /Restorehealth

When DISM runs, you won't see much happening in screen and for about five minutes, it'll just show a flashing cursor. But eventually, the figure of 20% will appear followed about 15 mins later by 100%. You'll also get a report on whether the tool was successful or not.

Solution :-

In case if you have changed hard drives and then trying to restore. It happened with me last night and spending over 2 hours I am not sure if I was lucky or this is really the way to resolve this issue. 

1. Boot OS 

2. Click on INSTALL , once you have seen your new/empty drive.

3. Click NEW, and press APPLY.

4. You must receive a prompt for creating system partition automatically, press yes.

5. Once the partition is created, don't go further. Cancel the process, go back and then click on REPAIR YOUR COMPUTER, use SYSTEM IMAGE RESTORE and now try to restore should work without any issues.
