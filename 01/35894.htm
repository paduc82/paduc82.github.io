slance310 say:

Hi,

It sounds like you are reinstalling to a machine with UEFI Bios...

One big difference between standard Bios and UEFI Bios is that UEFI Bios requires a GPT type disk...

In order to install windows on to a UEFI machine the installation must be done from a specific type of bootable Flash Drive... a DVD might be able to work, but would require a lot of changes to your machines settings...

If you want to install using a DVD you need to DELETE the disk and then INITIALIZE it as a MBR instead of GPT... But you may get an error that way as well, again, because of the UEFI Bios...

The easiest thing to do would be to get a ISO Image of Win 8.1 Pro and put it on a bootable Flash Drive...

You will need a tool called RUFUS to create the Flash Drive of at least 4gb, and preferably 8gb... The Flash Drive needs to be Formatted as FAT32, and not NTFS...

This is a requirement for the UEFI Bios...

You MIGHT be able to use your DVD to create the Flash Drive with RUFUS, but I have never tried or even checked...

Hi,

With UEFI Bios you can't use the Windows tools for creating a bootable Flash Drive...
