Hi.

To import a calendar you can go to the Calendar on Outlook.com and then select Import to import ics calendar files.

To export your calendar for backup purposes is a bit complicated, but still possible.

1) Go to the Calendar on Outlook.com and then select Share.

2) Then select Get a link.

3) Then click Create.

4) Then copy the link that starts with webcals://. Paste that link into a browser address bar, change the beginning of the URL from webcals:// to https:// then hit enter. The browser will then prompt you to download the ics file and save it.

5) After you've saved it on your PC, go back to Share, find the sharing link that you have created and remove permissions again so that the calendar isn't shared anymore.

It is not just a case of how big the Windows 7 installation is.  Your applications will also take up room.  As you refer to an SSD, I assume you will have a separate drive for your data.
