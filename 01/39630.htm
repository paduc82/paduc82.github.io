Is this your first installation of FSX?   Sometimes, when many re-installs are done, then it can no longer be activated.   We used to be able to get another activation code by ringing a Microsoft number, but I don't think that facility is available now since FSX is no longer supported. 

The other thing you can try is the following.   However, be careful when moving about the System Registry as any incorrect or deleted entries could render that program unusable.   As well, the sub-key shown below is given in lower case - yours maybe in upper case:  it's the same thing.

1. Close FSX and run Registry Editor (press Windows key + R and type   regedit    and     Ok)

2. Expand the following registry key:            

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Class\{4d36e967-e325-11ce-bfc1-08002be10318}      

3. After you find the folder, double-click Upperfilters on the right side.

4. Delete any value there and type in   PartMgr  (note, this is case sensitive;  type it as shown here) and click OK.  

5. Close Registry Editor and reboot Computer.

6. Start FSX and check whether the Activation box comes up.   Hope this helps you.
