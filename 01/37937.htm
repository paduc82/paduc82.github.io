Windows 10 is updated at least once a month. ¡firedog say:

OK, thank you. This shouldn't have happened! If the program opens, then it's still installed. But it sounds as if your store folder and other program data didn't survive the upgrade. They may still be there, squirrelled away in the folder called C:\Windows.old. So long as it's not more than ten days since this happened, you may be able to recover these data. Can you locate that folder? If so, close down running programs including Windows Live Mail and then look inside it for a folder called

Users\{your Windows username}\AppData\Local\Microsoft\Windows Live Mail

If it's there, copy it to the current location, i.e.

C:\Users\{your Windows username}\AppData\Local\Microsoft

replacing the one that's probably already there.

Similarly, look in Windows.old for

Users\{your Windows username}\AppData\Local\Microsoft\Windows Live\Contacts

and copy it to

C:\Users\{your Windows username}\AppData\Local\Microsoft\Windows Live

replacing the one that's probably already there.

Please let us know what happens.

Hello,

As per the description, I can understand that you have issues with the above mentioned error message.

Actually, the file system structure on the volume listed in the message might be corrupt because of one or more of the following reasons:

  • The disk might have bad sectors.
  • I/O requests issued by the file system to the disk subsystem might not have been completed successfully.

So, I would suggest you to perform check disk and see if that helps to fix the issue.

Run check disk to find and fix errors on the drive.

a) Click Start, type cmd, right click on cmd and click Run as Administrator.

b) On the prompt, type chkdsk /f /r and hit enter.

c) Try to check the issue after restart.

For more information on "Check a drive for errors" check this link:

Check disk (Chkdsk) runs when you start Windows 8 or Windows 7

Important: While performing chkdsk on the hard drive if any bad sectors are found on the hard drive when chkdsk tries to repair that sector if any data available on that might be lost.
