robert aldwinckle on forums say:

receive a message saying the update failed as I have insufficient space

That's not just a message but a recovery procedure which allows you to specify an attached drive to be used for temporary storage by the update.   An alternative would be deleting stuff to make room.  For example, I was able to use the Disk Cleanup procedure to do this.  The biggest source for more storage was Windows.old which would be needed to do a recovery if the install still failed for another reason, so it would be a good idea to either backup stuff that you are going to delete or make sure that you properly attach another volume.  Unfortunately it can't be just a Vdisk which is what I wanted because I didn't want to risk pointing it at my USB backup drive (e.g. in case the whole thing would be reformatted) and I didn't want to bother having to create a real partition on it.
