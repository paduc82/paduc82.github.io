If the account is a local account, you first type in a password incorrectly then, when it tells you it's wrong, click OK to tell it you want to try again.  You will now see a link to Reset password underneath the password entry box so click on that.  That starts the Password reset wizard which guides you through the process - connect the PRD USB then tell it which drive it's on then tell it the new password you want to use in place of the old one [the PRD is used to set up a new password not recover the old one].

Do note that the only PRD that will work is the latest one created for that user account on that computer [all previous PRDs are rendered invalid by the act of creating a new one].

There is an illustrated guide for Windows 7 at RM2  Password reset disk and, in pdf format, at Password reset disk and the only difference is the appearance of the sign-in screen.

In addition, if a password is rejected because the user account profile is corrupted then a PRD will not help so make sure you have also created two additional password-protected Admin-level local user accounts in case your normal user account profile ever becomes corrupted. 

For Windows 10, if the user account is an MS Account, you don't use the PRD but instead you go to https://account.live.com/resetpassword.aspx
