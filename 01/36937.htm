greg carmack - mvp 2010-2020 say:

You are correct that with MBR disks you can create a Logical Extended partition then create as many sub-partitions as you want to install as many other OS's as you want, however the boot volume must be Primary.  So for example if you have Windows 7 System Reserved as boot volume it would be the only partition that would need to be Primary (although you can still have up to 3 primary and one Logical Extended, or a total of four primary).  Partition Extended Logical Drives - Windows 7 Forums

You will also need to use MBR to install 32 bit to UEFI. 

You are testing the limits of multi-boot although theoretically you should be able to install them all but would only expect them to auto-configure the Boot menu in the OS's since Vista.  The others you can add using EasyBCD (click Download - no Name or Email required) specifying the System Active partition by letter as the boot partition.  If you need advanced help ask in the NeoSmart forums where my colleague Mahmoud is the top expert on multi-booting in the world. 

It's confusing because the label Boot only means the OS which is presently booted - it's Active which points to the intended System partition and the "System" label which confirms it when booted. 

There are some anomalies where you may lose System Restore points in a multi-boot but otherwise the OS's should act independently even while booted by the same System partition. You can of course make them all dead to each other by using separate hard drives booted via the BIOS Boot Menu Key.

  Do not try to install common programs for myriad reasons, but you can share files and even move each OS's User folders to the data partition, although each will then want to include the data partition as a System partition if you use built-in imaging on 7 and later.  Solution is to use a more flexible imaging app like excellent Macrium Reflect which can handle anything and is fully automated (and free!).

Feel free to ask back any questions and let us know how it goes.

davem121 say:
Hi Ian

See if Microsoft Support can help you . . .

If not, then yes, reformatting and clean installing Windows 10 will give you the laptop back . . .

You can reinstall Windows 10 at any time and it will not cost you anything !

Since you previously had Windows 10 installed and activated on that PC during the installation process skip the steps which ask for a product key and select the option 'I am Reinstalling Windows 10 on this PC', and activation will not be an issue, your PC will have a digital entitlement stored on the Microsoft Activation Servers

Make sure you install the same Edition of Windows 10 (Home, Pro . . . etc.) you previously had installed to avoid Activation issues

Click this link:
https://www.microsoft.com/en-us/software-downlo...
to download the Media Creation Tool (Click on Download Tool Now), with that you can download the latest Windows 10 ISO (Select Create Installation Media for Another PC), you can create a bootable USB flash drive (min 8GB) using that tool

Then, Boot your PC from the Installation Media you just created (change Boot Order in your BIOS) to begin installing Windows 10

If you have problems booting from a boot disc, you may have UEFI BIOS:
Insert the Bootable Installation Media, then go into your BIOS and make the following changes:
1. Disable Secure Boot
2. Enable Legacy Boot
3. If Available enable CSM
4. If Required enable USB Boot
5. Move the device with the bootable disc to the top of the boot order
6. Save BIOS changes, restart your System and it should boot from the Installation Media

camera/phone/etc was in when the video was recorded.

when importing and Photo Gallery cannot rotate video.
