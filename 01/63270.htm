¡firedog say:

I am using Live Mail to synchronise with my Zoho Mail account, through IMAP ...

It sounds like you haven't mapped the Zoho folders to Windows Live Mail's native ones, so you might need to review all the settings. Start by selecting the folders you want to see in Windows Live Mail at Zoho. 
 

Here are some suggestions for local settings - you may find on experimentation that different settings work better:
 

  1. Right-click on the account name in the folder pane and select Show or hide folders to open the Show/hide IMAP folders dialogue. Click Reset List to refresh the folder list, then use the buttons to show only the folders you want to see in the folder pane. Note the exact names of the 'special' folders.
     

  2. Right-click on the account name in the folder pane and select     Properties.
    On the IMAP tab, select     Store special folders on IMAP server,  then enter the server's names for the special folders in the boxes. If the folder for junk email at the server is called Spam, then enter Spam (not spam or SPAM) into the box alongside Junk email path and so on. You may have to provide a Root folder path to make this work properly.
     
  3. Make Deleted items folders work the same way for all accounts
    When viewing your message list, press Ctrl-Shift-O for Options.
    On the     Advanced tab, select Use the 'Deleted Items' folder for IMAP accounts.
     
  4. Avoid duplication of sent items
    Messages you send will be automatically saved in the Sent folder at the server. There is thus no need to have Windows Live Mail save them as well.     When viewing your message list, press Ctrl-Shift-O for Options.  On the Send tab, deselect Save a copy of sent messages in the 'Sent Items' folder. Note that this applies to all mail accounts, so be aware of the consequences for any other mail accounts you may have.

¡firedog say:

As initial trouble shooting, kindly remove and re-add your account in Windows Live Mail.

Removing the account and adding it back with the same settings as before will not change anything; all it will have done is permanently delete all the account folders and their contents and cause all messages in the account's folders to be downloaded again. That isn't troubleshooting - it's a major, and possibly expensive, exercise in futility.

If there are account settings that need changing, then that's all that has to be done. There is no need to remove and re-add the account to achieve this.
