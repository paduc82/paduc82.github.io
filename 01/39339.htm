  1. Type Sound in the search box. Select Sound from the list of results.

Update your Surface Dock if disabling the audio enchantments didn't work. Click here on how you can update the dock.

a. user say:


That's not what the error message really says, is it?

You don't have an XP problem, you have a Norton WinDoctor problem.

These files usually fall into the area of influence by the Windows XP implementation of Side by Side (SxS) technology.

Usually, DLL files would be found in the c:\windows\system32 folder, but sometimes there is a need (depending on the applications
installed) to have two different DLLs with the same name on one computer at the same time (this would be side by side).  Ouch!   How are you going to do that?

Since the names of the DLL files are the same they can't both be in c:\windows\system32 (this is called DLL H*ll) and you might not be
able to just overwrite one, so XP can keep the odd one in another folder:  c:\windows\WinSxS and find and use the correct DLL there when it needs it.

The mix up is usually from something that has recently been installed, uninstalled or updated (like the .NET Framework for example).

You have some options:

Some people will "solve" the problem by searching their system for the files Norton is complaining about and copy them (usualy from somewhere in the c:\windows\WinSxS folder) into the c:\windows\system32 and think that is a solution.  This is really a silly idea but it counts as a "solution" for some people.

You could ignore the errors.  In some circles, this is also known as a "solution".

You could configure Norton WinDoctor to ignore these errors (this is advice from Norton - but then why use it at all?)

You could not use Norton WinDoctor since they acknowledge the issue on their WWW site (at least in the older versions of WinDoctor).

You could see if the latest version of Norton WinDoctor will resolve the situation  (you neglected to specify your version).  At some point after the 2006 and 2007 versions Norton thinks they fixed this problem.  If you have a valid Norton WinDoctor subscription, updates are free.  I don't use Norton, so I don't know if the new version is better or not.

You could ditch Norton all together and replace it with something a little smarter.

Here is the Norton support URL if you want to stick with them:

http://www.symantec.com/norton/support/index.jsp
