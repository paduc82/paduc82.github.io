If you have not already - Start a thread on the AMD forums -

AMD Game Forums - AMD Graphics

With laptops , both AMD, and Intel  graphics (Nvidia too), drivers should be downloaded from

the mfg's site.

This is because many laptops have 'custom' built graphics that require their own modified driver version.

- Never use Windows Updates for graphics drivers on a gaming machine (laptop or PC).

(The graphics driver updates are found in the 'optional' updates section and should be disallowed.

Most 'Optional' updates should be installed only if you know you need them).

Unfortunately, laptop mfg's  very often provide only limited driver support, even with high end models, and when a laptop does not work with drivers downloaded from AMD you are very often stuck with

the latest the laptop mfg provides.

(Laptops are several steps above PC on the built in obsolescence, future e-waste scale).

Quote from the AMD download page -

" If you experience difficulty while using the AMD Catalyst Mobility driver(s), we recommend reverting to the driver provided by your notebook vendor. Please contact your notebook vendor for their most recent graphics drivers."

- However, with a very few exceptions, games shouldn't be incompatible due to not having the

most recent graphics driver.

Which games are you suspecting are incompatible ?

Various games do get optimisations in the drivers. These may provide some extra fps with AA, or other

settings, and in some games they may advertise 15% improvement overall (Not always true, and more often with newer series card). 

In some cases a driver update can fix a serious bug in a game, but most bugs are in the game, so waiting for a patch for the game is more often the fix for a poor running game.

(Assassins Creed Unity, BF3/4, FC3/4... the list is long)

But in none of the above should using an older driver make a game incompatible.

You can find out just what each driver does improve on in the Release Notes found on the download page. (Mostly Xfire and CAP updates. Similarly Nvidia's driver updates are more about SLI and the GeForce 'experience' (Nvidia's version of CAP files).

Actually, if a driver,  or using too powerful a gpu, leads to incompatibility it's far more likely to be

with older games, eg.- I have several games (older, or low spec games) that run on my GTX560ti that

I have to use Intel HD4000 graphics with on my machine running a GTX770.

If the Certificate of Authenticity (COA) sticker is damaged or letters or not legible, you will have to consider the following options:

  • Use a digital camera to capture a picture of the COA then enlarge it on your computer.

  • If your recovery partition is not available or damaged, you should contact the manufacturer of your computer and request a recovery disc set you can use to reinstall Windows 7. They might charge a small shipping and handling fee. This will exempt the need to use the product key also.

Buy Windows 10 Home/Pro

don cuthbert say:

X: drive is a RAM disk, created during Windows install, or if you boot from Windows install media. So it is not the same as your HDD Windows partition and will vanish if you restart before checking the log file referred to. It has a similar folder structure to the usual C: drive, as it is running Windows PE, or Windows RE.

There are some changes that may be a factor between build 15063 and 17134. You might consider upgrading your HDD Windows to 17134 or wait a week or more and then upgrade both installs to version 1809 of Windows 10 (once they resolve the current issues with that version and it becomes available again).

If there are any corrupt files on your system partition, you should be able to fix these by running:

bcdboot C:\Windows /v (/v just shows you some details while it runs)

and

bootsect /nt60 SYS /force

I would do this while running Windows from the 17134 SSD, so that you are using a more recent version of Windows files.

Also, it may be worth checking if there is another BIOS update from HP, although your BIOS date is very recent.

don cuthbert say: If there are any corrupt files on either of your Windows partitions, you could try fixing it by running sfc /scannow from an Administrator Command Prompt.  If errors are found but not repaired by sfc, there is a DISM command that might fix things.

There are two ways to change rip settings while in the Player Library. You can insert a CD into your CD drive and click Rip settings, or you can click the Rip Music tab in the Options dialog box. The Rip Music tab in the Options dialog box offers additional options that aren't available on the Rip settings menu.

To rip music from a CD, do the following:

1.      For best results, make sure that you are connected to the Internet before you begin.

When your computer is connected to the Internet, the Player attempts to retrieve media information about the tracks being ripped from an online database. The Player then adds the information to the files during ripping.

Note that you can rip a CD when your computer isn't connected to the Internet. However, the Player won't be able to identify the name of the CD, the artist who created it, or the names of the songs until you go online.

2.      Insert an audio CD into the CD drive.

3.      (Optional) If you want to select a different format (for example, MP3 instead of WMA) or bit rate (for example, 192 Kbps instead of 128 Kbps) for the files created during ripping, in the Player Library, click the Rip settings menu, and then click Format or Audio Quality.

4.      (Optional) If you don't want to rip every song, in the Player Library, clear the check boxes next to the songs you don't want to rip.

5.      Click the Rip CD button in Now Playing mode or Rip CD in the Player Library to begin ripping.
