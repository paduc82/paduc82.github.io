    ralf_g say:

    SSDs can have bad sectors. Up to a certain quantity of them is presumably allowed for by extra hidden capacity on the drive that automatically/silently replaces the damaged bits as needed.

    I'm not sure how many ways an SSD can fail but an intermittent mis-write/read might not be detected during a single pass disk check. Fast Startup writes a large virtual memory file onto the diskdrive whenever you shut down the computer and then subsequently reloads that saved data the next time you boot up.  Any glitched bit could show up as a harddrive or memory error at either stage, or not at all if it only happens intermittently.

    Hi Stella,

    This only works if you had done an in place upgrade from Windows 8 or 7.

    Follow these steps : How to rollback to your previous version of Windows from Windows 10.

    1.    Start by opening Settings – Click Start > Settings or type Settings in the search bar and click it or press Windows key + I.

    2.   Click Update and Security

    3.   Click Recovery then click Get started under Go back to Windows 7 /8 (depending on the previous version you running)

    4.   Here you can make multiple selections why are going back to your previous version of Windows along with additional details or you can choose just one and click Next

    5.   As noted, any configurations made to Windows 10 since upgrading will be lost, so if you installed new programs or hardware drivers or made personal settings, those will be deleted.

    6.   Make sure you have your password ready to sign into your old version if you had been using one. Click Next

    7.   The process can take a while depending on the amount data, applications and settings. Click Next

    8.   Click Go Back to begin the process.

    9.   Windows 10 will Restart

    This will be your screen throughout the recovery process.

    When the process is complete, you will be booted into your previous version of Windows environment.
