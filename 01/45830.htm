kosh vorlon say:

Hi TarekAl,

 

This is NOT elegant, but it may work for what you want to do (and with anything you want to print - not just .xps files).

 

Try this (if supported by your OS which you still didn't identify):

 

Open printers.  Click to Add a Local Printer.

 

Click Use an Existing Port and from the dropmenu, choose FILE: (print to file).

 

Under Manufacturer, choose Generic and under Printers, choose Generic/Text Only.

 

Now go to what you want to print and don't use the default printer but instead choose File / Print and select this new printer.  It will ask where you want it to print (which in this case means where do you want to create the file).  Choose the full path to your desktop along with a filename followed by a .txt extension (like C:\users\<username>\desktop\filename.txt).  When done nothing will print, but a .txt file of the print output named as you selected should exist on your desktop.

 

Now, since the result is a simple text file, this new file should have no trouble printing as text only on the real dot matrix printer.

 

No, it's not pretty - but it may be worth a try to see if it works and might be a workaround until a better solution can be found.  For that, I'd contact the manufacturer of the dot matrix printer (which you also did not specify) to see if they have a way to configure the printer or adjust the printer properties (or have a special driver) to accomplish your objective in one step instead of two.

 

Good luck!

 

Kosh
