1) Plug the problematic USB flash memory stick in your computer

2) Go to Control panel

3) Go to Device Manager

4) Click on Portable devices

5) You should find there an "unknown device". It should be the USB flash memory that can't be recognized by your computer.

7) Click on "uninstall". Your computer should uninstall the USB memory stick and then automatically re-install it. After reinstallation, the USB memory stick should be visible on your computer.

8) (Optional)  You may see a pop-up notification asking you to fix and scan the USB memory drive (i.e. the stick). You should do it if the memory drive is empty. Otherwise, you should first backup the data which is stored on that drive and then allow the computer to do fixing and scanning.
