 OK, for your question about the 200 GB vs 100 GB discrepancy, in the latter case there were hidden files that weren't included when you did the CTRL-A.  This would include things like the NTFS file mapping tables, your System Restore points, paging file, possible hidden system folders, etc.  So believe the 200 GB figure.

 

 Now the question about restoring Library files.  When you have things listed under Libraries, they are only LINKS to where the data really exists on the C: drive, so when you delete from Libraries, you're only deleting the link, not the data itself which is still on your C: drive under C:\Users somewhere.  This is something MS came up with in Windows 7 to help keep people from fiddling around in the C: drive itself.  It took me and the folks I work with a while to understand this new wrinkle, but when you think about it, if's probably a good idea.  That's especially true from a support person's perspetive who doesn't want to be hassled by problems created by users fiddling around in places that cause problems.  :-)

  1. Login to the router by entering the default gateway IP address.
