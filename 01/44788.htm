c)       Right click on the Command Prompt and you will see the pop up window at the bottom of the screen select the options Run as administer.

e)      Reboot the computer.

 

Now on the start screen you will see the Administrator User profile click on it and create a new user account and transfer all files from corrupted profile to New User Account.

 

Refer to the link to create a user account,

 

Create a user account

 

http://windows.microsoft.com/en-IN/windows-8/create-user-account

 

Transfer old files to new user account profile:

 

a)      Navigate to C:\Users\Corrupted_Profile_Name\. The profile name will be the corrupted profile name.

b)      Copy all the files.

c)       Open Control panel and Folder View options uncheck Hide protected operating system files.

d)      Navigate to C:\Users\Corrupted_Profile_Name and copy Appdata folder.

Note: Do not copy these three files. NtUser.dat, NtUser.ini, NtUser.log.

e)      Navigate to C:\Users\New_Profile_Name\.

f)       Paste all those files here.

g)      Reboot the computer

 

Once you are done with the transferring files disable the default Administrator again

 

To disable default Administrator account and check,

 

a)      Press Windows Key + Q on the keyboard.

b)      Type in Command Prompt and on the left hand side you will see the Command Prompt.

c)       Right click on the Command Prompt and you will see the pop up window at the bottom of the screen select the options Run as administer.

d)      Type in “Net user administrator /active:no” and click on Enter key.

e)      Reboot the computer.

If when you installed whatever operating system you now have you used the entire hard drive, then you will have erased the hidden partition that permits the reinstallation of the factory image.

Use a tool provided by your operating system to show all partitions on the hard drive.  If there is a relatively small partition in addition to the partition where the operating system is located, you may have a possibility of recovering the factory image.  If possible, it probably will not be easy.  See the "Answer" here with respect to Linux --> http://superuser.com/questions/34116/will-installing-linux-wipe-my-restore-partition

If you deleted the restore partition when you installed your current operating system, your only recourse is to contact the Lenovo Support Center as I suggested in my first response to you.  Given that Lenovo would have stopped selling computers with Windows XP shortly after October 2010, my guess is that they will no longer be willing to send you XP recovery media ... but it's worth a try.

Lenovo Support phone list --> https://support.lenovo.com/us/en/supportphonelist?linkTrack=Contact%20Us_step2:link9_Support%20Phone%20List

It's also possible that you will see offers to sell Windows XP from various online vendors.  I would be very cautious in buying anything this way.  You would also need to download all the drivers from http://support.lenovo.com/us/en/products/laptops-and-netbooks/thinkpad-x-series-laptops/thinkpad-x40

and install them.
