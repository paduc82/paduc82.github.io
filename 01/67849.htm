Call the Windows 7 Remote Desktop [RDC] host PC by its pubic IP if connected directly to the public internet or the public IP of any router its behind. With that said you may find it easier to call using a fully qualified domain name [FQDN]. You can get a free for personal use FQDN from No-IP.com or similar service. The servers then know your current public IP and map that to your FQDN and propagate the information across the public internet. You call the FQDN.

Also remember that TCP Port 3389 (or whatever you custom seletct via a registry hack) must be open/forwarded on any router the RDC host PC is behind. You can test port forwarding from the RDC host PC by going to the http://www.canyouseeme.org site.

    How to change default 'Organization' used when installing - Microsoft Community

    keith- say:

    How to change default 'Organization' used when installing software

     

    From: lllusion

    On a recently purchased a pre-configured laptop, when installing software the "Organization" field is already filled in with the company name from whom the laptop was purchased.

    How does one change the default organization for the computer so that it reflects my preference when new software is installed?

    keith- say:  

    From: Andrew McLaren

    I don't think there is any way to change this, through the normal user interface.

    The Organization value is stored in the Registry, under HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion

    The value is called "RegisteredOrganization".

    Run regedit as administrator, navigate to the RegisteredOrganization value, and double-click to modify it. Change it to your desired value. Click OK and exit Regedit. When you reboot, the new value will be displayed. You might want to change the RegisteredOwner value at the same time.

    Hope it helps,
