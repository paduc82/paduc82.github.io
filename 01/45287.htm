Click in the Select edition list box and choose your edition, Windows 10 contains both installation files for Home and Professional allowing you to select either edition during setup. Follow the instructions to use either the

Step 2:

Prepare Windows 10 ISO file

After you have downloaded the Windows 10 ISO, you need prepare it, this is especially important for Windows 7 users..

After obtaining the .iso file you can use the built in Disc Image burning tool or Microsoft .iso to USB/DVD tool to create a bootable DVD or USB (requires a blank DVD or USB flash stick of at least 4 GB).

After downloading the .ISO file, right click it and click Burn disc image

Insert a blank DVD

Click Burn

Windows 8/8.1 users can mount the .ISO file and start the upgrade automatically. Learn more here

Important:

If you are creating a bootable USB thumb drive, make sure the thumb drive is first formatted as FAT32.

Connect the thumb drive

Open Computer

Right click thumbdrive

Click Format

Make sure FAT32 is selected as the file system.

For UEFI based systems

If your computer is UEFI based, these are normally systems that come pre-loaded with Windows 8 or later, you will need to prepare the ISO file for such a configuration or you will receive an error message during setup. The thumbdrive needs to be formatted as FAT32 and use the GPT partitioning scheme. To do this, you need to use Rufus, a small tool you can download for free.

 

After you have installed Rufus:

  • Launch it
  • Select ISO Image
  • Point to the Windows 10 ISO file
  • Check off Create a bootable disk using
  • Select GPT partitioning for EUFI firmware as the Partition scheme
  • Choose FAT32 NOT NTFS as the File system
  • Make sure your USB thumbdrive in the Device list box
  • Click Start
  • Close when complete

lemp say:

The cable you use to charge your iPad should work. 

The same cable is used for charging and synchronizing the iPad with the computer.  The connector between the cable and the part that plugs into the wall outlet is an USB connector. 

The only problem will be if your computer has the new USB-C ports and your iPad charger uses the older "standard" USB connector -- or vice versa.

Disconnect the cable from the charger and look at both the end of the cable and the port on the charger.  The cable end should look like one of the 3 types shown in the circle:

Now look at your computer.  It will have one or more ports that look like one (or both) of these:

If the end of the cable matches the port on your computer, you're good to go.  The only difference between USB 2.0 Type A and USB 3.0 Type A is the color of the plastic part:  USB 3.0 is blue; USB 2.0 is black or white.  It doesn't matter whether the port is USB 3.0 and the cable USB 2.0 or vice versa. 

The only situations in which you would have to buy a new cable are (a) your cable has a USB-C plug and your computer has only Type A USB 2.0 or USB 3.0 ports OR (b) your cable has a Type A USB 2.0 or USB 3.0 plug and your computer has only USB-C ports, you'll have to buy a new cable. 

This is the charger that came with my fairly old iPad.  The wall plug flips up when not in use, and can be replaced with a different wall plug for some non-U.S.A. countries.  This charger uses a USB 2.0 Type A port:
