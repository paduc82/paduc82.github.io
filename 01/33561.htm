Hi,

 

Thanks for posting on Microsoft Community. We will assist you in resolving the issue.

 

From the above description, I understand you want to correct a misspelled user account name on Windows 10 PC.

 

Before assisting further, are you editing Microsoft user account name or Local user account name?

 

You will not be able to edit/rename Microsoft user account name on the computer. If you are using a Local user account, I suggest you to follow the steps below and check if the issue is resolved.

 

  1. Press Windows logo + X keys on the keyboard and select Control panel from the context menu.
  2. From the top right corner of the window, click on View by and select Large icons.
  3. From the window, click on User accounts and select Manage another account.
  4. From the window, select your user account (the one to edit/rename) and click on Change the account user name.
  5. Type correct user name in the box and click on Apply and OK.
  6. Close the window, lock the PC (press Windows logo + L keys on the keyboard) and check if the changes are effective.

 

The above changes made will reflect only on the Login/lock screen. To change the name of the user folder in C: drive, follow the steps below:

 

  1. Press Windows logo + E keys on the keyboard to open File Explorer.
  2. Click on This PC from the left side panel of the window and open C: drive.
  3. Open Users folder and select the user folder with a misspelled user name.
  4. Right click on the misspelled user name and select Rename option from the context menu to rename the folder.
  5. Close the window and check if the changes are effective.

Hi,

Where you are "stuck" is critical to the right answer. If you are truly stuck at a BIOS password prompt, then this is easily resolved by removing the CMOS battery or shorting a jumper on the motherboard. This would be a password prompt for boot-up that you get almost immediately after the system is powered on and before Windows even begins to load.

If you at a Windows User password prompt, then you can either do a wipe and reload (which is recommended if you bought a used machine, why would you want someone else's junk?), or boot to Safe mode by hitting F8 as soon as Windows begins to load and log into the default administrator account (the password is blank by default). Once there, you can reset your user account password from the Control Panel/Users applet.

As an additional point of information, these forums do not allow for anyone to post hacks and other means of bypassing passwords, so please do not ask for them.
