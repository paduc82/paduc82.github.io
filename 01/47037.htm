MAC Filtering

  1. Connect a computer to the gateway with an Ethernet cable.
  2. Open a web browser on a computer.
  3. Type the gateway address: 192.168.0.1 into the web address bar, select apply.
  4. This will take you to a login page, type admin for the username and password (located on the bottom of the device).
  5. Select Wireless Setup on the main page.
  6. On the left hand side select MAC Authentication.
  7. In the ‚ÄúSSID‚Äù drop down select the desired Wireless network name to enable MAC filtering.
  8. Alternatively you can select ALL SSID‚Äôs to enable MAC filtering on all wireless networks from this device.
  9. Next to ‚ÄúMAC Authentication‚Äù select Enable.
  10. Select Allow Device List.
  11. Select the device you would like to add next to ‚ÄúSelect Device Name‚Äù. Or manually enter in the MAC address.
  12. Select the Apply button to save changes.

 

Disable MAC Filtering
