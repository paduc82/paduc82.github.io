    •    Type cmd and hit Enter. In the command prompt type the command:

    Syntax:
    Subst (invalid drive letter): (existing drive letter):\
    Example:
    Subst I: C:\

    In your case, you can type Subst G: C:\

    Once you’ve done this, inspect the following keys in the registry editor to check if they are pointing to the correct drive (should be C: drive). If it does not point to the correct drive you may right click and modify it to the correct drive letter.

    Click on Start; in the start search type regedit and hit Enter. Browse through the following registry keys in the registry editor.
     
    HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\User Shell Folders

    HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\Shell Folders

    HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion

    HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\Shell Folders

    HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\User Shell Folders

    Make sure that directory paths on the right side are on a correct/existing drive. If it is using an invalid drive right click and modify the directory path by changing the drive letter to a correct one.
