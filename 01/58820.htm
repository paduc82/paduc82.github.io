lemp say:

When you say that the "problem persists even when the troubleshooter fixes the problem," do you mean that the issue is temporarily fixed after setup is run but then the problem returns sometime thereafter?

Generally this sort of problem happens because your printer is (by default) set to obtain an IP address automatically from your wireless router.  That works fine when you install the printer because Windows is configured to print to that IP address.

If your printer is turned off for a period of time (e.g., more than a day or so), the router "assumes" that the IP address is no longer being used.  If at that point another wireless device is connected to your wireless router (e.g., your smartphone), your router may assign the IP address to the new device.  When you next try to print, Windows attempts to send the print job to the new device -- which ignores it because it isn't a printer -- and nothing happens.

The temporary solution to this is to determine the printer's new IP address (usually through the printer's display panel or by printing a "network status page" -- see your printer's User Guide) and changing the printer port on your computer to print to the correct IP address.  See, e.g., https://youtu.be/YcNETLD4Xto

The permanent and preferred solution is to configure your printer to use a fixed or static IP address and then change the printer port on the computer to use that static IP address.  Your printer's User Guide should explain how to set a static IP address.  You will need to access your router's web configuration page to determine the range of IP addresses potentially assigned by your router so that you can select a static IP address that is outside this range but still reachable by your router. 

For example, if your printer has been automatically assigned an IP address of 192.168.1.101, your router may have as its own address 192.168.1.1 and may be configured to assign IP addresses in the range 192.168.1.100 through 192.168.1.254.  In this case, you can set your static IP address to be anything between 192.168.1.2 through 192.168.1.99.
