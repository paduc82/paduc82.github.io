I would like to verify if there is any other user account (administrator) setup on your computer? If so, then you can log in using the other user account to back up your files. If there is none, then you can enable the hidden administrator in WinRE.

To enable a hidden administrator in WinRE, you can refer to the steps below:

Download the Windows 10 ISO from this link and use it to create an installation media. Follow the instructions under the Using the tool to create installation media (USB flash drive, DVD, or ISO file) to install Windows 10 on a different PC section on the link.

Once you're done creating an installation media, follow the steps below to access WinRE:

  1. Insert the installation media.
  2. Reboot your computer.
  3. Press any key to proceed when prompted.

  4. On the Install Now page, click Repair your computer.

  5. Choose Troubleshoot.

  6. Select Advanced Options.

  7. Click Command Prompt.

  8. On the Command Prompt, type this command followed by hitting Enter:
             Net user administrator /active:yes

  9. Close the Command Prompt.

Once you're done with the steps, reboot your computer and log in using the hidden administrator that you have enabled. Create a new user account (administrator) and then delete the hidden administrator. To create user account in Windows 10. Follow the steps below:

  1. Select the Start  button, then select Settings  > Accounts > Family & other people > Add someone else to this PC.

greg carmack - windows mvp 2010-2020 say:
Hi Andrew. I'm Greg, an installation specialist and 9 year Windows MVP, here to help you.

The hard drive is configured that way so that you can move your User folders to D to keep C smaller for imaging purposes. That way if WIndows ever becomes irreparable you can boot rescue media to apply the stored backup image to C in 20 minutes and be back up and running and your files will be current and safe in their separate partition. If the files were kept on C they will be dated from when the image is taken

Here's how to move your User folders to D: http://www.dummies.com/computers/operating-syst...

Then create your System Image backup to store externally - or also on D, though it's better to have it on another hard drive along with files backup in case your hard drive dies - using the best free imaging app Macrium: https://www.howtogeek.com/howto/7363/macrium-re.... It will help you make bootable recovery media to use in case Windows ever fails.

If you don't want to use this modern arrangement for which your PC was set up, then you can recover space into C from D using this tool which allows annexing space from any other partition: http://partitionwizard.com/video-help/extend_pa...

But to be sure there are no unseen difficulties Please post a screenshot of Disk Management, which I read like a doctor reads X-rays. Follow the steps here so I can see everything needed to advise you: http://answers.microsoft.com/en-us/windows/foru...
