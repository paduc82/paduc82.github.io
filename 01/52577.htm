I have err HAD! the exact same issue....and I figured out what it was. Apparently wmc needs to create a new user profile if you are configuring an extender and my ntuser.dat located under C:\Users\DEFAULT was missing! This file is essential for creating ALL new user profiles and wmc needs this file there to properly create the Media Center Extender Profile (You can hide the Mcx folder later). So in order to see if this is your problem navigate to C:\Users\DEFAULT folder and inside you will see a ntuser.dat file, if not here is what you need to do (Make sure to click view > Show hidden Files and Folders).


Okay so if your ntuser.dat is missing your registry key might still be saved! So in order to save yourself the time of doing a full system restore you can just go to regedit instead! (What a relief) Now navigate to HKEY_Users and make sure the .DEFAULT registry key with subfolders is there- if it is you are in luck!


Now open command prompt by typing cmd in the windows 7 search bar and right-click and click Run as Administrator. Now what we are going to do you see is create a new ntuser.dat file from the .DEFAULT registry key! So in command prompt navigate to C:Windows\System32 by typing in cd c:\windows\system32


Now type in this command prompt command: reg save HKEY_USERS\.DEFAULT "C:\users\%USERNAME%\desktop\ntuser.dat" /y [/c]

So now a new uncorrupted ntuser.dat file should be on your desktop. COPY it to the C:\Users\DEFAULT


Make sure the security permissions for the C:\Users\DEFAULT folder are set to SYSTEM Full Control, Administrators Full Control, and Everyone Read.


Now try creating a new user profile and if it works then your media center extender service should work with no problems after these next few steps!


1. Type in the windows search bar: services.msc and click on services. When the UAC control pops-up click okay. Now scroll down and look for Media Center Extenders make sure the service is started automatic. If not double-click on it and click start. Do the same for the Network List Service, Quality Windows Audio Video Experience, UPNP Device Host and the Windows Media Player Network Sharing Service. Also make sure the Internet Connection Service (ICS) is disabled.


2. Type in the windows search bar: Windows Firewall. Click on Allow a program through Windows Firewall. Now make sure the Media Center Extenders and Windows Media Player Network Sharing Service's Home/Private Check-boxes are checked and Public is unchecked. If this isn't so do this.


3. If the Media Center Extender firewall rule is missing follow this guide to add it in. If you do need to follow this guide make sure to go back into "Allow a program through Windows Firewall" and uncheck public for both the Media Center Extenders and Windows Media Player Network Sharing Service check-boxes.


4. Delete the Mcx User account located under C:\Users (This is the account that is corrupted that wmc creates also I am assuming you don't have any media center extenders already connected properly)


5. Now in the My Computer file location bar type in C:\Users\%USERNAME%\Appdata\Local\Microsoft and delete the eHome folder. (I am also assuming you have no media center extenders connected properly)
