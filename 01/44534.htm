You reached the Dutch forum.  Normally I would then continue in Dutch (and transfer the post to the English forum if prefered).  but since I am replying late I will continue in English.   

Getting bluetooth to work can sometimes be complicated: 

- First pairing then connecting.  (A bluetooth device can be paired but won't do much if it is not connected).   

- A bluetooth device makes use of Bluetooth profile.  If the wrong profile is used during connection there will be a problem.  For example there is a profile for mice and keyboards, a profile for Human Interface Devices, a profile to transfer file, a profile for audiostreaming, etc.   

- There can be an issue with bluetooth-stacks or bluetooth drivers when multiple drivers or stacks are used.  Usually to solve problems in this category the remedie is to delete all drivers and bluetooth stacks first then install the correct bluetooth driver and then connect the bluetooth device.  If you first connect the bluetooth device (for example USB blluetooth stick ) before installing the bluetooth drive Windows might assign the wrong driver to the Bluetooth hardware.  What matters here is to remove and disconnect all bluetooth software/drivers/ and portable hardware, then visit the website of the manufacturer then connect/pair/use the bluetooth hardware.   

Since you are using Dualshock 4 wireless controller it could be an idea for you to visit forums (Playstation) to see if this is a common issue.  It could be that other users of such controller solved similar issue.

how do I NOT search subdirectories - Microsoft Community

niborjus say:

how do I NOT search subdirectories

Windows 7 Explorer

Using Windows 7, there is no longer all of the help in performing a search from Windows Explorer. One must learn/memorize/search for a syntax guide if one wants to do any advanced searches.

I have not been able to find any documentation that describes how to limit the number of subfolder/subdirectory levels that one can search or even say to just search the current folder.

How do I do that?

david perdue say:

I have not tested this with natural lanuage search ON, but if you're using the standard Advanced Query Syntax.  The answer is:

 

*.* folder:"\Current_Folder_Name"

 

Where "*.*" is whatever filename search template you need, and "Current_Folder_Name" is obviously not those literal words but the name of the folder the Explorer window has open.

The magic is the leading backslash, which in typical undocumented style tells the AQS parser "just this folder".

 

 

 

david perdue say:

...and don't forget the quotes around the backslash + folder_name, they are required to limit the search.
