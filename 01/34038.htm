  2. What happens when you insert the recovery disc and restart the computer?

Error code 0XC0000034 appears when the Windows Boot Manager (Bootmgr) entry is not present in the Boot Configuration Data (BCD) store or if the boot\BCD file on the active partition is damaged or missing.

Let’s try these methods below to repair corrupted boot configuration data through Windows recovery environment.

Method 1:

 

Booting to Recovery Environment with Windows 8 installation media.

 

Please follow the steps given below. Insert the Windows 8 installation DVD in the DVD ROM and restart your computer.

  1. After you boot your computer using Windows 8 DVD or System Repair Disc, a black screen appears with gray text "Press any key to boot from CD or DVD". Press any key.
  2. Select the correct time and Keyboard type.
  3. Click Repair your computer in the lower left corner.
  4. Select Troubleshoot from Choose an option screen.
  5. Click Advanced options in Troubleshoot screen.
  6. Click Automatic Repair from Advanced Options screen.
  7. Next, choose the operating system by clicking on its button.
  8. Windows 8 will look for problems on hard drive(s) and verify that all required files are intact.

Method 2:

If automatic repair doesn’t help to repair the startup files, you may try to repair them using these commands through Command Prompt under Advance Options.

  1. After you boot your computer using Windows 8 DVD or System Repair Disc, a black screen appears with gray text "Press any key to boot from CD or DVD". Press any key.
  2. Select the correct time and Keyboard type.
  3. Click Repair your computer in the lower left corner.
  4. Select Troubleshoot from Choose an option screen.
  5. Click Advanced options in Troubleshoot screen.
  6. Click on command Prompt.
  7. Type these following commands and hit enter after each line of command:
    Bootrec /fixmbr
    Bootrec /fixboot
    Bootrec /scanos
    Bootrec /rebuildbcd

If the computer is not able to read the repair disc or installation media and it does not give option “press any key to boot from CD/DVD” when you restart your computer with disc inserted and straightaways goes to the error code, possibly CD/DVD is disabled as a boot device in BIOS (Basic Input and output System) and you might have to change boot order priority from BIOS and set CD/DVD as the first or second boot device. I would suggest you to contact your computer manufacture or a local tech shop to get boot order priority in BIOS changed.

BIOS Disclaimer:Modifying BIOS/ complementary metal oxide semiconductor (CMOS) settings incorrectly can cause serious problems that may prevent your computer from booting properly. Microsoft cannot guarantee that any problems resulting from the configuring of BIOS/CMOS settings can be solved. Modifications of the settings are at your own risk.
