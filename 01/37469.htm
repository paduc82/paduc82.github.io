considerate_guy say:

The VOB file (from a DVD) is encoded in MPEG-2 format. While older versions of Windows came with built-in decoding of MPEG-2 video, the later versions (Windows 8/8.1/10) dropped support for the MPEG-2 codec.

You'll see this in the notes, if you look at the Supported Video File types for Movie Maker. You'll see the .vob is listed as an MPEG-2 file and at the bottom it says,

"Support for using MPEG‑2 and Microsoft Recorded TV Show video files in your movie is available only if your computer is running one of the following editions of Windows: Windows 7 Home Premium, Windows 7 Professional, Windows 7 Enterprise, or Windows 7 Ultimate, Windows Vista Home Premium, or Windows Vista Ultimate."

If you are on a later version of Windows that doesn't support the older MPEG-2 codec, you have a couple options. You can either pay to install a licensed MPEG-2 codec (not recommended), or you can convert the .vob file to a supported format like .wmv instead. There are lots of conversion programs so do some research on which ones can handle a .vob to .wmv conversion and create a compatible file for Movie Maker.

I've used Freemake Video Converter to do this with success. The only caveat is Freemake is a freeware option and its installer tries to install a bunch of "add-ons" and browser changes. If you are going to install it, you must choose a Custom Install and then carefully uncheck anything it wants to install or change. You only want the program; not the add-ons.

Hi Ganicholson,

Thank you for showing interest on Windows 10.

Try the following methods and check if it helps:

Method 1:

I suggest you to change the display resolution and check if it works.

Kindly refer to the following Microsoft article on Change your screen resolution.

http://windows.microsoft.com/en-US/windows7/Change-your-screen-resolution

If the issue persists then follow the below method.

Method 2:

I suggest you to update the Display adapters driver from the Device Manager and check if it works.

Follow the below steps:

  1. Press Windows + R on the Keyboard.
  2. Type in devmgmt.msc
  3. Select the Graphic driver from the list by extending Display adapters.
  4. Right click on the Graphic driver and click on Update driver Software.

If the issue still persists then it might be the Monitors problem, I would recommend you to contact the manufacturer for further assistance.
