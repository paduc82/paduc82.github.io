Hi Jay. I'm Greg, an installation specialist and 9 year Windows MVP, here to help you.

After WIndows 10 was released, there was massive failure of the Digital Activation for Clean Reinstalls of Upgrade which I was involved in triaging for several weeks before it was resolved in MS activation servers. They also spent a fortune collecting every retail, embedded and COA Product Key from the qualifying Windows 7, 8 or 8.1 to enable Activating the free copy by using the old OS Product Key in the very next version update 1511.

So I was not surprised that even after the free Upgrade period ended, they quietly extended a Grace Period for activating with the Windows 7, 8 and 8.1 Product Keys.

If you use Microsoft Accounts it will then link it to the account so that changes of hardware can be managed via the Activation Troubleshooter, which also we have found allows a one-time exception for changes even to underlying OEM licenses.
