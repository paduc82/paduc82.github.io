How to Delete a Wifi Network Profile - Windows XP - Microsoft Community

lucerova say:

How to Delete a Wifi Network Profile - Windows XP


How to Delete a Wifi Network Profile - Windows XP

To delete a wifi profile:

 

1. Open Control panel, select and double clickNetwork Connections.

2. Select Wireless Network Connection, right click it and select Properties.

 

3. Choose “Wireless Networks” tab and select existing network in preferred networks, then click Remove.

 

4. Click OK.

 

 

 

To reconnect to a wifi network:

 

1. Open Control panel, select and double clickNetwork Connections.

2. Select Wireless Network Connection, right click it, select and click View Available Wireless Networks.

 

3. Click Refresh Network list to view available wireless networks. Select the correct network which you want to connect, double click it or click Connect button.

 

4. If the wireless network is secured, it will pop up a window asking for the key/password.

 

5. Enter the correct Key then click Connect, wait for a while, you will connect to this wireless network.
