chetarruby say:

Refurbisher Question: What is Windows 10 factory restore?

I am a computer refurbisher.  I take in donated laptops and recondition the ones viable for re-use, for resale.

Prior to the release of Windows 10, when an OEM recovery partition was present on a laptop, it could be used to restore the computer to it's factory defaults.  Using this procedure was permitted to avoid having to purchase a new Windows 7 license for the system.

However, on Windows 10 machines, the 'factory restore' is actually just "Reset This PC" option in Windows 10.  Does this count as a factory restore and negate the need to purchase a brand new Windows 10 license?  I really need a Microsoft representative to answer this question.

This "gray" area is very confusing and I really need a straight answer.  What counts as a "factory restore" on a Windows 10 computer?  When do I need to purchase a new license for refurbished computers, and when can I re-use the license when the computer is already activated?

***Post moved by the moderator to the appropriate forum category.***

xylee del say:

For your queries about Windows 10, changing the owner of a licensed Windows 10 doesn't make it invalid. As long as the Windows 10 license was installed previously on the machine, the Operating System stays on the hardware.

If the owner sells the computer to a 3rd party or to a new owner, the license will still be valid. We highly suggest that you perform a clean install on that certain computer. You can check MVP Andre Da Costa's article if you need a walkthrough: How to: Perform a clean install using Reset this PC in Windows 10.

Hardware changes are fine such as hard disk replacement, it will not affect the Windows 10 license. Unless you are going to make major changes such as a Motherboard replacement, you might have to buy a new Windows 10 license.

Feel free to reach out to us if you have any additional queries about Windows 10.
