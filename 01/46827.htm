This could be an issue if your power plans is set to be in Power Saver Mode.

To change the option follows the steps:

1.     Click on Start.
2.     Go to Control Panel.
3.     Select System and Security.
4.     Go to Power Options.
5.     If the power saver mode is checked change it to Balanced
6.     Restart the system and see if now the issue is solved.
