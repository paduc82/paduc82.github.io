  • For IMAP, if you delete an email its gone
  • For Hotmail account, please refer to “Recovering Deleted Emails from your Hotmail Inbox”. Emails can still be recovered by Microsoft.
  • All deleted emails are stored in the email account's 'Deleted' folders’ for a short period of time for all three types of accounts mentioned above. Check if these folders contain any of the deleted emails.

- In Windows Live Mail <Email account>/Deleted Items
- In Explorer, %USERPROFILE%\Local Settings\Application Data\Microsoft\Windows Live Mail\<Email account name>\Deleted Items

  • If email is no longer available on server itself, there is a low chance of these mails will be recovered.
  • 'Storage folders' Deleted Items only contains items deleted from Windows Live Mail’s user created folders. If the program is functioning properly and the deleted messages are not present in the email account's ‘Deleted Items’ folder or not archived elsewhere in Live Mail or the user's hard drive, unfortunately, these files cannot be retrieved.

1.     Tap F8 during startup to get to the Startup Options menu.

2.     Choose Repair Windows.

3.     After choosing the location and logging on, click on the Command Prompt option.

4.     Type the following command followed by the ENTER key:

6.     When CHKDSK has completed, restart and see if we can get into normal and/or safe mode.

vaseem ahmed say:

Hi,

Do you get any error message when you turn on Window firewall?

I would suggest you to follow the steps mentioned below:

Method 1:

Give permission to folder and check if the issue occurs. Follow the steps in the link:

http://technet.microsoft.com/en-us/library/cc753659.aspx

Method 2:

Run fix it from the following link and check if the issue persists:

http://support.microsoft.com/mats/windows_firewall_diagnostic/en-us

Method 3:

Check if Windows Firewall service is started and set to automatic. To do this, follow the steps below:

a.       Click on Start.

b.      In the start search box, type services.msc and press enter.

c.       In the services window,  Right-Click on Windows Firewall Service and click on Properties.

d.      Check if the startup type is set to automatic, if not set it to automatic.

e.      Also make sure Log On is set to Local system Account

f.        Click Apply, General tab, Start service. Click on start button, this will start the Windows firewall Service.

g.       Restart the computer and check if you can enable Windows Firewall service.
