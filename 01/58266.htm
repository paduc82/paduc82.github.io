cmdrkeene say:

The percentage is for the offline cache, not for how much of the drive it will use.  It will use the entire drive up until the drive has no more space.  That's why the first thing it does when you setup a File History drive is... erase the drive.

The offline cache is how much space to use of the local system drive if the File History drive is not available (such as if it's at home and you're using a laptop).
