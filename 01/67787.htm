WARNING: Edit the registry at your own risk.  I make no claim that the registry edits I specify below are "recommended," "correct," or "appropriate."  Only that they worked for me.

  • I used regedit to delete the entire key:

HKLM/Software/Microsoft/Windows/CurrentVersion/WindowsUpdate/OSUpdate

  • Restarted
  • I don't remember, but believe that Windows Update continued to indicate that a reboot was required to install pending updates.
  • Configured Windows Update to allow me to choose which updates would be installed rather than auto-install.

  • Used Control Panel > Programs and Features > View Installed Updates to uninstall KB3035583 (Get Windows 10), then restarted the PC
  • Windows Update indicated: Restart to install important updates.  Restarted as requested, but the "Restart to install important updates" remained after restart.
  • Used regedit to examine HKLM/Software/Microsoft/Windows/CurrentVersion/WindowsUpadate/AutoUpdate

  • Installed them and Restarted one last time.
