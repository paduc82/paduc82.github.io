These complexity requirements are enforced upon password change or creation of new passwords.

A strong password:

• Is difficult for anyone to guess or hacker to decode.

Hello RyanO1080,

Thank you for posting to the Microsoft Answers Forum.

The best way to sync your files with another computer is to use Sync Center. In order to use the Sync Center, you must have a shared directory that you would like to sync. Navigate to a shared folder, right-click the directory and choose Always available offline.

You first will need to set up a Sync Partnership. 

1.

Turn on the device and plug it into your computer. If it is a wireless device, make sure that a wireless connection is established between the device and your computer or connect it by using a universal serial bus (USB) cable.
2.

Open Sync Center by clicking the Start button , clicking All Programs, clicking Accessories, and then clicking Sync Center.
3.

In the left pane of Sync Center, click Set up new sync partnerships.
4.

Click the name of the device in the list of available sync partnerships.
5.

On the toolbar, click Set Up.
6.

Select the settings and schedule to determine how and when you want to sync your device with your computer.
7.

To sync immediately, click View sync partnerships, click your device in the list of sync partnerships, and then, on the toolbar, click Sync.
Notes
  • If your device does not appear in this list, try connecting it again or turning it off and on. Then try clicking the refresh button ‌ in Sync Center. If the device still does not appear, see Sync with a device for more information.
  • Some sync partnerships do not have any sync settings or a schedule that you can adjust.
In order to ensure that your folders are continually synced choose The computer has been idle for at least 5 minutes. While this may be excessive, it ensures that you will have the newest copy of your files at all times. In Sync Center you can initiate a manual sync, stop an in-progress sync, view the status of all current sync activities, and receive notification of the need for conflict resolution.

Hi Haveaux,

BCCode: a  0x0000000A

Cause: This bug check is issued if paged memory (or invalid memory) is accessed when the IRQL is too high.

The error that generates this bug check usually occurs after the installation of a faulty device driver, system service, or BIOS.

If you encounter bug check 0xA while upgrading to a later version of Windows, this error might be caused by a device driver, a system service, a virus scanner, or a backup tool that is incompatible with the new version.

Refer this link for more information:

http://msdn.microsoft.com/en-us/library/ms793589.aspx

You may also unplug any external devices, if any and check if the issue gets resolved.

Method 1: Try booting into “Last Known Good Configuration” mode and check if you’re able to resolve the issue.

Refer the links below for more details:

Advanced startup options (including safe mode)
http://windows.microsoft.com/en-US/windows7/Advanced-startup-options-including-safe-mode

Using Last Known Good Configuration
http://windows.microsoft.com/en-us/windows7/Using-Last-Known-Good-Configuration

BCCode: 3b or (0x0000003b)

Cause: This error has been linked to excessive paged pool usage and may occur due to user-mode graphics drivers crossing over and passing bad data to the kernel code.

Refer this link for more information:

http://msdn.microsoft.com/en-us/library/ms793232.aspx

BCCode 109 refers to kernel corruption either through bad RAM or drivers.

Method 2: Hence, I’d recommend you to run Windows Memory Diagnostic Tool. Windows Memory Diagnostic Tool scans your computer's memory for errors.

Refer the following article for more details to run Memory Diagnostic Tool:

What are the system recovery options in Windows 7?
http://windows.microsoft.com/en-US/windows7/What-are-the-system-recovery-options-in-Windows-7  

If the tool detects errors, you should contact your computer manufacturer for information about fixing them, since memory errors usually indicate a problem with the memory chips in your computer or other hardware problem.

For most accounts, the Drafts folder can't be deleted, accidentally or otherwise. It's a system folder, like Inbox, Sent items, Junk email and Deleted items. The only exception I know of is for Outlook Mail accounts on short-lived builds of the program introduced in late 2015 and withdrawn a few months later. If one of these builds was installed, messages in the account drafts folder were moved to the Drafts folder under Storage folders, because the version of Exchange ActiveSync in use at the time didn't allow for the synchronization of the account Drafts folder.
