FAQ - Have questions or need help? Look here first! - Microsoft Community

barbarossablade say:

FAQ - Have questions or need help? Look here first!

Taptiles FAQ

 

Why are there so many ads?

The short version is that we have a dedicated, passionate development team that wants to continue to improve Taptiles with updates and bring you more fun and more features – they love the game just as much as you do! That development team is not a part of Microsoft, so they rely on the revenue from Taptiles ads and Premium subscriptions to keep developing. Unfortunately, this means that sometimes new features that take a lot of work – like supporting brand new Daily Challenges – comes with additional ads. The team is very aware of the ad experience and its impact, and are always looking for ways to make it less intrusive and a better overall experience for the players while sustaining development.

 

Why are there no ads showing, or the same ad over and over?

Ad management, inventory and serving is a pretty involved process, and sometimes there aren’t ads to show, or only a few active campaigns. As with all aspects of the ad experience, the development team and Microsoft are both very aware of the impact of this and are working hard every day to get more variety in there!

 

Why does the game immediately close on startup?

This is a known issue that has been a problem on different devices at different times, and the team is working diligently on trying to track down the problem(s), but in all honesty, so far it hasn’t been easy. Each update contains potential fixes for some of you experiencing this, and it is top on the list of priorities - please be patient while the investigation continues!

 

What happened to my Medals/Achievements/Badges?

If you’re not seeing these show up, or you use multiple devices and one of them doesn’t seem to be up-to-date, please be sure you are signed in to your Xbox account properly, since this data is synchronized in the cloud.

 

Why am I not getting the Time in a Bottle time bonus anymore?

This is a known issue that was tracked down recently, and will be fixed in Taptiles update 2.5, coming soon!

 

Why aren’t there different timers for mouse and touch players?

Based on lots and lots of player data, there actually isn’t a noticeable speed difference between these types of players. The mouse may intuitively seem slower, but touch players obscure the view of their next match with their fingers and hands, so for players of the same skill level (including the fastest players the team has seen) it actually balances out – it just depends on which input they are most comfortable using.

 

How can I prevent the mouse from rotating the puzzles?

You can completely turn off mouse rotation under Settings -> Game Options -> Mouse-drag Rotation. Note that turning this off means you will need to click the in-game rotate arrows or press A and D on your keyboard to rotate puzzles.

 

How do I enable or disable color-blind support?

You can find this option under Settings -> Game Options.


I have a specific issue not covered by this FAQ. What should I do?

Please search the forum for your problem using a keyword, such as “Dash” or “account”, to see if a support thread already exists. If not, please create a new post with a descriptive title such as “Dash mode no longer works”, and in the post, please describe exactly what happens step by step, and include the manufacturer, model, and/or processor and graphics card of your device so that we can pass this on to the team for troubleshooting.

tryx3 say:

Login problems can occur because a password has been forgotten as in this case or because a user profile has become corrupted.  There is no Windows 7 utility for then bypassing the login requirement.

Here are proposed solutions to the problem in rough sequence of total effort necessary.  If the first solution cannot be completed because the appropriate conditions or prerequisites do not apply then review the next solution in the list & so on.

1     Click on the text Reset password underneath the login box & insert the Password reset disk that was made when the account was set up.  Then follow the Password reset wizard’s procedure.

2     Get an Admin-level account [or, if on a network, a Domain administrator] to login & reset the problem user account’s password.  See What are the risks of resetting a password?

3     Use System restore to step back a short period to before any corruption to the system occurred.  See How to use System Restore to log on to Windows 7 or Windows Vista when you lose access to an account  It would then be worth Fixing the corrupted user profile to avoid a reoccurrence of the problem.

4     Insert the System repair disk & connect the drive that has the latest System image, reboot into the System repair disk & restore the system using the latest System image.  All existing data on the system drive will be lost but data on different partitions of the same physical disk often survives this restoration.  Afterwards, restore missing personal files from a separate backup if such a backup exists.

5     If Windows 7 was preinstalled on the PC then the PC maker will have provided a means of restoring Windows 7.  This may be by means of a Factory Image Restoration procedure described in the PC manual.

6     Use your Windows 7 Installation DVD to reinstall Windows 7 from scratch.  To do this, boot from the Installation DVD then follow the onscreen instructions.

7     Make your own Windows 7 Installation DVD / USB [you will need a 4GB USB stick for this]. 

7.1     To make an installation DVD/USB, download the correct* ISO from Official Windows 7 SP1 ISO from Digital River then transfer it to a DVD or USB using the Windows 7 USB-DVD download tool

7.2     * You can only run a successful installation with the edition of Windows [Home Premium & so on] that is identified on your Certificate of Authenticity.  If the PC is only specified as being capable of running 32 bit Windows 7 [Windows 7 “x86” but often referred to as x32] then only the 32 bit [x86] ISO file will suit.

7.3     Install the USB-DVD Tool then run it with the ISO file stored anywhere on the system [except the target DVD/USB drive, I discovered, although others say that it can be on the target drive].

7.4     The installation DVD/USB can also be used as a Windows repair disk for repairs & for restoring system images.

7.5     To install Windows 7, boot from the installation DVD/USB then follow the onscreen instructions.
