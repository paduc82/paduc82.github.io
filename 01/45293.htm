xircal say:

It appears to be a USB 2.0 device, so the first thing to check is to make sure you haven't connected it to a USB 3.0 port. Those are blue while USB 2.0 ports are white.

Also, disable USB Selective Suspend in the Power options menu. To access that, do the following.

  1. Right click the Start button and go to Power Options.
  2. In the Power Options menu which opens, click the link on the right called "Change plan settings".
  3. In the following menu, click the link to Change advanced power setttings".
  4. Scroll down and expand USB Settings | USB selective suspend settings and disable both.
  5. Click Apply/OK

This will prevent the USB driver sending Windows a message to idle the device.

If this doesn't resolve the problem, go to this site and explain to them that you purchased the webcam on the premise that it's compatible with Windows 8.1, but you cannot get it to install and this Community site was unable to help. http://support.microsoft.com/contactus/?ws=support&ln=en-us
