n01d3a say: I have noticed that downloading/installing apps from the windows store will not complete if the computer has windows updates that are waiting to be installed.

To check if there are updates that can be installed, go to the start screen. While at this screen, just type the word "update." The screen will change, and off to the right, please select the "settings" option. You should now see a list appear to the left, select "windows update" from the list. Now, click the button that says check for updates now.

This may take a minute or so, and you should see dots moving across the top of the screen to let you know windows is working. After the check, you should see purple writing that shows how many updates need to be installed, or it will say "no important updates are scheduled to be installed." If there are updates, you can click on that text, scroll to the bottom and choose to install update now.

Install all the updates, restart the computer. Then try to download the apps from the store again.
