hairyfool say:

There are two problems that I can see although I am not entirely see how it has got to that state.

 

1stly you have assigned the drive letter A: to the drive. A: and B: are normally limited to the floppy drives and I thought that if you tried to use them it would generate an error. Floppy drives cannot be partitioned so this might be when you give one the label all get assigned the same. Try assigning the next letter free above the last one used. If they all change then try giving one of the others a different letter afterwards to see if it will take the change on its own.

 

Secondly you have two adjacent unallocated partitions. These would normally have concatenated into one single unallocated space. This rather indicates that there is a problem with the partition table. Also none of the partitions are primary. This might however be a result of being a dynamic drive.

 

I would recommend that you back up any data in those partitions and remove them all and make sure that the drive then collect to a single unallocated space. When empty revert it back to a basic drive and recreate partitions as required. Unless there is a specific need basic will be all that you need.
