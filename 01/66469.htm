Each version of Windows has its own Product ID and can only be activated by a key for that SKU. This always has been the case, 7 cannot be activated by a Vista key, Pro cannot be activated by a Home key and so on.

The Windows 10 free upgrade program uses a different method to activate installation, although no new key is issued it is not activated by the Windows 8.1/7 key itself but by the fact that the qualifying installation is valid.
