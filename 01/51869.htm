copying folders - Microsoft Community

kenw100 say:

copying folders

Is there a way to copy the folders only from a directory and paste it to another directory, but without copying the contents of the folders?

a. user say:

I believe the xcopy command is still included in Windows 10. You can use it with switches to duplicate a folder structure without copying the folders' contents.

The /t switch copies the folder structure (only) from the source folder to the destination folder without copying the files, but it won't copy empty folders in the source folder. If the source folder includes empty folders you will need to add the /e switch as well. So the full syntax would be:

xcopy source target /t /e

From the School of Hard Knocks: Practice with a set of dummy folders first.

Just remember: You need to have adequate permissions in the source and destination folders.

whagi say:

If the details on your current PSU say 110 - 220 and it has a tolerance of 50hz- 60hz, then you could use that.

US is 120v at 60Hz

Canada is 110v at 60Hz

China is 220v 50Hz.

But, if your PSU voltage rating is restricted to say 100v-127v. Then you will need to purchase the correct PSU for the country, in this case, China. Then you have the whole NTSC and PAL issues if not using HDMI.

To be honest, your best buying a new Xbox in China to save you all the hassle, and then you know your 100% correct/safe.
