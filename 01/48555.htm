lemp say:
These ".NET Frameworks" are added by applications that you install that need them to run. 

Unfortunately, they are not all merely "upgrades," and some applications need specific versions of the .NET Framework.  For example, an application that needs .NET 3.5 will probably not work if the only thing installed is .NET 4.

You can uninstall any or all of them, but if you do, it's likely that some application that you have installed will stop working.   And no, I don't know any easy way to determine which applications may use which .NET Framework versions.

For technical details, see http://blogs.msdn.com/b/astebner/archive/2009/04/20/9557946.aspx

Hi Merlinboss,
 
As the hard drive is listed in the device manager, it sounds like the drive letter is not assigned to the hard drive. This could be one possibility why the device is not listed in Computer.
 
Try the following steps and check the result.
Step 1: Assign a drive letter
a. Click Start, type diskmgmt.msc and hit enter.
b. Check if the external hard drive is listed, if yes, then assign a drive letter.
See Add drive letter
c. Restart the computer and check if the issue persists.
 
Step 2: If the issue persists, run the Microsoft diagnostic tool to find and fix the issue
a. See Tips for solving problems with USB devices
b. Click Microsoft Fix it and run it successfully.
c. Check if the issue persists.
 
Note: Few hard disks are compatible only with Mac and may not work on Windows.

srimadhwa b say:

Hi Bill,

 

Xcopy provides an excellent tool for backing up selected folders. With appropriate switches, a variety of backup scenarios can be created. One possible backup configuration would be to copy only those files that have been changed.

 

Here is an example command: xcopy C:\somefolder E:\backupfolder /D /E /C /R /H /I /K /Y

 

This command will copy all files, including those in sub-folders that are newer in the source folder. It will copy hidden as well as read-only files and will create the destination folder and/or sub-folders if they do not already exist.

 

Additional Information.

Windows 7 has Robocopy which can be much faster.
