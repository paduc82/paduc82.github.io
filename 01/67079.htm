What happened is that classic versions of Movie Maker had a snapshot feature that let you take a full frame sized one from a clip in the collection bin.


Newer versions such as MM2012 don't have a collection bin and snapshots can only be taken from a clip in the project.... the same size as your current preview window. You can maximize the size of the preview window but it can't be higher than that.


You could install the classic version MM2.6 and use it for snapshots of clips in the collection bin.... or use other software to take the snapshots.
