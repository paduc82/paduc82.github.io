What is BitLocker Encryption?

BitLocker (formerly BitLocker Drive Encryption) is a full disk encryption feature included with the Ultimate and Enterprise editions of Windows Vista and Windows 7, the Pro and Enterprise editions of Windows 8 and Windows 8.1, and Windows Server 2008 and later. It is designed to protect data by providing encryption for entire volumes.

For reference:

http://windows.microsoft.com/en-US/windows7/products/features/bitlocker

 

In this case, letâ€™s try using the BitLocker Repair Tool to recover this drive and check if that helps. The BitLocker Repair Tool (Repair-bde) is a command-line tool included with Windows 7 and Windows Server 2008 R2. This tool can be used to access encrypted data on a severely damaged hard disk if the drive was encrypted by using BitLocker Drive Encryption.

Repair-bde can reconstruct critical parts of the drive and salvage recoverable data as long as a valid recovery password or recovery key is used to decrypt the data. The Repair-bde command-line tool is intended for use when the operating system does not start, or when you cannot start the BitLocker Recovery Console. However, if the drive has been physically damaged, it may not be recoverable.

Please refer to the TechNet article given below and follow the instructions given in it to run the BitLocker Repair Tool using command prompt.

Using the BitLocker Repair Tool to Recover a Drive
