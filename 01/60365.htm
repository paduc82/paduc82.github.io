If pressing F1 does not result in booting to Windows, your hard drive likely has already failed.  Stop using the computer.

If you have been regularly making backup system images, you could simply replace the hard drive and restore your backup image.

Otherwise, you might be able to recover some data by removing the hard drive, putting it in an external enclosure, and connecting it to another computer.  You would then have to install a new hard drive in the laptop, install Windows and all of your application programs, and restore whatever data had been salvaged.
