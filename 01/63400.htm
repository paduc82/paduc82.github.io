frederik long say:
The vast majority of network sharing problems are due to one or several of the following reasons:
  • People ignore the need to have identical account names & non-blank passwords* on each PC.
  • They do not share specific folders.
  • They do not set the share access rights to "Full" for "Everyone".
  • They do not set the folder access rights to "Read" or "Full Access" for selected accounts.
  • They do not activate "Network Discovery".
* Many people do not wish to bother with passwords. In these cases Windows can be configured to remember the password for the default account. However, it then becomes imperative to have a fully tested and documented spare admin account on each machine. Without this spare account it is only a matter of time until the user locks himself out.

1. Open Notepad

@echo off
FOR /F "tokens=1,2*" %%V IN ('bcdedit') DO SET adminTest=%%V
IF (%adminTest%)==(Access) goto noAdmin
for /F "tokens=*" %%G in ('wevtutil.exe el') DO (call :do_clear "%%G")
echo.
echo Event Logs have been cleared! ^<press any key^>
goto theEnd
:do_clear
echo clearing %1
wevtutil.exe cl %1
goto :eof
:noAdmin
echo You must run this script as an Administrator!
echo ^<press any key^>
:theEnd
pause>NUL

4. Run the file as an Administrator
