Whenever I see a Win10 PC running slow, I first check to see if it is installing updates as this is frequently a cause for slow down. Alternately, when Win10 is in standby mode, it will try to do maintenance tasks in the background, so I have found that it is common for the PC to act sluggish for a minute after waking up.

These two issues are disk related slow downs and, as such, you should be able to see the HDD maxing out in task manager. The best fix I have found to avoiding both of these issues is upgrading to an SSD if you are running Win10 Home or Pro. If running Enterprise edition, you can avoid some of these issues by pushing out updates at night instead of switching to an SSD.
