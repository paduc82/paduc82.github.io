¡firedog say:
... I use some software to clean the registry as well junk files.


This is almost certainly the problem. There is absolutely no need to 'clean the registry' - please see here: Why you should not use Registry Cleaners and Optimization Tools. Which program are you using?

The Windows Live Mail databases use .log files to save transactions temporarily, and other files with common extensions like .chk and .tmp to keep track of which log files have been replayed. This is the sort of file that your 'clean-up' program probably deletes, rendering the database unusable.

All I can suggest is that you copy what's left of the store folder containing your messages to your documents folder, then when Windows Live Mail launches with its new database, use the File > Import messages function to import from the store folder in your documents folder.
