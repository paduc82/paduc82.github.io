If you have a user account that you want to make unavailable, you can disable it. A disabled account can be enabled again later. Disabling an account is different from deleting an account. If you delete an account, it can't be restored.

 

Follow the steps below to enable the inactive account on your PC and check if the issue is resolved.

 

  1. Press Windows logo key on the keyboard, type mmc and select the top most search result. Enter the administrator password if prompted.
  2. From the left side panel of the window, select Local Users and Groups.
  3. If the option is not visible, click on the File option from the file ribbon of the window and then select Add/Remove Snap-in option. Click on Local Users and Groups and then click on Add. Select Local computer, click Finish and OK.
  4. Double click on the Users folder, right click on the user account you want to activate and select Properties from the context menu.
  5. From the General tab, uncheck the Account is disabled option and then click on OK.
  6. Close the window and check if the changes are effective.

You may be facing the above issue due to the corrupted user profile on the PC. Enable built-in administrator by following the steps below, create a new administrator user profile, transfer all the data from the previous administrative user profile and check if the issue is resolved.

 

Step 1: Enable built-in administrator account.

 

  1. Open elevated command prompt by entering Safe mode with Command prompt. To go to safe mode, use the power button on the sign-in screen and press Shift + Restart. This will take you to the recovery boot menu. Click Troubleshoot> Advanced options> Startup settings. You then see a list where you can choose Safe Mode with Command prompt.

  2. Type the following command: net user administrator /active:yes

  3. Login into the built in administrator user profile and create a new user profile with administrator privileges.

  4. Delete the corrupted user account from the built in administrator.

  5. Then lock your screen by pressing Windows key + L, and sign-in at the Administrator account.

  6. net user administrator /active:no

 

Step 2: To create a new administrative user account follow the steps below:

 

  1. Open Elevated command prompt by right clicking on the Start icon and select Command prompt (Admin).

  2. Type net user NewAccount password /add and click Enter.

  3. Type net localgroup Administrators NewAccount /add and click Enter.

  4. Log off from the system and login to the new user account.

I have now resolved the issue. Having re-installed Live Mail and also recreated the email account without any effect I came to the conclusion that there was corruption in the email database operations.

This requires a rather drastic series of steps

1. Back up your emails

2. Uninstall Live Mail.

3. Delete everything in the Windows Live Mail folder. The Path is C:\Users\User\AppData\Local\Microsoft\Windows Live Mail. Note AppData cannot be seen unless it is made visible  by clicking on Organise at the top left of the Windows Explorer window. The Path is: Organise\Folder & Search Options\View\ then click on the radio button next to: Show Hidden Files, Folders and Drives then click on Apply.

4. Install Live Mail re-create the email accounts, these will then sych with the server causing many emails and contacts to be restored. If you have other POP email accounts you may loose old emails as these accounts will not synchronise.

5. Test the LIVE mail operation for several days. If all is well you can restore emails for the POP accounts.

I have been testing for more than a week without any more problems. Beforehand the most disturbing problem seen was that an email would sit in the Inbox for a few days then the contents would disappear.

Not a simple problem to identify. Hopefully this will help others.
