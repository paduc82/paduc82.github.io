Okay.  Your P7-1534 has an MSI MS-7778 motherboard, HP name 'Jasmine'.  It has an AMD A75 chipset for socket FM2 processors.  Right now it has a factory installed A8-5500 AMD with the Radeon HD-7560D graphics embedded in the CPU.  We already established 512MB of shared main memory is how it is limited.  The motherboard can take up to an AMD A10-5800K processor with Radeon HD-7660D graphics built in.  It still pulls from the main memory, and the highest memory you can take is 1866 Mhz, up to 16GB and 1600 Mhz up to 32GB. 

For video purposes the graphics processor sees it as 933 Mhz memory.  Pretty good for 2013.  A tad slow today, and bottlenecked at 512 MB of usable memory and DirectX 11 capable.  W10 is capable of DX 12..  To upgrade the processot to get a better onboard graphics processor built in is the least bang for the buck and bigger money.
