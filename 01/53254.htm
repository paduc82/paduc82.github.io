Hi,

There may be a compatibility issue between the Bluetooth of your computer and Windows 10 that is why the Bluetooth is not working properly. You also mentioned that the latest driver for the Bluetooth is available only to Windows 7. This is also the sign that the specific component is not compatible to Windows 10. A component must have a driver for Windows 10 so that it will function properly. There is still a way to use the drive that you saw from the manufacturer's website. We recommend that you run the installer of the drive in compatibility mode so that your computer's Bluetooth will run in Windows 10. To do this, follow these steps:

  1. After downloading the driver from the manufacturer's website, right click it.
  2. Click Properties.
  3. Click the Compatibility tab.
  4. Under Compatibility mode, click the Run this program in compatibility mode for: check box.
  5. Click the drop-down menu, and then choose Windows 7.
  6. Click Apply, and then OK.

After following the steps above, the installer will now run in compatibility mode. Please take note that you need to uninstall the Bluetooth driver of your device first before performing the steps above. Access this link to know how to uninstall a driver. If you will see a prompt to restart your computer after uninstalling the Bluetooth driver, kindly cancel it so that Windows 10 will not install a generic Bluetooth driver. While uninstalling the Bluetooth driver, you need to click the Delete the driver software for this device option so that the Bluetooth driver will be completely removed from your computer.
