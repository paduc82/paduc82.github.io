Hello,

Try to convert MBR disk to GPT disk through the diskpart command line. The following will convert disk 1 to GPT disk.

1. Open Command Prompt as an Administrator.
2. In the command prompt, type diskpart and press enter.
3. Type list disk and press enter.

4. Type select disk 1 and press enter.
5. Type detail disk and press enter. This will give you a list of all the volumes on the selected disk. If you do not have any volumes listed, then you can go to step 9.

9. Type select disk 1 and press enter.

10. Type convert gpt and press enter.

11. Type exit and press enter.

You can change the MBR disk to GPT disk.

Method 2: Check your hard disk for any errors:

  1. Click Start.

  2. Type: CMD, from the results, right click CMD.

  3. Click 'Run as Administrator'.

  4. At the Command Prompt, type: chkdsk /r.

When you restart your system, your computer will be scanned for errors and attempts will be made to correct them.

Important: While performing chkdsk on the hard drive if any bad sectors are found on the hard drive when chkdsk tries to repair that sector if any data available on that might be lost
