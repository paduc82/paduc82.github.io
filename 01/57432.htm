Try to run checkdisk on your windows drive.

If still your problem is not solved you can create a new user profile by following options.

Boot from bootable dvd or pendrive of windows.

hit Shift + F10 to open command prompt

navigate to windows\system32 on your installation drive using

 d:  (if 2nd prartition contains windows)

cd windows\system32

rename utilman.exe file to something else temporarily.

ren utilman.exe temp.exe

now in place of it put cmd.exe

copy cmd.exe utilman.exe

restart your computer

on login screen click the assistance button it will open command prompt.

type netplwiz and hit enter. you will see screen to create new user

after login to new user navigate to users folder in c drive and your previous user and move all required files.

Hope it solves your problem. But rename back your utilman as this process can be used to manipulate your system from login screen.

frederik long say:

You need to tell us which line generates the error message.

You also need to change this line

if %ERRORLEVEL% == 0 goto decryptWinXP

to

if %ERRORLEVEL%==0 goto decryptWinXP

or even better: replace

ver | find "XP" > nul
if %ERRORLEVEL% == 0 goto decryptWinXP

with

ver | find "XP" > nul && goto decryptWinXP
