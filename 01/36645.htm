Unless someone out there has got it to work.

Stop searching network locations

How can I stop Windows from searching a network attached storage [NAS] drive? It is making search run slowly.

I cannot remove or uncheck these folders using Indexing Options>Indexed Locations because they do not appear in the list of indexed locations.

cmdrkeene say:

If you went ahead and indexed them, your searches would be fast because they'd come from the index instead of scanning the NAS drive over the wire.

But you can also set Windows to no longer search contents in a non-indexed location.  This speeds up searches, but it will still scan those locations for matching names (just not content). That's explained how here (do the opposite to turn it off).

You can also change the scope of where you want to search.  If you want to only search the C: drive, for example, just open the C: drive before you type your search term.  The NAS drives won't be searched.
