    Method 1: Troubleshooting Hardware & Devices

    1. Press Windows Key
    2. Type Troubleshooting in search box and press enter to open Troubleshooting
    3. The Troubleshooting window will open
    4. Click View all
    5. Click Hardware and device from a list
    6. Click Next and follow the instructions

    Method 2: Services

    1. Press Windows Key + R
    2. Run window will open
    3. Type services.msc and press enter
    4. Right Click on Bluetooth support service and click Properties
    5. Set the Startup type as Automatic
    6. Click Start under Service Status.
    7. Click Ok

    Method 3: Connect a Bluetooth Device

    1. Windows Key
    2. Type Bluetooth settings and press enter
    3. Turn on Bluetooth
    4. The device list will updated
    5. Select any device to pair
    6. Note: System and Bluetooth device will automatically connect, when two devices are in range and Bluetooth is turned on.

    Method 4: Lenovo Support(Lenovo G500)
    1. Please visit: Lenovo Support - "Lenovo G500"
    2. Download Latest Drivers for Bluetooth

    Method 5: Check & Restore windows integrity

    1. Windows Key
    2. Type CMD
    3. Right click and run as Administrator
    4. Command Prompt Windows will open in Administrator Mode.
    5. Type the following command and press enter
    6. DISM.exe /Online /Cleanup-image /Scanhealth && DISM.exe /Online /Cleanup-image /Restorehealth
    7. (Do you get any error??)
