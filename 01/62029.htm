Change font color on desktop

cmdrkeene say:
From our contributor Ronnie Vernon:

You cannot change the Desktop font color.

Because Desktop wallpaper images have varying colors, it is easy for the font colors to be unreadable if they are on top of the same color as a wallpaper image. For instance a black font on top of a dark part of the background image or a white font on a white part of a background image.

The Desktop icon fonts are dynamic and will show white on dark backgrounds and change to white with a black shadow effect on a light background.

In previous versions of Windows, it did not matter what color you picked for the Desktop fonts, some would always be rendered invisible because of the background color of a desktop wallpaper. The user was constantly moving icons around the desktop to a different colored area where the font could be read. This was the source of a large number of complaints from users asking for a font that was always visible, regardless of the color that was behind the font. 

The white/shadowed font solves this problem.
