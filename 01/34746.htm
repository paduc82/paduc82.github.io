naman r say:

 

Hi,

Try using diskpart to assign a drive letter to the sisk and check if the issue persists.

a)     Click Start then type diskpart in the search bar.

 

b)     Right-click diskpart in results above > click Run as administrator

 

c)      Once diskpart loads type list volume, press enter

 

d)     You should notice that you will see your partition that you use for your data, it should not show a drive letter assigned to it.

 

e)     Type select volume 6 (replace 6 with the volume number assigned to the partition in question)

 

f)      Type assign letter Z (replace Z with the letter that you want to assign to it)

When you look in Computer, you should be able to see your partition.

You can also refer to the link provided below for more information on adding and removing drive letters.

http://windows.microsoft.com/en-US/windows-vista/Change-add-or-remove-a-drive-letter

Note:

Even though the link provided above refers to Windows Vista it applies to Windows 7 as well.
