motobarsteward say:

FIXED!!!

Full procedure: -

Download zip file from

https://downloadcenter.intel.com/download/25150/Intel-Iris-Iris-Pro-and-HD-Graphics-Production-Driver-for-Windows-10-64-bit

Unpack the zip file.

Open Display settings-> Advanced Display Settings-> Display adaptor properties

Click the button (marked with an Admin shield) called Properties next to Intel(R) HD Graphics 5000

Select the driver tab and click on Update Driver

Then select 'Browse my computer for driver software'

The select 'Let me pick from a list of device drivers for my computer'

Click the 'Have Disk...' button

Click 'Browse...' and navigate to where you unpacked your zipped driver files. Navigate down to the Graphics folder and select igdlh64.inf. Click Open.

It will install the drivers and your screen may flash a couple of times.

When complete, reboot. I know it doesn't say to do this but do it anyway as this is where I was going wrong.

You will now see the Intel HD Graphics Icon in your system tray. You can also get to it through the control panel. Right click on it and select properties if opening form the system tray. And Intel(R) HD Graphics Control Panel should have opened up.

Click Display and then select the 'Custom Resolutions' option on the left. A warning pops up so click yes as we know exactly what we are doing, don't we?

Remember to select you external monitor in the 'Select Display' drop down menu on the left!

Set the width as 3840, the Height as 2160, the refresh rate at 50Hz and the Timing Standard as CTV-RB. Finnaly, click on the '+' add button on the bottom right.

A side about refresh rates: I can only get 50Hz on my monitor (a Samsung U28D590). You must enable DP1.2 in the monitors menu. You may find the gui will only let you select 30Hz or 25Hz

Reboot again so Windows knows about this new resolution.

Right-Click the desktop and select Display Settings.

Select Advanced display settings and select your external monitor. Your new 4K resolution is now available in the drop down menu. Select it and click Apply.

You should now be seeing your desktop in full 4K glory!!!
