a. user say:

Windows signs you in to a temporary profile when your regular user account becomes corrupted, which can happen from configuration mistakes (with which I have considerable personal experience) or malware.

The purpose of the temporary profile is to give you an opportunity to fix your corrupted user account or, more likely, to create a new user account, copy the files from the old to the new, and then get rid of the old user account. The temporary profile is not supposed to be your regular user account, so it doesn't work like a regular user account.

If you find yourself signed in to a temporary profile:

  • create a new user account;
  • copy your files and folders from the old user to the new user (tip from the field: don't forget anything); and
  • delete the corrupt old user account.

If you find this happening frequently, you need to figure out what it is you're doing wrong that corrupts your profile. Self-knowledge goes a long way with a computer.

Original Title: Windows live movie maker

considerate_guy say:

You've apparently associated the .mp4 extension with opening up Movie Maker for editing.  It sounds like you'd rather have it just immediately playback your .mp4 file in a media program.  All you need to do is right-click on an .mp4 file and choose 'Open With'.  Choose Windows Media Player, VLC, or whatever media program is your choice and then make it the default for the .mp4 extension.

As for the preparation time in Movie Maker, you can go to File > Options and there is a setting to tell Movie Maker to skip the preparing step.  That's assuming you *do* want to edit the clip and not just play it back.

Hi Jerry,

To resolve this issue, we suggest that you perform Startup Repair using an installation media. If you do not have a Windows 10 installation USB or disc, you can download and create one from the Microsoft software download website. To create an installation media and use it to repair your computer, follow the steps below:

  1. On a working PC, go to the Microsoft software download website.
  2. Download the media creation tool and then run it.
  3. Select Create installation media for another PC.
  4. Choose a language, edition, and architecture (64-bit or 32-bit).
  5. Follow the steps to create installation media, and then select Finish.
  6. Connect the installation media you created to your nonfunctional PC, and then turn it on.
  7. On the initial setup screen, enter your language and other preferences, and then select Next. If you're not seeing the setup screen, your PC might not be set up to boot from a drive. Check your PC manufacturer's website for info on how to change your PC's boot order, and then try again.
  8. Select Repair your computer. On the Choose an option screen, select Troubleshoot > Advanced options > Startup Repair.
