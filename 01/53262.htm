    Hello quiltcoast,

    1.     Have you checked if the ‘Esc’ key works when you connect the computer to a different keyboard?

    To add to what 'Mike Hall MVP' says, if this is a new keyboard and the ‘Esc’ key is not working, I suggest that you connect the keyboard to a different computer and check if it works fine as this could be caused by a faulty keyboard.

    If you have made sure that you have installed all the correct drivers for the keyboard, I suggest that you check if the ‘Esc’ key works fine in ‘Safe Mode’. Here are the steps to boot the computer to ‘Safe Mode’:

    Follow the below steps to boot the computer to ‘Safe Mode’:

    a.     Restart your computer if it is powered on.

    b.     Tap the F8 key after your computer initially powers on.

    c.      Once you see the Advanced Boot Options menu you can stop tapping.

    d.     Use the up/down arrow keys to highlight your selection.

    e.     Select Safe Mode with Networking and press Enter.

    f.       Logon to your computer using an account with Administrator privileges.

    Link you may refer to:http://windows.microsoft.com/en-US/windows7/Start-your-computer-in-safe-mode

    If it works fine in ‘Safe Mode, the issue could be caused by some of the third party services in the background, I suggest that you set your computer to clean boot and check if you have the same issue.

    a.     Click Start, type msconfig in the Start Search box, and then press ENTER.

    b.     If you are prompted for an administrator password or for a confirmation, type the password, or click Continue.

    c.       On the General tab, click Selective Startup.

    d.      Under Selective Startup, click to clear the Load Startup Items check box.

    e.     Click the Services tab, click to select the Hide All Microsoft Services check box, and then click Disable All.

    f.        Click OK.

    g.      When you are prompted, click Restart.

    After the computer starts, check if the issue is fixed, then follow the steps mentioned in the following KB article to narrow down the exact source:

    http://support.microsoft.com/kb/929135

    You can also set the computer to normal startup once you determine the service causing the issue. See the section on how return your computer to a Normal startup mode by following the steps under “Reset the computer to start as usual” in the above article.

    I decided to create a new post because none of the proposed fixes worked for me. I knew that my routers were fine after testing it on windows 2000, XP and Vista

    I'd like to share a problem that i had recently with windows 7 and wireless connectivity.

    Over a period of 3 months more or less ive been trying to get wireless dongles to work on a couple of machines. I wont name the specifics in the hardware because i think its irrelevant. Routers used were linksys,netgear,dlink and buffalo. Wireless adapters used were senao, zyxel and dlink.  ( Atheros, Ralink and Realtek chipsets ) ( 3 units of each model just to make sure its not hardware related issues )

    The problem i had was whenever i installed any of these dongles on any pc with windows 7 (any version) it never worked. The dongles installed fine but there was always an issue with "an unexpected error ocurred" or "no wireless networks detected". I tried various drivers,fixes, websites including this one etc. over the 3 month periiod. Ive always installed the windows 7 drivers (and even vista drivers as a test) because the dongles do not autoinstall.

    What i found near the end of this whole endeavour was not to install any wireless drivers at all. My experience was that if i installed any manufacturer driver and it didnt work that i would have to reinstall my OS anyway. The solution to my problem eventually was to ONLY install the "windows update" driver. This method has been working flawlessly on various systems eversince!

    I tested this on windows 7 home premium, pro and ultimate (all 32 bit). Clean install with manufacturer driver and no go. Clean install with windows update driver and it works perfect. I havent found a fix yet except for reinstalling the OS.

    I hope microsoft comes up with some kind of fix because i dont want to reinstall my 2 pc's at home hehehe.

    Hope this post might help someone.
