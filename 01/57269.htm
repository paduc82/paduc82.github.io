Solution for Surface Book 2 unable to turn using Power Button - Microsoft Community

jimmewong say:

Solution for Surface Book 2 unable to turn using Power Button

Hi All,

There was a Windows update on my SB2 on Thursday 13/9 and the following day, the same symptom unable to power up using the button occurred again.

I followed the instruction I posted few months ago and fortunately solved the problem. Seems like this problem is still intermittent and Microsoft didn't fix this permanently.

Anyway, I'm posting this again as I believe, there will be more people facing this issue and looking for solution. I'm not so sure whether these are the right procedures but it works!!!

Below are the solution:

1. Download latest Surface Book 2 firmware from here https://support.microsoft.com/en-us/help/4023482

There are 4 files to choose from. I selected SurfaceBook2_Win10_17134_1802209_1.msi assuming this is the latest version number. Mine is 15" SB2.

2. Double click the file downloaded. During installation, prompted option to Change, Repair and Remove. Only Remove option is available to be selected. Removed previous firmware installation and after that run the downloaded firmware again. This time the installation just proceed without prompting for Change, Repair and Remove.

After install and restart with the new firmware, Surface Book 2 able to turn on as normal by pressing the power button once. However, there is another problem occurred; Detach icon is in RED with X and stated "Harware Failure". Previously I'm able to detach the screen with keyboard Detach button but this time I'm unable to do that. Reboot few times....RED icon still appears.

3. Force restart solved the Detach RED icon issue.

Perform force restart:

i.  Hold the Power button for 30 seconds and then,

ii.  continuing to hold down power, press the volume up side of the volume button for 15 seconds.

iii. You will see some unfamiliar Surface Setup Page. Don't change anything.

iv. Select exit and save the setting.

v. Once Windows 10 restart, Detach icon become normal.
