richard diver say:

My system has 128GB, so I thought would take a look at my install to compare to yours and see if I could managed 32GB:

C:\Program Files = 2.3GB

C:\Program Files (x86) = 1.9GB

C:\ProgramData (hidden) = 1GB

C:\Windows = 18.3GB

Recommended 10% free space = 3.2GB (10% of your 32GB)

Minimum Windows Install: 26.7GB

Available space: 5.3GB

Now that does depend on which programs you install, and how often you clean up the system files after updates etc., But, then you have your User Data - even if you keep most of it in the cloud, you definitely need more storage locally for those times when you need offline access.

I'd suggest 64GB is a bare minimum for a useable system on full Windows 7, 8.1, and most likely for Windows 10 also. 128GB provides good breathing space (I run virtual machines in Hyper-V on my machine and have 20GB free space).
