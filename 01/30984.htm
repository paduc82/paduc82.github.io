lemp say:

There are two problems here.

If you leave all the printers connected, you can assign them unique names as described below.

However, if you disconnect a printer and then reconnect it to a different USB port, Windows probably will detect it as a "new printer" and install it.  Part of the installation process assigns a name based on the make/model of the printer, e.g., GoDex G500.  If a printer with this name is already installed, Windows adds "(copy N)" to the name.  I'm sorry, but I don't know any way to keep this from happening (other than not disconnecting and reconnecting the printers)

To rename a printer: 

Go to Devices and Printers

Right-click on the icon for a printer and select "Printer properties"

Enter the name you want on the "General" tab and click OK
