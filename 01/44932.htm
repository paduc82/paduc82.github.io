Hi Dejan,

I would like to inform you that if you are using proxy, you will not be able to access Modern User Interface Apps and Windows store. So in order to access Modern User Interface Apps and Windows store you need to disable the proxy settings on the computer.
 
Follow steps to disable proxy.
a. Go to Control Panel from start screen by pressing Windows key + X
b. Now click on Internet Option.
c. Under the Internet Option dialog box click on Security tab.
d. Under the Security tab click on Custom level button.
e. After clicking you will get the Security settings dialog box.
f. Under that find the Proxy connection.
g. Now check the radio button says Disable.
h. After that click on Apply and then OK button to confirm change.
This way you can disable Proxy Connection.
 
Hope this helps.
