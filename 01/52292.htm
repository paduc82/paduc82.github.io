mehdibens say:

Hello,

I think Windows Defender does a decent job in protecting you computer from the most prevalent and widespread threats. Combined with Windows Firewall, you also get a protection from online unauthorized access to you computer and LAN protection from local networks attacks (like in open wifi zones). However, Windows Defender scores low in many AV tests given that these tests really use very sophisticated malware that are zero-day malware (Windows Defender scores very low in this category). Normal users do not typically encounter those creepy malware. Also, Windows Defender does not provide strong protection against ongoing exploits (some other security products can detect exploit activity on vulnerable apps and prevent it), so make sure you keep all you software updated, especially Windows, as Microsoft continuously provide new updates to secure you system.

Now, you are the one to decide if you need an additional security software or not. I just can give you some points to consider while deciding.

1- Are you a cautious user? This mean, for example, do you easily recognize suspicious websites or downloads or not. Or can you spot signs of infection?

2- How often do you shop online using your banking information?

3- Do you have very important data in your computer (this may sound silly :) )?

4- Do you install infamous software often?

5- Do you mind running MalwareBytes scans weekly?

6- etc...

Just understand that if you want premium protection, Windows Defender is far from being the best product, however, it provides decent protection against the most common threats. Advanced security products often offer advanced cloud protection, very impressive detection of zero-day malware (very recent unknown threats), automatic prevention from exploits, protection from fishing websites (higher than the smart screen filter)...

You choose, I think you are the only one to know if you need high protection or only a basic one.

I used Windows Defender in Windows 8/ Windows 8.1 for quite some time and it did a great job protecting my system until once I realized I had a rootkit that it did not detect. I am now using another security suit.

I hope this helps.
