a. user say:

Your idea is a good one, however the default Documents folder needs to be on an internal drive.

You can certainly create Documents folders on an external drive for backup purposes, but the default save location has to be an internal drive.

You can understand why it has to be this way: the default save location needs to be accessible to Windows at all times, and an external drive can be removed, even if you never actually remove it.

pappacracker say: Thank you for your response. I should have explained it a bit more. I bought a solid state drive. I am going to switch the mechanical hard drive with the ssd and use the ssd as the system drive and the mechanical hard drive as a secondary one. Both will be internal. Will I be able to create the 4 folders as stated earlier on the secondary drive? Thank you BulldogXX.

a. user say:

Yes - that is exactly as I have done with my users.

You can move these folders: Desktop, Documents, Downloads, Music, Pictures and Videos. Moving those folders is supported by Microsoft; you can do it from the Settings app or from the individual folder properties (the Location tab.)

Moving the remainder of a user's profile is not supported and I do not recommend you do this.

Two tips:

1- Even after you move these folders, you will still see (empty) folders with the same name in their original locations on the C: drive. This is done for backwards compatibility with older software that is hard-coded to look for these folders in the C: drive.

2- When you move folders to a new drive they automatically inherit the access permissions of the root folder, which in your case is J:\. This may lead to unexpected file access problems. I changed the access permissions for the root folder and the data folders to match the way they were when they were on the C:\ drive and you might want to do this as well.

Be very careful when you change access permission! Read about this on the web to learn all the details and prepare your access settings on paper before you start. Making the wrong move on folder access permissions can lock you out of your own folders. (Please don't ask me how I learned this lesson.)

a. user say:

The best way to backup Windows is with third-party backup software. Windows' two backup utilities are actually not best for this task.

The first Windows backup utility is called File History. It is intended to make copies of individual files, and while it works well enough doing that, you need more functionality than File History can offer. It was included with Windows to provide a very basic backup for people who are anxious about installing a third-party program.

The other Windows backup utility is called System Image Backup, which was designed to provide a very basic image backup capability for the same group of people. In addition to lacking important features commonly available in third-party software, Microsoft has deprecated System Image Backup, which means that Microsoft is no longer developing SIB and recommends that we use a third-party program instead.

There are so many terrific third-party backup programs, both free and paid, that it's easy to see why Microsoft didn't feel the need to re-create the wheel. I'll give two examples below, but here are the key features you are looking for in a third-party product (which neither of Windows' own backup utilities provide:)

  • create and save any number of different backup jobs, and run them with one click or tap;
  • easily schedule backups to run automatically at a time you choose - my backups run overnight while I'm sleeping;
  • set a retention schedule to tell the program how many old backups to keep - for example, you might tell the program to keep the last 7 backups or the last month of backups;
  • receive an email when a backup fails (or succeeds!) with complete details; and
  • (for an image backup,) restore the image to different hardware, e.g., a new computer

among many other useful features.

A good thing to do is to run a search for: best Windows backup software 2018 . The search results will include a ranking and review of the best backup programs, in the opinion of the reviewer.

Read the reviews from professional organizations, take advantage of the free trials and make a choice that feels good for you. Two good examples of backup software are Macrium Reflect for making image backups, and Backup4All for making file and folder backups.
