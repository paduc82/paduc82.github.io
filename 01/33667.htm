Hi RG. I'm Greg, an installation specialist and 9 year Windows MVP, here to help you.

What did you do exactly in Safe mode? How did you enter it and leave it?

If you updated drivers that were problematic then this could be the problem. You should go back into Safe Mode to undo what you did, of update driver to one you know which works, then see if Windows will Start Normally.

Safe Mode doesn't cause problems, it's only a refuge if Windows cannot start - normally because of drivers, which is why Safe Mode works since it doesn't load them all - so if you did something in Safe Mode and now Windows cannot start, try first to undo it in Safe Mode, then if not try accessing Repair Mode to do a System Restore to before the changes were made: https://www.tenforums.com/tutorials/2294-advanc.... If accessing Safe Mode or doing a System Restore won't any longer work then do a Reset saving files.

You may need to create bootable media to access Repair Mode or even do the Clean install:

To create Windows 10 Installation Media on another PC install Media Creation Tool and follow the directions here: http://windows.microsoft.com/en-us/windows-10/m.... Uncheck the box for Recommended Settings to choose the exact version and bit rate for the target PC only.

Insert media, boot it by powering up PC while pressing the BIOS Boot Menu Key: https://www.sysnative.com/forums/hardware-tutor...

If the media won't boot you may need to enter BIOS/UEFI Setup (pressing key given in chart in link above) to turn off Fast Boot or Fast Startup first.

Choose the boot device as a UEFI device if offered, on second screen choose Repair Your Computer, then Advanced Troubleshoot Options, then System Restore. If that fails go back to Troubleshoot Options to try a Reset.

If that fails choose Install Now, then Custom Install, then at the drive selection screen delete all partitions down to Unallocated Space to get it cleanest, click Next to let it create needed partitions and start install - this makes it foolproof.
