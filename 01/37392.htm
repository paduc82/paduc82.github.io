I have been in a similar boat many times. The system boot disk is actually the DOS system disk. It appears the master boot record of the HD (hard drive) is messed up and may indicate a bad hard drive OR the DOS system just needs to be put back on the HD. To do this, you will need a DOS boot diskette (or CD if your computer will boot from a CD). Simply boot the computer from the original boot floppy disk. Then you can check to see if the hard drive is still there and/or the files are still there. 

The file you need is sys.com and should be on your hard drive root folder. This is where it can get tricky. You need to know EXACTLY what version of DOS is installed on the computer. Type VER at the command prompt, and it should give you the DOS version. Presumably, it is most probably version 6.22. Look at the file date for SYS.COM on the floppy disk and compare it to the dates of the sys.com on the hard drive. Also look for IO.SYS and COMMAND.COM. The dates should be the same or very close (around May 1994)  If this is true, you can safely assume that the floppy system disk matches what is on your hard drive. 

At the A:\ [prompt], TYPE sys c: [enter]

it should say:

'System Transferred'

Remove your floppy and reboot your old computer.

Good Luck,

You upgraded to Windows 8, but didn't have a previous version of Windows installed (error 0xC004F061)

If you see error 0xC004F061 when you try to activate Windows 8, it means that you're using a product key for an upgrade version of Windows 8 and a previous version of Windows wasn't on your PC when Windows 8 was installed. To install an upgrade version of Windows 8, you must already have Windows 7, Windows Vista, or Windows XP installed in your virtual machine (parallels). 

If you formatted the drive before the upgrade version of Windows 8 was installed, you won't be able to use your upgrade product key to activate Windows 8. To activate Windows 8, you'll need to install your previous version of Windows, and then reinstall Windows 8. For help with the activation process, contact support.
