There one important thing in that what Sujan already said - you need to open that website using Internet Explorer. As far as I know url shortcuts use cache of Internet Explorer. If they manage to find that specified website was previously opened with Internet Explorer they can use remembered icon. Url of the shortcut and url of previously opened website with IE must be identical.

or...

Sujan already said that you have possibility to change icon using shortcut Properties. This is true but then you need to point out icon on your hard drive. I guess you are interested to display icons of certain pages like for CNN: or for MSNBC: instead of system icons. First you need to download these icons on your hard drive and then you can right click on the shortcut > Properties > Change icon... > point out where you saved downloaded icon.

You are wondering how to download these icons? There is one rule.. it should usually work. I will show you using examples:
  • url for CNN is http://www.cnn.com/ and icon (favicon) is located here: http://www.cnn.com/favicon.ico (you just need to add favicon.ico to default url). You can open http://www.cnn.com/favicon.ico (it should display icon) > right click on it > choose Save image as. And now you have an icon on your hard drive and you can use it for your shortcut
  • situation looks similar in MSNBC.. default url is http://www.msnbc.msn.com/ and the icon you can find here: http://www.msnbc.msn.com/favicon.ico
