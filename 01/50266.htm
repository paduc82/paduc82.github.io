davids kruze say: Ok, so I've found a temporary workaround before the Drivers get signed.

1. Find another HDD to put your Windows 8 on and connect it to the motherboard,
    not the RAID Controller.

2. Make sure you have enabled the AHCI. (EXTREMELY IMPORTANT)

    2.5. If you already have Windows 8 installed using IDE,
           follow this guide and pay close attention to the second reply:
           GUIDE
           (skip step 3)

3. Install Windows 8 like you normaly would on your Temporary HDD.

4. After installing, connect and install your RAID Controller
    with the available drivers for Windows 7.

5. After restarting your PC for a couple of times, you should be able to see your
    RAID array inside the Windows 8. Download and open a program of your choice
    for duplicating the HDD/Partition.
    I used a free software called EaseUS Backup Free: 
    Download Here

6. Now, you should go ahead and clone the HDD/Partition (I would suggest to use the Partition
    Cloning). Use your current Windows 8 Partition as the source and select the RAID 
    HDD/Partition as your Destination. Make sure you check the Sector by Sector Clone.

7. After Cloning is done, shut down your Computer and pull out the temporary HDD only leaving the
    RAID Array. After that, you should be able to boot into previously cloned Windows 8 with your
    RAID Array.

Note. If you have more than one Partition on your RAID Array or 
         Temporary (Existing) Windows 8 HDD, it should be ordered like this:

                 Temporary (source) Windows HDD > C:\
                 Second Partition on the Windows HDD > D:\
                          Destinaion Partition on the RAID > E:\
                          And the other RAID partitions should be F,G,H.......

         This order is important if you want your Windows 8 Partition to boot as C:\ after
         you pull out the Temporary HDD. Hopefully you get the idea :)


I'm pretty sure that this guide applies not only to this particular RAID Controller but also others with similar problems.
