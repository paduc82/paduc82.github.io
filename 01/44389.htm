The Upgrade-Tool should not even look at files that are stored on the secondary drive, but it does.
Or maybe it checks the Windows Search Index and does a simple but stupid check of every file against a list of known files that are incompatible...

For some ridiculous reason, the installer found these drivers (seven different versions for different computers)
in the archive on the second drive while doing the compatibility check.
