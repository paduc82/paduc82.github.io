If the issue persists, proceed to Method 2.

A clean boot is performed to start Windows by using a minimal set of drivers and startup programs. This is to check if some File Explorer add-on is interfering with the smooth functioning of explorer.exe. To perform a clean, kindly follow the steps provided in this link.

Powershell is a command-line tool to configure the system settings and automate Windows tasks.

  1. Press the Windows Key, and then type powershell.
  2. Right-click Windows Powershell (Desktop App) and select Run as administrator. Select Yes in the UAC window that pops up.
