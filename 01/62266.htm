lemp say:

You have no idea what may be lurking on that computer, and thus you should either do a clean install of Windows 7 (for which you'll have to obtain a Windows 7 installation source) or reset the computer to its "factory state."

Dells come with a means to do the latter, so try that first.  See the directions here: 

https://www.dell.com/support/article/us/en/19/sln291879/restore-or-factory-reset-windows-7-on-your-dell-computer?lang=en

If you restore a computer to its factory state, it will be just as it was when it was first purchased.  You will need to do the following things:

Remove all free trials of any antivirus program using the appropriate removal tool supplied by the antivirus company.  There's a list of these here:  https://support.eset.com/kb146/?locale=en_US  Download the appropriate tool using another computer (such as the one you're using to ask your question here) and save it on a USB flash drive.  While you're at it, don't forget that you'll have to install your own antivirus program now.

If the laptop didn't come with Windows 7 Service Pack 1, you'll have to install that.  Go here and again download and save to a USB flash drive:  https://www.microsoft.com/en-us/download/details.aspx?id=5842  There are several files at that page.  You want windows6.1-KB976932-X64.exe if you have a 64-bit system and windows6.1-KB976932-X86.exe if you have a 32-bit system.

After Service Pack 1 is installed, reboot the computer twice.  Then follow the instructions in the first post in this long thread (the one that says "Update Dec. 4, 2016"):  https://answers.microsoft.com/en-us/windows/forum/windows_7-update/windows-7-update-solution/f39a65fa-9d10-42e7-9bc0-7f5096b36d0c

At this point, you can begin using the computer.  You will have to install any programs you want to use.  If you want Microsoft Office (Word, Excel, Powerpoint), you'll have to buy that separately.  Again, don't forget to install an antivirus program.

Error code 0x80070005

Symptom
When you try to use a KMS host, a KMS client, or a MAK to activate one or more computers, you may receive the following error message:

0x80070005
Access denied the requested action requires elevated privileges.

Cause
This issue occurs if User Access Control (UAC) does not let the activation processes run in a non-elevated command prompt.
Workaround
To work around this issue, follow these steps:

   1. Click Start, click All Programs, click Accessories, right-click Command Prompt, and then click Run as.
   2. Click The following user, and then select Administrator in the User name list.
   3. Click OK.
   4. At a command prompt, type the following command, and then press ENTER:
      slmgr.vbs
