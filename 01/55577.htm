Win7 when configured on peer-to-peer Network has three types of Sharing configurations.

HomeGroup Network = Works only between Win 7 computers. This type of configuration makes it very easy to Entry Level Users to start Network sharing.

 Work Network = Basically similar to the previous methods of sharing that let you control what, how, and to whom folders would be shared with.

 Public Sharing = Public Network (like Internet cafe) to reduce security risks.

 For best results log to each computer's System Screen and set all the computers to be on a Network with the same name while each computer has its own unique name.

http://www.ezlan.net/Win7/net_name.jpg


Make sure that the Software Firewall on each computer allows free local traffic. If you use 3rd party Firewall On, Vista/XP Native Firewall should be Off, and the active Firewall has to adjusted to your Network IP numbers on what is some time called the Trusted Zone (consult your 3rd Party Firewall instructions

General example,  http://www.ezlan.net/faq.html#trusted

Please Note that some 3rd party Software Firewalls keep blocking aspects of Local Traffic even it they are turned Off (disabled). If possible configure the Firewall correctly, or totally uninstall it to allow clean flow of local network traffic. If the 3rd party software is uninstalled or disables, make sure that Windows native Firewall is On .

How to fix Critical Error on Windows 10 - Microsoft Community

a. user say:

How to fix Critical Error on Windows 10

Hi everyone! Today, I am going to explain how to fix Critical Error on Windows 10. Critical Error happens for me approximately every 1 or 2 months and it can last up to 2 weeks. The Critical Error sign only shows when your Start Menu and Cortana are not working. You also cannot use your taskbar when you have Critical Error. On the Critical Error sign, there is an option to sign out of your computer. There are several ways to fix this.

Option 1:

Step A: Using your keyboard, hold Ctrl+Alt+Del at the same time.

Step B: Some security options will then load. There should be 4 options.

Step C: At the bottom-right corner, there will be a power (on) button. Click that.

Step D: Then select "Restart" and then your computer will start to restart. This means it will turn off and then come back on.

Step E: Your Critical Error should now be gone and your computer should now be back to normal.

Option 2:

Step A: You need to create/copy a (new) user account by doing the following;

  1. Press Windows + X keys on the keyboard, select Control Panel.
  2. Select System and Security and then System.
  3. Click Advanced System Settings.
  4. Under User Profiles, click Settings.
  5. Select the profile you want to copy.
  6. Click Copy to, and then enter the name of, or browse to, the profile you want to overwrite. This will usually be in C:\Users\username, where username is the name of the profile to which you are copying.
  7. Click Permitted to use, and then add all user accounts to which you want to give full profile control.
  8. Click OK.

Step B: This problem should now be fixed and Critical Error should be gone now.

Option 3:

Step A: Click "Sign Out" on the blue Critical Error sign.

Step B: You will now be on your login screen.

Step C: Click the power (switch on) button.

Step D: Select "Restart" and your computer will now start to restart.

Step E: When your computer comes back on, Critical Error will now be gone.

I hope you have enjoyed reading this and I hope it had helped you. Please comment with feedback!
