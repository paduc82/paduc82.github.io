robert aldwinckle on forums say:

all of my games function perfectly in safe mode  

For another tack, if they work in safe mode that means that they are not modern apps which means there is something in that machinery which could be causing the lag.  One way to cause a related test then would be to kill explorer.exe when you want to run your games.  You can use Task Manager to kill explorer.exe and then use Task Manager instead of the normal Desktop for switching or starting other programs.  Another alternative would be to have a command shell (e.g. cmd.exe or powershell.exe) open to switch to which would facilitate starting other programs.  You could even try having a File Explorer window open in that scenario, since that instance of explorer.exe would not be used for any kind of task management.

I just thought of another possible test:  change the dispatching priority of the task you want to run the most.  E.g. use Task Manager to set your game task's priority to Above Normal.  I doubt doing that would have as dramatic effect as you saw by running in safe mode but it may help smooth out the task switching enough that you can then run both kinds of tasks acceptably at the same time.  Unfortunately we are not allowed to go the other direction--reducing the priority of modern apps (or their machinery) to see if that is enough to let more conventional tasks run with less interference.

Heh.  I just noticed that we have already decided that the OS has more to do with your problem symptom than drivers based on the results of your test using a virtualized driver with a longer processing path!  (and I still "don't know how we could find out what the OS is doing in either case. ")

BTW another possible explanation for your problem symptom would be "thrashing".  That could be mitigated by strictly controlling how much else there was to do or how much RAM you have to prevent switches involving Swaps or page faults.  Again, we would need more information about how to measure performance for such factors, something that I only have a vague inkling of but something that I think could be done using normal tools.
