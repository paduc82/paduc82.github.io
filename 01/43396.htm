    Create a restore point before messing with this. You didn't specify what you wanted to remove/hide. You'll note that the computer's drives and mapped locations show up in the "send to" menu You can easily add a folder or drive to that menu by creating a "drive" map. You can disconnect mapped drives to remove them from both Explorer and the Sendto menu.

    Shortcuts in Sendto can be deleted - namely the Fax and Mail Recipient items. Better yet, you could move them somewhere else for safe keeping. The "Documents" links to the Documents library, and items sent to "documents" will end up in the default save location of that library (see library properties).

    So if you really really want to delete "Documents" you need to delete that library. "Compressed (zipped) folder" is best just marked as hidden, if you just don't want to see it. This link would be hard to recreate if you deleted it. Note that hiding Documents won't work out the same as you would hide the items inside the default save location - probably not what you wanted to do. Renaming the library won't help either. The "Send to" directories are stored in:

     C:\Users\Default\AppData\Roaming\Microsoft\Windows\SendTo C:\Users\YOURPROFILENAME\AppData\Roaming\Microsoft\Windows\SendTo

    see http://www.winhelponline.com/articles/98/1/How-to-Add-Items-to-the-Send-To-Menu-in-Windows-Vista.html to add items to Sendto

    Vista Send To Power Toy: http://www.winvistaclub.com/d61.html This lets you easily select which items to show(or not) on the sendto right click menu. Yes it was written for Vista, and no I haven't tried it on Windows 7 but you might like to try and report back to the forum.

    I would first try reinstalling Windows 8 on the system.

     

    New computers that come preinstalled with Windows 8 often have what is called a recovery partition. This is used to reinstall the operating system in the event of a system crash. To access it, you will need to boot into when you start your computer by pressing a function key. This can be either F1, F2, F9, F10, F11, F12 or even DEL or Tab key. Consult the owners manual that came with your PC for instructions about how to reinstall Windows.

     

    This is how the recovery partition is accessed for the most popular makes..

    For Dell, press CTRL + F11 directly after powering up the unit

    For HP, press F11 directly after powering up the unit

    For LG, press F11 directly after powering up the unit

    For Toshiba, press and hold '0' BEFORE and during the power up

    For Acer, press and hold ALT + F10 as soon as you see the logo

    For Asus, press F9 as soon as you see the Asus logo.

    For Samsung, press F4 at power up..

    For Fujitsu, press the F8 key repeatedly directly after powering up

    For Advent, restart your computer. Then Press F10 repeatedly until the message "Starting System Recovery" is displayed

    For Sony VAIO, restart and press "F8" or "F10" repeatedly until the "Advanced Boot Options" screen appears.
