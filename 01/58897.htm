microsoft update has disabled all MS start button apps MS10 - Microsoft Community

igilmour say:

microsoft update has disabled all MS start button apps MS10

15/June/2017 MS update today has disabled all my start button MS apps. Click and a screen opens for around 5 - 10 seconds then closes and back to desktop anyone else had this or know how to fix it...

***Post moved by the moderator to the appropriate forum category.***

igilmour say:

Hi Thanks for replying  However since I posted last I have managed to find a solution to the problem.

My problem was that after receiving a substantial update via MS to my Windows 10 system I found I could not open any of the MS apps via the tiles on the start pane.(inc: Store, news,photos,etc. It would flash up a blank app screen and then timeout after a few seconds and throw me back to desktop. Also I could not save files in either MS Word or file manager any attempt would result in a yellow warning pane that said "No more files "  However, I found a post on line that suggested that on ASUS machines a program called ADSM (Asus data security manager) that comes with the machine could conflict with some MS updates. I therefor stopped the program in task manager and then deleted it via control panel and rebooted. This entirely corrected the problems I was having. Therefor a conflict in the latest MS update and a native Asus program was the problem. ADSM is not required or essential so if you encounter a similar problem on other ASUS machines this is the solution.

How to always show enhancements toolbar in latest Windows Media - Microsoft Community

usp say:

How to always show enhancements toolbar in latest Windows Media Player?

I made the switch from XP to Win7 64-bit and I would like to get WMP to behave like the prior version where the play speed settings were always visible after opening WMP. I can't figure out how to enable this visibility option in the latest version. In the current version I have to open the enhancements menu, select the toolbar, then rearrange it on the screen to make it visible, then adjust the speed.

Any clues on how to solve this so that I see the toolbar automatically whenever I open the program? Thank you.
shekhar s say: Hi usp,

I am afraid to say that you shouldn't find this option opened by default as it was in the previous versions of Windows Media Player. And there is no workaround to get it opened by default as it is by architecture of the software.
 
However there is an improvement in the same, instead of the default toolbar that we had earlier we have a arrow below the Now Playing tab from where you can browse the enhancement option.
Here is how:
1. Begin playing a file.
2. Click the arrow below the Now Playing tab, point to Enhancements, and then click Play Speed Settings.
3. Move the Play Speed slider to the speed at which you want to play the content, or click the Slow, Normal, or Fast link.
4. To select speeds between the labeled play speeds, in the Enhancements pane, clear the Snap slider to common speeds check box.
5. To hide the settings, click the Close button in the Enhancements pane.

For more information you may refer to the articles below:
http://windows.microsoft.com/en-US/windows7/Change-playback-speed-in-Windows-Media-Player
http://windows.microsoft.com/en-us/windows7/Getting-started-with-Windows-Media-Player

Hope it helps.



Thanks and Regards:
Shekhar S - Microsoft Support.
