a. user say:

Hello Clare,

There are several methods that we can perform to pause the download for Windows Update. However, please note that pausing or stopping Windows Update is not recommended unless you want to pause Windows Update for a short while. Here are the methods:

Method 1: Pause Windows Updates via Setting app

(Note: This feature is available in Windows 10 build 14997 or later builds only.)

  1. Open Settings app. Click Update & security. Click Windows Update.
  2. Under Update settings page, you should see Advanced options link. Click Advanced options hyperlink to open Advanced update settings page.
  3. From there, turn on Pause Updates to pause updates that are being downloaded. When you turn on Pause Updates, Windows 10 will pause currently available updates up to 35 days.

Method 2: Pause Windows Update downloads via Command Prompt

  1. Right-click on the Start button and click Command Prompt (Admin). Click Yes button when you see the User Account Control prompt.
  2. Type the following commands one after another and don‚Äôt forget to press Enter key after pressing each command.

          net stop wuauserv (Might take a while to stop the Windows Update service.)

          net stop bits (Stops background intelligent transfer service.)

          net stop dosvc (Stops delivery optimization service.)

      3. Windows Update will/should now pause.

           Resume Windows Update downloads

  1. Open Command Prompt with admin rights.
  2. Execute the following commands to start all stopped services and resume Windows Update downloads.

          net start wuauserc (press Enter key)

          net start bits (press Enter key)

          net start dosvc (press Enter key)

      3. Navigate to Settings app > Update & security > Windows Update. You should see Windows Update resuming downloads. If not, wait for a while.

Important notes:

  • If Windows Update is not checking for updates after resuming Windows Update, you might need to clear Windows Update cache to make Windows Update check for updates again and begin downloading available updates all over again.
  • Use Windows Update Troubleshooter to fix issues related to Windows Update.

Feel free to post back if you need further assistance.
