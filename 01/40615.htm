Windows 8.1 Create a Recovery Drive Image - Microsoft Community

keithss say:

Windows 8.1 Create a Recovery Drive Image

xircal say: keith,

Normally, when you create a Recovery drive for troubleshooting purposes, it will only be 512MB allowing you to use a low capacity USB pen drive. There is an option to copy the data from the laptop's own Recovery partition, but if you go that route, you'll only be duplicating the one which is already installed on the machine.

See this article on how to create a Recovery drive: http://blogs.technet.com/b/ouc1too/archive/2013/10/22/windows-8-1-upgrade-reboots-to-black-screen-of-nothingness.aspx

Take note though that the link provides instructions on how to troubleshoot a startup issue and therefore it's not intended that you follow those instructions since you don't have that issue.

A system image on the other hand is rather like a screenshot in that it contains the system state as it was when you created it. So for example, if you were to create a system image today and then in six months time you use that to restore the operating system, what you'll get is the system state as it is today. This means you'll have to update Windows with all the security patches, hotfixes etc., which were released between now and six months time.

To create a system image as opposed to a Recovery drive, you'll need a much larger capacity storage unit. That's not a problem these days and even a 500GB external hard drive will only set you back about $60. But it's inadvisable to save a system image to the same hard drive that the operating system is already installed on because if the drive fails, you lose everything.

But as I already mentioned, most laptops ship these days with a Recovery partition which you access with a certain key or key combination when the machine boots up.

You didn't mention the make and model of your own machine, so here's a list of laptops and the key combinations needed to access the Recovery partition and how to use it:
http://www.pcadvisor.co.uk/how-to/laptop/3462995/factory-reset-laptop/

mlg63 say: A Recovery Drive doesn't contain a system image, just the tools to access a system image on an external hard drive or the factory restore partition on your hard drive. To make a System Image--also a compressed file--go to File History to create it on an external hard drive or on the network or a second separate hard drive on a desktop computer. The Recovery Drive can contain the Factory Restore file to return to original factory settings if your computer came with windows 8. If you upgraded to Windows 8.1 from windows 7, skip the factory restore copy because it is no longer usable since it had Windows 7 factory settings.

Be sure to test the recovery drive to make sure you can boot to it and access the Recovery Tools. Cancel and backout of it before restoring anything.

  1. Start Registry Editor by clicking Windows Start Button, type regedit and press Enter.

  3. On the Edit menu, click New>Key

  5. With newly created key still selected, click Edit>New>Multi-String Value
  6. Newly created value should be selected and waiting for you to type its name. Type VDD and press Enter.
  7. Quit Registry Editor.

    https://support.microsoft.com/en-us/help/403431...

    Explorer Network Browsing

    The Computer Browser service relies on the SMBv1 protocol to populate the Windows Explorer Network node (also known as "Network Neighborhood"). This legacy protocol is long deprecated, doesn't route, and has limited security. Because the service cannot function without SMBv1, it is removed at the same time.

    However, if you still have to use the Explorer Network in home and small business workgroup environments to locate Windows-based computers, you can follow these steps on your Windows-based computers that no longer use SMBv1:
    Start the "Function Discovery Provider Host" and "Function Discovery Resource Publication" services, and then set them to Automatic (Delayed Start). When you open Explorer Network, enable network discovery when you are prompted.

    All Windows devices within that subnet that have these settings will now appear in Network for browsing. This uses the WS-DISCOVERY protocol. Contact your other vendors and manufacturers if their devices still don't appear in this browse list after the Windows devices appear. It is possible they have this protocol disabled or that they support only SMBv1.

    Additional troubleshooting steps you can apply:

    - Try browsing to some of the computers manually, press Windows key + R, type: \\computername, hit Enter
    - Press Windows key + R
    Type: services.msc
    Hit Enter

    1. Start the "Function Discovery Provider Host" and "Function Discovery Resource Publication" services, and then set them toAutomatic (Delayed Start).
    2. When you open Explorer Network, enable network discovery when you are prompted.
