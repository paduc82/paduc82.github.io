Well to start with the RegCure Pro option:
RegCure Pro is a program that edits the Windows registry, the registry is stored on you're harddrive and is an integral part of Windows. 

You've tested your son's laptop with HP's System Quick Test, it indicated that there is a problem with the harddrive in the laptop. 
System Quick Test is a test where the hardware is tested, it checks if the hardware is okay, it does not test if the Windows install is in a normal (working) state.

Since the problem is in the hardware of you're computer (you're harddrive) and not in Windows, there's no registry program that can fix this problem.

According to the HP website: "Testing for Hardware Failures (Windows 8) | HPÂ® Support" there should be a option to test the harddrive more thorougly, by selecting, component tests in the main menu, then clicking harddrive and then click extensive test. 


If this extensive test only contains errors then there isn't hope for you're harddrive as it is, if some test pass then it isn't completely broken.
If isn't completely broken then there is a minute change that booting form a Windows Setup disk or usb disk and repairing the install might help, but don't count on it.

What you could do is run the extensive test and if all of these tests fail you might consider unscrewing the bottom of the laptop and checking if the harddrive is properly connected to the connector (the manual how to was posted in my previous post).
WARNING:
although i've did some googling on the topic and it seems that opening the bottom doesn't void the warranty of the laptop there is a chance that it will.
If the laptop is under warranty, play it safe get it fixed under warranty.



Bottom line is that the laptop needs to be repaired, whether it is just a case of reconnecting the harddrive properly or replacing it.
