lemp say:

Well, that's what happens when making assumptions in order to give a quicker response.

If the D: partition is something that you created and use to store your data, then go ahead and turn on System Protection.  For this data partition, you might consider configuring System Protection to only save "previous versions."  See Fig. 2 here --> http://programming4.us/desktop/1368.aspx

If I were you, however, I'd seriously consider using some external media for backup -- a networked drive, a USB-connected drive, and/or Internet "cloud" storage -- in addition to turning on System Protection. 

System Protection creates "Restore Points" on the same partition that it is protecting.  If the problem is that you inadvertently delete an important file, or if it gets corrupted in some way, then having a "previous version" available right on the same drive is very convenient.  But what happens when the entire drive fails (hard drives do fail, some more so than others)?  If System Protection is your only backup, you're left with nothing.  If you're a digital artist and these files are work-related, don't rely solely on System Protection.
