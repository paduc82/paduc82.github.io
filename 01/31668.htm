How to get a vba user window to appear on top of the Vista Window? - Microsoft Community

cam10 say:

How to get a vba user window to appear on top of the Vista Window?

I have inserted code that was supposed allow a vba win32 window to appear on top of the Vista
window, however, this does not happen.  I am developing on a 64 bit machine running under XP (SP3),
and the end result works fine on a 64 bit machine with XP (actually don't need this code), however when run on a 64 bit machine running under Vista, the user has to "ALT + TAB" in order to see the form.

Here is what I used...

Public hWndSolidWorks As Long
Public hWndOriginalOwner As Long
Public hWndForm As Long
Public Const GWL_HWNDPARENT = (-8)

Sub Main()

    Set swApp = Application.SldWorks
    Set Dwg = swApp.ActiveDoc
    Set swModel = swApp.ActiveDoc
   
    On Error Resume Next

    hWndSolidWorks = swApp.Frame.GetHWnd           'get the handle of the main SolidWorks window
   
    If Dwg Is Nothing Then                                                      ' Is doc loaded
        MsgBox ("No Drawings are loaded!")
        Exit Sub
    Else                                                                        ' Doc loaded?
        FileTyp = Dwg.GetType                                                   ' Get doc type
        If FileTyp = swDocASSEMBLY Or FileTyp = swDocPART Then                  ' Model ?
            MsgBox ("This is not a Drawing File!")
        ElseIf FileTyp = swDocDRAWING Then                                      ' Drawing ?
            Load frmProjProps                                                                ' load the form first 
            hWndForm = FindWindow(vbNullString, frmProjProps.Caption)           'get the handle of the userform
            hWndOriginalOwner = SetOwner(hWndForm, hWndSolidWorks)              'set the SolidWorks window as the owner
          ' of the userform and return the handle of
          ' the original owner
            frmProjProps.Show vbModal                                                   ' show the form
        Else                                                                    ' Else doc type
            MsgBox "Active file is not a SolidWorks document.", vbExclamation
        End If                                                                 
    End If
   
    Set swApp = Nothing
    Set Dwg = Nothing

End Sub

Public Function SetOwner(ByVal hWndToUse As Long, ByVal hWndOwner As Long) As Long

    ' NOTE: GWL_HWNDPARENT is a misnomer as it actually sets the owner, not the parent.
    SetOwner = SetWindowLong(hWndToUse, GWL_HWNDPARENT, hWndOwner)
   
End Function
