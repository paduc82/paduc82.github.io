xlapp say: For lots of people having the same problem, the issue seems to be with the Teredo networking adapter. To check if it is that stopping you connecting to Xbox services on PC, open the Xbox app, go to settings on the left menu, go to network and it will test your connection. If you see along the lines of "teredo adapter cannot connect" then you have a Windows service disabled that allows connectivity through Xbox servers. Here is how I fixed this issue for myself (copy pasted from another forum post).

  1.  Press windows key and search for "services", open it.
  2.  Locate "IP Helper" in the list and check if it is running - if not, this is the issue.
  3.  Right click "IP Helper" and go to properties, then dependencies tab.
  4.  Look for each of these dependencies in the services list and check each one is running and set to "automatic" startup type. (The one disabled for me was the "WinHTTP Web Proxy Auto-Discovery service")
  5. Once you find the service that is not running, right click it and press start. If start is greyed out and you cannot restart the process then continue to the next step. If you can start the process, do so - restart your computer and check "IP Helper" is now running, you should be done.
  6. If the process cannot be started with the right click menu, you likely have a registry issue causing it to not function properly.
  7.  Press windows key and search for "Regedit", then run as administrator.
  8.  A quick google search will tell you where to locate the particular dependancy service that is not running.
  9.  Find the default values for the service and change them in the properties. In my case I had to change a "0" to a "2" in the properties of the "WinHTTP Web Proxy Auto-Discovery service".
  10. Restart your computer
  11. Check "IP Helper" is running
  12.  Press windows key and search for "wsreset", run as admin
  13.  Profit! 

Hope this helps at least one other person, was scratching my head for a while on this and have seen many people given up entirely.
