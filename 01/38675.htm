Windws 7 CBS logs and Cab files - Microsoft Community

jackin say:

Windws 7 CBS logs and Cab files

What exactly creates CBS and CbsPersist_20160216121732.cab files on Windows 7 and is it safe to empty this folder and what causes to quickly fillup again?

gerry c j cornell say:

Cbs stands for component based services.

 

System File checker writes the details of each verification operation and of each repair operation to the CBS.log file. Each SFC.exe program entry in this file has an [SR] tag. The CBS.log file is located in the %windir%\Logs\CBS folder.

 

The Windows Modules Installer service also writes to the CBS log file. (The Windows Modules Installer service installs optional features, updates, and service packs.)

 

The CBS.persist.log is generated when the CBS log gets to be around 50 meg in size. CBS.log is copied to cbs.persist.log and a new cbs.log file is started.

 

The logs can be useful for troubleshooting issues. If you are sure your system is running fine, you can delete this file. SFC.exe will create a new one, next time it is run.

 

The System Update Readiness tool writes entries to the CheckSUR.log and the CheckSUR.persist.log. In Windows 8 and 10 the System Update Readiness tool is replaced by the DISM tools and these logs are replaced by a DISM log

Hi,

I can understand the inconvenience this may have caused, but rest assured, we are here to help you with your concern.

To have the Drop Box folder in File Explorer, you have to first download the desktop version of Drop Box onto the computer. Then follow the steps below.

  1. Open a File Explorer window and go to the Dropbox folder location (%USERPROFILE%\Dropbox).

  2. Right click on Favorites and select Add current Location to Favorites.

If the above steps did not help, then follow the steps below.

  1. Open File Explorer.

  2. On the left, look for Favorites option.

  3. Right click on it and select the Drop Box option.

  4. This should apply to all folders.

Hi lubincaussat, you can reinstall Windows 10 at any time and it will not cost you anything !

Since you previously had Windows 10 installed and activated on that PC during the installation process skip the steps which ask for a product key and select the option 'I am Reinstalling Windows 10 on this PC', and activation will not be an issue, your PC will have a digital entitlement stored on the Microsoft Activation Servers

Make sure you install the same Edition of Windows 10 (Home, Pro . . . etc.) you previously had installed to avoid Activation issues

Click this link: https://www.microsoft.com/en-us/software-downlo...
to download the Media Creation Tool (Click on Download Tool Now), with that you can download the latest Windows 10 ISO (Select Create Installation Media for Another PC), you can create a bootable USB flash drive (min 8GB) using that tool

Then, Boot your PC from the Installation Media you just created (change Boot Order in your BIOS) to begin installing Windows 10

If you have problems booting from a boot disc, you may have UEFI BIOS:

Insert the Bootable Installation Media, then go into your BIOS and make the following changes:
1. Disable Secure Boot
2. Enable Legacy Boot
3. If Available enable CSM
4. If Required enable USB Boot
5. Move the device with the bootable disc to the top of the boot order
6. Save BIOS changes, restart your System and it should boot from the Installation Medi
