It seems that you are unable to find the taskbar on your computer. I would like to have some information regarding this issue.

1. Are you referring to the taskbar on the desktop interface on Windows 8?

2. Have you made any recent changes on the computer prior to this issue?


If you are referring to the Modern User Interface, then I am sorry to inform you that we do not have any options to display the taskbar in the Modern User Interface of Windows 8 and it is by the new Windows 8 design.


However, if you are referring to the desktop interface, then the reason why the taskbar is not shown on the screen may be because the taskbar is set to automatically hide or it might be some issues with the “explorer.exe” service or it might be stopped manually.


Method 1: Disable the Auto-hide feature of the Taskbar.

If the task bar appears on the desktop interface, when you move the mouse pointer towards the bottom of the screen, then the auto-hide option of the taskbar might have enabled on the computer.
To disable this option follow these instructions and check the issue:

a) Right click on the taskbar from the Desktop Interface of Windows 8.

b) Select “Properties” to open the “Taskbar Properties” window.

c) On the “Taskbar Properties” window and under the “Taskbar” tab, uncheck the second option which says “Auto-hide the Taskbar”.

You will also have the option to change the position of the taskbar on the same window. You can change it to top, left or right.

Check if you are able to view the taskbar. If it is not visible at all from the desktop interface, then there might be some issues with the explorer.exe or its service might be stopped manually.

To start the “explorer.exe” service on the computer, try the next method:


Method 2: Check the “Explorer.exe” service and start it if it is “Stopped”.

a) Press “Alt” + “Ctrl” + “Del” keys on the keyboard and select “Task Manager” to open it.

b) On the “Task Manager” window, click on “File” menu and select “Create a new task”.

c) Type “explorer.exe” in the “Open” box.

d) Click on “OK” and check if you are able to find the taskbar now.

ameybarve say: My problem has been solve by following steps :

1. Click Start -> Type CMD -> Right click on CMD from the result -> Click Run as Administrator
2. Type the command net stop winmgmt and press enter
3. Leave the command prompt open and click Start and type system32 -> Open the folder system32       from the result pane.
4. From the system32 folder open Wbem and look for the folder Repository
5. Right click on Repository and click rename
6. Change the folder name from Repository to Repository.old
7. Switch back to command prompt and type net start winmgmt and hit enter to execute
8. Close all the windows and reboot the computer and try Msinfor32.
