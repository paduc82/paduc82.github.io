Wireless Security - Microsoft Community

xbr dionysus say:

Wireless Security

Login Information

  • Default Gateway - gateway.2wire.net / 192.168.1.254
  • Username - N/A
  • Password – “Device Access Code” on the side of the device
   
How To Find Wireless Passphrase (WPA/WEP)
  1. Plug the computer into the LAN port of the device via an ethernet cable.
  2. Open Web Browser.
  3. In the address bar of the browser enter in "gateway.2wire.net" (if this does not work, try one of the following IP address: 192.168.1.254).
  4. If asked for a password it ill be located on the side of the device labeled “Device Access Code”
  5. From the menu at the top select Settings.
  6. From the sub-menu select LAN.
  7. From the sub-sub-menu select Wireless.
  8. If password is set to "Use Default Encryption Key", check the side of the device for the password (may be listed as wireless passphrase).
  9. If Password is set to "Use Custom Encryption Key", the password will be in the box to the right.

 

Change Wireless Security Type/ Disabling Wireless Security

  1. Plug the computer into the LAN port of the device via an ethernet cable.
  2. Open Web Browser.
  3. In the address bar of the browser enter in "gateway.2wire.net" (if this does not work, try one of the following IP address: 192.168.1.254).
  4. If asked for a password it ill be located on the side of the device labeled “Device Access Code”
  5. From the menu at the top select Settings.
  6. From the sub-menu select LAN.
  7. From the sub-sub-menu select Wireless.
  8. In the drop down menu labeled Authentication Type select desired encryption.
  9. Select the radio button for Use custom encryption Key.
  10. Enter the desired wireless password.
  11. Select Save.
