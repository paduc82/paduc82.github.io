guruaid say:

The Mail app in Windows 8 and 8.1 does not have a archive feature. But the messages are stored on a local folder in the AppData folder. The messages are stored as EML files. To read the email outside of the Windows Mail app, you'll need an email client such as Microsoft Outlook, Windows Live Mail, Mozilla Thunderbird etc.

The emails are stored at the following location:
C:\Users\<username>\AppData\Local\Packages\microsoft.windowscommunicationsapps_<random_ characters>\LocalState\Indexed\LiveComm\<random_ characters>\Mail
In the Mail folder, the subfolder named '1' is the folder containing emails for the default account. The other folders represent different email accounts that have been configured on the Windows Mail app.

Double-click on the folder and then type a '*' in the File Explorer search bar that is by default present at the top right corner of the window to see email details such as sender and title.
You can select the messages that you want to back up, or copy the entire folder to a different location to make a backup of the emails.

If you are unable to save at all, it usually points out that Movie Maker is having problems with one (or more) of your source files.  Most often, the file type on one or more files is not 100% compatible with Movie Maker.  To figure out the problem, I usually make a copy of my project and begin paring it down by removing things from the end and attempting to save in between.  At the point it saves, it means the problem file is in the group I deleted.
