james vinyard say:

Well, before, a person could just run disk cleanup and they got a small panel with some settings for what you want to cleanup before it took action.  Looks like 8.1 changed that and you get a simple execution prompt with the drive letter.


Open the Control Panel and go to "System and Security", "Administrative Tools", "Free up disk space" and that is about all you can do.


The system actually does all of this by itself if you leave it idle for about 20 minutes.


For the Pro series, there is a utility called "CCLEANER" that does a great job.  The free version of that program should be good enough and it has been updated for Windows 8.  If you decide to use that utility, do a web search for it and download it only from the developers' web site and nowhere else.


Whatever you do, though, is never go to the Windows folder and try to clear out the cache folders manually.  You can shut down the machine doing that.  Disk Cleanup on its own is usually good enough and it even handles the cache generated by Internet Explorer.


Don't expect a speed boost if that is what you are looking for.  In fact, that is what the cache is for.
