  • The PC boots into Windows RE (Recovery Environment).

  • Windows RE installs a fresh copy of Windows.

  • The PC restarts into the newly installed copy of Windows.

Hi NikosFreggidis,

I'll need a few more details to provide an accurate solution: 

  • How are the monitors connected to the main PC?

  • Which version of Windows you are using? (To check, type: winver in start menu and hit enter.)

This issue is an indication of faulty drivers or incompatible graphic card driver. This issue can be fixed mostly by updating the drivers.

Method 1:

  1. Press Windows and R key together on your Keyboard.
  2. Type devmgmt.msc and hit enter.
  3. Expand the display adapter option from the list.
  4. Select the driver > right click on the driver and click on Properties.
  5. Click on Driver tab > check if you have the option to Roll Back Driver.
  6. If yes, Roll back the driver or uninstall it and restart the computer.

Uninstalling the driver will be automatically installed after the system is restarted. Check if the issue is fixed. If not, go to method 2.

                                 

Method 2:

Remove all the other monitors, connect the monitor which is not working with the main monitor.

  1. Select Settings from the start menu> click on System > Click on Display.
  2. Click on Detect under multiple displays. If it is detected, go to the next step.
  3. Open the drop-down, select Show only 1> then "Keep changes".
  4. And finally, open the drop-down again, select "Extend these displays" it will display on both the Monitor 1 and Monitor 2 now.

If both methods doesn’t fix the issue, you can follow Method 3.                      

Method 3:

Install the latest drivers from the manufacturer website.

  1. As per the given information, there are latest drivers for your graphic card.

    Click on the following links and download the drivers.  

  • Chipset driver listed in the manufacturer website.

  • Gtx770 Drivers for graphic card. 

  • GV-N770OC-4GD Drivers for graphic card.  

2.  Download the above drivers and install it.

3.  Restart the computer and check if it is working.
