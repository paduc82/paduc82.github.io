anannya podder say:

Hi Tabitha,

A loopback driver is a piece of software that lets one program send MIDI to
another program running on the same computer. Think of it as a virtual MIDI
cable. Whatever you send to a loopback driver's output port will be received by
the other program on the corresponding input. A loopback driver's in- and
outputs appear as normal MIDI ports in music software.

In short it is an interface that makes a file  accessible as a block device.
