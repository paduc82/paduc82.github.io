You keep mentioning a "driver" -- do you mean the decoder (which is what the utility checks)? A driver refers to a piece of software that tell hardware (like a video card or printer) how to interact with the operating system. The decoder is softare that works in conjuction with a media playing application to tell it how to decipher a file into music or video.

As I mentioned, if your system came with a DVD decoder, then it is possible that something you installed recently removed it. You mentioned Nero, did it work before installing that? If so, the Nero app may have replaced the original decoder, leaving you with no decoder after the Nero trial ran out. If a decoder came with the system originally, then find the setup/install discs and see if there is a way to reinstall it (or check at the manufacturer's web site and see if it can be downloaded).
