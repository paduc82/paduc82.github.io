Auto update search the first release on a newer version so, if you check, you find that it is trying to download and install Windows 10 1809 as it was on December (first working version of Windows 1809). But your version is actually supported and updated, so you have Windows 10 1803 updated as February 2019. That's way you see a difference in time when comparing 1803 to 1809.

1. Type "cmd" in windows search bar
