Technical Level : Basic

Summary
With the return of the Start menu in Windows 10 so are favorites such as All Apps (formerly All Programs). In Windows 10, launching a program is a common actions users perform. In this article, we take a quick look at how you can do so using the new and improved All Apps list in Windows 10.
Details

If you are accustomed launching programs by scrolling through your All Programs list, you can do so in Windows 10 normally. If you have a lot of programs though, it can become tedious.

All apps list

Say I want to launch Photos on my Windows Phone, all I have to do is swipe to the left, click on any of the alphabet headers and I am presented with a Alphabetical view of my installed apps, click on the letter P and I am presented with all apps that start with P.

In Windows 10, you can do the same.

Just click any of the alphabet headers and you will be presented with a alphabetical listing of your install applications:

a. user say:

Hi Devashish,

The probable cause for this issue could be due to driver for webcam in not compatible with Windows 8.1.
Please try the following methods and check if this helps.

Method 1: Run Hardware and Devices Troubleshooter
Follow the below steps to run the Hardware and Devices troubleshooter.

1. Press windows key + W and type troubleshooting.
2. Select troubleshooting.
3. Select view all on the top left corner.
4. Run Hardware and Devices troubleshooter.

If the issue still persists, I would suggest you to download the latest drivers for the webcam by choosing the correct model number from the computer manufacturer website:
 
Get the Latest Drivers, Manuals, Firmware and Software
http://www.samsung.com/us/support/downloads
 
Method 2: After downloading the drivers, uninstall the webcam drivers from the device manager and install the drivers which you downloaded from the manufacturers website.
 
Uninstall the webcam driver from “Device Manager”.
 
a. Press “Windows key + W” from the keyboard.
b. Type “Device Manager” without quotes in the search box and hit Enter.
c. Search for “Webcam” from the list of devices.
d. Right click on your “Webcam” and then select “Uninstall”.

Method 3: Download and install all updated drivers.
 
Download and install drivers
http://windows.microsoft.com/en-us/windows-8/all-drivers

Now check if your inbuilt webcam Works.

Hope this information helps, please reply for any further assistance on Windows.
