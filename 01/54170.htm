It‚Äôs not safe to change the Guest account to Administrator account due to security reasons. If incase if any of the user accounts, either Administrator or Standard account gets corrupt, it‚Äôs always safe to create a new user account and transfer the settings from the old account to new account rather than converting the Guest account to any one of them.

 

I would recommend you to enable the Built-in administrator account, change the Guest account back to standard account, create a new user account and then use it. You can go through the following steps for the same.

 

Enable and Disable the Built-in Administrator Account:

 In Windows Vista, the built-in administrator account is disabled by default. In previous versions of Windows, an Administrator account was automatically created during Out-of-Box-Experience (OOBE) with a blank password.

 An Administrator account with a blank password is a security risk. To better protect the system, the built-in Administrator account is disabled by default in all clean installations and upgrades of Windows Vista.

 

There are two ways to enable the built-in Administrator account:

 

Method 1: Use the Local Users and Groups MMC console.

Change the properties of the Administrator account by using the Local Users and Groups Microsoft Management Console (MMC).

¬∑         Click Start -> type lusrmgr.msc in Start Search box.

¬∑         Open the MMC console and select Local Users and Groups.

¬∑         Double click on Users folder on left pane.

¬∑         Right-click the Administrator account and select Properties.

¬∑         The Administrator Properties window appears.

¬∑         On the General tab, clear the Account is Disabled check box.

¬∑         Close the MMC console.

Administrator access is now enabled.

 

Method 2: Use the net user command.

Run the following command in an Elevated command prompt.

  • Click Start, type cmd in the Start Search box, right-click in the Programs list, and then click Run as administrator.
    Note: If you are prompted for an administrator password or for a confirmation, type your password, or click Continue.

¬∑         Run the command net user administrator /active:yes

 

Note: It is always recommended to disable the Built-in admin account, for disabling the Built-in admin account run the command net user administrator /active:no

 

Follow the steps mentioned below to change the Guest account type:

¬∑         Click Start, type netplwiz in the Start Search box and press Enter.

¬∑          Click Advanced tab.

¬∑          Under Advanced user management click the Advanced button.

¬∑         In the left pane, click on Users folder.

¬∑         In the middle pane, double click on Guest.

¬∑         Click on the Member Of tab.

¬∑         Click on Administrator to highlight it.

¬∑         Click the Remove button.

¬∑          Click OK on all Windows, and close the others.
