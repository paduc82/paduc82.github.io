Start date/time/tz: 7 bytes (this can probably be taken down to 4 to 5 bytes as we only need minute-precision, but let's be easy on the programmer)

Status: a few bits

Alarm settings: a few bits. Lets say one byte

Source: a few bits to 1 byte
