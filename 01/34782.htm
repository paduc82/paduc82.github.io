  3. Click OK.

  • If the problem does not occur after you restart the computer, follow these steps:

    2. If the problem still does not occur after another restart, click to select half of the cleared check boxes again. Repeat this process until you have selected all the check boxes and the problem still does not occur.
    3. If the problem occurs after a restart, click to clear half of the check boxes that you last selected. Repeat this process until you can isolate the one service that is running when this problem occurs.

a. user say:

Whether you can transfer your Windows License depends on how you acquired it in the first Place:

If your Windows 10 was bought as a full retail License or an upgrade from a full Retail Windows 7 or Windows 8.1 Than you can transfer it to a new motherboard or Computer.

If your Windows LIcense was bought OEM or acquired by an upgrade from an Orignal Oem Copy of Windows 7 or Windows 8.1 then it is non-transferable so you would need to purchase a new License. An OEM version of Windows is Never transferable as it is tied to the computer it was originally installed on.
