john_barnett say:

Hello Sparrow,


The one supplied with Windows 8 Pro is called Hyper V. Whether you can use it, of course, will depend upon whether your system (more importantly the 64 Bit processor) supports SLAT (Second level Address Translation). If it doesn't then you will not be able to install Hyper V.

To access Hyper V proceed as follows:

1. Open Control panel

2. Click the programs and features Icon

3. In the programs and features window look on the left hand side of the window and click on the Turn Windows features On or Off link.

4. In the Turn windows features on or off window, look down the list of available programs and place a check mark (tick) by clicking your mouse, in the box to the left of the Hyper V option.

5. Finally click the OK button

6. Hyper V will now install. You may have to restart your PC after the installation.

7. When your PC restarts, look for the Hyper V icon on the Start page.

You might like to do a Google search for setting up Hyper V as it is a little more complex than using such as Virtual PC.

Also if you have any other Virtual machine software installed, for example, VMware Workstation, you may find that Hyper V conflicts. To rectify you may have to uninstall any other VM software.

    alden rey say:

    Hi,

    Windows has always used RAM to store components of every program, service and library it runs. Windows 7, however, does one more thing with your RAM: it stores the data of every program it thinks you might use. This feature is otherwise known as Superfetch, and has many people complaining of Windows 7â€™s memory usage. Windows basically pre-records the memory for certain applications you often use in order to reduce the time it takes them to load. If you have 2 GB of physical memory or more, you usually end up benefiting from this extra feature. Any less memory, and Microsoft turns your computer into a snail begging for mercy. You may benefit more from disabling Superfetch. To disable this feature kindly follow the steps below:

    1. Hold the Windows Key, while pressing R to bring up the Run dialog box.
    2. Type services.msc, then press enter.
    3. The Services window displays. Find Superfetch in the list.
    4. Right-click Superfetch, then select Properties.
    5. Select the Stop button if you wish to stop the service. In the Startup type dropdown menu, choose Disabled.
