tunepunk say:

I had to resort to installing a language that does not support handwriting.

I added Eesti to my languages, which does not support handwriting and made sure Override for windows display language in advanced settings was set to English. Removed all other languages, since I don't use any spelling correction.

This workaround does exactly like i wanted. Although I can't use auto-corrections and suggestions, but i never use this functionality anyway.

Current Language Settings: (Control Panel)
Language: Eesti
Keyboard: (Swedish as I'm Swedish and prefer Swedish keyboard)

Advanced Settings:
Display language Override: English (United states)
(Checked) * Don't let websites access my language list. The language of my date, time and number formatting will be used instead.  - This makes sure correct language (English not Eesti) is displayed on websites etc.

Region (Control Panel)
Formats tab - English
Location Tab - United States (I just chose this option to get Cortana, although I live somewhere else)

Settings > Typing
Autocorrect Misspelled words (off)
Highlight Misspelled words (off)
Add a period after i double tap the Spacebar (off)

....

Hope this helps anyone else who has the same problem. As long as you can live without spelling correction etc this is the best solution for now.
