This usually happens when the operating system still recognizes a device, but it is no longer connected. You can follow these steps to fix the issue:

  1. Press the Windows + R key on your keyboard to open the Run box.
  2. Type Regedit in the Run box to open the Registry.
  3. Then go to this location: HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\MyComputer\NameSpace\
  4. Right-click NameSpace, choose Export, Save it to your desktop as NameSpace.
  5. Once done, right-click NameSpace again, and choose Delete.
  6. Just choose Yes on the prompt that says Are you sure you want to permanently delete this key and all of its subkeys?.
  7. There will be an error, but it should delete the files inside NameSpace.
  8. Go to your desktop and double-click the .reg file that that we've created so that it will import the files back to the Registry editor.
