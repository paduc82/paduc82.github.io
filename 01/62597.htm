System Reserved cannot be formatted if active and by default does not have a drive letter, you cannot change the OS drive letter after install as all the pointers will break. Because of this Disk Manager will not let you do either. If it lets you format the disk it is not protected and therefore not the active boot partition.

During setup for a clean install where it offers where to install Windows select the Advanced option.
