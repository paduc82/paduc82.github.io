rickee say: It is a myth that the AE1200 is not compatible with Windows 10. The reason there is no driver update is because the Windows 7 driver for the AE1200 works with all later versions of Windows, including Windows 10. What is causing the problems here is that when you upgrade to Windows 10, or a major Windows 10 update is pushed onto your computer, the AE1200 driver temporarily stops working. The fix is to simply uninstall it in Windows Device Manager and then reinstall it. This is a bug in Windows 10 installation that can be fairly blamed on Microsoft, but its Windows 10 Ready utility is not itself making an error. The AE1200 and its Windows 7 driver are compatible with Windows 10.
