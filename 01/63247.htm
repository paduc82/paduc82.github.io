If you have more than one user account on your computer, Fast User Switching is an easy way for another person to log on to the computer without logging you off or closing your programs and files. For instructions on creating user accounts, see Create a user account.

  1. Click the Start button, and then click the arrow next to the Shut Down button.

  2. Click Switch User.

Note

If you are logged on to a remote computer (for instance, using Remote Desktop Connection), you can't use Fast User Switching on that computer.


Warning

Make sure to save any open files before switching users because Windows doesn't automatically save files that are open. If you switch to a different user account and that user shuts down the computer, any unsaved changes you made to files that are open on your account will be lost.

Your solution depends on what you are using to connect the headset to your controller.

Firstly - regardless of which set up you are using, make sure you have updated your controller with the latest updates.

Now...

If you are using the "Headset Adapter" then the buttons on the right hand side control your volume settings.  The buttons on the left control the Game Volume to Chat Volume ratio.  If you don't have this set up correctly you may not be getting the right volume for your game or your chat.  Try pushing on the chat volume button (bottom left) until the game volume dissapears completely.  Then push the game volume (top left) up one so you hear the game sound... then adjust the chat volume using the controls on the right hand side.  Once that's comfortable... hit the game volume button (top left) until it's a comfortable game/voice balance.

If you don't have a headset adaptor and are instead using the 3.55mm jack directly in your controller (only newer models of the controller have this) then you can go to the dashboard, hit "left" on the D pad, scroll down to "Settings" and you have volume sliders available to customise as whatever works best for you.
