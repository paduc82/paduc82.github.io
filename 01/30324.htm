lemp say:
I don't know what you mean by the icon for "open with."  It sounds as if the previous user had associated the "DAT" file extension with Nero. Instructions for undoing this association are at the bottom of this response.

The file extension "DAT" is a very common extension.  Most such files are "DATa" files and not meant to be opened by the casual user.  You really should not be deleting random files like this.  There is a good reason that you couldn't delete ntuser.dat -- it's part of the Windows Registry that specifies a lot of information about your Windows profile. 

If by "Eset" you mean that you ran the Eset online scanner, that's OK.  If you mean that you have Eset NOD32 Antivirus, then you have too many antivirus applications (I would remove SEP in favor of NOD32, but see the next paragraph).

Many people would consider it prudent to completely reinstall Windows on any computer that you obtain second-hand.  You don't know what is on that computer, and it this case, what you don't know could certainly harm you -- in more ways than one.

Most Dell computers come with a "hidden restore" partition.  What you want is either #2 or #3 on this page --> http://support.dell.com/support/topics/global.aspx/support/kcs/document?docid=52182  Back up any files you created since you got the computer before you restore to the original factory condition.
