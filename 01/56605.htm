You can ignore the Microsoft Teredo Tunneling Adapter" error, at least for now.  That's used for IPv6 communications.  Unless you have an Internet service that uses IPv6 or you want to use the "Homegroup" feature for setting up your local network, you don't need IPv6 -- yet.  See http://appuals.com/microsoft-teredo-tunneling-adapter/

andre da costa say:

The 'N' versions are only available in European countries and minus media features such as Windows Media Player and Windows Media Center. You will have to manually install it yourself if you live in Europe due to a consent decree.

The best edition of Windows 7 is debatable since this depends on your needs and what you plan to do with your PC. If you want to know the most complete edition of Windows 7 that contains all features, that is Windows 7 Ultimate.

Each edition of Windows 7 is a subset of the other. So the higher you go the more features and better functionality you get.

Windows 7 Starter > Home Basic > Home Premium > Professional > Ultimate/Enterprise

Home Premium - best for Entertainment, this edition provides full functionality on the latest hardware, easy ways to network and share media using HomeGroups. You can watch shows for fee when and where you want with Internet TV on Windows Media Center.

Professional - Has all the features of Windows 7 Home Premium, it has everything you need for work and home. The business-focused edition is great for small and medium sized companies and people who have networking, backup and security needs and multiple PCs or servers.

Ultimate - Has all the features of Windows 7 Home Premium and Professional in addition to that, you get features unique to if you are working in a large Enterprise environment that is running Windows Server 2008 R2:

  • Microsoft BitLocker and BitLocker To Go drive encryption and data protection on internal and external drivers and storage devices.
  • Direct Access - which helps give you more secure connection to your corporate network over the Internet.
  • Microsoft BranchCache - which helps provide faster access to content from remote file and Web Servers at branch locations.
  • Microsoft AppLocker - which helps prevent unauthorized software from running on worker's PCs.
  • Enterprise Search Scopes - which make it easy to discover and search content on intranet portals
  • Multilingual user interface packs - which let you work in any of the available 35 languages.

32 vs 64 bit

http://windows.microsoft.com/en-US/windows7/32-bit-and-64-bit-Windows-frequently-asked-questions

Is my PC running the 32-bit or 64-bit version of Windows?

http://windows.microsoft.com/en-us/windows7/find-out-32-or-64-bit

http://windows.microsoft.com/en-US/windows7/Installing-and-reinstalling-Windows-7

Windows 7 64 bit comes in handy when you need to address at least 4 GBs or more of RAM. Windows 7 32 bit can utilize up to 3.2 GBs of RAM. Because the memory address space is much larger for 64 bit Windows, that means, you need twice as much memory than 32 bit Windows to accomplish some of the same task, but you are able to do so much more, you can have more applications open, do things like run an Antivirus scan in the background without it affecting your system performance. Windows 7 64 bit is more secure too, malicious code cannot easily infiltrate it, drivers are more reliable since they must be signed before they can work with 64 bit Windows 7.

As for compatibility, you will need 64 bit device drivers for any hardware devices you might have. Also, there is no 16 bit subsystem in Windows 7 64 bit, which means, your applications must be 32 bit only, not 16 bit installer or uninstallers. Also, if you decide to move to Windows 7 64 bit in the future, there is no upgrade path from 32 bit Windows, clean install only.
