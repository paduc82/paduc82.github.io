The way to end a remote session is to click the Red "X" at the top right of the window.  It will immediate terminate the remote user's access to your computer.  I now begin any session with, "PLEASE DO NOT MAKE ANY CHANGES ON MY COMPUTER UNTIL YOU ASK ME."  If they start trying to open things I do not want opened, I take control of the mouse, rather than allowing them to do it, and they usually start typing.  I use this time to capture a screenshot/picture by hitting Shift+PrtScn, have a Word document open, use Shift+Tab (move between windows) to go to Word, hit Ctrl+Alt+V (paste special) and choose "Device Independent Bitmap" which should be highlighted, and this inserts a picture of exactly what the remote technician is doing, the time, etc.  Then hit Shift+Tab to go back to the support chat window.  Have both open side by side so you can watch what the remote tech is doing while you copy what he/she is doing.  He/She can see that you are doing it as well.  Another trick is to hit Ctrl+A (which selects all), then Ctrl+C (which copies it), then Ctrl+Tab to go to Word, then Ctrl+Alt+V (paste special) to paste it in "formatted" or "unformatted" format.  If you don't like the way it looks, hit "Ctrl+Z" (which undoes your most recent action).  Keep in mind, the remote tech may be able to use shortcuts (macros), I just do not know.  So if anything is suspicious, just end the session by hitting the Red "X" and ending the session.  The window will stay open long enough for you to copy the contents so that you can keep a record of it.  Be careful choosing any type of paste except "unformatted text" if any hyperlinks or web addresses are included in the chat transcript.  If you paste them in formatted or HTML format, you could infect your computer with malware if they are a bad apple or an imposter/scammer.

davem121 say:
Hi sivamurali87, you can reinstall Windows 10 at any time and it will not cost you anything !

Since you previously had Windows 10 installed and activated on that PC during the installation process skip the steps which ask for a product key and select the option 'I am Reinstalling Windows 10 on this PC', and activation will not be an issue, your PC will have a digital entitlement stored on the Microsoft Activation Servers

Make sure you install the same Edition of Windows 10 (Home, Pro . . . etc.) you previously had installed to avoid Activation issues

Click this link:
https://www.microsoft.com/en-us/software-downlo...
to download the Media Creation Tool (Click on Download Tool Now), with that you can download the latest Windows 10 ISO (Select Create Installation Media for Another PC), you can create a bootable USB flash drive (min 8GB) using that tool

Then, Boot your PC from the Installation Media you just created (change Boot Order in your BIOS) to begin installing Windows 10

If you have problems booting from a boot disc, you may have UEFI BIOS:
Insert the Bootable Installation Media, then go into your BIOS and make the following changes:
1. Disable Secure Boot
2. Enable Legacy Boot
3. If Available enable CSM
4. If Required enable USB Boot
5. Move the device with the bootable disc to the top of the boot order
6. Save BIOS changes, restart your System and it should boot from the Installation Media
