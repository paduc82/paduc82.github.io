Hi Elroy,

Thank you for posting your query in Microsoft Community.

There are a few reasons why you might get error 0xC004C003:

  • You performed a clean installation of Windows 10 instead of first upgrading from your previous version of Windows. When this happens, we aren't able to register your entitlement with the activation servers because you aren't upgrading to Windows 10. To fix this, you'll need to go back to your previous version of Windows, make sure the previous version is activated, and then upgrade to Windows 10. For more info about how to go back to a previous version of Windows, see the section in

  • You upgraded to Windows 10, but the activation servers were busy. If you upgraded from an activated copy of Windows 7 or Windows 8.1, your copy of Windows 10 will be automatically activated, or you can wait a while and then select Activate now. If you performed a clean installation instead of first upgrading to Windows 10, see the info on how to fix this problem in the previous item.

  • You might also see this error if you're using an invalid product key. For example, you can't use a Windows 7 or Windows 8.1 product key to activate Windows 10.

Additional information:

Why can't I activate Windows 10?

tryx3 say:

1         Preparations

1.1       Make a DFIR USB

The Dell Factory Image Restoration [DFIR] partition contains Windows, drivers & many but not necessarily all Dell-bundled utilities &, as its name implies, it returns your system to its out-of-the-box condition except for any Dell utilities that they haven’t put in it.  If the Recovery CD to which you referred is really just a CD then it will just contain drivers & utilities not the OS.  Confusingly, Dell also refer to the DFIR USB or a DFIR on DVD as a Recovery disk so you do want to check exactly what you have.

Get an 8GB USB stick.  I was advised to get a 16GB stick but found afterwards that 8GB would have been big enough.

Start the Dell DataSafe Local Backup application that has been bundled with your system by Dell.  This almost useless application has one capability that is essential – making a DFIR USB.  See also What is Dell DataSafe Local Backup 2.0 & Frequently Asked Questions: Dell DataSafe Local Backup 2.0 or find the equivalent pages for your version of this [my Inspiron is from Jan 2010].  It’s possible that the Dell-bundled backup utility now has a different name so you’ll have to browse through the Dell program fodlers if its identity is not immediately obvious.

Click on the Backup tab.

Click on Create Recovery Media.

Select USB as your media.

It then runs through & creates a bootable version of the DFIR that you can use to return your PC to the state it was in when you first started Windows on it.  I was able to test this USB as I was doing a complete reinstallation from scratch but you will not have this opportunity.  I think the creation procedure ended with the advice that the DFIR USB should only be used if the conventional restoration from the DFIR partition itself should ever fail [this procedure is written out in your PC manual].

You might be tempted to delete Dell DataSafe local afterwards but don’t do it – getting it replaced is very difficult & even when I found the download site I couldn’t get it to install again.

1.2       Make a Windows Repair Disk

For an explanation of what this is see What are the system recovery options in Windows 7? using the link or just by looking in Windows help.  The Repair Disk gives you access to repair & restoration options if your PC is so corrupted that you cannot do this directly in Windows itself.

Many Dell users have reported that they are unable to make successful Repair Disks – theirs [as mine used to] take half an hour to report error 0x4001100200001012 & then just sit there laughing at you.  Some people say that the error is caused by something that’s non-standard in the structure of DFIR but I’ve never seen any proper confirmation of that explanation [but in my case the problem disappeared when I deleted the DFIR].  The Windows Installation Disk [see below] can do everything that a Repair Disk can so you can use that instead.  I have described the procedure purely in the irrational expectation that Dell will have fixed this problem by now.

Open Windows Backup, look in the left-hand part of the Window & you’ll see the link for Create a system repair disk.  Put a CD-RW or DVD-RW in the drive & hope for the best.

Now, keep the disk in your drive & reboot to test it.  If it works then you will be taken to a screen where you select the region & language settings & then to the System Recovery Options menu.  This proves that it works.  If you spend ages waiting on a slowly moving progress bar then you can expect the error message to appear eventually.

You’ll find fuller explanations of this procedure in Windows Help & Create a System Repair Disc [Seven Forums].  You can also search this MS Answers forum for repair disk information as there are several discussions.

Others say that you can also get someone else to make a system repair disk as long as it is the right x32/x64 to match your Windows & it should work on your PC [but I’ve never tried this].  Once again, use a CD-RW or DVD-RW so you are not throwing money away.

You can also download ISO images of the repair disk – there is one for x32 & one for x64 on this link Create a Windows 7 System Repair Disc — MAXIMUMpcguides – Windows 7 tips, tricks, help, and how-to guides

1.3       Transform a Windows Repair Disk into a USB

However you have made a System Repair Disk, you’ll save time using it if you make it into a USB instead.  USBs are generally regarded as much more reliable than DVD-RWs as well. 

Use the smallest USB stick you have as the Repair Disk only uses 200MB or so. 

I used Paragon Backup & Recovery Free Edition for this task but the backup applications you’ve already mentioned might also be able to transfer a bootable DVD into a bootable USB.  Paragon can also do this directly from an ISO file if you’ve chosen that source for the Repair Disk.  To create the USB, select the Recovery media option & tell Paragon which source DVD/USB/ISO to use & which is the target drive.  I imagine the other utilities will work in a similar way.

Once again, do test it before believing that it will help you in any emergency.

1.4       Make a Windows Installation DVD & transfer it to a USB stick

You will need a 4GB USB stick for this.

To make an installation DVD/USB, download the correct* ISO from Official Windows 7 SP1 ISO from Digital River then transfer it to a DVD or USB using the Windows 7 USB-DVD download tool

* You can only run any future successful installation with the version of Windows [Home premium & so on] that is identified on your Certificate of Authenticity - this is the label that’s probably inside your battery compartment & that shows your Product Key.  If you do ever need to reinstall from scratch you’ll find that you won’t need to actually enter your Product Key or even to Activate Windows as Dell have saved the necessary codes in your BIOS.

You need to install the USB-DVD Tool then run it with the ISO file stored anywhere on your system [except the target DVD/USB drive].

There is something non-standard about the structure of the Win7 ISO so the USB-DVD Tool might be the only tool that can successfully make this USB for you [it’s free so you might as well use it].

The installation DVD/USB can also be used as a Windows repair disk for repairs & for restoring system images.  Hence a good test of it is to boot with it just as you did with the repair disk.

1.5       Backup your Dell utilities

At the Dell site, register for Dell MyWay & download all the installation files for the various Dell applications. Check all the updates in the Dell update/download part of their support site as well. Put them on an external HDD. Check all the installed Dell utilities & make sure nothing is missing [Factory Restore might not have some of the utilities so you'd otherwise be at risk in case of system failure - DataSafe Local Backup used to be one of the missing ones but they might have corrected that by now].

1.6       Backup all other installation files

As what you are about to do carries some risk of wrecking your system, assume that you are going to have to re-install everything.  I ran through this whole procedure on my old HDD before fitting the new one & didn’t once bugger the system up but I’d still make that assumption if I did it all again.

Don’t forget to backup the ISO you’ve just downloaded.

1.7       Back up all datafiles & personal settings

Similarly, backup everything that you want from the folders visible in the C:\Users\YourUserName subfolders.  Whilst you are about to make a system image, you will want a simple copy of your datafiles, Favourites & so on as you will soon by copying them across onto your new D:\ drive [partition].

Some people would quite sensibly recommend that you also backup the hidden folders within C:\Users\YourUserName [such as AppData] but I disagree.  If I made such a mess of the whole task that I needed them then I would actually decide to restore from my system image so I could start the whole procedure again.

The Insert or Overtype mode toggling is not global on Windows 10 or even for a single application but rather local to the text input window in which the Insert key was pressed. By default, Insert or Overtype mode is disabled by design.

You may be able to find some instructions on the web to set Insert key always on but it includes modifying registry. However, serious problems might occur if you modify the registry incorrectly.
