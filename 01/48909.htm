Any firewall that has solely minimum/maximum settings is a very poor firewall implementation. a. user say:

Hello,

Thank you for posting your question on Microsoft Community Forum.

I would like to know some information about the issue so that we could assist you further.

1. Are you facing any issue on the computer?

2. Do you want know, how much memory the graphic used on the computer?

3. What is the make and model number of the computer?

The graphics card does indeed have built-in memory for its own use. This is often referred to as 'dedicated memory'. Then, since many games and applications require even more memory, part of the system's RAM is shared with the card. This amount will vary based on your graphics card, and your system's available RAM.

Some systems even provide system RAM exclusively for the graphics card. Though this is rare, it may occur if your graphics card has very little memory and your system has a lot. On most systems, however, a portion of the RAM is only 'shared' with the card, meaning that it can use it if it needs to. If not, it will be available for use by other programs.

The maximum shared memory is for the integrated (onboard) graphics memory. It can use up to 1 GB of your system RAM to run the onboard graphics.

32-bit Windows has an address space of 4GB. Part of that is used by system BIOS and graphics memory.

How to check the amount of built-in memory on your graphics card:

1. Open System Information (search for it in the Start Menu).

2. In the 'Components' section, click on 'Display'.

3. The total built-in memory will be listed under 'Adapter RAM'.

How to find the amount of shared system memory:

1. Search for and open 'dxdiag.exe' in the Start Menu.

2. Go to the 'Display' tab (you may have to click 'Yes' for an option to verify files, but that is irrelevant).

3. Your total available RAM for the graphics card will be listed under 'Approx. Total Memory'. This value is the sum of the built-in RAM and the shared system memory. To find the total shared RAM, simply subtract the amount of dedicated memory you found above.
