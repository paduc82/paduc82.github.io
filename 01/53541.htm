Force Windows 7 to use the wired rather than wireless adapter and KEEP - Microsoft Community

cindybode say:

Force Windows 7 to use the wired rather than wireless adapter and KEEP using it (not change back periodically)

We have a classroom in which we have a wireless adapter to connect to a wireless projector, but need the wired adapter to provide network and internet access. I can change the binding order (steps below) so that Windows will use the wired network as the first priority, but every so often, for no apparent reason, it switches back to using the wireless. I've also tried changing the metric, (instruction also below) but that did nothing at all to correct the issue. We have multiple instructors, but it doesn't seem to be tied to a profile. Any suggestions would be appreciated.

 

 

Change order of network binding

1.    1.    Open Network Connections by clicking the Start button  , clicking Control Panel, clicking Network and Internet, clicking Network and Sharing Center, and then clicking Manage network connections.

2.    2.    Press the ALT key, click Advanced, and then click Advanced Settings.  If you are prompted for an administrator password or confirmation, type the password or provide confirmation.

3.    3.    Click the Adapters and Bindings tab, and then, under Connections, click the connection you want to modify.

4.    4.    Under Bindings for <connection name>, select the protocol that you want to move up or down in the list, click the up or down arrow button, and then click OK.

 

Change Interface Metric

1. Open the Network Connections folder (Start > Run > ncpa.cpl)
2. Right click the desired connection.
3. Click Properties > Internet Protocol Version 4.
4. Click Properties > Advanced.
5. Un-check "Automatic metric".
6. Enter a number between 1 and 9999 for the "Interface metric.
