The Page File shouldn't impact on how much RAM the game uses. The Page File is not a substitute for RAM. It is meant to help HDD's

provide the cpu and RAM quicker access to the games files, and in the process also helps avoid HDD 'thrashing'

a given level of graphics settings regardless of if there is a Page File or not.

If the machine has only the minimum RAM for the game the Page File may get a bit more use if the User trys to push settings higher than minimum.
