Here's a good article on the file type: http://www.sharpened.net/helpcenter/file_extension.php?qtch

Here's an excerpt from that article:

"Cache of a QuickTime audio or video file downloaded or streamed from the Internet; if the file was created by QuickTime 6.5 or earlier, it may be opened in QuickTime Player by changing the file extension to match the original file (i.e. .MP3, .MPEG, .MOV). QuickTime 7 and later scrambles the header information of QTCH files so they cannot be played back by simply changing the file extension." (which means you need the later version of the program and I assume the same holds true for iTunes).  There is more information in the article but it seems you need either Quicktime Player or iTunes to open and play this file type.
