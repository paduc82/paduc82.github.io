Hello,

Thank you for posting your query Microsoft Community.

As per description, you are unable to access the administrator account.

I would suggest you to enable hidden administrator account and once that gets enabled we can create a new administrator account.

To enable hidden administrator account follow the step given below.

  1. Press Windows key + X key and select Command Prompt.

  2. In command prompt type net user administrator /active:yes  and hit enter.

This will tell Windows to enable the hidden default build-in Windows Administrator account. You need to logout the current user in order to see the newly “added” Administrator account.

Once the hidden administrator is active. Login into that account and try to create new user account.

Create new user account with administrator privileges and check.

I would suggest you to follow the article given below to create a new user account and check

Create a user account 

http://windows.microsoft.com/en-us/windows/create-user-account#create-user-account=windows-8

once the you successfully created the new administrator account I would suggest you to disable the hidden administrator account.

To disable hidden administrator account follow the step given below.

  1. Press Windows key + X key and select Command Prompt.

  2. In command prompt type net user administrator /active:no  and hit enter.

This would disable the hidden administrator account.
