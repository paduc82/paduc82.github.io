What does 'something about socket error 11003' mean? It would be much better if you posted the entire error message so we don't have to guess what it said.

Error 0x800CCC0D means that the server could not be found. Socket error 11003 is usually a DNS error: the server name you entered does not exist or can't be translated into an IP address. Since you didn't tell us which server is involved, it's impossible to suggest what you should enter instead.

It's quite sure, though, that changing server settings won't help if the server can't be found. Fiddling with the firewall is dangerous, and won't help in this case either. Nor will removing and replacing the account, unless you find out what server name to use and enter that when you re-add the account. You could just as easily do that without removing the account to start with.
