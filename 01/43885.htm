Call of Duty Advanced Warfare NAT Strict on Motorola Surfboard - Microsoft Community

boheathen say:

Call of Duty Advanced Warfare NAT Strict on Motorola Surfboard SBG6782-AC Resolved

Hey Everyone,


      This isn't a request for help but I helped my brother resolve a NAT issue last night and I thought I'd share what worked so maybe others can benefit. Initially the NAT issue was only for the Call of Duty Game, not the Xbox One.

Modem: Motorola Surfboard SBG6782-AC

http://www.arrisi.com/products/home/sbg6782-ac.asp

* This was a store bought Wireless Gateway modem that was an upgrade from Comcast Cable's standard issue modem for the speed they signed up for. It was purchased at Bestbuy in FL.

ISP: Comcat Cable

Connection: Wired

The issue: NAT Moderate/Strict (after trying to manually set static IP address on both console and modem configuration interface) and NAT Open on Xbox One, NAT Strict on Call of Duty Advanced Warfare.


What we tried: After I logged into the modems configuration interface we followed the recommended enable/disable/reboot UPnP, then Port Triggering/Forwarding, and DMZ. I was able to set a static IP on the console but the modem configuration static IP we set resulted in many connectivity errors despite using the correct DNS settings on both machines. For his modem it didn't seem to work. After we restored the modem to factory settings I didn't bother to redo the static IP on either machine and things worked out.

What worked: After trying all three with no success, I reset the modem back to factory default settings, disabled the UPnP feature since it wasn't working anyway, and set the DMZ host to match the Xbox IP Address on the network and console. I saved the settings, rebooted both the modem and the Xbox One and everything worked out great. I verified everything both on the console and the COD game to make sure there were no more issues.

I've read around and it seems others are having similar issues but I think if you bought a upgrade modem for your Comcast cable service this solution may work. I know that DMZ is a last resort but for this set-up it worked out. I recommend if you have a similar Motorola Surfboard modem trying the three steps above and it that fails just use DMZ.

Hope this helps!

-Jessica

Now, my question is - is there a way to prevent windows from installing localized (croatian) versions of apps?

jane lla say:
Hello,

Have you tried removing the Local Language from the Region and Language settings in Control Panel?

Also since you've mentioned that you're using a Croatian keyboard layout, check if this workaround sorts out the issue. Here's what to do:

1. Open Control Panel.
2. Click on Clock, Language and Region.
3. Select Language.
4. Make sure English is on top of the list. Move the rest below by using the Move down button. The store uses the language that is at the top of the list.
