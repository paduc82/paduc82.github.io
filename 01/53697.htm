A warning here that if you do the following and subsequently decide to go back to Windows 7/8, fiddling with accounts may cause the reversion to fail.  In that case you would have to do a clean install of either Windows 10 or 7/8. So to minimise this possibility, before attempting to go back, login with your Microsoft account and remove the new local account.  Also make sure that you have backed up all your important files on USB devices.

 

How to add a new local admin account user to Windows 10

Go Start > Settings > Accounts > Family & other users, then Add someone else to this PC
Click The person I want to add doesn't have an email address, then Next
Click Add a user without a Microsoft account
Enter the user name and password then click Next
Now change this new local account to Admin user
Start > Settings > Accounts >Family & other users, then select the new account, then click Change account type, Select Account type Administrator, click OK

Then logout and login with the new account and see if it works.

If it works, Logout, then login with your corrupt account, and backup your documents, pictures and anything else you want to keep onto a USB storage device (Onedrive files will be OK provided that they are all synced)

Then logout, login with the new local account, go to Go Start > Settings > Accounts > Family & other users and remove your Microsoft account, deleting the data.

Then add your Microsoft account again, and make it type Admin

Then logout, login with your Microsoft account and it will set up your profile again and it should work.  (You will need to verify your account.)
