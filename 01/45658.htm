xbf chris j say:

Hello Yousef,

Mac filtering is a setting accessed through your routers settings, and not through the Xbox settings.

Wireless routers can prevent unauthorized network access by using a feature called MAC filtering. Unfortunately, MAC filtering can also prevent your console from connecting to Xbox Live.

If MAC filtering is turned on, try one of the following:

  • Add your Xbox One console's MAC address to your router's authorized list.
  • Or, turn off MAC filtering temporarily on your router to see if MAC filtering is preventing your console from connecting to Xbox Live.

You can find your consoles mac address in your network settings.

  1. On Xbox Home, navigate to the left until you reach Pins.
  2. Select Settings.
  3. Select Network.
  4. Select Advanced Settings.
  5. Write down your Wireless MAC address.

how do you move files from one user account to another?
lisa m. say:

Michelle fay,

Welcome to the Microsoft Answers Community.

Copy files to the new user profile

1.

Log on as a user other than the user whose profile you are copying files to or from.

2.

In Windows Explorer, click Tools , click Folder Options , click the View tab, click Show hidden files and folders , click to clear the Hide protected operating system files check box, and then click OK .

3.

Locate the C :\Documents and Settings\ Old_Username folder, where C is the drive on which Windows XP is installed, and Old_Username is the name of the profile you want to copy user data from.

4.

Press and hold down the CTRL key while you click each file and subfolder in this folder, except the following files:

•

Ntuser.dat

•

Ntuser.dat.log

•

Ntuser.ini

5.

On the Edit menu, click Copy .

6.

Locate the C :\Documents and Settings\ New_Username folder, where C is the drive on which Windows XP is installed, and New_Username is the name of the user profile that you created in the "Create a New User Profile" section.

7.

On the Edit menu, click Paste .

8.

Log off the computer, and then log on as the new user.

 

*regarding email: You must import your e-mail messages and addresses to the new user profile before you delete the old profile.

 

I hope that helps!
