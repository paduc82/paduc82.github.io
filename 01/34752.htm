I still don't understand what "InkJet icon" you're clicking on if "there are no InkJet Printers listed in the Devices and Printers" screen.

I also don't understand why the Canon Pixma iP90 wasn't added to the list when you clicked the Windows Update button; it should have been added.  However, because it didn't get added, you can get the driver manually.

  1. Using Internet Explorer (only) go to http://catalog.update.microsoft.com/v7/site/Home.aspx
  2. At the Microsoft Update Catalog, search for Canon iP90 ( do not include the word "Pixma")
  3. The search should result in a list of 7 hits; 3 of these are for Windows 7 (among other versions of Windows)
  4. When you click on the link for one of the Windows 7 drivers, you'll get a screen like this:



  5. Look at the "Architecture" line.  "X86" is the driver for the 32-bit version of Windows 7.  "AMD64" is the driver for the 64-bit version of Windows 7.  You do NOT want the driver for "IA64."
  6. Once you have decided which of the 3 Windows 7 drivers is the correct one for your system, click the "Add" button at the right end of the line.
  7. Click "View basket" (just under the search box)
  8. Click the "Download" button
  9. You should get a pop-up window that says "Download options."  Click the "Browse" button, respond to the UAC prompt as necessary, and select the directory (folder) where you would like to save the downloaded file.  I suggest that you save the file to your Desktop.
  10. Click the "Continue" button and wait for the download to complete.
  11. When the download completes, click "Close."  You can also close Internet Explorer at this time if you like.
  12. The download will be a folder (if you selected your Desktop in step 9, the folder will be on your Desktop) named "Canon driver update for Canon Inkjet iP90."  In that folder will be a file with a name comprising a very long string of characters.  The file will be a "Cabinet" type file (i.e., the file extension will be .cab).
  13. If you have an archiving utility such as 7-zip (http://7-zip.org/) installed, you can extract the contents of the *.cab file using that utility. Otherwise, you'll have to use the built-in "Expand" command in a Command Prompt window.  For help with the Expand command, open a Command Prompt window and type
        expand /?   If you need more explicit directions, please ask (although using 7-zip would be faster).
  14. With your printer NOT connected to the computer, get back to the "Add printer driver wizard" (if you can't remember how you did it the last time, please ask for directions)
  15. When you get to the screen that lists printer manufacturers, select Canon, and then click "Have disk"
  16. Browse to the folder where you extracted the files from the *.cab file in step 13.  Select the file with an icon like this:

    and click "Open."
  17. Click OK, then select Canon iP90 from the list and complete the wizard
  18. Connect your printer to your computer and turn the printer on.  It should now be recognized and installed.
