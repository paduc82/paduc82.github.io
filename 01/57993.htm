considerate_guy say:

Right after adding new clips, those clips are automatically selected for you.  So maybe you are trying to immediately add a transition and it applies to them all?

The key is to only select the clips you want to be affected, first.  The selected clips have a light blue outline around them in the storyboard area.  Be sure to select the clips you want first.  To select a group of clips, hold down <Ctrl> and click on the clips in the storyboard area. 

Once the clips are selected, just double-click the transition you want.  Do not click the 'Apply to all' button or it will override the selected clips and apply that transition to all clips. 

Again, do not click 'Apply to all' unless that is what you want.  Instead double-click on the desired transition and it will apply to the selected clips.

lemp say:
I need to replace this deleted file. What is a trusted site I can use to complete this?

msjet.35.dll is part of the Microsoft Jet Database engine.   Primarily, this is used in Microsoft Access, but other database applications may also use the Jet engine and thus need msjet35.dll.

Probably the best way to replace this file (other than restoring it from the Recycle Bin, if that's possible) is to reinstall the application that is no longer working.

If you think you need to replace the file for a reason other than a non-working application, please explain.

You might also want to ask in the Microsoft Access forum .

msjet35.dll is not an "application."  It is a component of an application, such as Microsoft Access, that deals with databases.  AFAIK, there is no "trusted site" from which you can download msjet35.dll, although I'm sure that using Google you'll find lots of sites that claim to have such a file .

I don't know what you mean by "the Start...COMMAND...etc."

If you get "msjet35.dll is missing" error on startup, use Autoruns to determine how the file is being called and thus, possibly, what application is looking for it.  Once you know what application is trying to use msjet35.dll, you should be able to uninstall the application if you no longer need it or reinstall it using whatever mechanism you used to install it in the first place.
