Your Recovery Partition was imaged on your Hard Drive during the Manufacture of your Computer by the Manufacturer for these reasons.

considerate_guy say:

First, save your work as a movie. From the File menu, choose Save movie. (Note: that is different than Save project). Choose the format that can be used by your video DVD software on the other computer. For example, if you are using Windows DVD Maker on a Windows Vista or Windows 7 computer, it likes .wmv format video. Or if your software can handle .mp4 files, use that instead.

Now you can copy that file onto a USB flash drive and transfer the file to the other computer. Or you can use cloud storage like Dropbox or OneDrive. Login and retrieve it on the other computer.

Once the video file is on the other computer, you can use the chosen software to burn a video DVD. Remember though, if you are trying to create a DVD that will play in a standard DVD player, it must be created as a video DVD. It's not enough to just burn the single video file onto a DVD disc.

A video DVD uses the MPEG-2 codec and the structure of a standard DVD so that your player will understand it and be able to play the movie. Windows DVD Maker will create a video DVD, but just burning the file onto a DVD via the operating system will not.

If you don't have Windows DVD Maker, other options are Freemake Video Converter, DVD Flick, DVD Styler and others. Just be sure, if you are installing any freeware options like this, you always want to pick the Custom install option. That way you can uncheck any add-ons it may try to install.

kommerendj say:

Normally a wired connection is better. That is because it is not subject ot interference or signal strengh degradation.  You should have a stronger, more stable connection.

 

You may or may not notice an increase in speed, as your speed is limited to the bandwidth you are paying for and getting from your internet service provider.

 

In other words, if you have a 20 MBPS download speed package from your ISP, if you are connected to a gigabit LAN network card and router, you're still only going to get 20 MBPS.  The gigabit transfer comes in handy if you have two PC's connected by wire to your router and want to transfer files between them.

 

There should be nothing you have to do. Plug in the network cable, and bingo.

 

If you want to switch the wireless card off to save battery power if you are not connected to A/C, that is fine, but you can normally just plug in the wire and be connected.
