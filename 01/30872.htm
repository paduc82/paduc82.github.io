    jamescadd say:

    I hit the same issue that you did originally.  I finally found a way to at least get the monitor to show up in device manager as ASUS PB278 - no idea if this is a good or proper setup but here's what I did:

    1. Follow ChrisPapadas' steps to download the WHQL driver (note you have to choose 8.0 not 8.1 on the ASUS download page).

    2. After download make sure to right click and unblock the zip before extracting (not sure if important).

    3. Extract the zip and right click the .inf file and choose "Install"

    4. You will have to agree to install an unsigned driver

    5. In device manager right click the generic display, choose update driver software.

    6. Browse my computer

    7. Let me pick from a list

    8. ***Uncheck the "Show compatible hardware" checkbox***  - this is key

    9. Look through the list for ASUS PB278, select it and install.

    Now it should at least show up in device manager as a recognized device.  Again I'm not positive that the drivers are working properly as I just tried this out so YMMV.

    Hi,

    There is no way to guarantee complete security on a wireless network. However, there are precautions you can take to help minimize security risks when you use a wireless network.

    If you have a laptop or a PC with a wireless network adapter, you can see a list of available wireless networks and then connect to one of those networks. So, it may be the other Wifi network connections available in your area. A wireless network security key is the password required to be able to connect your PC to a secured access point (ex: router) on a wireless network.

    Whenever possible, you should connect to security-enabled wireless networks. If you do connect to a network that's not secure, be aware that someone with the right tools can see everything that you do, including the websites you visit, the documents you work on, and the user names and passwords that you use.
