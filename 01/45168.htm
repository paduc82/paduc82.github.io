It's unfortunate that you "cleaned" your registry.  That may have caused more problems down the road.  If there's an UNDO function available from whatever utility you used, I would use it.

The nslookup failure almost sounds like a DNS problem, but if your DNS server was bad, I would think that the ping command would also have failed.

Perhaps your hosts file has become corrupted somehow.

  • Click the Start Orb, type Notepad in the Search box, and then right-click on the link to Notepad and select "Run as administrator."
