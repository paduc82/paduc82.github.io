¡firedog say:

I did not actually delete any Hotmail folders ...

[Is] there ... any way in which deleting a folder in WLM could also delete it in Hotmail


The whole point of the DeltaSync protocol that you're using to access your Hotmail account in Windows Live Mail is to ensure that as far as possible the account folders and their contents are identical on both client and server. If you delete a message or a folder in WLMail, it will be deleted in Hotmail at the next sync. This seems to be what happened in your case.

Using DeltaSync, you can save messages locally while not keeping them on the server, by moving them out of an account folder and in to a folder under Storage folders. You can't save messages on the server but delete them locally, e.g. to save space on your disk drive.

[Is] there ... an easy way for me to import the messages (from WLM) to the 'missing' Hotmail folders


All you have to do is re-create the folders in WLMail and populate them. At the next sync, they should be restored to Hotmail. It may not all happen in one go if there are a lot of messages involved, so don't be impatient.
