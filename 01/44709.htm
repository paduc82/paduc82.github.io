canadian tech say:

To the best of my knowledge, DVD's come pre-formatted and do not need to be formatted by you.

It is possible that your DVD drive is failing.   It is one of the most common failure points in both laptops and desktops.

I have been servicing hundreds of PCs for years.  I have yet to see this be a software problem.   CD/DVD drives are delicate and fail often.  

 

Laptops:  Expensive to replace the component because it is a unique part only available from the OEM.  Better solution is to buy an external DVD drive.  Typically under $30, quite portable and small.  Buy the one that does not require a separate power source

 

Desktops:  Easy and cheap to fix.  Cost about $20,  Quite universally shaped and sized.

  1. Pull the power cord
  2. Lay it on the side down where the wires are
  3. Hold the ON button in for at least 7 seconds
  4. Open the side on top -- usually 2 Phillips head screws
  5. You may have to remove the front too.  Typically it is held in place by clips that you will see on the inside edge.  Just carefully lift those clips to release the front panel
  6. Undo the 2 different plugs in the back of the CD unit
  7. Remove the 2 Phillips head screws that hold it in the chassis
  8. Slide it out.  It is considered hazardous waste.
  9. Reverse the procedure to install the new one.
  10. Windows will install the driver itself -- you do not need to install any software.

License Transfers - New Consoles



You can transfer your content licenses to your new console by following these steps on the console you wish to transfer the licenses to:

 

  1. Navigate to the Settings Hub and select Account.
  2. Select License Transfer.
  3. Carefully read the prompt and select Continue.

 

The License Transfer process can also be completed online here. Please be aware however that this process can only be completed ONCE EVERY 4 MONTHS

 

  • As said, this can only be completed once every 4 months, so DO NOT transfer your licenses onto a secondary console unless you wish that console to be the primary console to use your DLC.

 

  • The point of the license transfer is so you can play your content whenever you wish, whether that be offline or online. If you are on a console without the licenses, the account will not be able to use the content unless it is connected to LIVE.

 

  • Any console with the account that purchased the DLC will be able to access the DLC as long as that account is on the console, regardless if the licenses are on that console or not.

 

  • Any other profile that is on the same console as the primary account that purchased the DLC will also be able to access the DLC as long as the primary profile remains on that console.

bossdweeb say:

How much free space is created by Disk Cleanup is entirely dependant on how often your run it, and when.

Disk Cleanup generally only cleans up a large amount if you have recently done an Win10 version update, and have run

the Cleanup System Files option.

Reason being, the version updates create download files and create backups that you can delete once you are satisfied the version

update has not stuffed up your Windows. Once done there should be no large amount to be 'cleaned' until the next version update.

200mb would be a fairly large cleanup under normal circumstances if you have not run Disk Cleanup for some time, particularly

if you never empty the Recycle BIN manually, in which case it could be more than 200MB.

- Your secondary HDD would rarely have much, if anything to be cleaned up as your games and fixed data such as video, music don't

create junk files. The main thing that needs be done with a secondary HDD is that it gets defraged.

Disk Defrag can be set to be done automatically, but I like to run it manually after installing a game. A well defraged and optimised

(files sorted contiguously on the platter by the defrag tool ) reduces the likelihood of disk 'thrashing' (when the laser is forced to

have to do a lot of moving to read the files).
