Windows 7 Back-Up onto External Harddrive vs System Repair Disk

lemp say:

A "system repair disk" is a CD (or DVD) that will enable you to boot up your computer and potentially fix problems when you can't start it up normally.  It is not a backup of anything.

Yes, you should have one of these.  You should make one now for Windows 7 and then make another after you successfully install Windows 10 (System Repair disks are specific to the operating system).

Windows 7 ("system repair disc"):  https://support.microsoft.com/en-us/help/17423/windows-7-create-system-repair-disc

Windows 8/Windows 10 ("recovery drive"):  https://www.pcworld.com/article/3140449/windows/everything-you-need-to-know-about-windows-10-recovery-drives.html

Windows 8/10 ("system repair disc"): https://www.howtogeek.com/131907/how-to-create-and-use-a-recovery-drive-or-system-repair-disc-in-windows-8/

grumpyofthehomecounties say:

Thanks for that.

I am happy to make two system repair disks as you suggest, but what capacity would be required?

Please be aware that the laptop in question has an 80GB capacity.

lemp say:

As I said, a system repair disc isn't a backup, so the size of the hard drive in the system is not relevant.

If you're using optical media (CD or DVD), you need one disc for each "system repair disc," but the actual files on the CD occupy less than 200 MB (for a Windows 7 system repair disc).

I haven't created a Windows 10 "repair disc," but again, it should fit on one optical disc.

As explained in the articles I linked, Windows 10 has the option to create a "system recovery drive" that includes some components of the operating system.  This allows you to repair/reinstall Windows should that become necessary.  As a result, if you choose to make a "recovery drive" rather than a "repair disc" you'll need to use a USB flash drive that has a capacity of 8 to 16 GB, depending on your particular system.  As part of the creation process, Windows will analyze your system and tell you how large a USB flash drive you need.
