Let's start with the three easiest potential solutions.  If one doesn't work, go to the next.

A.  Download and run HP's Print & Scan Doctor:  http://ftp.hp.com/pub/printers/hppsdr/HPPSdr.exe

B.  Remove and reinstall the printer

  1. Make sure that there are no pending print jobs (right-click on the icon in Devices and Printers > See what's printing)
  2. Disconnect the printer from the computer
  3. Right-click the icon and select "Remove device"
  4. Reboot the computer
  5. Connect the printer, turn it on, and let Windows reinstall it.  This will use the existing driver.

C.  Remove the printer and its driver and reinstall

  1. Make sure that there are no pending print jobs (right-click on the icon in Devices and Printers > See what's printing)
  2. Disconnect the printer from the computer
  3. Go to Control Panel > Programs and Features and remove any HP software
  4. If the printer's icon still is in Devices and Printers, right-click the icon and select "Remove device"
  5. Select any other icon under "Printers and Faxes" (for example, the "Microsoft XPS Document Writer") and then click the link to "Print server properties" that will appear in the toolbar at the top of the Devices and Printers window
  6. Select the "Drivers" tab and "Remove" any drivers related to the HP Deskjet 2130 Series.  Remove both the driver and the driver package
  7. Reboot the computer
  8. Download the driver and related software from https://support.hp.com/us-en/drivers/selfservice/hp-deskjet-2130-all-in-one-printer-series/7174550/model/7174551 Before downloading, double-check to make sure that the web page correctly identified your operating system, including whether it is 32-bits or 64-bits.
  9. Install the software you downloaded; don't connect the printer until prompted to do so by the installation routine.

There are additional measures to take if none of the above processes work, but they're a bit more complicated.
