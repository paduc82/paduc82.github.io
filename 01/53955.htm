More on RAID :


RAID often causes more issues than it solves. Especially for software RAID which is included on some motherboards and Windows.

RAID
http://en.wikipedia.org/wiki/RAID

Today there are 3 "General Types" of RAID

Software, Hybrid (some hardware with software checks), and Hardware.

Personally I think the Software and Hybrid are a waste of time and misleading when claims of reliability are usually made. In most cases an effective backup plan is better than these. All too often the RAID software and/or Hybrid model actually cause a failure. Or what is supposed to protect the system in drive crashes and failures ends up being the cause of them. Hardware RAID is by far the best.

Hardware RAID
http://www.bestpricecomputers.co.uk/glossary/raid-controller.htm

Hardware RAID is a physical hard drive controller that provides the drive interactions needed to perform a reliable redundant array.

Intel RAID controllers
http://www.intel.com/products/server/raid/index.htm

Adaptec RAID controllers
http://www.adaptec.com/en-US/products/

Many low cost add-in hardware RAID controllers are really a Hybrid where some of the action is done by the hardware card and the rest (and often most important functions) are performed by the CPU of the system using software RAID.

Windows 10 Home Password issue - Microsoft Community

marknyy say:

Windows 10 Home Password issue

Windows is telling me my password will expire in 5 days. Since I'm on windows 10 Home Edition I can't get to the settings for complexity and length. Any ideas?

a. user say:

A default install of Windows 10 Home or Pro doesn't put a time limit on passwords. That has to be setup after Windows has finished installing. So the question is: Who setup the time limit and how did they do it?

There are at least two ways to enforce or remove the time limit on a password (and probably more) which is known as the passwords age. You'll need Administrator permissions to do this.

One way is with the Local Security Policy snap-in. Search for Local Security Policy and press Enter. When the Local Security Policy window opens choose Password Policy underneath Account Policies (on the left). One of the choices on the right is Maximum password age. Double-click on that and choose 0 days.

The other way is with an elevated command prompt. Type cmd in the Search and press Enter. Right-click on cmd and choose Run as Administrator. Type the command

net accounts /maxpwage:unlimited

arminatorx say:

The console does not "read" Xbox 360 discs. 

It simply detects the copy protection on the Xbox 360 disc, and then downloads a backwards compatible emulation image from the Xbox marketplace. 

So make sure your console is connected online and the disc is detected. 

If the disc isn't detected properly, try to eject the disc and hold the power button on the console for about 10 seconds. 

Restart the console again, and make sure the console shows you a green boot screen. If not, hold the power button on the console for 10 seconds again. 

Then insert the disc again, and check if the console now detects the disc properly, and downloads the backwards compatible version.
