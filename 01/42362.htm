My (nonexpert) intution says yes, since the image is not a physical copy of the fragmented source.  So when you extract the contents back into a partition, it would make sense that the contents are placed contiguously.  However, I haven't been able to find confirmation of this through surf-fu.
trekdozer say: Unless the imaging program has a defragment feature - no, restoring an image does not result in the files being defragmented. The built in Windows 7 system image program doesn't defragment.

In general restoring a system image results in the files being in the same location they were originally. Although one exception to that would be if you restored it to a smaller partition then the original partition, but even then the files would not be defragmented.


If you want a defragmented image, defragment the hard drive before creating the image.
