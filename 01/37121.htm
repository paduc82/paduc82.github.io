If drivers for Windows 8 are not available then there is option to download the drivers designed for Windows 7 and install them on Windows 8 using the compatibility mode.

 

Please note that this method is not a guaranteed workaround as the working or failure of the device depends on the hardware manufacture design of the drivers however you may give it a try.

 

Follow the steps to install drivers in compatibility.

 

a)      Enter the Start screen

b)      Press Windows key + C on your keyboard to show the Charms bar (If you have a touchscreen: Touch the right edge of your screen and slide your finger to the center of the screen to show the Charms bar)

c)       Search for Troubleshooting, Click Settings in the right column.

d)      Click Troubleshooting. Click Run programs made for previous versions of Windows

e)      Click Next, select Not Listed. Click Next

f)       Click Browse and select the installer that you downloaded using the link above.

g)      Click Next, click Troubleshoot program

h)      Check The program worked in earlier versions of Windows but won't install or run now box

i)        Click Next, select the last known working operating system for this driver (probably Window 7)

j)        Click Next

You may be viewing compressed files showing in blue color. You can change that color by turning off "Show encrypted or compressed NTFS files in color".
1. In Explorer "Folder Options" menu, click on the "View" tab.
2. Under "Advance settings", "Files and folders", scroll down to "Show encrypted or compressed NTFS files in color".
3. Check to clear the box before it.
4. Click on Apply and OK buttons.
