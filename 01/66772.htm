lemp say:

I've seen too many reports of problems with the backup system available in Windows XP ("Windows Backup" or ntbackup.exe) to suggest that anyone rely on it.  http://en.wikipedia.org/wiki/NTBackup

That said, almost any backup system will implement at least three types of backup schemes:  full backup (which is what it sounds like), incremental backup, and differential backup.

Incremental and differential backups are similar.  Both start with a full backup. 

An incremental backup creates backups of files that have changed (or are new) since the last backup.  In this case "the last backup" includes prior incremental backups.  For example:

Full Backup 1:  backs up everything

Incremental Backup 1:  backs up files that have changed or are new since Full Backup 1

Incremental Backup 2:  backs up files that have changed or are new since Incremental Backup 1

Incremental Backup 3:  backs up files that have changed or are new since Incremental Backup 2

etc.

A differential backup creates backups of files that have changed (or are new) since the last full backup.  For example:

Full Backup 1:  backs up everything

Differential Backup 1:  backs up files that have changed or are new since Full Backup 1

Differential Backup 2:  backs up files that have changed or are new since Full Backup 1

Differential Backup 3:  backs up files that have changed or are new since Full Backup 1

etc.

The main difference is that each incremental backup is smaller than the corresponding differential backup, but when restoring an entire system, you need the last full backup and all the intermediate incremental backups but only the latest differential backup.
