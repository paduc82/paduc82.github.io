Well, OEM versions of Windows 7 are really meant to be sold to System Builders building new PC's. Its not an enforced restriction though. OEM versions of Windows 7 are identical to Full License Retail versions except for the following:

- OEM versions do not offer any free Microsoft direct support from Microsoft support personnel

- OEM licenses are tied to the very first computer you install and activate it on

- OEM versions allow all hardware upgrades except for an upgrade to a different model motherboard

- OEM versions cannot be used to directly upgrade from an older Windows operating system

You don't encounter the following restrictions above the retail versions, thats why Microsoft recommends that it be sold to system builders instead of the general public, but there is nothing other than preventing you from using it. In fact, its even more affordable.

I have now established that the contacts that reappear are stored somewhere on my laptop,

Contacts you see in the Windows Live Mail contacts window are always stored on your computer. In your case, there is some dysfunction in the synchronization process that tries to ensure that the contacts you see in Windows Live Mail are the same as those you see on the People page. I don't know, but this might be a result of using an obsolete version of the program. On Windows 7, you should be using Windows Essentials 2012.

However, we can probably cure this by forcing the program to download the latest version of the contacts list at People. To do this,

  1. Add a New contact on your People page that currently only contains Echo123.
     
  2. Sign out of Windows Live Mail 
     
  3. Close down all programs (at least, all those that might be accessing your Windows Live Contacts - e.g. OneDrive, Movie Maker, Photo Gallery ... )
     
  4. In Windows Explorer, navigate to the folder containing your contacts databases. This will normally be
    C:\Users\{user}\AppData\Local\Microsoft\Windows Live\Contacts
     
  5. Right-click on the folder named with the email address of the Microsoft Account you sign in with and select Rename. Remove the '@' symbol from the folder name.
     
  6. Launch Windows Live Mail. When the default contacts list has loaded (it's empty in your case), Sign in.

This should re-create the contacts database for the account concerned, with only two entries.
