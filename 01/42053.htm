[SOLUTION] Windows 10: How To Keep Preview Pane The Same - Microsoft Community

andrewx3n say:

[SOLUTION] Windows 10: How To Keep Preview Pane The Same Size [File Explorer]

It took me awhile to track down a solution to this, and it was buried in some other thread's responses. So I want to post this here simply to be easily available to anyone who Google, or Bing (lol kidding) searches for a solution. This is apparently a pretty common issue that has seen not a single Microsoft fix in over 6 years. 

Here's how to fix it:

1.) Close all file explorer windows

2.) Open one file explorer window

3.) Make the window re-sizable (not maximized)

4.) Click and drag on the edges to make it full size (not using the maximization button)

5.) Adjust the preview pane to how you want it

6.) Close that file explorer window

7.) Open another one (or the same one)

8.) Maximize the file explorer window with the maximization button

And it should now save in that position when you open new file explorer windows.

Don't allow the MS Updates driver to install over any other version of the ATI driver except other MS provided drivers. They aren't compatible, as you've seen. The HP drivers are based on ATI drivers that were current at the time your PC was being configured by them. To update them either use newer HP drivers if they have such available or download the latest ATI drivers for your card/graphics chipset from AMD. Uninstall the existing driver before installing the new one. The full Catalyst ver. 10.12 happens to be available in 2 versions but the actual drivers are the same in both. The larger download has a new Catalyst Control Center while the smaller one is the older version CCC. (MS drivers have no CCC version)
