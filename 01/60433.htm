cbarnhorst say:

As Andre said, there is no such thing as upgrade and full versions of the dvds.  The dvds are identical for all retail Home Premiums, upgrade or not.  (The same with Professional, the same with Ultimate, and so on.)  It is only the product key that determines whether the installer on the dvd will require the presence of a Windows installation to be upgraded.  The upgrade PK tells the installer to search for a copy of Windows to be upgraded.

You cannot do an in place upgrade with an OEM dvd.  OEM means "Original Equipment Manufacturer".  They are legitimate copies of Windows with perfectly legitimate product keys.  But you cannot use them to upgrade.  The upgrade functionality is not there.  That is what is meant by "for installation on a new computer only." 

The OEM product key requires you to start the installation by booting with the dvd and only a custom installation (aka clean install) is available.  Also, the OEM license only grants the purchaser the right to install the software on a new computer being built for sale to customers.  The OEM license does NOT grant the purchaser the right to run the software on his own computer.  OEM licenses are called System Builder licenses.  The packs on Tiger Direct are meant for use by those who custom build machines for customers.  Outfits like mom and pop stores, consultants, etc.  It is not legitimate for consumers to install them on their own DIY computers.  That is what retail copies of Windows are for.  Read about OEM licenses here:

http://www.microsoft.com/oem/en/licensing/sblicensing/Pages/index.aspx

In particular review the FAQs.

The secondary function (such as Volume Up/Down, brightness etc.) is referred to as the Action Key mode.

Switching between Action Key Mode and classic mode (where the Fn Key is required for an Action) is normally controlled in the BIOS. With Action Key enabled secondary functions are the default, with it disabled then normal operations take priority (F1 = Help).
