How to schedule to chkdsk on weekend automatically? - Microsoft Community

oemsoft say:

How to schedule to chkdsk on weekend automatically?

I would like to set schedule to perform chkdsk on sunday automatically.

Does anyone have any suggestions on how to do it?

omniplex say:

Do you have any suggestions?


As HairyFool said, if you have no hard disk problems running chkdsk weekly is a waste of time and not necessarily good for your disk, but if you have a real disk problem, you'd run chkdsk as soon as possible.  

If that's clear, of course you can create weekly jobs, open the task scheduler:  Enter taskschd.msc /s on a command line or in the Win-key+R "run"-box, or find it in the control panel.  (On my German Windows it's in "Verwaltung - Geplante Aufgaben", this might be "administration - scheduled task" or similar for you.  Win 7 home premium won't let me install an en-US UI to figure out the correct term ;-)

Once you are in the task scheduler enable the "action" panel (the rightmost column of three), there you can start to create simple tasks.  Try something obvious first, e.g., create a job starting calc.exe.  If that works as intended modify it into your weekly chkdsk /f magic.   

If your PC is not running at the planned time you'll need some advanced options to wake it up, otherwise your chkdsk could run as soon as possible after the planned time, exactly when you want to use your box for something else, so that won't do.  But you can plan a job running only if the system is started.   

Presumably your job will consist of two parts, a command line with chkdsk /f on C: to set the "I want a chkdsk at the next reboot" flag on C:, followed by the reboot -- for details see shutdown /? --; both commands separated by &&.   Try to start calc && notepad in one job, and if that works, replace it by chkdsk && shutdown.
