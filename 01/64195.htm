Where Event 41 reports are coupled with Event 1001 reports you have a blue screen error and, providing the system is configured to produce dump file, a dump file is generated.

When you get Event ID:41 errors without the corresponding Event 1001 report, you have a simple unexpected system failure and no dump file is produced.
