Hello Mike-Not-Michael,

Have you just activated Windows (which you must in order to use it) or did you also register the software on the Microsoft website?

Registration is optional and, frankly, even I don't register my Windows operating system products.

If you 'registered' the product then only Microsoft knows who you are and where you live - no one else will find out your details from Microsoft.

When you activated your copy of Windows Microsoft simply took an inventary of the hardware on your PC, gave it a specific identity number and Windows simply checks for that number every time you boot up.

If you want to give the PC away to charity then format the hard drive and re-install Windows Vista. When it comes to the point of asking for a user name etc, simply type in Admin or Adminstrator - do not use your own name. When the charity start to use it all files will be named as 'administrator' rather than 'Mike'.

With you talking about re-installing Vista I am assuming that you have a Windows Vista DVD from which you can install the operating system? If you don't and you are your PC came pre-installed with Windows, then the chances are that there is a hidden image on your hard drive which allows you to restore back to factory conditions. If this is the case then simply follow the instructions in your PC user manual to restore back to factory conditions. After the restore has completed, boot up the machine and, whne asked for your computer name and other personal details simply typoe in Admin or Administrator - or Owner if you wish.

Explorer

Explorer (explorer.exe) is the program that is the Start menu and taskbar, folder windows, and the desktop (which is a folder window).

 

To See if a Fix is Available

In Control Panel (and select Classic view in the left hand pane) choose Problem Reports and Solutions (type problem in Start's search box), go to Problem History, right click your error and choose Check For Solution.

 

You may also right click and choose Details for more info. Post those details here. The Fault Module Name is the important information.

 

If the problem affects Control Panel press Winkey + R and type wercon (or type it in a command prompt).
 

Close Explorer and Start a Command Prompt

  • Close any Explorer windows
  • Start - All Programs - Accessories - Right click Command Prompt and choose Run As Administrator.
  • Click Start. Ctrl + Shift + Right click a blank spot (just above the power buttons is one place) then Exit Explorer.
  • Press Ctrl + Alt + Delete then Task Manager. 
  • Check all explorer processes are closed. On the Process tab select explorer and right click and choose End Process, repeat if more than one explorer in the list.

 

Then to restart explorer after trying each of the following

  • Press Ctrl + Alt + Delete and choose Task Manager 
  • In Task Manager click the File menu then New Task (Run) and type explorer
 

If You Can't Start Explorer at All

 

  • Press Ctrl + Alt + Delete and choose Task Manager 
  • On the Process tab click Show Processes From All Users to elevate to Administrator
  • In Task Manager click the File menu then New Task (Run) and type cmd
  • Other things you can try typing

    Explorer /e,c:\
    wercon
    control
    iexplore
    rstrui

1) How is your PC connected? Is it Homegroup/workgroup?

2) What is the exact error you get when you try to share files with PC2?

I suggest you to follow the below steps and check if it helps.

1) Go to each computer in the homegroup and delete the following directory

 C:\windows\serviceprofiles\localservice\appdata\roaming\peernetworking
 (This directory is protected, so you will have to confirm when you try to access it)
 Shut the computer down and leave it off

2) On a Windows 10 desktop connect it to your network.
Setup a new Homegroup on your network
Write down your password (At this point, you will need to put that password in a safe place)

3) Go to each computer and add them to the new Homegroup.

4) Test and make sure you can get to your devices and directories

If you cant from one computer, leave the Homegroup and re-add the computer and try to share the files.

Hope this helps. Please reply to us with the status of the issue.
