poohdette say:

Hard Drive Died on Windows XP computer and Need Recovery Disc

If the hard drive died, is there still a way to connect it to something and pull the XP recovery program from it.  It seems you can upgrade a Windows XP to Windows 7.  Is this true?

lemp say:

If the hard drive really "died," then the only way to get data from it -- if that is even possible -- is through very expensive forensic methods that require a clean room and considerable expertise.

If the problem is not the drive itself but something such as the drive controller on the motherboard or the power supply, then you can remove the drive from the computer and put it in an external enclosure.  In this case, you can get the data relatively easily, but you can't use anything on the drive to "recover" XP to a different machine (a "recovery program" implies that the computer came with XP pre-installed and included a mechanism to restore the computer to its factory condition; the license for such computers is tied to the original computer).

Although Windows XP was a "qualifying product" that entitled you to use a Windows 7 "upgrade edition," you could not do an in-place upgrade from XP to 7.  A "clean install" of Windows 7 was required.  I used the past tense in the preceding sentences because finding a legitimate copy of Windows 7, upgrade edition or not, is very difficult these days.

jimwynne say: If the drive is dead, it's dead. 

lemp say:

There are reasons why a computer might not recognize a hard drive other than the drive itself having failed.

There are a couple of things you can do to diagnose things, but if you have an "IT person" who already has offered an opinion, option 1 would be to ask that person to figure out what is wrong.

Assuming that option 1 doesn't work or isn't available, and further assuming that this drive is the system drive on the computer which therefore doesn't boot to Windows, the thing to do would be to run disk diagnostics provided by the manufacturer of the hard drive.

If you have opened the computer and know what the make of the hard drive is, go to the web site of that manufacturer and look for downloadable disk diagnostics.  Or if you look up the specifications of the computer at the computer manufacturer's web site, that may tell you what make hard drive you have.

Two of the most common hard drive manufacturers are Western Digital (includes Hitachi/HGST)and Seagate (includes Maxtor and Samsung), so if you are unable to determine the make of your drive, download

1.  SeaTools for DOS http://www.seagate.com/support/downloads/item/seatools-dos-master/

2.  Western Digital Data Lifeguard for DOS http://supportdownloads.wdc.com/downloads.aspx?DL

Follow the instructions at the download site to create bootable media.  SeaTools is probably easier and more universal.  You download an ISO file and create a bootable CD.  If necessary, you can use (free) ImgBurn to do this.  The WD download is much smaller (it's an exe file) and the download page explains how to create a bootable USB flash drive.

Put the CD in the drive or connect the USB flash drive and boot from it.  You may have to configure the BIOS to boot from the appropriate device first.  Then run the diagnostic.

What you do next depends on the result of the disk diagnostics.

Yes, you can. I also find it quite annoying as well as I use my PC as a media extender and is impacts the quality of playback when it kicks in.

 

Create two batch files (startindexer.bat and stopindexer.bat)

 

In stopindexer enter the line

net stop wsearch

 

In the startindexer enter the line

net start wsearch

 

Now create 2 scheduled tasks one to run the first batch file that stops the service every day at a given time. The other to start it back up again at a time you won't be using the computer. Point to note. You have to run the scheduled tasks as the system account otherwise you probably won't have rights to stop and start the service.

 

It works fine.
