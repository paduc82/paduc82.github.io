cristhianescobarpq say: Hello, I am an independent consultant willing to help.

Windows 10 in S mode is designed to offer security and performance and runs exclusively in Microsoft Store applications. If you want to install an application that is not available in the Microsoft Store, you must exit the S mode. Exiting the S mode can not be reversed. If you make the change, you will not be able to go back to Windows 10 in S mode. There is no charge for exiting S mode.

1 On your PC, running Windows 10 in S mode, open Settings> Update and security> Activation.

2 In the Switch to Windows 10 Home or Switch to Windows 10 Pro section, select Go to Store. (If a "Update Windows Edition" section also appears, be careful not to click on the "Go to Store" link that appears there).

3 On the Exit S (or similar) mode page that appears in Microsoft Store, select the Get button. When you see a confirmation message on the page, you can install applications from outside the Microsoft Store.

Source: https://support.microsoft.com/en-us/help/445606...
