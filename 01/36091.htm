a. user say:

Too bad you did not say what the trouble is or anything about your system.

FYI - The Volume Shadow Copy Service is not supposed to be set to Automatic start.

The VSS Service gets installed as Manual start which means it will start when it needs to start and stop when it needs to stop - that's how it works.

Whenever there is any kind of VSS error message the Microsoft engaged alleged "experts" invariably will suggest changing the Service to Automatic but that is a wild goose chase idea. 

If the VSS Service was supposed to be set to Automatic it would get installed that way.

If having the VSS Service set to Manual was wrong (the way it gets installed) then millions of installed systems all over the world would be configured incorrectly and malfunctioning - and they're not.

Setting VSS to Automatic means it will be configured improperly and will then be running all day and night with nothing to do wasting your system resources which you would probably rather have being applied to more important things - and it will not solve your unspecified problem.
