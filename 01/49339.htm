Firmware Update

Login Information

  • Gateway - 192.168.2.1
  • Username - N/A (None)
  • Password - N/A (None)

Firmware Update

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open a web browser.
  3. In the address bar at the top, type in 192.168.2.1
  4. Click Login in the upper right.
  5. Input the password and click Login (by default, the password should be left blank).
  6. Select Firmware Update from the menu on the left under the Utilities section.
  7. The firmware version should be listed at the top of this page.
  8. Open another tab in the web browser.
  9. Go to www.belkin.com.
  10. Scroll down and click on Support.
  11. Enter the model number of your device in the search box.
  12. Scroll down to the "Downloads" section under the search results.
  13. Click the link with the model number of your device labeled Firmware Update.
  14. Locate the version of the router you have and click where is says download to download the latest version of the firmware.
  15. Save the file to somewhere that is easy to locate such as the desktop.
  16. Go back to the original tab that you opened the router's GUI page with.
  17. Click on browse.
  18. Find the firmware bin file and click open.
  19. Click Update and let the software install without interruption.
