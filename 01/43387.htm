a. user say:

Hello Kolby,

 

Thank you for posting on Microsoft Community. We regret the inconvenience caused and will assist you in resolving the issue.

 

An administrator account is a user account that lets you make changes that will affect other users. Administrators can change security settings, install software and hardware, and access all files on the computer. Administrators can also make changes to other user accounts.

 

You will be able to delete another user account only if you are the administrator with complete user rights on the PC. If you are using a local or administrator with limited user rights account, you will not be able to delete the account unless you know the administrator password of the actual administrative user profile.

 

Since you do not know the password of the Administrator user profile, you will not be able to make any major system changes on the PC.

 

Step 1: If you get to know the administrator password, I suggest you to convert the user profile to administrator profile by following the steps below:

 

  1. Press Windows logo key on the keyboard, type Control panel and click on the top most search result.

  2. Select User Accounts or Change account type from the window and click on your user account name.

  3. Select Administrator and click on Change account type.

  4. Close the window and check if the changes are effective.

 

Step 2: Follow the steps below to delete the user profile:

  1. Press Windows logo + X keys on the keyboard and select Command prompt (Admin) from the context menu.
  2. Enter the administrator password when prompted and click OK
  3. Enter net user and press Enter. You will now see all user account in the system.
  4. Then type net user accname /del and press Enter. For example: If you want to delete the user account named computer, then the command line net user computer /del.
  5. Reboot the PC and check if the changes are effective

Write to us with the status of the issue on the same post for further assistance. Your reply is most important for us to ensure we assist you accordingly.

 

Thank you.
