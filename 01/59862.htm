Printers are not "64-bit" or "32-bit" although printer driver software may come in those versions.

Windows XP 32-bit (the normal, most common edition of Windows XP) includes a driver for the HP IIp.  I can't recall if this printer uses a USB connection or a parallel connection.  If it uses USB, you should be able to just plug it in and Windows should recognize it and install the proper driver.  If it uses a parallel connector, use the Add a printer wizard from Start > Printers and Faxes.  See
