1.       What is the model of the computer?

2.       Does the computer have the feature of touch screen?

 

You should be able to use the touch screen feature when you install the operating system.

 

If you are unable to use, then I would suggest you to update the driver for touch screen.

 

Method 1: Update the driver

 

  1. Hover the mouse to the top right hand corner and click on search
  2. Type devmgmt.msc and hit enter
  3. Now locate the touch screen device, right click on it and click update.

 

Method 2: Install the driver program in compatibility mode

 

If the issue persists, then I would suggest you to install the driver for the operating system of lower version in compatibility mode.

 

a. Right click on the setup file of the driver and select Properties

b. Select Compatibility Tab

c. Place a check mark next to Run this program in Compatibility mode and select the operating system accordingly from the drop down list

d. Let the driver to install and then check the functionality.

 

If there are no touch drivers, then I would suggest you to contact the manufacturer.

 

Hope this helps. If the issue is still not resolved, come back and I will be glad to help you.
