Christoph,

There are different date-time stamps [Created, Modified, Accessed] and I do not know which ones you are referring to.  By default, File explorer will just show a column for Date [unspecified] and that can cause some confusion.

If you are working in File explorer then right-click on a column header and add Date modified & Date created as columns until you get used to things.  Ditch the column for Date.  Don't bother with Date accessed because it can be a pain as it only works as a column entry under particular conditions.

If you copy a file from your computer to the memory stick then

  • both will initially have the same Date modified.
  • the copy on the memory stick will have a Date created that matches the date-time you copied it.

Note this potentially confusing aspect for large files that take some time to complete copying

  • while copying is in progress the copy on the memory stick will have a Date modified that matches the date-time you started copying it.
  • when copying is completed, its Date modified will revert to its true value.

Christoph,

I think you are correct about previous OS versions showing Date modified by default in File explorer, Details view.

I have never been found out what the Date entry is supposed to be showing.  It sometimes seems to match one of the others but then fails to match them in other examples.  It might well be no more than a relic of, say, Windows 95 that they have not bothered to remove.

You can set up default folder views for all your folder types [Music, Documents, Videos, General, Pictures] that include Date modified if you wish.  [I cannot find a procedure for this written out already but I bet there is one somewhere] 

For each folder type [Music, Documents, Videos, General, Pictures] -

[none of this is destructive so you can play around as much as you like]

01  Find a folder of the type desired [Music, Documents, Videos, General, Pictures] or create a new folder temporarily to use in this procedure,

02  In File explorer, right-click on the folder name, Properties, Customise - set this to the type desired if it is not set already [including sub-folders if relevant to the case in hand],

03  In the bottom-right of File explorer's right-hand pane, click on the Display information about each item icon,

04  In File explorer's toolbar, select the View menu then, in its Layout section, select Details.

05  In File explorer's right-hand pane, right-click on any column header then select More... at the bottom of the list,

06  Choose the columns you want to see then click on OK.

07  Drag column headers across each other to get them into the layout you want.

08  Grab the separator lines between the columns and drag them to set the required width for each column - the default width for Name is likely to be too narrow whilst the default widths for Date modified, Size, Attributes are likely to be too wide & hence wasteful of your limited screen space. 

09  Sort the entries the way you want by clicking on the relevant column header.  You'll see a little v symbol [descending order] or a little upturned v symbol [ascending order] appear at the top of the column you have sorted by.  I can never remember which v means what so I always have to look at the entries in the folder I am working in to check they are sorted the way I want.

10  Personally, I set all this up in a maximised window then I use WinKey-LeftArrow to show it occupying half the screen width so I can check that what I have set up is sensible for both these common ways of seeing File explorer windows.  Even in the half-size window, I like to be able to see the Name & the Date modified columns because I find that avoids a lot of scrolling around whilst working.

11  In File explorer's toolbar, View menu at the right-hand end, click on Options then select the View tab of the dialog that appears.

12  Click on Apply to folders to set all folders of the same type to match the layout you have just designed.

Because setting up folder types is a bit time-consuming & can get reset by some Windows updates, I backup the settings.  See Backup and restore Default folder views - Wiki

bilo cape town say:

Hi. Many techs don't know all the facilities provided by the Disk Manager.

If the drive has been properly connected but does not appear in Disk Management - it has not been automatically detected.

Click on 'Actions' in the menu at the top - and click on 'Rescan'.

It may not show up on the first try. Repeat the Rescan a few times until the drive is detected.

If it shows up - RIGHT click on the band relating to that drive.

A menu should appear that lets you make it a 'Simple volume'.

If it displays a title in the band but you cannot get the menu to make it available to your system

it may be because it is a drive that was used on another system.

On the extreme left of that window it normally shows the drives as Disk 0 - Disk 1 etc with the 'Type'

as Basic / Dynamic.

The Boot Drive will always have the System Partition shown as C: in the band on the right.

The drive you have added may show as 'RAW' or unallocated space.

Now RIGHT CLICK on the Disk No. of the drive you have connected.

In a small menu that shows up it may indicate the drive as 'FOREIGN'.

It will also ask if you wish to 'Import this drive'.

When it is 'imported' it may show any partitions that were previously created.

Right click on each partition and assign a drive letter to each one.

If it shows as 'Unallocated space' in the band on the right - it does not have any partitions to which data can be written.

RIGHT CLICK in the band where it indicates 'Unallocated space'.

In the menu that shows up - select 'Simple volume'.

Options are presented with a default available Drive Letter (D / E / F)

which you can also change.

The default size of the partition is the total of the free space but you can

reduce the size to your needs - and it will leave the balance as unallocated.

Repeat the process of RIGHT CLICKING on the remaining unallocated space

to create another 'Simple Volume'.

It will format the new partition (NTFS) and show up with it's Drive Letter.

When you have completed this process - open the File Explorer and the new partitions should be seen when clicking on 'Computer' (Windows 7) or 'This PC' (Windows 8.1)

The new Drive/s can now have files copied or created in them or folders created with

relevant topical names eg Documents / Music / Photos etc to which your files can be copied

or in which new files can be created.
