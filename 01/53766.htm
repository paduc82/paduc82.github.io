bruce hagen ~ mvp say:

A simple solution. You have to tell Windows Search to stop indexing OE.

Windows Search. Set Desktop Search Options
http://www.microsoft.com/australia/windows/desktopsearch/search/options.mspx

In the Windows Control Panel | Indexing Options | Modify. Clear the check box for Outlook Express.

Bruce Hagen
MS-MVP Oct. 1, 2004 ~ Sept. 30, 2010
Imperial Beach, CA

rudy_g say:

Hi Bruce,

Thanks so much for the tip. While it did not exactly work like I expected (the path via control panel>Indexing Options>Modify did not solve it), I had to open the OE folder from WinExplorer>Properties>Advanced and unchecked the indexing box there. That stopped the problem.

After I followed your directions I did not re-start the machine, so I am unsure if it would have required that action or not. But the indexing check box in the OE folder was definitely not unchecked.
