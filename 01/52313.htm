zeke009 say:

I've read the thread a few times and I'm still unclear why you are looking for 32bit media when you are building a PC with a 64bit OS.


However, I've reimaged many a device using a USB drive. So, either grab your DVD or your ISO file and head on over to a working PC.


  1. Install this tool: http://images2.store.microsoft.com/prod/clustera/framework/w7udt/1.0/en-us/Windows7-USB-DVD-tool.exe
  2. Plug in your 4GB or greater USB drive.
  3. Either point it to your ISO or your DVD drive (more info about the tool). I've found the tool to be self explanatory.
  4. Build the media
  5. Take to your new PC and give it a test. Make certain you are either booting from removable media first or you are using the boot select menu.

I've never run in to issues using that tool for Win8 media. But I have to say I have not built the media on an XP machine before. It's always been either Win7 or newer.


I applaud your willingness to learn, but trial and error is the best tool as well as forums. Hopefully this helps and you're able to build your PC before your DVD drive arrives.


slance310 say:

Hi,

 

One thing I thought I should mention that you may or may not be aware of is the file format of the USB drive...

 

If you are planning on using a UEFI BIOS feature on your new machine, the USB requires a FAT32 format to be read by the UEFI setup...

 

The USB tool will create a NTFS format... this is OK for regular install, but won't work for UEFI...

 

I've not done this, but have come across this little detail recently... just a heads up.
