¡firedog say:

A couple of general points.

  • Blocked and safe senders lists are synchronized with the ones in use at mail.live.com for the Live ID you use to sign in with. Signing in or not thus may have a bearing on the lists in use.
  • These lists are also stored in the registry under the sub-key called
    HKEY_CURRENT_USER\Software\Microsoft\Windows Live Mail\PerPassportSettings
    Which of those lists is in use at any given time is a complete mystery, but one thing is sure: any third-party software that claims to optimize or defrag or clean up your registry is likely to interfere with the proper updating of the data under this sub-key.
  • Blocking senders works differently for HTTP and POP3 accounts.
  • The aim of blocking a sender is to keep messages from him out of the inbox. I can't see how this can ever work properly with any given IMAP account, because of the many different ways IMAP servers have of dealing with deleted messages.
  • There is also a mystical file in the store folder that contains rules for spam and safety. It's an XML file called SAS.Rule, and it's far from perfect. It supposedly governs the display of red and yellow nanny bars in messages.

For all these reasons, I can't place any reliance on blocking senders. However, using the Safe List Only option together with trusting mail from contacts seems to ensure that unwanted mail never arrives in my inbox. Regularly dealing with the Junk e-mail folder finds any false negatives.

I don't remember seeing a message like the sender is not in your contacts or allowed senders list. When does it appear?

I can't understand why you would ever end up in situations where the contact was added to the Block Senders List. Contacts should by definition be trusted... 

So, my advice would be: forget about blocked senders and rely on Safe List Only + Contacts, and scrap any third-party 'security' software that does more than monitor viruses and other malware.

Noel

Hi Torey,

Thank you for posting your query on Microsoft Community.

Kindly reply to the following questions:

1. Is the issue confined to a single video or all the videos?

2. Which media player you use to watch videos?

3. If you have more than one media player, did you verify if the video plays correctly on other media players?

I would suggest you to update Graphics card driver and check if the issue is resolved.

Method 1: Update the graphics card driver.

1. Press Windows key + X and click on Device Manager.

2. Expand the Display Adapter.

2. Right-click on the graphics card driver and click Update driver software.

Method 2: Uninstall and then reinstall the graphics card driver.

1. Press Windows key + X and click on Device Manager.

2. Expand the Display Adapter.

2. Right-click on the graphics card driver and click Uninstall.
3. Restart the computer, let Windows detect the driver and install by itself. 

I would recommend you to refer to the article by Andre Da Costa and try the mentioned steps.
