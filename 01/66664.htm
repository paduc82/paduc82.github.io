After the number one question of "how do i play AVI files?", how to play MPG or MPEG files is probably the second most sought information by newbies to digital video. In this short article, we try to address this question as well as we possibly can.


Things you should know about MPEG


MPEG stands for Moving Picture Experts Group. This group specializes in the development of video and audio encoding standards. When you have come into possession of a file with an .MPG or .MPEG extension, you usually have either MPEG-1 or MPEG-2 video. MPEG-1 video is used on VideoCD (VCD) discs and MPEG-2 video is used on Super VideoCD (SVCD) and DVD discs.


Play MPEG-1


I you are using the Windows operating system, Windows Media Player supports MPEG-1. For the record, if you want to play the video on a VCD disc in Windows Media Player, you will have to browse onto the disc in explorer and locate the AVSEQ file(s) in the MPEGAV folder. These files will have a .DAT extension. These will play back in Windows Media Player, but you will need special software to use all the features VCD offers. Software DVD Players such as WinDVD and PowerDVD are recommended for VCD playback.

If you are using a Mac or Linux operating system, then it is recommended that you download the VLC player.

Mac: http://www.afterdawn.com/software/alternative_platforms/mac_software/vlc_for_mac.cfm

Linux: http://www.afterdawn.com/software/video_software/video_players/vlc_linux.cfm


Why MPEG-2 won't play


The reason you cannot play MPEG-2 video in WMP on a fresh Windows XP installation is simple; the player cannot find a suitable decoder for the file. It is possible that WMP may try to download an appropriate codec to play back the file however. Assuming that doesn't happen, you will have to install some software on your computer to play MPEG-2 video.


Easiest Solution


Download and install the VLC player. The VLC package already includes software to play MPEG-2 video content.

  1. Download Windows 10 at https://www.microsoft.com/en-us/software-download/windows10 make sure to select x64 or x86 based on the PC that you are downloading it on, not the PC you are trying to reset/reinstall/fix.

  2. Run the Media Creation Tool and select "Create installation media for another PC" and fill the fields appropriately for Language, Edition, and Architecture.

  3. Choose USB or Disc, I personally used USB so the instructions will follow that route but I assume Disc would just require a little more work in order to burn it and use it (If your device that you are trying to fix has a disc drive, which mine did not).

  4. Select a USB (The drive needs to be empty, move all files that you wish to keep onto your HDD and then do this step) and let the program do it's work.

  5. Once finished you are ready to boot from the USB. Plug it into the device that you wish to fix and upon boot press and hold "F2" until you get to the BIOS settings. You need to disable Secure Boot which is under Security -> Secure Boot Menu -> Disable.

  6. Save and Exit the BIOS and hold F2 again until you are back into the BIOS. Go to the BOOT tab and on the boot list should be the USB drive along with one or two other options. (If it doesn't show up as mine didn't save and exit the BIOS two to three more times and see if it shows up then as the BIOS is loading before the USB drive and this fixes that).

  7. Once you have selected the USB drive and successfully boot from it you are presented a few options. For me I chose to reset and lose all my files (There was an option to keep the files I believe) as I had nothing of importance. The computer underwent its uninstall and reinstall of Windows 10 and I ended up back into the desktop, albeit a long trip to get there.
