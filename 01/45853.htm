daniel martín say: Hi, landonblaine:

When the option Store and display recently opened items in the Start menu and the Taskbar is unchecked and grayed out, it is a symptom that two Windows policies are applied to your system.

Open Registry Editor (Regedit.exe) and browse to this key: HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer. If there is a registry value NoRecentDocsMenu set to 1, delete it. If there is a registry value NoRecentDocsHistory set to 1, delete it as well.

Repeat the same steps in the registry key HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer (it may not exist).

Restart your computer and see if the problem is solved.

a. user say:

It is not recommended to turn off Volume Shadow Copy. It manages and implements Volume Shadow Copies used for backup and other purposes. If this service is stopped, shadow copies will be unavailable for backup and the backup may fail. If this service is disabled, any services that explicitly depend on it will fail to start.

However, if you still want to disable Volume shadow copy than follow the steps as below:

1. Press the Windows + R, type services.msc and press enter.
2. Locate the service "Volume Shadow Copy" and right click and click "Stop".
3. Close Services windows.
