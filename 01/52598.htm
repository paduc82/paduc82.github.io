slance310 say:

Hi,

The problem most likely is being caused by the fact that your computer is "New"...

Almost all new computers use UEFI Bios... It is almost guaranteed that the machine from which you copied Windows 7 was NOT a UEFI machine, but "Legacy"...

As long as your Bios is configured for UEFI Bios it will not boot from the Win 7 drive..

A hard drive set up to run on Legacy Bios is configured much differently than one that is configured to run on UEFI Bios... They are actually "Initialized" differently just to start with...

If your Win 7 was preinstalled on another machine, then it is not licensed to run on the new machine... That probably would not stop it from booting though...

If you are not careful, you could mess up your new machine trying to modify Bios settings to boot from the Win 7 drive, but you may be able to reset your Bios to Defaults if a problem arises...
