Select multiple files in same folder from a list of file names - Microsoft Community

airborne4905 say:

Select multiple files in same folder from a list of file names

I am a wedding photographer.  Each wedding folder has 700-1000 pictures.  When our brides order their pictures we have to select each individual picture to upload them to the printer.  I want to know if there is a way to just put a list in the folder search bar that will bring up only the pictures in the list out of that folder?  I can only get it to do 1 picture at a time.
alexander tarvid say:

Hi, Airborne4905!

As I understood, you have a list of files that you want to be printed automatically. Well, for this you should create a script. Below I show you a simple script.

Now, assume that you have a file d:\print.txt that contains names of the files in folder c:\users\public\pictures\sample pictures\ with one filename on each line, e.g.:
Greek.jpg
Dock.jpg
Forest.jpg

Then, you have a file called print.vbs that contains the following script code:

Set objShell = CreateObject("Wscript.Shell")
Set shell=CreateObject("Shell.Application")
Set objFS = CreateObject("Scripting.FileSystemObject")
Set objFile = objFS.OpenTextFile("d:\\print.txt", 1)
strFileContents = objFile.ReadAll

arrLines = Split(strFileContents,vbNewLine)
For Each strLine in arrLines
    strFile = "C:\\Users\\Public\\Pictures\\Sample Pictures\\" & strLine
    shell.ShellExecute strFile,"","","Print"
    objShell.Run "mspaint /p " & chr(34) & strFile & chr(34),0,True
Next

objFile.Close

If you want to print all the files you have listed in your d:\print.txt file, you just execute the print.vbs file.

Note:
1. The line strFile = "C:\\Users\\Public\\Pictures\\Sample Pictures\\" & strLine contains path to your pictures folder--if you need to browse another folder, simply edit this string (note that you should keep double-backslash instead of single-backslash that you use normally!).
2. The line Set objFile = objFS.OpenTextFile("d:\\print.txt", 1) contains path to the file where you list images to be printed. Edit the path to reflect the actual path to your file.
3. If you need a more elaborate version or some further questions about scripting, you are advised to ask questions in the Microsoft Scripting Guys Forum on http://social.technet.microsoft.com/Forums/en/ITCG/threads/.

Hope this helps!

blackhawkdrunk say:

Are you doing the quest called Waking Nightmare? If so, I'm guessing you're freezing coming out of the Dreamstride sequence. If so, it's a known bug fixed in the patch that is currently waiting to be certified by Microsoft. However, if this is place you're freezing, just don't save until you're back out in Skyrim & you'll be fine -- there's literally less than 5 mins of gameplay after that point, so it shouldn't be any trouble getting outside & safely saving w/o freezing. Also, if you have auto save turn on (which you shouldn't, btw), turn them off until you're back outside.

blackhawkdrunk say:

Bethesda games always work better w/ autosave turned off. Something to do with the frame rate drop when an autosave is triggered -- this can lead to corrupted save files. Also, autosaves tend to happen after a load screen & I'll tell you this; there's nothing more scary in gaming than a Bethesda load screen. Some people have been lucky & not affected by this, many have been unlucky -- myself included. Nevertheless, I'm glad I could help.
