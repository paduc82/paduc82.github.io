In the lower left corner of the *icon*? It means it's a shortcut?
 
A shortcut is not the file itself, but just a pointer to it. So you
can have many shortcuts (pointers) to a particular file.
 
So, for example, lets say you have a file called 2010budget in a
folder someplace. You also have folders for 2010 data, for budget
information, and for financial information. You can have a shortcut to
2010budget in each of those folders. You don't have the overhead of
multiple copies of the folder, just the tiny shortcuts to it, so you
can access it from any of those folders.

A small arrow on an icon represents a shortcut.

 

A shortcut is an icon that represents a link to an item, rather than the item itself. When you double-click a shortcut, the item opens. If you delete a shortcut, only the shortcut is removed, not the original item. You can identify shortcuts by the arrow on their icon.

 

If you delete a shortcut, the shortcut is removed from your desktop, but the file, program, or location that the shortcut links to is not deleted.
