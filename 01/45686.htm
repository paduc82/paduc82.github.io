lemp say:

Your XP sp2 CD is an "OEM" version rather than a retail version, but if you have the appropriate key, it will work fine.  The catch with OEM installations is that they are tied to the computer on which they were first installed ... so once installed, it can't be installed on another computer (and replacing the motherboard counts as "another computer").

Service Pack 3 is available for download and will continue to be available for the foreseeable future.  If you did a fresh install using the SP2 CD, you would download sp3 and install it manually (you have to have sp3 before Windows Update will work to give you the post-sp3 updates). 

That said, it is easy to create a sp3 installation cd by a process known as slipstreaming.  You can do this yourself, but any competent computer tech should know how to do it.  See http://www.bleepingcomputer.com/tutorials/slipstreaming-windows-xp-to-create-bootable-cd/

You can even create an installation CD that includes sp3 plus all of the critical updates that came after sp3.  See http://xdot.tk/

There are a few caveats to keep in mind before installing sp3:

Before installing sp3, read this article --> http://support.microsoft.com/kb/950717

If your PC is an HP Pavilion, Presario or Media Center and has an AMD processor , this patch should be installed before installing sp3 -->

http://h10025.www1.hp.com/ewfrf/wc/softwareDownloadIndex?cc=uk&lc=en&dlc=en&softwareitem=pv-60484-2

Install SP3 by downloading --> http://www.microsoft.com/download/en/details.aspx?displaylang=en&id=24

ignore the text that says that it is for network installations and also ignore the text advising Windows/Microsoft Update if only updating one PC.

As far as using sfc /scannow, that command will require an installation CD that is the same service pack that is installed on the computer, BUT using sfc /scannow will almost certainly NOT help you in any difficulty that you are likely to encounter.  Despite the many websites you may see that advise using this command, it simply does not do what people think it does, at least for XP.
