1. Hibernate was taking up like 15GB, go to an elevated command prompt and type

Unfortunately you need this on to do the really fast windows 8 boot times
2. Pagefile was taking up like 10GB
Depends on how much RAM you have, as I have 32GB, I only need roughly 1GB in case of a system crash, can change this under >System >System Properties
3. System Restore was taking up 15GB
>System >System Properties, go to protection settings, I moved mine to my RAID drive
