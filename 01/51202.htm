casey claassen say:

Is there a way to rename an existing task in the Windows Vista task scheduler library, within the computer management console?

The "Name" field seems to be grayed out on the existing tasks, even though other attributes of those tasks can be changed.
naman r say:

Hi Casey,


Thanks for posting the issue on Microsoft Community Forum.

 

As you want to know how to rename a Task in Task Schedular, I would suggest that you try the steps mentioned below.

 

a.       In Task Scheduler, highlight the task in question.

b.       Click on Export and choose a location to save the file (Desktop is as good a place as any).

c.        Click Save.

d.       In Windows Explorer, browse to the location where you saved the file.

e.        You can then rename it as normal. Back in Task Scheduler, click on Import Task.

f.        Browse to the now renamed file.

g.        Click OK.

h.       Now highlight the original task (remember that there will now be 2 tasks, one with the original

name and one with the new name) and click Delete.

 

Hope the steps provided helps.
