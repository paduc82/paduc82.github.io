    How to manually open Port 6699 in Windows Firewall?

    Windows Firewall may block port 6699 by default. If you want to allow a program to communicate using Port 6699 through the firewall, you can usually do that by selecting the required program on the Exceptions tab in Windows Firewall. However, sometimes the program is not listed in the Exceptions tab. In this case you need to open the port manually.

    1.Open Windows Firewall by clicking Start, Control Panel, Security, and then Windows Firewall.

    2.Click 'Allow a program through Windows Firewall'.

    3.Click 'Add port'.

    4.In the 'Name' field, type a friendly name for the port like 'Port-6699'.

    5.In the 'Port number' field, enter the port number '6699'.

    6.Select TCP or UDP, depending on the protocol required for port 6699
