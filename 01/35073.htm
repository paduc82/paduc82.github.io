1)   Have you made any changes prior to this issue?

 

If the reparse points to a location that contains any compressed file format (multimedia files, .zip files, and so on), you receive this error message.

 

To work around this problem, remove this reparse point from the library, and then run the backup wizard again.

a. Go to Start / All Programs / Accessories / System Tools / Disk Cleanup.

b. In the drop down menu, select the drive you want to clean up and click OK. The drive will be scanned.

c. In the next window, select the Clean up system files button.

d. Select the drive again and click OK.

e. In the next window, select the More Options Tab. In the System Restore and Shadow Copies section, click the Clean up button and click OK.

f. In the next window, select the Delete Button.

 

This will remove all of the temporary files and all but the latest System Restore Points.

 

rse point, select the absolute path of this location from the Windows Backup configuration user interface.
