mitch h. say:

Hi Sayyid,


Since you're using a SIM to be able to use the internet, Windows 10 automatically set your connection as metered if your Internet service provider limits the data you can use.

You can turn the Set as metered connection option to OFF by following these steps:

  1. Click the Internet access icon from the Taskbar.
  2. Click the network where you are connected, then click Properties.
  3. Locate and toggle the Set as metered connection to OFF.

Let us know if the message still appears after following the above mentioned steps.


Regards.

a. user say:

Hi Sayyid,

There are other options that you can do to turn off the "This PC is on a metered connection" notification. You can try to follow the steps below:

  1. Swipe from right to left and tap on Settings.
  2. On the Settings menu, tap the Change PC settings link.
  3. On the PC settings menu, tap on Network.
  4. Tap to select the network your device is using to connect to the internet.
  5. On the Date Usage column, Turn off the Set as a metered connection.

robert aldwinckle on forums say:

I have spent a lot of time trying to find instructions on how to use the search function in File Explorer, and I would appreciate someone's helping me locate such instructions, assuming they even exist.

You mean the "discoverable UI" was not intuitive for you?  <eg>

Shawn has already pointed you to the discoverable part but I suspect you had to be a keyboard user before to realize how readily discoverable they are.  For example, I always start searches from Win-E with Ctrl-e.  (Doesn't everybody?)  Notice what pops up then?  Hint: pay particular attention to what is hidden under the Other properties menu.   Anyway, I am also used to typing name: (at least).  However, since my machine is apparently so low power (or perhaps so filled with junk) I have had to learn how to optimize my queries even more than that, so have had to master the rest of the patterns available in the search UI query syntax.

http://windows.microsoft.com/en-us/windows7/Advanced-tips-for-searching-in-Windows

I imagine that you would be particularly interested in how the tilde character (~) is used in modifying search terms, so do a find (e.g. Ctrl-F in IE) for that character (~) in that document to get an overview of how it is used.  FWIW I seem unable to remember the tip that  name:~=  means "contains".  Fortunately  name:~~  seems to do the same thing and that's what I use frequently.  It is an important difference.  For example I recently tried to find all my Notes files (e.g. files which had that string in their name) and stupidly only searched for  name:notes   at first.  Fortunately I realized that one important file I wanted included in those results was missing, so I changed the search to name:~~notes  and found a whole lot more that I otherwise would have missed.

Another tip I can give is that I have found it very useful to use the Saved Search feature, not just to save a search but also to be able to inspect it to find out how my query has been interpreted.  Sometimes doing that has revealed that I hadn't really correctly expressed what I was trying to do before the UI syntax was converted to the query syntax which is passed to the search engine.

 

FYI

Hi whistler318, I see this is your first post here.  Welcome to our community.

For the most part, the little search box in the File Explorer works a lot like the search box when searching the internet.  You can type a phrase, or several words.  Also like a web search, you can get an exact match by "using quotes in the search" around the whole phrase.

It also has some functions you might remember from old versions of Windows you've used.  You can type things like *.jpg to find all jpg picture files, or type *bonus* to find all files that have the word bonus in them, even if it's not the first word in the file name.

Limiting the search to just file names can still be done the way it was in Windows 7, by typing filename: followed by the name (or part of a name) you want to find.

I would suggest you to check whether the printer spooler service is started.

C:\Windows\System32\Spool\Drivers\w32x86

andre da costa say:

Q: Why did Microsoft remove Windows Media Center and DVD Play back?

A: Microsoft chose to remove these features formally built into Windows because of the changing landscape of how users consume video. Users are turning to online services such as YouTube, Netflix, HULU, Vimeo and other sources for movies, videos and other types of content. Also, the types of devices Windows 8 will target on a large scale will likely not include an optical drive. This means, Microsoft would be licensing a codec to decode video for something that might not be included with the cost of the product on a shipping device.

Q: Suppose I have an existing computer that does have an optical drive or I currently use Media Center and Media Player for DVD playback and want to upgrade to Windows 8?

A: You have a few options:

·         If you upgrade to Windows 8 Pro, you will have the option of purchasing Windows Media Center as an add on feature for a small cost.

·         If you purchase the introductory edition of Windows 8, you will need to upgrade to Windows 8 Pro then purchase Windows Media Center.

·         If you purchase a new PC or Notebook with an optical drive, it is likely the OEM will include a third party decoder or bundle Windows Media Center as part of the overall cost of the system.

·         If you need DVD playback, you can purchase or use a third party program such as Intervideo, Cyberlink or VLC Player.

Q: Which editions of Windows 8 will Windows Media Center be available to?

A: Windows Media Center will only be available to Windows 8 Pro.

Q: Are there any major changes coming to Windows Media Center in Windows 8?

A: No, Windows Media Center on Windows 8 is identical to the commercially available version in Windows 7 today.

 

You can add Media Center using Feature Packs:

http://windows.microsoft.com/en-US/windows-8/feature-packs
