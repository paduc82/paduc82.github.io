Firstly, to go back from Windows 10 to previous version of Windows, you'll need to:

  1. Keep everything in the windows.old and $windows.~bt folders after the upgrade.

  2. Remove any user accounts you added after the upgrade.

  3. Know the password you used to sign in to Windows 7 or Windows 8.1 (if you used one).

  4. Have the USB drive you used to upgrade to Windows 10 (if you used one).

So, pertaining to the situation you are in, there can be two scenarios under which you will face such issues.

Scenario 1

You have modified Windows.old, $Windows.~BT and $Windows.~WS folders required by the system to perform the rollback operation. These modification can adversely affect the rollback feature in Windows 10.

Scenario 2

You have created a new user profile after setting up Windows 10.

In this case, you will need to delete the account or accounts (if more than one was created) that are specified in this screen before the customer can go back.

IMPORTANT: Before deleting the user account please make sure to back up any data that is associated with that user profile (Documents, Music, Videos etc.) as deleting the account will also delete the data associated with that user profile

4. Click on Settings  under the User Profiles section
