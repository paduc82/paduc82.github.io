The rest of the command I originally gave you should have searched through the output from the bcdedit command to find just the line with "osdevice" in it (4th from the bottom in the sample above).

Unfortunately, the results from the second command you used seem to confirm that there's a hardware problem with the hard drive.

This part

    On computer: MININT-9QCNML4

is no problem.  That's just the name of your computer.  It isn't significant in this context.

The rest of what you reported, however, does not look good.

This ...

Volume     0        D   Repair disc     UDF   DVD-ROM    172MB       Healthy

is the repair disc you used. 

The other line, however, seems to describe something like an SD card reader.  What it does not show is your hard drive.  This is consistent with the message that you received -- "Reboot and select proper Boot device or Insert Boot Media in selected Boot device and press a key" -- that is, the system is not finding any device from which it can start Windows.

When you were in the BIOS to set the boot order, was there anything there that suggested that the BIOS recognized your hard drive?  Is there a hard drive test available in the Toshiba BIOS?

It's really difficult to find a hard drive diagnostic utility for Toshiba products.  Because your system won't boot, you need a utility that runs from its own bootable CD rather than a Windows-based utility.  The utility that I think has the best chance of working on your Toshiba laptop is Seagate SEATools for DOS.  http://www.seagate.com/support/downloads/seatools/  There is a guide for using the utility on the same page.

The download from Seagate is an ISO file.  If you have a working Windows 7 system, you should be able to just double-click the file and burn it to a CD using Windows 7 built-in utility.  Otherwise, download (free) ImgBurn to create a bootable CD from the ISO file.  http://imgburn.com/

After you've created that SeaTools CD, just put it in the CD drive and boot from it as you did with the system recovery disc.

If the drive has failed, which seems likely, you can purchase a new drive from any number of online suppliers.  Laptop drives are pretty standard, and you are not required to buy one from Toshiba (which, if they sell them at all, will be fairly expensive).  You'll have to take the laptop apart to some degree in order to replace the drive.  Some laptops are relatively easy; others are quite difficult and I've not researched yours to learn where it falls on that spectrum.

If SEATools doesn't work on your laptop, or if all of the above seems too complex, take the laptop to an independent computer repair shop.  Avoid chain stores, consumer electronics stores, office supply stores, and the like.

Good luck!
