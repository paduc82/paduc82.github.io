marilyn-- say:  

From: PA Bear [MS MVP]

 

Did you install Vista SP1 manually or via Windows Update?

 

Try the following, logged-in as Administrator:

1. Stop the Automatic Updates service:

Start > Run > (type in) services.msc > [OK]

Double-click Automatic Updates > Click on Stop (Stopping the service will take a moment)

 

2. Delete the contents of the Download folder:

Start | Run | (type in) %windir%\SoftwareDistribution | [OK]

Open the Download folder and delete its contents Close the window.

 

[NB: You may need to delete C:\Windows\SoftwareDistribution <=this entire folder manually.]

 

3. Start the Automatic Updates service:

 

Start > Run > (type in) services.msc > [OK]

Double-click Automatic Updates > Click on Start (Starting the service will take a moment)

 

4. Read & heed: http://support.microsoft.com/kb/822158
