Can your programs work in Compatibility Mode in Windows 10?

There's no such thing as an emulator per se but you can run Windows 7 or XP in a virtual machine on Windows 10. Use Windows' own Hyper-V or a third-party application (VMware or VirtualBox.)

You're going to run into a licensing issue with a virtual machine. Because you upgraded from Windows 7, your license to use Windows 10 is a continuation of your license to use Windows 7. Here are few steps to resolve the issue:

1. Load the following link in your browser: http://support.microsoft.com/kb/290301

2. On the above link run the FIXIT tool for Office 2010. This may take a few minutes or complete in a few seconds, and will remove any corrupted installations and remnants of Office 2010.

3. After this is completed run the FIXIT tool for Office 2007 if you had a previous version of Office: Office 2007 installed on the computer before you installed to Office 2010. This may also take a few minutes or complete in a few seconds.

4. After the above process is completed, close any open window and return to the desktop.

5. Now we will check and delete the 2 Office Registry Keys in the Registry Editor.

1. Click on Start and then on Run. If the Run option is not available in the Start Menu navigate to this location: Start > All Programs > Accessories > Run.

6. Type the following command in the Run box and click on OK: regedit

7. Expand the 2 Sub Registry Key HKEY_CURRENT_USER and then expand the SOFTWARE Key below. Similarly expand the MICROSOFT Key. Find the OFFICE Key under Microsoft and right-click on the key and delete the Key. Come back all the way to the top of the Registry Key and collapse the HKEY_CURRENT_USER Key.

8. Expand the 3 Sub-Registry Key: HKEY_LOCAL_MACHINE. Navigate to SOFTWARE > MICROSOFT. Find the OFFICE Registry Key and then delete the key.

9. In case you are not able to delete one of the above Registry Keys ignore the step.

10. Close the Registry Editor window.

We shall now clear the Temp Folder. Follow these steps:

a. Click on Start > Run.

b. Type the following command in the Run box and click on OK: %temp%

c. In the Temp folder window click on the Organize drop-down button and click on Select All.

d. Click on the Organize drop-down button again and click on Delete.

e. In case you are not able to delete any item click on Skip and let the deletion process move to the next item.

f. Once completed close the Temp folder window.

11. Once this is done - restart your computer.

12. Download Office Professional Plus 2010 from the following direct download link:

http://msft.digitalrivercontent.net/01/106018573-10238044--NOA//office2010/X16-32250.exe

Download and install the above to your desktop because it's the easiest place to find the downloaded file on your computer.

Now try to install the updates (Start - All Programs - Windows update).

josephmcmahonvz say:

how do i connect my computer to my tv

How do i watch computer content on my tv
kari finn say:

It depends on both your computer and TV, or rather what kind of connection methods are available on both.

 

If the computer and TV both have HDMI, simply connect an HDMI cable from PC HDMI output to TV HDMI input, select signal source on TV accordingly, click an empty spot on Windows desktop to open Screen Resolution tool and set your dual display properties to Duplicate or Extended mode as you wish.

 

HDMI usually also carries the audio signal you can use TV's own speakers or even a connected Home Cinema system for audio output.

 

You can also do this with a VGA cable if both TV and computer have a VGA slot. VGA does not carry audio.

 

Read more about multiple displays: http://windows.microsoft.com/en-us/windows-8/multiple-monitors#1TC=t1
