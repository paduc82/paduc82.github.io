sumit (volunteer moderator) say:

If you are running a pro version,Open group policy editor and disable defer upgrades.

  • In the Run dialog box type in gpedit.msc
  • Open Computer configuration>Administrative templates>Windows Compnents>Windows updates
  • On the right hand side, double click defer upgrades and updates and the windows that open, set it to disabled.
  • Restart and check if the update is back.
