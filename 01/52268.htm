¡firedog say:

I see - thank you. Examples aren't much use when troubleshooting messaging problems, because mail servers are idiosyncratic. What works for one will not necessarily work for another.

Socket error 11001 indicates that Windows couldn't find the server. This means either

  1. The server was temporarily offline, or
  2. You were offline, or
  3. There's a problem with DNS, the service that translates mail.docmelamed.com into the numbers - an IP address - that computers understand, in this case 66.242.128.130

I had no trouble connecting to the server, which means that if (1) were the case, it's back online again. Incidentally, it reported that your mailbox was empty.

If (2) were the case then you should have seen other error messages, like "You are currently working offline. Do you want to go online now?"  Did you?

 

(3) might need a bit more work, so we'll wait with that until we've heard back from you.
