¡firedog say:
  1. 1.  I used the nickname to hold short mnemonic abbreviations of the contact's name, to be able to quickly enter a contact in the To: or Cc: fields.

    2.  "Name" field, which apparently doesn't exist).

    3.  if I don't put in a nickname, the names come out formatted as "Lastname Firstname". Shouldn't Live Mail put a comma in between?


Taking (2) first, Name certainly does exist. If you select View as > List on the ribbon, you can configure a column to show this field. Name is the value you'll see in a contact picker list (the one you see when you click the To... button or start typing in an address box), and the one used in the To: line of a sent message.

Name is constructed by WLMail from contact data fields. It will consist of the first match found when scanning data fields in the following order:

  1. Nickname
  2. First name + Last name (in the order specified at Sort by in the Contacts window)
  3. First name or Last name
  4. Company
  5. E-mail address

So Nickname should be used with circumspection.

To (1), you could use the same technique by putting your mnemonic in the Notes field. Otherwise, you can start typing a contact's Nickname, First name, Last name or Company, or part of his e-mail address (one of three) or any word you've put in his Notes field. So, if you type bil, for example, the list that appears will include:

  • everyone whose Nickname starts with Bil, e.g. Billyboy
  • everyone whose First name starts with Bil, e.g. Bilbo Baggins
  • everyone whose Last name starts with Bil, e.g. Jim Bilko
  • everyone whose Company name starts with Bil, e.g. Billingsgate Market
  • everyone whose e-mail address starts with bil or has bil straight after a period or the '@' symbol, e.g. 'bilbo at hotmail.com' or 'jim at billingsgate.com' or 'jim.bilko at hotmail.com'
  • everyone who has a (non-trivial) word in the Notes field starting with bil, e.g. Malaria and bilharzia expert
  • every category whose name begins with bil

I can't answer your no. (3), because I can't imagine why you'd want to address a message to Doe, John rather than plain John Doe. You can change the order using Sort by in the contacts window.
