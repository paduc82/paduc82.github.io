Hi Blue eye sand male, the size of your hard drive will not determine which version to install . . .

Open Control Panel - System

Under System Type, what type of processor do you have

If you see x64 then your PC can run the 64bit version of Windows 10, if you see x86, then your system cannot

Under Installed RAM, do you have 4GB or more, if so your system can run the 64Bit version, if you only have 2GB, then your system will run the 32bit version better

What Processor is reported on that page?

If your Processos is a Core 2 Duo, then the 32bit version of Windows 10 would be best, if you have a Core i3, i5 or i7, then the 64bit will run just fine . . .
