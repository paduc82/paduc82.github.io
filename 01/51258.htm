kevinglynn429 say:

ThrottleStop shows CMod at 75.0%.  That stands for Clock Modulation which is a method used for throttling a CPU.  Many Dell laptops have issues in the bios that cause this.  You need to set that to 100.0% by using ThrottleStop, check that option off and then click on the Turn On button.  Laptops can be miserable to use when Clock Modulation throttling is in progress.  You can ask Dell why this happens but good luck getting a straight answer from them.

I would also check off the Set Multiplier option and set that to the maximum value (10).  This is called Intel Dynamic Acceleration (IDA).  The maximum speed when both cores are active is 9 x 199.44 MHz = 1795 MHz and when only a single core is active, IDA mode lets the CPU increase to 1994 MHz.  In the newer Core i processors, IDA got renamed as Intel Turbo Boost.  Same thing though.

Your CPU is running at approximately 1600 MHz in your screenshot.  ThrottleStop separates the CPU speed from the clock modulation throttling so you can see exactly what is going on.  Looks like the Task Manager is combining these two items together and is calculating an equivalent MHz value after taking internal throttling into account.  

75% of 1600 MHz = 1200 MHz

The normal idle MHz for a T7100 is

8 x 100 MHz = 800 MHz

75% of 800 MHz = 600 MHz

If this is what the Task Manager is doing, it sort of makes sense.

     CHKNTFS  /X  Y:

     CHKNTFS  Y:

EDIT #2: Well guys, after over a month of troubleshooting, I found out this whole thing was a motioninjoy conflict, possibly triggered by a windows update.  I was using motioninjoy drivers for using my PS3 controller with PC for gaming.... turns out motioninjoy was the culprit.  I uninstalled/deleted motioninjoy through Device Manager, then rebooted PC.  My bluetooth radio was now visible again, but I still had problems connecting my keyboard.  I then opened services.msc and restarted the Bluetooth service.  After doing these things, my Apple Wireless Bluetooth keyboard works great again in Windows 7!  TL:DR...
