If it is changing when you reboot, then either the BIOS time is incorrect or the CMOS battery is dead. You can access the BIOS when the system is first turned on. Defragment & SSD - Microsoft Community

reza_mesgaran say:

Defragment & SSD

Hi.

I have installed a Samsung SSD 850 Evo on a Windows 10 build 10.0.14393.

In the Defragment software, I saw "trimmed" instead of "defragmented".

What is trim? Is it necessary to run defragment on SSD? 

Running defragment makes any problem to SSD or not?

Thank you.

rdrewd2 say:

No, it is not necessary to ever defragment your new SSD.  Doing so would only add many needless read/write cycles to the drive that would reduce it's lifespan.  TRIM (oversimplified), cuts down on the work required for the drive to write to  previously-written-to sections by clearing out any old data beforehand.  By doing this, the drive can start writing the new data immediately without having to pause and erase that section first, and keeps the SSD from having to maintain or re-write any useless junk data.  Generally speaking, TRIM helps your new SSD to start writing faster and do less work while being more efficient.

I suspect that your BIOS password prevents you from changingsettings in the CMOS chip. It does not prevent you from bootingup. Note also that BIOS stands for Basic Input/Output System. It is pronounced "By-oss" and cannot be "bois".

If you forgot the BIOS master password then I can see two options for you:

  • Check the FAQs on the Acer home page. There might be guide for resetting the BIOS to factory condition.
  • Burn a DOS bootable CD from www.bootdisk.com. Make sure that it contains a copy of debug.exe. Now see if the debug program you mentioned before will reset the BIOS password. I have not seen this method before and I do not know if it works.
I am not aware of any tool that lets you modify the BIOS password from within Windows.
