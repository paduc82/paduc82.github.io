jrosenfeld say:
Click start, run, type serrvices.msc, click OK. Scroll down to help and support. Click on it to highlight it. Check what it says in the column headed Startup type (you may need to expand the window to the right).
1. If the startup type says automatic, click the blue link 'restart' the service. If it fails to start post back (automatic means it should start each time you startup your PC, so if it is stopped despite that there may be an underlying cause).
2. If the start up type says manual or disabled, double click on the help and support service to open its properties, and in the startup type dropdown box select automatic. Click Apply, OK, to close the properties window. Then click the blue link click 'start the service'.
If it starts successfully, close the services window and see if help and support now opens.
