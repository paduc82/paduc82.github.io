You can't move Windows 10 system files to the SD card, you can offload some Microsoft Store apps and maybe redirect your personal folder (Documents, Music, Pictures) to the SD card and use services such as OneDrive to keep some files in the cloud.

Insert the SD card, browse it and do the following:

Create a folder named FILES, open it and Create a folder in it and named it: achapman72 (you can use your account name if you wish, this is just for example sake).

Create another folder and name it APPS

Press Windows key + R
Type: %userprofile%
Hit Enter

Right click your Documents folder
Click Properties
Select the Location tab
Click Move
Browse to the Files folder we created earlier then select achapman72 folder.

Repeat the same steps for other folders such as Pictures, Videos etc.

Open Start > Settings > System > Storage
Under More storage settings, click 'Change where new content is saved'

In New apps will save to list box, choose the 'Apps' folder we created earlier.

Open Start > Settings > Apps > Apps & Features
Select any large app you have downloaded from the Windows Store
Click Move, choose the 'Apps' partition

You need all three. You can use Windows Defender, the antivirus and antimalware software that's included with Windows, or you can use another antivirus and antimalware app.

DMZ - Microsoft Community

xbr dionysus say:

DMZ

Login Information

 

  • Gateway: 192.168.1.1
  • Username: admin
  • Password: admin

DMZ

 

  1. Connect a computer to the gateway with an Ethernet cable.
  2. Open a web browser on a computer.
  3. Type the address 192.168.1.1 into the web address bar.
  4. When prompted to login type admin for both the username and password.
  5. Under the Advanced Menu click on the Application option.
  6. Click on the DMZ Settings option.
  7. Check the Enable DMZ box.
  8. Select the Xbox’s IP address from the “Select a LAN IP address” drop down box.
  9. Click on the Submit button.

 

 

Disable DMZ

 

  1. Connect a computer to the gateway with an Ethernet cable.
  2. Open a web browser on a computer.
  3. Type the address 192.168.1.1 into the web address bar.
  4. When prompted to login type admin for both the username and password.
  5. Under the Advanced Menu click on the Application option.
  6. Click on the DMZ Settings option.
  7. Uncheck the Enable DMZ box.
  8. Click on the Submit button.
