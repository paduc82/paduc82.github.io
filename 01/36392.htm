I resolved my issue. The step I seemed to miss was using Device Manager, with Show Hidden, as noted below.

What eventually worked was:

1) Removing all of the iTunes software: iTunes, Bonjour, Software Update, Application Support, Mobile Device support.

2) Using Windows Device Manager, removing all instances of iPhone (NOTE: This required that I set the option to SHOW HIDDEN).

3) Rebooted.

4) Installed only Application support and Mobile device support (there are posts elsewhere on the interwebs about how these are the only two files that are part of itunes that are needed for tethering - you simply unzip the iTunesSetup.exe to get them - I imagine that a full iTunes install would have worked as well).

5) Rebooted.

Tried tethering, and this time I saw that the Ethernet driver was installing. Success!

a. Remove all floppy disks, CDs, and DVDs from your computer, and then restart your computer using the computer's power button.

b. Do one of the following:
