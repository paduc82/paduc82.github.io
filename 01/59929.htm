Service Pack 3 was released in June 2008.  Since then Microsoft
have released many critical and security updates - which address
threats and other problems that have been discovered since 2008.

It is most important to turn on Automatic Updates.  For the avoidance of doubt, you MUST download and install all critical updates without delay.  I hope this advice is not too late, because some Malware interferes with the automatic update function.

Open Control Panel and double click on Security Center. The
resulting menu offers the following options:

Option 1.    Turn off automatic updates. - Definitely NOT
recommended

Option 2.    Notify me, but don't download or install them. - Not
ideal, but might be appropriate, if you have a very slow
connection to the Internet.  This option allows you to choose
which updates to download and when.  Consequently, you can
download and install updates, when you are not using your
computer for any other purpose.

Option 3.    Download updates, but let me choose when to install
them. - That is my personal preference, because I want to be in
control of any changes to my computer.  Incidentally, I have just
received a pop-up notification that Updates are ready for my
computer.  I am about to click on the yellow shield icon to
install this month's critical updates.

Option 4.    Automatically download recommended updates for my
computer and install them.  -  That is Microsoft's
recommendation.

You should select from Option 3 or Option 4.

Once you ensured that automatic updates are turned on, use your browser to
visit the URL at
http://update.microsoft.com/microsoftupdate/v6/default.aspx?ln=en-us.
This will enable you to perform an immediate one-off search for
updates (without waiting for the next automatic update).

The express option searches for critical updates.  The custom
option will also include additional updates that you might, or
might not want to install.  From the optional updates list, I
always install root certificate updates - but I avoid hardware
drivers (because it is better to download updated drivers from
the hardware manufacturer's website).
