Hi Azizz,

 

Windows 8 automatically performs a number of maintenance tasks for you, including smart system diagnostics, checking to see if any updates are available for your software and performing security scans. Automatic maintenance does too many functions, so turning it totally off is not advisable.

 

The Automatic Maintenance normally runs at a time when the computer is idle. When it finds that the computer is in use at a time scheduled for the automatic maintenance, then it does not start automatically and instead it waits for the next time when the computer is idle.

 

It is apparently not possible to disable it completely. You could at least disable it from waking your computer.

Steps:

a. Click on the Action center flag on the right hand bottom corner to enter Action center.

b. Expand Maintenance and click on Change maintenance settings

c. Uncheck the option which says allow computer to wake up my computer at the scheduled time

d. Set the Automatic maintenance Schedule time to a time when the computer is powered off / not in use

Note: The task will remain enabled, but with triggers disabled, it won't actually start.

 

If the computer is running slow, I would suggest you to follow the steps given in the Microsoft KB article:

Ways to improve your PC's performance

http://windows.microsoft.com/en-us/windows-8/improve-pc-performance

 

Hope this helps. If you have any further queries/issues related to Windows, write to us and we will be glad to assist you further.
