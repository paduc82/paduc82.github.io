Hi Graham,

It is possible that part of the problem is being caused by the same system recovery partitions being used by both your Windows 10 and Windows 7. In this case, the best way to dual boot is to have two separate partitions for your Operating Systems - including the extra system/recovery partitions, and before booting either of the two operating systems is to "hide" the other one and its partitions.

If the Operating Systems are totally on a separate HDD, you could disable one HDD possibly in the BIOS before booting the second HDD with the other Operating System. It depends though on your BIOS. As of the moment, these are the only workaround for this issue.

Understanding downgrade rights

Consumer versions of Windows 8 Pro that come preinstalled on PCs have downgrade rights to a previous version of Windows (example: Windows 7, Windows Vista).

If your copy came pre-installed on your system, you may downgrade your copy to Windows 7.

If not you need purchase a copy of Windows 7 and install the same on your computer.

• Downgrade rights are only available for licenses obtained through OEM or Volume Licensing (VL) and not apply to Full version (FPP).

• OEM downgrade rights apply to Windows 8 Pro and allows for downgrades up to two prior version

• Software Assurance through VL (Volume licensing) provides the greatest flexibility, allowing for downgrades to all prior versions and editions besides Windows 7 Ultimate and Vista Ultimate editions.Without Software Assurance coverage, VL has downgrade rights to prior versions of Windows with the exception of Enterprise.

• It is not possible for a Windows 8 copy with multi-language function to downgrade to Windows 7 Pro or Windows XP Pro as multi language function is exclusive to Windows 7 Enterprise edition and not available for Professional edition.

If you’re eligible for a downgrade, you can use any Windows 7 Professional DVD to start the process, but you’ll need to call Microsoft at 866.592.8221 to get a one-time activation code. Without it, you’ll have 30 days before Windows will go into ‘Reduced Function’ mode and you won’t be able to get important security updates.

a) Click on Start.

d) Right click on the program that you need to uninstall and click on Uninstall.

Method 2: If the above step fails then check if the Windows firewall service is running properly on the computer.

a) Click Start type services.msc in the search box, and then click OK.
b) In the Services (Local) list, right-click Windows firewall and then click Properties.

Make sure that the service status is set to started and startup type set to automatic.
