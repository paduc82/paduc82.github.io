The mshtml.dll file became unregisterable after Internet Explorer 6

If you are running IE7 or above, you will get an informative message like this if you try to register it:

RegSvr32
mshtml.dll was loaded, but the DllRegisterServer entry point was not found.
This file can not be registered.

There are many reasons you might have lots of copies of certain MS files on your system. 

Whenever you get a MS update involving these files, before the update installs, it makes a copy of the current file(s) in a new update specific folder (usually a hidden folder) so that you can uninstall the update later if needed.  If you uninstall an update later, it will put back the original file(s).

After a few years, those things accumulate but generally don't take up too much room on your HDD.  I have over 130 copies of mshtml.dll on my system (many are the same) but that's what I get after about 7 years of the same XP.

Most of the time, the files that "matter" are in these folders (assuming XP is installed on your C Drive):

c:\windows\system32\
c:\windows\system32\dllcache   (this is where XP keeps copies of important files)

If you delete the file(s), and if they are part of the 3498 files that Windows File protection looks after, they will swiftly and silently be replaced and you will see an Event like this In the Event Viewer System log:

Event Type:    Information
Event Source:    Windows File Protection
Event Category:    None
Event ID:    64002
Description:
File replacement was attempted on the protected system file c:\windows\system32\mshtml.dll. This file was restored to the original version to maintain system stability. The file version of the system file is 8.0.6001.19328.

So that means your Windows File Protection is working as designed and means it takes extra effort to delete important files even on purpose.

Since we don't know anything about your system (as usual, MS Answers Forum "Owners"), it is hard to tell if that is the correct version of mshtml.dll or not but if you are running > IE6, you will not be registering it.
