a. user say:

Your Windows Update history could be empty if you have followed any popular Windows Update troublehsooting advice that involves renaming/recreating the Software Distribution folder(s).

It is popular advice but sadly when folks suggest that they never seem to mention that the Windows Update history will be wiped out (but not your installed Updates).  There are a few other consequences too.

Your system is presenting symptoms of a corrupted user profile.

Some symptoms are:

Wallpaper missing/empty/changed

Desktop shortcuts missing maybe only showing Recycle Bin and a few other things

Personal files (photos, images, documents, etc.) missing but they really are not missing

Command Prompt opens in TEMP profile (sometimes)

USERNAME environment variable = SYSTEM instead of your Windows login name

Sometimes Windows has a problem reading your profile when it starts and sometimes simply restarting your system will "fix" things.

One observation/rumor is that for some reason after some recent Windows updates and on 32-bit systems with Microsoft Security Essentials (MSE) installed profiles can get corrupted and unreadable.  Is your system 32-bit and running MSE?

System Restore might help, a chkdsk with error correction (chkdsk  /r) won't hurt, a scan with Malwarebytes (MBAM) might be in order.

It is not possible to repair a corrupted user profile.

If the problem is chronic and nothing seems to help you need to create a new user profile and copy parts of the corrupted profile to the new user profile and from then on use the new profile in Windows.

Microsoft has an article that tells you how to do that:

https://support.microsoft.com/en-us/help/14039/windows-7-fix-corrupted-user-profile

The problem is likely that one of these files in the afflicted user profile is corrupted so don't copy these to the new user account:

  • Ntuser.dat
  • Ntuser.dat.log
  • Ntuser.ini

One of those files has the corruption in it so you don't want to bring those corrupted files (or any that look like it) into the new user account you have created.

a. user say:

5 beeps means "with ketchup."

Okay, seriously, when a computer emits beeps while starting - more than just one beep, which is normal for some computers - that's the sign of a hardware failure. These beeps actually have a name: they're called 'beep codes' and different beep codes mean different things.

Beep codes are unique to each manufacturer, so if you contact your computer manufacturer's technical support, they'll tell you what your beep code means.
