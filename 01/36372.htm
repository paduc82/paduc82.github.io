How to boot your system into Safe Mode…

1.     Restart your computer if it is powered on.

2.     Tap the F8 key after your computer initially powers on.

3.     Once you see the Advanced Boot Options menu you can stop tapping.

4.     Use the up/down arrow keys to highlight your selection.

5.     Select Safe Mode with Networking and press Enter.

6.     You should see drivers loading, and then Please wait.

7.     You should then be at the Welcome Screen.

8.     Logon to your computer using an account with Administrator privileges.

How to troubleshoot a problem by performing a clean boot in Windows Vista
http://support.microsoft.com/kb/929135
To help troubleshoot error messages and other issues, you can start Windows Vista by using a minimal set of drivers and startup programs. This kind of startup is known as a "clean boot." A clean boot helps eliminate software conflicts. This article describes how to troubleshoot problems in Windows Vista by performing a clean boot. This article also describes how to start the Windows Installer service and how to reset the computer so that it starts as usual.

Note When you perform a clean boot, you may temporarily lose some functionality. When you start the computer as usual, the functionality returns. However, you may receive the original error message, or you may experience the original behavior.

How-to run Windows in Selective Startup (Clean Boot)…

1.     Click Start, click Control Panel, click System and Maintenance, click Administrative Tools, and then double-click System Configuration.

2.     Click Continue, or provide Administrator credentials if prompted.

3.     Click the General tab, and click Selective startup.

4.     Under Selective startup uncheck Load startup items.

5.     Click the Services tab, check the Hide all Microsoft services box, and then click Disable all.

6.     Click the Apply, then OK, and then Restart.

Some things to check before you start your replacement of the computer's components.

If this computer is a brand name (as opposed to home built) such as a Dell, HP, Toshiba, Acer, Etc.

1) Make absolutely certain the case support the mounting of a standard ATX motherboard.

2) Check the power supply cable to the motherboard to insure the wires to each of the pins is also conforms to the industry standard.

3) If you plan to replace the power supply because the wattage rating is to low, check the case mounting hole locations and the AC power cord / switch location. Some companies did on occasion use non-industry standard mounting.

4) Cooling fans and space inside the case for the new CPU heatsink/fan.

a. user say:

Hi,

If you’re having trouble connecting to a network, we suggest following the steps provided on this link that can help fix connection issues on Windows 10.

To help you further, we recommend following the steps below to stop Service host (svchost.exe) that keeps on downloading:

  1. Right click on the Start menu.
  2. Click Run.
  3. On the Run box, type in services.msc then click Ok.
  4. On the Services window that will show up, right click on the Background Intelligent Transfer then click on Properties.    
  5. Under the General tab, look for Service status, click Stop then click Ok.

 

Let us know if you need further assistance.

a. user say:

Hi,

Background Intelligence Transfer (BITS) is one of the crucial services to run Windows Updates. Disabling it permanently will interrupt Windows Update installation, which is not recommended.

However, if you really want to stop it , we suggest following the steps below:


  1. Right click on the Start menu.
  2. Click Run.
  3. On the Run box, type in services.msc then click Ok.
  4. On the Services window that will show up, right click on the Background Intelligent Transfer then click on Properties.   
  5. Under the General tab, look for Startup type tab, then click the dropdown arrow.
  6. Choose Disabled, then click Apply and Ok.

NOTE: When updates are available, we recommend starting the Background Intelligence Transfer (BITS) for you to install the updates.
