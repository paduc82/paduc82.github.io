bill smithers say:

Hi,

 

No; if the Operating System on the other Computer came pre-installed (Dell, HP, Acer, etc) you can not use it on another PC.

 

 

If the Laptop/Computer came with the Windows Operating System preinstalled (Dell, HP, Acer, etc), that Product Key that came with the Laptop/Computer is for the preinstalled OEM License and is non-transferable.

 

The OEM License/Product Key (preinstalled Operating Systems on Dell, HP, etc machines) is tied to the first machine it is installed on and dies when that machine dies.

 

Only Retail Operating System License, that you buy and install yourself, are allowed to be transferred to another machine.

And Retail Licenses can also only be installed on one machine at any particular time.

Hi Renee. I'm Greg, an installation specialist and 9 year Windows MVP, here to help you.

What works for most to reset PIN so it works is here:

If you cannot Add or Remove the PIN in Settings > Accounts > Sign in Options even after a restart, then unhide Hidden Files in File Explorer > View tab > Options > Folder and Search Options > View tab > tick box to Show hidden Files.

Browse to C:\Windows\ServiceProfiles\LocalService\AppData\Local\Microsoft\NGC

Delete or rename to NGC.old this folder to reset the PIN.


You may need to first take ownership of the folder: https://www.tenforums.com/tutorials/3841-add-ta...

Once you delete or rename the folder it will recreate it so you can add a PIN normally in Sign-In Options. You may need to restart the PC first.

I hope this helps. Feel free to ask back any questions and let us know how it goes. I will keep working with you until it's resolved.
