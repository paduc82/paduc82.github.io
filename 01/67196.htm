You might have items in your Windows Media Player Library that have missing or incorrect media information, such as song titles, track numbers, artist names, album titles, or genres. For information about how to change the columns displayed in the Player Library.

See Change how you display items in the Windows Media Player Library.

a. user say:

It sounds like your tech either is trying to sell you some additional support services you don't need or doesn't know enough about how Windows Services work.

There are about 165 Services on a clean Windows 7 Service Pack 1 installation.  Depending on what other things you have chosen to install you may have more or less than that.

Services startup types can be set to an Automatic (Delayed Start), Automatic start, Delayed start, Manual or Disabled.  Some Services are designed to only start when the are needed and then stop automatically (those would usually be the ones set to Manual start). 

Depending on your configuration it is not unusual to see many Services that are not running (Stopped) in Task Manager.


On a system here I can count 109 Services in Task Manager that are showing Stopped:

Just because a Service shows a Stopped condition doesn't mean you have a problem or that it needs to be running.

If your system is not misbehaving, things are probably just fine the way they are.

If somebody tells you that some Service(s) need to be running they should be able to tell you why it needs to be running and what issue will be resolved if you have the Service running.

You do not describe any symptoms consistent with a need to run sfc /scannow.

You do not describe any symptoms consistent with a need to create a new user profile.

You do not describe any symptoms consistent with a need to clean boot.

For what you describe, those ideas are just wild goose chases.

And if you were to perform a clean boot, I would never follow this advice:

reset the computer to start as usual after clean boot troubleshooting

If you do that, you will enable every startup item and every Service on your system - even those things that don't need to be started and running.  The result will be long boot times and sluggish/poor performance.  The fewer things you have running on your system, that's just more resources that can go to the things you do need running.  Why waste CPU cycles and memory on running things all day and night that have nothing to do?

These ideas for operations that have nothing to do with the issue described, things to try, Methods and Steps are akin to saying "I have no idea what the problem is or how to fix it".

If you want to find out about Windows Services, what they do and if they need to be running or not here is one of the best places to look:

http://www.blackviper.com/service-configurations/black-vipers-windows-7-service-pack-1-service-configurations/

If you are not sure about any Services come back and ask questions about the specific Services of interest.

Opening a website with Cortana on a Windows 10 desktop computer - Microsoft Community

bobzeiss say:

Opening a website with Cortana on a Windows 10 desktop computer

I spent a couple of hours looking for the answer to my  question...  "how to open a website with Cortana".   I found a relevant discussion about it, but the Microsoft Community failed to answer it correctly.  I tried all the ideas and suggestions and all Cortana would do is open Bing and do a search for the name of the website I gave her.  Very frustrating.  Nothing worked, the thread was closed, no correct answer was ever given.  How pathetic.

So, I took it upon myself to find the answer to what should have been a very simple thing to do, you think.   This is the real solution that actually works.

First thing is to make sure Cortana is accepting voice commands using "Hey Cortana".  If not go into Cortana settings and turn this on.

 

Start by opening File Explorer and then navigating to the folder where Windows 10 stores your program shortcuts: C:\Users\{your user name}\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Settings.

 

Opening that folder should display a list of program shortcuts and subfolders. In this example, we want to add a shortcut to a website, so right-click any empty space in the File Explorer window, click New > Shortcut, enter the URL for the website (http://mywebsite.com) then click Next and give it the name you want to call it.   Don't use browse.

 

 

Repeat this process for any other websites you want to add. These custom shortcuts won't appear in the Start Programs list. (maybe they will after I restart my computer, have not done that yet.)

 

To open the website with Cortana, say LAUNCH {the name you gave the shortcut}, and it should find it and open for you in your default browser.  Don't say OPEN, you must say LAUNCH.  Make sure the voice recognition has spelled the name correctly. If not, rename it or say it differently.

Hope this helps all those that were frustrated with this, as I was.

bobzeiss say:

After setting up numerous website to open on a voice command to Cortana, I learned a little more that I can share with you.

First I created a shortcut to the Settings folder on my desktop so I didn't have to navigate step by step to get to the right folder where the settings are located to open websites on voice command.  See my previous post on where this folder is located.  Having the shortcut available really comes in handy and saves time and keystrokes.  C:\Users\{your user name}\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Settings.

 

The next thing I learned is that Cortana knows a lot of the names I used as a voice command.  If she already knew the name I used, she would take me to her website or URL instead of mine.   So instead of using Info or Data or File as a voice command, I had to think of a name she did not know, like My Club or My Bank, or The Unknown.   I had to keep trying names until I got one that worked.   Some of them she did not pronounce very well and took me to the wrong place.  So it takes a little trial and error to get a name that works.  Just so you know.  Leave your URL the same and change the name until you get one that works.

If you still have trouble, I found she did not like some of my URL's..... like those that begin with https.   If I used http instead, it would work and I would be redirected to the https site anyway.  Sometimes she wanted to see www in the URL.   So try these tricks as well if you are having trouble getting her to go to the site you want.
