rohn007 say:
Exactly what files do you want to "transfer"?

If you are talking about data files, you can just copy them to USB drives or OneDrive, from old machine then to the new machine.

If you are talking about installed programs, there is no simply way to "copy" them.

If you use a backup program to "restore" a backup of your old machine to the new machine you will likely have problems with drivers, because the 2 computers almost certainly use different drivers.

Installed programs do setup program files in specific locations, BUT they also setup a bunch of entries in the Windows Registry. Simply copying the installed program files from the old computer to the new one won't allow the programs to run. You have to run the setup/install program, and most likley provide a "product key" or "registration number" to prove you have the right to install that program.

Hi,

 

Thank you for posting your query on Microsoft Communities. I understand that you want to copy the profile settings from one user account and create multiple accounts with the same settings. I will do my best to assist you with this. 

 

You can copy the settings for desktop backgrounds, screen savers, pointer preferences, sound settings, and other features but not the applications.

 

Refer to the following suggestions and check if you can create a user account with the same settings.

 

a. Log on as a user other than the new user you just created or the user that you want to copy files from.

b. Press Windows key and x together and select File Explorer.

c. Click/tap the View tab, and then Click/tap Hidden items.

d. Now navigate to C:\Users\Old user\

Note: Old user is the user account that is not working.

e. Copy all the files, except these three files NtUser.dat, NtUser.ini, NtUser.log.

f. Navigate to C:\Users\New user\

Note: New user is the user account that you have created now.

g. Paste all the files here.

h. Restart the computer and login to New user and check.

 

You can do the same for other user accounts that you create.

    andre da costa say:

    Installation fails or becomes non-responsive at certain point during setup 18%, 62%, Driver Migration, Applying Settings.

    Power down computer.

    Wait 10 minutes.

    Start computer.

    Setup will automatically restore your previous Windows 10 build or previous version of Windows.

    Launch command prompt with administrator privileges.

    Press Windows key + X

    Click Command Prompt (Admin)

    Type the following command:

    rundll32.exe pnpclean.dll,RunDLL_PnpClean /DRIVERS /MAXCLEAN

    Hit Enter on your keyboard

    Exit command prompt

    Restart

    Try updating to build 10162 again.
