Sorry, I didn't recognize it. Since I have a laptop I have more options.

So your power button does a controlled shutdown of Windows. It is safe and when you restart you are starting with a "fresh" windows session, unlike if you set it to sleep or hibernate (which is what I have). Restarting the computer is often a good trouble shooting step for that very reason. If Windows code running in RAM "gets confused", restarting gets rid of that confusion.

Closing all of the applications before shutting down is a good idea. In the Fall creators update MS added an "improvement" that automagically restarts any applications you had open at shutdown. That has been confusing some people, plus it delays the restart.
