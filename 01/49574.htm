bossdweeb say:

"and they send me solutions that I already told them don't work."

Well, you haven't listed what you have already tried here, but...

- If you are running Win10 with the 1703 update installed you should be able to force wide screen with games

that run on low resolutions which are not native to wide screen ( 1024x768 / 1280x1024, etc).

Prior to 1703 Windows was controlling screen scaling, which forced games running 4:3/5:4 aspect ratios to run

with black bars on the sides on wide screen monitors. The user was unable to force wide screen.

With the 1703 update you can force wide screen in your graphics drivers control panel settings again.

- Gamehouse games look like they may be Browser/Flash based games. Have you installed or updated to the most 

recent versions of Adobe Flash and Shockwave players ?

Adobe - Install Adobe Flash Player

Adobe - Adobe Shockwave Player

Compatibility is not really a a very common issue, and the vast majority of games that ran on Vista/Win7 will

run on Win10.  Most on line game retailers, and services such as Gamehouse  do a pretty good job of

keeping their games up to date.

- However, the changes made with 1703 do bring back a minor problem that earlier versions of Win10 had

with low resolution games not fitting the screen correctly (related to screen scaling again), and you do need

to use Compatibility to fix this.

In the Compatibility tab find and enable - Override High DPI scaling behaviour. This option is required if

a game does not fit the screen correctly, but is mostly confined to old games that had a limited maximum

resolution eg -800x600, 1080x768, etc .

Windows being the way it is, will often fail to enable this option with games that need it, while enabling it on

games that don't.
