bossdweeb say:

- 'Gold Version' of what game ?

- Which Windows Update ? The update may have changed one of your security settings such as UAC.

It may also have caused your Anti-Virus to block the games silently, so you may want to try disabling your AV temporarily

to verify if this is happening.

"So I looked to the internet for answers "

What fixes have you already tried ? Just what you mention after that sentence, or more ?

"all of my mods are now rendered useless"

- This may be due to you having installed the game to the default location - Program Files (X86) ? The three Program folders

(Win7 as well) are protected system folders, and mods have been known to cause permissions issues when a game is installed to

the default location. Installing games that can be modded to a folder you create in C:\ or better, another Partition or HDD is

the safest way to avoid this issue.

Having said that, if the game ran before the update this would point more to the update changing a security setting.

"I looked to see if I still has secdrv.sys - I do "

Popping a secdrv.sys file, or keeping one installed by a game (a driver used by SafeDisk copy protection) into your Win10 System32 folder,

or elsewhere in Win10, has no effect on disc based games that use SafeDisk DRM.

- Any disc based game that uses SafeDisk requires using a noCD fix/hack to run games on Win10.

The other fixes are only applicable to Win7/8/8.1. This is because the DRM Blocking 'feature' was built into Win10,

but only added to Win7/8/8.1 via a 'security' update.

- Download versions of games are not affected by DRM Blocking as they don't rely on the secdrv.sys driver for a disc check.

"Forces of Corruption base game DLC"

"Star Wars: Empire at War: Forces of Corruption" ? Apparently this game uses a version of SecuROM DRM.

Game developers do strange things at times. Some games will use one DRM with the main game but use another with

the DLC (sometimes due to issues with the original DLC, and maybe sometimes because the DLC is not developed by the

same company).

However, if a game ran before a Windows update it should still run. The other main issue with DRM is some versions of

SecuROM, StarForce and Tages are not compatible with 64bit Windows.

"Really dont want to buy a crappy Apple Computer"

Only a relatively few game work on Mac anyway.
