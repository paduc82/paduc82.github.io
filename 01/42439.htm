Event Viewer tracks information in several different logs. Windows Logs include:

a.       Application (program) events. Events are classified as error, warning, or information, depending on the severity of the event. An error is a significant problem, such as loss of data. d.      System events. System events are logged by Windows and Windows system services, and are classified as error, warning, or information.
