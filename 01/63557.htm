xbr dionysus say:

Restore Factory Defaults

Login Information

  • Default Gateway - 192.168.0.1 / 192.168.1.1
  • Username - admin
  • Password - admin1/password1/<user generated on first login>

Restore Defaults

Through firmware

  1. Open a web browser.
  2. Enter http://192.168.1.1 or http://192.168.0.1 into the address bar. (Or click either link) Enter the username and password if the gateway asks. (admin/set at first login, write this password down)
  3. Select Manual Setup.
  4. From the menu across the top select Utilities.
  5. Select Restore Defaults from the side menu.
  6. Select the Restore Modem to Factory Default State button.
  7. Select OK on the following Pop-Up.
  8. Wait for the device to restart.

Through Hardware

  1. Locate Reset button on the back of the device.
  2. Press and hold this button using a toothpick or ball point pen for aprox. 30 seconds.
  3. Wait for the device to re-boot.

Hey there Clicked!

The "FlouryThrone03" Gamertag you're seeing is an auto-generated gamertag that pops up when you log into an Xbox console/website with an email that does not currently have a Gamertag tied to it.

Based on what you've said, it looks like the most likely scenario is that your gmail email was never associated with an actual Microsoft Account. It sounds like you tied your ClickedMilk64 Gamertag with your gmail email, but never tied anything with your Microsoft email. So, when you signed into Xbox.com with your Microsoft Account email, you auto-generated the Gamertag "FlouryThrone3".

It looks like your best option would be to head over to xbox.com/forgot and see if you can use the tools there to recover the account information for your ClickedMilk64 account.
