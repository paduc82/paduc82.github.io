OK, I stumbled on a translation of the above steps, cued in by the (erroneous) Microsoft instruction at https://support.microsoft.com/kb/313314?ppud=4&wa=wsignin1.0:

Go to Control Panel/Network and Sharing Center

Double-click on each connection that needs to be modified (e.g., LAN and wireless) individually and click Properties

Double-click on the Internet Protocol Version 4 entry

Click the Advanced button

Select the WINS tab and click the last button, "Disable NetBios over TCP/IP"

The results may be verified at the Command Prompt by entering "ipconfig/all" and examining the "NetBIOS over Tcpip" entry under each relevant adapter.

    The file "wer.dll" means "Windows Error Reporting". If this cannot run, you can probably fix it by doing the following.

    1. Right click the Start button and go to "Command Prompt (Admin)".
    2. Type (or copy/paste) sfc /scannow and hit Enter.

    System File Checker will check Windows system files for corruption and attempt to repair any found.

    If it can't, you'll see a message that says, "Windows Resource Protection found corrupt files, but was unable to fix some of them".

    In that event, use the same command prompt to run the following command which I suggest you copy/paste:

    DISM.exe/Online /Cleanup-image /Restorehealth

    If neither of these fixes the problem, open another Command Prompt (Admin) and run the following command to check your hard drive for errors.

    CHKDSK /f

    If you have a standard rotary hard drive, run this command instead:

    CHKDSK /f /r

    You'll get a message that the tool cannot be run because the disk can't be locked and whether you want to schedule a diags check on the next reboot. Click "Yes" and then either reboot immediately, or finish what you're doing and then restart.

    If CHKDSK finds bad sectors, it will mark them as such so that they can't be used. But if the tool appears to hang for more than a few of hours at a certain percentage, then that may possibly indicate that your hard drive is defective and will need to be replaced.
