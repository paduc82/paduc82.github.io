Hi Dan,

From what you wrote I understand where you're getting confused. So here are the steps to follow to resolve this problem.

Open the Windows Search tool, copy/paste the following and then click Search: qmgr.dll

Search should show that file in the following locations:

C:\WINDOWS\system32
C:\WINDOWS\ServicePackFiles\i386
C:\WINDOWS\system32\bits

Provided they're all present, do the following to register the service.

Go to Start, Run and copy/paste: regsvr32 qmgr.dll and click OK
(If you haven't got the Run command on the Start menu, hit Windows key + R instead)

Do the same with this file: regsvr32 qmgrprxy.dll and then reboot.

Next steps are as follows.

Copy/paste this command into the Run field and click OK
%windir%\System32\rundll32.exe setupapi,InstallHinfSection DefaultInstall 132 %windir%\inf\qmgr.inf

Windows is going to launch a dialog box asking you to insert the Windows installation CD. Ignore that and change the path at the bottom to C:\WINDOWS\ServicePackFiles\i386

The reason for changing the path to the i386 folder is because the files located there are newer than the ones on the Windows installation CD.

Reboot the PC again next and then proceed with the rest of the troubleshooting steps.

Go to Start, Run and copy/paste the following and click OK: services.msc

You should now be able to see Background Intelligent Transfer Service, or just "BITS" displayed in the list of Services.

Double click it to open BITS and then click the Start button. This is just a test to ensure it's working properly.

If you get Error 2: cannot find the file specified, go to Start, then Run again and copy/paste the following command and then click OK:
Reg add HKLM\SYSTEM\CurrentControlSet\Services\BITS\Parameters /v ServiceDll /t REG_EXPAND_SZ /d %windir%\System32\qmgr.dll

Reboot and then try and start BITS again.

If you get the same error again, copy/paste the following into the Run field and click OK:
Reg add HKLM\SYSTEM\CurrentControlSet\Control\BackupRestore\FilesNotToBackup

Reboot once more and then try and start BITS again.

Let me know how you get on please.
