How do you know they're random and useless? Have you looked up each service?

Search index for instance uses idle CPU time to sort through your hard drive so searches take less time - hardly useless. And when you need the processing power, the activity level should drop, to nothing if necessary.

You do have the ability to disable services in Task Manager, same as always. But I'd recommend taking a little time to learn about the services before you disable them - you're likely to break something if you turn off a service you think is useless that is critical to Windows.
