    chriso747 say:

    Quite often FSX cannot install in Win10 unless the Licencing System is reset.   Error in installing FSX usually gives error codes 1605 and 1622.   However, you can try the following in the hope that a reset will fix your failure to install FSX.

    Go to:  https://support.microsoft.com/en-us/kb/928080

    and download the Software Licencing System Reset Tool Package.

    Follow the instructions given (it works for Win10 also) but when you are asked to go to Command Prompt, make sure you click on Command Prompt (Admin).   If this method doesn't work, there are other things you can try, though this Tool Pkge may be your best bet for now.

    To answer your other questions:  FSX was the last Sim in this category and detail that Microsoft produced.   After this, they produced MS Flight, which is a simplified version with very limited capabilities and scenery packages (I can send you the link for this if you like).   However, FSX Steam Edition by Dovetail games is a good replacement for FSX.  It gives all the features of FSX and it is downloadable off their site at a cost and doesn't require a product key, etc.  But try all options to install FSX first.
