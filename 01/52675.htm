davidnovexa say:

How to disable auto refresh in Windows Explorer?

When I create or move files and folders to a folder, windows explorer auto refreshes the listing. This makes hard to track what has changed in this folder. Is there any way to disable / enable the auto refresh? With auto refresh, If I want to rearrange the list I press F5.

cmdrkeene say:

You can't stop it from refreshing, but you can tell it not to change the order of items as you are making changes.  That way the refresh won't actually cause any movement or change.

To do that, just sort by something that isn't going to be modified, such as the "sort by date created" (which can obviously never change), or "sort by owner" for example.
