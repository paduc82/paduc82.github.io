charlesjones32 say:
Hi. AnTiMaTteR

I'm Charles, an Independent Advisor trying to help.


First, check if the CD is write-protected or recorded. This error in the reading unit can occur for several configuration reasons, but also for hardware damage.

I enclose the resolution steps that you can verify:

1. Run the automatic repair tool. Then restart the computer and check if the problem persists.

2. Check the web page of your computer's brand if it exists to download a firmware update from the reader.

3. Uninstall any recording software or use the recorder as sometimes install drivers that generate conflicts. Restart the system and check if the problem persists.

4. Disable the antivirus you have, or add the reader as an exception. These programs scan all files for viruses and can slow down the reading of the compact disc, locking it or giving the impression that it is not being read.

5. Enter Start, Control Panel, Device Manager. Display the Units section of DVD or CD-ROM and press right click on the unit name that appears, there select Uninstall. Restart the system and check if the error persists.

6. Check if you have the latest BIOS update offered by the manufacturer of your computer (if it was assembled by a company) or the manufacturer of the motherboard (if it was assembled by a technician).

Best Regards,
Charles.
