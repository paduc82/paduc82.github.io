xb1r athena say:

Wireless Security

Login Information

  • Gateway - 192.168.0.1
  • Username ‚Äì user
  • Password ‚Äì user

How To Find Wireless Passphrase (WPA/WEP)

  1. Open a web browser.
  2. In the address bar at the top, type in the gateway address: 192.168.0.1.
  3. The device will prompt for a user name and password. By default the user name is ‚Äúuser‚Äù and the password is ‚Äúuser‚Äù.
  4. Click on the Wireless from the menu across the top.
  5. In the Wireless submenu on the left, click on the Security subtab.
  6. If your network is secured by a WEP key, the passphrase will be listed where it says Network Key 1.
  7. If your network is secured with a WPA/WPA2 Key, the passphrase is listed where it says WPA Pre-Shared Key.

 

 

Changing Wireless Security Type/ Disabling Wireless Security

  1. Open a web browser.
  2. In the address bar at the top, type in the gateway address: 192.168.0.1.
  3. The device will prompt for a user name and password. By default the user name is ‚Äúuser‚Äù and the password is ‚Äúuser‚Äù.
  4. Click on the Wireless from the menu across the top.
  5. In the Wireless submenu on the left, click on the Security subtab.
  6. In the drop down boxes at the top of the page, select the type of encryption you desire. The options available should be WPA, WPA PSK, WPA2 or WPA2-PSK.
  7. If you are setting up one of these types of encryption, in the box labeled WPA Pre-Shared Key type in a passphrase between 8-64 characters. This password will be case-sensitive and can use any key on your keyboard.
  8. If you desire to set up a WEP key, make sure all the boxes at the top are set to disabled (WPA, WPA-PSK, WPA2, WPA2-PSK).
  9. In the drop down box labeled WEP Encryption, select Enabled.
  10. If you are setting up a WEP key, type in a 10 digit combination of letters and/or numbers A-F and/or 0-9 in the box labeled Key1.
  11. If you are setting up no security on the network, make sure all  of the drop down boxes (WPA, WPA-PSK, WPA2, WPA2-PSK and WEP Encryption) are set to disabled.
  12. Click Apply at the bottom of the page.
