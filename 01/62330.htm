kari finn say:

I am afraid you are not going to like my reply...

That is a typical issue caused by a user. Try this slowly and you'll see what I mean:

Hold down CTRL, select one file. Now point another file, press left mouse button and before letting it go, drag it just a millimetre or two on any direction. Release the mouse button and you'll see that both files were copied.

2. Windows key + Pause key. This should bring up System. Click Advanced System Settings on the left > Advanced > Performance > Settings > Advanced > Ensure there's a check-mark for 'Automatically manage paging file size for all drives'.

If you cannot get into normal mode to do any of this, please do this via Safe Mode.
