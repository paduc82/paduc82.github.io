First of all, all Surface machines have underpowered USB ports.  This is via hardware and no firmware patch can correct this.

I ran a test on a Surface Pro 3 by wiping it out and installing Ubuntu Linux on it and tested the USB port and sure enough, could not power an external hard disk.

3rd party makers of Surface like tablets will have full powered USB ports.  I do have a low end, Asus T102 that even has a 12 hour battery life that will power an external hard disk via the USB port.

Microsoft did this with the Surface line to cut power usage and the USB port is only good for flash media storage devices or external hard disks designed to low powered ports (I have a WD external that will work with a Surface "as is").

You need to find something else instead of a Surface model to power an external monitor.  There is no fix solution for the Surface models to make them full powered via the USB ports.

There are two scenarios when punctuation mark is used in naming folders. 

If you are using only punctuation mark to save the file then you can search the files with the search string (that is only punctuation mark).
