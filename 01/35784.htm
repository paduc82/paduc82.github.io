techfreak_ say:
Stereo mode would be an AD2P profile. That mode is playback only as stereo requires more bandwidth. Mono mode would probably be Hands-free or Headset Profile (lowest bandwidth). Skype would use the Headset Profile. If the stack software for the external or onboard Bluetooth device allows the device to be configured using the Headset Profile, it may work. However, if the device chooses to mute playback in that mode, that probably points to a limitation/locked configuration in that hardware (firmware). The only other settings that may have any effect is toggling exclusive mode settings. Of the instances I have read online, in regards to this hardware, people simply purchase another vendor's hardware that does not exhibit this limitation.
