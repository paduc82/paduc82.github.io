In Computer Management it does not show up under Removable Storage

Cameras that take RAW images can take just regular, just raw or both at the same time.

Your raw image should have a file extension .NEF

Check your settings on your camera again, and check to see if you have any .nef files in your folder.

http://helpx.adobe.com/creative-suite/kb/camera-raw-plug-supported-cameras.html

shindeprataps say:

It turns out that machine.config file (as the error suggests) is the problem. The machine.config can be found on the CONFIG folder located under %SystemRoot%\microsoft.NEt\framework\v4.0.30319\config. So:

  1. Logon as admin
  2. Go to c:\windows\microsoft.NEt\framework\v4.0.30319\config and rename machine.config to something like machine.config.BAD
  3. In the same location create a folder called TEMP
  4. Copy machine.config.default from the CONFIG folder to TEMP folder
  5. Go into TEMP folder and rename machine.config.default to machine.config
  6. Copy machine.config from TEMP folder back to CONFIG folder
  7. Restart your machine

Your C00CE556 error should be solved.
