daavee say: Cloning is ideal if you suspect your hard drive is on its way out and you want to replace it. Simply clone the old drive to the new one, remove the old drive and replace it with the new one, and you're good to go.

Imaging is essentially distilling the essence of the hard drive into one gigantic file (or it can be a series of files, depending on the software and hardware used). This "archive" file may be stored anywhere and it can be used whenever disaster strikes. Simply use the associated restore function of the imaging software and the hard drive will revert to the EXACT state the moment it was imaged. For best results, create regular incremental images for more flexibility. This can all be done automatically in the background.

Sometimes folks think they can use Windows Explorer to copy every single file on a hard drive to another hard drive and that this will work in the event of disaster recovery. Actually it won't. The new hard drive will still lack important data in VERY specific places. Although this method is okay for backing up files such as documents, photos, mp3s, etc., it is useless when it comes to the operating system.

http://www.whatsabyte.com/P1/Image_or_Clone.html

http://www.computorcompanion.com/LPMArticle.asp?ID=314
