pranavmishra5582 say:

Hi,

There could be many possible reasons for unidentified network and limited connection error while connecting to a network connection. The very basic reason is IP address conflict. I would suggest you to follow the steps given below to manually set the IP address.

  1. Right-click the Network icon located at the bottom-right corner of the Desktop screen then click Open Network and Sharing Center.
  2. Click Change adapter settings present on the left hand side of the screen.
  3. On the Ethernet Properties window, select Internet Protocol Version 4 (TCP/IPv4) then click Properties.
  4. Click the radio button beside Use the following IP address.
  5. It will give boxes to enter the IP address you can type the address for example If you have a number of machines begin by setting the IP address to 192.168.1.100 then go to 192.168.1.101.
  6. The subnet mask has a default value of 255.255.255.0 in most cases.
  7. Copy the Default Gateway from the Network Connection Details window information.
  8. Copy the DNS Server info from the Details window--there will likely be two addresses to copy.
  9. Close all the windows and test your internet connection.
  10. Save the changes.

You can follow the steps given under the link below.

Why can't I connect to the Internet?
