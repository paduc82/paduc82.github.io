Adding to ITknowledge24s suggestion:

If running the application as administrator works then you may follow the steps as below.

a. Locate the programs shortcut icon or the executable file and right click on it.

b. Select properties.

c. Click the Compatibility tab.

d. Under Privilege Level, place a check mark next to Run this program as an administrator.

e. Click Apply and then click Ok.

Note: By doing this you do not have to right click and select Run as Administrator every time you want to run the application.
