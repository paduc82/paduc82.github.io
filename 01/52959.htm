aileen alf say:

Hi,

By default, Windows Defender is turned on in Windows 7. However, if you installed Microsoft Security Essentials (MSE), then it will disable Windows Defender by default, and MSE will be used instead. You will not be able to turn on Windows Defender as long as you have MSE installed.

To turn on Windows Defender through the Control Panel, kindly check and make sure that the Windows Defender Service is enabled and set to Automatic. Please refer to the steps below:

  1. Type services.msc in the search box and press Enter.
  2. If prompted by UAC, then click on Yes.
  3. Right-click on Windows Defender Service and click on Properties.
  4. Next to Startup type, click on the drop-down menu and select Automatic.
  5. Click on the Start button.
  6. Click on OK.
  7. Restart your PC for the changes to take effect.

You should now be able to turn on Windows Defender through the Control Panel.
