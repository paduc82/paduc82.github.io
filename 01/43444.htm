Hi Roger,

Thank you for posting your query in Microsoft Community.

The BCC and CC options are hidden in the Windows Live Mail by default and the users will have to manually enable it once to use the options. For doing that, the users should open the Windows Live Mail and proceed to compose a new mail. The recipient address fields and other options will be displayed on screen and the 'Show CC and BC' option will be displayed at the end of the subject field. Selecting the link will enable the BCC option and from now on, the users can use the BCC without manually activating it. Issues with the auto complete/auto-fill feature, unable to send to the addresses mentioned in BCC, freezing of the program when using the BCC options etc are some of the problems that can occur with the BCC in Windows Live Mail.

Discussed below are some points that will help the users to fix issues the BCC option in Windows Live Mail:

  • Restart Program and Reconfigure BCC
  • Update or Repair Windows Live Mail
  • Resolve Valid System Issues

Restart Program and Reconfigure BCC

The issues and errors with the BCC option in the Windows Live Mail might be caused by some minor errors in the program or the system. The users in these kinds of cases are advised to restart the program or the entire system and reconfigure the BCC options by opening Windows Live Mail. This will fix the errors and will enable the users to use the BCC option in Windows Live Mail.

Update or Repair Windows Live Mail

If the issue with the BCC in Windows Live Mail is not solved by restarting and reconfiguring the program, then it might be due to program file or configuration corruption. The users are therefore advised to update the program or the Windows Operating System as it will solve the issues. If not, go for a program reinstall to completely solve all issues with the with the BCC and the Windows Live Mail.

Also, we would like you to install the updates on your computer. Please follow the steps below:

1. Click on the Windows logo on the lower left corner of the screen.
2. Select Control Panel.
3. Click System and Security then selectWindows Update.

Note: If there's a needed update indicated, please install the available update. After doing so, please re-boot your computer for changes to take effect.

Hope this information is helpful. Please feel free to reply in case you face any other issues with Windows in future.

Disconnect the drive that has XP installed, disconnect ant other drives and only leave the drive that has Windows 7 installed on it. Next try booting, you should the following message or blank screen. Follow the instructions list below;

This error message is displayed when either the bootmgr file is corrupt or missing or the BCD (Boot Configuration Data) is corrupt or missing.

How to Find Ur ip address? - Microsoft Community

mℓiksh say:

How to Find Ur ip address?

My ip address doesnt match when i send an email to some one. For Exp when i send an email and trace the ip i looks like 182.185.19.**
But wen i look it in comand prompt or from control panel It Looks like this  192.168.**  .  so which one is real? i also visited whatismyip.com and it showed 182.185.19.** ??now can u solve this issue....i want a satisfying answer.....
k.c.s. say:

The "IP Address" that your network card has been assigned either statically or dynamically falls into two categories, IPv4 and IPv6 (different versions of the IP specification).  You mentioned IP Addresses that are formatted as IPv4 Addresses (the ones still most commonly talked about).

 

Your network card itself has one globally unique id called a "MAC ("mack") Address" and it can sort of be assigned "other names" like an "IP Address", depending on needs.  So, the address you see of 192.168.**  has most likely been automatically assigned to your network card by your router/switch that you have the computer connected to.  If you didn't manually hard code and assign a "static/fixed" IP Address to your network card then what is happening is your computer wakes up and says:

 

"Hi, my network card wants an IP Address.  Is there anyone listening on the network that can give me one?"  Your router hears that requests and says, "yeah, I can give you one, here, your network card can also be known as IP Address 192.168.1.5 (as an example) for now".

That process is called "DHCP", Dynamic Host Configuration Protocol.

 

So, the thing you see at the comand prompt IS your IP Address BUT that type of IP Address is NOT publically visible to the rest of the Internet, it only means something to all other devices on your LOCAL network.

 

So if you see something like "182.185.19.**" in your research, it is likely the IP Address that your Internet Service Provider assigned to your ROUTER which is the device and network adapter that is actually connected to the Internet (you want it that way so that it can filter out the bad stuff and keep it away from your computer that is happily on your home network and not directly connected to the Internet).

 

In short, to all devices on your same LOCAL network your computer is known as "192.168.1.5" but to any computers on the shared GLOBAL network called the Internet, your computer SEEMS to have the IP Address of "182.185.19.**" (the address of your ROUTER provided by your ISP).  So, when they want to talk to your computer they send it to your ROUTER and then your ROUTER has some magic in it to know, "oh this is really for that computer that I call 192.168.1.5" and it sends the info over to you.  Your router is acting as a "middleman" to get data to/from your computer over the Internet.

 

Please mark this as the Answer if it answers your question.  Thanks.
