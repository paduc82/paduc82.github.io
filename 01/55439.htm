Switching from Apple to Microsoft?

andre da costa say:

Microsoft had a site a couple years ago, but it looks like it has been retired.

A lot the work to switch will involve a lot of heavy lifting.

Applications

The first thing you will have to realize, all your apps will need Windows equivalent, you will pretty much have to purchase new Windows compatible versions or install the Windows version if the desired app includes such an option on the install disc.

Certain platform specific applications that do not have Windows specific versions will need to be replaced with close alternatives:

iWork - Microsoft Office or iCloud.com (which includes online versions of iWork) if you have documents in these formats. This will require an Internet Connection.

iLife - Windows Essentials which includes Photo Gallery, Windows Mail and other apps.

If you use Microsoft Office already, it should be easier to transfer documents over the Windows, but data like emails will be much harder.

Files

You will have to backup these manually, specifically to a FAT32 drive (example flash drive) with enough space. This will involve copying and pasting, so if you have a lot of documents, photos, music, videos, it will take a lot of work.

Specifically, iTunes data can be backed up easily to an external drive and restored on Windows in iTunes.

Personally, what I would do is setup a Boot Camp partition using the Windows 8.1 Enterprise 90 day trial and try copying over as much over your data as you can from the Boot Camp partition and then use PC mover to move that data from Windows installation to your new Windows PC.

Its a lot of work and patience. Personally, I would take the computer to a reputable computer service center and let them help. A lot of what is involved can't be easily explained in a post. Lot of complexities involved. Surprisingly, Apple makes it easy to switch from a PC to a Mac,  but not vice versa. Microsoft doesn't seem to invest much because well, its such a small market, 1.5 billion versus 100 million.
