a. user say:

how do I update my graphics driver on windows 10 64 base acer laptop so I can run droid4x 

If you're sure that you need to update your computer's graphics driver in order to run Droid4X, the first thing that you should do is to visit the Acer website to see if they have a more recent graphics driver for your computer.

The reason you'd want to go to Acer first is that manufacturers sometimes install customized versions of graphics cards which require specialized drivers that are only available from the computer manufacturer. This is particularly true in the case of laptops, which frequently use customized hardware.

If Acer offers a more recent version of your computer's graphics driver, you should follow Acer's instructions on how to install it.

Only if that driver causes problems for Droid4X, and you're really sure that you will need an even more recent driver, should you go to the website of the manufacturer of your graphics chip (AMD, nVidia or Intel) to see if they have an even more recent driver for your graphics card. In that case you should follow their installation instructions.

A chipset is not a graphics device. The chipset manages the flow of data on a motherboard. WDDM is the Windows Display Driver Module, which is the architecture used by graphics card manufacturers to build graphics cards for Windows. Version 1.1 of WDDM was written for Windows 7.

How do I trim multiple sections from 1 long video? I just bought a - Microsoft Community

eddiehines say:

How do I trim multiple sections from 1 long video? I just bought a new laptop that has Windows Live Movie Maker.

I am wanting to trim 1 long video (40+ minutes) into multiple short clips. I can only see ability to make 1 start and end point from any single video. So how do I make and save multiple clips from 1 video?
considerate_guy say:

It depends on whether you are trying to pare down the long video removing parts you don't want, or if you are really trying to break it into several separate files.

For the first case, imagine a 1 hours show with commercials you want to remove.  Or perhaps a long clip with some boring parts you no longer want.

In this case, you just do this by making splits between the good and bad parts.  You can click the 'Split' button on the Video Tools | Edit tab, or just by pressing 'M' on the keyboard.  Once you have split it into parts you want to keep and other parts you don't want, you select each clip you don't want and press the 'Delete' key.

Then you can save this as a new movie with just the clips you want to keep.

If your question is instead how to split your video into individual files, that takes more work.  Movie Maker is not the best tool for this because it is really used to make one edited movie from multiple clips.

The way to workaround this would be to make the splits desired using the steps explained above.  Save a copy of this project in case.  In fact, I'd make a second copy of the project now and work with that.  In that copy, delete everything but the first clip, save that as a movie (e.g. Clip 1.wmv).  Now either use Undo (Ctrl+Z) or reload the full project.  Now delete everything but the second clip and save that as a movie (e.g. Clip 2.wmv).  Repeat for each of the other clips.

arya s asok say:

Hi NicApicella,

 

 

Backups are saved in this format: <backup location>\<computer name>\Backup Set <year-month-day> <time>. For example, if your computer name is Computer, your backup location is E, and you backed up on April 2, 2006 at 16:32:00, that backup would be located in E:\Computer\Backup Set 2006-02-04 163200. You would right-click the folder named Backup Set 2006-02-04 163200 to delete that backup.

 

When you make a full backup, a backup folder is created and labeled with the date for that day. As you add updates, that date stays the same, but your backup is not out of date. The next time you make a full backup, a new backup folder is created and labeled with the date for that day, and any updates are then added to that new folder. You should not delete the current backup folder.

You can delete a backup files by following these steps:

1.     Open the Drive location where the backup is saved.

For example, if you backed up your files to hard disk labeled "E," then open drive E.

2.     Right-click the folder containing the backup you want to delete, and then click Delete.

 

Hope this helps. Let us know the results.
