Seriously, when a program refuses to close, this usually indicates that the program is performing a critical function and closing could cause corruption in the program. This can happen even though the system may view the program as being in a non-responding state. Forcing a program to close can cause any one of a long list of problems.

Never pull the plug on a computer. Doing so can immediately turn the machine into a large brick.

What you need to do is try and determine the cause of the problem.

Check with the developer of the program for any known issues.

Open Control Panel and type Troubleshoot in the Search Box. In the results you will see many installed troubleshooters that can be run to discover and fix problems.

Run the System File Checker. Open an elevated command prompt, type SFC /SCANNOW and press Enter.

How to use the System File Checker tool to troubleshoot missing or corrupted system files on Windows Vista or on Windows 7

Check the system drive for errors.

Check a drive for errors

If a problem is recent, you can use System Restore to go back to a point before the problem started.

System Restore: frequently asked questions

Clean things out.

Delete files using Disk Cleanup

Finally.

Optimize Windows 7 for better performance

Hope this helps.
