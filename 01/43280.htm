Hi,

To assist you with your concern with Windows 10 update using up a lot of bandwidth, you may try the following steps:

  1. On the search box, type gpedit.msc, right-click Use as Administrator.
  2. Go to Computer Configuration.
  3. Select Administrative Templates. 
  4. Select QoS Packet Scheduler.
  5. Select Limit Reservable bandwidth.
  6. Right-click Edit, click Enable.
  7. Set to 60.

To manage the updates:

Method 1: 

  1. On the search box, type gpedit.msc, right-click Use as Administrator.
  2. Go to Computer Configuration.
  3. Select Administrative Templates. 
  4. Select Windows Components, then Windows Update.
  5. Select Configure Automatic Updates, right-click Edit.
  6. Select Enable.
  7. Set to Notify for download and notify for install.

Method 2: 

  1. Go to Settings, then Update and Security, Check for Update. 

Method 3: 

  1. Go to Settings, then Update and Security.
  2. Advance options, Choose how update are installed.
  3. Set to Notify for download and notify for install.

Method 4:

  1. Go to Settings, Update and Security.
  2.  Advance options, Choose how update are installed.
  3. Choose how updates are delivered, then off or pcs on my local network
