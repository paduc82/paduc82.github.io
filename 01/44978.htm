nicky radonskiy say:
Hi Matias, I suppose that your issue is caused by the task manager policies that have been affected by the changes. I recommend that you modify the group policy.
1. Create a restore point before starting the process because serious problems might occur if you modify the group policy incorrectly https://support.microsoft.com/en-us/help/402753...
2. Go to Start menu - type in Gpedit.msc in the command line and press on Enter button.
3. Go to User Configuration - Administrative Templates - System - Ctrl+Alt+Del Options.
Find the option Remove Task Manager in the right side of the screen, verify that it is set to Disable or Not Configured
Close Gpedit.msc MMC.
Go to Start menu - type gpupdate /force in the command line and click on Enter button.
Restart the PC.
If the issue persists, post your query on TechNet forum https://social.technet.microsoft.com/forums/en-...
