HI sherlocc,

Thanks for posting. Please do the following to see who the master brower is on your workgroup.

1. Click Start.
2. In Search area type "cmd" and press "Enter"
3. in the command prompt type "net view" (this will list the computers in your workgroup).
4. After you have identified your computer names type "nbtstat -a COMPUTERNAME" so if your computer name is //john-pc type "nbtstat -a john-pc"
5. This will list information on the computer that will look like the following:
                           NetBIOS Remote Machine Name Table

                            Name               Type         Status
                         ---------------------------------------------
                         JOHN-PC       <00>  UNIQUE      Registered
                         WORKGROUP      <00>  GROUP       Registered
                         JOHN-PC       <20>  UNIQUE      Registered
                         WORKGROUP      <1E>  GROUP       Registered
                         WORKGROUP      <1D>  UNIQUE      Registered
                         ..__MSBROWSE__.<01>  GROUP       Registered

6. If the comptuer is the Master Browser you will see the  ..__MSBROWSE__.<01>  GROUP       Registered

Hope this helps!
