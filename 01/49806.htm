Go into the Control Panel and then into Administrative Tools.

Click on Services.

Scroll down until you see the entry for Windows Time.  Mine is set to Manual (which means it should only start when I actually ask Windows to syncronize the time), so if it is set to Disabled, then Windows cannot start it.  Either set it to Manual or to Automatic (Delayed Start) and then click on Start Service (top left of the lines of Services) and see if it will now syncronize your PC with Internet Time.
