This isn't the answer for this particular question. We know how to download cursors.

We need to know where to drop the .cur files in order to make them usable. I've done this on older versions of windows, but it's been ages and I'm not sure the way I did it even exists on Windows 10.

glaskaki say:

It seems that the way I did this on older versions of Windows still applies. Use windows explorer/Windows 10 equivalent is called This PC to navigate to "Local Disk (C:)>Windows>Cursor". Make a new folder for your custom cursor set, and drop it in. This should not be compressed in any way, i.e. it should be just a regular folder with the .cur files in it, not a .zip or .rar. You should have at least the basics, like the resting cursor, hovering over a link cursor, etc etc, not every single arrow is required to make it work.

After it is dropped in, hit the start menu, then settings. Type "cursor" in the find a setting bar. Pick "Change how the mouse pointer looks" and it will give you a pop up with a new menu.

From here, it's pretty easy.

Make sure you are on the pointer tab, the scheme is set to none, then click each 'type' of cursor one at a time in the Customize menu. When the cursor you want to change is highlighted blue, hit browse and navigate to that new folder you made right inside the default location it opens to. Select the corresponding custom cursor, then hit open. This will replace the default option with your corresponding custom cursor. Do so for all options you'd like to change, then hit Save As under the scheme selection drop down box. Give it a name, then hit Apply and then Okay. Your cursor should change right before your eye!

Windows 7 Out Of Date Drivers

canadian tech say:

Drivers are not like Windows Updates.  They should be updated only when a specific problem needs to be solved.  Your only trustworthy source of drivers is the OEM of the computer.  OEMs tend to issue revised drivers only shortly after a model comes out.   After that, mostly you will not see new drivers. 

One exception is Intel.  You can get Intel drivers from the Intel web site.

The internet connection problem you describe is not likely due to out of date driver, it is possible, but unlikely.  Try just uninstalling and re-installing the existing driver.  After that that kind of problem are usually:

  • A failing piece of hardware
  • Poor signal quality
  • The wrong network password

Most often, I find simply buying a wireless USB device for about $20 solves this kind of a problem.  Most of them are better than the built in ones anyway.
