For Windows 7, .NET Framework 3.5 with SP1 is shipped with it as one of OS component.
For .NET Framework 3.5 with SP1, it includes .NET 2.0 with SP2 and .NET 3.0 with SP2 as prerequisite.
This means, you have the .NET Framework 2.0 SP2, 3.0 SP2 and 3.5 SP1 plus a few post 3.5 SP1 bug fixes on Windows 7. They are OS components. .NET 3.0 SP2 and 3.5 SP1 can be added or removed via the Programs and Features control panel.
Thus, we don't need to manually install them.

This is my add on:

Each .net framework is a individual program, that developers use to create programs.

Programs created by a .net framework need the same .net framework installed, if not found the

program requesting this item will ask for the item to be installed..

You can download and install .net framework 4.0.

I have installed .net framework 4.0 client profile,

and .net framework 4.0 extended. These will show in add and remove list.

Keep them all on, they are individual programs.
