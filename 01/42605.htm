ken blake - microsoft mvp say:

On Mon, 31 Jan 2011 19:08:48 +0000, Mainman407 wrote:

Yes, I have three computers. The Operating System on my office computer is Windows XP Pro 32-bit. My Laptop has Windows XP Home Edition-64-bit, and my wife's office computer has Windows Vista-32-bit.

I would like to upgrade my office 32-bit [O]erating [S]ystem to a 64-bit OS. I would also like to upgrade my wife's 32-bit Windows Vista OS to 64-bit OS.

It's not possible to do it by upgrading. You can not change the
"bitness" of Windows from 32-bit to 64-bit, nor vice-versa. The only
way to get there is via a clean installation.

My wife's PC (the one running Windows Vista) has a Dual-Core Athlon 2.66-GHz processor. It also has four DD2 Memory slots. From what I have gleaned from the web, since this system is 32-bit, I  was told that the system recognize only 4-Gigs of DDR-2 RAM (max.).

All 32-bit client versions of Windows (not just XP/Vista/7) have a 4GB
address space (64-bit versions can use much more; see below). That's
the theoretical upper limit beyond which you can not go.
But you can't use the entire address space. Even though you have a 4GB
address space, you can only use around 3.1GB of RAM. That's because
some of that space is used by hardware and is not available to the
operating system and applications. The amount you can use varies, depending on what hardware you have installed, but can
range from as little as 2GB to as much as 3.5GB. It's usually around
3.1GB.
Note that the hardware is using the address space, not the actual
RAM itself. If you have a greater amount of RAM, the rest of the RAM
goes unused because there is no address space to map it to.

However, supposedly, a 64-bit OS will recognize 8-Gig of DDR-2 RAM.

How much RAM a 64-bit version of Windows will recognize depends on the
version and the edition of that version. The number goes as high as
192GB for 64-bit Windows 7 Professional, Ultimate, and Enterprise.

I want to fill each slot with a 2-Gig strip so that I can perform certain CAD operations, without slowing down the system.

Does anyone know what steps I will have to perform in order to revert my Windows XP Pro and Windows Vista systems to 64-bit systems?

Two steps:

1. Buy the 64-bit version you want.

2. Clean install the version you bought.

i am having a desktop with windows 10 pro by mistake I changed it to enterprise can I go back to pro without clean install

chapa :) say: to perform a Clean installation in order to install Windows 10 Professional.

andre da costa say:

There is a possibility you might be able to.

  1. Open Registry Editor(WIN + R, type regedit, hit Enter)

  2. Browse to key HKEY_Local Machine > Software > Microsoft > Windows NT > CurrentVersion

  3. Change EditionID to Pro (double click EditionID, change value, click OK). In your case it should at the moment show Enterprise

  4. Change ProductName to Windows 10 Pro. In your case it should at the moment show Windows 10 Enterprise

  5. Browse to key HKEY_Local Machine > Software > Wow6432Node > Microsoft > Windows NT > CurrentVersion, change the same two values as in steps 3 and 4 above

  6. Close the Registry Editor

  7. Insert the Windows 10 Proinstallation media, do an in-place upgrade by launching the setup from desktop (not booting with Windows 10 Home install media)

  8. When asked, enter your validWindows 10 Proproduct key and select what to keep(settings, personal files and apps, only personal files or nothing)

  9. Windows 10 Proin-place upgrade will be done now.
