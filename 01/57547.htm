Anytime Upgrade to Windows 7 Ultimate

Type Anytime Upgrade in the Search programs and files box in the Start menu and click on the Windows Anytime Upgrade icon. From there, you can buy an Anytime Upgrade to Windows 7 Ultimate. Then your can enter your Anytime Upgrade product key and perform a simple upgrade to Windows 7 Ultimate so you can install a different language. All programs and files will stay on the computer. If you're unable to buy online in your area a local store may sell Anytime Upgrades, or you can buy the version of Windows 7 you want.

http://windows.microsoft.com/en-us/windows7/products/windows-anytime-upgrade

Install New Multilingual User Interface (MUI) in Windows 7 Ultimate:

1. Click Start and type Windows Update in the Search box.

2. Click Windows Update in the Programs list. 

3. Click Optional Updates and go to the section Windows 7 Language Packs.

    Note: If the optional update link can't be found, click Check online for updates from Microsoft Update to search the optional updates. 

4. Check the Language Pack you want and click OK.

5. Proceed with Install Updates.

How to change the Windows 7 Display Language:

1. Go to Start -> Control Panel -> Clock, Language, and Region / Change the display language

2. Switch the display language in the Choose a display language drop down menu.

3. Click OK

4. Restart for the changes to take effect (this will affect only the user profile under which you are currently logged on).

Sign-in Options in Windows 8

Technical Level : Basic

Summary
Describes how to customize or disable the sign-in options for Windows 8. This article only requires basic computer skills.
Details

Many people want to know how to disable or customize their password in Windows 8.  There have been several questions posted in the Microsoft Community Forums, and my post about this on my blog has currently received 347 views as the most viewed post on my blog - the next closest post has only received 59 views.  This article will teach you how to customize and remove the sign-in requirements on your computer.

Windows 8 has three ways to log-in to your computer - passwords, pins, and picture passwords.  These options can also be disabled if you dislike having to sign-in every time you access your computer, although your computer will not be as secure then.  I will teach you how to customize your sign-in options to your preference.

To begin the process, open the PC Settings from the Settings Charm, detailed instructions can be found here.  Select the "Users" tab in the side bar:

To create a pin, click the "Create a pin" button, which is not visible in the picture above as I already created a pin.  Enter your password when prompted, then enter a four digit combination.  Click "Finish" to save your pin.  You can use your pin next time you sign-in to your computer.

To create a picture password, click the "Create a picture password" button, which is just above the "Create a Pin" button.  Enter your password when prompted, then choose a picture to use.  Now you need to confirm that you want to use this picture and enter a combination of three gestures, which can be clicks, lines, or circles.  Confirm the gestures and click "Finish."

First option to bypass the password requirement - removing the password:

To by-pass the sign-in requirement, simply go to the same PC Settings page and click "Change your password."  Verify your current password, then leave the other fields blank.  This has the advantage of being easy, but does not allow the computer to remember site passwords in Internet Explorer as you need a master password for that option to work.  Sometimes the computer unexplainably does not allow you to leave the password fields blank, so next I will describe a more complicated way to remove the sign-in feature, which happens to allow Internet Explorer to remember site passwords as well.

Second option to bypass the password requirement - removing the sign-in requirement:

Type NETPLWIZ into the Start Screen and open the program the comes up, you could also use the Run command or the Command Prompt.  Uncheck the "Users must enter a user name and password to use this computer" check box and click OK.  Now open the Control Panel and search POWER.  Open the "Change what closing the lid does" result and click the "Change settings that are currently unavailable" link.  Select the "Don‚Äôt require a password" bubble and click "Save changes":

The last step is to navigate to the Screen Saver Settings window in the "Change the Theme" settings under "Appearance and Personalization" in the Control Panel.  Uncheck the "On resume, display logon screen" box and click OK.  You should never need to enter your password to access your computer again after following these instructions to disable your password.

If you have any problems, feel free to leave a comment here or post a question in the forums by clicking the "Need help?  Ask a Question" link at the bottom of this article.

considerate_guy say:

Windows DVD Maker will take a set of images or a mixture of images and videos and make a DVD. It will also generate a menu of "scenes" from those items. But it doesn't give you direct control over where those "scenes" are split. So you can't give it multiple images and tell it to group them into different slideshows.

However, if you give it a set of videos, it will prioritize the scene breaks to hopefully match with the start of each of those videos.

So, as skier-hughes noted, you just have to make individual slideshows in Movie Maker. Then take those videos and use them to make a DVD in Windows DVD Maker.

Movie Maker is part of Windows Essentials which you can download, but Windows DVD Maker is not; it is part of Windows Vista or Windows 7. (It isn't present in Windows 8 or higher and can't be downloaded).

So, in summary:

  1. Add your set of slides to Movie Maker. Arrange them, add optional music and titles or an optional AutoMovie theme, or you can add transitions and effects as wanted.
  2. Preview the project and save the project if you want to come back for later editing.
  3. Save this as a movie but be sure to pick the "Windows Media Video (*.wmv)" format because that's what Windows DVD Maker expects (not .mp4)
  4. Repeat the process for each slideshow.
  5. Open Windows DVD Maker and add the individual movies you saved previous.
  6. Follow the rest of the steps for choosing the menu layout, previewing, etc.
