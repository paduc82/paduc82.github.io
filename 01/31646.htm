trung n nguyen say:

Hi Kevin, Windows 8.1 does support Windows Accounts that don't have  a Microsoft Accounts tie with them. May I asked what have you try? There are a couple of ways that you can manage a Windows Account that are not ties to a Microsoft account, the only prerequisites are: 1) The parent's (you) Windows Admin account as to be tied to a Microsoft account. 2) The Windows Account you want to manage online has to be a "Child" account type. I believe in Windows 8.1, Microsoft introduced an account type called "Child" in addition to the Standard and Administrator account types. Below are a couple of ways to manage a Windows Account that does not have Microsoft Account ties with it.

1. Existing Windows Account:

- For any existing Windows Account that you want to manage make sure you set them to "Child". To do this, on the PC, search for Your account settings and select it from the results, click on Other accounts, select the account in question and click on Edit, then select Child.

- Now on the PC search for "Family Safety" and select it from the results.

- Click on Manage settings on the Family Safety website. When prompt to sign into the Family Safety website, enter your credential and verify the child is added to the family.

2. Creating a new Child account without adding a Microsoft Account.

- On the PC, search for Your account settings and select it from the results.

- Click on Other accounts.

- Click on Add an account.

- Click on Add a child's account.

- Click on Add a child's account without email.

- Enter a User name and password for this child.

- Click Next.

- Click Finish.

- Click on Manage Family  Safety settings online and sign in using your Microsoft Account, and verify the new child is added.

Hope this helps.

The option to choose is USB FDD (Flash Disk Drive)

If you have problems booting from a boot disc, you may have UEFI BIOS:

Insert the Bootable Installation Media, then go into your BIOS and make the following changes:

2. Enable Legacy Boot

4. If Required enable USB Boot

6. Save BIOS changes, restart your System and it should boot from the Installation Media
