You are mentioning two seperate issues and products that are unrelated (except for name similarities).

Issue #1: Skype for Windows 10 App installation and association issue

Skype - UWP Windows 10 App - Standard client/non-business - The Skype for Windows 10 part of the Answers Community.

When you use launch skype on the Skype.com page, it will utilize the SkypeUWP: protocol. This issues a call to whatever program is associated to that protocol. This will either be no program or the Skype for Windows 10 App. To resolve the problem:

a. If Skype for Windows 10 is installed, manually fix  the association for the SkypeUWP protocol by selecting the Skype for Windows 10 App (UWP).

b. If it does not work properly, install or uninstall then reinstall the Skype for Windows 10 Appto fix the program and its associations.

c. If you prefer not use the Skype for Windows for Skype (non-business) communication, install and utilize the classic Skype (7.x) desktop application instead.

Issue #2: Skype for Business Web plug-in installation or possible compatibliy issue with your web browser

Skype for Business (part of Office 365) - Web client plugin/guest (non-client login) - The Skype for Business/Office 365 part of the Answers Community.

When you join a Skype for Business as a guest/without a compatible client/Office 365 subscription, the link sent will forward you to a page where you can install the Skype for business web client plug-in, then launch Skype for business within that browser.

a. Upon clicking on the link your default web browser will be used. If the web client plugin is not installed properly, blocked by your antivirus or is incompatible with your default web browser, the Skype for Business web client will not launch.

b. Try another web browser, You do not necessarily have to change your default web browser. Simply copy and paste the link sent to you and try to install the Skype for Business plug-in, then launch the client by clicking on join the meeting.

c. If the launch of the web application will not work initially, you may simply have to fully exit the web browser used, reload the invite page, then click join the meeting again.
