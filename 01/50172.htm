HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Policies\system\

There, in the right pane, modify the value of the DWORD named

EnableLUA

and set it to 0. Reboot the computer

or, alternately,


(2) at the same key, create a DWORD value named

LocalAccountTokenFilterPolicy

and set its value to 1.
