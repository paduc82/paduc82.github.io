Quicken Backup - Microsoft Community

dan517 say:

Quicken Backup

I cannot backup my quicken file in onedrive?
a_user_here say:

What is the problem?

There should be no reason you can't backup your Quicken file to OneDrive.

On the other hand it might be a bad idea to run Quicken where it is reading data file out of OneDrive.

It all depends on how OneDrive treats the lock that Quicken puts on the data file while it is running.

If OneDrive honors the lock and doesn't do a sync while Quicken is running, then this would be OK. 

If OneDrive on the other hand goes ahead and syncs the data file while Quicken is using it, it could be in an inconsistent state.  Also you should never change the same data file in two different places, before it syncs.  Because there is no way to merge the changes from two different locations.

I have not tested with OneDrive to see how it treats the locks, but the test would be basically make sure it doesn't sync to another machine while you are using Quicken.  I have heard that DropBox does this properly, so maybe OneDrive would too.

chiefjoboo say:

Unfortunately you can't move Xbox accounts to another Microsoft account. You could change the login email but that will affect your login info as well. You can't set some parental controls on an adult account - such as screen time limits, and with others, the person on the account can disable any limit on anything if they want to and know how to.

The best thing to do in this situation is to create a new Microsoft account as a child account (Xbox accounts are under Microsoft accounts) but your child will have to gain all the progress again in all the games they play - this can't be avoided if you absolutely need to set parental controls on the account.

Please note that changing the birth date does NOT change the account to a child account - that age you entered when you created the account is permanently connected to the Xbox account of that Microsoft account. There is currently no way to change an adult account to a child account - it only goes the other way around and only when that first entered date of birth reaches 18 (20 in Japan and South Korea).

  4. Type the name you want to give the user account, click an account type, and then click Create Account.

Transfer your old profile information to your new one, here's how:

fter you create the profile, you can copy the files from the existing profile. You must have at least three user accounts on the computer to complete these steps, including the new account you just created.

  1. Log on as a user other than the new user you just created or the user that you want to copy files from.

  2. Open Documents by clicking the Start button , and then clicking Documents.

  3. Click the Tools menu, and then click Folder Options.

    If you don't see the Tools menu, press ALT.

  4. Click the View tab, click Show hidden files and folders, clear the Hide protected operating system files check box, and then click OK.

  5. Locate the C:\Users\Old_Username folder, where C is the drive that Windows is installed, and Old_Username is the name of the profile you want to copy files from.

  6. Select all of the files and folders in this folder, except the following files:

    • Ntuser.dat

    • Ntuser.dat.log

    • Ntuser.ini

  7. Click the Edit menu, and then click Copy.

    If you don't see the Edit menu, press ALT.

  8. Locate the C:\Users\New_Username folder, where C is the drive that Windows is installed, and New_Username is the name of the new user profile you created.

  9. Click the Edit menu, and then click Paste.

    If you don't see the Edit menu, press ALT.

  10. Log off, and then log back on as the new user.

    If you have e-mail messages in an e-mail program, such as Windows Mail, you must import your e-mail messages and addresses to the new user profile before you delete the old profile. If everything is working properly, you can delete the old profile.
