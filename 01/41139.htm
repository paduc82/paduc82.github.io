Your diagram (photo/image) shows only restore points that were created via System Restore.  The fact that one type is named "backup" is not because it contains your System Image that you created but because System Restore ran prior to the System Imaging process starting.  The restore point called Test1 was probably a name created by you when you created a manual restore point.  The name Automatic Restore Point was given by System Restore itself, it will every now and again create a System Restore point to automatically save important files as a precaution.
