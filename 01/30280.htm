a. user say:

Hi Tom,

 

I see that this is your first post in this forum, welcome to the Microsoft Community forum.

 

The start screen displays the most frequently used programs in your computer for easy access. If you didn’t find a program that you frequently use in the start screen, you can always pin that app to start screen.  You can also access a list of all the programs installed in the computer by navigating to the ‘All apps’ screen. Check out this article for a quick guide and some great tips on using Start screen:

 

http://windows.microsoft.com/en-gb/windows-8/start-screen-tutorial

 

http://windows.microsoft.com/en-gb/windows/how-to

 

You have also mentioned that you couldn’t find the ‘Shared documents folder’. You can find the shared folders named as public folders. These folders are located in C:\Users\Public (where C:\ is drive where Windows is installed). This contents of these folders might not be shown in the start screen apps of other user accounts since it is not included in the library. The start screen apps shows contents from the Library. To add the “public documents’ folder to library, log into each user account, navigate to the ‘Public documents’ folder and right click on it to select ‘Include in Library’. Click on ‘Create a new Library’ and add a name to the new library. Now the contents of these folders will be available in the start screen apps.

 

The apps that you download in a user account will be available only in that user account. Since most of the apps are connected to Microsoft accounts or other social accounts, the apps of one user account won’t be available in other user accounts. The other user accounts can download the same app by going to the Windows store app in their user accounts.

 

http://windows.microsoft.com/en-gb/windows-8/apps-windows-store-tutorial
