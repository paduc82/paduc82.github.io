  1. Right-click the Start button.
  2. Click Search.

  4. Double-click HKEY_LOCAL_MACHINE.
  5. Double-click SOFTWARE.
  6. Double-click Policies.

  16. Double-click the NoLockScreen DWORD you just created.
  17. Type 1 in the Value data field.
  18. Click OK.

Step 1: First step is to open Run command box by simultaneously pressing Windows logo and R keys (Windows+R). In the Run dialog box, type Netplwiz and then press Enter key.
