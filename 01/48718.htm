Essentially, I am looking to log into my email, and have a mirrored image/storage/account on both computers.  How do I do this??!

 

(Note:  I have already uninstalled and reinstalled both Essentials and Windows Live Mail on both computers, and deleted and readded my email account on both computers...)

 

THANK YOU SO MUCH!!!!

ron sommer mvp 2009-2017 say:

Open Windows Live Mail.

Right click on the account name in the folder pane, Properties.

Uncheck 'Include this account when receiving or synchronizing'.

 

Add the account again.

Enter the full email address, password, and display name.

Click next and Finish.

A Microsoft account will automatically use http protocol.

Do a Send/receive.

After the folders and messages have synced with the server, move messages from the pop3 account to the new account.

After doing a Send/receive, the emails and folders will sync to the server.

If you have messages in the pop3 account that you do not want on the server, create folders under Storage folders.  Move the messages to the created folders.

After all messages have been moved, delete the pop3 account.

Port Forwarding - Microsoft Community

xb1r athena say:

Port Forwarding

Login Information

  • Gateway – 192.168.0.1
  • Username – admin
  • Password – admin

 

Port Forwarding

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. If prompted to login enter “admin” as the username and “admin” for password.
  4. Select Virtual Server from the menu on the left.
  5. Select Port Range Forwarding from the sub menu.
  6. In the Start and End Port enter 3074.
  7. For the To IP Address enter the Xbox’s IP Address.
  8. For the Protocol drop down select Both.
  9. Check the Enable box.
  10. Click the Apply button.

Disable Port Forwarding

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. If prompted to login enter “admin” as the username and “admin” for password.
  4. Select Virtual Server from the menu on the left.
  5. Select Port Range Forwarding from the sub menu.
  6. Find the port you want to remove then uncheck the Enable box
  7. Check the Delete box.
  8. Click the Apply button.

vinod n say: Hey Athena,
 
Here are some advantages of HomeGroup:
 
1. Easily share the photos, music, and files on each of the PCs running Windows 7 on your home network. It's especially handy when you want to share a printer.
 
2. HomeGroup takes the headache out of sharing files and printers on a home network. Connect two or more PCs running Windows 7, and HomeGroup makes it easy to automatically start sharing your music, picture, video, and document libraries with other people in your home.
 
3. HomeGroup is password-protected and puts you in total control. You decide what gets shared—and what stays private. You can also make your files "read only," so other people can look at—but not touch—your stuff.
 
4. Easier to just put the file on a USB drive and take it to the PC with the printer.
 
5. Print a file at home when you aren’t connected to a printer.
 
6. Homegroup makes workgroups easier. If you have 10 PC's in a workgroup with a password shares, you need to create user accounts on all the PC's. With homegroup you can just create 1 password for shares, no accounts needed.
