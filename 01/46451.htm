I doubt the real problem has to do with DirectX. Win10 comes with DX9,10, & 11 as well as DX12, and it's not

something that you down or upgrade.

Games determine which DX files they use, not Windows, and many games use both DX9 and DX10\11. A lot of more recent games

only use DX11 or DX12 (not many games using DX12).

In this instance Windows probably has no idea what the source of the problem is, only that Direct X is involved, as it is with all games.

- If you have already reinstalled DirectX as Muralidhar suggested, you should also update Win10  to 1903. Some recent games

require Win10 to be up to date, (but they are generally more likely to not launch if Win10 is not up to date).

The disc based version of GTA5 does not have a DX installer in the games folder as it does in Steam. There is one on the disc though.

To verify if DX is up to date you can run the Direct X Web Installer, but if there is a suspected corrupt DX file you may need to reinstall

DirectX using the Direct X Redist (June 2010), as this overwrites all DX files, replacing any corrupt files

Download DirectX End-User Runtime Web Installer from Official Microsoft Download Center

Download Details - Microsoft Download Center - DirectX Redist (June 2010)

"1. The games are not installed from Steam, all are from external hard disks."

- This is likely the main problem. Games often develop issues when they are copied to another HDD rather than being

installed properly, as they loose a number of connections created with the Windows they were installed with originally.

Rockstar use several tools as part of their extended DRM system, so it's also possible one of them may be blocking the sound because

the game is not properly installed on your machine from a legitimate source (retail disc, Steam, or other on line game retailer).

or previous Windows install, ie- you didn't buy it)....

It seems you have enough money to purchase the game and install it properly then. Or if you purchased and installed the

game on another machine, reinstalling it may fix the audio issue.


"these Directx things are just some software type things "

That is correct. Here is a list of most the game related DirectX files you should find in your Windows.

- You will find all the DirectX files in the Windows\System 32 & sysWOW64 folders.

They are in alphabetical order and will start with d3dx9 - 24 > 43 . Then d3dx10 - 33 > 43 & finally d3dx11 - 42 > 43.

There should also be - d3d9, d3d10 and d3d11 (among others).
These come before the D3Dcompilers-33>43+47.

Audio - X3DAudio1_0 through 7 & Xaudio2_0 through 9

XInput & dinput are DX files used by controllers.

- Win10 currently has only two d3d 12 files. One in System 32,and SysWOW64 .
- There are a few more DX files, but I don't know what they are used for.

a. user say:

Your computer sounds like an attacker's playground.

When your computer is seriously messed up and you don't know where to begin to fix it - and you don't backup - you have two choices: 1) call in a professional; and 2) start over from the beginning, i.e., a clean install. For someone lacking in knowledge, calling in a professional is the wisest choice. To do a clean install the right way, avoiding mistakes and unwise choices along the way, you'll need some technical knowledge which you can certainly learn with enough research, but do all that after your computer is free of malware and working properly again.

To answer your questions:

1- There are dozens of system and user accounts built in to Windows, each with a specific set of permissions. That's because Microsoft follows the principal of least permissions, a PC industry standard whereby computer accounts have specifically defined purposes and receive only the permissions needed for their purpose.

2- Every time you go on the internet there are thousands of people all over the world who are trying to break in to your computer. You can't stop them. The most you can do is to make it so difficult and inconvenient to successfully attack your computer that they give up and look for an easier target.

The industry-standard system of user and system accounts is one of the many defenses that you have to protect yourself. Attackers possess exquisite knowledge of computers. If there were an account that allowed complete and unfettered access to every possible function of a computer, and I was an attacker, I would attack that account and you would never know it.
