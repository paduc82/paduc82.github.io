  1. Press Ctrl + Shift + Esc keys together to open Task Manager
  2. On the Startup Tab look for anything relating to this file (LogiDA)
  3. If you find something related, select it and click 'Disable'

  5. Look for anything relating to this file (LogiDA)

  7. Click Apply and Reboot

If you are still having this problem, this worked for me:

Open the registry editor.  If you don't know how, open search/Cortana and type regedit

Using the folders on the left hand side, navigate to the following folder:

Change the value of this key from 1 to 0.

After you reboot you should have your power plans back, and even be able to change advanced power options.

cmdrkeene say:

There's two sets of permissions.  Most likely one has been set but not the other, which can cause what you see.

  1. Right-click the folder you want to share, and go to properties.
  2. On the Sharing tab, click Advanced Sharing.  Make sure the "Share This Folder" checkmark is on, then go to properties and set the permissions.
  3. Click OK to save everything.
  4. Go back to the same properties again (same as step 1 above) and this time go to the Security tab.
  5. Set the permissions here also.
  6. Click OK to save these and give it another try.

As long as you set the same permissions on both step 2 and 5, it should be good to go.
