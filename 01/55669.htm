sathish s v say:

Hello Ronald,

 

In Windows 8 we can download the English language pack only through Language Control Panel.

 

I would suggest you to perform the following troubleshooting methods and check if you can download English Language packs.

 

Method 1:

 

I would suggest you to perform System File Check (SFC), and then check if this fixes the issue.

Follow the steps:

a.       Press “Windows Key + Q” to open Charms Bar.

b.      Type “cmd” without quotes in the search box.

c.       On the left pane, right click on the “cmd” option and select “Run as Administrator”.

d.      Type “sfc /scannow” without quotes and hit enter.

Note: It may ask for windows installation DVD to fix and to enable SFC to make more than minor repairs. Some files saved on your computer might get deleted when fixing corrupted files.

 

Method 2:

 

Step 1:

 

Remove these two “WUServer, WUStatusServer” registry keys from Windows registry.

 

Important: This section, method, or task contains steps that tell you how to modify the registry. However, serious problems might occur if you modify the registry incorrectly. Therefore, make sure that you follow these steps carefully. For added protection, back up the registry before you modify it. Then, you can restore the registry if a problem occurs. For more information about how to back up and restore the registry, click the following article:

 

Follow these steps to backup registry values:

a.       Press “Windows key + R” from the keyboard.

b.      Type “regedit” without quotes in the search box and hit Enter.

c.       Click “Arrow” next to “Computer” option in the left panel and make sure you have only “Computer” option listed.

d.      Right click on the “Computer” option and select “Export” option.

e.      Save it on the External Memory device.

 

Follow these steps to remove these “WUServer, WUStatusServer” keys.

a.       Press “Windows key + R” from the keyboard.

b.      Type “regedit” without quotes in the Run window and hit Enter.

c.       Navigate through these registry keys

HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\WindowsUpdate

d.      If you find “WUServer, WUStatusServer” these two keys, right click and select “Delete”.

e.      Close the Registry window.

 

Step 2:

 

Un-register and re-register Windows Update serve.

Follow these steps:

a.       Press “Windows key + Q” from the keyboard.

b.      Type “cmd” without quotes in the search box.

c.       You will find Command Prompt” option in the left panel.

d.      Right click on it and select “Run as Administrator.

e.      Type the command Net stop wuauserv and hit Enter.

f.        Once it is finished, type Net start wuauserv and hit Enter.

g.       Close the Command Prompt.

h.      Open Language control Panel and try to install the language packs.

 

I hope this helps you to fix the issue. If the issue persists or you have any other issues with Windows, please do post it in Microsoft Community. We would be happy to help you.
