Hi,

This might be due to file corruption. To try and fix it, please do the following.

  1. Hit Windows key + X and click "Command Prompt (Admin)".
  2. Type: sfc /scannow and hit Enter.

System File Checker will scan Windows system files for corruption and attempt to repair any found by replacing the files using the dllcache repository.

Occasionally though, the tool might fail in which case you'll get a message which reads: "Windows resource protection found corrupted files but was unable to fix some of them".

If you see that message, copy and then right click and paste the following command into the same command prompt:

DISM.exe/Online /Cleanup-image /Restorehealth

Unlike SFC which replaces files locally, DISM downloads original system files from Microsoft's servers to replace the corrupted versions. So please make sure you maintain your Internet connection while it runs.

There's no animation to watch in either case though I'm afraid and particularly with DISM, all you'll see most of the time is a plain black screen. But eventually, the figure 20% will appear followed about 15 minutes later by 100%. You'll also get a report of what was fixed.

If you see any error messages, please post those here.
