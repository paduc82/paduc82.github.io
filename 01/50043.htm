burrwalnut say:
I am trying to delete a program that is not listed on the remove/uninstall list in Control Panel.  How do I get all programs listed?


You say ‘delete’ a program. It’s possible that it hasn’t been installed, i.e. it is a standalone program. If it has been installed, it’s uninstall information is stored in a registry key.

Click the Windows Orb (Start), type regedit, press Enter and in the left pane navigate to HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion.

Expand the Uninstall key in the left pane and scroll down to find the program entry then right-click and delete it. To make sure you have the correct entry, check the name in the right pane. DisplayName contains the name that is (normally) listed in Programs and Features and UninstallString contains the name of the program that is used to do the uninstall.

If your program doesn’t have a registry entry, it has not been installed and the .exe can be deleted. It's unlikely but I suppose it could have been partially uninstalled.
