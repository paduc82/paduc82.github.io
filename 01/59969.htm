That error should only come up if the person dialing the Skype Number is restricted from reaching that number because the number is long distance for them and their line lacks a long distance provider. If the Skype Number is a local call for the person calling, a long distance provider would not be necessary. If the Skype Number is a long distance call for the person calling and they have a long distance provider, the call should go through. In either case they would have to pay the provider for the duration of the call if their existing service does not cover the cost of the local or long distance call. You would not incur any additional charges beyond the cost of the Skype Number itself.

One other possibility might be the lack of voicemail. Make sure voicemail is enabled in your account settings. If you are not online, callers should be routed to your voicemail. If no voicemail is enabled, this might generate various phone errors to people attempting to call.

If this is not what is occurring and no one is able to reach that number, I suggest you contact Skype Customer Service over live chat.
