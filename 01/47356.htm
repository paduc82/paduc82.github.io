Wireless Security - Microsoft Community

xb1r apollo say:

Wireless Security

Login Information

  • Gateway: 192.168.0.1
  • Username: admin
  • Password: admin

 

How To Find Wireless Passphrase (WPA/WEP)

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted, enter in "admin" for the user name and "admin" for the password.
  4. Select "Wireless" from the menu to the left.
  5. Select "Security" from the menu at the top.
  6. If the "Encryption" is showing WEP, the passphrase will be based on which key is selected next to "Default Key," and will be labeled as "Encryption Key".
  7. If the "Encryption" is showing WPA, the passphrase will be labeled as "Pre-shared Key".

 

Changing Wireless Security Type/ Disabling Wireless Security

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted, enter in "admin" for the user name and "admin" for the password.
  4. Select "Wireless" from the menu to the left.
  5. Select "Security" from the menu at the top.
  6. Select the desired security under "Encryption". (To turn off security, select "Disabled")
  7. Select Apply.

diana d say:

Hi Smiley_sr,

• Which of the following versions of VNC are you using? (RealVNC, UltraVNC, TightVNC)

Some versions of VNC and World of Warcraft are not compatible with 64 bit-Vista.
Refer to the link: http://www.microsoft.com/windows/compatibility/windows-vista/Search.aspx?type=Software&s=VNC&os=64-bit

http://www.microsoft.com/windows/compatibility/windows-vista/Search.aspx?type=Software&s=World%20of%20Warcraft%20I&os=64-bit

So you may try running those programs in the compatibility mode and check if World of Warcraft Iis freezing.
i. Click Start, and then type the name of the program in the Start Search box.
ii. In the Programs list, right-click the program name, and then click Properties.
iii. On the Compatibility tab, click to select the Run this program in compatibility mode for check box.
iv. In the list, select an operating system as Windows XP, and then click OK.

More information, check the article “How to troubleshoot program compatibility issues in Windows Vista” http://support.microsoft.com/kb/927386

If the above does not help, you may try running VNC in the user mode. Right click on “Configure VNC Server” menu item and select “Run as” for administrative privileges so that the VNC configuration tool can access to the relevant parts of the registry.
