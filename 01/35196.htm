Hi,

One possible reason why the Family & other users option is not showing may be due to lack of administrative privileges. We would recommend that you check your current account if it is set as an administrator by typing netplwiz on Cortana's search field, then press Enter.

If your current user account is set as an administrator, the next step is to make sure that you check for updates and install any available updates for Windows 10 to resolve this issue.

However, if your current user account is set as a standard user, you may need to enable and login using the built-in administrator account in Windows 10 and switch your user account into an administrator account to resolve this issue.

Kindly refer to the steps below:

  1. Type cmd on the search bar.
  2. Right-click on Command Prompt, then select Run as administrator. Note: If you are prompted to enter a password, enter the password of your current administrator account.
  3. Once the elevated command prompt launches, type this command: net user administrator /active:yes, then press the Enter key.
  4. You should see a message saying that the command completed successfully.
  5. Sign out of your current user profile, then sign in using the built-in administrator account that you've enabled. You would see it as one of the choices on the sign-in screen.
  6. Type netplwiz on the search bar.
  7. Select the user profile that you were using and click on Properties.
  8. Go to the Group membership tab and select Administrator.
  9. Click Ok to finish.

Let us know  how it goes.
