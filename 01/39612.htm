You can download the ISO i.e. Installation Media using the Media creation Tool provided by microsoft and burn it into a DVD/USB for later use.

Use this tutorial for this:

http://www.tenforums.com/tutorials/2376-usb-flash-drive-create-install-windows-10-a.html

First of all download the ISO and then use Method 3/2.

As Windows 10 uses Windows as a service model, you would get new features after every 6 months or so.So you can use the DVD created to install Windows 10, but you would have to spend 3 GB to download the newer version of Windows 10, if a newer version is available.
