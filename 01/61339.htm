    bill smithers say:

    Hi,

     

    If you upgrade your Motherboard to gain better performance with an OEM preinstalled Windows License, (HP Compaq, Dell, etc), you have created a new Computer and void your Windows OEM License.

     

    This explains it:

     

    http://www.microsoft.com/oem/en-gb/licensing/sblicensing/pages/licensing_faq.aspx#fbid=iePP5a9JaWJ

     

    Q. If my customer asks me to upgrade a PC with new hardware components, when is a new operating system needed? When would the PC be considered "new"?

    A. Generally, an end user can upgrade or replace all of the hardware components on a computer—except the motherboard—and still retain the licence for the original Microsoft OEM operating system software. If the motherboard is upgraded or replaced for reasons other than a defect, then a new computer has been created. Microsoft OEM operating system software cannot be transferred to the new computer, and the licence of new operating system software is required.

    If the motherboard is replaced because it is defective, you do not need to acquire a new operating system license for the PC as long as the replacement motherboard is the same make/model or the same manufacturer's replacement/equivalent, as defined by the manufacturer's warranty.

     

    You need a Windows Retail License to do what you want to do.

     

    Cheers.

    andre da costa say:

    OEM versions of Windows 7 are identical to Full License Retail versions except for the following:

    - OEM versions do not offer any free Microsoft direct support from Microsoft support personnel

    - OEM licenses are tied to the very first computer you install and activate it on

    - OEM versions allow all hardware upgrades except for an upgrade to a different model motherboard

    - OEM versions cannot be used to directly upgrade from an older Windows operating system

    lemp say:

    1) Is the "Open With" window above now a default pop-up window following boot that CAN'T be disabled without the Registry Editor?

    No, the "Open With" is not a default pop-up.  You're seeing it because something -- most likely malware -- attempted to configure your computer to start a process at boot time.  Windows, however, doesn't recognize the process name (i.e., the file name) as an executable program so it's asking you to specify what program should be used to open the file.

    No, you don't need to manually edit the Registry to deal with this problem.

    2) There's no program in Control Panel I can open which has a simple toggle to turn it on or off?

    No, there no program in Control Panel you can use to deal with this problem.

    Here's how you DO fix the problem.

    A.  The easy way (won't work unless you're very lucky)

    • Click the Start Orb, All Programs, and then scroll down until you find "Startup" and click on it.
    • If you see anything listed under "Startup" that you don't want starting when you log on, right-click and delete it.
    • If you don't know what the listed items are and are reluctant to delete something you don't understand (a very good policy to follow), list all of the items in your next Reply and someone here will attempt to advise you.

    B.  The slightly more difficult way (that will almost always work)

    Well, there are two channels when using an Xbox, game audio and chat audio. By connecting the headset to the controller, you already have chat audio channel and should be able to hear and talk over Xbox Live Chat. As for game audio, stereo RCA cables need to be connected to the headset transmitter. What connection type do you use for your TV? If HDMI, then you need a Audio Dongle in order to use TosLink or RCA. However, only RCA will work as the device was designed to use the Red & White RCA to Green 3.5mm Jack.
