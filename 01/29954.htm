Increasing ethernet connection speed - Microsoft Community

superseth say:

Increasing ethernet connection speed

My ethernet connection is at 100 mbps.  A friend told me it should be higher.  How do I increase it?
steve winograd say:

You might have to buy new networking equipment to increase the connection speed, and it probably wouldn't make any difference if you did.

A computer's Ethernet connection speed is determined by two devices:

  • the Ethernet adapter hardware in the computer.
  • the network switch, broadband router, or broadband modem that the computer's Ethernet adapter connects to.

A higher connection speed is only possible if both devices support it.

For most computers, 100 mbps (megabits/second) is the highest possible speed.  Some recent computers have Gigabit Ethernet adapters that are theoretically capable of running at 1000 mbps.  You can install a Gigabit Ethernet adapter in a computer that doesn't have one.

But the fastest available home Internet services run at about 50 mbps.  You can't get higher Internet speeds than that, even with the latest, fastest hardware.

The one place that Gigabit Ethernet would make a difference is in copying files between two computers that have it.  If you don't do a lot of file copying, it wouldn't be worth the trouble and expense to upgrade the network hardware.

a. user say:

Is the backup system built into Windows 10 completely useless?

To a significant degree - yes. Windows 10 does not have a supported method for making image backups, and the sole supported method for backing up individual files and folders - File History - has limited capability. You should use third-party software for both types of backup.

I can be candid about this because I'm not trying to earn cash payouts from Directly, unlike our Independent Advisors, so I'm not limited to writing whatever I think will persuade you to give my answer a good rating whether or not that's really the best advice for you.

Frankly, I'm not sure you should blame Microsoft for its lack of good backup utilities. Today there are many terrific image backup utilities written by organizations that specialize in this technology, which wasn't the case when Microsoft first added image backup to Windows. Perhaps Microsoft decided 'why are we still doing this?'

You could say the same about File History. It's no match for most any third-party file backup utility, but then I don't believe it was supposed to be. File History is great when you want to go back to an earlier version of a file, the way you can with Google Docs (for example,) and although you can use it to backup all your personal files, and it does work, it's convoluted. A third-party backup program makes backup much easier and has many more features.

For truly good backup and restore that you can count on, you really should be using third-party software. This is not a complaint against Microsoft and I suspect they would be happy to get backing up out of Windows altogether. The future for Microsoft is the cloud; you already see how they're trying to persuade Windows users to store their files in OneDrive.
