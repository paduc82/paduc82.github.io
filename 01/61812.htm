obsessive power say:

Hello,

The play and charge only charges the controller, it can not turn a wireless Xbox 360 controller into a wired controller.  To use a wireless controller, you will need a Wireless Gaming receiver.  However, Microsoft no longer makes these devices.  All the ones being sold on the likes of websites such as eBay and Amazon are third party.  They do work, but you may need to do the setup manually.  If you run into issues, bookmark this link.  The link is for Windows 7, however the same procedure should work for Windows 8 and Windows 8.1

The play and charge driver that was installed, just allows the controller to sync automatically with the receiver if you connect it up via the charging USB cable.  It doesn't do anything else.  Other than that, you could buy a wired controller.

There is no controller driver to download for Windows 8 because the driver is already installed as part of the Operating System.  This applies to both the Desktop version of Windows 8/8.1 and Windows 8/8.1RT for Windows Surface Tablets. 

If you have a wired Xbox 360 controller, just plug it in, it will work straight away.  For wireless controllers you still need a wireless gaming receiver.  And yes, the receiver does work with Windows 8RT.
