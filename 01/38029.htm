These are questions about  running Win7, 8 (10?) ReadyBoost using more than one device, and also how "parallel" in performance the devices need to be.

Many laptops have an ExpressCard slot that could be used with an adapter to run a flash card. However, there are also a couple of ExpressCard SSD's on the market. (ExpressCard has capability to run on either of two interfaces--USB 2.0 and PCI-E--which is MUCH faster.) This company makes an SSD (64 & 128 GB) for this slot that runs off the PCI-E, so it should be pretty fast:

http://www.wintecind.com/spec/FileMate/SSD/WF_ExpressCard_SSD.html

One generally cannot use this SSD as a boot drive, as most laptops will not boot off of the ExpressCard slot.  (Well, I suppose you could do it by using a boot loader on the internal hard drive.)  In either case, the C: drive would still be on the installed HDD in the laptop.  However, so long as the boot drive remains a standard HDD, there are some things I am wondering about:

1. Will Windows 7 allow the ExpressCard SSD to be used for the ReadyBoost cache? I know it will not allow it if the SSD is the OS drive.

2. Access times on an SSD are MUCH faster than an SDHC flash card, which otherwise would be the best thing to use for ReadyBoost. (SDHC is the fastest SD protocol that will run in the SD slot of most older laptops, or via an SD adapter for the ExpressCard slot.)  I'm thinking of putting the ReadyBoost cache on this "external" SSD.  I'm not planning on replacing the main HDD with an SSD on this older machine.

3. With the 64GB SSD, there is enough room to also put the Page file and perhaps some frequently used programs on the SSD. This should make the external SSD even more worthwhile. Could other Windows components be moved to this drive with benefit in this situation?

4. Since with Win 7, one can run two or more devices with ReadyBoost, I could instead use the SD slot AND the ExpressCard slot also with an SD adapter to run TWO SDHC cards for readyboost. (I understand that by running several cards, Win7 will use them like a RAID array, increasing the speed of transfers by using both cards intermittently--so potentially much faster.)

5. Alternatively, I could use the two slots for ONE SDHC card, and ONE ExpressCard SSD. In this case, the SSD is much faster (it is running on PCI-E),...so I am wondering if the discrepancy in speeds would be a problem by not running two identical Flash cards.

6.  Finally, here's one more alternative:  I could run ReadyBoost on an SDHC card in the SD slot, and use the SSD to put the Paging File on (if Win7 will permit it), and also perhaps put some frequently used programs and browser cache on the SSD.  But if I use the SSD as one of two ReadyBoost caches, I could still put the paging file on the SSD as well.

By the way, the laptop in question is maxed out with 4 GB of DDR2 RAM.

What do you think?  I realize these are complex questions. Anyone willing to take this on?
