a. user say:

There's a reason why a short time period is specified for rolling back, although 30 days is an arbitrary number: The longer you use Windows 10, the more changes you make to your computer, the harder it is to successfully roll back. If you're actively using the computer, the rollback could fail after 3 days, let alone 30.

The Windows rollback feature is best for people who want to have a look at Windows 10, to decide if they want to keep it. Like trying on clothes you bought on the internet. The longer you actually use Windows 10 - and the longer you wear those clothes - the harder it is to return successfully.

There is a way for you to test-drive a new operating system for longer than 30 days and then successfully go back. But let's solve your problem, first. Computers haven't shipped with re-install disks for years, but you can still go back.

Every computer that ships with Windows has to include a mechanism for re-installing all the software that was on the computer when it left the factory. In the good old days, that meant computers came with re-install disks. Now what was formerly on those disks is actually on the computer's hard drive in a hidden partition. Instructions for restoring the 'factory download' should have come in the box but if you can't find them, no worries, just contact the computer manufacturer's technical support and they'll walk you through it. Just remember to backup everything you want to keep, because restoring the factory download erases the hard disk.

This won't help you now, but in the future you should learn how to make a disk image backup. A disk image is a copy of your entire hard disk, compressed into a single file. When you restore an image, your computer goes back to the way it was when you made the backup. The only form of time travel that actually works. You can go back after 30 days, 60 days ... 60 months. And it takes only minutes to restore an image backup.

Here are the Ttickmode settings and the locations of the registry keys to allow for custom trickmode settings.

 

Registry key to modify Seekbar settings

 

Camping = "Hold down the trickmode key on the remote" which sends key repeats.

 

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Media Center\Settings\SeekBar]

"CampingFastSkipBackMs"=dword:000493e0      ; 300,000 ms = 5 minutes

"CampingFastSkipForwardMs"=dword:000493e0   ; 300,000 ms = 5 minutes

"CampingFastSkipTickMs"=dword:000001f4      ;     500 ms = 0.5 sec

"CampingSkipBackMs"=dword:00007530          ;  30,000 ms = 30 sec

"CampingSkipForwardMs"=dword:00007530       ;  30,000 ms = 30 sec

"CampingSettleDelayMs"=dword:00001388       ;   5,000 ms =  5 sec

"CampingKeyRepeats"=dword:00000007          ; 7 repeats

"ThumbnailDisable"=dword:00000001           ; disables thumbnails if set to true (1)

 

 

Reducing the number of key repeats will cause the seekbar to enter camping mode with less latency

when you mash-down on the Skip/Replay buttons

you‚Äôve once pressed-and-released the Skip/Replay buttons just once without holding it down).
