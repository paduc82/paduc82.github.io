ken blake say:

You can *not* do this.  XP is not upgradeable to 10.

You *may* be able to run Windows 10 on it, but if you can, you would have to clean install Windows 10, not upgrade to it.

Note that I said "may be able to."  It depends on two things:

1. Whether your hardware is compatible with Windows 10. With a computer as old as one running XP, it probably is not.

2. Whether any needed Windows 10 drivers are available for your computer.  With a computer as old as one running XP, they are probably not.

And one other point: points 1 and 2 above probably make doing this not possible, but even if it is possible, with such an old computer's hardware, its performance running XP would probably be so poor as to be unacceptable.

So given all the above, *if* you can do it, you would probably need to upgrade your hardware. Add the cost of buying and installing new hardware to the cost of buying a copy of Windows 10, and the total is probably more than the cost of buying a new computer with Windows 10 preinstalled. And besides a new computer being less expensive, it will undoubtedly be considerably better and last longer than your old one.

So it's an easy choice:  don't try to install Windows 10 on your computer; either stay with XP or buy a new Windows 10 computer.

Corrupted profile data associated with the game, an interruption with the game server, and software conflicts that occur when running a game on Windows 10 are possible reasons as to why you're encountering gameplay issues with Microsoft Solitaire Collection. The following troubleshooting methods can help in resolving this issue:

Solution 1: Run apps troubleshooter

This can help resolve issues with games and apps from the Windows Store. Click this link to proceed.

Solution 2: Check for updates

Keeping your game and the Windows 10 device up-to-date can help resolve common issues with Windows 10 games. Refer to the following links for the steps:

  • Check for updates for apps and games from Windows Store
  • How to get software and driver updates for Windows 10

Solution 3: Reset the game from the Settings menu

This process can help restore corrupted game data. Here are the steps:

  1. Click Start then Settings.
  2. Select System.
  3. Select Apps and features.
  4. Find the game then click Advance options.
  5. Click Reset.
  6. Click Reset again to confirm.
  7. Restart your device so that the changes can take effect. Launch the game and check if the same issue occurs.

Solution 4: Sign in to a different user account

This can help isolate if there's a corrupted user data causing this issue. Refer to this link for the steps on how to create a new user account on Windows 10. Try playing the game using the new account. If it works, kindly connect your Microsoft to the new user account then follow the steps below on how to transfer your files from the old user profile to the new one:

  1. Press the Windows + X keys.
  2. Select File Explorer.
  3. Choose This PC and double-click Local Disk (C:).
  4. Click Users.
  5. Copy all the folders inside the old user profile to the new user profile.
  6. Restart your device and sign in to the new user account.
  7. Check if you can see all the files you've copied. Your new user account should look like your old one. However, you'll need to change the desktop background.

You may also get in touch with the Microsoft Solitaire in-game support team for in-depth assistance regarding this issue. Click this link to submit a post.

For additional information regarding the game, kindly visit the Microsoft Solitaire FAQ page.

If you encounter the same issue after performing the aforementioned solutions, kindly visit this link and check if there are other solutions that you haven't tried yet.
