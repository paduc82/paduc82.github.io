As for the multiple display, the challenge there at most cases would be how will it be used exactly.

If it'll be used inside the class room as a projector, then it should post any problem as it'll just be needed to be configured to one computer in the room. If you'll have the display connected through the WiFi, then this will be treated no different by the network as another device running on WiFi.

Basically, in a large network such as a school, it would be best to have [1.] the computers connected through the local network (except for mobile devices), [2.] use your display for one main computer inside the classroom with a wired connections used. To make this two basic factors possible, I suggest that you have this requested to your school's I.T. This way, restrictions as well can be added. By doing so, it'll be easy for network administrators to add permissions for both students and teachers. It would also be best to consult your school I.T for the devices that would be need.
