david scammell say:
Hi RichardMcFarland,

As the current generation of Microsoft Surface RT tablets are based on Windows 8 RT, which is based on the ARM processor architecture, I highly doubt they will come out with a video editor for the RT version.

However, all existing Windows-based editors should work on the Surface Pro version when it is releases, as it is based on the x86 processor architecture, the same as current Windows 7 devices.

The reasoning is based on the processing power of the processor. ARM is built as a lightweight, low power consumption, processor. As such, using it for video editing would be like using a hand scissors to bring in a crop of wheat.

If you're looking for a full laptop replacement that also performs video editing, you'll need the Surface Pro.

I hope this helps!
