That sounds about right...  with a SATA drive and the XP Recovery Console, you will not end up in the C:\WINDOWS folder, you RC commands will not work, you will get drive not found errors, enumeration errors, etc.  Sound familiar?  Even if you boot into the RC using a genuine bootable XP installation CD, you are still going to have these problems with SATA drives.  The RC just does not come equipped to deal with SATA drives.

You could mess around in your BIOS and change the interface of your HDD to something other than SATA and that usually works, but unless you have configured your system in advance when it was working to allow access to floppies and USB devices, you will not be copying any files from any floppy or USB device - even if you can get the RC to behave.  There are too many ifs, maybes, and mights.  You need to eliminate the ifs, maybes and mights.

You should make a Hiren's Boot CD instead.  With no BIOS adjustments, it deals with SATA drives just fine, doesn't care about the Administrator password that you might need to get into the Recovery Console, comes already equipped to allow access to floppies, USB drives and has all the same tools you might use from the RC and many more.

There is a Fix NTLDR tool with Hirens' but I have never tried it and have no intention of trying it, but it might work (another "might").

From a working system or form my SkyDrive, you should get a copy of NTLDR and put it on a floppy or USB thumb drive make a bootable Hiren's CD and boot your system on that and then from the Windowsy looking desktop that you will recognize, open My Computer, copy the NTLDR from your preferred media and paste it into the C:\  folder of your HDD where it belongs and restart your system from your HDD.

Download the Hiren's Boot CD ZIP file from the bottom of the page here:

http://www.hirensbootcd.org/download/

Unzip the file to find the .ISO file that you need to burn to a CD using a program that is capable of burning an .ISO file to create a bootable CD.  Hiren's comes with a way to do that, but I have never tried it - I like ImgBurn instead.

The ZIP file is large, so the download will probably take a little while to complete but it is worth it.  Then unzip the download to extract the Hirens.BootCD.ISO file that will be used to create your new bootable CD.

Creating a bootable CD from a .ISO file is not the same as just copying the .ISO file to a blank CD.  You have to use software that understands how to burn a .ISO file to a CD to create a bootable CD.
