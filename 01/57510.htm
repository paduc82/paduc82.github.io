riskybiz ;-) say: OK. What I did to fix this issue was as follows;

1. Remove the hard drive, as this was trying to resume from hibernate at each boot and blocking access to the bios, preventing the recovery disk from loading.

2.  Boot the Windows 7 recovery disk.

3. Restart

4.  Can now access the bios. (F2 at startup on this machine).  Set bios to boot from optical drive i.e. Windows recovery disk.

5.  Shutdown.

6.  Refit the hard drive.

7.  Boot into the Windows recovery environment.

8.  Allow the recovery process to fix the startup issue it found.

9.  Reboot.  TA DA..... Windows desktop appears.

10.  Relax again.

Hope this is helpful to someone in the future.

32 vs 64 bit

Windows 7 64 bit comes in handy when you need to address at least 4 GBs or more of RAM. Windows 7 32 bit can utilize up to 3.2 GBs of RAM. Because the memory address space is much larger for 64 bit Windows, that means, you need twice as much memory than 32 bit Windows to accomplish some of the same task, but you are able to do so much more, you can have more applications open, do things like run an Antivirus scan in the background without it affecting your system performance. Windows 7 64 bit is more secure too, malicious code cannot easily infiltrate it, drivers are more reliable since they must be signed before they can work with 64 bit Windows 7.

As for compatibility, you will need 64 bit device drivers for any hardware devices you might have. Also, there is no 16 bit subsystem in Windows 7 64 bit, which means, your applications must be 32 bit only, not 16 bit installer or uninstallers. Also, if you decide to move to Windows 7 64 bit in the future, there is no upgrade path from 32 bit Windows, clean install only.

Wireless Security

Westell 9100

Login Information

Gateway - 192.168.1.1

Username - admin

Password - <User Created Password> or password1 or  The serial number on the back of the device

 

How To Find Wireless Passphrase (WPA/WEP)

  1. Open a web browser.
  2. Enter http://192.168.1.1 in the address bar. (Or click the link) Enter the username and password if the gateway asks. ( admin/password, admin/password1, admin/password on sticker, admin/serial number begining with CSJ).
  3. Click Wireless Settings.
  4. Verify wireless passphrase on the status page.

 

Changing Wireless Security Type/ Disabling Wireless Security

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open a web browser.
  3. Log into the router by entering the default gateway into the address bar of the internet browser. Example 192.168.1.1
  4. The router will then prompt you to enter the username and password.the default user name and password are: "admin" and "password"/"admin"/"password1".
  5. Select Wireless Settings from the tabs at the top.
  6. Select Basis Security Settings
  7. In the Turn WEP On section select off.
  8. Click Apply. (If no wireless security is desired stop here)
  9. Select Advanced Security from the menu on the left.
  10. In the Level 1 section select WPA.
  11. On the following screen set the Authentication method to Personal(Pre-Shared Key)
  12. Set the WPA Type to WPA Any.
  13. Enter the desired wireless passphrase into the WPA Shared Key space.
  14. Click the Apply button.
