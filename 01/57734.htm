Change Default DPI for scanning in Paint - Microsoft Community

thelema say:

Change Default DPI for scanning in Paint

How do I change the scanning defaults for Paint so that every scan is 300 dpi?

 

gcsrss say:

I'll give an answer. What the questioner does not understand is that MS Paint does not have a setting for the DPI. There is no default DPI setting in MS Paint. The setting is controlled by the scanner driver or the scanner itself. MS Paint simply communicates with the driver for the scanner, which then takes over to do the scan, then upon completion, it hands the scanned data back to MS Paint. Therefore, the settings the questioner claims to know how to change are not in MS Paint; they are in the scanner driver interface. That driver determines how it handles the settings after returning the scanned data to the calling/connecting program, in this case MS Paint. Most all scanner drivers revert right back to their default setting. In this user's case, it is 100 dpi, but for a different scanner, it could be 150 dpi (common with many scanners a few years back), and more recent scanners often default to 200 dpi. After each scan, you have to change from the default settings all over again, because once the scanner driver has turned the scan data back over to the calling program, it no longer is communicating with it. It doesn't matter whether you use MS Paint, Picasa, IrfanView, or any other program that connects to the TWAIN scanner. If you start out with a scanner that has a default setting of 300 dpi, you wouldn't have thought it was an issue to ask about. You can't be insistent that no one is answering your question if it is based on the wrong premise to begin with.

Some scanners allow control from the scanner itself. I use an HP combination (All-In-One) printer/scanner. I can set the default up on the printer/scanner control panel to be 300 dpi or whatever value I want; I often set it to 600 dpi. Then I can scan directly from the scanner/printer and always get 300dpi/600dpi scans. Depending on the default program to handle the saved file type determines which program will be executed when the image is sent to the computer. It could be MS Paint, or IrfanView, or any other. Very often, I send the scans to an SD card on the printer instead. When I am finished, I have an SD card with the images scanned at 300 dpi that I take to a different computer. Since the scanner/printer has been set to use 300 dpi as its default, MS Paint, IrfanView or any other program that initiates a scan will also get 300 dpi scans.

What the questioner needs is a scanner that allows its default to be set or one that defaults to 300 dpi to begin with. If you try a newer scanner, you may find that the settings is not returning to 100 dpi but instead to whatever setting the scanner has as its default.

Since most simple scanner drivers always revert back to the devices default setting, and in this case, that scanner setting is only 100 dpi, you can use software that is specifically designed to solve this program by having its own settings for resolution. One example is called "Scan it to." In its interface, you choose what resolution you want, then when it communicates with the scanner, it takes care of changing the setting each time a scan is requested.

Whether you use color or grayscale is a separate choice apart from the dpi setting. Some scanners can scan at a different resolution for grayscale than they do for color. Regardless, the problem and solution is the same.
