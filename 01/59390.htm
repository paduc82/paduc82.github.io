johnperno say:

Hello Shawn,

cmdrkeene say:

If you have another user account on your computer, try logging into it and see if it is there.  If it is, you can copy it to a temporary place (such as C:\Temp or C:\Users\Public\Documents) and then copy it into your Start menu in the "broken" account.

 

If you don't have another account, or it doesn't show there either, try making a new account.  You can delete it when you're done, but the new account should have this folder for you.

 

 

Tip:  To quickly open your Start menu as a folder so you can copy in folders or shortcuts, press +R to open the Run box, and run the command shell:programs

johnperno say:

Shawn,

 

Opened new account and accessory folder was there.  Followed your instructions on how to get it to my account with no success.  Need more detailed instruction, if you can.  Computer literate  Thank You John.

cmdrkeene say:

No problem.  Here's how.

 

  1. Login to the new account.  Press +R to open the Run box, and run the command shell:programs to show the Start Menu's programs folder.  Right-click on Accessories and choose "Copy".

  2. Choose Start / Documents.  Expand the Documents library, and right-click the Public Documents folder and choose Paste:


  3. Logout and into your 'broken' account.  Go to the same Public Documents folder.  Use the shell:programs command to open your Start menu again, and drag-drop the accessories folder into the Start Menu Programs folder to put it there.
