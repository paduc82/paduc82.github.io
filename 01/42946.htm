So, lets say you have successfully installed your device, here's what you do:

Right click on the Sound Icon at the bottom right corner of your screen.

Set the headset as default

Click on the Recording devices tab

Click on the Headset Microphone (You might start hearing sounds playing from your media player if it is in motion)

Click on set default

Select Properties

Click on the Listen Tab

Check the box beside 'Listen to this device'


You should be good to go now. Worked for me, might do so for you too. Cheers!

In Windows 8.1, Windows Store is set up to automatically download and install app updates. But here's the thing. Since Windows 8.1—like Windows Phone, by the way—doesn't have a notification center, there's also no indication that anything has changed. The next was to ensure that the correct static IP address pool had to be allocated to the VPN IP range that will be issued dynamically when users log in via VPN.
