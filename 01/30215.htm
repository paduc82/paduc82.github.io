considerate_guy say:

I'm curious what added features in Movie Maker 2012 you don't like that you want to downgrade to Windows Live Movie Maker 2011?


The only differences I can see are that the default format for saving is now MP4 rather than the proprietary Microsoft WMV format.  However, you still have the option to save in WMV, if you like.  All the transitions, effects and titles are the same as in WLMM 2011, with the addition of optional colored outlines around the text for improved readability.  You can also add a narration track, but if you don't use that feature, ignore it.  Finally, there is the ability to display the audio waveforms, but if you don't want them appearing, turn them off from the Project tab.  Do those things and Movie Maker 2012 is no different than WLMM 2011.


On the other hand, if you are having problems with Movie Maker 2012, please provide further details. Movie Maker 2012 is a little stricter in terms of the GPU driver requirements.  If you have an outdated driver for your graphics card, it would be prudent to upgrade to the latest.  If unsure of the make/model/driver date of your GPU, run dxdiag and go to the Display tab.  If the driver date is more than a few months old, go to the GPU manufacturer's site (e.g. Intel, NVIDIA or AMD) and get an updated driver.  See if Movie Maker 2012 then works for you.


One caveat if you do decide to forego Movie Maker 2012 and are able to revert to WLMM 2011: the project files that are created in Movie Maker 2012 will not work in WLMM 2011.  So if you have created a project file in the newer version, or even opened and saved a former project file from WLMM 2011, it will be upgraded to Movie Maker 2012 format.  You can't then read that file in the older version.
