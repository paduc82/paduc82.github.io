Login Information

 

  • Gateway: 192.168.1.1
  • Username: admin
  • Password: admin or a custom password was created during setup.

 

How to Find SSID (Network Name)

 

  1. Connect a computer to the gateway with an Ethernet cable.
  2. Open a web browser on a computer.
  3. Type the address 192.168.1.1 into the web address bar.
  4. When prompted for the username type in admin, use the password created during the setup process. If you did not setup one then the use the default password admin.
  5. Once logged in click on the Wireless tab on the top of the page.
  6. Click on the Basic Wireless Settings in the subcategory.
  7. Click on the Manual setup option.
  8. The Network Name (SSID) will be located on this page.

 

How To Find Wireless Passphrase (WPA/WEP)

 

  1. Connect a computer to the gateway with an Ethernet cable.
  2. Open a web browser on a computer.
  3. Type the address 192.168.1.1 into the web address bar.
  4. When prompted for the username type in admin, use the password created during the setup process. If you did not setup one then the use the default password admin.
  5. Once logged in click on the Wireless tab on the top of the page.
  6. Click on the Wireless Security in the subcategory.
  7. The wireless pass phrase will be shown on this page.

davem121 say:
Hi martinbarnes3,

In the back or bottom of your Router is a sticker with a UserName, Password and IP address (something like 192.002.005.001) printed on it

Note those down

Go to your PC and open a Web Browser

In the address type the IP address and hit Enter

That will bring up your routers configuration page

Somewhere on that page you will find settings

In the settings menu, you will be able to switch from 5GHz to 2.4GHz

Before you can change that, you will need to enter the UserName and Password you noted down from that sticker on the router

That's It!
