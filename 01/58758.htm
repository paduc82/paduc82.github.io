"Waiting for Background Programs to close" isn't really an error, it's a notification that asks you if you want to go ahead and "force" a shutdown. It also lists the tasks that are causing this condition so you'll know the cause.

The reason for this is could have an unsaved file open, Windows default background services, antivirus which may take time to close. It's really a nice feature because some applications stubbornly refuse to handle the exit they're supposed to take when Windows notifies them of a shutdown in progress.
