a. user say:

Looks like another Support Engineer who is engaged by Microsoft wild goose chase...

Last Known Good Configuration will not replace missing files...  that is not what it does, that is not what it is for and that is not how it works.

Implementing KB307545 will not replace missing files either and you do not describe symptoms consistent with a need to implement KB307545.

If the samlib.dll file is missing, you will see a message like this:

lsass.exe - Unable To Locate Component : This application has failed to start because SAMLIB.dll was not found. Re-installing the application may fix this problem.

If the samlib.dll file is corrupt, you will see a message like this:

lsass.exe - Bad Image
The application or DLL C:\WINDOWS\system32\SAMLIB.dll is not a valid Windows image.  Please check this against your installation diskette.

The lsass.exe is the Local Security Authentication Server and if it will not start, your system will not allow a logon, therefore, you will not be booting successfully past logging in into

After acknowledging the message by clicking OK, you will just have a black screen with a moveable mouse cursor.

The samlib file should never be missing, so if it is, it may have been quarantined (effectively deleted) by your unspecified antivirus/antimalware application.

There may be backup copies of the file on your system and you can replace the missing file from the XP Recovery Console.  If you do not have a genuine bootable XP installation CD from which you can get into the Recovery Console, you can create a bootable XP Recovery Console CD (no XP media required).

If the backup copies of the file have also disappeared, you will need to boot on something like a Hiren's BootCD.

Then using an unafflicted system, you can download a copy of samlib.dll from my SkyDrive, copy it to a floppy or USB thumb drive, carry that to the afflicted system and replace the missing file from the Mini XP mode of the Hiren's BootCD.

An Anytime Upgrade from Windows 7 Starter to any of the premium editions of Windows 7 such as Home Premium, Professional or Ultimate is equivalent to what you would get if you purchased a retail upgrade or full version license.

The benefits of a full version or upgrade version license is they are transferable to another computer. Also, they can be used to upgrade previous versions of Windows such as XP or Vista.

An Anytime Upgrade can only upgrade existing editions to a higher edition, example, Windows 7 Starter to Windows 7 Home Premium.

Windows 7 - Anytime Upgrade - Frequently Asked Questions:

http://windows.microsoft.com/en-us/windows7/Windows-Anytime-Upgrade-frequently-asked-questions

Windows 7 - Anytime Upgrade features:

http://windows.microsoft.com/en-us/windows7/products/features/windows-anytime-upgrade

If you are in Australia, Belgium, Canada, France, Germany, Italy, Japan, the Netherlands, Spain, Sweden, Switzerland, the UK, and the US you can use Anytime Upgrade. If not you will need to buy the version of Windows 7 you want.

http://windows.microsoft.com/en-us/windows7/products/windows-anytime-upgrade

How to do an Anytime Upgrade:

Click Start, type Anytime Upgrade, click the option to purchase a key. Wait while the Anytime Upgrade page is loaded, click the Windows 7 edition you want to Upgrade to.

Click Next, fill out your contact information (it is important you do this, since the product key will be emailed to you as a backup just in case you need to reinstall it) enter your credit card information, click Next to begin the upgrade, wait while the key is verified, accept the license agreement, click upgrade, wait while the software upgrades, (it might take 10 minutes or more depending on if updates are needed), your computer will automatically restart, after restarting, Windows 7 will notify you its updating the computer, the computer will restart one more time automatically and the upgrade will be completed, a windows will pop up notifying the upgrade was successful and your computer is ready to use, click close, you should be upgraded to Windows 7 edition you selected, your files, programs and settings preserved, proceed to activate Windows 7 edition you bought.
