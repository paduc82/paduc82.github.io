Hi ettenna,

The reason you are not able to view the PDF in IE is that unlike Edge which is able to show PDFs without any additional plugins, IE needs a plugin from other vendors like Adobe Reader, Foxit, etc., to be able to show PDFs.

Without these plugins in IE, you will only be able to save the PDF and won't be able to view it.

Clicking 'Open' in the 'Open' or 'Save' dialog thrown by IE when you don't have these plugins would launch your default PDF viewer, which would be Edge. This is probably why you see a Flash in the screen, since your default PDF viewer opens up now.

- When you initially tried printing the PDF from Edge, did you run into this issue with just one PDF file? Or weren't you able to print any PDF files?
