Close Devices and Printers.

B. Install the Driver
Go to the Printer manufacturer’s website and search for the latest Windows 10 drivers
Reinstall the Printer driver
Reboot the computer

leonard_su say:

Hi,

 

We apologize for the confusion, in order to remove the restriction of your child account you need to configure it on web using the parent account. To do this, please follow these steps:

 

1. Sign in to http://fss.live.com using the parent account.
2. Click Edit settings under the name of the child you want to remove the restriction.
3. Click Contact management, it is located on the left part of the web page.
4. Under Contact management for Child name tick the option button beside Child manages their own contact list.
5. Click the Save button to update the settings.

If  you want to remove the Family Safety completely on your child account, please follow these steps:

1. Sign in to http://fss.live.com using the parent account.
2. Click Edit settings under the name of the child you want to remove in Windows Live Family Safety.
3. Under Accounts for Child name click Remove link beside the e-mail address of the child account. It is located on the bottom right part of the page.
4. New message box will pop-up then click Remove ID to completely remove the relationship on the account.

 

Let us know the result afterwards.
