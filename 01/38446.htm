@BusyBuzz - I guess you are answering @JimNidiffer's problem # 2.

@JimNidiffer - in regards to problem #1, I think I may have a direct workaround (if not a solution) for the issue affecting your Gateway NV52 Laptop on Win 7:

A friend called me over for help with his HP Pavillion desktop computer, where he was getting exactly the same symptoms.

When I ran powerfg -requests and also powercfg -energy, I was consistently seeing the same "Legacy Kernel Caller" device reported as being the problematic device.

To bypass that issue, we ran this:

Powercfg -requestsoverride Driver "Legacy Kernel Caller" System

This disabled all power calls from the device, and provided a quick workaround, but we wanted to find out what the root cause was.

Since "Legacy Kernel Caller" was a bit generic, and didn't point me towards anything specific, I had a look at the devices he had attached to the system, and at any cards he'd added. After some trial and error of disconnecting and reconnecting devices and cards I found it was an old cheapo brand Wireless g expansion card. Since we couldn't find any new Win10 drivers for it, we just uninstalled its driver completely, and Windows detected it at the next start up and installed the default USB drivers, which worked fine.

From what I've seen, a large percentage of these power issues are down to incompatible device drivers left behind after an in-place upgrade. In many cases, they might be drivers that have survived several upgrades (e.g. XP to Win7 to Win 10)... 


Perhaps you've already resolved the issue yourself long ago, but I hope this will be helpful to someone, in any case :)

I faced the same issue trying to install CodeWriter app from Store. And the following worked for me.

Open this folder:

C:\Users\{username}\AppData\Local\Packages

Get-AppxPackage -allusers *WindowsStore* | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register "$($_.InstallLocation)\AppXManifest.xml"}

What apps you will be using?

- You use just the bare essential applications such as Office productivity suite, web browsing and email.

bossdweeb say:

What exact ACER Aspire One model ? They have used this name convention on numerous variations since 2008.

Generally BIOS is accessed by holding the Delete key while booting into Windows, but if you user manual says F2

it looks like they do things differently (as laptop mfg's sometimes do).

- I think you may be more successful posting on the ACER forums, but ...
Note that unless your machine has a dedicated graphics card it has no VRAM. Dedicated video RAM is only available on dedicated graphics cards. Integrated graphics processors use the system RAM.

If your machine does have a dedicated grahpics card you cannot increase the VRAM in any case. What you see is what you get.

A lot of laptops these days don't have an option in BIOS to increase the amount of system RAM that gets allocated to graphics.

The main reason they don't have an option to increase the amount of system RAM reserved for graphics processing is that even

if the graphics processor and cpu were able to actually process the extra data over the amount that is reserved/dedicated for graphics,

they can still access and use any available RAM that has not been 'dedicated' to graphics.
