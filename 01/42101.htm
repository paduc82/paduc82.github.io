hairyfool say:

Good question from Palcouk, many of the classic reason for having additional partitions on older O/S are not so significant with Windows 7

 

However...

 

Press the Windows Key (next to Alt key) and R to bring up the Run box. Replace anything in there with diskmgmt.msc  click OK.

 

After a few seconds a Window will open with a list of the drives in the top and the drives as a Bar Graph in the bottom. Your HDD will probably have 3 partitions as Mike has said, a System Reserved, the Boot Partition (C:) and Recovery.

 

Right click on the Boot partition in the Bar graph display and the menu should include Shrink Volume as an option so select that.

 

A message box will display for a short while as it analyses the disk then a dialogue will open with 4 boxes one of which is settable. The default will be the maximum you can shrink but as it is the O/S drive this would be too much. Type in the value you need for your additional partition, it is in MB so 102400 would be 100GB. You should find a link to the Help file entry for shrinking a volume, you have made no changes at this stage so click on that and have a read.

 

If you are happy click on Shrink and it should start the process. How long it takes will depend on how much space you are leaving in the C drive and how fragmented the disk is. Eventually you will end up with unallocated space on the disk in which you create your new drive. As a 4th partition it is a 2 stage process. 1st you create an Extended partition with the space then create logical drives in that partition.
