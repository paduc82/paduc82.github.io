a. user say:

As I understand it, there is a registry modification for this, changing the folder type to [Folder] instead of some shell filetype, but I cannot remember how, which is annoying. I had this fixed before I upgraded to Win10, and now I can't remember how I did it, nor can I find the page that details the registry hack.

EDIT: Aha! Found it in the registry on my own.

First, before editing the registry for even trivial things, it's a good idea to back up the registry to be safe.

Second, make sure you have administrative access, then run regedit.

Third, Go to HKCR\Directory. Find the Friendly Type Name key. Change the value there from @shell32.dll,-10152 to [Folder]. Click ok.

Fourth, Restart Explorer.

Fifth, Enjoy your folders grouping above the other file types.

Perform a hard reset:

Performing a hard reset erases all of the information that is stored in the temporary memory and forces Windows to do a complete scan of the system, including the battery.

Method 1: To perform a hard reset, follow the steps below.

  1. Unplug the AC adapter from the notebook PC and remove the PC from any port replicator or docking station.
  2. Remove the battery from the notebook battery compartment.
  3. Press and hold down the power button for about 15 seconds to drain any residual electrical charge from the capacitors.
  4. Once started, insert the battery and plug the AC adapter back into the notebook PC.
  5. Press the power button to turn on the notebook PC.
  6. Use the arrow keys to highlight Start Windows Normally, if it is not already highlighted, and then press the enter key.

Could you please help me with the below questions:

  • Which version of Windows you are using? (To check, type: winver in start menu and hit enter.)

  • Which program are you using to stream movies on Laptop?

Meanwhile, please follow the troubleshooting steps as mentioned below: 

Method 1:  Run Inbuilt troubleshooter.

  1. Click on the Settings icon from the start menu.

  2. Click on Update and Security > click on troubleshoot > Click on Hardware and devices on the left pane > Run the troubleshooter.

  3. This should fix the issue automatically or it will show the error message.

If this doesn’t fix the issue, go to Method 2.

Method 2: Set the default sound device.

  1. Press Windows and R key together on your Keyboard.
  2. Type Control and hit enter.
  3. Click on Hardware and sound > click on sound on the right pane.
  4. Choose recording tab and find the HDMI Output on the devices list.

  • If you can’t find it: Right click on an empty space and select, “Show disconnected devices” and “Show disabled devices”.

  •   Select “HDMI Output” and click on “Properties” and make sure that the HDMI Output is enabled.

 5.  Select the HDMI Output and click the button Set Default.  

                         

Method 3: Roll back Audio drivers.

  1. Press Windows and R key together on your Keyboard.
  2. Type devmgmt.msc and hit enter.
  3. Expand the Sound, Video and game controllers option from the list.
  4. Select the driver > right click on the driver and click on Properties.
  5. Click on Driver tab > Click on Update or Uninstall driver.

Uninstalling the driver will be automatically installed after the system is restarted. Check if the issue is fixed. For more information, check Update drivers in Windows 10.

Hope it helps.
