canadian tech say:

Cutting power to a Windows computer is a very bad thing to do.  It is a bit of a roulette game that CAN cause serious corruption of the system.  Your computer does not have an OFF button.  It has on ON button.  Windows should only be shut down by clicking the button offered by the system.  Windows does a lot of stuff during the shutdown and if it is not able to do those things, your system can be left with corrupted operating system files.  Consequently the next time it is turned on, it will not properly start.

Windows has a lot of great stuff built-in that helps it repair the damage but that does not always work.

There are incidents when the only way to shut down a system is to cut power, but they should be rare and with full knowledge that you are taking a risk.  In most cases, use of the Task Manager to shut down an errant program works just fine.

When your system is up and running, create a System Repair disk.  It is simple and fast.  You need a blank recordable CD or DVD.   Start globe, Maintenance and you will see it there.   If your system is not up and running, you can make it on another Windows system as long as its bitness (32 or 64) matches yours.  The System Repair disk is designed to be booted from.  It takes a while for it to load its own version of Windows 7, but will then offer you some repair options.  Sometimes people make the mistake of thinking the System Repair disk is not working.  That happens when the system has not in fact booted to that disk.  To make sure it does, you need to get to a start up options screen.  That usually happens when you press F12 key about once per second on startup.
