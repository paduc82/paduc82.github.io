mark bu say:

Hello Karen, 

Removing an account from the Windows 10 Mail app will not remove the account from Office Outlook, however it will remove the accounts from the system for other apps such as  People and Calendar or any other apps that use the email account.  If you remove the account and find that you are using another app that used it, you can easily add it back to the Mail app. 

In addition to removing the account, you could leave the account and disable notifications for the Mail app. This can be done from Start -> Settings -> System -> Notifications & actions, and then turn off notifications for Mail (listed under the heading "Get notifications from these senders".

If you do decide to remove the email account from the Mail app, and don't want to receive the notification to setup Mail, you can set Outlook as your default Mail app from Start -> Settings ->  Apps -> Default apps, and then clicking on the icon under the heading "Email" and then selecting Outlook.

a. user say:

Hi Dejan,

 

Thank you for posting this question on Microsoft Community Forums.

 

I see that you wish to login to Windows 8 without having to type the password, every time.

 

I would like to inform you that Microsoft security policies require the user to enter the password every time to login to the account.

 

We can try this workaround where the account will not lock, as long as the computer is not shut down.

 

Follow these steps to avoid the login screen after the computer wakes from sleep.

 

a.     On the desktop, press Windows key + X and select Power Options.

b.     Click on Require a password on wakeup.

c.      Click on Change settings that are currently unavailable.

d.     Click on Don’t require a password and click on Save changes.

 

Hope this method helps.

 

If you have any further questions regarding Windows operating systems, please contact us on this forum. We are happy to assist.
