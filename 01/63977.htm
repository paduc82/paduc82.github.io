Hi Guys,

So I’m assuming you’re quite frustrated by now that you can’t set your printer to default.

I’m sure the frustration isn’t because you can’t set it default, but because you can’t print PDF’s (because Adobe doesn’t list your new printer in the list) or Word/Excel doesn’t print. No worries. Keep Calm And Read Below!


The following is a lengthy FIX but it will work. This FIX is for Windows 7.

Warning: This involves changing you’re registry key therefore, under NO CIRCUMSTANCES delete anything or change anything that you don’t know.


So many of you may be confused about HKEY or HKCU and etc or how to get to HKEY\Software\Microsoft\Windows NT\CurrentVersion\Windows\ and etc.

So here is how it begins:-

Where the problem is

Step 1. Click “start” and type “regedit”. Right click on regedit and click “Run as Administrator”. Expand “HKEY_Current_User” then expand “Software”. Then expand “Microsoft” then expand “Windows NT” and then “CurrentVersion” and finally “Windows”.

Step 2. Look to your right. There is something called “Device”. Look at the 3rd column (Data). It probably shows the data for the old printer. This is the device name you need to change in order to fix the issue. Double click on device and here you can see we need the new “value data:” of your printer. If you already know the new value of your printer, then add it here. (DO NOT GUESS IT. NO, the name and model of your printer is not the value.)

So far, I’ve only showed you where the problem is.

How to Fix

Now we need to find out what the correct “value data:” is for your printer. The only way I was able to find mine was to make a new user account and give it administrative rights. For some reason, the new user account will have the correct registry keys (names, values, data) everything.

To do this, go to control panel and under “User Accounts and Safety” click “Add or remove User accounts”. Once created, go to the shutdown menu and “switch users”. Go to the new user. It will take some time to load.

Once loaded, go through the same steps as mentioned above under “Where the problem is”. Open the “device” and look at the new name. This new name should be your new printer. Now write this whole thing down. i.e. for my printer, the new name was “Brother MFC-J435W Printer,winspool,LPT1:”. Some printer’s names are “NE04” or something like that, which is why I told you not to guess it above.

Once you have this name, switch users again and come back to your original account.

Now again, go back to “regedit” and open “Device”. Put your new printer name here and press ok.

This should work.

If error occurs: FIXING “error writing to registry”

To Fix this error, do the following:-

Make sure you opened “regedit” as “Run as Administrator”. i.e. by right clicking it.

If you did, then within regedit, go back to the “Device” section. In the left column where it says “Windows”, right click on “windows” and select “Permissions..”. It should have the “special permissions” block checked. Uncheck this block and check “Full Control” “Allow” (This should automatically check Read as Allow).

Now try changing the name to the one you found from the other user.

And this will work!!! Close everything and try printing a PDF/Word file or whatever.

﻿Create system repair disc

  1.  Open Backup and Restore by clicking the Start button, clicking Control Panel, clicking System and Maintenance, and then clicking Backup and Restore.
  2.  In the left pane, click Create a system repair disc, and then follow the steps. If you're prompted for an administrator password or confirmation, type the password or provide confirmation.




Create A System Image In Windows 7

After you install and activate Windows 7 create a system image as a backup for your current computer in case you ever need to reinstall it. This will restore your computer to it's exact state when the image was made - that will save you the trouble of having to reinstall Windows 7 and everything currently installed again, and it takes much less time to do.

1.  Click the Start Menu and type "Back Up" then click on Back up your Files.  

2.  In the Control Panel window that appears, Click the blue Create a system image link.

3.  The Create a system image window should pop-up.  Select which hard disk, DVD, or network location you are going to save your system image to.  Click Next to continue.

4.  On the next page check both your System and System Reserved drives.  Once you have selected the drives Click Next to continue. Note: You can select as many drives as you want (if available), but the more you choose to include means the larger the image size will be.

5.  The next window is just a review, make sure everything is correct then Click Start backup to start the process.  This can take 10 minutes or more so patience is required.

6.  After the backup completes, you?ll be prompted with the option to create a system repair disc.  This is a good idea just in case your system has any problems in the future.  Click Yes.

7.  Choose which Disc Recording drive you?ll be using to burn the disc.  Insert a blank CD or DVD to the physical drive and then Click Create disc.
