andre da costa say:

If the original Windows 7 license is an OEM license that came with the previous computer, no.

So you will have to purchase a new full version Windows 10 license.

When I upgrade a preinstalled (OEM) or retail version of Windows 7 or Windows 8/8.1 license to Windows 10, does that license remain OEM or become a retail license?

If you upgrade from a OEM or retail version of Windows 7 or Windows 8/8.1 to the free Windows 10 upgrade this summer, the license is consumed into it. Because the free upgrade is derived from the base qualifying license, Windows 10 will carry that licensing too.

If you upgrade from a retail version, it carries the rights of a retail version.

If you upgrade from a OEM version, it carries the rights of a OEM version.

Full version (Retail):

- Includes transfer rights to another computer.

- Doesn't require a previous qualifying version of Windows.

- Expensive

Upgrade version (Retail):

- Includes transfer rights to another computer.

- require a previous qualifying version of Windows.

- Expensive, but cheaper than full version

OEM :

OEM versions of Windows are identical to Full License Retail versions except for the following:

- OEM versions do not offer any free Microsoft direct support from Microsoft support personnel

- OEM licenses are tied to the very first computer you install and activate it on

- OEM versions allow all hardware upgrades except for an upgrade to a different model motherboard

- OEM versions cannot be used to directly upgrade from an older Windows operating system

What happens if I change my motherboard?

As it pertains to the OEM license this will invalidate the Windows 10 upgrade license because it will no longer have a previous base qualifying license which is required for the free upgrade. You will then have to purchase a full retail Windows 10 license. If the base qualifying license (Windows 7 or Windows 8.1) was a full retail version, then yes, you can transfer it.

From the Windows 10 end user license agreement:

b. Stand-alone software. If you acquired the software as stand-alone software (and also if you upgraded from software you acquired as stand-alone software), you may transfer the software to another device that belongs to you. You may also transfer the software to a device owned by someone else if (i) you are the first licensed user of the software and (ii) the new user agrees to the terms of this agreement. You may use the backup copy we allow you to make or the media that the software came on to transfer the software. Every time you transfer the software to a new device, you must remove the software from the prior device. You may not transfer the software to share licenses between devices.

cbarnhorst say:

As Greg says, "retail" is not necessarily transferrable.  It says "retail" because it might be transferrable.  Greg gives the conditions under which it would be transferable, namely if the underlying license is transferrable.  Perhaps the term "standalone licenses" best describes a license that can be transferred.  

The Windows 10 freebie is an upgrade to the base license.  It is not itself a standalone license.  However, if the base license is standalone then the two licenses together are transferrable.  Your base license is not a standalone license so the Windows 10 upgrade is not either.  If you had upgraded a copy of Windows that was standalone (you bought it in a store) to Windows 10 then both could be transferred together.

Backgound:  In the past, there were two retail consumer licenses:  Full Package Product, and Upgrade.  

The "full" license was independent (standalone), i.e., did not require a previous version of Windows in order to install it.  The box had printed on it "For a new computer or a computer without an operating system."  

Upgrade licenses did require a previous version of Windows.  The Upgrade box was labelled "Upgrade for users of (list of operating systems)"

Windows 10 does not have retail Upgrade boxes.  Only FPP.  The retail upgrade license is no more.  (The retail prices of Windows 10 Home and Pro are equivalent to the Upgrade prices for previous versions of Windows, however, so Microsoft played nice with pricing.)

Why I can't transfer my OEM license:  OEM licenses are not tranferrable because of support.  In order for the OEM to to be able to install Windows on the computer you bought, the OEM paid a royalty to Microsoft that is a fraction of a copy's retail price.  To get such a low price from Microsoft, Microsoft requires the OEM to assume all support for your Windows.  But the OEM cannot be expected to support your copy of Windows on any other computer but the one he sold you.  HP is not about to support your copy of Windows on any computer of your choice, like a Dell, Gateway, etc., so the OEM is within his rights not to give you a transfer rights.

1. Right click on Windows Start > Device Manager > Click the topmost entry i.e. your device > Click Action in Menu > Click Scan for hardware changes
Now you should be able to see your USB drive

2, Right click on Windows Start > Device Manager > Expand Disk Drive > Locate the Drive > Right click on this > Uninstall > Reboot your computer
Now it should be detected

3. Use the Hardware & Devices Troubleshooter

Type Troubleshoot in Windows Start Search box > Click Troubleshoot > Click Hardware & Devices > Run the Troubleshooter

4. Whether the Drive is assigned a Drive Letter or not

Right click on Windows Start > Click Disk Management > If you see your Drive is listed there but there is no Drive Letter > Right Click it > Click Change Drive Letter and Paths > Add > Click Assign the following drive Letter > Click in the Drive Letter List Box > Choose an appropriate Drive Letter > OK > OK
