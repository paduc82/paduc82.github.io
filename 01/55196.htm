al jarvi say:
Attempting to connect new I-Pad to wireless router and cable modem but have lost the access codes


Do you have a Windows 7/Vista/XP wireless computer handy that is connected to your wireless network? You can easily get the network profile onto a flash drive for later use in case you need to recover a wireless client or add a new client.

http://theillustratednetwork.mvps.org/LAN/SoHoWirelessSecurity.html#Windows_Connect_Now

If the client can not use WCN, ie. like a MAC or Linux client for example, the wireless key is in a plain text file in the \Smrtntky\Wsetting.txt file on the flash drive. Since the iPad apparently does not include a USB port simply type the key into the iPad.

On a Windows 7 machine you can also, as an administrator, easily see the key on your wireless PC. Open Network and Sharing Center and click on Manage wireless networks at the left. Select the network, right click and select Properties. Click on the Security tab and check the Show characters checkbox.

http://cid-25ab668da65c8fbe.photos.live.com/self.aspx/Windows%20images/RecoverKey.png

Otherwise many wireless routers allow you to see the encryption key in plain text. Log into the router and go to the wireless security/encryption tab to view the key. If that is not available then redoing the key from scratch is your best bet. There should be no need to reset the router to the factory defaults unless you want to. Simply login to the router and reconfigure the encryption type, ie. WPA2 or WPA, if desired and a new key value. Configure your wireless clients with the new encryption type and key.
