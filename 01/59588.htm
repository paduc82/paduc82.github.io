    1.    Make sure that the USB cable is not connected to the product. If the USB cable is connected to the product, disconnect it.

    2.    From the Windows taskbar, click the Windows icon , and then click Devices and Printers . The Devices and Printers window opens.

    3.    Click Add a printer . The Add Printer window opens.

    4.    Click Add a local printer .

    5.    Click Use an existing port .

    6.    Click the Use an existing port drop-down list, select LPT1: (Printer Port) , and then click Next .

    7.    Under Manufacturer in the Install the printer driver window, click HP .

    8.    Under Printers , select the appropriate alternate driver for the product (listed in the table in the previous step), and then click Next .

     

    NOTE:

    If your HP product is available through Windows Update, click the Windows Update button to select additional print drivers. You must connect to the Internet before using Windows Update.

    9.    In the Printer name text box, type the product name and then click Next . The print driver starts the installation process.

     

    NOTE:

    The name of the alternate driver displays in the window as the default name. If you prefer, you can change the name of your product. This step is optional.

    10.  When the installation is complete, a message displays on the computer stating that the product was added successfully.

    11.  In the message window, click Finished .

    12.  Connect the USB cable to the product, and connect the USB cable to the computer. A Device driver software was not successfully installed window opens.

    13.  Click Close to close the window.

    Follow these steps to configure the installed driver.

    1.    In the Devices and Printers window, right-click the name of the product and then click Printer Properties . The Printer Properties window opens.

    2.    Click the Ports tab.

    3.    Click the drop down menu, and then change the port from LPT1 to USB or DOT4 .

     

    NOTE:

    If there is more than one USB/DOT4 port available, select the USB/DOT4 port with the highest port number. To see the full port name, widen the ports row in the Properties dialog box by placing the cursor over the vertical line between Port and Description , and then click and hold the mouse button. Move the mouse to the right until the port name is visible.

    4.    Click OK .
