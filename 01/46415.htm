Proceed as follows to get the drivers: http://pcsupport.about.com/od/driverssupport/ht/driverdlmfgr.htm.  Once you have the drivers, you can install them through Device Manager as follows:http://www.vistax64.com/tutorials/193584-device-manager-install-driver.html.

I think this will fix the problem and alllowt the microphone to be detedted.  If that stilll doesn't work, go to Start / Control Panel / Sounds and check the entries for the microphone to make sure it is enabled and setup correctly.

I hope this helps.  Keep me posted.

Good luck!
