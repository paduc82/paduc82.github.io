Hi there,


the Xbox 360 consoles (and some games) are regionally locked, which means that the console will only be able to run Discs from the region where it originates from.

If you receive a region error code then the Disc does not match the region from the console.


The only way to make it work is to get matching regions on console and disc. There is no way around it.

It is not necessary to update Windows 7.  None of the updates have anything to do with Windows 10 except the last one, the Get Windows 10 app that appears in the tray.  The user can bypass that and go directly to launching the upgrade from a dvd or usb.  Setup will find what it needs from Windows 7 to activate Windows 10 for free.  The user will be able to exercise the rollback option in Windows 7 regardless of the update status of the previous OS (in this case 7).

In point of fact an upgrade is not even required.  The user can perform a clean install of Windows 10.  The data needed for qualification for Windows 10 for tree is in the BIOS of his computer (the OEM marker) or if he has a retail Windows 7 product key he can enter that when asked for a product key by Windows 10 Setup.

There is no need for a rollback option in Windows 10 if the user simply makes a system backup of his Windows 7 system before installing Windows 10.  In fact it is more reliable to do so.
