Hi,

Some games may crash when you play them in full screen mode if the display settings are not set up correctly. To resolve your concern, you may perform the following solutions on your computer:

Solution 1. Set the display scaling to 100%.

  1. Press the Windows key on your keyboard and click the Settings icon.
  2. Go to System and select Display.
  3. Under Scale and layout, make sure that you change the scale under Change the size of text, apps, and other items to 100%.
  4. Click Apply to save the changes.

Solution 2. Change the Nvidia Control Panel settings.

  1. Open the Nvidia Control Panel.
  2. Go to Manage 3D settings and select Global settings.
  3. Change Preferred graphics processor from Auto-select to high Performance Nvidia processor.
  4. Save the changes and try running the games in full screen mode again.

Solution 3. Disable or turn off the Game bar.

  1. Press the Windows key on your keyboard and click the Settings icon.
  2. Go to Gaming and toggle off Record game clips, screenshots, and broadcast using Game bar.
  3. Uncheck Show Game bar when I play full screen games Microsoft has verified.

Keep us updated with the results after performing the suggested solutions above.
