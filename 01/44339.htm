a. user say:

Unfortunately, there are no shortcuts to upgrading an operating system. Before you begin to upgrade to a new operating system there are several tasks that you must perform:

1- Make sure that all your hardware is compatible with Windows 10. This includes less obvious hardware, like your motherboard and hard drives. You need to make a list of all your hardware and then determine whether each and every device is compatible with Windows 10 or can be made compatible with Windows 10 by updating the driver. Device Manager lists your computer's hardware. If your hardware is not compatible you are taking a risk by upgrading to Windows 10.

2- Make sure all your software is compatible with Windows 10. Again, make a list of all the software installed on your computer and find out whether each is compatible with Windows 10 or can be upgraded to a version that is compatible to Windows 10. You can find a list of the software on your computer at Control Panel > Programs and Features. You are taking a risk by trying to upgrade with incompatible software.

3- Make sure that your computer is free of problems. Upgrading never solves problems in your current configuration; they just migrate to the new operating system with unpredictable results. Your current configuration needs to be trouble-free and supported by Windows (e.g., no 'improvements' by so-called registry cleaners.')

4- Go on the internet and do some research on how to upgrade to Windows 10, including best practices to follow. Make a list of all the steps you will follow in their proper order.

If all this sounds like it's going to take a long time, trust me: It will. These are the little details that the advertising 'forgets' to mention. In return for your efforts you will have a trouble-free upgrade. You either spend the time before and have a successful upgrade, or after when you try to figure out what went wrong while pulling your hair out.

And always, always backup before you upgrade an operating system. (You should backup every day.) It's the ultimate defense against things going wrong.

    abdullah umar naseef say:

    is uninstall same as delete

    No, Unless it is a portable program.

    When you install a program, It copies the executable, dlls and other files required to run the program to the installation directory. It is something like "C:/Program Files/<Company>/<Program Name>/". And also adds some keys to windows registry.

    If you just delete *.exe file.You are leaving the dlls and other files. Even if you delete the entire folder. The registry keys will not be deleted. If any other program depends on it, You will get errors. So use add or remove programs to uninstall a program.

    EDIT:

    A portable program is a program that you don't need to install it to use it. It won't comes with a installer. It won't make changes to the system or registry. So deleting it is enough to remove it from your pc.

    Hope you got it.
