a. user say:

Hi Rishabh,

 

Follow the steps given below and check if it helps.

 

a)      Press Windows key + X, select Control panel.

b)      click " NETWORK and INTERNET"

c)      Click " NETWORK and SHARING CENTER" and Click "Change adapter settings"

d)      Make sure that your original connection through internet does share its connection to your Wireless Network Connection 2"

e)      Then go to "Sharing Tab" then click the box "Allow other network users to connect through this computer's internet connection"

f)       Then choose "Wireless Network Connection 2". Then Click "OK".

g)      It will prompt "Since this connection is currently active,some settings will not take effect until the next time you dial it" then just dial again.

 

 

Also try the steps given below.

 

a)      Press Windows key + X, select Command prompt (Admin)

b)      Then type the following:

netsh wlan set hostednetwork mode=allow ssid=VirtualRouter key=123456789

c)      Enter

d)      Then type :

netsh wlan start hostednetwork

e)      Then try connecting to internet.

 

 

Note: ssid is the name of your wifi so you can change the VirtualRouter in any name you want.

Key is your password so you can change the value 12345678 to any password you want.
