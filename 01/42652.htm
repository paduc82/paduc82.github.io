considerate_guy say:

A project file (.wlmp) is not a movie.  It just saves the state of your project so you can come back to it later.  It has a list of all the source files (photos, videos, music) but doesn't store a copy of these within the project.

 

Since the project just remembers the folder and the file names, if you have done any sort of clean-up since last using the project (such as renaming folders, renaming files, moving photos, deleting photos, etc.) then Movie Maker won't be able to find them anymore.

 

However, you can hover over a missing item (one showing the yellow triangle and exclamation point) and it will show you the location and file name it is looking for.  You can also double-click on one of these problem files and navigate to wherever it may now be (assuming you haven't completely deleted the file).  Once you point it to the file again, Movie Maker will again load the items into the storyboard and let you preview them in the project.

 

Remember, should keep all your source files in their original locations if you want to make further edits to your project.  Or if you are done, you can save it as a movie (File > Save movie) and it will create a complete video file (.wmv or .mp4) that you can then share and view elsewhere.
