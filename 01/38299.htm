But it's probably not where you want to go.


In Windows 7, the directory corresponding to the Windows XP folder C:\Documents and Settings\All Users is C:\Users\Public


In Windows 7, there is no actual folder named C:\Users\All Users. 


If you manually enter C:\Users\All Users in the Address Bar of Windows Explorer, you go to C:\ProgramData because when Windows 7 was installed, a "Symbolic Link" that causes that behavior was created for you.

You'll notice that there is one hidden "real" directory (Default).  You can see this in Windows Explorer if you set the option to view hidden files and folders.


"All Users" is a directory symbolic link and "Default User" is a junction.  Both of these objects are created by Windows 7 for backwards compatibility with applications programmed for Windows XP that expect to find things in directories with those names.  Essentially, both the "junction" and the "symbolic link" are pointers to another object on the hard drive, similar to the more familiar desktop shortcut. 


A junction (also called a soft link) differs from a hard link [the file system representation of a file by which more than one path references a single file in the same volume] in that the storage objects it references are separate directories, and a junction can link directories located on different local volumes on the same computer.  

A symbolic link is a file-system object that points to another file system object. The object being pointed to is called the target.
