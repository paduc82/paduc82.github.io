Hi,

As there are 14 updates to be done, it might have taken long time to install the updates on the computer, it also depends on the file size of the update that had to be downloaded on the computer.

Step 1: If it is still stuck on installing update page, then restart the computer and choose boot normally and check what happens.

If you get into Windows and not all of the updates were installed, then you can install the updates one at a time and see if one of them is having a problem.
Microsoft download Center: http://www.microsoft.com/downloads/en/default.aspx  
Enter the KB article number in the search field and download the same.

Step 2: If fails, try booting into Safe mode(http://windows.microsoft.com/en-US/windows7/Start-your-computer-in-safe-mode ) ,

Step 3: If starting in safe mode also fails, then do the following:

a.  Please insert the Windows 7 DVD and reboot the computer.
b.  When we are prompted to press a key to boot from the DVD, please press Enter or the space bar.
c. Click "Next" and click "Repair your computer".
d. Please highlight "Windows 7" and click "Next".
e. In the dialog box titled "System Recovery Options", click "System Restore"

Step 4: If that does not return you to a working configuration, try using the DISM command to see what update is in a problem state:

dism.exe /image:d:\ /get-packages
a.      When you run this command, you should see one or more packages listed in a Pending state. Check for known issues with the particular update or package. There may be a more direct workaround.
b.  If you are unable to find another solution, use the /cleanup-image switch with DISM to revert the pending actions:

dism.exe /image:d:\ /cleanup-image /revertpendingactions

c.  Once this is complete, try starting the PC into Windows.

Once the computer starts up normally, check which were the updates installed/failed and then try installing updates one by one and check.

I was having the same issue and I edit hundreds of photos a month for my company. I figured out that not all of the features that Microsoft want's to implement into Paint 3D are not fully tested and ready. I took a quick peek under "Image">"right click">properties>details tab. I noticed that the resolution of the images with the errors ,(I have dozens of input sources), was insanely high. I mean like well beyond my display resolution high. If you go under "Image">"right click">edit, you should see an option near the upper left (see attached images) called Resize. 

The resizing tool is quite simple to use. If you keep the "Maintain aspect ratio" box checked, the "Horizontal" and "Vertical" numbers (highlighted below) adjust together. These numbers are a scaling ratio by percentage. The default value is set to "100" for the original size. Lowering the numbers from "1-99" will shrink the photo down. If you increase the numbers "101-500," the photo will expand. 

Shrinking the photos down doesn't change their shape if you keep that "Maintain aspect ratio box" checked. It allows the "Horizontal" and "Vertical" numbers to adjust in sync. Once you think you have it down do a manageable size, just save the file. Use the "Save-As" feature if you want to keep an original file and the newly resized one. 

Take your new photo and attempt to edit it with Paint 3D and you should find that your text boxes no longer get cropped out. 

This worked perfectly for me, and I hope it works for anyone else who has this issue!!
