sharief k say:

Hi,

Sorry for the delay in response.

If you receive Windows Update error 8024000b, it means that Windows Update can't read a file (Update Manifest) that is required for installation.

To fix this problem, follow these steps:

  1. Press Windows key + R.

  2. Copy the following text, and then paste it into the Run text box:

    Cmd /c ren %systemroot%\System32\Spupdsvc.exe Spupdsvc.old

  3. Click OK.

  4. Run Windows Update again.

If the issue persists, reset Windows update components from the following link and check.

http://support.microsoft.com/kb/971058

 

Important: Backup the registry: http://support.microsoft.com/kb/322756/en-us
