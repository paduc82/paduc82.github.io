jack mvp say:

Hi

To be able to connect to a remote computer, the remote computer has to be ready to accept the connection.

That includes the setting in the OS to accept remote help, and opening the listening port of the Remote Desktop (RDT)

To port has to be open through the software firewall and forwarded trough the Router to the computer IP.

The default port of RDT is 3382.

Since a port can be forwarded only to one IP if you need to remote to few computers that are on the same network, you need to change the ports of other computers to be unique, and forward each port to the IP of the corresponded computer.

This page deals with other remote software, but there is description toward the end of the technique mentioned above.

http://www.ezlan.net/vnc.html

Changing the RDT port, http://support.microsoft.com/kb/306759
