Contana on Surface pro 4 - Microsoft Community

tillyuk say:

Contana on Surface pro 4

i have a new surface 4 pro. region language set to uk and English yet Contana is telling me it isn't supported in my region and won't work. anyone else with a similar issue?
greg [msft] say:

Hi tillyuk,

To use Cortana, your region, language, and speech settings have to be aligned. View Cortana's regions and languages article to see the corresponding language for each of those regions. In your case, since your regions is United Kingdom, your language needs to be English (United Kingdom), and the same with your speech setting.

Based on your description it sounds like you are set, but need to check the speech setting. Here are the steps to do that and some languages might require manually downloading a Speech language pack. 

  1.  On Start, go to Settings > Time & language > Region & language.
  2. Select the language and then Options.
  3. Under Speech, select Download. It may take a minute or two for the speech pack to load.
  4. Sign out and then sign back in for the new speech pack to be added to speech options.
  5. Go back to Time & language, select your new language then Set as Default.
  6. On  Settings > Time & language > Speech make sure the Speech language setting is aligned with the previous settings.
  7. Sign out and then sign back in for the new settings to take effect.

Hope this helps,

Greg

Windows 10 version update - Microsoft Community

david brown (dave) say:

Windows 10 version update

We run Windows 10 on thin clients with 4 mb RAM, 30 gb hardrives.  We have 8.5 GB free space.  Windows is trying to upgrade to a new version of Windows 10.  It says I need 8 gb free space, which I have.  I am getting an error message "Windows can not update reserved partition".   Any Ideas?  Thanks! 

[Moved from: Windows / Windows 10 / Install, upgrade, & activate]

a. user say:

Hi David, 

Based on the details provided, I recommend that you manage your hard drive space for better and smooth performance or better yet, install an additional hard drive or upgrade your PC. However, basing on the error message, the System Reserved Partition (SRP) is full. The System Reserved Partition (SRP) is a small part of your hard drive that stores all the boot information of your system or Windows. To troubleshoot this, we need to determine if your System Reserved Partition (SRP) is GUID Partition Table (GPT) or Master Boot Record (MBR) partition style. To do this, follow these steps: 

  1. Press Windows key + R, type diskmgmt.msc and hit Enter. 

  2. Right-click on the Disk (Disk 0) that contains the System Reserved Partition (SRP). 

  3. Select Properties. 

  4. Under Volumes tab, check the Partition Style if it is GUID Partition Table (GPT) or Master Boot Record (MBR). 

If it is with GUID Partition Table (GPT) partition, follow these steps: 

  1. Run Command Prompt as Administrator. 

  2. Enter the following commands: 

    1. Mountvol y: /s 

    2. Y: 

    3. Cd EFI\Microsoft\Boot\Fonts 

    4. Del 

    5. Y 

If it is with Master Boot Record (MBR) partition, follow these steps: 

  1. Press Windows key + R, type diskmgmt.msc and hit Enter. 

  2. Right-click on System Reserve and select Change Drive Letter and Paths. 

  3. Choose Add. 

  4. Enter Y as drive letter and click OK. 

  5. Run Command Prompt as Administrator. 

  6. Enter the following commands: 

    1. Y: 

    2. Cd Boot\Fonts. 

    3. Takeown \d y /r /f 

    4. Icacls Y:\* /save %systemdrive%\NTFSp.txt /c /t 

    5. Whoami (record the user name) 

    6. Icacls . /grant <username you got from whoiam>:F /t 

  7. Make sure that you still located in Y:\Boot\Fonts and enter this command: 

    1. Del *.* 

    2. Y

  2. Stop BITS, Cryptographic, MSI Installer and Windows Update Services. To do this, type the following command:    

    1. Net stop wuauserv    

    2. Net stop cryptSvc    

    3. Net stop bits    

    4. Net stop msiserver

    2. Net start cryptSvc    

    3. Net start bits    

    4. Net start msiserver
