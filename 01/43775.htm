alexanim-kwapong say:

windows 10 mobile hotspotpot

Please does the mobile hotspot offer any security?

venancio khu say:

Hi, 

 

Thank you for posting your concern on our Microsoft Community.

 

New portable hotspots in a device comes with some security turned on by default. This is a good thing as it ensures that the manufacturer at least provides some form of out-of-the-box security protection. Usually the manufacturer enables WPA-PSK encryption and places a sticker on the unit with the default SSID and network key that was set at the factory.

 

The main problem with most default portable hotspot security setups is that sometimes the default encryption strength may be set to either an outdated encryption standard, such as WEP, or it might not have the most secure form of encryption enabled, even though it is available as a configuration choice.

 

Some manufacturers opt not to enable the latest and strongest security standard in an attempt to balance security with compatibility for older devices that might not support the latest encryption standards.

You should enable WPA2 as the encryption type as it is currently (at the time this article was published) the most secure of the available choices

 

Change your default SSID or network name and create a strong password. Share this information to your trusted people only.

 

Hope this helps.

I imagine this issue *may have* been resolved by now, but just in case...

Keep in mind that the "Driver" is a piece of software used by Windows to communicate with the peripheral device (built-in or external).  Many Drivers can be updated by visiting the manufacturer's Support pages and seeking out model-specific "Drivers and downloads" for a specific model PC.

By contrast "Firmware" is a special file that may be listed "under" a different category when you visit the manufacturer's "Drivers and downloads" page, and it may be necessary to download a SPECIFIC "Utility" or "Diagnostic" tool in order to "apply" a firmware update to a device.  In other words, updating the firmware for a Device may be more difficult than simply downloading a single file and running it.
