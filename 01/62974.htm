  • Drive:\56eb37a890142\windowsinstaller\x86\windowsxp-kb942288-v3-x86.exe

  • windowsxp-kb942288-v3-x86.exe

56eb37a890142 random folder, and double-clicked on the following file

  • setup.exe

REVERTING EXTERNAL HARD DRIVE FROM XBOX BACK TO PC HELP!

I plugged an external drive in my Xbox one and now it won't recognize the drive when I plug it in my pc how do I fix that?

internet expert say:

The Xbox One formats the drive in a way that a typical PC may not be able to browse it normally for files. However, it should still be detected in your disk partition manager.


If you are using Windows: 

  1. Plug the drive in.
  2. Open "Disk Management"
  3. Find the external drive in the disk list.
  4. Reformat as intended.


If you are using macOS:

  1. Plug the drive in.
  2. Open "Disk Utility"
  3. Find the external drive in the disk list.
  4. Erase, and reformat as intended.
