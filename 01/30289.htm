a)      If Windows Media Player 12 is running, exit Windows Media Player 12.

 

b)      Locate the DRM folder on the hard disk, and then rename the DRM folder to DRMbackup.

Note By default, this folder is in the following location in Windows 7:

                   \ProgramData\Microsoft\Windows\DRM

c)       Visit the following Microsoft Web site:

http://drmlicense.one.microsoft.com/Indivsite/en/indivit.asp

d)      Follow the instructions on the Web site to update the security component of Windows Media Player 12. This procedure enables you to obtain new licenses for media content. You can also reset licenses for purchased content so that you can play that content on a computer.

Step 2:

Note

 On a Windows 7-based computer, you must run Internet Explorer as an administrator before you update the security component of Windows Media Player 12. To do this, follow these steps:

a)      Click Start, and then click All Programs.

 

b)      Right-click Internet Explorer, and then click Run as Administrator.
If you are prompted for an administrator password or for confirmation, type your password, or click Continue.

Note

 You must contact any third-party stores from which you purchased music to reset the licenses for that music. If you have copied CDs to protected Windows Media Audio files, those CDs must be copied again.
