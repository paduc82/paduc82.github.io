Problem:

A script-heavy website--in this case, specifically Facebook--may in some cases cause the Surface Pro 2 to completely shut down and restart.  This will happen without warning and typically occurs when the website in question is open and running in the background.  The problem happens specifically when using the 'desktop view' version of Internet Explorer, running in standard mode, and does not happen when using other brands of browser.

Solution:

Using the Modern UI version of Internet Explorer will avoid the problem.

Using another brand of browser for Facebook, such as Chrome or Firefox will avoid the problem.

Running Facebook in the desktop-view version of Internet Explorer can also work when using Compatibility View.  Telling Internet Explorer to run Facebook in Compatibility View solves the shutdown problem.

To enable Compatibility View for Facebook in the desktop view version of Internet Explorer:

1) Navigate to the website

2) Click the Settings icon in Internet Explorer (upper right-hand corner, looks like a cog) to bring up the Settings menu.  From the Settings menu, choose Compatibility View Settings.

3) The "Add this website" blank will contain the website address.  Click the Add button to add it to the list of sites that Internet Explorer will run in Compatibility View.

4) Click Close.  The site will now run in Compatibility View every time it is loaded.

It appears that you are getting an error Video TDR failure on the computer. This happens most often when playing graphic intensive games or when playing high quality videos. It can also happen with any process that uses a large amount of graphic resources, such as when editing or creating videos. TDR is the Timeout, Detection, and Recovery component in windows. When the graphics device gets too low on resources and stops responding, windows will attempt to recover by restarting the graphics device.

Method 1: I appreciate the troubleshooting steps that you have tried. I would suggest you to install the latest graphics driver from the manufacturer website and check it helps.

Strange Message on the Windows Defender Program

When I click on the 'View Details' link on the 'History' tab of Windows Defender, I receive the following message:  'The requested operation requires elevation.  Click Help for more information about this problem.'  What does this mean?

Thanks,

MarkVF

sarahkong say:
Means that your account is not an admin account.
To check that go to start - settings - accounts
See if your account is listed as an administrator.
If not, then go to start - settings - accounts - other people
See which account has administrator rights and login with that account.
Then logged in with that account go to start - settings - accounts - other people and click your account and give yourself administrator rights.
Should work after you log back in as yourself.
