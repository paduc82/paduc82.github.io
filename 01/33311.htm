You'll need to update an Internet Explorer record in the Windows Registry in order to run Microsoft Money 2003.

I'll assume that you're a novice (No Offence Intended) and guide you through the steps, one by one.

1. We'll need to open the registry editor, to do this you'll need to search for 'regedit', this can be achieved in the Windows 10 search.

2. Once open, you'll need to navigate your way through the registry, following the below path.

[Computer\HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Internet Explorer]

3. Click once on the Internet Explorer folder and you'll see a list of items on the right.

4. Change 'Version' so that the last digit is a zero.

For example, my current version number is [9.11.10240.16384], I simply changed it to [9.11.10240.0].

5. Once saved, you'll need to restart your PC.

From this point on you should be able to run Microsoft Money 2003 with no problems.
