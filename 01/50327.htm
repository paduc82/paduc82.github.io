Simply restart *Windows Explorer*!

To do that, open *Task Manager* and  look down the "Processes" list. Near the bottom of that list you will find "Windows Explorer".

a. user say:

Hello,
I'm Samuel, I'm a Microsoft expert. I will help you in this case today.

There are 3 different ways:

Disabling the Windows Update service.
Creating a group policy
Enabling the measured connections

Disable the Windows Update service

A method that is a little bit **** but quite effective is that of disabling Windows Update from the service panel. Type "services" in Cortana or you can also right-click on the Windows start button and type "services.msc" on the option to execute "Run". Both paths will take us to the Windows service panel.

Now we just have to locate the service with the name "Windows Update", double click on it, mark the start type as "Disabled" and save the changes.

Restart the computer and Windows will not download or install any updates again. If you want to reverse, you can always activate the service start type automatically again. A little elaborate but easy to carry out.

Disable updates by creating a group policy

Another option to get rid of automatic updates is to create a group policy with the GPE editor. Open the "Run" window again (you can press the "Windows" + R key to make it more comfortable) and type "gpeditor.msc".

Display the left menu until you reach "Computer Configuration -> Administrative Templates -> Windows Components -> Windows Update" and right click on "Edit" in "Configure Automatic Updates".

A new editing window will open in which you only have to check the option "Disabled" and apply the changes.

Activate the connections for measured use
Go ahead that this last safe-conduct only serves for wireless / WiFi connections. That is, it is mainly oriented to laptops. We only have to go to "Start -> Settings -> Network and Internet -> WiFi -> Advanced options" and activate "Measurement connections".


If you have any other questions, feel free to write again. Thank you!

I can imagine the inconvenience you are experiencing. I will certainly help you with this issue.

I would suggest you to Create installation media for Windows 8.1 and repair Windows.

Please follow the methods below with the new bootable DVD. 

Method 1: Booting to Recovery Environment with Windows 8 installation media.

Please follow the steps given below. Insert the Windows 8 installation DVD in the DVD ROM and restart your

computer.

Please reply us if the issue still persists.

  1. After you boot your computer using Windows 8 DVD or System Repair Disc, a black screen appears with gray text "Press any key to boot from CD or DVD". Press any key.
  2. Select the correct time and Keyboard type.
  3. Click Repair your computer in the lower left corner.
  4. Select Troubleshoot from Choose an option screen.
  5. Click Advanced options in Troubleshoot screen.
  6. Click Automatic Repair from Advanced Options screen.
  7. Next, choose the operating system by clicking on its button.
  8. Windows 8 will look for problems on hard drive(s) and verify that all required files are intact.                      

If the issue persists, follow method 2.

Method 2: If automatic repair doesn’t help to repair the startup files, you may try to repair them using these commands through Command Prompt under Advance Options.

Follow the steps:

  1. After you boot your computer using Windows 8 installation media, a black screen appears with gray text Press any key to boot from CD or DVD. Press any key.
  2. Select the correct time and Keyboard type.
  3. Click Repair your computer in the lower left corner.
  4. Select Troubleshoot from Choose an option screen.
  5. Click Advanced options in Troubleshoot screen.
  6. Click on command Prompt.
  7. Type these following commands and hit enter after each line of command:

         Bootrec /fixmbr

         Bootrec /fixboot

         Bootrec /scanos

         Bootrec /rebuildbcd

    8. Restart the computer and check the issue.
