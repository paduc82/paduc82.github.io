lemp say:

Windows 7 (which is no longer available from Microsoft but may be available from online vendors such as Amazon or Newegg) included a feature called XP Mode (for Windows 7 Pro or better).  This was intended to ease the transition from XP to Windows 7.  XP Mode is Windows XP Pro service pack 3 running in a "virtual machine" hosted by Windows 7.

When Windows 8 was introduced, Microsoft apparently decided that it really didn't want to help users stay with XP any longer and did not include an "XP Mode" capability for Windows 8. 

If you have a retail installation CD for Windows XP (including the license key), you could install that on a virtual machine hosted by Windows 8 ... but I suspect that your dad's old computer came with Windows XP pre-installed and so this option is not available to you (it is very difficult at this point to find legitimate copies of Windows XP for sale).

Theoretically, you could use that CD and product key -- if you have them -- to dual boot Windows 8 and Windows XP ... but only if there were Windows XP drivers for your new computer -- which there likely would not be.

If you were to get Windows 7 Pro and install XP Mode, you still might have a problem using the dongle.  See, e.g., http://answers.microsoft.com/en-us/windows/forum/windows_7-hardware/xp-mode-on-windows-7-is-not-detecting-keylok/23f84d19-b328-4d1f-9f80-fb13d43a5fbe and http://superuser.com/questions/128011/how-to-connect-a-usb-dongle-to-windows-xp-mode
