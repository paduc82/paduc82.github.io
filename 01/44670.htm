golddragon007 say:

Finally I solved this.

Current wallpaper is stored here:

c:\Users\{User profile name}\AppData\Roaming\Microsoft\Windows\Themes\TranscodedWallpaper

TranscodedWallpaper is a file. This place can be view easily with an user (some folders are hidden).

Current lock screen and login screen background stored here:

c:\ProgramData\Microsoft\Windows\SystemData\{random string}\ReadOnly\LockScreen_{random char}\{lockscreen file name}.jpg

You need to take over the ownership on this folder c:\ProgramData\Microsoft\Windows\SystemData\ then you need to add 'Everybody' to c:\ProgramData\Microsoft\Windows\SystemData\{random string}\ReadOnly\LockScreen_{random char}\ folder, then you will be able to create a hard link to the previous file with a command line.

Note windows will restore permissions if you lock your screen, and I don't know how it will behave if the background will be changed when the screen is locked.

mklink /H LockScreen.jpg %AppData%\Microsoft\Windows\Themes\TranscodedWallpaper

mklink /H LockScreen___1920_1080_notdimmed.jpg %AppData%\Microsoft\Windows\Themes\TranscodedWallpaper

Note the second may be changed, like if you have different screen resolution.

And now always my current wallpaper shows up on my lockscreen and my login screen. Sooo much fun.

What is the exact error message you get? Is that a question like "Do you own this game or app" and asking you to provide a disc or to sign in online?

Maybe the following explanation of the Xbox One's copy protection helps diagnose the issue better: 

Games need a digital license to be able to be started on your console. If it's a game you installed from disc, the disc has to be in the drive and be recognized by the console (make sure the game icon is visible on the home screen with a disc symbol in the corner).

If the console can't recognize or read the disc, it tries to check if maybe you own the game digitally, and tries to check it's own "Home Console" shared storage for shared digital Home Console licenses, and if it doesn't find those, it'll try to check if your account owns the game digitally with the license in the account's online license keyring. 

So, if you have a disc based game that doesn't seem to find it's license, make sure the disc is readable. Even if the disc appears clean, try the following:
Eject the disc from the drive, then hold the power button on the console for about 10 seconds to turn the console off completely. Turn the console back on, and make sure it shows a green startup animation. 

Take the disc to a sink, and apply a small drop of mild (dishwasher)soap on the silver side of the disc. Wet your fingers, and "massage" the soap around the silver side with spiraling motions. 

After the silver side is completely covered in soap, rinse the disc under running water. From now on, only hold the disc with your index finger in the center hole, and your thumb on the edge of the disc. 

Shake the disc to get the water off, and wipe the remaining drops with a soft, lint-free cloth. 

Take the disc back to the console, and check if the console detects the disc now, and if you're able to play. 

If you have bought the games digitally, make sure you have NOT unset the "Home Console" setting under the console settings under "Personalization". 

In order to turn the console into your "Home Console" again, the console HAS TO be connected online to be able to download your digitally purchased licenses again. After that, you can again use the console offline.

"How to Refresh Windows 8 and Windows 8.1"

  • The PC boots into Windows RE (Recovery Environment).
  • Windows RE scans the hard drive for your data, settings, and apps, and puts them aside (on the same drive).
  • Windows RE installs a fresh copy of Windows.
  • Windows RE restores the data, settings, and apps it has set aside into the newly installed copy of Windows.
  • The PC restarts into the newly installed copy of Windows.

"How to Reset Windows 8 and Windows 8.1"

"When you reset your PC, the drive will be completely reset, and all your apps and files will be removed from it. Windows 8 will be reinstalled, and you'll need to change any settings and reinstall any apps that you want to get back after resetting. If you know if any files that you don't want to lose, then you should back those files up before starting the reset"
