"audiodg.exe" is a part of Windows 10. System services like audio drivers run in different and isolated login session from the locally logged-in user in Windows 10. This ensures that content and plug-ins cannot be modified by other applications (e.g. by spyware).

Note: The audiodg.exe file is located in the folder C:\Windows\System32. In other cases, audiodg.exe is a virus, spyware, or worm!


Method 1: Disable the audio enhancements.

1. Select the Start button, type Sound in the search box, and then select Sound Control Panel from the list of results.
2. On the Playback tab, right-click (or press and hold) the Default Device, and then select Properties.
3. On the Enhancements tab, select the Disable all enhancements check box and try to play your audio device.
4. If that doesn't work, select Cancel and, on the Playback tab, select another default device (if you have one), select the Disable all enhancements check box, and try to play audio again. Do this for each default device.


Method 2: Reinstall Audio driver.

1 Select Start .
2 In the search box on the taskbar, type device manager.
3 Locate the Sound, video and game controller and expand it.
4 Right click on the audio driver icon and Select Uninstall.
5 Follow the onscreen instructions to uninstall the audio driver.
6 Restart your PC, and Windows will attempt to reinstall the driver.
