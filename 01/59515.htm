microsoft barbie say:

Hi Dan,

Have you been able to recover your data?  If not, I have another idea (I can't believe I didn't think of this earlier).  Although I stand by my previous statement that just running System Restore will not restore your user files; the act of creating a restore point creates "Previous Versions" of files and folders.  All you need to do is right-click on the folder your data was located in (or the folder above) and choose Properties.  Click on the Previous Versions tab.  Find the restore date you manually ran before the data disappeared and click Open. 

What you will see now are what's called Shadow Copies.   You can copy files or folders from this window to the location you would like to have them restored to. Make sure to close the window when you are done, so you don't confuse the Shadow Copies with your current files/folders.  If you look carefully at the address bar, you'll notice it includes the date and time of the restore point as well as the folder names.  

This would be a better resolution than the Restoration program above.  You'd only need to run that program if no restore points are available.

Barbara

where do you set up windows mail. not windows live microsoft windows e-mial


Windows Mail as in Vista? And you want to use this in Windows 7?

First, you need access to a Vista machine.

On the Win7 machine, find the "Windows Mail" folder in Program Files. Rename it to "OldWindows Mail" as a precaution.

Using a Flash or CD, copy the "Windows Mail" folder in its entirety from the Vista machine.

Go back to the Win7 machine and copy the "Windows Mail" from the Flash or CD to a location in Program Files.

Open it and right click on the WinMail.exe file and send a shortcut to the Desktop. Done.

A few caveats: EML and NWS files will not open in WinMail, but rather another e-mail client that you will need to have. WinMail also cannot be made the default mail or news client.
