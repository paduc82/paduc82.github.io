  1. Start Windows Live Mail.

  2. On the lower left of the window, click the Contacts button.

  1. Select Import.

  2. Select .csv from drop down menu.

  3. Click Browse and select the location of the .csv file.

  4. Click Next and select the check boxes next to the fields that you want to import.

  5. Click Finish.


To import a .VCF file


  1. Select Import.

  2. Select .vcf from drop down menu.

  3. Click Browse and select location of .vcf file.
