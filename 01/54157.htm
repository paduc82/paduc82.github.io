To allow a device from waking your computer:

1. Open Device Manager by clicking the Start, clicking Control Panel, clicking System and Security, and then, under System, clicking Device Manager.

trekdozer say: Verify that you do not have 32 bit Windows installed - to use over 4GB of RAM you need 64 bit Windows. If you have 64 bit Windows 7 you need to turn on memory remapping in the BIOS. Also make sure your motherboard supports the amount of RAM you have installed.


The usable memory may be less than the installed memory on Windows 7-based computers:

http://support.microsoft.com/kb/978610





Memory Remapping For Over 4GB Of RAM

If you have 64 bit Windows 7 you need to turn on memory remapping in the BIOS.


BIOS - How to Enter the BIOS

Every computer is different, when you first turn on the computer it should tell you what button to push to enter the BIOS. It's usually one of the following and you must push it within the first 5 seconds: F1  F2  F10  F12 ESC  DEL

Once in the BIOS you have to find the menu with memory remapping, it's different on each computer.


Enable the memory remapping feature

Check the BIOS settings to see whether the memory remapping feature is enabled. Memory remapping gives Windows access to more memory. You can enable the memory remapping feature in the BIOS by booting to the system setup. See the User's Guide for your computer for instructions on how to boot to system setup on your computer. The name for the memory remapping feature may be different for different hardware vendors. This can be listed as memory remapping, memory extension, or something similar. Be aware that your computer may not support the memory remapping feature.
