eileen gal say:

Hi,

This error message occurs when the boot firmware on the computer generates lots of memory fragmentation. This can also be caused by an issue with Boot Configuration Data (BCD).

To help you with this, we suggest that you rebuild the BCD. To do this, please follow the steps below:

  1. Using the bootable USB flash that you already have, plug it in or insert it on your device.

  2. Select Command Prompt and type the following commands, then press Enter:

    Bootrec / fixmbr
    Bootrec / fixboot
    Bootrec /rebuildbcd

    Note: Please enter each command in order.

  3. Close the Command Prompt window, then restart your device.

greg carmack - windows mvp 2010-2020 say:
hi Skip. I'm Greg, an installation specialist and 9 year Windows MVP, here to help you.

You get those arrows when you have more icons on the space you've allowed for taskbar icons than can fit. The arrows then provide a second level of icons to choose from.

To get rid of it, right click an empty area of task bar to Unlock Task bar, find the divider between task bar icons and System tray items which will appear as a ultra-thin double line, put your cursor on the line and drag it back all the way to the right to again divide the System tray over on the right. Now it should relax the taskbar icons room so they all fit without a "loft" Then you can lock the task bar again.

More about customizing task bar here: https://www.thurrott.com/windows/windows-10/412...
