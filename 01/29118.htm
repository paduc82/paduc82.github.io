    a. user say:

    That's two separate problems.  We;ll try to address each.

    For the USB, is there any reason you can't format it (is there data on it you don't want to lose)?  Perhaps you can copy the data to a temporary folder on your PC and just let the system format the USB even though it's already formatted.  Hopefully it will be a one time thing and not prompt you to format every time you stick it in the machine (then we'll need to check your system files for corruption or check the USB for corruption).

    For the music files, go to C:\Users\<username>\Music and see what's there.  Can you double-click on a song and have it play?  If so, select the songs you want to copy and right click and click copy.  Then move to the USB drive and right click and select paste.  If it won't copy the files but will play them then I'll be very confused because it obviously is able to see them.  Let me know.

    If you still can't copy and paste the music, then let's check some of your system files:

    Go to Start / All Programs / Accessories / Command prompt and right click on command prompt and click run as Administrator.

    Type sfc /scannow and enter and let it run.  It will scan and try to fix some of your system files.  Hopefully it will complete with no corruption it could not repair (if there is such corruption post back here or try to analyze it to find the problem file(s) using     http://support.microsoft.com/kb/928228.

    While in Command Prompt, type chkdsk /f /r and enter and let it run.  It will scan and try to fix any corruption or bad sectors on your hard drive and mostly remove that as a cause.

    If that doesn't work, then please post any repeating error messages from the Event Viewer concerning copy or paste functions (Start / Control Panel / Administrative Tools / Event Viewer).  Here's how to use Event Viewer: http://www.bleepingcomputer.com/forums/topic40108.html.

    If that doesn't work (or if SFC found corrupt files), we'll need to do a system repair/upgrade using the genuine Windows Vista Installation Disk (one you own or one you can borrow from ANYONE).  Here's the procedure:     http://www.vistax64.com/tutorials/88236-repair-install-vista.html along with the upgrade from an earlier version of Windows section of the following: http://support.microsoft.com/kb/918884.  Although this will not affect your data, settings, or programs, you should still backup your data before starting just to be on the safe side. If the other procedures don't work, THIS one is almost certain to work.  You may have a lot of updates to re-install (including any service packs you had to remove).  If the version on the system came with SP1 or SP2 pre-installed and the disk is an earlier version, then you'll need to make a slipstream disk as follows: http://www.vistax64.com/tutorials/151606-vista-sp1-slipstream-installation-dvd.html.  

    Hopefully one of these procedures will resolve your problem.  If not, post back and we'll try something else.  Post back in any case so we'll know how it turned out.

    Good luck!
