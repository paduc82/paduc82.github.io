Hi,

 

If you have failed to connect the hotspot connectivity using connectify, you may have to contact connectify support since creating a Wi-Fi hotspot feature is not available in Windows 8.

 

As a work around, you may follow the steps provided.

 

If you set up wireless ad-hoc connection on your windows computer, you’ll be able to connect other devices to your computer’s Wi-Fi network and thus share the internet connection of your computer with other devices without the need of external router or access point, etc.

 

a. Press win key + x and Choose command prompt (Admin) option from the menu displayed.

 

b. Now in the command prompt window, type netsh wlan set hostednetwork ssid=nameofnetwork key=networkpassword

 

c. Instead of typing nameofnetwork in the above step, type any name that you want to give to your network. Instead of typingnetworkpassword in step above, type secure hard-to-guess password that you want for your network so that anyone trying to connect to your network must know the password.

 

d. Now type netsh wlan start hostednetwork and press enter. You windows 8 laptop/tablet/computer will now be configured as Wi-Fi Hotspot. Any other device could now connect to your computer’s Wi-Fi using the password that you set in the step above. So, Turn Wi-Fi on other devices like your smartphone or another laptop running any version of windows be it windows 7, vista or windows xp. On the second device, your WiFi network name will be shown, connect to it. It’ll ask for the password. Enter the same password you created in second step. Done. Your second device will be connected to your computer’s WiFi.

 

e. If you want to remove or stop the newly created ad-hoc connection, typenetsh wlan stop hostednetwork in command prompt (admin).
