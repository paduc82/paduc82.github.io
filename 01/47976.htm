canadian tech say:

This involves more than the power settings.

Ensure that you have it set to let the screen go to sleep before it is set to go into sleep or hibernation mode.

  1. Click the Start button (bottom left)
  2. type screensaver into the text box that pops up
  3. Click on change screen saver
  4. Ensure that you have selected a screen saver and set the wait time to some time less than the time for your screen to go to sleep (step 7 below)
  5. Click on Change power settings
  6. Click on Change Plan settings
  7. Turn off the display should be set at a time greater than you set in the screensaver
  8. Put the computer to sleep should be set at that greater time or more.
