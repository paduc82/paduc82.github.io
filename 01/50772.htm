    hairyfool say:

    Right click anywhere on the empty desktop (not on an icon) and select screen resolution.

     

    Below the box with the image of the screen the middle button should be labelled Resolution with a value in the button  X * Y. Click the drop arrow and is the slider at the top? Does the highest option have "recommended" on it.

     

    If the second answer is no then the graphics card probably is not installed correctly and needs a driver update as it has not interrogated the screen to get its capabilities properly.

     

    If it has the recommended option does the resolution match what you believe the screen is capable of. Slide to that setting and select apply and see if the black bars go and the image fills the screen.

     

    A LCD screen has a fixed number of pixels wide and high and has a specific ratio. The standard ratios are 4:3, 16:9 and occasionally 16:10. With black bars either side it is most likely the screen is a 16:9 ratio but the graphics output is 4:3. Some screens stretch to make it fit others use the black bars.

     

    Any setting other that the correct ones for the screen  will produce a soft image with a loss of sharpness.
