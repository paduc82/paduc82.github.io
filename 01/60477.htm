      2. If any of the devices listed under the Bluetooth group have a "yellow bang" (small yellow triangle with exclamation point), there is a problem installing that device.
      3. If there was a problem with any device, right-click on that Device to open the Device Properties window, and look at the Device status on the "General" tab.  Reply back here with that status.
    3. If there is not a node for Bluetooth devices, is there a node for Other devices?
      1. If so, it's likely any devices listed under the "Other Devices" node will have a "yellow bang" (small yellow triangle with exclamation point).  When devices appear here with the yellow bang, it's typically because no driver could be found for that device.
      2. Right-click on the device(s) with the yellow bang, choose properties to open the Device Properties window, and look at the Device status on the "General" tab.

      4. If "Update Driver" fails, in the Device Properties window, select the "Details" tab, as we need to find out what the Hardware IDs for this device are.
