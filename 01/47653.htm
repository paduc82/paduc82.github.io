Running Disk Defragmenter from the Command Line for More Control - Microsoft Community

aakkam22 say:

Running Disk Defragmenter from the Command Line for More Control

Technical Level : Intermediate

Summary

In addition to using using the Disk Defragmenter UI, some users may prefer to use the Command Line to defrag with more granular controls.


Details

The Syntax of the Command

Running the operation from the Command Line uses the same base command each time: 

defrag.exe

You can then specify options for the command which are added a space away from the base command.

  • <volume> Specify which drive you want to defrag using the drive letter.
  • /C Defrags all local volumes on the computer
  • /E Defrags all local volumes on the computer except those specified
  • /A Displays defragmentation assessment statistics without defragmentiting it
  • /X Performs an operation called 'free space consolidation'. You can run this when you need to shrink a volume, such as in Disk Management. It also can help reduce future fragmentation of files.
  • /T Tracks an operation that's currently running on the specified volume
  • /H Runs the defrag at normal priority instead of the default low priority. You can use this if you will leave the PC while it's defragmenting, or you want the operation to receive priority over other currently running tasks.
  • /M Defrags mutliple volumes at the same time. You can only use this option if your hardware supports accessing multiple disks simultaneously.
  • /U Provides a visual indicator of the progress on-screen
  • /V Verbose mode. Useful if you'd like to see an extended report for the volume.

Here is an example of the syntax for novice users:

defrag.exe C: /U /V /X

In this example, volume C: is being defragmented, and the user added 3 options. Note the space after each option, and the space after defrag.exe when specifying the drive letter.
