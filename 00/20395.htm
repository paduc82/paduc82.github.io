suzanne s. barnhill say:

It's incredible that this old bug should still be surfacing, but it certainly sounds like it. Try this:

  1. On the Page Layout tab, in the Page Setup group, click Columns and choose More Columns...
  2. In the Columns dialog, click the preset picture for One (even if 1 column is already selected).
  3. If possible, set "Apply to" to "Whole document." If you can't, you'll need to repeat it for every section (it may help if you select the entire document with Ctrl+A before going to the Columns dialog).
  4. Click OK.

If this solves the problem in a given document, and you are experiencing the problem in all new documents, you will need to use the same solution in the Normal template (Normal.dotm), which you will find by selecting Templates in the navigation pane of Word's Open dialog.
