bob jones aka: cybertaz say:
The first suggestion I'd make - especially since this isn't your default printer - is to check the mfr's site for an updated driver for that printer. In most of the few similar reports appearing here driver updates were at the root of the problem.

Also, make sure Office is fully updated (14.1.4), repair disk permissions & restart the Mac.

As for the default font, what specific "motions" are you going through? Changing the default font (Format> Font) used by Word has no effect on the fonts that were used in existing documents. That only determines what font will be used in new documents created from that point on. It also causes that font to be used as the default for normal text content as well as a variety of other aspects of the documents.

You can most easily change the default font used for only envelopes by going to Tools> Envelopes. There you can set the preferred font for both the Delivery & the Return addresses. Doing so modifies the appropriate Styles on which the formatting is based.

If you continue to have a problem it may be due to font issues on your Mac. Follow the procedure here for cleaning things up: Font Weeding.
