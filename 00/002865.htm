bernard liengme say:

mike middleton say:

If you want a static list, you could use the Random Number Generation feature of the Data Analysis tools.

If you want a dynamic value in a worksheet cell, use =NORMINV(RAND(),mean,stdev). That is, use

=NORMINV(RAND(),0,1)

and then press F9 to recalculate.
