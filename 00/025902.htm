Go to your People page.
Locate and click the group name in your contact list.
Click Edit in the command bar on top of the page.

To add a member, type the contact's name in the Add member box and click the desired contact from the drop down option.

Dim fso As Object 'Scripting.FileSystemObject
Dim SignaturePath As String

Private Sub CommandButton1_Click()
  'Ok-Button
  Dim mItem As MailItem
  Dim TS As Object 'Scripting.TextStream
 
  'Create a mail
  Set mItem = CreateItem(olMailItem)
  'Signature choosen?
  With Me.ListBox1
    If .ListIndex > -1 Then
      'Add it
      Set TS = fso.OpenTextFile(SignaturePath & .List(.ListIndex) & ".htm")
      mItem.HTMLBody = "<br>" & TS.ReadAll
      TS.Close
    End If
  End With
  'Close ourself
  Unload Me
  'Show the mail
  mItem.Display
End Sub

Private Sub CommandButton2_Click()
  'Abort-Button
  Unload Me
End Sub

Private Sub UserForm_Initialize()
  Dim SignatureFile As String
 
  'Setup the form
  Set fso = CreateObject("Scripting.FileSystemObject")
  Me.Caption = "Choose a signature"
  With Me.CommandButton1
    .Caption = "Ok"
    .Default = True
  End With
  With Me.CommandButton2
    .Caption = "Abort"
    .Cancel = True
  End With
 
  'Read in all signatures
  SignaturePath = Environ("appdata") & "\Microsoft\Signatures\"
  SignatureFile = Dir(SignaturePath & "*.htm")
  Do While SignatureFile <> ""
    Me.ListBox1.AddItem fso.GetBaseName(SignatureFile)
    SignatureFile = Dir
  Loop
End Sub

rohn007 say:

It looks like your text file has a fixed maximum width. I didn't count, but typically text files are limited to 80 char or slightly less.  Anything longer than that is put onto a new line.  It is a very old computing standard.  I notice that the columns are not comma separated, so I'm assuming they are all fixed width, filled with spaces or zeros to make the lines all fit the 80 char length.

You could open the file in Word, then use find/replace to find all paragraph marks / returns after any 80 (or what ever your line length number is) characters and replace it with a space.

Then you can use the Excel import function and define where the columns are to be set.
