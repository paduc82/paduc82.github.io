Enterprise Resources in Project 2013

I'm working with Microsoft Project 2013 and am having difficulty creating an Enterprise Resource Pool. I have tried multiple views, resources, and jobs to fix the problem. I can see the Enterprise Resource Pool option under the Resource tab/Resource Pool drop-down menu, but am unable to select it. How can I create this resource pool? 

dale howard [mvp] say:

Nichole --

Are you using Microsoft Project 2013 with either Project Server 2013 or with Project Online (the Project Server 2013 system hosted by Microsoft)?  If so, you do not create the Enterprise Resource Pool in Microsoft Project 2013.  Instead, you create the ERP in Project Web App (PWA) on the Resource Center page.

If you are not using Project Server 2013 or Project Online, then you cannot create an Enterprise Resource Pool.  The reason you are seeing this option on the Resource Pool menu is because you are using the Professional version of Microsoft Project 2013, which is the version required to work with Project Server and Project Online.

What I recommend you do instead is to create a local shared resource pool file.  To do this, create a new blank project, apply the Resource Sheet view, and then enter the names of all of the resources who will work in all of your projects.  Save the file using a descriptive name.  Next, open a project that you want to share the resources with the shared resource pool file and click Resource Pool > Share Resources.  In the Share Resources dialog, select the option to use the resources in the shared resource pool file, and then click the OK button.  Save and close the project, and repeat this process with every project in which you want to use the resources in the shared resource pool file.  Lastly, save and close the shared resource pool file.

Hope this helps.
