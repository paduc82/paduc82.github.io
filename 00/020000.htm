brian tillman [outlook mvp 2007-2019] say:

You can't.  Whether you like it or not, the fact is that Address Book is just a special view of the Contacts folders you have defined as address books in their properties (or that Outlook had defined automatically). You cannot export from it or import to it.  The difference is that your Contacts folders will show you all of your contacts, including those that are incomplete and don't have an email address or fax number defined for them. The Address Book view however, will show you only those contacts that have valid email addresses or fax numbers defined for them.  The entire purpose of the Address Book service is to convert contact names into addresses.

If you want to see only those contacts that are likely to appear in the Address Book view, display your Contacts folder in the List view.  Choose List in the Current View chunk or click View>Current View>List.  Then sort by the Email field.  This will pull all your contacts having addresses into alphabetical order by address.  You can then select those entries and copy them to an Excel spreadsheet., saving them in an xlsx or csv format.

  • Change back to the interactive mode (worksheet) and go through the required procedure to print but this time select the user usual or default printer so user's normal printer is set.
  • Now the following code will identify the usual or normal printer and save it, change to the required new printer, print the data and return to the usual or normal printer.
  • In the line of code that sets the Active Printer, copy your printer name from where you saved it (omit the single quote which makes it a comment) and replace the printer name that I have used with the printer you want to select for the print job. Note that the printer name is enclosed in double quotes.

Sub ChangePrinter()
    Dim strUsualPrinter As String
   
    'Save the Usual printer
    strUsualPrinter = Application.ActivePrinter
   
    'Change to the required different printer and Print
    Application.ActivePrinter = "FX DocuPrint CP105 b on Ne01:"         'change this line to your required printer name
    ActiveSheet.PrintOut
   
    'Change back to the usual printer
    Application.ActivePrinter = strUsualPrinter

End Sub

To update a specific link, say the first Excel link source

ActiveWorkbook.UpdateLink Name:=ActiveWorkbook.LinkSources(xlExcelLinks)(1)
