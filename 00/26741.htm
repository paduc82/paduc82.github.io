a. user say:

You can do most of what you could do on the Database toolbar in Word 2010 but it was clear from Word 2007 that Microsoft regards this stuff as specialised.

Go into the File tab and click Options. Then if all you need is the Insert Database icon, click Quick Access Toolbar. Under "Choose commands from", select "Commands not in the ribbon." Locate "Insert Database" in the list, select it, and click Add.

By default, the Quick Access toolbar is displayed at the top left hand corner of the Word window.

If you need the other tools on the Database toolbar, it may be better to go into Options, then click Customize Ribbon, then add a new New and/or a new Group to the ribbon, locate each function you need and add it to that group. I don't know if all the functions are available, and some of them may not have the same names as the tooltips on the Word 2003 toolbar, but ask again if you need more help on that.
