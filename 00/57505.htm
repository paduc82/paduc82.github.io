considerate_guy say:

A few relevant points:

  • You can still download Windows Movie Maker 6.0 and continue using it on your computer.
  • The "newer" Windows Movie Maker 2012 doesn't have a traditional timeline. Instead, you directly add photos and videos to the "project" area. This area behaves a little like a wrapped timeline. You can reorder your clips and photos, add music below, etc. In the default zoomed view, each media item is represented by a single thumbnail and you can use this like the storyboard mode in the original WMM 6.0. Or if you use the zoom slider in the lower right, you can zoom in and the project area behaves more like a traditional timeline. Try using it a little and it should start to make sense.
  • The "newer" Windows Movie Maker 2012 was actually retired back in 2017. It never really had any online manuals but any resources that Microsoft had are now removed from their servers.
  • The intended path for editing videos is to use the Photos App. There are menu items for Create Project or Create Video that can work from your selected clips and photos. It's more restricted than Movie Maker in that you can only pick a single song for the soundtrack but you might want to take a look at that for quick movies.
