luc sanders say:

Swoklatt,

Have you tried selecting the text in Word, then paste special in PPT choosing the Paste link radio button, click Microsoft Word object and click OK. john korchok say:

Luc's suggestion is a good one for your specification that you only want a part of the document to appear in the presentation. As an alternative approach, you can edit the Word file to have only the information you want to appear in PowerPoint, then save it as a separate file. Then follow these steps to link it to PowerPoint as an updatable source:

  1. In PowerPoint, choose the INESRT tab. In the Text section, choose Object.
  2. Select the Create from file option.
  3. Click on the Browse button and find the Word file. Select it and click on the Open button.
  4. Click on the Link checkbox, so that changes in the Word file will be reflected in PowerPoint. OK out.

If you have Acrobat Reader installed, you should be able to follow the same sequence, but choosing a PDF to link to instead of the Word file.
