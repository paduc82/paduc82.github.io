apollo67 say:

Your problem is in your where condition.  You are using a LIKE statement and applying wild cards to either side of the criteria value.  So of course you are going to get every record where the field contains a 3.  You need to correct your where statment to perhaps equality (=) versus LIKE.

"[qryCUSTOMER_LISTING].[RT] = " & Me.txtSEARCH

It appears that you're querying for numbers only.  However, be aware that if your field in your criteria is of type Text, you will need to supply quotations on either side of your sought value using the Chr() function.  (i.e. Chr(34))

Derek Cunningham,

Just modify the UDF.  Say cell A1 will hold the word Monday and the background color red.  Then you could modify the UDF to be this: 

Function CountCcolor(range_data As Range, criteria As Range) As Long
    Dim datax As Range
    Dim xcolor As Long
xcolor = criteria.Interior.ColorIndex
For Each datax In range_data
    If datax.Interior.ColorIndex = xcolor And _
       datax.Value = criteria.Value Then
        CountCcolor = CountCcolor + 1
    End If
Next datax
End Function

Then you would call it like this   =CountCcolor(F:F,A1)

erica blair say:

Hi Kevin,

Another option besides filtering is to leave your tables alone and use the VLOOKUP() function to implement a search.

Do this by completing the following steps:

  1. Set the first of your data ranged and insert an excel table, then give it a name like Teams (the arbitrary names i will use for this example). Note you only need one copy of your table for this example so we will not discuss the second table.
  2. Choose two cells to be your input - these will be reserved for typing in a team name. If you'd rather a dropdown list, with these two cells selected (we will name them t1_ and t2_ arbitrarily) go to Data on the ribbon and select Data Validation > Change the type to list and put this in the formula bar: =Teams[Teams] (assuming the first column in the Teams table is named Teams)
  3. Now your t1_ and t2_ cells should have dropdowns that select between the teams. Create another quick table or tables where data for one team will be shown, either one atop another or side by side (however you like). For the data for the first table make the formula in the data cells: =VLOOKUP(t1_,teams,2,false). The '2' in this formula picks a column so you will need to change it to get different data columns, but if you have 3 data points per team, you just have 3 different VLOOKUP()s with 2, 3, and 4 in that parameter.
  4. Repeat for the other lookup table but use t2_ instead.
  5. Now you should be able to lookup exactly two teams from your tables by selecting the dropdowns accordingly.

Please let me know if this helps and if you need further assistance.
