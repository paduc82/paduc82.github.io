zigzag3143 (ms -mvp) say:
To access People from Outlook.com, hover the cursor over the Outlook logo in the top left and click on the down-arrow that appears to the right of the Outlook logo.
Select People. 

Returning to your Inbox from People:
Hover the cursor over the People logo and click on the down-arrow that appears on the right of the People logo. Select Mail.  
You can also get to Calendar or Skydrive in the same manner.

Filtering your list:
Above the search box at the top of the Contact list on the left, is a filter option which says "All ". 
If you click on this you should see the groups you created in the past.  
Click on any of these to filter your list to just the contacts in that group.

Selecting one or many Contacts:
If you click on a contact name in the list, the details for that contact will appear on the right.  
Alternatively, if you click on the check box to the left of the contact picture, you can build up a list of selected contacts on the right by clicking on the check box to the left of multiple contacts.

Adding/Removing from Group membership:
Whether you select one or many contacts, you will see a top menu bar option for Groups.  
If you click on this, then it will display the list of groups with a check box checked for each group that the contact(s) is/are a member of. 

You can uncheck to remove or check to add, the selection of contact(s) to the chosen groups, and then click on Apply to apply the changes.  
Your selected contacts will then be updated to be added or removed from the groups you chose as appropriate.

Delete
When you select one or many contacts, you will see a top menu bar option for Delete.  
If you click on Delete and click Delete in the confirmation box, the selected contacts will be deleted.

Edit:
You can choose edit if you have clicked on a single contact (name or picture) in the list on the left to see their details on the right.  
When viewing their details, click on Edit in the top menu bar to edit or add details.

imaquila say:

vlookup hit and miss return values

I have one worksheet with  columns (D) ItemID , (E) Description,(G) Price  and a 3 column table spanning J3:L10 (1st column ItemID, 2nd column Description and 3rd column Price)
I've attempted to use a VLOOKUP in column E using the ItemID in column D  to compare with the first column of the table for a match and then return the Description in the second column of the table.  Likewise in Column G has a similar VLOOKUP to return the Price from the 3rd column of the table.
I currently only have 6 rows and most of them return the price correctly but none of the rows return the description.
The following is an example  taken from Row 6:  Col D -ItemID 309, Col E-Description   =VLOOKUP(D6,$J$3:$L$10,2,False), Col G-Price  =VLOOKUP(D6,$J$3:$L$10,3,False).   The Result is that D6 contains the ItemID of 309 (which has been formatted as text), E6 contains the VLOOKUP formula for Description and  "N/A" is in Cell G6.    The other 5 rows all display price correctly but none of the rows return the Description from the ItemList Table.

Clearly, I'm missing something in the understanding of VLOOKUP and I would be grateful for any clarification that would solve these issues now and in the future.
Thanks
pete_uk say:

The original file seems to be still available at that mediafire link you quoted. Your description column contains the VLOOKUP formula as text - you should format cell E2 as General and then double-click on the cell as if to edit it and then press <enter>, and this will effect the change in format. This can then be copied down.

 

Hope this helps.

 

Pete

pete_uk say:

Re-enter the value in cell J9 as '309 - the apostrophe will not show in the cell but will be interpreted as a text value and thus will return the correct Description in cell E6 (and in G6:H6).

 

To remove the errors in column G, you can change the formula in G2 to this:

 

=IF(D2="","",VLOOKUP(D2,$J$3:$L$10,3,FALSE))

 

and then copy down. This will return 0 in column H, but if you want that to be blank as well, then you can change the formula in H2 to this:

 

=IF(D2="","",PRODUCT(F2,G2))

 

or you could simplify it to:

 

=IF(D2="","",F2*G2)

 

and then copy it down.

 

Hope this helps.

 

Pete
