andreas killer say:

If you open a new file, Excel creates the window state in the same state as the window you choose to open the file.

Resp. if your active window is maximized and you double click a file within the Windows Explorer, the file is opened maximized.

So if the Excel window is only on a part of a monitor, the new file is opened in that size also. AFAIK there is no way to change the behavior.

I work also on PCs with multiple monitors for many years and facing similar issues. There are 2 solutions:

a) You can add code to your PERSONAL.XLSB and catch the events when a file is opened. And if the filename match, you can move that window to a specific monitor at a specific place.

b) The same (and much more) can be done with a 3rd party tool, I use the Actual Window Manager to move and place my windows. Very easy to use and IMHO not expensive.

https://www.actualtools.com/windowmanager/
