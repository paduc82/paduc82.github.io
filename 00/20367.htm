In MS WORD 2010 How do I make large fonts on the same page? - Microsoft Community

mountaindont say:

In MS WORD 2010 How do I make large fonts on the same page?

In MS WORD 2010 How do I make large fonts on the same page? I want to make large letters appear on the In MS WORD 2010 How do I make large fonts on the same page? I want to make large letters appear on the same page in landscape mode. The letters are size 270. The first line has 4 letters and the second line has 5 letters. I can get the first line but when I start the 2nd line it goes to the next page. Any suggestions you can pass along would be appreciated
pamcaswell say: 270 pts times 2 divided by 72 = 7.5 inches.  But the height of an 8.5x11 landscape page with 1" margins is only 6.5".  You can reduce the margins to something a little less than .5"  or reduce the font's point size to 234 pts, or a little less.  Also make sure there's no paragraph before and/or after spacing.  Otherwise, the margins and point size will have to be smaller yet to fit

EDIT:  My calculations ignored the fact that, normally, the line spacing is 120% of the point size.  So you also will need to set the line spacing to exactly 270 pts.



HTH,
Pam

doug robbins - mvp office apps & services (word) say:
In MS WORD 2010 How do I make large fonts on the same page? I want to make large letters appear on the same page in landscape mode. The letters are size 270. The first line has 4 letters and the second line has 5 letters. I can get the first line but when I start the 2nd line it goes to the next page. Any suggestions you can pass along would be appreciated

You will need to set the spacing between the paragraphs to 0 and the line spacing can be set to about 215 pts before it starts to cut off the top of upper case characters.  If you are using lower case, to prevent cutting off the bottom of the descenders, a larger line spacing will be required.  You should also minimise the top and bottom margins.
