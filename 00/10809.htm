john mcghie say:
Oi Rafael:

Não se pode dizer qual chave do produto foi usado em uma máquina específica.

A chave de produto não é armazenado na máquina de destino. A chave do produto é utilizado para semear um algoritmo de criptografia que gera a identificação do produto.

Se você olhar no menu de aplicativos para o item Sobre e mostrar que, ele irá mostrar a identificação do produto para você.

Se você encontrar duas máquinas em que todos, mas o último jogo de cinco dígitos, a mesma chave foi usada em duas máquinas.

Não vai dizer-lhe "que" chave foi usado, apenas que a chave "mesmo" foi usada.

Apenas tentativa e erro vai deixar você resolver qual é a chave que pertence ao computador. Você pode mudar as licenças sem remover e reinstalar no entanto.

Pode ser mais rápido para você remover todas as 16 chaves e começar de novo: Em cada máquina:

1) Feche todos os programas da Microsoft (Command + q)
 
2) Exclua os seguintes arquivos:

/ Applications / Microsoft Office 2004/Office/OfficePID (normalmente um arquivo invisível)
Usuários / ~ / Library / Preferences / Microsoft
     - Banco de dados de Registro de carbono
     - Cache de registro do Office
     - Microsoft Office Configurações

Onde "~" é o seu nome de utilizador: clique em "House" símbolo no Finder

3) Reinicie o computador

4) Inicie qualquer aplicativo do Microsoft Office (por exemplo, Word)

Ele irá pedir para re-introduzir a chave de produto. Desta vez, manter o controle deles :-).

Espero que isso ajude


Hi Rafael:

One cannot tell which product key was used on a specific machine.

The Product Key is not stored on the target machine. The Product Key is used to seed an encryption algorithm that generates the Product ID.

If you look in the Application menu for the About item and display that, it will display the Product ID to you.

If you find two machines on which all but the last five digits match, the same key was used on those two machines.

It won't tell you "which" key was used, just that the "same" key was used.

Only trial and error will let you sort out which key belongs to which computer. You can change the licenses without removing and reinstalling however.

It may be quicker for you to remove all 16 keys and start again: On each machine:

1)  Quit all Microsoft programs (Command + q)
 
2) Delete the following files:

/Applications/Microsoft Office 2004/Office/OfficePID (normally an invisible file)

Users/~/Library/Preferences/Microsoft
    - Carbon Registration Database
    - Office Registration Cache
    - Microsoft Office Settings


Where "~" is your login user name: click the "House" symbol in the Finder

3)  Reboot the computer

4)  Start any Microsoft Office application (e.g. Word)

It will prompt you to re-enter the Product Key.  This time, keep track of them :-).

Hope this helps

    ActiveDocument.Comments.Add range, ExWb.Sheets("Words").Cells(i, 2)

graham mayor say:

The following should work for you

Option Explicit

Sub InsertCommentFromExcel()
Dim objExcel As Object
Dim ExWb As Object
Dim strWorkBook As String
Dim i As Long
Dim lastRow As Long
Dim oRng As Range
Dim sComment As String
    strWorkBook = Environ("USERPROFILE") & "\ReplacementWords.xlsx"
    Set objExcel = CreateObject("Excel.Application")
    Set ExWb = objExcel.Workbooks.Open(strWorkBook)
    lastRow = ExWb.Sheets("Words").Range("A" & ExWb.Sheets("Words").Rows.Count).End(-4162).Row
    For i = 1 To lastRow
        Set oRng = ActiveDocument.Content
        Do While oRng.Find.Execute(ExWb.Sheets("Words").Cells(i, 1)) = True
            sComment = ExWb.Sheets("Words").Cells(i, 2)
            ActiveDocument.Comments.Add oRng, sComment
        Loop
    Next
    ExWb.Close
lbl_Exit:
    Set ExWb = Nothing
    Set objExcel = Nothing
    Set oRng = Nothing
    Exit Sub
End Sub
