prashantkumar96 say:
From Wikipedia:

Firewall -->
Its primary objective is to control the incoming and outgoing network traffic by analyzing the data packets and determining whether it should be allowed through or not, based on a predetermined rule set.
Network Monitoring Software -->
While an intrusion detection system monitors a network for threats from the outside, a network monitoring system monitors the network for problems caused by overloaded and/or crashed servers, network connections or other devices.

Primary difference:
Firewall scans incoming and outgoing traffic. Network Monitoring Software prevents intrusion from outside and monitors network's problems.
