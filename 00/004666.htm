This is the expected behavior. Outlook will always send on the same account that downloaded it. The Send emails option is for including the account in a send/receive, not what account is used for replies and forwards.

If you want to send replies and forwards from a different account, go into Account settings (tools menu) and change this account's outgoing server to the one you want to use or can use.  Go into More Settings and set the outgoing server authentication so its correct for the desired smtp server. If you want replies directed to a different account, change the email address and reply to addresses for the account also.

A common scenario for this is you changed ISPs and the old one lets you receive mail at the email address for 6 months. You need to send replies from your new ISP.  You'll use the old incoming mail server and logon as usual, but change the email address and the outgoing server to the new one.     Dim x, mailItem As Outlook.mailItem

Sub fooMail(ByRef mItem As Outlook.mailItem)
'code that numbers subject line

 Dim sAppName As String
 Dim sSection As String
 Dim sKey As String
 Dim lRegValue As Long
 Dim lFormValue As Long

 sAppName = "Outlook"

     iDefault = 1 ' adjust as needed

     lRegValue = GetSetting(sAppName, sSection, sKey, iDefault)
