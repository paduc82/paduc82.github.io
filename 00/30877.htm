If an upgrade fails during the SAFE OS PHASE you'll have to do a clean install.

During the SAFE OS PHASE of an upgrade the image is placed on the hard drive; in this case the 'image' means the bits constituting the upgrade. The error 0x80070057 refers to file corruption.

Hi,

Thank you for the response and keeping the status updates.

Because Windows Firewall with Advanced Security plays an important part in helping to protect your computer from security threats, we recommend that you do not disable it unless you install another firewall from a reputable vendor that provides an equivalent level of protection.

To disable the firewall portion of Windows Firewall with Advanced Security from a command prompt

  1. Open an Administrator: Command Prompt. To do so, click Start, click All Programs, click Accessories, right-click Command Prompt, and then click Run as administrator.

  2. If the User Account Control dialog box appears, confirm that the action it displays is what you want, and then click Continue.

  3. At the command prompt, type the following command:

    netsh advfirewall set profiles state off

    where profiles is AllProfiles, CurrentProfile, DomainProfile, PrivateProfile, or PublicProfile.

To disable the firewall portion of Windows Firewall with Advanced Security by using the Windows Firewall Control Panel program

  1. Click Start, click Control Panel, click Network and Internet, and then under Windows Firewall, click Turn Windows Firewall on or off.

  2. On the General tab of the Windows Firewall Settings dialog box, select Off (not recommended), and then click OK.

misses joust say:

Using Twitch to Stream

G'day all! We've seen that some of you are running into some issues when trying to stream your games to Twitch. A lot of these issues can be solved by tweaking your privacy settings. 

Please note that Child Accounts will not be able to stream. That is, Accounts that are under the age of Majority (18 years in most regions). Changing the account's privacy settings to "Adult" will not change that.

In order to stream on Twitch, you do not need your Privacy settings set to "Adult," but you do need to have them set in the following way:

  • In your Profile, make sure your presence is set to online.
  • In the Settings app under "Privacy and Online Safety," make sure of the following:
    • "See if you're online" is set to Everyone
    • "Broadcast Gameplay" is set to Allow.
    • "Share content made using Kinect" is set to Allow.
  • In the Settings app under "Preference," check Allow broadcasts and Game DVR Uploads.
  • In the Settings app under "Kinect," check Kinect on.
  • If you want to include Kinect audio in your broadcast, also check Use Kinect microphone for chat.

If your settings reflect all of the above and you're still unable to stream, then please start a thread here and describe exactly what's going on. Include error codes, error messages, games you're trying to stream, and what you've tried already. We'll help you find a solution!
