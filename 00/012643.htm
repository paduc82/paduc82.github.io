    This is a very well know issue.

    Excel, like most math apps (other than expensive ones like Mathematica, Maple) store numbers in binary notation using the IEEE 754 convention. Within the precision of the method some decimal number cannot be exactly represented in binary.

    You can fix the problem by doing some rounding. So replace formulas such

    as   =IF(((C2-B2)=0);0;‚Äùerror‚Äù) 

    by   =IF(Round(C2-B2,13)=0;0;‚Äùerror‚Äù)
