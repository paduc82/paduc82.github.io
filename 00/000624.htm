  2. Select Compatibility Tab
  3. Place a check mark next to Run this program in Compatibility mode and select the operating system accordingly from the drop down list
  4. Click OK and run the setup.

Restart the WLAN Auto Config service

Follow the steps below to restart the WLAN Auto Config service:

 

  1. Press Windows Key + R
  2. Type services.msc
  3. Right click on the WLAN Auto Config service and select Restart, or click on Start if it‚Äôs not started.
