eric lu msft say:

Hi Jessie,

I'm glad to help you.

The maximum of storage for each site is 25 TB, it is a limit in SharePoint Online you need to know. If you need more storage, you should Add storage space for your subscription. 6.84 TB is your available storage now.

The number of total GB used by existing sites includes all documents in each site. You can regard it as total used storage.

If you want to know the details about used storage in each site, click the title of the site and check under Insights tab. 

Manage sites in the new SharePoint admin center for your reference.
