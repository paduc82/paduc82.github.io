chemcal say:

Generally, there could be a way to do it by a circumvention. The method of an auxiliary (helping) column parallel to active cells was used currently in the older programming; now is being partly forget. If you set a flag (number, character etc.) into a cell adjacent (or equally distant) to the cell you want to color up, you can set each of cell colors by a conditional format. It is usually required that the flag column would be hidden, though this need not to be a common situation. The trick is then in setting such a flag. The best way is a macro subroutine. After you activate a cell to be colored up, you call the macro that changes the flag (i.e. from empty to 1). The macro should be best activated by a key shortcut. An example of macro that even toggles, as the result, the color on and off:

Option Explicit

Sub ColorCell()

Const Flag As Long = 1, Zero As Long = 0

If ActiveCell.Offset(0, 1) <> Flag Then

  ActiveCell.Offset(0, 1).Value = Flag

Else

  ActiveCell.Offset(0, 1).Value = Zero

End If

End Sub

As you can see, you will obtain the readily activated conditional sum of A1:A20 with input 1’s in B1:B20 as the function

=SUMIF(B1:B20, 1, A1:A20).

As you can also see, when there is no need for hiding the auxiliary column, you can input the flags in a direct way.
