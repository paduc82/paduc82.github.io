douglas eckert say:

PERCENTILE.EXCE VERSUS PERCENTIL.INC

What is the difference between the formula which uses PERCENTILE.EXC versus PERCENTILE.INC?  I understand what they are, but I do not understand their significance.  How would a user choose between them?  Why?

 

Douglas Eckert

York PA

hansv mvp say:

The syntax is PERCENTILE.EXC(array, k) and PERCENTILE.INC(array, k)

 

PERCENTILE.EXC only works if k is between 1/n and 1-1/n, where n is the number of elements in array.

 

PERCENTILE.INC uses a slightly less accurate algorithm, but it works for any value of k between 0 and 1.

petra hall_808 say:

Automatic email from Excel based on date in cell

Hello,

Please can anyone tell me how to get Excel 2013 to send an automatic email via Outlook based upon the 'review' date in a cell?

Many thanks

BTW, I am a total Excel VB newbie so be gentle please ;)

***Post moved by the moderator to the appropriate forum category.***

dario_sch say:

This code needs the workbook to be opened to check for the review date. I don't think it would be possible to execute the code without opening the workbook.

__________________________________________________

Private Sub Workbook_Open()

Dim OutMail As Object
Dim OutApp As Object

    If Sheets("Admin").Range("A1") = Date Then
        If MsgBox("A review is necessary, would you like to send an e-Mail?", vbOKCancel + vbQuestion, "Accept/Reject") = vbCancel Then Exit Sub
    Else
        Exit Sub
    End If


Set OutApp = CreateObject("Outlook.Application")
Set OutMail = OutApp.CreateItem(0)


    With OutMail
        .To = "*** Email address is removed for privacy ***"
        .CC = ""
        .BCC = ""
        .Subject = "Review Necessary"
        .Body = "Hi," & vbCrLf & vbCrLf & "You need to review..."
        .Send
    End With


Set OutMail = Nothing
Set OutApp = Nothing

End Sub

Paste this into the module and it will ask you whether you want to send an e-Mail if the date you're opening the file is the same as the date in the cell A1 in the "Admin" Tab. You can change these values.
