a. user say:

We'll need to do a general troublehooting to see if we can identify and resolve the problem.  You may need to use the disk to do some of these procedures.  Skip any you can't do because you can't access the drive (and come back to them one access is re-established.  Can you get in using Safe Mode?  Can you get in using the disk.

Do you know when this problem began?  Try a System Restore to a point in time BEFORE the problem began.  Here's the procedure: http://www.howtogeek.com/howto/windows-vista/using-windows-vista-system-restore/.  Be sure to check the box to show more than 5 days of restore points.  If the first attempt fails, then try an earlier point or two.  NOTE: You will have to re-install any software and updates you installed between now and the restore point, but you can use Windows Update for the updates.  Use the recovery disk if the system prompt doesnt work. The recovery disk works a bit different from the above procedures but if you follow the prompts from the System Restore menu option with the above information you should be able to restore with no problems.

If the System Restore doesn't work, do a Startup Repair by booting to the genuine Windows Vista Installation Disk (or one you can borrow from ANYONE) or from a Recovery Disk.  Here's the procedure: http://www.bleepingcomputer.com/tutorials/tutorial148.html.  To boot to the CD you may need to change the BIOS to make the CD-drive first in the boot sequence.  To do that, wait for the screen that tells you the F key to push to access the boot menu or boot setup.  Push it quickly. Make the changes, save your work, and exit.  Put the CD in the drive and reboot.  When prompted, push any key to boot from the CD.

If you don't have either disk, you can make a bootable Recovery Disk using http://neosmart.net/blog/2008/windows-vista-recovery-disc-download/ along with burning software like: http://www.snapfiles.com/get/active-isoburner.html and, of course, a blank CD (you may need to do this from another computer)

If that doesn't work, try to boot into safe mode (repeatedly click the F8 key while booting and go to safe mode with networking or do so from the command prompt on the disk).  Then let's check some of your system files:

Go to Start / All Programs / Accessories / Command prompt and right click on command prompt and click run as Administrator (you can skip this step if using the disk).

If using the disk, cd to C:\Windows\System32. Type sfc /scannow and enter and let it run.  It will scan and try to fix some of your system files.  Hopefully it will complete with no corruption it could not repair (if there is such corruption post back here or try to analyze it to find the problem file(s) using http://support.microsoft.com/kb/928228.  Try to post any corrupted files here so we can see if they can be repaired with good copies from the installation disk (unless there are too many).

While in Command Prompt, type chkdsk /f /r and enter and let it run.  It will want to schedule itself to run at the next restart.   Answer yes and then reboot to run the program.  It will scan and try to fix any corruption or bad sectors on your hard drive and mostly remove that as a potential cause.

If that doesn't work, then please post any repeating error messages from the Event Viewer concerning startup (Start / Control Panel / Administrative Tools / Event Viewer or typing eventvwr from the command prompt starting from C:\Windows\System32).  Here's how to use Event Viewer: http://www.petri.co.il/vista-event-viewer.htm.  Look in the System section.

Hopefully this information will help us identify and/or resolve the problem.  If none of them work, we'll try something else.

Good luck!

Most modern video editing programs today will be able to do all the things you mentioned, even better than Movie Maker. considerate_guy say:

I understand the convenience of a DVD in that you don't need a computer but just a standalone DVD player connected to a TV. For your needs, that may be just right so I don't want to discourage you in that.

If you have some slightly more modern equipment (e.g. an HDTV and a Blu-ray player), then you can just burn the .wmv or .mp4 file that Movie Maker creates directly onto a disc (e.g. a DVD disc). This would not be in a format playable on a standard DVD player but would be understood by an HD device like the Blu-ray player. It would have better quality compared to a video DVD played in a DVD player. But again, no pressure to do that if a DVD is working for you.

As to why you often see parts cutoff when using a DVD, the main cause is overscan. I'm sure you remember back to the advent of television with CRTs (cathode ray tubes). The picture is formed by electrons being "drawn" onto a screen over and over at a rate of 60Hz (or maybe 50Hz in Europe and other places). But because of the technology in CRTs, these scans would actually go beyond the edge of the screen as "overscan". So content near the edges would often not be seen. If you were a television station, you would just always make sure that you wouldn't put any important information, especially text, near the edges and so the viewer wouldn't really notice that they weren't seeing all the picture.

So if CRTs are dead then why do we still have overscan? This is where the bad news comes in, but the short answer is because it used to be there. The source of the problem is that broadcasters expect the TV to crop the image so they don't mind putting garbage on the edge like a misc yellow line on the left or black and white dots along the top containing "metadata" such as captions, content ratings, show title, etc. To hide that, even modern HDTV manufacturers will often zoom in the content and not show the edges. It's happening less and less with HD broadcasts, but if you are using a standard-definition source (like a DVD player) it may still be zooming in the picture to hide these edges in the "overscan" area. The amount of overscan varies with the combinations of television and DVD players, so you may get different symptoms if you are playing it at home on your television or if you are going to a relative's house and using theirs. 

Anyway, I hope that explains things. Let me know if there is any other knowledge I can impart on this or similar subjects.
