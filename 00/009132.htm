anna bon say:

Hi Kiliee,

Outlook generates and attaches a winmail.dat file at the end of each message that only Outlook can use. It's indicated at the end of the message so that on the receiving side, Outlook can display the message correctly. Outlook is the only email program that can use this information. Therefore, recipients using email clients other than Outlook might receive the winmail.dat file instead of the original message.

To help prevent email recipients from receiving winmail.dat files, the formatting needs to be changed to Plain Text. This is applicable if you are sending your email using the Outlook application. Please follow the steps below:

1. Open Outlook application.

2. Go to File.

3. Select Options.

4. Click on Mail tab on the left pane.

5. Under Compose message, select the drop-down list under Compose messages on this format. Select Plain Text, and click the OK button.

If you are using an Outlook.com, please check this link for the resolution.
