thechosenone22 say: Both your statements are correct - for the following reason.

If you buy a SIM free version of a Nokia phone, then you are entitled (and able) to put any SIM card in it and it'll work - however you're looking at an up front cost of - for example - 500 GBP for the N95. The firmware for this type of phone is then readily accessible from Nokia and available for download from Nokia once released.

However if you purchase or receive your phone as part of a deal from a network operator (e.g. Vodafone, Orange etc.) then chances are the phone will be locked to that particular network - so you will be unable to use any other SIM card in the phone without first purchasing an unlock code from the network operator). With these phones, the firmware that the phone runs will be customised to the particular requirements of the network operator. In this case any new versions of firmware have to be approved for release by the network operator before it can be downloaded, hence your phone is listed as still having the latest version of firmware, when technically it doesn't if you compare it to a SIM free phone.

You won't be paying anywhere near as substantial up front cost, but instead it'll be like a continuous wasp sting - they'll take money from you every month instead to recoup the cost of the phone.

niyamath u khan say:

This happens because one (or multiple) sheets in your workbook has a different dpi setting for printing

In order to have all the individual sheets printed within a single PDF file (not multiple PDFs) you need to set the same DPI printing value for all of them. 

To set the same Page Setup options for all individual Excel sheets:

1.     Open the Excel workbook

2.     Right-click the tab for any of the sheets you have in your workspace (by default they sheet tabs are named Sheet1, Sheet2, ...)

3.     Choose the Select all Sheets option (which will select all the sheets in your workspace, you can tell they're selected if all the sheet tabs have a white background like an active sheet tab would have)

4.     In Microsoft Excel 2007 go to Page Layout and click on the expand button for the Page Setup dialog (for Microsoft Excel 2003 or earlier go to File->Page Setup). Set the Print Quality to the DPI value you want and press OK
