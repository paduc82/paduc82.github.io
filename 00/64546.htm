saltgrass say:

You don't really state why you need to have the Safe Mode option.  MSconfig.exe will, using the Boot tab, set the system to boot into Safe Mode the next time you reboot.  Once in Safe Mode, you can change msconfig.exe back to a normal boot.  Msconfig adds a line to the BCD store to allow it to boot into Safe mode.

You can also probably boot into Safe Mode by using the Settings-Update-Recovery options.

If you change the BCD store, you really need to make sure you know how to change it back in case that may become necessary for whatever reason.

Commands need to be done in the Administrative Command Prompt window.  If you want to see all options with bcdedit, type bcdedit /? and look through the listing.

anannya podder say:

Hi,

Let me assist you with your concern.

An administrator account is a user account that lets you make changes that will affect other users. Administrators can change security settings, install software and hardware, and access all files on the computer. Administrators can also make changes to other user accounts.

The first time you set up Windows, you created a user account. This account is automatically an administrator account so that you can finish setting up your PC and install any programs that you would like to use. When you add other accounts to your PC, however, they'll usually be standard accounts. Standard accounts are best for everyday use. The sign-in screen in Windows shows what accounts are set up on that PC.

I would like to know how many user accounts are shown in the sign-in screen. Also, which user account was used on the PC to set up Windows 8.1?

From the lock screen window, you can Hold down  SHIFT key while tapping or clicking on Restart (click on the power button and it will give the option to Restart). "My ASRock X99 Extreme4"

Apparently the only difference between SATA and S_SATA is that the 6 SATA ports support RAID arrays and the 4 S_SATA ports do not. You could try the drive in any of the available ports, or perhaps try switching the data cable in case it is the source of your problem.

According to the specs I see below, that board should support 10 SATA drives? 4x S_SATA ports -0 through 3 and 6x SATA ports -0 through 5. The eSATA port is shared with one of the S_SATA ports.

http://www.asrock.com/mb/Intel/X99%20Extreme4/

The only limitations in the SATA port usage that I noticed was this:

"*S_SATA3_3 connector is shared with the eSATA port, S_SATA3_2 connector is shared with Ultra M.2 Socket"

So if you are connecting a drive on the eSATA port then S_SATA3_3 should be unavailable.

Why don’t I have the Get Windows 10 app?

If you don’t see the Get Windows 10  app on your PC, it might be because:

  1. Your device isn’t up-to-date with at least Windows 7 SP1 or Windows 8.1 Update.
  2. Windows Update is turned off or is not set to received updates automatically.
