how to change margin and extend to 80 ch.length in wordpad screen? - Microsoft Community

peternovas say:

how to change margin and extend to 80 ch.length in wordpad screen? fontsize is o.k.

I am copying formatted TXT files from my dbasevisual software ito C:\USERS\PETER\docume~1 wordpad txt file which is formatted as a

10ch margin and 65ch length fixed.I dont know how to change the wordpad format to a 3ch margin and 80ch length.

jay freedman say:
First, I'll assume you really are using Wordpad (the program that comes with Windows) and not Word (the program from Office). They're considerably different.

Second, this takes some math. The margins and line widths are in inches, centimeters, or points, not in characters, because in most fonts some characters are wider than others. Setting up for characters makes sense only if you're using a fixed-width font such as Courier New or Lucida Sans Typewriter, and then only if you know what the font size is (and it's all the same size). In either of those fonts, 12-point type fits 10 characters per inch, so let's use that. (If you need smaller text, 10-point type fits 12 characters per inch, but the math isn't as simple then.)

With that assumption, you're asking for margins that are 0.25 inch and a text column that's 8 inches wide in a total (letter-size) page width of 8.5 inches.

  • Click the dark blue rectangle in the top left corner of Wordpad and click Page Setup.
  • Make sure the paper size is set to Letter 8.5"x11" and the orientation is set to Portrait.
  • Change the left and right margin boxes to 0.25". (You can change the top and bottom margins or leave them as is.)
  • Click OK.
  • In the document, the markers on the ruler should be shown at 0" (which is 0.25" from the left edge of the paper) and at 8" (which is 0.25" from the right edge of the paper).
  • Set the font to Courier New or Lucida Sans Typewriter, and set the size to 12 pt.
  • Paste in the text.
