  4. Follow instructions at https://www.symantec.com/connect/blogs/windows-installer-safe-mode to start windows installer service.

  9. Look under Display Adapters, Monitors, and anything else with a similar icon, for duplicate devices/caution symbols.  If you have an integrated and a discrete graphics card, try disabling the integrated display adapter.  Right devices/conflicting devices.  IMPORTANT: Leave your primary graphics adapter and monitor alone.

It's unfortunate the messages that come up don't have an message number to indicate where in the Windows code or which module generated them.   The text "make sure all required file system drivers are loaded" is not much more help, because there is no indication of which driver might be missing.  This gets right up my nose because if this is a general message, it should not be generated, or if it is more specific, the module that detected the error must recognize the driver(s) that it needs and should tell us.  There may be log messages, but I'm not sure they will be any more useful.
