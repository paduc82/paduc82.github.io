APA 6th edition upgrade available?

yves dhondt say:
Hi

Check the folder containing the stylesheets to see if it is there.

In Windows Explorer (My Computer), go to the location where Office is installed. If you don't know where that is, do a search for 'winword.exe'.

 

If you made a default 32-bit installation on a 32-bit OS, the folder will be "C:\Program Files\Microsoft Office\Office14" for Office 2010.

 

In the folder containing Office 2010, there should be a subfolder "Bibliography" which in turn should have a subfolder "Style". If the installation of the SP was successful, the following 4 files should be added to that directory (next to the ones that were there originally):

  • APASIXTHEDITIONOFFICEONLINE.XSL
  • HARVARDANGLIA2008OFFICEONLINE.XSL
  • IEEE2006OFFICEONLINE.XSL
  • MLASEVENTHEDITIONOFFICEONLINE.XSL

 

If they are not there, then something probably went wrong during the installation of the service pack and you might want to try installing it again.

 

If the files are there, then they should show up in Word next time you start it.

 

If you are convinced that the installation of the SP was successful and you can't find the styles, you can always try to install them manually. I made a post on this topic to get the styles available for 2007: http://answers.microsoft.com/en-us/office/forum/office_2007-word/apa-6th-edition-and-mla-7th-edition-in-office-2007/7ec9a800-b406-4a04-b257-c2d7d64f5fc3 . The steps are the same for 2007 and 2010.
