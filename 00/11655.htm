If you look at the terms of the license you'll see that it allows you to have office installed on 3 computers at the same time. Re-installing on the same machine, as you have, still counts as the same "installation". The activation system is able to recognize it is the same computer. 

You are also allowed to "transfer" those installation to new machines. Normally there is a 90 day interval, but transferring from a dead machine is allowed to bypass that time limit.
