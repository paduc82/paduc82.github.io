Sub Test()
  Dim StartC As Range, LastC As Range, This As Range, All As Range

jay freedman say:

You can insert footnotes only in Rich Text content controls. If the content control is of the plain text kind, footnotes cannot be inserted, either manually or by code.

If the empty master report is created from a Word template, then the solution is to change all plain text content controls in that template to rich text content controls (or at least the ones that might have to accept components with footnotes). If the master report is created entirely by code, then the code must be revised to insert rich text content controls instead of plain text ones (that is, specify .Type = wdContentControlRichText, where the constant on the right has the value 0).
