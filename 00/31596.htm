Assuming the latter, do a search for some of them. It may be that just the shortcuts were lost and the files are still on your drive.

If the files themselves were on your desktop, you're probably out of luck. But for the future, two points:

xbr dionysus say:

Wireless Security

Login Information

  • Username - admin
  • Password - telus
  • Gateway - 192.168.1.254 

How To Find Wireless Passphrase (WPA/WEP)

  1. Open a web browser.
  2. Enter http://192.168.1.254 into the address bar. (Or click the link) Enter the username and password if the gateway asks. (admin/telus)
  3. Select Wireless Setup from the top menu.
  4. Select Basic Settings on the left.
  5. The passphrase will be displayed at the bottom of the page in the box that is located in the Security Type section.

 

 

Changing Wireless Security Type/ Disabling Wireless Security

  1. Open a web browser.
  2. Enter http://192.168.1.254 into the address bar. (Or click the link) Enter the username and password if the gateway asks. (admin/telus)
  3. Select Wireless Setup from the top menu.
  4. Select Basic Settings on the left.
  5. In the drop down menu for Security select the desired security type. (To turn off the security, select Off)
  6. Click Apply.
