tony.dowler say: Good news, I managed to find a resolution to this issue. The problem was with the aoddriver2.sys file. I was able to navigate to C:\Program Files\ATI Technologies\ATI.ACE\Fuel\amd64\AODDriver2.sys and rename the driver to AODDriver2.old. I was able to run msoobe again and get the machine to boot.
