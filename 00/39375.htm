castle214 say:

Go to the folder where all the "libraries" are and make sure you are showing all folders - there should be the my documents, favorites, etc but there should be one marked "appdata".  From there go to "local", "Microsoft", "Windows Live Mail" and there are all the accounts and storage folders from Windows live mail - you can drag and drop them onto a flash drive, DVD, CD or whatever external media you've got.

 

No "extra" backup program required!  It's almost the same as the old Outlook Express path, you've just got to make sure the "AppData" folder is visible but that should be easy to display.
