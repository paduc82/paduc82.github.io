arosta---01 say: Hi Everyone, 
I also bought this non-original CA-42 cable, and I had problems installing it. I read this thread and finally I figured it out myself so here is my solution, I hope it will help someone. 

This one regards installation on Windows VISTA, but I guess it will work on XP as well. my cable came in a box that looked like Nokia (fonts like Nokia) but in the manual they wrote "NOK" everywhere instead of "Nokia", the cable turned out to be a "Prolific PL-2303" cable. 

So, first of all, to start up clear unplug the cable, uninstall nokia pc suit and any previous installation of the CA-42 driver and anything related to that. after you done it, its important to reboot your computer. then when windows finished to load up, explore the installation CD and run "PL-2303 vista driver installer" for windows vista, or "Nokia CA-42 USB driver installer" for winXP. 
let it finish and ONLY THEN plug the cable in. if it went right you will see the "found new hardware" notification, and it should identify it automaticly and say something like successfully installed Prolific PL-2303 (COM X) if you use VISTA, or successfully installed Nokia CA-42 cable on XP. 
after that I recommend you to go to "device manager" and see that it looks OK under the category "ports (COM & LTP)", on the same chance see on which port the cable is working, on my computer it installed it on "COM 8". 
now, you would like to install Nokia PC Suite, but don't install the version that came on the CD because it won't work. download the latest version from Nokia and install it. after yo done that you should open Nokia PC Suite and go to settings -> manage connections. there you will see several options - 1)bluetooth, 2)IR, 3)Serial Cable CA-42, 4)USB. make sure the CA-42 option is marked with a "V", then click on it (it should be painted in blue background) then click on the settings button (it looks like a working tool for opening screws) there you shloud choose the serial port you saw the cabel is using in the device manager and click OK, if it wont let you use this port then plug the phone and it will let you choose the right port and recognize the phone.

grandpa jim say:

The Metadata for your initial post indicates that you are using Office 2013 running under WIndows  7. If that is the case, references to a Command key are not relevant.

ALT+F9 does NOT remove links. What is does is toggle between the display of the field codes { } and the result of those fields of ALL of the fields in a document. SHIFT+F9 does the same thing for ONLY the field that is selected.

If you use CTRL+a to select everything in a document, CTRL+SHIFT+F9 will unlink ALL of the fields in the document, converting their results to ordinary text.
