teylyn say:
Hello,

this can easily be done with Index/Match as a combination formula.

If Postcode is in column A and the products are in columns B to Z and the table has 1000 rows, then use this formula. 

=INDEX($B$2:$z$1000,MATCH("2345",$A$2:$A$1000,0),MATCH("Product3",$B$1:$z$1,0))

This formula basically says: look at the values in B2 to Z1000. Return the row where column A has the code 2345. In that row, return the cell where the product name in row 1 is Product3.

In this sample formula, the postcode "2345" and the product "Product3" are hard-coded for illustration purposes. You can use a cell reference instead. So if the table with the rates is on Sheet1 and you want to have Sheet2 show the rate for the postcode and product entered in Sheet2, cells A1 and A2, then the formula would be

=INDEX(Sheet1!$B$2:$z$1000,MATCH(A1,Sheet1!$A$2:$A$1000,0),MATCH(A2,Sheet1!$B$1:$z$1,0))

cheers, teylyn
