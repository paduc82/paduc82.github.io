bernie deitrick say:

Here's one way to use Excel to do what you want.

 

On the sheet with the currnet season's data, insert a new column, labeled Season. Put 2012/13 into cells of that column for every row of data that you have. Then copy the values from the other seasons' sheets, and place those values below the current season in the correct columns (do not copy the headers over). For every row of copied data, put the correct season in the season column. When you capture the new updated data, make sure that you completely overwrite the current season's data.

 

Delete the other worksheets so you have only one with all the data.

 

One option is to then select your data table and create a pivot table.  Play around with the pivot table: put the stats you are interested into the data fields, with player as the row field, and season as the column field. You can filter by top ten, or create other analysis columns....
