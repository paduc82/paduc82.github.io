    As you go down column B the string will get too large - Excel has a limit for text in a cell (think it is 256 characters)

    The answer is to use VBA.

    Here is code that worked form me

    best wishes

    Sub Tryme()
      Dim myFile As String, LastRow As Integer, k As Integer
      myFile = Application.DefaultFilePath & "\testfile.csv"
      Open myFile For Output As #1
      'assuming data to be in column A
      LastRow = Cells(Rows.Count, "A").End(xlUp).Row
      For k = 1 To LastRow
           Write #1, Range("A" & k).Value,
      Next k
      Close #1
    End Sub

    iliru say:

    Hello Andrea,

    I think this that you require, normally should be done with the help of an UDF because as I know Excel not have any formulas / functions to concatenate all the cells and put them in a single cell according to the criteria you are looking for.

    However, there is a simple method that I am explaining to you below. This method has a limitation since it can not collect more than 8192 characters from different cells.

    I suppose your data starts from cell A1 and continue to cell A10000, and also I'm clarifying the fact that in my example I have used decimal numbers. Using decimal numbers gives me the opportunity to select up to 2300 cells in my case. If the numbers used are bigger than decimal numbers then the number of cells involved will need to be lower. So it's always necessary to apply the condition that the characters involved will have to be less than 8192.


    in an empty cell write this:

    = A1:A2300

    do not press Enter but use the F9 key from the keyboard

    after this action, all the numbers contained from cell A1 to cell A2300 will be released. Selected these numbers except the ={ and } and copy them to Notepad.

    Then go back to the Excel page and instead of the selected cells shortly select new cells. For example:

    = A2301:A4600

    and then repeat the above to send these data back to Notepad.

    Note: When dropping new data in Notepad, make sure that at the end of the previous string of numbers you place a comma and space and then drop new data.

    Return to the Excel page, select other cells and resumes previous actions.

    If I assume that you have 10000 cells and can copy up to 2500 cells then the above actions will be done 4 or maybe 5 times.

    After you have paste all the data in the Notepad, from the keyboard press the CTRL+H and in the window that will give you to Find what put ; (only a semicolon) and to Replace with put , (comma and a space behind it). Press the Replace All button.

    After all these actions, the data will be released in the format you want and you can save these data as .txt


    Method II

    Use the CONCATENATE function. In an empty cell, place this formula:

    = CONCATENATE(A1:A10000)

    Instead A1:A10000 use A1:A2300 and press the F9 key on the keypad.

    The continuation is the same as Method I.

    In my example I have used cells from A1 to A10000, but you may have more or fewer cells. So change the cell data according to your case.

    In addition, there is no guarantee that paying the ransom will actually result in the restoration (decryption) of your files. Remember that you are dealing with cybercriminals.

    As with most ransomware infections, the best solution for dealing with encrypted data is to restore from backups. Please note that Locky shall delete all Shadow Volume Copies with vssadmin.exe so that you cannot restore your files via System Restore or using a program like Shadow Explorer... but it never hurts to try, so as to check in case the ransomware failed to do everything it's supposed to do.
