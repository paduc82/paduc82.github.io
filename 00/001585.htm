Which user updated a cell in Excel - Microsoft Community

lizbeef say:

Which user updated a cell in Excel

Hi

 

I would like a fomula in cell G1 that will tell me which user updated cell A1. is this possible?

jllatham say:

Put all of the following code into the worksheet's code module where you want this to be done.  To do that, open the workbook, select the worksheet, right-click on its name tab and choose [View Code] from the list that appears.  Paste the code into the module and save the workbook as a macro enabled workbook, type .xlsm

 

Make changes to cell A1 and you should see the name in G1 change once.

 

This actually gives you the logged in name of the user as opposed to Application.Username which would have been used during the installation of Office/Excel and if you're in an environment where the IT department handles installs, the name used then may not be the actual computer user's name.

 

Here's the code:

 


#If VBA7 Then
Private Declare PtrSafe Function apiGetUserName Lib "advapi32.dll" _
    Alias "GetUserNameA" (ByVal lpBuffer As String, nSize As Long) As Long
#Else
  Private Declare Function apiGetUserName Lib "advapi32.dll" _
    Alias "GetUserNameA" (ByVal lpBuffer As String, nSize As Long) As Long
#End If

Private Sub Worksheet_Change(ByVal Target As Range)
  If Target.Address = "$A$1" Then
    Range("G1") = fOSUserName()
  End If
End Sub
Function fOSUserName() As String
' Returns the network login name

' NOTE: the following source reference with link should all be on 1 line!
'source: https://groups.google.com/forum/?hl=en#!Original/comp.apps.spreadsheets/qOMGQY9vbkM/19V8HhGYGv8J

Dim lngLen As Long, lngX As Long
Dim strUserName As String
    strUserName = String$(254, 0)
    lngLen = 255
    lngX = apiGetUserName(strUserName, lngLen)
    If lngX <> 0 Then
        fOSUserName = Left$(strUserName, lngLen - 1)
    Else
        fOSUserName = ""
    End If
End Function
