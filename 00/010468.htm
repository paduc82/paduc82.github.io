ossiemac say:

Try the following code.

Sub test()
Dim dateToFind As Date
Dim lngYear As Long
Dim lngMth As Long
Dim lngDOW As Long
Dim lngOccur As Long

lngMth = Month(Date)
lngYear = Year(Date)
lngMth = Month(Date)
lngDOW = vbMonday
lngOccur = 1    '1st occurrence

dateToFind = NDow(lngYear, lngMth, lngOccur, lngDOW)

MsgBox dateToFind
End Sub

Public Function NDow(Y As Long, M As Long, _
     N As Long, DOW As Long) As Date
    
'Y is Year, M is Month, N is the occurrence, DOW day of week

NDow = DateSerial(Y, M, (8 - Weekday(DateSerial(Y, M, 1), (DOW + 1) Mod 8)) + ((N - 1) * 7))

End Function

Using Countif to check for dupes in dynamic range - Microsoft Community

jw aldridge say:

Using Countif to check for dupes in dynamic range

Headers are in row 5. As I add data in table, I am checking the value on the current row (o6) to see if there was an entry made already with the same value.  

I am using this is in p6.

=COUNTIF(O2:O6,O6)

Instead of the last four, I need to dynamically check the last 50 rows/entries in column O. 

ossiemac say:

Let me see if I am interpreting your question correctly. You place a formula in column P at row 6 and you will be copying it down to match the number of rows of data in column O. If a value entered in column O has already been used then the value in column P will be greater than 1. Is this correct?

If above is correct then the formula as follows by making the first row of the range absolute (with the $ sign).

=COUNTIF(O$2:O6,O6)

Note that you could use conditional formatting to highlight the cells in a color if a value is used multiple times as follows.

  1. Select the entire column O
  2. Select Conditional formatting (On Home ribbon)
  3. Select New rule
  4. Select "Use a formula to determine which cells to format"
  5. Enter the following formula in the field:        =COUNTIF($O:$O,O1)>1
  6. Select format button
  7. Select Fill tab (at top of dialog)
  8. Select a color such as red.
  9. Click OK and OK again.
  10. Now all cells that have duplicate entries will be red.

amit tandon say:

Enter below formula in cell P6 & copy down:

=COUNTIF(OFFSET($O$5,0,0,MIN(ROW()-ROW($O$5),50)),$O6)

This formula checks if entry in cell O6 is already present in range $O$5:$O5 & returns the number of times it is present. When copied to cell P7, the formula will determine if the entry in cell O7 is already present in range $O$5:$O6. When copied to cell P8, the formula will determine if the entry in cell O8 is already present in range $O$5:$O7.

On reaching row 55 (P55), the the formula will determine if the entry in cell O55 is already present in range $O$5:$O54 ie. in 50 cells above. After this, in row 56 onwards, the formula checks 50 cells above in column O.

Hope your query has been correctly understood. You may update the range per your requirement.
