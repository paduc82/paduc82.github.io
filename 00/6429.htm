joeu2004 say:

Without showing us the formulas and any input numbers, we probably cannot help you.

If you format the cell as Number with 4 decimal places and use a calculation of the form ROUNDDOWN(expression,4), the result should be rounded down to 4 decimal places, as desired.

However, there are "corner cases" (read: unlikely) where even what you might expect to be 0.989999999999999 is "converted" to 0.9900.  That is because that actual value is 0.989999999999999+5.55E-16 or greater, but Excel formats only the first 15 significant digits,  rounded.

charleskenyon (ia) say:

Hi

You could actually round or truncate to four digits.

When you do this, any digits greater than 4 disappear. If you just change the number of digits displayed, the digits are still there. They would be used in any calculations based on the number.

If you base a calculation on a rounded or truncated number, you use the number of the digits allowed by the formula.
