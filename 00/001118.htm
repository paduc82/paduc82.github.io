AFAIK you are correct in saying that VBA is does not support multi-threading.

Now - it could be possible to install a second licensed copy of Excel on the same physical hardware - but in order to use it separately you'd need to set up a Virtual Machine: and that means not only getting an added license for Excel/Office but also another for the operating system to use on the VM.

In my opinion the most robust VM hosting software is probably from VMWare.  They offer several options for a desktop system:

VMWare Workstation - again not free, but an outstanding product (I have a couple of machines using it to create and manage VMs).

jay freedman say:
In a minute I'll tell you how to uninstall Word if that's what you really want to do, but first let me tell you why it probably won't do any good if you then reinstall it.

Most problems described as "Word stopped working" arise from one of three sources: add-ins that are damaged or meant for a different Word version; a damaged Normal.dotm template; or damaged or missing registry entries. Uninstalling and then reinstalling Word does nothing to change any of these. After the reinstallation, the same problem will still be there.

Instead, use the procedures in http://support.microsoft.com/kb/921541 to find the cause and fix it.

If you're just determined to remove Word, here are the steps:
  • In the Control Panel, go to Add and Remove Programs.
  • Select the Office 2010 item and click the Change button.
  • Accept the "Add or Remove Features" option and click Continue.
  • In the list of features, click Microsoft Word and choose "Not Available".
  • Click the Continue button and answer any prompts.
