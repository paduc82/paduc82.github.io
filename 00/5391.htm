Word has a very specific design.

One of the key features Word is designed around is "Styles".  It comes with 9 predefined "Heading styles". You can modify these styles to look any way you wish.  These 9 styles are the ones "automagically" used to generate TOCs, with up to 9 levels.

Yes, the default is to use only 3 levels in the TOC, but you can adjust that to up to 9 levels.

Yes, there is an option to manually mark text using "field codes", but as you surmise, that is a PITA that you only want to fall back to in very specific circumstances.

jay freedman say:

It is not necessary to use the Heading 1 - 9 styles or to redefine your custom styles to make the table of contents work. You need only two steps:

  • For each of your custom styles, open the Modify dialog, click the Format button in the bottom left, and click Paragraph in the menu. In the Paragraph dialog, change the Outline Level dropdown to the level that you want the TOC to show for that style.

  • If you're using the Custom Table of Contents menu item on the Table of Contents button, that dialog lets you choose how many levels you want to show. If you're using one of the Automatic tables, press Alt+F9 to display field codes; edit the field code from { TOC \o "1-3" \h \z \u } to { TOC \o "1-9" \h \z \u } (or whatever range of outline levels you want). Then press Alt+F9 again and update the TOC.

Sometimes you have to open the template itself and make the modifications there.

If you want to do a restore, verify there is a backup at OneDrive > settings (cogwheel) > options > Device backups

If there is not, go to settings > backup on the phone and do one. Media files in public storage need to be backed up manually through USB (using MTP), best is to attach to USB on a PC and just copy the files to a folder on your PC.
