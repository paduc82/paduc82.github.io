Microsoft Word stopped working - Microsoft Community

diane poremsky mvp (slipstick.com) say:

Microsoft Word stopped working

When Microsoft Word (or other Office applications) is not working, hanging, or crashing, check the Fixes or workarounds for recent issues in Word for Windows list new issues then try the steps below. 

How to troubleshoot problems that occur when you start or use Word has additional troubleshooting methods for problems that occur when you use Microsoft Word.

Step 1: Open Word in Safe mode.

The first troubleshooting step is to see if Word works in Safe mode. If it works in Safe mode a COM add-in may be causing problems. 

  1. Close Word.
  2. Press the Windows key+R on the keyboard to open Run window.
  3. In Run window and type winword /safe (There is a space before the /).
  4. Press Enter or click OK to start Word in Safe mode. 

If Word works ok in Safe mode, disable all COM Add-ins and restart Word. Re-enable the add-ins one a time to identify the culprit. 

  1. Click on File > Options > Add-ins.
  2. Under Manage select COM Add-ins.
  3. Click on GO beside Manage, uncheck all the add-ins and click Ok.
  4. Close the dialog and Word. Reopen Word and check if it works.

Repeat Steps 1 - 3 to enable the add-ins, one at a time. 

See I get a "stopped working" error when I start Office applications on my PC for a fix it tool to disable the add-ins automatically. 

Note: If one of the add-ins is the Office Compatibility Pack, uninstall it from your computer.  Open Control Panel > Add/Remove Programs. 

Step 2: Repair Office installation.

If Word doesn't work in Safe mode, try repairing your Office installation.

  1. Right click on the start menu and choose Apps & Features
  2. Find and select the Office installation.
  3. Choose Modify then select the Quick repair option. Use the Quick Repair option first as it will fix many things and complete fairly quickly. 

If Quick Repair doesn't help, use the Online Repair option to uninstall Office and reinstall Office. It will not delete your existing Office settings or your Outlook profile and data files. 

For more information, see  Repair an Office 2016 application. The article includes an "easy fix tool" that will wipe Office from your computer, including all folders and registry entries. This utility will delete your Outlook profile and data files, custom templates, and other files stored in the C:\Users\%username%\AppData\Local and Roaming paths.
