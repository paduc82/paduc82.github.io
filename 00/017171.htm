How to change number in a cell to phone number format

bobgreenblatt say: The problem is that Excel's format only governs the way numbers are displayed. Since your list has various configuration of numbers, the ones that are not numbers are interpreted as text by Excel. (These are usually right adjusted in the cell as you have noted.) You will need to undergo several manual steps to remove the unwanted punctuation in the cells. do this by putting a formula in an adjacent column such as: =substitute(A1,"-","") This will produce a string from A1 with no hyphens. You can fill this down for as long as needed. you may then have do this again (in anopther adjacent column) to remove spaces, and again for parens, etc.

When you end up with a "clean" column of only numbers, copy it and then Paste special values to yet another column. Then, enter a 1 in a spare cell somewhere, and copy theat cell. then select you new clean numbers and paste-special-multiply. This will force any text values to numbers. You can then apply a number format to get them to appear the way you want. Lastly, copy these and past back into the original column. You can then delete all the intermediate columns.
