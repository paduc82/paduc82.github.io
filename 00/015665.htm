For the OneDrive desktop application, if you are using the new sync client of OneDrive for Business (OneDrive.exe) , it can only sync the default library in our OneDrive for Business. Moreover, if you already use the new sync client, you need to ask your tenant admin to configure the following settings:
1. Set up the SharePoint Online Management Shell environment.
In step 4 of the article, we need to run the following command to connect to the SharePoint Online admin center:
Connect-SPOService -Url https://contoso-admin.sharepoint.com -credential admin@contoso.onmicrosoft.com

Set-SPOTenantSyncClientRestriction -GrooveBlockOption "OptOut"

Windows 10 (Powerpoint Specs) - Microsoft Community

corywoods say:

Windows 10 (Powerpoint Specs)

What is the size of the newest powerpoint screen/pixels or dimensions? ie: 1350 x 750 etc 

 

steve rindsberg say:

PowerPoint doesn't really think in pixels.  Depending on what locale you have your system set to, it works in inches or centimeters unless you're using VBA or other programming languages, in which case you work in Points (72 points to the inch).

The default page size is 13.33 x 7.5 inches.

When you view the presentation in slide show view, it fills the screen, assuming that the screen and the page size have the same aspect ratio.

On a really old video projector that only goes up to 800x600 pixels (yes, they still exist), that'd mean the slide is 800 pixels wide.  On a modern high resolution laptop display, you might see almost 4000 pixels on screen (from the same presentation).

That said, if you really need to work in pixels, you can enter them in at least some of the sizing dialog boxes.  Insert a shape, for example, go to the size dialog box and you can enter 96px.  On most systems, PPT will translate that into 1" since most Windows systems operate at kind of an imaginary 96dpi.  Whether that has any real-world value or not ... an exercise for the reader. ;-)
