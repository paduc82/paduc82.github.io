Yes, there is:

  • Select System Restore when prompted.

¡firedog say:

It might be an idea to review all your account settings. These are suggestions only; you may find on experimentation that different settings work better for you. As these instructions were written for an ordinary Gmail account, there may be some minor differences in what you see in webmail. 

 

In webmail,
  

  • Sign into your account at mail.google.com and select Settings.
    1. Forwarding and POP/IMAP, select Auto-Expunge Off
    2. they were separate folders, so fewer is usually better to avoid duplication. In particular, you don't want All mail (which is just that: all messages in all folders). I have opted to hide Starred and Important also.
    3. Sent mail, Drafts, Spam and Bin.
    4. account security page and check that you allow Windows Live Mail (a 'less secure app') to access your Gmail account. Despite what you might think, this setting does not compromise your email security. It's still protected by the same TLS method as Google uses to authenticate users.

 

In Windows Live Mail,
  


  1. Right-click on the account name in the folder pane and select Show or hide folders to open the Show/hide IMAP folders dialogue. Click Reset List to refresh the folder list, then use the buttons to show only the folders you want to see.
     

  2. Right-click on the account name in the folder pane and select Properties. On the IMAP tab, select Store special folders on IMAP server, then check that the Gmail folders are listed with the names you noted earlier with the prefix [Gmail]\
     
  3. Deleted items folders work the same way for all accounts
    When viewing your message list, press Ctrl-Shift-O for Options. On the     Advanced tab, select Use the 'Deleted Items' folder for IMAP accounts.
     
  4. When viewing your message list, press Ctrl-Shift-O for Options. On the Send tab, deselect Save copy of sent messages in the 'Sent Items' folder, unless you have other accounts which don't synchronize folders with the server, i.e. that use POP3.
