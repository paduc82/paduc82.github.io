Header date formatting

charles kenyon say:
Format the date in your letter in a character style called Date. Use a StyleRef field in the header to reflect that date.

Ideally you have a letter template which has a CreateDate field in it in the body and a header set up to show that date on the second and subsequent page using a StyleRef field.

That way when you write the letter the date will be the date on which the letter is created. It will stay that date even if you open the letter later to print another copy. You can type over it and change it if needed. The date in the header will be whatever is typed in the Date style in the body of the letter.

You may want to look at the tutorial on StyleRef fields and letters. It has samples.

ossiemac say:

Secondly, I want to be able to temporarily lock those numbers when applying different ordering.  How to please?

The only way you can do above on a temporary basis is to remove the formulas by using Copy -> Paste Special -> Values over top of them selves and then re-insert the formulas again when they required

However, on the first part of your question. Select the column of numbers then select menu item Insert -> Table. If you don't like the banded rows then while any cell in the table is selected you will see a ribbon icon at the very top called "Table tools". Select Table tools ribbon and then in the "Table styles" block click the down pointing triangle with the horizontal bar on top to expand the available styles and select the first one at top left.

In the first cell of the table insert the following formula and copy down. (The negative value is dependent on the row that you start on so that you get a result of one for the first cell of the range.)

=ROW()-1

Now when you insert a row the table grows and inserts the formula in the new cell and updates all of the cells below.
