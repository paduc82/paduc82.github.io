cmdrkeene say:

The Users folder has always shown your username as the folder name (going back to Windows 95 and Windows NT).  It has never shown your PC name under the Users folder (if it did, then you just used your computer name as your username).


To change your computer name, use the System Dialog (this is also been the same since Windows 95).  A quick way to access this is +R to show the Run box, then run sysdm.cpl to show the System Properties dialog.




While the username you use to login to Windows can be changed, as well as the display name as it appears in various places (like the login screen), the folder inside C:\Users (formerly c:\documents and settings\) cannot be changed.  This isn't new, this also goes back to Windows 95.


The only supported way to change that folder is to create an account with the name you want. The username is used to name the folder (just like always).
