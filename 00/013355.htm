Is it just stored as part of the OneNote file or is it saved in this location (C:\Users\username\AppData\Local\Microsoft\OneNote\14.0\OneNoteOfflineCache_Files) as an actual excel file with some crazy long alphaneumerica name and then it is just referenced by the actual OneNote file whic is stored at C:\Users\username\Documents\OneNote Notebooks\Personal stuff..

Embedded files are part of the section (*.one) files. If you open one of these files from Onenote a temporary copy is created in the OfflineCache_Files folder to hand it over to the corresponding application. A changed file is copied back to the section file. The cached copy is not immediately deleted, but subject to some sort of garbage collection.

Because these embedded files blow up the section files and result in a performance penalty (two copies for each change, no differential update possible) you could/should prefer hyperlinks to a central folder, as you mentioned.

But this will only work within a LAN, not in the cloud aka Skydrive.
