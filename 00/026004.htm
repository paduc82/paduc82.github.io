Directory in Word with pagination/line separation issues

I'm creating a directory using mail merge with an existing Excel file. In the resulting directory, I need whole records to stay together, and not be split across a column or a page. I've selected 'keep with next' and 'keep lines together' and 'widow/orphan control' in various combinations, but the resulting file is still splitting records across the column, so that the name and phone number are at the bottom of a column, and the email and address are at the top of the next. Any ideas for me? I have not had this problem in earlier versions of Word.
suzanne s. barnhill say:

If each address block is a single paragraph, then you want just "Keep lines together," not "Keep with next." And don't even try to use the Address Block feature in Mail Merge; instead, insert your merge fields one at a time, using line breaks (Shift+Enter) at the end of each line of the address block and a paragraph break (Enter) only at the end.

If you do have paragraph breaks within the address block, then you need "Keep lines together" on all the paragraphs and "Keep with next" on all but the last.
