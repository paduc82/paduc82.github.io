Hello, I need a formula to cause the contents of cell A1 to become strikethrough based on the value of cell A2. For example, if the value of A2 is "open" no change to A1, if the value of A2 is "completed," then cause the text in A1 to be strikethrough. lars-Åke aspelin say:

Hello, I need a formula to cause the contents of cell A1 to become strikethrough based on the value of cell A2. For example, if the value of A2 is "open" no change to A1, if the value of A2 is "completed," then cause the text in A1 to be strikethrough. Also, if I can cause cell color of font color to change, that would be very helpful

Thanks in advance

Try conditional formatting of cell A1.

This is how to do it in Excel 2007, maybe close enough to get you on track for Excel 2010.

Select cell A1.

On the Home Tab / Styles section, click on Conditional formatting and then New Rule...

In the Select a Rule Type , choose Use a formula to determine which cells to format

In the Format values where this formula is true, enter the following formula:

=$A$2="completed"

Click on Format...

Check the box Strikethrough in the Font tab and choose any other formatting like Fill color etc that you want.

OK yourself out.

Note: If you want the same conditional formatting on several cells and the rule is not to look at cell A2 but on "the cell below", just omit the $'s in the formula.

Hope this helps / Lars-Åke
