davem121 say:
Hi, I am a software developer, most Windows Applications / Apps are written in one of the C languages (C, C++, C#). The reason being faster execution of code, since these programs are compiled into an exe (executable) file.

Java programs are not compiled, they use a system called just-in-time (JIT), which means when you perform an action (pressing a button . . . etc.) that code is then sent to the JRE to be run.

When writing software, The user interface - the components used on the Window (Form) are pulled form a library and a lot of the Java Libraries are not written to scale properly with Windows 10 DPI Scaling. Whereas a lot of modern Windows apps written in say C# use a language called XAML to describe the user interface and the resulting UI scales perfectly . . .

The Windows.old folder contains your old Windows installation after an upgrade. If you delete it, you can't go back to your old OS if you don't like Windows 10.

3. Windows will analyze the drive. Be patient as this can take a few minutes.

4. After Windows analyzes the drive, a window will open that looks similar to this:

5. Click Clean up System Files, and provide confirmation if necessary. 

6. Windows will re-analyze the drive. When it's done and the window opens again, find and check the box labeled Previous Windows Installations. Then click OK.

Windows will now delete the Windows.old folder.

a. user say:

Hi Jimenez,

 

Thank you for posting your query in Microsoft Community.

 

I understand the inconvenience you are facing. Let me help you in resolving the issue.

I suggest you to follow the steps mentioned below and check if it helps:

  • Press the Windows key on the keyboard.

  • Click on Settings, then click on Time & language option.

  • Click Region & language on the top left of the screen.

  • Click on Windows Display language to expand, and click on Options.

  • Select the Keyboard layout, and click on Remove.

If you wish to uninstall Language Packs then try the steps below and check if that helps:

  • Open a command prompt windows, type the following and hit Enter.
  • Lpksetup /u
  • The Install or Uninstall display languages panel will open.
  • Select the language, click on Next and the Language Interface Pack will start uninstalling. You will then be asked to restart your computer.

Keep us posted on Windows related queries and we will be happy to assist you further.
