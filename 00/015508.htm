davetomato say:

Add Character Spacing to the Ribbon

Hi, in Word 2010 I would like to add the Character Spacing commands (expanded / condensed) to a custom ribbon or quick access toolbar, but can't find these commands in "All Commands" - where would I look?

Furthermore, is there a way to search all the commands? There are hundreds!

Cheers, Dave

jay freedman say:

Considering the hundreds of commands that Word does contain, it's hard to believe -- but true -- that there are tons of things you can do that don't have a dedicated command. This is one of them.

Often (but not always) the way to make a complex action into single button-press is to write a macro, and then add a button to run the macro.

If you have a specific amount of expansion or compression that you use most often, you can use a macro like this, which sets the expansion of the selected text to 0.2 point:

Sub Expand_02pt()
    Selection.Font.Spacing = 0.2
    Application.ScreenRefresh
End Sub

You could perhaps have two or three of these macros, one for each amount you prefer. (For compression, put a minus sign in front of the number.) That's probably a bad idea, though. Instead, you could have the macro ask for the amount, like this:

Sub Expand_Compress()
    Dim amt As Single
    On Error GoTo done
    amt = InputBox(prompt:="Enter number of points (negative for compression):", _
        Title:="Expand/Compress Selected Text", Default:="0.1")
    If IsNumeric(amt) Then
        Selection.Font.Spacing = CSng(amt)
        Application.ScreenRefresh
    End If
done:
End Sub

Another possibility is simply to open the Font dialog, but have the macro tell it to open to the Advanced tab:


Sub ShowFontAdvanced()
    With Dialogs(wdDialogFormatFont)
        .DefaultTab = wdDialogFormatFontTabCharacterSpacing
        .Show
    End With
End Sub
