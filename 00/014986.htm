Sounds like in your case it may be delivered instead via an advertising network.  This is common for the more often seen Tech Support browser hijacks, but since this item also appears to be nothing more than a browser popup, it could be delivered any way that a browser might be launched.

Do you have a browser open on a website when this popup appears, especially one that might already display more typical ads within its pages?  If so, this is how the Tech Support popups are most often delivered and can typically be reduced, if not stopped by using an ad blocker within whatever browser is being used.

If no browser is currently known to be open when this occurs, then the possibility that something installed locally is causing it seems more likely, which could include browser components like plug-ins.

The first thing to determine is which browser is involved if more than one is installed, since this helps identify where to start looking for how the popups are being injected.  Then try removing or disabling any plug-ins that might be suspect.

Personally I avoid all 3rd-party browsers like the plague they are, since they're typically difficult to protect properly without lots of research and add-ons.

I only run the built-in Microsoft browsers, which includes both Edge and IE11 on Windows 10, with IE11 configured for high-security settings including Send do not track requests and Tracking protection lists from EasyPrivacy and Stop Google Tracking, which shuts down most ad networks in that browser.
