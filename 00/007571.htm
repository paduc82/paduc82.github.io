andreas killer say:

Why not copy the template itself?

Please never use SELECT, SELECTION, ACTIVECELL, it is slow and error prone. Always refer to the objects directly.

Andreas.

Sub Test()
  Dim Ws As Worksheet
  Dim NewName

  'Write A1 into G17
  Range("G17") = Range("A1")
  'Fill G17:G18 down
  Range("G17").AutoFill Destination:=Range("G17:G18"), Type:=xlFillDefault
  'Get the new name
  NewName = Range("G18")

  'Copy the template to the end
  Sheets("Month Template").Copy After:=Sheets(Sheets.Count)
  'Refer to that sheet
  Set Ws = Sheets(Sheets.Count)
  'Rename the new worksheet
  Ws.Name = NewName
  Ws.Range("A17") = NewName
End Sub
