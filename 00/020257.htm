a_user_here say:

There in fact may not be any way for you to do this without reinstalling Windows 7, and in fact you might not be able to do that either.  When you buy a laptop the manufacture puts an OEM license version of Windows on the machine.  That license is tied to that hardware.  That is the deal, the manufacture gets Windows at a reduced price provided it is tied to one computer.  When that hardware goes away, so does the license.

A person buying a "Retail" version of Windows (like on a DVD) has the right to transfer it to other machines.  It is possible for a manufacture to include a Retail version of Windows, but highly unlikely.

Starting with build 10565 it is possible to active using the Windows 7 product key (before that it would have to be an update or a Windows 10 product key), but only if it is considered a valid product key.

Most likely even if you tried to use that Windows 7 disk image/product key on another machine it wouldn't activate either.
