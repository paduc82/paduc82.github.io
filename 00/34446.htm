6) Wait until the WiFi light on the router is on; you may or may not use WiFi, but it's still a good indicator your router is finished booting up.

Here is a simple explanation of what is happening.

 

When working with streaming audio or video that uses interrupts, DPCs (Deferred Procedure Calls) are used to process the audio in each buffer as they stream in. If another DPC (from a poorly written driver or app) takes too long and another interrupt generates a new buffer of data, before the first one can be processed, a drop-out results.

 

Perform the following.

 

Open File Explorer and navigate to the following folder.

C:\Windows\Minidump.

 

In the folder locate the latest .dmp file that was created when the error occurred.

 

The file(s) will be in the following format.

031413-11419-01.dmp

The first group of numbers designates the date that the error occurred and the file was created (in this example March 14, 2013).
