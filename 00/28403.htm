You should be able to use worksheet_calculate. Right click your sheet tab, view code and paste this in on the right

Private Sub Worksheet_Calculate()
Dim c As Range
Application.EnableEvents = False
For Each c In Intersect(Columns(2), ActiveSheet.UsedRange)
If Len(c.Value) > 0 And c.Value = 0 Then

Next

End Sub

mike h.. say:

Hi,

This could just as easily be done with conditional formatting but you wanted code so try this. Red is VBRed but there is no color constant for orange so use 49407

Sub somesub()
Dim lastrow As Long, c As Range
Dim MyRange As Range
lastrow = Columns("X:Y").Find(What:="*", After:=[X1], _
SearchOrder:=xlByRows, _
SearchDirection:=xlPrevious).Row
Set MyRange = Range("X1:Y" & lastrow)
For Each c In MyRange
    If c.Value < 0 Then
    c.Interior.Color = vbYellow
    'vbred
    '49407 for Orange
    Else
    c.Interior.Color = xlNone
    End If
Next
End Sub

mohan suryanarayan say:

Hello,

Welcome to Microsoft Community.

I am sorry to know that your computer was stolen on which Office 2013 had been installed.

To blacklist Office 2013 from the stolen computer and to reinstall Office 2013 on the new computer, you may deactivate Office 2013 from the My Account page. copy of Office 2013 with your Microsoft Account? If yes, then you can login to your My Account page (URL for My Account page: http://office.com/myaccount) with your Microsoft Account credentials (User name and password used for registering/redeeming your copy of Office 2013) and to deactivate Office 2013 on the stolen computer follow these steps:

On the My Account page check below and you will see the computers on which you have installed Office 2013. Deactivate the compute by clicking on the Deactivate button next to the name of the computer (the stolen computer).

To install Office 2013 on the new computer click on the Install or Download button on the same page (My Account page).

If the above steps do not work then you will have to contact Microsoft Commerce for further assistance.

You may contact Microsoft Commerce by referring to the following link:

http://commerce.microsoft.com/help

In the help page, under Getting help click on Contact Customer Support. Select a Product, Problem type and Category. Then either click on Call Me or Start Chat for more help on this issue.

Hope this information helps you. Please let us know if you need additional information.

Thank you.
