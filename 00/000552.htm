Sum ASCII values of letters - Microsoft Community

hsg1 say:

Sum ASCII values of letters

Is there any way to sum ASCII codes of individual letters
of a word existing in a cell? For example, A1 has "ALEX"
and B1 should have "298" [65(A)+76(L)+69(E)+88(X)].
Is it possible to realize this with a formula?
chip pearson say:

Assume your word in in cell C1. The following array formula will return the sum of the ascii values of the characters in C1.

=SUM(CODE(MID(C1,ROW(INDIRECT("1:"&LEN(C1))),1)))

So if C1 contains ABC, the result is 198, the sum of 65+66+67.

This is an array formula, so you MUST press CTRL SHIFT ENTER rather than just ENTER when you first enter the formula and whenever you edit it later. If you do this properly, Excel will display the formula in the formula bar enclosed in curly braces { }. You don't enter the braces -- Excel does it automatically. The formual will NOT work correctly if you do not use CTRL SHIFT ENTER. Seewww.cpearson.com/Excel/ArrayFormulas.aspx for an in-depth disucussion of array formulas and how to use them.

Cordially, Chip Pearson Microsoft MVP, Excel Pearson Software Consulting, LLC www.cpearson.com

dave peterson say:

You could use =sumproduct() and avoid the ctrl-shift-enter stuff:

=SUMPRODUCT(CODE(MID(A1,ROW(INDIRECT("1:"&LEN(A1))),1)))

Or maybe with a little checking:

=IF(A1="",0,SUMPRODUCT(CODE(MID(A1,ROW(INDIRECT("1:"&LEN(A1))),1))))
