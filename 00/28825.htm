I can not give you a complete example, because that's one of the things that I make my living.

But the concept is quite simple:

Within the template sheet create named cells, but the names must refer to the template sheet, not the Workbook!

The named cells should reflect the heading of your data sheet, means the simplest possible scenario is that you have a named range "Test" in the template sheet and a heading "Test" in your data sheet.

After you've copied the template sheet, visit each cell in the headings in the data sheet and try to access the named range in the copied sheet (use an error handler to handle not exists names).

If you found a named range, use INTERSECT and get the value from the column with the heading and e.g. the current row => Value = Intersect(ThisHeader, ActiveCell.EntireRow) and write it into.

t. valko say:

What does this do?

 

=AVERAGEIF(D2:D48,OR("=10","=12"),G2:G48)

 


It returns an error because it's not a logical formula.

If you want to average G2:G48 if D2:D48 = 10 or 12...

Try this array formula**:

=AVERAGE(IF(ISNUMBER(MATCH(D2:D48,{10,12},0)),G2:G48))

** array formulas need to be entered using the key combination of CTRL,SHIFT,ENTER (not just ENTER). Hold down both the CTRL key and the SHIFT key then hit ENTER.

Open Outlook in safe mode and check if reminder window flashes in safe mode too:

·         In Run window and type outlook /safe

Note: There is a space between 'outlook' and '/'.

·         In Outlook, click on File< Options< Add-ins.

·         Under Manage select COM Add-ins.

·         Click on GO options beside Manage, uncheck all the add-ins and click Ok.

·         Try to open Outlook and check if it works fine.

 

If the issue occurs in safe mode too then run the command mentioned below which will remove the faulty reminder and check if Outlook works fine.:

·         Close Outlook.

·         Click on Start< All Programs< Accessories< Run.

·         In the Run window type Outlook /cleanreminder and click Ok.

·         Restart Outlook and check if it works fine.
