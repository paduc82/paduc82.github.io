'Ease of Access' in Windows refers to various settings that allow those who might be visually, hearing, or otherwise impaired to use a computer. It's not about workflow.

If I read you post correctly, you want to set default folder views to your liking. This is accomplished in Explorer by setting a folder's view to your preferred settings, then clicking the 'View' tab -> Options -> Change folder and search options' -> View tab -> Apply to folders.

You will not the setting has the wording "...folders of this type...". This refers to the primary folder templates of General, Documents, Music, Pictures, and Video. THerefore, you need to perform this action once for a folder of each type.
