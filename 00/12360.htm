jay freedman say:
As a first attempt, each of the Office programs lets you set the number of files you want to show in the Recent Documents list. In Office 2003 programs, click Tools > Options and go to the General tab; or in Office 2010, click File > Options > Advanced and scroll down to the Display section. In each of those places you'll find a box for "number of Recent Documents", and you can set that to zero.

MSPaint does not have an Options dialog, and there is no setting you can change for recent documents in that program in either Windows XP or Windows 7.

None of this is permanent in the sense of being irreversible -- any user can change the value back to a nonzero number, and the recent document list will start accumulating again.

However, the real solution to this question is to create a separate login for each user of the computer -- each with their own user name and password. When you do that, no user will be able to see any other user's Recent Documents lists, because they'll be stored in separate areas of the registry. Only an Administrator will even be able to look at another user's lists. Setting this up will probably require reinstalling the Office suites, though, because the installer has to be told to make the software available to all users, not just the one account being used to do the installation. (That's an option you choose early in the installation wizard.)

barb reinhardt say:

Try this.  Make sure you are on the active sheet when you do this. 

Option Explicit

Sub FindJeffersonAndFillDown()
Dim myWS As Excel.Worksheet
Dim i As Long
Dim myRange As Excel.Range
Dim myCell As Excel.Range
Dim lRow As Long
Set myWS = ActiveSheet

lRow = myWS.Cells(myWS.Rows.Count, 6).End(xlUp).Row

For i = lRow To 1 Step -1
    Set myCell = myWS.Cells(i, 6)
    If LCase(myCell.Text) = "jefferson" Then
        Set myRange = myCell.Offset(1, 0).EntireRow
        myRange.Insert Shift:=xlDown, CopyOrigin:=xlFormatFromLeftOrAbove
        myRange.Offset(-1, 0).FillDown
    End If

Next i
MsgBox ("Complete")

End Sub

I think I'd replace this:

 

 If Format(myCell.Text) Like "AL-JEFFERSON CO*" Then

with this

 If lcase(myCell.Text) Like "al*" and _

    lcase(mycell.text) like "*jefferson*" and _

    lcase(mycell.text) like "*co*" Then

This looks for something

  • Starting with "al"
  • Containing "jefferson"
  • Containing "co"

IIRC, there are a couple of different variations of a dash that may appear the same, but aren't recognized as the same during comparisons.

Barb Reinhardt
