julie fka profjulie say:

Resco Audio Recorder is a nice piece of software, but you may not need to purchase or install any additional software to record phone calls.  On the HTC Touch Diamond2, I am able to record phone calls with the software that is already on the device.

All you have to do is start recording, using the Voice Recorder software that is on the device, either before you place the call or after you dial the number.  Then the conversation gets recorded.  If you record phone calls often, you might want to assign a programmable button to the Voice Recorder software, so that when it is pressed, you can start recording without fiddling around much with the phone.  You can find the Voice Recorder software by tapping Start > Voice Recorder.
