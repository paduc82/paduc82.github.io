mike hall... always say:


This is how the recovery partition is accessed for the most popular makes..


For Dell, press CTRL + F11 directly after powering up the unit

For HP, press F11 directly after powering up the unit

For LG, press F11 directly after powering up the unit

For Toshiba, press and hold '0' BEFORE and during the power up

For Acer, press and hold ALT + F10 as soon as you see the logo

For Asus, press F9 as soon as you see the Asus logo.

For Samsung, press F4 at power up..

For Fujitsu,  press the F8 key repeatedly directly after powering up

For Advent, restart your computer. Then Press F10 repeatedly until the message "Starting System Recovery" is displayed

For Sony VAIO, restart and press "F8" or "F10" repeatedly until the "Advanced Boot Options" screen appears.

 

You may have to try more than once to get any of the above to work. If after repeated attempts, you have no luck or the machine reports that there is no recovery partition, your only option is to contact the computer manufacturer and ask for recovery disks for the model..

 

After the reset, you should get all updates from Windows Update EXCEPT for hardware devices. Next, you should install a good anti-virus solution. ONLY AFTER you have done this should you replace stuff from backups or install anything else.

a. user say:

That's an interesting problem and I could not find a way to disable it like the others you have already done.

When you right click something and drag it Windows looks at the number of pixels the mouse has moved to decide to offer that copy/move/shortcut menu. 

Have you ever noticed that when you are holding down Ctrl to select a bunch of files suddenly they are all copied? 

What happened is the mouse pointer was dragged (accidentally of course) a certain number of pixels to signal windows that that copy/move menu should be used so it made copies on the next click.

I don't think you can turn that feature off but you can change it so you have to move the cursor a really long way before it pops up that menu.


A bad side effect is that it affects all operations where you click and hold to drag something but you can see how this works and then you can undo it easily if you need to.

The default value on my system seemed to be 4 pixels (not much) so you can increase it to a really big number like 1000 pixels which for me means i have to drag way to the right side of my screen before I see that menu - hopefully your user won't figure that out.

The adjustment is in the registry and since there is no quit without saving or undo options in the registry editor be sure to make a System Restore point and export the registry key before making adjustments.

Click the Start orb/button and in the box enter:

regedit

Then choose to run regedit.exe as Administrator:

Navigate to here and make a note of the current values of DragHeight and DragWidth (mine was 4) then change to some big number like 1000:

Since this is a per user setting a logoff/logon cycle will be required but I always like to restart the system just to be sure the changes stick.

For me that means I have to right click (hold) and drag more than half way across the screen for that menu to pop up.

Adjust those values to whatever works for you.
