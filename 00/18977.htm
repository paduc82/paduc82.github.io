joeu2004 say:

Robert wrote:

finding NPV
Initial investment =$100,000.00
Length (in years)= 5
Annual cash flow= 35,000
cost of capital= 8%
I have been trying for hours to figure this out on my own.

 

There are many ways to set this up.  The most general is:

1. Enter the initial investment into A1 as -100000.  It is negative because it is an outflow.

2. Enter the net cash flows into A2:A6 as 35000.  They are positive because they are inflows.

3. The NPV can be calculated as follows:

=NPV(8%,A2:A6)+A1

formatted as General, Number, Currency or Accounting.

Some gotchas to be aware of:

1. If the cell is formatted as General to begin with, Excel chooses to format the cell as Percentage after entering the formula.  All you need to do is change the format after entering the formula.  Alternatively, format the cell as Number, Currency or Accounting (not General) before entering the formula.

2. Note that A1 is added to NPV.  We cannot write NPV(8%,A1:A6) as we might expect because of the odd way that Excel (actually, probably Lotus 1-2-3 or VisiCalc before it) defines the function.  If you look at the help page, you will see that the discounting starts with i=1 instead of i=0 as NPV is normally defined.

PS:  Since the net cash flows are equal (35000), we could use the PV function instead of NPV.  If that is what your instructor wants, we can show you how to set up that alternative.  It might seem a little counter-intuitive because of the signed values that Excel returns.

rajesh c. say:

You can achieve this by a slight workaround.

In each sheet, if you key-in the following formula in say cell A1 then you will get the current worksheet name in cell A1 as an output of the formula.

=MID(CELL("filename",A1),FIND("]",CELL("filename",A1))+1,255)

(Note: this formula will work only for a saved workbook - in other words - a newly created workbook will not give you desired result until it is saved)

You can do this for each worksheet - so in each worksheet , the cell A1 will give you that particular worksheet name.

You can then refer to the cell A1 of each worksheet to get that name in your desired worksheet.

Alternatively, have a look at the youtube video on the following URL, which will guide you to create a list of all the worksheet names in your desired worksheet.

https://www.youtube.com/watch?v=VfGreju7Tc0
