¡firedog say:

Click Compact view on the View tab and you'll see what it does. You can achieve the same effect by dragging the right-hand border of the folder pane to the left. Only folders that you've added to compact view will be visible.

Storage folders enable you to store messages locally that you don't want to keep on the server. Moving a message from an HTTP or IMAP account folder to a user-created folder under storage folders will remove it from the server. If you remove an account, all its folders and their contents are permanently deleted. You can avoid the consequences of doing this by accident by saving messages in Storage folders.

The special folders that you see under storage folders are required for newsgroup accounts and some IMAP accounts. You can't delete them. TheDeleted items folder there is a temporary repository for items deleted in storage folders.

As for help, there's not much of it around. These forums may be your best bet, but you'll have to use the Find answers box to search for the help you need.
