I never like to contradict my friend, John, but sometimes I just can't avoid it... This is one of those times :-)

We agree that Word for Mac -- regardless of version -- has no built-in ability to open PDFs, but neither does it have a PDF converter. Consequently, I never have seen a PDF "fall directly through" to another program [such as would happen if doing the same in a browser] when attempting to open it using File> Open in Word. Succinctly put, a PDF cannot be opened from within Word. Unless it has been added in O2013, I haven't seen that behavior in the Windows version 2010 or earlier, either.

As for the Convert File dialog, my experience is that the preference determines whether the dialog displays when attempting to open a non-Word file type for which Word has one or more applicable filters. If the Preferences is checked, the dialog displays allowing the user to verify the intended filter or choose a different one: If the box is not checked Word will use the filter it believes to be the right one without opening the dialog. Even if the box is not checked, however, the dialog will launch if Word does not know which converter to use or does not have a usable converter. As mentioned above, PDF falls into this latter category. It's understandably misleading because PDFs do display as a viable file type [depending on the Enable: list stetting] when in Word's Open dialog window.

I wouldn't suggest that you wait for a fix because it isn't something that's broken... it simply is not an included functionality. You may want to submit a suggestion or feature request via https://word.uservoice.com to explain what you would like to see. IMHO, I doubt that there will be any ability to edit PDFs added to Word in the near future, if ever, so there's no reason for the program to be able to open them. And given the posture Microsoft appears to have taken toward Mac Office it doesn't seem likely that any sort of 'pass through' would rank very high on the priorities list. I'm reasonably sure that their position would be 'just switch to your PDF reader program or Finder'.
