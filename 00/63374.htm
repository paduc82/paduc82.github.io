We are really sorry for that the Fresh Paint doesn’t have this option for changing the DPI.

But yes we can help you resizing the image by following the below steps.

To resize a picture using Paint
1. Click the Start then type Paint.
2. Click the Paint button click Open, click the picture you want to resize, and then click Open.
3. On the Home tab, in the Image group, click Resize.
4. In the Resize and Skew dialog box, select the Maintain aspect ratio check box so that the resized picture will have the same aspect ratio as the original picture.

If the Maintain aspect ratio check box is selected, you only need to enter the horizontal value (width) or vertical value (height). The other box in the Resize area is updated automatically.
5. Do one of the following in the Resize area, and then click OK:

• To resize your picture by a certain percentage, click Percentage, and then enter a percentage to reduce the width by in the Horizontal box or a percentage to reduce the height by in the Vertical box.
• To resize the picture so it's a specific size, click Pixels, and then enter a new width in the Horizontal box or new height in the Vertical box.
6. Click the Paint button, point to Save as, and then click the picture file type for the resized image.
7. Type a new file name in the File name box, and then click Save.

4. On the troubleshooter select Reset > Reset keeping games and apps.

If a disc is not working read this. - Microsoft Community

fishman2409 say:

If a disc is not working read this.

If you're having a problem with a single disc but other discs are working fine, then here are a couple of solutions to try.

1: Make sure the disc is clean. Clean the disc with a soft, clean, slightly damp cloth. Hold the disc by the edges without touching the top or bottom surface.

2: Try the disc on another console to see if the disc is the problem. If the disc works on a different console then refer here: https://support.xbox.com/en-US/xbox-one/console/disc-error#mydisc.

3: If the disc is a cd, dvd, or blueray disc then make sure you have the dvd and blueray app. To do this refer here: https://support.xbox.com/en-US/xbox-one/apps/dvd-blu-ray-setup. Make sure the cd, dvd, or blueray disc is from the same region as the xbox you purchased. To troubleshoot with this refer here: https://support.xbox.com/en-US/xbox-one/apps/watch-bluray-dvd-movies.

4. Replace the game with the xbox disc exchange: https://support.xbox.com/en-US/games/troubleshooting/disc-replacement-program.

If none of your discs play then there could be two problems. The first is that the instant-on power mode can render a small number of consoles unable to read discs. The other problem is perhaps the console's disc drive needs servicing. To turn off instant-on power mode go to all settings and power and change it to energy saving. If the console's disc drive needs servicing then here is a link to the online service center: https://devicesupport.microsoft.com/en-US.
