  3. Open cmd and navigate to %windir%\Microsoft.NET\Framework64\v4.0.30319
  4. Run regasm <path_to_dll>\MyAssembly.dll
That's it, now it works.

Just like Linux or OSX, when an app is uninstalled it's not necessarily "purged". It still resides in a few different read only folders under Windows. SO, to eliminate the unwanted updates in later builds do this....

Open powershell(admin) and stay in system32 dir

Type Get-AppxPackage | Select Name, PackageFullName  (and you will get a list of pkg's on the system)

Look through the list on the LEFT side and find the package you want gone and replace *_* with the simple name AFTER the (.) So if you want Zunemusic gone put that. Don't get carried away with this since some apps which are part of the Windows build will report an error in RED, they will re-appear in future builds. Also note ccleaner can not perform this operation..... SO..... type this

get-appxpackage *-* | remove-appxpackage   (replacing *_* with pkg name)

This will purge the pkg

doug robbins - mvp office apps & services (word) say:

One way is to insert an ActiveX command button into the document and add a picture that button

Then insert the fullsize picture into the document and select it and the add a bookmark (in this example, I used Image1).

Then add the following code to the Command Button

Private Sub CommandButton1_Click()
With ActiveDocument.Bookmarks("Image1").Range.Font
    If .Hidden = False Then
        .Hidden = True
    Else
        .Hidden = False
        Exit Sub
    End If
End With
End Sub

Then, clicking on the command button, toggles on and off the display of the image by changing the hidden attribute of the font (The display of Hidden Text and of non-printing characters must be turned off)

Hi,

You're seeing this message because you do not have have an email program installed and a default e-mail program set.

The default e-mail program must be installed on the computer- e.g. Outlook Express (XP), Windows Mail (Vista), Windows Live Mail, Thunderbird, etc.

You cannot use a web-based e-mail service as the default. On Windows 8-8.1, you can't use the mail app as the default.
