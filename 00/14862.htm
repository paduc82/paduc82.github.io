steve rindsberg say:

Assuming you want to do this during a slide show rather than in Normal view:

Use the selection pane to name the image you want to hide "Image"

(or anything else you like, if you modify the following code accordingly).

Add this code to the presentation:

Sub ToggleMeElmo(oTriggerShape As Shape)
    Dim oSh As Shape
    For Each oSh In oTriggerShape.Parent.Shapes
        If oSh.Name = "Image" Then
            oSh.Visible = Not oSh.Visible
        End If
    Next
End Sub

Then add another shape or select an existing shape and give it an Action setting of

Run Macro: ToggleMeElmo

Just don't use the image itself for this; if you do, you'll be able to make it go hidden but not unhidden, since once invisible, it can't be clicked.

It's not "finicky", it's precise. Each field has a data type. The Unique ID field is a long (i.e. number) type and therefore the "equals" test is required. The "contains" test is used for string type (i.e. text) data. So, you can't use equals with string

The default in Word is to display deletions and comments in balloons in the margins of the document. However, you can change the display to show comments inline. Inline comments can be viewed when you rest your pointer on the comment indicator.
