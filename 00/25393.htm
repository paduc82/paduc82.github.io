No, but you can set up "gutter" formatting to have even and odd space preserved in your margins to hold the three holes when you have paper.  You could put a graphic of three holes in your margins to simulate the punching but I wouldn't.

rich michaels - mvp (greatcirclelearning.com) say:
FYI  and FWIW based on experience - in inches the absolute bare minimum margin plus gutter would be 0.7" and a 3-hole punched paper stock will work.

suzanne s. barnhill say:

I would strongly advise punching after printing to avoid having to figure out how to feed the paper in order to get the holes in the right position. If you have a mopier that is able to punch paper as part of the print process (I know some of them can staple), then this will be a feature of the hardware, accessed through the printer Properties. But you can certainly purchase pre-punched paper and experiment to find out the correct feed orientation.

Refer to the other replies about suggestions regarding the gutter or mirror margins setting required to allow room for punching. Don't make this excessive, however. If you use the default 1" margins, there's no need to add extra on the binding side; keep in mind that any binding format that allows the paper to be seen full width (this includes comb and spiral binding as well as ring binding) does not require the same gutter as a binding process that obscures part of the page. If you add too much inside margin, the pages will look off-center.

I couldn't post to the forum for some reason, so I worked on the problem for a couple of days and finally pieced together this solution for the next person. I wish I knew where to look in a manual or tutorial for this kind of thing. But the only way seems to be to cobble solutions together from forums on the net. Here's my contribution! :-)


Sub AlignTopRight()
' Paste an image into Word so it is still selected
' Then invoke this macro to align it to the top right corner
' And to set it behind text, fixed position on the page

Application.ScreenUpdating = False
Dim Shp As Shape
On Error Resume Next
    'I'm not sure if this block is required, but it works
     Set Shp = Selection.InlineShapes(1)
     If Not Shp Is Nothing Then
          Set Shp = Selection.InlineShapes(1).ConvertToShape
     Else
          Set Shp = Selection.ShapeRange.Item(1)
     End If

    If Not Shp Is Nothing Then
    With Shp
      .LockAspectRatio = True
      ' for absolute positioning
      '.Left = CentimetersToPoints(5.5)
      '.Top = CentimetersToPoints(0.5)
      '.Width = CentimetersToPoints(2.5)
      'put the image behind text
      .WrapFormat.Type = wdWrapBehind
      'this was the tricky part, discovering this
      .RelativeHorizontalPosition = wdRelativeHorizontalPositionPage
      .RelativeVerticalPosition = wdRelativeVerticalPositionPage
      .Top = wdShapeTop 'if you say =0, it sets the AbsolutePx in the dialog
      .Left = wdShapeRight 'these wdShapeXX objects set the Align field in the dialog
    End With
    End If
Set Shp = Nothing
Application.ScreenUpdating = True
End Sub
