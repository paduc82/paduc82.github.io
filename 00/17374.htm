rich007a say:

Does something like this help:

 

Private Sub MyCalc()
    Dim pres As Presentation
    Dim sld As Slide
    Dim shp As Shape
    Dim dbTotal As Double
   
    Set pres = ActivePresentation
    For Each sld In pres.Slides
        For Each shp In sld.Shapes
            If shp.Name = "SlideTotal" Then
                dbTotal = dbTotal + CDbl(shp.TextFrame.TextRange.Text)
            End If
        Next shp
    Next sld
   
    pres.Slides(1).Shapes("PresentationTotal") _
        .TextFrame.TextRange.Text = dbTotal
End Sub

 

Running this code will look on each slide for a shape called "SlideTotal" (named via the Selection Pane: Home tab > Editing > Select).  It will sum the values contained in these shapes and then enter that total in a shape named "PresentationTotal" on slide 1.

You could assign the above macro to a shape so it is run when the shape is clicked.

Cheers
Rich
