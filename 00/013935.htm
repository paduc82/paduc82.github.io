Conditional formating: Any cell in given range = any cell in - Microsoft Community

taupe marie say:

Conditional formating: Any cell in given range = any cell in 2nd given range

If the value in any cell in column A equals the value of any one cell in column B, both cells need to be highlighted (filled)

At the same time, if any cell in column A equals any other cell in column A, both cells need to be highlighted (I know how to do this one, but I'm not certain about the two rules together).

Thank you!

jeeped say:

I would take a 5 rule/3 colour approach to this. Something like:

  • Yellow - values in column A are duplicated in column A only
  • Red - values in column A are duplicated in column B only
  • Blue - values in column A are duplicated in both column A and column B

First make three rules for column A. Select all of column A and create these rules:

  1. Use a formula to determine which cells to format =COUNTIF(A:A,A1)>1  fill colour yellow
  2. Use a formula to determine which cells to format =COUNTIF(B:B,A1)  fill colour red
  3. Use a formula to determine which cells to format =AND(COUNTIF(A:A,A1)>1,COUNTIF(B:B,A1))  fill colour blue

Next, select all of column B and create two more rules.

  1. Use a formula to determine which cells to format =COUNTIF(A:A,B1)  fill colour red
  2. Use a formula to determine which cells to format =COUNTIF(A:A,B1)>1  fill colour blue

And finally, go to Manage Rules then set Show formatting rules for: to This Worksheet (circled in red below). Select each rule and use the Up and/or Down arrows (also circled in red below) to set up a progressive logical order like blue, blue, red, red, yellow.

If that logical progression does not appeal to you, you can create your own by moving the rules up or down and turning the Stop if true option on or off.

This might seem like quite a bit of extra work, but your conditions seemed important so identifying what each highlight means would likely be an advantage.

hansv mvp say:

Select the cells in column A that you want to apply conditional formatting to. Let's say A2 is the active cell within the selection.

Select Conditional Formatting > New Rule...

Select the option "Use a formula to determine which cells to format".

Enter the following formula:

=COUNTIF($B$2:$B$1000,A2)>0

where B2:B1000 is the used range in column B.

Click Format... and specify the formatting that you want.

Click OK.

The rule you already have can exist side by side with the new one. If they can be combined, for example if one rule sets the font color and the other the fill color, they can both be applied to the same cell, if appropriate.
