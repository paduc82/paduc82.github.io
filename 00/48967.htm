What is the difference between Windows® updates and HP Support Assistant updates?

        

Both updates are important, and cover different aspects of your computer. Windows updates are limited to operating system updates. HP Support Assistant provides software updates specifically related to your HP or Compaq computer, including driver updates, BIOS updates, and HP software updates.

In rare cases, such as a software driver update, both Windows and HP Support Assistant may try to provide the same update. Both applications will check, however, if the new driver version was already installed. If it was, neither application will download or install the driver again.

Will it  be possible to install the 64-bit version of Windows 7 Home Premium along with about 3 GB of other programs on a 32-GB solid state boot drive?
j w stuart say:

That will work.

1) Windows 7 on the SDD. No Hibernation, pagefile on second hard drive (first partition, set to about 20GB).

2) Applications installed using "Custom" option so that the install path points to the hard drive (2ns partition).

3) Data stored to its own partition on the hard drive.

4) Temp files also set to utilize the hard drive.

This will of course loose some of the speed advatages of using a SSD, but no all of the benifits.
