AN,

Slideshows will skip .mov files.  Even if your .mov files only contain pictures, they are still recognized as video files by your computer, and the slideshows in photo gallery and in screensaver won't include video files.  If you have access to the original photos that make up your .mov files, save those individually as .jpg or .png files; then they'll be recognized as photos and can be included in your slideshows.  My brother sends me groups of photos in .mov format, and the only way I can view those is with media player; to add them to any slideshow, I have him send them one at a time so that windows sees them as individual photo files (.jpg) and not as a single video file (.mov).

r2d2-76540 say: From my side, in the registry, opened with "Regedit" in administrator mode; I found this:
Key: HKLM / Software / Policies / Microsoft / Windows / WindowsUpdate / DisableWindowsUdateAccess = 1
I changed: HKLM / Software / Policies / Microsoft / Windows / WindowsUpdate / DisableWindowsUdateAccess = 0
I validated by closing "Regedit"; and I was able to make updates without problems.
I watched on another computer and the key "HKLM / Software / Policies / Microsoft / Windows / WindowsUpdate" does not exist. So on the one that was the mistake I delete this key.
Since then all updates are done correctly.

If it helps others.
