johnniejohnson. say:

percentages in Resource Names Column?

Hello, I am getting a percentage in my Resource Names column.  All of my resources are set to 100% in Resource availability.

As you can see it doesn't happen all of the time.   So basically I have 2 questions, 

1. What do these percentages mean?

2. How do I get rid of them?  I have found how to hide percentages on the Gantt chart but not how to get rid of them in the column.

dale howard [mvp] say:

Johnnie --

The percentages appended to some of the resource names indicate the Units value applied to the resource on the task.  The Units value indicates whether the resource will be working full-time (100% Units) on the task, or part-time (such as 20%).  In your example, Colleen will be spending approximately 20% of each day working on the task, meaning she is working part-time.  There is no way to hide this number.  Microsoft Project shows it by default for every resource that is not assigned to work full-time on a task.  Hope this helps.

johnniejohnson. say: It is set for fixed work and the Effort Driven Box is not checked.

dale howard [mvp] say:

Johnnie --

There are two types of Units values that you specify in a Microsoft Project schedule.  As you already know, there is a Max. Units column available in the Resource Sheet view.  The Max. Units value you enter represents the maximum amount of an average working day that a resource can work in the project.  When you entered 100% in this column for the resource in question, you told Microsoft Project that the resource is available to work full-time in the project.  For part-time resources, enter values less than 100% in the Max. Units column.

The other Units value is called Assignment Units.  This is the value you which you specify, or which is calculated, when you assign a resource to a task.  This is the percentage value you are seeing to the right of the resource name for each Gantt bar in the project schedule.  When you assign a resource with an Assignment Units value of 100%, Microsoft Project does not show this number to the right of the resource name.  But when you assign a resource to work part-time on a task, Microsoft Project DOES show the Assignment Units value.

To see the Assignment Units value for the resources assigned to each task, do the following:

  1. Display the Gantt Chart view.
  2. Right-click anywhere in the Gantt Chart pane and select the Show Split item on the shortcut menu.
  3. Select any task in the top pane (the Gantt Chart pane).
  4. In the bottom pane (the Task Form pane), you will see the names of the resources assigned to the task, along with their Units value (which is the Assignment Units value) and their Work value.

Because you are using Fixed Work as the Task Type, I suspect that you assigned the resource by entering a Work value and a Duration value, which would mean that Microsoft Project would calculate the Assignment Units value.  Keep in mind that if you change the Units value for the resource, Microsoft Project will recalculate the Duration value because the Work value is locked.  Hope this additional explanation helps.
