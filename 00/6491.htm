Does this happen in the new blank document that appears when you open Word by itself?

If so, the most likely cause -- given how long you've used your copy of Word -- is that some kind of protection has been applied to the Normal.dotm template, possibly by accident. To check this:

  • Use the Office button > Open command to open the Normal.dotm file. (If you don't know where it is, enter %appdata%\Microsoft\Templates in the address box of the Open dialog and press Enter; you should see Normal.dotm in the file list in that folder.)
  • Go to the Review tab of the ribbon and click the Protect Document button near the right end. This opens the Restrict Formatting pane on the right. If there's a Stop Protection button at the bottom of the pane, click it.
  • If you aren't prompted for a password, the protection has been removed. Save the template and close it, close and reopen Word, and make sure the protection is gone.
  • If you are prompted for a password (and presumably you don't know what the password is), close Word, use Windows Explorer to go to the Templates folder, and delete the Normal.dotm file. Then restart Word.


If protection isn't the problem, it's possible that you never activated your copy of Office, and it has now (after a fixed number of sessions) gone into reduced functionality mode (http://support.microsoft.com/kb/927921). That article explains how to activate Office, which will restore normal function.


If that still doesn't help, read http://word.mvps.org/FAQs/AppErrors/ProblemsStartingWord.htm.

suzanne s. barnhill say:

You say, "I think I have read somewhere that this is because fonts in general do not include old style figures as superscripted and since the footnote reference style that is used in the text is connected to the footnote reference at the bottom of the page, the number at the bottom of the page cannot be in old style." I think that's a red herring because Word "superscripts" any character just by reducing the font size and raising it (though this is transparent to the user and not reflected in the font formatting as displayed). It should be able to do this with OSF as easily as with lining ones. In fact, I have tested with applying both Superscript and OSF formatting to ordinary text (not a footnote reference), and it does work.

But I tried modifying the Footnote Reference style to use OSF, and it doesn't seem to "take," so I guess that's a nonstarter. What you really need to do is modify the macro used to "unsuperscript" the note numbers to include formatting them as OSF. I should think that might work.

stefan blom say:
I've had a quick look at this and it appears that the footnote reference mark itself simply doesn't support setting/customizing the option for "Number forms."

jay freedman say:

Even adding the necessary statement (.Paragraphs(1).Range.Font.NumberForm = wdNumberFormOldStyle) to the macro doesn't cause the footnote reference numbers to appear as old style. The Advanced tab of the Font dialog claims that the property is set, but it's ignored.

I think this is a result of the behind-the-scenes manipulation by the Word program that causes the reference numbers to be hyperlinks without appearing in blue or underlined. It seems that it would take a change in the program to allow old style numbers. I'm afraid that's a low-probability request, but you can post it at https://word.uservoice.com/ and see how many votes you can get for it.

stefan blom say:
Indeed, as Suzanne wrote, changing the "Number forms" for a footnote reference mark is permitted (that is, the option isn't greyed out), but the display isn't affected.

make folders and subfolders in Word 2013 - Microsoft Community

lizjelinek phd say:

make folders and subfolders in Word 2013

How do I make folders in Word 2013

john korchok say:
Usually you would make the folders in Windows, not in Word. However, when you're saving a file, the File>Save dialog has a New folder button that you can click to create a new folder.        

jay freedman say:

If you're asking about making folders in which to store individual document files, you do that in Windows Explorer -- or in any of the file management dialogs in Word, such as the Save As or File Open dialogs, which are really just Windows Explorer with different window controls -- by clicking the New Folder button.

If you're asking about making "folders" or something like that within a Word document, the answer is that you can't. Word doesn't have any such concept. If you want a tabbed interface with multiple separate "documents" within a single file, use OneNote instead of Word.
