Running Windows 7 and Office 2010.

cmdrkeene say:
Hey Geoff,
I'm a statistician at my day job and I spend my days in Excel.  I've wished for this many many times... and for some reason I've never investigated it until I saw your question.

Well, I believe I just made both our lives easier.  Here's how I did it.

  1. Download the open-source Types utility, which allows you to change file type settings in Windows 7 (the interface for which is no longer present in the Folder options).

  2. In the Classes dropdown menu, find Excel.Sheet8 (that 8 refers to the version number of Excel you have installed, or more appropriately the version of Excel that made the files).  With that selected, click the Gear icon to open the properties of the class.

  3. On the Actions tab, click in any blank white space to make sure nothing is selected.  Type a new action in the top box (I used "Open in new window"), then click the + plus button to add it.

  4. Select your new entry, then put this into the bottom box (your path to Excel may be different, you can copy from one of the existing actions).

    "C:\Program Files (x86)\Microsoft Office\Office14\EXCEL.EXE" "%1"


  5. When you're done, just close out of every window and give it a try.  It worked great for me here.

Also - one side effect you are likely already aware of:  Excel files open in separate windows are isolated from each other.  It makes it exceedingly difficult to write formulas that reference the other file, among some other oddities.
