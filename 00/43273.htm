Port Forwarding

  1. Login to the router by entering the default gateway IP address.
  2. The router will then prompt you to enter the user name and password.the default user name and password are: "admin", "admin".
  3. From the menu along the top select Advanced Setup.
  4. From the sub-menu select NAT.
  5. Select Virtual Servers.
  6. Set the Rule Index to a number not in use in that table below.
  7. For the port start and end numbers enter 3074.
  8. Enter the xbox‚Äôs IP address to the Local IP Address box.
  9. Select Apply.

 

 

Disable Port Forwarding

  1. Login to the router by entering the default gateway IP address.
  2. The router will then prompt you to enter the user name and password.the default user name and password are: "admin", "admin".
  3. From the menu along the top select Advanced Setup.
  4. From the sub-menu select NAT.
  5. Select Virtual Servers.
  6. Select the rule number using the Rule Index drop down.
  7. Select Delete.
  8. Select Apply.
