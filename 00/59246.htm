I completely understand that the 64-bit version is a clean install and not an upgrade, and I understand from reading articles and forum threads that 32-bit Windows 8.1 will upgrade to 32-bit Windows 10.

Hence why I want to be able to have the two installed on the same box for few weeks or so.

Am I able to do this?

Thanks!

cmdrkeene say:

You can't run it side-by-side.  This is an upgrade copy that can only be used in a single instance.  You can go to 64-bit if you want, but you have to do it with a clean install following your first 32-bit upgrade.  It would work like this:

1) You have 32-bit today. You get the upgrade and have 32-bit Windows 10.

2) You activate. Now the activation server will allow you to clean install in the future.

3) You download the 64-bit "create install media" tool and create a USB/CD for installing. Boot up from it directly (don't run it from inside your 32-bit OS).

4) You boot from it, wipe the drive, and install the 64-bit version.

If you downloaded the 64-bit install media, then installed it to another drive partition for side-by-side use, you'll have problems activating.
