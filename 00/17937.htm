Sub Make_Values()
    Dim rng As Range

    Set rng = Nothing

If everyone has his or her Windows and Outlook time zones selected properly and all have applied the Windows updates for DST, then Outlook will handle events across time zones properly and each person will see the event in his or her local time.  Thus if a person in the Eastern time zone makes an event for 9:00 AM and sends it to someone in the Pacific time zone, the time will be 6:00 AM and if they send it to someone in London, that person will see it at 2:00 PM.

http://windows.microsoft.com/en-us/windows-10/getstarted-protect-your-pc
 
On Windows 8.1/10 when a third party antimalware program (such as McAfee, Norton, etc) is installed the Windows Defender (WD) program is disabled.
 
If you wish to use WD on Win 8.1/10 you should ensure that any third party antimalware program that may be on the computer (or was previously on the computer before it was upgraded to Win 10) has been uninstalled. It is recommended that you also run the appropriate software removal tool for the third party program(s) and then reboot the computer. See List of anti-malware product removal tools
 
WD should then be enabled though you might need to restart WD in Action Center.
 
For further information see Problems starting Windows Defender in Windows 8/8.1/10
