If the date being compared to create the number is duplicate then the row number will be duplicated at the highest value.

Try this --

SELECT (SELECT Count(*) FROM WPRWeeklyHistory AS [XX] WHERE [XX].WeekCode&[XX].[Date] <= WPRWeeklyHistory.WeekCode &WPRWeeklyHistory.[Date]) AS Rownumber, WPRWeeklyHistory.MvT AS tempHOLDER, WPRWeeklyHistory.[User name], WPRWeeklyHistory.DateCode

Sub Test()
  Const wdInlineShapeOLEControlObject = 5
  Dim wdApp As Object 'Word.Application
  Dim wdDoc As Object 'Word.Document
  Dim wdISh As Object 'Word.InlineShape
  Dim ctlObj As Object 'This is not a MSForms.Control !
  Dim ctlOB As Object 'MSForms.OptionButton
 
  On Error Resume Next
  'Try to access Word
  Set wdApp = GetObject(, "Word.Application")
  If wdApp Is Nothing Then
    MsgBox "Open your word file first"
    Exit Sub
  End If
  'Get the opened file
  Set wdDoc = wdApp.ActiveDocument
  If wdDoc Is Nothing Then
    MsgBox "Open your word file first"
    Exit Sub
  End If
  On Error GoTo 0
  'Check each shape
  For Each wdISh In wdDoc.InlineShapes
    'Is it a OLE control?
    If wdISh.Type = wdInlineShapeOLEControlObject Then
      'Get the object inside
      Set ctlObj = wdISh.OLEFormat.Object

      Select Case TypeName(ctlObj)

Function Append(ParamArray Args()) As Variant
  'Combines all passed arguments in one array, must be called as matrix formula
  'The result is a two dimensional array with the same size as the matrix formula
  'Arguments could be a cell reference, single value or matrix array
  'Example: {=Append(A1:A5,"x",{7;8;9})}

  Dim Result(), Data
  Dim a As Long, Size As Long
  Dim i As Long, j As Long, r As Long, c As Long

  'Step 1: Determine the size of result array
  For a = 0 To UBound(Args)
    If TypeOf Args(i) Is Range Then
      'cell reference => "A1:B3"
      Size = Size + Args(i).Rows.Count * Args(i).Columns.Count
    ElseIf IsArray(Args(i)) Then
      'array => "{1,2,3}"
      Size = Size + UBound(Args(i)) - LBound(Args(i)) + 1
    Else
      'value
      Size = Size + 1
    End If
  Next

  'Step 2: Allocate the result array
  If TypeOf Application.Caller Is Range Then
    'Get the size of the matrix formula
    With Application.Caller
      ReDim Result(1 To .Rows.Count, 1 To .Columns.Count)
    End With
    If Size = 0 Then
      'Fill with empty strings
      For i = 1 To UBound(Result)
        For j = 1 To UBound(Result, 2)
          Result(i, j) = ""
        Next
      Next
      GoTo ExitPoint
    End If
  Else
    If Size = 0 Then
      'Return empty array
      GoTo ExitPoint
    Else
      'Return onyl one column
      ReDim Result(1 To Size, 1 To 1)
    End If
  End If

  'Step 3: Copy the data into the result array
  r = 0
  c = 1
  For a = 0 To UBound(Args)
    Data = Args(a)
    If IsArray(Data) Then
      'cell reference => "A1:B3"
      'array => "{1,2,3}"
        For j = 1 To UBound(Data, 2)
      For i = 1 To UBound(Data)
          r = r + 1
          If r > UBound(Result) Then
            c = c + 1
            If c > UBound(Result, 2) Then GoTo ExitPoint
            r = 1
          End If
          Result(r, c) = Data(i, j)
        Next
      Next
    Else
      'value
      r = r + 1
      If r > UBound(Result) Then
        c = c + 1
        If c > UBound(Result, 2) Then GoTo ExitPoint
        r = 1

swischner say:
Do I understand you correctly that you clipped a webpage from the browser to OneNote using the web clipper extension and now you want to cut off unwanted portions of that web page in OneNote?

If you used the "Full Page" or "Region" options when clipping the page, you now have a screenshot (bitmap image) in OneNote. As OneNote does not offer any image editing features, you can not crop the picture in OneNote. Your only option would be to export the image from the OneNote page, edit it in any image editing application and import the new, edited picture back into the note.

If you choose the "Article" option when clipping web content, it will only take relevant portions of the page and save that as text and images into OneNote, so you could remove unwanted portions (if there are any, because the "Article" mode tries to clean up the content anyway). A similar option is to not use the web clipper menu at all, but select the portions of the web page in the browser, right-click the selection and choose "Clip to OneNote" from the context menu. This also produces editable text and graphics in OneNote, similar to the Article-mode.
