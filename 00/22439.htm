rich007a say:

Hi,

I think you would need to keep everything in one slide.  You would have to make your text appear/disappear using Animation, but it would all have to be on one slide.

 

If you have the "same" video appearing on several slides, it will be restarted each time you move to the next slide.  While it looks like the "same" video, to ppt, it's just another video that needs playing - there is no link between the slides.

 

Hope that helps.

Cheers
Rich

ute-s say:

Do not insert the video in the slide master. If you insert it there, it will re-start with every slide.

 

Insert your video on the first slide. Change the animation to "Play". Set it to play across slides. Then it will continue to play, if you change slides. BUT: You cannot display the text of the second slide in front of the video, it will be placed behind it. So this method works only for animated borders, not for full-size backgrounds.
