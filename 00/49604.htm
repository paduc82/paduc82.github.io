I would check the status of the Print Spooler Service.

You should be able to configure this type of setup by modifying the registry using the Registry Editor (regedit). Please take note that the method below contains steps that tell you how to modify the registry. However, serious issues might occur if you modify it incorrectly. Therefore, make sure that you follow these steps carefully. For added protection, back up the registry before you modify it. Then, you can restore it if a problem occurs. For more information about how to back and restore the registry, refer to this support article.

  1. Click the Start button, type regedit, and then press Enter.
  2. If prompted, choose Yes or enter the administrator password.
  3. On the left-side pane, navigate to the path below.
    • HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Policies\Explorer
  4. Right-click Explorer and then select New > DWORD (32-bit) Value.
  5. Name the value as NoStartMenuMorePrograms and then hit Enter.
  6. Double-click NoStartMenuMorePrograms, set the Value data to 1 and then click OK.
  7. Close the Registry Editor and then restart the PC.

Thank you for that information. The policy: Remove All Programs List from the Start menu applies to your goal.

  1. Click the Start button, type gpedit, and then hit Enter.
  2. Navigate to User Configuration > Administrative Templates > Start Menu and Taskbar.
  3. Double-click Remove All Programs List from the Start menu.
  4. Select Enabled.
  5. Under Choose one of the following actions, you can choose between two configurations:
    • Collapse and disable setting - this will hide all Start menu icons. App icons can still be accessed by clicking the All apps button.

  6. Click Apply and then OK to save the changes.
  7. Restart the PC.

  1. Press the Start button, then type cmd, then right-click Command Prompt, then select Run as administrator.

  2. From there, type the following command to find and repair errors on drive C or to your other drives, then press Enter:
    • chkdsk /f C:

After running the command, Check Disk will perform an analysis of the disk and then repairs any errors it finds, provided that the disk isn't in use. If the disk is in use, Check Disk displays a prompt that asks whether you want to schedule the disk to be checked the next time you restart the system. Click Yes to schedule this check.
