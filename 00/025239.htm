t. valko say:
Try this...

With your dates in the range A2:A13...

Enter this formula in B2:

=TEXT(A2,"mmmm")

Enter this array formula** in B3 and copy down until you blanks:

=LOOKUP("zzz",CHOOSE({1,2},"",INDEX(TEXT(A$2:A$13,"mmmm"),MATCH(0,COUNTIF(B$2:B2,TEXT(A$2:A$13,"mmmm")),0))))

Thanks Biff,

Thanks for your solution. Works perfectly well. Could you please also clarify me the use of"zzz" & Choose({1,2}in your above formula?

 


We're using that as an error trap.

The INDEX(...) expression will return either a month name that is a TEXT string or a #N/A error.

When INDEX returns a month name we have:

=LOOKUP("zzz",{"","month_name"})

When INDEX returns a #N/A error we have:

=LOOKUP("zzz",{"",#N/A})

So, we're using LOOKUP to return the *last* TEXT string in the the array that's generated by the CHOOSE function.

When the array is {"","month_name"} the last TEXT string is the month_name.

When the array is {"",#N/A} the last TEXT string is the blank "". The #N/A error is ignored.
