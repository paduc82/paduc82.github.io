This gives you just messages from 2011 as .olm file. You have to import .olm files in order to view messages. Not the same as .pst file, but that's our option on a Mac.

Another option is to drag a folder to the Desktop. This creates an .mbox file. Each folder and subfolder has to be dragged individually. Even a smart folder can be dragged as a .mbox file.

Try this...

I just bought a 6230i and have found almost no information about using an MMC.

 

Two most burning issues are ..

 

1) how do I copy info from my phone memory to my MMC?

 

2) how do I make sure that in the future, when I save files on my phone (images or audio) that they get saved to my MMC and not to my phone memory?

shezmyster say:

Hi Mike;

 

1) As for copying information (Files) from phone memory could be done once you have selected the file you want to move.

 

Eg: Moving an image from the phone memory to MMC

Select Menu> Gallery> Images>*highlight the image you want to move* > Options> Move > *Then it will open the Gallery again > Select target folder > Press middle key (move to)

Thats it!

 

2) As for making sure your information is saved to the MMC, you have to define the settings before hand.

 

Eg: Saving your Photos taken from Camera directly to MMC

Select Menu> Media > Camera> Options> Settings> Image Storage> Other Folder> * Then it will open the Gallery again> Select the MMC (*if you have not renamed it you will see "NO NAME")> press middle key (SET)

 

 

Hope this helps :)

jim gordon mvp say:

Be sure Outlook is at version 14.7.7 before proceeding (Outlook menu > About Outlook)

Copy the backup identity to the Microsoft User Data/Office 2011 Identities folder in your Documents folder.

If it is the name of your default identity, the next time you open Outlook, it will rebuild the identity and you should be good to go. If you have more than one Identity, hold the Option key down when you start Outlook and set your copied identity as the default identity, then open it. Outlook will immediately rebuild it, and you should be fine.
