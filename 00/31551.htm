syed n say:

Hi Shawn,

 

Backups are created in sets known as backup periods. To help maximize your disk space, Windows Backup backs up all selected folders the first time it runs and then it only backs up files that are new or have been modified since the last backup was made.

 

Periodically, Windows creates a new, full backup. Each full backup is known as a backup period. When you view your file backups, you see all of the backup periods labeled with date ranges. If you decide to delete file backups, you should always keep the most recent file backup.

 

The decrease or increase in size of the backup file can have many reasons; it could be due to the programs and data files that you have deleted\added. Also the temporary internet files add to the size of the data files accessed during the period of the backup taken.

 

For more information on Backup:

a. user say:

You forgot to say which backup isn't working or how it's not working.

Windows 10 has two backup utilities: File History backs up individual files and System Image Backup creates backups of entire drives. Neither was intended to take the place of a fully featured third-party backup utility, but not everyone feels comfortable installing third-party software. For these people Microsoft includes in Windows, for no extra charge, a very basic backup capability.

File History works as advertised, but its limited feature set doesn't come close to what you get with a third-party file and folder backup program, of which there are many excellent examples, both free and paid. If backing up isn't convenient, people won't do it.

System Image Builder has been deprecated which is what your girlfriend means when she says "we can still be friends." It's no longer being developed and Microsoft recommends the use of a third-party program. It's not easy to see why: There are so many terrific image backup programs, created by developers who specialize in that type of software.

To find a backup program that is both reliable and convenient - two hallmarks of a worthy backup program - search the web with this search term:

best windows backup 2018

The search results will include reviews of excellent backup programs. Read the reviews from professional organizations, take advantage of the free trials, and choose a backup program that makes sense to you.

You probably know this already, but System Restore is not a backup and neither is Backup and Restore (Windows 7) which was included in Windows 10 so that users who made backups with the utility in Windows 7 can restore their backups on Windows 10. It can also make backups but it's not supported for that purpose so it doesn't always work properly. The whole idea of a backup program is that you can count on it.
