julie fka profjulie say:

Windows Phone only syncs directly to an online calendar.  The online calendar can be a part of an Outlook.com, Live.com, Hotmail.com account or it can be related to a 365 account.

If you also use Outlook 2013 on a computer to manage your calendar, you need to sync that calendar to an online calendar that can be accessed synced to a Windows Phone.  Calendars that are stored in the personal folder within Outlook 2013 do not sync any where and will never be syncable to an online Calendar.

If you use a 365 account, you can set it up to sync to Outlook 2013 and to a windows phone.  In both cases, you'll set up an account (in both Outlook 2013 and on Windows Phone) that uses the same user name and password that you use when you log into the 365 account via a computer browser.

When you set up the 365 account within Outlook 2013, a folder gets created in the software that includes Calendar, Contacts, etc.  Add any new calendar entries to the Calendar in that folder. Copy any calendar entries that are currently stored in the Calendar in your personal folder to the 365 Calendar.  Any calendar entries that are stored in the Calendar in the 365 folder  will sync online to your 365 account.  

When you set up the 365 account on the phone, again, using the same user name & password that you used when you set it up in Outlook 2013 and that you use when you log into it from a computer browser, any Calendar entries that have been synced from Outlook 2013 to the online 365 calendar will be synced to the phone.
