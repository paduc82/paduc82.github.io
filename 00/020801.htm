macropod say:

FWIW, I use a couple of macros to export all citation/bibliography entries to a document for backup, and for re-importing if necessary:

 

Sub SourcesExport()
Dim oSrc As Source, StrSrc As String
With ActiveDocument
  .Range.Text = vbNullString
  For Each oSrc In Application.Bibliography.Sources
    StrSrc = StrSrc & vbCr & oSrc.XML
  Next
  .Range.InsertAfter StrSrc
  .Paragraphs.First.Range.Delete
End With
End Sub

 

Sub SourcesImport()
Dim i As Long
'In case the source is already present
On Error Resume Next
With ActiveDocument
  For i = 0 To UBound(Split(.Range.Text, vbCr))
    Application.Bibliography.Sources.Add Split(.Range.Text, vbCr)(i)
  Next
End With
End Sub

 

Simple and effective for both backup & restoration and for replication across multiple PCs.

These are controlled by the LocPinX and LocPinY cells in the shape's shapesheet so they cannot be set independently. The rotation handle may appear to be of a short fixed length but when you drag it with your mouse you can move your action outwards to get finer control of the angle. P.S. the angle can be set using the Angle cell in the shapesheet, and also via the Size and Position window, but that works with values only and will overwrite any formulae that have been setup!

Recover unsaved document HELP - Microsoft Community

phyl12 say:

Recover unsaved document HELP

How do I recover the document I was working on and did not save in Word....... 2010

I do have the auto save feature on but I cannot find the file ?

jay freedman say:

Click the File tab in Word. If "Recent" isn't highlighted on the left side, click it. At the bottom right, click "Recover Unsaved Documents". A dialog will open to the folder that contains any auto-saved files that exist.

If the document was open for less time than the interval set for AutoSave in the Options dialog, there won't be an ASD file for it. Also, Word removes auto-saved files that are more than four days old.
