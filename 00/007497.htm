ute-s say:

When a sound file is recorded and saved, a codec is used to compress the file. When you want to play this file, you need to have the same codec installed on your computer and PowerPoint has to be able to use it. WAV can use some different codecs, and you need specialised programs, if you want to find out which one a specific file uses. You could then add it to the 2010 computer in case the codec is missing there (please install the single codec, no codec packages. And be aware that this solves the problem only for one computer.). Additionally many codecs used in WAV tend to produce quite large files.

MP3 is a more modern and the currently most widespread file format. The files usually are 10 times smaller than most WAV files. So you could use a audio editing program like the free Audacity to convert your old files to MP3 and re-insert them. They should play well on all Office 2010 computers.

Restrict usage of stationery in microsoft outlook

Can someone tell me whether it is possible for me to restrict the usage of the different types of stationery in ms outlook ? Whether i can setup a default stationery for use by all users on the local network and how to prevent any user from changing  the stationery on his PC.
Is this possible, and if yes, how to do it ?
sue mosher say:
Try using this registry key to enforce stationery usage: 

Key: HKEY_CURRENT_USER\Software\Policies\Microsoft\Office\12.0\Common\MailSettings
Value name: NewStationery
Value type: REG_EXPAND_SZ
Value: Name of the HTML file, without the .htm tag

For example, if the name of the stationery file is “Company Letterhead.htm,” the value for NewStationery would be “Company Letterhead.”

Similarly, a value called NewTheme in the same key controls the default theme if you're using a theme and not stationery. You can create a new .adm file if you want to add either key to the group policy editor.

Although the Organization Chart add-in is included with the 2007 Office programs, it is not automatically installed when you install a 2007 Office suite. To install the Organization Chart add-in, follow these steps:

Exit all programs.

Click Start, click Run, type appwiz.cpl, and then click OK.

In the Add or Remove Programs dialog box, click Change.

Click the 2007 Microsoft Office suite that is listed, and then click Change.

In the 2007 Microsoft Office suite Setup dialog box, click Add or Remove Features, and then click Continue.

Expand the Microsoft Office folder.

Expand the Microsoft Office PowerPoint folder.

Click the button next to Organization Chart Add-in for Microsoft Office programs, and then click Run from My Computer.

Click Continue to install the Organization Chart add-in.

After you install the Organization Chart add-in, follow these steps to open the Organization Chart add-in in PowerPoint 2007, in Microsoft Office Excel 2007, in Microsoft Office Outlook 2007, or in Microsoft Office Word 2007:

On the Insert tab, click Object in the Text group.

Click Organization Chart Add-in for Microsoft Office programs
