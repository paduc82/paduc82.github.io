Deepa Raj wrote, "That is, if first computer crashes, or if any hardware is changed, Office cannot be reinstalled using the same product key. "

 

The above is why there is an official Microsoft policy that prohibits employees from offering "interpretations" of the software licensing terms.

 

The above statement by Deepa is only 1/2 correct:  The PKC license is nontransferable, meaning it cannot be activated on one computer, then uninstalled and activated on another computer.

 

The confusion comes from what constitutes "a computer" especially in a scenario where the computer crashed and parts had to be replaced.

 

Essentially, Microsoft's policy is that only a motherboard replacement that is not the functional equivalent of the original motherboard constitutes "a new computer" in a situation where a computer crashed and had to be repaired.  See the "More Information" section of this MS KB:  http://support.microsoft.com/kb/824125  The More Info section was written when only OEM licenses were permanently tied to the original computer.  Recent new license forms and terms (MLK, PKC, and now all Office 2013) effectively make all Office 2013 licenses, Office 2010 PKC, and Office 2007 MLK, the same as a traditional OEM license with respect to moving the license.

There are two places to set reverse print order. It sounds as if you are accessing the printer Properties through Word's Print dialog.
