KB3097877: remove using Remote Desktop or PSExec - Microsoft Community

dalemahalko say:

KB3097877: remove using Remote Desktop or PSExec

I have a bunch of computers in an organization that are all set to install Windows Updates automatically, and today I am getting hit by KB3097877 breaking several of them all at the same time, flashing at the logon screen over and over.

Fortunately I have discovered that even if the main logon screen is broken, you can still logon to it from another computer via Remote Desktop.

From there: Start -> Control Panel -> Programs -> View installed updates -> Windows updates -> KB3097877 -> Uninstall

If you have Home or Ultimate I don't think this will work as Remote Desktop may not be available on those editions. 

If you don't know the machine's name or network address this is going to be quite difficult to locate, because you can't find the machine name via the usual route at the logon screen. I was able to find it by rebooting the computer and then checking our organization's DHCP server to see what network address leases were renewed recently.

Another option for removing it is using PSExec from another computer. Here too you need to know the machine name, and you need to be able to find it over a common network connection such as a wired LAN:

Psexec -d -s \\remotemachine wusa.exe /uninstall /kb:3097877 /quiet /norestart

PSExec is available from Microsoft SysInternals, and should work for Home / Ultimate:

https://technet.microsoft.com/en-us/sysinternals/psexec.aspx
