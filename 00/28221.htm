ossiemac say:

Try the following. Not sure of your expertise with VBA so have included some guidelines to install.

Excel has a convert function so I have used the function instead of the multiplier.

To install the code:

  1. Right click the worksheet tab name and select View code.
  2. Copy the code below and paste into the VBA editor.
  3. Close the VBA editor (Cross with red background (or background turns red when cursor hover over it).
  4. Save the workbook as Macro Enabled.
  5. Ensure Options set to allow macros. See Help for how to do this. (The option to "Disable all with notification should be OK").

Private Sub Worksheet_Change(ByVal Target As Range)
    On Error GoTo ReEnableEvents
    Application.EnableEvents = False
   
    If Intersect(Target, Range("D8:E25")) Is Nothing Then GoTo ReEnableEvents
   
    If Target.Column = 4 Then
        'Convert function does not have kilometres so use "m" argument for metres and divide by 1000
        Target.Offset(0, 1).Value = WorksheetFunction.Convert(Target.Value, "mi", "m") / 1000
    Else
        'Convert function does not have kilometres so multiply Target value by 1000 before conversion.
        Target.Offset(0, -1).Value = WorksheetFunction.Convert(Target.Value * 1000, "m", "mi")
    End If
   
ReEnableEvents:
    If Err.Number <> 0 Then
        MsgBox "Error occurred in Sub Worksheet_Change"
    End If
    Application.EnableEvents = True
End Sub

jay freedman say:

The macro recorder has always had severe limitations. It should be considered as, at best, a starting point for learning how to write macros ab initio. That's one of the main points in this article.

In this case, the code you need (which is specifically for inline pictures) looks like this:

Sub Resize_5()
    Selection.MoveLeft Unit:=wdWord, Count:=1, Extend:=wdExtend
    If Selection.InlineShapes.Count > 0 Then
        Selection.InlineShapes(1).Width = InchesToPoints(5)
    End If
End Sub

rob rolnick say:

Hi kvan44,

This forum is dedicated to the Office Web Apps on SharePoint, SkyDrive, Facebook, and docs.com.  For questions related to Windows Live Mail you should really ask on  their forum, similarly for questions related to Windows Live SkyDrive you should really ask on  their forum.  By asking in the correct forum, you are more likely to get someone who is knowledgeable with the product/service and capable of giving you both timely and correct advice.

That said, to delete photos from Windows live SkyDrive, you can do the following:

  1. Go to  http://photos.live.com
  2. Sign-In.
  3. On the left column, click on "View all albums"
  4. Click on the album containing the photo(s) you want to delete.
  5. If you'd like to delete an entire album, click on "More" (to the right above the photos) then select "Delete" in the flyout menu and agree to the dialog.
  6. if you'd like to delete a specific photo click on the photo you wish to delete.  Then once it loads click "Delete" above the photo and agree to the dialog.
Hope that helps.
