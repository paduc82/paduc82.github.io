pamcaswell say:

 

1. What does it mean to select the numbers at a given level?  What does it allow me to do?  For these quirky levels, I find that regardless of which set I select a number from, I can make changes that apply to all entries at that level.  That is, any actions I perform don't seem to be restricted to the sub-division of that level.

You can apply direct formatting (such as bold, color, font size) to the  selected number level without affecting the paragraph formatting.  If you set numbering value or restart numbers for the number you actually selected, the change will be reflected in  the other numbers in that level and section.  Selected numbering levels have gray backgrounds in W2010; the  selected number also has a blue background.

 

2. Why did this level get broken into two selection sets like this, and what does it mean?  Is it just a bug?

A restarted or reset number does not get highlighted with the rest of the numbering level. I recall noticing that other things can cause this, but I can't duplicate it today.  As you noted, numbers that do not get the gray background still get the change applied.  Because of this quirk, you can find the numbers that  have restart or reset numbering applied.  So maybe it's a feature.

3. In general, where is there an explanation of how multilevel lists work?  In particular, how can I create multiple multilevel lists that are interleaved; how do I indicate that a given entry belongs to a given one of the multiple lists.

A good guide to outline and heading numbering is here:   http://www.shaunakelly.com/word/numbering/OutlineNumbering.html .  Though it describes the  steps for W2003 & before, the steps are mostly the same except for how you start.  In W2007/10, place the cursor in the first (potential) numbering level 1 in your document (often Heading 1), and click the multilevel list icon and then any list gallery icon that is close to what you want (say 1. Heading 1, 1.1 Heading 2, etc). If you need to customize the list, click the  multilevel list icon again and then Define new multilevel list. 

I am not sure sure what you mean by "interleaved" lists.  But you can have  more than one multilevel list in a document. The built-in  list number and list bullet paragraph styles are  linked to different multilevel lists.  You can create your own multiple lists. I've never done so myself because the nine numbering levels in a list have always been enough.

Note that the separate lists are independent; for example,  a level from one cannot not trigger a restart of a level in the other. You cannot link the numbering levels to the same styles.

4. Maybe related to the above, what are "list styles"?

A list style  is a multilevel list that can be saved and modified.  A highly customized multilevel list that must be used repeatedly is a good candidate for a list style.

 

Pam

  • Check and make sure OneNote is running in the foreground.
  • Logout and log back in to OneNote to reset your credentials and avoid authentication problems.
  • Force a OneNote to sync

Reset credentials

  1. Open OneNote, tap the menu in the upper-right corner of the screen, and choose Settings.
  2. Tap Account.
  3. Tap Sign Out, and then click OK.
  4. Tap the recent apps button on the bottom of the screen (the one next to your home button), find the OneNote app and swipe it sideways to close it.
  5. Start OneNote and sign in to your account.

Force Sync

  1. In OneNote, find the notebook that's experiencing the sync issues.
  2. Tap the menu in the upper-right corner of the screen, and choose Sync all.
  3. Leave the device on until the sync is done. You may have to leave the device on overnight, especially in the case of very large notebooks or first time syncs.

petrib say: You enter the PUK code, when the SIM has been locked by too many attempts at entering the PIN code for the SIM.

When the PUK code is needed, the phone will specifically ask you to enter it. If you want to trigger the PUK code request, enter the PIN code for the SIM card incorrectly a few times. There is no key press codes to trigger the PUK code prompt.

Each PUK code is specific to each SIM card, and has nothing to do with the phone itself.

And the PUK code is also not related to the phone security/lock code (the default for the security/lock code on the phone is 12345 on Nokia phones).
