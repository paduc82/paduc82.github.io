I understand how Word 2010 uses the Normal.dotm as the default template for a new blank document, how you can change the defaults which amends Normal.dotm and how you can create new templates for creating new documents.  However, can you use a custom template as the default template - e.g. when you start Word and it opens with a new blank document, that is based on a custom template rather than Normal.dotm?  Can you do that and how?

 

Thanks.

 

graham mayor say:
The default template is the Normal template. Only Word can create a Normal template and it will do that when the template is missing from the User Templates folder. You can open that template Normal.dotm in Word to make changes to it.

You can create as many document templates as you wish, and it is always a good plan to create templates for particular documents types that you will use regularly and make few changes to the normal template.

suzanne s. barnhill say:

Yes, you can create a shortcut to Word that will open with a new document based on a specific custom template. See the article “How to use startup command line switches to start Word 2010, Word 2007, Word 2003, Word 2002, and Word 2000,” which explains that you can start Word with the /ttemplatename switch. If you create a shortcut on your desktop or Taskbar with a command line including this switch, Word will start with a new document based on the named template. Note these specific instructions:

 

If you specify a path or file name that contains spaces after a switch, enclose the path in quotation marks. For example, if you want to start Word and automatically create a document based on the Contemporary Fax template, the switch should look similar to the following:

     /t"c:\program files\microsoft office\templates\1033\contemporary fax.dot"

 

Therefore, the complete startup command would be similar to the following:

     "C:\Program Files\Microsoft Office\Office\Winword.exe" /t"c:\program files\microsoft office\templates\1033\contemporary fax.dot"

 
Note that the specified template will be used only for the first document created using that shortcut. If you press Ctrl+N or use a toolbar button or menu item to create a new document, it will be a Blank Document based on Normal.dotm.
