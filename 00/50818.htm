a. user say:

Instead of wasting your time with the so-called Microsoft Support Engineer, here is some information about System Restore in Windows 10 that will explain things.

1- System Restore is turned off by default in Windows 10. You can turn it on if you wish. More on that below.

2- Restore points have a limited lifetime, after which they get deleted. Originally that was 90 days. I understand the time period is shorter in Windows 10, but I'm afraid I don't know the exact period.

3- Restore points are also deleted after every major update.

The reason that restore points get deleted (#2 and #3 above) is that they get obsolete real quick. If you mix old restore points into updated Windows you could have a mess on your hands. And the reason they get obsolete is because Windows 10, being a new operating system, is being updated at a rapid pace (for an operating system.) A major upgrade is coming - the so-called Anniversary Upgrade - so you can expect all your restore points to get deleted after that.

System Restore is turned off by default because its usefulness has sharply declined and people never really understood it. As a method of rolling back Windows to an earlier time, it sometimes worked, sometimes didn't work and sometimes made things worse. People often confused restore points with backups. A restore point is definitely not a backup.

Windows 7 introduced system image backup which is a far superior method for rolling back to an earlier time. Windows 10 improved it and added an improved file and folder backup (File History.) Granted, third-party image and backup software gets the job done even better, but System Image and File History are built into the operating system, so there's nothing to install. I would not be surprised if System Restore eventually gets removed.

If you want the best way to roll back after something messes up your computer, learn to make images using either Microsoft's own tool or with third-party software. (My current favorite is Macrium Reflect.) It only takes a few minutes - once you know how - and it's the only form of time travel that actually works. Well, it can't get you the future. Not yet, at least.
