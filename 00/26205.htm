For example, if I have a task that is expected to take 5 days and I only have one person available 50% of the time. Is there a way to show that this task is going to take 10 days to complete the 5 days worth of work? Let me know if I need to elaborate and I will!

john - project say:

Dan,

First of all, I think you may be getting duration and work confused. In Project they are two different things. Duration is the time span during which a task is performed. Work is the effort one or more resources will expend accomplishing the task.

Project uses what is called the work equation to schedule tasks

Duration = Work/Units

In your case, I believe you mean you have a task that has 5 days worth of effort (i.e. work) to complete. If a single resource is assigned full time, that task will take 5 days (duration). However, in your case your resource is only available 50% so for a fixed work task, assigning that resource will lengthen the task automatically to 10 days (duration).

So to answer your question, yes, if you set the task Type to Fixed Work and first set the Work field to the desired value, when you assign a resource at 50%, the duration will automatically be adjusted per the work equation.

Hope this helps.

John

' To allow multiple selections in a Drop Down List in Excel (without repetition)
Dim Oldvalue As String
Dim Newvalue As String
Me.Protect UserInterfaceOnly:=True
Application.EnableEvents = True
On Error GoTo Exitsub
If Target.Address = "$E$25" Then
  If Target.SpecialCells(xlCellTypeAllValidation) Is Nothing Then
    GoTo Exitsub

    Application.EnableEvents = False
