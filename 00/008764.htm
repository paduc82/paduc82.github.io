jay freedman say:

There is a Unicode character đ with the character number hex 0111. It's present in most fonts that contain more than just the 128 ASCII characters, such as Cambria Math, Times New Roman, and Arial Unicode. To insert it, you can use the Insert > Symbol > More Symbols dialog, or just type the number 0111 in the document and immediately press Alt+X.

In regular text you can apply italic formatting to the character. In an equation control, unfortunately, I don't see any way to italicize it to match the ħ character.

Things are going to be different for Windows 10. During the initial upgrade from Windows 7/8.1 to Windows 10 your devices hardware will be scanned and submitted to a Microsoft Product Activation Server.

Windows 10 has an inbuilt feature to make a Windows 10 Recovery Drive. It allows for a vanilla Clean Reinstallation (locked to Edition, Architecture). suzanne s. barnhill say:

If the line numbers have been created using the Line Numbers feature in Word, you can remove them just by disabling this feature (on the Layout tab of Page Setup). More likely, however, if the transcripts were prepared by legal transcriptionists, the line numbers are in a text box anchored to the header. Access the header by double-clicking in the header area; select the text box and delete it.

As for the Q's and A's, if the transcriptionist was knowledgeable, she might have used Q-and-A styles (as described in http://WordFAQs.mvps.org/QandA2007.htm), using Q and A as bullets. If so, you could remove them by modifying the styles not to have these bullets. If the Q and A were typed, however, then you can just use Replace to search for Q and any following space (a tab character, for example) and replace with nothing, then repeat for A and following space.
