rich michaels - mvp (greatcirclelearning.com) say:

Pardon my jumping in but I can't stay quiet on this...

Regarding an AutoRecovery file that you can't open... use Finder and make a copy. Rename the copy taking the file extension off of the name. Thus if the file is "AutoRecovery My File.docx"... remove the ".docx" from the name. Ignore the warning the system will give you.

Now, from Finder right click (A.K.A Control + Click) on the file and use Open With... to open the file with Word. The file will open as Read Only and just Duplicate it so you can get a new working copy.

Now some personal advice...

  1. I don't trust the default AutoRecovery feature and its unspecified "secret" location. I prefer to assign AutoRecovery files to a location that I specify.
  2. With MacOffice 2016 the location must be within the "Sandbox" designated by Apple. I use: ~/Library/Group Containers/UBF8T346G9.Office/User Content/AutoRecoveryDocs
  3. Open Finder and press Shift+Command+G and a GoTo box will open. Paste the above bolded path, including the tilde "~" symbol into the box, remove the "AutoRecoveryDocs" folder name and click Go.
  4. Now create the "AutoRecoveryDocs" folder as a subfolder in the "User Content" folder.
  5. Open Word > Preferences > File Locations and set the AutoRecovery files path to this location. The path will will look like /Users/<your login name>/Library/Group Containers/UBF8T346G9.Office/User Content/AutoRecoveryDocs
  6. Set your AutoRecovery time period to no more than 10 minutes but not less than 3 minutes.
  7. Quit and restart Word, allowing the new setting to be stored in Word's internal preference file.
  8. Finally, make sure you don't just rely on AutoRecovery as a "backup" function, because it's not. It only works if Word crashes. If you accidentally write over something you didn't mean to... you are SOL... so...
  9. Turn on Word > Preference > File Save > Always Create a Backup so at least you have one previous copy before you accidentally destroyed your file and every time you pause to think as you are editing, press Command+S to save your file.
  10. However... if your files are REALLY important to you then download GUB (Zapps Pro GOT-UR-BACK) from my website. It FREE so I'm not trying to sell you anything. In MacOffice it will optionally keep (your choice) up to 5-levels of backup on Word and Excel documents.
  11. It's important for you to understand, this is not an "auto" backup, it is a file overwrite protection mechanism. Every time you click Command+S or click the Save button it stores the previous version of the document you are editing, and it will store up to 5 of them. They are held in a convenient folder that you can access and do not clutter up your Documents directory. If you use GUB, you turn off Word's "Always Create a Backup" feature.

Hope this helps
