Hi,

Win 8, 8.1, and 10 have a new feature called Fast Startup. It can sometimes cause problems like you mention as well as others.

The easiest way to tell if this is the problem is to turn it off and see what happens.

This feature only works when you do a SHUTDOWN and then Boot.

It doesn't effect a RESTART.

To turn it off or on...

Go to Control panel...Power Options, and select Choose What the Power Buttons Do on the left.

Then select Change Settings That Are Currently Unavailable near the top center of screen...

Lower down on the window, uncheck Fast Startup.

This may or may not help your problem, but is easy to check.

If this resolves the problem, then you can investigate what isn't loading properly.... most likely a driver.

hansv mvp say:

Here are two user-defined VBA functions:

Function MaxBlanks(rng As Range) As Long
    Dim n As Long
    Dim c As Range
    For Each c In rng
        If c.Value = "" Then
            n = n + 1
            If n > MaxBlanks Then
                MaxBlanks = n
            End If
        Else
            n = 0
        End If
    Next c
End Function
 

    Dim n As Long
    Dim c As Range
    Dim f As Boolean
    MinBlanks = rng.Count
    For Each c In rng
        If c.Value = "" Then
            n = n + 1
            f = True
        Else
            If n > 0 And n < MinBlanks Then
                MinBlanks = n
            End If
            n = 0
        End If
    Next c
    If Not f Then
        MinBlanks = 0
    ElseIf n > 0 And n < MinBlanks Then
        MinBlanks = n
    End If
End Function

Use like this to calculate the maximum number of consecutive blanks in E2:E26:

=MaxBlanks(E2:E26)

And the minimum number of consecutive blanks:

=MinBlanks(E2:E26)
