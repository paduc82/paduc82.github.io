jack cook - aka help_line say:

So this is what I would do if it were my device. Keep in mind that the internal memory of your device is only 8GB and that can be used up very quickly. the fastest and probably the most efficient way would be to hard reset your device and start fresh.

I would uninstall any programs you may have already installed on your SD card and just keep Music, Photos, and Docs on it if you have them already installed. I would then Hard Reset and NOT install from a backup but rather start fresh. Use the same Live ID so you have access to all your Contacts, Calendar and Photos you have backed up to OneDrive (I assume you have enabled auto upload of photos via Wi-Fi). If you have not saved photos and Docs to OneDrive, do it if they are important to you.

Once it is Hard Reset, only install those programs you want in Main Memory then under Settings, System, Storage change your storage location from Main Memory to SD card and install those rarely used programs to SD card. Keep in mind that even though you are installing to SD card, parts of each program have to be installed to Main Memory. I would be selective on how many you install.

If you do it carefully you will be good to go. Just one final point: Before you install anything after it is initially set up, check and see if your device has any updates to install. If you do, they will install and then install programs.

Your device is a good device but keep in mind it is an entry-level device and does not have infinite memory. Just install the important things. Unimportant games etc install, use then uninstall once you are done with them. Don't fill it up with junk you will rarely use. Just my thoughts....

tryx3 say:

Keenan,

The symptoms you describe sound to me to be consistent with requiring updates to display drivers & the BIOS, if any have been issued.  I suspect that the computer on which your colleague recreated the user profile later developed the symptoms again.  If it did not then do not bother reading this post.

You cannot rely on Windows for hardware updates.  It often gets them wrong.  The reliable solution is to go to the computer maker's website then download & install the latest display card driver & any BIOS update.

Your question is a valid question for this forum.  You might end up getting help in TechNet but this is the appropriate forum.

  • TechNet forums are intended for IT admins such as yourself to correspond with other IT admins about IT admin issues.
  • Similarly, MSDN forums are intended for developers to correspond with other developers about development issues.
  • This MSCommunity / MSAnswers forum is intended for consumer [end user] issues such as the problem you reported.

I assume, by the way, that both Windows & Office are fully up-to-date. 

  • If you have computers running Windows 10 then they also need hardware drivers from the computer maker to be routinely kept up to date [much more than was the case with, say, Windows 7].
  • Windows 10 is much more closely integrated with its hardware and consequently the demands it makes on hardware are much more intense & the need for changes [to suit Windows updates] happens much more frequently. 
    • On a Windows 7 computer from 2010 I had one display card driver update in the whole of the first two years. 
    • On a new Windows 10 computer from the same manufacturer I have a display card driver update every other month.
