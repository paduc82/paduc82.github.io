bob jones aka: cybertaz say:

No, you cannot renumber the rows... Besides, the row number is not an integral part of the record on the row, so if you sort the records, row n remains row n even though it displays different data. It also would play havoc with the sheet in other respects, as well.

Without resorting to a programmatic method the easiest way is to include a (e.g.) Record Number field as the first column & enter the number just like any other data. If the numbering is to be sequential you can use the Fill Handle or Edit> Fill> Series.
