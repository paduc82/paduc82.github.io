I know its very easy with data validation, but user has to go in to drop down and select. But here my user wants to have just click on desired cell which is visible and my report should be updated.
hansv mvp say:

Selecting an item from a dropdown list is a deliberate action. Selecting a cell can easily done by accident, so it is error-prone.

 

Right-click the sheet tab.

Select View Code from the context menu.

This opens the worksheet module in the Visual Basic Editor.

Copy the following code into the module:

 

Private Sub Worksheet_SelectionChange(ByVal Target As Range)
    If Not Intersect(ActiveCell, Range("A1:C1")) Is Nothing Then
        Range("H1").Value = ActiveCell.Value
    End If
End Sub

 

Switch back to Excel.

Save the workbook in a format that supports macros: .xlsm, .xlsb or .xls, but not .xlsx.

Make sure that users enable macros when they open the workbook.
