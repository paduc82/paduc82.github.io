Drop down menus in onenote - Microsoft Community

taddraper say:

Drop down menus in onenote

I was wondering if there is a way to transfer dropdown menus created in Word to Onenote or to create dropdown menus in OneNote.
Thanks for the help,
Tad
omer atay say:

If I understand correctly you want a dropdown in the ribbon which will have a bunch of items in it and when you click on one of them, it will insert some pre-defined text at the cursor position? If that is what you want, you can do that with a macro using the Onetastic addin.

 

Here are the steps:

  1. Go to http://omeratay.com/onetastic
  2. Download the Onetastic addin, and install it (pay attention to 32 bit vs. 64 bit)
  3. Start OneNote
  4. Click on New Macro button on the Home tab
  5. Click on Import
  6. Paste the following:

<?xml version="1.0" encoding="utf-16"?>
 <Macro name="Goal 1" category="Goals" description="Insert Goal 1">
   <ModifyVar name="TextToInsert" op="set" value="Goal 1" />
   <For first="Text">
     <That hasProp="selected" op="eq" value="true" />
     <ModifyVar name="TextToInsertAndSpace" op="set" var="TextToInsert" />
     <ModifyVar name="TextToInsertAndSpace" op="add" value=" " />
     <ModifyProp name="value" op="set" var="TextToInsertAndSpace" />
   </For>
   <For each="Text">
     <That hasProp="selected" op="eq" value="true" />
     <That hasProp="value" op="eq" var="TextToInsert" />
     <ModifyProp name="selected" op="set" value="false" />
   </For>
 </Macro>

 

7. Click OK then Save, then close the Macro window.

 

Now you will have a drop down named Goals, and Goal 1 under it, clicking on it will insert Goal 1 at the cursor position. To change the inserted text, edit the pasted XML where it says Goal 1 (in three places) and put the goal text you want. To add more of these, change the XML again and create new macros. You need to create macros once, and then you will have your dropdown there forever.

 

Let me know if this is what you want and if that works for you or not.

Try this ARRAY formula , see below on how to enter an ARRAY formula.

=COUNT(1/FREQUENCY(IF(LEFT($A$1:$A$7000,14)="Direct Consign",IF($C$1:$C$7000<>"",MATCH($C$1:$C$7000,$C$1:$C$7000,0))),ROW($C$1:$C$7000)-ROW($C$1)+1))
