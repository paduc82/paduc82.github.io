Start slack and Finish slack - Microsoft Community

basavaraja kumbar say:

Start slack and Finish slack

Hi, Hello

I want to know the difference between the ''Start slack and Finish slack".

I am getting the same values for both the fields.

If anyone knows then please revert...

Thank you.

tom_bpc say:

Start Slack is (Late Start - Early Start), while Finish Slack is (Late Finish - Early Finish).  The early dates come from the forward-pass calculations, while the late dates come from the backward-pass calculations.  These are described in any basic project management textbook that addresses network scheduling.

In Project, start slack and finish slack are typically the same unless the task is in progress.  Here are some basic observations:

1. For 100% completed tasks, the early and late dates are overridden by actual dates, so start slack and finish slack are zero.

2. For in-progress tasks, the early and late start are overridden by the actual start, so start slack is zero.  Early and late finish dates are separately calculated according to the project logic, so finish slack is typically non-zero.

3. For un-started tasks, both start slack and finish slack are computed according to the project logic - these are always the same since the task duration is a constant.
