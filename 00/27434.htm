max, singapore say: Try this in a copy of your source data sheet ...
Insert a new col A, your names (with duplicates) are now running in B2 down
Put this in A2: =IF(B2="","",IF(COUNTIF(B$2:B2,B2)>1,"","x"))
Copy A2 down all the way, this will flag all unique names with an "x"
Apply autofilter on col A, choose: x to filter out only the unique rows
Copy the filtered data set, then paste special as values into a new sheet (Delete col A in the new sheet) 
That's it, you have reduced the original source data to one containing only the unique lines to use for all your downstream analysis, etc

Hi downingtownryan7759,

I notice only one user account could not be migrated. Could you perform the following steps to see if there is any improvement?
