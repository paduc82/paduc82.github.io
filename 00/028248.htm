Going back to your original question you say  "(I am using the shape as a button)". Does this mean that you have a macro assigned to the Click event of the shape. If so, then when you click the shape it does not select but it does run the assigned code.

If above assumption is correct then you can assign as many shapes as you like to the same macro code and identify the shape that was clicked with Application.Caller.

Sub Shape_Click()
    Dim strText As String

    Set shp = ActiveSheet.Shapes(Application.Caller)

    strText = shp.TextFrame.Characters.Text
    Range("e23") = strText
   
End Sub

Build 10162 "Failed to play test tone" slight workaround - Microsoft Community

halosheep say:

Build 10162 "Failed to play test tone" slight workaround

So I, like many others, have had an issue with audio once installing 10162. Here's how I (kind of) got around it and got my headphones working. TL;DR: disable all enhancements for the audio device and it should work. Here's how I did it:

1- Make sure you have the latest driver installed (default drivers seem to work as long as it's a fresh install)
2- Go to Control Panel > Hardware and Sound > Sound

3- Right click on the device that isn't working and open properties (mine are all just generically labeled "Speakers", except for my headphones)
4- Go to the enhancements tab and check "disable all enhancements", even if you have none of them on in the first place

Go back and test the device and it should play audio. Don't mess with the default audio format or you'll have to do it this again for it to work. You can actually re-enable enhancements after you've made sure the device is working again and it seems to run fine.

This really only fixes it if your problem stems from the default audio format being changed.
Hopefully this helps some people, and hopefully it works for everyone!

I tested this with my Corsair Vengeance 2100 headset and my default speakers.

If you mean that you want to create a custom stationery for Outlook 2016 client, please follow the instructions below:

 

1 . Create a new message, then set the background, font, etc

2 . Click File->Save as. In the Save As dialog box address bar, enter %appdata%\microsoft\stationery, then press Enter.

Formula for sum of highlighted cells - Microsoft Community

amy dehart say:

Formula for sum of highlighted cells

What is the formula to Sum only the cells with a specific fill color?
mike h.. say:

Hi,

To date no version of Excel can do that with inbuilt function so we need to create our own. ALT+F11 to open VB editor, right click 'ThisWorkbook' and insert module and paste the code below in on the right. Close VB editor.

back on the worksheet call with this

=SumColour(A1:A20,B1)

Where A1:A20 is the range to sum and B1 is a cell that has the same fill colour that you want to sum.

NOTE. Changing a cell colour does not make Excel calculate so you will need to force recalculation by tapping F9. Otherwise this will calculate whenever Excel does.

Function SumColour(rng As Range, clr As Range)
Application.Volatile
Dim c As Range
For Each c In rng
    If c.Interior.ColorIndex = clr.Interior.ColorIndex Then
        SumColour = SumColour + c.Value
    End If
Next
End Function
