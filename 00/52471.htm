In Windows 10, you can now set DPI (Dots per Inch) scaling to make text and other items, such as icons, fonts, and windows, to display larger or smaller for all displays or per-display.

Windows 10 DPI scaling improvements are primarily focused on:

  • Optimizing the usability and readability of high-DPI displays

  • Providing a uniform experience multi-display systems

  • Empowering developers to optimize app-specific scaling based on display DPI

    If you are running a screen resolution of 1366x768 on a tablet, chances are that UI will look good at 100% DPI settings. But what about when you connect that tablet to an external high resolution display? In Windows 8 you can choose either 100% to optimize the UI on the tablet display or up to 150% to optimize the UI on the external display. You have to compromise.

    Windows 10 takes care of this issue by supporting per-display DPI scaling to address. By default, Windows 8.1 will choose the optimum DPI scaling value for each display based on the value chosen for overall scaling in the control panel.

Hi Ivan,

 

Perform the steps mentioned below to create a recovery image.
a: Press Windows(logo) + X
b: Go to Command Prompt(Admin)
c: Then type mkdir C:\RefreshImage click Enter
d: Then type recimg -CreateImage C:\RefreshImage click Enter

e: Waiting for the  ISO Image of Windows 8 to create until finished
f: After that Restart your computer
g: Then go to Computer>Local Disc(C:)>RefreshImage>CustomRefresh.wim>right click to copy that ISO image of Windows 8 to the preferred devices that you would like to keep that ISO image
