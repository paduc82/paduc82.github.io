andreas killer say:
The VBA.Rnd function supports a seed, therefore you can follow this steps:

Right-Click the sheet tab
Choose "View Code"
Within the menu click Insert\Module
Paste in the code below
Close the VBA editor 

Function RandSeed(Seed As Variant) As Variant
  RandSeed = Rnd(Seed)
End Function


Make a new sheet and setup the cells:
A1: Seed
B1: Multiplied
C1: Random number
A2: 0.5284
B2: =A2*997
C2: =RandSeed(A2)
A3: =B2-INT(B2)
B3: =A3*997
C3: =RandSeed(A3)

Select A3:C3 and drag down as needed.

You are not handling the resource assignment correctly.  Here is what you need to do:
  1. Apply the Gantt Chart view, if not already visible.
  2. Click the View tab to display the View ribbon.
  3. In the Split View section of the View ribbon, select the Details checkbox. Microsoft Project 2010 displays the Task Entry view, with the Gantt Chart view in the primary pane (top pane) and the Task Form view in the secondary pane (bottom pane).
  4. Select the task to which you want to assign the resource at 20% Units.
  5. In the Task Form pane, select the resource from the pick list in the Resource Name column.
  6. In the Units column for the resource, enter 20%.
  7. In the Work field, enter the amount of work you estimate the resource will perform on this task.
  8. Click the OK button.
The system will now calculate the Duration of the task using the Units and Work values you supplied.  Hope this helps.

benhoran- [o365] say:

OK... Using both the download URL available to me for the Visio 2016 installer from within the my Office 365 portal, and the info gleamed from the above thread on downgrading the main Office 2016 distribution, I managed to piece together the URL to download the Visio Pro 2013 installer.

English (US), 32-bit:  c2rsetup.officeapps.live.com/.../download.aspx

English (US), 64-bit:  c2rsetup.officeapps.live.com/.../download.aspx

Basically, the only thing that needs changing in the download link is to change "version=O16GA" to "version=O15GA" to get the 2013 installer rather than the 2016 version.

As per the previously listed threads, other languages can be downloaded just by changing the "language" parameter (EN-US in the above links), and the difference between 32 and 64-bit versions is controlled by the "platform" parameter.

I hope this helps others out if they get caught in the same situation of having to downgrade back to Office 2013, and lose Visio in the process.
