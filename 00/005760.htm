stephen boots say:

I suppose you need to understand what an antivirus program, antispyware program, and firewall are.

 

Antivirus is a program that monitors all activity on the PC -- memory, disk access, I/o -- for virus behavior and/or signature match against a database of known malware. It performs two functions: real time protection and scanning, manually or on a schedule.

 

Antispyware programs might also provide similar monitoring, but it typically looks for specific activity that attempts to "spy" or gather information, redirect searches, etc.

 

A firewall prevents access to your computer over a network connection that is not a response to your request. A firewall can also block access from the computer outbound, but the Windows Firewall does not do this by default.

 

Some antivirus programs also perform antispyware functions, Microsoft Security Essentials is on, so it disabled Windows Defender to prevent a conflict.

 

Some security suites by 3rd party vendors will include all 3 of the above, so will disable Defender and the Windows Firewall so that there is no conflict.

 

 

You should never have more than one security product installed on the PC providing active protection/scanning. This can cause performance issues, system instability, and can hinder the effectiveness of both products at providing protection.

 

EDIT:  I don't know what you mean by "even more programs on their way for updates?" However, Automatic Updates for Windows and Microsoft Programs should be enabled so that security patches (fixes problems) can be deployed as soon as they are available.

    Sub Sheets_Count_2()
    'count all visible sheets, and exclude 2 specific sheet names ("Confidential1" and "Confidential2")

    Dim counter As Long, sh As Worksheet
    counter = 0
    For Each sh In Sheets
        If sh.Visible And sh.Name <> "Confidential1" And sh.Name <> "Confidential2" Then
            counter = counter + 1
        End If
    Next sh
    MsgBox counter

    End Sub
