Assumptions:

1) you want all results in Sheet1:A:B to be Constant1*Sheet# where # is the last digit of the sheet name.

2) Sheet1!E5 will contain the sheet name like "Sheet2" or "Sheet3"

The formulas are:

* In Sheet1:A2 enter: =$E$1*INDIRECT($E$5 &"!" & "A" & ROW()-1) 

* In Sheet1:B2 enter: =$E$1*INDIRECT($E$5 &"!" & "B" & ROW()-1) 
* Copy both down as many rows as needed

How this works:

* ROW() This is the row of the formula so in A2 this = 2. 

* ROW()-1 The data on Sheet2 starts in row 1, which is 1 higher than the results start row

* $E$5 &"!" & "A" & ROW()-1 This creates a string that looks like Sheet2!A1 which holds the address of the coordinate

* INDIRECT() This converts string addresses to cell references
