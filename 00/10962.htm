dr. nitin paranjape say:

Just to clarify a little more:

Aligning requires at least two objects.

If you have only MORE THAN ONE object selected
In this case PowerPoint gives you a choice.
Now Align Selected Objects and Align to Slide options will be available.

Now you can choose whether you want to align selected objects to each other
OR
Align ALL selected objects with respect to the slide.

In this situation this is like a radio button - Choose Align to slide OR Align Selected Objects

If you have only ONE object selected
Then Align to Slide will be shown as selected.
The Align Selected Objects will be grayed out (because it is irrelevant).

In this case it means that you can align the object to the slide itself.
For example, select a single object and then click Align Middle and then Align Center.
This is the fastest way to put any object EXACTLY in the center of the slide.

At this point there is only one option available.
It is like having two radio buttons in a group and one of them being inactive.
Therefore, you are forced to choose the active option.

I do agree that it is slightly non-intuitive.
However this is more of a UI design constraint because you could not possibly put a radio button in a dropdown menu.

Please note that in earlier versions of Office (2003 or earlier), if you had only one object selected, all the align options would be inactive except Align to slide. This was a toggle and it was unselected by default. Very few users understood that they had to click the Align to Slide option. Therefore since 2007 version this UI was improved.

rohn007 say:

The problem is not Click to (not) Run, you have run into the 32/64 bit clash.

When MS recompiled Office applications to create 64 bit versions the various program files had the same file names, no surprise. 

When MS created the 64 bit installer, they cheaped out and install the 64 bit program files to the same locations as 32 bit Office installs.

This means you CANNOT install 32 and 64 bit versions of ANY Office programs or components (like the Access run time). The installer checks, and displays the message you see.

The Access run time appears to be 32 bit.

MS changed the default install to 64 bit earlier this year. Uninstall and Re-install is just installing 64 bit Office.

Since there does not appear to be a 64 bit Access Run Time, you have to uninstall Office and explicitly install 32bit Office

Use the Office offline installer MS

https://support.office.com/en-us/article/use-the-office-offline-installer-f0a85fe7-118f-41cb-a791-d59cef96ad1c?ui=en-US&rs=en-US&ad=US

The Offline Installer should allow you to install Office without an internet connection (after the file is downloaded). This page also gives you the option to explicitly select 32 or 64 bit installation.
