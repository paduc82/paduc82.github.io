Hi RT,

The following adaptation worked for me:

'============>>
Option Explicit

'------------>>
Public Sub Tester()
    Dim WB As Workbook
    Dim SH As Worksheet
    Dim Rng As Range
    Dim iLastRow As Long
    Dim lngStart As Long
    Dim lngEnd As Long

    Set WB = ActiveWorkbook
    Set SH = WB.Sheets("SheetB")
    iLastRow = SH.Cells(Rows.Count, "A").End(xlUp).Row

    With SH
        Set Rng = .Range("A1:I" & iLastRow)
        lngStart = Range("AB9").Value
        lngEnd = Range("AC10").Value
    End With

    Rng.AutoFilter Field:=3, _
                   Criteria1:=">=" & lngStart, _
                   Operator:=xlAnd, _
                   Criteria2:="<=" & lngEnd
End Sub
'<<============

I have removed the +1 and -1 from your date criteria as the use of the  >= and <= operators caters for your requirement to include the start  and end dates in the autofilter output. Additionally, I saw no reason to adjust the dates to delete the time values. Note also that rather than define the range as extending to row 65000, I have used column A to establish the last row of the autofilter range.

The above code successfully extracted all data in the specified range where the date in column C matched the criteria dates.
