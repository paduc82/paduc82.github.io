keratl say:

If you put your original code in the userform_load event of the userform, it would work as expected. However, once your form is loaded, that label caption doesn't know if something else has changed (in other fields on the userform) unless you write an event handler that uses some event to refresh your caption.

For example, if you have state in a Combobox linked to cells in a worksheet, then that combobox will have a combobox_change event, into which you could put your

Label1.caption = "Label caption here" & FindScriptInfo(FaxNumber) & "."

and the label would update each time the combobox value is changed (assuming your case statement included the selected state)
