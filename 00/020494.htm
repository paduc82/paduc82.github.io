jack cook - aka help_line say:

Don Sorcinelli ( from Boston Pocket PC) gave this response on a previous post.  I hope it helps!

 

To start, there are some types of Exchange folders that will never show up on your device. They are -

 

1) Public folders;

2) Archive folders;

3) Other than the default, folders designed to hold items other mail messages (i.e. - a second calendar folder).

 

In addition, not all folders are synchronized to the device by default. To view the sycnrhonized folders, you should open Messaging -> Microsoft Outlook -> Menu - >Folders. If a folder in question is not displayed there, you can then go to Menu -> Manage Folders. From that screen you can add other folders to synch.

 

Some additional notes -

 

1) Remember that additional synchronized folders mean more space required on the device;

2) If your Exchange Server is Exchange Server 2007, you can fogro the additional synchronizing and take advantage of the new "Server Search" functionality. Simply select Menu -> Tools -> Server Search while in the Inbox folder. You have the ability to enter keywords, date ranges and either Inbox, Sent Items or All Folders for the search area. Search results are displayed and you can view/interact with any results you find. Personally, this feature alone was compelling enough to make me move from a hosted Exchange 2003 provider with no plan to upgrade to a new hosted provider with Exchange 2007.
