brian tillman [outlook mvp 2007-2019] say:

Typing in the To field does not reference your contacts in any way. You are referencing a cache of addresses to which you have sent mail in the past. Clearly some of those will be contacts, since presumably you've emailed some of them in the past, but other entries will be for those one-off messages you've sent as well.

If the next time you send a message to a person whose name doesn't appear when you start typing, add that contact to the message by clicking To to the left of the recipient field and choosing him or her from the Address Book window (which is a view into your Contacts folder). Now that you've sent mail to that person, their name and address will be remembered in that cache and the next time you start typing in the To field with the letters that begin that name, that person will be included in the list that pops up.

Add new contacts by putting them in the Contacts folder. Once you've sent the first message to that person, that person's name and address will be added to the cache as well.

WIRED INTERNET CONNECTION TO MOBILE PHONE - Microsoft Community

abhyudayalva say:

WIRED INTERNET CONNECTION TO MOBILE PHONE

I'm using lumia 535 with windows 8.1 and windows 7 on my PC. Two questions-

1. How do I connect my phone to broadband through my PC ? (is it possible?)

2. Can I connect my phone directly to the USB port of modem and get internet access on my phone ?  ( Is any software required for this?)

albuquerquefx say:

Our resident Microsoft rep is exactly right: these phones only perform data connections via WiFi.

 However, for the sake of simplicity, he perhaps chose to gloss over possible options involving your PC.

Does your PC have a wireless card, even if you do not use it? If so, depending on the Operating System installed, you may have a way to create an "Ad Hoc" wireless network that could then share your wired internet connection to the phone wirelessly.

Let me give an example:

Assume that your PC is running a Windows 7 operating system.

Assume that your PC has a wired network connection to your internet provider (cable, DSL, satellite, tin cans and string, etc.)

Assume that your PC has a wireless card that is not currently being used.

With those assumptions in place, you can perform the following steps:

Click the start button, and immediately type the words ad hoc. Your start menu will show an option at the very top that says Set up an ad hoc (computer to computer) network.  Click on that option.

Click next at the "Set up a wireless ad hoc network" screen.

Per the instructions on the next page, give your wireless network a name and a security key (password.) Optionally, if you're going to do this often, click the box that says "Save this network."  Click the next button when complete. Click finish at the completion window.

You have now created a wireless network, it will appear in your list of available networks if you click on the network icon in your task tray. If you "connect" to that network on your PC, it will start the ad-hoc network for other devices to join in. If you "disconnect" from that network on your PC, it will turn off the ad-hoc network and disconnect any other devices that may have joined.

Before you get excited, you need to perform one more step: share your internet access with that new ad-hoc network.

Click the start button, and immediately type the words network connections. Your start menu will have an option at the very top that says View Network Connections. Click on that option.

In this window, find your Local Area Connection adapter that shows connected to the internet. Right-click that adapter, select Properties. In the properties window, click the Sharing tab, and enable the checkbox that says "Allow other network users to connect through this computer's internet connection". From the drop-down menu, find your ad-hoc network and select it. CLick OK when done.

Your phone can now join the wireless network (look for the wireless network name that you created and use the password that you specified. Once your phone has joined that wireless network you created, it can further interact with the internet.

Other operating systems will handle this differently. Windows Vista is nearly the same, XP requires a few tweaks. Windows 8, insofar as I'm aware, isn't capable of this feature but there are free apps that make it work. I'm clueless about Mac operating systems, but I'm reasonably confident that OSX can do this too.

 brian tillman [outlook mvp 2007-2019]  say:

    


            
            
            
            
Outlook doesn't really have an address book.  It uses Contacts folders and then makes the list of addresses in those folders available for entering into email messages via a view called the "Outlook Address Book"


By "using the White Pages", I assume you mean 
http://www.whitepages.com/.  When you click "Save to Outlook":, the web page creates a vCard (vcf) file.  VCards open as Outlook contact records and you can add it to your Contacts simply by clicking Save and Close.


However, the vCard created by the White Pages service does not contain any email address for the person you're adding.  Since the whole point of the "address book" in Outlook is to provide email addresses for adding to email messages, unless the contact
 record contains an email address, it will not appear in the address book view.  Since you can't send email without an email address, what would be the point?  It will still be in your Contacts folder, however, and you can reference it any time you wish to
 obtain the phone number or street address.

            Two things stand out:



1.  I assume that you want to test for the Internal Counterparty Facts CP Type column having a value of "E" or being Null.  If so the IIF function call should be amended as follows:



    IIF(Nz([Internal Counterparty Facts CP Type],"E") = "E" And [Ms Compression] etc....



The Nz function will return "E" if the column is Null, so this part of the expression will evaluate to True if the value in the column is "E", or if it is Null.

 of "No" or "Yes" only, you really should be using columns of Boolean (Yes/No) data type.

Hi Zoran,

Share this information to assist you better.


    

  1. What steps you tried to switch accounts?
  2. Have tried any troubleshooting steps?


This issue could occur due to some incorrect settings or corrupt user profile files. In this case, I would suggest you to create a new administrator user account and check if the same behavior is noticed. If yes, then try these steps and check:


    

  1. Open an Administrator CMD window.  If you can’t find a way, use these steps:

  2. Press CTRL+Shift+ESC to open Task Manager. 
  3. Click File > Run New Task 
  4. Make sure you have a check mark beside “Create this task with administrative privileges”

  5. Type CMD 
  6. Type the following commands at the CMD prompt: 
    
    dism /online /cleanup-image /restorehealth
    
    sfc /scannnow     
  7. If EITHER command does not show a good outcome, we need to stop and repair Windows. We have 3 options:

  8. Reset: If possible, remove Antivirus/Sec Software and perform a Reset (keeping data and settings). users will need to reinstall apps with this option.

  9. If the above commands both completed and show no remaining issues, run these commands:

    
powershell
    
    Get-AppXPackage -AllUsers |Where-Object {$_.InstallLocation -like "*SystemApps*"} | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register "$($_.InstallLocation)\AppXManifest.xml"}    

  10. Check to be sure this command ran, showed a blue progress indicator and then returns to the prompt.

  11. If you see a red error, then the command may have been typed or pasted incorrectly, or there may be an larger problem with Windows 10.  Try again.

  12. Check to see if the issue is resolved. 
  13. If not, restart the PC.In this case try as a new administrator user account.



Hope this information was helpful and do let us know if you need further assistance. We will be glad to assist.
