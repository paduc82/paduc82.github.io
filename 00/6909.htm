jeeped say:
The answer is both â€¦

The best long term solution would be to consolidate the inventory into some form of universal table that standard formulas could be run against, but you are definitely going to run into some stumbling blocks with the alphabetic S, M, L, etc vs. the numeric 7.5, 8, 9, etc. Not only are they letters vs. numbers, but the question remains as to whether to treat  7.5, 8, 9, etc. as numbers or textual representation of numbers.

One of the biggest problems with multiple tables per sheet is in the phrase "the closest row above the product code â€¦". Excel simply does not like to work in a backwards direction. This fact together with the requirement to examine multiple worksheets leads me to believe that a User Defined Function or UDF is going to be the best (if not only) solution to your current situation.

Tap Alt+F11 and when the VBE opens, immediately use the pull-down menus to Insert, Module. Paste the following two UDFs into the new pane titled something like Book1 - Module1 (Code).

Public Function fcnStockBySize(vCd As Variant, vSz As Variant, ws1 As String, ws2 As String) As Variant
    Dim w As Long, r As Long, iSiz As Long, vStk As Variant
    On Error Resume Next
    For w = Sheets(ws1).Index To Sheets(ws2).Index
        With Sheets(w)
            vStk = WorksheetFunction.Match(vCd, .Range("A1:A9999"), False)
            If Err.Number = 0 Then
                For r = vStk To 1 Step -1
                    If .Range("A" & r).Text = "Style" Then Exit For
                Next r
                iSiz = WorksheetFunction.Match(vSz, .Range("A1:Z1").Offset(r - 1, 0), False)
                If Err.Number = 0 Then
                    vStk = WorksheetFunction.Index(.Range("A1:Z9999"), vStk, iSiz)
                    If Err.Number = 0 Then
                        fcnStockBySize = Int(vStk)
                        Exit Function
                    End If
                End If
            End If
        End With
        Err.Clear
    Next w
End Function

Public Function fcnSizesByStyle(rCd As Range, ws1 As String, ws2 As String) As Variant
    Dim w As Long, r As Long, vStk As Variant, vSiz As Variant, iCnt As Long
    On Error Resume Next
    iCnt = WorksheetFunction.CountIf(Range("A1:A" & rCd.Row), rCd.Value)
    For w = Sheets(ws1).Index To Sheets(ws2).Index
        With Sheets(w)
            vStk = WorksheetFunction.Match(rCd.Value, .Range("A1:A9999"), False)
            If Err.Number = 0 Then
                For r = vStk To 1 Step -1
                    If .Range("A" & r).Text = "Style" Then Exit For
                Next r
                vSiz = WorksheetFunction.Index(.Range("A1:Z1").Offset(r - 1, 3), 1, iCnt)
                If Err.Number = 0 Then
                    fcnSizesByStyle = vSiz
                    Exit Function
                End If
            End If
        End With
        Err.Clear
    Next w
End Function

Tap Alt+Q when done to return to the worksheets. The second function will return the sizes of each inventory item. Usage:
=fcnSizesByStyle(<cell with item code>, <first sheet>, <last sheet>)
Typically something like,
=fcnSizesByStyle(A2,"Sheet2","Sheet3")
â€¦ which is filled down to match the item codes in column A. This will return the sizes in the format that they already exist. A 0 is returned if a size cannot be found. The first function uses the item code and the size to retrieve the inventory in stock. Usage:
=fcnStockBySize( <cell with item code>,  <cell with size>, <first sheet>, <last sheet>)
Typically something like,
=fcnStockBySize(A2,B2,"Sheet2","Sheet3")
â€¦ which is filled down to match the item codes and sizes from columns A & B.

sam mathai chacko say: OK, go to the VBA editor, and paste this in the ThisWorkbook class. I have assumed that the name of your data sheet is Sheet1. Rename as required. You will be prompted to select the required columns that you need to be displayed before printing.

Private Sub Workbook_BeforePrint(Cancel As Boolean)

    Dim rngColumnHeaders As Range
    If ActiveSheet.Name = "Sheet1" Then
        Set rngColumnHeaders = Application.InputBox("Select the column headers you want to display", "Print Page", , , , , , 8)
        ActiveSheet.Range("Print_Area").EntireColumn.Hidden = True
        For Each rngColumnHeaders In rngColumnHeaders
            rngColumnHeaders.EntireColumn.Hidden = False
        Next rngColumnHeaders
        Application.EnableEvents = False
        ActiveSheet.PrintOut
        Application.EnableEvents = True
    End If
   
End Sub
