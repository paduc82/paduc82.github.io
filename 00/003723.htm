This is fairly straightforward

Option Explicit

In an ordinary module, setup the calling macro that does all the work e.g. as follows. Change the Userform name  (here Userform1), the option button names, the template path and template filenames as appropriate. I have only included two case statements. You will need one for each button.

Option Explicit

Sub SelectADoc()
Dim ofrm As New UserForm1    'The name of the userform
Const sTemplatePath As String = "C:\Path\"    'The path where the templates are stored
    With ofrm
        .Show
        If .Tag = 0 Then GoTo lbl_Exit
        Select Case True
            Case .OptionProposal.Value
                Documents.Add sTemplatePath & "Proposal.dotm"
            Case .OptionProgress.Value
                Documents.Add sTemplatePath & "ProgressReport.dotm"
                'etc
        End Select
    End With
lbl_Exit:
    Unload ofrm
    Set ofrm = Nothing
    Exit Sub
End Sub
