Sub ClearSignInSheet()
' ClearSignInSheet Macro
    Sheets("Signin").Activate
    Application.EnableEvents = False
    Range("D2:E55").ClearContents
    Range("D2").Select
    Application.EnableEvents = True
End Sub

c.Value = Int(c.Value / 100) / 24 + (c.Value Mod 100) / 1440

Mod return the Modifier (i.e. remainder) of a division. Think of it as the opposite of Int.

3. Go to the folder we opened in the Step 1, find out the original workbook and open it.

(2) Specify a worksheet which you will place the deleted worksheet before;
(3) Check the Create a copy option and click the OK button.
