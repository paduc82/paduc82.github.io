Hi Nicole,

Thank you for posting your query on Microsoft Community.

You may create an email rule on your account. This will prevent junk emails from your email account. Please follow the steps below on how to create specific email rules and check if this helps:

Blocking emails using keywords that might appear in the sender's name:

  1. Open your inbox.

  2. At the upper right part of the page, click on the gear icon beside your Profile name then choose More mail settings.

  3. Under Customizing Outlook, click on Rules for sorting new messages.

  4. Click New.

  5. Under Step 1; on the first box, choose Sender's name; on the second box, choose contains; then on the third box, type in the keyword that you want to block.

  6. Under Step 2, choose the radio button Delete these messages—don't even put them in my Deleted folder.

  7. Click Save.

Blocking emails using keywords that might appear in the subject:

  1. Open your inbox.

  2. At the upper right part of the page, click on the gear icon beside your Profile name then choose More mail settings.

  3. Under Customizing Outlook, click on Rules for sorting new messages.

  4. Click New.

  5. Under Step 1; on the first box, choose Subject; on the second box, choose contains; then on the third box, type in the keyword that you want to block.

  6. Under Step 2, choose the radio button Delete these messages—don't even put them in my Deleted folder.

  7. Click Save.

99,44% of all PowerPoint presentations are "traditional", "linear". You click from one slide to the next.

But for a while now there has been a "new concept" called "relational" / "showcase" / "interactive" / "hyperlinked". The basic idea is that all slides are easily accessed with very few clicks. For example every page has a link back to a TOC that has links to every slide. The next level is a TOC page that has links to other files.

This way the presenter has a few options.
#1 Present the "deck" of slides in linear flow
#2 Due to time limitations, the presenter can easily skip slides
#3 During presentation, the presenter can easily skip directly to a specific slide to answer disruptive questions
#4 Design "custom" presentations using a large collection of existing slides.

steve rindsberg say:

This won't do ALL you want it to, but it'll at least get you partway there and probably save a good bit of time. This bit of VBA lets you specify the FULL path to the file you want to pull the custom show slides from, the name of the custom show, then inserts each of the slides in the named custom show into the current presentation.

How do I use VBA code in PowerPoint
This explains how you can use VBA code in PPT

Sub InsertCustomShow()
    Dim x As Long
    Dim sShowName As String
    Dim oSourcePres As Presentation
    Dim sSourceName As String
    Dim oTargetPres As Presentation
    Dim oTempSlide As Slide
        
    Set oTargetPres = ActivePresentation
    
    ' get the name of source presentation
    ' containing custom show to be inserted
    sSourceName = InputBox("Full name of source presentation:", "Source Presentation")
    'if blank, quit
    If Len(sSourceName) = 0 Then
        Exit Sub
    End If
    
    ' get the name of custom show
    sShowName = InputBox("Full name of custom show:", "Custom Show Name")
    'if blank, quit
    If Len(sShowName) = 0 Then
        Exit Sub
    End If
    
    Set oSourcePres = Presentations.Open(sSourceName)

    With oSourcePres.SlideShowSettings.NamedSlideShows(sShowName)
        For x = 1 To .Count
            Call oTargetPres.Slides.InsertFromFile(sSourceName, _
                oTargetPres.Slides.Count, _
                oSourcePres.Slides.FindBySlideID(.SlideIDs(x)).SlideIndex, _
                oSourcePres.Slides.FindBySlideID(.SlideIDs(x)).SlideIndex)
        Next
    End With
    
    oSourcePres.Close
    
End Sub

steve rindsberg say:

OK, another bit o' VBA that should help.

This will delete any existing show named MyNewShow (edit the code to change the name to whatever you like), create a new custom show by that name and add the currently selected slides to it.  A bit clunky but it should help.

If you want to use it often, I'd use InputBox (as in the earlier bit of code) to collect a name for the show instead of of having to edit the code each time.

There's probably some way to ADD the selected slides to the named custom show but I didn't have a chance to work that out.  Offer to hunt me down and force beer on me and we might change that. ;-)

Sub CreateNewCustomShow()
    Dim sShowName As String
    Dim aSlideArray() As Long
    Dim x As Long
    
    sShowName = "MyNewShow"
        
    On Error Resume Next
    ActivePresentation.SlideShowSettings.NamedSlideShows(sShowName).Delete

    ReDim aSlideArray(1 To ActiveWindow.Selection.SlideRange.Count)
    
    For x = 1 To ActiveWindow.Selection.SlideRange.Count
        aSlideArray(x) = ActiveWindow.Selection.SlideRange(x).SlideID
    Next
    
    ActivePresentation.SlideShowSettings.NamedSlideShows.Add sShowName, aSlideArray
    
End Sub
