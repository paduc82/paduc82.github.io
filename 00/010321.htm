To download and install the Microsoft Speech Platform, follow these steps:

Download additional TTS language engines

As stated - if your account was configured as an Exchange ActiveSync account - you would not have been able to create a contact group inside the contact folder for that account - no exceptions.

If you're account was configured as a POP account - CG's would most certainly have been possible

Similarly if the account was configured as an IMAP Account - the contacts (and groups) would not have sync'd with the server - and once the IMAP account was removed from the profile so too would have anything inside the contact/calendar folders that had the words "This Computer Only" in the description.

To summarize - if you currently have your Outlook.com account configured as EAS - you can't create CG groups in OL'2013 or OL'2016 and if any groups created in your Outlook.com account via the web interface are disappearing - then it's an issue that needs to be dealt with on the Outlook.com server side - has nothing to do with Outlook (desktop client).
