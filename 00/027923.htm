Rather than initiate text messages by opening up the text messaging application and then selecting Menu > New > and letting the text messaging application select a mobile number for you, try initiating new text messages from the Contacts application:

  • Open up the contact you want to send the text message to, move down to the phone number that you want to send the text message to and select Menu > Send Text Message. 
  • Then the text messaging application will open up, create a new text message and insert the number you selected in the TO field.

Creating text messages in this fashion allows you to send a text message to any phone number you've got for a contact.  You won't be able to change the label on the phone number fields, but you won't be limited to only sending text messages to one phone number. 

Of course, you might accidentally send a text message to a landline, in which case, you'd probably get some kind of "failure to send" message.

Accelerators only show when the context from which they are called matches their action....


for example....


If you highlight some text on a page then only Accelerators that accept a text selection will appear in the list of Accelerators on the context menu or the Accelerator icon.


If you just click on a blank area of a web page, then those accelerators which accept a url for an action will appear....


some parts of webpages may not actually be empty... they may contain parts of other websites embedded in the webpage of a different site. Your IE security zone settings may be preventing Accelerators to Restricted websites from showing.


When asking internet related questions it is helpful if you can include the address of any websites you are having issues with.
