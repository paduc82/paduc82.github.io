Hi Jac,

Try this...

  1. Close all open Word documents and then quit Word and Outlook completely.
  2. Open the Activity Monitor.app program, which is located in Applications/Utilities
  3. Type "Microsoft" into the search box and verify that there aren't any running copies of Microsoft Word or Microsoft Outlook open. If there are, select the application so that it highlights and then use the X button to Force it to Quit.
  4. Quit the Activity Monitor application and open Microsoft Word. A new blank document should be displayed.
  5. Adjust the window of the new blank document to the size you want.
  6. Save this blank document. It will probably have the name of Document1 and that is fine, you can delete it later.
  7. Quit Word completely and reopen

    Assuming that you do mean that you want to use Full Screen View, the fastest & easiest solution is to use the built-in keyboard shortcut Control+Command+F when you want to send a document window to or restore from Full Screen. As many times as this has been covered over the years I would think most of the regulars here would know that a template solution won't work [e.g., How do I get Word 2011 to remember its window assignment? ]. Regardless of experience, though, testing the proposed 'solution' also would have revealed that to be the case.     A number of macros have been published relevant to specific releases of the program. Unfortunately, VBA support has changed a bit with each release, so what works for one version won't work for another.     change order of label in mailmerge tool in Microsoft office word 2007 - Microsoft Community

    trannhanquyen say:

    change order of label in mailmerge tool in Microsoft office word 2007

    Hi there,

    I'm Quyen. I have a question about a tool in Microsoft office word 2007

    I made labels of mail merge

    • By defaut labels are ordered: from left to right, from top to bottom (as image below)
    • But i need them are arranged in a different way: from right to left, from top to bottom (as image below)

    So, can i do that? if can how can i do it?

    I look forward for your help
    Thanks very much

    a. user say:

    There are two ways you could do this.

    1. Resequence your data source so that record 1 and 2 are swapped, 3 and 4 are swapped, and so on How you could do that depends on the data source (is it Excel? A Word document? Or what?)

    2. Change the table direction. This may be easy, if you already use a language with a right-to-left script, but if not, it is more difficult. See below...

    For a label merge, Word creates a table that typically has one cell for each label.

    Select the table.

    Right-click, and click Table Properties...

    Near the bottom of the Table tab in the dialog, if there is an option call "Table direction", select Right-to-left. If there is not, please see below.

    After that, lay out your fields etc. in the top right-hand cell, then use Update Labels to copy your layout to the other cells. There should be a { NEXT } (<<Next record>>) field at the top of each label cell except the cell at the top right.

    I cannot check now, but that right-to-left option may not appear unless you have enabled an Editing Language in Word that uses a right-to-left script (Arabic/Hebrew/Persian/Urdu etc.) . Be aware that adding such a language adds quite a few options to your menu/ribbon, and may have other effects on your documents.

    To add an Editing Language in Word 2007 I would

     a. close Word

     b. try to save a backup copy of my normal.dotm template, because Office may replace your existing one. Its location may vary from system to system, but mine is in

    C:\Users\<username>\AppData\Roaming\Microsoft\Templates

    where <username> is my Windows user name. So if it is pjj, the file would be in

    C:\Users\pjj\AppData\Roaming\Microsoft\Templates

     c. in Windows, find the Microsoft Office 2007 Language Settings program (the option is not built into Word in the 2007 version) and start it. How to do that will also vary depending on your version of Windows, but in my case in the WIndows Start menu I found it in All Programs->Microsoft Office->Microsoft Office Tools.

     d. Add a suitable RTL Editing Language.

    If you do not see any suitable languages, you may have to enable such a language in Windows Control Panel or Setting (again, depending on your version of Windows). But that is another story.
