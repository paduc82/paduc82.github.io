suzanne s. barnhill say:

Entries in the table of contents (TOC field) are by default hyperlinked to the corresponding headings. To create hyperlinks elsewhere, you will need to manually bookmark the headings and then insert hyperlinks to those bookmarked locations.

To follow a hyperlink (in the TOC or elsewhere) you must, by default, press Ctrl while clicking. When you mouse over a TOC entry, do you see "CTRL+click to follow link"? If not, press Alt+F9 to reveal the TOC field code and add the \h switch to the end of the field code, then Alt+F9 to toggle back and F9 to update the TOC. You should now see hyperlinks.

If you have created your TOC manually instead of inserting a field, see http://www.shaunakelly.com/word/numbering/tableofcontents.html.

How to set up different margins on letterhead second page - Microsoft Community

a. user say:

How to set up different margins on letterhead second page

I'm building a corporate letterhead template that has a 2" top and left margins and 1" bottom and right margins. I want to provide a template that automatically creates a second page with 1 in margins all the way around when the user fills the first page. I've conquered the issue of a different first page and keeping the page one header and footer on page one only.

I've spent the entire day reading tutorials and forums, but can't find a solution that works in my case. I know this can be done, but I'm darned if I can find the answer.

Using Word 2013.

jay freedman say:

Text in Word cannot flow across a section break. What you need is a way to change the apparent margin without changing the actual margin, and there is a workaround for that.

As explained in detail at http://wordfaqs.mvps.org/LetterheadRibbon.htm#Complex, you must set the actual margins to 1" all around, as you want on the second and later pages. Then turn on the Different First Page header option, insert the corporate information and logo in the first page header, and insert a text box (with no fill and no borders) that pushes the first-page text an additional 1" to the right.

When the text flows naturally to the second page, there won't be a text box there (it's only anchored to the first page header), so it will obey the actual 1" margin.
