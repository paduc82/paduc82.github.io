larkhalpern say:

Many thanks to David and Anthony here on the forum as well as to Jessica and Manuel who responded to my query through Microsoft Help.  My issue has been resolved, but why my new PC's system repeatedly rejected the correct password is still a mystery and the steps outlined below do not address the actual problem.  

Here are the steps I took to resolve my problem:

  • Locked out of my desktop PC, but knowing my password was correct, I just kept entering it repeatedly until it was finally accepted.
  • Once logged in, I set up a local account with administrator access so if I got locked out again, I could log in through the local account.  Here are the steps to create a local account:
    1. Press the Windows key and R to bring up the Run box.
    2. Type in netplwiz and click OK.
    3. Put a check mark on "User must enter a user name and password to use this computer."
    4. Click Add.
    5. Click on "Sign in without a Microsoft account (not recommended)."
    6. Click on Local account.
    7. Type in a new user name in the User Name field, but do not enter a password.
    8. Click on Next, then Finish.
    9. Double click on the newly created user name.
    10. Select the Group Membership tab.
    11. Select Administrator, click apply, then click OK.
  • Then I changed the password on the user name set up with my Microsoft account.  

    Here are the steps to change the password of the account you're logged in with: 
    1. Click Ctrl+Alt+Del and click on Change Password.
    2. Fill in the Old password field with the old password, the new password in the New password field and retype the new password in the Confirm password field.
    3. Click on the arrow.

    Here are the steps to change the password for a user name you're not logged in with:
    1. Press the Windows key and R to bring up the Run box.
    2. Type in netplwiz and click OK.
    3. Click on the applicable user name to highlight it.
    4. Click on Reset Password...
    5. A dialog box will open.  Type the new password in the New password field and retype it in the Confirm new password field.
    6. Click OK.
    7. Click Apply and OK.

I'm now able to shut down and restart and log back in with my Microsoft account without any problems.  Why the original password was only accepted 1-2 times out of 100 remains unexplained.
