hansv mvp say:

As far as I can tell, the macro could be shortened to

 

Sub OptionButton1_Click()
    Worksheets("SLA").Range("SLAs").FormulaR1C1 = _
        "=OFFSET(Parameters!R[-8]C[2],Parameters!R2C8*6,1)"
End Sub

 

You can declare a variable at the top of a module (before all Subs and Functions), but you cannot assign a value to a variable outside a Sub or Function.

 

So this is valid:

 

' -----
Option Explicit
Dim SLA As Worksheet
 
Sub OptionButton1_Click()
    Set SLA = Worksheets("SLA")
    ...
End Sub
' -----

 

but the following is not valid:

 

' -----
Option Explicit
Dim SLA As Worksheet
Set SLA = Worksheets("SLA")
 
Sub OptionButton1_Click()
    ...
End Sub
' -----

jan kuera say:

Hello Sandra,

the numbers are actually pretty easy, you can use page number for them.

The small calendar could be added with the Insert | Calendars button.

The weekday is the most tricky part. Maybe you could use separate masterpage for each weekday. I would not be afraid to use VBA to fill what you need in.

 

However, this is how can you do it with other programs:

1. Start Excel. In the B1 cell, type 1/1/2012 (so you have first row blank).
2. Drag it at the right bottom corner to fill series with dates until the day you need, feel free to fill the whole year which will be at row 367 for 2012.
3. Copy the A column to columns B and C so you have three columns of dates.
4. Select the column A, click on it with right button and select Format Cells... In the list of types select Custom, and type mmmm into the Type box.
5. The same with column B, just use dddd format.
6. The same with column C, just use d format.
7. In the first row, type Month, Weekday and letter D.
8. Save it on a disk, make sure you select Unicode Text File format (*.txt) and accept all warnings about loosing format.
9. Close Excel.
10. Start Publisher with a blank publication.
11. In the Mailings tab, click on the Mail Merge dropdown button and select Step By Step Mail Merge Wizard.
10. Step 1: Use an existing list, click next ad select the text file you created in the Excel. You can accept the defaults - Tab as separators and that first row contains headings as long as you did it in step 7.
11. You will be presented a Mail Merge Recipients. You can just click OK here.
12. Step 2: Design your single page daily planner. Do not use master pages. You can drag Month, Weekday and D from the Mail Merge taskpane to the publication (or double click it). This will create placeholders for the data. You can freely format it as you wish.
12. Step 3: Click next (Create merged publications). Click merge to a new publication.

That's It! If you need to fix something just close the new publication, change the original one and Click to merge again.

Tip: To make different even and odd pages, I would create two publications with all days, one formatted as even pages, on as odd pages. Then I would print even pages from one publication only, odd from the other. Since Publisher cannot do it and if your printer driver cannot do it either, you can either use VBA to print it (http://support.microsoft.com/kb/294748) or export it to PDF and do the print from Adobe Reader for example.

Hope this helps,
