Hey Tiggerday2000

I have been having the same problems. Luckily, I was able to fix it, and I did so by combining multiple methods that I found on various trustworthy tech websites. This is what I did, but depending on your version of Windows 10, well...I don't know as to whether it will work, but please do not hold be responsible to anything that goes wrong.

First step: (VERY IMPORTANT)

Either in safe mode or the standard windows desktop, press Win+X on the keyboard. Select "Command Prompt (Admin) or "Windows Powershell (Admin)" 

In the desired command console, type: net user administrator /active:no

It should say that it completed successfully. To check, type: net user administrator. See if it says "No" under "Account Active"

Next step: Make sure you are either signed in to your Microsoft account or a local account with a password. For a local account, go to Settings>Accounts>Sign-In Options>Password

Restart your PC

Third Step: if the message keeps coming up; don't worry. This means that it is still going wrong, meaning you can still fix it. Select your account from the bottom-left corner and sign in. The, on the desktop, press Win+R and type netplwiz and press OK or hit enter.

In the pop-up window, select "Users Must enter a username and password to use this computer. 

Then, down under the Box labeled "Users for this computer" select your account and then press the properties button underneath.

Another pop-up should open. Go to the "Group Membership" tab, and select your account as "Other." In the dropdown menu, select t group "Administrators." Hit apply and then apply again. 

Restart your PC.

This time, you should go to the lock screen (which displays the time, internet status, etc..) Press the spacebar or click anywhere. Your account should be there and no others. You can only now sign in to yours, and it does not keep saying that the admin account is disable Hooray!!

Bonus step:

If you ever do need to use the built-in admin account, you can either undo everything you just did, or boot into safe mode. (Press the recovery options F key on boot or hold Shift while pressing Restart)

Go to Troubleshoot>Advanced Options> Start Up- Behavior. Then, hit restart in the options and when the menu comes up, press F4. This boots in safe mode and is using the Built-in Admin account.

Alternatively, you can just go into Admin Command prompt and just type net user administrator /active:yes.

Hope this helps, as it did for me.

MAC Filtering

Login Information

  • Gateway - 192.168.1.1 
  • Username - admin
  • Password - password/password1/admin

MAC Filtering

 

  1. Open a web browser.
  2. Log into the gateway by entering “192.168.1.1” into the address bar of the internet browser.
  3. Once logged in, click on “Wireless Setup” on the top of the page.
  4. When  prompted to enter the username and password. Type in the current username and password (see login information).
  5. Click on “Wireless advance settings” on the left hand side.
  6. On the main page, check the “Enable Access List”
  7. Select “Accept all devices listed below”.
  8. Enter in a MAC address from the device you would like to add.
  9. After adding the devices, click on Apply.
  10. You can also remove the MAC addresses by selecting them in the “List” box and selecting the “Remove” button.
