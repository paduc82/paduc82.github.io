In Powerpoint 2016, how do you change the paragraph spacing (after) of a default text box to 6pt?

In powerpoint, when I paste text using the Paste Special -> Paste as text only command, the resulting text box has all the formatting I want except that the paragraph/line "Spacing After" always defaults to 0pt and I must click the arrow in the paragraph section of the home tab and set it to 6pt manually (for each text box I paste/draw).

In an older question I posted here, I was able to change the default text size to 12pt by editing the presentation.xml file and changing every sz=1800 entry to sz=1200. (The old question is here: https://answers.microsoft.com/en-us/msoffice/forum/msoffice_powerpoint-mso_win10/how-can-i-change-the-default-text-size-obtained/d9eca664-906a-4292-bb15-3eb6ace35dad )

Is there a way to set the default "Spacing After" value of text boxes to 6pt? Either through powerpoint's UI or perhaps by adding or modifying an entry in the xml code of my ppt file like I did to fix the text size default?

john korchok say:

I'm in the middle of writing a multi-part article about this very subject. Here's part 1: XML Hacking: Text Box Styles Your question will be covered in part 2, but I won't keep you in suspense. The text box styles are stored in ppt/presentation.xml in the <p:defaultTextStyle> section. You can create up to 9 levels of different styles that you access the same as in regular text placeholders, by click on Increase List Level or Decrease List Level.

Inside each definition of a text level, space before and after look like this:

  <a:lvl2pPr marL="457200" indent="0">
      <a:spcBef>
          <a:spcPts val="0"/>
      </a:spcBef>
      <a:spcAft>
          <a:spcPts val="1200"/>   <Comment - 12 points after
      </a:spcAft>
      <a:buNone/>
      <a:defRPr sz="1200"/>
  </a:lvl2pPr>

The syntax is identical to <a:lstStyle> section of a slide layout. In fact a quick and dirty way to set up text formatting is to follow these steps:

  1. Create a dummy slide layout. I usually start with a duplicate of the Content with Caption layout, because it already has a text box with small type on the left. Then I delete the title and content placeholder.
  2. Create up to 9 levels in the text placeholder and format each level as you would like your text boxes to look.
  3. Save the presentation, then close it and expand the XML.
  4. Open the slide layout your modified. If you use the Content with Caption layout, look for ppt/slideLayouts/slideLayout8.xml and open it in a text editor
  5. Just below <a:lstStyle> are 9 level definitions with all the formatting you created in the program interface. Each level starts with <a:lvlXpPr where X is the level number. Copy all 9 level definitions, but not the starting <a:lstStyle> or closing </a:lstStyle> tags.
  6. Open ppt/presentation.xml.
  7. Select all 9 level definitions (starting just below </a:defPPr>) and paste in the 9 levels from slideLayout8.xml.
  8. Save the file, zip everything back into a presentation. Each inserted text box should have up 9 levels of styles.

Edit: The second part of the article is online now: XML Hacking: Styled Text Boxes Complete
