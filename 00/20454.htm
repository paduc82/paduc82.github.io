Hi TrentQueen,

I see that you have provided the mail flow process of a workable Exchange Online mailbox, and it shows that the mail is delivered to Exchange Online and then Exchange on-premises. Therefore, if the mail flow goes the same for the affected shared mailbox, the root cause of the issue should be the fact that there is no corresponding mail-enabled object for this shared mailbox in Exchange on-premises, thus a valid recipient cannot be found.

Regarding this, my suggestion is that you create an object (a user account) in the on-premises AD, match it with the cloud shared mailbox (which means they must have the same SMTP address), and then enable a remote mailbox for it by running Enable-RemoteMailbox.

I want to use the following code in a working word mail merge doc. It fails at

  If Trim(.DataFields("Last_Name")) = "" Then Exit For  (The requested member of the collection does not exist)

LAST NAME FIRST NAME ADDRESS CITY ST ZIP  CONTRIBUTIONS

Sub Merge_To_Individual_Files()

Application.ScreenUpdating = False
Dim StrFolder As String, StrName As String, MainDoc As Document, i As Long, j As Long
Set MainDoc = ActiveDocument
With MainDoc
  StrFolder = .Path & Application.PathSeparator
  For i = 1 To .MailMerge.DataSource.RecordCount
    With .MailMerge
      .Destination = wdSendToNewDocument
      .SuppressBlankLines = True
      With .DataSource
        .FirstRecord = i
        .LastRecord = i
        .ActiveRecord = i
        If Trim(.DataFields("Last_Name")) = "" Then Exit For

        StrName = .DataFields("Last_Name") & "_" & .DataFields("First_Name")
      End With
      .Execute Pause:=False

    For j = 1 To 255
      Select Case j
        Case 1 To 31, 33, 34, 37, 42, 44, 46, 47, 58 - 63, 91 - 93, 96, 124, 147, 148
        StrName = Replace(StrName, Chr(j), "")
      End Select
    Next
    StrName = Trim(StrName)
    With ActiveDocument

      .SaveAs FileName:=StrPath & StrName & ".pdf", FileFormat:=wdFormatPDF, AddToRecentFiles:=False
      .Close SaveChanges:=False
    End With
  Next i
End With
Application.ScreenUpdating = True
End Sub
