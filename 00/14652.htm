rob koch say:

What you are requesting already exists and really always has, since Windows has had the administrator and standard user account concept since early versions of the operating system.  You're simply confusing the idea that a separate password is required, since this ability has existed since the changes to ACLs included in Windows Vista.

Security: New ACLs Improve Security in Windows Vista

In truth, you should be creating a separate standard user account for day-to-day use by less technically capable users and only give an administrator account to those users (e.g. yourself) who might recognize the risks involved when something requests this higher level of access.  This is the security model which Windows has always contained for this purpose and various features such as User Access Control (UAC) have made this even more effective in terms of security and ease of use.

How User Account Control works (Windows 10) - Microsoft Docs

Even if you should need administrator access for some reason, this will typically generate a UAC prompt to login as the administrator, which creates exactly the scenario you were wishing for, an additional password being required in order to perform the higher security function that's been requested.

In truth, even the administrator is often limited within Windows 10 as to what operating system items they can directly affect, since many of the critical functions of the filing and operating system are off limits by default, though the administrator user has the ability to change this security on a folder-by-folder basis when they attempt to browse into a protected location.  If the administrator user is stupid enough to accept the prompt to change this folder security, the operating system will let them, but that's why it's always best to create standard user accounts for children or other less knowledgeable users who might answer these prompts incorrectly.

Another example of this multi-layered approach to security is the Trusted Installer service, which is also discussed in the New ACLs article I referenced above,  This Trusted Installer service is what actually performs the installation of both Windows Updates including those for Defender, as well as any applications that are installed, so standard users have only Read & execute and List folder contents access to these folders by default.

If you wish to protect from the risk that even you might make an error as administrator, then it's always been suggested that using a standard account for day-to-day tasks is safest, while the ability to respond to a UAC prompt with the credentials for your separate administrator account username and password means that you don't need to logout and back in to perform most higher security functions when the need arises.

In truth, you should be creating a separate standard user account for day-to-day use by less technically capable users and only give an administrator account to those users (e.g. yourself) who might recognize the risks involved when something requests this higher level of access.  This is the security model which Windows has always contained for this purpose and various features such as User Access Control (UAC) have made this even more effective in terms of security and ease of use.

How User Account Control works (Windows 10) - Microsoft Docs

Even if you should need administrator access for some reason, this will typically generate a UAC prompt to login as the administrator, which creates exactly the scenario you were wishing for, an additional password being required in order to perform the higher security function that's been requested.

In truth, even the administrator is often limited within Windows 10 as to what operating system items they can directly affect, since many of the critical functions of the filing and operating system are off limits by default, though the administrator user has the ability to change this security on a folder-by-folder basis when they attempt to browse into a protected location.  If the administrator user is stupid enough to accept the prompt to change this folder security, the operating system will let them, but that's why it's always best to create standard user accounts for children or other less knowledgeable users who might answer these prompts incorrectly.

Another example of this multi-layered approach to security is the Trusted Installer service, which is also discussed in the New ACLs article I referenced above,  This Trusted Installer service is what actually performs the installation of both Windows Updates including those for Defender, as well as any applications that are installed, so standard users have only Read & execute and List folder contents access to these folders by default.

If you wish to protect from the risk that even you might make an error as administrator, then it's always been suggested that using a standard account for day-to-day tasks is safest, while the ability to respond to a UAC prompt with the credentials for your separate administrator account username and password means that you don't need to logout and back in to perform most higher security functions when the need arises.

OneNote 2016 OneNote for Windows 10

OneNote for Windows 10 (simply labeled “OneNote”) is the best and most up-to-date version of OneNote on Windows. It was designed to feel natural with any input method — from mouse and keyboard to pen and touch — and it contains numerous improvements for better performance, battery life, and reliability.

OneNote for Windows 10 also has a number of features that are not available in OneNote 2016 — including ink effects with a dramatically improved ink-to-text option, Researcher, a notification center, deep integration with Windows 10, and much more.

OneNote for Windows 10 comes pre-installed on every version of Windows 10, and it’s included in Office 365 and Office 2019.

By comparison, OneNote 2016 is the original Windows version of OneNote, which came with Office 2016 and earlier. It has some legacy functionality that might be important if you use this version at work or school, but OneNote 2016 is no longer being updated with new features.

Microsoft recommends using OneNote for Windows 10, but OneNote 2016 is still optionally available for anyone who needs it — especially if you have locally stored notebooks or are running an old version of Windows.
