The Endnote Text Style is for the Endnote Text, as its name implies. If you were getting superscripted endnotes, that suggests you were formatting the text with the Endnote Reference Style.

formula to copy cells from one sheet to another - Microsoft Community

danielwood1985 say:

formula to copy cells from one sheet to another

Hi, is there a formula to look in one sheet, down column D, and if the criteria is met, copy the row into the second sheet where the formula will be running?
rich007a say:

OK, so you need to return multiple items from a lookup, you can't use VLOOKUP for that.  Take a look at this: http://www.ashishmathur.com/lookup-where-search-string-appears-multiple-times/

 

But if you just want to return the result of the sum of all the charges next to Yellow, use the SUMIF function:

 

=SUMIF($A$2:$A$500,"Yellow",$B$2:$B$500)                    if on the same sheet

or

=SUMIF(Sheet2!$A$2:$A$500,"Yellow",Sheet2!$B$2:$B$500)      if on a different sheet

 

Note that from Office 2007, worksheets contain over a million rows, so using A:B references to reference an entire column can be quite a resource drain.  It is good practice to avoid this if possible.  Just use a sensbily large number to cover your data (I used 500 in the above example).

 

Cheers
Rich
