If using Word 2007, go to the very top next to the Word Orb and the furtherst button to the right on the left side of the screen is Customize Quick Access Toolbar.  Click on that.  On the left side find the entry Fonts... and click on it and then click on Add so it now also shows on the right side and then click OK and save this.  Now go to the new button you clreated and click on it.  Set the defaults you want (fonts or anything else listed there) and when done click on the Default button at the bottom.  It will ask if you want to change the normal template to these new changes and answer yes.  Then close out of Word and re-open it.  The new defaults should now be in place.  That's it.  You're done.  Anytime you want to change it again, all you need to do is click the fonts button you added and follow the same procedure.

Since new douments are based on normal.dot, this font you used when you saved it should be the default font.  If you can find normal.dot, then I believe this will work on whatever version of Word you are using.

I hope this helps.

greginmich say:

Edge’s loop protection isn’t a panacea. It’s only effective against malicious webpages that use alert loops to lock up the Edge browser. Since the dialog loop is created with a script that runs on the webpage itself, the “malware” will be gone as soon as you succeed in closing the malicious page.

Many popups don’t fit this scenario at all, and are actually the handiwork of binary malware that has been installed on the system and thus needs to be removed with specialized malware-removal tools:

https://malwaretips.com/blogs/remove-reimage-repair/

In order to distinguish between browser-locking webpages and conventional malware, use the old method for removing a Tech Support Scam page from Edge:

1. First, right-click on the taskbar and select Task Manager

2. Then select the browser app and click on the End task button.

3. Now, restart Edge indirectly by entering a search topic (like “tesla”) for a Cortana web search.

4. Then right-click on the search-result tab and choose “Close other tabs”.

If Edge will then close and reopen without bringing up the browser-locking scam page, then the issue was probably just with the webpage itself.

GreginMich

marshall barton say:

You need a field to provide a unique sort before you can calculate the -1, -2, -3, part of the invoice number.

Assuming you have such a field (let's call it SortField) the query would look like:

INSERT INTO [existing table]
SELECT Format(Date(), "yyyymmdd") & "-"
      & (SELECT Count(*) FROM [one table] As X
          WHERE X.SortField <= [one table].SortField
              AND X.[creat Invoice] AND X.[type] = "lender"
         ) As InvoiceNum, *
FROM [one table]
WHERE [creat Invoice] AND [type] = "lender"

Think you misunderstood.  The ClearConditionalFormatingBeforePaste macro should be used INSTEAD of paste to stop the c.f.s building up.  You must copy something from a (clean*) FROM sheet first, then go to the RECEIVING sheet/file, and the click where you would if you were going to paste, but run the macro instead.

stefan blom say:

You have multiple sections because a section break has been inserted somewhere in your document. Word will insert section breaks automatically if you change section properties (such as Page Setup) for a selection.

Once you have more than one section, header (or footer) content can be linked or unlinked from the corresponding header (footer) in the preceding section.

A section can have up to three different headers (and three different footers); this is the case if you have enabled both "Different first page" and "Different odd and even" in the Page Setup dialog box (or the similar options found on the Header & Footer Tools Design tab on the ribbon). Each type of header (or footer) must be separately unlinked from its "neighbor" in the preceding section.

For more, see http://addbalance.com/usersguide/sections2007.htm.
