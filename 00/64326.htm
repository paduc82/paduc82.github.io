Apps greyed out and underlined in Windows 10 start menu. - Microsoft Community

igofficenesa say:

Apps greyed out and underlined in Windows 10 start menu.

There was another post like this, but it's closed and I couldn't reply to it.

New Windows 10 install, and some of the start menu entries have underlines, and those apps are greyed out in the tile section.

I read a method to reset the windows store, and it had quite a few steps, including a reboot.

All I had to do was one step.

Open Windows Powershell (Admin) and enter:   wsreset.exe    to reset the store. That's it.

System will churn a bit and eventually it will open the Windows Store.

Close the store and click on one of the greyed out icons. For me this opened the update section of the store, and all the greyed out apps started updating.

When done, close the powershell and check the start menu. Everything should be working normally.

If Windows 10 can grey out or underline apps that are not working properly, why can't it reset itself? Obviously it can detect that the apps are not working. Or maybe it got stuck trying to update or something. In any case, it seems Windows could inform the user and perform the reset itself.

celexa say:

This was the only way I found to fix this:

 

Be very careful editing windows registry

 

For Windows Vista/7

 

Click Start and type regedit in the Search Programs and Files field. Then right-click on regedit in the programs list and select Run as Administrator.

 

Within the Registry Editor, navigate to: HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\WindowsUpdate\AU.

 

On the right side of the window, locate AUOptions and NoAutoUpdate. Delete both of these registry keys.

 

Reboot

 

You should now be able to edit Windows Update
