r.taylor say:

I had exactly the same problem on Win10 after a couple of crashes and an attempt to update. It refused to install or run, always giving error 0x80070003 in Store update. 

I was able to finally clear the problem.  What I did was this:

1.) First, open an elevated command prompt window, and run the command to clear Store cache:

    wsreset

2.) Now start PowerShell console and do the steps you mentioned earlier:

   a.)  Get the full package name:

         get-appxpackage *castle*

    b.) Then force remove the package using the fullname you find from step a:

         remove-appxpackage Microsoft.AgeCastles_1.14.36.0_x86__8wekyb3d8bbwe

   *This time the remove worked without the error (only if Store cache was cleared first).

3.) Close Store app (from Task Manager) and restart it again.  Do update or install.

In my case I still had a failed update attempt listed in the Downloads and Updates section (I forgot to clear it before doing the remove-appxpackage), so I just did a retry, and it ended up downloading the entire game over (not just an update) and it worked fine.  I used similar steps for a different game I had problems with and it worked there as well, but that one I had to search for game again and do Install.  So either retry update or install depending on what Store thinks your state is. 

I hope that helps anyone else seeing this same (or similar) problem.
