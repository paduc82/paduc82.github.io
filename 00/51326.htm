    hairyfool say:

    Use the Windows System Image tools to create a backup of the entire drive to an external disk and make the recovery drive.

    Disconnect the old disk and connect the new one, boot the recovery drive and do a restore pointing it to the external disk. If you have USB 3 ports you may have to connect it to a USB 2 port in order to see the system image.
