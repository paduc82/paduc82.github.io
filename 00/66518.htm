Split View - the hamburger menu

So, if you are looking for a function, this is the first place you should look.

Split View is a function commonly associated with mobile apps that have made the jump to the desktop

In addition to replacing the older standard Calculator Windows App, the new Calculator is optimized for touch just as well as it works with keyboard and mouse. Not all Universal apps feature the Split View menu.
