Apparently, the behaviour of the "/golive" command has changed now in the new Skype 8.X version.

 

In the classic Skype (the 7.X version), the "/golive" chat command activates a group call with the camera disabled by default. In the new Microsoft Skype for Desktop (the 8.X version), this chat command activates the group call, but with the camera enabled.
