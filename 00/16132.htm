andreas killer say:

A Shape has a GroupItems property

https://docs.microsoft.com/en-us/office/vba/api/excel.shape.groupitems

If you have grouped shapes you can access all single shapes like this

Sub Test()
  Dim MainSh As Shape, Sh As Shape
  Dim Item
 
  Set MainSh = ActiveSheet.Shapes(1)
  For Each Item In MainSh.GroupItems
    Set Sh = Item
    Debug.Print Sh.Name
  Next
End Sub
