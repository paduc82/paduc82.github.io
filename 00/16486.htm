The code below is very basic. Because you said you are going to auto insert the initials when the name is selected, the code runs when a change occurs against the EPA Representative name.

To install the code.

  • Right click the Worksheet tab name and select View code. (The VBA editor will open)
  • Copy the code below and paste into the editor.
  • See the comments that appear in green in the editor and then edit the code for the date/time format and if you don't want the message that the date/time previously entered then delete the code between the asterisk lines. Private Sub Worksheet_Change(ByVal Target As Range)
    Dim msgResponse As Variant
    'Using Select case with multiple match values
    'is similar to having multiple If statements
    Select Case Target.Address
        Case "$C$76", "$C$83", "$C$90"
            '*********************************************************
            If Target.Offset(1, 0).Value <> "" Then
                Beep    'Optional line. Delete if beep not required
                msgResponse = MsgBox("Date/Time previously entered. " _
                                & "Do you want to update?", vbYesNo)
                If msgResponse = vbNo Then Exit Sub
            End If
            '*********************************************************

            Target.Offset(1, 0).NumberFormat = "dd mmm yyyy h:mm:ss AM/PM"
           
            Target.Offset(1, 0) = Now()
    End Select
End Sub

andreas killer say:
That is possible, but it is security critical, because virus would use the same way to get your email addresses.

A better way is to resolve the recipients with a macro in outlook if you create the email.

Paste the macro below into a regular module in Outlook.
Create a new email and paste the ID's into the To field as usual.
Press TAB (to leave the To field)
Press ALT-F8 and execute the macro.

Andreas.


Sub ResolveRecipients()
  Dim mItem As MailItem
  Dim r As Recipient
  'Get the current email
  Set mItem = ActiveInspector.CurrentItem
  'Try to resolve all
  mItem.Recipients.ResolveAll
  'Delete unresolved recipients
  For Each r In mItem.Recipients
    If Not r.Resolved Then r.Delete
  Next
End Sub

There is, but it requires some complex VBA: it's more trouble than it is worth.

Far better to use a Master Document to print your document.  That will automatically paginate for you.

A couple of pointers:

1)  The master document should be used ONLY for printing.  It's not a good idea to attempt to edit the text of a master document: unless you are very careful, you are likely to get corruptions.

2)  All chapters should have been created from the same template.

3)  The style definitions in each chapter must be the same.

4)  Do not place any text at all in your master document file.  You should be able to delete the master document without losing any text (because if it corrupts, you will have to).  The master document is simply a list of linked chapters.

When you open the Master, you can Update Fields and the table of contents, figure numbering, pagination etc will all update correctly.
