rickcp say:
what is a rootkit? 
The term rootkit is used to describe the mechanisms and techniques whereby malware, including viruses, spyware, and trojans, attempt to hide their presence from spyware blockers, antivirus software, and system management utilities.

Because of its ability to hide and continuously compromise a system from within, it's considered one of the most dangerous or insidious weapons and tricks in an attackers arsenal. Malware uses rootkits, or rootkit functionality, in order to 'hide its presence on an affected computer and thus impede its removal'. Once compromised by a rootkit, any information returned by an affected system can no longer be trusted and must be regarded as suspect, which is exactly how they hide themselves and their components from you - by modifying requests for information that might give them away.

*Source: You may read more about it in the 'MMPC Threat Report: Rootkits - June 2012' (PDF file download link provided in this blog post: know your enemy - protect yourself, Oct. 19, 2012) which outlines how an attacker uses rootkits and how these rootkits function on affected computers.-

[Le Boule said] Without knowing the name of the malware we're limited as to how much advice we could give you.
Some of the most prevalent malware families today consistently use rootkit functionality. There are several rootkit classifications depending on whether the malware survives reboot and whether it executes in user mode or kernel mode. Moreover, they are difficult to detect and remove and can remain hidden for extended periods, possibly years, if gone unnoticed.

In addition, some recent rootkits sometimes referred to as MBR rootkits or "bootkits", can modify the Master Boot Record (MBR) to gain control of the system and start the process of loading the rootkit at the earliest possible point in the boot sequence.
One notable example of this type of rootkit is the "DOS/Alureon.A", which is MS detection for a variant of the "Win32/Alureon" malware family that infects the MBR.
Please also note that alert notifications or detections of this malware from installed antivirus or security software may be the only symptoms.

Furthermore, if a system has been determined to be compromised by this type of malware, then an additional tool might be warranted to allow you to boot to a known good or trusted environment so that appropriate remediation measures can be taken. In this case, the Windows Defender Offline (WDO) may be useful. You may also find this WDO User Guide (from sevenforums.com) to be helpful.

As is the case of Alureon, there may be some instances where this type of 'bootkits' may modify certain driver files so that they become corrupted and unusable. In order to restore functionality to the computer, the corrupted file must be restored from backup. Users must boot into a recovery environment and manually replace the file with a clean copy.
Similarly, because some rootkit variants may cause damage to the Master Boot Record (MBR) and the Boot Configuration Data (BCD), additional 'manual' action should be taken. You will need to run the following commands using the "bootrec.exe" tool to ensure a complete repair of your computer:
- bootrec /fixmbr
- bootrec /fixboot
- bootrec /rebuildbcd
For more details on these commands, please refer to Microsoft Security Article KB927392, with specific focus to the options "/fixmbr", "/fixboot" and "/rebuildbcd".

*Warning: It's a known issue for many offline scanners (including WDO) that can not 'automatically' repair the MBR, but only detect and partially remove threats, so this repairing needs to be done 'manually'.
*Note: There have been reports at this Community [Source], however, that Anti-Rootkit utility TDSSKiller (from Kaspersky Labs) may automatically repair the MBR (though not guaranteed) when cleaning the system off this type of threats, thus it's most recommended. See > How to remove malware belonging to the family Rootkit.Win32.TDSS (aka Tidserv, TDSServ, Alureon). You may also find this TDSSKiller User Guide (from bleepingcomputer.com) to be of assistance.
