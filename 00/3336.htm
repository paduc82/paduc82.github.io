don cuthbert say:

That error number means that the specified product key has been blocked.  No product key will work to activate Windows 10, unless you buy one.  The free upgrade will not use or require any product key to be entered.

 

The embedded product key is blocked, as it is never used to activate Windows 10.  No product key should be entered during upgrade or a clean install of Windows 10 free upgrade.

 

Auto-activation happens only after detecting that an upgrade has been done from an activated, qualifying OS (Windows 7 or 8 or 10 Insider Preview).  The device is then registered as a licensed Windows 10 device, allowing future clean installs of Windows 10 without any product key.

You will have to reinstall and activate Windows 7 with its product key and then upgrade from within the activated Windows 7 to result in an activated Windows 10.

Directions are on this page:

http://windows.microsoft.com/en-us/windows-10/media-creation-tool-install

All we are trying to say is that unless you upgrade a certain way it will not activate.  There are only two official ways to upgrade to Windows 10 for free.  Both have to be launched from within a running, activated Windows 7 SP1, activated Windows 8.1, or activated Windows 10 Insider Preview.

  • Media Creation Tool - "Upgrade this PC now"

a. user say:

Hello Mridul25:

 

Thank you for posting to the forum regarding your update issue.  Error code 80040154 most often indicates an issue with the software or the computer you are currently using.  There are a couple things that may help resolve this.

 

The first thing we suggest you do is uninstall the Zune software from your computer, http://support.xbox.com/en-US/zune/software/remove-and-reinstall-zune.  After this completes, please verify it is fully removed, click start > computer > C:/ > program files > look for Zune folder.  If you find this folder, delete it and all of it's contents.

 

Once the software is fully uninstalled, check your computer for Microsoft updates, http://update.microsoft.com/microsoftupdate.  Be sure you click the link that reads "check online for updates from Microsoft update".  Run and install any and all updates that are found, then restart the computer.

 

Once the computer reboots, install the software again, http://www.microsoft.com/download/en/details.aspx?id=27163.  After the software as installed to the computer, connect your phone and try to update the phone one more time. 

 

If you continue to get this error we have narrowed the issue to the PC itself, the phone or the network.  The next thing to do would be try the update from another computer on another network.  If you get the same exact error under these circumstances, the issue is likely the phone.  If you get further through the update, there is likely an issue with a third party piece of software blocking the update or registry corruption causing this issue.  If you get the same error, the phone may have hardware or firmware issues that prevent the update from completing.

 

Please let us know if you have any other questions or concerns.
