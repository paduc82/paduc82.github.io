mel77084 say:

Laptops often collect lint that packs against the screen where air exits, and when enough of it accumulates, the cooling is no longer adequate, so the overheated processor shuts down during activities such as a scan. The only solution is to disassemble the laptop and clear the airflow path.

The degree of difficulty varies from one to another, but going through it once is enough to make one watch to do everything reasonable to prevent dust/lint from entering the air intake in the future.

A cooling pad might help if the blockage isn't too severe, but there is no real substitute for a proper cleaning.

mr.johnson89 say:

I have been able to create a drop down list being able to save multiple selections in the same cell.

I would like to figure out how to have MULTIPLE of these multiple selection drop down cells.

Here is the code i used for the first drop down cell.

Option Explicit

Private Sub Worksheet_Change(ByVal Target As Range)
'Code by Sumit Bansal from https://trumpexcel.com
' To Select Multiple Items from a Drop Down List in Excel
Dim Oldvalue As String
Dim Newvalue As String
Application.EnableEvents = True
On Error GoTo Exitsub
If Target.Address = "$C$109" Then
  If Target.SpecialCells(xlCellTypeAllValidation) Is Nothing Then
    GoTo Exitsub
  Else: If Target.Value = "" Then GoTo Exitsub Else
    Application.EnableEvents = False
    Newvalue = Target.Value
    Application.Undo
    Oldvalue = Target.Value
      If Oldvalue = "" Then
        Target.Value = Newvalue
      Else
        If InStr(1, Oldvalue, Newvalue) = 0 Then
            Target.Value = Oldvalue & ", " & Newvalue
      Else:
        Target.Value = Oldvalue
      End If
    End If
  End If
End If
Application.EnableEvents = True
Exitsub:
Application.EnableEvents = True
End Sub
