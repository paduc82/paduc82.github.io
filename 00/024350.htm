This is one case where ActiveX controls will work best and I would use Radio Buttons for the Options as by assigning a common group name to each of them, they will act as mutually exclusive, and then use CheckBoxes for the Sub Options.

Then, with the following code in the This Document object, the check boxes will be checked as you require, depending upon which of the radio buttons is true

Private Sub OptionButton1_Click()

If OptionButton1.Value = True Then

    CheckBox1.Value = True

    CheckBox4.Value = False

    CheckBox6.Value = True
