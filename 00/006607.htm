There is no such version as Word 97-2003. There are Word 97, Word 2000, Word 2002, and Word 2003. If you have any one of those, you can't rotate a text box. Your screen shot shows that you have Word 2013 or 2016, probably from Office 365, but your document is in Compatibility Mode, meaning that it is in an older format. You can rotate a text box in a .docx document created in your current version of Word, but you cannot rotate a text box in a legacy .doc document (in Word 97-2003 format) or a .docx created in Word 2007. If you want to be able to rotate the text box, you can convert it to the newer format using Convert or Save As.

Solution: Save the document in the .docx file format.

Important

The following procedure changes the file format of the document. If you plan to share the document with people who use an earlier version of Word, you might want to work in compatibility mode and leave the document in its current file format. This is because other people may be unable to edit parts of a document that you create by using new or improved features in Word 2011.

  1. On the File menu, click Save As.
  2. On the Format pop-up menu, click Word Document (.docx).
  3. Make sure that the Maintain compatibility with Word 98-2004 check box is clear, and then click Save..

daniel g. grau say:
You can be old in body, but not in spirit. The fact that you are using a computer attests to that. Myself, I'm pushing 60, but a long way from pushing daisies.

Some years ago, they decided to change the preferred format for Office files and now most Office document names have file extensions ending with x (.docx, .pptx, .xlsxâ€¦). The new format supports new features that are not available in the older versions of Word.

When Word shows "Compatibility mode," all it means is that the file is old, saved in the "Word 97-2004" format (their extension is .doc). Other than that, working with .doc files should not be a cause for concern. Note that manually changing the extension won't suffice to convert a fileâ€”you need to do a Save As and choose the new format. 

Word V.x is originally released in 2001. (How time flies, don't it?) In keeping with renovation, "Normal View" is now called "Draft View," while the old "Page View" has become "Page Layout View." I feel younger already.

A document will open in whatever view it was last saved in. Thus, the only way to change the view permanently is to force a re-save. For this, you have to enter a dummy change (type something, then backspace), so Word will allow you to save it. Alternatively, you could do a Save As and select the newer format (first on the list, .docx), but I would not bother.

To improve how an older document looks in your screen, go to Preferences > Compatibility and select the version of Word in the dropdown menu.

To conclude, you don't need any conversion to share those filesâ€”they will all open on the PC, unless their Word version is really old.

gabriel bratton say:

Something you could try is adding your account to the current profile but as an IMAP account.  I think IMAP is the default preference for Yahoo.  A difference is that POP accounts use PST files and IMAP accounts use OST files to store the data.  

For example, put your email address in this page, https://www.apple.com/support/mail-settings-lookup.  It will tell you the IMAP settings to use to connect it in Outlook.  Add the account in Outlook, File, Account, Add.

Try out the newly added IMAP account and see if it causes the same problem.  You can always go back and remove it if you don't like it for some reason.  Go to File, Account Settings, Account Settings.

If you really want to stick with POP I would try a secondary profile and add the account as POP to that new profile.
