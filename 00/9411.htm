What's the time limit for OneNote page version preservation? - Microsoft Community

ukrainetrain say:

What's the time limit for OneNote page version preservation? Can the OneNote page versions be extracted programmatically?

I need OneNote page versions to be preserved in perpetuity. So, is there a time limit for their preservation? If the older versions are discarded automatically, is there a OneNote page version-specific API or SDK that would enable me to extract OneNote page versions to some DB programmatically before they are discarded?

Thanks a lot in advance.

neha singh msft say:

Hi UkraineTrain,

We create a new version any time a change is made to a page and either a) the author is different from the last change, or b) it’s been an hour since the last version we pinned.

Every time we create a new version, we review the existing versions to see if any should be pruned. The default pruning rules are:

  • We keep all of the changes for the first 2 days.
  • Older than that, we remove all but the first from any 1-hour period for the next 5 days.
  • Older than that, we remove all but the first from any 1-day period, for up to 10 unique days.
  • After all of the above, we delete all versions.

All three of those numbers (2, 5 and 10) are all customizable by policy. Furthermore, there are two more settings that can be changed.

  • Version pruning can be turned on or off globally.
  • A fourth number can be set which is the age (in days) after which any version will be pruned, regardless of the other rules. By default, this is -1 (off).

Regardless of what numbers are plugged in, versions are only pruned when either a new version is being created, or when the containing notebook is being optimized.

To extract OneNote page programmatically, I’ll suggest you create a new thread in our MSDN forum for expert assistance.

For your reference: onenote api for page versions  

Regards,

Neha

    Dim ws As Worksheet

Word2013 document automatically adds strikethrough and bold - Microsoft Community

zygiella say:

Word2013 document automatically adds strikethrough and bold

In a Word2013 doc, every time I type an email address it automatically changes it to BOLD and STRIKETHROUGH.  It does this only on email addresses.  This is only on ONE particular document that is used and updated frequently.  I have tried copying all and pasting onto a new document, but it still does it, since the formatting comes with it.  I cannot find anyplace to undo this weird automatic setting (which I did not ever set).  This document originated on an old computer with the previous version of Word, but I have since made brand new docs with the pasted content, and the issue continues. It's been happening for ages.   This is Word 2013 on MS Office Professional 2013. Any ideas? This is driving me nuts.
jay freedman say:

Word has an option in File > Options > Proofing > AutoCorrect Options > AutoFormat As You Type for "Internet and network paths with hyperlinks". As part of its operation, it applies the character style named Hyperlink to the link it creates.

The Hyperlink style's default definition includes a single underline and the color defined for hyperlinks in the current document theme (usually blue). In your case, the Hyperlink style in the old document has evidently been redefined as bold and strikethrough (you didn't mention a color).

Copying text that has a style applied to it and pasting it into another document carries the style along and overwrites the style's definition in the new document.

The solution is to redefine the style back to its original settings. To do that:

  • First press Ctrl+Alt+Shift+S to open the Styles pane. If you don't see the Hyperlink style in the pane, click the Options link at the bottom of the pane; in the dialog, set the two dropdowns to "All styles" and "Alphabetical" and click OK. 
  • Right-click the Hyperlink style in the Styles pane.
  • Click "Delete Hyperlink", and click Yes in the prompt that appears. Contrary to the wording of the prompt, Word won't really "delete" the style because it's a built-in style; instead it will be returned to its default settings.
  • Save the document.

Why won't the symbols in my excel spreadsheet print?

kergga say:

A few things to check;

 

(1) It might be your printer driver. Try printing the document to a pdf, to a different printer, or even from a different computer to see if the boxes print. If so, then it is most likely a printer driver issue.

 

(2) It may be an issue with the order of the objects on your page, depending on how your form is set up. Are there any other objects (shapes, textboxes, controls, etc) underneath the symbols that are not printing?

 

(3) Some issues with image filetypes have been reported to cause problems on certain printer drivers. If your symbols were pasted in as images (EPS files, etc) try pasting one in a different format (JPG or BMP) and see if that has an impact

 

(4) If the symbol is a shape: right-click the image or shape, select "size and properties", click the "properties" tab, and verify that the "print object" box is checked.

 

HTH,

Keith
