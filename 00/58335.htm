Face recognition seems stalled in Windows Live Photo Gallery - Microsoft Community

a. user say:

Face recognition seems stalled in Windows Live Photo Gallery

User reporting issue with facial recognition and grouping processes because Windows Live Photo Gallery stopped providing further suggestions for people who have been tagged before.

User reporting issue with facial recognition and grouping processes because Windows Live Photo Gallery stopped providing further suggestions for people who have been tagged before.

a. user say:

The reason for this behavior is because of the current Windows Live Photo Gallery implementation (trade-off between accuracy and coverage - the internal recognition and grouping algorithms are designed to focus on accuracy as a first priority and coverage second). The implication of this design decision is that WLPG only predicts on a fraction of faces.

 

Early on in the tagging process, there are a large number of untagged photos that are easy to group together – these are the group that are presented early, and which users will quickly complete tagging. When the easy ones are done WLPG makes not only a lower rate of predictions, but this rate of predictions is on an even smaller number of untagged faces and with time the system stops finding more matches.

 

Reason:

Face recognition in its current version is designed to provide accurate people tagging suggestions to users.  Microsoft is working on improved recognition capabilities in the next version of Windows Live Photo Gallery.

 

Workaround:

  1. Finish all suggestions until the batch people tagging dialog is empty
  2. Click on Tag them or select the All detected people link in the People section in the ribbon
  3. Once that is completed the batch people tagging windows will show all detected faces from your pictures
  4. Select and mark faces that you want to tag

Another method of increasing the number of recognized faces/people suggestions (by decreasing accuracy) is to modify a registry key as shown below:

  1. The value is under the key: HKEY_CURRENT_USER\Software\Microsoft\Windows Live\Photo Gallery\Library
  2. Add a DWORD value of name FaceRecognitionThreshold
  3. There are 4 values that you can set this as: 0-3, with 1 being the default. 0 is more stringent, 2 is moderate, and 3 is loose.
  4. Close PG and reopen it to force recognition to rerun. To force recognition to re-run without closing the app, you can open a command prompt and run
    C:\Program Files\Windows Live\Photo Gallery\WLXPhotoGallery.exe /RunFaceRecognition . You will then need to hit F5 after the clock icon goes away. (if you’re on an x64 machine, then it’s c:\Program Files (x86)\...)

The issue could be related to Display drivers then please try the steps provided to uninstall the Display Adapter from Device Manager, uninstall the device drivers from program list and then download the latest drivers from the manufacturer's website.

 

Step 1:

Uninstall the Device from Device Manager

  • Open Device Manager
  • Expand the Display Adapter section.
  • Find the Display card Driver.
  • Right click Display card Driver and select Uninstall.
  • Select the check box delete the driver software for this device.
  • Reboot the computer after uninstall process has finished.

 

Step 2:

Uninstall the Device Drivers from Program List in Control Panel

  • Click on start, select Control Panel
  • Now click on Programs and select Programs and Features.
  • Find the Display card Driver.
  • Right click Display card Driver and select Uninstall.
  • Reboot the computer after the uninstall process has finished.

how to create a new local disk on windows xp? - Microsoft Community

shreyaskhan say:

how to create a new local disk on windows xp?

how to create a new local disk on windows xp?

tricky300 say:

You cannot "create" a disk you have to seek out, buy one and then install it to your PC. Make sure you buy the correct type and capacity. Make sure that your PC can accomodate it physically and technically. You might also need to buy a (compatible) data cable and power supply connector/converter.

 

Or, if by "disk" you really mean partition, you must have unpartitioned space available on the current disk, in which case you can use disk manager. If there is no unpartitioned space (or too little to create a viable partition) then you must shrink and/or move partitions on your current disk using 3rd party partitioning software.

 

To start disk management...

  • Click Start
  • Click Run
  • Type diskmgmt.msc and hit Enter

There are many partition manipulation packages available, some are free some are not. Google partition management software. Do research and choose carefully.

 

Warning! Manipulating partitions without fully understanding the potential consequences can lead to an unbootable system and possibly an unrecoverable system too. Make sure you are fully backed up first and have the means to effect recovery from external media if necessary.
