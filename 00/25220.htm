Row numbers are meaningless since a record's position in a continuous form or datasheet view is dependent on the current sort order. Change the sort order and a stored row number is out of order.

If you need to preserve the sort sequence of a set of records, is there some other value that can do that? If you are importing the data from a text file. If you use an autonumber in your table, then the autonumber should preserve the sequence and you can sort by it.

Sub SaveOLEMsgs()

    Dim shp As Word.InlineShape

    Dim itm As Object

    Set doc = ActiveDocument
