jay freedman say:

The Greek characters might be appearing because you accidentally applied the Symbol font at the point where they start (maybe by inserting a symbol from that font). Like all other formatting, if you continue typing from that spot, the font continues in effect until you change to another font or remove the formatting. To remove it, select the Greek characters and press Ctrl+spacebar (the shortcut for clearing all character formatting that isn't part of the current style).

Similarly, the "All Caps" font formatting could have been applied by pressing Ctrl+Shift+A when you meant to press only Shift+A, and all typing from that point onward would appear in all capitals. It, too, can be removed by selecting the affected text and pressing Ctrl+spacebar.

rick rothstein, mvp - excel say:

I don't think there is a way to control that using manual copy/paste. Here is a short macro that can be used to straighten out these kinds of dates. Select the range containing the dates that need to be converted and then run this macro...

Sub StraightenOutDates()
  Dim Cell As Range
  For Each Cell In Selection
    Cell.Value = CDate(Format(Cell.Value, "m/1/19dd"))
    Cell.NumberFormat = "mmm-yy"
  Next
End Sub
