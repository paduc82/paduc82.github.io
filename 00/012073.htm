ken sheridan say: You only need code in the parent form's Current event procedure:

Private Sub Form_Current()

    Me.AllowEdits = Not Me.Verified
    Me.sfrmTransactions.Form.AllowEdits = Not Me.Verified

End Sub

Note that sfrmTransactions is the name of the control in the parent form which houses the subform.  This might or might not be the same as the name of its source form object.

Similarly the code to unlock the form and subform would be:

Private Sub OfficeUse_Click()

    Me.AllowEdits = True
    Me.sfrmTransactions.Form.AllowEdits = True

End Sub
