To restore desktop icons in Windows 10, you need to rebuild the Icon Cache. In order to rebuild the Icon Cache, you need to delete the IconCache.db file.

Here's how:

  1. Open File Explorer and navigate to the following folder:
    C:\Users\%username%\AppData\Local\Microsoft\Windows\Explorer
  2. In the Explorer folder, you will see a lot of files with the iconcache name. Delete all of them.

Once you have deleted all the iconcache files, a folder names IconCacheToDelete will appear. Simply restart your PC and this folder will disappear.

    arminatorx say:

    If there is a long time until the boot logo appears, then this is similar to a PC that has been shut down unexpectedly. 

    During that time on the black screen, the Xbox One console is checking the file system on it's internal drive for logical errors that might have happened when it was writing on the disc, but was interrupted unexpectedly (e.g. power outage, or hardware failure). 

    If the file system is in an inconsistent state, the console checks that during the next boot. 

    It could also mean, that the system partition of the hard drive became corrupted. 

    You could try to use an USB drive to do a factory reset and try to refresh the system installation via USB.

    If that doesn't help, it might be that the hard drive of the console is faulty and the console has to be repaired/replaced.
