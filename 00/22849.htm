How To Copy Style - Microsoft Community

trip_to_tokyo say:

How To Copy Style

On the screen in front of me I have a slide that:-

1. has one shape (rectangle) with white text on black background.

2. has another shape with yellow text on orange background.

I want to copy the style from number 1. above to number 2.

How do I do that please?

TIA for any replies.

beth melton say:

I tend to use Copy Style and Paste Style instead of the Format Painter. To copy the object style, press Ctrl + Shift + C. To paste the object style, press Ctrl + Shift + V.

If you like these commands and prefer to click a button you can add them to your Quick Access Toolbar:

  1. Right-click the Quick Access Toolbar and then click Customize Quick Access Toolbar
  2. From the Choose commands from drop down, select Commands not in the Ribbon
  3. Select Apply Style and then click Add to add it to your Quick Access Toolbar.
  4. Do the same for Pick up Style

Also note you need to make sure the object is selected and that the text isn't active in order to copy/paste the shape formats.

~Beth Melton Microsoft Office MVP
