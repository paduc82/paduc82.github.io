    Offset function stopped updating

    Hi, I'm using excel 2016 and I'm using the offset function to have rolling charts. So after extending my table how do I get my offset set function to update so my charts will display the current date and data?

    bernie deitrick say:

    Post your formula - but as a starter, use offset with a reference cell from outside your table, so that when you delete rows of your data, you are not deleting the reference cell used in the formula. 

    So, continue to use your macro, but make sure you are not deleting the row with that reference cell.

    josephflowers say:

    An example of the formula is

    =offset(data!$M$3,count(data!$M:$M),0,-184,1)

    The table is A3 to Y2190 so the only thing that changes in the formulas is the column. The vba code deletes row 3 when data is entered in row 2190

    bernie deitrick say:

    Change from

    =offset(data!$M$3,count(data!$M:$M),0,-184,1)

    to

    =offset(data!$M$2,count(data!$M:$M)+1,0,-184,1)

    Then deleting from row 3 will not matter

    And if you want to make the formula able to be copied easier, try:

    =offset(data!M$2,count(data!M:M)+1,0,-184,1)

    Which can be copied across to return other columns.

    Also, if you always have values to row 2190, then you could use this, for example, which will ignore row deletions and insertions, and always start from row 2190:

    =SUM(OFFSET(INDIRECT("Data!M2190"),-183,0,184,1))

    mike h.. say:

    Hi,

    As soon as you start concatenating like this you lose the date format and you get the number so you must reformat the date as you need it in the concatenate formula. So if E3 contains a date then this will return the year

    =TEXT(E3,"YY")

    we can then concatenate that with another cell

    =TEXT(E3,"YY")&G3

    We may now have a problem with G3 because you said it was formatted to display as 3 digits and you'll lose that formatting too so we may need to put that back like this

    =TEXT(E3,"YY")&REPT("0",3-LEN(G3))&G3

    You can of course change the date format in this formula to be whatever you want.
