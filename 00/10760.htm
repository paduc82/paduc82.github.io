Starting with the backup question............

Your contacts are located in the default contact folder which will be located in the pst file set as the "default data file" for the profile. Going to guess that the 2nd pst file in this profile is an "archive" pst file unless you added your own or created a new one. In a simple environment, your default data file for a single POP account will be the same file where your email is delivered. In other words, the contact folder is in the same folder tree as your <Inbox>.

If the above is accurate, all you need to do is take note of this pst's folder/file name. With Outlook closed, make a copy of that PST file and save it whereever you want (preferably not on the same drive as where the original is located).

BTW - if you go back to File --> Account Settings --> Email tab --> when you highlight the POP account and look down to the <Change Folder> button - the data file shown there should be the same as your default data file if the above assumption is correct.

As for contacts from Outlook desktop to WIn10 Mail - there's no connection between the two. If there is some pressing reason for this requirement, quickest solution would be to create an Outlook.com account, move the contacts to the Outlook.com account at which point any device connected to the account will be able to share the info. If you do decide to go that route, suggest you make the .ost data file associated with the Outlook.com account as the default data file so that the default contact folder for the profile are one and the same. No sense going any further into this option unless you plan on following that route.
