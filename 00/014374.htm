scottgem say:

Generally this is done BEFORE you call the report. 

For example. Lets say you had a report that reported on Sales figures by store. And you wanted the ability to run the report for only a selected store. You would create a form with a combobox where the user can select the store. You would also have a button to run the report.  The code behind the button would use the WHERE clause of the OpenReport method. For example:

DoCmd.OpenReport "rptSales",,,"StoreID = " & Me.cboStore

Access reports are generally not interactive. But there is a Report view that provides some interactivity, but not completely.

Okay David,

so, using 'Folder picker' method, select the desired folder

in the below vba macro

change: vHLeft / vHCenter / vHRight as needed

sample vba macro

Sub Edit_PageSetup_Headers()

'## 13 Apr. 2019 oAnast ##

Dim wb As Workbook

Dim sh As Worksheet

Dim sPath

Dim fName

Dim vHLeft

Dim vHCenter

Dim vHRight

vHLeft = "&B&I&""Arial Narrow""&10" & "Date: " & "&D" & vbCr & "Place: ABCDEFG (line-2)"

vHCenter = "&B&I&""Arial""&12" & "Center Title (line-1)" & vbCr & "Center Title (line-2)"

vHRight = "&B&I&""Arial Narrow""&10" & "data1-data1 (line-1)" & vbCr & "data2-data2 (line-2)"

With Application.FileDialog(msoFileDialogFolderPicker)

.Title = "Please, Select a Folder"

If .Show = True Then

Application.ScreenUpdating = False

sPath = .SelectedItems(1) & "\"

fName = Dir(sPath)

Do Until fName = ""

Set wb = Workbooks.Open(sPath & fName)

For Each sh In wb.Sheets

With sh.PageSetup

.LeftHeader = vHLeft

.CenterHeader = vHCenter

.RightHeader = vHRight

End With

Next sh

wb.Save

wb.Close False

fName = Dir()

Loop

Application.ScreenUpdating = True

End If

End With

End Sub

In that situation, it may be easiest to create a batch file to issue the commands.  Then you can put it on your desktop or task bar to have quick access to run it as required.  The brain dead simplest form of this .BAT file would be:

net stop “Microsoft Office Service”

net start “Microsoft Office Service”

net stop

if %errorlevel% == 2 echo Could not start service.

if %errorlevel% == 0 echo Service started successfully.

 

net start “Microsoft Office Service”

if %errorlevel% == 2 echo Could not start service.

if %errorlevel% == 0 echo Service started successfully.
