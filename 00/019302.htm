bob jones aka: cybertaz say:

Mac PPT does not support Object Linking & Embedding (OLE) for any file types other than certain Media files (images, sound, video) & those shown in the Insert> Object dialog on the Create New list.

An existing PDF can be inserted using Insert> Photo> Picture from File, but only the 1st page will display if there are multiple pages. The PDF file, itself, cannot be embedded as an object, nor can a new PDF be created within PPT.

Slides from another presentation can be inserted as slides by using the Insert> Slides From... command or you can open the source presentation, save slides as Pictures, then insert the Pictures into the target presentation file.

Unfortunately, the instructions in the "Sorting a list" article will treat each "line" (paragraph) of the text as a separate item, so sorting that way will make a complete mess  of everything. Don't do it!

The article http://www.gmayor.com/convert_labels_into_mail_merge.htm has a procedure for changing each label's text into a single paragraph.     Dim cBox As CheckBox

Run a macro containing the following code

Dim i As Long
Dim fcode As Range
With ActiveDocument
    For i = 1 To .Fields.count
        Set fcode = .Fields(i).Code
        If InStr(fcode, "SEQ Figure") > 0 Then
            fcode = "SEQ Figure \# 00#"
        End If
    Next i
    .Range.Fields.Update
End With
