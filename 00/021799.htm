    tiny_chef say:

    Hi johng1959,

     

    Welcome to Nokia Forum!

     

    To update the phone through Nokia Suite, it requires a USB connection. Upon checking on the device specification, Nokia 113 does not support USB cable. However, you can still update your device by downloading it directly on the phone (FOTA):

     

    Select Menu > Settings > Device > Device updates.

     

    1. To check if there is an update, select Download device software.
    2. To download and install an update, select Download device software, then follow the
    instructions shown on the phone. The update may take several minutes. If you encounter problems, network service provider.

     

    For detailed instructions on how to backup and update your phone, see page 44-45 of the manual. You can download the user guide here: http://nds1.nokia.com/phones/files/pdf_guides/devices/111_113/Nokia_111_113_UG_en.pdf

     

    armagonfin say:

    Hey,

     

    as it was already stated above, Nokia 113 does not have USB-connector, so updating the device with Nokia Suite is not possible. The device does support Nokia Suite via Bluetooth connection, but installing device SW updates via Bluetooth is not possible. So the only end-user update option for this device is via FOTA. Find the instructions from here: http://www.nokia.com/global/support/software-update/belle-asha-symbian-software-update
