In the source document, bookmark the numbered paragraph that you want to cross-reference. Insert a cross-reference to the item in a SET field (for example in the header paragraph). Each SET field creates an invisible bookmark which can be used to reference its field result.

{ SET mybookmark { REF tryout \n } }

1. Bookmarked text here.

In the target document, insert an INCLUDETEXT field which references the bookmark defined by the SET field (this is for the paragraph number). Insert another bookmark which directly references the bookmarked text (this is, obviously, for the bookmark text).
