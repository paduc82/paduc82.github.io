lironl say:

I found the solution.

The problem was that my home directory (C:\Users\<username>\) where <username> is substituted for my real user name) had the system and the hidden attributes set.  Clearing these attributes caused My Pictures and My Videos to appear in Windows Live Photo Gallery.

To fix the problem:

1) Start the command prompt

    Click the start button, choose All Programs, then choose Accessories, then choose the command prompt,

2) If the command prompt starts in the home directory, execute the following command:

   attrib -h -s .

Otherwise, execute the following command:

    attrib -h -s (full path name, enclosed in quotes if there are spaces)

e.g.

     attrib -h -s C:\Users\Example

or

     attrib -h -s "C:\Users\Example Username With Spaces"

3) Exit command mode.

4) Close and restart Windows Live Photo Gallery.

My Pictures and My Videos should now appear correctly.

Hi Andrew,

I am following this thread.  Posted my first answer when initial searching of the Internet didn't turn up the syntax for searching for empty properties via the Search box.  The problem with the Column filtering I suggested in my first reply is that if you apply more than one, you are creating an 'AND' condition for the search, but what we want is 'OR'.

Finally found the key nugget here: Using Advanced Query Syntax To Find Files in Windows 7

To enable Winhlp to work in Windows 10, follow these simple instructions.

  1. Download the updated Winhlp32 installation files, which are compressed into a zip file (winhlp32-windows-7-10-x86-64.zip) from the following link:  https://drive.google.com/open?id=0B_tSrOg4FLp1T0dyTEppaW9SYm8

  3. Run the Install.cmd as Administrator by right-clicking on Install.cmd, then select Run as Administrator.
  4. Acknowledge any privilege warnings and allow Install.cmd to proceed.

That is all there is to it.  You now you have Winhlp working in windows 10!
