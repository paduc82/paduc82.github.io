afzal taher say:

Hi nomisma,

Welcome to Microsoft Answers!

First make sure both machines are members of the same Workgroup which by default is named Workgroup. If both are administrator accounts and
if you are unable to login from only one administator account, then you may have to change some settings to to get access to printers in workgroup.

Check if you have followed all these steps while setting up file and printer sharing from Windows 7 to Windows XP.

Steps to setup file and printer sharing from Windows 7 to Windows Xp:

1. On the Windows 7 machine go into Control Panel \ All Control Panel Items \ Network and Sharing Center then click on Change advanced sharing settings.

2. You will want to verify the following settings under Advanced Sharing Settings for the Home or Work and Public profile.

· Network discovery – on

· File and printer sharing – on

· Public folder sharing – on

3. If you want any user to have access the public shares turn off password protection.  This is located in Advanced Sharing Settings toward the
    bottom of the list.

4. Now if you go into Network in Windows 7 you should see your XP machine and the Windows 7 as well.

5. To share the printer on the Windows 7 machine go into Devices and Printers from the Start menu and double click on the printer icon.

6. Next double click on “Customize your printer”.

7. In the Properties screen click on the Sharing Tab and check the box to share the printer and type in its share name.

8. You can install Additional Drivers before setting up the XP machine. To find the shared folders and devices double click on the Windows 7 machine
     icon under Network.

9. Continue into the Users folder and Public to see the shared folders.

10. Over on your XP machine open up My Network Places to find the Windows 7.

11. Double click on the Share folder to find a list of shared folders in the Public folder on Windows 7.  If you have password protection enabled you will
      need to type in the username and password of the user account on the Windows 7 machine first.

Setup XP With Shared Printer

1. To set up the shared printer in XP you will need to go into Printers and Faxes from the Start menu and kick off the Add Printer Wizard.

2. Now select “A network printer, or a printer attached to another computer” then hit Next.

3. Next select “Connect to this printer…” and type in the path for the printer connected to the Windows 7 machine and click next. 

4. Now click Yes to the confirmation message.

5. Then click Finish the printer to install and complete the Wizard.

NOTE : In some cases you will need to install the x86 XP drivers for the shared printer because the Windows 7 drivers are not compatible with XP. 
When everything is installed open up Printers and Faxes to find the shared printer.

Windows 10 will be free for the first year to all Windows 7, Windows 8.1 and Phone 8.1 users. This means, as long you upgrade within the first 12 months (July 29th 2015 to July 29th 2016), you will not have to pay for Windows 10 at all.

This is more than a one-time upgrade: once a Windows device is upgraded to Windows 10, we will continue to keep it current for the supported lifetime of the device – at no additional charge.
