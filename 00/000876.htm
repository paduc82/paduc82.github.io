brian tillman [outlook mvp 2007-2019] say:
In Outlook 2013, all of your folders are kept in the same file.  The Contacts folder is not a Windows file system object by itself.  There is no easy way to separate your calendar, contacts, and mail folders into different Windows folders.  I can, however, give you a not-so-simple way to share contacts, if you're still interested.  There could be some confusion possible, but it should work.  Write back if you want to know.

sjr29 say: Thanks Brian and yes, I'd like to know the not-so-simple method please.

brian tillman [outlook mvp 2007-2019] say:

Each of the people using the PC should have his or her own account.  This will allow you to share some information and keep other information private.

Your Documents library should be a combination of two folders: your My Documents folder and Public Documents.  In Outlook, create an Outlook Data File in the Public Documents folder with New Items>More Items>Outlook Data File.  Browse to the Public Documents folder, give the data file a name like "Shared Outlook File" or "Shared Contacts" (you can share more than just contacts with this file, if you wish, like a calendar), the click OK.  This data file will appear in your Navigation Pane on the left side of Outlook.  Create a Contacts (or "Shared Contacts" or any other name you like) folder in that data file by right-clicking the root of the data file and choosing "New Folder", entering the name you've decided to use for the folder, and selecting "Contact Items" in the "Folder contains" drop-down.

Now, copy into that folder the contacts you want to share.  You can keep the contacts you want private in your own Contacts folder.  Make sure the folder will appear in the address book interface by right-clicking it, choosing Properties, then selecting the Outlook Address Book tab and making sure the "Show this folder as an e-mail Address Book" check box is selected.

Now, each person should log into their Windows account, start Outlook, and add that data file to their Outlook profile by clicking File>Open>Open Outlook Data File, browsing to the shared data file, selecting it, and choosing OK.  The can then put any contacts they want to share in the Contacts folder in that data file as well.  They should make sure it shows up in their Address Book interface as well, as described above.  In the Address Book interface, you can arrange which Contacts folder is searched first by clicking Tools>Options and selecting the Custom radio button, rearranging the folders as desired.  The one on top will be searched first.  If a match isn't found there, the next one will be searched, etc.

You can share other data, as I said, in this data file.  Have a common calendar there as well, if you wish.  The drawback is that you won't be notified when events reach their start date.

If any of this isn't clear, let me know.

garym123 say:

A soft reset does not lose data. it is like pulling the battery.
Hold the volume down button and the power button down until the phone vibrates, release. This takes about 10 seconds.  The phone will shutdown and restart. 

Ignore the slide down to power off.

It is not a 'factory reset'. A phone reset returns the phone to an out of the box condition
