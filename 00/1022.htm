chekyek say:

Nokia N81 8GB

Using Nsysinfo, the ARM processor-the speed only clock at 192.0 MHz,total RAM 81199104 B.
Why this figures is not the same as advertised by Nokia?
petrib say: Because Nsysinfo asks for the processor speed using the Symbian HAL (Hardware Abstraction Layer) API (Application Programming Interface) to get the speed, and that API returns a static/fixed value.

Most often in the case of Nokia Symbian phones usually is not set correctly, and thus applications using the API report the value of the variable, but not the actual hardware/CPU clock rate.

Of course, it'd make sense if Nokia was careful in changing the value for every device they make, so that it accurately reflects the actual speed.

On the other hand, for 99.999% of the world's people whether the value is accurate representation of the processor speed is irrelevant. (Note also that 76% of reported statistics is made up on the spot.) ;)

jim gordon mvp say:
When font characters do not display, it means these particular characters (technically they're called glyphs) are not available in the active font on your Mac.

This can be caused by one of two things:
  • The version of the font you have on your Mac doesn't have all the glyphs that were in the font that was used to create the content
  • The font that was used is not present at all on your Mac and some other font was substituted and the substituted font doesn have all the glyphs that were in the font that was used to create the content.
The best cure is to install onto your Mac the font that was used to make the content, and delete the old version of that font if you had one.

An alternative would be to select the text and apply a font that you already have that has the glyphs you need.

Quite often, over time users accumulate old and new versions of the same fonts. A good starting point for troubleshooting fonts is to use Font Book (itâ€™s in Applications > Utilities folder) or other font utility to validate ALL your fonts. Delete old versions of duplicate fonts. Delete fonts that don't validate completely. Restart your Mac and empty the trash to get rid of the unneeded fonts and free up disc space.]
