Syncing your Microsoft Edge favorites, reading list, and other data between Windows 10 devices including Windows Phone is by signing in with the same Microsoft account on each device.   1. Select the Start button and select Settings.

To turn on syncing in Microsoft Edge on your PC:

  1. Select More  > Settings.

  2. Turn on Sync.

On your Windows 10 Mobile, launch Microsoft Edge, then go to Settings, and scroll down and tap the Sync your content switch to On.

Then, you may need to restart Edge on one or both devices/machines to force the syncing process.

qa guy commelec say:

You are using Access as if it was a spreadsheet instead of a relational database.

You should have a table for individuals with a primary key and another table using that data as a foreign key in a one-to-many relationship with referential integerity.  Then use a form (Name)/subform (your dated activities) with Master/Child links set using the primary/foreign keys.  You would scroll to the name and add activity date record and the foreign key field would automatically be filled in.

To get there, and to find what you are looking for, use a union query like this --

SELECTName, Date1 AS ActDate

FROM YourTable

UNION ALL SELECTName, Date2 AS ActDate

FROM YourTable

WHERE Date2 Is Not Null

UNION ALL SELECTName, Date3 AS ActDate

FROM YourTable

WHERE Date3 Is Not Null

   ….

UNION ALL SELECTName, DateX AS ActDate

FROM YourTable

WHERE DateX Is Not Null;

 

Then use a totals query –

SELECTName, Min([ActDate]) AS MinActDate

FROM qryYourTable

GROUP BY Name;

 

Build a little, test a little.
