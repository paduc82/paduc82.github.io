Hi xsculder,

Windows Tool tips are small information boxes that appear when we place the cursor on any icon or shorcuts on the desktop. This can be enabled or disabled from the registry.

1. Click Start and type REGEDIT and press enter. This will open the Windows Registry Editor.

2. Now, navigate through to

   HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Advanced

3. Look for the DWORD “ShowInfoTip“, right-click and select Modify and change the value data to 0 to disable Tooltips.

I would like to make some clarifications to fully diagnose your concern.

 

> Are the contacts also reflected on you MS account (Hotmail account)? If you are concerned with contacts that you have on your Windows Live Mail, when you synch your account on it, it will automatically copy the contacts from MS account to your Windows Live Mail. This is also the same with with your stored emails.

 

> When you reinstall the Windows 7, did you reformat your driveC? If so all the data will be deleted and therefore cannot be recovered. If you the back up feature to restore the files, it will only recover the data when you activated it.

 

You can recover the emails deleted in Windows Live Mail by following the steps below. You will need to know the name of the folders that were deleted. Here is the step by step procedure.

 

1. You will first need to select the Show hidden files option.

    a. Right click on the Windows Button (start button)  and then select Open Windows Explorer and click Organize / Folder and search options. Click the View Tab.

    b. Place a check mark in the Show Hidden files, folders and drives option. Click Apply/OK.

    c. Close and restart Windows Explorer.

 

2. Open the WLM Store folder.

    a. Navigate to the following folder.

    C:\Users\your name\ AppData\ Local\ Microsoft\ Windows Live Mail

    (Where 'your name' is the name of the account that you use to logon to Windows 7)

    b. In this folder you will see all of the WLM storage folders. Each email account will have its own group of folders.

    c. Any general storage folders that you created will under the main Storage Folder item.

 

3. Make a copy of the Windows Live Mail folders.

    a. Right click the Windows Live Mail folder and select Properties / Previous Versions Tab.

    b. After the list populates, select a version of the folder with a date and time prior to when the folders were deleted.

    c. Click the Copy Button. In the Copy dialog box, select the Desktop as the location to place the copied folder.

 

4. Move the previous versions of the deleted folders to the WLM store folder.

    a. Go to the Desktop and open the copied WLM folder. Locate the folders that were deleted.

    b. Right click the folder and select Copy. Go to the current WLM storage folders and paste the folder into the same location where it was deleted from.

 

After you have retrieved all of the deleted folders, you can delete the copied WLM previous version folder from the Desktop.
