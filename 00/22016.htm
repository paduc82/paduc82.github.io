stefan blom say:

Which specific options are you referring to? Note that some options let you temporarily hide markup but doesn't actually remove it. To get rid of existing markup in a document, you have to accept (or reject) it, and then turn off the tracking.

To control which kind of changes Word should track in a document, select or clear "Track moves" and "Track formatting" in the Advanced Track Changes Options dialog box.

stefan blom say:

You have to open your template as a document and turn off the tracking options there. The changes will apply to future documents created from the template. Existing files (documents and templates) must be dealt with one by one.

If you are not using a specific template, your document makes use of the Normal template, so that is the one you must edit. To open it quickly, you can use the following trick: In Word, press Alt+F11 which will activate the Visual Basic Editor. Click View | Immediate Window. In the Immediate window, type

NormalTemplate.OpenAsDocument

and press Enter. The Normal template opens in a separate window (and you should now close the Visual Basic Editor). Activate that window, turn off the setting in the dialog box, save and close the template.
