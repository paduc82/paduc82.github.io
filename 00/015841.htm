mandatory form field in MS Word - Microsoft Community

jmhf say:

mandatory form field in MS Word

I have a form set up in MS Word 2007. 

 

How do I make the form fields mandatory to fill in and not be left empty?

 

Thanks!

jay freedman say:
Yes, that can be done. In the template for the form, open the macro editor (Alt+F11). In the Project pane of the editor, expand the project for that template,  expand the "Microsoft Word Objects" icon under that, and double-click the ThisDocument icon. In the blank window on the right, paste in this code:

Option Explicit

Private WithEvents oApp As Word.Application

Private Sub Document_New()
    'assign Word to the application variable
    If oApp Is Nothing Then
        Set oApp = ThisDocument.Application
    End If
End Sub

Private Sub Document_Open()
'assign Word to the application variable
If oApp Is Nothing Then
    Set oApp = ThisDocument.Application
End If
End Sub

Private Sub oApp_DocumentBeforeSave(ByVal Doc As Document, SaveAsUI As Boolean, Cancel As Boolean)
    If Doc.AttachedTemplate <> ThisDocument Then Exit Sub
   
    If Doc.FormFields("Text1").Result = "" Then
        Cancel = True
        MsgBox "An entry is required in the Text1 field.", , "Error"
        Application.OnTime When:=Now + TimeValue("00:00:01"), Name:="GoBacktoText1"
        Exit Sub
    End If
   
    If Doc.FormFields("Text2").Result = "" Then
        Cancel = True
        MsgBox "An entry is required in the Text2 field.", , "Error"
        Application.OnTime When:=Now + TimeValue("00:00:01"), Name:="GoBacktoText2"
        Exit Sub
    End If
End Sub

Sub GoBacktoText1()
    ActiveDocument.Bookmarks("Text1").Range.Fields(1).Result.Select
End Sub

Sub GoBacktoText2()
    ActiveDocument.Bookmarks("Text2").Range.Fields(1).Result.Select
End Sub


Then replace all occurrences of "Text1" and "Text2" in the code with the actual names of the fields you want to check. Save the template.

If you have more fields to check, you need to duplicate the If ... End If sections in the DocumentBeforeSave macro and change the field names, and add corresponding GoBack procedures.

Good Day All,

I am using Word 2010.  When I add captions to my tables, using Reference/Insert Caption, some captions are created in a text box and some are just in a plain text line.  In the same document, both show the style as caption and some are text boxes, others are not.

The biggest problem is that I like to wrap words around my tables and the caption goes in a crazy place (e.g. I select below the table and it is off to the side and I cannot move it.

suzanne s. barnhill say:

Please see my reply in the referenced thread. If you really want a caption to be in a text box, you can insert the caption and then insert a text box around it.

 

donaldcatanzaro say: Hi Suzanne,

It is not clear to me how to change Word Wrapping to something like  In Front of Text when that is not an option in table properties.  There is only "None" and "Around" and my table is already on "Around".

How does one insert a text box around something ?

Also, often times when I place an image or table in a text box and then put the caption in that same text box, the linkage between the label (e.g. Table 2 or Figure 44) is no longer stable.
 

stefan blom say:
The option you mention ("In front of text") is available for text boxes, shapes, and other objects, but not for tables.

To wrap a text box around a piece of text, just select the text and then click Insert tab | Text Box | Draw Text Box.

suzanne s. barnhill say:

I didn't suggest that you could wrap your text box around the table. I said that "you can insert the caption and then insert a text box around it." That is, you can insert a caption, select it, and then use Insert | Text | Text Box | Draw Text Box to put that text in a text box (that is, create a text box around it).
