1. On your keyboard, press the Windows key and R at the same time. The Run window opens.

rik_uk say:

In the Excel Help files it states the following...

Up to 64 IF functions can be nested as value_if_true and value_if_false arguments to construct more elaborate tests.

For example the formula below works just fine in 2007...

=IF(B1=1,1,IF(B1=2,2,IF(B1=3,3,IF(B1=4,4,IF(B1=5,5,IF(B1=6,6,IF(B1=7,7,IF(B1=8,8,IF(B1=9,9,IF(B1=10,10,0))))))))))

This sure breaks the 7 levels of If statements rule in Office 2003! Of course this will cause backward compatibility issues for anyone viewing your files in earlier versions, or saving in a 2003 (or earlier) format...

OK - so for the OR/AND part - consider that maybe some of the true parts in the formula above were actually common results. This means instead of testing for each case individually we can create a logical test for each common result once.eg

=IF(B1=1,"LO",IF(B1=2,"LO",IF(B1=3,"LO",IF(B1=4,"LO",IF(B1=5,"LO",IF(B1=6,"HI",IF(B1=7,"HI",IF(B1=8,"HI",IF(B1=9,"HI",IF(B1=10,"HI",0))))))))))

then this could be simplified by

=IF(OR(B1=1,B1=2,B1=3,B1=4,B1=5),"LO",IF(OR(B1=6,B1=7,B1=8,B1=9,B1=10),"HI",0))

or as the cell reference is the same in each logical test it can be further condensed to

=IF(OR(B1={1,2,3,4,5}),"LO",IF(OR(B1={6,7,8,9,10}),"HI",0))

As the equation above is refering to one cell, the AND function can't be used as it is mutually exclusive (B1 can only be one value at once, not many).

So if we add a reference to cell C1 into the logical test we could have

=IF(AND(OR(B1={1,2,3,4,5}),C1="A"),"LO",IF(AND(OR(B1={6,7,8,9,10}),C1="B"),"HI",0))

This means that both the conditions for cell B 'and' cell C have to match the logical test for the true result to be shown, otherwise the result will be 0.

if you were to write this in its long form it might look like:

=IF(B1=1,IF(C1="A","LO",0),IF(B1=2,IF(C1="A","LO",0),IF(B1=3,IF(C1="A","LO",0),IF(B1=4,IF(C1="A","LO",0),IF(B1=5,IF(C1="A","LO",0),IF(B1=6,IF(C1="B","HI",0),IF(B1=7,IF(C1="B","HI",0),IF(B1=8,IF(C1="B","HI",0),IF(B1=9,IF(C1="B","HI",0),IF(B1=10,IF(C1="B","HI",0),0))))))))))

This is for illistration.

 

I hope this is of help...

=IF(AND(G5<5,F5>75),"Pass","Fail")

In each formula the OR and the AND functions have been used within the IF function. This is what nesting is, using one function within another function.

What the sample shows is that the two IF statements used will produce different out comes based on the 'nested' function.

Firstly the OR function will look at G5 to see whether the attendance mark is less than 5 OR the average pass mark is more than 75. If either one of these conditions is true then it passes the TRUE result to the IF function, which will then result in "Pass" being returned. Should both of the conditions be false (attendance was 5 or greater, average pass was 75 or less) then the OR function would pass a FALSE result to the IF function, which returns a "Fail".

Secondly the AND function will look at G5 to see whether the attendance mark is less than 5 AND the average pass mark is more than 75. If both of these conditions is true then it passes the TRUE result to the IF function, which will then result in "Pass" being returned. Should either one of the conditions be false (attendance was 5 or greater, average pass was 75 or less) then the AND function would pass a FALSE result to the IF function, which returns a "Fail".

This is the bases of nesting functions, one function passes its result to the next.
