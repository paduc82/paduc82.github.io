One of the quickest ways to determine if a message is phishing scam or not is to be sure to check the actual underlying links within the message.

This is done by hovering over any link even if it <looks> like a legitimate address and check the actual underlying link OR right-click on the link --> select <Copy> --> paste it in any text editor (i.e. Notepad) and see where it points to. If it's not a valid MS URL (or not a valid for the vendor from whom this email alleges it's from) then immediately delete it. No reputable company is going to ask you to re-enter your credentials to verify your ID from a link within an email message.
