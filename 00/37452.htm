considerate_guy say:

Are you judging this in the preview of Movie Maker?  Most likely Movie Maker is unable to keep up with a real time full screen preview of your pictures.  As it is loading a photo completely, it might show you a lower-resolution version until the full resolution is loaded.  But perhaps by then, you are on to the next photo so you don't see it.

 

There are a couple things you could check:

  1. Check that you have your photos on your internal hard disk and you aren't loading them from some slower external location (e.g. thumb drive, network drive, etc.)
  2. Instead of forcing Movie Maker to try and keep up in real time, save your movie first.  That will give it all the time it needs to load and process the photo to be in your final movie.  To save your movie, use File > Save movie.  That will create a .mp4 file that you can play (full screen if desired) in Windows Media Player.  That should not stall/blur like the preview does.

Thanks for providing sufficient information. You're correct, WMI service is an internal service that allows programs in a computer to communicate with each other and its disablement could be the root cause.

There's two ways we can turn it on. Kindly try these and see if the service starts running again:

I. Changing Windows Management Instrumentation's Startup Type.

1. Press the Windows key + R to open the Run prompt.

2. Type "services.msc" and press OK. This will open the Services window.

3. Click on any service and type "windows management". This will not input anything but will instead browse to Windows Management Instrumentation.

4. Right-click on it and hit Properties.

5. Change the Startup type to Automatic. You must use an account with administrator privileges for this.

II. Start WMI Service through CMD.

1. Press the Start menu and type "command prompt" and click on the result.

2. Right-click and choose Run as administrator.

3. Type this code and press Enter: net start winmgmt.

Once done, please see if you can use your apps again. We'll be here if you need more help.
