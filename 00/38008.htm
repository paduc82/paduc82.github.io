lemp say:

The "Print Screen" button is poorly named.  Pressing the button merely captures a screen image and saves it (in RAM) to the Clipboard.  To print it, you need to open a graphics program, e.g., Paint, and PASTE the Clipboard contents into the program, then print.

In Windows 7, it's far easier to use the Snipping Tool (click Start, type Snipping Tool into Search box, press Enter), but you'll still have to save the captured image before printing it. 

See, for example, How to Take and Print a Screen Shot in Windows 7: http://www.library.illinois.edu/it/helpdesk/howto/snippingtool.html
