hairyfool say:

If stuck at the motherboard initialisation key then it means the Hardware initialisation is not completing. This would not be down to Windows.

Once Power On Self Test (POST) completes it then tries to initialise any hardware attached before blanking the screen and handing control to the 1st bootable device. If there was a problem with the drive or the O/S there are a specific set of errors it would generate.

Unplug EVERYTHING apart from the keyboard and display and try the boot again. If that works then attach the mouse and see if everything is OK. The commonest cause are USB drives, printers with storage, Hubs.
