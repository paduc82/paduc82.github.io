  4. Swipe Next and select Repair your computer.

  5. You will then see a blue screen and an option to choose.

  6. Choose the option Troubleshoot and select Advanced options.

  7. Choose Automatic Repair to restart from Advanced boot option.

  8. Once restarted, choose an account with administrative privileges from Automatic repair window credentials of which you want to use to repair Windows 8.

Hi gprr1234.

Unfortunately it is true.

Skype ID or Login cannot be moved / split / deleted.

I could suggest only one workaround, it depends on what domain you using.

Example, you have Skype ID - name_skype and it is merged with email @domain.com.

What you can do.
1) Create a new alias for this account, @outlook.com
2) Make this alias primary
3) Delete @domain.com alias after 24 alias.
4) Create a new account with name @domain.com.


Result: name_skype and outlook - one account, @domain.com is different account.


Important:
It works only if @domain.com is still available for registry (for live/msn etc it will not work).

If domain belongs 3rd party services that after deleting alias account could be re-created in the same time.

If domain is outlook/hotmail - only after 30 days
