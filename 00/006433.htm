rob koch say:
It started about a month ago, and it is on a Dell Inspiron 1525 laptop so there is nothing to clean.

Most likely this is an issue with either the driver for the Touchpad or a setting related to the Mouse/Touchpad configuration.

It could also be a physical problem with the Touchpad, but based on your description of the symptoms this sounds like a common problem with the driver, possibly caused by a recent Windows Update or configuration change.

I'd first go to the Dell website, select Support, then Drivers and Downloads, and finally Mouse, Keyboard & Input Devices.  Under this you'll find a Dell Touchpad / Pointing Stick driver to download and execute.  It's always best to install the latest version of this driver, since older versions sometimes display flaky operation due to changes in core Windows support components delivered via Windows Update.

If this doesn't resolve the problem, check that various settings relating to the Mouse in Control Panel such as "Pointer Options, Automatically move pointer to the default button in a dialog box" aren't checked.  Other settings related to sensitivity of the Touchpad such as Touch Pressure or Palm Rejection can also affect whether a slight brush of the touchpad while typing causes the pointer to move randomly around the screen.

Other settings may also have unexpected effects, so resetting the Touchpad using the Default button is a way to test whether earlier changes may have created the problem.

I've seen and heard of these random movement issues myself with touchpads in the past and most commonly they are resolved via driver updates or settings.  In rare cases the physical touchpad is the problem, but that should be easily determined by running a finger around the pad area in a pattern to see whether bad spots exist.

The movements are often so strange and unexpected that they appear to be in someone else's control, but usually this is simply an illusion based on the fact that they don't match your expectations from your own motions.
