a. user say:

FileCopy will require you to use the same extension as the original workbook.  And since you want to save a copy of the workbook as an .xlsx workbook, maybe this will do...

Private Sub Workbook_BeforeClose(Cancel As Boolean)

    Dim Msg As String
    Dim Ans As Integer
    
    Msg = "Would you like to file an archive copy of this file?"
    Ans = MsgBox(Msg, vbQuestion + vbYesNo)
    If Ans = vbYes Then
        Application.DisplayAlerts = False
        ThisWorkbook.Sheets.Copy
        ActiveWorkbook.SaveAs Filename:="C:\Users\user name\Archive\NEW FILE NAME (" & Format(Date, "YYMMDD") & ").xlsx", FileFormat:=51 'xlOpenXMLWorkbook
        ActiveWorkbook.Close
    End If

End Sub

Note that if a workbook by the same name already exists, it will be automatically overwritten.  To avoid this, you can add a time stamp to your date stamp.  For example...

Format(Now, "yyyy-mm-dd hh-mm-ss")

Hope this helps!
