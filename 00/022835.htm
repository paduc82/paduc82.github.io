anula d'souza say:

Hi Sheena,

Thank you posting in Microsoft Community. As I understand you want to synchronize the multiple calendars configured in Outlook 2010 with Windows 8 calendar.

 

Outlook 2010 calendar and Windows 8 calendar can be synchronized only if you have a Windows Live account.

If the same Windows live account is configured in both Outlook and in Windows 8 Calendar then any changes you do in any of the calendar will be reflected on the webmail and gets synchronized with the other calendar application.

 

You must have set Windows Live account as default account in Outlook 2010 and the same Windows live account must be used for calendar application in Windows 8 hence the main/default calendar in Outlook (i.e. Windows Live account calendar) is synchronizing with Windows 8 Calendar.

 

If you want to synchronize other calendars from Outlook 2010 to synchronize with Windows 8 then you have to import the data from the other calendars in Outlook to the Windows Live calendar (default) so that the data gets synchronized.

  3. Repeat step 2 several times to load addition tabs with different pages.

  4. Launch Task Manager and view Processes or Details

  5. Observed and make note of the instances of the IE processes that are currently running.
