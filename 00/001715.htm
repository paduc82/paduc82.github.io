Sub SavePagesAsDoc()

    Dim orig As Document
    Dim page As Document
    Dim numPages As Integer
    Dim idx As Integer
    Dim fn As String
   
      
    ' Keep a reference to the current document.
    Set orig = ActiveDocument
   
    ' Calculate the number of pages
    numPages = ActiveDocument.Range.Information(wdActiveEndPageNumber)
   
    For idx = 1 To numPages
        ' Make sure the document is active
        orig.Activate
               
        ' Go to the page with index idx
        Selection.GoTo What:=wdGoToPage, Name:=idx
   
        ' Select the current page
        Selection.GoTo What:=wdGoToBookmark, Name:="\page"
               
        ' Copy the selection
        Selection.Copy
       
        ' Create a new document
        Set page = Documents.Add
       
        ' Activate it
        page.Activate
               
        ' Paste the selection
        Selection.Paste
       
        ' Generate the file name
        fn = "Page" + CStr(idx) + ".doc"
       
        ' Save the document as Word 97-2003
        page.SaveAs FileName:=fn, FileFormat:=wdFormatDocument, AddToRecentFiles:=False
       
        ' Close the document
        page.Close
       
    Next

End Sub

charles kenyon say:
Hello Bruce,

This can be frustrating, I know from personal experience!

How to Open and Edit PDF Document in Word 2016
http://www.wikigain.com/open-and-edit-pdf-docum...

A pdf can be created from a Word document. Those can easily be opened and edited by Word.

It can be created from a web page. Those can easily be opened and edited by Word.

It can be opened from a scan or a picture taken by a phone camera. Those are pictures of words saved as pdfs. Just as you can have a picture of a car. You can see the car in the picture, but you can't change the timing of the engine in that picture. You can't change the order of text or otherwise edit it with a picture of text. Word can open such a file, but it can't edit it.

There is an intermediate kind of program or process called OCR. That is, Optical Character Recognition. If run on a picture of text, it gives text, more or less accurately (but seldom perfectly). Most scanners come with such software. It is built into Adobe Acrobat (but not Acrobat Reader). It is also in OneNote.

Once translated into text, it can be edited in Word.

Windows Phone 8 - contacts on phone, imported from SIM card, want to - Microsoft Community

maparik75 say:

Windows Phone 8 - contacts on phone, imported from SIM card, want to FORCE a sync.

I have contacts that were imported from SIM card about 200+.

How do I FORCE this to be copied to my connected primary hotmail account. 

I had earlier saved the sim contacts as vcf and csv and both did not import at all in outlook.

I need to take backup of my contacts and reset my phone as intermittently restarts. I tried various contacts backup software and none backup the notes field.
julie fka profjulie say:
Any contacts you entered into the phone BEFORE you set the phone up with a Microsoft account are not associated with any online account that they can be synced to.  Setting up a Microsoft account after the fact, will not add any of these contacts to be synced to the newly added Microsoft account.....they are orphans. (however, any new contacts you add to the phone after the Microsoft account is set up on it will be automatically assigned to the Microsoft account and synced to that account).

There is no way to automatically assign these orphan contacts to any online account that they can be synced to.

You can send each contact that is currently stored on the phone to your computer as vCards via Bluetooth connection (assuming the computer is Bluetooth enabled) by:
  • Turn on the Bluetooth radio on the phone and your computer
  • Then open up the phone's people hub, and selecti a contact
  • Tap the three dots in the lower right hand corner of the screen, select "Share contact" > Bluetooth (this option will not appear unless Bluetooth is turned on)
But having to do this for 150 contacts will be pretty tedious. 

There are apps you can get from the marketplace that allow you to save your contacts to a CSV file (I have not used any of these apps, so I can't suggest any one app) or vCards that can then be uploaded to your SkyDrive account.  Once the file(s) are uploaded to SkyDrive, you can download the file(s) and:

  • if you are working with a CSV file, you can import that file into your the People Hub of your Microsoft account (once the contacts are imported in the People Hub of your Micropsoft account they will sync to your phone); or
  • if you are working with vCards, you can send each card to the phone (either via email or by bluetooth), open them up (individually) and save them to the phone's People Hub....at which time they will be saved and associated with your Microsoft account.
