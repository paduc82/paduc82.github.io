    I get offers like that ... only they're for swamp land in Florida and the Brooklyn bridge.

    This is a come-on that you should just ignore. Allow The Magnificent BulldogXX to set you straight:

    The only way to know for sure whether a driver is out of date is to compare the version of the driver on your computer with the latest version of the driver on the hardware manufacturer's website. Don't trust a program to do that for you.

    The mere fact that a newer driver is available does not necessarily mean that you should install it. Drivers are modified to fix problems. If your hardware works fine, there's no need to install an updated driver no matter how new it is. If you really want to impress your geeky friends: Each driver update comes with an explanation that describes specifically what problems are being fixed. If you don't have those problems, you don't need the new driver. That simple.

    Microsoft delivers drivers through Windows Update but it generally does not write the drivers. The hardware manufacturer has the knowledge (and legal authority) to write the driver, which it then passes on to Microsoft. These drivers have acquired a bad reputation because they're not always the same driver you would get if you went to the manufacturer's website. That's why your geeky friends avoid drivers delivered through Windows Update.

    Finally: Drivers are always free, so don't let anyone try to charge you.
