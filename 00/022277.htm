resourceful say: Hello. There is limited language support at Start>settings>system>regional settings. You can set that to English but the overall OS may still be in Spanish. The best way would be to flash an English ROM. You can change the user interface to English by using SPB Mobile Shell but the apps would still be in Spanish.
