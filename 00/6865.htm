How to convert .ost file to .pst file?

karl timmermans [outlook mvp 2012-2018] say:

I want to open .ost backup file in outlook 2013, but I can not.

For sake of clarity - there is no such thing as a backup ost file. These files can only be opened in Outlook by the profile/email account that created it. OST's represent the contents of the server the last time Outlook connected to the account. Exception to this are the folders designated as "This computer only" - those folders must be copied or exported to a separate PST file for future access.

Getting back to your original question - your only option is going to find a 3rd party tool that can convert a stand-alone OST file to a PST file. Do a google/boing search for available options.

3. On the column on the left side of the screen, put the date when the event will start on When box including the time on the Start box.
