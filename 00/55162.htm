bossdweeb say:

"In my User folder, there is a folder called "Pat" which I cannot change the name of, nor the location...By the way, I have already removed "Pat" from the control panel, and have replaced her with my information."

I wonder how another User folder got into your User folder ?

So you have deleted the Pat user from User Accounts, and your games still have their data in your User\AppData folder ?

"A lot of my games run off of appdata,"

Hopefully you are not saying you have installed games in this location.

Games don't 'run off' the AppData folder. Some games store data such as configuration files or saves in AppData,

and this can add up to several GB's.

Hopefully you are not installing your games to the default location [ Program Files (X86) ]  on the SSD at all.

There is no benefit running games from an SSD other than a couple of seconds faster loading time, and games take

up a lot of space.

I also install my Steam Client on another HDD because I've tried not using the default Steam Apps location a while

back and several games that I had Steam install to that location didn't run properly. Now I keep the Steam Client

and all Steam Games together on the same HDD/Partition.

- Windows does not allow you to change the location of AppData folder for good reason. Doing so would cause problems 

for any software/game that uses the AppData folder as they are programmed to look in this location for their files.

Renaming is not an option for the same reason as the name of the AppData and the sub-folder within are part of the file path.

Change the file path from the one used at the time a software was installed and software will not be able to find the files

they are looking for.

While Windows does make it possible to change the location of some of your User folders I would limit doing this

to the Music, Photo and Video User folders.

( I don't eve bother with that. Leave the folder created by Windows in the original location and make a new folder on the HDD.

Then move the music, etc to the new folders and create a shortcut to those folders (or keyboard hotkey, etc), and

pin the shortcut to the File Explorer Taskbar icon or the Start Menu. )

If changing location of data you have created, such as music. etc does not leave enough space your SSD is too small.

Either get a larger SSD or go on in an endless battle moving files to the HDD.

I was able to self-resolve by doing the following:

  1. Launch admin command prompt.
  2. Run (with quotes): "%PROGRAMFILES%\Windows Defender\MPCMDRUN.exe" -RemoveDefinitions -All
  3. Wait for finished message.
  4. Run (with quotes): "%PROGRAMFILES%\Windows Defender\MPCMDRUN.exe" -SignatureUpdate

Once the final step completed, I am now able to check for updates via command center and within Windows Defender without error.

SOLUTION for everyone missing the Stereo Mix feature.
I've have been following this topic for 2 years and finally found something that works. Go to CNET and download FREECORDER.
