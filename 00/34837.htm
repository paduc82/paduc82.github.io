If your machine is a desktop vs laptop and uses Realtek HD audio, and you have installed the

current Win10 Realtek drivers, the setup for headphones with speakers working independently is as follows -

(If it's a laptop you may want to visit the mfg's forums.)

Depending on which version AC97, and the software you are using ...

With the headphones plugged into the front panel jack -

- Open the Realtek HD Audio Manager (Realtek HD  should be in Control Panel, but you
can also add a short-cut to the Task Bar Notification area).

- Click on the yellow folder icon in the upper right of the window and make sure the tick box
next to  Disable front panel jack detection  is clear.

- In the Device advanced settings option if you select
- Make front and rear output... simultaneously , the speakers will be disabled when the headphones are plugged in,

and the speakers will function when the headphones are unplugged.
- This option also allows you to set-up the configuration (Sound Effects, etc.).  for the headphones
to be different from the speakers.
(The option  Mute the rear...plugged in  does not offer an option to set-up a different configuration
for the headphones from the speakers, but works the same otherwise).

- If you tick the box next to  Disable front panel jack detection and select
Make front and rear output... simultaneously the speakers will not work. Headphones only.
- If you select - Mute the rear...plugged in  both speakers and headphones will operate simultaneously,
and when the headphones are unplugged the speakers will still function.
- This option combines both speakers and headphones, so there is no separate option for EQ on the headphones.

- Other brands of PC audio may have different terms and setup options, but you should be able to get the
same results from experimentation.
