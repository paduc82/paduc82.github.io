1. Install the Microsoft Online Service Sign-in Assistant for IT Professionals RTW.
2. Install the appropriate version of the Azure AD Module for Windows PowerShell:
• Azure Active Directory Module for Windows PowerShell (32-bit version)
• Azure Active Directory Module for Windows PowerShell (64-bit version)
3. Run the command “Connect-MsolService” , then enter you admin credentials.
