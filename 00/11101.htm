Method 1:
Let us create a new Calendar folder in the default Outlook data file.

 

1. Open Outlook 2010 -> go to folder list (Ctrl + 6).
2. Right click on the Calendars -> Click on New Calendar -> Click OK.

Now create a conditional formatting and check if it helps.

Method 2:
If the above steps do not help, the let us create a new Outlook data file and check.

 

1. Open Outlook 2010.
2. Click on File -> Account Settings.
3. Under the Data Files tab -> Click on Add, give a name to the data file -> Click OK.
4. Select the new data file under the Data Files tab -> Set as Default.
5. Re-open Outlook and check if the conditional formatting filters are working.

 

Note: If it does not work, go back to Data Files tab and select the data file which had all the information and set as default.

Method 3:

http://support.microsoft.com/kb/829918

 

I hope the conditional formatting retains in the new profile. If not, then provide us additional information so that we can check and provide you further steps to fix the cause.
