Managing file size when embedding custom fonts in PowerPoint - Microsoft Community

nicholas_w_ say:

Managing file size when embedding custom fonts in PowerPoint

When you embed a custom font in a PowerPoint template, the file size increases by approximately the size of the fonts used in the document. To facilitate editing, PowerPoint also embeds Regular, Bold, Italic, Bold Italic weights of the fonts used in the document. If you use bullets, then fonts such as Calibri may be included. 

To reduce the file size, only embed the font you want. Follow these steps to make sure standard fonts like Calibri are not embedded in your document.

  1. Select View from the ribbon, and then select Slide Master.
  2. In Slide Master ribbon tab, select fonts, then select Customize fonts.
  3. Select the Heading font and Body font for the font you want (if you need complex script support then those can be specified as well).
  4. Select Save.
  5. Select Close Master View.
  6. Select File, then Options, then select Save.
  7. In the right side of PowerPoint Options dialog, Select the option Embed fonts in the file and next choose Embed all characters.
  8. Select Ok.

PowerPoint will embed fonts on the next save.

If youâ€™ve already embedded multiple fonts, you can remove them by following these steps:

  1. Select File, then Options, and then Save.
  2. In the right side of PowerPoint Options dialog, unselect the option Embed fonts in the file and next choose Embed all characters
  3. Select Ok

PowerPoint will remove all embedded fonts on the next save.

Note: Microsoft Office checks the embedding rights of each font before embedding. Fonts which do not have embedding privileges are not embedded. To understand embedding privileges, check out the OpenTypespec
