stefan blom say:

One option could be running the following macro which actually inserts comment text in line with text. Ensure that you do not save the document after you have run the macro, though.

Sub InsertCommentTextInline()
Dim c As Comment
For Each c In ActiveDocument.Comments
Set myrange = c.Reference.Duplicate
With myrange
.InsertAfter "["
.InsertAfter c.Range.Text
.InsertAfter "]"
End With
With myrange.Font
.Bold = True   'Make comment text stand out
End With
Next c
End Sub

For installation instructions, see http://www.gmayor.com/installing_macro.htm.

Hi

The result of restarting in Safe Mode suggests there is a problem with the installation of Excel itself, or there is a font issue that needs to be resolved on your Mac.

When booting to safe mode, several things happen. Mac OS Disk Utility runs in the background and fixes file system errors such as directory and volume problems. The font cache is emptied. Apps that normally would run at startup are blocked from opening. A minimal set of fonts is loaded.

Excel should run normally, but if your workbook uses a font that is not part of the base Mac OS and Office install, then characters using the special font will not display. It sounds like this is the case, so I would not be alarmed about that. You could probably choose a different font and that would display what does not show up.

However, other aspects of Excel should work normally. Lines and boxes should work normally. Excel should be faster, not slower, because not as many fonts are being used by your system.

Let's rule out problems with your installation of Office. Force an update to the upcoming general release build of 16.17. To do this, go to the Help menu in Excel and choose Check for Updates, which launches AutoUpdate app. In AutoUpdate, click radio button to have updates download and install automatically, click the check box to join the Insider program, and then set the pop-up to Insider Slow. Click the button to check for updates. This is a way to put yourself first in line for general release updates. Installing 16.17 will replace your current install of Office and gives you the latest bug fixes without you having to uninstall or reinstall Office.

Try using 16.17 and see if things are better. Is speed back to normal?

If not, I think you may have to reinstall your system fonts. You can do this using Font Book app, which is in your Applications > Utilities folder.
