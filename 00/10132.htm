This is not the best idea. As a general rule, we don't store calculated values and that's what you are doing.

you should have a query that sums the quantity shipped by order and items. something like this:

SELECT OrderNo, ItemID, SUM(QtyShipped) As Shipped

FROM Shipments

GROUP BY OrderNo, ItemID

ORDER BY OrderNo,ItemID

.................surely it is better to have it stored rather than having to calculate it each and every time?

No.  Computing the value whenever needed guarantees that it is correct.  Storing it leaves the database open to the possibility of update anomalies, thus putting the integrity of the data at risk.

WHERE ((([Forms]![Invoice]![subshipments].[Form]![order_number_box_shipments])=[Sales_orders_add_cookers]![Order_number]) AND (([Forms]![Invoice]![subshipments].[Form]![Item_box_shipments])=[Sales_orders_add_cookers]![Cooker_ID]));

hansv mvp say:

Try this:

 

Function RandomSampleAry(ByVal InArray, ByVal N As Long)
    Dim i As Long
    Dim j As Long
    Dim m As Long
    Dim sngTmp As Single
    Dim varTmp
    m = UBound(InArray)
    If N > m Then
        N = m
    End If
    ReDim outarray(1 To N)
    ReDim ix(1 To m) As Single
    Randomize
    ' Fill array with random numbers
    For i = 1 To m
        ix(i) = Rnd
    Next i
    ' Bubble sort - replace with more efficient sort if needed
    For i = 1 To m - 1
        For j = i + 1 To m
            If ix(i) > ix(j) Then
                sngTmp = ix(i)
                ix(i) = ix(j)
                ix(j) = sngTmp
                varTmp = InArray(i)
                InArray(i) = InArray(j)
                InArray(j) = varTmp
            End If
        Next j
    Next i
    For i = 1 To N
        outarray(i) = InArray(i)
    Next i
    RandomSampleAry = outarray
End Function

rick rothstein, mvp - excel say:

I'm not 100% sure of your setup... you say InArray will be an array Dim'med 1 to something seeming to mean it is created in VB code, but you are asking for a worksheet formula to pass it into the UDF? Anyway, maybe you can make use of the function which I know works in code when you pass it a single-dimensioned array (RandomSampleArray will be an array with the same lower bound as the array passed into it and will contain N elements)...

Function RandomSampleArray(InArray As Variant, N As Long) As Variant
  Dim Cnt As Long, RandomIndex As Long, Temp As Variant, TempArray As Variant
  Static IsRandomized As Variant
  If IsEmpty(IsRandomized) Then
    Randomize
    IsRandomized = True
  End If
  TempArray = InArray
  For Cnt = UBound(TempArray) To LBound(TempArray) Step -1
    RandomIndex = Int((Cnt - LBound(TempArray) + 1) * Rnd + LBound(TempArray))
    Temp = TempArray(RandomIndex)
    TempArray(RandomIndex) = TempArray(Cnt)
    TempArray(Cnt) = Temp
  Next
  ReDim Preserve TempArray(1 To N)
  RandomSampleArray = TempArray
End Function

Note that I made the function take care of issuing the Randomize statement call once per session (Randomize should only be issued one time per session so as not to compromise the size of the random numbers Rnd will make use of).
