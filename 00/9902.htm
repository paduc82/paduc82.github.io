I don't know of a way to do that Elario.  I was under the impression that it is all hard coded and cannot be changed. If you try and use any other port number you will just get an error.

The only experience I have had for changing the COM port is via bluetooth syncing.  Although it is not what you are looking for, I thought I would reference it here for others that might need it.
That process of seting up was like this:
(via Juggs Ravalia [MSFT])

Step 1 Set up a Bluetooth COM port
1. Open the Bluetooth Devices Control Panel.
2. On the Options tab, select Turn discovery on and Allow Bluetooth devices to connect to this computer.
3. On the COM Ports tab, click Add.
4. Select Incoming (device initiates the connection) and click OK.
5. Note the number of the COM port that was added. The COM port number must be no higher than 20.
6. Click OK to close the Bluetooth Devices Control Panel.

Step 2 Set up ActiveSync to use the Bluetooth COM port
1. In ActiveSync on the PC, on the File menu, click Connection Settings.
2. Select Allow connections to one of the following.
3. Choose the COM Port with the number you noted in Step 1 Set up a Bluetooth COM port.
4. Click OK.

Step 3 Start Bluetooth sync from your device 1. In ActiveSync on the device, tap Menu, then tap Connect via Bluetooth.

Hi-light the cells that you wish to convert and run this tiny macro:

 

Sub ConVertRM()
Dim r As Range
Dim v As String
For Each r In Selection
    v = r.Value
    If InStr(v, "RM") > 0 Then
        r.Value = Replace(v, "RM", "")
    End If
Next
End Sub

 

so data like:

RM1234.56
-RM678.09
RM-12.34

will become:

 

1234.56
-678.09

-12.34

 


Macros are very easy to install and use:

1. ALT-F11  brings up the VBE window

    ALT-M opens a fresh module

To remove the macro:

   1. bring up the VBE window as above
   2. clear the code out
   3. close the VBE window

To use the macro from Excel:

   1. ALT-F8
   2. Select the macro
   3. Touch RUN
