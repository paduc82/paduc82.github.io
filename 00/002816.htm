Hi,

use this array formula: =AVERAGE(IF($B$2:$B$6<>0,$B$2:$B$6)) and =AVERAGE(IF($C$2:$C$6<>0,$C$2:$C$6))

This is an array formula which must be entered by pressing CTRL+Shift+Enter and not just Enter. If you do it correctly then Excel will put curly brackets around the formula {}. You can't type these yourself. If you edit the formula you must enter it again with CTRL+Shift+Enter. (Mike H)

The AVERAGE() function calculates only numbers, no characters like "-".

There seems to be a contradiction in your facts.  We need to be very careful about how we use terminology.

The AVERAGE function ignores cells that have non-numeric data such as text.  So if the "-" is text, AVERAGE would ignore it no matter how that text gets there (e.g. "referencing another formula").

Since you say that the "-" is treated as zero by AVERAGE, pulling down the result, it seems that the cell value is actually zero, and you used formatting  to have the zero appear as "-", for example the Account format or a Custom format like 0.0;-0.0;\- .

But if that is the case, there is no way to distinguish zero as a placeholder ("no data") from zero as actual data (no catch).

I suggest that you use  text  "-" to mean "no data".  In that case, you do not need to do anything special with AVERAGE.  Simply write =AVERAGE(B2:B6) and =AVERAGE(C2:C6).

For example, the formula in B2 would be:

=IF('Boat 1'!AJ17="","-",'Boat 1'!AJ17)

If you want the text "-" indented, you can use the Custom format 0.0;-0.0;0.0;@_._2 with Right horizontal alignment.  That displays numbers with 1 decimal place, and it indents text ("-") from the right by the width of ".2".

guessed say:

The 'proper' way to do this is kind of labour intensive and involves editing field codes. If you are up to the challenge your fields should look like this

Caption 1a -> {SEQ Figure \* ARABIC }{SEQ FigX \* alphabetic }

Caption 1b -> {SEQ Figure \* ARABIC \c }{SEQ FigX \* alphabetic }

Caption 1c -> {SEQ Figure \* ARABIC \c }{SEQ FigX \* alphabetic }

Caption 2 -> {SEQ Figure \* ARABIC }

The \c switch tells the sequence to repeat the last caption number.

stefan blom say:
Note that once you have started editing the SEQ fields manually, you can't use the Caption dialog box again (in that same document), or the numbering will break, because Word tries to recreate the original numbering sequence.
