First, I don't think you can do this with macros. Macros are very limiting. You will need to use VBA (not SQL) code to populate the arguments of the SEndObject method.

This is really not that difficult. The SendObject method uses the same arguments that you see using a macro. But the advantgae of VBA is making it easier to populate the values. What you can do is create variables for each of the arguments like:

Dim strTO As String,  strSubject As String, strBody As String

Then you would use an assignment like

strSubject = "Today is " & Format(Me.datecontrol,"mm/dd/yy")

strBody = Me.Jobcontrol & "Completed at " & Me.completioncontrol & " which was " & Me.differencecontrol & " from yesterday's run."

Then you add

DoCmd.SendObject arguments

substitute the variables for the arguments.

john sr wilson say:
The easiest way is to add a slide (either blank or a duplicate with no animations) just before the slide in question.

Give this slide a zero second auto transition and have a button on the animated slide that links to it.

If you do need a macro

Sub rptSlide()
With SlideShowWindows(1).View
.GotoSlide (.CurrentShowPosition), True
End With
End Sub
