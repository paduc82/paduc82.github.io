lemp say:

wuauclt.exe was "regenerated" because it is one of the files that is monitored by Windows File Protection.  It actually was replaced as soon as you renamed the original file, not when you re-booted.

If you disable Windows automatic updates, then wuauclt.exe shouldn't run. 

In order for the file to be malware masquerading as a legitimate Windows file it would have had to have been replaced both in Windows\system32 but also in Windows\system32\dllcache.  That's possible, but unlikely.  However, you can upload the file (or any file) to VirusTotal and it will be checked against a wide variety of a/v programs.  https://www.virustotal.com/

On second thought, perhaps a malware issue is not as unlikely as I first thought, given that you are using an extremely outdated and vulnerable version of Windows.  Windows XP service pack 2 went end-of-life in July 2010.

Error code 80070571 occurs when your Xbox LIVE profile has been corrupted.
Overview

If you see error code 80070571 when you’re using Xbox LIVE, it means your profile has been corrupted. To fix this error, you need to delete the corrupted profile that’s stored on your Xbox console and then download your profile again.

Read this page to learn more about 80070571 and what you should do about it.

On this page
 
Error code and message

Error code 80070571 is preceded by the following message:

Sorry, Xbox LIVE profiles can't be downloaded right now. Try again later.

Status code 8007051

Problem

If you’re seeing this message and error code, it means your profile has been corrupted.

Solutions

Solution 1: Delete the corrupted profile that’s stored on your Xbox console and then download your profile again

Delete the corrupted profile that’s stored on your Xbox console

  1. Go to Settings, and then select System.
  2. Select Storage.
  3. Select All Devices.
  4. Select Gamer Profiles.
  5. Select the gamertag that you want to delete.
  6. Select Delete.
  7. Select Delete Profile Only. This deletes the profile but leaves saved games and achievements.

Download your Xbox LIVE profile

  1. Press the Guide button on your controller.
    1. Select Download Profile.    

      Note If you don’t see Download Profile, it probably means you’re signed in. Press X to sign out of your profile.

    2. Enter the Windows Live ID that’s associated with your Xbox LIVE gamertag. If you don’t know your Windows Live ID or password, see Manage your Windows Live ID

Note If your account is an Xbox LIVE child account, you have to use the parent’s Windows Live ID to download the profile.

  1. Select the storage device for your profile (for example, Hard Drive).

    Your profile will begin downloading. Once your profile has downloaded, you can choose whether to require a Windows Live ID password to sign in to your profile.

Remember Password option

The Remember Password option is off by default, which means you’ll be prompted for your Windows Live ID password when you sign in to your Xbox LIVE profile.

  1. Leave Remember Password unchecked if you want to be prompted for your Windows Live ID password when you sign in to your profile on this console.

You should now be able sign in to this console using your Xbox LIVE gamertag.

One step you probably should have taken is to do a System Restore; however since you've reinstalled Windows, the backup directories are most likely history.

Safe Mode is just Windows running with the bare essentials of programs and services.  Usually when Safe Mode works fine but Normal Mode doesn't, it's because a Program or Service that starts up in Normal Mode is interfering somehow.  The way to attack this is to use Microsoft's "msconfig" utility to selectively enable/disable various optional programs and services until you find the one that is giving you grief.  This can be a long trial-and-error method.  Also, it is unlikely you have a hardware problem if you can access the drives in safe mode.

"How to troubleshoot configuration errors by using the System Configuration utility in Windows XP"
  < http://support.microsoft.com/kb/310560 >

In any event, my first step if I were you would be to back up the critical data on your other internal drives to an external USB drive.  If you can't do this under Windows, then create a free Live Linux CD (such as Knoppix ), boot from this CD, and then back up the data on your disks to a USB drive or Network drive.  Creating a Live Linux disk is as simple as downloading a ".iso" file and use your CD-burning program's "Burn Image" feature to create the bootable CD.

HTH,

Reinstall Windows Preinstalled On Computer

OEM computer manufacturers are required to provide at least one way for you to reinstall Windows if you buy a computer with Windows preinstalled - your computer should have included either a restore partition on your hard drive, or Windows install/restore disks. ﻿Most computer manufacturers also have a feature to make your own install disks.

If the restore partition is still there and was not removed, it will reinstall Windows. There is usually a prompt when you first start the computer telling you what key to press to start the restore﻿ - check your manual if you're not sure how to. Or if you have install disks - insert them, restart and boot from them to start the reinstall process.

If you restore it to factory condition, it will be like it was when you first bought it - all preinstalled programs will be installed. Anything you installed after buying it must be reinstalled. This will remove any personal files on the computer, back them up first.


If you don't have the restore partition or install disks:

Contact the OEM computer manufacturer and ask them to send you an install disk for your computer - they may or may not do it. If they agree to send you one they may charge a small fee. Contact information for the manufacturer of your computer can be found here:
