marshall barton say:

That's the kind of thing you get when you use code to calculate the running sum.  You can not use code to calculate anything that involves more than values in the current detail section becaue report sections are processed in whatever order and as many times as required to deal with things like KeepTogether, jumping around in PrintPreview, etc.

Instead, you need to use a text box with the value you want to sum and set its RunningSum property to Over All or Over Group as needed.
