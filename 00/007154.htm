Excel 2007 Paste While Filtering Pastes in to Unfiltered Cells - Microsoft Community

mdsleeds say:

Excel 2007 Paste While Filtering Pastes in to Unfiltered Cells

I have a large excel 2007 file with circa 15,000 rows and 30 columns. I have filtering on and I am trying to copy only visible data from a particular column in to a new column at the end of the current columns, however, when I paste, excel is pasting in to the unfiltered cells and I only want it to paste in to the visible cells. This issue only seems to occur with large data files; is it a known bug in excel when working with large data files?

 

I did search Google for help and located a support link that seemed to fit the bill perfectly for me, but it still pasted in to the unfiltered cells. This workaround was to copy the column I was looking to paste, then select Find & Select, then select Go To Special and select Visible cells only.

 

Does anyone know how to paste in only the visible cells when using a large excel file (it does not have any macros in it)???

ossiemac say:

You cannot copy the data from the visible range to another range of only visible cells. What occurs is that the visible data that is copied is compacted into one contiguous range and at the destination that is how it is pasted from the first visible cell and the data will go into a contiguous range equivalent to the size of the compacted data and will be in visible and non visible cells as a contiguous range.

However, there is a workaround because you can copy the contents of a single cell to an entire visible range only so I will attempt to explain it.

  1. Insert a helper column that can be deleted later.
  2. Without any filters set, copy the entire column of source data to the destination.
  3. Next set the filters as required. (So only the data you finally require is visible).
  4. Type an X (or any character) into a single blank cell.
  5. Select the single cell with the X and then Copy.
  6. Select the range of the Helper column then select Find & select -> Go To Special -> Visible cells only and then right click over the selection and select Paste. (This places an X in all of the visible cells only of the column).
  7. Turn off the filters to show all of the data.
  8. Now on the Helper column select the Filter DropDown and then select Text filters In the top section of the dropdown) then select "Does not equal" and type an X in the field beside it and click OK.
  9. Now it should display all of the data you do not want in the pasted destination column.
  10. Select the data in the pasted destination column and then Find & select -> Go To Special -> Visible cells only and OK.
  11. Now press Delete key and the unwanted data should be gone.
  12. Turn off the filtering and delete the helper column.
