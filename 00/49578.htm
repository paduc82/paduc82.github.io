Has your HD Texture Pack installed correctly. - Microsoft Community

waykam say:

Has your HD Texture Pack installed correctly.

If the HD Texture Pack stopped during installation it might be worthwhile re-installing it as I believe the data may be corrupted by this error. I've done this and was suprised by the difference and I got a friend who had had the same error do the same and he noticed the difference as well, another friend whose pack installed without stopping did the same and didn't see any change. I think if the game partially installs the Pack and stops, then starts again the data is corrupted.

-

You re-install by going into the Xbox menu, go to the system blade (far right), select memory and scroll to Battlefield 3, open the folder and delete the file called HD Textures.

-

To reinstall correctly insert the BF3 disc and start the game, wait till the loading screen with "Press Start" on BUT DO NOT DO ANYTHING. Wait until you hear the disc stop spinning (takes a couple minutes), once it stops spinning press start and then install the Texture Pack.

-

If you do try this and see a difference can you post here and let others know.

how can I find out how many network filter drivers I have using - Microsoft Community

sylviahaw say:

how can I find out how many network filter drivers I have using Windows 7 and uninstall if too many

I have been told to locate the following registry subkey HKEY_LOCAL_MACHINE/SYSTEM/CurrentControlSet/Control/Network but cannot find it.  I have clicked Start,click Run, type regedit, OK but still cannot find it
sralpert say:

Apparently, there is a key:

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Network\MaxNumFilters

that shows the maximum count. Further, if you look at:

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Network\{4d36e974-e325-11ce-bfc1-08002be10318}\

Each entry corresponds to a possible filter.

 

In my case, I had 13 filters but the MaxNumFilters was only set at 8! Some installs (like Virtual PC) will not complete if you have 13 or more.

rpggamergal say:

Hi Han_BFY,

IPC is the abbreviation of  "Inter-Process Communication"

It's used to start a connection session, password/user


"A capability supported by some operating systems that allows one process to communicate with another process. The processes can be running on the same computer or on different computers connected through a network.
IPC enables one application to control another application, and for several applications to share the same data without interfering with one another."
