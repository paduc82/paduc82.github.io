The following will work if cells E18 and F18 are entered manually; it won't work if they contain formulas.

Right-click the sheet tab.

Select 'View Code' from the context menu.

Copy the following code into the worksheet module:

Private Sub Worksheet_Change(ByVal Target As Range)
    If Not Intersect(Range("E18"), Target) Is Nothing Then
        Range("N1:O1").EntireColumn.Hidden = (Range("E18").Value = "")
    End If
    If Not Intersect(Range("F18"), Target) Is Nothing Then
        Range("P1:Q1").EntireColumn.Hidden = (Range("F18").Value = "")
    End If
End Sub
