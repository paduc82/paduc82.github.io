suzanne s. barnhill say:
In addition to enabled Different Odd and Even, you must enable Different First Page. This will give you a separate First Page Header and First Page Footer. Leave the header blank and put the page number in the footer (I usually center it).

suzanne s. barnhill say:

Charles has answered your question about starting on an odd page. You should in any case not be using Continuous section breaks when you want to start on a new page: always use Next Page, Odd Page, or Even Page. As for the headers, each one is linked to (or unlinked from) the corresponding header in the previous section. That means that the First Page Header in Section 2 is linked to the First Page Header in Section 1 and the same for Even and Odd headers; same for footers. Your Even Page Headers will be the same in any case because they have the title of your thesis. And you won't have to unlink the Odd Page Headers if you use a StyleRef field to pick up the chapter title instead of typing it in.

From your screen shot, I see that you have an extra, empty paragraph in each of your footers. If you want the page number where it is, it would be better to increase the footer margin, but do get rid of those empty paragraphs, as they're just excess baggage.

Build 10122 Realtek Audio Issue on Dell Inspiron 3531 - Microsoft Community

markhall_78 say:

Build 10122 Realtek Audio Issue on Dell Inspiron 3531

Here is a bug from the 10122 install and how to work around it.

  1. After installing build 10122 my audio was not working.  
  2. I downloaded and installed the Realtek audio driver (64x) from the Dell site and ran the install procedure.  This did not fix the problem.
  3. I had to pull up the Device Manager and noticed under the Sound, video and game controllers section there was a generic "High Definition Audio" driver in addition to the Realtek driver.
  4. I right clicked and Uninstalled the generic High Definition Audio driver and the audio started working again.

I hope this helps if you have the same issue.
