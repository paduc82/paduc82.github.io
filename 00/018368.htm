Changing font color using a macro - Microsoft Community

chrigrav say:

Changing font color using a macro

I am trying to add a Macro to my document to change font color. It is as simple as:  If the value of D3 is less than the value of F3 I want the font color changed to red. Help please?!
doug robbins - mvp office apps & services (word) say:

Assuming it is the color of the font used in cell D3 that you want changed, run a macro containing the following code when the selection is in the Table:

 

Dim D3 As Range, F3 As Range
With Selection.Tables(1)
    Set D3 = .Cell(3, 4).Range
    D3.End = D3.End - 1
    Set F3 = .Cell(3, 6).Range
    F3.End = F3.End - 1
    If Val(D3.Text) < Val(F3.Text) Then
        D3.Font.ColorIndex = wdRed
    Else
        D3.Font.ColorIndex = wdAuto
    End If
End With

john korchok say:

There are three possible approaches:

  • A full copy of Acrobat will open XPS files and save them as text. You can script Acrobat using JavaScript. Here's a page about scripting Acrobat: Scripting Acrobat
  • You can set up a text file printer (Text File Printer Setup), so all files sent to print are turned into text files. Then you can script XPSViewer using Windows Scripting Host or .NET VB to print all files in a folder to the text file printer. 
  • You can script an XPS library to print each file to a text file printer: SANAtech XPSViewer Control NiXPS View (uses JavaScript)

Your programming language preference may be the deciding factor. All of the above create text files that you would then run through your current macro.
