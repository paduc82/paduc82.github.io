ken sheridan say: To enable you to delete a questionnaire and all data associated with it you just need to edit each of the relevant relationships and enforce 'cascade deletes' in each.  This will delete the questionnaire and all answer and response data associated with it when a row is deleted from the Questionnaires table.  It will not delete the set of available questions in the Questions table or the answers associated with the questions; for that you'd need to delete the rows in the Questions table also.  Similarly the respondents and associated response sessions won't automatically be deleted; for that you'd need to delete the rows in the Respondents table also.

An improvement to the database would be to create an enforced relationship between Questionnaires and ResponseSessions.  This would allow all response data relating to a questionnaire to be automatically deleted if a questionnaire is deleted.

I will update the file in my SkyDrive folder to incorporate these changes.

So to wipe the database completely clean of all data you'd execute the following three queries:

DELETE * FROM Questionnaires;
DELETE * FROM Questions;
DELETE * FROM Respondents;

which you could easily do from a simple little function, procedure, or macro, though these sort of global set operations should be handled with extreme care, and it is absolutely imperative that the database be backed up before doing so.

transferring word to my Mac - Microsoft Community

madisonbroome say:

transferring word to my Mac

how do I transfer Microsoft word to my new Mac?
john korchok say:

You have to reinstall it.

If you purchased it originally, you'll need both the installation media or download, plus the 25-character Product Key to reinstall.

If you had 2011 through the Office 365 subscription plan, you can no longer reinstall if from your Office 365 account page. Instead, you'll have to install Office 2016 for Mac.

If you have custom toolbars, macros, autotext or styles stored in your Normal.dotm template, you'll want to save that file to copy to your new computer. If you have custom dictionaries or lots of autocorrect entries, you should also save those files as well. Here are their default locations on your old computer:

Normal.dotm - Users/ YourUserName/Library/Application Support/Microsoft/Office/User Templates/

AutoCorrect (ACL) and Custom Dictionary (DIC) files - Users/ YourUserName/Library/Preferences/Microsoft/Office 2011

The Library folder is normally hidden. Hold down the Alt key while clicking on the Go menu and choosing Library to make it visible.
