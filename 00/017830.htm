You can certainly stop the notification by turning off CFA. When you do, however,

W10 v1809 has simplified this process tremendously. When an App is blocked, you

Clicking on the ADD button presents an option for "Recently blocked Apps". Select

No reason to kill CFA anymore!  Unless you are Ransomware Proof.

Caveat: This procedure will clear your Update History.  Only the history. The Updates currently

In Command Prompt (admin), type the following, and press enter after each entry.

Stop Bits             <enter>

Stop Wuauserv   <enter>

Stop Appidsvc    <enter>

Stop cryptsvc      <enter>

Ren C:\windows\SoftwareDistribution SoftwareDistribution.old    <enter>

Start Wuauserv    <enter>

Start Appidsvc     <enter>

Restart your PC and Windows will rebuild these two folders. Eliminating any corruption that

they might contain.

10. The repair takes about an hour and a half. The PC will restart several times. Be patient and

      don't interrupt it.

      updates that were not in the ISO, and also Windows Defender definitions.
