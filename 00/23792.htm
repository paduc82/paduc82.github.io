Hi,

In the ThisWorkbook module of your PERSONAL.XLSB workbook, paste the following:

Private WithEvents app As Application

Private Sub app_WorkbookOpen(ByVal Wb As Workbook)
   Call DoStuff
End Sub

Private Sub Workbook_Open()
   Set app = Application
End Sub

There are two events here.  The first is the App_WorkbookOpen event.  This will fire when you open any other workbook.

The second event is the Workbook_Open event.  This runs when ThisWorkbook (I.e your Personal.xlsb) is opened, i.e. when you start Excel).  Once you have pasted the above code into the ThisWorkbook of Personal, place the cursor within the Workbook_Open code and hit F5 to run it.  Now app is the Excel application, which has been declared "WithEvents".

In a separate (normal) code module (Insert > Module), paste the following:

doug robbins - mvp office apps & services (word) say:

If the template is being used as intended as the basis for creating documents using File>New and selecting the template, you should just use a

{ CREATEDATE \@ "yyyy" }

field in the template, using CTRL+F9 to insert the field delimiters { }

When that is done, any document created from the template will display the year in which it was created.

Try the following

Range("A25").Select
Application.DisplayAlerts = False
ActiveCell.ClearNotes
ActiveCell.NoteText Text:="My comment"

If you wish to add the content of a cell (say  B25) as a comment in another cell (say A25) then

Range("A25").Select Application.DisplayAlerts = False ActiveCell.ClearNotes

MyComment = Range("B25").Value ActiveCell.NoteText Text:=MyComment
