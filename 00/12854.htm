On First Date, I have a plan of 725 while my actual value is 715 which is less than my plan value so i want that my rows from C15 should be highlighted red till C9. But C9 stands for 100 while my actual value is only 15 points greater than 700. So it should highlight some portion of C9 also.

myall_blues say:

To make the explanation slightly easier I'll call the values in row 3 (100,200 etc) the target values.

Data bars work by just shading a portion of the cell according to the value in it. In a series of cells, data bars by default take the maximum value and treat it as 100% (ie shade the whole cell) and values less than the maximum are treated as some value less than 100%. The formula I provided just provides values for the data bars to use by calculating how much of the cell needs to be shaded and setting the value between 0 and 1 (effectively 0% and 100%).

So the formula =IF(D$3<=$C4,1,MAX(0,(100-(D$3-$C4))/100)) works as follows:

If the target value is less than or equal to the actual value, set the cell value to 1. Otherwise take the cell width (100 units, because the target values are in increments of 100), subtract the difference between the actual value and the target value, then divide the whole lot by 100 to change it to a value between 0 and 1.

So in row 4, the actual value is 715 - we've gone over 700 by 15 so the next cell with target value 800 has 15/100 parts shaded, or 0.15.

For the next and subsequent target values (900, 1000 etc) the result of the formula is <0 (negative) so the formula just finds the maximum of the calculated value and zero to reset any negatives to zero.

No to the VBA bit - one of the limitations of data bars is they don't allow multiple colours. You can change the colour of data bars but not individually - it's all or nothing. But with a bit of VBA trickery you can beat that limitation - the VBA statements just set the conditions for which each bar colour applies. The first statement, selection.formatconditions(1).formula="=if($c4<$b4,true,false)", says the red bars apply when actual is less than plan. The second statement says the green bars apply when actual is greater than or equal to plan.

Hope that helps.
