tryx3 say:

Clive,

You stand a better chance of success if you use http://windows.microsoft.com/en-us/windows-10/media-creation-tool-install to create an installation disk [DVD/USB] - it will include the Creator's update.

  • Then start your computer normally [do not boot from the installation USB].
  • Connect the installation USB.
  • Open File explorer and browse to the USB.
  • Run its setup.exe file then just respond to its prompts [there are some at the start & some at the end].
  • You will be asked for your permission to search for updates during the Repair install but it will keep things simpler to say no [I would have included this comment even without your comments about Windows update problems, it just keeps things simpler during this evolution to let it go with what it's got]. 
  • It will work its way through overwriting Windows files so whatever is causing your update problem will probably be fixed as well.
  • You should backup your own files first but they ought not to be affected.  I have never had this procedure fail on me.
  • Your installed applications are not affected either.
  • You will need to run Windows update afterwards because there will be further updates that have been released since the Creator's update was released.

This procedure is called a Repair install or an InPlace upgrade.  It is a common way of updating to a new version and it is very reliable.  There are instructions in Repair Install Windows 10 with an In-place Upgrade - TenForums but you will probably not need them.

Later on, you can use the same installation USB as a repair tool as well.  It can fix [by overwriting] many Windows problems but not, in general, user account problems because it does not affect user accounts.
