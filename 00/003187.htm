macropod say:

Hi Andrew,

 

It would be most unusual for Word allow you to have multiple footnote references on a page and all of the footnotes on another page. Basically, footnotes give you two options: 'bottom of page' or 'below text'. The only real difference the latter makes is to move the footnotes up close to the text on a partly-filled page. Endnotes, have different options: 'end of document' and 'end of section', the latter means they'll be placed on the last page before a Section break. That seems consistent with what you're describing.

 

Select one of the notes, then right-click. The pop-up should contain mention of footnotes or endnotes, as applicable. Right-clicking on 'note options' allows you to swap the note type between footnote and endnote.

ted maz say: Update: The only workable, but very cumbersome, "solution" is both obvious and time-consuming. I only applied it to one of my Excel workbooks (with 10 or so worksheets). The procedure is simple: After opening the file in "normal" Excel, First, select the entire content of a given worksheet (click the upper left corner cell), then select font 10 and zoom 100%. Second, adjust the column widths and row heights (this can be done semi-automatic for all columns / rows at a time). Third, fine-adjust (where needed) larger fonts and/or column / row sizes, as well as certain cells' formats (e.g., merging cells, wrapping text, etc). These 3 steps (for each worksheet) in effect create new differently sized file that, when uploaded in Excel online, will be readable and editable by others.

Hi,

The row or column assumptions should be hard coded values.  Infact they should never refer to another cell - specially the one that you provide the reference of in the Data Table dialog box.  A Data Table works on the concept of looping i.e. the assumption cells values feed into the cell which you have referred in the Data Table dialog box.  So the two cannot be linked.
