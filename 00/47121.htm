Hi,

 

This error message usually happens if it’s actually been used by some other application which could be running at the background. The best way to check that is Task Manager.

 

Refer to the article shows the details on how to identify a program, if running without your concern:

http://windows.microsoft.com/en-in/windows-8/task-manager

 

If this is not the case, it could also occur if explorer.exe had the thumbs.db file open while accessing the folders and files in File Explorer. Well, thumbnail cache file used for Windows Explorer’s thumbnail view.

 

Follow the steps to check and make sure that explorer.exe is not using thumbs.db:

a) Hit Win+R, type gpedit.msc in the Run window and click OK. The Local Group Policy Editor will be launched.

b) Navigate through the tree to User Configuration > Administrative Templates > Windows Components > File Explorer.

c) In the right-hand pane, double-click “Turn off the caching of thumbnails in hidden thumbs.db files”.

d) Select Enabled. Click on Apply and then on OK.

e) Close the regedit window.

 

Check if that resolves the issue within File Explorer. If not, proceed further.

First, you may run the Microsoft Easy fix diagnostic tool for network to automatically detect and fix any incompatibilities with your device and driver. Here are some that may apply to you:

  • Find and fix problems with devices and hardware
  • Find and fix problems with connecting to the Internet or to websites
  • Find and fix problems with wireless and other network adapters

More easy fix solutions are available on this article: How to use Microsoft easy fix solutions.

  1. Go to Control Panel, on the Search Control Panel bar located on the upper-right corner of the Control Panel window, type Troubleshooting.
  2. Click Troubleshooting on the results pane.
  3. On the left pane, click View all.
  4. Look for Connecting to a Network, right-click, and have it Run as administrator.
  5. Follow the onscreen instructions.
  6. Once done, please check if issue still persists.
  7. Please repeat the steps, and then run troubleshooter for Network adapter.

Method 2: Update driver

It is suggested to update a device driver to resolve any incompatibility issues with the latest version of Windows 10.

  1. To Update a driver, please go to the Cortana search bar beside the Start button. Type devmgmt.msc. Hit Enter.
  2. Expand Network adapters. Look for your adapter.
  3. Right-click, then select Properties.
  4. Navigate to Driver tab. then click Update.
  5. If options is grayed out, click Update driver instead.
  6. Choose Search automatically for updated driver software.
  7. Follow the onscreen instructions.

For more troubleshooting steps about Network concerns, please refer to this article: Fix network connection issues in Windows 10.

Method 1: Reset EDGE

  1. Press the Windows key + X button.
  2.  Select Control Panel.
  3.  Go to Network and Internet.
  4.  Click on Internet Options.
  5.  Choose the Advanced tab.
  6.  Click on Reset button. (You will see another box. You can put check in "Delete Personal Settings")
  7.  Click on the Reset button.

Note: You may save your favorites and take note of your settings as this method will reset all the changes made in your browser.

First, restart your computer into Safe Mode.

  4. Close the window.
  5. Go to Start, then type in Windows PowerShell.
  6. On the results, right-click on Windows PowerShell, then run as administrator. If prompted. Please type your administrator credentials.
  7. Copy this command:

Get-AppXPackage -AllUsers -Name Microsoft.MicrosoftEdge | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register "$($_.InstallLocation)\AppXManifest.xml" -Verbose}
