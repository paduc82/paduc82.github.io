How to delete lines beginning with specified word? - Microsoft Community

er_975 say:

How to delete lines beginning with specified word?

I have a long text and I need to erase (or at least select) all the lines beginning with, e.g. "John says:".

(By line I mean an actual line, not a paragraph, sentence etc. Also the text will not be reformatted so the lines will not change.)

How should I proceed?

Thank you.

jay freedman say:

Turn on nonprinting marks by clicking the ¶ button on the Home tab, or by pressing the shortcut Ctrl+Shift+8.

What appears at the end of each line that you want to remove?

If it's a paragraph mark (¶), then:

  • Open the Replace dialog box and click the More button to expand it.
  • Check the box for "Use wildcards".
  • Put this text in the Find What box (modify the words at the beginning to the actual text you're looking for):  John says:[!^13]@^13
  • Leave the Replace With box empty, and click the Replace All button.

If it's a manual line break (), do the same as above, except use this in the Find What box:  John says:[!^11]@^11

If the line ends with a space character -- that is, the text is automatically line-wrapped -- the best you can do without a macro is to select the next occurrence of the text you're looking for, and manually press Shift+End to extend the selection to the end of the line. Then you can press Delete or Enter or Shift+Enter, depending on what result you want. For your long document, you need a macro (see http://www.gmayor.com/installing_macro.htm if needed) like this one (again, change the "John says" to the actual text to look for):

Sub DeleteJohnSays()
    If Selection.StoryType <> wdMainTextStory Then
        With ActiveDocument.ActiveWindow.View
            .Type = wdPrintView
            .SeekView = wdSeekMainDocument
        End With
    End If
    
    Selection.HomeKey Unit:=wdStory
    With Selection.Find
        .Text = "John says: "
        .Format = False
        .Forward = True
        .MatchWildcards = False
        .Wrap = wdFindStop
        While .Execute
            Selection.Expand Unit:=wdLine
            Selection.Delete ' Unit:=wdLine
        Wend
    End With
End Sub

er_975 say:

I'd like to know- what do [!^13]@^13 and [!^11]@^11 represent?

 Thanks for the rest.

doug robbins - mvp office apps & services (word) say:

See the article "Finding and replacing characters using wildcards” at:

http://www.word.mvps.org/FAQs/General/UsingWildcards.htm

jay freedman say:

To save you a bit of time wading through "all you ever want to know about wildcards", the short answer is:

^13 represents the paragraph mark, which is ASCII character 13. Similarly, ^11 represents the manual line break mark.

[!^13] matches any character that is not a paragraph mark.

[!^13]@ matches any sequence of characters that are not a paragraph mark.

So the expression [!^13]@^13 matches any sequence of characters that are not a paragraph mark, followed by a paragraph mark. The other expression has the same meaning, but for line breaks.

brian zelt say:

SUGGESTION: Keep With Previous

in MS WORD->Paragraph->Pagination->Line and Page breaks 

There is a "Keep with Next" which can be used to keep a style setting with the next object/paragraph.

For instance, a Table Caption is normally placed above a table, The "Keep with Next' is valuable to keep the Caption attached to the table below it.

For Figures, Captions are normally placed below the graphic, therefore the FIGURE CAPTION style requires a "Keep with Previous" to keep the graphic wit the Caption below it. 

Please consider adding "Keep with Previous" to the next update.

john korchok say:

This is an effect that can be caused by Autofit. To turn it off:

  1. With the presentation open in PowerPoint, choose VIEW>Slide Master.
  2. Select the larger slide at the top of the left-hand list. This is the Master.
  3. Right-click on the main text placeholder and choose Format Shape... The Format Shape pane opens at the side of your screen.
  4. In the Format Shape pane, click on TEXT OPTIONS, then click on the Textbox icon (a square with an A and 5 lines).
  5. Select Do not Autofit.

In a normal presentation, that setting on the Master should ripple through the Slide Layouts below it. But many hands have touched this deck, so check the text or content placeholders on the slide layouts as well and make sure the same Do not Autofit option is applied to them.

After making this change, some slides may have more text than will fit. A common rule of thumb is to limit each slide to 50 words.
