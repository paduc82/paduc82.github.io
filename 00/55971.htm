roger bye say:

Use a wide Explorer window. If you don't have too many columns chosen, the space to the right of the last column will serve as your empty space.  If you have something selected, you have to click once in the empty space to deselect. Then shift-rightclick in the same space and you'll get the context menu with "Open Command Window here".

 

As an alternative, create a batch file, say newcmd.bat, with one line like this:

@for %%f in (%1) do @cmd /k cd /d %%~dpf

 

Then right-click any file, choose "Open with", and get to your new .bat file.

 

If you only have folders in view (they won't give you "Open with"), shift-rightclick one of them and choose "Open Command Window here".  When the window comes up, enter "cd..".  (You don't need a space after "cd", but of course, it wouldn't hurt.)  Lame maybe, but pretty quick.

 

If the .bat file line doesn't make sense, read up on the FOR command.  You're missing out on some great automation enhancements.

a. user say:

You know, Microsoft is famous for giving multiple, confusing names to the same or similar products. But let's cut to the chase:

Microsoft Outlook is an application, where 'application' is the new word for 'program.' To make matters more confusing, Microsoft likes to call everything an 'app' just because it sounds modern and edgy. No matter what you call it, a program/application/app is something that you install on your computer.

Outlook.com is not a program/application/app because you don't install it on your computer. It's actually a web page that provides email service. It offers many of the same email features as Microsoft Outlook - but not all - without you needing to install anything on your computer. And because it's on the internet, you can send or get your email from any computer in the world (as long as it can get on the internet.)

Windows Mail is, technically, an app because it's installed on your computer. But it doesn't actually do anything by itself. Instead, it acts as a front end (there's another word to explain) to email services like Outlook.com or Gmail, which are web pages. A front end is something that passes commands to other apps or web pages, so you don't have to deal with the app or web page directly. In similar fashion, a server passes your food order to the kitchen so you don't have to deal directly with all the chefs.

Microsoft Outlook 2016 can work completely offline and creates distribution lists very well. No version of Outlook has ever been able to resize a photo - that task is performed by other software which can sometimes be used by Microsoft Outlook. Finally, my copy of Microsoft Outlook 2016 runs quickly on my 7-year old computer. It should run even faster on newer hardware.
