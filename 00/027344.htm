bob jones aka: cybertaz say:
In later versions of Word (including 2010) the default setting for the Normal paragraph style is to not update automatically. Additionally, the check box for enabling Automatically Update has been removed for Normal. The most likely reason the check box is there for you is that the document was originally created using an older version of Word -- 2003 or earlier for PCs, 2004 or earlier for Mac. If you create a new blank document in 2007/2008 or later, then go directly to Format> Style> Modify for Normal you'll see that the check box to Automatically Update isn't even there :-)

That property is not a factor when modifying a style in the Modify dialog. Any changes you make to the style's definition will automatically apply to any text in the document carrying that style -- That's the standard behavior of a Style.

The purpose of the check box was to enable the style modification based on any direct formatting changes made to the text, itself. That's what is happening to you & that's why the check box was removed... to keep users from having to go through exactly what you're experiencing.

IOW, using the Modify dialog updates the Style regardless of whether the box is checked, but directly formatting the text doesn't modify the Style. If the box is checked, formatting the text also redefines the Style.
