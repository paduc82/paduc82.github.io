hansv mvp say:

You can use a combination of INDEX and MATCH:

MATCH("monster",A2:G2,0) will search for "monster" in A2:G2 and if it's found in the 6th cell, it will return 6.

=INDEX(A1:G1, MATCH("monster",A2:G2,0)) will return the value of the corresponding cell in A1:G1.
