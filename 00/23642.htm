graham mayor say:
You cannot use 'wdFormatDocument' or 'wdFormatDocument97' when using Late Binding from Excel. These are Word commands. You must use the numeric values instead and SaveAs2 to specify save switches. You can get these from Word's VBA help.

The command lines would be

Dim strFilename as String

strFilename = "c:\Path\" & Who1 & Who2 & ".docx"
wdDoc.SaveAs strfilename 'Default Word format i.e. docx

or for DOC format from Word 2010

strFilename = "c:\Path\" & Who1 & Who2 & ".doc"
wdDoc.SaveAs2 strfilename, 0

or to specify the format - here DOCX

strFilename = "c:\Path\" & Who1 & Who2 & ".docx"
wdDoc.SaveAs2 strfilename, 12

You may not be able to save to the root of the C: drive with some operating systems, so choose a sub folder with read/write access as the Path. It will crash if there are illegal filename characters in Who1 or Who2. If this is likely you will need to add trapping for those characters.
