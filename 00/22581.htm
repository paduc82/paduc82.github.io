byhamc say:

Yes it is - Excel stores your personal macro workbook in a directory called XLSTART. On my laptop (running Win 7), the file can be found under:

C:\Users\[USERNAME]\AppData\Roaming\Microsoft\Excel\XLSTART

On my previous laptop (running Win NT), it was something like

C:\Users\[USERNAME]\AppData\Microsoft\Excel\XLSTART

 - the exact location of this will depend on your system set up so you may have to search around for it. Also, the AppData folder is hidden by default, so you may have to change the view settings to find it.

Once you've located the folder, you'll find a PERSONAL.xlsb file, which you can then copy and paste onto another PC in the same location.

Any files added to this folder will auto-start when you open Excel.
