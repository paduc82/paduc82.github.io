Windows Live is a loose descriptor, now deprecated, for a number of Microsoft applications and services that were an early attempt to integrate local activity with activity on the web. I think the only two things left bearing the name are the mail client program Windows Live Mail and the custom domain service known as Windows Live Admin Center.

Newsgroups are much older than Windows Live. Newsgroup posts are stored on dedicated servers and accessed from news clients over the Internet using a special protocol. A MAC (media access control) address is a unique numeric code that is permanently assigned to each unit of most types of networking hardware, such as network interface cards (NICs), by the manufacturer.
