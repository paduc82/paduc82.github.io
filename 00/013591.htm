babu v say: Hi Sheryl,

When you install any particular program that requires authentication to start. These generic credential programs (or websites) would need a username and password to initialize. The user values stored in the Generic credential manager are meant for long-term use and prevent user from needing to enter these credentials too often.
Note: These Generic credentials are different from that of Windows credentials.
