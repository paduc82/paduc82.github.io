john582n say:

I have it working on an Aspire One AOA150.  You need to do a simple registry edit then away you go!

  1. Press the Windows key + R simultaneously to launch Run.
  2.  Type in regedit and press Enter to launch the Registry Editor.
  3. Press Ctrl + F simultaneously to open the Find dialog box.
  4. In the Find dialog box, type in this: Display1_DownScalingSupported then click on Find Next button and Windows will automatically search for you the registry key and high light it for you.
  5. Click on the high lighted key (Display1_DownScalingSupported) and change the Value from 0 to 1.
  6. Click on Find Next and do the same change to the second one (if it is present)
  7. Once finished, exit the Registry Edit, restart your PC.
  8. After you've restarted, go to your Desktop, right mouse click, choose Screen Resolution and you should be able to choose 1024x768. Please choose this option, save and close.

(Modified from another post by JosephiineN

graham mayor say:
Using a bit of lateral thinking, and assuming that you want the paragraphs to split where they wrap naturally (assuming there are longer than one line), the following might work. In this case it works in cell 2 on each row.


Dim oTable As Table
Dim oRow As Row
Dim oCell As Cell
Dim oRng As Range
Dim oNewRng As Range
Set oTable = ActiveDocument.Tables(1)
For Each oRow In oTable.Rows
    Set oCell = oRow.Cells(2)
    Set oRng = oCell.Range.Paragraphs(1).Range
    Set oNewRng = oCell.Range.Paragraphs(1).Range
    With oRng
        .Collapse wdCollapseStart
        .Select
        Selection.MoveDown Unit:=wdLine, Count:=2, Extend:=wdExtend
        .Start = Selection.Range.End
        .Collapse wdCollapseStart
        If oRng.InRange(oNewRng) Then
            .End = oCell.Range.Paragraphs(1).Range.End - 1
            .Delete
        End If
    End With
    Set oNewRng = oCell.Range.Paragraphs(2).Range
    With oRng
        .Collapse wdCollapseEnd
        .Start = .Start + 1
        .Select
        Selection.MoveDown Unit:=wdLine, Count:=1
        .Start = Selection.Range.End
        .End = oCell.Range.Paragraphs(2).Range.End - 1
        If Selection.InRange(oNewRng) Then
            .Delete
            .Collapse wdCollapseEnd
            .Select
        End If
    End With
Next oRow
