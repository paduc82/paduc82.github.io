hansv mvp say:

Let's say the names are in column A, in A2:A500.

  • Select this range.
  • Activate the Home tab of the ribbon.
  • Click Conditional Formatting > New Rule...
  • Click "Use a formula to determine which cells to format".
  • Enter the formula
  • =COUNTIF($A$2:$A$500,A2)>3
  • Click Format and specify the text and/or fill color that you want.
  • Click OK twice.

Another option is to create a pivot table with the doctor name in the row labels, the expertise in the column labels, and the expertise also in the values area.

Hi,
Your question is not clear, but I'll try.

First, you have used a Worksheet Change event (Private Sub Worksheet_Change(ByVal Target As Range)) so presumably you want the code to run whenever any cell on the worksheet containing the code is changed.

 

If you have this code in Sheet1 in sample1.xlsm and you want the total to appear in sample2.xlsm then you could try the following:

 

 

Private Sub Worksheet_Change(ByVal Target As Range)
    Dim rng As Range
   
    If Target.Cells.Count > 1 Or IsEmpty Then Exit Sub
    If IsNumeric(Target) Then
        Set rng = Workbooks("sample2.xlsm").Worksheets("sheet1").Range("B1")
        rng.Value = rng.Value + Target.Value
    End If
End Sub

 

 

If you paste the code above in the Sheet Module of Sheet1 in sample1.xlsm, then each time a cell in that sheet is selected, if its value is numeric that value will be added to the value in cell B1 in sheet1 in sample2.xlsm.

Hi Mazuren,

Thank you for posting your query in Microsoft Community.

I understand the inconvenience caused to you. I will be glad to assist you with the Issue.

I would suggest you to follow these steps and check if it helps:

1. Press Windows Key + R

2. Type regedit in the run box.

3. Locate the following path HKEY_LOCAL_MACHINE\SOFTWARE\MICROSOFT\WINDOWS DEFENDER\

4. Right click this file and select permissions

5. Press Advanced

6. Where it says owner press change

7. Change the owner to your username

8. Now press apply then ok

9. Select Full Control from the permissions menu

(click the Widnows Defender folder)

10. Find the value DisableAntiSpyware and DisableAntiVirus and set these values to 0

(explaination: double click each and where the 1 is put 0 and press ok)

11. Open services.msc

12. Find Windows Defender

13. There should be 2 services: Windows Defender Service and Windows Defnder Netwrok

14. Right click each and press start

15. After you have started both restart your computer

16. Run Windows Defender

http://windows.microsoft.com/en-us/windows-10/how-to-protect-your-windows-10-pc#v1h=tab02
