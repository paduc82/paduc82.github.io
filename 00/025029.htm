pete_uk say:

Put this in cell AZ55:

 

=IF(BF55="","",IF(BF55=INDIRECT("BE"&45-ROW(A1)),INDIRECT("BF"&45-ROW(A1)),VLOOKUP($BF55,BE$5:BF$44,2,FALSE)))

 

then you can copy it down. Similarly with your other two:

 

BE55:   =IF(BF55="","",IF(AZ55=INDIRECT("BF"&45-ROW(A1)),BF55&" "&AZ55,INDIRECT("BE"&45-ROW(A1))&" "&INDIRECT("BF"&45-ROW(A1))))

 

Not sure why you are adding a null space to the ends of that one.

 

BF55:   =IF(AF55>"",IF(AF55=INDIRECT("BE"&45-ROW(A1)),INDIRECT("BE"&45-ROW(A1)),INDIRECT("BE"&45-ROW(A1))),"")

 

although this one could also be written as:

 

BF55:   =IF(AF55>"",INDIRECT("BE"&45-ROW(A1)),"")

 

as the inner IF is not making a selection (typo ?)

 

Note that INDIRECT is a volatile function, so your sheet will recalculate if there is any change and this might make it appear sluggish. An alternative is to use INDEX, like this:

 

INDEX($BE$1:$BE$44,45-ROW(A1))

 

instead of:

 

INDIRECT("BE"&45-ROW(A1))

 

and so on.

 

Hope this helps.
