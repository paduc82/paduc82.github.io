vijaydeshpande say:

can I set up a password for my entire My Documents folder?

My Documents folder is visible to all. Can I set up a password before I can open the My Documents Folder?

lemp say:
My Documents folder is visible to all. Can I set up a password before I can open the My Documents Folder?

If your hard drive is formatted using NTFS (My Computer > right-click Local Disk (C:) > Properties), then you can use NTFS permissions.

You can -- and should -- set a password to access your XP user profile.  Click Start > Control Panel > User Accounts (you may have to click User Accounts twice).  Select your account and click "Create a password." If you have a floppy drive, you can also create a password reset disk. Otherwise, write down your password and keep it in a safe, secure location.

Are there other people who use your computer?  Each person should have their own user account.  By default, user accounts that are not "computer administrator" accounts can not see files in the My Documents folder of other user accounts.

You also can restrict access to your files, even from "computer administrator" accounts (although someone who is a "computer administrator" and knows about NTFS permissions can find ways around such restrictions).  See http://support.microsoft.com/kb/304040

I would suggest you to change your display settings to extend by following the steps mentioned below:
1.      Open Screen Resolution by clicking the Start button, clicking Control Panel, and then, under Appearance and Personalization, clicking Adjust screen resolution.
2.      Click the drop-down list next to Multiple displays, click Extend these displays, and then click OK.

Note

·         If you don't see an option named Multiple displays in your display settings, then Windows isn't detecting a second monitor plugged into your desktop computer. Or, it is not detecting an external monitor plugged into your laptop. When Windows detects this monitor, it will appear in the drop-down list next to Display. If you don't see it listed there, try clicking Detect.

·         Make sure the monitor you want to extend your desktop to is turned on and plugged into the correct video port on your computer. You might also need to select the correct source labeled on your monitor's controls to ensure the monitor is using the correct video port. For more information, check the manual for your monitor or go to the manufacturer's website.
