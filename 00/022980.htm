    The Outlook AddressBook is simply a specialized view of the contact folders that have been designated to be members of the Outlook AB. If you want to print a list of your contacts - you need to print the contents of a specific "contact folder".  If you have more than one contact folder, then each must be printed separately.

     

    Highlight the contact folder then
     

    go to File --> Print
     

    #2 - The options available to you under "Settings" in terms of printer layout are dependent on the "view" that has been selected for the contact folder you want to print - i.e. different options appear when a "list" type view is selected vs a "card' type view
