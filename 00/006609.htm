I am assuming that if the user picks 2, you want the sum of Jan + Feb which will be in columns C & F.  So I name this SumToMonth in the formula below.

This formula must be **array-entered**.  A2 is the first cell in the row you desire to sum

=SUM(N(OFFSET($A2,0,-1+(ROW(INDIRECT("1:"&SumToMonth))-1)*3+3)))

    @MP

    You don't need the GetSeriesRange() function at all. The Y-values can be fetched from the series in the chart.

    Here is a shorter version of the FormatLine() macro doing the same work as the longer version - I think...

    Sub FormatLine_2()
    Dim ser As Series
    Dim i As Long
    Set ser = ActiveSheet.ChartObjects("Chart 1").Chart.SeriesCollection(1)
    For i = 2 To ser.Points.Count
      If ser.Values(i) >= ser.Values(i - 1) Then
        ser.Points(i).Border.Color = RGB(0, 255, 0)
      Else
        ser.Points(i).Border.Color = RGB(255, 0, 0)
      End If
    Next i
    End Sub
