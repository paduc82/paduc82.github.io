Sent emails in outlook 2013 are sent but show up in outbox folder and - Microsoft Community

freya100 say:

Sent emails in outlook 2013 are sent but show up in outbox folder and not sent email folder

I have office 2013 and in my outlook when I send an email often it goes but does not show up in my sent file - it shows up in the outbox. This makes it very difficult to know if emails are actually sent and my outbox is filling up
brian tillman [outlook mvp 2007-2019] say:

If a message does not move from Outbox to Sent Items, it means something is interfering with the communication between Outlook and the mail server.  As part of the data stream, Outlook says to the server "here's the end of the message; did you receive it OK?"  The server replies, "yes, I received it."  Then Outlook moves it to Sent Items.  If Outlook does not receive the "message received" acknowledgement, it can't know the transmission completed and so doesn't move the message to Sent Items.

The most common cause of this is an add-in interfering.  Typically, this add-in is your antivirus scanner.  You should disable scanning email, particularly outgoing mail.  It's not needed and can cause your symptom.

If disabling the AV scanner doesn't help, then start Outlook in safe mode (hold Ctrl while you click its icon to start it) and see if Outlook behaves normally.  If it does, stop and restart it, then you can go looking for the add-in causing the problem by opening the Add-In Manager (File>Options>Add-Ins>Go) and unchecking all add-ins.  Click OK.  Restart Outlook and see if it works.  If it does, reenable the add-ins one at a time, testing between each enablement.  If it stops working again, then the add-in you just enabled will be the culprit.  Remove that add-in.

Method 1: Run system maintenance troubleshooter

Please follow the steps to perform System maintenance check to improve performance of the computer:

1. Press Windows and x key together and select Control Panel.

2. Click on Troubleshooting.

3. Click on View all on the left side panel.

4. Click on System Maintenance and follow the onscreen instructions.

Method 2: Perform Clean Boot

Placing your system in Clean Boot state helps in identifying if any third party applications or startup items are causing the issue. Check this issue in clean boot state.

  1. Press Windows key + X key. And select Run.
  2. Type msconfig in the Run box and hit Enter.
  3. On the Services tab of the System Configuration dialog box, tap or click to select the Hide all Microsoft services check box, and then tap or click Disable all.
  4. On the Startup tab of the System Configuration dialog box, tap or click Open Task Manager.
  5. On the Startup tab in Task Manager, for each startup item, select the item and then click Disable.
  6. Close Task Manager.
  7. On the Startup tab of the System Configuration dialog box, tap or click OK, and then restart the computer.

For more information refer the below link.

How to perform a clean boot in Windows

http://support.microsoft.com/kb/929135

Note: After troubleshooting, refer to this section "How to reset the computer to start normally after clean boot troubleshooting".

Improve performance by optimizing your drive
