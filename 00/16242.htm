Word does not allow hyphenation between pages (leads to diverse - Microsoft Community

trig123 say:

Word does not allow hyphenation between pages (leads to diverse number of lines per page)

This topic was already discussed in Word 2013 forum, but without resolution.

The problem ist still there in Word 2016 and - as I consider it as critical for professional authors - I'd like to reopen this unresolved issue:

Word does not allow hyphenation between pages which leads to diverse number of lines per page.

Consequence: If a page ends with a word that is hyphenated, the whole line of text is moved to the next page.

If you remove hyphenation of the word manually, the complete line drops back to the end of the previous page (where it belongs to).

  • This behavior is independent from the "Widow/orphan control" and

  • also happens when limiting the complete text to a single font with one contious font size

  • and without using any paragraph spacing.

If two consecutive bottom-most lines are auto-hyphenated, only the last is pushed onto the next page; so, a new page can start with a fragment of a word.

Maybe in the meantime someone found any workaround to this?

Thanks a lot,

Trig (DE)
