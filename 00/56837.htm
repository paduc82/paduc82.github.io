azeez n say: Hi simple00,

If for any reason, one or more icons are not showing correctly, you can repair by rebuilding the icon cache. Icon cache is a hidden file located at the Local Folder. Here is how to do it:

1. Open Windows Explorer (any folder/drive).

2. As the IconCache is a hidden file, you need to enable ‚ÄúShow hidden files‚Äù option to see the same. To do this, head over to Tools > Folder Options, switch to View tab, and finally enable Show Hidden files, folders, and drives option.

3. Now navigate to C:\Users\username\AppData\Local  folder and then delete IconCache.db file.

4. Reboot your computer to rebuild the icon cache.

5. All icons should be displayed correct now.

But, before rebuilding icon cache, I'd suggest you to create a restore point .
