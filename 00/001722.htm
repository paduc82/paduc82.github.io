Most USB devices can be removed and unplugged. xircal say:

PCSUUCC.exe is what's known as a PUP (Potentially Unwanted Program). PUPs usually get installed when a user installs other software and chooses the "Recommended" option which appears during the installation process. It's called "Recommended" because the developer of the application wants you to install additional applications for which he receives commission.

It's advisable when installing free software to always select the "Custom" option since this will display a list of whatever other applications have been bundled with it. These are usually checkmarked by default. Removing the checkmarks will ensure you only install the software you actually want.

See this link which describes PCSUUCC.exe: https://www.reasoncoresecurity.com/pcsuucc.exe-fb9c101a240af8b9d72b328115b56ca624f5585c.aspx

If the Microsoft Security Scanner which Ramesh Kumar recommended doesn't remove it for you, download the free version of Malwarebytes and run a full scan with that. I'd advise you to do that anyway since some PUPs are capable of downloading further unwanted applications without user  intervention.

Click the Download button on this link which will get you the link to the free version: https://www.malwarebytes.org/
