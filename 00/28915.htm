Windows 10 is really just Windows 8.2 - Microsoft Community

barthumphries say:

Windows 10 is really just Windows 8.2

This isn't a Windows 10 technical preview, it's a Windows 8.2 consumer preview because there's nothing technical about this preview. There's no ADK, no API, no method changes, as far as any of us non-Microsoft employees can tell, there really aren't any changes from Windows 8.1 other than:

  • Windows button instead of a Start button
  • Metro menu has been shoved into the Start menu.  Only a few apps to start with, but more can be added so that the Start menu fills the whole screen
  • Search and "task view" icons on the taskbar that can't be gotten rid of.

That's not a completely new version of Windows, it's just some slight features changes.

Microsoft: No, it's really different, we promise!

Us: Ok, how?

Microsoft: Well, we can't tell you yet, but it will be different.  We'll have Cortana!

Us: Do you have that now?

Microsoft: No, but we've added some drop shadows that we didn't have before!

Us: Seriously, that's the big difference that you're highlighting, drop shadows?

Microsoft: Uhm, uhm, you can search from the desktop!

Us: How does this qualify as a completely new version again?

Metro app icons are still as clunky as ever -- a giant desert of a weird random color surrounding a single small icon with the name underneath (or you can make it a small icon and remove the name).  If we wanted large icons on the desktop we would have increased icon size but Microsoft thinks we're all going blind and need ginormous icons.

Microsoft is still in love with horizontal scrolling, despite that practice being a big no-no in web development, Microsoft somehow thinks we'll welcome it on the desktop.  Metro apps on the Start menu arrange themselves horizontally by default.  Even if you only have 7, and they're all small, you can't put them in a column so you'll have a giant dessert of whitespace underneath them on the Start menu.  If you drag "documents" over to the right side of the start menu, you go from a normal favicon to a giant folder image for old blind people.

Basically, other than the Metro screen being shoved into the Start menu, there's nothing new.  There're promises of more, and when those resolve perhaps I'll change my mind.  Until then, I have but one lamp by which my feet are guided and that's the lamp of experience.  There's nothing in this consumer preview that's technical in any way.

cmdrkeene say:

Every version is just a few slight features changed. 2000 to XP was just a different color taskbar for the most part.  Blue instead of gray.  That's also why the internal versions reflect the true amount of changes:

Windows NT 4.0 (along the same time as Windows 95)

Windows 2000 was just NT 5.0

Windows XP was just NT 5.1

Windows Vista was 6.0 and was a major change, mostly to take advantage of new hardware for things like graphics acceleration that were never possible before.

Windows 7 was just 6.1 and really just changed some taskbar behaviors.

Windows 8 was 6.2 and although it looked different, the big change was just a bigger (full-screen) start menu. Although it was big, it had the same function (shortcuts and an 'all programs' directory).

Windows 8.1 was 6.3 and added some mouse functions and title bars

Windows 10 is just going to be 6.4.

Everything is gradual.
