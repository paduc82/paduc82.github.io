Software requirements:

  1. Microsoft Office 2013
  2. DAMO
  3. I only tested in Windows 8/8.1 but it should work in 7 as well.

My Solution:


  1. Uninstall DAMO.
  2. Install Microsoft Office 2013 (32 bits IMPORTANT) and restart.
  3. Install DAMO as Administrator (right click and Run as Administrator), but don't set up any user profile yet.
  4. Install any DAMO fixes as Administrator and this time set up your user profile (Name, Domino Server, Data folder, id, etc).
  5. Find the file mapisvc.inf inside the Office 2013 installation folder. You can use the Explorer Search Box.
  6. Take notice of mapisvc.inf file location, in my case it was located in "C:\Program Files\Microsoft Office 15\root\vfs\ProgramFilesCommonX86\SYSTEM\MSMAPI\3082\", but I've seen it located in other machines in "C:\Program Files (x86)\Common Files\System\MSMAPI\3082\".   9. Let the Initial Setup work, if for any reason Outlook terminates abnormally, just repeat step 8.
  10. Wait for the "Outlook will now close" dialog and click OK.
  11. Once Outlook has closed, restart computer.
  12. Run Outlook as normal user. No more Run as Administrator.
  13. Enjoy.

resourceful say: Hello. PIM info (contacts/calendar) only syncs to a full version of Outlook (2003/2007/2010) via a WMDC USB connection to your PC. It won't sync to the Live email client. The easiest thing for you to do is sync your contacts to Live with an internet connection (data plan or Wifi) via an exchange activesync connection. You will use your Live ID. Screenshot directions can be found here (per Prof Julie, a forum moderator):
http://frommypocket.pocketprimer.com/2012/03/29/sync-your-windows-mobile-6x-phone-to-your-windows-live-account.aspx

The info will not be stored on your computer but in the "cloud" (e.g. Microsoft servers). You can easily access your contacts at www.hotmail.com. 

If you want to backup more info, e.g. recent calls, call history, etc., create a backup file with the free program PIMBackup to an SD card. Take the card out before sending it in for repair.  When the phone comes back from repair, reload the program, and restore the backup. 

Lastly, if you want to create a system backup, you can buy a program like SPB Backup . Again, download and install the program, create the backup file to your SD card, take out the SD card before sending it in, and restore the backup when you get it back. It will be just like before you sent it in, with all the apps and settings in place.
