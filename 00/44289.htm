How to Run A Program in Above Normal Priority Mode Automatically - Microsoft Community

windows_has_flaws say:

How to Run A Program in Above Normal Priority Mode Automatically

I want to know how to run a program in Above Normal priority mode every time I click on it instead of having to go to Task Manager every time to manually set it. Thanks!
son of magni say: Run Regedit and go to the following key:

HKEY_CLASSES_ROOT\exefile\shell

Now under "shell" add a new key called "Run with Above Normal priority"
Under that key add another key called "Command"
In your "Command" key, double-click the "(Default)" Reg_SZ entry and change the value to:

cmd.exe /c start "Run with Above Normal priority" /abovenormal  "%1"


Now when you want to run an exe with above normal priority, right-click on the exe and select your new command from the context menu.
