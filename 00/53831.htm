The answer depends on what kind of item you want to pin.  Since the only kind of item you mentioned was a folder, you can easily do what you're asking about now and needn't wait for a later build.

In the Windows 8 desktop, when you drag an executable file (or shortcut) onto the taskbar, a message pops up that says "Pin to Taskbar." 

 

If you drag a document file (or shortcut), the message will say "Pin to [associated application]" - in other words, "Pin to [default browser name]" for an html file, "Pin to Notepad" for a txt file, etc.

 

For a folder, File Explorer is the associated application.  Therefore, if you open a File Explorer window, locate the desired folder, and drag it to the Taskbar, the message that pops up says "Pin to File Explorer."  Just click on that, and it will appear in the FE jumplist, above the Frequent Places, in a separate section called "Pinned."  You can drag things up and down there to order them as you wish.

 

(The Control Panel, though it is displayed in FE as a "System Folder," will only let you pin it to the desktop, not to the FE jumplist.  There may be some other special cases like that; I don't know, but it generally works like I described, including for other System Folders.)

 

Just to be clear, I'm a new Windows 8 user, very far from expert.  I only know how this works because I learned it by trial and error today.  So there may be more to it that I can't help with.
