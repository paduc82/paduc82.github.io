Hi,

There is a way you can put a password on your Internet Explorer to restrict its usage. This will be most effective, given that you do not have any other browsers on your computer aside from this. Please do note that this option is only available for Internet Explorer 9 and earlier. Kindly check the steps provided below:

Internet Explorer 9

  1. Open Internet Explorer.
  2. Go to Tools.
  3. Go to Internet Options.
  4. Click on the Content tab.
  5. Look for Content Advisor and click Enable.
  6. In the Content Advisor dialog box, click on General tab.
  7. Please make sure that Supervisor can type password is checked.
  8. Select Create password.
  9. In the Create Supervisor Password box, make any password that you want.
  10. Click OK.
  11. Click Apply.
  12. Restart Internet Explorer.

Internet Explorer 10 onward

  1. Press Windows Key + R.
  2. Type RunDll32.exe msrating.dll,RatingSetupUI.
  3. Click General tab.
  4. Please make sure that Supervisor can type password is checked.
  5. Select Create password.
  6. In the Create Supervisor Password box, make any password that you want.
  7. Click OK.
  8. Click Apply.
  9. Restart Internet Explorer.

Let us know if you need further assistance.

Contrary to common belief running a macro doesn't clear the undo stack. It's the action of a macro writing to or changing the workbook that clears the stack and while the difference may seem subtle it can be used to advantage.

You can write your own UNDO routine but in my view this is more trouble than it's worth. Have a look here.

http://spreadsheetpage.com/index.php/tip/undoing_a_vba_subroutine/

You could output your macro results to another workbook but depending on what your doing this may not be possible. If you do adopt this approach then you can read the result back in using a formula and the undo stack is preserved

'lots of code
myresult = Range("A1") ^ 3 / Range("A2").Value
Workbooks("Results book.xlsx").Sheets("Sheet1").Range("A1").Value = myresult

and in the original workbook you read back the value with a formula.

='[results book.xlsx]Sheet1'!$A$1

I should have pointed out how you change the ranges. From the original (shorter) formula you need to change it in 3 places all underlined below. Bear in mind this will on;y work of the lookup data start in row 6, the -4 bit provides the offset.

 

=INDEX($V$6:$V$20,SMALL(IF($U$6:$U$20=$A$3,ROW($A$6:$A$20)-ROW($A$1)-4),ROWS(B$1:B1)))
