robert_foster say: I have having this problem too, and it is really frustrating.  I did not have this problem with my previous edition of Word.

I did find an imperfect workaround. 

Insert > Symbol > More Symbols.  In the window that opens, Select the symbol, but also make sure that font line near the top left is set to the font that you normally use.  Then enter your shortcut.  When you use the shortcut, the the symbol should appear in the desired font. 

The drawback, of course, is that if you happen to use a different font in a particular document, the symbol will appear in the original font that it is set to.

glenprouty say:

Hi Suneel,

Your problem is most frequently caused by the presence of "another" anti-

virus, or the fragments of one. If you desire to use Windows Defender as

your Real Time Protection, you must uninstall any other antivirus present and

also run the removal tool that is supplied on its web site. Windows Defender

sees the fragments that are left.

In some cases Malware can also cause your problem. Both malware, and

3rd party antivirus set a "Policy" in Windows Defender that disables it.

We will undo that Policy, but first we must get rid of the cause.

1. Ensure that any other antivirus is completely uninstalled.

While MalwareBytes and WD normally run in harmony, there have been

cases where MalwareBytes Preminm caused a problem. For this exercise,

Turn off the Real Time protections in MalwareBytes. Turn them on later.

2. Scan your PC to get rid of malware.

   If you don't have Malware bytes you can download a copy here:

https://www.malwarebytes.com/trial/

  Turn off its Real Time protections and scan your PC. Include "scan for

Rootkits".

You can also run an online scan from here:

http://www.eset.com/us/online-scanner 

When you are satisfied that your PC clean, and no other antivirus exists,

you should observe the Registry key that contains the WD Policy changes.

Enter the Registry by clicking "start" and type "regedit". Then click on "Run as

Administrator". In the Registry, navigate through the following branch.

Hkey_Local_Machine\Software\Policies\Microsoft and select the Windows

Defender folder.

This folder should only contain one Key, "Default". Any other key present

has been set by Malware, Another Antivirus, or by You (W10 Pro).

Expect to find keys for "DisableAntiSpyware" and/or "DisableAntivirus".

If you do, you should modify their DWords from 1 to 0.

Modify the DWord by right-clicking it, and then click "modify".

Type 0 and the highlighted 1 should change. Click "OK".

Back out of the Registry by retracing the steps that you used to enter.

This keeps your Registry from becoming cluttered. When you reach HKLM,

click "File" then click "Exit".

Restart your PC    Then access Windows Defender again. The message

should be gone. If Windows Defender is not enabled, go to Virus & Threat

Protection Settings and turn on Real Time Protection.

Once you are satisfied that Windows Defender is functional, If you had

Malwarebytes previously installed, turn on its Real Time Protections.

If the problem returns, Malwarebytes was your cause!

Good luck,  Glen
