greg carmack - windows mvp 2010-2020 say:
Hi OdongoBenard. I'm Greg, an installation specialist and 9 year Windows MVP here to help you.

A new PC with WIndows 10 preinstalled should have a Product Key embedded in the UEFI firmware chip on the motherboard. To check this run ProducKey: http://www.nirsoft.net/utils/produkey-x64.zip to look for an OEM-BIOS Product Key.

If there is no such key then ask the seller how you are to activate preinstalled WIndows, because it is illegal piracy to sell Windows software without a way to activate it.

If you have a Digital License on another PC, it is possible it could be moved to this one if it's the exact same version following the procedure here:
https://support.microsoft.com/en-us/help/20530/...

As long as the underlying license you upgraded or that came with WIndows 10 is retail and not factory preinstalled or builder's OEM, then it can transfer since Upgrade inherits the rights and restrictions.

It is possible to do yourself if you'd previously signed in with your Microsoft account and it linked to the Digital License as shown in Status at top of the Activation Settings page. Then when you change motherboard, you'd sign in again and run the Troubleshooter. We recently learned it will even make exceptions for OEM if all else is proper. So that's your best bet.

Servers are overloaded lately so you may get a Try Back later error message. So you can wait and try back in a day or two.

If not contact Microsoft Support: https://partner.support.services.microsoft.com/...

I hope this helps. Feel free to ask back any questions and let us know how it goes. I will keep working with you until it's resolved.
