stefan blom say:

In Word, each section can have up to three headers (and three footers); this is the case if you select both "Different first page" and "Different odd and even" in Page Setup.

By default, a header (or footer) is linked to the corresponding header (footer) in the preceding section, unless you manually unlink it using the Same as Previous command on the Header & Footer Tools Design tab of the ribbon. Each type of header (footer) must be unlinked separately from its "neighbor" in preceding section. (For example, you have to unlink the First Page header in section 2 from the First Page header in section 1. And you have to unlink the First Page Footer in section 2 from the First page Footer in section 1. and so on for Odd headers. Etc.)

phil f say: With the help of Renjie Yu in the Windows Mobile Community Forum, I have created a batch file solution to this problem. 


 

 

The steps to the batch file solution are as follows:

 

1.       Create a batch file named WmdcOpen.bat containing the following arguments:

@Echo off

REM Starts wmdc.exe and shows the WMDC program screen

 

Start C:\Windows\WindowsMobile\wmdc.exe /show

Exit            

                  

2.       Create a shortcut to the WmdcOpen.bat file by right clicking the file and selecting create shortcut.  The name of the shortcut is WmdcOpen.bat â€“ Shortcut.  Right click the shortcut, select Properties and select â€œMinimizedâ€ in the Run window and select â€œOKâ€.  Clicking the shortcut opens the WMDC program screen via the batch file.  The command prompt window runs in the minimized state then closes on completion of the file arguments.

3.       Create a batch file named WmdcCloseOpen.bat containing the following arguments:

 

@Echo off

 

REM Quits wmdc.exe process and the WMDC program screen

REM and restarts wmdc.exe process in Task Manager for future syncing

 

Start C:\Windows\WindowsMobile\wmdc.exe /quit

Timeout /t 4

Start C:\Windows\WindowsMobile\wmdc.exe

Exit

4.       Create a shortcut to the WmdcCloseOpen.bat file by right clicking the file and selecting create shortcut.  The name of the shortcut is WmdcCloseOpen.bat â€“ Shortcut.  Right click the shortcut, select Properties and select â€œMinimizedâ€ in the Run window and select â€œOKâ€.  Clicking the shortcut closes the WMDC program screen, quits the wmdc.exe process in Task Manager then restarts the wmdc.exe process in Task Manager without opening the WMDC program screen.  The command prompt for the file runs in the minimized state and exits on completion.  The Timeout argument of 4 seconds is necessary to allow time for the WMDC program screen to close.  The restarting of wmdc.exe leaves the computer ready for the next synchronization.

5.       Put these two files and two shortcuts in the C:\Batch Files directory created on the computerâ€™s hard drive.

6.       Navigate to the following registry key via regedit.exe on the PC:

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows CE Services\AutoStartOnConnect

7.       Create a new string value by right clicking in the registry editorâ€™s right pane of this key and selecting New>String Value and name the string value WmdcOpen

8.       Right click the new WmdcOpen string value and select Modify.  Place the following text in the  Value Data box:

"C:\Batch Files\WmdcOpen.bat - Shortcut" with quotes

Select â€œOKâ€

9.       Navigate to the following registry key via regedit.exe on the PC:

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows CE Services\AutoStartOnDisconnect

10.    Create a new string value by right clicking in the registry editorâ€™s right pane of this key and selecting New>String Value and name the string value WmdcCloseOpen

11.    Right click the new WmdcCloseOpen string value and select Modify.  Place the following text in the  Value Data box:

"C:\Batch Files\WmdcCloseOpen.bat - Shortcut" with quotes

Select â€œOKâ€

12.    Close the registry editor.

diane poremsky mvp (slipstick.com) say:
'This computer only' tagged on a folder means it is on this computer only; it is not synced up to the server.

Switch to the folder list - Ctrl+6 - and look for the folder. (Ctrl+1 or the Mail icon will get you out of this view.)

If you cant find it, do another search for contents in the folder. Open one message then open Advanced Find and click Browse. This will show you were it is. (Screenshots here: https://www.outlook-tips.net/how-to/find-folder...)

Make sure you have a copy of these messages or move them into a folder that syncs. If anything happens to the ost, you'll lose the content of these folders.
