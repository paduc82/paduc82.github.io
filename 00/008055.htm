brian tillman [outlook mvp 2007-2019] say:
If they're IMAP folder sets, every time you delete the account from Outlook, the PST associated with that account should be deleted and every time you add an IMAP account back in a new PST will be created.  Sometimes, the PST doesn't get properly deleted when the account is removed.  Remove all IMAP accounts from Outlook, close Outlook, then delete the IMAP PSTs (don't delete the PST you're using as the default data file, though).  Either re-add the IMAP accounts using the Mail applet in Control Panel or by starting Outlook and adding them there.

Most Product Key Cards (PKC) come with a 27 character PIN that has to be entered on a website to "activate" it and retrieve the 25 character "Product Key".  Product Key retrieval using the PIN is a one-time process so you have to keep track of the retrieved Product Key.

The 25 character product key has to be entered in one of the Office applications to "activate" the Office installation.
