suzanne s. barnhill say:

If you don't want a number on the first page, then, when using roman numerals, you must insert a section break between the first and second pages and start numbering at i in the second section. In order to leave the first header/footer blank (unless you enable Different First Page for that section), you will need to unlink the second section from the first. suzanne s. barnhill say:

Number format can be changed for any given section, and numbering can be restarted in any section, even without unlinking the header or footer.

  1. Start with the insertion point in the header of the first section of the front matter.
  2. On the Header & Footer Tools | Design tab, in the Header & Footer group, click Page Number and choose Format Page Numbers...
  3. In the Page Number Format dialog, choose "i, ii, iii" for the number format and click OK.
  4. On the Header & Footer Tools | Design tab, in the Navigation group, click Next Section one or more times to get to the header on the page where you want to start numbering with 1.
  5. Open the Page Number Format dialog again and choose "1, 2, 3" for the number format.
  6. Under "Page numbering," select the "Start at" radio button and (if necessary) choose 1 for the starting number.

It's best to do those last few steps after you have more than one section of 1, 2, 3-numbered pages because, when you restart numbering in a given section, when you insert a section break to create a new section, the numbering will restart in that section as well. If this happens, you'll have to open the Page Number Format dialog for every section that is restarting and choose "Continue from previous section." But if you wait to restart numbering until you have at least one more section after the one where you're restarting, you won't experience this problem.

suzanne s. barnhill say:

Like Charles, I don't have any experience with using Word on a tablet, but on a PC, the solution for this specific situation (doesn't work if you want to start with another number) is this:

  1. On the Header & Footer Tools | Design tab, in the Options group, check the box for "Different First Page." This will give you a separate First Page Header/Footer that you can leave blank (or at least without a page number).
  2. Then, in the Header & Footer group, click Page Number and choose Format Page Numbers... In the Page Number Format dialog, select "Start at" and enter 0 as the starting number. This will give you 0 (not displayed) on the first page and 1 on the second page.

samsaifan say: Solved!!

So it's an AV problem and usually happen when the name of the PST file ends with .com. Some AV's treat it as a .com application file and either delete it or just mess it up. I would suggest giving the PST file a friendly name that has no AV alarm-triggering file extensions.

Recovering your vanished PST file -Hope it's not too late:

1- Browse to the folder where your missing PST was saved in (Usually "Outlook Files" under "My Documents"
2- Right click on the folder containing the PST file and select "Properties"
3- Click on the "Previous Versions" tab
4- You should now see at least one folder that has a date close to when the PST file disappeared
5- Click on the folder then click on "Open"
6- Now you'll see your dearly missed PST file, just copy the file to some other folder
7- Re-associate your email account with the PST file recovered
