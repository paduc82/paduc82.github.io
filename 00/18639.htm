  2. Click on the Contacts icon to show your contacts.

  1. Sign in to Skype.

david marcovitz say:

It seems easy to do this with slide masters if all the titles are in the title placeholder. If they are not, then, for a macro to work, you would need some way to distinguish what a title is. Normally, the title is shape #1 on each slide, but you can't always count on that. If that is the case, then a macro would look something like this (untested, just typed off the top of my head):

Sub FixTitles()

    Dim oSld as Slide

    For Each oSld in ActivePresentation.Slides

        With oSld.Shapes(1)

            If .HasTextFrame Then

                .TextFrame2.Font.Size = 48 ;something like this and a line for each fixed property

            End If

        End With

    Next oSld

End Sub

This probably needs a bit of tweaking, but it should get you started on the right idea.

john sr wilson say:

You need to explain EXACTLY what you mean.

Are the "Titles" really Title type placeholders?

Are all of the slides using the same layout? (And therefore all titles should be in the same position)?

If these are both true this might work :

Sub fixTitles()
Dim L As Long
Dim sngL As Single
Dim sngT As Single
Dim sngH As Single
Dim sngW As Single
If ActivePresentation.Slides(1).Shapes.HasTitle Then
With ActivePresentation.Slides(1).Shapes.Title
.PickUp
sngL = .Left
sngT = .Top
sngH = .Height
sngW = .Width
End With
End If
For L = 2 To ActivePresentation.Slides.Count
If ActivePresentation.Slides(L).Shapes.HasTitle Then
With ActivePresentation.Slides(L).Shapes.Title
.Apply
.Left = sngL
.Top = sngT
.Width = sngW
.Height = sngH
End With
End If
Next L
End Sub
