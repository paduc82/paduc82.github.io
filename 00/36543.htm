If he is able to install and activate XP on the computer licensed to run the OEM version of XP, yes you can install Windows 7 Upgrade on that computer (not another one). You may have to call Microsoft to get XP activated. You can't install the OEM XP on a computer other then the one it came with.

According to Microsoft an OEM version is basically tied to the first motherboard it's installed on and you can only install a new motherboard in that computer if you are doing so because the original motherboard becomes defective.

Difference Between a Full License and an OEM License
OEM licenses of Windows:
are bound to the first computer installed on and are not transferable
cannot do an in place upgrade from older versions of Windows
offer no free support from Microsoft
allow hardware upgrades of all components except for the motherboard (unless because the original motherboard becomes defective )

      Hello,

      Most USB to PS/2 adapters will only work with keyboards and mice.  They only support a limited set of input-related commands.  In your description, it sounds like whenever one of those commands is sent (like when you swipe), it causes an error on the USB device, and Windows shuts it off.
