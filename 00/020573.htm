Hi Judith,

 

I would stay far, far away from Norton.  AVG is fine, but not as good as you can do.  And using several real-time products at the same time not only causes problems and should not be done, but actually reduces rather than increases your security because the conflicts between them mean neither function at their full capacity leaving holes which would otherwise not exist.  In the end, the choice is yours.  I'll give you two examples and show you how some products can be combined to increase security (but by after-the-fact catching rather than running concurrently in real-time to avoid the conflict which would otherwise occur).  It you're security-conscious as evidenced by trying to use multiple products at the same time, then you may want to go with what I do - but it's your computer, your decision, and your risk.

 

Our forum Moderator only uses MSE and has for years and hasn't been seriously infected yet, but I personally prefer to use a more diverse approach.  For most users, MSE will probably be fine on its own as long as they use good habits (we'll talk about that shortly).  My job requires me to take more risks by downloading and trying many different types of software, some I normally would avoid but need to in order to help someone having a problem with it and can do so better if I have it in front of me than if just reading articles about it, and visiting sites where others are having trouble that I might otherwise avoid for that very reason.  So I like a bit more protection.

 

Here's what I do (and it isn't 100% but nothing is, and in some cases only catches things after they've already gotten into the system, but it's a good process while remaining reasonable and they can all work together - as proven by the fact that I use them - and everything is free {if it costs money, you are either in the wrong place or selecting a paid version rather than the free version and not only is that not needed - but the paid versions often contain features that may cause conflicts and shouldn't be installed with MSE}):

 

I use MSE for real-time protection and run scheduled daily quick scans with a scheduled bi-weekly full scan.

 

I update and periodically (every couple of weeks or so) run full scans with Malwarebytes: http://www.malwarebytes.org/ and SuperAntiSpyware: http://superantispyware.com/ and ESET online scanner: http://www.eset.com/us/online-scanner Get only the free versions. Any that also have paid versions include features like real-time protection that cannot be used along with MSE anyway (in these cases), so there's no point in spending the money anyway.

 

I use Secunia PSI http://secunia.com/vulnerability_scanning/personal/ to make sure all my programs are updated and current (since updates often plug security holes in the software and thus can be important as many people never check and/or update their software unless prompted - and this way I don't need to check everything since notifications and sometimes updates are automatic).

 

I always follow the following advice from Stephen Boots - MVP (and you should do this no matter what products you decide to use from above):

 

-Make sure that the Windows Firewall is enabled. No other firewall truly offers better protection or is needed. This one is perfectly fine.

 

-Make sure that all important/critical updates, including service packs for the operating system and programs are installed from Microsoft Update (Windows Update).

 

-Make sure Internet Explorer is at version 8 or 9 and updated with all patches.

 

-In Internet Explorer 8 or 9, use the SmartScreen Filter.

 

-Make sure that IE Internet Security settings are at least set to medium-high (default).

 

-Enable the pop-up blocker in IE.

 

-On Vista and Windows 7 make sure that User Account Control (UAC) ON and not running with elevated privileges.

 

-Make sure that Windows Automatic Updates are set to at least notify, but the preferred setting is to download and install automatically. If you update manually, be sure to update as soon as possible after being notified of available updates.

 

-Make sure that installed applications, especially Adobe Acrobat, Adobe Flash, and Java are at their latest versions. Many vendors are regularly updating and patching for security holes.   Secunia PSI can help greatly in automating and reminding you about this.

 

-Never click through links from unknown sources and use caution even if they are from a "trusted" source.

 

-Never open unsolicited email attachments.

 

-Practice safe web browsing.

 

I hope this helps.  If you would like detailed instructions on how to install MSE, just post back and I'll send them to you.  The others are all quite simple and don't really need instructions but MSE can be a bit tricky in terms of requirements and being properly prepared before installation and even in choosing the best way to configure it to start.

You cannot.

 

The solution is to assign a format other than General, for example Number with 2 decimal places.

The SQL is: 

UPDATE tblCustomers SET CurrentBalance = DLookup("txtBalance", "frmPlaceOrder", "txtCustomerID= [CustomerID]")
WHERE DLookup("txtBalance", "frmPlaceOrder", "txtCustomerID= [CustomerID]");

There is also a select query that also contains the balance, in case that would be a better option as oppose to a form.

Please advise.

scottgem say:

First Balance is a calculated value and should not be stored. It should be calculated on the fly using an expression.

Second, A Dlookup forks against a table NOT a form. The proper SQL should be something like:

UPDATE tblCustomers SET CurrentBalance =Forms!frmPlaceOrder!txtBalance 

But, again, you really shouldn't be storing this value.
