bernie deitrick say:

You could use the workbook open event of the user's personal.xlsb - or any other method to run the macro code below - to copy all the files from a source folder (if they exist) to replace existing addins: This code uses the Alternate Start Up folder - you could use StartupPath instead, and assumes that the new add-in versions are places into a folder for each user: "Z:\My Documents\Excel\New Addins"

Private Sub Workbook_Open() ' Or use as a regular macro

    Dim wbAddin As Workbook

    Dim strNewAddin As String

    Dim strAI As String

    Dim strPath As String

    strPath = "Z:\My Documents\Excel\New Addins"


    strNewAddin = Dir(strPath & "\*.xlam")


    While strNewAddin <> ""

        Set wbAddin = Workbooks(strNewAddin)

        If Err = 0 Then

            wbAddin.Close False

        End If

        FileCopy strPath & "\" & strNewAddin, Application.AltStartupPath & "\" & strNewAddin

        Workbooks.Open Application.AltStartupPath & "\" & strNewAddin

        Kill strPath & "\" & strNewAddin

        strNewAddin = Dir()

    Wend

End Sub

From your description, I understand that you want to disable the Exclamation sound alert while closing an unsent/unsaved email.

This is a shared Windows feature which cannot be disabled only for Outlook as other applications uses it as well. And disabling this sound alert, would disable the other applications which uses it.

Follow the below mentioned step to disable Exclamation sound alert:

1) Go to Control Panel and double click on Sound.
2) Select Sounds tab and select Exclamation from the list under Program.
3) Now click on the drop down arrow for Sounds and select None, Apply and OK.
4) Open Outlook and try closing an unsent/unsaved email and verify the result.
