general ledger say:

Yes, there is a formual for that.

New to Excel 2007 are the functins SUMIFS and COUNTIFS.  Both work like the old SUMIF and COUNTIF except they accept multiple criteria.  This eliminates the need for creating keys by concatenating values.  I think the maximum number of conditions is 256.  What ever the maximum is, I think it is enough for most users.

Another method is using Array Formuals.  With these, you can test for several true/false values and then perform a calculation.  Array formulas are a little more challenging to create and require extra computer resources to calculate but often can solve calcuations that would not otherwise be possible in one formula. 

Here is an example of an array formula:  {=SUM(ROUND(E5:E8,2))}  This array formula rounds each value in the range E5 through E8 to two decimal places and then sums those rounded values.  The { } brackets are not typed in while entering the formula.  You enter the formula and the last step is to press Ctrl+Shift+Enter at the same time to end the formula.  Pressing these three keys together tells Excel this formula is an array formula and Excel automatically appends the { }

Here is an array formula that might closely fit your needs:  {=SUM(($A$1:$A$10=Market)*($B$1:$B$10=Size)*($C$1:$C$10=Type)*($D$1:$D$10))}

This will look at the values in A1:A10 and compares each to the value Market.  Each of these ten comparisons returns a true or false.  A true is a value of 1 and false is 0.  Then it  looks at each value in B1:B10 and compares each to the value Size, returning a 1 or 0 for each test.  The same for range C1:C10 and Type.  When the test for Market returns a 1, the test for the Size returns a 1 and the test for the Type returns a 1, then all the conditions are met and the formula sums the corresponding value in the range D1:D10

I hope this helps,
