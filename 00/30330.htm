Unfortunately, this was also a problem on Ghosts, because for some reason Activision's servers communicate through two ports which are not on the list of open ports in Xbox One's port forwarding. I strongly recommend giving the port forwarding a go, because www.portforward.com has a great set of step-by-step instructions on how to forward ports for every brand and model of router out there.     It will tell you how to log into your router, AND how to forward specific ports. The ports which will need to be forwarded are:

    Port 88 (UDP)

    Port 3074 (TCP and UDP)

    Port 53 (TCP and UDP)

    Port 80 (TCP)

    Port 500 (UDP)

    Port 3544 (UDP)

    Port 4500 (UDP)

    Port 3075 (UDP - Activision Specific Port)

    Port 3076 (UDP - Activision Specific Port)

    When I first played Ghosts, I had this identical issue - open NAT, but Moderate on the Ghosts MP menu. I forwarded 3075 and 3076, and magically, everything worked beautifully! :-)

    How do you turn off the Num Lock key? - Microsoft Community

    neverendingnoise say:

    How do you turn off the Num Lock key?

    Yes I have googled this! I have spent two hours googling this!

     

    I have followed instructions on editing my registry and changing BIOS settings. I have come to the conclusion that these solutions only change the starting setting for Num Lock. But I want Num Lock completely disabled. When I press Num Lock I dont want anything to happen! I dont want the 6 key on the num pad to point right.... I dont want the toggle!

     

    What do I do?

    teen geek say:

    Hi neverendingnoise,

    You might have tried this before, but here's how to disable it through the registry:
    1. Click Start, click Run, and type regedit to open the Windows registry editor.
    2. In the registry, open the below folders.

      HKEY_CURRENT_USER\
      Control Panel\
      Keyboard\
       
    3. Within the keyboard folder, you should have a string value named "InitialKeyboardIndicators" with a value of 0, 1, or 2. Below is the explanation of each of these values.

      0 = Num Lock is turned OFF after the logon.
      1 = Disable Num Lock.
      2 = Numlock is turned ON after  the logon.

    Hi,

     

    Did the laptop come preinstalled in German language?

     

     

    let's check which is the base language on the laptop. Try these steps.

     

    a) Hit Windows key + X

    b) Open Command prompt (Admin)

    c) Enter the following command:

     
    dism /online /get-intl

     

    d) Hit Enter.

     

    This will tell you the base language of your Windows 8 (Default system UI language).

     

    If the language is German, you will need to install language pack for English. If it lists English as the base language, you will need to reinstall/remove the language packs/ language options
