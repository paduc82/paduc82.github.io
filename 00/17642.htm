jim gordon mvp say:

.doc is a proprietary Microsoft file format. It became the defacto standard for word processing documents, but open source advocates opposed it. I think the last version of Microsoft Word to use this format by default was Word 2004.

.docx is a plain text file format (zipped).

.odf is another plain text file format advocated for by open source enthusiasts, but is rarely encountered in the "real world."

.docx is the most modern of the three and is the format most people use today. This format supports more features than the other two.

Microsoft Word defaults saving to .docx, but supports the other formats, too.
