a. user say:

Hi Sue,


This error message might occur due to the fact that the print queue is there are pending documents to be printed within the queue. It might also be an issue with the print spooler service.


I would suggest you to clear the print queue list and then download and install the latest firmware and driver updates. Test if you are able to print using the printers.

Follow the steps:

Step 1:

Let’s clear the print queue:

Press Windows+X keys and select Control Panel.

Click on Devices and Printers.

Under “Printers,” right-click your default printer and select “See What’s Printing.”  This shows a list of print jobs that are in the queue.

Click “File” and select “Cancel All Documents.” If you continue to see print jobs in the queue, restart your computer.

Alternatively: You can use Command Prompt to stop all the pending documents and reset the print spooler service:

  • Press Windows+X keys and select Command Prompt(admin).
  • Type the following commands and hit Enter after each:

Net Stop Spooler

del %systemroot%\System32\spool\printers\*

Net Start Spooler

Net Start Spooler


Step 2:

Refer to the link to download and install the latest driver and firmware updates:

For Printer HP LaserJet CP2025:

http://h20565.www2.hp.com/portal/site/hpsc/public/psi/swdHome/?lang=en&cc=us&sp4ts.oid=3673581


For printer Photosmart B209a:

You would need to choose the appropriate the correct operating system from the drop down menu and then download the latest driver;

http://h10025.www1.hp.com/ewfrf/wc/softwareCategory?cc=us&lc=en&product=3777182


Step 3:

Start printing again and check if you still receive the same error message.


For reference, you can run the troubleshooter and follow other steps from the link to troubleshoot printer problems:

http://windows.microsoft.com/en-NZ/windows/printer-problems-in-windows-help#fix-printer-problems=windows-8&v1h=win8tab1&v2h=win7tab1&v3h=winvistatab1&v4h=winxptab1

a. user say:

Hi Sue,

Try resetting the print spooler service by navigating to the location and check if it fixes the issue. Follow the steps:

  • Press Windows+R keys to open Run.
  • Type “Services.msc” and press Enter.
  • Click on Print Spooler service and right click on it.
  • Click the "Stop" button to stop the Print Spooler Service if it is running.
  • Check whether the Startup type is set to Automatic. If not, then set it to Automatic.
  • Go to the following location and delete all the pending documents from the “Printer”. C:\WINDOWS\system32\spool\PRINTERS”
  • Delete all the queued print jobs (select and delete all the files) in the folder so the folder is devoid of pending print jobs.  The folder should be empty when you are done.
  • Restart your system and check the Print Spooler Service again to be sure the Print Spooler Service restarts automatically on reboot (since the Startup type is set to Automatic).
Also, I would suggest you to download and install the latest printer driver first and check if that fixes the issue. If not, then, download the latest firmware updates.

slance310 say:

Hi,

 

I know you mentioned starting the Print Spooler in "SERVICES", but also make sure Bidirectional Support is Enabled... If this doesn't help, you will probably need to uninstall\reinstall.

The "Print Spooler" can be found in Control Panel... Administrative Tools... Services.

To find Bidirectional Support, right click on the printer in Devices and Printers, select Printer Properties, PORTS tab, and it will be near the bottom...

I have found that the printer needs to be physically unhooked from the computer to change the Bidirectional Support. Otherwise the change won't stick..

If you attempt to reinstall, turn off the Print Spooler, REMOVE your printer from Devices and Printers, and unhook it from your computer, and uninstall the software\drivers.

When reinstalling, make sure the Print Spooler is turned off... The reinstallation should turn it back on... It is supposed to start automatically.

 

You might also go to Control Panel... Administrative Tools... Print Management, and delete any drivers in the driver folder other than Microsoft, before reinstalling.

 

BTW... do you have any kind of RESET button on the actual printer... if so give that a try.
