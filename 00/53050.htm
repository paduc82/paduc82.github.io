DHCP Reservation

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. If prompted to login enter “admin” as the username and “admin” for password.
  4. Select DHCP Server from the menu on the left.
  5. Select DHCP List&Binding from the sub menu.
  6. Enter the desired last digit in the IP Address box.
  7. Enter the Xbox’s MAC Address.
  8. Select the Add button.

omniplex say:
is it necessary to install a third party AV software to solve the issue??  
There is a NAP-agent service (network access protection), maybe you simply have to start it, check it under "services".  Under the requirements tab (right click, properties,  dependencies) it lists RPC as required.  If one of these two services is shown as "disabled" instead of "manual" or "automatic" it can't work as expected.

NAP is an Internet Protocol, in essence it checks that a client (you) has the security patches required to use a server (your office).  Otherwise the server is supposed to show you a page with working links to whatever it takes to get the required "posture".  Microsoft wouldn't offer the NAP-agent if their own MSE (security essentials) can't do that, you don't need third party software for this.

But the server might insist on a third party solution working with its own "NAP" ideas.  Ask the system admin at your office how that works.  I've never used NAP and read an Internet Draft in the "last call" (before publication as RFC) years ago, so actually I don't know what I'm talking about ;-)

Booting from GPT is only supported for 64 bit editions of Windows 7 (and Vista) on UEFI based systems (UEFI – Unified Extensible Firmware Interface).

 

 

If your computer does not support UEFI, delete all the hard drives partitions and create a new partition:
