Problem with Favorites in Internet Explorer - Microsoft Community

janva say:

Problem with Favorites in Internet Explorer

The favorites won't work in IE. I have IE9, but it didn't work in IE 8 either. I cannot save Favorites, organize Favorites or import Favorites. Please help
don varnau say:

Hi,

I suspect that you (or someone else) copied the Favorites folder from another computer into Windows 7. If so, you have a permissions problem- Windows 7 doesn't want to work with a Favorites folder that was created on another computer.

This procedure has helped others...

1. Create a new folder on the desktop named faves.

2. Right-click the Favorites folder at [C:\Users\your account name] and Move it to that faves folder. Click "move" or "replace" to any desktop.ini questions.

3. Open the relocated Favorites folder and try (double-click) a few favorites to verify that they work.

4. Restart the computer.

5. You should end up with a new, nearly empty Favorites folder at [C:\Users\your account name\Favorites]

6. Copy favorites (not folders) from the Favorites folder in the faves folder to the new Favorites folder. If you had created subfolders in the Favorites folder, recreate those, then copy favorites into them.

7. If everything is working OK in a few days, delete the extra copy of the Favorites folder.

Don

box 1 selects the color

box 2 selects the quantity of add ons

box 3 selects the material type

based on these 3 criteria excel would go look up the price in a table which has a column of color, qty of add on's, then price based on the material type (4 columns total).  There are 2 types of material Mild steel & Stainless. there is a price for each.  I want to generate a formula that will pull the price from the table and put it in a summary sheet on another tab.  Any ideas?

jllatham say:

As you are using Excel 2007, you can use the SUMIFS() function.

Assuming your table is on Sheet1, columns A:D and goes from row 1 down to row 100, and the drop downs are on Sheet2, A1, B1 and C1 then:

=SUMIFS('Sheet1'!$D$1:$D$100,'Sheet1'!$A$1:$A$100,A1,'Sheet1'!$B$1:$B$100,B1,'Sheet1'!$C$1:$C$100,C1)

But if you have any users with older versions of Excel, SUMIFS() won't work and you should use SUMPRODUCT() for compatibility with the older versions of Excel, as:

=SUMPRODUCT(--('Sheet1'!$A$1:$A$100=A1),--('Sheet1'!$B$1:$B$100=B1),--('Sheet1'!$C$1:$C$100=C1),D1:D100)
