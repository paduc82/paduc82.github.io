*******Left click the text :"Download Latest Version"...

just wait for the download screen to open*******

go to...Tools / Preferences / Video button / Video
Snapshots section.

create a new folder on your desktop and select it)

At...Format...choose JPEG or PNG.

capture.

When you install the printer driver and get this message: "Installation has been interrupted" how do you do it?
You can solve the problem:

1) First hold down the Windows key and press key R

2) Type services.msc and press Enter
3) In the Service window: Click on any line and type the letter P
4) Find the Print Spooler service
5) Double click or right click on Print Spooler service and select Properties
6) In the Startup type listbox, select Automatic, and then click Apply.
In the Service status line click the Start button and wait a few seconds, then click OK.
7) Now you can get back to installing printer drivers. Good luck! Please watching my video solve this problem
