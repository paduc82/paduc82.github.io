To answer my own question (after help from a friend):

The "Network Password" is the user password that belongs to the user of that computer.

Additionally, new computers may have a default user profile pre-installed, but it may not have a password set. To create a network it is necessary to set a password for the computer and the user profile. That becomes the "Network password".
