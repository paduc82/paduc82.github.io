jay freedman say:

Although Microsoft would REALLY, REALLY prefer to have you log in with a Microsoft account every time you use Office 2013, there's an almost-hidden option in the setup to let you opt out. You can see it at the bottom of the first screen shot in http://www.companionlink.com/support/kb/How_to_configure_Office_2013_PC_Install_without_using_Office_365_Cloud. Once you set up Office to use a local account, you don't have to be connected to the Internet (except to send/receive email in Outlook). There are a few features that won't work without the Microsoft account (for example, synchronizing settings between your desktop and laptop, and collaborating with other users on the same document), but even the ones that are convenient really aren't necessary.

This is not to be confused with Office 365, which includes Office 2013 programs plus OneDrive and other programs. Because that's a subscription model, Office 365 requires a connection at least once every so many days to verify that the subscription payment is current. It also can get updates more frequently than the once-a-month Patch Tuesday updates that Office 2013 gets.

You don't have to uninstall Office 2007 first before installing Office 2013. It's true that you can't have more than one version of Outlook installed, but the Office 2013 installer knows how to uninstall Office 2007 for you -- and that doesn't remove your which of the 2007 programs you want the installer to remove. If you uncheck Excel, for instance, you can have both Excel 2007 and Excel 2013 simultaneously; you can't uncheck Outlook.

After Office is installed, go into the Options dialog of any of the programs and go to the Save page. Check the option "Save to computer by default" and make sure that a suitable folder is listed in the "Default local file location" box. That will then be where the Save As dialog defaults to, instead of a folder in OneDrive. You choice in any Office program will then show up in all the Office programs.

Equal numbers treated like uneven

Type the following numbers: 3351.45; 6.38; 918.38; 281.89 and 302.8

Make a cell with the sum of the above numbers. The result will be 4860.9. Type in another cell manually the number 4860.9. Finally go to another cell and subtract the two number by multiplying the result by 1 trillion. You now see that the result is 0.9094.

bernard liengme say:

=A1-B1 returns 0

=(A1-B1) returns 9.09495E-13

because Excel in some cases tries to 'anticipate' the round off error

=IF(A1-B1=0;"Yes";"No") returns No

=IF(A1=B1;"Yes";"No") returns Yes

Never test is two numbers are equal but test if the differ by something more than a tiny amount

=IF(abs(A1-B1)<1e-10"Yes";"No")

Or use the ROUND function

=IF(ROUND(A1-B1,12)=0,"YES","NO")
