NO signature on reply

talking moose say:
Sorry, that is not a feature supported by Outlook for Mac.

If you are using a POP account then you can do the following to work around this issue:
  1. Create a new "send only" POP account under Outlook menu -->  Preferences... --> Accounts. Configure this account identically to your other POP account and add "No Signature" to the end of the account name.
  2. Under Outlook menu -->  Preferences... --> Schedules --> Send & Receive All, you want to disable this new "No Signature" account from receiving mail. After all, no use in receiving mail again since your first account is already doing so.
  3. By default, this account will not be configured with a signature. No need to apply any signature to it.
  4. Create a new incoming POP rule under Tools menu --> Rules. For all messages coming into your regular POP account you want to "Change Account" to your new "No Signature" account. Save the rule.

As you receive messages the rule will change the account to the "No Signature" account. When you reply Outlook will not append a signature.

Hope this helps!
