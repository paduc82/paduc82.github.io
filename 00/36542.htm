Hi

There is a very common mistake that is easy to make when you are configuring Folders in Windows 7 and later versions of Windows.

When you open your Pictures Folder, you may actually be opening the Pictures Library, because they look similar.

A Library is designed to display the contents of any other folder on the system. Multiple folders can be added to any Library.

When you open a Library, if you look at the Top of the folder window, you will see a link that shows "Includes: x locations" where "x" is a number. If you click that link, you will see a dialog box similar to the following.

This dialog displays all of the 4 folders that have been added to the default Pictures Library.

You can easily remove any of these folders. Simply select the folder you want to remove and click the Remove Button. This does not have any effect on the files in that folder, or on the library. You just will not see all of the files in the Removed folder when you have the library open.
