Imort text from txt file, divide to columns withaout wizard, by default settings

Hello!


I need import text from txt file, I found following solution, but that open text import wizard, is possible write script with out wizard

Sub ImportTextFile()

       Dim DestBook As Workbook, SourceBook As Workbook
       Dim DestCell As Range
       Dim RetVal As Boolean

       ' Turn off screen updating.
       Application.ScreenUpdating = False

       ' Set object variables for the active book and active cell.
       Set DestBook = ActiveWorkbook
       Set DestCell = Range("A2")

       ' Show the Open dialog box.
       RetVal = Application.Dialogs(xlDialogOpen).Show("*.txt")

       ' If Retval is false (Open dialog canceled), exit the procedure.
       If RetVal = False Then Exit Sub

       ' Set an object variable for the workbook containing the text file.
       Set SourceBook = ActiveWorkbook

       ' Copy the contents of the entire sheet containing the text file.
       Range(Range("A1"), Range("A1").SpecialCells(xlLastCell)).Copy

       ' Activate the destination workbook and paste special the values
       ' from the text file.
       DestBook.Activate
       DestCell.PasteSpecial Paste:=xlValues

       ' Close the book containing the text file.
       SourceBook.Close False

roger t.yokubaitis say:

Upgrading PST file from earlier version to 2010 version

I have Office 2010 Premium, including Outlook 2010.  My earlier PST files do not hold as much data as the current 2010 PST files and I need to transfer ALL of my data from my latest old PST file to a brand-new 2010 PST file.  Does anyone know how to be absolutely certain to transfer every last bit of data from the older, smaller capacity PST files to the newer, larger capacity PST files such as are included with Outlook 2010?
karl timmermans [outlook mvp 2012-2018] say:

The suggested approach presumes the following in your question


- you are referring to PST files associated with existing POP accounts - if not adjust accordingly where applicable (i.e. if PST file is not associated with an email account - then nothing to change in that regard)


- your current PST file shows as "Outlook Data File (97-2002)" as the format

(right click the root of the PST file in the navigation pane --> Properties --> Advanced --> see description beside Format


- export this PST file to a new PST file ensuring that you export from the Root of the original PST and include all sub-folders


- add the newly exported PST file to the profile by opening it (File --> Open --> Outlook data file (review it to ensure that you have correctly exported all the data)


- point the email account(s) that are pointing to the old PST file to the new one

File --> Account Settings --> email tab --> highlight each email account involved --> click on the <Change Folder> button and point it to the Inbox located the new PST file


- ensure that the newly created PST file is designated as the default for the profile instead of the old one (if applicable)

File --> Account Settings -- Data File tab


- when done, remove the old PST file from the profile
