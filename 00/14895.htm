Hi Andrey,

Since this issue happened only to you, please try disabling the forwarding feature by using PowerShell. Below are the detailed steps:

1. Connect to Exchange online using PowerShell.

2. Run the command:

Set-Mailbox -Identity <UserMailboxEmailAddress> -DeliverToMailboxAndForward $false -ForwardingSMTPAddress $null

If you are not an admin of your organization, please contact the admin of your organization to run the PowerShell command for you.
