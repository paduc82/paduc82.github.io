Hi Daniel,

For the option 1, if you don't use Safe Links (I understand "do not rewrite" option?), for the unknown potential malicious URLs will be rewritten and checked again a list of known malicious links when use clicks on the link (you can turn on it). 

For the option 2, the URLs in "do not rewrite" option should be the URLs you trust for your organization, it's also like a whitelist list. Even if you enter your internal phishing link in the list, users can also click the link to access the website. If you don't add the internal phishing link in the list, it will be checked in a list of known malicious links.

For the option 3, we don't have a option that is a safe link with rewrite.

Is there a method of creating a date axis which includes the day/month but excludes the year from my date data?

shane devenshire say:

Hi,

 

One way to do this is enter all the dates for a given year in one column, pick a leap year.

 

In an adjacent column enter the formula like =MONTH(A2)&"/"&DAY(A2)  - here I put all the dates in column A and the dummy date field uses the formula is in column B

 

Enter the values in columns C and D.  Say 2012 data in column C and 2013 data in column D.  The titles for these columns would be 2012 and 2013.

 

I'm not sure of the validity of a multi year trendline but

 

In column E enter

 

=IFERROR(AVERAGE(C2:D2),"")

 

And title in Average. 

 

Plot the range B:E.  Select the Average series and generate a trendline.  Then format the Average series to Markers Options = None and for Line color = No line
