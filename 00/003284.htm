andreas killer say:

I wonder if there is a function in VBA Excel which permit us to delete, let say the second line and to save it without having to read the third line?

No.

Many people think that if they delete only one line that the file is modified only at this point, but that's wrong, a computer works different.

When ever you open a text file, the whole file is read into memory (means you have a copy in RAM), you can modified the file there (with your editor) and when you save it, the whole file is written back (not only a part).

In the opposite, it is possible to read only a part of a file with VBA...
