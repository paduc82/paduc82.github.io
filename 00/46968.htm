Did you install or Upgrade Windows 10. If part of the free upgrade program then it would only install the bit count version that matches the original operating system. If you had Windows 7/8 32bit then the Windows 10 version must be 32bit. 

This is irrespective of the capabilities of the processor itself as most processors sold in the last 10 years have been 64bit capable.
