shah---01 say: Hi all,

I too got an N80 recently and after struggling for days trying to set up the Wifi access I finally have web browsing and uPnP streaming working.

I have a NetGear DG834G router.

Originally I tried WEP 64-bit, WEP 128-bit and WPA-PSK (TKIP) encryption. Each time I'd set up the router first and then set up the connection on the N80.

But, everytime I tried to access the web on the N80, I'd get a blank page and then a "Gateway not found" error.

Eventually, after hours of trial and error, I changed the channel number that the router has for the wireless connection and lo and behold the N80 now connects.

I tested it with WEP 64-bit, WEP 128-bit and WPA-PSK (TKIP) connections, all worked okay.

From me experience, I can say that it is important to set up the conneciton on the router before setting it up on the N80.

And, I did find that the N80 was a bit quirky about the WEP keys that it liked. The following ones I tested (not my actual keys) and they worked fine :-

64-bit :- 0A0B0C0D0E0F

128-bit :- 0A0B0C0D0E0F1A1B1C1D1E1FAB

The settings on my Netgear DG834G router that I can remember (I'm at work so don't have access to the router) are :-

Network name: Banana
Broadcast SSID: No
Wireless mode: g
Encryption mode: WPA
Key: Pineapple

I have access restricted by MAC address and have added the N80's MAC to the access control list.

My N80 setting for my WLAN access point are (Tools->Settings->Connection->Access Points) :-

Connection Name : Home
Data bearer : Wireless LAN
WLAN netw. name : Banana
Network status : Hidden
WLAN netw. mode : Infrastructure
WLAN security mode: WPA/WPA2

WLAN security sett.

WPA mode : Pre-shared key
Pre-shared key : Pineapple
TKIP encryption : Allowed

Hope some of this helps.
