This regular formula returns the first name listed:

This array formula, committed with CTRL+SHIFT+ENTER (instead of just ENTER)
begins the list of subsequent unique, non-blank names:
B3: =IF(SUM(($A$2:$A$20<>"")*ISERROR(MATCH($A$2:$A$20,$B$2:B2,0)))<>0,INDEX($A$2:$A$20,MATCH(1,--ISERROR(IF(ISBLANK($A$2:$A$20),0,MATCH($A$2:$A$20,$B$2:$B2,0))),0),1),"")

To answer this, please be informed that when using IMAP, your Inbox is stored on the mailserver whereas the Sent Items are still stored locally (unless otherwise specified).
