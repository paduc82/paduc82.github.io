Sub MyTest()

Const strInformation As String = "This is the info line"

    ' -----------------------------------------------------------------------------
    ' Document title and information

    Set oDoc = ActiveDocument
    Set oRange = oDoc.Range
    oRange.Collapse wdCollapseStart
    oRange.InsertParagraphAfter

    With oRange
        .Text = strTitle & vbCr
        .Font.Size = 24
        .Font.name = "Arial"
        .Font.Bold = True
        .Font.Color = RGB(18, 75, 122)

        .Text = strInformation & vbCr

        .Font.name = "Arial"
        .Font.Size = 10

steve rindsberg say:

OK, so you're getting shapes from the old theme on the masters/layouts when you apply a new theme.

This, believe it or not, is deliberate.  It happens when you (or someone) has added shapes to the master/layouts.  PowerPoint marks these as "user drawn" and doesn't delete them when applying a new theme, because it has no way of knowing whether you really want them to stay put or to go away with the other shapes that are really part of the theme.
