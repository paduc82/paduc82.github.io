The point is that each word can have its own proofing language setting. That setting continues (like Bold or Italic formatting) when you continue typing from a word so formatted, but this is invisible.

If you turn on Automatically Detect Language in your proofing options, in my opinion, the number of such problems will increase, rather than decrease. In my experience, Word is not that good at detecting languages. I've kept this turned off for many years, so my experience may not be worth that much. Also, I seldom work with multiple languages (other than Latin) in a document.

Text copied from other documents or from a web page will bring such language formatting with it unless pasted as plain text.

If you installed Word using one language and then changed it, you definitely want to check the styles in your normal template and any AutoText entries.

charleskenyon (ia) say:
Here is a macro (in three variations) that changes the language settings for all styles in a document. 
Sub StyleEnglishUK()

'   Written 21 March 2018

'   Charles Kenyon

'   Intended to set all styles to English UK, proofing, not automatitically update

'

    Dim aStyle As Style

    On Error Resume Next

    For Each aStyle In ActiveDocument.Styles

'        aStyle.AutomaticallyUpdate = False ' probably do not want this for TOC styles - future work

        aStyle.LanguageID = wdEnglishUK

        aStyle.NoProofing = False

    Next 'aStyle

'    ActiveDocument.UpdateStylesOnOpen = False ' For information on using this line, see:

'       http://www.shaunakelly.com/word/sharing/willmyf...

    On Error GoTo 0

End Sub

jeeped say:
... when I hold down control and hit delete, it deletes whole words at a time, ...

Just to clarify, CTRL+Delete (e.g. DeleteWord) does not always delete the entire word. What is does do is delete from the text insertion point (e.g. blinking cursor) to the next blank space. So, yes, it will delete the entire word if the TIP is at the beginning of the word, but only half of the word if the TIP is in the middle of the word.

In any event, you should check the Keyboard Shortcuts within Word Options to see how the keystroke combination is assigned. Click the Office button then the Word Options button in the lower right. Select Customize from the list on the left, then click the Customize... button beside Keyboard shortcuts: at the bottom.


Figure 1.0 - Customize Keyboard: DeleteWord

  1. In this Customize Keyboard dialog, select Commands Not in the Ribbon from the Categories: list, then DeleteWord from the Commands: list.
  2. If there is anything (even Ctrl+Del) in the Current Keys: list, select it and click Remove at the bottom.
  3. With the cursor within the Press new shortcut key: text box, press CTRL+Delete, then click Assign at the bottom.
  4. While you are here, you may want to check the settings for DeleteBackWord. The keys should be CTRL+Backspace. This deletes from the text insertion point to the previous space.
  5. Click Close in the lower right. Remember to click OK to save your changes and close Word Options.

Did that set the keystroke shortcut correctly?
