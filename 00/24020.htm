a. user say:

You could consider VBA along the following lines (needs more error checking, e.g. what if the ActiveDocument is the Chap1.doc, what if the bookmark isn't there, what if the bookmark does not contain a number):

Sub getStartPageNumber()
'
' Get the starting page number for the active document's first section from 

' the specified file

' Modify these Consts as necessary

Const SourceDocFullname As String = "c:\Chapters\Chap1.doc"
Const SourceBookmakeName As String = "Chap1Pages"
'
Dim docActive As Document
Dim docChap1 As Document
Dim lngCount As Long
Set docActive = ActiveDocument
Set docChap1 = Documents.Open(SourceDocFullname)
lngCount = CLng(docChap1.Bookmarks(SourceBookmarkName).Range.Text)
docChap1.Close savechanges:=False
Set docChap1 = Nothing
docActive.Activate
Set docActive = Nothing
With ActiveDocument.Sections(1).Headers(wdHeaderFooterPrimary).PageNumbers
  .RestartNumberingAtSection = True
  .StartingNumber = lngCount
End With
End Sub

Word MVP Graham Mayor has some instructions for installing macros here:

Beyond that, you'll probably need to consider when and how to run this macro - I suggest you have a look around a little for more info. on that.

You could modify this VBA so that you pulled the page count directly from Chap1.doc rather than from a bookmark inside it. However, if you are already putting the correct value in the bookmark, it may be best to stick with that.

You have 2 problems.

Try this.

Sub Workbook_Open()
 Dim DateCompare As Date       '=Today() on a summary page
 Dim DateCompare2 As Date     '=Automatically created date on a different worksheet within the workbook
   
     If (Range("DateCompare").Value = Range("DateCompare2").Value) Then
         Exit Sub
     Else
         MessageBox
     End If
 End Sub

 Sub MessageBox()
 Dim OutPut

rich007a say:

Hi,

No way to do that built-in to office.  But you could run this macro in Word AFTER you have pasted it as one single table:

Sub EveryOtherRowToText()
' Place cursor in a table before running.
' This procedure will convert every other line from row 2 to text.
'
    Dim tbl As Table
    Dim i As Long, lnRowCount As Long
   
    If Not Selection.Information(wdWithInTable) Then
        MsgBox "Can only run this within a table.  Place cursor inside a table."
        Exit Sub
    End If
   
    Set tbl = Selection.Tables(1)
   
    lnRowCount = tbl.Rows.Count
    If lnRowCount Mod 2 <> 0 Then lnRowCount = lnRowCount - 1
   
    For i = lnRowCount To 2 Step -2
        tbl.Rows(i).ConvertToText Separator:=wdSeparateByParagraphs, NestedTables:=True
    Next i
End Sub

Copy the code above.

In Word press Alt+F11 to open the VBE.

In the top left corner (in the pane that looks like a folder tree) right-click on Normal and click Insert > Module.

Paste the code on the right.

In the VBE, click the Save button to save the Normal template.

Close the VBE.

Paste your data from Excel into a Word document.

Place the cursor anywhere within the table.

Do View tab > Macros > select "EveryOtherRowToText" > Run

Job done?

Cheers
Rich

Sub SplitEveryRowAndAddText()

    Dim tbl As Table
