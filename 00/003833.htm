greginmich say:

I accidentally uninstalled windows defender. How can I get it back?

It isn’t possible to accidentally uninstall Windows Defender – but if you installed a third-party AV app, then that would have turned off Windows Defender, as Le Boule indicated. To check the operational status of Windows Defender, type “Defender” into the Cortana search bar and then hit the enter key when “Windows Defender” is selected in the search list. If you’ve installed a third-party AV app and this is what you see, then everything is fine:

But if the Windows Defender UI comes up with this message, then the installation of the third-party AV app failed to disable Windows Defender in its registry key, and this screen could easily fool you into thinking that Windows Defender should still be in an operational state:

That’s emphatically not the case. Windows Defender is not in a state where it can be turned on, and you shouldn’t be trying to turn it on. This is a very common issue, but it’s not a serious one – so fixing it is optional. It’s also possible that this issue might be resolved automatically with the upcoming Windows 10 Anniversary Update, where Windows Defender actually will be allowed to run alongside any third-party AV app, but strictly for the purpose of running Automatic Maintenance scans in order to provide supplemental protection. This Limited Periodic Scanning feature is described in more detail here:

http://answers.microsoft.com/en-us/protect/wiki/protect_defender-protect_scanning/limited-periodic-scanning-in-windows-10-to-provide/3dcfc8dd-36b4-43f2-9a0a-732e43191cf3

Nevertheless, this issue with Windows Defender not being turned off in its registry key can easily be fixed by turning it off with Group Policy:

http://answers.microsoft.com/en-us/protect/forum/protect_defender-protect_updating/definition-update-for-windows-defender-kb2267602/ea933090-9301-4c27-b915-1d3ba92b301f
