Hi KateLumsdaine,
I hope you are having a good day. Here's some information that can help you.

To open a PDF file without converting it to a Word document, open the file directly wherever it's stored (for example, double-click the PDF file in your Documents folder).

However, if you want to edit the PDF file, go ahead and open it in Word. Word makes a copy of the PDF, converting it to a Word document and attempting to match layout of the original PDF. You always have the original PDF file, in case you don't want to keep the version that Word converts.

Converting from PDF to Word works best with files that are mostly textâ€”for example, business, legal, or scientific documents. But when you open a PDF file in Word, it might not look exactly the way it looked as a PDF.

To edit a PDF, simply open it in Word. This works best with PDFs that are mostly text.

Go to File > Open.

Find the PDF, and open it (you might have to select Browse and find the PDF in a folder).

Word tells you that it's going to make a copy of the PDF and convert its contents into a format that Word can display. The original PDF won't be changed at all. Select OK.

jay freedman say:

If the scanner's software doesn't include Optical Character Recognition (OCR), then the PDF will contain just pictures of the scanned pages. Opening that PDF in Word will also contain just pictures, not editable text.

If you have OneNote, paste the scanned pictures into a OneNote notebook. Right-click a picture and click "Copy Text from Picture" in the menu. That "reads" the picture and puts text on the clipboard, and then you can paste the text into either OneNote or Word. This is very basic OCR; it won't retain the formatting of the original and it may make a lot of errors that will need corrections.

You scanner may include an OCR program in its software bundle. If there is one, try using it. If you need a more capable OCR, use a web search to compare reviews of the many third-party programs that are available.

charleskenyon (ia) say:
Since Word 2013, Word has had the ability to directly open pdf document. However, some pdf documents (from scanners mostly) are really pictures of text rather than the text itself. Just as you can't look under the hood of a picture of a car, you can't edit a picture of a document.

In that case, you need to convert the picture to text. This is a process known as optical character recognition. This is built into Adobe Acrobat and is also in Office OneNote. Most scanner software comes with an OCR component as well. Once you have text, you can edit it with Word.

If you simply want to write on the document (but not in it) you can add a Text Box floating on top of the document layer, whether or not it has been put through the OCR process.

Word documents that have been saved as PDF will not need the OCR process, they retain their text, although not all of their Word structure and formatting.

Finally, documents converted from pdf (or really any other format) to Word can be tough to edit because the conversion process never has a one-to-one matching of how formatting is done. This means that a converted document will seldom be formatted in Word in a way that makes sense. An example is multiple section breaks to change margins, where in Word you would simply change the paragraph indent.

Yes.

2) Is there a way to have just one macro that executes regardless of which optionbutton is chosen?

Yes.

First we must establish a class module, which attaches itself to the events of the buttons.

Insert a class module in your workbook, rename it to "clsButtonHandler" and insert the following code:

--- schnipp ---
Option Explicit

Public WithEvents OptionButton As MSForms.OptionButton

Private Sub OptionButton_Change()
  Dim R As Range
  'Determine the cell in column A in the row of this button
  Set R = Intersect(OptionButton.TopLeftCell.EntireRow, _
    Columns(1))
   'Remarks:
  'This event is called when a button changes, one time for _
    each button in a group.
  'But always the button whose value is False will call this _
    event first!
   If OptionButton.Value Then
    'I assume that the caption is "Yes" or "No"
    R.Value = OptionButton.Caption
  Else
    R.ClearContents
  End If
End Sub
--- schnapp ---

To get this to run, we must identify all the option buttons and create a handler for each and keep him alive.

The best place for this is the Workbook_Open-Event. Place the code below in the module "ThisWorkbook".

And for your first question have a look at Sub SetAllButtonsFalse.

Andreas.
 Option Explicit

Private OptionButtonCollection As Collection

Private Sub Workbook_Open()
  Dim WS As Worksheet, S As Shape
  Dim ButtonHandler As clsButtonHandler
   'Next line will be never execute, except you run this sub a _
    second time
  If Not OptionButtonCollection Is Nothing Then
    'Remove all buttons from our collection
    Set OptionButtonCollection = Nothing
  End If
   'Initialize collection
  Set OptionButtonCollection = New Collection
   'Setup sheet with all those buttons
  Set WS = Sheets(1)
   'Visit all objects in that sheet
  For Each S In WS.Shapes
    'Is it a control?
    If S.Type = msoOLEControlObject Then
      'Is it an optionbutton?
      If S.OLEFormat.progID = "Forms.OptionButton.1" Then
        Set ButtonHandler = New clsButtonHandler
        Set ButtonHandler.OptionButton = _
          S.OLEFormat.Object.Object
        OptionButtonCollection.Add ButtonHandler
      End If
    End If
  Next
End Sub

Public Sub SetAllButtonsFalse()
  Dim ButtonHandler As clsButtonHandler
  If OptionButtonCollection Is Nothing Then _
    Workbook_Open
  For Each ButtonHandler In OptionButtonCollection
    ButtonHandler.OptionButton = False
  Next
End Sub

townfield say:

Les,

As ever in life, there are many ways to skin a cat - what are you really trying to achieve here - rather than what's the approach you've taken?

If you are using Excel Form controls, you can link a cell to the radio button, a group of radio buttons is usually linked to the same cell - if all the buttons are OFF the cell value will be ZERO and if button "N" in the group is ON the cell value will be N.  Rather magically, setting the CELL VALUE to 0 will switch all the buttons off.

You could therefore have the two buttons on each of your 85 rows referring to cells Q1 through Q85.

You could then have a (simple) macro which swicthes off all of the buttons:-

range("Q1:Q85").value = 0

 

To determine the Yes / No status, you could use the following formula on each row:-

=if(Q1=0,"Not chosen",if(Q1=1,"Yes","No"))

 

The above assumes that all linked cells have been set to zero by the above macro, that button 1 means "Yes" and button 2 means "No".

 

Alternatively, if you are only looking for Yes / No / blank answers, you could do this more simply by using data validation:-

1. Write your 85 questions in column A

2. In column B select the 85 rows

3. Select DATA / VALIDATION - Choose Allow "List", in Source type "Yes,No" (without quotes), tick "Ignore Blank", tick "In-Cell drop-down"

This will produce a cell with a drop down arrow, which when clicked will expose the options "Yes" and "No" - all without any complex programming.

 

HTH

Townfield
