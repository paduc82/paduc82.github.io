talking moose say:
Your database is permanently damaged. Youll need to revert to the last version you could open and export your data, create a new identity and import your data.

Launch Outlook with the Option key held down. This will launch the Database Utility application. Choose the latest backup that will still open and use the cog menu button at the bottom to make it the default identity. Quit the Database Utility.

Launch Outlook normally. When you see the Main Window choose File > Export > Outlook for Mac data file. Save this to your desktop or some place where you can find it later.

After the export is done, quit Outlook and launch Outlook with the Option key held down again. Create a new identity and use the cog menu button at the bottom to make it the default identity. Quit the Database Utility.

Launch Outlook normally. Choose File > Import > Outlook Data File and navigate to your saved exported file. This will import your data.

You will need to re-create your accounts, rules and preferences but at least your data will be restored.

Hope this helps!

jay freedman say:
Changing the user name in the Options > General dialog will cause Word to "see" you as a different author and change the comment color.

The quick way to get there is to click the down arrow on the bottom half of the Track Changes button and choose Change User Name. That opens the Options dialog to the right place. Changing as little as one character in the user name will do the job.

Changing the user name back to a previously used name will let you insert more comments in the same color as that "author's" other comments.

An aside for people trying out the Word 2013 Preview: This trick doesn't work there. In that version, comments and tracked changes ignore the user name in the Options dialog and instead use the name associated with your Live ID or Active Directory login.
