Public Function udfWsVlookup(rLV As Range, rTA As Range, lCI As Long, _

How can I make it not get stuck in the loop?

jeeped say:
The Worksheet_Change is an event driven macro that is triggered by a change in the value of the cells. That is why you need to put,

Application.EnableEvents = False

... so that the event macro does not trigger itself. If you want to change (aka Clear) a group of cells and not have the Worksheet_Change macro triggered then you have to turn off the events trigger. So your button macro should read,

Application.EnableEvents = False
Range ("B6:1005").ClearContents
Application.EnableEvents = True

That turns off the event trigger, clears the cells and finally turns the event trigger back on.

On a related note, you should probably not turn off the events within the Worksheet_Change loop. If you are processing more than a single cell then you are unnecessarily turning off events multiple times. Once is enough. You could also check to see if the cell contains a value before trying to make it proper case. Try this,

Private Sub Worksheet_Change(ByVal Target As Range)
  On Error Resume Next
  If Not Intersect(Target, Range("B6:B1005")) Is Nothing Then
    Dim cel As Range
    Application.EnableEvents = False
    For Each cel In Intersect(Target, Range("B6:B1005"))
      If Cbool(Len(cel.Value)) Then _
        cel.Value = StrConv(cel.Value, vbProperCase)
    Next cel
    Application.EnableEvents = True
  End If
End Sub


hansv mvp say:

In the code behind the command button, use

 

    Application.EnableEvents = False
    Range ("B6:1005").ClearContents
    Application.EnableEvents = True

 

This prevents the Worksheet_Change event from being called - you don't need it to run when you clear the cells.

 

By the way, I'd structure the Worksheet_Change code slightly differently:

 

Private Sub Worksheet_Change(ByVal Target As Range)
    Dim cel As Range
    If Not Intersect(Target, Range("B6:B1005")) Is Nothing Then
        Application.EnableEvents = False
        On Error Resume Next
        For Each cel In Intersect(Target, Range("B6:B1005"))
            cel.Value = StrConv(cel.Value, vbProperCase)
        Next cel
        Application.EnableEvents = True
    End If
End Sub

 

There is no need to call Application.EnableEvents each time within the For ... Next loop.
