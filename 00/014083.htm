You can apply a border to text, as opposed to the whole paragraph. Select the text. This is actually a job much better done in Publisher. You can apply a much wider variety of borders to text boxes in Publisher than in Word. But you can still use text boxes in Word. Hello Karen,

You have a couple of options:

Add the percent complete value to the right of the bar.
Add a progress line on the Project summary task.

Both are accomplished through the Bar Styles command in the consolidate project.

To add the percent complete value:
  1. Right click on the Gantt chart to show the shortcut menu.
  2. Select "Bar Styles" from the shortcut menu to show the Bar Styles dialog.
  3. In the list of tasks - select "Project Summary"
  4. Click the "Text" tab in the lower portion of the dialog and select the "% Complete" field from the list where you want the value to appear.

To add a progress line:

  1. Right click on the Gantt chart to show the shortcut menu.
  2. Select "Bar Styles" from the shortcut menu to show the Bar Styles dialog.
  3. Select the bar underneath the "Project Summary" bar.
  4. Click the "Add row" button.
  5. Name the new task Project Summary Progress - for example
  6. Set the appearance as you'd like.
  7. Show for "Project Summary".
  8. From "Task Start"
  9. To "CompleteThrough"
