Style Changes When I Press "Enter"

I can press ctrl+a to select the whole document, select the default style but then when I hit "Enter" at the bottom of the document the style change.
jay freedman say:
When you say the style changes, are you referring to the style applied to the new paragraph where the cursor goes after you press Enter? Or is it the style of the paragraph that you just finished that changes?

If the new paragraph has a different style, that might be because of the way your styles are defined. For each style, there is a property "Style for following paragraph". This determines which style Word applies to the new paragraph when you press Enter. For many styles, such as Normal and Body Text, the following paragraph will have the same style as the one you just finished. For others, such as Heading 1, the following style is a different one, and this is usually deliberate -- it means that you won't have to manually select the different style after pressing Enter.

Open the Styles pane by pressing Ctrl+Alt+Shift+S, and hover the mouse over the name of the style that's applied to the paragraph you just left. The popup shows the definition of that style. If there is a notation in the popup about "Following style", you may want to right-click the style's name, click Modify, and change the entry in the "Style for following paragraph" box.

Sub PrinterSelectionDemo()
 'Display list of printers to user - selecting an item changes active printer - Jim Cone
  Dim bChoice As Boolean
  bChoice = Application.Dialogs(xlDialogPrinterSetup).Show(ActivePrinter)
  If Not bChoice Then
     MsgBox "User cancelled"
  Else
    MsgBox ActivePrinter
  End If
End Sub
