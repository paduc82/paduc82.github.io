jack cook - aka help_line say:
There are two types of files you can use to install a program on your device.

One is an EXE file which you install from your computer.  You attach your device, click on the EXE file and it will automatically install it on your device.

The second is called a CAB file.  When you download a CAB file, you must copy it to your device.  Then use File Explorer to find the file you just copied and click on it.  It will then install on your device.  After it is installed, you can safely delete that CAB file to save memory space.

It sounds to me that you have a CAB file so copy it over to your device, click on it and let it install.

BTW, if you are given an option to install in MAIN MEMORY or on STORAGE CARD, choose the STORAGE CARD.  You always want to preserve the MAIN MEMORY as much as you can.  Most programs (not all) will run perfectly from your storage card.

rich michaels - mvp (greatcirclelearning.com) say:

Headers and Footers are controlled by Sections within the document. So to have the cover and intro page as you described with different Headers and Footers from the remainder of the document, you insert a Next Page Section Break at the end of your Introduction page.

You can have up to 3 different Headers and 3 different Footers in each Section of the document. They breakdown like the following:

Header for the Section

  1. First Page
  2. Primary Page (odd page)
  3. Even Page

Footer for the Section

  1. First Page
  2. Primary Page (odd page)
  3. Even Page

Take a look at Word's Help and what is written there about setting up various Headers and Footers.

Hope this helps

I suspect you may have added the Exchange Server service to your Outlook profile by mistake.  Unless you know for a fact that you have an email account on an Exchange Server, with Outlook closed, go to Control Panel > Mail > Email accounts, select the Exchange Server account and Remove it, then add a new account of the correct type. If Outlook complains about a password and won't let you remove the account, Control Panel > Mail > Show Profiles, click the Add button to add a new profile, then select the old profile and Remove it.  Make sure the new profile is shown under "Always use this profile" and click OK.  You can now either restart Outlook, which will ask you to setup an email account, or use Control Panel > Mail > Email Accounts to add one.
