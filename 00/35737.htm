All desk icons open Media Player - Microsoft Community

riverkat say:

All desk icons open Media Player

Someone else used my desktop, and now every icon I click takes me to Windows Media Player.  How to fix?
jeeped say:

Since you are referring to the icons on your desktop, you are likely speaking about shortcuts (e.g. .LNK files)

I've seen this happen when upon opening a program or document through a shortcut (e.g. .LNK file), you are asked what program to open it with. After selecting Media Player (or some other program) via the Select a program from a list of installed programs, and checking the Always use the selected program to open this type of file (circled in red below), the document or program referenced by the original shortcut opens in Media Player as it should.

The trouble is that now all shortcuts (e.g. .LNK files) on the desktop or throughout Explorer will want to be opened by Media Player and use the Media Player icon to display the shortcut regardless of what the shortcut was originally intentioned to open. This is known as a file association and default icon. The same condition could occur if you performed those 'Open With' actions on a .JPG or even an .EXE, but the .LNK explanation is far more likely for what you've described.

The good news is that a small free utility has been written to undo the .LNK file association that was created. There is also a registry hack to accomplish this, but I recommend the unLNK utility for its ease of use.

An Utility to Unassociate File Types in Windows 7 and Vista
http://www.winhelponline.com/articles/231/1/An-Utility-to-Unassociate-File-Types-in-Windows-7-and-Vista.html

Just download, then use the utility to unassociate the .LNK files. Direction for use are provided on the web page above (the same place to get the utility).
