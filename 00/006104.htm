hansv mvp say:

Mini Toolbar:

  • Edit the contents of a cell, preferably in the cell itself.
  • Select some text.
  • Move the mouse pointer slightly up and to the right.
  • The mini toolbar should appear.
  • The mini toolbar will also appear when you select some text in the formula bar, but if you move over a ribbon button, the mini toolbar will vanish, so it's trickier to use it.

Live Preview:

  • Enter values in some cells.
  • Select those cells.
  • On the Home tab of the ribbon, in the Font group, click the dropdown arrow on the right hand side of the Font button.
  • Move the mouse pointer over the fonts in the dropdown menu, but don't click.
  • The cells within the selection should take on the font you're hovering over.
  • The same holds for the Font Size dropdown, the Font Color dropdown and the Fill Color dropdown.
