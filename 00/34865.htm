digitalamoeba say:

Windows 8 also doesn't include proper working A2DP support and Microsoft has tried to block install of Windows 7 Broadcom Bluetooth drivers.

 

I was having problems which meant that a Samsung HS3000 Bluetooth wireless headset was no longer working correctly with a HP Pavilion G6 laptop after upgrade from Windows 7 64bit to Windows 8 64bit

 

The workaround solution I found was to

 

1) Download and attempt to install the latest HP Broadcom drivers for my HP laptop under Windows 7 64bit (sp52717.exe) from http://ftp.hp.com/pub/softpaq/sp52501-53000/sp52717.exe via this HP support page http://h10025.www1.hp.com/ewfrf/wc/softwareDownloadIndex?softwareitem=ob-93400-1&cc=uk&dlc=en&lc=en&os=4063&product=5116748&sw_lang=

 

2) The initial automated install will extract files to C:\SWSetup\SP52717 by default but then fail during install

 

3) Make sure that you have fully uninstalled any previous paired Bluetooth devices from within Control Panel\Hardware and Sound\Devices and Printers list

 

4) Consider whether you want to reboot just to make sure things are reset and clean then go into C:\SWSetup\SP52717\Win64 and manually run Inst.exe (not the main Setup.exe as that's blocked)

 

5) After install reboot to make sure drivers are fully installed

 

6) On restart go to Desktop and then Control Panel\Hardware and Sound\Devices and Printers list > Click 'Add device' and make sure your Bluetooth device is in pairing mode so it can be found.

 

7) During install you will likely get one or more crash reports while a discovering Bluetooth services panel is onscreen. Give each a little bit of time and then clear/close them leaving main panel in front.

 

8) After a while of no apparent activity right click Start bar Volume icon and choose 'Playback devices' - hopefully you should now see expected 'Bluetooth hands free audio' or equivalent new device in your playback devices list :-)

 

9) Do one further reboot to clear Bluetooth services panel and hopefully find everything working normally as expected - i.e. Bluetooth headset should get priority when it's switched on, otherwise sound comes out of laptop speakers

 

Hope that helps others...and that Microsoft and Broadcom get round to releasing full A2DP support drivers for Windows 8 soon!
