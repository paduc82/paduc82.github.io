scottgem say:

Don't do this in the Union. Create a new query that uses the Union as the source and set the criteria there. 

Then, for you education, look at the query in SQL View to see how to create a WHERE clause. 

ken sheridan say:

If you are not confident about writing WHERE clauses within the UNION query then you have two options:

1.  Create each part of the UNION query in design view, adding the criteria to both.  You don't have to save these, just switch to SQL view in each case and copy the SQL statements.  Then paste these into a new query in SQL view with the UNION operator between them.  You only need the one terminating semi-colon though, at the very end of the UNION query.  Save and use this query.

2.  Create the UNION query without criteria and save it.  Then base another query on the UNION query and add the criteria just once in design view.  Save and use this second query.

PS:  If the two parts of the UNION query cannot return duplicates, or if it can, you don't need to suppress the duplicates, then use the UNION ALL operator rather than the UNION operator.  It will perform faster.

Adnan Hamza,

With regard to your main issue. As I understand you want to insure that all tasks start and finish the same day (i.e. no tasks slip over to the following day). Number1=IIf(day([Start])<day([Finish]),1,0)

Alternatively, instead of using styles to create the illusion of a box, you could just draw a text box and adjust word wrapping so that surrounding text moves around the text box, but the text and the text box itself stay in place.

 

Step 1: Go to the Insert Tab, click on the Text Box menu, and select "Draw Text Box" from the menu.

Step 2: Draw your text box, typing in any text and formatting as you so desire.

Step 3. Click on the edge or border of the text box to select the entire text box. You should see a "Drawing Tools" contextual tab appear on the ribbon.

Step 4: In the Arrange Group on the Drawing Tools | Format tab, click on the "Wrap Text" menu and select "Square."

Step 5: Position the text box where you'd like it to appear in your document.

I guess I am a bit confused by what you mean when you say you want the text to "move with other text in [your] document."

Step 2.  Give your new style a name and click on the "Modify..." button.

Step 3. Select the Style Type: "Linked (paragraph and character)."

Step 4. Towards the bottom of the Create New Style from Formatting dialog box, select the Format button in the lower left corner of the window and click on "Border."

Step 5. In the Borders and Shading Dialog box, click on the "Box"  button in the far left hand side of the window to apply a border to the paragraph and click OK.

Step 6. Click OK to close the Create New Style from Formatting dialog box and to apply the new style to your selected text.

 

For whatever text you add, the border will appear to "lengthen" or "grow" with the added text and paragraphs.

To stop "the box" from "growing," simply apply a different style (such as the normal style).
