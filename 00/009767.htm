bob jones aka: cybertaz say:

To go to the beginning of a line within a cell use the Home key, use End to go to the end of the line.

Use the Command key with Home/End to go to the start/end of multi-line cell content.

If you have something other than a full keyboard [such as a laptop or Apple wireless keyboard] you may need to use the fn key in conjunction with the appropriate Arrow key.

The consideration is that the paradigm you refer to is for document navigation, but a cell is not a document. Those keystrokes work as per the paradigm if you are in Ready Mode rather than Edit or Enter Mode... the worksheet is the document :-)
