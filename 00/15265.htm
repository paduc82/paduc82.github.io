tushar mehta say:

You cannot have two (or more) consecutive Else's.  The structure of an If statement is

If condition-1 Then
ElseIf condition-2 Then
...
Else
    End If

where the ElseIfs and Else are all optional

The intent is to look through the each of a series of cells-- the first IF is to select the top cell, based on the top border, of a range later to be merged.

Each subsequent, nested IF is to look for the bottom border. Once it find the border on the bottom of the range, it merges the rows from topcell to botcell.

Yes, there is a second ELSE, outside, to handle a missing top cell.

END IF

suzanne s. barnhill say:

You just need a single list. As Doug has said, detailed instructions for setting up numbered headings can be found at http://www.shaunakelly.com/word/numbering/numbering20072010.html, but this is really simple if you just start with the list style that displays 1 Heading 1, 1.1 Heading 2, 1.1.1 Heading 3. This list is already linked to the nine levels of headings, so all you have to do is type "Chapter" before the 1 in the "Enter formatting for number" box, and you're all set. This will give you:

Chapter 1 (Heading 1)

1.1 (Heading 2)

1.2 (Heading 2)

1.3.1. (Heading 3)

1.3.1.1 (Heading 4)

1.3.2 (Heading 3)

1.4 (Heading 2)

and so on

When you start Chapter 2, you apply Heading 1, which will automatically be numbered "Chapter 2," and all the subheads (Heading 2, Heading 3, and so on) will follow suit (2.1, 2.1.1, and so on).

Whether you use styles or outline levels (or both) for the TOC, you'll get a TOC that represents this list structure. As Doug said, you modify the TOC formatting by modifying the TOC styles. By default, they are set to update automatically, so changes you make to the formatting of one entry at a given level should be automatically propagated to all entries at that level, but I have found that, while this works for paragraph formatting, it is less effective for font formatting. For that you may actually have to go to the Modify Style dialog for that style. Doug's method will work if you haven't inserted the TOC yet, but it's more problematic with an existing TOC. For that:

  1. Open the Apply Styles dialog using Ctrl+Shift+S.
  2. Type TOC 1 (or whatever level you want to modify) and click Modify....
  3. In the Modify Style dialog, make the desired changes and click OK.
