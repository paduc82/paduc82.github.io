gord dibben say:

Assuming you have A1:A10 with CF to color the cells.

Copy those cells, paste to any area then run macro on the pasted cells.

Sub CopyOnlyFormatCondionColor()
    Dim c As Range
    Dim x As Long
    On Error Resume Next
    For Each c In ActiveSheet.UsedRange. _
            SpecialCells(xlCellTypeAllFormatConditions)
        x = c.FormatConditions(1).Interior.ColorIndex
        c.FormatConditions.Delete
        c.Interior.ColorIndex = x
    Next c
End Sub
