average loan balance of a term loan without an amortization table? - Microsoft Community

orfishome say:

average loan balance of a term loan without an amortization table?

I have used the AVERAGE function in conjunction with an amortization table to figure the average lifetime loan balance of an amortizing loan.  That was within a static spreadsheet.  Now, however, I am creating a more dynamic spreadsheet in which I need to figure the same average lifetime loan balance but for multiple loans.  Due to space limitations, I cannot embed a separate amortization table for each loan but need to constrain my calculations to one or two cells.  Is it possible to construct a formula that will "internally" calculate this for me?  Basically, based on inputs of the rate, balance, and period via links from other cells, I need to calculate remaining principal balance for each of the number of periods and then average those to arrive at one AVERAGE figure.  This is essentially an average daily balance calculation but for months instead of days and done "behind the curtain" in one formula.  Any suggestions? 

 

ORFisHome

joeu2004 say: ORF wrote:

Basically, based on inputs of the rate, balance, and period via links from other cells, I need to calculate remaining principal balance for each of the number of periods and then average those to arrive at one AVERAGE figure.  This is essentially an average daily balance calculation but for months instead of days and done "behind the curtain" in one formula.  Any suggestions? 


Try the following array formula [*]:

=AVERAGE(FV(B2,ROW(INDIRECT("1:"&C2))-1,E2,-A2))

***EDIT***    Or this normal formula (just press Enter as usual):

=SUMPRODUCT(FV(B2,ROW(INDIRECT("1:"&C2))-1,E2,-A2)) / C2

where:

A2:  loan amount (positive number)
B2:  periodic interest rate
C2:  number of payment periods
D2:  balloon payment (positive number), usually zero
E2:  periodic payment (positive number), usually:  =ROUND(PMT(B2,C2,-A2,D2),2)

Since B2 and C2 are based on payment periods, the formula is independent of the payment frequency and the method of computing the periodic interest rate, which varies from country to country.

For example, for US loans with monthly payments, if the annual interest rate is 4%, B2 would be =4%/12.  If the loan term is 15 years, C2 would be =15*12.

-----
[*] Enter an array formula by pressing ctrl+shift+Enter instead of Enter.  Excel will display an array formula surrounded by curly braces in the Formula Bar, i.e. {=formula}.  You cannot type the curly braces yourself.  If you make a mistake, select the cell, press F2 and edit, then press ctrl+shift+Enter.

joeu2004 say: ORF wrote:

Is it possible to construct a formula without including a balloon payment amount?


Simply set D2 to zero, as I indicated.  Zero balloon payment is the same as no balloon payment.

Alternatively, simply omit D2 altogether.  So E2 becomes =ROUND(PMT(B2,F2,-A2),2).

I allowed for a balloon payment for generality.

ORF  wrote:
> Without an amortization table in the mix, I can't figure a balloon
> amount at the end of the term (3 years).

First, sure you can.  The FV function tells you the outstanding balance after any number of payments.  That would be the balloon payment if the loan terminated then.

But second, in structuring a variable-rate loan, the balloon payment applies to the end of the total loan term (15 years), not the end of the initial interest rate term (3 years).

In any case, if you do not want to allow for a balloon payment, so be it.  That's certainly your prerogative.

Good, because the first is easy.


On PowerPoint's Slide Show menu choose Set Up Show (bottom option)

Under Show Type choose Browsed at a Kiosk (full screen)

Under Show Options choose Loop Continuously until 'Esc'


All done. With these settings, when you play the slide show it will run continuously until the Escape key on the keyboard is pressed. This presumes that you have set animations and timings appropriately so that your presentation doesn't stop and wait for someone to click the mouse because you set an animation to play "when clicked" (unless you want it to, of course - you can make interactive presentations loop, too).

You only have to make a movie if you want to use the second option. In that case you would use screen recording software (QuickTime in Lion does this and there are others such as SnapZPro and Camtasia for more recording options). Then you can use iDVD to burn the movie to a DVD.

gord dibben say:

Can't help with your error but try this macro to delete names from workbook.

Public Sub Delete_Names()
 Dim N As Variant
 
 If ActiveWorkbook.Names.Count < 1 Then
  MsgBox "Hidden Ranges for " & _
   ThisWorkbook.Name & _
    " cannot be found", vbInformation, ThisWorkbook.Name
 Else
 
  Dim i, j As Integer: j = ActiveWorkbook.Names.Count
  On Error Resume Next
  For Each N In ActiveWorkbook.Names
   N.Delete
   i = i + 1
  Next N
 
  MsgBox i & " Named Range(s) were Deleted out of: " & j
 
 End If
 
End Sub
