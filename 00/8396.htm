john w. vinson say:

An Alias is just another name for something. Aliases can be used in SQL queries, or in the query grid, for various reasons.

 

For example, if you have a table field named txSurname and you want to include it in a datasheet or an export as LastName, you can type:

 

LastName: [txSurname]

 

in a vacant Field cell in the query grid; or equivalently use

 

SELECT txSurname As LastName...

 

in the SQL view of a query. This defines "LastName" as an alias for the field txSurname.

 

You can also define aliases for Table names if you use SQL view. For instance, if you had a table with a long unwieldy name like tblUserIdentityAndDemographics you could use a Query like

 

SELECT Bio.Thisfield, Bio.Thatfield

FROM tblUserIdentityAndDemographics As Bio

WHERE...

 

Often just having a shorter, more meaningful name for a table is useful; sometimes it's essential, for instance if you need to add a table to a query twice. For example, you might have a police dispatching table with fields for the ResponderID (who responded), DispatcherID (who took the call), and ClerkID (who entered the data into the database); in a query for a report you might want to join each of these three fields to the Officers table:

 

SELECT O.LastName AS RespondingOfficer, D.LastName As DispatchingOfficer, C.LastName AS Clerk

FROM ((Responses INNER JOIN Officers AS O ON O.OfficerID = Responses.ResponderID)

INNER JOIN Officers AS D On D.OfficerID = Responses.DispatcherID)

INNER JOIN Officers AS C On C.OfficerID = Responses.ClerkID

WHERE...

 

This is using the single Officers table in three different contexts, so you need aliases (C, D, O) for the three instances of the table, and aliases (RespondingOfficer, DispatchingOfficer, Clerk) for the names of the three officers.

    gord dibben say:

    Makes PDF file from each worksheet and names as value in A1 of each sheet.

    Adjust the Path to suit.

     

    Sub Sheets_To_PDF()
    Dim w As Worksheet
    Dim rng As Range
        Application.ScreenUpdating = False
        For Each w In ActiveWorkbook.Worksheets
        Set rng = w.Range("A1")
             w.ExportAsFixedFormat Type:=xlTypePDF, Filename:= _
            "C:\GordStuff" & "\" & rng.Value & ".PDF", Quality:= _
            xlQualityStandard, IncludeDocProperties:=True, IgnorePrintAreas:=False, _
            OpenAfterPublish:=False
        Next w
        Application.ScreenUpdating = True
    End Sub
