My apologies, here are the instructions for PowerPoint:

  2. Choose View>Master>Slide Master. The master thumbnail view displays the slightly large Slide Master at the top, with the smaller Slide Layouts below them.

How do I delete a duplicate favorites folder? - Microsoft Community

johngw14 say:

How do I delete a duplicate favorites folder?

Inside my favorites folder there is a duplicate "favorites" folder.  I have tried deleting and re-naming this duplicate folder in regular mode and safe mode.  I have also tried deleting my entire favorites folder, but each time I get a message that these folders cannot be deleted because they are being used in another program.

The path to this duplicate folder is C:user\john\favorites\favorites

How do I delete this folder?

 

robert aldwinckle on forums say:

How do I delete this folder?


First find out which directory is being used.   E.g. Run... (press Win-R and enter:)

shell:Favorites

Then in the resulting Windows Explorer window press Alt-d to see the path involved.   My guess would be that it is pointing at the one that you are trying to delete?   ;  )

A related test would be to use the Add to Favorites Bar or Add to Favorites commands and see where new .url files are written.

Once you know which one is the current shell:Favorites target you can right-click, Properties, Location and try to change it.   Then, once it is not the shell:Favorites target I would expect you would be able to delete it the same as any subfolder of Favorites.

Alternatively, if the problem is really just that the same name is in use and that creates a lock on it you could boot in safe mode Command mode only, e.g. under the Administrator account, and then delete the directory that way.   Logging in with another account and logging out of yours would be another way of achieving the same result.

And I just found another diagnostic you could use to find out what is going on.   Performance Monitor, CPU tab, Associated Handles list, search for Favorites.   Presumably you would find something has a handle opened for your  Favorites\Favorites  directory.   Then provided it was not the one pointed at by shell:Favorites it would probably be safe to kill all of those tasks and do your deletion with another one which wouldn't need it open, such as a cmd window (aka Command Prompt).

 

Good luck

Robert Aldwinckle
