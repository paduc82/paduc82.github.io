ken blake say:

There are some exceptions, but almost all programs that ran under Windows 7 will also run under Windows 10. The major exceptions are some utilities and some games.

Also note that regardless of the version or versions of Windows, a 32-bit program can run under 64-bit Windows. However the reverse is not true--a 64-bit program can not run on 32-bit programs.

Note that that last sentence does *not* say that a program that was running on 64-bit Windows can not run on 32-bit Windows. If it was a 64-bit program running on 64-bit Windows it can not run on 32-bit Windows, but if it was a 32-bit program running on 64-bit Windows, it *can* run on 32-bit Windows.

1. When you receive a BCC (Bug Check Code or informally, a BSOD) error, the STOP error details are displayed on the screen. After noting the error, you will have to manually turn off the computer, i.e. press and hold in the power button, wait 10 seconds then restart.
