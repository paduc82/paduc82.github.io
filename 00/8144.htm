That option must be set via the Modify Style dialog box, for the style you want to change. By default, the option is enabled for List Bullet and List Number (and some other styles as well).

I suspect you may have added the Exchange Server service to your Outlook profile by mistake.  Unless you know for a fact that you have an email account on an Exchange Server, go to Control Panel > Mail > Email accounts, select the Exchange Server account and Remove it, then add a new account of the correct type. tom ogilvy say:

Excel looks locally for the personal.xlsb file.  Also, the workbook itself has no ties or relation to the personal.xlsb if all you did was run macros through the menu.  So opening a workbook would not open the personal.xlsb.  Opening excel opens the persona.xlsb and as I said, it would be the local version (if one exists).

Perhaps you need to download the personal.xlsb as well and install it in the correct location or manually open it.

Tom Ogilvy

sharanukumar say:

Hi,

 

If you are using Airtel India network and  error message is displayed (e.g. "No response. Try again", or "Unable to open file", or "Operation failed").


Go to main menu -> press "Menu" -> "Settings" -> "Configuration" -> "Preferred access pt." -> choose "Mobile Office" (instead of "Airtel Live!") -> press "Select".

In case there is no APN named "Mobile Office" to choose, please contact your operator to get APN settings directly.

If you get APN information as plain text, values must be entered manually by selecting "Personal config. sett." in "Configuration" menu. After having entered the correct values, the newly created APN must be activated as described above.

NOTE:Using a different APN than preconfigured by your operator may result in high costs if not covered by your data plan. If in doubt, check your contract details with your operator before manually activating new APN settings!

 

I hope this helps...:)
