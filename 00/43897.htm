arielmt say:

Assuming you've reset your IP configurations to automatic:  If you have tried the "Diagnose Problems" button offered or the "Troubleshoot problems" link in the Network and Sharing Center (Start -> Control Panel -> Network and Internet -> Network and Sharing Center) without success, then the next thing I would try is deleting the wireless network and connecting to it anew:

  • Start -> Control Panel -> Network and Internet -> Network and Sharing Center -> Manage wireless networks (link on the left) -> Select your apartment's network -> Remove (button in toolbar)
  • Connect to your apartment's wireless network again as if it were the first time.

If that still doesn't work, make sure it actually is something with your PC instead of a rogue elsewhere in your apartment.  Take your laptop to a wi-fi hotspot, such as a restaurant offering free wireless, and connect to their Internet service wirelessly.  If you're still not able to access the Internet, then it's almost certainly something with your laptop or wireless adapter.

But if you are able to access the Internet elsewhere using the same hardware and software, then I'd put money on someone else in your apartment using static IP settings within your network's DHCP range of addresses, and your apartment's DHCP server happened to assign you that already-used address, thus creating the IP addressing conflict you're seeing.
