You used to be able to disable the Address Bar in the registry.  Don't know if it works in IE11.  Might give it a try...

  Open regedit and create the following keys:

  • HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Internet Explorer\ToolBars\Restrictions
  • Under Restrictions Key create a DWORD registry with the name NoNavBar and a value of 1
  • Logoff or reboot and the Address Bar is gone. To enable it again change the value to 0 or remove the key.

hansv mvp say:

The average of 0 is 0.

 

Did you mean that you want to average pupils with a non-zero score?

If so, you need to create an array formula. To make a formula an array formula, confirm it after entering it with Ctrl+Shift+Enter instead of with Enter or Tab.

When you do so, you'll see { } around the formula; this indicates that it is an array formula. You shouldn't type the { } yourself though; it's Ctrl+Shift+Enter that adds the brackets.

 

The formula is

 

=AVERAGE(IF((E3:E47="M") * (S3:S47>0),S3:S47))

hansv mvp say:

Excel 2007 and later have a new function for COUNTIF with multiple criteria: COUNTIFS, but that is not available in Excel 2003. In all versions of Excel, one can use SUMPRODUCT for this purpose:

 

=SUMPRODUCT((E3:E47="M") * (T3:T47="2c"))
