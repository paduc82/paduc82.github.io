Apple does not allow changes to attachments while they are still attached to an email message.

You must move the attachment into your Documents folder (or other folder where you have read/write capabilities) or use Save As and save them to a usable folder.

I should have been more specific. Sorry.

BEFORE you open the attachment, right-click on it. Choose Save-As from the pop-up menu.

In the Save As dialog, in the left side choose Documents. In the right side choose a folder in which to save the file.

Use an application to open the saved file or navigate to the file in Finder and then double-click the saved file's name.

-OR-

While still in Outlook, open the attached file. In whatever application that opened the file, use File > Save As and choose a folder in the Documents folder in which to save the file.

What you cannot do is open the file and the click Save in the application that opened the file while that file is an Outlook attachment. Writing to an attached file's location in Outlook is not allowed.
