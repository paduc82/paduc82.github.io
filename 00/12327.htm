hpholm say:

How much capacity does the SD memory card have?
Some PC's with a built-in card reader are not able to correctly handle SDHC or SDXC memory cards. (There may be a driver update from the PC manufacturer).

The phone can format the memory card itself:
App list > Settings > storage check, tap SD-card, swipe up and select the option to format the SD card at the bottom. This will erase everything on the SD card and force the phone to re-index all thumbnails and metadata regarding the SD card.

If formatting in the phone fails, then the SD card is probably broken.
