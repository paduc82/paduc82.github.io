Actually, after upgrade to Windows 10, my Wifi connection was set as Public and I wasn't able to change this location and it seems that Cortana doesn't work on a public network.

You can see it if you go to Network and sharing center (type this as a search in the start menu):

NB: Sorry, my Windows is in French, I hope translations will be good.

So, here is what you can try if you have the same problem:

1. Click on your Wi-Fi icon in the notification area

2. Disconnect from your network

3. Click on Network settings

4. In the new window, click on Manage Wi-Fi settings

5. Choose your Wi-Fi network and click Forget

6. Try to reconnect to your Wi-Fi and, Click Yes when you're asking if you want your computer to be found from the others.

7. Finally, check in the Network and sharing center  if your wifi is now on a Private network.

If so, Cortana should work! \o/

If you're still on a Public Network, repeat steps 1 to 5, then we'll try to go deep:

  • Press Windows + R on your keyboard.
  • Type secpol.msc and click OK
  • Here, check in Network List Management Policy if your network isn't set to public location and if Unknown network isn't set to public location, by default it should be set to "Not configured", change it if needed.

You can close secpol window, next:

  • Press Windows + R on your keyboard.
  • Type gpedit.msc and click OK
  • Go to Computer Configuration > Windows settings > Security settings
  • Again, check in Network List Management Policy if your network isn't set to public location and if Unknown network isn't set to public location, by default it should be set to "Not configured", change it if needed.

Close gpedit window.

And to be sure that Windows has forget your Wi-Fi profile:

  • Press Windows + R on your keyboard.
  • Type regedit and click OK
  • Go to HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\NetworkList\Profiles
  • Here, delete without regrets all the sub-keys (it looks like folders named {1234-11...})

Then you can go back to steps 6 and 7.

I hope everything is clear and could help others.
