The Pros And Cons of Microsoft Security Essentials. What Are - Microsoft Community

jackplatt say:

The Pros And Cons of Microsoft Security Essentials. What Are Yours?

Dear Microsoft Answers,

I am very interested in community feedback regarding the following: Pros and Cons of Microsoft Security Essentials.

The Pros:

 Lightweight-There are very few system resources used compared to other anti-virus and malware scanners.

Auto-Updates-You don't have to do anything, It runs in the background and keeps its mouth shut.

No Ads or Promotion Pop-Ups-The other companies always have these toaster pop-ups trying to get you to update.

No Add-On Prompts During Installation-You don't have to opt-out to prevent some tool bar, mini-browser, or other junk from installing.

The Cons:

No Control Over Exact Timing For Updates-Because MSE relies on Windows Updates in conjunction with data from the Microsoft Security Center and other sources such as IE's Smart Screen Filter, there is a strong Windows Network integration with the product. However, users have no control over exactly when the product will update. You have to "check to make sure."

Scanning Interval is unreliable and sometimes does not work- The default 2am scan time is extremely approximated! My own experience has shown scans to start around 1:30am to as late as close to 3am. If you are using your computer during that time, the scan will be missed, I once changed it to "Daily Scan" but the scan did not pick up, so I went back to the Sunday 2am default. It's like MSE scans "When it feels like it." The timing interval could be improved.

If Updates are missed it can take up to 48 hours (some have said 72 hours) before they are force downloaded- Supporters of MSE have said that it is OK for MSE to be out of date for a few days, and when a new threat is found any time before that 48 hour forced update, MSE will force a new definitions download. My concern is, wouldn't that be too late if the threat has already compromised a users' PC or data? Maybe several daily definition updates may be old school, but are customers really getting MSE updates at least once a day? I either reboot my system once a day OR shut down giving it a six hour or more rest each night. On any cold boot or restart MSE should be connecting to its servers and downloading those updates regardless of the day or time interval of last system check.

No Notice of New Features in MSE Program Updates- Microsoft, likely for security reasons, does not publish a list of bug fixes or security ehancements with new releases of MSE. Users could benefit from a list of enhancments from version to version, bug fixes, and any bugs that are still being evaluated by Micorsoft. Hope others can contribute to this discussion!

Jack
