mike h.. say:

Hi,

 

To Use autofilter on a protected sheet select the option to allow autofilter when applying the sheet protection. users will then be able to use existing autofilters but not create new ones.

 

Sorting is a diferent matter. There is an option to allow sorting in the protection dialog but this only allows sorting on un-locked cells so you have choices:-

 

1. Un-lock the cells that you want users to sort by right click - format - protection tab and un-check locked. Note that users will be able to change these cells.

 

2. Do the sorting with a macro. If you want this option then more detail of what and the extent of sorting is required.

Hi Guys,

I've been trying to find a solution to this for a few days now, and I've found two workarounds... So far, no solution is in sight by Microsoft.

First workaround:

Go to the webpage on IE, save it as an HTML file.

Open a new e-mail message, and click the "Attach" button.  Navigate to the HTML file, select it, and click the down arrow next to "Attach" - Select "Attach as text" and it will show up.

Alternatively, you can right click the page, send to OneNote.

In OneNote, you can click the down arrow in the Quick Access menu at the very, very top, select "More Commands," and add "Email Page" to the quick menu.  As a warning, this destroys all formatting of the page, and looks pretty terrible... but it at least somewhat works.
