If the same issue persists, follow the steps below using command line:

  3. On Command Prompt, type command slmgr /upk and hit Enter to uninstall product key.
  4. Then, type command slmgr /cpky and hit Enter to uninstall the product key form the registry.
  5. After that, type command slmgr /ipk followed by your product key and hit Enter (e.g. slmgr /ipk XXXXX XXXXX XXXXX XXXXX XXXXX) to install the product key on the system.
  6. Finally, type command slmgr /ato and hit Enter to activate Windows.
