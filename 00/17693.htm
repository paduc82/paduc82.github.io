Plan B

Create a userform (see http://www.gmayor.com/Userform.htm). Add a series of check boxes (the following code shows only two, but the principle is the same for more) a label and a comamnd button. The code uses the default names for the checkboxes, label and command button.

Associate the following code with the userform

Option Explicit
Private oVars As Variables
Private bProtected As Boolean

Private Sub CommandButton1_Click()
With ActiveDocument
'Unprotect the file
If ActiveDocument.ProtectionType <> wdNoProtection Then
    bProtected = True
    ActiveDocument.Unprotect Password:=""
End If

    Set oVars = .Variables
   
    If Me.CheckBox1.Value = True Then
        .Bookmarks("Clause1").Range.Font.Hidden = True
        oVars("varClause1").Value = True
    Else
        .Bookmarks("Clause1").Range.Font.Hidden = False
        oVars("varClause1").Value = False
    End If
   
    If Me.CheckBox2.Value = True Then
        .Bookmarks("Clause2").Range.Font.Hidden = True
        oVars("varClause2").Value = True
    Else
        .Bookmarks("Clause2").Range.Font.Hidden = False
        oVars("varClause2").Value = False
    End If
    'etc
End With

'Reprotect the document.
If bProtected = True Then
    ActiveDocument.Protect _
    Type:=wdAllowOnlyFormFields, _
    NoReset:=True, _
    Password:=""
End If
Unload Me
End Sub

Private Sub UserForm_Initialize()
On Error Resume Next
Set oVars = ActiveDocument.Variables
With Me
    .Caption = "Hide unwanted clauses"
    .CommandButton1.Caption = "Continue"
    .Label1.Caption = "Check the boxes to hide the clauses."
   
    .CheckBox1.Value = oVars("varClause1").Value
    .CheckBox1.Caption = "Clause 1"
   
    .CheckBox2.Value = oVars("varClause2").Value
    .CheckBox2.Caption = "Clause 2"

    'etc

End With

End Sub

lars-Åke aspelin say:

This solution proposal uses conditional formatting:

Select the cells with data i Sheet2, e.g. cells B2:C10

On the Home Tab, click on the Conditional Formatting in the Styles section

Select New Rule...

In the New Formatting Rule pop-up window, select Use a formula to determine which cells to format

In the Format values where this formula is true: box enter the following formula

=IF($B2<>"",OR(NOT(ISERROR(IF($C2=Sheet1!$C$2:$C$100,FIND(","&$B2&",",","&SUBSTITUTE(Sheet1!$B$2:$B$100," ","")&","),NA())))))

Note: Change the 100 in both places to fit the size of the table in Sheet1

Click on the button Format...

Choose the format you like in the Format Cells pop-up window and click on OK

Finally, click on OK.

I put in the SUBSTITUTE in the formula to allow for entries with extra spaces, like

2,5 , 15, 24

in the sequencenumber column, i.e. column B, in Sheet1 as you mentioned this to be manually entered.

If there is no risk for extra spaces in this column, you can skip the SUBSTITUTE part of the formula.
