what is a delimiter? - Microsoft Community

best1can say:

what is a delimiter?

I run into this term many times but never had a good definition of it.  I see it in VBA,Excel, and others.
zorvek say:
A delimiter is any character chosen to separate one element in a string from another element. As Ronnie pointed out above it is commonly used in files to separate the fields in each record (comma or tab), and the records from each other (carriage return and/or line feed).

A delimiter is also used in dates (dash or slash): 12/25/2012 or 12-25-2012

A delimiter is used in times (colon): 10:32:00 AM

A delimiter is used to separate parameters in a call to a subroutine (comma):

    SubroutineName Parameter1, Parameter2, Parameter3

or in a function call in a cell:

    =SUM(A1,B1,C1)

A space is even considered a delimiter in a line of code:

   A = B + C

Kevin

suzanne s. barnhill say:

There are several possibilities here:

  1. The default body font in Word 2007 is Calibri, a sans-serif font with quotation marks that are not as curly as those you are used to in Times New Roman, but they are slanted, not straight up and down like "straight quotes." Are you sure your quotation marks are straight?
  2. Make sure you've actually changed the right option. Go to Office Button | Word Options | Proofing and click AutoCorrect Options... Make sure you have checked the box for "Straight quotes" with "smart quotes" on the AutoFormat As You Type tab as well as AutoFormat.
  3. No change in formatting in Word 2007 should affect existing documents; they should be just as you created them in your earlier version. For the new documents you've created with Word 2007, if they really do have straight quotes, you can easily fix them. After enabling "smart quotes" on the AutoFormat As You Type tab, search for ' or " and replace with ' or ". Word will autocorrect the quotes as it replaces them.

For what it's worth, Word 2007 is not "the newest version of Microsoft Word." Word 2010 and 2013 are newer. If you do actually have one of those versions, then the path to the AutoCorrect Options dialog is File | Options | Proofing | AutoCorrect Options...
