I think it would make sense to use Ethernet first, get IP addressing established and then switch.

Try using SMB.  When I do that I get IPv6 addresses, not IPv4 ones.  Then a relevant question would be how would the IPv6 be transmitted?  E.g. perhaps you would have to use a tunnel, such as ISATAP.  But I don't think that one is working well in W10.  E.g. when I upgraded from W8.1 to W10 I had horrible problems with my modem's router which was IPv4 only.  I had to get a new router which actually supported IPv6 and not rely on a tunnel.
