mitzi09 say:

How do I get rid of the " startnow browser hijacker" ?

My home page changed from Google to Startnow.  How can I get rid of it?
shenan stanley say:
  1. To stop all Startnow.com processes, press CTRL+ALT+DELETE to open the Windows Task Manager.
  2. Click on the "Processes" tab, search for Startnow.com, then right-click it and select "End Process" key.
  3. Click "Start" button and selecting "Run." Type "regedit" into the box and click "OK."
  4. Once the Registry Editor is open, search for the registry key "HKEY_LOCAL_MACHINE\Software\Startnow.com."  Right-click this registry key and select "Delete."
  5. Navigate to directory %PROGRAM_FILES%\Startnow.com\ and delete the infected files manually.


Then download, install, update and perform full system scans with:

  • MalwareBytes (FREE)
  • SuperAntiSpyware (FREE)


Removing everything they find.  Rebooting when needed.  (You can uninstall one or both when done.)

Then perform an online scan with the eSet Online Scanner.
