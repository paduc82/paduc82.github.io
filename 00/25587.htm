allen browne say:

As you found, the form's Unload event is too late. The record has already been saved by this time.

Instead use the BeforeUpdate event of the Form (not controls.) This is the last event the fires before the record is saved. If you cancel this event, the bad data is not saved in the table.

Here's an example of checking for a blank control. The code is designed so you can duplicate the first If block to check other controls also:

Private Sub Form_BeforeUpdate(Cancel As Integer)
    Dim strMsg As String
    Dim strControl As String
    If IsNull(Me.SomeControl) Then
        Cancel = True
        strMsg = strMsg & "SomeControl can't be blank." & vbCrLf
        strControl = "SomeControl"
    End If
    If Cancel Then
        strMsg = strMsg & "Oops:" & strMsg & "Correct, or press <Esc> to undo."
        MsgBox strMsg, vbExclamation, "Invalid data"
        If strControl <> vbNullString Then
            Me(strControl).SetFocus
        End If
    End If
End Sub

sub SaveAsdPDF()

  Application.Wait Now + TimeValue("00:00:03")
  A = ThisWorkbook.Path & "\FC\" & Format(Now, "ddmmyy-ss") & ".pdf"

  If IsValidFileName(A,True) then

end sub

Function IsValidFileName(sFileName As String, Optional bShowIllegal As Boolean) As Boolean

    Dim lstIllegal As Variant
    Dim i As Long
    Dim result As Boolean

    lstIllegal = Array("/", "", ":", "*", "?", "< ", ">", "|", """")
    result = True

    For i = LBound(lstIllegal) To UBound(lstIllegal)
        If InStr(1, sFileName, lstIllegal(i)) > 0 Then
            result = False
            If bShowIllegal Then
              MsgBox "Illegal character in filename: " & lstIllegal(i) & vbCrLf & sFileName
            End If
            Exit Function
        End If
    Next i
    IsValidFileName = result
End Function

You will find blank spaces are quite legal!

I discovered a Bug while working with (Power)Pivot in Excel 2013. You can replicate the issue with the following steps:

1) Create a simple source table with the following data:

Name Important

Test 03 FALSE

2) Create a PivotTable based on this source table with the option "Add this data to the Data Model" checked

3) Then use the "Important" column as ROWS and the "Name" column as VALUES. You'll get the following result: 1x (blank) and 2x FALSE

Row Labels Count of Name
(blank) 1
FALSE 1
FALSE 1
Grand Total 3

When creating the PivotTable without the option to add a data model, you'll get the correct result: 1x (blank), 1x FALSE and 1x TRUE

Row Labels Count of Title
(blank) 1
FALSE 1
TRUE 1
Grand Total 3

payrollnorth canterbury sports say:

Using COUNTIF across multiple worksheets

Hello,

I have a workbook that includes several worksheets from which I need to extract certain answers onto a cumulative worksheet.  To be less general, I have put accident reports in an Excel workbook, one sheet per accident.  I need to know for example how many times a particular type of accident happened where staff were involved as one example. I am having difficulty getting Excel to pick-up the data on the additional sheets.  I am currently using the following on the Report sheet to calculate: 

=SUM(COUNTIFS(Sheet6!B21,"First Aid",Sheet6!C18,"Staff/Employee"),COUNTIFS(Sheet7!B21,"First Aid",Sheet7!C18,"Staff/Employee"))

However, I am having to manually add the sheets as they are completed, rather than Excel picking up on them.  Is there actually a way to do this? I am trying to save time going over the data and entering it into a report monthly for our Trustees manually by putting it all in Excel and let the program do the work, but it's not working.  I've spent all day (6 hrs) researching ways to do it, but I'm getting no where. 

amit tandon say:

Create a range with relevant sheet names, name this range TabList, and use below formula:

=SUMPRODUCT(COUNTIFS(INDIRECT("'"&TabList&"'!B21"),"First Aid",INDIRECT("'"&TabList&"'!C18"),"Staff/Employee"))

You can enter sheet names in cells B1 (Sheet2), B2 (Sheet3), B3 (Sheet4) and name this range TabList (B1:B3). You can update the named range TabList every time you add/include a worksheet by expanding the TabList - in cell B4 you enter Sheet5 and TabList is to be expanded to (B1:B4).

This should be a simple way to use COUNTIFS across multiple sheets. Please note that the named range should not have blank cells.
