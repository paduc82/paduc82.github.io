You need to step through each of the array elements to see if they are contained in the cell.

Sub Test()
Dim rng As Range
Dim cell As Range
Dim ContainWord As Variant
Dim i As Integer, Found As Boolean

  Set rng = Range("A1:A100000")

    Next

john p spencer say:

You can use open args to pass the value of the checkbox.  It will get passed as a string (either "-1" or "0")

I would then create a module level variable and set its value.

Dim tfFax as Boolean

Private Sub Form_Load()

tfFax = Me.OpenArgs <> "0"
End Sub

Then you can use the detail event to set the value of an unbound control in the detail event

Private Sub Detail_Format(Cancel As Integer, FormatCount As Integer)

IF tfFax Then
   Me.SomeControl = [Fax Number] + "(F)"
Else
   Me.SomeControl = Null
End IF

End Sub

There are of course other methods to do this.  This was (in my opinion) the easiest to explain.
