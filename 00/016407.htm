The File > Save As page that has the Add a Place button has become a total disaster in Word 2016 since the removal of the Recent Places list. The Add a Place function has never had any options other than SharePoint and OneDrive, and I don't expect it to get any better. The solution is to give up on the File > Save As page altogether. To replace it:

  • In File > Options > Save, check the two boxes for "Don't show the Backstage when opening or saving files" and "Save to computer by default". ["Backstage" is what Microsoft calls all the things on the File tab.] In the "Default local file location" box, enter the path to the folder you use most often, or the parent folder of a group of often-used subfolders.
  • To open an existing document, use the Ctrl+O shortcut to pop up the Open dialog box. To save a document for the first time or to save it as a different file, use the F12 key to pop up the Save As dialog box.
  • For folders you use frequently, add them to the Quick Access list. In the Open dialog, the Save As dialog, or the File Explorer program (which are all the same code), right-click any folder and click Pin to Quick Access. The folder will appear in the list at the top of the left panel until you remove it.
