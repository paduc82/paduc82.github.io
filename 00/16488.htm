hi vzion117,

to bulk remove a specific license, you can see the following script:

https://gallery.technet.microsoft.com/disable-yammer-in-office-432f584d

note: this script will disable yammer services for all your users. if you want to be selectively, since our forum focuses on office 365 for business online services, it is suggested that you seek help from script center, thanks for your understanding.

you can also refer to the following steps to un-assign only the yammer license to users in bulk, 

1. create a .csv file and add users under the column userprincipalname. here is a sample users.csv.zip.

2. install the azure ad module.

3. run connect-msolservice to initiate a connection to the online service. the credentials of the global admin administrator account are required.

4. run get-msolaccountsku to find the skuid of the office 365 plan, then run the following command to get each office 365 service plan skuid. in my case, it's the office 365 enterprise e3 plan. you can change the enterprisepack based on your scenario:

get-msolaccountsku | where-object {$_.skupartnumber -eq "enterprisepack"} | foreach-object {$_.servicestatus}

5. use the following script to remove the yammer service.

$users = import-csv d:\users.csv

$noyammer = new-msollicenseoptions -accountskuid organization:enterprisepack -disabledplans yammer_enterprise

note: you can change the organization:enterprisepack, skuid in the disableplans according to your situation.

$users | foreach-object{set-msoluserlicense -userprincipalname $_.userprincipalname -licenseoptions $noyammer}

i recommend you do a test on one user before performing all the scripts above.

dan at it associates say:

Sorry, No, a Product Key Card license is not permitted to be moved from the computer on which it was first activated, even if the computer no longer works...the PKC license lives and dies with that first original computer.

 

Assuming you are planning to purchase a new computer and may also be shopping for a new Office, please note that ALL consumer licenses for Office 2013 (the current locally-installed Office version) have become just like PKC licenses, good only on that first computer.

 

Just as an FYI, Office 365 Home Premium is OTOH a cloud-based product that you subscribe to (yearly fee), that allows you to concurrently activate the product on up to 5 computers, both PCs and Macs, and also allows you to move the activations to other computers if any of them are damaged, sold, die from old age, etc.
