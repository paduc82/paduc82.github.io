¡firedog say:

... trying to replicate Windows Live Mail on a new laptop because I have a ton of saved email on my old laptop that I would like to import.

It's quite possible. First, copy the entire store folder from your old machine to the same spot on the new one. By default, the store folder is this one:

C:\Users\{user}\AppData\Local\Microsoft\Windows Live Mail

where {user} is the name of the Windows user account concerned. Note that {user} may not be the same on the new machine, depending on the name you gave it.

Then download the Windows Essentials 2012 full installer from the wayback machine. There are two English versions; if one doesn't install, try the other one:

Universal English:

http://web.archive.org/web/20170112124505/http://wl.dlservice.microsoft.com/download/C/1/B/C1BA42D6-6A50-4A4A-90E5-FA9347E9360C/en/wlsetup-all.exe 

British English:

http://web.archive.org/web/20160423192125/http://wl.dlservice.microsoft.com/download/1/9/4/194B1919-A211-441F-BC0E-34F6B533B876/en-gb/wlsetup-all.exe

The installer should find the store folder you created and take it into use. You will then have to re-enter all your saved passwords.

If you want to set up a new Outlook Mail account using IMAP, you'll find detailed instructions in this article: Windows Live Mail and Outlook Mail
