rajesh sinha. say:

As far as my knowledge is concerned there is no Date & Time Format available in Excel could convert Weekday's name in Capital. But I can suggest you a Formula will help you to do so. 

=Upper(Text(A2,"dddd"))&Text(A2, " mmmm yyyy")

  • You can adjust cell reference as you need.

If you want to apply Weekdays Name as fastest then, I would like to suggest a MACRO.

Sub WeekDayCaps()

    Dim ws As Worksheet

    Dim Rng As Range


    Set ws = ThisWorkbook.Sheets("Sheet1")

    Set Rng = ws.Range("B2:B2")

    Rng.Formula = "=IF(A2="""","""",upper(TEXT(WEEKDAY(A2),""dddd   "")))&Text(A2,""Mmmm YYYY"")"

End Sub


N.B.

  1. Copy & Paste this as Standard Module. 
  2. Write Date value in Cell A2 and RUN it.

First things first, is the ISO image the same build as your computer? You can find out by opening the Start menu and typing About your PC. It will be the OS Build.

If it is, please run this command on an elevated command prompt: DISM /Online /Cleanup-Image /RestoreHealth /Source:WIM:F:\Sources\Install.esd:1 /LimitAccess.

You will notice that we simply changed the extension to .esd. Also, kindly exclude the period when running the command.
