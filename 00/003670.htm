PowerPoint High Quality Full HD Video Export in Power Point 2016 - Microsoft Community

royi a say:

PowerPoint High Quality Full HD Video Export in Power Point 2016

How can one export a video from Power Point in Full HD Resolution (1080p) in Power Point 2016?

In Power Point 2013 on could do the following:

  1. Open the Visual Basic Editor (ALT + F11).
  2. Click "Insert -> Module" and past the attached.
  3. Save the module and go back to Power Point.
  4. Go to "View -> Macros -> Run" (Or Click ALT + F8) and chose "MakeMp4Video".
    Chose Run and let the program run.
  5. The output will be named "Test.mp4" and placed on the Desktop.

The Code:

Sub MakeMp4Video()
If ActivePresentation.CreateVideoStatus <> ppMediaTaskStatusInProgress Then
ActivePresentation.CreateVideo FileName:=Environ("USERPROFILE") & "\Desktop\test.mp4", _
UseTimingsAndNarrations:=True, _
VertResolution:=1080, _
FramesPerSecond:=30, _
Quality:=100
Else: MsgBox "There is another conversion to video in progress"
End If
End Sub

brian tillman [outlook mvp 2007-2019] say:

Nothing will happen to your documents.  Office 2007 and 2010 both use the same formats for their generated documents.  If converting from the trial of Office 2010 to a released 2007 Office suite, there will be no conversion issues.  If you wish to use Office 2003, it is a good idea to download and install the Microsoft Office Compatibility Pack, which will allow Office XP or Office 2003 to accept Office 2007/2010 Open XML document format.

Outlook 2003 through 2010 all use the same format Personal Folders file and there should be no conversion issues there, either.  If you need to use Outlook 2002, however, you'll need to convert the PST to the older ANSI format.  This is a manual operation but it's not all that difficult.  The technique is described here: Convert an old PST to a Unicode PST.  Although this described the process of converting from the older to the newer format, the reverse process is nearly the same.  You must choose the "Outlook 97-2002" Personal Folders file format when creating a new PST for the conversion.
