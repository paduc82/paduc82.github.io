how do i convert 2010 outlook files, address book, emails to office 365?

Just going by your question, the short answer is that there is nothing to convert.

If this is a situation where you are doing an "in-place" upgrade of '2010 to Office 365 on the same machine, the profile settings for '2010 will be used by '365

If this is a case, where you are installing Outlook on a new machine, you would just need to copy the files to the new machine. However, <what> you need to copy depends on the type (POP3, Imap, Exchange) of email accounts were configured in '2010. A definitive answer can be provided once the info to that question is provided.

If you're not sure about you're email accounts, open Outlook '2010 and go to File --> Account Settings --> email tab --> type of email account is shown in the right hand column

Next go to the Account Settings --> Data Files tab --> and take note of each data file listed. Any PST files associated with any IMAP files would not need to be copied but you do want to copy any PST file set as the default data file.

Rather then go through all the possible combinations and permutations, a more definitive answer can be provided once the above info is provided.

Your answer implies that all my files, including Outlook addresses, emails, etc. will automatically get transferred into 365. Did I read this correctly?
<<

That's correct. Would additionally suggest that you uninstall '2010 before installing 365. As mentioned, the profile used by '2010 which contains all the info regarding your Outlook files/email accounts will be migrated by '365.

Two things to make note of

#1 - Do NOT use the Microsoft uninstall <Fix-It> tool since that also removes existing Outlook profiles. Just uninstall '2010 via Control Panel <Programs>. If you uninstall '2010 after installing '365 then you should do a full online repair of '365 after the '2010 uninstall completes

#2 - The key difference between '2010 and '365 relates to IMAP accounts where the PST file used for IMAP email folders will be converted to an .ost file and the email folders will be sync'd with the server. No action is required on your part.
