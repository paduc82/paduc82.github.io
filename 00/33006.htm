Let me describe the general idea:

Task that responds to the Sleep Event that starts a batch-file that shuts down any applications when going to sleep or starts any applications when waking up from sleep. "

With help of some "test sleeping" I denoted that System - Kernel Power - ID 42 puts my system to sleep, and System - Power-Troubleshooter - ID 1 wakes my system up from a sleep state. Without loss of generalty I suppose these ID's will be the same for every computer.

The fix

  1. Right-click on the taskbar and and select Start Task Manager
  2. Make sure your process is running, right click on the process and click Properties. 
    If this option is grayed out, make sure to expand the process and click a subprocess of the main branch.
    (Make sure that force quitting this process will shut down the whole branch!)
  3. Under General denote the name of the process next to its icon (e.g. "chrome.exe")

Finding your processdirectory (of process you want to start when going to sleep):

  1. Right-click on the taskbar and and select Start Task Manager
  2. Make sure your process is running, right click on the process and click Open File Location
    If this option is grayed out, make sure to expand the process and click a subprocess of the main branch.
    (Make sure that force quitting this process will shut down the whole branch!)
  3. Shift Right-click on the .exe-file and click on Copy as Path
  4. Note that when putting this in your batch-file you need to use quotation marks (" ... ") around things like with spaces
    For example: The PROCESSDIRECTORY for chrome would be C:\"Program Files (x86)"\Google\Chrome\Application\chrome.exe
    Note that I removed the extra quotation marks (" ... ") around it.

The Batch-files:

  1. Create a new folder somewhere for your new OnSleep and FromSleep batch files.
  2. Right-click inside the folder and select New > Text Document and rename it to OnSleep.bat or FromSleep.bat. and click OK.
  3. When creating a OnSleep.bat-file use the command "taskkill /IM PROCESSNAME /F"
    When creating a FromSleep.bat-file use the command "start PROCESSDIRECTORY"
    (Don't use these quotation marks (" ... ")  in your batch-file by the way.)

The Tasks:

  1. Open up Task Scheduler and click on Show/Hide Action Pane if there is no Action Pane on the right of your window
  2. Click on Create Task... and give a proper name and description.
  3. Make sure under General > Security Options that it will Run with highest privileges
  4. Under Triggers click on New... and select Begin the Task: On an Event
  5. Fill in the event information based on what kind of task you are making and press OK.
  6. Under Actions press New... and select your batch-file and press OK
  7. Create your task by pressing OK one more time.
