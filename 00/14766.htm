muditha jayath chathuranga say:

Hi Alex,

Yes this is possible to achieve. What you need to do is, perform following tasks from Exchange Admin Center (EAC).

  1. Go to Recipients -> Mailboxes
  2. Search the common mailbox
  3. Edit by clicking on the pencil icon
  4. Select mailbox delegation
  5. Add users to Full Access section.

Users who have Full Access will only be able to access the mailbox to read and manipulate items in it. Any attempt to send from this mailbox will fail if they are not in either Send As or Send on Behalf permissions boxes.

Once this is done, make sure to remove the existing mail account from Outlook. (I assume you used the regular user name and password to add it.) Then follow below steps to add the mailbox to Outlook.

  1. Open Account Settings
  2. Select the user's mail account and click Change.
  3. Click More Settings ...
  4. Go to Advanced tab.
  5. Click Add in Mailboxes section.
  6. Enter the mailbox ID and click OK.
  7. Close all dialog boxes by clicking OK and restart Outlook.

What this error means is, your Windows 10 installation folder is unfinished and/or corrupted. Now, how do you fix that?
