ossiemac say:

Without the copy of the workbook all suggestions are only a guess at what you might try. I am assuming that this is not occurring with all workbooks; just one particular set of data. If the problem is with all workbooks then you probably have a problem with Excel which might be fixed with repair or re-install.

I am wondering if maybe the original data came from another system (eg. downloaded from the internet) and if so, it might have additional non visible characters appended to the actual data. When format is set to wrap cells then the additional non printable characters can cause them to wrap and it appears that the rows are all too wide. In a blank cell away from the data enter the following formula and see if the number of characters match the number of visible characters in the cell (Where B5 is one of the problem cells). If length if longer than the visible data then there are non visible characters present.

=LEN(B5)

You can also check it out as follows:

  • Select a problem cell
  • Click in the formula bar some distance out to the right of the visible data.
  • The cursor should automatically position itself at the end of the data.
  • Check if there are any blank space between the visible data and the cursor. If there is then there are non visible characters appended to the data.
  • Also check that the visible characters start at the very left of the formula bar because the non visible characters can be there also. To check this, click anywhere in the visible data and use the left arrow key to move the cursor to the left and it will be obvious if there are any non visible characters before the visible characters.

Removing non visible characters is the next step.

Try both TRIM and CLEAR functions to see if it will fix the problem. (If necessary, see Help for how to use the functions). These need to be inserted in the cells to the right of the data and then copy the data with the formulas and Paste Special -> Values to remove the formulas and then test as above to see if the non visible characters appear to be removed. If they are then the original columns of data can be deleted leaving the clean data.

If above did not remove the non visible characters (and it quite often does not) then the following procedure.

  • Select a problem cell
  • Position the cursor in the formula bar out to the right of the data and the cursor will then auto position at the end of the cell's non visible characters.
  • Use the left arrow to move the cursor back ONE character
  • Carefully use the mouse to click where the cursor is and hold the left button and drag to right to highlight the non visible character. (It may be a very narrow amount but it should highlight grey)
  • Hold Ctrl key and press C to copy.
  • Press Esc to get out of Edit mode in the formula bar
  • Click Find & Select  then select Replace
  • Position cursor in the "Find what" field and then hold the Ctrl key and press V once. (Insert the non visible character)
  • Leave "Replace with" field blank and click Replace all.
  • Test as above to see if the non visible characters are removed.

If the above does not help then I will need a copy of the data to experiment with and see if I can identify the problem.
