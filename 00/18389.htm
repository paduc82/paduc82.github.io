I received a sample workbook from the asker. There are two problems with the solution given the workbook:

1) There is a header with non-time text values in row 1 - this will cause an error of #VALUE when evaluating the header row.

2) The times, while formatted correctly, are stored as text values.

This formula correctly finds 3 rows that are between 1 and 2 minutes in duration:

=SUMPRODUCT((VALUE(A2:A100)>=TIME(0,1,0))*(VALUE(A2:A100)<=TIME(0,2,0)))
