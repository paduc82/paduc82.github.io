rob koch say:

salvemundum,

Though I generally agree with and in fact use some of the same security tools that you do, I want to point out some potential issues with your choices which you may not be aware of, as well as try to clarify some of these specifically as they relate to Windows 10 which the original poster has indicated they are using.

First, the primary issue already mentioned that the Windows Defender Full scan takes too long at several hours.  The reason that this takes so long is that it performs a very complete "deep scan", unlike many of the 3rd-party security applications which often use speed optimizing techniques to perform their scans faster, which may not be as thorough.  Microsoft actually doesn't recommend using the Full scan unless a malware infection is already suspected, since Quick scans and Real-time protection normally detect any active malware by checking commonly infected areas of the operating system.

Some of the items you've listed aren't necessary with Windows 10, since like Windows 8 it already contains Defender so #1) MSE AV is unneeded and #2 as well, since Windows 10 includes a simple PDF reader app sufficient for most consumers.

#3, 4 and 8 I am going to ignore entirely, since these applications aren't often required by most users, so their use would be based on the needs of the specific user. In fact, either Windows 10 or the Outlook.com web email includes access to free basic video/audio viewers or limited versions of the most common Office apps, so few consumers need these any longer.  In general terms, the less 3rd-party applications that are installed the more secure the system will be, so offering these as optional for those who require them might be more effective when discussing security.

The EMET toolkit is a valuable addition for those with the technical understanding to use it, but it was really designed for more technical users like network and security administrators in large organizations, so it might be better to offer other options such as the Malwarebytes Anti-Exploit application to most consumer users.  Part of the reason I say this is that even you have apparently missed the fact that the 3rd item in the EMET Security Mitigations table on the page you provided is "Data Execution Prevention (DEP) Security Mitigation".  This means that DEP is already configurable using EMET, so the manual instructions you provide later for this are unnecessary and in fact will conflict with EMET's control of this function.

Though your advice to maintain effective backups is good, since Windows 10 contains both cloud-based synchronization via OneDrive as well as other backup options within the operating system itself, it might be better for most consumers to investigate these before adding a 3rd-party alternative.  This is especially true since it's not possible to be certain that any of these 3rd-party vendors will still be around or free in a few years, while Microsoft is likely to still be supporting either the current or possibly newer versions of what they provide in Windows itself.

I'm not currently running Windows 10 yet myself, but I believe that the Enhanced Protected Mode setting that you mentioned is only available in Internet Explorer 11, while the default Edge browser is so limited in the ability to run extensions (only the built-in Flash player and possibly PDF I believe?), that there's no reason for this setting for that browser currently.

rob koch say:

There is no such thing as a "best" antivirus, since the best for you may be different than for someone else.

This seems strange to most people, but the truth is that most reputable security programs work well for some and badly for others.  This often has more to do with how the program displays and handles a malware attack then how well it detects them, since some people understand very technical details and others don't.

If you find that one program provides much easier to understand or otherwise "better" notifications, then that's likely the best program for you personally.  In my own case I so rarely experience malware in the first place, that I prefer not to see any notifications at all unless there's something critical occurring that I need to be aware of.  Since that's how Windows Defender operates, this has been the best program for me personally.

You'd need to discuss how ESET works with others who use it regularly, so that's typically best done in their own support forums.  Remember that wherever you are asking you are likely to hear more who prefer the program(s) commonly supported there, so don't assume this is helpful.  Instead, try to learn how the program actually works when malware is detected and whether you like and understand the way it does this, since you are the reason for the program and not either the PC or the malware.

Rob
