Compound Interest Formula - Microsoft Community

whendidwe say:

Compound Interest Formula

Is there a formula in Excel that will calculate the final value of periodic payments at a fixed interest rate compounded quarterly?

As an example, suppose I invest $200 every month. The interest rate is fixed at 5% per annum, compounded quarterly. What is the value in 20 years?

I know I can do this on a month-by-month basis, but it seems there should be a formula that you can just plug in the variables and get the results.

Thanks in advance for your help.

joeu2004 say:

[My previous response was incorrect.]

=FV(5%/4, 20*4, -200*(3 + 6*5%/12), 0, 0)

5%/4 is the quarterly interest rate.  It might be (1+5%)^(1/4)-1 if the annual rate is a compounded rate ("yield" or "rate of return").

20*4 is the number of quarters.

200*(3+6*5%/12) is the 3 monthly payments plus interest for each quarter.  5%/12 is the monthly interest rate.  It might be (1+5%)^(1/12)-1 if the annual rate is a compounded rate.  6 is the sum of 1+2+3, the cumulative months of

(In general, if the compounding period has n months, replace 3 with n, and replace 6 with n*(n+1)/2.)

The quarterly payment is negative because of Excel's convention of signed cash flows:  opposite signs for inflows and outflows.  The choice of signs is arbitrary.  I choose negative for payments ("inflows") so that FV ("outflow") returns a positive value.

The first 0 is the initial account balance.  You did not indicate any.

The second 0 is for payments "at the end" of the quarter.  What it really means is:  the quarterly payment does not earn additional interest because we included the cumulative quarterly payment amount.

The following demonstrates the correctness of the FV formula above [EDIT: A more intuitive implementation].

Formulas:

C3:     =IF(MOD(A3-1,3)=0, B3, B3+C2)

D3:    =C3*5%/12

E5:     =SUM(C5, D3:D5, E2*(1+5%/4))

F14:   =FV(5%/4, 1*4, -200*(3+6*5%/12), 0, 0)

F242: =FV(5%/4, 20*4, -200*(3+6*5%/12), 0, 0)

Note that the result of the FV formula in column F is essentially same as the "manual" calculations in column E.

(Caveat: Sometimes, there is an infinitesimal difference due to anomalies of Excel and binary computer arithmetic.)

rich007a say:

You could of course, use this formula for custom data validation in cell A2 (assuming A2 is where your data starts):

 

=AND(COUNTIF(A$2:A2,A2)=1,LEN(A2)=11,ISNUMBER(1*A2),INT(1*A2)=1*A2,LEFT(A2,1)="0")

 

However, I would probably find working with this quite painful, as there is no easy way to tell which condition has not been met.

 

Therefore, I would consider ditching DV (select column A, Data tab > Data Validation > Any Value > OK) and adding the following formulas to the following cells:

 

B2:  =IF(A2<>"",COUNTIF(A$2:A2,A2)=1,"")

C2:  =IF(A2<>"",LEN(A2)=11,"")

D2:  =IF(A2<>"",ISNUMBER(1*A2),"")

E2:  =IF(A2<>"",INT(1*A2)=1*A2,"")

F2:  =IF(A2<>"",LEFT(A2,1)="0","")

 

With the following headings in row1:

 

  Is Unique

  11 Characters

  Numeric

  Integer

  Leading Zero

 

You could then apply Conditional formatting to those cells.  Select cell B2 then press Ctrl+Shift+Right Arrow.  Do Home tab > Conditional Formatting > Use Formula...  and enter the following formula:

 

=NOT(B2)

 

Click the Format button and select sensible formatting, e.g. red from the Fill tab.  OK, OK.

 

Now select B2:F2 and drag down a sensible number of rows.  You will then be able to make mistakes, but you will be told which mistakes you have made...

 

Just a thought.

 

Cheers
Rich

 

PS.  FYI, the reason Ashish included the 1*A2 in the ISNUMBER and INT tests is because (from the Help file for ISNUMBER: "in the formula ISNUMBER("19"), "19" is not converted from a text value to a number value, and the ISNUMBER function returns FALSE", however when a numeric string ("19") is multiplied by 1, Excel converts it into a number.  Since you have to format your cells as text to keep the leading zero, your phone numbers are treated as strings.
