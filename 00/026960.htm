rob koch say:

MSE does generally clean up fairly well, though on occasion I had noticed what appeared to be leftover definitions in one of the folders, but these typically weren't over the 60-70MB of definitions and engine that are downloaded in the monthly package.

The mpcache- entries are a long-term cache of all of the files that have been scanned on the system, though I can't say I've ever seen one that large.  However, if you have a large file system, especially containing a NAS or external drives that you scan as mapped drives, then that's a believable figure.  The old file dates would be because the folder or drive that they relate to hadn't changed since that date, which could be likely with backup drives or application folders for example.

Overall, though your figures are a bit larger than most I've seen, this sounds reasonable and I wouldn't be concerned.  If you are scanning the archive files during a full scan this is likely also inflating the mpcache figure, especially if there are any ISO files on the drives as these will then be scanned as well.

If you want to stick with something like your current approach, you could have FILLIN fields on the later pages, too, and use Ctrl-A, F9 to force Word to display their prompts upon opening. Alternatively, you could use ASK fields on the first page, with cross-references to their bookmarks later in the document; in that case, you have the option of using Ctrl-A, F9 or of selecting the ASK field bookmarks and pressing F9 to get them to update.

In either case, you shouldn't use \* CHARFORMAT  & \* MERGEFORMAT switches in the same field.

chitrahaas nalla msft say:

Hi L_L123,

 

Based on your requirement, we would like to convey the following:

When you opened the Excel file which is already saved with specific format and if you save the changes made to saved Excel file, by clicking on Save icon in Quick Access Toolbar or File in Menu bar> Save. Changes will be saved to same format and will not get (cannot) saved in other format (which is set at Excel Preferences).

 

As your requirement, is to save every Excel file as .xlsx format. It is possible when you create a new Excel workbook. But, when you click on Save icon or File> Save, it will be opened with window and default format will be the format selected in Excel Preferences.

(the window appears as above, when you click on Save icon or File> Save for a new Excel workbook (which are not saved))

 

As a conclusion, what youre looking for is not possible for the files that are already saved as specific format.
