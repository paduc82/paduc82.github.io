Now for the problem:

Sometimes I'll save several recipes from the internet in one single standard (default Normal.dotm) Word document.  I will apply my styles as noted above.  The formatting may not be the same (I may have chosen a different font and text size for Heading 2, for example), but the style names are the same.

Later, when I want to print one of the saved recipes, I'll copy just that text from the original file (standard <ctrl>+C).  I'll open a new document with my recipe card template.  But when I paste the text into the new document, the margins of the new template are overridden by the old margins.  In other words, I don't have a 4x6 card layout, I have an 8 1/2" x 11" layout.

I should be able to copy the text into the new template, and have it look exactly as if I'd typed the text into the page manually.  Setting up the correct margins took a LOT of time, and the whole point of the template was to remove that need.

How do I prevent the old information from overriding the new template?  I don't want to lose all the formatting, or I'd just paste it into Notepad first.  I want to EASILY paste the text into the new template and have it look right.

I'd be glad to attach or send sample files if needed.

Thanks, all!

jay freedman say:

There are two things you need:

  • In File > Options > Advanced, under the "Cut, copy, paste" heading, make sure the option for "Pasting between documents when style definitions conflict" is set to "Use Destination Styles (Default)". It probably is set that way, but check it anyway.
  • When copying from the first document to the second document, be careful not to include any section breaks or the final paragraph mark of the document. To be able to see and avoid those things, turn on nonprinting characters by clicking the ¶ button on the Home tab. It's in those places that Word stores the information from the Page Layout dialog (paper size, margins, etc.), and copying them from one document to another will override the destination's settings.

IP Address : 192.168.137.1

Subnet mask : 255.255.255.0

Default gateway :  leave blank

 

Prefered dns server : leave blank

Alternate dns server : leave blank

 

now click ok > ok > close

h) Give network name "wifi" ( without quotes)

i) security type "wep"

j) security key  "1q2w3" ( Any 5 character passeword)

k) check save this network

l) next> close

e) Phone IP address > 192.168.137.2

     Primary DNS Address: 192.168.137.1
