paulcordts say: There are a couple of issues here

First  - If ActiveCell.Value = "1" Or "3" Then is not a correct format to use for what you want to do 

If ActiveCell.Value = 1 Or ActiveCell.Value =3 Then

Second - ActiveCell.Offset(1, 0).Select puts you in the wrong place

It the first row was deleted, then this ends up selecting column A in the second row of remaining data (which was the original 3d row - thus missing original row 2)

Remember that deleting rows changes both the active row and the row count.

I like to work from the bottom up to avoid these issues

try this

Sub test()

Dim i As Integer, LR As Integer

LR = Cells(1000, 1).Value
ActiveSheet.Cells(LR, 6).Select
For i = LR To 1 Step -1
    If Cells(i, 6).Value = 1 Or Cells(i, 6).Value = 3 Then
        ActiveSheet.Rows(i).Select
        Selection.Delete
    Else
     Cells(i,6).select
    End If
Next i
Cells(1, 6).Select
End Sub

This does select a cell or row to make it easier to follow when stepping through, but you dont even need to do this.  You can just directly delete the rows and not bother selecting cells and rows

Sub test()

Dim i As Integer, LR As Integer
LR = Cells(1000, 1).Value

For i = LR To 1 Step -1
    If Cells(i, 6).Value = 1 Or Cells(i, 6).Value = 3 Then
        Cells(i,6).Entirerow.delete
    End If
Next i
Cells(1, 6).Select
End Sub

I left the Cells(1,6).select at the end so you will have a visual cue when the routine is done.

Hi Erik van Broekhoven,

By design, when your Windows Phone's screen turns off, it is instituting several power saving features, including disabling the WiFi. This will happen everytime you turn off your screen or if Windows Phone's screen time out function kicks in and turns off your screen. When the screen comes back on, WiFi will be returned to its previous state.

If you are charging your phone and the screen stays lit, then yes, WiFi is still enabled. But if the screen goes dark, then WiFi is temporarily being disabled. Perhaps, since you are charging the device, the radio itself is not powering down, which would explain a quicker reconnection. Not 100% sure but it sounds like a viable solution.

Please note that if you do not have cellular data on your device and your screen goes blank, then you will not be able to get Internet-dependent items like e-mail, WhatsApp messages and other background tasks needing an active Internet connection. This is not a setting you can change.

Additionally, if you go into your device's Setting and then swipe right, you'll see "background tasks." Go into this and check that any apps you are expecting to run in the Background are listed and "on." If you don't see an App listed, tap on the "Advanced" button at the bottom of the list and see if the app appears in this list. If it does not appear, then either the app is not configured to run in the background.
