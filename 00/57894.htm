    Hi Alexander,

     

    Do you use a third-party cleaner utility or have you used it earlier?

     

    You might face this error after running a cleaner utility, which may have cleared out certain folder path references in the registry, assuming they are obsolete.  Or the problem may surface only after upgrading to Windows 8.1 (and registry cleaning done prior to the upgrade).

    If you have a registry cleaner utility called System Mechanic, then you need to undo" feature provided by the utility.

     

    Important: Follow the steps in this section carefully. Serious problems might occur if you modify the registry incorrectly. Before you modify it, back up the registry for restoration in case problems occur.

     

    You may also follow these methods and check if it helps.
    a. Press Win Key + R to launch the Run box.

    b. Type regedit.exe and press OK

    c. Navigate to the following key: HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows Search\FileChangeClientConfigs
    d. Right-click FileChangeClientConfigs and Rename
    e. Rename it to FileChangeClientConfigs.OLD.
