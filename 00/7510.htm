> What am I doing wrong?

You missed this in my post:
>> change the path (be sure to keep the backslash at the end)

So you need to change this line:
     strFilePath = "C:\LOCKIT"
to this:
     strFilePath = "C:\LOCKIT\"

The reason nothing was happening is that the Dir$() command takes that string and adds "*.doc*" to the end of it; without the backslash you're getting "C:\LOCKIT*.doc*" as the file to look for, and there is no such file (which would have to be in the root folder, C:\).

jay freedman say:
The following macro is based on the one in http://www.word.mvps.org/FAQs/MacrosVBA/CheckIfPWProtectB4Open.htm. Besides being specialized to apply a password, it has a few enhancements such as handling both Word 97-2003 (*.doc) files and Word 2007-2010 (*.docx and *.docm) files, and suppressing any Auto macros that might be in the documents.

You'll need to change the path (be sure to keep the backslash at the end) and password where they're set near the beginning of the macro.

Public Sub PasswordWholeFolder()
' based on http://www.word.mvps.org/FAQs/MacrosVBA/CheckIfPWProtectB4Open.htm
 
Dim strFileName As String
Dim strFilePath As String
Dim oDoc As Document
Dim PWD As String
  
    ' Set directory for batch process
    strFilePath = "C:\Temp\lockme\"
    ' Set password for documents
    PWD = "FooBar"
  
    ' Get name of first .doc* file from directory
    strFileName = Dir$(strFilePath & "*.doc*")
  
    ' Suppress any Auto macros in .doc or .docm
    WordBasic.DisableAutoMacros 1
          
    While Len(strFileName) <> 0
  
        ' Set error handler
        On Error Resume Next
       
        ' Attempt to open the document
        Set oDoc = Documents.Open( _
                   FileName:=strFilePath & strFileName, _
                   PasswordDocument:="?#nonsense@$")
      
        Select Case Err.Number
            Case 0
                ' Document was successfully opened
                Debug.Print strFileName & " was opened."

            Case 5408
                ' Document is already password-protected and was NOT Opened
                Debug.Print strFileName & " is password-protected " & _
                    "and was NOT processed."
                ' Clear error object and disable error handler
                Err.Clear
                On Error GoTo 0
                ' Get next document
                GoTo GetNextDoc

            Case Else
                ' Another error occurred
                MsgBox strFileName & vbCr & Err.Number & ":" & Err.Description
        End Select
      
        ' Disable error handler
        On Error GoTo 0
      
        ' "Dirty" the document so it will save
        oDoc.Saved = False
       
        ' Save with desired password
        oDoc.SaveAs2 FileName:=strFilePath & strFileName, _
            Password:=PWD, AddToRecentFiles:=False
      
        ' Close Document
        oDoc.Close
      
        ' Clear Object Variable
        Set oDoc = Nothing

GetNextDoc:
      
        ' Get Next Document from Specified Directory
        strFileName = Dir$()
 
    Wend
       
    WordBasic.DisableAutoMacros 0
  
End Sub
