jay freedman say:

The rotation of the second page is controlled only by the printer driver, not by Word. In the Print dialog, click the Properties button at the top right -- that opens the Printer Properties dialog, which is supplied by the printer driver and is different for each manufacturer (and maybe for different models).

I have a Canon MX860, which I hope has the same Properties dialog as your MX512. If your dialog is the same or similar, look at the Page Setup tab for a way to control which side of the paper would be stapled or bound if you were making a booklet. If it says "long side" change it to "short side", or vice versa:

If you get a choice of "Top" or "Bottom", it shouldn't matter which you choose; that affects only which edge gets an additional margin (to accommodate the staples) if you click the Specify Margin button. By default the margin is zero.

  • Place your cursor where you want the bookmark placed (or select some target text)
  • Do Insert  > Bookmark and name it (some characters are not allowed)

Once your bookmark is defined, you can refer to it:

  • Select the source text (typically, something like "This is explained in Chapter 8."
  • Type Command K
  • Select the Document tab
  • Click on the Locate button next to the Anchor field
  • Display the Bookmarks and select the target

Then, clicking on the hyperlink should take you to your destination within the same document.
