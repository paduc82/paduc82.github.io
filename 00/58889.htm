john rubdy say:

Hi,


  •  Are you facing any issue with COM+ System Application Service?


COM+ System Application Service will be set to “Manual” by default on Windows 7. There is no need to start the service.


The COMSystem Application Service is used for tracking and configuring COM+ components. COM+ can be used to develop enterprise-wide, mission-critical, distributed applications for Windows.


It Manages the configuration and tracking of Component Object Model (COM)+-based components. If the service is stopped, most COM+-based components will not function properly. If this service is disabled, any services that explicitly depend on it will fail to start.

 

Basically, this is the application that controls COM+ Event System. One of the support files that you'll probably never have any use for, but if you disable it, the warning notices you receive are worse than leaving it enabled. Disabling this service will generate EventLog entries noting it isn't running. An annoyance, but not harmful.



If you want to start the service you have to login as Administrator on the computer. Follow these steps:


a)      Press Windows logo key + R

b)      Type “services.msc” without quotation and press enter.

c)       Scroll down to COM+ System Application

d)      Right click on it and select Properties.

e)      Click on Start button on general tab.

To convert to a Local account, you can do the following:

  • Sign in to your Microsoft account on Windows 10 Technical Preview.

  • Click Users and accounts and under Your Profile click Connect to a Microsoft account.

  • Choose if you want OneDrive as your default file saving location, click Next.
  • Click Switch.
