How do I define a formatting style to automatically format the first - Microsoft Community

notrons say:

How do I define a formatting style to automatically format the first sentence of a paragraph?

I want to be able to create a style that makes all words in a paragraph BOLD until I insert a colon. Can someone help me? I know it's possible, but I'm not quite sure how to get there.

graham mayor say:
The macro is easy enough. The following will format each paragraph in the document up to and including the first colon as bold, provided the paragraph is formatted in the 'Heading 2' style indicated at the top of the macro. Change Heading 2 to the style in question.

Sub FormatParaStart()
Dim oPara As Paragraph
Dim oRng As Range
Const strStyle As String = "Heading 2"
For Each oPara In ActiveDocument.Range.Paragraphs
    If oPara.Style = strStyle Then
        If InStr(1, oPara.Range, ":") > 0 Then
            Set oRng = oPara.Range
            oRng.End = oRng.Start + InStr(1, oRng, ":")
            oRng.Font.Bold = True
        End If
    End If
Next oPara

Note that the style name must be exactly as it is shown in the style list.
What the macro will not do is format the paragraph on the fly.

Many combinations of arithmetic operations on floating-point numbers in Microsoft Excel and Microsoft Works may produce results that appear to be incorrect by very small amounts. For example, the equation
