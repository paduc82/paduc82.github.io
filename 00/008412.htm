Reset IE 9 :-

 

Here I am explaining how IE 9 reset takes place if you want to do it manually.

Launch IE 9 and click on tools from menu bar and select internet options, or you could simply type inetcpl.cpl in start search box to get the same.

Under General tab find browsing history where you can click on delete and select options you want to flush, i.e. cookies, temp files, form data etc.

Under browsing history if you select settings button you will get new window and you can click on view object and delete all downloaded files. (Sometimes this location does keep corrupt files which may affect IE performance)

On the Next tab Security make sure all the zones are on their default settings/levels.

Next tab privacy should also be on default level and make sure the pop-up blocker is ON unless you dont have any specific need to keep in off.

Under next tab Content, make sure Content advisor button is enabled and then click on Clear SSL state to flush the secure socket layer.

In the next tab Connections make sure it is set to never dial a connection if you are using broadband or DSL, FYI Dial connection only for single phone line connection.

Then click on LAN settings and make sure only first choice has check mark rest 2 should be unchecked.

Under next tab Programs click on Manage Add-ons and disable all the non-Microsoft provider unless you have a specific need to keep one of them enabled. Also make sure that IE is set as your default browser.

Under next tab Advance uncheck the line item called enable third party browser extensions to disable all unwanted toolbars.

At the end click on Apply and OK and close all the open windows for IE and launch IE.

Following this process will save you most of the troubles.
