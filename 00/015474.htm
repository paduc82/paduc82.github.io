Hi Bren,

Thank you for posting your query on Microsoft Community forums.

Is the issue specific to ebay website?

I appreciate your efforts towards resolving the issue. However, the issue may also arise due to an add-on incompatibility or due to an issue with GPU rendering. So, I would suggest you to try the following methods and check if the issue persists.

Method 1:

Try to run Internet Explorer without add-ons and check if the webpage is displayed correctly.

  1. Click Start, click All Programs and then click Accessories.
  2. Open System Tools.
  3. Click on Internet Explorer (No Add ons).
  4. Open the ebay website and check for the issue.

If the website is displayed correctly, then try to manage add-ons as detailed in the following link.

Manage add-ons in Internet Explorer

If the issue still exists, please try the next method.

Method 2:

The issue may also arise if the Internet Explorer move all graphics and text rendering from the CPU to the Graphics Processing Unit (GPU). So, you may check the website after enabling or disabling the the hardware acceleration to check for hardware or software compatibility issues with viewing certain Internet websites.

Please try the following steps:

  1. Click Start, and then click Internet Explorer.
  2. Click the Tools icon in the upper-right corner, and then click Internet Options.
  3. Click the Advanced tab, and then under Accelerated graphics, select the Use software rendering instead of GPU rendering check box. Clear the check box, if it is already selected.
  4. Click Apply, and then click OK.
  5. Close Internet Explorer 11 and then restart it so that the change takes effect. 

If the issue still exists, try Method 3.

Method 3:

If the ebay website was working fine with Internet Explorer 9, then you may try to change the user agent string in the developer tools:

  1. Open Internet Explorer and press F12.
  2. Scroll down to the option Emulation in the navigation pane, on the bottom-left of your screen.
  3. Click on the drop down menu for User Agent String.
  4. Select the version of Internet Explorer in which the webpage had the sign in option.

Create a registry entry ArchiveIgnoreLastModifiedTime and verify the result.

To create the ArchiveIgnoreLastModifiedTime registry value, follow these steps:

  1. Click Start, click Run, type regedit in the Open box, and then click OK.
  2. Locate and then click the following registry subkey:
    HKEY_CURRENT_USER\Software\Microsoft\Office\16.0\Outlook\Preferences
  3. On the Edit menu, point to New, click DWORD Value, type ArchiveIgnoreLastModifiedTime, and then press ENTER.
  4. Right-click ArchiveIgnoreLastModifiedTime, and then click Modify.
  5. In the Value data box, type 1, and then click OK.
  6. Exit Registry Editor.

    bob jones aka: cybertaz say:
    If the document has been formatted using Styles recognized by Word you need only Modify the Style. When you do, any text formatted using that Style will update to reflect the change.

    You can either use the Modify Style dialog or you can reformat one heading to which the Style was applied then use Update to Match Selection in the Style names list of the Toolbox.

    Additionally, Advanced Find & Replace can be used to replace one style with another or change specific formatting attributes... although doing the latter can be a bit dicey :-)
