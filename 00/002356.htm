andreas killer say:
Whether it is possible to create the files with the customer's account number depends on the layout of your file.

Run the macro below to export each printed page as single PDF file.

Andreas.

Sub Test()
  Dim WS As Worksheet
  Dim i As Integer
  Set WS = ActiveSheet
  'For 1 to NumberOfPages
  For i = 1 To ExecuteExcel4Macro("Get.Document(50)")
    'Export the page as single PDF
    WS.ExportAsFixedFormat xlTypePDF, _
    ThisWorkbook.Path & "\" & WS.Name & "-" & i, _
    From:=i, To:=i
  Next
End Sub

When it comes to the world of icons and graphics in general, comes down to where those items come from - there is a whole eco-system of providers of icon libraries which can be purchased on a royalty-free basis but the copyright remains with the creator of the library.

To be honest, I have no idea as to whether MS designs all their own icons or has purchased a existing commercial library. john korchok say:
The rule of thumb is that you can use Microsoft icons, clipart or fonts in works of your own. What is prohibited is reselling (or giving) those assets to others for use in their projects. Reproducing the user interface gets dicey. If it's for instructional use in a blog or book, you wouldn't have a problem. But if you were selling or giving away a collection of button-faces harvested from Office, you could get in trouble.
