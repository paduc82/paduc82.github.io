graham mayor say:

The code, which goes in the ThisOutlookSession module, is as follows. Change the display name of the account to the actual display name. Run Application_Startup manually from the VBA editor (any time you make changes to the code), or restart Outlook (saving the project at the prompt).

Because of Outlook's security, you will almost certainly need to self certify the macro project, or it won't run when you restart Outlook - hence the second link.

Option Explicit

Private WithEvents Items As Outlook.Items
Private Sub Application_Startup()
Dim olApp As Outlook.Application
Dim olStore As Outlook.Store
    Set olApp = Outlook.Application
    For Each olStore In Session.Stores
        If olStore.DisplayName = "Account Display Name" Then 'Change to what you have
            Set Items = olStore.GetRootFolder.folders("Inbox").Items
        End If
    Next olStore
End Sub

Private Sub Items_ItemAdd(ByVal item As Object)
Dim msg As Outlook.MailItem
    On Error GoTo ErrorHandler
    If TypeName(item) = "MailItem" Then
        Set msg = item
        msg.categories = "Red Category"
    End If
ProgramExit:
    Exit Sub
ErrorHandler:
    MsgBox Err.Number & " - " & Err.Description
    Resume ProgramExit
End Sub

Replace Account Display Name with whatever is displayed in Outlook as the name for that account and similarly use the spelling for the folder as shown. The following macro will loop through all the stores and folders and list the Account Display Name and the Folder Names

Sub Macro1()
Dim olStore As Store
Dim strStores As String
Dim olFolder As Folder
    For Each olStore In Session.Stores
        For Each olFolder In olStore.GetRootFolder.folders
            MsgBox olStore.DisplayName & vbCr & olFolder.Name
        Next olFolder
    Next olStore
End Sub
