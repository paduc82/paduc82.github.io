jay freedman say:

Come to think of it, that could help elsewhere in the document. Could you please advise me how to accomplish that, Jay?

The basic idea is that you insert a bookmark surrounding the first dropdown, and in each other place you insert a field that repeats the content of that bookmark. To insert the field, go to Insert > Quick Parts > Field, select "Ref" in the list of field names, select the bookmark name in the second column, and click OK. The result in the document is a field whose field code is {REF bookmarkname}, and when the field is updated it displays what's in the bookmark. You can insert the same field as many times in different parts of the document as you wish.

If the dropdown is a content control, either a combo box or a dropdown list*, you'll need a small macro to cause all the fields to update when the cursor leaves the dropdown. This goes in the ThisDocument module of the document (which must be saved as a macro-enabled document,  with a .docm extension):

Private Sub Document_ContentControlOnExit(ByVal CC As ContentControl, _
        Cancel As Boolean)
    Dim objCtrl As ContentControl
    If CC.Type = wdContentControlComboBox Then
        For Each objCtrl In ActiveDocument.ContentControls
            If objCtrl.Type = wdContentControlComboBox Then
                objCtrl.Range.Text = CC.Range.Text
            End If
        Next
    End If
End Sub

When you click on the REF fields, they'll appear as working copies of the dropdown content control. If you change the choice in one of those copies, the macro will change all the others, including the "real" control. Also, if you edit the list in the Properties dialog of the "real" control, then update the fields, all the lists will change to match.

If you use a legacy dropdown formfield instead, then you don't need a macro but you do have to use the Restrict Editing pane to allow only "Filling in forms". That means you can't change anything outside formfields unless you unprotect the document.

To set this up, insert a dropdown formfield with the needed choices, and check the two boxes in the Properties dialog for "Drop-down enabled" and "Calculate on exit". The bookmark name you enter in the Properties dialog is what you use in the REF fields elsewhere in the document, and the "Calculate on exit" causes the REF fields to update automatically when the cursor leaves the dropdown.

In this case, the REF fields will show only the text of the formfield and won't be working copies of the dropdown.

---

* The difference between a combo box and a dropdown list box is that the user can type anything into the box at the top, whether that text is one of the choices or not; a dropdown list box restricts the user to choosing one of the list entries.
