kristine arp say:

Hi Nafe,

Outlook has a default sent items folder where all the sent messages are being saved. You can find it on the left navigation pane, under Favorites. See the image below:

However, if you wish to consolidate all the sent messages to a specific folder, you can use Outlook rules. You have to create the folder where you want to move the sent items by following these steps:

  1. Click Folder.
  2. Click New Folder.
  3. Type the name of the folder.
  4. Choose Mail and Post items from the drop-down under Folder contains.
  5. Select where to place the folder.
  6. Hit OK.

To create the rule, follow the steps below:

  1. Click Rules from the Outlook ribbon.
  2. Choose Manage Rules & Alerts.
  3. Under E-mail Rules, click New Rule.
  4. On the Rules Wizard window, choose Apply rule on messages I send.
  5. Set the rule condition if you want to move specific messages. However, if you want to apply the rule to all the sent messages, do not choose any conditions and click Next.
  6. Check the box beside move a copy to the specified folder.
  7. Set the folder where to forward the sent messages by clicking the underlined word â€œspecifiedâ€.
  8. Choose the folder and hit OK.
  9. Click Next and choose the rule exception if necessary.
  10. Click Next.
  11. Edit the rule name if necessary.
  12. Check the box beside Run the rule now on messages already in â€œSentâ€.
  13. Hit Finish.

You can use this article as a reference on how to manage emails by using rules.

dan at it associates say:

OEM licenses for Office are not permitted to be used on any computer other than the original computer that came with the program.  Therefore, the license does no permit you to move it from the old dead computer to a new computer.

 

Regardless of whether the product went thru activation (either automatically or manually by phoning), moving the OEM license from an old dead computer to a new computer is in contravention of the End User Licensing Agreement (EULA) for the software so the installation is a nongenuine installation.

 

In order to maintain compliance with the EULA, uninstall the moved OEM software from the new computer and install properly licensed Office software on that computer.

dan at it associates say:

You wrote, "I'll use the license from the disk I used to load it."

 

Which I understand to mean that your impression is that the license is the disc.  That is not so.

 

According to your original post, I get the impression that you have two existing OEM licenses for Office 2003 that came on their respective computers.  You were trying to install one of these on a new computer.  One did not activate on the new computer, but the other one did.  You took this to mean that since it activated, it must be good.

 

I am trying to tell you that just because it activated it does not mean it must be good.  By definition, NO OEM license is permitted to be moved to another computer, period.

 

If you have a new computer that does not have an Office license, NO OEM license that came with another comuter can be legitimately installed on this computer.  According to the End User Licensing Agreement (EULA) for OEM licenses for Office 2003 the license lives and dies with its original computer and cannot be moved to another computer when the original computer goes bye-bye.

 

Just wanted to ensure the right informaton is presented, not just for you but for others who may see this post on the forum at a later date.
