fazil m say:

Hi,

 

A protocol is the special set of rules that end points in a telecommunication connection use when they communicate. Protocols specify interactions between the communicating entities.

 

One such protocol is the DHCP (Dynamic Host Configuration Protocol). DHCP allows a computer to join an IP-based network without having a pre-configured IP address. DHCP is a protocol that assigns unique IP addresses to devices, then releases and renews these addresses as devices leave and re-join the network.

 

Internet Service Providers (ISPs) usually use DHCP to allow customers to join the Internet with minimum effort.
