rohn007 say:

Win 8 the Search field is still present in Word 2007, 2010 and 2013 in the upper right corner of the Open dialog.  You can search for file names and text in indexed files.

 

In Word 2003 the Search is hidden under the "Tools" drop down in the Open dialog, you can search for text in files that are indexed by Windows Find service.  You can tell Windows which folders to include and exclude.

 

The indexing service "reads" files in the "index locations" and creates a list of keywords that you can search for.

 

In the past I turned off the indexing service (I didn't bother (forgot to) after doing recent Windows re-installs) and used a 3rd party tool called "Everything" from Void tools. It ties directly into the Windows File System, and therefore can provide effectively instant file name searches (but not  content searches).  It is simple to use and does not create the system overhead that the indexing service does.  I use LONG descriptive file names and I add keyword tags to the file names. I usually can find any file using the file name only.
