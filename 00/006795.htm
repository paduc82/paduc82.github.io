jay freedman say:

Macros of this kind have been written and posted on the Internet many times. There are lots of variations, depending on what specific settings each user requires. A couple of examples of setting a specific size are in this thread. A web search for "word picture resize macro" turns up many more.

Considerations for modifying the posted macros include (but aren't limited to) these:

  • Are all of the pictures set to In Line with Text, or are they all floating (i.e., any of the other text wrapping types), or are they a mixture?
  • When the pictures are resized, do you want to keep the original aspect ratio (height/width) or not?

The picture style requirement isn't as simple as it seems. The "styles" you see on the Picture Tools | Format ribbon don't exist in VBA; each one is a combination of many properties that the macro must set separately. There is no reliable guide to exactly which properties and values to set to match the "styles"; it's a matter of trial and error.

The following variation of HansV's second macro in the thread I mentioned will approximate the Simple Frame, White style after setting each picture's height to 2.5 inches and keeping the original aspect ratio.

Sub ResizeEm()
     Dim shp As Shape
     Dim ish As InlineShape
     For Each shp In ActiveDocument.Shapes
         Select Case shp.Type
             Case msoPicture, msoLinkedPicture
                 shp.LockAspectRatio = True
                 shp.Height = Application.InchesToPoints(2.5)
                 shp.Line.Style = msoLineSingle
                 With shp.Line
                    .Style = msoLineSingle
                    .Weight = 7
                    .ForeColor = wdColorWhite
                 End With
                 With shp.Shadow
                    .Blur = 4
                    .OffsetX = 2
                    .OffsetY = 2
                    .Transparency = 0.75
                    .Visible = msoTrue
                 End With
         End Select
     Next shp
     For Each ish In ActiveDocument.InlineShapes
         Select Case ish.Type
             Case wdInlineShapePicture, wdInlineShapeLinkedPicture
                 ish.LockAspectRatio = True
                 ish.Height = Application.InchesToPoints(2.5)
                 With ish.Line
                    .Style = msoLineSingle
                    .Weight = 7
                    .ForeColor = wdColorWhite
                 End With
                 With ish.Shadow
                    .Blur = 4
                    .OffsetX = 2
                    .OffsetY = 2
                    .Transparency = 0.75
                    .Visible = msoTrue
                 End With
         End Select
     Next ish
 End Sub
