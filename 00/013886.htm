doug robbins - mvp office apps & services (word) say:

By default, the Figure caption and the Table caption (in fact all captions), use the same Caption Style so, if you are modifying the style for one set of captions, the appearance of the other set will assume those modifications.

You could a separate Style for one set of captions, but then, when you want to insert another one of that type, as you will not be able to do that via the normal Caption facility, you would have to do it by copying and pasting an existing caption of that type and then editing the text as required.

elaine donald say:

Display size of all worksheets in a workbook

I have multiple workbooks and each has 15 sheets, all moved over to the MAC from a PC.  When each is worksheet is opened the display size is small and I want to change the size to 125% without having to change each worksheet.  How to you apply a global change to all worksheets?
bobgreenblatt say: The short answer is: you can't. The zoom size of each window is saved with the file. You can do it with VBA. Add a module, and then add this code:

Sub Zoom150()
For Each xx In Workbooks
    xx.Activate
    ActiveWindow.Zoom = 150
    Next
End Sub

Or, you could alternatively add a workbook open event macro to each workbook to zoom to 150%.

j.e. mcgimpsey say: One way:

My startup  add-in configures my windows, including default zooms. I also allow the use of a Named value (name: jem_WindowZoom, scope: worksheet) to override the default. Here's how (I'll include only the code relevant to zooms):

In a new workbook:

 1) Insert a new class module. In the Properties window, change the name to CApp. Put this code in it:

     Public WithEvents oApp As Application
    Const cnZoomDefault As Long = 125 'Change to suit.
    
    Private Sub Class_Initialize()
        Set oApp = Excel.Application
    End Sub
        
    Private Sub oApp_SheetActivate(ByVal Sh As Object)
        Dim nZoom As Long
        On Error Resume Next
        nZoom = Evaluate(Sh.Names("jem_WindowZoom").Value)
        On Error GoTo 0
        If nZoom = 0 Then nZoom = cnZoomDefault
        ActiveWindow.Zoom = nZoom
    End Sub

2) Insert a new code module. Put this code in it:

Public clsApp As CApp

3) In the workbook's ThisWorkbook code module put this code:

Private Sub Workbook_Open()
    Set clsApp = New CApp
End Sub

Save the workbook as an Excel add-in (.xlam). Add it using Tools/Add-ins... (or put it in your startup directory).

If you want any particular worksheet to display in a different zoom, create a name (Insert/Name/Define):

    Names in workbook:     Sheet1!jem_WindowZoom
    Refers to: =85

(change the sheet name to the particular worksheet).

arkengel say:

Hi Nick, 

There is no such thing as a stupid question. There are simply questions to things we don't know and would like to know more.

Unlocking your phone simply allows you to use SIM cards from other carriers that support the same wireless bands. (Example, an ATT phone could now use T-Mobile SIM card to be used in the T-mobile network)

This will have no bearing on how the phone receives updates. Using the same example of an ATT phone that's being used on T-Mobile, would still be subjected to when ATT releases updates. 

The reason this happens is because the ATT phone has what's known as an AT&T ROM version, which means that, when it's looking for updates it's checking with the server if AT&T has already released the update. So even if T-mobile had already released the update, it doesn't mean you could download it on this phone.

There are ways to install an unbranded ROM, but that requires some knowledge and of course is risky in the sense that if your phone stops functioning your carrier might refuse service (if they find out it's been fiddled with.)

In the Apple world this doesn't happen because Apple releases the updates directly from their servers to every phone regardless of network. They have this advantage since Apple has only a few models to support.

Android and Windows Phones live in a very big range of devices, therefore carriers don't want to risk a flood of problems happening with all the different devices. They also use this technique to withhold updates to encourage people to buy newer models.

I hope this helps.
