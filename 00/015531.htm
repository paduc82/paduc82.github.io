I originally thought that the OP might have been using a third-party AV app because the Limited Periodic Scanning switch appears in the Windows Defender settings. However, I just reproduced the issue on a machine that's running Windows Defender in a standalone configuration – by running this command at the elevated command prompt:

REG ADD "HKLM\SOFTWARE\Policies\Microsoft\Windows Defender" /v DisableAntiSpyware /t REG_DWORD /d 1 /f

 

And this was the result…

 

 

When I subsequently turned Windows Defender back on by running this command:

REG DELETE "HKLM\SOFTWARE\Policies\Microsoft\Windows Defender" /v DisableAntiSpyware  

 

The Windows Defender settings returned to their normal configuration:

Lithium Polymer batteries like other Lithium battery types in phones have a low voltage cutoff.

If the battery voltage is allowed to drop below 2.5 -3.0 volts the battery may be permanently damaged and the charging circuit will not charge the battery when the voltage gets that low. This is for safety, overheating/battery fires.

The battery has to be disconnected from the phone to measure the voltage.

You have to replace the battery to use the phone again.
