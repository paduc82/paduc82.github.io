See your point OssieMac. 

Nothing is bullet-proof since opening the workbook without macros disables completely the event catching.

In this case I propose a similar to read-only solution:

With the Template file:

Save As / Tools / General Options / Password to modify "myPass"

Now only authorised users can Open and Save template, all others can open as read-only but must save with other name...



Just for "personal" reasons I post a more complete VBA code. Works far better than the previous, yet macros must be enabled for it to work...


Private Sub Workbook_BeforeSave(ByVal SaveAsUI As Boolean, Cancel As Boolean)
Dim myPath As String, newFname As String, TemplateName As String, SMFl As Boolean

myPath = ThisWorkbook.Path
TemplateName = ThisWorkbook.Name 'Fix the templates name here, ex:"MyTemplate.xlsm"
Do
newFname = InputBox("Give NEW file name to save to:", "Cannot save as Original")
If newFname = "" Then Cancel = True: Exit Sub
If InStr(LCase(TemplateName), LCase(newFname)) > 0 Then SMFl = True
Loop While SMFl

newFname = myPath + "\" + newFname + Right(TemplateName, 5) ' For excel version earlier than 2007 replace 5 with 4

Application.EnableEvents = False
ThisWorkbook.SaveAs (newFname)
Application.EnableEvents = True

Cancel = True

End Sub

Thank you for posting to the forum regarding your Windows phone 7 experience.  The issues you are describing with the phone randomly shutting off and issues with battery life can be caused by a couple of things.  One of the most common is issues with the hardware.  This could be as simple as the battery or larger issues with the phone.  If this becomes the case you will need to contact your mobile operator or phone manufacturer in regards to getting the phone repaired or replaced.

 

The other possible causes are issues with the software.  There are a couple of things we can do to try and correct these.

  • First, we suggest you try performing a soft reset of the phone.  The easiest way to do this would be to remove the battery for about 2 minutes, replace and restart the phone.  If you are unable to remove the battery, consult your owner's manual or contact the phone manufacturer for the manual steps.
  • If the issue continues check the battery saver settings.  If the battery saver box is currently checked, uncheck the box.  If it is unchecked, check the box.  After you have done this, restart the phone and monitor the battery life.
  • The next thing to do would be to let the phone completely die.  Once it is completely drained, connect the phone and charge for about 30 minutes.  Disconnect the phone, then reconnect the phone and let charge for 4 hours +.
  • If you notice no change in battery life, the next thing to try would be another battery.  If you know someone else with a Windows phone, use their battery or have them use your battery in their phone.  This is to try and determine if the issue is the phone or the battery.
  • Finally, you can try a hard reset of the phone.  This will set the phone back to it's factory settings.  Before you do this you will want to back up your media, http://support.xbox.com/en-US/zune/software/syncing-media.  Contacts and calendar events should be backed up to the cloud.  You can verify this by signing into http://contacts.live.com and http://calendars.live.com.  All the information on your phone should be listed here.  Text messages and game progress will be reset.

Please let us know if you have any other questions or concerns.
