mrc89 say: I think I have managed to solve the problem...
The boot + start up time has been reduced from 1 min 15 sec to about 30 seconds (each including logon) with these steps (in case someone else has the same problem):

- WIN+R --> msconfig --> ENTER --> tick "no GUI boot" in the boot tab and disable all non-microsoft services in the services tab, set timeout to 0
- Disable all startup items using task manager, ccleaner and Tuneup Utilities
- Open control panel, search fonts --> hide all unnecessary fonts [5 seconds of boot time saved]
- Uninstall all old/unused programs
- Uninstall antivirus/antispyware (in my case Bitdefender & Spybot) and switch to Windows Defender [this saved me about 15 seconds of boot time!]
- set automatic logon (search for "netplwiz", uncheck "users must enter username + pw")
- clean up registry (ccleaner + tuneup utilities)
- defragment registry + system disk multiple times in a row until 0% fragmented (even if it says it is not necessary, it can make a difference)
- put HDD into first place in BIOS boot order
- run chkdsk (or tuneup disk doctor, essentially the same thing)
- Disable loading of programs using "Autoruns" (but be careful, some might be important)
- Run system scan (antivirus)
- In BIOS, set HDD to "optimal performance" rather than "quiet" (I can't hear a difference yet)
- Open device manager and disable all unused/unnecessary devices (in my case: bluetooth, synaptics biometric device (fingerprint reader), ethernet lan, unused printers...)

I think that was about it. Neither of the above tweaks was THE solution, but altogether, they saved me about 45 seconds of boot time.
