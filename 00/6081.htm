    How do I make custom Smart Art in PowerPoint themes? - Microsoft Community

    karenhebert say:

    How do I make custom Smart Art in PowerPoint themes?

    How do I customize the look of SmartArt?  I can set the colors, but SmartArt often uses colors that are too faint for proper viewing.  So I would like to create a custom palette that would come up for SmartArt that would be darker than the default settings.  
    bob jones aka: cybertaz say:
    Plainly stated, there is no facility in Mac PPT for doing what you describe wanting to do. It would require an Add-In to be written for the purpose.

    Two major concepts of SmartArt are (a) to obviate the need for a user to spend time creating diagrams, & (b) to insure consistent appearance throughout the presentation. As with most "packaged conveniences" there are few options for altering their design. You can individually modify each component of any given SmartArt diagram but there is no universal method for re-applying those same specifications.

    The colors used in SmartArt are determined by the Color Scheme imposed by the Theme or the Color Scheme selected from the list on the Themes tab. You can create & save a custom Color Scheme but, for several reasons, that won't give you what you're asking for...

    First of all, Themes [Theme Colors & Theme Fonts] are not object-specific. I.e., you can't apply 1 Theme to 1 object & a different Theme to another. Themes are global to the presentation, so when you select any given Theme all objects [all SmartArt, all WordArt, all Placeholders, & all Shapes] in the presentation are affected.

    Secondly, there is no provision for color palettes dedicated to any specific type of object.

    Enough of what can't be done :-) Here are some things that can be done;
    • Add custom colors to the basic Fill Color & Line Color palettes by opening one & selecting More Colors. Once added the swatches will remain available in any presentation. Once a custom color is applied in a file it is placed on the main color palettes as a new line of Recent Colors so it can be conveniently reselected from there.
    • For a specific type of SmartArt diagram, format it as you wish then add it to the Scrapbook. When you need another like diagram drag an instance from the Scrapbook to the slide. Scrapbook items are available in all Office documents, not just all presentations.
