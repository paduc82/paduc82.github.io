If the upgrade gets interrupted with the error code 0xc1800103-0x90002(I.e. with readable text message as "Something happened"), then the reason might be any of the below:

* Problem with Background services

* Antivirus running on you PC

* Unknown application creating conflicts on background

* Some drivers malfunctioning
