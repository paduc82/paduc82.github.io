stefan blom say:

AutoFit to Window should be the default when creating new tables in Word. However, to change the setting for existing tables, you can use the following macro:

Sub SetFitToWin()
Dim t As Table
For Each t In ActiveDocument.Tables
t.AutoFitBehavior wdAutoFitWindow
Next t
End Sub

For assistance with the macro, see http://www.gmayor.com/installing_macro.htm.
