sathish s v say:

Hello Naveen Gautam,

Windows 7 does not support Command Prompt to be run in full screen.

This happens because in Windows 7, the device drivers do not support running all of the DOS video modes. The device drivers are based on the Windows Display Driver Model (WDDM).

Follow the steps mentioned below to increase the layout of the command prompt window and chekc how it works.

a.       Click on Start button, and then click Accessories.

b.      Right click on Command Prompt option and select Properties option.

c.       Select layout tab.

d.      Under Windows Size option, you can increase the Height and Width of the Command Prompt window.

(or)

e.      Click Start button.

f.        Type CMD in the search box and hot enter.

g.       It will open the command prompt Window.

h.      Right click on the Title Bar at the top and select Properties.

i.         Select Layout tab.

j.        Here you can adjust the Height and Width of the Window.
