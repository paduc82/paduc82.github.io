Sounds to me like a display adapter driver problem.  This is a known issue currently, rather widespread for devices using AMD graphics cards.  It resulted from MS providing less than perfect driver updates for AMD Radeon display adapters.  Do you have an AMD display adapter?

Determine what brand name graphics card aka (display adapter, video card, graphics processing unit) you have on your device... by right clicking the Start menu>Device Manager>Display Adapter>Expand it>Properties>all right clicks.

If AMD is your brand name display adapter, you should update the driver by going to the manufacture's website HERE  selecting "Autodetect" then download what autodetect finds for you.  This is the easiest way to do it if you have an AMD display adapter.

If you don't have a AMD Radeon Display Adapter on your device, then this wouldn't apply to you.

My display resolution problems were corrected immediately after updating the Display Adapter Driver from the AMD website found above using AutoDetect.  Sleep mode was again found in the shut down options

jessemoll say:

Hi,

The command prompt (or "cmd") defaults to 80 characters wide.  You can make it wider by right clicking on the title bar on the top and selecting properties.  In the layout tab, change the "Screen Buffer Size" Width to something larger than 80 (depending on how wide your screen is... try 160 or 200).  This will allow you to make the command prompt window wider.
