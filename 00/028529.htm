Most VBA beginners thinks that they have to fill formulas in cells to get the results they want, because they are experienced with formulas. But that is the wrong way.

In VBA you can write formulas, and you can select and activate and goto and do all the other things that you see within the recorded macro. But you should not write VBA code as a recorded macro shows.

Please never use SELECT, SELECTION, ACTIVECELL, it is slow and error prone. Always refer to the objects directly. The macro recorder doesn't know where your goal is, that is the reason for that &%$"! code.

You said you want to check a list for a unique value, so let us start with a simple example. I assume that the value is in cell A1 and the list is in column B, then this is all what you need:

Sub Test()
  Dim R As Range
  Set R = Range("B:B").Find(Range("A1"), LookIn:=xlValues, LookAt:=xlWhole)
  If R Is Nothing Then
    MsgBox "Not found"
  Else
    MsgBox "Found in row " & R.Row
  End If
End Sub

That is that same as you search for the value manually, easy, fast, effective. Paste this code into a regular module, place the cursor at .Find and press F1.

john mcghie say:
That message means that the document has corrupted.

The cure is a Maggie:  see at the bottom of this post.

Just to add a little more explanation:  "not enough memory or disk space" means "something to do with saving the document failed".  The only possible reason the coders thought that could happen was because you were out of memory or disk space.  These days, that is the LEAST likely explanation.

The reason the document won't save is because its internal code has become corrupted.  The cause is often the use of Tracked Changes.  Chances are, that document has been tracking changes all its life, and you have switched off the display of tracked changes so you do not have to look at them.  If you turn the display of tracked changes back on, you will see that the document has now become impossibly complex, and that is why it has corrupted: it is now so complicated Word can't read its own handwriting.  If you "Accept all changes in document" then save and close the document, that may resolve the issue for you.

If the "Ignore All" list becomes large (more than five or six items) in a document, that usually indicates that either the text is in the wrong language, or the custom dictionary is in the wrong language.  Either way, the spelling checker is not working "well" if you have to ignore sufficient words to give you this problem.  Fix that issue and your work will be a lot more productive.

The Maggie is very likely to cure your issue.

This is not the place to be asking questions about Word 2010: that's the PC version, this forum is for the Macintosh version which is very different.

The Maggie:


1. Create a new blank document in .docx format

2. Carefully select all of the text in the bad document EXCEPT the last

paragraph mark

3. Copy it.

4. Paste in the new document.

5. Save under a new file name and close all, then re-open.


This technique for de-corrupting is known as "Doing a 'Maggie'", after

Margaret Secara from the TECHWR-L mailing list, who first publicised the

technique.


Hope this helps

jay freedman say:

The only way to make the change permanent is to modify the Footnote Text style, and to make that change apply to the Normal template or whatever other template your thesis documents are based on. Here are the steps (there are several ways, but this is the quickest):

  • Click in the text of any footnote.
  • Press Ctrl+Shift+S to open the Apply Style box. The name of the Footnote Text style should appear in the box; if some other name appears, then open the dropdown and select Footnote Text.
  • Click the Modify button to open the Modify Style dialog.
  • At the bottom of the dialog, click the "New documents based on this template" option.
  • Click the Format button and choose Paragraph in the menu.
  • In the Paragraph dialog, set the spacing Before and After to 0 (zero).
  • Click OK in both open dialogs. You can close the Apply Style box or leave it open; it won't affect how you work.
  • The next time you shut down Word, if you're prompted to save changes to the template, click Yes.

If you start a new document based on the same template and the spacing is still wrong, you'll have to open the template itself for editing (use the Open command instead of New) and make the same change there. Sometimes the "New documents..." option fails to save the settings to the template, maybe because the template is damaged or marked read-only.
