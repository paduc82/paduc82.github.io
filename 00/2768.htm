steve rindsberg say:

Sub Thing()

Dim oSl As Slide
Dim oSh As Shape

' Get a reference to the last slide:

Set oSl = ActivePresentation.Slides(ActivePresentation.Slides.Count)

' Look at each shape; if it's a footer placeholder, delete it:


For Each oSh In oSl.Shapes
    With oSh
        If .Type = msoPlaceholder Then
            If .PlaceholderFormat.Type = ppPlaceholderFooter Then
                .Delete
            End If
        End If
    End With
Next

End Sub

Hm.  Could toss an Exit Sub in there after the .Delete line if you want.  It'd save an immeasurable amount of time.
