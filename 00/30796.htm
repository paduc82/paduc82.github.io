Below is my usual troubleshooter for this sort of issue however also check these :

1. Uninstall ALL 3rd party antivirus and use Windows Defender which in Windows
8/8.1 & 10 is a full antivirus. WD provides both exceptional protection and superior
compatibility.

2. Suspect that there may be an issue with the USB platform on that computer.
In Device Manager check the WiFi network adapter and USB settings for any
detectable issues. You probably need to check with the System/Motherboard Maker's
Support or a real computer shop (one that does its own in-shop service).

3. Check for any other electrical/electronic devices in the computer area which
could be causing intermittent connection issues. Even phones can cause issues.

It is possible that the drivers you are using, even if the latest now available, are not
100% Windows 8.1 compliant. Check with your System Maker's Support (Motherboard
Maker for custom systems) and/or the actual device maker. The latter, such as Intel,
RealTek, Atheros and others may have newer versions not available at the System/Mother
-board Maker's site. And continue to check with them regularly over the next year +.
Also update the BIOS, low level chipset drivers, and the major on-board and separate
device drivers.

Method:

How to Format a Compact Flash Card?

 

Things you’ll need:

a.       Compact flash card reader

b.      Compact flash card

 

Instructions:

1.      Insert the compact flash card into your card reader. Make sure to face it the correct way to prevent breaking pins and reading strips.

2.      Connect the card reader to your computer via one of its USB ports. Allow the card a few seconds to connect before continuing.

3.      On the pop-up window on the screen select the option 'Explore Files'.

4.      Navigate up one folder until you can see the drive you just inserted. It should something like drive "H:".

5.      Right-click on the drive and click on the "Format" option.

6.      Make all the settings are at the default level.

7.      Make sure that the format is set to the "FAT32" file system. This is the file system that compact flash cards require to be navigable by computers.

8.      Click on "Format" at the bottom of the window. Allow the process several minutes to complete.

 

Tips & Warnings:

If you recently formatted the compact flash card, you can use the "Quick Format" option in the formatting window.

 

All content on the card will be erased when you format it, so back up any files that you will need in the future.
