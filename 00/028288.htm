How can I determine the column letter of the last visible column in a - Microsoft Community

camma47 say:

How can I determine the column letter of the last visible column in a range?

I have a worksheet that contains monthly data in columns.  I want to automatically show the results on another worksheet.  The months that are not yet completed are hidden.  How can I use a formula to automatically extract the column letter of the last visible column so I can use it in formulas that pull data from specific rows, so that the last unhidden month automatically populates the second worksheet?
jllatham say:

Here is a routine that would put the last visible column's letter from a sheet into a specific cell on another sheet.  The idea is that this code would be associated with the sheet where you want to pull data into (and I presume you're probably using INDIRECT() to use a variable column ID?).

Open your workbook, go to that second sheet and right-click on its name tab; choose [View Code] from the list that comes up.  Copy the code below and paste it into the code module presented to you.  Edit the code to change the two Const values to hold the proper sheet name of that other sheet, and the address of the cell where you want the column letter id to appear.  Press [Alt]+[Q] to close the VB Editor and return to the workbook.

Test it: choose any sheet, then choose the one you just put the code into - you should see the proper column letter as "$F" or "$A" or "$M", etc.  Go to the sheet with the hidden columns and unhide some or hide some more; go back to that other sheet, you'll see that the column ID has changed.

Save the workbook as a macro enabled workbook, type .xlsm or .xlsb (or even Excel 97-2003 type .xls)

Here is the code

  'this determines the last visible column on Sheet1
  'and places that column's letter ID in cell E1
  'on this sheet (Sheet2)
  'change these two Const values as needed
  'the name of the sheet with hidden columns
  Const hcSheetName = "Sheet1"
  'the cell on THIS sheet you want to put
  'the last visible cell's column ID into
  Const destCell = "E1"
 
  Dim dataWS As Worksheet
  Dim lastCol As Integer
  Dim lastVisCol As Integer
  Dim colPtr As Integer
 
  Set dataWS = ThisWorkbook.Worksheets(hcSheetName)
  On Error Resume Next
  'find last used column, visible or not
  lastCol = dataWS.Cells.Find(What:="*", _
   After:=dataWS.Cells(1), _
   Lookat:=xlPart, _
   LookIn:=xlFormulas, _
   SearchOrder:=xlByColumns, _
   SearchDirection:=xlPrevious, _
   MatchCase:=False).Column
  On Error GoTo 0
  'lastCol can be 0 if data sheet is empty
  If lastCol = 0 Then
    lastCol = 1 ' to prevent errors, set A as default
  End If
  'find last visible column
  lastVisCol = lastCol
  With dataWS
    For colPtr = lastCol To 1 Step -1
      If .Cells(1, colPtr).EntireColumn.Hidden = False Then
        lastVisCol = colPtr
        Exit For
      End If
    Next ' end colPtr loop
  End With
  'output results
  Range(destCell) = Left(Cells(1, lastVisCol).Address, _
   InStrRev(Cells(1, lastVisCol).Address, "$") - 1)
  Set dataWS = Nothing
End Sub

doug robbins - mvp office apps & services (word) say:

You need to insert a Section Break between pages 2 and 3 (probably a Next Page Section Break would be the most suitable, but you may have to remove any hard page break that you also have between those pages.
