So the feature was intended for the following two scenarios but I think what you want is slightly different:

 

1. You want to temporarily look at a summary of your tags. In this case you don't want dimming and you will have a look at the summary page and shortly after that delete the page (probably before your next tag search). This case will duplicate the search results if you search again before deleting the page.

 

2. You have your tags scattered and you want to consolidate them in a single page and work off of that from now on. In this case you want dimming and won't care about originals any more. You won't delete the summary page because it is the page of record now.

 

What you want if I understood correctly is the first case, but to be able to keep the summary page a little longer, and do more searches before deleting it. Unfortunately this is not available.

 

Summary pages are created as a snapshot and aren't dynamic.

What is the data type of the returned value? If it's text, try this:

=T(VLOOKUP(AR6,'Jan-31'!$AR$6:$BA$66,10,0))

Or, this generic version which will work for both text and numbers:

=IF(VLOOKUP(AR6,'Jan-31'!$AR$6:$BA$66,10,0)="","",VLOOKUP(AR6,'Jan-31'!$AR$6:$BA$66,10,0))
