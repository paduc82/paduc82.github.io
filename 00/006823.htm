 What about copying? That too is not a problem: Simply hold the [Ctrl]-key down while you drag a section to a new location. As soon as you let go of the mouse button, the section is copied. This even allows you to create a duplicate copy of a section inside the same notebook.

 

Please note that this only works with sections, not with section groups.

pkjt say: Hi there again,

I finally found a solution. Within the connection settings of WMDC there is a checkbox to allow or dissalow connections over various interfaces. By using this checkbox wmdc.exe is being called with some command line parameters. In my case (COM communication) they are:

C:\Windows\WindowsMobile\wmdc.exe /WRTCON 0 1 1 0 0 ""                 // for connection disable
C:\Windows\WindowsMobile\wmdc.exe /WRTCON 0 1 1 0 1 "COM1"       // for connection enable

Keeping this in mind, I wrote a small batch file and called it in admin mode. After that the reconnect works for me. Don't know what the flags 0 1 1 0 0 ... are being used for. The last one seems to control de-/activation.
