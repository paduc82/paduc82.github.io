   On Error GoTo Error_jpValuesOnly_Action

scottgem say:

Anita,

What you are missing is that you are using a Continuous form. A Continuous form is a SINGLE set of controls that is repeated for each record. This is a key point because what is happening is when you change the RowSource of the Floors combo then you are changing that Rowsource for ALL records in the subform. So any other records that have a different location and therefore a Floor value that is no longer in the current RowSource will now not be able to display that data.

The answer to this is to use a hybrid setup. You have a text box that overlays the combobox and displays the value so when the RowSource changes it doesn't change the textbox. Ken Sheridan has a sample app on his OneDrive that illustrates this. This thread has a link to his demo:

Cascading demo
