there are some situations where Windows doesn't know where to find the recovery environment (WindowsRE) any more and when that happens it can't create recovery media.

try this:

Unable to Create Recovery USB Drive

If you have imaged your drive, there is a chance the System Reserved partition is not in the same place that your Windows installation thought it was, and attempts to create a recovery drive will fail with the following error:

We can’t create a recovery drive on this PC

Some required files are missing. To troubleshoot problems when your PC can’t start, use your Windows installation disc or media.

When this happens, you can fix this problem by telling the program that creates the recovery drive where the Recovery Environment is located by entering the following commands into a command prompt run with administrator privileges:

reagentc /disable

result “REAGENTC.EXE: Windows RE is already disabled.”

reagentc /setreimage /path \\?\GLOBALROOT\device\harddisk0\partition1\Recovery\WindowsRE

This assumes your 450mb Recovery partition is partition1.  In my case, it was partition3.  Keep trying by swapping partition1, partition2, partition3, etc in the above command until the command is successful. 

You can also go into disk management by right-clicking the Windows 10 Start icon and click on Disk Management.  Then look which partition is the 450mb Recovery one for disk0.  For me (which was a result of a Windows 7 upgrade), it was the 3rd partition.  For a fresh installation it is probably the first partition and may be 500mb.

Then try again to create your recovery drive and it will probably work!

a. user say:

You won't want to hear this, but most people have no trouble with updates. Historically, the numbers show that virtually all Windows 10 PCs are completely up to date by the time a new feature update is released.

Unlike earlier versions, Windows 10 is not a complete product that gets updated every now and then. Instead, it consists of a continuous stream of releases that build out the operating system over time. They aren't going to stop.

For people who don't know how to deal with regular updates, or don't want to, this is really bad. And it's true that Microsoft is not very good at releasing updates - its competitors do this much better. But Microsoft is not going to stop updating its products and let itself fall behind Google, Apple, Amazon and Facebook, among many other competitors, because some people can't keep up.

1- Do not jump on a new update as soon as it gets released. The conventional wisdom is to wait at least 2 or 3 months before installing a new update. That gives professionals and corporate testers time to root out the problems that are pretty much inevitable in a huge project like Windows, and it gives Microsoft a chance to fix them.

2- If your computer is totally messed up, and you don't backup so you haven't got a known good version of your computer, your only choice is to pay a professional or do a clean install.

3- If your hardware isn't supported on Windows 10, and/or you're not the type who wants to keep up with the latest features, don't use Windows 10. You won't hear that from the Marketing Department, but that's the way it is.
