katrina men say:

Hi Gayla,

When you make a Contact private, this means that other people within the network cannot see the details of that certain person if you include them in your recipients. For example, you send an email to contact 1 with contact 2 and 3. Contact 1 is the private contact. When contact 2 and 3 hovers over contact 1's name, they will only see his email but not other details you've included in his information.

If you are using an Outlook app, this feature is available and you can make a contact private when you click on the lock icon in Outlook's ribbon beside the Follow Up flag.

talking moose say:
This is common where the sending server is Exchange and youre sending plain text messages. The servers text encoding is set to Windows text encoding. The sender cant change this but he can override it. The recipient, however, has no control.

By default Outlook for Macs setting is to automatically try to detect the correct encoding but it doesnt always do that. If you manually change the messages encoding from Automatic to Unicode (UTF-8) (Format menu > Text Encoding) then the message will arrive with the correct characters.

You can set Outlook to always use Unicode (UTF-8) in Outlook menu >  Preferences > Composing and enabling Preferred Encoding For New Messages: Unicode (UTF-8).

jason (office engineer) say:

Hi Amy,

Duplication is one option.  The gotcha with duplication is that it changes the URL of the Sway, but it's still the easiest way to move your Sway to someone else.  There is no limit to how many times a Sway can be duplicated.  With that said, owners of a Sway can disallow duplication of a Sway via a checkbox in the Settings pane.

We do now support transferring ownership of Sways, but it can only be done by your Office365 organization's admin and can only be done for Sways in an account that has been deleted.  Once the account is deleted, the Sways that belonged to it will begin to appear as deleted.  The admin has 30 days during which they can do this transfer, otherwise the Sways will become permanently deleted.  Here is an article on this:  https://support.office.com/article/Reassign-Sways-from-a-deleted-user-account-Admin-Help-9580E618-3C3E-4D28-A6EF-74C00A997248

Hope this helps!

Jason
