Thank you for clarifying the situation. You can select an existing backup when you modify the File History configuration through Control Panel.

  1. Connect both the current File History drive and the new one.
  2. Click the Start button, type Control Panel, and then hit Enter.
  3. Go to System and Security > File History.
  4. Click Select drive.
  5. Select the new drive and then under Select an existing backup, click the existing backup of your PC.
  6. Click OK.
  7. Choose between Yes or No to move the existing files and use if continuously for File History.
