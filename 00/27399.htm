The meaning of the slow ring is that you want to be conservative and not receive a build until more adventuresome folks have found the more serious bugs and Microsoft has made some fixes.  Microsoft only releases relatively stable builds to the slow ring.  The slow ring represents a later stage in software testing.

In other words, if you prefer to give feedback about usage and performance rather than dig around trying to break something (finding bugs) then stay in the slow ring.  That you are not seeing a build means its not ready for prime time yet.

If this is not your desire and you're ready to play, then by all means change to Fast Ring.
