In the past Usage history was only shown for a period of 6 months requiring you to contact support for a longer duration. May 2010 example

techfreak_ say:
Contacts are not deleted. Contacts linked through another outside source (Local Outlook program, MSN, etc.) are either not designed to follow you or have been deprecated. Only actual Skype created/imported contacts will remain from machine to machine. Your Skype history is only retained from 30 days to 2 years in the cloud. A period over that time in the application would only be located in the local history file (main.db) of your older machine/Skype 7.x software. This information could be transferred to another computer running classic Skype 7.x, as long as you had a copy of the undamaged main.db file. Any other data outside of the application or the Skype cloud history, is in the usage history.

nj joey say: Unfortunately, Palcouk's response is pretty much correct (despite being somewhat curt).  Atom powered Windows 8 tablets "should" be able to run all the apps that you might find in the Windows 8 App Store that run under the Metro tablet interface.  You'll find that anything more than basic apps running on the desktop will be a bit sluggish. 

There are some significant benefits to the setup, but some big drawbacks as well.  These machines run about the same price as the first batch of Windows 8 RT machines.  Unfortunately, RT machines can only run apps from the App Store.  They cannot run standard Windows desktop applications.  That's a big drawback unless you don't have a need for any legacy apps and you're happy with the limited options in the App Store. 

The Atom powered full Windows 8 tablets (like the ATIV 500T) basically give you the same full tablet experience as the RT machines, but they allow you to also run all standard Windows desktop applications.  These machines are tablets first and foremost.  However, for those times when you might need to run standard Windows applications, these machines will support that.  They're not really meant to be full Windows 8 replacement machines, but rather tablets that can run standard Windows applications when you need do.  The benefit to these machines is that they can be passively cooled (no fan) and they generally get great battery life.

If you want a tablet with more laptop-like performance with the desktop applications, you need to step up to one of the systems with the Core series processors.  These are basically laptops in tablet form (or convertible between the two).  These machines are generally significantly more bulky than the Atom tablets, have fans for cooling the processor, and typically have shorter battery life. 

So the ATIV 500T (and similar tablets) are sort of hybrids.  They're fully functional tablets and should run applications in the Metro interface smoothly.  They have the added benefit of also being able to run standard Windows applications (unlike RT) when you're need to.  You just have to accept that when running those applications, they're not going to be nearly as fast as when you run them on a laptop or tablet with a Core series procesor.

Unfortunately, there's not a whole lot you can do to speed it up.  These machines definitely have their place, but they do make some compromises in favor of less heat and longer battery life.
