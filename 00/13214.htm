gekras say:

Suresh,

 

I see nobody's replied for a long time, and this is an easy and useful thing to do in Excel, so here you go:

 

Type this into B1:

=FORMULATEXT(B2)

that gives (I think what you want): =Schedule!D105

 

Even better, try this:

=MID(FORMULATEXT(B2),2,(FIND("!",FORMULATEXT(B2))-1))

That gives Schedule!

 

And THEN, try =INDIRECT(MID(FORMULATEXT($B$2),2,(FIND("!",FORMULATEXT($B$2))-1))&B3&B4)

Now B3 specifies the column, and B4 the row, of the data you'd like to retrieve! By changing only which sheet B2 points to, you can have calculations that operate on whichever of however many worksheets you have! Yay!

 

So far, FORMULATEXT() is the only reason I've found to upgrade to Excel 2013.

suzanne s. barnhill say:

Web Layout view will always wrap to the window. If you want the window to take up half the screen, then you'll have to use the mouse to drag it to that size. If it is this size when you close Word, it should retain that size when you reopen it.

 

As for the "weird horizontal size" in Print Layout view, assuming that you haven't actually selected Landscape orientation, see http://WordFAQs.mvps.org/WhiteSpace.htm.

suzanne s. barnhill say:
When you open Word or a Word document, the size of the Word window will be the size last used. If this size was created by snapping (especially if you snapped a maximized window), then this probably wouldn't apply. You would need to change the window to a "Normal" window (using the Restore Down button) and then size it as desired. Unfortunately, the position of the window is also saved, so if you move the window to the far left or right of the screen, then any new window is going to open there, not on the other side.
