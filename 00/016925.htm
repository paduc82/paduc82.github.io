chip pearson say: Try the code below. If you want to prompt the user for a folder to list, leave the code in SECTION 1 as it appears here. If you want to use the current working directory, delete SECTION 1.  The code will list all files in the folder and their sizes in bytes. As written, the full file name (including the folder path name) will be listed. If you want just the file name with no folder path information, delete the line of code inside the Do loop:

FName = FolderName & "\" & FName 

This code will list only the files in the selected folder. It does not loop inside subfolders within the folder. If you want code to list files in subfolders, subfolders of subfolders, etc., post back with more detail.


CODE:

Sub ListFiles()

Dim FolderName As String
Dim WS As Worksheet
Dim R As Range
Dim FName As String
Dim FullName As String

FolderName = CurDir
' START SECTION 1
With Application.FileDialog(msoFileDialogFolderPicker)
    .AllowMultiSelect = False
    .InitialView = msoFileDialogViewList
    .Title = "Pick A Folder"
    .Show
    If .SelectedItems.Count = 0 Then
        Exit Sub
    End If
    FolderName = .SelectedItems(1)
End With
' END SECTION 1

Set WS = Worksheets("Sheet1") ' change sheet name as desired
                              ' or use Set WS = ActiveSheet to use active sheet
Set R = WS.Range("A1") ' change to desired listing start cell

FName = Dir(FolderName & "\*.*", vbNormal)
Do Until FName = vbNullString
    FullName = FolderName & "\" & FName
    FName = FolderName & "\" & FName ' delete this line if you don't want to show folder names
    R.Value = FName
    R(1, 2).Value = FileLen(FullName)
    Set R = R(2, 1)
    FName = Dir()
Loop
    
On Error Resume Next
ChDrive FolderName
ChDir FolderName
End Sub

Note how I changed the instruction to say that, when selecting the Chart Data, the left-most column in the selection will become the X axis data, and the right-most column in the selection will become the Y axis data.  Any columns in between will be ignored.

You will see below how to set the min and max for each axis.  (I commented out the code for the y axis, but it is there if you need it).

Edit: I've tweaked the code so that the data range is requested first, and the current selection is used as the default so you can select the data range first.  I also added the necessary error handling so that the code will exit if the user cancels either of the first two input boxes.

Here's the code:

Sub CreateChart()
    Dim rngChtLocation As Range, rngChtData As Range
    Dim strCompany As String
    Dim cht As Chart
    Dim ser As Series
    Dim ax As Axis
   
    ' What range contains data for chart:
    On Error Resume Next
    Set rngChtData = Application.InputBox( _
        prompt:="Select, with your mouse, the complete range of data you wish to graph (for a single Company)." & vbCr _
              & "The left column should be the X values and the right column should be the Y values.", _
        Title:="Select Chart Data", _
        Default:=Selection.Address, _
        Type:=8)
    On Error GoTo 0
    If rngChtData Is Nothing Then Exit Sub 'user hit cancel
   
    ' What range should chart cover:
    On Error Resume Next
    Set rngChtLocation = Application.InputBox( _
        prompt:="Select, with your mouse, a range where the graph should appear. The range can be in either workbook.", _
        Title:="Select Chart Position", Type:=8)
    On Error GoTo 0
    If rngChtLocation Is Nothing Then Exit Sub 'user hit cancel
    
    strCompany = Application.InputBox("Please enter the Company name", "Excuse Me", , , , , , 2)
   
    ' Cover chart location range with chart:
    Set cht = ActiveSheet.ChartObjects.Add( _
        Left:=rngChtLocation.Left, Top:=rngChtLocation.Top, _
        Width:=rngChtLocation.Width, Height:=rngChtLocation.Height).Chart
   
    ' Format the chart:
    cht.ChartArea.AutoScaleFont = False
    cht.ChartType = xlXYScatter
    cht.HasTitle = True
    cht.ChartTitle.Characters.Text = "Market Cap Progression for " & strCompany
    cht.ChartTitle.Font.Bold = True
    cht.ChartTitle.Font.Size = 12
   
    ' Delete all series from the chart:
    Do While cht.SeriesCollection.Count > 0
        cht.SeriesCollection(cht.SeriesCollection.Count).Delete
    Loop
   
   ' Add a new series:
    Set ser = cht.SeriesCollection.NewSeries
    ser.XValues = rngChtData.Columns(1)
    ser.Values = rngChtData.Columns(rngChtData.Columns.Count)
   
    ' X axis:
    Set ax = cht.Axes(xlCategory, xlPrimary)
    ax.HasTitle = True
    ax.AxisTitle.Characters.Text = "Time"
    ax.AxisTitle.Font.Size = 10
    ax.AxisTitle.Font.Bold = True
   
   ' X axis min and max:
    ax.MinimumScale = Application.Min(rngChtData.Columns(1))
    ax.MaximumScale = Application.Max(rngChtData.Columns(1))
   
    ' Y axis:
    Set ax = cht.Axes(xlValue, xlPrimary)
    ax.HasTitle = True
    ax.AxisTitle.Characters.Text = "MC Value"
    ax.AxisTitle.Font.Size = 10
    ax.AxisTitle.Font.Bold = True
   
       ' Y axis min and max:
'    ax.MinimumScale = Application.Min(rngChtData.Columns(rngChtData.Columns.Count))
'    ax.MaximumScale = Application.Max(rngChtData.Columns(rngChtData.Columns.Count))
End Sub
