hansv mvp say:

The B25 doesn't belong in there. I don't know what exactly you are trying to accomplish, but {=ROUND(MIN(IF(B4:B10>0,B4:B10)),-2)} will round the minimum of the positive values in B4:B10 to the nearest multiple of 100.

Warning: if ALL values in B4:B10 are either <=0 or blank, the formula will return 0.

Note for others: the { } are used to indicate an array formula; they should not be typed. The formula should be entered as =ROUND(MIN(IF(B4:B10>0,B4:B10)),-2) and confirmed with Ctrl+Shift+Enter. Excel will then display {=ROUND(MIN(IF(B4:B10>0,B4:B10)),-2)} in the formula bar.

teylyn say:

The Round() function needs a positive integer as the last parameter. Your formula has a negative number. Why? What are you trying to round to?

Also, your formula is missing a closing bracket for the Min() function, which should be after the B25 cell reference.

This formula works for me (confirmed with CTRL-Shift-Enter):

=ROUND(MIN(IF(B4:B10>0,B4:B10),B25),2)

Just to be sure, you can run the following macro that sets both AlternativeText and Title to the empty string, for both Shapes and InlineShapes:

Sub RemoveAltText()
'Stefan Blom, Microsoft Word MVP, May 2019
Dim x As Shape
Dim y As InlineShape

For Each x In ActiveDocument.Shapes
x.AlternativeText = ""
y.Title = ""
Next x

For Each y In ActiveDocument.InlineShapes
y.AlternativeText = ""
y.Title = ""
Next y
End Sub

Since Title isn't used in recent versions of Word, perhaps the document was initially created in an older version?
