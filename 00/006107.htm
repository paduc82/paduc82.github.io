john korchok say:

Text Box: An optional box that you insert using Insert>Text Box.

Text Placeholder: A preformatted area on a slide layout that includes a prompt for you to Click to add text. 

Shape: Inserted with the Insert>Shapes command, these don't include text by default, though you can right-click on a Shape and choose Edit Text to add text to it.

To set the text size of a Text Box, create a text box, set the point size to 16 points, then right-click on the text box and choose Set as Default Text box.

To set the text size of a text placeholder, choose View>Slide Master. The slide layout responsible for the current slide is displayed. If you only want to set the size for that layout, select the text levels and set them to 16 points. If you want to set the text size for all layouts, scroll up to the larger slide at the top of the thumbnails (this is the actual Slide Master), select the content placeholder and set the text levels to 16 points. The change will ripple through most layouts automatically.

To set the size of text of a shape, insert a Shape, right-click and choose Edit Text. Set the text size to 16 points, then right-click on the shape and choose Set as Default Shape.

When you paste, you don't want to bring any of the source formatting with the pasted text, so choose Home>Paste>Paste Special, choose Unformatted Text and click on OK. You can automate this a bit by adding a command to the Quick Access Toolbar:

  1. In PowerPoint, choose File>Options>Quick Access Toolbar.
  2. Change Choose commands from: to All Commands.
  3. Change Customize Quick Access Toolbar: to For PresentationName (the actual name of your presentation. This ensures that the QAT command travels with the presentation to all users.
  4. In the left-hand list, scroll down to select Paste and Keep Text Only.
