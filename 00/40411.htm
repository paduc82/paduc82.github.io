anil kumar b say:

Hi Winifred,

 

It is possible that some wireless configuration using the same address range could get pass the filtering. There could be a router that is set with the default IP address range. There is also a chance that the SSID could be the same - a lot of people call the network something such as Home or Wireless and the chance is there that two nearby systems could be named the same - this is more likely when the routers are from the same manufacturer and are left at default.

 

I would suggest you to refer to the methods and check if it helps.

 

Method 1:

To Delete Wireless Network Profile using "Forget this network"

 

a. On your desktop, click/tap on the Network icon in the taskbar notification area. (see screenshot below)

 

b. Right click or press and hold on the wireless network (ex: Brink-Router) in the network list, and click/tap on Forget this network. (see screenshot below)

 

 

If issue still persists, refer to method 2:

 

Method 2:

To Delete Wireless Network Profile in a Command Prompt


a. Open a command prompt, by pressing Windows key + X and clicking Command Prompt (Admin).
b. In the command prompt, copy and paste the command below, and press Enter.

NOTE: This command will give you list of all wireless network profiles on your PC.


netsh wlan show profiles

c. In the command prompt, type the command below, and press Enter.
NOTE: This command will delete the wireless network profile.


netsh wlan delete profile name="WirelessProfileName"

For example, if I wanted to delete my listed Brink-Router (name) wireless network profile, I would type the command below exactly.

netsh wlan delete profile name="Brink-Router"

d. When finished, you can close the command prompt if you like.

andre for directly say:
Open Start > Settings > Update & security > Troubleshoot
Scroll down
Click Network adapters
Click Run the Troubleshooter

When complete, restart to see if the problem is resolved.

If that does not work

Press Windows key + X
Click Device Manager
Expand Network Adapters
Select the Network card listed,
Select Update Driver
This should give you a choice of drivers - select the oldest one
Wait a few minutes while it installs the old driver

or

press Windows key + X
Click Device Manager
Expand Network adapters
Right click your adapter
Click Uninstall


Exit Device Manager, restart. Windows 10 should detect the network adapter then reinstall it. Check if you can connect and browse.

also try:

Open Start > Settings > Network & Internet > Status


Scroll to the bottom then click Network reset.
