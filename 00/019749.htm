Conversation view sorts messages by Subject not message ID. If you have messages where the sender has left the Subject line blank then all messages will appear together when grouped by Conversation, regardless of whether the content is related.

This is currently how Outlook for Mac is designed to work. Please be sure to let Microsoft know you'd like to see this feature changed in future versions by using the Help > Send Feedback mechanism in any Office application.
