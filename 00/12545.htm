jim gordon mvp say:

VPN is a very slow technology. When using VPN through DSL or even cable networks you should expect very slow performance. If your network connection isn't 100 Megabits per second with a fast ping on both your end and the server side you will definitely notice a slowdown.


You can save yourself a lot time by copying the file from the server to your local hard drive. Work on the file locally, then copy the file back to the server when you're done.

jim gordon mvp say:

Additional thoughts...


The default Excel file format (.xlsx) is plain text with folders for media content that has been zipped. This file format is called Office Open XML and is the international standard for all spreadsheet applications (not that most of them adhere fully to the open standard - Excel is about the only one that does). This file format may be the source of your consternation, as there is a lot of zipping and unzipping going on at both ends of your server connection, plus saving in plain text is the absolute slowest way to move computer bits from one machine to another.


Excel offers another file format called Excel Binary Workbook (.xlsb). Try saving your workbook in (.xlsb) format and see if things speed up a bit over VPN. Binary is very efficient, from a computing point of view, but you can not easily hand-code the files saved in this format using a text or XML editor instead of the Excel interface.

daniel g. grau say:
OK, I found another setting will do that too, although it definitely should not: Keep lines together. This setting should only trigger when a paragraph reaches a page boundary, not when wrapping around a picture.

Nevertheless, in a document laid out similarly to yours, applying KLT to the second paragraph causes it to elude the bottom of the picture, like in your document.

You probably have that applied to both paragraphs, given that the problem still manifested itself when you deleted the first Return. (Deleting a Return to join two paragraphs makes the format of the first paragraph prevail.)

So go to Format > Paragraph, select the Line and page breaks tab at the top right, and take it off if it's on.

Else, post back and we'll keep on trying.
