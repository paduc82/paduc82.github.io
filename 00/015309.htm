Hello,

Greetings !!

Thank you for posting,

I understand your office 365 tenant has multiple Yammer network, You can browse to Yammer Admin center > Network Migration to see the list of Yammer networks associated with your office 365 tenant.

 

Multiple Yammer networks may get created if the domains that are added in office 365 have a past history of Yammer either a free Yammer network was created using this domains or domains might be connected with other tenant which was using Yammer enterprise.

 

Once you verify this domains in Office 365 the Yammer network associated with that domains gets connected to your office 365 tenant and you have multiple Yammer network scenario.

 

I would like to inform that having multiple Yammer networks in one office 365 tenant is not recommended by Microsoft, since in Multiple Yammer network scenario, we might get issues like users from same tenant redirected to different Yammer networks and not able to see the data / groups on other  Yammer network, user profile, profile pictures not syncing from office 365 to Yammer, users of same tenant getting redirected to different Yammer networks with separate URL.. Also the new feature  Yammer Integration with Office 365 groups Now Rolling out is not available in multiple Yammer networks scenario.

 

In order to fix this issue, we will have to perform Network Migration and consolidate multiple Yammer networks into one so all the domains that are verified in your office 365 admin centre gets synchronised to a single Yammer network

 

 

It is recommended to initiate a Network Migration from Yammer network whose Data is important and need to be preserved.

 

 

If you wish to have export data of the Yammer network before migrating, you can login as an admin to respective Yammer networks by creating a test Global Admin account with the domain of  the respective Yammer networks that you wish to migrate and export the data from Yammer network Admin > Export Data. (Note: the exported Data will be available only for reference purpose and cannot be imported back to Yammer.

 

Please follow below steps to perform network migration.

 

1.       Login into yammer home network with Global admin rights

2.       Click on gear icon> Network admin> Network migration

3.      Step 2 of 3 - Choose a Yammer Network to Migrate page.

6.       Click next

7.       Enter the domain name of the domain to migrate > Start Migration

Network migration: Consolidate multiple Yammer networks
