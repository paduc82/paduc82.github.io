brian tillman [outlook mvp 2007-2019] say:

If you see "Outlook Email", then your account is also a migrated one. For some reason, US accounts show "Outlook Mail", while accounts in the UK and other countries can show "Outlook Email".

The first time you click the Archive button, you should see this:

The drop-down to the right of "Select folder" should just give you a list of your existing folders or the option to create a new one. If you don't see this, it's a sign you already chose a folder for the archives, perhaps without realizing what you were doing. It's also conceivable that there is a bug, but let's see if we can find the Archive folder.

Turn on the Reading Pane. Send yourself a very distinctive message from some other mailbox, if you have one. When it arrives, select it and click Archive to move it. Then use the "Search Mail and People" function above the folder list and look for that specific message by entering that distinctive text in the search field and pressing Enter. When the message appears, select it. It should appear in the Reading Pane and the name of the folder containing the message should appear below the sender's name, like this:

You now know what your Archive folder's name is. You might also slide down to the bottom of the folder list on the left to see if there is a folder whose name is not in alphabetical order. That could be your archive folder.

Frankly, the Archive feature is unnecessary and misleading. All the Archive button does is to move a message to another folder in your mailbox, whether named Archive or something else. It's operationally no different that clicking "Move to", except that the destination folder is predesignated so there are fewer clicks. The message is still in your mailbox and still taking up space in that mailbox.
