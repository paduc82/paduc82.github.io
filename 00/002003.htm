ronrosenfeld say: If you don't want to do this manually, you will need a VBA macro.  Since you haven't shown in detail your data or your desired output, I put the following together to give you some ideas.

The macro assumes
  • Your data starts in A1
  • The first row consists of labels
  • The Names are in the first column
  • The data is in a contiguous area bounded by an empty row and and empty column.
  • The results will go into a few columns to the right of the data, on the same worksheet.
  • The results will have the "search name" in the upper left.

See how it works out for you, and post back for clarifications.

And please, run this on a COPY of your data, not the original.


To enter this Macro (Sub), <alt-F11> opens the Visual Basic Editor.
Ensure your project is highlighted in the Project Explorer window.
Then, from the top menu, select Insert/Module and
paste the code below into the window that opens.

To use this Macro (Sub), <alt-F8> opens the macro dialog box. Select the macro by name, and <RUN>.


===========================================
Option Explicit
Sub Report()
    Dim rSrc As Range, c As Range
    Dim rDest As Range, rCrit As Range
    Dim colNames As Collection
    Dim i As Long
    Dim vNames As Variant, v As Variant
    Dim vRes() As Variant
    Dim FirstAddress As String
Set colNames = New Collection

'You need to ensure that rSrc is set to a
'cell in your data range

Set rSrc = Range("A1").CurrentRegion
Set rCrit = rSrc(1).Offset(columnoffset:=rSrc.Columns.Count + 2).Resize(rowsize:=2)
Set rDest = rCrit(1).Offset(columnoffset:=3)
    rDest.Resize(columnsize:=2).EntireColumn.Clear

'Get list of names
For Each c In rSrc.Offset(rowoffset:=1).Resize(rowsize:=rSrc.Rows.Count - 1).Columns(1).Cells
 vNames = Split(c.Text, ",")
 On Error Resume Next
    For Each v In vNames
        colNames.Add Item:=Trim(v), Key:=Trim(v)
    Next v
On Error GoTo 0
Next c

'Generate report for each name
For Each v In colNames
    rCrit(1) = rSrc(1): rCrit(2) = "*" & v & "*"
    rSrc.AdvancedFilter Action:=xlFilterCopy, _
        criteriarange:=rCrit, _
        copytorange:=rDest, _
        unique:=False
    With rDest
        .Value = v
        .Font.Bold = True
    End With
    Set rDest = rDest.End(xlDown).Offset(rowoffset:=2)
Next v

rCrit.ClearContents

End Sub

Additionally, I need workbook 2 to be saved in the same location as workbook 1 is saved in, with a name that references workbook 1 and also says "Paste Values" (for example, "Workbook 1 - Paste Values"). I hope that is enough detail. If anyone could help, I would appreciate it!

andreas killer say:
Sub Test()
  Dim DestWB As Workbook
  Dim WS As Worksheet, DestSheet As Worksheet
  Dim Data
  Dim i As Long
  Dim NewName As String
  Dim fso As Object 'Scripting.FileSystemObject

  'Create a new workbook
  Set DestWB = Workbooks.Add(xlWBATWorksheet)
  Set DestSheet = DestWB.Sheets(1)
 
  'Visit each sheet in this file
  For Each WS In ThisWorkbook.Worksheets
    'Read the data into an array
    If WS.PageSetup.PrintArea <> "" Then
      Data = Intersect(WS.UsedRange, WS.Range(WS.PageSetup.PrintArea))
    Else
      Data = WS.UsedRange
    End If
    'Be sure we have an array
    If Not IsArray(Data) Then
      ReDim Data(1 To 1, 1 To 1)
      Data(1, 1) = WS.UsedRange
    End If
    'Count the sheet
    i = i + 1
    'Add one if necessary
    With DestWB
      If i > .Worksheets.Count Then
        Set DestSheet = .Worksheets.Add(After:=.Worksheets(.Worksheets.Count))
      End If
    End With
    'Write the data into the sheet
    DestSheet.Range("A1").Resize(UBound(Data), UBound(Data, 2)) = Data
  Next

  'Create the filename
  Set fso = CreateObject("Scripting.FileSystemObject")
  NewName = fso.BuildPath(ThisWorkbook.Path, _
    fso.GetBaseName(ThisWorkbook.Name) & "- Paste Values." & _
    fso.GetExtensionName(ThisWorkbook.Name))
  'Save the file
  ActiveWorkbook.SaveAs NewName, FileFormat:=xlWorkbookNormal
End Sub
