suzanne s. barnhill say:

What icon? Just selecting single spacing (no matter how you do it) does not remove any Before/After spacing that may be applied. To remove it you can

  • Select the No Spacing style (this actually changes the style, however, which you won't want to do if something other than Normal is applied).
  • Click on the Line and Paragraph spacing button in the Paragraph group on the Home tab and click Remove Space Before Paragraph or Remove Space After Paragraph if either is available.
  • Go to the Paragraph dialog and select all your spacing and other formatting options in one go.

Note that, while Remove Space Before/After Paragraph will remove any amount of Spacing Before/After, using the Add Space Before/After Paragraph option adds 12 points in either location. If you want to add a different amount, you do have to do it through the Paragraph dialog.

I'm writing a document, that contains programming code. For programming code I created specific style and want to create macros, that will find and replace all curly quotes to straight in this specific style. 
jay freedman say:

The macro that does the same steps is below. Change the name of the style in the indicated line to the name of your style.

Sub CodeStraightQuotes()
    Dim rg As Range
    Set rg = ActiveDocument.Range
    Options.AutoFormatAsYouTypeReplaceQuotes = False
    With rg.Find
        .Text = """"
        .Replacement.Text = """"
        .Forward = True
        .Wrap = wdFindContinue
        .Format = True
        .Style = ActiveDocument.Styles("Code")     '  <== change if necessary
        .Execute Replace:=wdReplaceAll
    End With
    Options.AutoFormatAsYouTypeReplaceQuotes = True
End Sub
