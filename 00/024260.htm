The picture fill function would be on the Text Fill menu rather than the Shape Fill menu if it was available, but it isn't there -- but only for the "new" WordArt version. Fortunately, you can get the old WordArt version by making a Word 97-2003 (*.doc) file in Word 2013.

If you've made a new blank *.doc and created the WordArt there, you can copy/paste it into your *.docx and it will continue to be old-style WordArt.

Hello,

Excel starts each sheet on a new page. This is by design. If you want the contents to appear on one page, then you need to pull it onto one sheet first. 

You can do that with a helper sheet, where you reference the cells from the other sheets, or copy and paste the content on one sheet before printing.

teylyn say:
Add a new sheet to the workbook, just for printing. In cell A1 of that sheet, enter a reference to cell A1 of the first sheet.

=Sheet1!A1

Copy down and across until all data from Sheet 1 shows. You can use the Format Painter to copy and paste the formatting from Sheet1 to the new sheet.

Then, below that data, create a reference to Sheet 2

=Sheet2!A1

Copy down and across and copy the formatting. Repeat for the third sheet, then print only the helper sheet.
