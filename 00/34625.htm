The reason why this happens has to do with technical limitations of Windows and Thumb Drives.  The memory used in flash drives has a number of write-cycles. To minimize wear on the drives Windows saves all write operations to a "cache" file on the local HD. When a full "block" worth of information is in the cache, it gets sent to the flash drive in a single write operation.

If you don't properly "flush" the cache by properly closing the flash drive before disconnecting it you stand the risk of leaving part of the file behind on the HDD! That causes file corruption.  Unfortunately it also appears that for DOCX files, the body text portion of the file structure is at the end of the file. Making it the part most likely to be lost.

a. user say:

Hi!

 

The system might be restarting because of various reasons.you might need to change some system settings or some settings in the BIOS.

 

You can try to find out what woke up Windows from its last sleep.

1.       Click on start and type ‘cmd’ in the search.

Open command prompt and type “powercfg –lastwake’

Note down the details provided after running the command .

 

Also run the command powercfg -devicequery wake_armed_in the command prompt to get a list of the devices that are currently configured to wake up Windows from sleep.

 

Next, disable the network card to wake the computer.follow the below steps :

1.       Click on start

2.       Open Control Panel
Click Device Manager
Select Network Adapters .Expand it and select the ethernet controller or LAN card.

3.       Right click on it and open properties.go to power management. Deselect "Allow the computer to turn off this device to save power" .

4.       Go to "Advanced" tab. put value off for "Wake from shut down" --->value none for "Wake up capabilities"
press OK

 

Other possible causes that can wake up the computer are :

 

1.      Windows Update (if set to automatic) – check the time of install (usually 3:00 AM). Go to Control Panel. Then open windows updates .click on change settings on the left side and check the setting for the time.

2.      Automatic Backup programs are able to wake up the computer to run scheduled backup. In Control Panel ,open windows backup and restore and check the settings.

 

 

There might be some options in your BIOS too that you might need to turn off. Refer to the manufacturer’s guide on how to get in the BIOS.

Look for any options relating to Power Management/ Automatic Power Up/Power Loss Restart etc  and turn them off.Also check for BIOS APIC options .

 

It could also be possible that the system is not shutting down properly.please ensure that when you shut it down,it turns off completely.

 

Please revert back with the results

 

Hope this helps
