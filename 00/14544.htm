Create index of cross-references? - Microsoft Community

tom r. 303 say:

Create index of cross-references?

Hi,

I am trying to figure out if there is a way to create an index of cross-references in Word 2011. Specifically, I am writing a legal opinion and I have a list of different requirements. I'm using a customized auto numbering format so that the list items appear as "Requirement 1" and so forth. There are different types of requirements, and I would like to create an index that would list the different requirement types as a main entry (e.g. "Division Order Requirements") and then list the numbered items as a sub entry. Hopefully I would end up with something that looks similar to this:

Advisory Comments
   Requirement 1
   Requirement 2
   Requirement 5

Division Order Requirements
   Requirement 3
   Requirement 4

Operational Requirements
   Requirement 6

Thanks in advance for any help or advice.

Tom R.


john mcghie say:
Hi Tom:

A "normal" Word index will do this, but you need to hand-tag the document text.

So an index tag of { XE "Advisory Comments:Requirement 1" } would produce:

Advisory Comments
     Requirement 1

Each colon (:) you add to the XE tag text steps in another level.

Somewhere in each paragraph you need to add an XE tag.  If you want to be very clever, you can add a REF tag inside the XE tag to copy in the number of the paragraph.  That allows you to edit the text and have the numbering change without having to re-do all the XE tags.

So { XE "{_Ref247119503 \r \h} Advisory Comments:Requirement 1" }

I wrote an article about generating indexes in Word years ago: http://word.mvps.org/FAQs/formatting/createindex.htm

I am not sure it will be much help to you: you are not "actually" creating an index, you are using the index generator for a nefarious purpose :-) 

Microsoft has an article that covers the basics here:
http://office.microsoft.com/en-au/word-help/create-an-index-HP005189283.aspx

Note that anything for Word 2003 will work in Word 2011, anything later -- the PC Word Ribbon is not the same as the Word 2011 ribbon.

That article refers to a Concordance Index to "automatically" mark entries.  Sorry: It won't work for you, so don't try it.  That article also refers to "Choosing a format" for your Index.  There is only ONE choice:  "From template".  If you choose any other format, you lose control of the formatting because Word will reset all the styles used every time it generates the index.

This should be enough, but please don't hesitate to get back to us if you need a little more detail.

Hope this helps
