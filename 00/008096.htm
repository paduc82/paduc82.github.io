john w. vinson say:

What kind of object is vwCollegeMatrix?

 

You need not have a control on the report to use the field in conditional formatting. Just use the fieldname.

 

The other problem I see is that you're comparing the date field to Now().  Now() does not return today's date; it returns the current date and time, accurate to a few microseconds. It's almost certainly NOT going to match any value in your table. If [Primary Status Date] contains just a pure date with no time component, compare it to Date() instead of to Now(); if the table field contains a time component and you want it to match any time on the current day, use a criterion

 

>= Date() AND < Date()+1

  [txtStatusDate] = Date()

jan kuera say:

Save Word 2013 document with 2010 compatiblity

Hello,

how can I save Word 2013 document so that it can be opened in Word 2010 and have the same layout? That is, how do I turn on compatibility mode for document that is not using it?

The section Word Options | Advanced | Compatiblity options for: is empty. I guess that's what I am looking for to change.

Thanks,

Jan

suzanne s. barnhill say:

Word 2013 doesn't allow you to directly change the Compatibility setting on a new document, but you can use a snippet of code (courtesy of Jay Freedman) to do this.

Press Alt+F11 to open the macro editor, and press Ctrl+G to open the Immediate window at the bottom of the editor. In that window, paste this line and then press Enter:

 

ActiveDocument.SetCompatibilityMode wdWord2010

 

Close the macro editor. Then open the Options dialog, go to the Advanced options, and scroll to the bottom. Change the dropdown "Lay out this document as if created in" to "Microsoft Word 2010". At that point, the list of 2010 layout options will appear below the dropdown.

 

For a full explanation, see http://answers.microsoft.com/en-us/office/forum/office_2013_release-word/where-can-i-find-suppress-extra-line-spacing-at/70bf7ca3-a884-40c4-ab59-34d2a04a1a8f.
