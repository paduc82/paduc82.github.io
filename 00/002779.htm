    To add a ringtone to your phone using a Windows Vista, Windows 7 or Windows 8 PC

    1. Connect your phone to your PC, then do one of the following:

      • On a Windows Vista or Windows 7 PC, click Start, then click Computer.
      • On a Windows 8 PC, click Desktop, click File explorer, then click Computer.

    2. Under Portable Devices, double-click your phone, double-click the Phone folder, then double-click the Ringtones folder.

    3. Find a sound file on your PC (for example, music files are often in the Music folder and organised by artist and album), then drag the file into the Ringtones folder.

      Note

      You need to move a file from your computer to your phone to add or rename a ringtone. You can't add a ringtone by moving a file from a folder that's already on your phone or renaming an existing ringtone.

    4. On your phone, go to Settings > Ringtones + sounds > Ringtone. Your new ringtone will appear under Custom in the list of available ringtones.

    is there a function similar to the SQL IN in excel - Microsoft Community

    adey_981 say:

    is there a function similar to the SQL IN in excel

    I need to be able use a function similar to the SQL "IN" function, is there such a thing in excel or would this have to be a UDF.
    jeeped say:
    ... I need to be able use a function similar to the SQL "IN" function, is there such a thing in excel or would this have to be a UDF.

    There are a number of ways to look into a part of a string or to match one entry in an array. Here's one:

    =MATCH(10, {5,10,15},0)
    ... returns 2. If 10 was 11 and therefore not in the {5,10,15} array, it would return #N/A. Use an IFERROR() wrapper to get your False return as in,
    =IFERROR(MATCH(10, {5,10,15},0),FALSE)
    ... any result that is not an error will produce a positive integer which is non-zero, aka TRUE.

    Essentially the SQL IN operator asks whether a value is IN an array of other values and returns a boolean true or false. Excel has numerous Lookup Functions that meet this criteria.
