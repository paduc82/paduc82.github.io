What i would like is 1 month 29 days, what am i doing wrong. 

thank you in advance. 

hansv mvp say:

DATEDIF with the "md" argument returns the number of days within one month.

You can use DATEDIF with the "m" argument to return the number of months:

=DATEDIF(TODAY(),W3,"m")

If you would like to return the number of days and months in one cell, you can combine them:

=DATEDIF(TODAY(),W3,"m")&" month(s) and "&DATEDIF(TODAY(),W3,"md")&" day(s)"

If, by occasionally, you mean that you're looking for programs that only scan, Malwarebytes' is a good program to install, partly because the free version actually removes items that it detects.

 

Please note that it is never recommended to run two real-time antivirus programs at the same time, as such a setup has a major impact on performance, and the programs can reduce effectiveness of each other. From the knowledge that I have gained from the internet regarding the difficulties users have had with McAfee, it may be the antivirus that would conflict with others, if the other way around would not additionally true.
