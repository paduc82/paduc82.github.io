j w stuart say:

RAID requires a minimum of 2 hard or Solid State Drives.

 

There are two types of RAID arrays.
1) Software RAID
2) Hardware RAID

Hardware RAID requires a controller chip on the motherboard, usually a Gigabyte or JMicron chip.

See:
Installing Hardware RAID on Windows 7
http://www.nikmakris.com/blog/post/Installing-Hardware-RAID-on-Windows-7.aspx

How to Create a Software RAID Array in Windows 7
http://www.howtogeek.com/howto/36504/how-to-create-a-software-raid-array-in-windows-7/

Information regarding Windows 7 software Raid:
http://www.alanjlee.com/interesting-bits/a-few-notes-on-windows-7-software-raid/

Nothing is absolutly fool proof, should the RAID controller chip fail then both drives are probably currupt.

If I had a choice it would be RAID 5 or 50.

Start PC, just as the manufacturers logo disappears and Windows attempts to load, press and hold Power Button for 5 - 10 seconds to perform a Hard Shut Down
