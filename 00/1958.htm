These file sizes are expected - we now use delta updaters which is a diff between what you have installed and the files you need to upgrade. Hotfix releases (where you see .1 or .2 at the end of the version) usually weigh in at 80MB, whereas version to version upgrades (e.g. 15.23.x to 15.24.x) are in the range of 130 - 180 MB. Microsoft AutoUpdate is responsible for inspecting your current machine state and offering you the smallest possible update.

Sub tryme()
  Dim myType(6) As Double
  myDate = Range("B1").Value
  For j = 1 To 6
    myType(j) = Cells(j + 3, 2).Value
  Next j
  With Worksheets("Output")
    LastCol = .Cells(1, Columns.Count).End(xlToLeft).Column + 1
    .Cells(1, LastCol) = myDate
    For j = 1 To 6
     .Cells(j + 2, LastCol) = myType(j)
    Next j
  End With
End Sub

An IME is an input method editor (see https://docs.microsoft.com/en-us/windows/desktop/dxtecharts/installing-and-using-input-method-editors). Apparently they're used only for Asian languages.
