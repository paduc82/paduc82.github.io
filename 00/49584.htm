rmac3 say:

First do a backup of windows, dont do an exact copy because that will have the old drivers and that is the problem. Then start REGEDIT and drill down to:

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\storahci\StartOverride

Change DWORD from 3 to 0 and reboot. Press down the key to get you into the BIOS and change IDE to ACHI. Reboot and hopefully it will go into safe mode and windows will load the new drivers. If it wont you can try troubleshooting to force the driver change. If that wont work then you have to reinstall.

neilpzz say: Try going to msconfig, select safe mode and restart. Then at restart enter the bios and change the setting to AHCI. Boot into windows in safe mode and go back to msconfig and change back to normal start and restart.
