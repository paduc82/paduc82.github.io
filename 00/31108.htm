Reduce Start Up Programs:

A. Click Start, type MSCONFIG in the search box, and then press Enter.

teching together say:

Device Manager Automatically Opens on Startup

Posting this question so I can answer it for anyone else who runs into this issue since the older thread about this is now locked with no complete solution listed.      

https://answers.microsoft.com/en-us/windows/forum/all/device-manager-automatically-opens-on-startup/480ed799-b3d6-4481-ae4f-3270545b5bf3

Basically, the issue is that Device Manager is oddly in the Startup programs, so tries to start automatically whenever a user logs in.     You don't notice with a local admin account, but standard users get a warning because they don't have admin privileges.  

Seems to only impact:

  • Windows 10
  • Computers from Refurbishers, so I'm guessing it's a leftover from some of their imaging techniques.


teching together say:

The problem appears to come from a registry key that adds 'devmgmt.msc' into the startup. 

At least on the computers we had that were impacted, I was able to clear up by deleting a Registry key located in:

HKLM\Software\Windows\CurrentVersion\Run

The key was different on different systems, but was always in that location.     Here's a screenshot for the more visual folks:

Usual caveats apply...be careful with editing the registry as you can break things....
