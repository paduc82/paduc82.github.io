What does error code: 0xc0000225 mean and how can I reboot my - Microsoft Community

chirojo say:

What does error code: 0xc0000225 mean and how can I reboot my computer?

A notification on my computer has indicated that my PC/device needs to be repaired

error code: 0xc0000225

What does this mean and how can I repair it.

a. user say:

Hi,

The error code you provided is a Stop error which indicates that the problem resides on the UEFI. The traditional BIOS utilizes Master Boot Record (MBR) while UEFI uses GUID Partition Table (GPT) to help the system achieve fast boot. In some cases, when the system confuses with these two schemes; it will generate the stop code 0xc0000225.

To fix this issue, we suggest that you Create Windows 10 Installation Media.

Then follow the steps under "Using the tool to create installation media (USB flash drive, DVD, or ISO file)".

After that, please follow the steps below:

1. Plug-in  the installation media and start your system.

2. You will see "Press any key to boot from CD/DVD..." message. Press any key and click on Repair your computer.

3. Next, select the Troubleshoot option and click on Advanced Options. Under Advanced Option screen, click on Command Prompt.

4. Then, in the Command Prompt, type these commands below (in bold letters). Press Enter key after each command:

  • Diskpart (this command will load DiskPart utility)

  • list volume (will lists all volumes available on your system)
  • select volume 2 (take note that you need to select the volume which is labelled as ESP)

  • assign letter=Z (this will assign Z temporarily to the ESP volume selected previously)
  • exit (this command will get you to the normal command prompt mode)

  • bcdboot C:\windows /s Z: /f UEFI (this command will generate boot files for UEFI)

Close the command prompt and reboot your computer. Your system should boot normally.

In any case that you need further help, don't hesitate to post back.
