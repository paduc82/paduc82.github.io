If you want to continue using the new installation, you will have to install all the missing programs again. You will also have to copy your data (not programs) from one of the old disks.

 

You should have persevered with correcting one of the identical disks, which you still can. All that is needed is:

2. Copy NTLDR and NTDETECT.COM from C: (new installation) to the root folder of the old disk, probably drive E.
