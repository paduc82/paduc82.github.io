BUGS: Pasting PNG results in Black instead of transparent. - Microsoft Community

thenonhacker say:

BUGS: Pasting PNG results in Black instead of transparent. Inserting PNG files results in wrong Pixel Dimensions

BUGS: Pasting PNG results in Black instead of transparent. Inserting PNG files results in wrong Pixel Dimensions

Pasting PNG results in Black instead of transparent. 
1. Copy a PNG image from any browser. 
2. Paste it in an Outlook Email. 
3. BUG: The transparent part of the PNG is black instead of transparent. 
4. The Pixel Dimensions are correct though.

Inserting PNG files results in wrong Pixel Dimensions 
1. Save small PNG icons on your computer. 
2. In an Outlook Email, click Insert Picture, then pick the small PNG files you've saved. 
3. BUG: The image dimensions of the inserted icons are larger than what they are supposed to be, and this also causes the image look resampled/blurred. 
4. The transparency is correct though. 
5. WORKAROUND: For each inserted image, edit image properties, type the correct width / height in pixels.

SOLUTION: Whether a PNG is pasted, or inserted from file: 
1. The background of the image should remain transparent. 
2. The image dimensions in pixels should be correct.

In the Mac OS X file system, a forward slash is used to separate objects (aka files and folders). Adding a forward slash to the file name tells the OS that the part prior to the slash is a folder and the part after the slash is the complete file name. While not all applications will have issues reading these files, I can assure you that adding forward or back slashes to a file name is a very poor practice.

Also worth noting is that files with a forward or backward slash in the name cannot be read by Windows. This is briefly addressed in the following help file next to Avoid certain characters in file names:
Best practices for working with Windows

OneNote 2007 to 2010 Upgrade

I have been asked to upgarde to OneNote 2010 and I have been using 2007 .. Is the upgrade seamless i.e. do I have to back up any files or will 2010 install over 2007 without any loss of data ?
malte ahrens say:

The upgrade is seamless (to the extent that when you open the upgraded version of OneNote, all your notebooks are there, if I remember correctly).

As far as I'm aware, the upgrade doesn't even touch your old OneNote files (e.g. upgrade them to the 2010 format) until you yourself do it.

All this being said, there is a tiny chance (as with every action you take on your computer) that something will go wrong, so if it's super critical stuff, I'd probably back it up first (you would likely have a backup of it anyway if it's that important).
