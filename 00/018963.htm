tom ogilvy say:
The equation of a line is generaly Y = a + bX or in Excel function terms Y = Intercept + slope times X if you want to provide a Y and solve for X it would be (using algebra) X = (Y - a)/b using functions provided by Excel you would make that into x = (Y - Intercept())/Slope() (you would need to provide the known Y and X's as arguments to the function same as you use them as arguments to the Trend and Forecast functions.) The trend and forcast functions solve for the slope and intercept behind the scenes so to speak. You can get them directly using the slope and intercept functions. If you plot a trend line on a graph you have have it display the formula of the trend lines which will show the slope and intercept as well. Just make sure you format the display to show a significant number of decimal places or you will be using truncated coefficients which might not give good results. -- Regards, Tom Ogilvy
