bob jones aka: cybertaz say:
PowerPoint is not an embeddable file type on a Mac. The file embedded as an icon on a Windows system will not open. Your best bet is to have the sender supply you with a separate copy of the presentation file.

However, if the Word document is in the .docx format there is another option to try... It may be effective but isn't completely guaranteed. You need a copy of StuffIt Expander or a comparable utility [although the process may vary with a different app -- I've never used anything but Expander].

  1. Drag the Word file to the Expander icon in your Dock. A folder bearing the same name followed by the word Folder will be created in the same location as the file.
  2. Open that folder, then open the Word folder within it.
  3. Open the enclosed folder named embeddings
  4. The presentation file will be there
  5. When you open it you may get notifications of repair or content removed, but the file should open once you accept the notifications
