hosch250 say:

Securing your computer with Syskey

Technical Level : Basic

Summary
This article will show you different methods of securing your computer by using different settings in Syskey.

Details

The Syskey control is designed to help secure your system against malware by requiring a password to be entered at boot.  To access and change settings in Syskey, open the Run command by pressing Windows-R, or by finding it in any of the numerous other ways available, and type "Syskey", then press Enter:

If you click the Update button, this window opens:

These different options require different types of interaction from the user.  The method selected in this screenshot is most likely the method used by your computer - this method does not require any user interaction, the system just enters the password at the right time.  The option above this one requires a disk to be inserted at boot; the tip says a floppy disk, but this may just be an archaic notification, I have not done this personally.  My web searches indicate a floppy drive is still required, and a USB stick cannot be used; if you have done this, feel free to leave a comment and I will update the article.  The top option requires the user to manually type a password when the system boots.  On Windows 8.1, the password must be entered even before the lock screen shows, almost as soon as the computer is booted.  When you are done, click OK to save your changes.
