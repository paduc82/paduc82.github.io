Hi Chau,

Do you mean the .PST file size limit in the Outlook client? If so, please refer to the information below:

1. The default .PST file size limit in Outlook 2010 and Outlook 2013 is 20GB.

2. When the file size is greater than 20GB, you will be unable to open the .PST file.

3. When the file size reaches 20GB:

a. Reduce the size of the .PST file (maybe delete some mails) with these steps:

1) Start Outlook 2010 or later with a profile that contains your original .PST file that is larger than 20 GB.

2) Click FILE->Info->Cleanup Tools->Archive.

3) In the Archive box, enable the Archive this folder and all subfolder option.

4) In the list of folders, select the .PST file that you want to archive.

5) In the Archive items older than drop-down list, specify a date during which a fairly large number of items will be archived.

6) In the Archive file box, select the destination .PST file for your archive messages.

Or you can refer to the following article to enable the archive mailbox in Exchange Online:

https://technet.microsoft.com/en-us/library/jj984357(v=exchg.150).aspx

b. Configure the size limit of .PST files described in the article below:

https://support.microsoft.com/en-us/kb/832925

If I have misunderstood anything, please let me know.

Micosolf office

 

doug robbins - mvp office apps & services (word) say:

Sentences are supposed to start with a capital letter.

If you mean at the beginning of every line, you should not be pressing Enter at the end of the previous line.  Word is far smarter than a typewriter and will start a new line for you automatically when the text will no longer fit on the line.  However, if you deliberately want to start a new line that is not a new paragraph (which would normally start with a new sentence and hence the capital letter), use Shift+Enter instead of Enter.
