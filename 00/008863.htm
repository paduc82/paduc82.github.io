Without getting into the weeds.............SPDY (pronounced ‚Äúspeedy‚Äù) is a web protocol developed by Google, which is yet to be certified as a web standard.  But it is already part of Google Chrome and was just incorporated into Internet Explorer 11.  Its main purpose is to improve the performance of websites by minimizing latency and reducing page loading time by 64% through a process of "layering" multiplexed connections.  Faster loading means better scrolling (in your case) because it is the image loading that causes erratic scrolling.

Since SPDY3.1,etc is not fully implemented in Internet Explorer 11 yet, it often has the reverse effect, interfering withTCP/IP connections (thus the ubuquitous, "this page cannot be displayed" message)  Disabling SPDY for the moment, reverts IE back to the standard HTTP/HTTPS protocols.
