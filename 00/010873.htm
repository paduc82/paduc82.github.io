van dinh say:

>>The problem is that I want the tip to display before the user has typed in the character ...<<

... then KeyPress is certainly an incorrect Event to use. There is no way you can display the help tip for the first character with the TextBox_KeyPress Event.

 

From what you described, I think you need to use 2 Events:

* TextBox_GotFocus Event to show the help tip for the first character to be entered.

* TextBox_Change Event to display the tip for succeeding characters after the first character. 

In the Change Event, your code must use the Text Property of the TextBox and not the Value of the TextBox since the Text Property value changes as the user enters character by character while the Value Property remains static (most likely Null) until the entered text is about to be committed, i.e. at the TextBox_BeforeUpdate Event.

 

If you are not familiar with 2 Events or the Text Property mentioned above, please check Access VBA Help / you Access books for detailed explanation as well as possibly, sample code.

The issue you describe sounds a lot like an issue with the digital Zune certificates.  I would like you to review the following article and try these steps on your computer, http://support.microsoft.com/kb/2468307.  The following steps are ones that can be performed on a machine running XP.  There are notations for Windows 8 but that is only for where steps may differ.

TRUST THE ZUNE CERTIFICATE

  1. Close the Zune software and disconnect your phone from your computer.
  2. Open Certificate Manager. To do this, click Start, type certmgr.msc in the Start search bar, and then press your Enter key.

    Windows 8 users:     On the Start screen, type Certificate, tap or click Settings in the right column, and then tap or click Manage computer certificates.
  3. In the pop-up window, tap or click the Personal folder in the left column, and then double tap or double click the Certificates folder in the right column.
  4. Click the heading that says Issued By to sort the column, and then look for one or more certificates that read "zune-tuner://windowsphone/...".
  5. Right-click the certificate file and click Copy. If there is more than one file, hold the Ctrl key, click each file, then right-click a file and click Copy.
  6. Tap or click the Trusted Root Certification Authorities folder in the left column, right-click the Certificates folder, and then click Paste.
  7. Click Yes to install this certificate.

Create a new certificate

  1. Close the Zune software and disconnect your phone from your computer.
  2. Open Certificate Manager, if it is not already open. To do this, click Start, type certmgr.msc in the Start search bar, and then press your Enter key.

    Windows 8 users:     On the Start screen, type Certificates, tap or click Settings in the right column, and then tap or click Manage computer certificates.
  3. In the window, tap or click the Personal folder in the left column, and then double tap or double click the Certificates folder in the right column.
  4. Click the heading that says Issued By to sort the column, and then look for one or more certificates that read "zune-tuner://windowsphone/...".
  5. Right-click each certificate file, click Delete, and then click Yes.
  6. Tap or click File in the menu, then Exit.
  7. Click Start and in the search box enter %userprofile%\Application Data\Microsoft\Crypto\RSA, and then press Enter.

    Windows 8 users:     Right-click in the lower left-hand corner of the screen and tap or click Run, or type Windows Key + R, and then type the string above.

    Note For some enterprise customers, the string will be %userprofile%\AppData\Roaming\Microsoft\Crypto instead.
  8. Find the folder that begins with "S-1-5-21-".
  9. Right-click the folder and then click Rename. Rename the folder so that it ends in .bak. This means you will likely be adding ".bak" to the end of the number.

    Note If there is more than one folder that begins with "S-1-5-21-", try this procedure with one folder at a time. If it does not work, rename the folder to the original name and try the next folder.
  10. As soon as you're done, open your Zune software and connect your phone to your computer. This creates a new certificate and folder.

If you have any trouble with these steps let us know by describing which step you get stuck on or what contents may not be found on your computer.  If you do have an issue like Squire_Guy describes where no personal certificates are found, try the following steps.

  • Create a new Windows user profile on your computer and then attempt the process again.  The steps to create a new profile can be found here, http://support.microsoft.com/kb/27978.

If you continue to have issues finding and resetting the digital certificate we need to verify this is an issue on the computers end and not the phone's.  The best way to do this would be to connect to another computer with the Zune software.  If the phone is recognized on the alternate PC, you will want to have your computer looked at as there may be some issues within the OS registry.  If the phone is not recognized on the second computer, it would be time to contact the mobile operator/phone manufacturer in regards to getting the phone serviced or replaced.

Please let us know if you have any other questions or concerns.

paul az say:

Proration based on dates - HELP!!!!

I have to figure out what percentage of time occurs between two different dates on a monthly basis.

For Example:   Start Date: July 1, 2010

                      End Date:  September 15, 2010

 

What percentage of this time is in June 2010, July 2010, August 2010.....

I know that there are basically four conditions:

All before = 0%

All After = 0%

Spans = need to calculate

All in = 100%

Now I need the formula

joeu2004 say: Paul wrote:
I have to figure out what percentage of time occurs between two different dates on a monthly basis.

For Example:   Start Date: July 1, 2010
                      End Date:  September 15, 2010

What percentage of this time is in June 2010, July 2010, August 2010.

Suppose A1 and A2 contain the start and end dates.  And suppose that A4:A8 contain the first-of-the-month dates 6/1/2010, 7/1/2010,..., 10/1/2010, which you can format to display just the month and year.

Then put the following formula into B4 and copy down through B8:

=MAX(0, MIN($A$2,EOMONTH(A4,0))-MAX($A$1,A4)+1) / ($A$2-$A$1+1)

formatted as Percentage.

If you get a #NAME error, see the Help page for EOMONTH.  If you do not want to rely on the ATP, replace EOMONTH(A4,0) with DATE(YEAR(A4),1+MONTH(A4),0) or with A5-1 if you are willing to ensure that there is always a date in the next row of column A.

a. user say:

Hi ali bai,

 

As what the previous Moderator said, if you have synced your account using POP, the changes that you made on your Gmail account would not reflect in Outlook.com since POP is a one way connection to send and receive emails. You need to mark the messages as read on both accounts. It would not automatically reflect on the other account that you have read the message.

 

Outlook.com does not have a feature where the messages will be marked as read if you open it using a different email address. You can only set your POP and deleting messages to the following:

 

  1. Login your account.
  2. Go to Inbox.
  3. On the right-upper part, click the gear icon then More mail settings.
  4. Under Managing your account, click POP and deleting downloaded messages.
  5. Check Don't let another program delete messages from Hotmail. (If your other program is set to "delete messages from the server," we'll simply move them to a special POP folder. They won't be deleted.) 
  6. Click Save.

Note: This only applies when deleting messages

 

If you want this feature to be available to the product, please send us a feature request on this link.

 

Hope this helps. Please let us know if you need additional assistance.
