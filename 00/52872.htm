Hi Anthony_1970,

Do you mean you encounter same issue that when searching in Outlook, you get this error: “Something went wrong and your search couldn’t be completed.”, or "Search results may be incomplete because items are still being indexed"?

If yes, on Windows 7, please check if you install the updates KB4022168 by going to the Control Panel and search for Updates.  Click View Installed Updates and look for updates installed ~6/28.  If you have the updates reboot the machine to make sure they had a chance to take affect.

After the updates some other things to check:

1.  In Outlook click into the search box, on the ribbon click Search Tools, Indexing Status.  If there are items still needed to index it might take some time for it to catch up.  It is a good sign if indexing is working.

2.  In Outlook click into the search box, on the ribbon click Search Tools, click Locations to Search.  Make sure there is a checkbox next to all of the items that you want to be able to search.

3.  If indexing is completed from Step 1 and you are still not seeing results or you see the message "Something went wrong and your search couldn't be completed" check to make sure Outlook is selected for Indexing.  

- In Outlook click into the search box, on the ribbon click Search Tools, Search Options, click Indexing Options, click Modify.

- If Outlook is unchecked click the checkbox next to it and click OK.

4.  If indexing is completed from Step 1 and you are still not seeing results rebuild the Index.  In Outlook click into the search box, click Search Tools, click Search Options, click Indexing Options, click Advanced, click Rebuild.

5.  There was a prior issue back in March that caused search results not to be returned.  If you originally configured Outlook or created a new Outlook Profile in late February or March this could have affected you.  The best solution to this issue is to create a new Outlook Profile, https://support.office.com/en-us/article/Outlook-search-stops-working-after-updating-to-build-7870-2020-and-higher-0bcf0b06-320d-40c3-8cf0-7e3df90793ed?ui=en-US&rs=en-US&ad=US.  

New Outlook Profile steps, https://support.office.com/en-us/article/Create-an-Outlook-profile-F544C1BA-3352-4B3B-BE0B-8D42A540459D.

Note:  you can keep the old profile and refer back to it if needed.  If this is the affecting issue it will not likely get resolved without a new profile.  When you follow the steps to create a new profile you can click the check box "Prompt for a Profile to be used".  When you start Outlook it will prompt you to select which profile to open.
