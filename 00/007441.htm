shirley wang msft say:

Hi Natalie,

If we shared some files with other users in SharePoint Team Sites and they sync these files to their local folder, with Edit permission at the library level, they can delete the files from local folders and the browser. Therefore, if they delete files on his computer after syncing the library, the files will be removed to the recycle bin in SharePoint Online.

To make sure this doesn’t happen in the future, we can remove the Delete Items permission at the library level for users. Then, they can only edit the files, but can’t delete them. Please follow the steps below to create a Edit without Delete permission level and then assign it to the users:

1. Go to the team site where we would like to share files with other users.
2. Click the Settings icon like a gear > Site settings > Users and Permissions > PERMISSIONS > Permission Levels > Edit.
3. Scroll to the bottom and click Copy Permission Level.
4. Create a name ”Edit without Delete” and un-tick Delete items in the permissions list > Create.

5. Go to the library where the users would like to sync.
6. Click LIBRARY > Library Settings > Permissions and Management > Permissions for this library > Select the users who have Edit permission > Edit User permissions > Select Edit without Delete permission level created newly and remove Edit permission used before > OK.
7. When the users access the files, there is no Delete option as below. When they try to delete files in synced folder on the computer, they will receive an error saying no permissions and can’t delete.

Shirley

Excellent. Not often having serviced Sony VAIOs I had no clue about that and I'm glad that worked out for you. Jeepers, you'd think they'd identify it better. It sure looked, sounded, and acted like malware to me. I'll have to remember that one.

 

As far as Malwarebytes, as long as you downloaded the free version I recommended it can remain on your system as a backup. While you probably are more of an average user than I am, I'll tell you how I protect my system. Now keep in mind that MSE (and Malwarebytes if you want) are perfectly adequate as long as you practice safe surfing and maintenance habits. I'll go over the programs I use and then tell you about the habits you need to learn until they become like instinct.

 

I take more security precautions than most because helping people often means I need to test software before recommending it, go to questionable websites where people are having problems to see if I can determine the cause, need to download sometimes dubious software because someone has a question about it and I can't help unless I have it in front of me, and so forth. Because of this I take more precautions than the average user who doesn't do (or shouldn''t do) these things.

 

Here's what I do (and it isn't 100% but nothing is, and in some cases only catches things after they've already gotten into the system, but it's a good process while remaining reasonable and they can all work together - as proven by the fact that I use them):

 

I use MSE for real-time protection and run 2-3 quick scans a week with a scheduled bi-weekly full scan. Our moderator runs a weekly quick scan and doesn't run full scans unless he suspects he might be infected or the quick scan picks up something and he wants to do a more thorough scan to be on the safe side - and that's all he does in terms of using programs for scanning. He has great faith in MSE.

 

I update and periodically (every couple of weeks or so) run full scans with Malwarebytes:http://www.malwarebytes.org/and SuperAntiSpyware: http://superantispyware.com/ and ESET online scanner:http://www.eset.com/us/online-scanner Get only the free versions. Any that also have paid versions include features like real-time protection that cannot be used along with MSE anyway (in these cases), so there's no point in spending the money anyway.

 

I use Secunia PSI http://secunia.com/vulnerability_scanning/personal/ to make sure all my programs are updated and current (since updates often plug security holes in the software and thus can be important as many people never check and/or update their software unless prompted - and this way I don't need to check everything since notifications and sometimes updates are automatic).

 

I always follow the following advice from Stephen Boots - MVP (and whether or not you add any of the above programs, the following applies to everybody and should be thought of as the "14 Commandments of keeping your computer secure"):

 

-Make sure that the Windows Firewall is enabled. No other firewall truly offers better protection or is needed. This one is perfectly fine.

 

-Make sure that all important/critical updates, including service packs for the operating system and programs are installed from Microsoft Update (Windows Update).

 

-Make sure Internet Explorer is at version 8 or 9 and updated with all patches.

 

-In Internet Explorer 8 or 9, use the SmartScreen Filter.

 

-Make sure that IE Internet Security settings are at least set to medium-high (default).

 

-Enable the pop-up blocker in IE.

 

-On Vista and Windows 7 make sure that User Account Control (UAC) ON and not running with elevated privileges.

 

-Make sure that Windows Automatic Updates are set to at least notify, but the preferred setting is to download and install automatically. If you update manually, be sure to update as soon as possible after being notified of available updates.

 

-Make sure that installed applications, especially Adobe Acrobat, Adobe Flash, and Java are at their latest versions. Many vendors are regularly updating and patching for security holes.

 

-Never click through links from unknown sources and use caution even if they are from a "trusted" source.

 

-Never open unsolicited email attachments.

 

-Practice safe web browsing.

 

I add a 15th. Do your normal work as a standard user. Use an administrator account only if you need to do something that requires those priviledges (since anyone who hacks into your system or malware that enters your system will have administrator access if you're logged in as an administrator and be able to get in deeper and do more harm than if you're a standard user). And have one or two backup Administrator accounts (all with good, strong passwords that you won't ever forget) in case problems happen with one of them because if it's the only one on the system and you can't use administrator rights with it or even access it for whatever reason (and it happens), then you're stuck with a restore to factory conditions as there's no other way to regain control of the computer. Always keep on UAC. Well, maybe that's more than one more, but that about covers it.

 

I hope this helps.

 

Good luck!
