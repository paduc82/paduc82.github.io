ronrosenfeld say: From your example of A3, it appears that you also want to remove any leading spaces.  Assuming you also want to remove any trailing spaces, I would recommend a User Defined Function.

To enter this User Defined Function (UDF), <alt-F11> opens the Visual Basic Editor.
Ensure your project is highlighted in the Project Explorer window.
Then, from the top menu, select Insert/Module and
paste the code below into the window that opens.

To use this User Defined Function (UDF), enter a formula like

B1:  =BetweenSlashes($A1,COLUMNS($A:A))

Fill right far enough to account of all of the possible instances of //zzzz// in the string
Then fill down as far as required.

======================================
Option Explicit
Function BetweenSlashes(s As String, Optional Instance As Long = 1) As String
    Dim re As Object, mc As Object
Set re = CreateObject("vbscript.regexp")
With re
    .Global = True
    .Pattern = "//\s*([\s\S]+?)\s*//"
    Set mc = .Execute(s)
        If mc.Count >= Instance Then _
            BetweenSlashes = mc(Instance - 1).submatches(0)
End With
End Function
