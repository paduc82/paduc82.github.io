When Maps is in Offline mode, it will still use A-GPS just fine. If you want the phone to really stay away from using any operator data, you need to disable A-GPS in the phone Navigation settings. Maps can't affect the A-GPS situation.

Online and Offline in Nokia Maps mean here the way how the addresses and POIs are searched and where the route for navigation is calculated.

In Online mode:
- The address or POI you search is searched from the Nokia navigation server
- The route calculation is done on the Nokia navigation server as well and the route is downloaded to the phone

In Offline mode:
- The addresses and POIs are searched from the map data you have downloaded in your phone
- The route calculation is done in the phone

Sometimes Maps can find certain addresses only in Offline mode, and certain POIs (such as pharmacies in Finland) can only be found in Online mode. This is an obvious bug in Maps 2.0, and hopefully it will be corrected in the future. In the newest version wk26 build02 version it is not yet corrected! :(

Need Help Locking a Word Document - Microsoft Community

melissa_zoe say:

Need Help Locking a Word Document

How do you lock a word document so it is "read-only" for other people?  Thanks
cindy meister say:
How do you lock a word document so it is "read-only" for other people?  Thanks


Hi Melissa

You've gotten four different answers, and all of them are right :-) The question for you is: which is the best answer for you. Word has various flavors of "read-only".

1. Graham's message relates to using the "Save As" option to save the document as "read-only". All this does is force the user to use "Save As" so that your document can't be over-written.

2. You can use IRM (Information Rights Management) to lock down the document. This is probably what you meant with your question, but if you don't have access to a server that supports that technology, it's not going to be much help.

3. Related to (2) are the steps Beth mentions, that are also in the article Ruchi points you at. As Beth says, this kind of protection is not very secure - unless you have access to IRM.

4. If what you really need is a document that people can read, but not alter, your best bet is to follow Doug's (and Beth's) advice to save the document as a PDF file.

I hope that clears things up a bit for you :-)

Cindy Meister, VSTO/Word MVP
