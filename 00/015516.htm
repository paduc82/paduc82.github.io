Ok to start you must have WordPad installed on your PC or MAC. From there you must identify the file you want to pass to MS Word, once you have identified the file you must open it with WordPad since it is the program where it was created and follow the following steps so that you can pass it to MS Word:

1. Open the WordPad document that you want to convert to Word.

3. Press "Ctrl" and then press "C". This will copy all the content of the WordPad document.

6. Navigate to the location of your computer where the Word file is stored in which you want to paste the content you copied from Wordpad, and then double-click on the Word file to open it.
7. Press "Ctrl" and then press "V". This will paste all the content that you copied from WordPad into the Word document. Press "Ctrl" and "S" to save the changes.

Photos App Displays Hidden Files - Microsoft Community

quinton aj stevens say:

Photos App Displays Hidden Files

In the default "Photos" app, it shows you images from your selected "Pictures" library within Windows Explorer. If there are hidden picture files (even within a hidden folder within this library), they will still show in the application.

e.g. If I set my Pictures library to "E:\Pictures\" and create a folder "E:\Pictures\Embarrassing Pictures Of SpongeBob At The Christmas Party\" and set that folder to "Hidden," and even all the files within the folder to hidden, the Photos app will still display them within the "Collection" section.

Better yet, if the Photos is set as a Live Tile in your Start Menu, these photos will also show up there! Not so hidden anymore, huh?

The only workaround, as far as I know, is to not use the Photos app and to disable the Live Tile within the Start Menu. You could also set your Pictures library to only include the specific folders, such as "E:\Pictures\Bananas\" "E:\Pictures\MyPetDolphin" but that is honestly fairly gimmicky and should have the option in the settings menu of the Photos app "Don't show Hidden Files & Folders" and have it be set as default.

What do you guys think? Has anyone else noticed this and has it caused them any problems? I, for one, would not use the Photos app if this is not changed.

markgamez say:

I was having the same issues, but I found a pretty simple solution for this. 

If you go inside the folder and shift-right-click a blank space inside the window, you can click "Open command window here"

After that you can type: "attrib -h" (without quotes)

If you haven't done it already, you'll also want to go to "File Explorer Options" which you can get to by typing that in the windows search bar.

Go over to the "View" tab and tick the option to "Show hidden files, folders , and drives."

Now when you go into the folder, the images or other files in the folder will be semi-transparent. You will still be able to open them in the photos app, but it shouldn't remember their location after that or display them in the "Recent" tab. (Although you may have to refresh the app by opening it and closing it once or twice before they disappear from the recent tab and the live tile.)

Let me know if you have any other questions or if this doesn't work for you. But for now, your

Embarrassing Pictures Of SpongeBob At The Christmas Party should be safe from visitors on your computer!
