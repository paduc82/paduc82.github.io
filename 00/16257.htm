I need help with an IF or VLOOKUP formula for a checkbook register. - Microsoft Community

tundra37 say:

I need help with an IF or VLOOKUP formula for a checkbook register.

I need help with an  IF or VLOOKUP formula for a checkbook register that I created in Excel 2007. I have a workbook with two worksheet tabs. The first worksheet tab is the Checkbook Register and the second worksheet tab is the Table of Accounts, such as Automobile Maintenence, Business Entetainment, Medical Prescriptions. In the table, each account ahs a short account 2 digit account number. What I want to accomplish is that in the checkbook register when I enter the account number I would like the account description to appear in the adjoining column without me typing it each time. The Table of accounts is about 50 items long.
chip pearson say:

Assume that your table of accounts is on Sheet2 in the range A1:B10 (codes in A, descriptions in B). Then back on your register sheet, assume the first entry is on row 1, with the code to be entered in column A. In column B, use the function

 

=IF(A1="","",VLOOKUP(A1,Sheet2!A1:B10,2,FALSE))

 

and copy down as far as you need to go for your firty rows. If A is empty, the result is an empty string. If A1 is not empty, the description of the code in A1 is returned. If you enter on Sheet1 a code that does not exist in your table of accounts, you'll get a #N/A error. If you want to have a different message if the code is not found, use a formula like

 

=IF(A1="","",IFERROR(VLOOKUP(A1,Sheet2!A1:B10,2,FALSE),"not found"))

 

and change the "not found" to the error message you want. Note that IFERROR is new to Excel 2007 and is not available in earlier versions.

Header on Second Page Only - Microsoft Community

eddyg11 say:

Header on Second Page Only

Hello everyone,

Is it possible to put a header only on the second page of a word document and nothing else? The document is around 15 pages and we need the header only on the second page and I can't find anything about that in my search. I use Word 2013. 

Thank you!

allison cs say:

Yes, it is possible!

You will need two Section Breaks, one on the first page and one on the second page. A regular Page Break won't suffice. To create these section breaks:

  1. Go to Page 1.
  2. Click once somewhere in the empty space near the bottom of the page. (This is to make sure you don't inadvertently split Page 1 content into two.)
  3. Go to the Page Layout tab.
  4. Click Breaks.
  5. Create a new Section Break Next Page.
  6. Go to the next page.
  7. Double-click the header to open it for editing.
  8. Click Link to Previous to unlink it from the previous page.
  9. Now double-click outside the header to return to "normal" editing.
  10. Repeat Steps 2-9 to create a new section break between pages 2 and 3.

Now you should have a unique header on pages 1 and 2. (If you don't want the header on page 1, just delete any content from the header.)

Hope this helps!

For more on section breaks in Word 2013, see here.

stefan blom say:
You can do this with one section break as long as you enable the "Different first page" option for the first section and put your content in the main header of the first section. The main header in the second section must be unlinked from its "neighbor" in the preceding section so that you can delete the content there. See also http://wordfaqs.ssbarnhill.com/FrontMatterRibbon.htm.

suzanne s. barnhill say:

A third option:

  1. Create the header content and cut it to the Clipboard.
  2. In the header pane, create the following field:
    { IF { PAGE } = 2 "[content]" }

To create the IF field:

  1. Press Ctrl+F9 to insert the field braces {  }.
  2. Between the braces, type IF, = 2, and the quotation marks.
  3. Insert the PAGE field by pressing Alt+Shift+P.
  4. Paste the Clipboard content between the quotation marks (where I have [content]).
  5. Update the field with F9 and the header will appear only on the second page.

This solution will work in situations where you don't need or can't have a section break between the first and second pages.

Use Find and Replace to format 4 digits - Microsoft Community

aussie_bel say:

Use Find and Replace to format 4 digits

Hi!

Is it possible to use Word's Find and Replace feature to search for 4 digit numbers that contain a space after the first digit? I want to remove that space. I have a huge document with inconsistencies in how the numbers are presented. 

In other words:

Find # ###

Replace with ####

I hope someone can help me and I apologise if this has been asked before. 

Bel

Word 2010

Windows 7

doug robbins - mvp office apps & services (word) say:

In the Replace dialog, click on the More button and then check the Use wildcards box.  Then in the Find what: control, enter

([0-9]{1}) ([0-9]{3})

Note, there is a space between the ) and (

Then, in the Replace with: control, enter

\1\2

Then click on Replace All.

For an explanation of the above, see the article "Finding and replacing characters using wildcards at:

http://www.word.mvps.org/FAQs/General/UsingWildcards.htm

aussie_bel say:

Thanks so much Doug. 

I notice that the query will find numbers consisting of more than 4 digits as well. I do not want to remove the space from those so I can't use Replace All. But this is still very helpful. 

doug robbins - mvp office apps & services (word) say:

What follows the 3 digit number?  If a space, put that after the entry in the find what box inside another set of ( ), if a carriage return, put ^13 there inside another set of ( ).  If a mixture, put that mixture inside ([ ]) after the entry in the find what box.

Then in the Replace with box, insert \1\2\3

robert19 say:

You could use the following wildcard Find/Replace:

Find What: ([0-9])[ ]([0-9]{3}[![0-9])

Replace With: \1\2

macropod say:

You could use:

Find = ([0-9]{1}) ([0-9]{3}>)

Replace = \1\2

Note that the Find is just you original one with the '>' tag to denote the end of a word. Replace All should work fine with that.
