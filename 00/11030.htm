scott a. helmers say:

Thanks for the kind words, Jim. I think your approach will work if you know that every shape should have a hyperlink and the absence of a hyperlink identifies a problem. However, a more general solution might be to use code below to do an inventory of all existing hyperlinks and write them to Excel. In Excel, you can sort by the column containing the target page to identify any pages that are not the target of any hyperlinks.

Sub FindHyperlinks()

    Dim pag     As Visio.Page    

    Dim shp     As Visio.Shape
    Dim hyp     As Visio.Hyperlink

    For Each pag In ActiveDocument.Pages
        For Each shp In pag.Shapes
            For Each hyp In shp.Hyperlinks
                'write to Excel: pag.Name, shp.Name, hyp.SubAddress
            Next hyp
        Next shp
    Next pag
End Sub

NOTE: the Subaddress cell of the hyperlink is the one that contains the target page name for internal hyperlinks.

Actually, a better idea might be to create a dictionary object, then

   for each page

      for each shape on the page

         if a hyperlink exists, write the target page name to the dictionary

      next shape

   next page

   sort dictionary by page name and write out the results

The advantage of a dictionary in a situation like this is that it won't accept duplicates, so each target page will appear either zero or one time in the dictionary. This is great if you don't care where links come from or how many there are if more than one.

bob jones aka: cybertaz say:
One option:

Create a 2 column lookup table anywhere you wish. It can even be on a different sheet. The example below happens to be in K4:L8. The values may not be exactly right because of the ambiguous use of the term "between" & you didn't indicate whether you're dealing strictly with positive, whole numbers, but this is the idea:


  K        L  
0 $150
46 $180
61 $210
90 $240
106 $270

In cell F41 use the formula: =VLOOKUP(F3,K4:L8,2)

Sample results are:

If the number in F3 is <46, F41 will display $150
If the number in F3 is >45 but <61, F41 will display $180
If the number in F3 is >60 but <90, F41 will display $210
If the number in F3 is >89 but <106, F41 will display $240
If the number in F3 is >105, F41 will display $270
