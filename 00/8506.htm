How do I add new pages while creating brochures? - Microsoft Community

wannier say:

How do I add new pages while creating brochures?

Original Title: Creating brochures 

Am trying to create a bi-fold brochure and thought using one of the templates such as "school program" or "picture program" would do the trick. However, when I added page numbers it did one page number per sheet rather than 2, which seems to defeat the purpose of using a template. Also, I can't see how I add new pages. Any advice?
daniel g. grau say:
In those two program templates, you still only have two pages, and that's what Word shows in the status bar at the bottom: "Pages: 1 of 2" and "Pages: 2 of 2." So I will instead refer to "panels," and there are two panels to a page.

To add panel numbers, I would advise you to just type them. This is feasible if you have just a few panels.

If you have many inside panels, then you'd have to introduce some calculations:

  • In the bottom tab of your Publishing Layout window, click on Master Pages
  • On the First Page master section, create a text box at the bottom of the left panel and add the following field formula: { =  { NUMPAGES } * 2} (this will be the last panel of the program, or back cover)
  • Under the left panel in the Master section, add this field:  { = ( { PAGE } * 2 ) -  2 }
  • Under the right panel in the Master section, add this field:  { = ( { PAGE } * 2 ) - 1 }

For an 8-panel program, including the first two cover panels (4 pages in total), your panels will be numbered as follows:

     8     NOTHING
     2     3
     4     5
     6     7

If you don't know how to insert fields:

  • Create a text box at the bottom of the panels.
  • Enable the display of fields with SHIFT F9 (this a show/hide fields toggle control)
  • Select Insert > Field and uncheck "Preserve formatting..." at the bottom
  • Click on "(All)" in the left list and then on "= (Formula)." This will insert { =  }.
  • Now place your cursor after the equal sign and repeat Insert > Field, but this time select "Page" (also, repeat the uncheck operation for "Preserve formatting").
  • So far, it should look like this: { = { PAGE } }
  • Now move in with your cursor and add the rest of the characters in the formulas, as per above instructions.

Finally, to see the panel numbers, toggle the display of fields with SHIFT F9.

Print Area - Microsoft Community

yer vang78 say:

Print Area

How do you print non adjacent areas on a single page?

gord dibben say:

Hide the ranges you don't want to print.

Or use a macro to print selected or preset areas.

Sub Print_Noncontiguous()
    Dim Destrange As Range
    Dim Smallrng As Range
    Dim Newsh As Worksheet
    Dim Ash As Worksheet
    Dim lr As Long

    Application.ScreenUpdating = False

    Set Ash = ActiveSheet
    Set Newsh = Worksheets.Add
    Ash.Select

    lr = 1

    For Each Smallrng In Selection.Areas
    
    'You can also use a range with more preset areas like this
    'For Each Smallrng In Ash.Range("A1:C1,D10:G20,A30:E41").Areas

        Smallrng.Copy
        Set Destrange = Newsh.Cells(lr, 1)
        Destrange.PasteSpecial xlPasteValues
        Destrange.PasteSpecial xlPasteFormats
        lr = lr + Smallrng.Rows.Count
    Next Smallrng

    Newsh.Columns.AutoFit
    Newsh.PrintOut   'PrintPreview

    Application.DisplayAlerts = False
        Newsh.Delete
    Application.DisplayAlerts = True

    Application.ScreenUpdating = True

End Sub
