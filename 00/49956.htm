a. user say:

Hello Alehuta,

Thank you for posting your query in Microsoft Community.

If you see a delay with Windows welcome sound on start up, then the possible cause is that the sound drivers might be taking time to load completely. In simple the issue is with the sound drivers which are incompatible with windows10.

Suggest you to check and update latest windows 10 compatible sound drivers from Manufacturer website for device to see if problem still persist. Do try the below troubleshooting steps and see if resolve the issue.

Method 1:

Try Update drivers from Device Manager

a) Open run command by pressing Windows key + R and type devmgmt.msc (without quotes) and hit Enter.

b) Spot and expand the Audio inputs and outputs.

c). Right click on the High Definition Audio Device and select Update Driver Software. Once this is done, restart the computer to check the problem fixed. If it doesn't fix the issue, then right click on the same driver, select "Uninstall", then restart the system and let the system install the driver automatically on Start.

Method 2:

If no updated drivers available on the computer manufacturers website for windows 10, then trying installing drivers in compatibility mode. Compatibility mode runs the program using settings from a previous version of Windows.

Follow these steps to install the drivers in compatibility mode and check if that helps.

a. Download the driver from the manufacturers website.

b. Right-click on the driver setup file and click on properties.                       

c. Click on the compatibility tab and check the box Run this program in compatibility mode for and select Windows 8/8.1/7 operating system from the drop down menu and proceed with the installation. Once this is done, restart the computer and check if the issue persists.

Keep us posted if your require further assistance, we're glad to assist you.

Microsoft ActiveSync

My Bernina Sewing machines uses Microsoft Activelync to communicate with my laptop and it will not download what can i do?
arya s asok say:

Hi Mcquiltbag,

 

ActiveSync is only compatible with Windows XP or earlier version of Windows. If you run Windows Vista or Windows 7, you will not be able to download and install Active sync on it.

Since you are using sewing machine and trying to sync with laptop, I would suggest you to contact the sewing machine manufacturer to check if they have any alternate software to Active sync that can be installed on Vista or Windows 7 computer.

Normally active sync is used to sync the computer with the mobile device. Currently we use Windows Mobile Device Center to communicate with the mobile devices instead of Active sync for Windows Vista and Windows 7.

Hope this helps.
