karl timmermans (outlook mvp 2012-2018) say:
>>
Somehow, I ended up with several inboxes in my Outlook 2016. I would like to combine them into one inbox. Is that possible?
<<

It all comes down to which protocol (POP3, IMAP, Exchange) was used to configure the email accounts.

Executive summary regarding the specific question

#1 - IMAP and Exchange accounts will each have their own <Inbox> and cannot be combined in a single <Inbox>

#2 - POP accounts "can" be configured to all use a single <Inbox> if configured to use the same underlying PST file which can be set when the account is initially configured or if the accounts are already configured, can be done by going to
File --> Account Settings --> Email tab --> select the POP3 email account --> click on the <Change Folder> button and select the <Inbox> in the PST file you want to use. The PST file must already be added (exist) in the Outlook profile.

diane poremsky mvp say:
If you are using POP accounts, yes, you can use one inbox. Go to File, Account Settings, open the account settings dialog. Select the account, then click Change Folder at the bottom and choose the inbox you want to use. Remove the extra pst files from the Data files tab.

If you using IMAP accounts, no, your can't combine them. You can use instant search to view all new messages or remove the accounts and add them back as POP accounts using the manual set up option (click Advanced and choose manual on the auto account setup screen).

See https://www.slipstick.com/how-to-outlook/how-to... for more information on using instant search method.

Hi WillKeller,

 

I would suggest you to try the steps mentioned below.

 

Step 1: Optimize Internet Explorer

http://support.microsoft.com/kb/936213

 

Step2: Enable “Launching applications and unsafe files (not secure)

a.    Open Internet Explorer, Open the Tools Menu and Internet Options

b.    Click Custom Level under the Security Tab

c.    Change the  Launching Applications and Unsafe Files to Prompt

 

After this Internet Explorer will show an error message (about: Security Risk). You can perform this step only when you know that you’re downloading files from a secured website.  

Note: If enabling the option did not fix the issue, disable it.

 

Step 3: To Add "*.exe" to the "Inclusion list for moderate risk file types" setting

a.    Click on Start; in the Start search type “gpedit.msc” without quotes and press Enter.

b.    Go to User Configuration and then to Administrative Templates.

c.    Click on Windows components and then choose Attachment Manager.

d.    Add "*.exe" to the "Inclusion list for moderate risk file types" setting.

 

This policy setting allows you to configure the list of moderate risk file types. If the attachment is in the list of moderate risk file types and is from the restricted or Internet zone, Windows prompts the user before accessing the file.

 

 

Hope this information is helpful and let me know if you need any further assistance.
