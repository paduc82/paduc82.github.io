¡firedog say:

This is a different question from that in the thread you posted in, so I've separated them   

 

 

To import into Windows Live Mail keeping categories intact, use Windows Contacts as an intermediary:

  1. Type contacts into the start search box and press Enter. This will open the Windows Contacts folder for the current Windows user.
  2. If the folder contains any contacts and is in use for some reason (like synchronizing with a mobile device), copy the folder and paste back to the same place to safeguard the contents of the folder. Empty the folder called Contacts.
  3. Create new folders within the Contacts folder to match the folders of VCF files.
  4. Use the Import > vCard function to import batchwise from each folder of VCF files. Use Select All (Ctrl-A) to select all the VCF files in a folder to import all of them at once.
  5. After each import, move the contact files into the corresponding folder.
  6. Open each of the folders in turn and use New Contact Group to create a group with the category name you want eventually to use in Windows Live Mail. Then use the Add button in the contacts group dialogue to select all of the contacts in the folder and add them to the new group.
    You should now have a number of folders inside Contacts, each containing a number of .contact files and a single .group file.
  7. In Windows Live Mail's contacts window, use Import > Address book for current Windows user. Categories will be created automatically containing the members of the Windows Contacts groups.
  8. You can either keep the new folders in Windows Contacts as a backup, or delete them all and then copy the contents of the Contacts - Copy folder back to restore the status quo. The Contacts - Copy folder can then be deleted.


You could have avoided all this simply by signing in in Windows Live Mail. This would synchronize all your contacts and categories with those at people.live.com, so to reproduce them on a new installation or another Windows account or another computer, just sign in with the same account.
