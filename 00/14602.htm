Merging Word Documents within Word for Mac? - Microsoft Community

lawlessig say:

Merging Word Documents within Word for Mac?

Is there anyway to merge many word documents (chapters in a book) into a single Word document, using Word for Mac?

john korchok say:

If it's a must to keep all pages in one file for long-term editing, you should follow Charles' advice and copy and paste all files into one. Depending on how the different documents are formatted, you might separate each one with a Page Break or with a Section Break. You would use a Section Break if each document has a different first page, even and odd pages, or if the margins or page orientation changes among the documents. If your formatting is simple, a Page Break may be sufficient.

If you can keep the files separate for editing, but want to combine them for printing, consider using Word's Master Document feature. This allows you to temporarily combine all the files into one large one, so that you can get page number sequences right. Master Documents are only a problem when users try to make them a permanent storage solution that they edit and update. This eventually leads to document corruption.

Here are the steps to create a Master Document:

  1. In Word, choose View>Outline on the Ribbon or View>Master Document from the menu at the top of your screen.
  2. If the Show Document button is not already highlighted, click on it.
  3. Click on Insert, then select the first document you want to add. Repeat this step until you have added all documents.
  4. Print the file.
  5. Delete the Master Document. The subdocuments will not be  deleted or otherwise affected.

I've used macros to automate master document production in Windows, but the Mac version is sandboxed and doesn't have a functioning DIR command to get a collection of files, which makes programming significantly more complicated.

You can set the profile that would open once you start Outlook via the Control Panel. Before we proceed, we would like to confirm if you are using the Outlook desktop app to manage your emails when you mentioned Outlook 2017. If yes, follow these steps:

  1. Exit Outlook.
  2. Go to Control Panel.
