labels from table of contents

In Word 2003, Is there a way to create labels from the table of contents other than copy and pasting one line at time?
doug robbins - mvp office apps & services (word) say:

NOTE - after following this procedure, do NOT save your original document. SAVE it BEFORE performing the procedure!

1.    Select the table of contents and then press Ctrl+Shift+F9 which will convert the table of contents to plain text.

2.    Copy the table of contents to the clipboard

3.    Start a new document and paste the table of contents into it.

4.    With the table of contents selected in that document from the Table menu, select Convert and then Text to Table to convert the table of contents into a word Table.

5.    Insert a row at the top of the table and into the cells of that row, insert names that can be used as the field names for a mail merge.

6.    Save that document and then use it as the data source for a label type mail merge with your desired label media.

If you need detailed instructions on the mail merge process, see the article “Mailmerge Labels with Word XP” on fellow MVP Graham Mayor’s website at:
