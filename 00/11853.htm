How can you make the default for fill as no fill in Office 2007 or - Microsoft Community

bgcsocal say:

How can you make the default for fill as no fill in Office 2007 or 2010?

We use the shapes a lot in one category of our Excel adn Word documents to circle stuff and then type in comments, arrows, etc. The circle/oval always comes up filled with a color. Is there a way to set a preference so that the shapes are always not filled? Most of our users have Office Professional 2007 and XP, but I can upgrade them to Office 2010. We also ahve some users on Windows 7 using 2007 or 2010.

jay freedman say:

For new shapes in new documents, the solution is easy, once and done:

- Open the Normal.dotm template (go to the folder %appdata%\Microsoft\Templates, right-click Normal.dotm, and click Open).
- Insert a shape, and format it as you want the default to appear.
- Right-click the edge of the shape and choose Set AutoShape Defaults.
- Delete the shape.
- Save and close.

Whenever you create a new document based on this Normal.dotm, shapes you draw in that document will have the default formatting you set.

Shapes in existing documents are harder to deal with. Each document maintains its own AutoShape default formatting, and changing that default doesn't affect existing shapes. The best workaround I
could manage was this: Store the following macro in Normal.dotm (see  http://www.gmayor.com/installing_macro.htm  if needed).

Sub ApplyShapeDefaults()
    ' to be used after at least one shape in the document
    ' formatted as desired, and you right-clicked and chose
    ' Set AutoShape Defaults

    Dim tmpshp As Shape
    Dim shp
    With Application.ActiveDocument
        ' insert a new shape, which will get the default format
        Set tmpshp = .Shapes.AddShape( _
                    Type:=wdShapeRectangle, Left:=0, Top:=0, _
                    Width:=10, Height:=10, _
                    Anchor:=.Paragraphs(1).Range)

        ' grab its formatting
        tmpshp.PickUp

        ' apply that formatting to each shape
        For Each shp In Application.ActiveDocument.Shapes
            shp.Apply
        Next

        ' remove the temporary shape
        tmpshp.Delete
    End With
End Sub

Choose any existing shape in the document, format it the way you want it, right-click its edge, and choose Set AutoShape Defaults. Then run the macro.
