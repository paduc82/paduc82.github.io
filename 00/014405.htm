HJB1954

given the locations you described being on the activesheet when the macro is run, it will construct the teams starting in cell F11.  

Sub abc()
Dim sh As Worksheet, cell As Range, i As Long, j As Long
Dim teams As Long, players As Long, c As Range
Dim prange As Range, output As Range, maxrow as long
Set sh = ActiveSheet
teams = sh.Range("C11")
players = sh.Range("B11")
Set prange = sh.Range("A11").Resize(players, 1)
Set output = sh.Range("F11")
    i = 1
    j = 1
    maxrow = 0
  For Each cell In prange
    If j > teams Then
      i = i + 1
      j = 1
    End If
    output(i, j).Value = cell.Value
    If maxrow < j Then maxrow = j
    j = j + 1
  Next
  For k = 2 To maxrow
    If k Mod 2 = 0 Then
    v = output(k, 1).Resize(1, teams).Value
    For l = 1 To teams
      output(k, l).Value = v(1, teams - l + 1)
    Next
    End If
  Next
End Sub

The WMA file type is a Windows Media Audio file which is not supported on the Mac operating system. At one time the Flip4Mac suggestion from Phillip was an option. Telestream, the developer, simply has not upgraded their product to keep pace with the rest of the Mac environment. I solved it by using and adapting to my needs the response from a Jay Freedman that can be found here:

https://answers.microsoft.com/en-us/office/forum/office_2007-word/double-space-in-word-outlook-2007-causes-changes/22011548-35d8-4f4e-9d6e-c31bf33341a7

There you will find the section Apply as you type.

Remove the mark in front of Built-In Heading styles

That solved it.
