marvin ma msft say:

Hi Novice,

1. For the old emails, many configurations may cause this. For example, you can navigate to File > Account Settings > Account Settings in Outlook client, then check if the account just enabled Cached Exchange Mode for the past certain months. If so, it will just download the emails within the past certain months from the server.

Besides, there's a default retention policy applied to the whole Office 365 tenant. In brief, retention policy is used to manage the emails (preserve, delete or move to archive) in different types of folders in your mailbox. The policy consists of many retention tags to perform the relevant actions when exceeding retention period.

One of the default retention tags is Default 2 year move to archive. It will move all older than 2 year (730 days) emails which were stored in default folders to Archive folder.

So to confirm it they can see the old emails, please firstly sign in Outlook Web App (OWA) to check it since OWA will sync all emails with the Exchange server. Then make sure whether these emails appear in the users' Archive folder. If you cannot find these emails, please try to use Content Search feature to search the total items in the user mailbox.

2. Generally, we can use Audit Log to check all activities in your tenant, including email activities. But it requires to enable the feature before. Since you have just migrated to Office 365, I assume you haven't enabled this feature. In this scenario, I suggest you enable the feature to monitor the activities in the future. Then you can move these emails back to Inbox folder and check if the issue recurs. 

On the other front, the shared mailbox will only attach to the specific users who have Full Access permission. This permission allows the user to behave as the mailbox owner. So we cannot prevent users to delete emails in shared mailbox, unless they have no access to shared mailbox. 

A workaround is to enable Litigation Hold of the shared mailbox, it will preserve all data permanently. However it requires Exchange Online Plan 2 license.

3. The archive folder syncs with the email server. So yes, it's accessible on all devices. For a shared mailbox, all users with Full Access permission of the shared mailbox can access Archive. As a side note, Outlook for mobile apps don't support shared mailbox feature currently.
