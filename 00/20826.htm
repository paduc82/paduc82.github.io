How to remove operator splash screen (boot animation) - Microsoft Community

fender77---01 say:

How to remove operator splash screen (boot animation)

I managed to remove the splash screen on mine. I'm still on release firmware and can "safily" operate on dev mode. However, this is done at your own risk!

 

You have to do this from the terminal.

 

1: Become root of your device: "devel-su" ->enter "rootme" ->enter.

 

You are now in charge!

 

2. bring out the correct folder: "CD /etc/mwelcome.d" ->enter

 

3.  "ls" ->enter

 

You will now see the files within that folder. The splash screen should be stored here.

On mine the file mentioned in terminal after this command was named 320.conf. This was the only file as I could see. I don't know if this is just the name of the telenor splash screen or if every operator splash screen will have that name, but if there's only one file in the folder, I guess it's that one.

 

4. To remove the splash screen (you only move the file to another folder (MyDocs):

 

"mv 320.conf /home/user/MyDocs" ->enter

 

You will now get prompted with that permissions and ownership can not be preserved. Nothing to worry about. Everything is fine.

 

5: "exit" -> enter and "exit"-> enter

 

Then reboot the device!

 

The file will be found in Mydocs now. I use filebox as a filemanager.

 

 

To redo:

 

Become superuser again

"CD /home/user/MyDocs/ ->enter

"ls"->enter and you will see the file - in my case 320 config.

Now,

"mv 320.conf /etc/mwelcome.d/->enter

Reboot.

 

I have no idea if the operator splash screen will be named something else on others devices, Mine is telenor Norway branded.

 

I guess this could also be used to apply other splashscreens. What I don't know yet if you will have to remove the one first (i guess) so the folder which the phone reads from has only one splash screen file.

 

What you basicly do is to move a file from a folder to another one.

 

I have tried this on my N9 a couple of times following the guide and succeded every time. BUT this is done at your own risk!

bill manville say:

For the case of a task crossing the year end, you will need to know the amount of effort in each year.

The simplest approach would be to use 2 lines for such a task, one for the period to 30th June and the other for the period in the new year.

The alternative would be to have columns for Rate_2014, Days_2014, Rate_2015 and Days_2015 and then Total would be Rate_2014*Days_2014 + Rate_2015*Days_2015

The only way you could do it in one row with the columns you currently have would be if the effort was spread uniformly across the time from start date to end date; e.g. if the task was from 1-June to 29-August, 1/3 of the effort was at 2014 rates and 2/3 was at 2015 rates.  In that case the formula could be

=IF(E2<DATE(2015,7,1),SUMIFS(Rate_2014,Role,A2,Location,B2,Type,C2),IF(D2>=DATE(2015,7,1),SUMIFS(Rate_2015,Role,A2,Location,B2,Type,C2),((Date(2015,7,1)-D2)*SUMIFS(Rate_2014,Role,A2,Location,B2,Type,C2)+(E2-Date(2015,6,30))*SUMIFS(Rate_2015,Role,A2,Location,B2,Type,C2))/(E2-D2)))
