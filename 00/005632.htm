a. To move the messages to an existing folder, choose Move to Inbox, and then click the folder name

b. To move the messages to a new folder, choose Move to a new folder and type the name for the folder you want to create.

c. To delete the messages, choose Delete these messages.

d. To forward messages, select forward messages and type the email address where you wish to forward the said emails.

7. Click Save.

 

Note: In an instance where the spammer just changes the domain (sending email address), it would be better if you would create an email rule that will specify the subject instead of the sender's email address. (e.g. If the email contains the word "Viagra", delete the email).
