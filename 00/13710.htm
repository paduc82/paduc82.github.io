don cuthbert say:

Fast Ring build ISO files are not normally made available. An exception was made for build 15002, which you can download from this link and then upgrade to 15014.

https://www.microsoft.com/en-us/software-download/windowsinsiderpreviewadvanced

The only way to download a single ESD file for the current Fast Ring build is to install an older build (such as the current public build 14393) and put it on the Fast Ring. Windows Update will then download a single Install.esd file for the new build. From this file you can produce a current Fast Ring ISO file, but you have to make a copy of this ESD file before the computer Restarts after the download. Otherwise the ESD file gets deleted during the upgrade. If you have an ISO or install media for the regular public build of Windows 10, you can wipe your hard disk and install that. Activation should be automatic after install, as long as you use no Product Key during install. If you are concerned that activation may not work, you can save your activation as described in the article I posted earlier. You can also install another copy of Windows without deleting what you have now and see if it activates. To do this you can first shrink drive C: and install the new copy to the empty space created when you shrunk C:.

If you have no other computer to use to download what you need, you may have to reinstall Windows. You can try creating a bootable USB stick by running recoverydrive.exe and choosing to include System files. This should create a USB stick to clean-install build 14955.

You can also try using the Reset Windows option, by holding SHIFT when you click on Restart to bring up a boot menu, which includes the Reset option in a submenu.

If Windows is too damaged these methods may fail. You can first see if you can fix anything by running sfc /scannow at an Administrator Command Prompt.

Hi Peter,

I can't claim that this will work, particularly with PowerPoint 2011, but in theory if you used a VBA macro to sum the lengths of any recorded audio in the slides you should get close to the number you are seeking. Here is the core code that locates the "sound" media within a shape of a slide.

Sub SoundMediaLength()
    Dim shp As PowerPoint.Shape
    If shp.type = msoMedia Then
        If shp.MediaType = ppMediaTypeSound Then
            Debug.Print shp.mediaFormat.Length
        End If
    End If
End Sub
