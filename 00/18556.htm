If the touch screen fails from time to time, update or reinstall the current software. Connect your phone to Nokia Suite and click on the green arrow to see if any updates for your device are available. If there is none, you will have the option to reinstall the current software.

 

You can also hard reset the device if the above doesn't help, this will whipe any bugs off the phone. Make sure you make a back up with Nokia Suite first, as it will also whipe all your phone's data. When you make a back up, don't tick the Settings box, as this will reinstall the bug as soon as you put the back up back onto your phone. Now hard reset the device by entering *#7370# into your phone. The default security code is 12345, unless you have changed this code yourself.

 

Once you've done the above, the touch screen shouldn't fail anymore. If it does, please visit a local Nokia care centre where they can check the phone on any bugs or damage, it might be that the device has some external damage which cannot be fixed with a software update or hard reset.
