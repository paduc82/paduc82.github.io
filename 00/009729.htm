Is there a way to change 'hidden until used' on a Mac? 

I am using Word 14.2.2 on a Lion
john mcghie say:
Go to Format>Style.

Change the "List" box from "Styles in Use" to "All styles".  You will then see all of the styles in the document.

You can apply from that dialog.  When you do, they will appear in the "In Use" list.

When you create a caption, Word automatically applies the built-in style "Caption".  If you have changed this style, your changes will be applied to all current and future paragraphs of that style in the document.

If you have changed the style in the template but not in the document, then you need to copy the changed style from the template into the document.  There are three ways to do this:

1)  Create the document from that template

2)  Attach the document to that template and click the "Automatically update styles" checkbox in that dialog.

3)  Open the Organiser (Tools>Macro>Organiser...) and copy the style across.

The Styles Management mechanism is only half there in Word 2011, they left all the useful bits out.

Hope this helps

anfy2010 say:

paragraph selection

I need to select all paragraphs in the document that has the style named AIO only using a macro vba

 

 

yves dhondt say:

If you want to select them all at the same time, that is not possible as far as I can tell.http://support.microsoft.com/kb/q288424/ states:

You cannot select a discontiguous range programmatically by using the Word object model. A discontiguous selection may only be created by the user in the Word UI.

However, if you want to perform a certain operation on them, you could execute a find which selects them one by one. You would then do the operation one at a time. The following macro selects all paragraphs having a style of "Heading 1" and concats their text. After all have been found, the text is displayed in a messagebox.

Sub ExampleMacro()
   
    Dim allText As String
   
    Selection.Find.ClearFormatting
    Selection.Find.style = ActiveDocument.Styles("Heading 1")
    With Selection.Find
        .text = ""
        .Replacement.text = ""
        .Forward = True
        .Wrap = wdFindContinue
        .Format = True
        .MatchCase = False
        .MatchWholeWord = False
        .MatchWildcards = False
        .MatchSoundsLike = False
        .MatchAllWordForms = False
    End With
   
    Do While (Selection.Find.Execute)
        allText = allText & Selection.text
    Loop
   
    MsgBox allText
   
End Sub
