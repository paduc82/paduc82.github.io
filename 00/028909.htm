How can I return the sum of values returned from a look up table?

What function can I use to return the sum of values from a specified column for records with a matching index value?

I am looking for something like VLookup(), except I want to return the sum of values. VLookup() goes to the first record where a where a match is found and returns the value in the referenced column. I want to return the sum of the values in the referenced column for all records with a matching index value. 

For example, if I had an order table including a part number column and a quantity ordered column, how could I return the sum of the quantities ordered for a specified part number? 

ashish mathur say:

Hi,

The easiest would be to build a Pivot Table.  Drag Part Number to the row labels and Quantity to the value area section.  Alternatively, you may write the SUMIF() function

=SUMIF($A$2:$A$100,D2,$B$2:$B$100)

A2:A100 has all part numbers, B2:B100 has quantity sold.  D2 has the specific part number.

amit tandon say:

Formula in cell D2:

=SUMIF(A2:A100,C2,B2:B100)

This returns the SUM of all value from column B, corresponding to column A values which match cell C2.

In case this is your requirement.
