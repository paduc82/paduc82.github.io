Windows ReadyBoost allows Windows Vista, Windows 7, and Windows 8 to use compatible USB flash drives as cache instead of the hard drive.

william radford say: when the libraries themselves do not exist, Windows will automatically default the start item paths to the actual folder locations. Quite intuitive actually, albeit lacking any actual notation that this happens within the system's Help documentation.

Open Start Menu User Folders to Actual Location Instead of Libraries

This method should only be used if you do not plan on using the default libraries included with windows relevant to the paths you'd like to modify.

Procedure

  1. Open the Start Menu, either by clicking the Start Menu button found in the taskbar or pressing the Windows Key
  2. In the Search/Run field, type Libraries and open the "Libraries" program that appears. You can simply start typing as soon as you open the start menu and your query will appear in the Search/Run field.
  3. Select the relevant libraries correlating to the User Folder paths you'd like to change and delete them, either by pressing the delete key or by selecting Delete from the Right-click Context Menu. Confirm the action to complete the process.

Potential Bugs

  • The user's "Documents" library may reappear within the "Libraries" 'Program' (Folder), this does not seem to affect the start menu item path change, however, if it does reappear, removing the "Public" folder included by default in the "Documents" library location is recommended, as this may prevent reversion of the start menu item path back to the library or change the default library view to the folder itself.
