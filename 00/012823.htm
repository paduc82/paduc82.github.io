ok, I've found a way to uninstall the apps...

execute this commands in a Powershell window with admin rights:

Get-AppxPackage

then you will see the complete list of the apps that are installed.

and to uninstall a specific app, use the following command:

Remove-AppxPackage -package PackageFullName

PackageFullName = is the PackageFullName of the app that you want to uninstall.

a. user say:

If you made a Windows install USB key it is a little easier to get to Recovery mode.

If you do not have the USB key then you can use the manual way to get in Recovery Mode.

Using the Power button to boot when it gets to the Mother board or manufacture Screen Press and hold the Power button to force shut down. do this 3 times should boot you to Recovery mode.

In Recovery Mode select Troubleshoot then Advanced then Startup options and you should see the Option to enable Safe Mode.
