Entering PPPoE - Microsoft Community

xb1r athena say:

Entering PPPoE

Login Information

  • Gateway - 192.168.1.1
  • Username - admin
  • Password - 1234

Entering PPPoE

  1. Open a web browser.
  2. Log into the router by entering the default gateway into the web address by typing 192.168.1.1 into the address bar
  3. The router will then prompt you to enter the username and password. The default username is admin and the default password is 1234.
  4. Expand the BASIC directory.
  5. Select the WAN subdirectory.
  6. Select Dynamic IP in the WAN screen.
  7. On the main page locate "WAN Mode" and select the PPPoE option.
  8. Enter in the PPPoE username and password.
  9. Click on Save Settings.
  10. Wait for the router to reboot.
