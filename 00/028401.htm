rohn007 say:
Yes you can (continue to) save macros to the "old" XLS" format. Graham meant that you cannot save macros in the new default XLSX format. In the new formats you have to explicitly select the XLSM file type if you want to save macros.

The old, XLS, file type is a "binary" format.  Think of it (very) loosely as being similar to a compiled program. The contents cannot be easily read using simple tools like notepad.  The new formats, XLSX and XLSM, are "XML based". That means they are primarily text based. If you change the file extension to ZIP, ie by simply adding .ZIP to the end of the file name, you can explore the underlying structure of the file. Most of the elements are text based so they can be read by simple programs like notepad.

Sorting a spreadsheet automatically. - Microsoft Community

adrian watts say:

Sorting a spreadsheet automatically.

I want to sort my data first by Column J and then by Column I un my spreadsheet - I know how to do this manually. How do I get EXCEL to perform the sort automatically? New data in columns A to F, will cause the values in Columns I and J to change. I want to see the ranking automatically, when the data changes.
shane devenshire say:

Here's one possibility:

Private Sub Worksheet_Change(ByVal Target As Range)
   Dim isect As Range
   Dim sRng As Range
   Set sRng = Range("A1").CurrentRegion
    Set isect = Application.Intersect(Target, Range("A2:F" & sRng.Rows.Count - 1))
    If Not isect Is Nothing Then
        With ActiveWorkbook.Worksheets("Sheet1").Sort
            .SortFields.Clear
            .SortFields.Add Key:=Range("J2"), _
            SortOn:=xlSortOnValues, Order:=xlAscending, DataOption:=xlSortNormal
            .SortFields.Add Key:=Range("I2"), _
            SortOn:=xlSortOnValues, Order:=xlAscending, DataOption:=xlSortNormal
            .SetRange sRng
            .Header = xlYes
            .MatchCase = False

Already answered your earlier.

Outlook is unrelated to the other three applications.  Consequently it does not use the text rendering engine used by the others.  Unlike the PC, the editor in Outlook is Outlook, not Word.

Consequently the text engine in Outlook can't do justification, kerning, ligatures and hanging punctuation.  But its simpler rendering engine is more forgiving of uncommon fonts.
