sanmartin say: The problem is that Vistalizator doesn't "install itself". The program changes system protection files and registry entries to make single-language editions of Windows accept the use of Multilingual User Interface packs (MUI's) limited to multi-language editions. These modifications are the reason that Windows is detected as "non genuine" afterwards. To revert that, you must revert the tampered files to their original state. To avoid a complete reinstallation of Windows, you could try an in-place repair upgrade (instructions: http://www.sevenforums.com/tutorials/3413-repair-install.html) which will restore the correspondent files and registry keys.
