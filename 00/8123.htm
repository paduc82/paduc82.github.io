julie fka profjulie say: When you install the Outlook Hotmail Connector and set it up to sync by inputting your windows live ID and password, a new file structure will be created within Outlook called something llike "Windows Live" - you'll see it in the left hand navigational pane of Outlook.  Within the Windows Live folder will be a Contacts folder, Appointments, folder, and etc. etc - the file structure will be similar to what you currently have under your personal folder.

If you don't have any contacts or appointments already posted to your Windows Live account those folders will be empty.  Then you can copy the contacts from your CONTACTS folder and copy them into the Windows Live Contacts folder and copy the BUSINESS CONTACTS from your personal folder and copy them into the Windows Live Contacts folder. You can do the same for your Appointments (copying them to the Windows Live Appointments folder). If you prefer to import this data directly into your Windows Live contacts & appointments (via your browser), you can do that instead.

After you've implemented and copied everything you want in your Windows Live folders, you can press the F9 button or select the Send/Receive option to manually sync this stuff in the Windows Live folders to your Windows Live account.  You can set up the Filter & Rules option for this folder to sync the Windows Live folders however often you prefer.  Once the data is in your Windows Live folder you can sync it to your phone.

The data in your Personal folders will never be synced to windows live, so you should stop using the Personal folders altogether...

teylyn say:
Hello,

to set this up for the first time, you need to change all occurrences of the date to a reference to a cell or, better yet, a named range.

1. Create a named range that points to a cell with the date you want to use. To do that, enter the desired date into a cell, then type a name like "TheDate" into the name box (the text box above cell A1) and hit enter. 

2. find all cells that use the date as a fixed value and replace it with the range name "MyDate". You can use this range name in any sheet in your workbook.  As an alternative to the range name, you can use a reference to the cell, like "Sheet1!$A$1"

A formula with a range name may be easier to read, especially if you use descriptive names for named ranges.

Compare

=EndDate-StartDate

with 

=Sheett1!$E$5-Sheet5!$F$13

Kind of messy, but I can't think of another way that's better:

SELECT *
FROM table
WHERE EmpID IN (
       SELECT TOP 2 H.EmpID 
       FROM table As H
       WHERE H.EmpID = table.EmpID And H.Pay >= 1000
       ORDER BY H.Random DESC)
UNION ALL
SELECT *
FROM table
WHERE Rate IN (
        SELECT TOP 2 M.Random
        FROM table As M
        WHERE M.EmpID = table.EmpID And M.Pay < 1000
        ORDER BY M.Random DESC)
UNION ALL
SELECT *
FROM table
