doug robbins - mvp office apps & services (word) say:

You have not indicated how the data should be arranged in Word, but the following should get you started:

Dim arrdata As Variant
Dim i As Long, j As Long
Dim appWord As Object
Dim docWord As Object
arrdata = Sheets(1).Range("A1:Z" & Sheets(1).Range("A" & Sheets(1).Rows.Count).End(xlUp))
On Error Resume Next
Set appWord = GetObject(, "Word.Application")
If Err Then 'Word was not running
    Set appWord = CreateObject("Word.Application")
End If
On Error GoTo 0
appWord.Visible = True
appWord.Activate
Set docWord = appWord.Documents.Add
With docWord.Range
    For i = 1 To UBound(arrdata, 1)
        If arrdata(i, 26) = "Y" Then
            For j = 1 To 3
                .InsertAfter arrdata(i, j) & vbTab
            Next j
            .InsertAfter arrdata(i, 4) & vbCr
        End If
    Next i
End With

jay freedman say:

The problem was caused by the author (or the editor) of the article you copied the code from: The <code><span> at the beginning of each line and the </span> at the end of each line are supposed to be HTML code to change the way the rest of the line is displayed. They are not valid parts of the VBA code, and they must be removed.

You can easily remove them with these steps:

  • Reopen the macro editor (Alt+F11).
  • Select the <code><span> at the beginning of the first line.
  • In the menu at the top of the editor, click Edit and then click Replace (or use the shortcut Ctrl+H) to open the Replace dialog. The selected text will appear in the "Find What" box, and the "Replace With" box will be blank.
  • Click the Replace All button. Click OK in the message that says how many replacements were made. The <code><span> has been removed from all of the lines.
  • Click the Cancel button in the Replace dialog.
  • Select the </span> at the end of the first line.
  • Open the Replace dialog again, and click the Replace All button. Click OK in the message box. The </span> has been removed from all of the lines.
  • Click the Cancel button in the Replace dialog.
  • Click File and then click Save.

Now test the dropdowns. If they still don't work, you may have some other problem, for which you can post here again.
