obsessive power say:

Actually we don't need to know your hard drive's make.

When you attach an external hard drive to the Xbox One and choose to format it for use with the console, the system changes the formatting of the drive to the one that the console uses.  Once this is done, your PC will not recognise the drive for data storage, but it should know that it is attached.

To enable you to use the drive again on your PC, it must be re-configured.  To do this you need to load the 'Disk Management' tool.  If you're running Windows 8 the quickest way you can access the Disk Manager is by Holding down the Windows Key and pressing X then selecting 'Disk Management' from there.  In Windows 8.1 you can right click on the START button and select it.

To quickest way to access Disk Management on Vista and Windows 7 is via the Run program on the Start Button and typing in diskmgmt.msc and pressing Enter or the OK button.

If your hard drive is in working order, it will be detected in Disk Management and you will be prompted to 'Initialise the device'.  This will then make the drive available in Windows, but it will be formatted again only Windows use.

At present it isn't possible to use an external hard drive for Xbox One and PC use, it has to be one or the other.

Hope that helped.
