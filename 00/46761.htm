                           net stop cryptSvc

                           net stop bits

                          ren C:\Windows\System32\catroot2 Catroot2.old

4. Now, restart the BITS, Cryptographic, MSI Installer and the Windows Update Services. Type the following commands in the Command Prompt for this. Press the ENTER key after you type each command.

                        net start wuauserv

                        net start cryptSvc

                        net start bits

                        net start msiserver

5. Type Exit in the Command Prompt to close it and then restart the computer.

6.After restarting the computer, now retry to install Windows Updates.

mgranback say:

Had the same problem with the Spotlight Lock Screen images on two computers and had went through all the numerous suggestions here and elsewhere without finding a resolution until I finally noticed the following:

1.  Before immediately trying to delete everything from the Assets and Settings folders, try the usual "first" suggestion of switching Lock Screen image from Spotlight to Picture.

2.  Get back out of settings and activate lock screen using Windows and "L" keys together and verify that the selected picture comes up.

3.  May not be necessary but I did a restart at this point.

4.  After restart go back to settings and switch Lock Screen back to Spotlight BUT DON'T EXPECT IT TO CHANGE TO A SPOTLIGHT IMAGE IMMEDIATELY.

5. Get out of settings and Use the Windows+L keys to activate the lock screen but, again, don't be surprised if you're still seeing the default "backpack on a lake" image. Then do another restart (again,  may not be necessary but just stating what I did to get mine working).

6.  After restart, and this is the key to what I've experienced, WAIT!  Don't expect Spotlight Images to start showing up immediately and you'll probably still have the same "backpack on a lake" image or whatever.  On my laptop, it initially appeared that the above steps weren't working.  About 45 minutes later, however, I noticed image had changed in Settings.  Tried activating the lock screen with the Windows+L keys again and, sure enough, Spotlight was working again.  When I tried the same thing on my Desktop, however, IT TOOK 36 HOURS before the first new Spotlight image came up even though I could see the new images in the Assets folder shortly after taking the above steps

7.  Once (if) you get spotlight to start working again DO NOT select "Not a fan" for any of the images for at least a few days to a week.  After I got mine working on both computers, Spotlight would work fine as long as I did nothing or "liked" the image.  As soon as I selected "Not a fan", however, Spotlight stopped working and Lock Screen went back to the default "backpack on a lake" image again and I had to repeat the above steps.  After letting it go for a week, however, I tried it again and selecting "Not a fan" worked normally again changing to another Spotlight image and has continued working normally for over a month now.  Perhaps it just takes that long for enough pictures to upload and system to adjust to be able to change to new picture ???.  I also discovered, however, that, even after Spotlight had been working fine for over a month, I tried selecting "Not a fan" for three images in a row.  The first two times it changed to a new image.  The third time it went back to the default "backpack on a lake" image and stopped working again.  So I'd suggest not selecting "Not a fan" for several images in a row.

8.  If, even after waiting a day or two, Spotlight lock screen images don't start working again it may still be necessary to delete the contents of the Assets and Settings folders or try the numerous other suggestions that have been offered.  Based on my experience, however, once you do get it working I'd still suggest not expecting immediate results and not selecting "Not a fan" for a few days to a week after you do get it working.
