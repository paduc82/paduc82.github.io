a. user say:

Hi Riff,

Thank you for posting your query in Microsoft Community.

I understand your concern, I will be glad to assist you.

The process is actually quite simple.

Note: Before you start, make sure you have an up-to-date backup of the system you want to keep. If you make one tiny mistake in the steps below, you could end up deleting the wrong partition, which would be very very bad!

Keep Windows and Remove Linux:

Follow the steps and check if it helps.

  1. Type "Disk Management" in search box, Open the Disk Management tool.

  2. Find Linux partition. It won't be labeled since Windows doesn't understand the Linux file system, so you'll need to figure out where it is on your hard drive. Make sure you have the right one before continuing!

  3. Right-click on the partition and choose "Delete Volume." This will delete the partition from your hard drive, leaving free space. If you get a green box of "free space," you'll have to right-click on it and choose "Delete Volume" a second time until you get the black "Unallocated Space."

  4. Right-click on your Windows partition and choose "Extend Volume." Extend it to fill the free space your Linux partition left behind.

  5. Lock the computer and  Press Shift key + Restart it will take you to Windows Recovery Environment there select "Troubleshoot,">>select Advanced options>> click on Command Prompt. Type the following command: 
    bootrec.exe /fixmbr
    This will remove Linux's bootloader and replace it with Windows'.
  6. Reboot your computer and you should find that it boots directly into Windows, with no Linux partition to be found.

Hope the information is helpful. Let us know the status of the issue.

sam___ say:

Hi Sharon,

To open multiple Windows of the same applet hold down the 'Shift' key and click on File Explorer again.

To view windows side by side you can drag one of the windows you want to view to either the left or right side of the screen and let go of the click. A new screen will show up that will allow you to select the other window you will view side-by-side. Another method, if you have only 2 windows open, is to launch the 2 instances of File Explorer, right click on the task bar and select the "Show Windows Side by Side" option.

Update the chipset drivers and check.

  1. Navigate to your computer or motherboard manufacturer's website.
  2. Click to enter the support or driver download page. If you have a prebuilt computer, enter the model number for your computer to see if you have motherboard drivers available to you. Otherwise enter the motherboard's model number.
