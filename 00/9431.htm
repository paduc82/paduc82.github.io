I've included the full macro but it's the bottom bit in bold that is going wrong. 

What I want to do, is once NAME YOUR STRING is done, I want to move two cells to the right and count the number of characters in the cell to the left (ie. NAME YOUR STRING is in A3, move two across to C3 and count what's in B3).

Any help much appreciated!

Thanks

Steve

Sub Edit_Boolean_Save_Strings()
'
'

' Copy

Sheets("Search String Result").Select
    Range("C2:M10").Select
    Selection.Copy

' Go to saved strings and select next empty row

Sheets("Saved Strings").Select
lMaxRows = Cells(Rows.Count, "c").End(xlUp).Row
Range("c" & lMaxRows + 1).Select

' Paste Special - Values

Selection.PasteSpecial Paste:=xlPasteValues
        
' Count characters in string
        
ActiveCell.Offset(0, 1).Select
ActiveCell.Formula = "=LEN(RC[-1])"

' Move one to the left

ActiveCell.Offset(0, -2).Select

' Escape

Application.CutCopyMode = False

' Autofit

Selection.EntireRow.AutoFit

' Select cell

Sheets("Search String Result").Select
Range("C2:M10").Select

' Name

Sheets("Saved Strings").Select
ActiveCell.FormulaR1C1 = "NAME YOUR STRING"

' Change font size and centre

    With Selection.Font
        .NAme = "Calibri"
        .Size = 12
    End With
    With Selection
        .HorizontalAlignment = xlCenter
        .VerticalAlignment = xlCenter
    End With

' Include characters

    ActiveCell.Offset(0, 2).Range("A1").Select
    ActiveCell.FormulaR1C1 = "=LEN(RC[-1])"
    ActiveCell.Offset(0, -2).Range("A1").Select
