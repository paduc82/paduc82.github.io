How to use dropintolist to drop Process shape into Swimlane in Visio - Microsoft Community

shootforstars say:

How to use dropintolist to drop Process shape into Swimlane in Visio Using VBA

I am trying to use Activepage.DropIntoList command to drop a Process/Decision shape into a specific Swimlane using VBA in Visio/Excel. I keep getting source & destination errors. A couple of working examples would be helpful.

Also, question, does the shape automatically align vertically & horizontally (by number) within the swimlane with this command? Currently Im using Drop with x, y coordinates which is clumsy and not easily scalable.

Any advise is appreciated. Thanks in advance

david j parker say:

I don't think you are using the correct method because the Process shape is not going to be a member of a list. It is just going to be a member of a container.

The following code will drop a Process shape into the first Swimlane .

Public Sub DropProcessInSwimlane()
Dim shpSwimlane As Visio.Shape
Dim shpProcess As Visio.Shape
Dim mstSwimlane As Visio.Master
Dim mstProcess As Visio.Master
Dim sel As Visio.Selection

    Set mstSwimlane = ActiveDocument.Masters("Swimlane")
    Set sel = ActivePage.CreateSelection(visSelTypeByMaster, 0, mstSwimlane)
    Set shpSwimlane = sel.PrimaryItem
   
    Set mstProcess = ActiveDocument.Masters("Process")
    Set shpProcess = ActivePage.Drop(mstProcess, _
        shpSwimlane.Cells("PinX").ResultIU + _
        0.5 * shpSwimlane.Cells("Width").ResultIU - _
        1.5 * mstProcess.Shapes(1).Cells("Width").ResultIU, _
        shpSwimlane.Cells("PinY").ResultIU)
    shpProcess.AddToContainers
End Sub
