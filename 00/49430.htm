cbarnhorst say:

I suspected that the upgrader would not be able to find the digital entitlement data in the firmware.  It will make no difference whether you use media.  Something corrupted the digital entitlement data.  Your copy of Windows 10 doesn't use a product key.  It uses the new activation technology, digital entitlement.  The only product keys that work are retail and special program keys.  The Get Windows 10 upgrades are a general program.

The Genuine Windows page is looking for a product key to see if you are using a blocked key.  It also checks for known activation hacks.  It may not be able to check digital entitlements yet.

The problem is that the Windows 7 Media Creation Tool is simply the Microsoft Store tool and is set to only create media for retail product keys.  The reason is that the Microsoft Store assumed customers with OEM product keys needed to contact the OEM for recovery media to restore the computer to factory condition.  Of course there are other scenarios where a user might need Windows 7 media.  The MC Tool for 8.1 corrected this mistake by making downloads available to anyone who wants one.  The Windows 10 MC Tool continues the broader view.

If the product key you have is from the sticker affixed to the computer then it should work with Windows 7 Home Premium installatin media.  The trick is to find one.  Call your local computer shops to see if they can make a copy of one for you.  Then you can recover to Windows 7 Home Premium and upgrade again to Windows 10 Home.  The shop needs to have a RETAIL dvd of Windows 10 Home, although if all else fails you can try whatever copy of Windows 7 Home Premium they can make for you.  Copies of OEM dvds may or may not work.  It varies.

Once you recover to Windows 7 HP you do not need to do any updates, not even Service Pack 1.  You can use the Windows 10 MC Tool and upgrade to 10.  The business about updates only applies if you want to use the Get Windows 10 icon on the taskbar because that requires a particular update before the icon will appear.

You don't really need to uninstall your old printer, but if you want to do so, first go to Control Panel > Programs and Features and uninstall any software related to the old printer.  Then click the Start Orb, go to Devices and Printers, right-click on the icon for your old printer, and select "Remove device."

Your new printer should have come with installation instructions.  If you don't understand those instructions, please provide the specific printer model you purchased.

Briefly, if you are going to connect the printer to the computer using a USB cable, make sure that your computer is connected to the Internet, then connect the printer to the computer, and turn on the printer.  Windows should detect your printer and automatically install it.  If Windows asks for permission to look in Windows Update for a printer driver, allow that to happen (that's why you need to be connected to the Internet).

The process described in the preceding paragraph does not, of course, install all of the junk software that HP bundles with its printers.  If you feel the need for such software, then follow the installation instructions that come with your printer.  Generally, this requires that you first insert a CD into your computer and then follow on-screen instructions until requested to connect your printer.

If you are going to connect the printer using wifi, then it is far simpler and easier to use the CD accompanying your printer to do the installation than to install the printer manually.

BT Homehub 4 Open NAT Solution! - Microsoft Community

dee dah dave say:

BT Homehub 4 Open NAT Solution!

This guide is for the Homehub 4 but should help with other routers too.

-

Okay I'm going to presume you know your way around your router settings a little bit and get straight to the point. I suggest having your xbox switched on and logging into your router with a PC or other device. First we want to make sure a few settings are in place.

-

Go to advanced settings > home network > devices. Your xbox should be one of the listed connections, click on it. Down at the bottom it will say "Always use this IP Address". Make sure no is selected and click apply.

-

Go to advanced settings >Firewall > UPnP. Make sure both "UPnP" and "Extended UPnP Security" are set to yes and click apply.

-

Go to advanced settings > Firewall > DMZ. Make sure "DMZ Enable" is set to no and click apply.

-

Go to advanced settings > Firewall > Port Forwarding. Click on "Add new game or Application". In the top box call it "My Xbox Live Manual Port Forward".

Copy and existing game/application choose no.

Now you want to add the following protocols and ports. Do each of the following clicking add after each . The translate to boxes can be left blank -

- Protocol UDP Port Range 88 - 88

- Protocol UDP Port Range 3074 - 3074

- Protocol TCP Port Range 3074 - 3074

- Protocol UDP Port Range 53 - 53

- Protocol TCP Port Range 53 - 53

- Protocol TCP Port Range 80 - 80

- Protocol UDP Port Range 500 - 500

- Protocol UDP Port Range 3544 - 3544

- Protocol UDP Port Range 4500 - 4500

Click apply. Click "back to Port Forwarding". In the "Game or Application" dropbox select your newly created "My Xbox Live Manual Port Forward". In the "Device" dropbox select your xbox. Click add. Click apply.

-

Go to Advanced Settings > System > Restart. Click "Restart My BT Home Hub". Wait a few minutes for the router to properly start back up. You can now close the window and exit out of your router settings.

-

Go back to your Xbox. Go to Settings > Network > Advanced Settings > Alternate MAC Address > Clear > Restart

-

Your Xbox will do a full shutdown and restart which will take a minute or two. When back on go to Settings > Network > Test Multiplayer Connection, all should be good.

-

You should now have an open NAT.

ronald san say:

Hi,

If you want to remove a user account from your Windows 10 computer because you don’t need it anymore, follow the steps below:

NOTE: It is always recommended you keep a backup of all your data before proceeding.

  1. Open the Settings app, click Accounts, and navigate to Family and other users.
  2. Click on the account you want to remove from your computer, and click Remove.
  3. You’ll be prompted to confirm, and if you really want to continue, click Delete user account and data.

This will remove the user account from the login screen, as well as any files, settings, and program data stored on that user account.

Hope the information helps.

ronald san say:

Hi,

Please make sure that you are not logged in using the user account you want to remove, and try the following procedures.

Method 1:

  1. Press the Windows + R keys to open the Run dialog, type lusrmgr.msc, and press Enter.
  2. In the Local Users and Groups windows, and click on the Users folder in the left pane to open it.
  3. Click on a user (your accidentally created account) you want to delete in the middle pane to select it, click on More Actions under the same user's name (your accidentally created account) in the right "Actions" pane, and click on Delete.
  4. Click on Yes to confirm.

Method 2:

  1. Press the Windows + X keys, and click on Command Prompt (Admin).
  2. In the elevated command prompt, type net users , and press Enter to see a list of all user account names on the PC.
  3. Should you see the account you want to remove, type net user "the_account_you_want_remove" /delete, and press Enter.

In case you need further help, feel free to let us know.
