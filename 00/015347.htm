=IF(VLOOKUP(B2:B43,E!A3:C795,2)<>" ",VLOOKUP(B2:B43,E!A3:C795,2),VLOOKUP(B2:B43,'R'!A3:C1555,2))

jeeped say:
Well... seeing as how its urgent, try,

=IFERROR(VLOOKUP($B2,E!$A$3:$C$795,2,0),IFERROR(VLOOKUP($B2,'R'!$A$3:$C$1555,2,0),""))

... which looks up B2 in the current sheet within the E sheet's A3:A795 range and returns the corresponding value in the B column (exact match required). If it is not found, it tries again in Sheet R's A3:A1555 range and returns the corresponding value  from column B if an exact match has been met. If neither have been accomplished, an empty string (e.g. nothing) is returned.

That formula can be filled down as necessary.

If only columns A and B are to be considered, I don't know why you are referencing column C, but perhaps you need the the static range reference for an additional lookup.

Addendum: if you were planning to use a similar formula to pickup the values in column C, use this instead:

=IFERROR(VLOOKUP($B2,E!$A$3:$C$795,COLUMNS($A:B),0),IFERROR(VLOOKUP($B2,'R'!$A$3:$C$1555,COLUMNS($A:B),0),""))
... which can be filled both right and down.

It may be of worth to note that the supplied formula uses IFERROR() which is proprietary to Excel 2007/2010. If the workbook needs to be used in previous versions of Excel then an IF(ISERROR()) conversion should be used. Post back if this is required.

VBA Variables - Microsoft Community

danielruff say:

VBA Variables

In functions like AVG, an unlimited amount of variables can be chosen until you close the parentheses to end the function. When writing a VBA, how can I make it so that my function has an unlimited amount of variables?  

jp ronse say:
In functions like AVG, an unlimited amount of variables can be chosen until you close the parentheses to end the function. When writing a VBA, how can I make it so that my function has an unlimited amount of variables?  

Unlimited is not really correct. It is limited to 255 entries but a range is considered as 1 entry.

Say that your worksheet formula is: =AVERAGE(A1:A1048576), you can use EVALUATE("AVERAGE(A1:A1048576)"). Preferrable with a reference to the worksheet. Activesheet.Evaluate.

jllatham say:

For the second part of your question, you can set up a variable number of parameters by using the OPTIONAL declaration.  It would go something like this:

Function MyFunction(P1 As Range, Optional P2 As Variant, Optional P3 As Variant) As Double

Check Help for the requirements for Optional arguments - one, as indicated is that they must be type Variant (naturally there are exceptions to this 'rule').

Also check Help for IsMissing

A good place to start is probably on this page at Chip Pearson's site:

http://www.cpearson.com/excel/optionalargumentstoprocedures.aspx
