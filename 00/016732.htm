djwiginc say:

Spell a number in a format of changing currency

shane devenshire say:

You need something to trigger a macro to do this.  Unfortunately Excel does not recognize a format change as a change event.  If you want to trigger it when a value is typed into a cell you could use:

Private Sub Worksheet_Change(ByVal Target As Range)
    Dim isect As Range
    Set isect = Application.Intersect(Target, Range("A1"))
    If Not isect Is Nothing Then

    If Range("A1").NumberFormat = "_([$$-1004]* #,##0.00_);_([$$-1004]* (#,##0.00);_([$$-1004]* ""-""??_);_(@_)" Then
        Range("B1").NumberFormat = "_([$$-1004]* #,##0.00_);_([$$-1004]* (#,##0.00);_([$$-1004]* ""-""??_);_(@_)"
    ElseIf Range("A1").NumberFormat = "_-[$¬£-452]* #,##0.00_-;-[$¬£-452]* #,##0.00_-;_-[$¬£-452]* ""-""??_-;_-@_-" Then
        Range("B1").NumberFormat = "_-[$¬£-452]* #,##0.00_-;-[$¬£-452]* #,##0.00_-;_-[$¬£-452]* ""-""??_-;_-@_-"
    'many more here
   End If

   End If
End Sub

You could use the Selection_Change event instead, which would mean it would run everytime the cursor was moved.  In any case there will be a lot of coding to get every currency format that Excel supports.

I would do it with a case statement rather than if.

1. To add this code to your file, press Alt+F11, 
2.  In the VBAProject window, top left side, find your sheet name under your file name and double click it.
3.  Paste in or type the code above.

nsilmike say: a hub port is a usb port that is shared to several usb sockets.  usb ports can supply up to 500mA max., so if you have devices connected to a hub that are demanding more power (combined) than 500mA, you get some sort of error such as you have seen.  Try different ports, and/or disconnect some other usb devices when you connect the zune.  And, in the device manager, you can see what devices and how much power they are using per port.
