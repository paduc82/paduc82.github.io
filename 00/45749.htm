original title: DDR3 RAM Speeds

worldtraveler58 say:

Download a utility called CPU-Z (google for it).  It will tell you a lot of things about your MoBo and memory.  Select the Memory tab and look for "DRAM frequency."  That will tell you the actual current clock speed of the memory.  On the SPD tab you can find the specs for the memory module in each slot.  It will tell you the rated clock speed of each module.

 

There are a number of reasons for memory running slower than its rated speed, but sub-standard memory is a possibility.  It's also possible that it's running slower to save power.  I have an older VAIO with memory rated at 400 Mhz and CPU-Z shows it running at 399 (I doubt if it's that accurate).

 

BTW, the speed ratings of memory can be confusing.  DDR3 with 1333MT/s (Mega-transfers/sec) transfers data on both the rising and falling edges of the clock pulse.  Therefore the clock is actually running at 667Mhz (megahertz).  It also transfers 8 bytes at a time, so it's catagorized as PC3-10666 (10666 = 1333 x 8).  So everything may be correct with your memory.
