Q: Why aren't my JPEGS importing correctly?

These were saved from Photoshop using the CMYK JPEG option, but Word 2007 can't handle them. How do I fix this?
graeme_bennett say: There's no intrinsic reason why these shouldn't import correctly. Office 2007 can handle RGB JPEG, CMYK JPEG, progressive JPEG, and even EPS with JPEG preview images with aplomb. Either some oddball plug-in was used to export these JPEGs in the first place (perhaps they were created by a wavelet-based JPEG 2000 plugin?) or the file has become corrupted somehow. Check that it isn't compressed as a zip file or otherwise archived. If all else fails, load the files back into Photoshop and save into any of the many other image file formats Microsoft Word supports, such as TIFF, BMP, PNG, etc.
