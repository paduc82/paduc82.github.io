rinus. say:

I experienced the same problem but found the solution (in my case at least).

I found out that when I programmed one of the pen buttons to 'right-mouse click' that when I pressed that when I used that button in Onenote I got the same ring-shaped indicator popping up as well as a right-click menu.

And then I started looking into windows pen settings and found there's a setting that simulates a 'right-mouse click' when you press and hold down your pen. Turning this option off solved the irritating issue in Onenote for me.

1. Click start.

2. Type ‘Pen Tablet Properties’ and press Enter

3. under 'pen action' select the line 'press and hold'

4. press 'settings' and disable the 'right-mouse click' simulation.

When you boot to Safe Mode, a whole bunch of stuff happens. The font cache gets emptied. Disk Utility runs and fixes any directory and volume problems it encounters, and only a minimal set of default fonts is loaded. Because not all fonts are loaded, if your document uses a font that is not one of the basic fonts, your text may appear blank. It's there, it just isn't displayed. You could select all and choose an available font in Safe Mode, but overwrite your existing file. Use File > Save As and choose a new name if you want to save a version for continued testing.

Apple says the waves of bars is to be expected sometimes in Safe Mode. I don't know why, and this wasn't a problem until a few months ago, but it seems Apple is living with this for the time being.

Booting to Safe Mode should prevent "helper" software such as TypeIt4Me from running, but Word add-ins such as Zotero and EndNote will probably still work.

Cross referance, bookmarks or Fields????? - Microsoft Community

chumio say:

Cross referance, bookmarks or Fields?????

Hi all,

For what seems to be a simple request may turn out to be quite tricky.

I would like to be able to create a a field or book mark that the user can type a word into and have it populate below in the table.

So The user would enter "12345" and "This is the description" and it will appear on the grid below as shown.

I was able to do this using either a bookmark or cross referance.

I would like to then save this selection/form as a Quick Part Selection and be able to insert it multiple times in the one document.

I was also able to do this however when I change the item number on a newley inserted Quick Part all the other referances change.

Is there any way a Quick selection or a Bibliography save can have all the referances contained within it so that if i change one it will not effect the other?

Cheers :~)

suzanne s. barnhill say:

There are basically two ways to repeat information of this kind:

  1. With a cross-reference (REF field). Unless the text you want to cross-reference is automatically bookmarked by Word (that is, it is in one of the built-in headings styles or is an autonumbered item, footnote, endnote, or caption), you will need to manually bookmark the text, then insert a cross-reference to the bookmark text.
  2. With a StyleRef field. You apply a unique style to the text to be repeated and insert a StyleRef field where you want it repeated.

In either case, the trick is how to enter information that will be bookmarked or will be formatted in the required style. Unless you use a UserForm that will insert information at a bookmark (and in that case you might as well go ahead and insert it everywhere it's needed--see http://gregmaxey.mvps.org/word_tip_pages/repeating_data.html for more), you will probably have to use a form field (which by default includes a bookmark) or a content control. Using form fields requires protecting the entire document for forms, which greatly reduces functionality. Using content controls can be tricky.

Consequently, for this type of application, I prefer to use MacroButton NoMacro fields for data entry and StyleRef fields to repeat the data entered. One special benefit of StyleRef fields is that they update automatically and immediately (unlike REF fields, which have to be updated explicitly). The article at http://word.mvps.org/FAQs/TblsFldsFms/UsingMacroButton.htm explains how to insert a "[Click and type]" MacroButton field, and http://WordFAQs.mvps.org/StyleRef.htm explains how to define a character style to apply to the field. When data is typed into this type of MacroButton field, it overwrites the field, becoming ordinary text, but it will be formatted with the special character style that is referenced by the StyleRef field(s) you have inserted where you want the entered data repeated.
