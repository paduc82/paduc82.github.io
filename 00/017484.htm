How to auto replace cell content once detected by Excel? - Microsoft Community

angelstory3000 say:

How to auto replace cell content once detected by Excel?

I extract a monthly report from SAP with a lot of date.  In that data a column with codes.  Is there a way to automatically replace all cells in one column in Excel once detected? For example I want A001,A002 & A003 = "Gas" and some other code will be replaced by "Oil" and so on?
ossiemac say:
  1. Ok then somewhere else in the workbook (either out to the right of your data or on another worksheet) Copy the column of codes from the data and paste to one column.
  2. Select the new column of codes and use Remove duplicates.(On Data Ribbon)
  3. Enter the replacement codes in the adjacent column.
  4. In the main data Insert a blank column beside the column with the codes.
  5. Use VLOOKUP function to look up the required replacement in the created list. (See Help for VLOOKUP function. Note that when entering the table_array part of the formula it is best to select the entire columns or ensure that the table array is in absolute reference style with the $ signs)
  6. Copy the formula to the bottom of the data.
  7. Select the column with the formulas and Copy.
  8. With the column still selected, PasteSpecial -> Values. (This pastes values over top of the formulas which removes the formulas and leaves the values)
  9. Delete the main column of data which leaves the newly created codes.
