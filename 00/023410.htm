todtown say: There is a VBA solution. You can use the worksheet change event. for this. Code in this event will run whenever there is a change on the worksheet. You can set criteria that will allow the code to only run with these cells are changed. Something like this:

'Put this code in the worksheet module
'Assuming the time stamp will appear in Cells A2 and B2, respectfully. 
Private Sub Worksheet_Change(ByVal Target As Range)
    If Target.Column = 3 Then
        ActiveSheet.Range("A2").Value = Now()
    ElseIf Target.Column = 11 Then
        ActiveSheet.Range("B2").Value = Now()
    End If
End Sub

steve rindsberg say:

There are a couple of ways of going about this.

There are free utility programs (one called Pixie comes to mind) that will "sample" any on-screen color that you click on and show you the color values in any of several specification systems. 

But you don't really need a separate program.

Draw a rectangle or other shape, set its fill to the value you're after or work with a shape that's already given that fill.

Rightclick the shape, choose Format Shape, and in the Fill section of the formatting pane, click the dropdown to the right of "Color" and click More Colors.  The shape's color will appear in the Colors dialog box with its RGB values set.

If you don't have any examples of the color to work with, let us know where the "Grey 80%, Text 1" spec came from.  A pre-2007 version of PowerPoint, maybe?

It might be close enough to start with 80% of RGB white; for white, each RGB value would be 255, 80% of that is 204, so try RGB 204/204/204.
