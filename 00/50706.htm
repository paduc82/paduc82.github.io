a. user say:

Using OneDrive for File History - or for any other regular backup - is not a good idea, which is probably why you're not finding much help on the subject. File History was designed to work with a local or networked storage destination, i.e., not on the internet.

Many PC users' upload speed to the internet is only a small fraction of their download speed. (Verizon FIOS is an exception.) Depending on how you use it, File History can easily suck up most of your upload bandwidth, which would make File History really slow and interfere with any other uploading you are trying to perform from your PC.

Another disadvantage to using File History with OneDrive: If you lose your internet connection, you've lost your backup.

But even at its best, File History is no match for a full-out backup program. It wasn't intended to be a full-out backup program. If you want 'the good stuff' in backup software, you want a third-party program.

Hi

Hardware device compatibility is the responsibility of the hardware manufacturer.

If HP does not provide drivers for that monitor you can try installing drivers for a previous version of Windows (Windows 7) using the Compatibility Mode.

Go to the following website.

HP 2159m 21.5 inch Diagonal Full HD LCD Monitor Drivers:

http://h10025.www1.hp.com/ewfrf/wc/softwareCategory?cc=us&dlc=en&lc=en&product=3884707

In the Operating System drop down menu select the Windows 7 64bit or Windows 7 32bit and click the Next button. This will display the Driver download page.

1. Download the driver for Windows 7 and place it on your Desktop.
 
2. Right click the driver installation file and select Properties / Compatibility Tab.
 
3. Place a check mark in the Run this program in compatibility mode for: option.
 
4. In the drop down menu, select Windows 7.
 
5. Click Apply / OK.
 
6. Right click the installation file and select the Run As Administrator option to install the driver.

Let us know the results.
