But if you want only a few frames arround some cells, you can replace the GroupBox with a rectangled shape. (Tab Insert\Shapes\Square)

Sub Makro1()
  ActiveSheet.Shapes.AddShape(msoShapeRectangle, 40.5, 24.75, 117, 57).Select

  Selection.ShapeRange.Fill.Visible = msoFalse
  Range("A1").Select
End Sub

This shape has the same behavior as the GroupBox, except that the frame has no title.

ipellant say:

Sounds like the battery management went nuts in Win 10 and has balked the charging. Or the battery died of old age.

A completely flat battery will stop some laptops from powering up.

Most laptops will power up (when plugged in) with the battery removed - this is in case of battery failure, the laptop can still be used.

It may need a hard reset...

Remove the battery and power cord. Press the power on button for half a minute.

Plug the power cord back in and see if the battery begins to charge. If not, the battery has died.

Only turn the laptop back on when the battery has charged, or remove the battery and  plug the laptop into a power source.

dianeoforegon say: If you switch to IMAP you won't have this issue. Each message is given an ID number. If Yahoo reboots their server, messages are given new ID numbers and Entourage will see them as new messages and download again. 

POP accounts only download from the Inbox, You can put emails on the server you want to archive in a folder you create online.

  • In Entourage, select Accounts under Tools in the Menu bar.
  • Click on Option tab.
  • Uncheck "Leave a copy of each message on the server" if you to immediately delete from server or select one of the options to delete after xx days.


Yahoo account setup:

IMAP

    Incoming Server - imap.mail.yahoo.com
    Incoming Port - 993 (requires SSL)
   
    Outgoing Server - smtp.mail.yahoo.com
    Outgoing Port - 465 (requires SSL/TLS)
   
User name: full email address (for example, bill @ yahoo.com* or bill @ rocketmail.com*)
   
Password: the password you log in to Yahoo! with.

*Spaces added to emails. Forum server strips out the info.

Note: If your Yahoo! Mail address does not match your Yahoo! ID, you will need to use the beginning of your Yahoo! Mail address when you set up your client. For example, if your Yahoo! Mail address is "*** Email address is removed for privacy ***", but your Yahoo! ID is "james", you will need to use "jim" as your user name when setting up the IMAP client.
