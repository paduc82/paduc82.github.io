I'm sure it isn't the only way, and probably not even the best way, but I usually get one cell working and then copy/paste format to the column or row, as appropriate. One thing to remember is that as you are building your formula, anchors ($) matter. So for example, if you want every cell in Row 1 to be red if A1 is less than 10, your CF formula (in B1) might be =$A$1<10, and then you can just copy the CF to the whole row.

Without more information, it is hard to tell if anchors is really the problem, but it is a common problem.

When creating a table of contents from the gallery, Ctrl + Click - Microsoft Community

apf22 say:

When creating a table of contents from the gallery, Ctrl + Click does not work correctly

When you create a table of contents from the gallery, Ctrl + Click does not work correctly. You have to select the title, such as Table of Contents, in order to select the entire table of contents. Then when you hover over an entry you see "Ctrl + Click to follow link". Unless the box surrounding the table of contents is visible, you can't click on the table entries to navigate to the desired page.

I worked around this by selecting Insert Table of Contents. If you select this with all the defaults then when you hover over any entry, you see "Ctrl + Click to follow link" and you can click on the link to navigate.
