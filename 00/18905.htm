possejohn say:

How to Force "Enable Macros"

Greetings,

I am using Word 2007 and am working on a LAN system at work.

The LAN administrator has made it so that ANY document opened with VBA code, the macros/VBA has to be enabled by manually by the user.

In Excel, I get 'around' this by displaying a welcome worksheet that tells the user to enable macros.  After macros/vba are enabled, the rest of the worksheets are taken from xlVeryHidden to visible.

I now have a Word document that has VBA code, which requires the user to enable to ensure certain characteristics are embeded into the word document, specifically, when the document is saved & closed, a copy of the document is saved and archived into a special folder, and that the archived document is password protected from changes.  This is done so that historical record of what was documented can be retrieved, and have some integrity.

QUESTION:  How do I 'block' any changes to the word document until the user has enabled macros/vba?  How do I 'force' the user to enable macros/vba before using the document?
