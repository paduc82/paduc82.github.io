4. Restart OneNote.

jay freedman say:
Word has two kinds of styles: paragraph styles and character styles.

A paragraph style, as the name implies, applies to an entire paragraph. Its definition includes both character formatting (font, size, color, bold, italic, etc.) and paragraph formatting (line spacing, indents, space before or after, etc.). All text in Word has some paragraph style.

A character style can apply to as little as a single character, and it contains only the character formatting. It can be applied to all or part of a paragraph; that text then has both the original paragraph style (which supplies the paragraph formatting) and the character style (whose character formatting overrides that of the paragraph style).

It sounds like you've been using the "save selection as a new Quick Style" command on the right-click menu, which by default creates only paragraph styles. When you enter the name for the style, click the Modify button at the bottom. In the dialog that opens, change the "Style type" box to say Character (unless you really mean to create a paragraph style).
