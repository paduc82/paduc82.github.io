There is no "checkmark that will make the file read-only". 

There are two ways to make a file "read only."

1) The "Locked" checkbox in the Finder's Get Info dialog. If this is enabled, the folder or file can no longer be modified. If you lock a file through that feature, Word will tell you that the file is read only.

doug robbins - mvp office apps & services (word) say:

To start with, anything that you type into the header or footer of a section, will be repeated on all of the pages in that Section, save for some caveats on that statement if you have the Different First Page option or Different Odd and Even options set.

If you have access to a copy of the document where the page numbering is still handled by the use of Page fields, try running a macro containing the following code:

Dim i As Long, j As Long
With ActiveDocument
    For i = 1 To .Sections.count
        With .Sections(i)
            For j = 1 To .Headers.count
                .Headers(j).PageNumbers.RestartNumberingAtSection = False
            Next j
            For j = 1 To .Footers.count
                .Footers(j).PageNumbers.RestartNumberingAtSection = False
            Next j
        End With
    Next i
End With

It should find any headers\footers that you may have missed.
