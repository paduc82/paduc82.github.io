a. user say:

Once legally activated, a computer's activation never expires. The only time you'll need to re-activate is if you change the computer's motherboard. That's because a computer with a new motherboard is considered a new computer.

For Windows 10, license and activation are separate things. You need a license to activate. The license has its own rules, but once the license has been used to activate the computer, the activation never expires.

When you buy a new Asus computer directly from Asus, the computer is activated at the factory, so there's nothing that you need to do in order to activate the computer. The type of license used to activate the computer is an 'OEM' license. That type of license can only be used to activate that computer and no other.

When you buy an Asus computer from somewhere else, there's no way of knowing what you're getting, so be sure you buy from a seller you trust.

a. user say:

Hi Edward,

We appreciate you for being a part of Windows 10.

When you upgrade your current version of Windows to a newer version, the upgrade process will create backups of important directories from your previous version of Windows. These directories include your user profiles, Windows folder, Program Files directory, and other important locations. These backups are then stored in two folders called Windows.old and $Windows.~BT, which can be used after the upgrade to recover any data that may not have been properly migrated over to the new version of Windows.

Please be aware the Windows. Old folder located at the root of the local disk where Windows 10 is installed and it will be deleted after 30 days automatically from the day you have upgraded to Windows 10. The Windows. Old folder is used to facilitate the rollback.

If you do not want to roll back to the previous Operating system from Windows 10, you may run the disk cleanup. You can use Disk Cleanup to reduce the number of unnecessary files on your drives, which can help your PC run faster. It can delete temporary files and system files, empty the Recycle Bin, and remove a variety of other items that you might no longer need. (It will also delete the previous Windows installation folders like Windows. Oldand $Windows.~BT)

  1. Press Windows + R keys on the keyboard to open run command prompt.

  2. Type cleanmgr in the run text box and hit Enter key on the keyboard.

  3. In the Drives list, tap or click the drive that you want to clean up, and then tap or click OK.

  4. In the Disk Cleanup dialog box, tap or click Clean up system files.  You might be asked for an admin password or to confirm your choice.

  5. In the Drives list, choose the drive you want to clean, and then tap or click OK.

  6. In the Disk Cleanup dialog box, select the checkboxes for the file types you want to delete, tap or click OK, and then tap or click Delete files. (You may check the option Previous Windows installations if it is present)
