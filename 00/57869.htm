thomas ferguson say:

Connect the HD from which you want the computer to boot to the power and to the controller. Boot the machine. As it boots, enter the BIOS setup in the usual way. Set the connected drive as thedrive from which the system boots. Note that the other drive is not connected. Restart.

If the system does not boot, insert the Windows Vista DVD into the drive. Make sure the DVD drive is set forst in the boot order. Restart the machine. Use the facilities on the DVD to repair startup. This process will result in changes to the boot loader. Restart the system.

It should start.

After you confirm proper operation, shut the system down and connect the other drive, Restart. You should now be able to delete whatever you want from the HD with the unwanted system.

Tom Ferguson
