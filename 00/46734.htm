cmdrkeene say:

You sure can.

If both computers are on the same network (like the same WiFi connection), then you can open Remote Desktop, type the IP address or PC name of the desktop computer, and connect directly.

If you take your Surface away from home and want to remote connect to your desktop while you're on another network (like a school or Starbucks WiFi), then you would need to do some configuring of your home router so that the ports are forwarded correctly.  Or you can use a program that does this automatically, such as TeamViewer, which has an RT app and is free for personal use.
