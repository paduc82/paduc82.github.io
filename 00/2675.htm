I agree with everything Jay told you. You are talking about the legacy styles drop-

down.

The above shows it from in the QAT along with the button to launch the styles pane and the font drop-downs.

One work-around would be to use shorter style names. You can use aliases for style names as well.

I suggest using functional names rather than descriptive names.

i.e.: "Body Text" vs "Garamond 11 pt. left"

Instead of: Centered Bold 14pt

use

h3a, Heading 3 a, Centered Bold 14pt, cb14

typing h3a or cb14 in the drop-down will apply the style

You can also attach keyboard shortcuts to your styles.

You want the first part of the name to be short and unique. That is what will appear first in the short area that MS has for display of style names.

Also, be aware of the Apply Styles dialog (Ctrl+Shift+S). That does have a slightly wider display of the style in use.

This gives you the wider display, but also opens a dialog box that grabs the focus from your document and has no keyboard shortcut to get rid of. It is a modal dialog, meaning that if you click back in your document, it remains open and the style shown will reflect the style in use at your insertion point in the document. When I use Word, I usually have the navigation pane displayed and move the Apply Styles dialog box to display in that area so it does not cover up my document.
