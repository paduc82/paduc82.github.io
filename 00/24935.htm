gilleslabelle say:

Hi,

A Microsoft account is a User account used to Login to Windows.

An Outlook account is a Microsoft Email account. (Hotmail)

Some people use the Outlook Email account to login to Windows, but are different things all together.

In Control Panel/Users, you can create Local accounts and Microsoft accounts for users.

Local accounts Login to Windows locally and do not need to sign in to Microsoft.

Microsoft accounts Login to Windows using a Microsoft user name and password - this can be an Email address and password.

The benefit of using a Microsoft account Login is that the User connects to the Microsoft server and has the account synced to their account.

The Favorites on the Internet browser are setup the same for example even if login is on a different computer using Windows 10.

Restoring Factory Settings on Nokia N80 - Microsoft Community

angeldust82 say:

Restoring Factory Settings on Nokia N80

If I restore the Factory setting on a Nokia N80 will that erase all of the pin codes and security codes that the previous owner put on there. I have tried to contact them and they will not reply to me.

psychomania---01 say: No unfortunately it won't. A hard reset has been known to reset codes on some models so it may be worth a try. It will delete any data that you have added to the phone since you got it.

Hard reset:

1. Switch off the phone.

2. Hold down the following three buttons: Green (the call answer button), * button, and '3' button.

3. While holding these buttons, hold the power button for a second to switch on the phone.

4. Release when the phone shows the nokia hands logo or shows other signs of life like the language selection screen.

It may take a few attempts and very flexible fingers!

If it doesn't work you will have to take the phone to a nokia care point to get the codes reset. You will have to take proof of purchase with you.
