john korchok say:

This is because you have done what most Word users do when they want to create a new page: Insert dozens of carriage returns. And your current formatting dilemma illustrates why you should never do this. You''ll be able to fix it fast if you turn on the display of non-printing characters:

  1. In Word, choose FILE, then Options.
  2. Select the Display panel.
  3. Check Show all formatting marks, then OK out.

Now the problem is obvious. Select all the carriage returns (the backwards P characters) and replace then with one Page Break (Insert>Page Break). Now the second page text stays where it should no matter how much text you type before it.

kourpa say:

Dynamic Sumif

Factory      Sales
A                100
A                200
B                300



=sumif(range, Criteria, [Sum_range])

Using following functions, 

=MATCH("Factory",1:1,0)        =1
=MATCH("Sales",1:1,0)           =2

=ADDRESS(1,D3)                   =$A$1
=ADDRESS(1,E3)                   =$B$1

=MID(G3,2,1)&":"&MID(G3,2,1) =A:A
=MID(H3,2,1)&":"&MID(H3,2,1) =B:B


From the above function how can I use the text "A:A" and "B:B" in my =sumif(A:A,"A",B:B).

hansv mvp say:

Let's say the last two formulas are in J3 and K3, respectively. You can then use:

 

=SUMIF(INDIRECT(J3), "A", INDIRECT(K3))

kourpa say: Many thanks for your solution.  Is there an easier function/combination of functions that will achieve the same results? 

hansv mvp say:

You could use

 

=SUMIF(INDEX(A:Z,0,MATCH("Factory",1:1,0)),"A",INDEX(A:Z,0,MATCH("Sales",1:1,0)))

 

You can, of course, replace "Factory", "A" and "Sales" with cell references.

kourpa say: Hi HansV,

Are there any online resources where I can learn more about these Excel functions?

hansv mvp say:
See for example http://www.randomwok.com/excel/how-to-use-index-match/

ronrosenfeld say: Another method is to use the OFFSET function:

=SUMIF(OFFSET(Sheet1!$A$1,1,MATCH("Factory",$1:$1,0)-1,
ROWS($A:$A)-1),A2,OFFSET(Sheet1!$A$1,1,
MATCH("Sales",$1:$1,0)-1,ROWS($A:$A)-1))

If there are no empty cells in the ranges, and no used cells below the data, my preferred method would be to use dynamic NAME'd ranges.

Factory  Refers To:
 =OFFSET(Sheet1!$A$1,1,MATCH("Factory",Sheet1!$1:$1,0)-1,COUNTA(OFFSET(Sheet1!$A$1,1,MATCH("Factory",Sheet1!$1:$1,0)-1,ROWS(Sheet1!$A:$A)-1))-1)

Sales Refers To:
 =OFFSET(Sheet1!$A$1,1,MATCH("Sales",Sheet1!$1:$1,0)-1,COUNTA(OFFSET(Sheet1!$A$1,1,MATCH("Sales",Sheet1!$1:$1,0)-1,ROWS(Sheet1!$A:$A)-1))-1)

And then just use the simpler formula:
=SUMIF(Factory,A2,Sales)
