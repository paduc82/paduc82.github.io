Data labels on XY Scatter chart - Microsoft Community

gnirmal say:

Data labels on XY Scatter chart

Hi, I used the VB code (from https://support.microsoft.com/en-us/kb/914813) to create data labels on my XY Scatter Chart and it worked fine, except that when there are two or more labels for a single data point, it overlaps each other and its difficult to read.

Can any one help how to separate the labels? I did try formatting labels with "comma separator". It didn't help.

chemcal say:

If this is what you wanted..

Provided you have labels in the third column (first x, second y values), the macro combines them into a single label for each point with the same coordinates.

 

 

Option Explicit

 

Sub AttachLabelsToReptPoints()

 

   'Dimension variables.

   Dim Counter As Integer, ChartName As String, xVals As String

   Dim xRng As Range, lbl() As Variant, x() As Double, y() As Double, CCounter As Integer

 

   ' Disable screen updating while the subroutine is run.

   Application.ScreenUpdating = False

 

   'Store the formula for the first series in "xVals".

   xVals = ActiveChart.SeriesCollection(1).Formula

 

   'Extract the range for the data from xVals.

   xVals = Mid(xVals, InStr(InStr(xVals, ","), xVals, _

      Mid(Left(xVals, InStr(xVals, "!") - 1), 9)))

   xVals = Left(xVals, InStr(InStr(xVals, "!"), xVals, ",") - 1)

   Do While Left(xVals, 1) = ","

      xVals = Mid(xVals, 2)

   Loop

  

    Set xRng = Range(xVals)

    For Counter = 1 To Range(xVals).Cells.Count

    ReDim Preserve x(1 To Counter), y(1 To Counter), lbl(1 To Counter)

       x(Counter) = Range(xVals)(Counter).Value

       y(Counter) = Range(xVals).Offset(0, 1)(Counter).Value

       lbl(Counter) = Range(xVals).Offset(0, 2)(Counter).Value

       For CCounter = 1 To Counter - 1

         If x(Counter) = x(CCounter) And y(Counter) = y(CCounter) Then

           lbl(CCounter) = lbl(CCounter) & ", " & lbl(Counter)

           lbl(Counter) = ""

         End If

       Next CCounter

    Next Counter

  

   'Attach a label to each data point in the chart.

   For Counter = 1 To Range(xVals).Cells.Count

    If lbl(Counter) <> "" Then

     ActiveChart.SeriesCollection(1).Points(Counter).HasDataLabel = _

         True

      ActiveChart.SeriesCollection(1).Points(Counter).DataLabel.Text = _

         lbl(Counter)

    End If

   Next Counter

 

End Sub

I'm personally a big fan of Threaded SMS. But I'm sure there are some out there who are not. For those uninspired few, I happened across a Registry Edit to disable threading on 6.1 devices. (The author claims it works on both Professional & Standard devices (PPC & SP).
