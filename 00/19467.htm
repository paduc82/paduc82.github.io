Calculate future date 90 days out based on Date Picker Content Control - Microsoft Community

kmorrissey13 say:

Calculate future date 90 days out based on Date Picker Content Control date

Hello-

I have inserted a Date Picker Content Control in a table cell (in a Word 2013 document) and named it "startDate". I would like to add another table cell containing a field named "dueDate" that is 90 days later than the start date chosen by the user. The due date would vary based on the selected start date. Any suggestions?

macropod say:

You could need to use a ContentControlOnExit macro for that, coded along the lines of:

Private Sub Document_ContentControlOnExit(ByVal ContentControl As ContentControl, Cancel As Boolean)
Dim EndDt As Date, StrFmt As String
If ContentControl.Title <> "startDate" Then Exit Sub
StrFmt = ContentControl.DateDisplayFormat
EndDt = DateAdd("d", 90, CDate(ContentControl.Range.Text))
With ActiveDocument.SelectContentControlsByTitle("dueDate")(1)
  .LockContents = False
  .Range.Text = Format(EndDt, StrFmt)
  .LockContents = True
End With
End Sub

I note you refer to the destination cell containing a field, presumably a formfield. Content Controls and formfields should not be used in the same document. The above code assumes the destination is a Content Control titled 'dueDate'.
