'

' Assumes:

' Column A = Cost

'

Sub MultiGoalSeek

'

    Dim i as Long

    Dim nRows as Long

'

    nRows = ActiveSheet.Range("A1").End(xlDown).Row - 1

'

    For i = 2 To nRows + 1

        Range("C" & i).GoalSeek Goal:=Range("D" & i), ChangingCell:=Range("B" & i)

    Next i

End Sub

marshall barton say:

You can create as many duplicate labels as needed by creating a utility table (named Numbers) with one field (named Num) and populate the table with records containg 1, 2, 3, ... up some number that is more than the number of duplicate labels you will ever need.

With that in place, your query would look something like:

SELECT yourtable.*

FROM yourtable, Numbers

WHERE Num <= Copies\125 +1

Your second question will need an additional field (named Batch) in the subscribers table to indicate how many months worth of items to ship in a single batch.  I.e. regular subscribers would have a 1 in this new field and for subcribers every other month put a 2 in this field.  I think(?) you can get close to what you want by modifying the above query to something like:

SELECT yourtable.*

FROM yourtable, Numbers

WHERE (Num <= (Batch * (Copies+124))\125) And (Month(Date()) Mod Batch = 0)
