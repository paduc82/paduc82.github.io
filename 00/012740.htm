Theme Colors in Visio are stored in individual drawing/file level. Unlike other Office applications Visio doesnâ€™t store the Theme Colors externally. To make a custom theme available to new drawings you may have to create the custom Theme Colors, save the drawing as template and then use this template to create your new drawings. So, the steps involved in creating a template and creating a new drawing would be;
1. Create a new drawing in Visio 2010.
2. Create and define custom Theme Colors.
3. Save the drawing as a template (*.vst) and close it.
4. Double click on the template (*.vst) from Windows Explorer.
5. A new drawing will be created inheriting all the settings defined in template including Theme Colors.

To answer your other question, the theme colors are applied to entire drawing. Shapes dropped on to the drawing that has theme applied will inherit the colors already defined in that particular theme. For example, if you change colors of Accent 2, Line and Text in a Custom Theme Colors and drop a generic Box shape, the new colors will automatically get applied to that shape (since a Box shape is already predefined to use Accent 2, Line and Text colors).
