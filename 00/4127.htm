I thought I could surf the web and find VBA snippets for manipulating tracked changes, then scan for tracked changes corresponding to the captured unhighlighting, then accept each of those.  From perusing the Word object model, the Revision object seems to be the thing that holds the details for the tracked changes, but none of the properties seem to contain the details that I would see in the margin bubbles i.e. details that would let me determine whether it is a tracked unhighlighting.  Can such a detail be accessed by VBA from the Word object model?

jay freedman say:
This appears to work:

Sub x()
    Dim rev As Revision
    ActiveDocument.TrackRevisions = False
    For Each rev In ActiveDocument.Revisions
        If LCase(rev.FormatDescription) = "formatted: not highlight" Then
            rev.Accept
        End If
    Next
End Sub

Of course, try it first on a copy of the original document.
