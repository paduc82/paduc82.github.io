gord dibben say:

You could try this UDF to concatenate the cells with the de-limiter of your choice

I have edited to suit your semi-colon request.

Function ConCatRange(CellBlock As Range) As String
'usage   =concatrange(A1:A7)
'for non-contiguous cells  =ccr((a1:a10,c4,c6,e1:e5))
'note double parens for non-contiguous cells
Dim Cell As Range
Dim sbuf As String
    For Each Cell In CellBlock
    On Error GoTo fred
    'comma de-limited alter to suit
    If Len(Cell.text) > 0 Then sbuf = sbuf & Cell.text & ";"
        Next
    ConCatRange = Left(sbuf, Len(sbuf) - 2)
fred:
End Function

neha singh msft say:

Hi Sami,

The view does not change if multiple users are viewing the workbook at the same time with read only permission. For example, a workbook is shared with User A and B with view (read) only permission. If user B is looking for information in sheet 5, then it will not affect the user A viewing sheet 2 in the same workbook as the same time. All the users will have their own view unless any user with edit permission, updates the workbook.

The moment user with edit permission edits the workbook, the newly added information will start reflecting in the workbook, but this will not change the sheet tab for the user with read only permission.

The settings are synced every time you change them (OneDrive sync).
