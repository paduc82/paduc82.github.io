hansv mvp say:

If you have a field that uniquely identifies the records in the query, or a combination of fields that together uniquely identify the records, it is possible to create a calculated column that returns a sequence number.

But this is not very efficient; with large numbers of records the calculations will slow down the query noticeably.

 

Reports have built-in support for numbering records sequentially with very little overhead.

 

Also see http://answers.microsoft.com/en-us/office/forum/office_2010-access/query-to-count-cases-of-sets-in-a-running-manner/bbc7e126-e523-4a5b-be79-3504e8290cc9

bob jones aka: cybertaz say:

The Display for Review & the Markup Options controls in the Tracking group determine display choices only, they do not remove markup from the document.

Choosing No Markup does not create a "clean copy" for anyone other than the user currently viewing the document with that choice selected. IOW, it temporarily hides the existing markup. Furthermore, neither of those options travel with the document. They are viewing preferences specific to the local installation of the software being used.

When opening any document that contains Comments & Tracked Changes existing markup will display regardless of the settings which were in place when the document last was closed using a different installation of the program or the same one. More explicitly, the Markup Options remain in effect for that installation of Word until the user changes them, but have no influence when a document is opened in a different instance of Word; the Display for Review settings are not at all persistent even on the same system for the same document.

The only way to create a clean copy is to follow this 3 step process:

  1. Turn Track Changes OFF
  2. Use the Accept/Reject tools in the Changes group to finalize the document & the Delete tool in the Comments group to remove any Comments
  3. Save the file
