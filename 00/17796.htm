rick rothstein, mvp - excel say:

I think you will need to do this with a macro. To install the following macro, press ALT+F11 to go into the VB editor, click Insert/Module on the VB editor's menu bar and then copy/paste the following code into the code window that appears...

Sub SplitDigits()
  Dim X As Long, Cell As Range, CellText As String, Digits() As String
  For Each Cell In Selection
    CellText = Cell.Value
    If Len(CellText) Then
      For X = 1 To Len(CellText)
        If Mid(CellText, X, 1) Like "[!0-9]" Then Mid(CellText, X, 1) = " "
      Next
      Digits = Split(WorksheetFunction.Trim(CellText))
      Cell.Offset(0, 1).Resize(1, UBound(Digits)) = Digits
    End If
 Next
End Sub

To use the macro, just select the cells you want to split out first, then press ALT+F8, choose SplitDigits from the list that appears and click the Run button. Note, as currently structured, the split out digits will be placed starting with then next column over. If, instead, you would want to split the list out starting with the cell containing your text (this will overwrite the cells entry and you will lose the original strings text), then change this line....

Cell.Offset(0, 1).Resize(1, UBound(Digits)) = Digits

to this...

Cell.Resize(1, UBound(Digits)) = Digits

mike h.. say:

Here's a way with a formula and text to columns.

Enter this ARRAY formula in B1. See below on how to enter it

=MID(SUMPRODUCT(--MID("01"&A1,SMALL((ROW($A$1:$A$25)-1)*ISNUMBER(-MID("01"&A1,ROW($A$1:$A$25),1)),ROW($A$1:$A$25))+1,1),10^(25-ROW($A$1:$A$25))),2,25)

Drag down as far as required to extract the numbers. Copy this new range and paste special|Paste values on top of the formula cells.

Then text to columns, fixed width and manually insert beraks in the string.

If there could be more than 25 numbers in the cell then change all instances of 25 in the formula to a larger number.

This is an array formula which must be entered by pressing CTRL+Shift+Enter
and not just Enter. If you do it correctly then Excel will put curly brackets
around the formula {}. You can't type these yourself. If you edit the formula
you must enter it again with CTRL+Shift+Enter.

If the department dropdown in cell A1 contains numbers like

then define the range names for the dependent drop-down with names like

d_1

Then, in the dependent dropdown validation formula enter
