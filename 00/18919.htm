bob jones aka: cybertaz say:
There is nothing in Word that translates text from one language into another.

Any portion of the text can be identified as nearly any Language you prefer by selecting the text then going to Tools> Language to choose the one you want, but that doesn't alter the vocabulary, itself... IOW, if it was typed in English it will continue to be in English, but you'll be telling the program it's something else strictly for Spelling & Grammar purposes.

You can us the Translation feature in the Toolbox> Reference Tools to translate the entire document or use dome other translation service, then copy/paste from the translated version to your original document.

If you're going to do that I'd suggest using a copy of the document in which you've first converted the Bibliography to static text. Cut/paste it to a new blank document & translate just the bibliography. Copy the result back into the document from which you cut it.

I'd also suggest that you carefully proofread the translated result in order to avert any rude surprises :-) Online translators are quite literal & not 100% reliable.
