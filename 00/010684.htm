tushar mehta say:
Is it possible to plot a series with the secondary horizontal axis and primary vertical axis? Thank you Jlalib6


It's quite easy.  I checked with a column and a line chart.

You have to have at least 2 series.

Right-click the series you want on the secondary axis and select Format Chart Series...

In the Format Data Series dialog box, select the Series Options tab and select Secondary Axis.

From the Chart Tools contextual tab select Layout | Axes group | Axes dropdown list | Secondary Horizontal Axis >  and choose the desired axis type

From the Chart Tools contextual tab select Layout | Axes group | Axes dropdown list | Secondary Vertical Axis >  and choose None
