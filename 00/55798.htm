In addition to the steps suggested already, I'd recommend you run the DirectX Troubleshooter which you can access as follows:
  1. Go to Start, then Run (hit Windows key + R if you can't see "Run" on the Start menu) and copy/paste the following and then click OK: dxdiag
  2. After the animation finishes loading, click the "Sound" tab and then click the "Test Direct Sound" button and respond to the prompts accordingly.
  3. If you still don't hear anything, click the "Music" tab and try testing each port. By "port" I mean the options available to you which you should be able to see in the list.
  4. If the above results remain negative, click the "More Help" tab and then click the "Sound" button. This will launch the Windows Sounds Troubleshooter where you'll get a number of options to choose from.

In the event that it doesn't, you can try using System Restore to wind back the clock to a time when you know the PC was working properly, To run System Restore, do the following:

  1. Click Start | Programs | Accessories | System Tools | System Restore.
  2. Click the option: "Restore my computer to an earlier time" and then click Next. You'll get a calendar with a number of Restore Points which appear in bold type. Choose one where you know audio was working properly.

System Restore takes a few minutes to run during which time the PC will reboot. Don't interrupt this cycle. When the PC reboots, it will load the System Restore menu where you'll see a message as to whether the restore was successful or not. Click OK to boot to the desktop and then try and play something to verify whether audio is now working.

Be aware that when you run System Restore, any programs you installed after the date you choose to restore to will have to be reinstalled again. The same goes for antivirus updates and security patches.

jack - support engineer say:

Hello Kissmyne,

Thank you for posting.  I understand how the Aero score being lower than your Gaming score could be confusing to you at first glance.  I think if you take into account how the scores are arrived at it will make more sense to you.  The Aero package is way more graphics intensive then running games when you are utilizing ALL of the Aero advanced features such as dual displays performing a task like video editing:

A computer with a base score of 3.0 is able to run Windows Aero and many new features of Windows Vista at a basic level. Some of the new Windows Vista advanced features might not have all of their functionality available. For example, a machine with a base score of 3.0 can display the Windows Vista theme at a resolution of 1280 × 1024, but might struggle to run the theme on multiple monitors. Or, it can play digital TV content but might struggle to play High Definition Television (HDTV) content.

http://www.microsoft.com/windows/windows-vista/get/experience-index.aspx

When you think about it in these terms it becomes easier to understand how your graphics card can have a higher Gaming Index Score than Aero Score because running the game simply isn’t as taxing on graphics card.  I would recommend that you read the entire article I provided the link for.  After you read the article the Vista Performance Index should be easier for you to interpret.  I hope this information is helpful.
