rich007a say:
Is there any way to tweak the macro to skip over these broken links


Yes, you can do something like this:

'=======================================
Sub OpenHyperlinksInSelection()
' Opens all hyperlinks in the current selection.
'
    Dim hl As Hyperlink
    For Each hl In Selection.Hyperlinks
        On Error Resume Next
        hl.Follow NewWindow:=False, AddHistory:=True
        If Err.Number <> 0 Then
            On Error GoTo 0
            With hl.Range.Interior
                .ColorIndex = 3
                .Pattern = xlSolid
                .PatternColorIndex = xlAutomatic
            End With
            Err.Clear
        End If
        On Error GoTo 0
    Next hl
End Sub
'=======================================

Cells with dead links are filled red.

ken sheridan say:

The following will return the rows from the table with the latest date of entry per ID_Article:

SELECT T1.*
FROM Traffic AS T1
WHERE DateOfEntry =
    (SELECT MAX(DateOfEntry)
     FROM Traffic AS T2
     WHERE T2.ID_Article = T1.ID_Article);

You can join the query to another instance of Traffic on ID_Article and return the Price column from the above query as the latest price if you need to do so in multiple rows of the final result table.

PS: This should be much more efficient than calling the nested domain functions in an expression.

Step 1: Restart your computer.

Step 2: Make sure the Microsoft Security Essentials service is set to automatic and started

  1. In Windows XP, click Start, click Run, type services.msc, and then press ENTER.
    or
    In Windows Vista and Windows 7, click     Start, click in the Start Search box, type services.msc, and then press ENTER.
