An IMAP mail account in MS Outlook is a copy of data/folders on the IMAP server. Recreating that account on any device will show the data as held on the server.

Whereas a pop mail accounts data file, the *.pst file, is unique to the PC. By default a pop mail account downloads all mail from the server. You can create sub-folders within the Personal Folders set, but these are unique to that PC.

To use the old pst data files, when setting up MS Outlook, or creating a new Profile with a pop mail account it is necessary in the data files tab to add the previous *.pst and set this as the default msg store

john w. vinson say:

Is this a Number (Double or Decimal) or a text string? You can use either but the syntax is a bit different. If it's Number try

DateSerial([numdate] \ 10000, [numdate] \ 100 MOD 100, [numdate] MOD 100) + TimeSerial(100 * ([numdate]-fix([numdate]) MOD 100, 10000 * ([numdate]-fix([numdate]) MOD 100)

For text, try

cdate(Left([textdate], 4) & "-" & Mid([textdate],5,2) & "-" & Mid([textdate],7,2) & " " & Mid([textdate],10,2) & ":" & Right([textdate], 2))

There are dozens of ways to do this using the CDate, DateSerial, TimeSerial and substringing functions, but I did test these and they both work.
