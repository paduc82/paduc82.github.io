"So i concluded that the problem was Directx12 "

Wrong conclusion, unless the game in question is one of the handful of new games that actually uses DX12.

But that won't be the case if you were playing these game before Win10.

Win10 has DX9, 10, 11 and 12, and games uses specific DX files. The majority of games still use DX9, many with

provision for DX10 or 11 effects, but there are an increasing number that are limited to DX10 or 11.

Visual C++ is not a  DirectX component. The problem with uninstalling all the C++ versions is that

there are many sub-versions, and some games use specific sub-versions, so when you reinstall what MS has to

offer on their download page you may not get the 'altered' version that the game developer used for their game.

- Installing a game that is having an issue may fix any issue with C++, as the game will reinstall the sub-version

it uses (eg - My Windows has eight different versions of C++ 2005 alone - including 32 and 64bit versions ) , and believe it or not, some current games still use a C++ Library from 2005 .

I was surprised to see this myself when installing a recent game from disk a couple of weeks ago

(don't recall which, but it may have been either Alien Isolation or The Evil Within).

- Your error can also be generated by an issue with .NET Framework, or the game is trying to run the wrong

version of a file, eg-  if you have only reinstalled 32 or 64 bit versions of C++ you may get this error.


"Im trying to install the dxwebsetup but it keeps on saying theres a newer or equivalent version of directx installed

and im wondering if theres a way around there. "

This is because your Windows has all of the DX files. There is no need for a way around it. The Web Installer

only installs any 'missing' DX files.

However, while it's uncommon,  DirectX file or files can at times get corrupted when a game reinstalls DX.

In this instance you need to reinstall DX using the DirectX Redist (June 2010) installer package.

This will overwrite all DX files (except for the couple of DX12 files in Win10).

Download Details - Microsoft Download Center - DirectX Redist (June 2010)

( You first unpack the files to a location of your choice. The open the folder and run the DXSETUP.exe

to reinstall DX. You can delete this folder after reinstalling. )

You can easily change display language in windows 8. Changing the language in windows 8 is pretty easy.

 

Language packs are available for Windows 8 and for Windows RT.

Change Language of Windows 8:

a.      Click Control Panel icon from windows explore menu bar.

b.      In the Change your language preferences pane, click or tap Add a language.

c.       Search for the language that you want to install. For example, select Catalan, and then click or tap Add. Catalanis now added as one of your languages.

d.      In the Change your language preferences pane, click or tap Options next to the language that you added.

e.      If a language pack is available for your language, click or tap Download and install language pack.

f.        When the language pack is installed, the language is displayed as available to use for the Windows display language.

g.      To make this language your display language, move it to the top of your language list.

h.      Log off & then log back on to Windows for the change to take effect.
