Data in the named range Company.

On some sheet...

A1:A3 = 1,2,3

Enter this formula in B1:

=INDEX(Company,MODE(MATCH(Company,Company,0)+{0,0}))

=INDEX(Company,MODE(IF(COUNTIF(B$1:B1,Company)=0,MATCH(Company,Company,0)+{0,0})))

=COUNTIF(Company,B1)

How can I add (or remove) a watermark conditionally? - Microsoft Community

carole kushner say:

How can I add (or remove) a watermark conditionally?

We would like to be able to add a watermark conditionally to a Word mail merge.   How can this be accomplished in Word 2013?

macropod say:
That depends on the condition and is done via an IF field.

carole kushner say:

The condition would be a value in a mail merge field.  Can you provide me with a little more detail about how it can be done via an IF field.  Thanks.

Best,

Carole

macropod say:

Basically, you'd use an IF field in the header, coded along the lines of:

{IF{MERGEFIELD ConditionField}= "TrueCondition" "WaterMark Text & ImageAnchor go here"}

If the Watermark is just text or an in-line image, you can simply insert it where indicated, between the double quotes. If it's a floating image, you'll need to change the horizontal layout alignment to 'left of > character' and move the anchor so it's between the double-quotes Once you've done that, lock the anchor in place. Do note that you may need to need to re-position the image after this, so that it displays in the correct location when the merge executes; it's position when the field code is displayed isn't the same.

Note: The field brace pairs (i.e. '{ }') for the above example are created in the document itself, via Ctrl-F9 (Cmd-F9 on a Mac); you can't simply type them or copy & paste them from this message. Nor is it practicable to add them via any of the standard Word dialogues.

jason (office engineer) say:

Graeme,

You are correct that Heading cards cannot be grouped with other content.  Content in Heading 1 cards will always appear on its own slide.  Content in Heading 2 cards, however, will appear across the top of all slides following that Heading 2 card (until a new Heading 1 or Heading 2 card occurs).
