if the test like:

 

If DCount("*", "TableNameHere", "[FieldNameHere]=" & Me.TextBoxNameHere) > 0 Then

ken sheridan say: One thing I'd add.  Make sure that Bob's message box is not informing you that it's duplicating the current record.  Data in another field could be edited in an existing record, in which case the DCount function would return a value of 1, so, where MyID is the (numeric) primary key:

Dim strCriteria As String

strCriteria = "[FieldNameHere]=#" & _
    Format(Me.[TextBoxNameHere],"yyyy-mm-dd hh:nn:ss") & _
    "# And [MyID] <> " & Me.[MyID]

If DCount("*", "TableNameHere", strCriteria) > 0 Then
   Cancel = True
   MsgBox "Your Custom Message Here", vbInformation
End If

Note the # date/time delimiters and the formatting of the value to the ISO standard for date/time notation to ensure international unambiguity.
