There are 4 editions of Windows 8:

 

Windows RT - comes preloaded on new devices that have an ARM based processor. Cannot run older apps designed for Windows 7 and older versions of Windows.

 

Windows 8 - introductory edition of Windows 8 that comes preloaded on most new computers. Its meets the needs of most home users. Does not support Media Center.

 

Windows 8 Pro - includes all the features of Windows 8 in addition business features such as: Domain Join, Remote Desktop, BitLocker drive encryption, File and Folder Encryption, Network Backup, Group Policy, support for up to 512 GBs of RAM, supports Media Center (must be purchased separately)

 

Windows 8 Enterprise - all the featues of Windows 8 Pro and additional features such as App Locker, VDI Support, UNIX Subsystem, Branch Cache support, DirectAccess (similar to VPN), Federated Search. So, the best edition would be Windows 8 Pro.
