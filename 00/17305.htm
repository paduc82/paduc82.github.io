jay freedman say:

I'm not seeing what you described. The Reset Form Fields button, either in the ribbon or added to the Quick Access Toolbar, is disabled when the form is protected; but it does clear all the form
fields when the form is unprotected. I don't know of anything that would make it behave as you described.

If you can't get it to work, though -- or even if you can get it working but only in an unprotected form -- you may want to store the following macro in your Normal.dotm template (see
http://www.gmayor.com/installing_macro.htm if needed) and assign a Quick Access Toolbar button to the macro. This will work even when the form is protected.

Sub ClearFormFields()
    Dim FF As FormField

    For Each FF In ActiveDocument.FormFields
        Select Case FF.Type
            Case wdFieldFormTextInput
                FF.Result = ""
            Case wdFieldFormCheckBox
                FF.CheckBox.Value = False
            Case wdFieldFormDropDown
                FF.DropDown.Value = 1
            Case Else
                ' do nothing
        End Select
    Next
End Sub
