macropod say:

If you bookmark the table, you can use formula fields to point to individual cell values. For example, suppose you apply the bookmark name 'Tbl' to the table and somewhere else in the document you want to refer to the value in C3. For that you'd use a formula field coded as:

{=SUM(Tbl C3)}

with whatever numeric picture switches you require to format the output.

Note: the field braces (i.e. {}) cannot be typed; they can be created via Ctrl-F9 (Cmd-F9 on a Mac) or you can skip that part and just input =SUM(Tbl C3) into the Insert|Quick Parts|Field dialogue for the Equation field (uncheck the 'preserve formatting' option, too).

macropod say:

The technique I described only works with Word tables, not with embedded/linked Excel worksheets. The 'name' is whatever you bookmark the table with. The cell reference points to a single cell. The formula can be made to point to a range of cells (e.g. C3:C5, or C:C), in which case it will sum all cells in that range. Adding new rows won't change which cells a specific cell reference (e.g. C3) points to.

If the Excel worksheet is inserted via a link from an external workbook, such a link can point to a range defined by cell addresses or a name, and the data can be output in a variety of formats - which you'll see if you copy from Excel then, in Word, use Paste>Paste Special with the 'paste link' option. Thus, if you want to display the contents of a specific Excel cell somewhere in your document, you can link to just that cell with the appropriate link format.

ron sommer mvp 2009-2017 say:

The mail feature in an Apple computer can only use pop3 protocol to access a Hotmail (Outlook) account.  Using pop3 protocol, the online Inbox messages are downloaded to the computer.  After the emails are downloaded, a message is sent to delete the messages from the server.  Instead of deleting the messages, the Outlook server moves the messages to the POP folder.

The mail feature should have an account setting to Leave a copy of messages on the server after downloading.  Checkmark that setting so the emails will stay in the online Inbox.

 

When pop3 protocol was developed back in the 60s (?), the server storage space was limited.  If the allotted space became full, new emails would be rejected. 

There is no syncing of messages and folders between your computer and the server.

 

When I go directly into my Hotmail account (via Outlook), I only see a small number of messages in my Inbox.  There does not appear to be a way to modify the viewable messages.

This is a different issue.  The Mac browser does not work well with outlook.com.

suzanne s. barnhill say:

Usually the setting will be in your printer Properties. Not all printers can print borderless, and those that do may require a special paper carrier or limit borderless printing to sizes smaller than Letter (e.g., 8x10) since this setting is primarily designed for photo printing.

If you want to print text clear to the edge, you can set the margins to 0 as suggested by Mary and Stefan, but more commonly it is a background that needs to be printed "full bleed." In that case, you would leave the text margins alone and insert the background graphic Behind Text, sized to fill the page, with position set to Top and Left relative to Page.
