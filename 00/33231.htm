    I want to disable my Surface Pro 3 (Windows 10) touch screen - Microsoft Community

    vincentwaterhouse say:

    I want to disable my Surface Pro 3 (Windows 10) touch screen so it works only as a traditional laptop with keyboard and mouse. The touch feature does not work properly and I don't need it

    I want to disable my touch screen feature and use as traditional laptop with a mouse and keyboard only.  No touch features.
    guia ram say:

    Hi Vincent,

    To disable the touch feature of your Surface Pro 3, we suggest following the steps listed below:

    1. Type device manager on Search/Cortana, then select Device Manager from the results.
    2. On Device Manager, look for Human Interface Devices and expand it.
    3. Under Human Interface Devices, right-click on HID-compliant touch screen and select Disable from the list.
      There may be more than one HID-compliant touch screen, you'll need to disable them as well.
    4. A pop-up stating Disabling this device will cause it to stop functioning. Do you really want to disable it? will appear, click Yes.
    5. The device's touchscreen functionality will be disabled immediately.

    If at any point you'll want to enable the touch feature again, you can go back to the Device Manager, follow the same steps and choose Enable.

    Let us know if you have any further concerns.
