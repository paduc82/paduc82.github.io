SUBTOTAL Function to Calculate Median - Microsoft Community

noybu say:

SUBTOTAL Function to Calculate Median

The Excel SUBTOTAL function can compute an average for a subgroup of numbers as well as various other statistics.  I use this function a lot in tables that I filter by different criteria.  Is it possible to use the SUBTOTAL function to compute a median?  I did not see this on the SUBTOTAL help page.  Is there a workaround that would work?  Thank you!
bernie deitrick say:

Use an array formula, entered with Ctrl-Shift-Enter, like this for data starting in A2:

=MEDIAN(IF(SUBTOTAL(2,OFFSET(A2,ROW(A2:A100)-ROW(A2),0)),A2:A100))

You can use a larger number than 100 - just don't use A:A instead  - that would be really slow.

Sub resetMe()

Dim osld As Slide

DoEvents

  Application.CommandBars.ExecuteMso ("SlideReset")
