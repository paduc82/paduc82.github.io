I spent some time searching for a 2016 solution today, and if "Deleted Items" is one of the folders you need to search, Microsoft created a separate checkbox just for "Deleted Items".

  • Under "File / Outlook Options / Search": Check Include messages from the Deleted items folder in each data file when searching in All Items, if that checkbox appears
  • Update Windows Indexing to index "contents" for .pst files
  • Rebuild the index
  • Relocate the index (which also rebuilds it, naturally)
