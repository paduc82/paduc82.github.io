jisgwani say:

Desktop Background settings disabled.

The options in the panel for selecting backgrounds for my desktop are all in grey text and clicking on any of them has no effect. Where is the registry setting for my desktop background and what is the value for enabling it? Other users backgrounds settings work fine, but mine as the primary administrator does not. Using XP, SP-3. 

a. user say:

If the following policy value in the registry for a current user is set to one (1=enabled), you will not be able to change the background image (wallpaper).

HKCU\Software\Microsoft\Windows\CurrentVersion\Policies\ActiveDesktop\NoChangingWallPaper

If you find it, you can change it to zero (0=disabled) or delete it entirely.

That might fix the symptom of your problem - which is not being able to change the background image, but that will not fix the real problem.

If you find the value enabled and you did not change it yourself manually or via the Group Policy Editor, and especially if you have XP Home, you have another problem.
