The new Microsoft Skype for Windows 10 (Skype UWP) application, which you have installed now, is using a different folder location for saving local settings, incl. chat history:

%localappdata%\Packages\Microsoft.SkypeApp_kzf8qxf38zg5c\LocalState\

However, pasting the files from your old Skype installation to this folder will not work either, as the chats are now saved in a different database file, the skype.db file with a different structure than the standard main.db file.
