steve averbach say:

After almost four hours of trying to cobble together my own solution (I obviously have failed), I'm turning to the community.

Scenario: a worksheet used to track cost-saving actions includes plan and actuals. The value in Column G is either "Plan" or "Actual". The first "Plan" row is Row 11 (and then every other row is a plan row). I'd like to enhance the worksheet where:

- the default view does not show the plan rows

- a button above Column G is named "Show Plan"

- when the user clicks the "Show Plan" button, the plan rows (11, 13, 15,...) appear AND the button name changes to "Hide Plan."

- -when the user clicks the "Hide Plan" button, the plan rows (11, 13, 15,...) are hidden AND the button name changes to "Show Plan."

Thanks for your guidance

mike h.. say:

Hi,

Go to the 'developer' tab and click on 'Insert' and then click the command button icon in the ActiveX toolbox. On the worksheet hold down the left mouse button and drag to draw a button on the sheet in the appropriate place. Right click the button and view code and paste the code below into the editor. Back on the worksheet click the button and the rows will either hide or unhide depending upon the current state and the button caption will change to the correct one.

Private Sub CommandButton1_Click()
Dim x As Long, LastRow As Long
LastRow = Cells.Find(What:="*", After:=[A1], _
SearchOrder:=xlByRows, _
SearchDirection:=xlPrevious).Row
For x = 11 To LastRow Step 2
Rows(x).Hidden = Not Rows(x).Hidden
Next
If Rows(11).Hidden Then
    CommandButton1.Caption = "Show Plan"
Else
    CommandButton1.Caption = "Hide Plan"
End If
End Sub

Hi Ken,

This is the only effective way to do it in any version of Word.

What happens when I attach a new template to my document? or How do I copy content and settings from a template to a document? by Shauna Kelly MVP The key is what templates do and how they act in Word. Attaching your template in the Windows version or the Mac version will not give you any of the following from the template:

  1. Headers or Footers
  2. a table or other text anywhere in the file
  3. Page numbers
  4. Margins
  5. Page Orientation
  6. Columns

None of those come to the existing document by attaching the template. I ask you to look at my web page on templates, written for the Windows version, which details what you get by attaching a template to an existing document. 

Templates in Microsoft Word 

http://www.addbalance.com/usersguide/templates.htm#gives_you_access

You can import by attaching:

  1. Macros
  2. AutoText (and in Windows versions, building blocks)
  3. Keyboard shortcuts
  4. Toolbar or QAT modifications
  5. Styles

This has been true since Word 3 on the Macintosh and Word 97 on Windows.
It likely was true before that, but I am speaking only to my own experience.

I believe there are programs where you can attach what that program calls a template and get the results you want. Those programs are not Word. In PowerPoint or Publisher, I believe there are ways to change the layout by changing the master slide or template.
