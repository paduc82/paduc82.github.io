How do I enable cookies to access Amazon?

Internet Explorer:

Internet Options> Privacy. First party & third party cookies, check Accept.

Check Always accept session cookies. It might help to click on Sites and add amazon.com as Accept.

Edge:

3 dots>Settings> Advanced settings> Cookies> Don't block cookies.

If these settings are correct (both IE & Edge) you may have a corrupt cookie and clearing all cookies to get a fresh start will resolve the problem.

Also, verify that the computers Date, Time and Time zone are correct. Incorrect settings can affect cookie function.

    Changing link source - Microsoft Community

    ghc1943 say:

    Changing link source

    I have a PowerPoint file (Generic 1.pptx) with multiple links to multiple Worksheets in an Excel Workbook (Generic 1.xlsx). I would like to save Generic 1.pptx as Generic 2.pptx and Generic 1.xlsx as Generic 2.xlsx and have Generic 2.pptx linked to Generic 2.xlsx. Can I change the links globally instead of changing one at a time?

    OS is Windows 7 Ultimate 64 bit

    Office 2007

    Files are stored in a TruCrypt local disc (file).

    Thank you,

    GHC

    shyam pillai say:

    Hi,

    You could open up the contents of the PPTX file using a zip editor and change the text references manaually and save it back or you can try running this macro on generic2.pptx. Change the file paths to reflect their locations on disk.

    Sub UpdateLinks()
    Dim oSld As Slide
    Dim oSh As Shape

    Const OLD_FILE = "D:\Workarea\Documents\Generic 1.xlsx"
    Const NEW_FILE = "D:\Workarea\Documents\Generic 2.xlsx"

    For Each oSld In ActivePresentation.Slides
        For Each oSh In oSld.Shapes
            If oSh.Type = msoLinkedOLEObject Then
                On Error Resume Next
                oSh.LinkFormat.SourceFullName = Replace(oSh.LinkFormat.SourceFullName, OLD_FILE, NEW_FILE)
                oSh.LinkFormat.Update
            End If
        Next
    Next
    End Sub

    Regards,

    Shyam Pillai
