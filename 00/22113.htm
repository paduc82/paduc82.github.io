srinivasu guntimadugu say:

Hi,

 

Welcome to Microsoft Office Community and thanks for posting the question. It looks like you need assistance in sharing the PST file between 2 computers.

 

Usually we do not recommend the default Outlook data file b/w 2 computers periodically as there are more chances of the Outlook data file getting corrupted when transferring through a thumb drive.

 

If you have the same e-mail account setup on both the computers, then you can import the recent ad updated Outlook data file from one computer to other. By default the new e-mail will be downloaded on the both computer here after.

 

Outlook 2003 and Outlook 2010 share the same data file format which is UNICODE. Refer the following link to check if the PST file is in older format 97-2003 which is ANSI or UNICODE format:
http://office.microsoft.com/en-in/outlook-help/determine-the-personal-folders-file-pst-format-HP001043556.aspx

 

If the data file is in ANSI format, then refer to the following link to convert to UNICODE format:
http://office.microsoft.com/en-in/outlook-help/convert-a-non-unicode-data-file-pst-to-a-unicode-data-file-pst-HP001038351.aspx

 

Here are some of the links on how to copy or move the Outlook data file between computers:

http://support.microsoft.com/kb/291627

 

http://office.microsoft.com/en-in/outlook-help/move-an-outlook-data-file-from-one-computer-to-another-HA001230892.aspx

 

http://office.microsoft.com/en-in/outlook-help/open-change-or-close-outlook-data-files-HA001232805.aspx

 

I hope the information provided helps.
