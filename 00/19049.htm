To use AutoText stored in a template other than the normal template, the holder template must either be:

  • The attached document template (normally the one used to start the document)
  • A loaded Global Template (normally in Word's Startup Folder)

The outer braces have to be inserted using either the Insert> Field dialog or by using the Command+F9 keystroke, they cannot simply be typed.

Also, MACROBUTTON fields don't "disappear when clicked". A single click selects the entire field which is then replaced by whatever is typed by the user. If you'd prefer one that does disappear when clicked, use a PLACEHOLDER Field instead. A completed PLACEHOLDER Field for the same purpose as above would be:

{ PLACEHOLDER "[Recipient]" \* MERGEFORMAT }

  • Assign the text NOT FOUND to column B and Column D of the Report Sheet.

hansv mvp say:

Try this macro:

Sub MatchData()
    Dim wshM As Worksheet
    Dim wshR As Worksheet
    Dim lngRow As Long
    Dim lngLastRow As Long
    Dim rngMatch As Range
    Set wshM = Worksheets("Master")
    Set wshR = Worksheets("Report")
    lngLastRow = wshR.Range("A" & wshR.Rows.Count).End(xlUp).Row
    For lngRow = 2 To lngLastRow
        Set rngMatch = wshM.Range("A:A").Find( _
            What:=wshR.Range("A" & lngRow).Value, _
            LookAt:=xlWhole)
        If rngMatch Is Nothing Then
            wshR.Range("B" & lngRow).Value = "NOT FOUND"
            wshR.Range("D" & lngRow).Value = "NOT FOUND"
        Else
            wshR.Range("D" & lngRow).Value = rngMatch.Offset(0, 1).Value
            wshR.Range("E" & lngRow).Value = rngMatch.Offset(0, 3).Value
        End If
    Next lngRow
End Sub
