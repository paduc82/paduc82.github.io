As you posted your question a while back you've probably already resolved the problem, but for others who stumble across this:

I had exactly the same problem.  Open up Word, click File, choose Options, then choose Add-ins.  Look to the bottom of the Add-ins page and select where it says Manage:  COM Add-ins and click go.  Deselect all the COM Add-ins and click OK.  Close Word and restart-it should start up fine.  Check that Word will now open previous documents.  If everything is OK, go back into the COM Add-ins and reselect one at a time and restart Word again.  Once the offending Add-in has been reselected Word will not open correctly so you'll now know which Add-in was causing the problem and just leave it unselected.  My problem was the Send to Bluetooth Add-in, which I believe does cause a few problems.

tom ogilvy say:

Here is some basic code.  It keys off of the selection when you click the button.

If you have one cell selected, it will prompt for the number of rows and copy and paste

If you have multiple adjacent cells selected in a single row, it will copy the selected cells, prompt for the number of rows and paste underneath the selected cells for that many rows.

If you have a rectangular area select, it will copy the first row of that selection and paste into the selected cells below the first selected rows so you don't have to specify the number of rows (you did by selecting them).

Sub btn_Click()
Dim r As Range, ans As Long
Set r = ActiveCell
If Len(Trim(r.Formula)) = 0 Then
  MsgBox "activecell is empty, nothing to do"
End If
If Selection.Rows.Count = 1 Then
  ans = Application.InputBox("enter number of rows to copy to", "Number Rows?", , , , , , 1)
  Selection.Copy Selection.Offset(1, 0).Resize(ans)
Else
  Selection.Resize(1).Copy Selection.Offset(1, 0).Resize(Selection.Rows.Count - 1)
End If
 
End Sub

If the activecell is empty when you click the macro, it tells you it is empty and does nothing.
