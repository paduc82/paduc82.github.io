to check if password sync has been enabled in aad sync service, please try the following steps:
1. open windows powershell as an administrator on aad sync server.
2. type in "import-module adsync" command to load the aad sync cmdlets into your current session.
3. run get-adsyncaadpasswordsyncconfiguration -sourceconnector "name of the ad connector".


generally, if you made some changes to ous, you need to run full aad sync. for running full aad sync service, please run c:\program files\microsoft azure ad sync\bin\directorysyncclientcmd.exe initial on aadsync server.

to use this script, you need to replace the names of the connectors with the values from your environment.

$adconnector  = "o15sso110.tk"

$aadconnector = "o365bpt.onmicrosoft.com - aad"

import-module adsync

$c = get-adsyncconnector -name $adconnector

$p = new-object microsoft.identitymanagement.powershell.objectmodel.configurationparameter "microsoft.synchronize.forcefullpasswordsync",  
string, connectorglobal, $null, $null, $null

$p.value = 1

$c.globalparameters.remove($p.name)

$c.globalparameters.add($p)

$c = add-adsyncconnector -connector $c

set-adsyncaadpasswordsyncconfiguration -sourceconnector $adconnector -targetconnector $aadconnector -enable $false

set-adsyncaadpasswordsyncconfiguration -sourceconnector $adconnector -targetconnector $aadconnector -enable $true
