1. Applications
2. iSync
3. Preferences (drop down)

6. Shutdown and reboot iPhone
7. Reconnect cable then Synchronise (making sure of course that you have also ticked the Outlook Sync boxes (go to Outlook, Tools, Sync Services).

If it still doesn't work then run it again. No guarantee but I hope that fixes it, Bruce

This is possible. To select multiple contiguous emails, select an email with the Action button. Then use the stylus to tap Shift on the onscreen keyboard. Then select another email to select all the emails in between.
You can also select non-contiguous emails by tapping CTL on the soft keyboard and use the stylus to select multiple emails.
