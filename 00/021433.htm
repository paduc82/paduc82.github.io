rick rothstein, mvp - excel say:

Give the following macro a try... it will identify the misspelled words in Column A, move them and the adjacent column's words to the top of the list and identify the misspelled word with a question mark so you can see where the misspelled word list ends (so you will not have to sort the list manually). Note, the macro assumes the list starts in Row 1, but you can change this by changing the row number assigned to the InsertRow variable in the first line of code after the Dim statemen (that is, if your data starts in Row 2, simply change that line from InsertRow=1 to InsertRow=2).

Sub MoveMisspelledWordsToTop()
  Dim ColumnACell As Range, InsertRow As Long, LastRow As Long
  InsertRow = 1
  LastRow = Cells(Rows.Count, "A").End(xlUp).Row
  On Error GoTo Whoops
  Application.ScreenUpdating = False
  For Each ColumnACell In Range("A1:A" & LastRow).Cells
    If Not Application.CheckSpelling(ColumnACell.Value) Then
      Rows(InsertRow).Insert
      Cells(InsertRow, "A").Resize(1, 2).Value = ColumnACell.Resize(1, 2).Value
      Cells(InsertRow, "A").Value = "? " & Cells(InsertRow, "A").Value
      ColumnACell.Resize(1, 2).Delete xlShiftUp
      InsertRow = InsertRow + 1
    End If
  Next
Whoops:
  Application.ScreenUpdating = True
End Sub
