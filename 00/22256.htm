ken sheridan say: The following function is one of a bunch of rounding functions I produced a while ago using logic originally described by mathematical guru James Fortune in the old Access newsgroup, and with further comments by the late David Fenton:

Public Function RoundUp(dblVal As Double, dblTo As Double) As Double

    Dim lngTestValue As Long
    Dim dblTestValue As Double
    Dim dblDenominator As Double
    
    dblDenominator = -1 * dblTo
    dblTestValue = dblVal / dblDenominator
    lngTestValue = Int(dblTestValue)
    RoundUp = -1 * lngTestValue * dblTo
 
End Function
