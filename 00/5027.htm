vijay a. verma say:

You will have to use a small Macro for this.

1. Save your file as .xlsm
2. ALT+F11
3. Locate your Workbook name in Project Explorer Window
4. Right click on your workbook name > Insert > Module 
5. Copy paste the below code in this
6. ALT+F8 to display Macro Window
7. Run your Macro from here

8. If you also want to print out all worksheets through this Macro, remove leading apostrophe from the bold line. With apostrophe, bold line will not execute.

'*** Macro to Insert Title Rows *****

Sub PrintSetup()
    Dim Ws As Worksheet
    For Each Ws In ThisWorkbook.Worksheets
        Ws.PageSetup.PrintTitleRows = Ws.Rows("1:6").Address
    Next Ws

    'Worksheets.PrintOut

End Sub

You need a macro for that.

Sub Repeat_Rows_All_Sheets()
    Dim wkSht As Worksheet
    For Each wkSht In ThisWorkbook.Worksheets
         With wkSht.PageSetup
          .PrintTitleRows = "$1:$6"
          .PrintTitleColumns = ""
         End With
    Next wkSht
End Sub

amit tandon say:

To get the Top spend item (cell D2):

=INDEX($A$2:$A$100,MATCH(LARGE($B$2:$B$100,1),$B$2:$B$100,0))

To get the top 3rd item (cell D3):

=INDEX($A$2:$A$100,MATCH(LARGE($B$2:$B$100,3),$B$2:$B$100,0))

To get the top 5 items, enter below formula in cell C2 & copy down - presuming column B spends are distinct (no repeats):

=INDEX($A$2:$A$100,MATCH(LARGE($B$2:$B$100,ROW(1:1)),$B$2:$B$100,0))

To get top 5 items if there are multiple spends of the same amount, enter below formula as an array formula (CTRL+SHIFT+ENTER) in cell C2 & copy down:

=INDEX($A$2:$A$100,MATCH(LARGE(($B$2:$B$100-ROW($B$2:$B$100)/10^10),ROW(1:1)),$B$2:$B$100-ROW($B$2:$B$100)/10^10,0))
