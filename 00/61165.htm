    1)     Click on the Start button.
    2)     Type cmd in the search bar.
    3)     Right click on the cmd.exe and select Run as Administrator.
    4)     Run the following commands one by one:
    netsh interface ipv4 set global dhcpmediasense=disabled

    Run script automatically when Windows goes into hibernate or wakes - Microsoft Community

    roplar say:

    Run script automatically when Windows goes into hibernate or wakes up?

    Hi,

    I am using the Group-Policy feature in Windows-XP to run scripts automatically when you logon or logoff.

    But is it also possible to make a script run automatically, whenever a Windows-xp machine is about to hibernate? And also when it wakes up from hibernate?

    Any clues how to do this?

    a. user say:

    One funny thing about Hibernation is that if you Hibernate and then power up and don't "do" anything - like press a key or move the mouse (just sit and watch), in 300 seconds (5 minutes), XP will go back into Hibernation automatically.  Try it!

    XP does this because after it comes out of Hibernation, if XP finds that there is nothing to "do", it thinks it may have come out of Hibernation by accident and goes back to "sleep" and says "Why did you wake me up?!"

    This is why when folks hibernate at night and power their machine up in the morning and it wakes up, they go make their coffee or breakfast and come back later only to find they have to power their machine up again.  They might think something is wrong, but things are working as designed.

    One way to fix that, is to use a third party tool that uses the command line to simulate a mouse movement.   That is enough to keep XP from rehibernating while you are out.

    To simulate a mouse movement as soon as your system comes out of Hibernation, you will need to download the nircmd.exe file.

    There are lots of other good utilities from the NirSoft main page:

    http://www.nirsoft.net/

    To simulate the mouse movement, download the nircmd.exe file from here

    http://www.nirsoft.net/utils/nircmd2.html

    Put the nircmd.exe file in a folder that is somewhere in your path (c:\windows\system32, etc.) or refer to the path directly in your batch file. I put the nircmd.exe file in my c:\tools folder where I keep other things.

    If you don't have Hibernate enabled in your Power Options, Hibernate tab, this will put your system in Stand By instead of Hibernation.

    Create a batch file (I called mine hibernate.bat and put the nircmd.exe in my c:\tools folder) and put these line in your hibernate.bat file:

    @echo off
    c:\windows\system32\rundll32.exe powrprof.dll,SetSuspendState Hibernate
    c:\tools\nircmd.exe movecursor 10 10
    c:\tools\nircmd.exe movecursor -10 -10


    Save the new hibernate.bat file to your desktop or someplace you can remember and double click it to run the batch file.  When you are sure it is working okay, drag the new desktop icon down to your Quick Launch area (just to the right of your Start button), change the icon, rename it, etc so it looks a little better and then you can just single click the new Quick Launch icon once to invoke the hibernate.bat file.  Now you just have to remember to use the hibernate.bat file (which could be on your desktop with a nice icon) instead of the Turn Off Computer menu.

    The first command will put the system into Hibernation and then when XP comes out of hibernation (even days later), the next commands in the batch file will run.  Those last two commands will simulate a slight mouse movement and that will fool XP into thinking there is some reason for coming out of hibernation and it will not go back to sleep.  Then when you get back from making your coffee, your system is still waiting for you.


    If your case, your hibernate.bat might look like something like:

    @echo off
    command 1 to run before hibernate
    command 2 to run before hibernate
    command 3 to run before hibernate

    c:\windows\system32\rundll32.exe powrprof.dll,SetSuspendState Hibernate

    command 4 to run when coming out of hibernate
    command 5 to run when coming out of hibernate
    command 6 to run when coming out of hibernate
