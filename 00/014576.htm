bob jones aka: cybertaz say:
Just to clarify: Word includes only Outline Levels 1-3 by default. However, it will include as many as all 9 Outline Levels by using the Show Levels spinner on the Table of Contents pane of the Index & Tables dialog.

john korchok say:

By default a table of contents will work with all styles the have the outline level set to Level 1, 2 or 3. (Format>Paragraph>Outline level). Normally the styles called Heading 1, Heading 2 and Heading 3 match this description, but other styles can have the outline level set to 1, 2 or 3 and they, too, will appear in the TOC.

So you can either apply those Heading styles to your headings, or you can change the styles you are using to be Outline Level 1, 2 or 3.
