An AutoCorrect entry for xxxx already exists

While working in MS Office (Word documents or Outlook emails) and correcting red underlined misspelled words, I often get the message, "An AutoCorrect entry for xxxx already exists. Do you want to redefine it?" This seems illogical. If an AutoCorrect definition for the misspelled word already exists, why would I want to redefine it? To make it incorrectly spelled? (I suppose this would apply if current A/C entry is itself incorrectly spelled.) I am probably misunderstanding the question. Could someone please explain this? In any case, I never know which to click, "Yes" or "No." I usually say "No" and jsut get on with it. What I really want to do is turn this thing off so I don't have to waste so much time being interrupted by this message. How can I do that? Any advice?

suzanne s. barnhill say:

You will not get this message unless you are trying to add the correct spelling as an AutoCorrect entry, not just correct the word.

When you right-click on a red-underlined misspelled word, the context menu you get has, at the top, suggestions for correction. You can select one of these, and the spelling will be corrected. Farther down, there is an AutoCorrect menu item; if you click on that, you will get the same list of suggested correct spellings, but if you click on one, not only is the word corrected, but the correction is added to your AutoCorrect list.

If there is already an AutoCorrect entry defined for a given word and it's being marked as incorrect, then there are two possibilities:

  1. Someone else created the document who did not have this AutoCorrect entry, or you created it before adding the AutoCorrect entry, because if you had typed that particular incorrect spelling after adding the AutoCorrect entry, then it should have been corrected automatically as you typed.
  2. If you are actively typing incorrect spellings and they're not being corrected (even though the relevant AutoCorrect entries exist), then you must not have AutoCorrect enabled. In the AutoCorrect Options dialog, you must have the "Replace text as you type" box checked for AutoCorrect to function.
