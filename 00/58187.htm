Perform the task mentioned below to set headphones as default:

a: On the start screen of windows 8 press the ‘Windows key + X’ key on the keyboard and then click on control panel and then Hardware and sound option.
b: Click in Sound, then a new window will open up.
c:  In the new windows click on the tab "Playback" and right click in the window and click on Show Disabled devices.
d: Now check if headphones are listed there and right click on it and choose enable.
e: Highlight that headphones and click on "Set as Default".

There does not appear to be a similar fix for Windows 10 available yet. There is a workaround for Windows 10, although it is not a solution to the problem that is causing the error.

When the message appears in Windows 10, the mouse is inactive and the keyboard appears to be unresponsive. You cannot click the "OK" button in the dialog box that displays the error message, and pressing <ENTER> or other keys does nothing. This is because the dialog box is not the active window and cannot receive inputs from the keyboard.

When the message appears on the screen, press ALT+TAB. You may see a change in the color of the dialog box's title bar indicating that it is now the active window. Press <ENTER>, and the dialog will close, allowing the logon process to begin.

This workaround succeeds on Windows 10 version 1511 (build 10586).

You can configure your keyboard to use a different language or keyboard layout, such as Canadian Multilingual, Spanish, or United States-Dvorak. A large selection of keyboard layouts is installed with the Microsoft Windows Operating System. Generally, you should configure your keyboard layout in Windows to match the actual keyboard that you use.

There might be additional keyboard languages installed on your PC with Windows Updates. To configure your keyboard to use the installed languages and to remove the additional languages, follow the below mentioned steps and see if it works.

  • Right click on Windows Start > select Windows PowerShell (Admin). 

  • Copy and paste the following command and Enter in PowerShell - Get-WinUserLanguageList

  • Now, you will get the list of languages which are installed. Make a note of language tag of the language which you want to retain. Let's say that this language tag is en-US.

  • Now, Copy and paste the following command and Enter in PowerShell - Set-WinUserLanguageList en-US

  • This will make other languages disappear. (In case, you want to add again, you can add again from language settings).

  • You may add Japanese after uninstall all the additional languages from Settings.
