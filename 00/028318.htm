leonardleedy say:

Excel test if Cell contains Hyperlink

Is there a way to check if a cell contains a hyperlink.  Column A contains imported data from the internet. In column B I want to use a Function/Formula that will test to see if the corresponding row in column A contains a Hyperlink.  I'm thinking a function that will return a True/False.

Thanks,

Lenny

norman david jones say:

Hi Leonard,

Is there a way to check if a cell contains a hyperlink.  Column A contains imported data from the internet. In column B I want to use a Function/Formula that will test to see if the corresponding row in column A contains a Hyperlink.  I'm thinking a function that will return a True/False.

Try a User Defined Function (UDF):

  • Alt+F11 to open the VBA editor
  • Alt+IM to insert a new code module
  • In the new module, paste the following code 
'=========>>
Option Explicit

'--------->>
Public Function HyperlinkFound(aRng As Range) As Boolean
    HyperlinkFound = CBool(aRng.Hyperlinks.Count)
End Function
'<<=========

  • Alt+Q to close the VBA editor and return to Excel
  • Save the file as a macro-enabled workbook with an xlsm extension

 This UDF can be used in Excel like a native function, e.g:

            =HyperlinkFound(A1)

for a single cell, or

            =HyperlinkFound(A1:D10)

for a range.

===

Regards,

Norman
