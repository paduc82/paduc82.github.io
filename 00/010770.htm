hpholm say:

Hi Jason,

Unrecognized files received via bluetooth pile up in the 'Other' category on the phone, and cannot be accessed later.

Try installing the 'Calendar Import' app from the store, and check if it will do what you need?

 

It will most likely be required to open the .vcs file when prompted after bluetooth transfer, so the app can handle the file and keep it in its own storage space. This is how most apps work in Windows Phone: Each app has its own isolated storage space, which other apps cannot access, not even the Files manager.
