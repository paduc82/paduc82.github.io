dianeoforegon say: 1) Do you have Sync Services checked in Outlook Preferences > Sync Services?

Note: sync in Outlook 'merges'. Previously we had the option to have Entourage overwrite Apple's applications or have the Apple applications overwrite Entourage but in Outlook we only get merge. Unchecking then adding back used to force a sync. You can try unchecking then adding back to see if this will get sync started again.

2) In iCal if you don't see "On My Mac" calendars, you need to add these under File > New Calendar.

3) Only fields that exist in both Outlook and the Sync Services database are synchronized. For example if you have calendars in iCal for Home, Family, Work only Family will sync to Outlook until you add those calendars as categories to Outlook. If you create an event and do not add a category, Outlook it will assign the "no category" to those items and a calendar in iCal named "no category" will show up.

4) Open the Activity Monitor in Applications/Utilities to monitor the sync process. In the top portion of the window sort by column CPU so highest activity is at the top. You should see the following plist file as active. (Deleting these files then logging out under the Apple in the Menu bar then logging back in will create new files.  This can help if these files are corrupt. You would need to reset Sync Services after this.) 

com.microsoft.Outlook.plist
com.microsoft.outlook.office_reminders.plist
iCalExternalSync.plist
com.microsoft.Outlook.SyncServicesPreferences.plist

Also in the User's Library/Preferences/Microsoft/Office 2011 folder delete OfficeSync Prefs

Log out/in after deleting preferences under the Apple in the Menu bar.

5) If you have a lot of items in iCal, it can take quite some time for sync to start the actual sync process. 

As a test you can create a new User in System Preferences to test Sync Services. Open Outlook create a few test events with Outlook in the name using different categories. Turn on Sync Services. Do these events show up in iCal? Do the same in iCal. Create events with iCal in the name to see if these events show up in Outlook. Remember that unless Outlook has the same category as the iCal calendar they will not sync. It can take up to 10 minutes for even a handful of events to start syncing. 

Seeing sync actually work with a small number of items in a clean environment will help see the sync process actually working.
