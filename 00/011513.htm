bernie deitrick say:

Change the CELL function to include the second argument, which is a cell reference for the file of interest:

=MID(CELL("filename",A1),SEARCH("[",CELL("filename",A1))+1, SEARCH("]",CELL("filename",A1))-SEARCH("[",CELL("filename",A1))-1)

will always show the name of the file that it is in rather than the active file name.

Determining the Word version used to create a Word file - Microsoft Community

roy keret say:

Determining the Word version used to create a Word file

Hi,

I have Word 2007 on one machine and Crunchbang linux distro on another.

I have a doc file and was wondering how can I find out with which version of Word it was created.

Regards,

Roy

hansv mvp say:

I don't think Word stores the version a document was created in, only the version it was last saved in.

For a .docx/.docm document:

  • Create a copy of the document in Windows Explorer.
  • Change the extension of the copy from .docx/.docm to .zip.
  • Windows will display a warning, click OK.
  • Open the .zip file.
  • One of the files within the zip file is docPropes\app.xml.
  • Open this file.
  • Near the end is a line of the form <AppVersion>n.0000</AppVersion>
  • The "n" specifies the version the document was last saved in:

n=12: Word 2007

n=14: Word 2010

n=15: Word 2013
