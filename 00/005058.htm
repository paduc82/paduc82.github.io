Hi Walter

First, if you have any files in the Windows 10 Technical Preview installation, that you want to save, save these files to external media.

1. Boot to Windows 7. Make sure you are using an Administrator account.

2. Click Start and type  msconfig  In the results click msconfig.exe. In the Configuration editor, click the Boot tab. Make sure the Windows 7 is set as the Default OS. If it is not the Default, select Windows 7 and click the Set as default button.

3. Click the Windows Technical Preview entry and select the Delete button. Exit  msconfig.

4. Click Start and type  Computer Management, In the results click Computer Management. On the left side menu, click Disk Management.

5. Locate the hard drive partition where the Windows Technical Preview is installed. Right click that partition and select the  Delete Volume  option. Confirm the deletion.

6. Exit Disk Management and reboot the computer.

This will effectively remove the Windows Technical Preview and the dual boot configuration.

Let us know if you have more questions.

Except for the initial action of inserting a Content slide after the first Title slide, when you select any slide in PowerPoint, then press Ctrl + M, PowerPoint duplicates the layout of the selected slide. If the selected slide was based on the Comparison layout, so will the new one. 

It would be possible to create a macro that inserts a slide based on a particular layout, then add that to the Quick Access Toolbar. QAT icons are automatically assigned a keyboard shortcut.
