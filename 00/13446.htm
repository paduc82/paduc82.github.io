Hi

The reason is that each build has to go through a very long process before it is stable enough to be released to the public testers.

1. During the development process for a new version of the Windows Operating System, the initial Build of the software is named the Canary Ring.

2. The next version is named the OSG Ring (Operating Systems Group) and is tested by the developers who are team members of that group. The members of this group test and Validate that ring.

3. The next version is named the Microsoft Ring. This version is distributed to Microsoft Employees for testing and validation.

4. The next version is named the Fast Ring. This ring is uploaded and made available to the public. These are the Expert Insiders who are Installing, Testing, and Validating, the Preview builds of the Windows 10 Technical Preview.

5. The next version is named the Slow Ring. This version is the last in the series of Rings for that build.

During this process, each group of testers report any problems with that Ring and these problems are fixed, prior to the next Ring being released (Validation). 

You can decide which Ring you would like to install in PC Settings. 

Most testers will want to install the Slow Ring simply because it has been validated by the most groups and will contain the fewest bugs.

julie fka profjulie say:

Windows Phone does not use any contacts that are stored on a SIM card, but you can import contacts from a SIM card to a Windows Phone as follows:

On the phone, tap the People tile which is on the phone's home screen

Tap the three dots in the lower right hand corner of the screen > tap Settings

Tap the "Import SIM contacts" button.

If you see contacts in the People Hub that were imported into the phone from the previous SIM card, the contacts are either stored in the phone's memory or they could have been synced from the Microsoft account that you used when you set up the phone. 

  • If these contacts are stored in the People Hub of the Microsoft account, then you can go there to delete the contacts, then sync the phone to the Microsoft account and the contacts will be deleted.
  • If these contacts are stored in the phone's memory, you'll need to delete each contact, one-at-a-time.  You can do this by pressing & holding your finger on a contact until a menu pops up, then select the "delete" option.

Contacts are generally synced between a windows phone and a Microsoft account.  So once your contacts are synced to a Microsoft account, they are available (until you delete them) to be synced to any device you currently use or plan to use in the future (computers, tablets, phones - no matter what OS is running on the device).

bernie deitrick say:

I don't know how to change the default behavior, but try a macro like this - select the csv file in the dialog...

Sub OpenCSVAsTXT()
Dim myFN As String
Dim myNewFN As String

'Get the filename
myFN = Application.GetOpenFilename("CSV Files (*.csv), *.csv")
myNewFN = Replace(myFN, ".csv", ".txt")

'Rename the file
Name myFN As myNewFN
Application.DisplayAlerts = False

'Open the file
On Error Resume Next
Workbooks.OpenText FileName:=myNewFN, Origin:=437, StartRow _
        :=1, DataType:=xlDelimited, TextQualifier:=xlDoubleQuote, _
        ConsecutiveDelimiter:=False, Tab:=True, Semicolon:=False, Comma:=True, _
        Space:=False, Other:=False
Application.DisplayAlerts = True
'Save the file as an Excel File
ActiveWorkbook.SaveAs Replace(myFN, ".csv", ".xls")

'Restore the orignal name
Name myNewFN As myFN
