Jage,
Re:  "so only column 5 is locked after data entry"

'First unlock all cells on the sheet  - only required once

'This is setup so that once an entry is made it can't be changed.
'Only blank cells can have a new value entered.

'---
Private Sub Worksheet_Change(ByVal Target As Range)
'Adds manual entries to a dynamic list on another sheet within the workbook

On Error GoTo BadCode   '<<<< Replaces On Error Resume Next

'Code stuff here

Call Macro2(Target)

Application.EnableEvents = True
Exit Sub      '<<< New

BadCode:  '<<< New
  Application.EnableEvents = True
  MsgBox "Error " & Err.Number & " - " & Err.Description
End Sub
'---

Sub Macro2(ByRef Target As Excel.Range) 'Password protects in a specific column
Dim confirm As Long
Dim Cell As Range

Application.EnableEvents = False
If Target.Column = 5 Then
  confirm = MsgBox("Do you wish to confirm entry of this Cell?", vbYesNo, "Confirm Entry")
  Select Case confirm
  Case Is = vbYes
    ActiveSheet.Unprotect Password:="password"
    With Application.Intersect(Target.Columns(1).Cells, ActiveSheet.UsedRange)
       .Cells.Locked = False
        For Each Cell In .Cells
           Cell.Locked = Not IsEmpty(Cell.Value)
        Next ' Cell
    End With
    ActiveSheet.Protect Password:="password"
  Case Is = vbNo
    Application.Undo
  End Select
End If
Application.EnableEvents = True
End Sub

How can I Sum up the numbers seperated by a hyphen - Microsoft Community

shawn7 say:

How can I Sum up the numbers seperated by a hyphen

How can I Sum up with a formula the numbers which are located in one cell and seperated by a hyphen. There is only 1 space. For example cell

A1: 2-5 4-6

B1: 3-6 4-5

C1: 5-5 4-4

jasperp say: Try this array formula:

=SUM(MID(A1:C1,1,1)+MID(A1:C1,5,1))

Enter with Ctrl+Shift+Enter, not just Enter. Excel will put curly braces { } around your formula.
http://office.microsoft.com/en-us/excel-help/introducing-array-formulas-in-excel-HA001087290.aspx

To add the numbers of the other player, replace 1 and 5 by 3 and 7, like so:

=SUM(MID(A1:C1,3,1)+MID(A1:C1,7,1))
