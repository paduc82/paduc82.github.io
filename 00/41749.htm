steef_1202 say:

Horizontal dashed line in Windows 10 network status!

We have experienced this behaviour also on some of our company laptops last week.

This behaviour can cause Office 365 to not find the license server anymore, if you take a look on your office account in for example word it will say: No internet connection.

However you are still able to browse the web and able to ping several webservices.

Although a restore can help with this issue, this is not the root cause.

In our case a restore did not help, neither did a firmware, bios and driver update for the network adapters.

We deinstalled several software packages but this was also not helpful.

The solution was pretty simple:

The network list service was disabled in services, this was most likely the case after a windows update or software that has been installed (for example a VPN software solution)

We suspect it to be because of a Windows update since we had this issue on several laptops in different countries all within a certain time frame (no software was installed during that timeframe, only windows updates)

Once we set the network list service back to manual startup and started the service the horizontal dashed line disappeared and

we were able to fully function again.
