Is there a way within Excel to sort the data within a cell? - Microsoft Community

laserray70 say:

Is there a way within Excel to sort the data within a cell?

By this, I mean take a cell (say A1) that has the following data:

and have excel reorder the data in the cell to:

item1 item3 item4 item9

 

I don't have time to create a VBA macro do to this, so if Excel doesn't have some builtin function, then I am looking for a canned macro or maybe a plugin.

 

Thanks much.

 

Ray

jeeped say:

... and have excel reorder the data in the cell to:
item1 item3 item4 item9


Tap Alt+F11 and when the VBE opens, immediately use the pull-down menus to Insert, Module. Paste the following into the new pane titled something like Book1 - Module1 (Code),

Sub mcrSortCellContents()
    Dim c As Range, i As Long, j As Long, s As String
    Dim vValues As Variant
    For Each c In Selection
        vValues = Split(c.Value, Chr(32))
        For i = 0 To UBound(vValues) - 1
            For j = i + 1 To UBound(vValues)
                If vValues(i) > vValues(j) Then
                    s = vValues(i)
                    vValues(i) = vValues(j)
                    vValues(j) = s
                End If
            Next j
        Next i
        c.Offset(0, 1).Value = Join(vValues, Chr(32))
    Next c
End Sub

Tap Alt+Q when the script has been pasted to return to your worksheet. Select all of the cells in a single column that are to be sorted and ensure that there is a blank column directly to the right. Tap Alt+F8 to open the Macros dialog and Run the script.

            

Note that my script depends upon a single space delimiting the values within a single cell as per your sample data. This is referenced by Chr(32) in two places if it needs to be changed.
