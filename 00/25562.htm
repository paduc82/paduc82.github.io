Outlook 2013 contacts - Microsoft Community

lboardman say:

Outlook 2013 contacts

If I uninstall Office 2013 and then install Office 2016, will I lose my Outlook contacts?

diane poremsky mvp (slipstick.com) say:
It depends on what type of email account you use and where the contacts are stored.

If you use outlook.com, office 365 Exchange, or on-premise Exchange, the contacts are safe in the mailbox only. You can remove the account and the contacts won't be deleted.

If you use a POP account, the contacts are safe in the pst file. It won't be deleted when you delete the account from outlook, but the pst will need to be backed up if you reinstall Windows.

If you use an IMAP account and the contacts are in a folder labeled 'this computer only' they will be lost unless you export them to a pst file (along with any appointments and tasks in their respective folders).

davidf2 say:

tim_ver,

Your question is a bit open ended and you have a number of options.

If you already have a website and just want to make a single page Publisher publication available on that website, you could convert it to a single page .mht file and upload it to your existing site. However it would probably be easier to just convert it to a PDF file and upload that file and make it available to download. A PDF file will look the same to everyone and you don't have to be concerned about cross browser compatibility.

A four page Publisher publication can also be converted to a PDF or if you want to convert to a web publication then you do File > Convert to a web publication. In general you need to understand that print is different than web and there are things that won't convert to html format, which means you will probably have to tweak the layout, design and/or formatting of a print publication when you convert it to a web publication. You should plan on testing your site in both IE and FireFox. If it renders correctly in both those browsers then it will also work in the other major browsers.
