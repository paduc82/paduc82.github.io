steve rindsberg say:
This macro deletes all ActiveX objects. I'd use it ONLY on a copy of your presentation, since it'll delete them whether or not they're registered:

Sub DeleteActiveX()
    Dim oSl As Slide
    Dim oSh As Shape
    Dim x As Long
    
    For Each oSl In ActivePresentation.Slides
        For x = oSl.Shapes.Count To 1 Step -1
            Set oSh = oSl.Shapes(x)
            If oSh.Type = msoOLEControlObject Then
                oSh.Delete
            End If
        Next
    Next

End Sub

stefan blom say:

If indentation changes unexpectedly for paragraphs in a multilevel list, that is a sign that indents haven't been set up properly from the start. All aspects of numbering, including indentation around numbered items, must be specified in the Multilevel List dialog box. (The Paragraph dialog box can only be used for indents from the right, when multilevel lists are involved.)

Click in the first Heading 1 paragraph of your document and then click Home | Multilevel List | Define New Multilevel List. Verify that paragraph styles are associated with the correct levels, that restarts are set appropriately, and make sure that the indentation options under "Position" (the options at the bottom in Jay's screen shot) are correct.

In the dialog box, "Aligned at" determines the position of the list number. "Text indent at" defines the position where second and subsequent text lines start. "Add tab stop at" defines the position of the text on the first line of text, following the list number. All measurements are from the left margin. If you want a hanging indent, set "Text Indent at" equal to "Add tab stop at" (and both of these measurements should be larger than "Aligned at").

jay freedman say:

To change the starting value for numbering, you don't have to set up everything. The entry "Define New Multilevel List" in the dropdown is misleading -- clicking it does not necessarily start a new list. When a list format is already applied to the document's headings, clicking that entry opens the Multilevel List dialog box with all the current information already in place. You can change just the starting number and click OK.
