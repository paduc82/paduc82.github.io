ramesh srinivasan say:

Start Regedit.exe and go to:

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Fonts

Backup the branch by exporting to a .REG file.

In the right-pane, delete the following values:

  1. HelveticaNeue-UltraLigCond (TrueType)
  2. HelveticaNeue-UltraLigCondObl (TrueType)
  3. HelveticaNeue-UltraLigExt (TrueType)
  4. HelveticaNeue-UltraLigExtObl (TrueType)
  5. HelveticaNeue-UltraLight (TrueType)
  6. HelveticaNeue-UltraLightItal (TrueType)

Exit the Registry Editor, logoff and login back.

Then manually delete following .OTF files from C:\Windows\Fonts using Command Prompt (Admin).

  1. HelveticaNeue-UltraLigCond_0.otf
  2. HelveticaNeue-UltraLigCondObl.otf
  3. HelveticaNeue-UltraLigExt.otf
  4. HelveticaNeue-UltraLigExtObl.otf
  5. HelveticaNeue-UltraLight.otf
  6. HelveticaNeue-UltraLightItal.otf

Running this command will show the list of .otf files you installed.

dir c:\windows\fonts\helvetica*.otf

(There zip file in the above website bundles 6 .otf files, which you need to delete from the Fonts folder.)

Then run:

del c:\windows\fonts\helvetica*.otf

    • The PC boots into Windows RE (Recovery Environment).

    • Windows RE installs a fresh copy of Windows.

    • The PC restarts into the newly installed copy of Windows.
