Combine Shapes drawn with Curve Shapes tool -- BUGGY? - Microsoft Community

vkayzy say:

Combine Shapes drawn with Curve Shapes tool -- BUGGY?

Been using the Combine Shapes tool to mostly combine or subtract shapes drawn with the Curve Tool. I found that most of the time that Combine Shapes didn't seem to do anything -- the shapes remain as they were. But sometimes it worked. I've outline below what works and what doesn't. My question up front is: IS THIS BY DESIGN OR A BUG? If by design, it would be good if Powerpoint put up a message saying "Shapes are not closed" so you know what's going on and don't waste time figuring out why some shapes work and other's don't. 

Through experimentation I found that successes depends on how the Curve Tool is used to create the shapes.If you don't close the shape properly the Combine Shapes tool will not work on the curved shapes. You can try this to see what I mean:

WILL NOT COMBINE EXAMPLE
1) With the Curves Tool selected,  click to mark the first point of the shape, move the mouse across the screen to make a line and click, then move the mouse down and back and double click to make a parabola. 
2) Select the color fill to make it look like a solid parabola with a flat side.
3) Repeat the above to create another solid parabola
4) Overlap the seemingly solid shapes and try to use Combine Shapes to combine or subtract -- it doesn't work.

It is possible to make it work by carefully closing the shapes when you make them. That is you must overlap the start and end points. Getting the end point "close" to the start point will not work. (But now the shapes will not have a flat side).

COMBINE WILL WORK EXAMPLE
1) Use the Zoom Slider to enlarge the screen to about 150% (easier to close the curve).
2) With the Curves Tool selected,  click to mark the first point of the shape, move the mouse across the screen to make a line and click, then move the mouse down and back and click to make a parabola, and now move the mouse so it exactly overlaps the first point and double-click. You have an arrow-head shape (parabola with a curved back).
3) Select the color fill to make it look like a solid shape.
4) Repeat the above to create another solid arrow-head.
5) Overlap the solid shapes and try to use Combine Shapes to combine or subtract -- it should work fine (if not, you didn't close either of the shapes properly).
