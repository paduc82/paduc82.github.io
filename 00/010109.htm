If you link to individual cells, the link references in Word are not affected by anything you do in Excel; they remain the same and will point to whatever now happens to be at those addresses. You could, of course, change the cell references in Word as well.

vijay a. verma say:

Here is final conclusion -

Your file is encoded in UTF-8. If a file is encoded in UTF-8 or Unicode,  Excel forces Wizard to open. If it is in ANSI encoding, Wizard will not be opened.

You can experiment by opening the file in Notepad and when you do Save As, you can choose encoding next to Save button.
