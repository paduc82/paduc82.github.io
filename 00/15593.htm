{ = { MergeField MyValue } * 2 }

macropod say:
Your post suggests some of the braces (i.e. { }) have been typed; you need to use actual field braces, created via Ctrl-F9 or insert the actual merge fields where you have { MergeField MyValue }, etc.

dlh_573 say: I'm very familiar with using Ctrl+F9 to create Word fields. That's not the problem. And the spelling of all my field codes and column names have been double checked.

macropod say:

Try using:

{SKIPIF{={IF{MergeField MyFilterColumn \# 0}= 5 1 0}+{IF{MergeField MyFilterColumn \* lower}= "none" 1 0}}=1}

and:

{={MergeField MyValue \# 0}*2}

The first construction necessitates on a single SKIPIF field and the second ensures MyValue outputs a number even when the record is empty. If the MyValue field contains extraneous characters, though, some additional coding may be required.
