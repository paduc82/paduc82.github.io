ken sheridan say: You can open a form as read only by means of the acFormReadOnly constant as the setting for the DataMode argument of the OpenForm method:

DoCmd.OpenForm "NameOfForm", DataMode:=acFormReadOnly

A potential fly in the ointment, however, is that this will also disable any 'editable' unbound controls in the form, such as a navigational  combo box, which you might want to be available to the user.  Command buttons are unaffected, however.  This is also true if you set the form's AllowEdits property to False in code.  To deny users the ability to edit selected controls in a form you can set the Tag property of the relevant controls to something like LockMe, and then disable/lock the controls like this in the form's module:

Dim ctrl As Control

' first move focus to a control not to be locked/disabled
Me.SomeControl.SetFocus

For Each ctrl in Me.Controls
   With ctrl
       If .Tag = "LockMe" Then
           .Locked = True
           .Enabled = False
        End If
    End With
Next ctrl

You can make this code conditional on a value passed into the form as its OpenArgs property if you want it to be read-only.
