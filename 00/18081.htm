Thanks, how do enter that code while recording a macro?

You would have to press Alt+F11 and manually edit the recorded macro sequence.
If you want to do this while recording try this sequence:
  1. Click Record Macro.
  2. With Use Relative References turned off, select B1.
  3. Toggle Use Relative References to on.
  4. Press Ctrl+<down arrrow> or alternately, Ctrl+End.
  5. Tap Home.
  6. Press Shift+Ctrl+<up arrow> or Shift+Ctrl+Home.
  7. Click Stop Recording.
Press Alt+F8 to open the Macros dialog then select the recorded macro and click Edit. It should look like this,

Sub Macro1()
    Range("B1").Select
    Selection.End(xlDown).Select
    ActiveCell.Offset(0, -1).Range("A1").Select
    Range(Selection, Selection.End(xlUp)).Select
End Sub

... which ultimately could be written as,

Sub Macro1()
    Range("A1:A" & Cells(Rows.Count, "B").End(xlUp).Row).Select
End Sub

steve rindsberg say:
Remember way back at the beginning of the thread where I said you'll need to create an add-in?
There was a reason for that. ;-)

PowerPoint never runs code automatically other than via the Auto_Open and Auto_Close subs in an add-in.  And of course, in response to events if there's an event handler active.

Auto_Open will run automatically when the add-in containing it loads.  That will happen automatically every time PPT starts up, once the add-in has been added via File | Options | Add-Ins | Manage Add-ins (choose PowerPoint Addins, click Go) and adding the PPAM or PPA file you've saved from the PPT or PPTM that contains your code.
