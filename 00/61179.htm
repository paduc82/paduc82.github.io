Error code 10 in device manager appears in one of the following situations:

a. user say:

Hi Cecile Cornell,

Thank you for letting us know your concern.

The message mfc110u.dll errors can be caused by several reasons:

•Invalid or corrupt mfc110u.dll registry entry.

•Virus or malware infection which has corrupted the mfc110u.dll file.

•Another program maliciously or mistakenly deleted the mfc110u.dll file.

•Another program uninstalled the mfc110u.dll file.

I would suggest you to perform the below methods and check if that helps:

Method 1: Install all the Windows updates. If you will face any issue with installing updates then run the Windows Update Troubleshooter and check if it works.

Follow the steps to run the troubleshooter:

  1. Press Windows key + C on your keyboard to show the Charms bar.
  2. Type Troubleshooting and click Troubleshooting under Settings.
  3. Click on View all option on the left.
  4. Click on Windows Update Troubleshooter to run the troubleshooter.

Method 2: 

Step 1: Place the computer on Clean Boot and try installing the drivers.

Refer to this link: http://support.microsoft.com/kb/929135

Note: Please go through the section: How to reset the computer to start as usual after troubleshooting with clean boot of the Kb article to boot the computer in normal startup after fixing the issue.

Step 2: Update your Device Drivers.

Download the Windows 8 latest device drivers from the manufacturer’s website. If Windows 8 drivers are not available then download the Windows 7 drivers and install it in Compatibility mode. Follow the below steps:

  1. Right-click on the driver and click on ‘Properties’.
  2. Click on the ‘compatibility’ tab and check the box ‘Run this program in compatibility mode for’ and select Windows 7 operating system from the drop down.
  3. Click on ‘Apply’ and click ‘OK’ and run the file to install it.

Method 3: Run the SFC.

Follow the below steps:

  1. Open Command Prompt.
  2. Select “Run as Administrator”.
  3. Type “sfc /scannow” without quotes and hit Enter.

Method 4: Perform a System Restore.

Refer the below link and perform the System Restore.

http://windows.microsoft.com/en-US/windows-8/restore-refresh-reset-pc

NOTE: The System Restore restores the computer back to an earlier point in time, called a restore point. System Restore doesn’t change your personal files, but it might remove recently installed apps and drivers and also reset the recently changed settings on the computer.

Keep us posted if you face any issues on windows in future. We will be glad to help you.
