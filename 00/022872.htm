How do I find the last value?

The question was: ...I just want to see the last date entered. As you know Dates are Numbers in Excel.

MAX(ISNUMBER($B$2:$B$5001)*ROW($B$2:$B$5001)) is an array calc. that checks if each value in B2:B5001 is a number. This generates an array of i.e. {TRUE,FALSE,FALSE,...,TRUE,...,FALSE}, multiplied by another array (the row numbers) that in this case is {2,3,4,5...,5001}.

TRUE*2=2, FALSE*3=0, FALSE*4=0...,TRUE*18=18, ...FALSE*5001=0. In the end we get an array like {2,0,0,...,10,...18,...,0}. MAX of this array returns 18 (last row where a number was found).

MAX(ISNUMBER($B$2:$B$5001)*ROW($B$2:$B$5001)) is an array calc. that checks if each value in B2:B5001 is a number. This generates an array of i.e. {TRUE,FALSE,FALSE,...,TRUE,...,FALSE}, multiplied by another array (the row numbers) that in this case is {2,3,4,5...,5001}.

In other words. by combination of ISNUMBER and ROW functions, you create an array, where the last entry is the maximum one.

If we want to find the last entry in a one row entry, it will be enough to change the ROW to COLUMN in your formula.
