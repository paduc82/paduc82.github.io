jay freedman say:
Yes, you can create an entirely new template, and in fact that's the recommended process -- it's a good idea to create a different template for each different type of document you frequently deal with. Each template can have its own set of styles, page layout, ribbon and Quick Access Toolbar customizations, macros, and AutoText entries if that's what you want.

The procedure for creating a clean new template is shown at http://www.shaunakelly.com/word/templates/how-to-save-a-word-document-as-a-template.html.

Once the template is open in Word, there are several ways to create new styles in it. You can modify the built-in styles, or open the Styles pane (Ctrl+Alt+Shift+S) and click the New Style button at the bottom of it, or format some paragraph the way you want the style to look and then right-click it and choose Styles > Save Selection as a New Quick Style.

After you create the template and save it (preferably in the Templates folder, which is usually %appdata%\Microsoft\Templates), you can use File > New > My Templates to select the template as the base for a new document. The "Blank Document" is the only one that's automatically based on the Normal template.
