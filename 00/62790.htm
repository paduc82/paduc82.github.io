The WinSxS folder is the location for Windows Component Store files. The Windows Component Store is used to support the functions needed for the customization and updating of Windows. Here are some examples of how the Windows Component Store files are used:

  • Using Windows Update to install new component versions. This keeps systems secure and up-to-date.
  • Enabling or disabling Windows features.
  • Adding roles or features using Server Manager.
  • Moving systems between different Windows Editions.
  • System recovery from corruption or boot failures
  • Uninstalling problematic updates
  • Running programs using side-by-side assemblies

Deleting files from the WinSxS folder manually or deleting the entire WinSxS folder may severely damage your system so that your PC might not boot and make it impossible to update. What this means is that it's extremely not recommended to manually delete any files from the WinSxS folder. The WinSxS folder normally has files in it, and if you want to find out how much disk space the folder really uses, you can use this in Command Prompt (admin): 

C:\>dism /online /cleanup-image /analyzecomponentstore

This command will result in something like this:

This size is normal. The other reliable method of cleaning up system files is using Disk Cleanup:

  1. Click Start.
  2. Type Disk Cleanup and select it from the list of results.
  3. Click Clean up system files.
  4. Select the files you want to delete.
  5. Click OK.

Once you've done this, along with the command above that Mike P advised, any files left in the WinSxS are essential to your system and should not be deleted.
