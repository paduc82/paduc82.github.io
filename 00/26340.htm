suzanne s. barnhill say:

The sort order in indexes is based on the same algorithm that is used to alphabetize text in a document. It obviously makes capital letters come before the corresponding lowercase letter and prioritizes spaces over hyphens.

There is probably no way to correct this except by manually editing the index. There's no point in any manual editing until the document is entirely finished and the index has been updated for the last time. Then you will have to lock or unlink the INDEX field and make the manual edits.
