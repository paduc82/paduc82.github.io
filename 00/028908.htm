It looks like this file has some "non-printing" characters in it (something like a line feed character), and that is causing the result you are seeing when you do Text to Columns.  Here's one thing you can do to fix that:

  • There is an Excel function named CLEAN that will remove non-printing characters from text.
  • In an empty column (let's say B for example), enter the formula "=CLEAN(A2)" in row 2, and then drag down for all the rows of data in the file.
  • Copy the contents in Column B, and then do a Paste Special..., Values back into Column A.
  • Now try the Text to Columns - it should work properly.

I'm not sure why Text to Columns fails with non-printing characters, but apparently it does.
