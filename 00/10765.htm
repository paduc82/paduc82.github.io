scott mcdaniel say:

Recovering from a backup in Access simply means using your backup file in place of your corrupted live file. To do that, just locate the backup and copy it over the corrupt file. If you want to save the corrupt file, rename it to something else.

If you want to recover the corrupt file, you can try this (make a copy first):

Decompile the file. To do that, create a standard Windows shortcut with this as the target:

"full path to msacess.exe" "full path to the database" /decompile

Run that, then see if you can open it.

You can try to compact it from a command line (assuming you cannot open the file - and again, be sure to make a copy of the file first):

Open a command prompt, then enter this:

"full path to msaccess.exe" "full path to your database" /compact

If that doesn't work then you're left with contacting a recovery service. Wayne at www.everythingaccess.com is the best in the field.
