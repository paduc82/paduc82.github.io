mike h.. say:

Hi,

 

This look at colum c and does a vlookup (effectively) on columns A & B, if the value is found in column A then the value in col B is returned otherwise the code asks for user input for the value to be returned to the target cell.

 

Right click your sheet tab, view code and paste the code in

 

Private Sub Worksheet_Change(ByVal Target As Range)
If Target.Cells.Count > 1 Or IsEmpty(Target) Then Exit Sub
   If Not Intersect(Target, Range("C:C")) Is Nothing Then
    Application.EnableEvents = False
       If IsError(Application.Match(Target.Value, Range("A:A"), 0)) Then
       Target.Value = InputBox("No match, enter your value")
       Else
       Target.Value = Range("A" & Application.Match(Target.Value, Range("a:a"), 0)).Offset(, 1).Value
        Application.EnableEvents = True
       End If
    End If
    Application.EnableEvents = True
End Sub
