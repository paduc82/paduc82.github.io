william zeng msft say:

Hi TJKroll,

From your description above, this is because the account credential is persisted on your computer. In this case, the best practice here is to remove OneDrive for Business from your computer, you can follow the steps below:

For Windows 10: OneDrive is built into Windows 10, so you can‚Äôt uninstall it like an app.

  1. Press Win+R to open Run dialog
  2. Type cmd to bring up the command prompt window
  3. Type the following to stop the OneDrive process:

taskkill /f /im OneDrive.exe

  1. If you are on a 32-bit system, type:

%systemroot%\System32\OneDriveSetup.exe /uninstall

  1. If you run a 64-bit version, type:

%systemroot%\SysWOW64\OneDriveSetup.exe /uninstall

For Windows 7 and 8.1:

  1. Right click the Start button to open Control Panel
  2. Select Uninstall a program under Programs section
  3. Find Microsoft OneDrive and Uninstall it.

You can reinstall OneDrive for Business if you want to continue use it.

'======================================

WORKBOOKABC.Sheets("PQRS").RANGE("A1").Select

IF WORKBOOKABC.Sheets("PQRS").RANGE("A1").Interior.Color = RGB(102,204,205) Then

    With Selection.Interior
        .Pattern = xlSolid

        .PatternTintAndShade = 0

    With Selection.Borders(xlEdgeLeft)
