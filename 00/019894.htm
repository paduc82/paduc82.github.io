Below are general troubleshooting steps for Outlook. Try one by one

1. Create new Outlook profile to rule out corruption in Outlook profile. - http://office.microsoft.com/en-us/outlook-help/...

2. Type command in your Windows Search Box > Right click on Command Prompt > Run As Administrator > Type """"sfc /scannow"""" without quotes > Run it and when it completes, configure Outlook again.

3. Please check if you have Lync or Skype installed on this computer which starts when the computer starts. If yes, then get out of Skype and start Outlook. Basically, Outlook need to start first than Skype.

4. Repair your Outlook data files - https://support.office.com/en-us/article/Repair...

5. Disable Anti-virus temporarily.

6. There is a possibility that some Add-in is causing the problem. To rule this out, start Outlook in safe mode. Hold CTRL key and click on Outlook icon. Don't leave CTRL key, till it asks for the confirmation about safe mode.

7. Login to Windows with a new profile to rule out problem in Windows profile.

8. Right click on Outlook icon and run as Administrator.

9. File > Options > Advanced > Check Disable Hardware Graphics Acceleration

10. Permanently delete all your deleted / spam mails, if not needed. And empty your Junk / Clutter / Spam folders also (Other is new name for Clutter in 2016), if not needed.

11. Account Settings > Open your account > More settings > Under Advanced tab, put INBOX in Root Folder Path. (If your all folders disappear, simply remove INBOX from here)

12. If not, you will need to see which startup application is conflicting with Outlook. (All startup apps can be seen by typing Msconfig.exe in Windows search and can be disabled and enabled. This detective work needs to be done

The watermark appears by default on all prerelease software, not just Windows 10 preview builds.  In the case of Windows 10 the watermark is controlled by the PaintDesktop registry setting.  "1" = show watermark and "0" = do not show.  The watermark is always available and the user can turn it on and off regardless of the status of the software.  The reason you didn't see it on Slow Ring until a couple of weeks ago was the Slow Ring was still on the public release (build 17763.xxx).  

The watermark has been used by MSFT for over 20 years to indicate pre-release software.
