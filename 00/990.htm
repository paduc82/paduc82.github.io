What is a Microsoft account?

A Microsoft account (Formerly Windows Live ID) is the email address you use along with your password to sign in to any Microsoft program or service such as Outlook.com, Hotmail, Messenger, SkyDrive, Xbox LIVE, or Office Live.  Some examples of Microsoft accounts end in @live, @hotmail, @outlook.com.

 

We remove accounts that have not had any logins for more than one year.  If you had a Microsoft account and have not used it for more than a year you will need to create a new account.

If you use any of the services listed above, please continue reading for hints and tips on how to discover your Microsoft account.  If you do not use any of these services, your best bet is to create a new Microsoft account.  If you are sure that you do not use any Microsoft services, click here to create a new Microsoft account.

 

Here are some suggestions that may help you remember a forgotten Microsoft account email address.

1.       If you used the account to send email, ask someone you sent an email to what address the message came from.

2.       If you used the account with Windows Live Messenger, launch Messenger to see if it automatically logs in with your account.  If it does, you can log out and the resulting login screen will display your Microsoft account.

3.       You can also ask your contacts to check their Messenger contacts for your email address.

4.       If you used another computer in the past to log into your account, on that computer go here, click into the account name box, and push the down arrow on your keyboard.  This may display a list of previous logins and reveal your Microsoft account name.

 

On Windows XP, Vista or Windows 7, check to see if your email is saved in the Windows credential vault.
Go to Control Panel, User Accounts and click Manage your credentials.  This credential vault can store user names and passwords.  Your email may be listed here if it was ever saved

 

Are you an Xbox User?  Great!  Please visit the Xbox support website by clicking Here for help in using your gamertag to retrieve your Microsoft account.

pete_uk say:

Okay, following my earlier advice, this is the recorded macro:

Sub merge_sheets()
    Sheets("Sheet2").Select
    Sheets("Sheet2").Copy After:=Sheets(3)
    Range("A1").Select
    Selection.End(xlDown).Select
    ActiveCell.Offset(1, 0).Range("A1").Select
    Sheets("Sheet3").Select
    Range("A2:H2").Select
    Range(Selection, Selection.End(xlDown)).Select
    Selection.Copy
    Sheets("Sheet2 (2)").Select
    ActiveSheet.Paste
    Application.CutCopyMode = False
    Sheets("Sheet2 (2)").Select
    Sheets("Sheet2 (2)").Name = "Sheet4"
    Range("A1").Select
End Sub

It copies Sheet2 (which gets named Sheet2 (2)), then moves the cursor down to the next empty cell in column A, then copies the range from A2:H whatever of Sheet3 into the newly-created Sheet2 (2) and pastes it at the bottom. It then renames that sheet to Sheet4 and selects cell A1.

However, this is not very efficient, and may fail the next time you run it. For one thing, if you already have a Sheet4 then the new Sheet2 (2) cannot be renamed to this, so you need to delete any existing Sheet4 (or rename it) before running the code again (tomorrow). Also, the macro recorder puts a lot of "selects" in the code which can be eradicated to tidy up the code.

I don't know how you intend to use this, so if you start with only Sheets 1 to 3 each day then it might do for you.

Hope this helps.

andrey kazak say:

Excel 2010: toggling Lock Picture Aspect Ratio of a comment

Dear community members,

  As you know, Excel allows to fill background of a cell comment with a picture.
  I want to create a VBA snippet to automate it, but with Lock Picture Aspect Ratio toggle switched on (see screenshot).

  Can you suggest me how to do it using VBA please?

ossiemac say:

I don't know of any method of directly setting or locking the comment frame to a size that directly correlates to the picture aspect ratio so that the picture fills the comment frame.

 

However, the code below is a workaround that actually inserts the picture in the worksheet first; (Not in the comment frame) with the aspect ratio locked and then it is possible to calculate the ratio and use the calculated ratio to set the dimensions of the comment frame before importing the picture into it. The picture inserted in the worksheet is then deleted.

 

Also the code example contains other code that you might find interesting.

 

Feel free to get back to me with any questions you have regarding the code.

 

Sub CreateAndFormatComments()
   
    Dim ws As Worksheet
    Dim cmt As Comment
    Dim strComment As String
    Dim strPictFileName As String
   
    Dim strFileName As String
    'Dim lngLeft As Long
    'Dim lngTop As Long
    Dim shpPic As Shape
    Dim dblWidth As Double
    Dim dblHeight As Double
    Dim dblRatio As Double

    Set ws = Worksheets("Sheet1")   'Assign worksheet to a worksheet variable
    strPictFileName = "C:\Users\User\Pictures\Facebook\Toys\Bicycle 4.jpg"
   
    'Following creates string for comment text. Edit as required
    strComment = "My Test Comment"
   
    'Test if comment already exists
    Set cmt = ws.Cells(4, "C").Comment      'Assign comment to a comment variable
   
    If Not cmt Is Nothing Then      'Not nothing then exists
        'Can use delete here if required and re-create
        cmt.Delete  'Optional
        'Create new comment and assign to Comment Variable
        Set cmt = ws.Cells(4, "C").AddComment   'Create new only if deleted
    Else        'If is nothing then must create
        'Create new comment and assign to Comment Variable
        Set cmt = ws.Cells(4, "C").AddComment
   
        'Alternative code to assign comment text at same time
        'Set cmt = ActiveCell.AddComment(Text:=strComment)
    End If
   
   '*********************************************************************************
    'Code between the asterisk lines to insert a picture and get it's aspect ratio _
     and assign the same aspect ratio to the comment shape prior to assigning the _
     picture to the shape fill. (Picture is then deleted from worksheet.)
   
    strFileName = strPictFileName
   
    'Note: the -1 parameters for Width and Height to maintain the aspect ratio _
     Left and Top are set to zero (top left of worksheet)
    Set shpPic = ActiveSheet.Shapes.AddPicture(strFileName, False, True, 0, 0, -1, -1)
   
    With shpPic
        dblRatio = .Width / .Height     'Calculate the aspect ratio
        .Delete       'Picture no longer required so delete
    End With
   
    With cmt.Shape
        'Entering the dot here will display intellisence dropdown _
         for all available code options.
        .Height = .Height * dblRatio    'Set ratio same as picture ratio
        .Fill.UserPicture (strPictFileName)
        .LockAspectRatio = msoTrue
        .Shadow.Visible = msoFalse
    End With
   
    '******************************************************************************
   
   
    'Add the string comment and format the TextFrame and Font
    With cmt.Shape.TextFrame
        .Characters.Text = strComment       'Adds text if new comment or replaces existing text
        'Following line is example to concatenate with existing text _
         if comment already exists.
        '.Characters.Text = .Characters.Text & " " & strComment
       
        '.AutoSize = True        'Don't AutoSize when set to picture size
        With .Characters.Font
            .Name = "Arial"
            .Size = 8
            .Bold = True
        End With
       
    End With
   
    Application.DisplayCommentIndicator = xlCommentIndicatorOnly
   

'XlCommentDisplayMode Enumeration options for above line (From Excel Help)
    'xlCommentAndIndicator      1 Display comment and indicator at all times.
    'xlCommentIndicatorOnly     -1 Display comment indicator only. Display comment when mouse pointer is moved over cell.
    'xlNoIndicator              0 Display neither the comment nor the comment indicator at any time.
      
    'Variable Cleanup
    Set ws = Nothing
    Set cmt = Nothing
    Set shpPic = Nothing
     
End Sub

Sub InsertPictureAsComment()

  Dim SelectedFilePath As String
  
  SelectedFile = Application.GetOpenFilename(FileFilter=Images (.jpg), .jpg, MultiSelect=False)
  If SelectedFile = False Then End
 
  SelectedFilePath = SelectedFile
  
  ' Turn on Microsoft Shell Control and Automation object (Tools - Reference) and use 'ExtendedProperty' to grab the dimensions
  Dim objShell As New Shell
  Dim objFolder As Folder
  Dim objFile As ShellFolderItem
  Dim DimensionString As String
  
  Set objFolder = objShell.Namespace(FilePath(SelectedFilePath))
  Set objFile = objFolder.ParseName(FileNameWithExt(SelectedFilePath))
  
  DimensionString = objFile.ExtendedProperty(Dimensions) ' Sample output 800 x 533
  
  Dim DimensionStrings() As String
  DimensionStrings = Split(DimensionString,  x )
  
  Dim ImageWidth As Integer
  Dim ImageHeight As Integer
  
  ImageWidth = CInt(Right(DimensionStrings(0), Len(DimensionStrings(0)) - 1))
  ImageHeight = CInt(Left(DimensionStrings(1), Len(DimensionStrings(1)) - 1))
  
  Dim ImageScaleFactor As Single
  
  With ActiveCell
    .ClearComments
    .AddComment
    .Comment.Visible = False
    .Comment.Text Text=""
    .Comment.Shape.Fill.UserPicture (SelectedFilePath)
    .Comment.Shape.Fill.Visible = msoTrue
     
    .Comment.Shape.Height = ImageHeight
