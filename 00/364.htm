mike h.. say:

Hi,

This can go in A1 and it will start from APR-16 and you can drag down and it will give you all the months by rolling back to 1 when it reaches 12 for December.

=INDIRECT("'" & TEXT(DATE(2016,IF(ROW(A4)>12,ROW(A4)-12,ROW(A4)),1),"MMM" ) &"-16'!P43")

You asked how it works. It evaluates from the middle outwards.

This bit now returns a number starting at 4 which rolls back to one when it's greater than 12. It simply uses the ROW function to return a number so ROW(A4) returns 4

=IF(ROW(A4)>12,ROW(A4)-12,ROW(A4))

We then have this bit which use that number to create a date which for APRIL is 1 Apr 2016 etc

DATE(2016,IF(ROW(A4)>12,ROW(A4)-12,ROW(A4)),1)

All we want for your purposes is the month of that date so we then use the TEXT function to format that as simply Apr, May etc.

TEXT(DATE(2016,IF(ROW(A4)>12,ROW(A4)-12,ROW(A4)),1),"MMM" )

Lastly we use the INDIRECT function to turn that month into a cell address that Excel recognises.

=INDIRECT("'" & TEXT(DATE(2016,IF(ROW(A4)>12,ROW(A4)-12,ROW(A4)),1),"MMM" ) &"-16'!P43")

The 2 bits in bold are concatenated onto the month we previously generated and this gives a valid address that looks like this.

='APR-16'!P43

='MAY-16'!P43

etc

suzanna schuster say:

ESET and Defender cannot run on your machine at the same time.  Windows Defender, being installed by default with your operating system, will turn itself off if you install any other AntiVirus on the box.  (Running multiple AVs simultaneously can cause memory conflicts and performance issues.)

Because ESET is on your box, Defender is unable to turn itself on.  The reason you are getting the notifications is because at some point the machine doesn't believe that ESET is on and protecting your machine suitably.  This could be because it is taking awhile to turn on at startup, or because it isn't registering correctly with Windows 10.

You have two options moving forward: do you want to be protected by Windows Defender or by ESET?  If you want to be protected by Defender just uninstall ESET and Defender should turn itself on during or immediately after the uninstall process.

If you want to use ESET the first step to try and fix your problem is to see if you are up to date.  Make sure you have the latest version of ESET SS that has been released, since Windows 10 is a new OS older versions might not be fully compatible (although these versions likely should have been blocked or told to remove in upgrade.)

If your ESET is up to date, and fully on, and you're still seeing this prompt please look at "Security & Maintenance" in Control Panel, and see what is listed for your AntiVirus and AntiSpyware.
