Bing feature suggestions - Microsoft Community

haaklong say:

Bing feature suggestions

Bing Vs. Google

Recently I've compared Bing to Google again, the fact is that both have some strong points, and when I take the ''Bing Vs. Google'' test I often chose Bing 4 out of 5 times, Bing has more features than Google, and is more flexible, but Google has different features that in a lot of cases makes it more functional, Bing and Google have different features, and I couldn't include all of Bing's counterparts as not all are branded under Bing, but so far I'd still go with Bing, but I did learn that there are still many things Bing can improve on. :-)

Bing Search Finance

As Bing had recently introduced a physical calculator you can type on on Bing, maybe Bing should do the same with currency exchanges, simply type in ''Dollar to Euro'' and then you'll get a menu with all available currencies from the Bing Finance app on Windows Phone, so far this is awesome, but this could and should also be implemented in Bing Search, another example is the M.S.N. Money Web App, Microsoft's online currency converter, instead of putting a link to M.S.N. Money Bing could implement this feature in itself.

Another feature should be the ability to find stock market courses, for example if I type in ''Microsoft'' or ''MSTF'' I could immediately see how much the stock has de- or increased.


Calendar reminders

To users who have a Microsoft-account Bing could be personalised by including personal events or related content in search terms.

Bing Smart Search (Music option)

I wanted to suggest a feature for Bing Smart Search on Windows 8.1 (Blue), the ability to search Music, on Windows Phone we have Bing Audio which can listen to a song, and look for that song in Xbox Music's library, I wanted to suggest that feature for Bing Smart Search, something that would make this feature more innovative would be if it would also search for music clips, and other places where you can hear/purchase/download the song, giving YouTube-alternatives like Vimeo and DailyMotion a better chance, so far Xbox Music is already integrated in Bing Smart Search, but I'd love to see some additional functionality.

Bing Dictionary

Bing's dictionary feature is a bit lackluster, and typing in 'definition'' doesn't make it always easier, so far I like the feature, but here are a few suggestions, maybe putting it on the list of Bing features (like in Chinese Bing), also it only supports English and Simplified Chinese, the Bing Dictionary doesn't work at all in Dutch, for Bing to be more successful it should be more globally orientated, not just on the American or Chinese markets.

But thank you for this great feature, as I have no complaints on either the English or the Simplified-Chinese versions. :-)

Future feature suggestions

Here are a few feature suggestions that I wanted to make that I found while I was inspired by Google.

Book Search

In Books Bing could be like videos, simply searching the book stores of Amazon, Nook, Google, Apple, Etc. rather than creating own content as that is an expensive business, you could search by genre, fiction, non-fiction, similar like the classifications of the Windows Store on books.

App Search

The ability to search apps from all over the web, and Microsoft should launch an App Store for Desktop Apps as well, but without the restrictive policies of the Windows Store, and independent from the Windows Store, but that's just an idea that I don't place much faith in, also because 4.000.000 apps are A LOT to index, and I'm sure that there is a double amount of those ''fake apps'' that are in fact viruses, spyware, and other malware, but that's another story, by idea for app search is the ability to search apps in the Windows Store, Windows Phone Store, Google Plus, Apple's App Store, Windows Desktop Apps, Etc. and showing the ratings of those apps, and warnings if they are or aren't ''safe''.

Advanced Search

Similar to that of Google, personally I'd like the idea to filter more.

Search Tools

More filters for searching, and more hints on tips and tricks, for example ''Smartphone -iPhone'' is looking for all results but iPhones, yet most Bingers (people who search on Bing) don't know this, an easier way to find out how would be helpful, maybe a list similar to filtering through images would make it easier, Google has this on almost all search types, so far Bing has done an excellent job, but in case of helping consumers search Bing still has to do a lot of catching up to do, if like me you're a Bing-power-user and understand how to use these features than Bing is indeed better than Google, but for a novice user who doesn't understand search engines and shortcuts Bing isn't really intuitive. :-(

Here is the best thing I should write here, UserVoice, just like on Windows Phone & Xbox Music, Bing needs a UserVoice, so far the Feedback feature is great, but Bing needs a UserVoice website. 

(The feature I wanted to suggest in Bing Finance, better integration between Bing & the M.S.N. would be improving for both platforms and brands.)


Bing's G.U.I.

Something I also wish to suggest is having ''Metro boxes'' like on Windows & Windows Phone above in Bing's search categories, I think that iconography would be visually easier for people, we know what an image is, for the general web-search Bing could have a magnifying-glass, Bing Images the Photo logo from Windows, Bing Video's Xbox Video's logo, Bing News the same logo as the Bing News app, Etc. the M.S.N. and Outlook.com could also have their respective logo's, maybe even OneDrive and Microsoft Office Online.

This would also allow Microsoft to put more services on the top, mostly because icons take less space than words.
