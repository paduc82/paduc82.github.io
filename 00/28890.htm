pamcaswell say: Linking to previous means that the same content, including fields, is in the linked footer /header sections.  But the same page field may be formatted differently for each section—different number styles, different values, and restarts, and such.
If the previous sections was set to start a 1, a new section will also be set that way.  Go into format page number and set it to continuous.

ernmeister say:

Ok, I figured it out.  Had I shown the code in my original post I probably would have had everyone telling me what was wrong. Here is what the original code was.

    ThisWorkbook.Close SaveChanges:=True
    
    Application.Quit

I changed it to:

    ThisWorkbook.Save

    Application.Quit
