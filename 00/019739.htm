kaminc say: I actually figured it out myself, but I'll leave the answer for anybody who might run into the same issue:

When you use the =QUOTIENT( function in Excel, it will always return only the integer portion of the result.  When using division in your calculations, instead of writing your formulas like this:

=QUOTIENT(C7*100000,C43)

Try writing them like this:

=((C7*100000)/C43)

mukarakaplan say: Here is an easy solution:

1- Insert a new date picker content control from the Developer Tab.
2- Select the Design Mode in the Developer Tab.
3- Delete the "Click here to enter a date." default text.
4- From the Insert tab, select Insert Date and Time.
5- Select the style of the default today's date style and check the Update automatically box.
6- Right click the date picker content control and select properties.
7- Select the matching date style for the date picker.
8- Turn of the Design Mode in the Developer Tab.

Now your date picker content control box will display today's date by default. So every time you open your document, that box will display today's date and you'll have the option to click the date picker and change the date to an earlier or later date.

carlos huang msft say:

Hi Gerg,

Since you can access the group conversations via Outlook client, here is the details steps to export the group conversations.

  1. Open Outlook and choose your group. 
  2. Use the Shift+ or Ctrl+ to select multiple conversations you want to export. 
  3. Then click the File on the upper left, click Save As. 
  4. Choose the location and type the file name.
  5. Click Save to export the conversations.

If you encounter any issue when you follow the procedure, please feel free to tell me.

  • Re-launch Skype.
