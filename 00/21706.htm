How can i get Excel to automatically convert multiple unit types to - Microsoft Community

mrimpossible say:

How can i get Excel to automatically convert multiple unit types to lbs?

I need a column within a table to convert all measurements to lbs.  Measurements are given in lbs oz and grams, so i want to find a way to have a column normalize all inputs into lbs and still be able to sum the results.  Ex. 256g would become 0.56 and 256oz would become 16.  I'm assuming that this will have to be done in VBA and my experience with VBA is extremely limited.  Thanks in advance for any help you can render. 

bernie deitrick say:

If you have entries in, say, cells A2:A4 like

256g

256oz

256lb

1) Using formulas - use this formula in a cell in row 2:


=IF(RIGHT(A2,1)="g",VALUE(LEFT(A2,LEN(A2)-1))/453.59,IF(RIGHT(A2,2)="oz",VALUE(LEFT(A2,LEN(A2)-2))/16,IF(RIGHT(A2,2)="lb",VALUE(LEFT(A2,LEN(A2)-2)),"")))

and copy down to match your data cells.  Then you can copy the formulas, and paste formats and values over the original values, and delete your formulas.

2) Use this macro - select a single cell in the column with the values, and run this macro:

Sub Convert2Pounds()
    Dim rngC As Range
    For Each rngC In Intersect(ActiveCell.EntireColumn, ActiveSheet.UsedRange)
        rngC.NumberFormat = "0.00"
        If LCase(Right(rngC.Value, 1)) = "g" Then rngC.Value = Val(Replace(LCase(rngC.Value), "g", "")) / 453.59
        If LCase(Right(rngC.Value, 2)) = "oz" Then rngC.Value = Val(Replace(LCase(rngC.Value), "oz", "")) / 16
        If LCase(Right(rngC.Value, 2)) = "lb" Then rngC.Value = Val(Replace(LCase(rngC.Value), "lb", ""))
    Next rngC
End Sub

In Access 2010, I want to create a toggle control on a form to either hide or unhide the Navigation Pane.

How do I determine if the Navigation Pane is hidden or not?
jeanette cunningham say:

Here is a way to toggle it.

Put a hidden check box on the form. Call it chkHidden.

 

Here's the code:

On the form's load event put

Me.chkHidden = False

 

On the button's click event put

 

Private Sub [ButtonName]_Click()
If Me.chkHidden = False Then
    DoCmd.NavigateTo "acNavigationCategoryObjectType"
    DoCmd.RunCommand acCmdWindowHide
    Me.chkHidden = True
Else
    DoCmd.SelectObject acTable, , True
    Me.chkHidden = False
End If
End Sub

When you "add Favorites to the Navigation pane" (renamed as Quick Access in Windows 10) and pin them there, they show up in the Open and Save dialogs of Office programs. lemain say: I've found the answer so for the benefit of those who follow....

In the Nokia 6310i there are TWO places where the GPRS modem needs to be specified. Under "Settings" and under "Services" All of the internet and data stuff is carried out under Services and the modem was correctly specified in there but on the off-chance I re-typed the designated name under Settings > GPRS modem settings > Edit acess point and I changed the name for Access Point 5 (an arbitrary one) to 'internet' which was the correct designation for the Yoigo service. Then the thing worked perfectly.
