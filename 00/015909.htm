melcom99 say: Okay, problem solved:
The ck-20w is not designed for using the stereo line out together with the mute function.

Either you use the mute function in mono mode or you use the stereo mode for a aux in radio input and then you have to select the the source at your radio when you want to here musik.

So if you want to use your phone for navigation over the car speakers you have to use your ck-20w in mono mode.
