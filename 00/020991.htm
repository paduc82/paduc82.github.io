mdawnm25 say:

Word 2007 documents have incorrect icon

My MS Word 2007 documents do not have the correct icon.  They have a blank sheet with lines and DOCX at the bottom.  How do I fix this?  My operating system is Windows 7.
jeeped say:
Sorry to here that my proposed solution did not work. There is an posted answer here that deals with your problem and seems more specific to your situation. This involves a registry rewrite on the entry for the icon associated with DOCX. Note that it does contain some hard coded paths that are typical of a default Windows installation and may have to be changed if your installation is non-standard.

jeeped say:
There could be a minor corruption in the cache file that stores commonly used icons in a quickly referenced storage file. Windows holds copies of the icons commonly used on your desktop and other places in a configuration file called IconCache.db. It has likely suffered a minor corruption and needs to be rebuilt.

Close all open windows and click Start then type or paste in  explorer but do not hit <enter>. In the list above, right click on Windows Explorer and select Run as Administrator. Acknowledge the UAC warning and provide credentials if required. When Explorer opens, navigate to,
c:\Users\<your username>\AppData\Local

If you do not have Show hidden files, folders and drives turned on, turn it on now by tapping the ALT key, then selecting Tools, Folder Options. On the View tab's Advanced Settings, check Show hidden files, folders and drives.

There should be a file called IconCache.db listed (e.g. IconCache  <Data Base File>). Rename it or delete it.
Restarting the computer will ask Windows 7 to rebuild the IconCache.db file.
