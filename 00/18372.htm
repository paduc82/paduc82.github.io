    To repeat (copy) the formatting of a selection of text to another selection of text, the shortcuts Shift+Ctrl+C and Shift+Ctrl+V can be used, for copying and pasting the formatting respectively.

    Is there anything similar to repeat the formatting of a picture inserted in a document, with everything that it entails, e.g. picture size, borders, alignment etc?

    jay freedman say:

    No, there is no built-in single "repeat picture formatting" command.

    If you record a macro while you format the first picture, you can then select the next and run the macro to reapply the whole sequence of commands. If you know (or are willing to learn) some VBA language, you can modify the recorded macro so it will format all the pictures in the document in one go.
