Formulas:

B2:  =AVERAGE(C2:D2)

E2: =A2*$B$14 + $C$14

B14:C14: =LINEST(B2:B11,A2:A11)

D14: =RSQ(B2:B11,E2:E11)

The formula in B14:C14 is array-entered:  select the two cells, type the formula, then press ctrl+shift+Enter instead of just Enter.  The formula is displayed with surrounding curly brackets.  We cannot type the curly brackets ourselves; that is just the way that Excel indicates that a formula was array-entered.

In actual use, you might discover that the formula does not exactly match measured data.  That is not unusual.  For example, according to the Am Diabetes Assoc, the following formulas relate A1c and blood glucose numbers:

est glucose = 28.7*A1c - 46.7

est A1c = (glucose + 46.7) / 28.7

A friend's actual measurements were 161 glucose and 6.9 A1c.  But based on 161, est A1c is 7.2; and based on 6.9, est glucose is 151.  Note that neither matches actual measurements; and the difference cannot be explained by rounding error.  That's just the nature of statistical estimates.

[EDIT] For that particular example, it might be noted that an A1c measurement reflects the average blood glucose over 3 months or so (it's just the nature of the protein), whereas the glucose measurement above is for the instantaneous moment.  The est A1c formula is intended for an average glucose measurement. But my statistical point is still valid.)

There are two Startup locations. The one under Program Files is actually the less convenient of the two, because you have to give Administrator approval to save files there. It exists primarily for the convenience of program installers for Office add-ins.

The preferred location is a hidden folder within your user profile. To get there, paste this path into the address bar of the file manager and press Enter:

%appdata%\Microsoft\Word\Startup

Also note that you could (although I don't recommend it) change the location used for the Startup folder to anywhere on your disk. In the Advanced page of the Word Options dialog, at the bottom, you would click the File Locations button; in the next dialog, select the Startup item and click Modify to browse to the desired spot. Since the Startup folder is used mainly for templates that contain macros, you would also have to go to the Trust Center and make the new location a Trusted Location to allow the macros to run.
