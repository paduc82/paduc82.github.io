I can't think of a way to do this when scrolling since there are no Excel events associated with scrolling.

You could have a Worksheet_SelectionChange event procedure that would change the header row according to where the cell you selected was.  Would that suffice?

This assumes the window is split (but not frozen) with one row above the split

Private Sub Worksheet_SelectionChange(ByVal Target As Range)
  Dim lHeadRow As Long
  If Target.Row >= Me.Range("Head2").Row Then
    lHeadRow = Me.Range("Head2").Row
  Else
    lHeadRow = Me.Range("Head1").Row
  End If
  If lHeadRow <> ActiveWindow.Panes(1).ScrollRow Then
    ActiveWindow.Panes(1).ScrollRow = lHeadRow
  End If
End Sub

Stephanie, I don't know if you are happy or not with Bill's solution so here is another.

 

Sub SwitchHeadings()
  'Headers of two ranges on row 1 and row 100
  'switch between these two range headers when run
  'depending on the row of the active cell


  Dim Hdr1 As Long, Hdr2 As Long
 
  Hdr1 = 1
  Hdr2 = 100

  'make sure you are on the right sheet
  'if this macro is activated from QAT
  'or keyboard shortcut
  Sheets("Sheet3").Select 'change to suit
 
  'remove freeze panes from sheet
  ActiveWindow.FreezePanes = False
     
  With ActiveWindow
    If ActiveCell.Row <= Hdr2 Then
      Cells(2, 1).Select
      ActiveWindow.ScrollRow = Hdr1
      ActiveWindow.FreezePanes = True
     
    ElseIf ActiveCell.Row >= Hdr2 Then
      Cells(101, 1).Select
      ActiveWindow.ScrollRow = Hdr2
      ActiveWindow.FreezePanes = True
   
    End If
  End With
 
End Sub

jay freedman say:

The shortcut to the macro could have been reassigned to some other command or removed, possibly by an add-in. You can reassign it manually to the macro.

Open the Options dialog and go to the Customize Ribbon page. Click the Keyboard shortcuts Customize button at the bottom left to open the Customize Keyboard dialog. Scroll the Categories list to the bottom, and click on Macros.

The list on the right now shows the names of all the macros contained in the file selected in the "Save changes in" dropdown (which you may need to change, depending on where the macro is stored).

Click on the name of your macro and look at the Current Keys box. If it doesn't show the shortcut you had before, click in the "Press new shortcut key" box and press the shortcut (don't try to spell it out, just press the intended shortcut).

If the shortcut has been reassigned to another command, that command will be shown below the Current Keys list; if it was removed, that space will show "[unassigned]". In either case, click the Assign button to reassign that shortcut to the macro.

If the Current Keys list does show your intended shortcut but that shortcut still doesn't run the macro, it's possible that some other program outside of Word has hijacked the shortcut. Investigate any program that was installed or reconfigured about the time the shortcut stopped working.

You are right, you can't do this by applying a style.
