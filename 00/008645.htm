@NeeraleePatel;

This level of archive corruption is usually due to working directly from a thumb drive or other removable media. If you do not use the Safely remove hardware utility, the read/write buffers may not be cleared before you remove the thumb drive (or other device). This is true even if you have saved and exited Word. Working directly from and email attachment is also a possible cause, but the removable media explanation is by far the most common.

With that said, I can often recover the DOCX up to the point of the last editing session by repairing the archive structure and reconstituting the word/document.xml configuration file. Typically everything beyond that point is scrubbed with programming NULLs.

If you would like me or someone else to attempt repair on your document, you will need to upload a copy of your damaged DOCX to a public file area like OneDrive, DropBox  or Google Drive and post the location to the publicly shared file back here.

Please try to upload a copy that has not already been subjected to repair attempts as previous attempts may have stripped content that was recoverable with a manual edit of the internal components configuration files.

don varnau say:

Hi,

It has been my experience that IE will eventually (perhaps for no visible reason) sort

Favorites by name. I've never seen a way to prevent this.

Workarounds:

1. Create enough sub-folders in Favorites that all Favorites will fit on the screen and the renaming is less of a problem.

2. . Rename individual Internet Shortcuts so that sort by name works for you.

Example: 1yahoo.com will  be listed before bing.com. Or aYahoo.com before bBing.com.

Preventive measure:

After organizing Favorites in your preferred order, export one or both of these registry keys every day. Double-clicking (merging) either key will

restore the value (sort order) when the key was exported.

Favorites:

HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\MenuOrder\Favorites

Favorites Bar (Links)

HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\MenuOrder\Favorites\Favorites Bar

Exporting those registry keys:

 From Start> Run (or Windows logo key + r) type in REGEDIT and hit Enter.

Export/backup the entire registry (for safety)

In Registry Editor select Computer in the left pane then File> Export. Save that REG file in a convenient location. Double-clicking that REG file will restore the previous registry and undo any changes, if necessary.

 
Then:

Drill down, by clicking the + to

HKEY_CURRENT_USER

   Software

      Microsoft

         Windows

            CurrentVersion

               Explorer

                  MenuOrder

                     Favorites 
Select Favorites, right-click> Export.

Repeat for Favorites Bar.

 
Exit Registry Editor.

Don

For the first question, the Quarantined Devices in the mobile section is used to configure Exchange ActiveSync access settings (including Allow, Block, or Quarantine) for synchronizing data from a mailbox stored on an Exchange server. This feature was designed to help IT organizations control which of the growing number of Exchange ActiveSync-enabled devices are allowed to connect to their Exchange Servers. With this feature, organizations can choose which devices (or families of devices) can connect using Exchange ActiveSync (and conversely, which are blocked or quarantined).

For the second question, if you block one of these Quarantined Devices, it means the account canâ€™t be configured via ActiveSync and the user will receive HTTP 403 Forbidden errors. Whatâ€™s more, the user will receive an e-mail message from the Exchange server saying that the mobile device is blocked from accessing the mailbox. However, the user still can configure accounts via IMAP or POP.
