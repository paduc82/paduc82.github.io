gongshow say:

Is there a way to change font color in microsoft word 2013 using a keyboard shortcut?

Is there a way to change the font color from black to red (for example) when the text is highlighted using a keyboard shortcut instead of Is there a way to change the font color from black to red (for example) when the text is highlighted using a keyboard shortcut instead of clicking the color change button everytime?
suzanne s. barnhill say:

You could define a character style whose only property is the designated font color and then assign a keyboard shortcut to that style. Note, however, that a character style based (as it should be) on Default Paragraph Font will remove any other direct formatting (such as bold or italic) that is applied to the selected text.

If you'll be doing a lot of text in one run, a simpler solution is to use the shortcuts for Copy/Paste Formatting. Apply the red font color to selected text and press Ctrl+Shift+C to copy the formatting and then Ctrl+Shift+V to paste the formatting on any other selected text. The "formatting Clipboard" will retain the selected formatting until you change it or quit Word, so you can continue to paste it throughout a Word session, regardless of other intervening actions.
