a. user say:

Since the time of my original question a lot seems to have changed. There are still no parental controls per se in Outlook.com, but Microsoft has made it possible for parents to exercise a substantial amount of authority over what their children can do or see on Microsoft properties.

The key, as I discovered, is to sign up your child for a Microsoft account using their real birthday. When the software detects that the person is a child (within the meaning of COPPA), a whole slew of protections kick into place.

To start, the child's Microsoft account must be linked to a parent's Microsoft account. (Technically, it can be any adult.) The 'parent' must prove they're an adult by allowing Microsoft to make a small charge to their credit card (50 cents, goes to charity.)

Once this is done, the parent's Microsoft account exercises a great deal of control over the linked child's Microsoft account. For example, the parent controls what games the child can play on Xbox Live, what music they listen to on Zune, whom they are allowed to have contact with via text, voice, video and chat, and more. I haven't explored all the features yet, but I think the parent can limit the child's email activity on Outlook.com to people in the contact list and Safe Senders list. All other mail - including spam - can be processed with rules, so the child doesn't even see it unless the parent lets it through.

And these protections don't automatically expire when your child turns 13. Only the parent can decide when to 'promote' their child's account to adult status. Smart!

An even greater degree of control is possible if you install Microsoft Family Safety on the child's account. We chose not to do this with our son since we use third-party software that, quite frankly, does a better job for our needs than Family Safety.

It is very much to Microsoft's credit that they have included these features with a Microsoft account. Nothing like this on Google or Yahoo. Our son signs in to Windows 8 with his Microsoft account credentials, and that puts us more at ease.

swischner say:
The folder that Vijay-Verma pointed you to is the default path for locally stored notebooks, which can only be done by OneNote 2016, which is installed with MS Office 2016. From Microsofts view the ability to store notebooks on a local drive is a kind of "leftover" from older OneNotes, Since 2014 they declared OneNote "cloud only" software, which means that all other versions (iOS, Mac, and even the Windows 10 OneNote app) can only store notebooks on OneDrive or OneDrive for business. There also is no backup mechanism in the app and the notebooks on OneDrive are hidden from direct access, so you are not able to even manually download them from there. Microsoft claims you won't need a backup anyway, because everything on OneDrive is much safer than on your PC. Well, this view is at least.... err.... discussable.

You might think that there MUST be a local copy of the notebooks on your device because you can access and edit your notes while not connected to the internet. Yes, there sort of is. It's called the OneNote cache and only usable by the app itself (the notes are stored in a split up, binary format, for sync purposes only. Copying those hidden cache files would not serve as a usable backup in any way).

Time for somewhat good news:
If you (like me) are somewhat getting nervous about not having a backup, there are in fact two ways to get a backup which is restorable (to restore those backups you would absolutely need OneNote 2016 from MS Office though!):

1. If you install OneNote 2016 (in addition to your Windows 10 app), it has a built in backup function that copies opened notebooks to a local folder / drive. The options are found under "Options", "Save & Backup". Be sure to have "Back up notes stored on SharePoint" checked, as this will also backup notes from the cloud, if they have been opened in OneNote 2016.

2. If your notebooks on OneDrive are not stored in the main folder but in a subdirectory. you can download that complete directory (not the separate notebook "files" which are just placeholders without actual content). This will result in a ZIP file that also contains the "real" notebook files in a format that OneNote 2016 (the version with MS Office) can handle in case of emergency. For all of this ignore the local OneDrive client (your OneDrive folder on your device) and use the web browser instead.

Summarized: Microsoft does not give you an official method to back up notebooks stored on OneDrive other than the "leftover" function in OneNote 2016 from MS Office. If you use that option for backups (or the workaround I showed you), the restoring of the notes absolutely need a "full" OneNote 2016 from MS Ofifce.

Create a command button on the form.

Make sure the command button is selected.

In the Format tab of the Property pane, set its Caption property to "STOP or DOWN".

In the Other tab, set its Name property to cmdStop.

In the Event tab, click in the On Click event.

Select [Event Procedure] from the dropdown list in this event.

Click the button with three dots ... to the right of the dropdown arrow.

Make the code in the module that appears look like this:

Private Sub cmdStop_Click()
    Me.[Stop Time] = Now
End Sub

where Stop Time is the actual name of the stop time text box/field.

Switch back to Access, and create a start button the same way.

cbarnhorst say:

You can do a repair install using the Media Creation Tool at

https://www.microsoft.com/en-gb/software-download/windows10

Make a usb or dvd and with Windows 10 running insert the media.  Click on Install Now.  Do NOT restart the computer with the media inserted.

Choose "Keep Everything" when asked.  This will completely reinstall Windows without disturbing your personal stuff and apps.  You will then be on the public release and Get Insider Builds will be turned off.

It's likely that is just how the images are stored.  Some apps will allow you to rotate them, but they don't always store the rotation data in the photo itself and Sway does not have access to that data.  You can fix this by opening the photos in a photo editing app, like MS Paint, and then rotating and saving them before putting them into your Sway.
