Activation depends on what type of license you have of Windows 7/8/10.

a. user say:

Augustinus Shiyengo,
This error occurs if your computer was purchased from an OEM and the computer uses an ACPI_SLIC table in the computer BIOS program.  An ACPI_SLIC table is used by an Advanced Configuration and Power Interface (ACPI)-compliant BIOS program to store Software Licensing description information.  This problem occurs if the KMS server does not find a valid Windows marker in the ACPI_SLIC table in the computer's BIOS program. This problem occurs if you purchased a computer that has a qualifying Windows operating system installed. However, the Windows marker in the ACPI_SLIC table is corrupted. To resolve this problem, update or replace the BIOS program by using a BIOS program that contains a valid Windows marker. To do this, you must contact the OEM (manufacturer) who supplied the computer.

Your computer manufacturer should supply you with the information you need to correct this issue.  If you need further assistance please let us know.
