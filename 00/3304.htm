john korchok say:

Avery label templates are laid out using a table. Because you're having trouble with logo positioning, I'm guessing that the images are floating above the text layer. You be able to get a more reliable arrangement if you make the logos inline with the text. Then they'll sit in each table cell. Please try this:

  1. Click on a logo to select it.
  2. On the Picture Tools>Format tab, click on Position and select the top option In Line with Text. The logo pops into position on the page. If that position is not inside one of the table cells, copy and paste the logo into a cell.
  3. Inline images act like text characters, so use text formatting commands to position it. As examples, use Home>Paragraph>Alignment>Centered to center the logo in the cell, or use Home>Paragraph>Spacing>Before and add 10 points to create some space above the logo.
  4. Repeat for all logos, applying the same measurements to each logo. When you're done, they should site in a neat grid and they won't accidentally move.

john korchok say:

If you have a Word file that just contains 2 copies of the PDF beside each other, you can send the Word file. There is nothing that is going to be rendered differently at the other end, because all formatting is embedded in the PDFs.

You'll get the most accurate result if you size the the PDF to A5 numerically. After using Insert>Object to place the PDF on the page, right-click on it and choose Format Object. On the size tab, set the Height to 210 mm or 8.3 inches and the Width to 148 mm or 5.8 inches.

For positioning the PDF on the page, you can do that 2 ways:

If the PDF is inline with text (image hugs the margins, you can't move it with your mouse), set the Word Page Layout>Margins to 0" for all 4 margins. Then you should just be able to fit 2 PDFs on a landscape A4 page.

If the PDF is floating (you can select it and drag it to any position on the page), then right-click on the PDF and choose Format Object. Select the Layout tab, then click on the Advanced button. On the Picture Position tab, Set Horizontal to 0" to the right of Page and Vertical to 0" below Page. Then for the second PDF, do the same, but set Horizontal to 148 mm or 5.8 inches to the right of page. The 2 PDFs should completely fill the page with no overlap.
