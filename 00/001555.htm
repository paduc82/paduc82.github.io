daniel g. grau say:

It all depends on how you create Book II.

If you make a copy of Book I (in the Finder or via Save As), then delete its entire contents, you'll still have your styles because they were part of the document.

If you create Book II from a blank document, you'll only have the styles as defined in whatever template the document was based on (typically, Normal.dotm). However, if you paste in there parts of Book I that have styles applied, those styles will be ported over to the blank document, even if you later delete the associated text. IOW, the new styles become part of the document.

If your target document already contains a like-named style as the text you're pasting into it, the style will be kept as is (i.e., as defined in the target document) and the pasted text will change the way it looks to reflect the pre-existing style.

If you already have text in Book II, you can copy the styles from Book I. For that, open both documents, then access Tools > Macros > Macro > Organizer in the main menu and select the Styles tab at the top.

There is a way to make any style available from now on in all your new documents:

  • Edit the style by selecting Format > Style… in Word's main menu
  • Highlight a style
  • Click on the Modify button
  • Place a checkmark in "Add to template"

steve rindsberg say:
Unless the projector has built-in intelligence like the one you cite, I don't think there'll be any difference in power consumption between dark/light slides.  Projectors have a constant light source that's filtered to get different colors/brightness levels on screen.  

Imagine window-blinds.  The sun's not getting dimmer when you gradually close them; you're just letting less sunlight in.  A projector functions analogously.

If you know that you'll be using a projector with this ECO mode feature, you could presumably save power by using a darker background.   You'd need to use your own judgement and observe the on-screen image from the specific projector under the specific projection conditions in order to make any reasonable judgement as to what presentation designs work best.

erica blair say:

Hi NY,

One way to do this (and keep the ratios of the current spread across the months) is to do the following:

For each monthly value, have another cell in which the monthly value is divided by the total (aka the percent of the total that value is)

Then you can have a cell in which you enter the goal ($276M in your case).

Then adjacent to each of the above calculated percentages you can place a formula that multiplies the percentage times the goal to get the required value for that month based on the same monthly ratios as the actual or entered data.

Manage Sources

Using Manage Sources, then Edit Source, I am having a problem in listing religious authors.  Most religious authors have a suffix. For example: Rev. Thomas Smith, O.Carm.  When I generate a Bibliography, Word sees the comma as the last name, first name comma and jumbles everything  I tried using hard spaces to no avail.

The problem entry is: Smith, O.Carm., Rev. Thomas

How can I get Word to recognize the comma for the suffix is different from the comma for last name/first name?

Joe
joe phelps say: I have on my own figured out how to insert suffixes on names if you want. Suffixes are, for example, Jr., J.D., O. Carm., etc.  First, in Manage Sources, do not manually type the name in the name line.  If you did it would look like Smith, O.Carm., Rev. John.  This method will not work. 

Instead, always, always use the Edit button   On the Last line type the Last Name, then comma, the hard space, then the suffiix. So the last line will be:  Smith,(Ctrl-Shift [together]space bar) O.Carm. Remember to use a hard space between the comma and suffix

Use the same idea to keep a title with a first name. On the First line, type Rev.(hard space)John. It could also be used for Dr., Pope, etc.  

Then select Add, then OK. Voila!  Your biblio is perfect.
