The solutions that work for me are:

1. In the T61's, I installed an Intel Ultimate-N 6300AGN Wifi card that was $12 on E-bay (includes 1/2 high adapter.  (Note: newer cards have a smaller form factor that will not fit the LAN card socket).

  a. Before flashing the BIOS, remove the old Intel 4965AGN WiFi card.

  b. Enter the BIOS with F1.

  c. Ensure that:

     i. Network -> Flash over LAN - Enabled (otherwise there will be an error message).

     ii. Network -> Ethernet LAN Option ROM - Enabled

     iii. Security -> BIOS update option -> Flash BIOS by End-Users - Enabled.

  d. Insert the bootable Middleton BIOS Mod 2.29 into the DVD/CD drive.

  e. Reboot and enter the Startup Menu with F12.

  f. Select the DVD/CD drive if not the default.

  g. The bootable CD immediately show a command line prompt; several commands will execute.

  h. Unlike the usual BIOS versions from Lenovo, there are no Install Options.  The BIOS will take several minutes to complete.  During that time, you may hear one or two "beeps", the same as you might hear for a POST error.  This is normal behavior.  At the end of the  flash, the word "DONE" appears.

  i. Shut the system down; remove the AC power supply and battery.

  j. Install the new WiFi card.  (Note: if you attempt to install the new card before flashing the modded BIOS, you will get a 1089 error.  You *cannot* get past this with the stock BIOS as the new card is *not* on the "whitelist".

  k. Windows 8 and 10 will use Windows Update to find and install the WiFi card drivers.

END RESULTS:

  a. The new card actually is more sensitive than the old one, allowing me to work farther from the Access Point.

  b. The new card does not have the problems of the old one - often having problems authenticating that required Disabling then Enabling the WiFi card.

 c. I did not benchmark the harddrive before and after the BIOS flash (I do get an device indication that it is SATA 2 drive); "anecdotally", I can't see much difference in drive performance before and after the flash.  If you care, benchmark the harddive before the  flash and after.

In the T60, I installed an IBM/Lenovo ThinkPad 11a/b/g/n FRU 42T0825 (an Atheros AR5008 5418 AR5BXB72) WiFi card that was $15 on E-bay.

  a. The Intel 3945ABG uses only two wires (black and white).

  b. The new card has terminal for three wires.

  c. Others have found that only connecting the two wires gives reasonable performance (I tried to find the 3rd antenna with FRU 42R9909 - it's not available).

  d. Windows 8 and 10 found the drivers without difficult.

END RESULTS:

  a. The new card actually is more sensitive than the old one, allowing me to work farther from the Access Point (with only the two wires, not three).
