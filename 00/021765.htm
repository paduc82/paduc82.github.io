  1. Draw a Shape the exact size and shape as the image you want to be transparent.
  2. Move it off to the side.
  3. Right click on the image and choose Save as Picture (choose jpg or png)
  4. Delete the image you just saved as a picture
  5. Take the shape you created in Step 1 and place it where you want it on the slide. Right click on the shape, Choose Format Shape to open the Format Shape dialog
  6. Click Fill > Picture or Texture Fill (the title of the dialog title will now change to "Format Picture")
  7. Click  Insert -- from File
  8. Browse to the saved picture from Step 3 above, Select it then click Insert 
  9. Before you hit Close in the Format Picture dialog, be sure to adjust the Transparency setting near the bottom of the dialog box

Note: it's best to place draw the shape in the Slide Master (View > Slide Master in the Ribbon), but you can repeat this step on every slide if you don't know how to work with masters.

I hope this helps.

Sandy

hansv mvp say:

We enter numbers in the decimal system; Excel stores numbers in the binary system. Because Excel uses a finite precision, tiny rounding errors can occur when numbers are translated from decimal to binary and back.

If you increase the number of decimal places in A3, you'll see

-0.0649999999999999

You can get round this by using the ROUND function:

=ROUND(A1-A2,3)

About setting criteria in an append query - Microsoft Community

chackra say:

About setting criteria in an append query

Hi

I want to append records form a table rows including PK field data to another table through a cmd button to the condition that

whenever I run the append query it should append only the new records added in the source table since the previous run.

How to set criteria for that.

Thank you. 

hansv mvp say:

Set the following condition in the Criteria row of the primary key column:

Not In (SELECT [PK] FROM [TargetTable])

where PK is the name of the primary key field, and TargetTable is the name of the table you want to append records to.
