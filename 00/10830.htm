You don't export at all.

All your Contacts (and all your other Outlook data as well - email, calendar tasks etc) are held in a single file - normally called Outlook.pst.

Copy that file (with Outlook closed), and paste it into your Documents folder on the new computer.

Open Outlook 2013 and o File-Open-Outlook data File.

Navigate to where you put the pst file.

You can then either set the old pst file as the default, r copy the data from the old file to the new.

To copy Contacts from Outlook 2010 on a Win-7 Desktop
to Outlook 2013 on a Win-7 Laptop

 

 

On the desktop computer

  1. Open Outlook / Contacts
  2. On the File tab, click Options
  3. On the resulting menu, click Advanced
  4. On the resulting window, click Export
  5. On the resulting Import and Export Wizard, select Export to a file, and click Next
  6. For Create a file of type:, select Outlook Data File (.pst)
  7. For Select folder to export from:, select Contacts
  8. For Save exported file as:, assign a filename (e.g., MyContacts) & browse for a location, (best choice being a thumb drive)
  9. Click OK, Next, Finish

 

On the laptop computer

  2. On the File tab, click Options
  3. On the resulting menu, click Advanced
  4. On the resulting window, click Export (Button is labeled Export, but leads to both)
  5. On the resulting Import and Export Wizard, select Import from another program or file, and click Next
  6. For Select file type to support from:, select Outlook Data File (.pst)
  7. Browse to find file to import (It will be on the thumb drive above, now moved over to the laptop) and click Next
