stefan blom say:

Word tries to guess which language you are using and may change it at the insertion point. To prevent that from happening, clear the "Detect language automatically" option in the Language dialog box (Review | Language | Set Proofing Language).

Note that once incorrect language formatting is present in a document, you may have no other choice than to select the text and change it manually, again via the Language dialog box.
