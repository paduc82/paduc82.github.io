I don't know who your is.

 

Your error is a COM HResult containing a Windows error.
 
To decode 0x80070422. The 0x means it's a hexadecimal number, the 8 means error, the first 7 means it a windows error, and the rest of the number, 422, is the actual Windows error.
 
To look up the error we need it in decimal format. Start Calculator (Start - All Programs - Accessories - Calculator) and choose View menu - Scientific, then View menu - Hex. Enter 422. Then View menu -  Decimal. It will say 1058.
 
Start a Command Prompt (Start - All Programs - Accessories - Command Prompt) and type
 
net helpmsg 1058
 
and it will say
 
C:\Users\David Candy>NET Helpmsg 1058
 
The service cannot be started, either because it is disabled or because it has no enabled devices associated with it.
 
Click Start - All Programs - Accessories - Right click Command Prompt and choose Run As Administrator. Type
 
sc start DCOMLaunch
sc start RpcSs
sc config wuauserv start= auto
sc start wuauserv
sc config trustedinstaller start= demand
sc start trustedinstaller
sc config COMSysApp start= demand
sc start COMSysApp
sc config BITS start= demand
sc start BITS
sc config spldr start= boot
sc start spldr
sc config slsvc start= auto
sc start slsvc
sc config EventLog start= auto
sc start EventLog
 
 
Right click the window and choose Mark. Select the output. Press Enter. Paste into your reply.
