Changing the partition configuration after install should have no implications on performance as partitions have to be contiguous space on the drive.

 

There are 2 ways to check the layout of the drive. Start Disk manager (diskmgmt.msc) and in the lower pane make sure the two active drives are each a single block. There may be a single small partition at the beginning of the drive called System Reserved which is not visible in Explorer.

 

Alternative Run Diskpart and after accepting the UAC warning enter list volume to see how the disk is set up.
