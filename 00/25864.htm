Custom TOC in Word 2011 - Microsoft Community

marimacadmin say:

Custom TOC in Word 2011

In Ms Word 2011, I have created a Table of Contents.  How do I add a hyperlink to the TOC item instead of the page number?  I've tried the \h and it doesn't work.  Also when I toggle field codes, I only see the code for the first level no matter where my cursor is.

a. user say:

I thought I'd at least have a lok at a VBA macro, and making a few assumptions, came up witht he following. Probably worth a try, but not putting too much effort in if it doesn't function properly. I can put in a bit more effort if you identify any easily located problems, but not a lot.

It assumes that you have a TOC field that generates a page number, and that the separator between the ToC entry text and the page number is a single character (I've actually assumed it's a tab). Youalso need a couple of blank paragraphs after the ToC. I last attempted Kindle book generation a couple of years ago, so will have to leave you to check that part.

Sub insertMyOwnToCBookmarks()
' Copy a ToC with page numbers and convert it into a numberless
' hyperlinked ToC for use with Kindle
' For Mac Word 2011
' Peter Jamieson 2014-09
' Thrown together and hardly tested!

' Before running this,
' make sure that the ToC generates page numbers, and generate the ToC
' There also need to be a couple of empty paragraphs after the ToC
' Each page number should be created using a { PAGEREF } field
' that references a specific bookmark in the document

' After running it, hiding the original ToC may cause problems when generating for
' a Kindle. In that case, I suggest we modify the original TOC field, perhaps by putting
' "ADDIN " in front of "TOC", then regenerating the field result

' the start of the bookmark name in a PAGEREF field generated by Word in a ToC
Const intBookmarkNameStart As Integer = 10
' We will bookmark our dupicate "ToC"
Const strMyToCBookmarkName As String = "MyToC"

Dim f As Word.Field
Dim fHyperLink As Word.Field
Dim i As Integer
Dim rPara As Word.Range
Dim rToC As Word.Range
Dim rOldToC As Word.Range
Dim strBookmarkName As String
Dim strToCText As String
' get the first ToC (you may need to modify this if
' you also have tables of figures etc.
For Each f In ActiveDocument.Content.Fields
  If f.Type = wdFieldTOC Then
    Set rOldToC = f.Result
    Exit For
  End If
Next
If (rOldToC Is Nothing) Then
  MsgBox "No ToC fields found in this document"
Else
  ' remove any existing copy
  If ActiveDocument.Bookmarks.Exists(strMyToCBookmarkName) Then
    Set rToC = ActiveDocument.Bookmarks(strMyToCBookmarkName).Range
    rToC.Text = ""
  Else ' we'll create a new "Toc" after the existing one
    Set rToC = rOldToC.Duplicate
    rToC.Collapse wdCollapseEnd
    rToC.MoveStart unit:=wdCharacter, Count:=2
  End If
  ' duplicate the ToC result
  rToC.FormattedText = rOldToC.FormattedText

  ' iterate through the PAGEREF fields from the end of the ToC result
  For i = rToC.Fields.Count To 1 Step -1
    Set f = rToC.Fields(i)
    If f.Type = wdFieldPageRef Then
      ' extract the bookmark name
      strBookmarkName = Mid(f.Code, intBookmarkNameStart, InStr(intBookmarkNameStart, f.Code & " ", " ") - intBookmarkNameStart)
      Set rPara = f.Result.Paragraphs(1).Range
      f.Delete
      rPara.MoveEnd unit:=wdCharacter, Count:=-1
      rPara.Characters(rPara.Characters.Count).Delete
      strToCText = rPara.Text

      ' Insert the hyperlink. Haven't checked how we could do the "\l" with Hyperlinks.Add
      Set fHyperLink = rPara.Fields.Add(rPara, wdFieldEmpty, " HYPERLINK \l " & strBookmarkName & " ", preserveformattting = False)
      ' Restore the text
      fHyperLink.Result.Text = strToCText
      Set fHyperLink = Nothing
      Set rPara = Nothing
    End If
  Next
  ' (re-)create the bookmark
  rToC.Bookmarks.Add strMyToCBookmarkName
  rOldToC.Font.Hidden = True
  Set rToC = Nothing
  Set rOldToC = Nothing
End If

End Sub
