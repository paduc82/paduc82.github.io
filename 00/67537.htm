hairyfool say:

You do not have to do anything whatsoever to Windows for a change of RAM except for 32bit being limited to around 3.2GB. As you are already on x64 that does not apply.

Windows manages RAM dynamically and configures itself to the available memory each time it boots. The one exception is if you have made changes to the Virtual Memory settings but as avoiding changing this from the recommended option of Let Windows Manage These Settings is by far the best you shouldn't have to do anything.

Everything you express concern about is entirely hardware related and has nothing to do with Windows unless the computer won't boot at all. The BIOS (UEFI or Legacy) will tune itself to the requirements of the memory as defined by the Serial Presence Detect chip built into the DIMM ( a small 8 leg IC normally in one corner). If those requirements cannot be met then either the bank gets locked out or the system will not boot.

The BIOS does not "test" the RAM, it in the 1st place gets told what settings to use by the chip. There is no upgrade "sequence", if it is compatible it works, if not it doesn't. RAM is initialised every time the computer starts and about the limit of stored parameters within the BIOS settings is its size, a few BIOS report when it detects a change in the amount of RAM and requires going into BIOS setup to register it. If it supports overclocking then it may go through a check routine to make sure it can operate any user setting values and reset back to default if boot fails.

Acer will have issued a memory compatibility table with the motherboard release, the problem with that is they are seldom updated as memory listed becomes unavailable and new memory gets produced. The alternative is to go to one of the better memory suppliers such at Crucial or KIngston who certify the memory they produce as compatible based on the motherboard spec (they do not physically test each motherboard/memory combination). Most motherboard manufacturers and memory supplier recommend installing memory in pairs if not a complete matched set with the accusation that they want to sell more but as you are finding mixed sourcing does cause problems.
