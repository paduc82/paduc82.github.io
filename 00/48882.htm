How do I create chapters in a video? - Microsoft Community

tb-nz say:

How do I create chapters in a video?

I combined a number of clips each with title, but when I burn a DVD, the chapters did not correspond to the start of each clip. Is there a way to force the beginning of each chapter?
considerate_guy say:

When you are burning to DVD, Movie Maker is really calling another program called Windows DVD Maker. It essentially passes the complete video file to this program, so it has no idea where the original breaks are.  As a result, it just picks 6 arbitrary points using an algorithm.  These 6 "scenes" may not match with your idea of chapters at all.

 

If you want to create a DVD with chapters, here is my recommendation.

  1. Create a separate project for each chapter. Save a movie for each chapter onto your PC.
  2. Manually start Windows DVD Maker.  Now you can give it the individual movies you created for each chapter.  In this way, it will know that you want to treat each movie as a separate chapter and it will create your "Scenes" menu accordingly.
