> . .\AnalyzeMoveRequestStats.ps1

 

2. Select the move requests that you want to analyze.

 

In this example, we’re retrieving all currently executing requests that are not in a queued state.

> $moves = Get-MoveRequest | ?{$_.Status -ne 'queued'}

 

3. Get the move reports for each of the move requests you want to analyze.

> $stats = $moves | Get-MoveRequestStatistics –IncludeReport

 

4. Run the ProcessStats function to generate the statistics.

> ProcessStats -stats $stats -name ProcessedStats1

You could use something like this:

 

Sub RowAndColCount()
    Dim rng As Range, ar As Range
    Dim lnFirstCol As Long, lnLastCol As Long
    Dim lnFirstRow As Long, lnLastRow As Long
    Dim lnRowCount As Long, lnColCount As Long
    Dim r As Long, c As Long
   
    Set rng = Range("$I$2:$J$7,$H$10:$I$11,$E$10,$E$15:$E$19,$G$22,$M$5:$M$7,$O$3:$O$4")
     'or
'    Set rng = Selection

     'Get extremes of selection:
    lnFirstCol = Columns.Count + 1
    lnFirstRow = Rows.Count + 1
    For Each ar In rng.Areas
        lnFirstCol = Application.Min(lnFirstCol, ar.Column)
        lnLastCol = Application.Max(lnLastCol, ar.Column + ar.Columns.Count - 1)
        lnFirstRow = Application.Min(lnFirstRow, ar.Row)
        lnLastRow = Application.Max(lnLastRow, ar.Row + ar.Rows.Count - 1)
    Next ar
   
     'Count selected rows between Rows(lnFirstRow) and Rows(lnLastRow):
    For r = lnFirstRow To lnLastRow
        If Not Intersect(rng, Rows(r)) Is Nothing Then lnRowCount = lnRowCount + 1
    Next r
    Debug.Print lnRowCount & " individual rows in the range."
    
     'Count selected columns between Columns(lnFirstCol) and Columns(lnLastCol):
    For c = lnFirstCol To lnLastCol
        If Not Intersect(rng, Columns(c)) Is Nothing Then lnColCount = lnColCount + 1
    Next c
    Debug.Print lnColCount & " individual columns in the range."
End Sub
