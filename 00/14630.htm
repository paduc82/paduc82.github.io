dianeoforegon say: We don't use the term profile in Outlook Mac so I'm not sure I totally understand your question.

You don't normally use multiple Identities in Outlook. We use them to test for issues. For separate accounts, the Mac way is to use a different User. However, Outlook can manage multiple email accounts in one Identity.


An identity contains preferences, email account settings, messages, contacts, events, notes and tasks.
Your identity is named "Main" by default. It is located in the Microsoft User Data folder. Creating a new Identity is a good way to test your database to see if corruption is the problem.

Each identity can have (almost) any number of accounts. Each account is associated with a distinct email address than can be used to send and/or receive mail, but all mail is stored in the same database, all calendar events are common to all the accounts, all notes & contacts are shared.

You can have multiple accounts in Outlook. Examples: POP, IMAP, Hotmail, Gmail, AOL, Exchange.

How to create a new Identity or switch to a new Identity in Outlook 2011

http://www.officeformachelp.com/outlook/faqs/#faq20
 


CREATE A NEW USER

Go to System Preferences --> Create a New User in Accounts. Switch to the New User by logging out/in or use Fast User Switching.
