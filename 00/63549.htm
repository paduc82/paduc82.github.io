Copy the below line
control /name Microsoft.IndexingOptions
Windows Key+R > CTRL+V the above copied line
Click on Advanced > Rebuild

4. Restart Windows Explorer
Windows Key+X
Click Task Manager
Head to Processes tab and select Windows Explorer
Right click > Click Restart

How to - https://answers.microsoft.com/thread/bc609315-d...

chuck skovira say:

Here's how I solved the issue in case this happens to anyone else:

I had to go into the games/apps menu and delete the game, and delete the lego Hobbit demo I had downloaded. Then shutdown the xbox one, pull the power cord for 30 seconds. Reboot the xbox one and then re-insert the disc and allow it to install again.

It looked like it was going to fail the update again, but I just let the installer run a little longer and it finally worked.
