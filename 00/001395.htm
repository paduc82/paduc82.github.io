john mcghie say:
Hi Phillip:

I never recommend turning AutoRecover OFF (the name of the control is AutoRecover, which makes clear that it is not an auto-save).

AutoRecover will protect you if a) You have saved the document previously, b) Word crashes,  and c) it knows it has crashed (i.e. if there is a power failure, or an error in Word's program code).  As Bob notes, it does no good at all if Word exits normally, or if the user makes a mistake.

There's a long list of circumstances under which it won't help.  However, I don't recommend turning it off, because on very rare occasions, it might help.

I DO, however, recommend turning ON "Always create backup".  That saves a copy of the file every time you save, in the same folder as the document, labelled "Backup of ...".

If anything bad happens to the original, open Backup of and continue on your way.  You will keep everything up to the most recent save.
