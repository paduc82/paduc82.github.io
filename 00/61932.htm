Where does the Sound Recorder app save the recordings? - Microsoft Community

analy [msft] say:

Where does the Sound Recorder app save the recordings?

I recorded something with Sound Recorder and I want to access the recordings. The help page mentions I can send the files to someone using the Share charm but is there any easy way to find the saved recordings directly? Where are they stored?
troy barnes say:
Using the Share charm to send the files to another place is the recommended way of sending the recordings out of the app easily.

If you want to access the recordings directly you can find them stored inside the application data for the Sound Recorder app. Note that apps store important information in these locations so it is not recommended that you mess with other files in there.

You can find the recordings of the Sound Recorder app stored under: <Your user profile>\AppData\Local\Packages\Microsoft.WindowsSoundRecorder_8wekyb3d8bbwe\LocalState\Indexed\Recordings\

To get to that location you can follow these steps:
  1. Open File Explorer
  2. Copy and paste the following:
    %localappdata%\Packages\Microsoft.WindowsSoundRecorder_8wekyb3d8bbwe\LocalState\Indexed\Recordings
  3. Hit Enter
  4. All the recordings from the app should be in that folder.

Keep in mind that if you modify the files in that folder those changes will be reflected in the app. For example, if you delete those files you will lose the recordings so proceed with caution.
