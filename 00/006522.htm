Well, your parents are going to find out, because your only option is to reinstall Windows 8.1 from scratch if it came preinstalled.

New computers that come preinstalled with Windows 8.1 often have what is called a recovery partition. This is used to reinstall the operating system in the event of a system crash. To access it, you will need to boot into when you start your computer by pressing a function key. This can be either F1, F2, F9, F10, F11, F12 or even DEL or Tab key. Consult the owners manual that came with your PC for instructions about how to reinstall Windows.

This is how the recovery partition is accessed for the most popular makes..

For Dell, press F8 on the keyboard until the Advanced Boot Options menu appears on the screen.

For HP, press F11 directly after powering up the unit

For LG, press F11 directly after powering up the unit

For Toshiba, press and hold '0' BEFORE and during the power up

For Acer, press and hold ALT + F10 as soon as you see the logo

For Asus, press F9 as soon as you see the Asus logo.

For Samsung, press F4 at power up..

For Fujitsu, press the F8 key repeatedly directly after powering up

For Advent, restart your computer. Then Press F10 repeatedly until the message "Starting System Recovery" is displayed

For Sony VAIO, restart and press "F8" or "F10" repeatedly until the "Advanced Boot Options" screen appears.

If you have any important data on the drive, please make sure you back up your data before reinstalling.

Possible registry problem:

You should remove "RecoveredFrom" from registry.

Before deleting registry key back it up 

  1. In Registry Editor, locate and click the registry key or subkey that you want to back up.
  2. Click File > Export.
  3. In the Export Registry File dialog box, select the location where you want to save the backup copy to, and then type a name for the backup file in the File name field.

_elaine_ say:

Hello, RichardMasterson

Here are several considerations, not presented in any particular order:

  • Skype does not work like a conference bridge - therefore, there is no number for you to give people to call in. A Skype Number would not serve the same purpose, and this is a separate paid-for feature.
  • You can set up a group call, and provide participants a link to join the group.
  • Key question: how many people do you anticipate participating?
  • The consumer market versions of Skype have stricter limits on the number of participants for audio versus video calls.
  • If all of your participants will start their own Microsoft/Skype accounts, then feasibly your conference calls would be free of charge. Otherwise, you would also need a subscription or credit to cover call costs, which means not a free way to conduct your committee meetings.
  • What types of features are you expecting - or would these only be audio calls?
  • What would you be using Skype on? Its operating system?

Well! There is a start.
