Calculate numbers and display result in another cell

hansv mvp say:

You can create a custom VBA function in a module in the Visual Basic Editor.

As a first step, select File > Options.

Select 'Customize Ribbon' in the navigation pane on the left.

Tick the check box for Developer in the list of Main Tabs on the right.

Click OK.

Activate the Developer tab of the ribbon.

Click 'Visual Basic'.

Select Insert > Module.

Copy the following code into the module:

Function Eval(s As String)
    Eval = Evaluate(s)
End Function

Switch back to Excel.

In E26, enter the formula

=Eval(C26)

Remark: if you want to include cell references in your expression, for example 12*A1+25*B1+14*C1, the function needs to be slightly more complicated to avoid problems:

Function Eval(s As String)
    Application.Volatile
    Eval = Application.ThisCell.Parent.Evaluate(s)
End Function
