dustiiwolf say:

It doesn't work like that, per se.

In order to play DVDs, Microsoft has to make use of specific codecs. However, some of these codecs (or the patents of them), from what I understand, cost Microsoft money to use and implement into the Windows Operating system.

There are no "free licenses", and you don't "make deals with the patent licenses". You have to pay licensing fees to the owners of the codecs or the patents of them thereof in order to be able to implement the proper codecs into the software.

As such, if Microsoft were to resume including DVD playback by default in Windows, it would inflate the cost of the operating system and make it impossible to offer for a lower price, let alone free for Windows 8.x users.

However, Microsoft still offers a way to obtain the ability to play DVDs. For Windows 8.x users, it's the "Windows Pro Pack", and for Windows 8.x Pro users, it's the "Windows Media Center Pack". Both of these can be obtained through the "Add features to Windows" wizard in the Control Panel, for a one-time fee*, that will install Windows Media Center and the appropriate codecs for standard DVD playback.

*Per copy of Windows
