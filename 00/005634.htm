This little macro shows how to edit the subject by code. Open the VBA editor, and paste it to the ThisOutlookSession module. To run it, press alt+f8.

 


Sub ChangeSubject()
  Dim Item As Object
  Dim Exp As Outlook.Explorer
  Dim Sel As Outlook.Selection
  Set Exp = Application.ActiveExplorer
  Set Sel = Exp.Selection
  If Sel.Count Then
    Set Item = Sel(1)
    Item.Subject = "changed subject"
    Item.Save
  End If
End Sub

This is a result of "sandboxing" which is a Apple requirement of software sold through their store. Microsoft Office 2016 for the Mac will be sold from the Apple store and so they have implemented the Apple rules in the 2016 product. You can read up on what sandboxing actually is by searching it on the web...

In essence what is happening is that Acrobat creates a temporary file "ARCXXXXX" and sends a request to Word to open it so that a PDF can be created. However, the file is in the protected "sandbox" area and Word needs your permission to access it.

Apple, Adobe and Microsoft need to work this out... IMHO.
