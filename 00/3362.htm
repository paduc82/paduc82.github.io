vijay b say:

Hi John,

Thank you for being a part of Windows 10 Technical Preview testing.

Shutdown sound is available in Windows 10 and was available in Windows 8. Just the feature is hidden and we will help you out.

Firstly we have to enable the feature from registry.

1. Press Windows + R, Type regedit.
2. On UAC screen, Press Yes.
3.  Navigate to HKEY_CURRENT_USER\AppEvents\EventLabels.
4. Here you will be able to see a list of sound events.
5. Scroll down and click on System exit.
6. To the right pane, locate the key ExludeFromCPL and double click on it and change its value to 0 and click OK.
7. Now come to your Desktop screen closing the registry editor and right click the Sound icon on the Taskbar and select Sounds.
8. Under program events you will be able to see the System exit option.
9. Choose the event and change the sound from sounds drop down box.

In this way you will be able to change the shutdown sound in Windows 10.

Disclaimer: These methods contains steps that tell you how to modify the registry. However, serious problems might occur if you modify the registry incorrectly. Therefore, make sure that you follow these steps carefully. For added protection, back up the registry before you modify it. Then, you can restore the registry if a problem occurs. For more information about how to back up and restore the registry, click the following article number to view the article in the Microsoft Knowledge Base: 322756
Refer (How to back up and restore the registry in Windows) (http://support.microsoft.com/kb/322756/ )

I did learn a few things from talking to them though....everything firewall, anti-virus, and VPN need to be turned off...

 

START --> RUN --> type in <services.msc>

 

right click on anything that has to do with firewall, anti-virus, and VPN

 

Once this is done....uninstall ActiveSync 4.5 or any other version of ActiveSync...you may also need to uninstall the Network Adapter driver for your Windows Mobile Device (Control Panel --> Hardware Manager --> Network Adapters)

 

Now plug in your phone into the computer and let it detect the device and install the proper drivers.....while still connected, install ActiveSync 4.5.....Once installed, uplug the phone from the computer and plug it back in.....it should now almost instantly find the phone and go through the Wizard setup....

 

I rebooted my computer to turn back on all the services I disabled and connected my phone again and it worked just fine....

 

I hope this helps a few people out there...i just about pulled my hair out trying to get this to work yesterday....
