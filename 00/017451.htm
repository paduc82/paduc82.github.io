ryanbane say: Having said that, I found a workaround that I don't much like. This question is still open because if there's a nicer way to do this, I'd really like to know about it. Posting here for the benefit of anyone else who wants to do this, but with a significant disclaimer: this edits and reverts the system's hosts file, and you can screw some things up with DNS resolution if you break the hosts file badly. Use at your own risk, you're responsible for your own machine. This worked for me on a Surface RT, and should work for any windows 8 where you have administrator access.


My workaround:

Open notepad as administrator (from the start screen, type notepad, right click/swipe down on the app, click run as administrator, say yes to the User Account Control prompt when it displays)

Notepad will open on the desktop.

Open this file: C:\Windows\System32\Drivers\etc\hosts (if you press file, open, and navigate to the etc directory, remember to select All Files so you can see files without a .txt extension)

If the url you want to go to is at host www.site.com (for example, you want to pin http://www.site.com/latestComic.html), add the following line to the end of the file and save:

127.0.0.1 www.site.com

This will cause resolution of the URL to go to your machine, rather than the actual site.

Leave the notepad on your desktop, go back to the start screen.

Open IE, go to the URL you want to pin (something at host www.site.com). IE will go to your machine rather than the actual website, so you'll get an error in IE, but the URL in the address bar won't change, and you can pin the URL. Pin the URL.

Once you've pinned the URL (and named it something besides the error page title), return to the desktop and notepad. Remove the line you added, and save the file. Close notepad.

Return to IE, close IE (swipe down from the top of the screen). Return to the start screen.

Click the pin for your URL, verify you get redirected as you expect.

Note that the Protected view settings have nothing to do with access rights in Windows; it's a setting in Word/Office.
