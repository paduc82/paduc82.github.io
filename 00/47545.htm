malkeleah say:

You shouldn't hate to ask - I'm here because I want to help. Each motherboard mftr. gets you into the BIOS slightly differently. When you first turn on the machine there will be a message that says something like "Press Esc" or "Press DEL" or "Press F11" to enter Setup. That will get you into the BIOS. Look for an entry for USB legacy devices, usually under the integrated peripherals category. You'll have to look around because each BIOS is different.

When you are in the drive utility (it sounds like you chose SeaTools), there is a menu where you can choose basic tests. I usually do the short test first because if it fails that one there isn't any point in doing the long test. If it passes the short test (90 seconds) I will usually do the long test just to be sure. At the end of the test you will get the results. If the drive fails any physical tests (as opposed to file system corruption), it needs to be replaced.

Of course, other components might be bad in your case. It was my opinion that you should start testing with the hard drive.

Hi Doug,

Backup and restore is the technology used in Windows 7 or earlier machines to do a backup. With the advent of Windows 8 File History and Windows image came into existence. Although they do the same job but File History is more effective than backup.

Hi Rogeh. I'm Greg, an installation specialist and 8 year Windows MVP, here to help you.

As long as the underlying license is retail, then you can transfer it or WIndows 10 upgrade of it (which inherits the same transfer rights) to new hardware.

shenan stanley say:

If it is a retail license, it can be installed on the new system.

With retail licensing, you can transfer the license from one computer to another over and over and over...

There's no real 'process' to it.  Install it and activate it on the new PC.

If you associated the Windows 10 license with your Microsoft account, you could install Windows 10 on the new system instead.
