You can't do this with a macro, at least not in 2007. This requires VBA code.

What you need to do is have a query that returns all records you want to output. (if you are outputting the whole table, then you don't need a query). You then need to loop through the recordset using the OutputTo method to create the RTF files. You can't break the report up, so you need to report on one person each time. So create a Query and set the criteria to look up a text box on a form (=Forms!formname!controlname). You need to create a form to kick off the job. Have a button on the form to run code like:

Dim db As Database, rs As Recordset
Dim strFilename As String

Set db = CurrentDB()
Set rs = db.OpenRecordset("queryname")

rs.MoveFirst
Do While NOT rs.EOF
    Me.txtRecordID = rs.Fields("RecordID")
    strFilename= Me.txtRecordID & ".rtf"
    DoComd.OutputTo acOutputReport,,"reportname",strFilename
    rs.MoveNext
Loop

Set rs = Nothing
Set db=Nothing
