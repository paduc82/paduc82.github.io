Can't remember document password - Microsoft Community

nicolaredman say:

Can't remember document password

Original Title: "Document passwords"
Hello, I set a password on a Word document and can't remember it. Is there any way at all to override it? I'm desperate! Help appreciated. Thank you
john mcghie say:
No.  There is no way to override the password on a document created in a recent version of Word.  There used to be, in the old Word formats, but the new password mechanism is a lot tougher.

In the old Word formats, the password was actually stored in the document.  In the new one, the password is not stored in the document, it is used as the encryption key for the entire document.  So if you have the correct password, the document is readable: if you haven't, it's not, not even by Microsoft.

As J.E. said, if you search the Internet, you will find companies that claim to be able to crack the password.  Whether their claims are true or not depends on how complex your password was: if you know how to set a secure password, it may take them more than a year to crack it.

If you are entirely comfortable with giving your credit card details to someone who cracks passwords for a living, then I guess you will find out.  Personally, I would not dare.

Hope this helps
