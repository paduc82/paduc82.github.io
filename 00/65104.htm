If the port setting on the Windows XP laptop is shown as "USB001", that means that the laptop is expecting to find the printer physically attached to one of its own USB ports.  That's not where I understand you have the printer connected.

Unless I'm totally confused about your setup (always a distinct possibility), you have the printer physically connected to the Window 7 desktop.  You want to be able to access the printer from the XP laptop over the network.

I assume that you can share files between the Windows 7 desktop and the Windows XP laptop.  If you can't share files because you haven't properly configured the network permissions, you won't be able to share the printer either.

On the Windows 7 desktop, go to Printers and Devices, right click on the icon for the HP 2050, and select "Printer properties."  Click the "Sharing" tab.  The box to "Share this printer" and the box to "Render print jobs on client computers" should BOTH be checked.  Make a note of the name in the box labelled "Share name" (this should be a relatively short name with no spaces and no special characters; something like HP2050 would be fine).

On the Windows 7 desktop, right click My Computer and select Properties.  Make a note of the Computer Name.

Because the instance of the printer currently installed on the Windows XP laptop expects to find the printer on a USB port, you need to install another instance of the printer on the laptop. 
  • Go to Printers and Faxes and start the Add New Printer wizard (click Add a printer)
  • Click the radio button to add "A local printer attached to this computer"
  • UNcheck the box to "Automatically detect and install my Plug and Play printer"
  • Click Next
  • Click the radio button to "Create a new port"
  • In the drop-down box, select "Local Port"
  • Click Next
  • In the "Enter a port name" box, type the following and click OK
         \\COMPUTERNAME\Sharename
    where "COMPUTERNAME" is the name of the Windows 7 computer that you noted above and "Sharename" is the printer's share name configured on the Windows 7 computer
  • Complete the Add New Printer wizard as usual.  If a message pops up saying that a driver is already installed for this type of printer and asks if you want to reuse the installed driver, answer Yes.
  • When you get to the step of naming the printer, add something to the name so that you will know that this instance is the one that is installed on the Win 7 computer.  For example, if the name of your desktop is DELLW7, you might name this printer "HP2050 on DellW7".
