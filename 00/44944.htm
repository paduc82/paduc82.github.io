This error message is as a result of Windows system corruption. There are quite a few events that can have resulted in file errors. An unfinished installation, a partial file deletion, incorrect deletion of applications or hardware. It could also be brought about in the event your laptop or desktop is contaminated with a virus or adware/spyware attack or by an incorrect shutdown of the laptop or desktop. Any of the aforementioned actions may very well end up in the removal or data corruption of Windows system files. This damaged system file will result in absent or wrongly linked information and archives required for the accurate functioning of the program.

1) Click Start button.
2) Type: CMD.exe into the 'Search programs and files' field
3) Right-Click on CMD.exe and select Run as Administrator

5) Type: cd %windir%\ServiceProfiles\NetworkService\AppData\Roaming\Microsoft\SoftwareProtectionPlatform
6) Type: rename tokens.dat tokens.bar
7) Type: cd %windir%\system32
8) Type: net start sppsvc
9) Type: slui.exe
10) After a couple of seconds Windows Activation dialog will appear. You may be asked to re-activate and/or re-enter your product key or Activation may occur automatically.
