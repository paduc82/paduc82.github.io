brian tillman [outlook mvp 2007-2019] say:

Are you using a POP account?  With POP, there is no such thing as read or unread mail, only old and new mail, with the server telling Outlook what's new or not.  If you want Outlook to download only messages that you have not read, then log into your mailbox on the server, create a folder other than Inbox, and move those messages to the other folder.  If they're not in Inbox, Outlook can't see them.

Alternatively, if your server supports it, employ an IMAP account, although that could lead to a PST large enough to hold all the data in the Inbox as well because Outlook will echo all server messages on the local PC as well.

bernie deitrick, excel mvp 2000-2010 say:

Don't link to a spcific cell - use VLOOKUP functions keyed on unique names to extract the data of interest.  For example, if you have two columns with Last names and First names, create a third column where you use formulas like this, on each sheet (and then copy down to match your table)

=A2 & " " & B2

to create a cell with first and last name combined, then use those cells for the VLOOKUP, like

=VLOOKUP(C2,Behaviors!$C$2:$G$100,3,False)

to pull values from the Behaviors worksheet. If you sort the Behaviors sheet, make sure that you sort the entire table, including the names. Use help to learn more about VLOOKUP, or post back if you have problems.
