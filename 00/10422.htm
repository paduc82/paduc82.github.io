e.j. gun say:

There is some cleanup you should do before worrying about the button issue.

First, do you really need to work with entire columns (such as "a:y") and entire sheets ("Cells.Select")?  That takes a lot of time and memory.  Try limiting the ranges to just those areas that actually contain data.  UsedRange, CurrentRegion, etc, can limit the range of the sheet you access, and usually make for more efficient code.

Second, try to be more explicit about which workbooks, worksheets, etc that you are referencing.  You should at least append "ActiveSheet" to the "Cells.Select" and "Range("A1").Select, etc, lines.

Finally, ask yourself if you really need to do so much select/copy/paste.  Instead, can you just directly tranfer ranges?  For example:

Workbooks(sThisbook).Sheets("Data").Range("A1:J45").Value = Activeworkbook.Sheets("Data").Range("A1:J45").Value

Anytime you see code with a lot of "Select" and "Copy" and "Paste" in it, it can be made more efficienct.
