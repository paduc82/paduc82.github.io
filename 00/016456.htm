File menu > Options command > Advanced option > Formulas section.  Make sure "enable multi-threaded calculation" is turned on. When that is on, make sure that the "Use all processors ..." is also selected. 

Note: "Hyperthreading" can split a physical cpu core into 2 logical cores so the number of cores reported may be double what you expect.

gilles messinger say:

All of the optimization steps described in the reply are important because truth must be said: Excel 2016 still runs with a single thread most of the time, even if there is an option trying to make you believe it can work with multiple cores.

If you need to prove this, just use a pretty complex Excel workbook and:

- Make sure the multi-core option is enabled

- Run an update on the Excel workbook

- Watch the processor utilization in the Resource Monitor

You will see that most of the time the % CPU used is equal to 100% divided by the number of threads of your computer.

This proves that Excel uses a single thread most of the time, even with the multi-core option enabled.

For instance, on my computer which has 8 threads (1 cpu quad-core with hyperthreading enabled), Excel is stuck at 12.5% CPU most of the time, which is exactly corresponding to a single thread (100/8 = 12.5).

File, Options, Mail, Spelling and autocorrect button, then Autocorrect options button. Use the replace text as you type section. If you type something, select it then open this dialog (from the file tab on the message form), the formatted text will be in the With field.
