joeu2004 say: Richard wrote:

how to analyze the value of paying acrued interest on student loans versus the cost of capitalizing them in the loan


The following shows one way of comparing the loan alternatives using the IRR (internal rate of return) of the cash flows.  But I want to emphasize that IRR analysis is not the whole story.  There are other ways to compare the alternatives.  Also, practical considerations might rule out some alternatives, even if they are the better choice.

This is not an appropriate forum in which to discuss those financial considerations; besides, you do not provide sufficient information about your financial situation.  But I interpret your question, in the context of your previous year-old thread (click here) [1], to mean that you are interested in the Excel formulas that you might use, not financial advice per se.  The following shows some of those formulas.

The following analysis uses the first loan as an example.  Similar analysis can be applied to the other loans individually.  Also, you can make different choices for each loan.  There are a total of 12 alternative combinations.

The first image below shows the values for the alternatives for the first loan.  The second image shows the underlying formulas.





Since the IRR represents cost to you, you might choose the alternative with the smallest IRR.  Note that Loan A and Loan B have the same IRR.  That is because the interest rates are the same; for a simple loan, the periodic IRR is the periodic interest rate.  When there is a tie, you might choose the alternative with smallest periodic payment.

(But as noted above, the IRR is not the only criterion that you might use for making your choice.)

We can use Excel RATE to calculate the IRR because in this model, all of the cash flows after the first one are the same, and they occur regularly (monthly).  They are the loan payments.

Note that for Loan C and Loan D, the PV of the RATE function is reduced by the accrued interest.  The accrued interest is also included in the total payment.  That is because the accrued interest must be paid at the end of the original loan, since it is not recapitalized.  [EDIT] It is a negative cash flow that offsets the positive cash flow (loan principal), reducing the net cash flow in time #0.

Nevertheless, the monthly payment for Loan C and Loan D is based on the full new loan principal, since that is the amount that must be repaid (plus interest).

The accrued interest is also implicitly included in the total interest for Loan C and Loan D, since that is simply the difference between the total payment and the loan principal.

However, that is not really the complete model for the IRR analysis.  The original loan of $37,963 occurred some time ago, and the cash flow representing the disposition of the $8474 in interest accrured for the original loan probably occurs 12 to 48 months later.  So there are some number of zero cash flows in between.  That changes the actual IRR values.  And under some assumptions, it changes the order of some loan alternatives based on the IRR.

For example, if the first original loan was 48 months, we might set up the following model.



The formula in B12 is =IRR(B13:B181); for D12:  =IRR(D13:D181).  The formula in C12 is =IRR(C13:C121); for E12:  =IRR(E13:E121).

greg maxey say:

Rohn007,

 

I know you qualified your "old 2003 WordArt options are not available, but all the changes from Wor2003 to 2013 are fairly complicated.  I still may not have my head fully around the subject, but I have been doing some tinkering and here is what I (think) I know.

1.  WordArt in 2003 and 2007 are basically the same thing.  The difference is the user interface for inerting and formatting (Menu and toolbar for Word2003, Ribbon and Contextual tab for Word2007).  And of course Word2007 offers significantly more formating options.

 

2.  Both begin life as an inlineshape.  To be rotated they must be converted to a shape either programatically or via layout formatting.

 

3.  Word 2003 and 2007 documents containing WordArt objects can both be opened in both Word 2010 and Word 2013.  The WordArt objects those documents contain can be altered using a WordArt Tools Formatting contextual tab.  New "classic" WordArt objects can be added to these documents by adding the WordArtInsertGalleryClassic control to the QAT or ribbon.

 

3.  Both can be copied and pasted into a Word2010 document. In these applications, both can be altered using  a WordArtTools Format contextual tab.

 

4. If either are copied and pasted into a Word2013 document, they are basically toast.  They are no longer recognized as WordArt and become a simply inline shape that can be edited using the Drawing Tools Format contextual tab.

 

However, here is the interesting part,  a classic WordArt object can be inserted into a Word 2010 and 2013 document programatically and when they are, they begin life as a shape (vice inlineshape) and the WordArt Tool Formatting contextual tab is available when the object is selected!!

Sub ClassicWordArtInsert()
  ActiveDocument.Shapes.AddTextEffect Office.MsoPresetTextEffect.msoTextEffect1, _
               "WordArt - Your Text Here", "Arial", 20, _
               Office.MsoTriState.msoFalse, Office.MsoTriState.msoFalse, 0, 0, Selection.Range
End Sub

As for the tweak:

Sub ClassicWordArtInsert()
Dim oShp As Word.Shape
Dim oILS As Word.InlineShape
  Set oShp = ActiveDocument.Shapes.AddTextEffect(Office.MsoPresetTextEffect.msoTextEffect1, _
       "Your Text Here", "Arial", 36, _
       Office.MsoTriState.msoFalse, Office.MsoTriState.msoFalse, 0, 0, Selection.Range)
  If MsgBox("Do you want your classic WordArt object inline with text?", _
            vbQuestion + vbYesNo, "Classic Inline") = vbYes Then
    Set oILS = oShp.ConvertToInlineShape
  End If
  On Error Resume Next
  oShp.Select
  oILS.Select
  On Error GoTo 0
lbl_Exit:
  Exit Sub
End Sub
