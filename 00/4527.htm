rob koch say:

The csrss.exe file is almost never a Trojan, though lots of online (often fake) web pages will try to convince you otherwise.  At least one of these must always exist for Windows to be running and two or occasionally even more are not uncommon.

To view the properties or Open file location for the file, simply click the Show processes from all users button in the lower left corner of Task Manager.  The csrss.exe will then be listed as running as user System and the properties details tab will show it to be the Client Server Runtime Process and part of the Microsoft Windows Operating System.

Though there have been malware created using this name, they are relatively rare and mostly quite old, so not often seen today, since most antimalware can easily detect and remove them.

Yes, but you have to use the MsgBox Function as part of your IF Action. Plus you can't use the vb Constants so you have to use the numerical equivalents.

If you add an IF to your macro in the condition box enter:

MsgBox("prompt",4)=6

Use whatever prompt you want. the 4 sets the box type with Yes and No buttons, the 6 checks if Yes was pressed (use 7 if you want to test for No having been pressed).

john mcghie say:

Hi Keri:

If those images really were 72 dpi PNGs, that's likely the problem.  72 dpi looks fairly disgusting when upsampled on a high-res Mac.

It may also be the "Placable Header" problem anyway:  Word used to create "Screen Display" images of low resolution, then get all confused and display/print the wrong one.  Make sure all your updates are in: there were some fixes to reduce this.

I would not worry too much about carryover corruption.  If you have removed the images, whatever is left will not cause too much trouble.  However, if you want to be bothered, there are two ways to do this:

A:  The Paste into New Document method

B:  The Organiser Copy method.

The Paste into New Document method is simpler.

  1. Create a new document.
  2. Set the paper size and margins correctly.
  3. Remove (Search/Replace) all the Section Breaks from the old document.
  4. CAREFULLY select all the text EXCEPT the very last paragraph mark.  Turn onDisplay non-printing characters () to reveal the paragraph marks so you can see what you are doing.  The last paragraph is where all the hidden code in the document is hiding: if you copy it, you copy all the rubbish in.
  5. Paste that into the new document.
  6. Add new section breaks if you need them, do NOT copy the old ones, they store rubbish too!

This method has the advantages that it is conceptually simple, and it copies only the styles in use.

The Organiser Copy Method is faster once you know how, and less work.

  1. Create a new document.
  2. Set the paper size and margins correctly.
  3. Ensure your insertion point (cursor) is in the new document.
  4. Go to Format>Style and click the Organiser button in the bottom left.
  5. Your new document should be shown in the left pane, the document Template will be shown in the right pane.
  6. Click Close File below the side that shows the template (all the styles in the template are already in the new document, copied there at the instant of creation).
  7. The Close File button changes to an Open File button. Click it.
  8. Change the Enable button below the file list to "Word Documents" (or All files).
  9. Navigate to the OLD document and click Open.
  10. You will see a list of the styles in the old document.  Sadly, there is no way to tell which ones are in use, and most of them are not.
  11. In the old file pane, select the styles you want to bring into the new document.
  12. Click Copy
  13. Say "Yes" if prompted to overwri

    The styles you selected will be instantly copied to the new document, overwriting any that already existed. You can save time by simply selecting them all and copying them all: styles are tiny, a few bytes each, you will not notice any extras.

    The benefit of this procedure is that it is fast: once you know how, you can do it in seconds, with no document preparation (you do not need to remove either the bad pictures or the section breaks).  Another major benefit is that it CANNOT copy any form of corruption or rubbish over: the Organiser looks only at the document property store, not the headers and footers or the text, and copies only the styles you select.

    You would then paste just the text into the new document, using Copy then Edit>Paste Special>Unformatted Text.  Save the document, then re-format the text using the known good styles you just copied in.

    Whereas the Paste Into New Document method gives you a very clean result, with almost no rubbish coming in, the Organiser Copy method gives you a known perfect result, with no possibility of any rubbish coming in.

    However, if you make a mistake and use an ordinary paste instead of an Unformatted Text paste, all the rubbish is now indelibly in the new document, so you will need to start again, creating a fresh new document.

    Hope this helps
