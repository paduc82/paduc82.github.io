a. user say:

Hi,

Your concern about the performance of your device when viewing photo thumbnails can be due to its configuration and/or corrupted system files. To better understand your concern, we'd like to know the following:

  • The last changes made/activities performed on your device prior to your concern.
  • If you've experienced your concern on multiple image file extensions. Examples are JPEG, JFIF, BAT, GIF, and PNG.
  • The version and build of Windows 10 that you're using on your device. To check this, you can perform the steps posted on this link.

Based on the steps that you've already performed, we'd recommend performing the following to recreate the thumbnail cache:

  1. Open File Explorer.
  2. On the left pane, click This PC.
  3. Right-click the C: drive where Windows 10 is installed, and select Properties.
  4. Click the Disk Cleanup button.
  5. Check the Thumbnails option, and uncheck all the other options.
  6. Click OK.
  7. Click Delete Files to confirm and to complete the task.

In the event that the steps did not suffice, we'd recommend going through the commands posted below to further isolate your concern:

  1. Open Start.
  2. Search for Command Prompt.
  3. Right-click the result and select Run as administrator.
  4. Type the following command to stop File Explorer and press Enter.
    taskkill /f /im explorer.exe
  5. Type the following command to delete all the thumbnail database files without asking for confirmation and press Enter.:
    del /f /s /q /a %LocalAppData%\Microsoft\Windows\Explorer\thumbcache_*.db
  6. Type the following command to start File Explorer and press Enter.
    start explorer.exe
  7. Close Command Prompt to complete the task.


Note: We understand that you've already performed two of the commands listed. We've detailed the known succession and format of commands that works on many occasions on other users.

Also, if you've only experienced this issue on images with JPEG/JPG/JPE file extensions, we'd recommend downloading the registry items that we've prepared for you and merge them on your existing registry configuration. To do this, you can perform the following:

  1. Once download is complete, right-click on one of the registry items.
  2. Select Merge.

Note: Changes in the registry may cause issues on your device when done incorrectly. To secure your device, we'd recommend backing it up by performing the steps posted on this link. The article also includes steps on how you can restore them in case you need to.

We look forward to your response.
