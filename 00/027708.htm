Outlook 2003 and up can use PSTs that start with a maximum size of 20GB but can be adjusted to allow growth beyond that limit.  If you ran into a limit, it's because you are using a PST carried over from Outlook 2002 or earlier.  You should have recreated that PST long before now.

But my question is, when I truncate the oversized file, what happens to the data that has been truncated (i.e. cut from the oversized .pst file to make it the correct size)?  I will obviously want all the data in my oversized pst file to be available to me in Outlook so I can delete/ archive it. If you truncate, the data is lost forever.  You will not be able to recover it.

There are third-party commercial PST repair tools that may be able to recover a little more data than the truncate-and-repair method you describe.  You can start here to see if you can find one: http://www.slipstick.com/problems/repair-a-damaged-personal-folders-pst-file/.  The very first ad on that page shows a tool that claims to be able to recover oversized PSTs.
