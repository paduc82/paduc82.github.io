If you select the existing chart before running the macro, you can use code like this:

 

Sub ChangeSeries()
    With ActiveChart.SeriesCollection(1)
        .Values = Array(56, 61, 45, 15, 30, 10)
        .XValues = Array("A", "B", "C", "D", "E", "F")

End Sub

Sub ChangeSeries()
    With Worksheets("Sheet3").ChartObjects(1).Chart.SeriesCollection(1)

suzanne s. barnhill say:

Despite the dire warnings about editing the Registry (if you're concerned you can always export the key first), this particular key is one that is quite safe to delete; as noted in the article Stefan referenced, "When you restart Word, it rebuilds the Word Data registry key by using the default settings." The article is an official Microsoft publication and can be trusted. The article provides step-by-step instructions to find and delete this key (under "To delete the Word Data registry subkey, follow these steps:"), but here's a summary.

  1. Close Word.
  2. To edit the Registry, press Windows Key + R to open the Run box.
  3. Type "regedit" (without the quotation marks) and press Enter. You'll be asked if you really, really want to do this; say yes.
  4. In the Registry editor, you'll see several folders. The one you want to expand is HKEY_CURRENT_USER.
  5. Navigate down through the folders to HKEY_CURRENT_USER\Software\Microsoft\Office\14.0\Word\Data.
  6. The article provides instructions for exporting the key if desired.
  7. Select the key and press Delete our use Delete on the shortcut menu (I don't see it on the File menu).
  8. Exit the Registry Editor and start Word.
