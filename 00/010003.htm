Although the formats can vary, the variability has to be defined
somehow, or else you would not be able to tell a phone number from an
address number; or other numeric data in the body of the email.

Are the phone numbers preceded by some defining word or indicator?

For the sake of this excersize, I assumed North American numbers which
would always have an area code.

I also assumed the area code could be surrounded by parentheses; and
that there might be a hyphen, dot or space used as separators (or no
separators).  Examples are given in the macro.

I further assumed that you might have multiple phone numbers within
the body of the macro (up to five) and that the bodies would be in
column D, and the phone numbers would start in Column J.

Furthermore, I figured you would probably want the extracted phone
numbers in some standardized format.

Those assumptions can be changed, but this should get you started.

==================================================
Option Explicit
Sub ExtractPhoneNumber()
    Dim s As String
    Dim re As Object, mc As Object, m As Object

    'Assume email body is in column D
    'But change it to match your ws
    Const BodyColumn As Long = 4

    'Assume PhoneNumbers will be written
    'starting in column "J"
    Const PhoneNumColumn As Long = 10

    Dim BodyRg As Range, c As Range, d As Range
    Dim i As Long
'set rg to relevant area
Set BodyRg = Range(Cells(1, BodyColumn), _
    Cells(Cells.Rows.Count, BodyColumn).End(xlUp))

'set up Regular Expression Engine to match phone numbers
'as written, matches 3334445555, 333.444.5555, 333-444-5555,
'333 444 5555, (333) 444 5555 and all combinations thereof,
'like 333 4445555, (333)4445555 or 333444-5555.
'Does not match international notation +13334445555,
'but matches domestic part in +1 333 4445555.
Set re = CreateObject("vbscript.regexp")
    With re
        .Pattern = "\(?(\b\d{3})\)?[-. ]?(\d{3})[-. ]?(\d{4}\b)"
        .Global = True
    End With

'iterate through bodies
For Each c In BodyRg
    'clear cells where phone numbers will be written
    'assume they will start at PhoneNumColumn
    'column and that there can be a maximum of five
    'phone numbers in the body
    Set d = c(1, PhoneNumColumn - BodyColumn + 1)
    Range(d, d(1, 5)).ClearContents
    s = c.Text
    If re.test(s) = True Then
        Set mc = re.Execute(s)
        i = 1
        For Each m In mc
            With d(1, i)
                .Value = m.submatches(0) & _
                    m.submatches(1) & m.submatches(2)
                .NumberFormat = "(000) 000-0000"
            End With
            i = i + 1
        Next m
    End If
 Next c
End Sub
====================================

All of those assumptions can be modified.
