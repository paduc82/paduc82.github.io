ken sheridan say: You need to assign the path to the OutputFile argument of the OutPutTo method.  The following is from one of my online demo files which does this, incorporating the customer name and invoice number in the file name and saving it to a sub-folder per customer within the default folder specified at start-up.  The sub-folder is created if it does not exist:

Private Sub cmdPDF_Click()

  On Error GoTo Err_Handler
    
    Const FOLDER_EXISTS = 75
    Const MESSAGE_TEXT1 = "No current invoice."
    Const MESSAGE_TEXT2 = "No folder set for storing PDF files."
    Dim strFullPath As String
    Dim varFolder As Variant
    
    If Not IsNull(Me.InvoiceNumber) Then
        ' build path to save PDF file
        varFolder = DLookup("Folderpath", "pdfFolder")
        If IsNull(varFolder) Then
            MsgBox MESSAGE_TEXT2, vbExclamation, "Invalid Operation"
        Else
            ' create folder if does not exist
            varFolder = varFolder & "\" & Me.Customer.Column(1)
            MkDir varFolder
            strFullPath = varFolder & "\" & Me.Customer.Column(1) & " " & Me.InvoiceNumber & ".pdf"
            ' ensure current record is saved before creating PDF file
            Me.Dirty = False
            DoCmd.OutputTo acOutputReport, "rptInvoice", acFormatPDF, strFullPath, True
        End If
    Else
        MsgBox MESSAGE_TEXT1, vbExclamation, "Invalid Operation"
    End If

Exit_Here:
    Exit Sub
    
Err_Handler:
    Select Case Err.Number
        Case FOLDER_EXISTS
        Resume Next
        Case Else
        MsgBox Err.Description
        Resume Exit_Here
    End Select

End Sub

ken sheridan say: You might find the following function helps:

Public Function GetAverage(blnIgnoreNulls As Boolean, blnIgnoreZeros As Boolean, ParamArray aVals() As Variant)

    Const DIVBYZERO = 6
    Dim var As Variant
    Dim n As Integer
    Dim dblTotalVal As Double
    
    For Each var In aVals
        dblTotalVal = dblTotalVal + Nz(var, 0)
        n = n + 1
        If blnIgnoreNulls And IsNull(var) Then n = n - 1
        If blnIgnoreZeros And var = 0 Then n = n - 1
    Next var
    
    On Error Resume Next
    GetAverage = dblTotalVal / n
    Select Case Err.Number
        Case 0
        ' no error
        Case DIVBYZERO
        ' anticipated error
        Case Else
        ' unknown error
        MsgBox Err.Description, "Error"
    End Select
        
End Function

The first two (Boolean) arguments determine whether Nulls and/or zeros are to be taken into account when computing the average, i.e. Nulls can be interpreted as zeros if desired; the third argument is a parameter array of variants which are to be averaged.  If the parameter array contains nothing but Nulls when calling the function then it will return Null if the first argument is True, zero otherwise.  If the second argument is False, i.e. zeros are not included when computing the average, and the parameter array contains nothing but zeros, then Null will be returned.  If the third argument is omitted completely, i.e. an empty array is passed into the function, then Null will be returned regardless of the second two arguments' values.
