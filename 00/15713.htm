don cuthbert say:

I think this is a feature of Insider Preview builds and if a computer is upgraded from Windows 7 or 8 to 10.  I just checked my SSD that was clean-installed with Windows 10 and then upgraded to the November update version of Windows 10.  It only has a single Recovery partition at the beginning of the drive, so I am not getting multiple recovery partitions now with subsequent Windows builds.

There normally is a Recovery partition.  This is used to hold WINRE.WIM which is used to launch Windows RE (Recovery Environment) when Windows needs to be repaired.  Because Insider Preview builds normally allow for a rollback to the previous version, they keep more than one Recovery partition.  As I noted above (back in May) after a sequence of several upgrades, there was a maximum of 3 of these Recovery partitions.  I found that the most recent one was the first one on the disk and the oldest was at the end of the disk.

As your image shows, you have one at the beginning and at the end of your drive.  I wonder if the 499MB partition at the end is an old one from some previous Windows install.  You can use Diskpart to assign it a drive letter, in order to view the files it contains from a Command prompt.  You may have to unhide the \Recovery\WindowsRE folder.  You will be able to tell by the file date of Winre.wim or by reading the ReAgent.xml file contents.

If the partition is old and of no use, you can also use Diskpart to delete the old partition.  If it won't delete it while Windows is running, you may have to boot into Windows RE to do it.  This can be done by using CTRL-ALT-DEL, clicking on the power icon at the bottom right and then, while holding down the SHIFT key, choose Restart.  Another way is to boot with Windows install media and choose Repair your computer, or by using Recovery Drive media (on a USB thumb drive).  Use the menus to choose a Command prompt to run Diskpart.

After deleting the old recovery partition, you can extend your C: partition to reclaim the disk space.

 

EDIT: I just checked my Insider Preview Build 11082, which was upgraded from a clean install of build 10586 and it only has the normal Recovery partition at the beginning of the drive.  So it is not creating multiple Recovery partitions.

Excel Conditional Formatting If and Statement

Hello,

=AND($G$2="renewed",$Q$2="yes")

What is the correct formula?

jasles say:

Your formula is only checking cells G2 and Q2 for all of your conditional formatting rules, so if those two cells say "renewed" and "yes", every cell that has the rule will turn green. That's because you have an absolute reference to those two cells.

You'll need to change $G$2 and $Q$2 to $G2 and $Q2 to only have the absolute reference to columns G and Q, while the rows will change.
