Hi Stacy,

 

Based on this article, you can move students' class notebooks: Move students between Class Notebooks in OneNote.

 

1. Open OneNote and then open both Class Notebooks. For example: Algebra P1 and Algebra P2.

 

2. Right click on each section of the student's private notebook that you need to move to your other class. For example: Homework, Quizzes, or Handouts.

How can I reload userform without clearing the fields? - Microsoft Community

tom wikle say:

How can I reload userform without clearing the fields?

I have worksheets that are opened by a button on a userform. On these worksheets is a button to return to the userform. The code for these buttons is : Userform1.Show. This reloads the userform, but with all the fields cleared. I want to reload the userform without clearing the fields. Is that possible?
jim thomlinson say:

The show command is not clearing the user form. The form is cleared when you unload it. Change your code from

Unload Userform1

or

Unload Me

to

Userform1.Hide

 

This just hides the userform until the next time you show it.

 

By way of an example add a userform to a new workbook. Put a command button and a text box onto the form. Add this code

 

'******************

Private Sub CommandButton1_Click()
    Me.Hide
End Sub

 

Private Sub UserForm_Initialize()
    TextBox1.Text = ActiveSheet.Range("A1").Value
End Sub
'***********************

Add a command button to sheet 1 with the following code

Private Sub CommandButton1_Click()
    UserForm1.Show
End Sub

When you click the button the first time it will pick up the value in cell A1 and put that into the text box. The value in that text box will persist even after you hide and unhide the form.
