I want to calculate the future value of a fund where the payments to - Microsoft Community

richardeglington say:

I want to calculate the future value of a fund where the payments to it are increasing (at a known rate).

I know that the FV only handles fixed payments - is there one that will handle increasing ones?

 

Many thanks,

rich007a say:

You're talking about the FV function in Excel right.

I think you want this formula:       fv = pv (1+r) + pmt ((1+r) - (1+q)) / (r-q)

where
fv future value
pv present value
r interest rate per period
q rate of increase in payment
n number of periods
pmt first period payment

which I found on this page here: http://ask.metafilter.com/19455/Excel-Wizards-How-do-I-get-the-Future-Value-if-the-payments-are-growing-by-a-fixed

 

If you put the inputs in these cells as follows:

B2  pv present value
C2  r interest rate per period
D2  q rate of increase in payment
E2  n number of periods
F2  pmt first period payment

you could use this formula:

=(B2*(1+C2)^E2)+(F2*(((1+C2)^E2)-((1+D2)^E2))/(C2-D2))

You could also set up a UDF using this code:

'========================================
Function FVx(pv As Double, r As Double, q As Double, n As Double, pmt As Double) As Double
' pv present value
' r interest rate per period
' q rate of increase in payment
' n number of periods
' pmt first period payment
'
    FVx = (pv * (1 + r) ^ n) + (pmt * (((1 + r) ^ n) - ((1 + q) ^ n)) / (r - q))
End Function
'========================================

and use it in a worksheet like this:  =FVx(B2,C2,D2,E2,F2)

 

I hope that helps.  (Note, I did not check the maths behind this formula, I just copied from the above referenced site!  You should espcially confirm I put the ()s in the right places.).
