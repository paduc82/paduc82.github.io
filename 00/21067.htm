I am giving to you a simple procedure that will help you build a citeria string for your where statement.  The procedure will accept a value and corresponding fieldname and append it to an existing criteria string.  If a a value is null, nothing will be appended to the criteria.

Function AddCriteria(Expression As Variant, FieldName As String, Optional DataType As Integer = 1, Optional CompOp As String = "=", Optional LogicOp As String = "AND", Optional Criteria As String, Optional Expression2) As String
    'Version    1.0
    'Created    12/9/2010 1:47:11 AM
   
    'Description
    'A simple procedure to build an SQL WHERE statement.  Each call to the function will build the constraint condition, adding the expression, field, comparrison operator, etc., and append
    'it to the current criteria conditions passed in.
   
    'Syntax
    'Criteria_Result = AddCriteria(Expression, FieldName, DataType, [Comparison Operator], [Logical Operator], [Current Criteria])
   
    'Arguments
    'Expression     The expression/value to search for
    'FieldName      The field to compare search expression against
    'DataType       1 = Text
    '               2 = Numeric
    '               3 = Date
    'CompOp         SQL comparrison operators (e.g. =, Like, <>, >, <, Between)
    'LogicOp        SQL logical operator AND/OR
    'Criteria       SQL WHERE criteria passed in to be appended to
    'Expression2    Second expression value to search for when using comparrison operator Between
   
    Dim sCondition As String
   
    If (Not IsNull(Expression)) Then
        If (CStr(Expression) = "Is Null" Or CStr(Expression) = "Not Is Null") Then
            CompOp = ""
        Else
            Select Case DataType
            Case 1
                Expression = Chr(34) & Expression & Chr(34)
            Case 3
                Expression = "#" & Expression & "#"
            End Select
        End If
       
        If (Not IsNull(Expression2) And CompOp = "Between") Then
            sCondition = FieldName & " " & CompOp & " " & Expression & " AND " & Expression2
        Else
            sCondition = FieldName & " " & CompOp & " " & Expression
        End If
       
        If (Len(Criteria) > 0) Then
            Criteria = Criteria & " " & LogicOp & " " & sCondition
        Else
            Criteria = sCondition
        End If
    End If
   
    AddCriteria = Criteria
   
End Function
