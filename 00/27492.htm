Hi VJ

The reason why the counter is not working might be because you have some extra space characters in your data. It's Very common typo

so use the function Trim to eliminate them.

I just tried the macro and it works fine.

*********************************************************************************************

Sub SIMcounter()



Dim SIMRng As Range

Dim L, R, S As String
Set SIMRng = ActiveSheet.Range(Cells(2, "B"), Cells(Rows.Count, "B").End(xlUp))


Counter = 0

For Each Sim In SIMRng

    L = Trim(Cells(Sim.Row, "A").Value)

    R = Trim(Cells(Sim.Row, "C").Value)

    S = Trim(Sim.Value)

        '''IF column col1 low and col 3 KER then count sim in col2

            If L = "low" And R = "KER" And S = "sim" Then

                    Counter = Counter + 1

karl donaubauer say:

Hi,

 

There are some mistakes: The syntax for the subform control is wrong and you can't use the sum() function for a control (never works for controls) or field in a different object. Instead you have to create a text box in the subform, e.g. in the footer or header - so it's not visible in datasheet view. Then in the parent form you can refer to this summing text box in the subform:

 

1) Control source of the new text box "txtSumLineTotals" in the subform: =Sum(LineTotal)

 

Where LineTotal has to be a field in the record source of the subform. If instead LineTotal is already a calculated control you have to repeat the expression of the calculated control, e.g.

=Sum(Quantity*UnitPrice)

 

2) Control source of the text box in the parent form: =[QuoteServiceLineItems]![txtSumLineTotals]

 

Where "QuoteServiceLineItems" has to be what you see in the "Name" property of the subform control in the parent form. This may differ from the name of the form you use as subform (="Source Object" property of the subform control)
