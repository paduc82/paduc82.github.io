bernie deitrick, excel mvp 2000-2010 say:

Just apply data filters to your master list. If you absolutely need separate sheets or workbooks, then one of the macros below will do it.

Bernie

 

Sub ExportSheetsFromDatabase()
'Based on the value in the first column
Dim myCell As Range
Dim mySht As Worksheet
Dim myName As String
Dim myArea As Range

Set myArea = ActiveCell.CurrentRegion.Columns(1).Offset(1, 0).Cells

Set myArea = myArea.Resize(myArea.Rows.Count - 1, 1)

For Each myCell In myArea
On Error GoTo NoSheet
myName = Worksheets(myCell.Value).Name
GoTo SheetExists:
NoSheet:
Set mySht = Worksheets.Add
mySht.Name = myCell.Value
With myCell.CurrentRegion
    .AutoFilter Field:=1, Criteria1:=myCell.Value
    .SpecialCells(xlCellTypeVisible).Copy _
        mySht.Range("A1")
    mySht.Cells.EntireColumn.AutoFit
    .AutoFilter
End With
Resume
SheetExists:
Next myCell
End Sub

 


Sub ExportDatabaseToSeparateFiles()
'Export is based on the value in the desired column
Dim myCell As Range
Dim mySht As Worksheet
Dim myName As String
Dim myArea As Range
Dim myShtName As String
Dim KeyCol As String
Dim myField As Integer

myShtName = ActiveSheet.Name
KeyCol = InputBox("What column letter to use as key?")

Set myArea = Intersect(ActiveCell.CurrentRegion, Range(KeyCol & "1").EntireColumn).Cells

Set myArea = myArea.Offset(1, 0).Resize(myArea.Rows.Count - 1, 1)
myField = myArea.Column - ActiveCell.CurrentRegion.Cells(1).Column + 1

For Each myCell In myArea
On Error GoTo NoSheet
myName = Worksheets(myCell.Value).Name
GoTo SheetExists:
NoSheet:
Set mySht = Worksheets.Add(Before:=Worksheets(1))
mySht.Name = myCell.Value
With myCell.CurrentRegion
    .AutoFilter Field:=myField, Criteria1:=myCell.Value
    .SpecialCells(xlCellTypeVisible).Copy _
        mySht.Range("A1")
    mySht.Cells.EntireColumn.AutoFit
    .AutoFilter
End With
Resume
SheetExists:
Next myCell

For Each mySht In ActiveWorkbook.Worksheets
If mySht.Name = myShtName Then
Exit Sub
Else
mySht.Move
ActiveWorkbook.SaveAs "Workbook " & ActiveSheet.Name & ".xls"
ActiveWorkbook.Close
End If
Next mySht
End Sub

 

 

max, singapore say:

Here's a simple formulas way to deliver it automatically from the source data sheet into each team member's sheet ...
 
Assume your 5 col source data is in sheet: x,
cols A to E, data in row2 down,
with the key col = col A (Names)

List in P1 across the various names eg: Tom, Steve, Kevin, etc
Put in P2:  =IF(OR($A2="",P$1=""),"",IF($A2=P$1,ROW(),""))
Copy P2 across/fill down to cover the max expected extent of source data, eg down to row 500?

Click Insert > Name > Define **
Put under "Names in workbook:":   WSN
Put in the "Refers to:" box:
=MID(CELL("Filename",INDIRECT("A1")),FIND("]",CELL("Filename",INDIRECT("A1")))+1,32)
Click OK
**In Excel 2007, it should be Formulas > Defined Name

The above defines WSN as a name we can use to refer to the sheetname in formulas
It will auto-extract the sheetname implicitly (Technique from a post by Harlan)
Note: The workbook is presumed saved beforehand

Then, in a new sheet named after one of the names eg:  Tom
With the same col headers pasted into A1:E1
Put in A2:
=IF(ROWS($1:1)>COUNT(OFFSET(x!$O:$O,,MATCH(WSN,x!$P$1:$IV$1,0))),"", INDEX(x!A:A,MATCH(SMALL(OFFSET(x!$O:$O,,MATCH(WSN,x!$P$1:$IV$1,0)),ROWS($1:1)),OFFSET(x!$O:$O,,MATCH(WSN,x!$P$1:$IV$1,0)),0)))
Copy A2 across to E2, fill down to say E100, to cover the max expected extent for any single name. Cols A to E will return only the lines for the name: Tom from the source sheet: x, with all lines neatly bunched at the top. Now, you just need to dress the sheet up as desired, then simply make copies of it. Rename each copy as the other names (a simple one time job) and you'd get the corresponding results for each of those names. Adapt to suit ..

Have you tried to perform a soft reset? If not please do so.

The soft reset is the same thing as pulling your battery (on devices where that is possible). This method will not remove anything at all from your phone and is completely safe to attempt in any problem situation.

You perform the soft reset by having your phone turned on. Then press and hold the volume down and Power buttons until your phone vibrates and restarts. Note that you should not touch the device screen during the process (not even when the 'slide down to power off' prompt appears).

Example of situations where the soft reset should solve the problem:

frankli_msft7108 say:

hi basdijkstra,

 

to meet this request, we need to combine two commands in one script. the following example shows how to export the license information for all the shared mailboxes(including unlicensed and licensed).

1. install the windows azure active directory module for windows powershell. you can access it via the following link:

    https://msdn.microsoft.com/en-us/library/azure/jj151815.aspx#bkmk_installmodule 

 

2. run the following command to connect the msolservice:

$cred =get-credential

connect-msolservice -credential $cred

3. run the following script to export the csv file. the file cound be found at the location c:\userlist.csv.

$reportpath="c:\userlist.csv"

add-content -value ("userprincipalname"+","+"displayname"+","+"islicensed"+","+"licenses") -path $reportpath

$shareduser = get-user -recipienttypedetails "sharedmailbox"

foreach ($user in $shareduser)

{

$msoluser = get-msoluser -userprincipalname $user.userprincipalname

add-content -value ($msoluser.userprincipalname+","+$msoluser.displayname+","+$msoluser.islicensed+","+$msoluser.licenses) -path $reportpath

}
