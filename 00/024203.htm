The most likely cause is a damaged Data subkey in the registry. (Documentation of what's stored in that key is skimpy, but there are no specific values for those functions in the Options subkey, so Data is the alternative.)

Use the procedure called "Method 3: Delete the Word Data registry subkey" in the article https://support.microsoft.com/en-us/help/921541/how-to-troubleshoot-problems-that-occur-when-you-start-or-use-word. One additional comment on step 2: If you run Windows as a standard (non-Administrator) account, right-click the regedit program in the search results, click More, and click Run as Administrator; otherwise you won't be allowed to make any changes in the registry.

Be sure to follow the link to https://support.microsoft.com/en-us/help/322756/how-to-back-up-and-restore-the-registry-in-windows to back up the subkey

 

HKEY_CURRENT_USER\Software\Microsoft\Office\14.0\Word\Data

before removing it, so you can restore it if the problem isn't solved.

When you restart Word in step 8, the program will create a new Data key with the same default settings as when it was first installed. If the problem is solved, you'll have to go through the Options dialog and its children to recreate the settings you want. [In versions up to Word 2003 it was possible to save most of the settings for restoration, as described here, but that no longer works.]
