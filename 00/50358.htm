MAC Filtering
 
  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter "root" for the username and "12345" for the password.
  4. Select "Advanced Setup" from the menu on the left.
  5. Select "Security" from the sub menu and then "MAC Filtering" below that.
  6. If the "MAC Filtering Global Policy" shows as "Blocked", select "Change Policy" to set it as "Forwarded".

DMZ
 
  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter "root" for the username and "12345" for the password.
  4. Select "Advanced Setup" from the menu on the left.
  5. Select "NAT" from the sub menu, then "DMZ Host" below that.
  6. Enter the IP address of the Xbox in the "DMZ Host IP Address" box.
  7. Select Save/Apply.



Disable DMZ
 
  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter "root" for the username and "12345" for the password.
  4. Select "Advanced Setup" from the menu on the left.
  5. Select "NAT" from the sub menu, then "DMZ Host" below that.
  6. Delete the IP address that is entered in the "DMZ Host IP Address" box.
  7. Select Save/Apply.



Port Forwarding
 
  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter "root" for the username and "12345" for the password.
  4. Select "Advanced Setup" from the menu on the left.
  5. Select "NAT" from the sub menu, then "Virtual Servers" below that.
  6. Select Add.
  7. Select the radio button for "Custom Server" and enter Xbox.
  8. Enter the IP address of the Xbox into the "Server IP Address" box.
  9. Enter 3074 into the "External Port Start", "External Port End", "Internal Port Start", and "Internal Port End" in rows 1 and 2.
  10. Select "TCP" from the "Protocol" drop down in row 1, and select "UDP" from the "Protocol" drop down in row 2.
  11. Select Save/Apply.



Disable Port Forwarding
 
  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter "root" for the username and "12345" for the password.
  4. Select "Advanced Setup" from the menu on the left.
  5. Select "NAT" from the sub menu, then "Virtual Servers" below that.
  6. Select the radio or check box next to the rule you would like to remove.
  7. Select Remove.



Firewall
 
  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter "root" for the username and "12345" for the password.
  4. Select "Quick Setup" from the menu on the left.
  5. Select Next.
  6. The information on the WAN IP settings page should be already configured. Select Next.
  7. Check or uncheck the "Enable Firewall" check box based on current configuration. (By default the Firewall will be enabled if the device is configured with PPPoE.)
  8. Select Next leaving the other settings as default until the quick setup is complete.
  9. Select Save/Reboot on the last screen.



Firmware Update
 
Due to the age of this device, the firmware is no longer available. This feature is unavailable.



Restore Factory Defaults
 
Through Hardware
 
  1. Plug the computer directly into the devices LAN port via an Ethernet cable.
  2. Locate the Reset button on the device.
  3. Using a toothpick/pen/paper clip, press and hold the button for 10 seconds.
 
Through Firmware
 
  1. Plug the computer directly into the devices LAN port via an Ethernet cable.
  2. Open a web browser.
  3. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  4. When prompted, enter "root" for the username and "12345" for the password.
  5. Select "Management" from the menu on the left.
  6. Select "Settings" from the sub menu, then "Restore Default" below that.
  7. Select "Restore Default Settings"



Bridging Guide
 
  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter "root" for the username and "12345" for the password.
  4. Select "Quick Setup" from the menu on the left.
  5. Uncheck the "DSL Auto-connect" check box.
  6. Select Next.
  7. Do not change the VPI and VCI numbers unless your ISP has given you specifics to enter.
  8. Select Next.
  9. Select the "Bridging" radio button.
  10. Select Next.
  11. Check the "Enable Bridge Service" checkbox, and enter a "Service Name".
  12. Select Next.
  13. Leave the default IP address and subnet mask. Select Next.
  14. Uncheck the "Enable Wireless" checkbox. Select Next.
  15. Select Save/Reboot.



Multiple Console Support
 
We have confirmed that this network device is unable to support more than one console with an Open NAT.



Known Issues
 
The CT-5071T is a wired ADSL gateway with one USB port, one Ethernet port and NO wireless broadcast capability.
