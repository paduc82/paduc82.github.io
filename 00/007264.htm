ebkh say:

What is the difference between quick parts and building blocks in 2010?

Is there a difference between quick parts and building blocks in 2010? The training I'm taking, keeps refering to building blocks and I'm not sure if those have replaced what used to be quick parts. 

suzanne s. barnhill say:

A very good question, as the distinction has never been clear (to me, at least). Quick Parts is an item (button) on the Insert tab. It opens a menu that includes AutoText, fields, and document properties. The menu also includes the Building Blocks Organizer but no building blocks per se (though AutoText is a type of building block). Where you actually see building blocks is in other galleries on the Insert tab such as the Header, Footer, and Page Number galleries, the Table of Contents gallery, and the Text Box gallery.

If you look at the Building Blocks Organizer, you can see what sorts of things are saved as building blocks. Oddly, although these include text boxes, you cannot save a custom text box as a quick part (Save Selection to Quick Parts Gallery); instead you have to save it as a text box (Save Selection to Text Box Gallery).

Accepted insertion color change in Word 2016

I have a query. I am using Word 2016. I want to know about a feature in which once I accept any changes in Word document coming via Track change should be highlighted with a color of my choice, say, yellow. Is it possible?

i.e. once I accept the change, the place should have a background color of yellow.

Say in a line there are 8 words, and someone inserted two words in between, and I accepted them. What I want is once I accept those two words, the background color of those two words should change .

doug robbins - mvp office apps & services (word) say:

Use a macro containing the following code:

ActiveDocument.TrackFormatting = False
Dim arev As Revision
For Each arev In ActiveDocument.Revisions
    With arev
        If .Type = wdRevisionInsert Then
            .Range.Select
            If MsgBox("Do you want to accept the insertion of " & .Range.Text & "?", vbQuestion + vbYesNo, "Accept and color insertions") = vbYes Then
                .Range.Shading.BackgroundPatternColor = wdColorYellow
                .Accept
            Else
                .Reject
            End If
        End If
    End With
Next arev
