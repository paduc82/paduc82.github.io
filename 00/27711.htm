karl timmermans [outlook mvp 2012-2018] say:

If both computers have the same settings, then the first one that accesses the account will get the emails - the other won't.

 

There is no hard and fast rule as to how to "best" configure the account since it all depends on how you use your computers. If you absolutely have to have both computers always contain all emails and there is no pattern in how these are used in terms of timing etc - then your only option would be to "manually" delete items once you're certain that both computers have all the data.

 

An alternative approach would be to copy the PST file from your "primary" computer to the secondary one but again this depends on your particular "work-flow" situation.

 

Yet another scenario that works in some situations is to use a program like "TeamViewer" whereby you remotely access Outlook on your "primary computer". Using this approach, your email will always be up-to-date since the second computer will actually be using Outlook on the primary system via remote access (but that will require that you always have an internet connection active).

 

Basically, using PST files (POP accounts) when multiple computers are being is generally the worst approach available.
