nic t. say:

doug robbins - mvp office apps & services (word) say:

ALT+SHIFT+t inserts a Time field

which will be updated to display the present time whenever the fields in the document are updated.

If you select the time immediately after inserting it and then press CTRL+SHIFT+F9, it will be converted to ordinary text and will not then update.

However, an easier way of doing it might be to create a macro containing the command 

Selection.InsertBefore Format(Time, "Medium Time")

and assign it to a button on the Quick Access Toolbar, or to ALT+SHIFT+T and then the time will just be inserted as ordinary text from the get go.

You can replace "Medium Time" with either "Short Time" or "Long Time", which will have the following effect

04:35 PM - Medium Time

16:35 - Short Time

4:35:39 PM - Long Time

(assuming you're using English (US), otherwise there will be a different language ID instead of 1033).

Since the real experts haven't yet responded, let me take a crack at it. I believe the displays are different because your various computers' "Save" options have been set differently. When a computer (or a single file) is set to "Compatibility Mode" (i.e., compatibility with Word 97-2003), only the older WordArt options are made available. Here's how to change this: In Word 2010, go to File>Options>Save. The resulting screen will offer the option "Save files in this format:" (it's right near the top).  Select "Word document (*.docx)" and click "OK." The "*.docx" format is used for Word 2010 files, and documents created after selecting that option should give you the Word 2010 WordArt options.

 

However, previously saved files that contain Word 2003 WordArt (usually *.doc files) will revert to that array of choices when you open them--even if you subsequently save them as *.docx files. And if you save *.docx files "down" to Word 2003, any embedded 2010 WordArt will undergo some often unpleasant changes. windowappreviews say:

When any sites show This site can't be reached error it happens most of the times due to dns issues. 

It is an easy fix. Just follow this solution to flush dns.

  • Go to Command Prompt by pressing windows key and X and select Command Prompt.
  • Now, just paste these commands one by one and after every command hit enter.
  • Commands: ipconfig /release

ipconfig /all

ipconfig /flushDNS

ipconfig /renew

netsh int ip set DNS

netsh winsock reset

Lastly, just reset your PC or the device and see if it makes any changes.
