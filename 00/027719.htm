"Pages" do not actually exist in a Word document (or any other word processor file).  Word makes up pages on the way out to the printer or to your screen: they don't exist in the document.  Word simply stacks the text one character at a time from the top of the document (the top left, if you are writing in English...) and when it has stacked as many as will fit on the paper size you have selected, it displays a new page and continues stacking.  This can go on for thousands of pages.

So to create another page: type more content.

julie sheets say:

joseph319 say: Not a problem...The code below is late bound so there is no need to make a reference to ADO.

Make sure to first create a database and define a table with the fields that you want to capture prior to running.

This will append to that table everytime you run it.  I add a timestamp so when i query for reports i know what data to pull.

Sub MSProjectUpload()

Dim cn As Object    'ADODB.Connection
Dim rs As Object  'ADODB.Recordset
Dim tsK As Task
Dim dbPath As String

On Error Resume Next

dbPath = "C:/MSProject.accdb"
'+++Connect to Access Database+++
    Set cn = CreateObject("ADODB.Connection")   'New ADODB.Connection
    cn.Open "Provider=Microsoft.ACE.OLEDB.12.0; " & _
        "Data Source=" & dbPath & ";"

'+++Open the Recordset+++
    Set rs = CreateObject("ADODB.Recordset")   'New ADODB.Recordset
    rs.Open "tbl_New_Upload", cn, adOpenKeyset, adLockOptimistic, adCmdTable


'+++Begin Task Retrieval+++
    For Each tsK In ActiveProject.Tasks
        With rs
            .AddNew                        ' add values to each field in the record
            .Fields("UID") = tsK.UniqueID
            .Fields("Task_ID") = tsK.ID
            .Fields("WBS") = tsK.WBS
            .Fields("Task_Name") = tsK.Name
            .Fields("Finish") = tsK.Finish
           
            ' add more fields if necessary...
            .Update                         ' stores the new record
        End With
    Next tsK

'+++CleanUp+++
    rs.Close
    Set rs = Nothing
    cn.Close
    Set cn = Nothing
End Sub
