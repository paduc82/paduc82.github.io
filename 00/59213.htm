The Visual C++ Redistributable is a DLL (Dynamic Link Library) file required by programs or games built using Microsoft‚Äôs Visual Studio software development environment. When a program requires a DLL or some other supporting file to run, this is called a dependency. The MSVCR.DLL stands for:

  • MS ‚Äì Microsoft
  • V ‚Äì Visual
  • C ‚Äì C++
  • R ‚Äì Redistributable.

Browsing your Apps and Features Setting or Programs and Features Control Panel, you might see multiple versions of the redistributable installed. Some apps might even require updated versions of the Visual C++ Redistributable such as 2010 SP1 or 2012 Update 4.
