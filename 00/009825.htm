Duration is always working days.  So, my guess is your calendar is still the Standard  - defined as Monday through Friday 8:00 am - 12:00 pm, 1:00 pm - 5:00 pm.  You have altered the definition of "Month" in File > Options to 31 days.  With that definition there are 8.42 "31 day" working time periods from January 1 to Dec 31.  If you want to count every day as a working day (including Saturdays and Sundays) you'll need to change the Standard calendar to make Saturday & Sunday working days.  If you want to count 365 calendar days - enter the duration as 365 edays. dianeoforegon say: Look in the Trash first.

Entourage expects the Microsoft User Data folder to be in your Documents folder. For OS X users, that's the Documents folder in your User's folder. When Entourage opens in a new blank Identity, it's usually User error.

Most likely you have accidentally moved your data. Hopefully, you have not deleted the file. If you have deleted the trash, it's gone. Sorry!

Select Switch Identity (under Entourage in the Menu bar) and see if your old identity is still listed.

To find your old database, search for "Microsoft User Data" (without the quotes). You can also search on "Office 2008 Identities" (replace 2008 with either 2004 or X if you are using an older version)

If you have not used a new identity in Entourage, then copy the entire Microsoft User Data folder to the correct location.

Fix if you have more than one Microsoft User Data Folder...

Move the entire identity folder you want to use to the Office Identities folder that Entourage is recognizing. Your identity is named "Main" by default. Be sure to rename one so you do not overwrite. Your identity folder can contain more than one identity. Switch Identities to view your old mail. If you have not used the new identity, delete it. If you have used it and want to combine the new mail with your original database, drag each folder to the desktop from one identity. This creates an MBOX file. Switch identities. Drag the MBOX files into the folder window of Entourage.

You can delete the old folder after you are satisfied with the transfer.

deepestblue---01 say:

Restore factory settings: Type *#7780# at the stand-by screen. Enter lock code, default is 12345

- You will not lose data

- All settings - profile, display, ringtones, etc will be set to factory defaults

 

Soft reset: Type *#7370# at the stand-by screen. Enter lock code, default is 12345

- Backup your data before this procedure

- Your phone memory will be wiped clean

- Your phone will be in factory condition

 

Hard reset: Turn off your phone, press and hold down the Shift, Space-bar and Del keys and power on the phone without releasing them. You will be taken to the region selection screen if it was successfull, after powering up.

- Backup your data before this procedure

- Your phone memory will be wiped clean

- Your phone will be in factory condition

- This is a last resort, to be done only if nothing else works.

 

Reset procedures  do not change your firmware, so you'll have the same firmware that you were running before reset.

 

If you are not running the latest firmware, I'd recommend updating to the latest one using FOTA and then doing a soft reset. See if your problems are solved, then restore your backup, one item at a time...
