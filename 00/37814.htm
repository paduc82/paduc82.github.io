mike hall... always say:
There is no way to get a Windows XP, Vista or Win 7 desktop 32-bit operating system to use all 4gb installed RAM. This is set for what Microsoft call compatibility issues.

If you want/need to be able to use all 4gb or more, yoiu will have to upgrade to a 64-bit OS.

Yes, 64-bit requires more RAM than 32-bit in order to function properly.

A motherboard which only supports a total of 4gb RAM will generally not have the memory mapping feature which allows 64-bit to use all of the 4gb installed RAM, so will only be able to use around 3.7gb.

RAM usage is not the most compelling reason to switch to a 64-bit OS. Most general users will not even get close to 2gb RAM usage unless running heavy memory intensive applications.

The most compelling reason is security because 64-bit makes very little concession to backwards compatibility in a time when security was not seen as an issue.

Do NOT update your BIOS unless you have a problem related to BIOS. It would appear that you do not.
