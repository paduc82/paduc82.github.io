a. user say:

The simplest way to do this is probably to add a nested SKIPIF field at the top of your mail merge main document, like this:

{ SKIPIF { MERGEFIELD datacolumn \@YYYYMMDD } <> { DATE \@YYYYMMDD } }

where all the {} are the "special field braces" that you can insert using ctrl-F9

I don't especially like this, as "SKIPIF" was supposedly deprecated a very long time ago (and it means that Word still has to process all the records in Excel), but all the alternatives are arguably even harder to implement.

don cuthbert say:

Once all of the downloading is complete you don't need the Internet connection. But I have found that when watching closely, there are sometimes additional portions downloaded. The new UUP process seems to do 3 downloads. The first one is very small and is processed to prepare for the rest of the downloading and processing of files. Then the main download is done. The final download that I have observed is a small ESD file and I believe there is some processing that takes place prior to that.

Because this is designed to be a background process, with low priority, it should be pretty resilient to interruption, as the average user might not even notice it is happening until the computer is ready to Restart for the actual OS upgrade. They might power off or restart the computer at various times while the downloading and processing is happening, but before it is done.

fraser xu msft say: Hi Anna,

The best way to share a calendar is to let all staff to access it directly from SharePoint. For iPad users, they can access it using Safari.

If these staff have Office 365 account that belong to the same Office 365 for Business tenant as you did and the team site is created with default settings, they will be able to access the calendar without extra action.

If these users are external users, then please following steps below to share the calendar with them:
1.If you don’t want them access other contents in the team site, please go to Site content> New subsite to create a new subsite. (Note: Under Template Selection please choose Team site. Under Permissions please choose Use unique permissions.)
2.Create a new calendar in this subsite.
3.Open this new calendar and click CALENDAR tab> Current View> All Events.

4.Click List tab> Shared with.

5.Enter those external users’ email address in Invite people section.


Regards,
Fraser
