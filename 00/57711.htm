First, confirm the correct password is getting entered with some simple checks before resorting to more extreme measures:

  1. Click on the “eye” icon in the password field and view the password before hitting Enter.

  2. Verify the keyboard layout wasn’t changed during the upgrade, causing you to enter unexpected characters when typing.  Press the Windows Key and the Spacebar at the same time to check the keyboard language and layout.  While holding the Windows Key, press the spacebar repeatedly to change the keyboard layout.

  3. From another device, verify that you can successfully sign into http://account.live.com with the same Microsoft account email and password.

Now that you’ve confirmed you’re entering the right password, if you upgraded from Windows 8.x on a Microsoft account, and you’ve never been able to sign into Windows 10, then you are likely impacted by this issue.  To recover, you’ll have to roll back the operating system to Windows 8.x, redo the upgrade to Windows 10, and make sure to enter the correct password on your first attempt:

  1. Insert Windows 10 installation media and boot to the media to enter the Windows Recovery Environment.  You can create media from another computer using the Media Creation Tool. 

  2. In the Windows Recovery Environment, select Troubleshoot > Advanced options.

  3. Select Go back to the previous build.  If that option doesn’t appear, select See more recovery options to view it on a second page.

  4. Follow the on-screen instructions to roll back to your previous version of Windows.

  5. Begin the upgrade to Windows 10 again and then sign in.
