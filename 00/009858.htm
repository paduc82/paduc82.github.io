Sub Macro1()

hansv mvp say:

When you record a macro, and open a dialog, the recorded macro will *not* display the dialog but merely perform the action of the dialog.

For example, if you display the File > Open dialog and open document A while recording a macro, the macro will open document A without displaying a dialog.

To display a dialog in a macro, you must write the Visual Basic code yourself, for example to display the File > Open dialog:

 

    Application.Dialogs(wdDialogFileOpen).Show
