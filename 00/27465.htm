Anyway, I'm not sure if all of this code is required, but it works:

                    

Sub TestMacro()

    Dim v As Variant
    Dim myColumns As String
    Dim i As Integer

I"ve never really understood why anyone would execute queries via
the QueryDefs collection, instead of just directly. This is going to
work just fine:

  Debug.Print db.OpenRecordset("SELECT @@IDENTITY")(0)

longer to run than it will take to initialize CurrentDB a second
time.

Outlook 2010
  • Navigated to Developer on the Ribbon Menu and left-clicked on COM Add-Ins, which displayed the list of Add-Ins available.
  • Left-clicked on COM Addin for Microsoft Office Outlook Hotmail Connector to highlight the line item.
  • Left-clicked on the Remove button to remove the highlighted COM Add-in, which could not be removed. Received the following error dialog box message: Microsoft Outlook - This add-in is installed for all users on this computer and can only be connected or disconnected by an administrator. OK

Control Panel
  • Navigate to Control Panel\Programs\Programs and Features\Uninstall or change a program and unistall Microsoft Outlook Office Connector.
Outlook 2010
  • Navigate to Outlook 2010 and archived all categorized emails from one (File Cabinet) of only two folders in *** Email address is removed for privacy *** account.
  • Archived 2,445 categorized emails to an archive file on local drive in standard Outlook folder location.
  • Deleted all 2,445 categorized emails in the File Cabinet folder.
  • Left-click on File in the Oulook 2010 Menu Bar.
  • Left-click on the Account Settings Button
  • Left click on the Account Settings... menu.
  • Left-clicked on Hotmail email address in the list to highlight it.
  • Left-clicked on the X Remove button.
Microsoft
  • Navigate to http://office.microsoft.com/en-us/outlook-help/microsoft-office-outlook-hotmail-connector-overview-HA010222518.aspx and download Microsoft Outlook Hotmail Connector.
  • Run the executable to install Microsoft Outlook Hotmail Connector again.
Outlook 2010
  • Left-click on File in the menu bar.
  • Left-click on + Add an Account button.
  • Key in first and last name in the Your Name: field.
  • Key in Hotmail email address in the E-mail Address: field.
  • Key in password in the Password: field.
  • Re-key password in the Retype Password: field.
  • Left-click the Next button and Outlook 2010 automatically configures the connection strings.
  • Hotmail and Outlook 2010 synchronization starts.
Note: If you use categories to categorize your emails, you will not loose category or categories assigned to each email. At least that was my experience.
