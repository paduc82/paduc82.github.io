Random Number Generator - Microsoft Community

nickpandelos say:

Random Number Generator

Afternoon. I am looking to create a random number generator. Here is the catch. I want to take a letter from A-Z and use a random number between a speciffic range. If that didn't make sense, here is an example.

A=1-34, B=1-38, C=1-134 .... and so on.

One cell would select a random letter A-Z. Lets say it selected B; then a second cell would come up with a random number between 1-38.

paulcordts say:

You just need 2 RANDBETWEEN functions and a table for your A=1-34, B=1-38, C=1-134 .... and so on.

In A2 use =CHAR(RANDBETWEEN(65,90))  65 is the Char number for A, 90 for Z, so this generates a random number between 65 and 90 and then returns the corresponding A-Z character for that number

In B2 use =RANDBETWEEN(VLOOKUP(A2,D2:F27,2),VLOOKUP(A2,D2:F27,3))

This will lookup the min and max values for each corresponding randomly generated letter.

for 64 bit office -

C. Open an desired office application
