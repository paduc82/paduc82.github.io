Minimize a popup form to a certain position - Microsoft Community

huaca2 say:

Minimize a popup form to a certain position

When I minimize a certain popup form with DoCmd.Minimize in a button macro, it goes to its default position on bottom left corner of the windows, so hiding part of my main form.

I would like to minimize it to a certain position near the bottom right corner, using VBA or just the macro in the button, but I cannot find  how to achieve this.

Any help would be much appreciated.

dirk goldgar say:

I have played with your suggestion about not minimizing the form, but resizing it, but cannot find a working solution, because, when using the DoCmd.MoveSize command from a button to resize and put the form in the adequate position, there is not option to restore the form (as a matter of fact it is already "restored"), so it can only be minimized to the left bottom corner or maximized to full screen!

Yes, it presents a problem.  My assumption was that the popup form would be (pseudo-)minimized and (pseudo-)maximized by code and not by direct user interaction, so wouldn't need to worry about that. 

Here's an approach that seems to work so long as the Access application is maximized.  It traps the form's Resize event, determines whether the form is being minimized, and if so, moves it all the way to the right of the Access window. It uses Nicole Calinoiou's brilliant FormWindow class, which can be downloaded from the Access Web here:

   

    http://www.theaccessweb.com/forms/frm0042.htm 

    Forms: Move and Resize form windows from code

With that class module imported into your database, then you can create an event procedure for your popup form's Resize event like this:

Private Sub Form_Resize()

    Dim fw As New clFormWindow
   

        fw.hwnd = Me.hwnd
        fw.Left = (fw.Parent.Left + fw.Parent.Width) - fw.Width

End Sub

For me, under the conditions you describe, that seems to work, although it looks a bit odd because the animation for minimizing the form shows it moving down and to the left, but then it appears on the right.

By the way, I tend to agree with Tom van Stiphout that overriding the default Access behavior can cause confusion.  Maybe it would be better to just hide your popup form, and then show it again when some appropriate event occurs.  However, I could never resist a challenge.

Convert Legacy Form Fields to Content Controls - Microsoft Community

neitherthisnortheother say:

Convert Legacy Form Fields to Content Controls

I am updating a fillable form created in Word 2003. I would like to change the checkboxes and text fields to the content control equivalents. What is the simplest way to do this? Thank you.
greg maxey say:

But that wouldn't be much fun ;-)

See: http://gregmaxey.mvps.org/word_tip_pages/installing_employing_macros.html

for instructions to employ the VBA code provided

Sub ScratchMacro()
'A basic Word macro coded by Greg Maxey
Dim oFF As FormField
Dim oCC As ContentControl
Dim strText As String
Dim bVal As Boolean
Dim arrVals() As String
Dim lngIndex As Long, lngDDItem As Long
Dim oRng As Range

  For lngIndex = ActiveDocument.FormFields.Count To 1 Step -1
    Set oFF = ActiveDocument.FormFields(lngIndex)
    Select Case oFF.Type
      Case 83
        ReDim arrVals(1 To oFF.DropDown.ListEntries.Count)
        For lngDDItem = 1 To oFF.DropDown.ListEntries.Count
          arrVals(lngDDItem) = oFF.DropDown.ListEntries(lngDDItem).Name
        Next lngDDItem
        strText = oFF.Result
        Set oRng = oFF.Range
        oFF.Delete
        Set oCC = ActiveDocument.ContentControls.Add(wdContentControlDropdownList, oRng)
        oCC.DropdownListEntries.Add oCC.PlaceholderText, vbNullString
        For lngDDItem = 1 To UBound(arrVals)
          oCC.DropdownListEntries.Add arrVals(lngDDItem), arrVals(lngDDItem)
          If oCC.DropdownListEntries(oCC.DropdownListEntries.Count).Value = strText Then
            oCC.DropdownListEntries(oCC.DropdownListEntries.Count).Select
          End If
        Next lngDDItem
      Case 71
        bVal = oFF.CheckBox.Value
        Set oRng = oFF.Range
        oFF.Delete
        Set oCC = ActiveDocument.ContentControls.Add(wdContentControlCheckBox, oRng)
        oCC.Checked = bVal
      Case 70
        strText = oFF.Result
        Set oRng = oFF.Range
        oFF.Delete
        Set oCC = ActiveDocument.ContentControls.Add(wdContentControlText, oRng)
        oCC.Range.Text = strText
     
    End Select
  Next
lbl_Exit:
  Exit Sub
 
End Sub
