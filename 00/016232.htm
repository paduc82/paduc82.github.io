Sub AddDays()
   Dim m As Integer, y As Integer, d As Integer
   
   Application.ScreenUpdating = False
   m = Application.InputBox(Prompt:="Month: ", Default:=Month(Date))
   y = Application.InputBox(Prompt:="Year: ", Default:=Year(Date))
   
   For d = 2 To Day(DateSerial(y, m + 1, 0))
      Sheets("1").Copy after:=Sheets(d - 1)

   Application.ScreenUpdating = True

Resize Row to Fit Picture - Microsoft Community

dak say:

Resize Row to Fit Picture

Can a macro auto resize a row to fit a picture upon pasting it into a given cell?

This would be incredible help!

ossiemac say:

I can't make the code run automatically when you paste the picture but you can have a button on the worksheet to run the code. You will need to select the required picture (ie. so you can see the sizing handles) and then click the button. If you have just pasted the picture then it should be already selected.

To install the code

  1. Alt and F11 to open the VBA editor.
  2. Select menu item Insert -> Module.
  3. Copy the code below and paste into the editor.
  4. Close the VBA editor (Cross top right of VBA editor screen. Background turns red when you hover cursor over it)
  5. On the worksheet screen, select the Developer ribbon. (If not visible then right click on a blank area of any ribbon and select "Customize the ribbon" and check the box against "Developer" in the right column of the dialog and then click OK).
  6. Click on Insert icon (in the controls block) and then in the Forms controls in top half of dropdown icons select button and then move the cross cursor on to the worksheet and click and hold the left button and drag out to required size.
  7. In the Assign macro dialog select "ResizePictureRow" and click OK.
  8. Now to reset the required row height, select the required picture and then click the button.
  9. Right clicking on the inserted button at a later time will select the button and you can move or resize as required and edit the button name.

Sub ResizePictureRow()
    Dim shpPic As Shape
   
    On Error Resume Next
    Set shpPic = ActiveSheet.Shapes(Selection.Name)
    On Error GoTo 0
   
    If Not shpPic Is Nothing Then
        If shpPic.Type = msoPicture Then
            ActiveSheet.Rows(shpPic.TopLeftCell.Row).RowHeight = shpPic.Height
        End If
    End If
End Sub

ossiemac say:

The code below will perform both the resizing of the picture and the row height as follows.

  • Ensures the picture is positioned with the top left corner at cell A9. (You can paste the picture anywhere without accurately selecting the destination.)
  • Adjust the width of the picture to fit A9:K9.
  • Adjust the Row height to the same as the picture.

Install the code and create a button as per my previous guidelines. You do need to ensure that the picture is selected before clicking the button otherwise the code will not do anything.

Sub ResizePicAndRow()
    Dim shpPic As Shape
   
    On Error Resume Next
    Set shpPic = ActiveSheet.Shapes(Selection.Name)
    On Error GoTo 0
   
    If Not shpPic Is Nothing Then
        If shpPic.Type = msoPicture Then
            'Locking Aspect Ratio ensures that when picture width is set
            'the height automatically adjusts by the same ratio.
            With shpPic
                .LockAspectRatio = True
                .Top = ActiveSheet.Range("A9").Top
                .Left = ActiveSheet.Range("A9").Left
                .Width = ActiveSheet.Range("A9:K9").Width
                ActiveSheet.Rows(shpPic.TopLeftCell.Row).RowHeight = .Height
            End With
        End If
    End If

End Sub
