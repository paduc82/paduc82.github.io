jay freedman say:

Insert a unique bookmark in the paragraph you want to refer to. Then, at the point of the reference, replace the # with a cross-reference (References > Cross-reference) to the bookmark, selecting "Paragraph number" in the "Insert reference to" dropdown.

Note that bookmark names can't contain spaces or any punctuation other than underscores. Since the paragraph numbers may change, I would not use them in the bookmark names.

Any time a referenced list item's paragraph number changes, select all (Ctrl+A) and update fields (F9) to update the cross-references. (Alternatively, press Ctrl+P to go to the Print page, and then press Esc to go back to the document body. This requires having the "Update fields before printing" option turned on.)

suzanne s. barnhill say:
If the numbered paragraphs are autonumbered (as these appear to be), it is not necessary to manually bookmark them, as the paragraph number can be inserted from the Cross-reference dialog as the paragraph number of a "Numbered item."

jay freedman say:
Depending on how the numbering scheme works, it may still be preferable to use bookmarks. If there are lots of groups of steps, each of which restarts from 1, the list of "Numbered items" in the Cross-reference dialog can become very hard to use.
