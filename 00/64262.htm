ruwim.b say:

You have Bitdefender security application installed.

 

Your webcam is blocked by Bitdefender. Bitdefender Privacy Filter Driver (bdprivmon.sys) is inserted into webcam's driver stack.

 

Stack: \FileSystem\bdprivmon,\Driver\ksthunk,\Driver\SPC620m,\Driver\SPC620,\Driver\usbccgp

 

https://www.bitdefender.com/consumer/support/answer/2028/

 

I don't know if a similar "protection" is implemented for the microphone, but everything is possible with Bitdefender application.

 

For testing purpose I would suggest completely to uninstall Bitdefender and see if this will help with you Skype issues.

 

https://www.bitdefender.com/uninstall/
