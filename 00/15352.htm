Hello
A contact which is linked to multiple accounts or phone numbers, follows a order. For example, a contact which has four different phone numbers, one Email-ID and contacts from Skype and Facebook too, will show the information for that contact in the following order:

  1. Mobile 1
  2. Mobile 2
  3. Home
  4. Work
  5. Email ID
  6. Skype ID
  7. Facebook ID

Now, as you have stated, for a contact which is connected to a mobile number, a Facebook ID and a Gmail ID, will obviously be pointing to it's first priority, which is the mobile number.  That's why, the contact is now showing the Nickname as the contact ID after linking all those accounts. If you want to change that Nick to Firstname-Surname combination, you must change the nickname connected to that mobile number (from your old SIM card), under that contact.

shane devenshire say:

A blank cell is treated as 0 which is a value <.85

You will need to modifiy your conditional format


In 2003:
1. Select the cells you want to format D1:D100 in this example
2. Choose Format, Conditional Formatting
3. Choose Formula is from the first drop down
4. In the second box enter the formula:

=AND(D1<>"",D1<0.85)

5. Click the Format button
6. Choose a color on the Patterns tab (or any available option)
7. Click OK twice.
