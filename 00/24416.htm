vasil l. michev say:

Well how do you distinguish student accounts from faculty ones? Just apply that criteria to filter them out either directly in PowerShell or via CSV file, then pipe the output to Set-Mailbox. For example:

Get-Mailbox -Filter {CustomAttribute11 -eq "student"} | Set-CASMailbox -OwaMailboxPolicy "blabla"

Where I've used CustomAttribute11 as the attribute that designates the account as student.

Or via CSV file:

import-csv .\users.csv | % {Set-CASMailbox -Identity $_.UserPrincipalName -OwaMailboxPolicy "blabla"}

Where the users.csv file has a column named UserPrincipalName, containing all the users in question.
