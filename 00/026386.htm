shane devenshire say:

You will need code, the question is when do you want it to run.  It will not run because you change the format of the cell.

Private Sub Worksheet_Change(ByVal Target As Range)
   Dim isect As Range
    Set isect = Application.Intersect(Target, Range("A1:A10"))
    If Not isect Is Nothing Then
        Target.Offset(0,1).PasteSpecial Paste:=xlPasteFormats
    End If
End Sub


1. To add this code to your file, press Alt+F11, 
2.  In the VBAProject window, top left side, find your sheet name under your file name and double click it.
3.  Paste in or type the code above.

This code would execute when the value in a cell in column A1:A10 was changed
