mcakins say: Awesome!

keith a. miller say:

Not sure if this applies to Windows 8, but in Windows 10, the Properties' dialog for the Screenshots folder has a 'Location' tab, which allows you to move the Screenshots folder like any other User folder.  When you change the location from the default, a value named {B7BEDE81-DF94-4682-A7D8-57A52620B86F} will be added under:

HKCU\Software\Microsoft\Windows\CurrentVersion\Explorer\UserShellFolders

with the path to the folder as string data.

Changing HKLM\...\FolderDescriptions will change the creation location for all new profiles on the computer.
