rohn007 say:

It is not obvious, but they have defined the text with a single column. But the column width is narrower than the page width after margins.  The single column is defined as 6.91", after the defined left and right margins are subtracted, there still is 7.32 inches of printable space, hence the gap!

If you turn on reveal formatting, I had to expand the "Section" portion of the definition. Column definition is at the very bottom.

To fix it, go into Page Layout tab > Page Setup group > Columns drop down.  Change the number of columns to 2 then back down to 1. This will reset the single column width to use all space between the defined margins.

I copied your text, inserted a section break, and applied the fix to the second para. I Have the insertion point located inside of the first para.  I did the "fix" in the columns dialog, note the changed column width which has NOT yet been applied does not match the column width in the reveal formatting pane which is showing what is currently being used.

  • Copy from Word.

retrojunkie say:

I think i may have had the same problem, and managed to solve it.

 

Issue: I had my phone plugged in via USB, had chosen 'nokia mode' in the mobile screen's menu,

however when I was in the PC Suite program, i got to the bit where it tells you to connect your mobile, and it wouldnt 'detect' my mobile via USB.

 

Solution: I did similar to what was suggested here: http://www.expansys-usa.com/ft.aspx?k=110959 (safe link)

 

1- Turn off the PC Suite program (any windows AND any system tray icons)

2- Plug in the mobile + usb cable into a usb slot, preferably one behind the computer to better our chances

3- Set the mobile to 'data storage'

3- The PC will detect the mobile as a new hardware device, and start finding drivers for it - it probably wont do much though.

 

4- Restart the computer

 

5- Plug in the mobile again. Choose 'data storage' again. If there were any remaining drivers for it to install, it should do it by now.

 

6- Plug out the mobile. Plug in the mobile again, but DONT select any of the 3 options (nokia, printing, data storage)

The computer will probably start detecting the mobile again, but as a different device again (rather than just detecting the microSD memory inside the phone, it'll actually detect the phone itself)

7- Restart the computer

8- Plug in mobile again. Set to nokia mode. Open up PC suite. It should detect it right away.

 

Its very possible you just need to do steps 6-8 (ie just dont select any menu options when you first connect the mobile to your pc. But give it a minute to start detecting the mobile, and still give the computer a restart at the end)

Remember, once you want to connect the mobile to PC Suite, you can THEN select Nokia Mode.
