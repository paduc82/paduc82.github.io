john mcghie say:
Hi Susan:

Not enough detail to answer.  If you suddenly typed a URL, Word will apply the Hyperlink style, and if that has been mis-configured with a fot size, the font will change.

If you applied an AutoText as an AutoCorrect, and the AutoText was specified as "Formatted", then it will come in with the style that it was stored with.

If you typed off the end of a paragraph and created a new paragraph, the new paragraph will be formatted with the "Style for following paragraph" which may be different.
