bill manville say:

Firstly, your (x)*AND(y)  is rather strange.
AND(x, y) is more usual

Secondly I see no reason for it to be an array formula since none of the content is an array.

 

Thirdly I don't understand why you are multiplying strings by cell contents.

 

A more usual expression of this formula would be

=SUM(IF(AND(G14=0,H14=0),0),IF(AND(G14=1,H14=0),0),IF(AND(G14=2,H14=0),0),IF(AND(G14=0,H14=1),2*I14),IF(AND(G14=1,H14=1),3*I14),IF(AND(G14=2,H14=1),10*I14),IF(AND(G14=3,H14=0),7*I14),IF(AND(G14=3,H14=1),150*I14),IF(AND(G14=4,H14=0),150*I14),IF(AND(G14=4,H14=1),10000*I14),IF(AND(G14=5,H14=0),250000*I14),IF(AND(G14=5,H14=1),"JACKPOT"))

 

Looking at the formula more closely, it is clear that there are combinations of G14 running from 0 to 5 and H14 being 0 or 1. Only one of the IFs will give a value other than FALSE. In each case except the last, the result is a number multiplied by I14.

This can be more elegantly written as
=IF(H14=0,CHOOSE(G14+1,0,0,0,7,150,250000)*I14,IF(H14=1,IF(G14<>5,CHOOSE(G14+1,2,3,10,150,10000)*I14,"JACKPOT")))
