How do I get video to play in the preview pane the way it used to?
aileen alf say:

Thank you for the clarification. If you're not able to play the video in the preview pane, it's possible that Windows Media Player Preview Handler is disabled. It can only be enabled through the Registry. Make sure that you follow the steps carefully if you want to proceed with this step. I recommend that you back up the Registry before you modify it. Then, you can restore it if problem occurs. For more information about how to back up and restore the registry, you may refer to this Microsoft article.

Here's how you can enable the Windows Media Player Preview Handler:

  1. Start Registry Editor by typing regedit in the search bar and selecting the best match.
  2. Locate and then select the following registry keys:

    HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\PreviewHandlers

  3. Locate and right-click {031EE060-67BC-460d-8847-E4A7C5E45A27}, and then click Modify.
  4. In the Value data box, type the following text, and then click OK:

    Windows Media Player Rich Preview Handler

  5. Exit Registry Editor.

Let us know how it turns out.
