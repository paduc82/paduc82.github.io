Add message on all blank pages of a word before publishing - Microsoft Community

sridharan1 say:

Add message on all blank pages of a word before publishing

Is there a VBA solution for this?
jay freedman say:

Numbered as in your questions:

1. Create the field

{ IF { =MOD({ PAGE \*Arabic },2) }= 0 " " "[page break]¶This page is intentionally left blank." }

Click in the "This page" part of the field code, open the Paragraph dialog, and set the alignment to Center and the Space Before measurement to half of the page height. Copy this field and use it to replace the section break at the end of each chapter. You need to do this only once. After each update of the document, use Ctrl+A and then F9 to update all the fields, and they will automatically place the message only where it's needed.

2. If you have a program such as Adobe Acrobat that enables you to edit the PDF file in this way, then it may indeed be easier than trying to do it in Word. However, you will have to look at the end of each chapter after every update to the document, to add the message where it is needed and to remove existing messages where they are not needed.

3. The macro, as amended below, will do the job of adding new messages, but it won't remove messages where they're no longer needed, so you'll still have to examine every chapter ending.

Sub insertblankpagemessage()

Dim j As Integer

Dim i As Integer

Dim NotEmpty As Boolean

Dim c As Variant

Dim vertCenter As Long


ActiveDocument.Repaginate

j = ActiveDocument.BuiltInDocumentProperties(wdPropertyPages)


For i = j To 1 Step -1

    NotEmpty = True

    Selection.GoTo What:=wdGoToPage, Which:=wdGoToNext, Name:=CStr(i)

    Selection.GoTo What:=wdGoToBookmark, Name:="\page"

    If Selection.Characters.Count < 3 Then

        NotEmpty = False

        For Each c In Selection.Characters

            If Asc(c) > 13 Then

                NotEmpty = True

            End If

        Next

    End If

    If NotEmpty = False Then

        ActiveDocument.Bookmarks("\page").Range.Select

        With Selection

            .Collapse wdCollapseStart

            With .PageSetup

                vertCenter = (.PageHeight - .TopMargin - .BottomMargin) / 2

            End With

            .Text = vbCr & "This page is intentionally left blank"

            .Font.Bold = True

            .ParagraphFormat.Alignment = wdAlignParagraphCenter

            .ParagraphFormat.SpaceBefore = vertCenter

            If .Range.ListFormat.ListType <> wdListNoNumbering Then

                ' remove numbering from message

                .Range.ListFormat.ApplyNumberDefault

            End If

        End With

    End If

Next


End Sub

    greg maxey say:

    Add plain text content controls at the places you want the staple stops to be.  Tag them staplestop_1, staplestop_2, etc.

    Using design mode set the placeholder text to a single space.  Set properties to content control cannot be edited and cannot be deleted.

    Sub ScratchMacro()
    'A basic Word macro coded by Greg Maxey
    Dim lngIndex As Long
    Dim oRng As Range
      On Error GoTo lbl_Exit
      For lngIndex = 1 To 1000 'maximum number of staple stops
        If lngIndex = 1 Then
          Set oRng = ActiveDocument.Range
          oRng.End = ActiveDocument.SelectContentControlsByTag("staplestop_" & lngIndex).Item(1).Range.End
          oRng.Select
          'Print Selection
        Else
         oRng.End = ActiveDocument.SelectContentControlsByTag("staplestop_" & lngIndex).Item(1).Range.End
         oRng.Start = ActiveDocument.SelectContentControlsByTag("staplestop_" & lngIndex - 1).Item(1).Range.End
         oRng.MoveStart wdCharacter, 2
         oRng.Select
         'Print Selection
        End If
      Next
    lbl_Exit:
      Exit Sub
    End Sub
