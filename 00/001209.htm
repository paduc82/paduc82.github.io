joeu2004 say: Tommy wrote:
I need to sum progressively numbers from one column only when figures appear on another colomn.

Your question is subject to much interpretation.  A concrete example is always good (real numbers, real cell reference, real formulas, real expectations), as well as a clear and complete (but concise) explanation.

For example....  I suppose you want to sum values in B1:B1000 when the corresponding cells in A1:A1000 appear nonblank.  Some cells in A1:A1000 might be empty; other cells might contain the null string ("").  Some cells in B1:B1000 might be empty or contain the null string when the corresponding cell in A1:A1000 appears nonblank.

In that case, try:

=sumproduct(--(A1:A1000<>""),B1:B1000)

The double negation (--) converts truth values (TRUE,FALSE) to numbers (1,0), which SUMPRODUCT requires.

Note that this assumes that the range of rows (1:1000) is known ahead of time.  If, instead, you want to be able to adjust the range as pairs of numbers are entered into the two columns, you need to include that requirement in your explanation.

chrisbedford say:

Win 7 error "the default mail client is not properly installed"

I have deliberately quoted the exact wording of a previous topic because that discussion addressed the exact problem I had, but did not solve the problem.

I found the solution elsewhere, and wanted to post it, but found the thread closed. So here for what it's worth is what I found: On a Win 7 (Home Premium) PC with Office 2003 installed, the user was getting the above erro when trying to select URL or email address links in emails. Attempt to fix this by conventional means failed, because Outlook was not offered as a program on the list of Default Programs, nor was MAILTO: one of the protocols in that same editor (I'm talking about Start - Default Programs: the first two options, "Set your default programs" and "Associate a file type...")

Here's the fix that worked for me!

Two steps: 1. To fix the issue of Outlook not appearing on the list of programs that can have their defaults set, run OL2003fix which you get from here: http://www.winhelponline.com/blog/wp-content/uploads/2008/02/ol2003fix.zip

Now go to Start - Default Programs - Set your default Programs
Now that you can see Microsoft Office Outlook in the list on the left, select it and "Set this program as default" on the right.

[The protocol MAILTO is still not listed under "Associate a file type..." but that doesn't seem to be a problem any more]

2. To fix the registry entry associated with the http: and https: links, copy-paste the text between the lines of equal-signs (not including the equal signs!) into your text editor (Notepad)
==========
Windows Registry Editor Version 5.00
[HKEY_CLASSES_ROOT\mailto]
"URL Protocol"=""
==========
"Save As" anynameyoulike.reg (on the desktop is fine - once you have used it once you don't need it any more and should delete it)
Double-click the .reg file
Accept all the warnings about making changes to your registry.

BAM you're done. You're welcome.

    steve rindsberg say:

    One trick that'd work:

    File | Save As and choose PowerPoint Picture Presentation.

    That saves a copy of the presentation as a series of pictures.

    Open the presentation and on each slide:

    Click to select the picture
    Drag it to the right so that the left edge of the picture now sits on the right edge of the slide (helps to zoom out to 50% or less first).
    Then drag the selection handle on the right side of the picture to the left side of the slide.

    Voila.  Your slide is now reversed left to right

    troy hedges say:

    You can use the command prompt to list the files. Go to Start, Run, type 'cmd' without the quotes and click OK.

    A new Command Prompt window will open up and from there you can navigate to the folder you want using 'cd' to change directories and 'dir' to list the files/folders.

    Next, right click at the top of the window and choose, Edit, Mark, and then hold down the mouse button and drag over the area that you want to copy to highlight the names of the files/folders. Right-click again when you have the list highlighted, and then go to Excel and Paste.

    vijay a. verma say:
    Pixel count (in Normal view) depends upon a number of factors -
    1. Display's resolution
    2. Zoom
    3. Print scaling

    Hence, you will see different pixels on different devices in Normal view.

    Page Layout view (in view tab) is the right place to show ruler measurements. You just hold the boundary of a column, it will show you width in inches and pixels. (If you want to change this inches, then file > options > advanced > Under display section, you can change ruler settings). Pixels will always show when drag or hold a column boundary.

    This Page Layout pixels will be constant across devices and this is what Pixels and other measurements really are.
