On Thu, 14 Oct 2010 20:37:54 +0000, bawoody wrote:

I have a list of names and would like to extract part of the name into another cell

example:

Mr. and Mrs. John Doe               Mr. and Mrs. Doe

Here is a User Defined Function (UDF) that assumes the entries might
be more complex than what you have presented.

1.  If there are initial titles, they will be terminated with a
period.

2.  The last name will be terminated by a comma, or it will be the
last word in the string.

3.  If there is not title or trailing descriptor, you only want the
last name returned.

To enter this User Defined Function (UDF), <alt-F11> opens
the Visual Basic Editor. Ensure your project is highlighted
in the Project Explorer window. Then, from the top menu,
select Insert/Module and paste the code below into the
window that opens.
To use this User Defined Function (UDF), enter a formula
like

:      =TitleLastName(A1)

in some cell.

============================
Option Explicit
Function TitleLastName(s As String) As String
    Dim re As Object
    Const sPat As String = "^(([^.]*\.)+\s)?.*?(\w+)(,.+|$)"
Set re = CreateObject("vbscript.regexp")
    With re
        .Pattern = sPat
        .Global = True
        .MultiLine = False
    End With
TitleLastName = re.Replace(s, "$1$3")
End Function
