      Hi Earl. I'm Greg, an installation specialist and 9 year Windows MVP here to help you.

      Normally you'd simply copy them out of OneDrive and back into their User folder, but this has been complicated recently by OneDrive actually moving User folders into OneDrive with a new feature it set by default on many PC's without the owner's knowledge.

      So if the new Folder Protection is chosen in OneDrive Settings > AutoSave Tab, then Documents, Desktop and Pictures folders will be moved into the OneDrive folder. See Folder Protection details here: https://www.tenforums.com/tutorials/116029-turn...

      Another feature can remove files from the hard drive and store them only in the cloud if Files On Demand is chosen in OneDrive Settings: https://blogs.office.com/en-us/2017/05/11/intro...


      Strangely, recent Updates have set these on by default and even caused file losses that resulting in withdrawing the October Version 1809 Upgrade. So I would check closely those OneDrive Settings are what you want, that only what you want is in the OneDrive folder at all times where it will be synced to the cloud.

      BTW there's a method to sync all of your User folders via Onedrive to your other devices which I compiled in this tutorial that shows how I've done this for ten years without problems: http://answers.microsoft.com/en-us/windows/wiki...
