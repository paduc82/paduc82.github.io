thorstenschmermund say:

Hi DavidHodapp!

I had the same problem, but solved it in the following way:

Go to the Windows Bluetooth device manager screen.

If the Beoplay S3 is already in your Bluetooth device list, remove it from the list of bluetooth devices.

Then, press the B&O Button on your Beoplay S3 for about 2 seconds.

The speaker will give an audio signal and the LED will flash blue.

Now, click "Add new bluetooth device" in the Bluetooth device manager screen.

When Beoplay S3 is shown as an option, add it.

It should now be recognized as an audio device.

The key is to put Beoplay S3 in "bluetooth install mode" by pressing the B&O button for about 2 seconds until it gives an acknowledge signal and the LED flashes blue.

Hope this works for you - it did work for me!

anil kumar b say:

Hi,

 

If you receive Windows Update error 80072f8f while checking for updates, your computer's date and time might not be the same as the date and time for the online Windows Update service. Your computer and the service don't need to have an identical date and time, but if they vary too much, it will prevent you from checking for updates.

 

I would suggest you to try these steps and check if it helps.

 

Method 1:

To change your computer's date and time

 

  1. Open Date and Time by pressing Windows key + X, clicking Control Panel, clicking Clock, Language, and Region, and then clicking Date and Time.
  2. On the Date and Time tab, make sure that your computer's date and time are correct. If they're not, click Change date and time.  If you're prompted for an administrator password or confirmation, type the password or provide confirmation.
  3. In the Date and Time Settings dialog box, do the following:
    • To change the date, under Date, click the left or right arrows to find the current month in the calendar, and then click the current date.
    • To change the time, under Time, double-click the hour, minutes, or seconds that you want to change, and then click the arrows to increase or decrease the value.
  4. When you've finished changing the date and time settings, click OK, and then click OK again.

 

Method 2:

If you have installed any third-party anti-virus application, I would suggest you to disable it and check if you can install updates.

 

Note: Running the computer without firewall or the anti-virus software is a potential threat to the computer; make sure to enable the security software after completing the troubleshooting steps and after identifying the issue

 

Method 3:

Place the computer in clean boot to check if any third party application is causing the issue, as clean boot helps in eliminating software conflicts.

Note: After troubleshooting, see the section Reset the computer to start as usual to return the computer to a Normal startup mode.

 

For reference:

http://windows.microsoft.com/en-ca/windows/troubleshoot-problems-installing-updates#1TC=windows-8

malkeleah say: If you have another user account with administrative privileges you can log into that account and change your original user account's password from the User Accounts applet in Control Panel. If you don't have another account like this set up or don't have the password to it, you'll need to log into the built-in Administrator account. In XP Home, boot the computer into Safe Mode. Do this by repeatedly tapping the F8 key as the computer is starting up. This will get you to the right menu. Navigate using your Up arrow key; the mouse will not work here. Once in Safe Mode, you will see the normally hidden Administrator account. The default password is a blank.

In XP Pro, you do not need to go into Safe Mode. At the Welcome Screen, do Ctrl-Alt-Del twice to get the classic Windows logon box. Type in "Administrator" and whatever password you assigned when you set up Windows.

If you have XP Pro or Media Center and were using the built-in Administrator account for your regular work and it is therefore the only user account on the system, there are ways to change the password for that account to a blank.
