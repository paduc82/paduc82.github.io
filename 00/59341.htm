hairyfool say:

If the files are on a network that is remote from the computer that needs to see them then that computer needs to know how to connect to the file store. That requires a) sending the information of how to connect to the file (ip addresses, computer name, user name and password etc) and b) configuring your network to allow inbound connections to the file store with its inherent security risks.

A better solution is to use a 3rd party file store on the internet which you can both connect to. This would include Dropbox or sharing the file via your OneDrive account. These allow you to create a web hyperlink in your email that once clicked on by the recipient he can authenticate and get the file.

Difference between window 10 purchased from microsoft & free media - Microsoft Community

sanaullahkhante say:

Difference between window 10 purchased from microsoft & free media creation tool

What is the difference between Windows media creation tool and window purchased from Microsoft.
andre da costa say:

Both are the same product in some respects, the only difference is the method of acquisition and installation.

The physical retail package is stuck at build 10240, which you have to update to the latest version (14393.726) through Windows Update.

The digital download through the Microsoft Store using the Media Creation Tool, includes the latest release (14393.0), which is updated to .726 either during the installation or through Windows Update after completing the installation.

The Media Creation Tool can be used to perform direct upgrades from Windows 7 or Windows 8; or you can use it to create a bootable copy on an empty USB thumb drive or blank DVD. You can also use the Media Creation Tool to also download an ISO image, which you yourself can use to create a bootable copy on DVD or USB using your favorite program.

The retail Windows 10 box license is the same, but the install media comes in the form of a small USB thumb drive and only in one specific edition based on your purchase (Home or Pro). When you boot from it, you will have to choose which architecture of Windows 10 you want to install (32 or 64 bit).

Another difference with the Media Creation Tool, it lets you specify which editions and architecture you want to download. In fact, you can download all architectures and edition, which you can use to upgrade a mixture of versions and editions. This normally requires a larger thumb drive.

Learn more about how it works:

How to download official Windows 10 ISO files
