You can't turn off Microsoft Sync according to several articles I've read - but you may be able to disable it or prevent the alert from showing up (but I'm just guessing here - I could find no articles on how to do it).

Go to Start / Search Box and type in msconfig and enter and then double-click on the program icon that appears.  Move to the Start-up tab and open it.  Scroll through the list until you find Microsoft Sync (it may have a slightly different name) and uncheck the box that indicates it should start at startup.  That should remove it from the startup group and you should no longer see the alert.

If that doesn't work, perhaps it's a service that's starting up.  Go to Start / Control Panel / Computer Management / Services and then scroll to find Microsoft Sync.  It is probably set on automatic.  Change it to manual.  That should prevent it from starting up unless you actually activate the program (and again, it should eliminate the alert).

Do you know when this problem began?  Try a System Restore to a point in time BEFORE the problem began.  Here's the procedure: http://www.howtogeek.com/howto/windows-vista/using-windows-vista-system-restore/.  Be sure to check the box to show more than 5 days of restore points.  If the first attempt fails, then try an earlier point or two.  NOTE: You will have to re-install any software and updates you installed between now and the restore point, but you can use Windows Update for the updates.  Use the recovery disk if the system prompt doesnâ€™t work. If you received a blue screen error, or stop code, the computer has shut down abruptly to protect itself from data loss.
