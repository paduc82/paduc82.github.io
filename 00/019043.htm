jim gordon mvp say:
Well, you can, but the problem is my instructions were crappy. Please allow me a second chance.

I had some fun with this one. I have objects going around clockwise and counterclockwise and in aberrant directions all at once on my slide. It's pretty amusing.

So let's do the easy stuff first. Make or put an object on a slide, copy it, and paste it 4 more times so you have 5 identical objects. I get confused easily, so I  put a number in each shape so I could keep track of them.

Next, I selected the first shape. Then, on the Animations tab of the Ribbon, I clicked the Motion button button and in the pop-down in the Basic grouping I chose Shape. That makes a perfect circle that starts at the top, as indicated by the green triangle. The trouble is, I can't simply copy the shape or the animation path and re-use it, because the motion will always start at the top, and you can't rotate the path.

Well, at least I have one shape whirling in a circle the way I want.

On to shape #2. I position shape #2 so it's where I want it to start in its own circle. Well, I am terrible when it comes to drawing lines with the mouse, so I cheat. For shape #2 I choose the Loop motion path. Then I right-click on the path and choose Edit Points. I only need one loop, so I delete the points in the outer loop. That leaves me with a poorly formed circle. I click on each point that's left and choose Smooth Point and adjust the little handles so that it makes a nice smooth circle, and then size the cirlce and position the circle to match the path of Shape #1.

If you're good with the mouse, you might be able to draw a good circle using Draw Freeform, Draw Curve or Draw Scribble and then using edit points to tidy things up. You make a new path for each of the remaining shapes. The previous advice about setting the Timing to With Previous still applies.

The result of all this is that you can set the starting and ending point for each shape by customizing the path. Right-click on a path and choosing Reverse direction adds interest, as does having erratic paths while you refine your path to a nice circle.

Increase row numbers in a formula - Microsoft Community

ashbash27 say:

Increase row numbers in a formula

Hi

I have the following simple sum formula in a cell in SHEET1....

=SUM(Sheet2!CB4/Sheet1!B7)

What I want to do is be able to drag down the formula to populate the below cells in the same column as follows....

=SUM(Sheet2!CB11/Sheet1!B8)
=SUM(Sheet2!CB18/Sheet1!B9)
=SUM(Sheet2!CB25/Sheet1!B10)
=SUM(Sheet2!CB32/Sheet1!B11)

So the first cell reference (CB*) increases by 7, and the second (B*) increases by 1.

Any help would be appreciated

Thanks
hansv mvp say:

SUM isn't needed here, you're not adding multiple cells.

Let's say the first formula is in a cell in row 1.

You can then use

=INDEX(Sheet1!CB:CB,7*ROW()-3)/Sheet1!B8

and fill down. If the first formula is in a different row, you'll have to change the -3 to -10 for row 2, -17 for row 3 etc.
