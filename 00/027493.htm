yaqub k say:

Hi,

1.       Are you facing any issues when you try to activate Active x?

2.       Do you receive any error message?

3.       Have you made any changes to the computer prior to this issue?

I would suggest you to follow these steps to enable Netflix ActiveX and check if it works.

a. Double-click the Internet Explorer icon on the desktop to start the browser. Click the Tools menu option

b. Click the Security tab in the Internet Options window. Click Internet icon and then Custom level to start the Security Settings - Internet Zone window.

c. Scroll down to ActiveX controls and plug-ins section.

d. Check Enable for Download signed ActiveX controls and Run ActiveX controls and plug-ins. Check Prompt for Download unsigned ActiveX controls.

e. Click OK to close the Security Settings window and then click OK in the Internet Options window.

This issue usually happens after installing the upgrade on your phone. Since you have tried installing the apps to a different storage but to no avail, we suggest that you perform a soft reset to refresh your phone.
