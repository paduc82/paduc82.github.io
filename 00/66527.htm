a. user say:

Hello Joe,

Thank you for posting your query in Microsoft Community Forum.

The issue is caused basically when NetBIOS over TCP/IP is not turned on (enabled) on one or more computers in the workgroup. Another reason could be that the Computer Browser service is not started or is turned off on one or more computers in the workgroup. So, I would suggest you to try the following steps and check if the issue is reproduced.

Step 1: Enable NetBIOS over TCP/IP

  1. Click on Start, type in ncpa.cpl in the Start search and hit ENTER.
  2. Right-click the network connection (Local Area Connection) and click Properties.
  3. Click to highlight "Internet Protocol Version 4 (TCP/IPv4) and click Properties.
  4. Click the General tab, and then click Advanced.
  5. Click the WINS tab.
  6. Under NetBIOS setting, click Enable NetBIOS over TCP/IP, and then click OK two times.
  7. Click Close to close the Local Area Connection Properties dialog box.
  8. Close the Network Connections window.

Step 2: Start the Computer Browser service

  1. Click Start and type services.msc in the search box.
  2. On the Services window, double-click on the Computer Browser service.
  3. Check if the Computer Browser service is Started set to Automatic. If not, please set it to this status.

Hope this information is helpful. If the issue still persists, please post back for further assistance, we'll be glad to assist you.
