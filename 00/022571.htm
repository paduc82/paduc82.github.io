Are you using the same version of Office?

If no, then I wouldn't transfer the toolbar files.

If yes, and you've created the toolbar manually and you're using xl2003 or
older, then...

Go back to the old computer
start excel
make a minor change to a toolbar
Close excel
look for a file named *.xlb with the correct last date updated time/date
property.
(use windows search)

Copy that file (to a thumbdrive???)

Then go to the new pc
Do the same thing

Then look for that *.xlb file. 

Copy that *.xlb file to the folder that holds that just modifed version of the
*.xlb file.

======
If you're using different versions of office, then don't do this.  The icons
don't always do the same thing between versions.

In fact, I'd say go back to the old pc and do a print screen (as a reminder of
what icons you've used in the past).

Then go to the new pc and do the Tools|Customize stuff.

You'll find that you'll have icons that you really don't need to add and even
better, you'll find that you have more choices to add to your toolbar -- invest
a little time to skim through all the new (and old) tool icons.

========
As for MSWord, I think the manually modified toolbars are kept in your
normal.dot file.  But again, don't copy the old one over if you're not using the
exact version of office.

========

ps.  Don't forget any personal.xl* workbooks that you may have created on the
old pc!

How to calculate weeknum and weekday for Saturday to Friday week - Microsoft Community

diane h2 say:

How to calculate weeknum and weekday for Saturday to Friday week span?

Our week starts on Saturday and ends on Friday. I need to calculate the week day number (1 to 7) and the week number.
rik_uk say:

OK this formula calculates the week number (even [dare I say it] better than the inbuilt excel function)... I'll explain why first, take a look at how the function handles the weeks between 2010 and 2011...

Take =WEEKNUM([serial num],[{1,2}]) which is the Excel Function for US week numbering, should week 53 be week 1? or is there only 1 day in week 1 of the US calendar this year? (some years do have full 53 weeks in them eg the end of this year...)

23/12/2010 Thu 52
24/12/2010 Fri 52
25/12/2010 Sat 52
26/12/2010 Sun 53
27/12/2010 Mon 53
28/12/2010 Tue 53
29/12/2010 Wed 53
30/12/2010 Thu 53
31/12/2010 Fri 53
01/01/2011 Sat 1
02/01/2011 Sun 2
03/01/2011 Mon 2

There is a formula found here for calculating the ISO week calendar (as accepted by MS), but this is based on the weekday function (ie Monday to Sunday or Sunday to Saturday), so it is modified to give Saturday to Friday week calculation. I have used B5 as my date field, please amend as required...

=INT((B5-IF(WEEKDAY(B5,1)+1=8,1,WEEKDAY(B5,1)+1)-DATE(YEAR(B5+4-IF(WEEKDAY(B5,1)+1=8,1,WEEKDAY(B5,1)+1)),1,4))/7)+2

which results in

24/12/2010 Fri 51
25/12/2010 Sat 52
26/12/2010 Sun 52
27/12/2010 Mon 52
28/12/2010 Tue 52
29/12/2010 Wed 52
30/12/2010 Thu 52
31/12/2010 Fri 52
01/01/2011 Sat 1
02/01/2011 Sun 1
03/01/2011 Mon 1
04/01/2011 Tue 1
05/01/2011 Wed 1
06/01/2011 Thu 1

This is tested from 1972 to 2071 and are no problems for these years.

To pull the week day starting from Saturday then this part of the equation is used...

=IF(WEEKDAY(B5,1)+1=8,1,WEEKDAY(B5,1)+1)

So with this in mind you could use this in part of the week number calculation - say the date is in B5, the week day is in C5, then in D5 put the following formula to calculate the week number :

=INT((B5-C5-DATE(YEAR(B5+4-C5),1,4))/7)+2

Hope this helps.
