todtown say: Do you want the line to break where there is no data, or do you want the two ends of the line to join?

I think your solution should work if you want them to join instead of the line going to zero. 

=If(B5=0,NA(),B5)
Where B5 contains the number to plot.

The result when B5 is zero will show #N/A, but you can use conditional formatting to hide it. 

To break the line where there is no data you'll need to delete the value from the cell.
