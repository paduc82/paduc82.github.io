MrRichofSTLMo,
Thank you for visiting the Microsoft Answers community forum.

The ISATAP #9 is related to Internet Protocol 6, which can cause issues with devices that don't use it and can safely be uninstalled as most networks today still use IPv4.

To remove IPv6 and all of it's associated devices:
Go into the properties for each network adapter, wired and wireless, and either uncheck the box next to Internet Protocol 6 (TCP/IPv6), that will disable it, or select it and click uninstall, which will remove it off the computer.  Then go into device manager and remove any 4to6 adapters, WUN miniport adapters, or ISATAP adapters.  Then reboot your computer.

If that does not work, try this:
To reset the TCP/IP stack go to this article and either click on "Fix it for me" or follow the instructions to fix it yourself:

http://support.microsoft.com/kb/299357

Hi TheDarkTrainer,

I will need some additional information in order to help you better.

1.       What is the make and model of the sound card installed on the machine?

2.       Is this the first time you are experiencing the issue?

 

For single channel sound cards, you will only be able to see the green bar for the Master Volume Control and not for the other applications such as Windows Media Player.

The controls on other applications are given to control the volume of the particular application with reference to the Master Volume Control.

 

For instance, if you scroll the Master Volume Control bar to 100% and then play a music file in Windows Media Player. You can try to open the Volume mixer and increase/decrease the volume control for Windows Media Player, you will notice that the Master control will set the green bar accordingly and will not let the green bar cross the bar of the Windows Media Player.

If you mute the Volume Control for Windows Media Player, it will automatically make the green bar disappear from the Master Volume Control.

However, if you have multiple applications running on the machine at the same time and using the sound card, you can try changing the levels on different controls to see the effect.

 

With multiple channel sound cards such as sound blasters, it may be possible to see green bar on individual applications also.
