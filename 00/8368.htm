Domande principali relative agli aggiornamenti software - parti da qui - Microsoft Community

candy_46 say:

Domande principali relative agli aggiornamenti software - parti da qui

Come aggiorno il software del dispositivo Nokia?
Puoi aggiornare il software di gran parte dei dispositivi Nokia scaricandolo direttamente sul telefono o utilizzando Nokia Suite.

 

Informazioni e metodi su come aggiornare il software del dispositivo sono disponibili all'indirizzo
http://www.nokia.com/global/support/software-update/update-your-phone/


Come posso sapere quando è disponibile un nuovo software per il dispositivo Nokia?
Puoi ricevere una notifica di nuove versioni del software per ogni dispositivo Nokia in diversi modi:

1) Tramite Nokia Suite
Ogni volta che connetterai il dispositivo all'ultima versione di Nokia Suite con un cavo USB compatibile, Nokia Suite verificherà se è disponibile un aggiornamento per il dispositivo. Questa notifica funziona solo quando utilizzi una connessione via cavo USB.

2) Verifica online
Puoi facilmente controllare l'ultima versione disponibile sul
sito Web del Programma di aggiornamento software Nokia.

Al momento Nokia non pubblica le date di rilascio previste per le nuove versioni del software.


Quali aggiornamenti sono stati apportati in ciascuna versione nuova del software?
I principali aggiornamenti sono pubblicati sul
sito Web del Programma di aggiornamento software Nokia.


Ecco alcune delle domande più comuni tratte da Domande frequenti sull'aggiornamento del software del dispositivo Nokia:

Perché il mio dispositivo ha cambiato lingua dopo l'aggiornamento?
Dopo un aggiornamento è possibile che il dispositivo Nokia si riavvii nella sua lingua nativa. Per ulteriori informazioni, visita il sito www.nokia.com/support.

Se il tuo dispositivo Nokia è stato prodotto inizialmente per essere utilizzato in un altro Paese, l'uso del Programma di aggiornamento software Nokia potrà ripristinare il gruppo originale di lingue del dispositivo. 

Altri utenti riescono ad aggiornare il software, mentre io no, anche se abbiamo lo stesso modello di dispositivo Nokia. Qual è il problema?
La disponibilità degli aggiornamenti dipende da molti aspetti, tra cui la versione esistente del firmware del dispositivo Nokia, il paese e l'operatore. A causa di una di queste variabili è possibile che il tuo dispositivo Nokia non possa essere aggiornato.

Il tuo fornitore di servizi di telefonia mobile o operatore potrebbe non avere approvato l'ultimo firmware disponibile per il dispositivo Nokia.

Nokia produce molte varianti differenti di ogni prodotto (per Paesi e lingue diversi) e non su tutte le varianti è caricato l'ultimo firmware. È una nostra priorità aggiornarle tutte il più rapidamente possibile e ci scusiamo per eventuali ritardi. Il nuovo software potrebbe essere disponibile successivamente, torna a visitarci presto.


Perché Nokia Suite non rileva il mio dispositivo?
A volte i driver del dispositivo Nokia e i relativi cavi di collegamento si possono danneggiare.

Per risolvere il problema, disinstalla e reinstalla l'ultima versione di Nokia PC Suite e del Programma di aggiornamento software Nokia:
- Nel Pannello di controllo del PC scegli Installazione applicazioni
- Disinstalla il Programma di aggiornamento software Nokia (se sono disponibili più occorrenze, disinstallale tutte)
- Vai a www.nokia.com/support per scaricare e installare l'ultima versione di Nokia Suite


Non riesco a utilizzare Nokia Suite su alcun PC e il dispositivo non supporta gli aggiornamenti direttamente via etere. Esiste un metodo alternativo per aggiornare il software del dispositivo?

 

Se non riesci a utilizzare Programma di aggiornamento software Nokia, puoi visitare il centro assistenza Nokia Care autorizzato più vicino e chiedere l'aggiornamento del software. Il servizio potrebbe essere a pagamento. Le località in cui sono presenti centri di assistenza Nokia Care sono riportate nella pagina del supporto del tuo Paese. Per dettagli, visita http://www.nokia.com/global/support/locations/

Saving Bing's Daily Backgrounds Summary
There is currently no provision for keeping your own collection of Bing daily background images but with a few clicks and keystrokes, you can save the image files locally or to your SkyDrive.
Details
The method of retrieving a Bing daily background image is essentially the same no matter which web browser you prefer but there are subtle differences in the mouse clicks and keystrokes used to accomplish the task. I will break this down for each of following three popular web browsers.
  • Microsoft Internet Explorer version 10 (aka IE10 or simply IE)
  • Google Chrome version 31.x.xxxx.xx (pending)
  • Mozilla FireFox version (pending)
  • Temporary Internet Files (proposed)
For each of these web browses, I will show you how to view the HyperText Markup Language (HTML) code that defines a web page's content and formatting. From the source code, you can locate the reference to the daily background image and use it to acquire a copy for yourself.

For the purpose of an example in this tutorial, I will use the Bing daily background image from 14-Nov-2013.
 

Figure 1: Little owl, Vieux, Calvados, Lower Normandy, France (© Getty Images)

  Internet Explorer 10
  1. Open a new instance of Internet Explorer and go to http://www.bing.com if the Bing site is not already your browser's home page.
  2. Use the Next and/or Previous buttons ((1) and (2) in Figure 1 above) to locate the image you want to save. Hover over the Information button  (noted in Figure 1 above by (3)) to get an idea of what the image entails. In this example, the information or title of the image is "Little owl, Vieux, Calvados, Lower Normandy, France (© Getty Images)". We will use this to locate the image reference within the web page's source code in the next step.
  3. The command you are looking for next is View, Source from the pull-down menus. This can also be initiated with Alt+V, C or through the Ctrl+U shortcut. This will open a new pane with the Bing search landing page's HTML code.
     

    Figure 2: Internet Explorer Source pane showing Bing.com HTML Code

    Yes, it looks like gobbledy-goop but hidden in there is the reference we need to get a copy of the image.
     
  4. The next step is to find the defining reference to the image used for the background. Choose Edit, Find from the pull-down menus (Alt+E, F or Ctrl+F) to open the Find dialog.
                  
    Figure 3: Source pane Find dialog 

  5. Type or paste background-image:url into the text box and click Next.
     
        
    Figure 4: Find results from Source pane search 

  6. The actual location on the Bing.com web site where the image is located immediately follows background-image:url within the brackets.
     
        

    This location is known as the URL or Uniform Resource Locator. It starts with a forward slash so all we have to do is prepend what we have found with Bing.com's base URL. Start in Notepad with Bing's base URL and then copy and paste the image's site URL to the end of it.
     
        http://www.Bing.com//az/hprichv/NormandyOwlHole_GettyRR_141825829_EN-CA2062962585.jpg
     
  7. With the full URL together, copy it and paste the entire location into IE10's address bar. Tap Enter and the image will appear in the web browser by itself. It can now be right-clicked to use the Save picture as... command.
     
            
You're done. You pulled the image using the coded resource location and saved a copy for your personal use.

Web browsers are updated constantly and completely new versions are either being released or being tested in limited markets. The methods I've described are pretty standard and should survive updates and upgrades. If you find anything that is 'broken' feel free to comment below and I will make all efforts to rectify the instructions.
