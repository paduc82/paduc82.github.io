macropod say:

You should be able to simply copy the contents of all the documents (including the actual bibliography from only one of the documents) into one new document and have the bibliography entries copy over as well. In the destination document, simply refreshing the single copied bibliography field should result in all the works being listed. Sometimes, references in footnotes/endnotes may copy over correctly; you should be able to use the source manager to copy those from your master list into the document's current list.

If you end up with duplicates, that's most likely because someone's edited one of the references. Replace any citations pointing to the incorrect reference with the correct one, then delete the unwanted entry in the source manager and refresh the bibliography field.
