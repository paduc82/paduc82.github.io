I think we are all confusing two things.

The case says "64 GB". This refers to the storage capacity of the Surface. You can verify this number by looking at the "properties" of drive C: by right clicking it. Look at the cake diagram and the largest number below this. If the number is greater than 32'000'000'000 but lower than 64'000'000'000, then you have the correct 64GigaBYTES of storage, what you paid for. If it is smaller, not good.

The 32 BIT operating system tells you the number of BITs Windows uses to form one single piece of data with which it works. This means, that one piece of data is 32 Bits long. Think of it is a number with 32 digits of ones and zeroes, just like the number 2475 consists of four digits out of 10 possible ones (1,....9,0).

This figure is only remotely related to the 64 GB of above.

It is however related to the Processor built into the machine. This can be a 32 Bit processor or a 64 Bit processor. I do do not remember, whether Surface RT were available at all with 64 Bit processors. But I do know, that Windows RT is a 32 Bit system and only uses these 32 Bits to work with and if the Surface was at all equipped with 64 Bit processors, these would have been set so a 32 Bit mode.

So, 64 BITs is definitively not what you paid for with Surface RT.
