Publisher 365 files changes format when working between Windows 7 and Windows 10 computers

Any publisher files created on Windows 7 changed after opening the file on a Windows 10. The file displayed as it should on the Windows 10 machine but when you opened the same file on Windows 7 machine again, that's when the format issues occurred. Words flowed onto different pages and the indentions were different. My solution to this problem was to copy and install the fonts from Windows 10 machine to the Windows 7 machine. On Windows 7 PC:

  • Backup fonts first!
  • Create a "Backup" Folder anywhere(Under documents, desktop, etc)
  • Navigate to C:\Windows\Fonts
  • Right click Font folder and copy
  • Paste in your "backup folder"

On Windows 10 PC

  • Copy C:\Windows\Fonts folder on usb drive

Back on Windows 7 PC:

  • Navigate to C:\Windows\Fonts and open the Fonts folder (It will open up special font window) *keep open
  • open the Windows 10 fonts folder on usb
  • Copy all fonts and drag them to windows 7 fonts window
  • They should automatically begin installing
  • Close any error dialog boxes for fonts that do not install. This is normal. Not all Windows 10 fonts will not work with Windows 7
