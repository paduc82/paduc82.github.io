ossiemac say:

Hello Sean,

Try the code below. I have assumed that you have the code in module for Worksheet "Billing Worksheet" and that is the sheet where rows are hidden and unhidden and that you are changing values in Worksheet "FCO Recap". If this is not correct then please get back to me.

A couple of points to note. If you are changing the values on FCO Recap then that is the active sheet. The code is making changes to Billing Worksheet. Me is the generic object for the worksheet to which the code module belongs and I have included it in the code otherwise Excel can be referencing the incorrect worksheet. I have also included code to provide a message if the code fails otherwise the user will never know that an error has occurred with the use of On Error.

I have removed your line of code "c.Activate". Apart from the code being incorrect because the worksheet is not the ActiveSheet and therefore the cell cannot be activated, there is no reason to select every cell in the loop. If you want the code to select the appropriate cell in Billing worksheet that matches the one that was changed on FCO Recap then a different approach is required and capture the worksheet change on FCO Recap instead of the Calculate event on Billing worksheet. If you want this then please get back to me.

Private Sub Worksheet_Calculate()
    Dim LastRow As Long, c As Range
      
    Me.Unprotect Password:="password"
   
    On Error GoTo ReEnableEvents
    Application.ScreenUpdating = False
    Application.EnableEvents = False
   
    With Me
        LastRow = .Cells(.Cells.Rows.Count, "c").End(xlUp).Row
    End With
   
    For Each c In Me.Range("c28:c" & LastRow)      'This line edited since initial posting
       If c.Value = 0 Then
            c.EntireRow.Hidden = True
        Else        'Does not require ElseIf
            c.EntireRow.Hidden = False
        End If
    Next
   
ReEnableEvents:
    If Err.Number <> 0 Then
        MsgBox "Error occurred in Sub Worksheet_Calculate for Sheet 'Billing Worksheet'"
    End If
   
    Application.EnableEvents = True
    Application.ScreenUpdating = True
    Me.Protect Password:="password"

 End Sub
