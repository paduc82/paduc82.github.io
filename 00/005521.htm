Sync MS Project 2013 with SharePoint 2013 with custom columns - Microsoft Community

gary2008 say:

Sync MS Project 2013 with SharePoint 2013 with custom columns

Without custom columns, the sync is easy.  but, say, you want your Gantt chart to be grouped by project phases, the steps below sync MS Project with SharePoint 2013 with custom columns:

1) Create a Task list in SP2013, add a custom column, say TaskGroup as a choice type (probably not necessary, but I set it as "required").
2) Optional -- Add a task item to the list.
3) In Project, create a new project by "Import from the SharePoint List", using the list above.
4) In the project file, add a custom field as TaskGroup, add one of the required value based on the choide list values in (1).
5) Optional -- continue editing the Project file.
6) In the project file, click "File". Under "Info" there is a Save button. Click "Map fields" to map the TaskGroup field from Project

side with TaskGroup collumn on SP side.
7) "Share" to sync Prject to SP side. -- Notice, when the sync appears completed on Project side, you may or may not see the new results immediately on SharePoint side.  It sometimes may take a few moments before you see the updated the task because Sharepoint uses timer jobs, I guess.
8) On SP side, "List setting" -- edit Gantt chart view, select "TaskGroup" as one of the view fields.  Under Group by, select

"TaskGroup".

Now your Gantt chart will have a group-by view nicely.

These steps are essentially 2-way synchronization between Project and SharePoint.

Any comments to make the sync more efficient are welcome.

This is lifted directly out of the Nokia Search and Solve FAQ database.....

 

Close Ovi Music Player and delete the MDataStore.db3 file:

XP C:\Documents and Settings\<user>\Local Settings\Application Data\Nokia\Nokia Data Store\DataBase\MDataStore.db3


Both of these locations are in hidden folders.  So, to view them users will need to: 

 

XP: Open Windows Explorer > Tools > Folder Options > View > Show Hidden Files and Folders


Vista: Control Panel > Folder Options > View > Show Hidden Files and Folders
