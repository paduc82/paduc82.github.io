hansv mvp say:

If you include line breaks - vbLf or Chr(10) - in the value of a cell, Wrap Text must be on, otherwise the line breaks will not be displayed.

Excel will not widen a column to autofit wrapped text, but it will narrow it if possible, so perhaps you can use something like this:

 

Sub AutoFitWrappedText()
    With Range("A1").EntireColumn
        .ColumnWidth = 255 ' maximum width
        .AutoFit
    End With
End Sub
