tony jollans say:
You are correct that the default custom dictionary in Word 2002 will not accept Cyrillic - it is ANSI-encoded - and the 'dictionary full' message is what you get when you try to add unicode to it.
 
Open Notepad and save the empty file; in the Save dialog ..
 
  • Change the Save As Type to "All Files"
  • Change the Encoding to "Unicode" (dropdown at bottom of dialog in Vista)
  • Navigate to wherever you want to put it - the default is %APPDATA%\Microsoft\Proof
  • Save the File as SomeName.Dic
Close Notepad and, in Word ....
 
  • In Tools > Options > Spelling & Grammar, click on Custom Dictionaries.
  • If you created the file in the default location (above) it will appear in the list
  • If you created it somewhere else, click on Add, and navigate to it and Add it
  • Check the box for your new dictionary and click on Change Default
If all goes well, your default dictionary will now accept unicode and you should be able to add your Cyrillic words.
