diskpart

list disk

Type create partition primary and press Enter.

Once diskpart successfully created the specified partition, type format fs=ntfs (or format fs=exfat) and press Enter. This command will format the drive with a file system.

Assign a drive letter to the newly created partition by typing assign, then press Enter.

After performing all the steps provided, try to install Windows 10 again.

a. user say:

 Hi,
The signal strength is a combination of the actual 802.11x signal + Phone + Microwaves + other sources that might generate 2.4GHz “Noise” in other words it is visual representation of all the 2.4GHz in the atmosphere around the your Wireless hardware.
That is, you might see High Signal Strength that Shows:  Excellent (5 bars), but it is actually  30% signal + 70% noise.Such a signal would be the reason for low bandwidth, and or frequent disconnection of the Wireless Network.
In contrast, a  medium level  Signal (3 Bars) that does not contains any noise would provide a match better connection.The real meaningful measure of the Wireless' signal capacity is Signal to Noise Ratio (SNR), a measure that takes into consideration the relevant Signal adjusted to the Noise Level.  Unfortunately, none of the Entry Level Wireless drivers provides such a measure.
 
There are a couple of freeware utilities that can be installed on a Wireless Connected computer and would provide SNR information like  Netstumbler.These softwares would show valid SNR only with Wireless cards that provide the raw information to the program.
 
If your card is Not on the list you can still use SNR soft wares to scan your environment, and find out if there is additional WIFI signals around.
 
As an example, if you find a neighbor using his Wireless Network set to channel 6, try to change your system channel to 11, and thus avoid noise interference between the Networks.

New computers that come preinstalled with Windows often have what is called a recovery partition. This is used to reinstall the operating system in the event of a system crash.
