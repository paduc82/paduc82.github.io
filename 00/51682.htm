cbarnhorst say:

The recovery partition is for your old OS.  

Just to be clear, "resetting" means using Reset this PC in Settings.  It refreshes the Windows 10 system files.  If you are trying to repair Windows 10 files, then that is one option.  If you are trying to revert to a previous version of Windows it is not.  

If you are trying to repair Windows 10 then I suggest you use the Media Creation Tool at

https://www.microsoft.com/en-gb/software-download/windows10

and create a dvd or usb.  With Windows 10 running insert the media and when you see the splashscreen click Install Now.  When you come to the "What to keep" screen, click the top button (keep everything).  This will preserve all your files AND apps.  

This is called an upgrade-in-place and will install new Windows 10 system files without touching anything else.  

If you are "cleaning house" then you can do an upgrade-in-place without keeping anything or you can do a classic clean install by starting the computer with the media.

cbarnhorst say:

Do a repair install.  That keeps your apps, files, and settings.  Go to

https://www.microsoft.com/en-gb/software-download/windows10

and create a dvd or usb.  With the computer running (at the desktop), insert the media.  When the splash screen appears, click Install Now.  Step through the screens to "What do you want to keep" and click the top choice to keep everything.  Continue.  That will completely reinstall Windows 10 without touching your apps or files.

Be sure NOT to start the computer with the media as that would wipe out everything.

Here's how Game Pass works. It gives your account a temporary license to run the game for as long as you subscribe to the service.
