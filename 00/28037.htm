Theory: there is no obvious successor to Binder, but both Word and Excel are capable of acting as containers for other objects - for example, you can create a Word document with independent, embedded Word and Excel objects. don cuthbert say:

UUP Dump gives the Architecture options x64, x86 and arm64.  Choose x64 (most will require this one) it is then referred to as amd64, which is the standard naming convention for 64-bit Windows (running on both Intel or AMD processors).  x86 is for 32-bit Windows.

Only a very few newer devices will use the arm64 choice, as that is an uncommon processor, used mostly in a few portable devices that operate primarily on battery power.

I believe they use "x64" on the first selection page to make it easier to make the correct choice.  Otherwise "amd64" and "arm64" look almost the same, at a glance, and it would be too easy to make the wrong selection.

After downloading the resulting ZIP file and unzipping it somewhere, the CMD script that you run downloads directly from Microsoft's Update servers (like Windows Update does when you run it from Settings).

UUP dump seems to be a safe and reliable method for quickly producing a "Fast Ring" Windows 10 ISO file.  There is nothing that it does that could not be done using only tools and utilities that are already included with Windows 10, such as PowerShell and DISM.  However, using UUP-processed files (to accurately produce a Windows 10 ISO that closely replicates an official ISO from Microsoft) requires a lot of intricate steps, which is why these users have created these scripts to do this automatically each time.
