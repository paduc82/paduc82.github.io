Ok, I found it.

https://support.office.com/en-za/article/Check-spelling-and-grammar-in-a-different-language-b872cb8a-7568-4512-9c8e-1da7ed5176ec

Updated 25th August 2016. Mac OSX 10.11 ElCapitan. Microsoft Office 2016 (Word)

To set the documents to English (UK) for example.

Open a word document that has text in it.

Highlight all the text . cmd+a

Go to Tools -> Language

A drop down menu will appear. Make sure you Untick the [don't correct spelling and grammar] option. It may start witha - sign in the box. tick and untick the box to clear it

Find your required language - English (UK) and click to highlight it.

THEN

Click [Default] on the same drop down box. It wil lthen ask you if you want ot make this the default for all documents based on the Normal template. Check Yes. and the jobs done.

If you don't click Default - it won't work.
