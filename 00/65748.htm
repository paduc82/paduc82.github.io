While you may have already tried this, this is what I did to resolve this exact same issue on my windows 8.1 computer.

1. open the window that allows changing your mouse pointer.

2. Select the pointer scheme (or custom pointers) of your choice, and click apply (For this problem, pointers will not change)

3. click OK to close the window.

4. Close all windows, and apps, then restart your computer normally

5. Login to your account.

If the above procedure does not fix your mouse pointers, then the cause of the problem is very likely a recent hardware, or software change.

If you have recently installed new hardware, try removing that piece of hardware, and restart the computer.

If you have recently installed new software, check to see if this software starts, when your computer starts. If it does, try disabling it from startup, and restart your computer.

Windows 8/8.1

1. Right click the taskbar at the bottom of your desktop screen, and choose Task Manager

2. If you see tabs in task manager, skip to step 3. If you don't, click "More details" at the bottom of task manager

3. Click on the startup tab at the top of task manager

4. Search the list for your newly installed software.

5. if you find your new software, click on it to select it, and click the "Disable" button on the bottom right of task manager.

6. close all running programs, restart your computer, and try again.

XP, Vista, 7 (You must have an admin account)

1. Press the key combination Windows + R to open the run dialog

2. type "msconfig.exe" (without quotes) and click OK. If you're prompted for permission (UAC) click yes, or enter your password.

3. Click the startup tab in the new window that appears

4. Search the list for your newly installed software.

5. If you find your newly installed software, uncheck the box on the left of it, then click OK.

6. Close all running programs, restart your computer and try again.

I hope this helps you.

sarahkong say:
Sounds like you might have a security software installed on your PC that EFS is picking up on.
Because you are right, you don't need a smartcard for EFS to encrypt files.
Try doing a clean boot to make sure that EFS is not picking up on a security software.
And if you have a 3rd party antivirus running i recommend removing it and using Windows built in Defender.

https://support.microsoft.com/en-us/help/331796...

Also you can create a GPO to disable smart card usage for EFS. If you are using home edition of Windows this option will not work.
https://www.free-online-training-courses.com/co...

FYI, I am not a big fan of EFS for the simple reason is if you do not back up your self signed certificate and export then you can not open that file if your cert gets corrupt or if you move that file to a new PC or do a restore.
Have you thought about using bitlocker instead?
https://www.windowscentral.com/how-use-bitlocke...
