(2) There will no longer be a danger that you'll forget to Save As and would instead overwrite the default file with the new letter.

For more about templates, read http://www.addbalance.com/word/templates.htm.

As for where the file goes: Do you have a OneDrive folder on your local disk that gets synchronized to the cloud, or are you trying to save directly to the OneDrive server? If there is such a folder, either save your letter there or save it in a folder that you've selected to be synchronized. If you don't have a local OneDrive folder, see this Microsoft support article.
