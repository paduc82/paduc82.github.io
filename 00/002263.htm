mechengsk say:

Steph,

I know its a bit late, but I've come across your question as I was facing the same issue and managed to do it. 

- Go to Project tab, Select Change Working Time
- Click create new calendar, in dialog box, give it a meaningful name, select Create new base calendar.

- In the Work Weeks, select the [Default] and click the Details button.

- In the Details dialog box, for every day, select Set day(s) to these specific working times. enter 0 in From cell and 0 in To cell. You should see 12:00AM and 12:00AM in each cell respectively. (Unfortunately you are not able to set all days to nonworking time, this step by-passed this restriction)

- In Work Weeks tab create new entry with finish date far in the future.

- In the details of the new work week set all the days to non-working

- Under Exceptions, set your given day you want to be work day

- In details of that exception make sure it is selected as Working Times and set you From/To cell values.

- Set you recurrence pattern options

- You are set.

I hope this helps

how do I copy e mail folders to a flash drive

a. user say:

Hi AllenAckley,

 

Please know that the feature to copy, archive or download messages via the Hotmail web interface is not yet available. You can directly forward your inquiry, feedback or suggestion to the product team to make them aware of it by referring to this link.

 

As a workaround, you can configure your Hotmail account to an email client such as Windows Live Mail so that you can download your messages locally to your computer and archive it from there. To download the latest version of Live Mail, click here. After downloading the application, you can now add your email account by following the steps provided in this article.

 

You can only archive messages and not the folders. Once you have successfully archive your messages, you can just create folders and move messages in it.

 

To archive your messages, please follow these steps:

 

Note: Before you start, please create a new folder in any location of your hard drive and please be sure that it is empty. This is where you will save the emails that you will export.

 

1. Start Windows Live Mail.

2. Sign in your email account.
3. Click the Blue tab with a drop down button on the left side of Home tab.

4. Choose Export email and then select Email messages.
5. Select the your preferred format of the emails:

 

     - Microsoft Exchange

     - Microsoft Windows Live Mail

 

6. Click Next and then Browse.

7. Locate the new folder that you have created earlier and then click Next.
8. Click the button beside All folders and click Next.
9. Click Finish.
