Hi,

try this (ADODB method)

in order to convert data from sheet WPM RCP Upload

into an .csv file

 

 

Sub SaveFile_CSV()
Const myDelim As String = ","
Dim WS As Worksheet
Set WS = Sheets("WPM RCP Upload")   '<< SOURCE SHEET
Dim r As Long, c As Long, i As Long, j As Long
r = WS.Cells.Find(What:="*", SearchOrder:=xlByRows, SearchDirection:=xlPrevious).Row
c = WS.Cells.Find(What:="*", SearchOrder:=xlByColumns, SearchDirection:=xlPrevious).Column
Dim myPath As String
myPath = "C:\Desktop\TEST\"
Dim myFile As String
myFile = "TEST Import File " & Format(Date, "ddmmyy") & ".CSV"
Dim obj As Object
Set obj = CreateObject("ADODB.Stream")
obj.Type = 2
obj.Charset = "UTF-8"
obj.Open
Dim v() As Variant
ReDim v(1 To c)
For i = 1 To r
For j = 1 To c
v(j) = WS.Cells(i, j)
Next
obj.WriteText Join(v, myDelim), 1
Next
obj.SaveToFile myPath & myFile, 2
MsgBox "file is ready"
End Sub

brian tillman [outlook mvp 2007-2019] say:
Open the Address Book interface and click Tools>Options.  Check Names will search in the order listed in the middle pane.  It stops checking when it finds a match.  If you have the GAL listed first and a name starting with the string you've typed is in the GAL, Check Names will find it and stop.  If it doesn't find a name starting with that string and you have your Contacts folder listed after the GAL in the Address Book options, then Check Names will find it in your Contacts.  If you want Outlook to search your Contacts first, change the search order.
