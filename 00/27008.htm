john sr wilson say:
It will but I would never set both the height and width. Video when added has LockAspectRatio set to True

If the video is the same aspect ratio as the slide it will fit with either the height or width if it isn't the last setting will apply and the video will be either too small or too large for the slide.

If you want to distort to fill the slide (why?) you need to set either the height or width and afterwards set LockAspectRatio to False and reset the height/width
