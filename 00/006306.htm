Ctrl+Alt+Right arrow -> Sentence case

but I can't find these in the 2010 command list for customize keyboards. Will I have to do macros for these or am I just not looking in the right place?
jay freedman say:
There haven't been separate commands for the various cases at least from Word 2003 forward. There's a ChangeCase command (Shift+F3) that cycles through upper case, lower case, and capitalize each word (but, oddly, not sentence case). All four, plus a toggle case command, are on the Change Case button in the Home ribbon.

If you want a shortcut for each one, then yes, you'll need macros. Here they are:

Sub UpperCase()
    Selection.Range.Case = wdUpperCase
End Sub

Sub LowerCase()
    Selection.Range.Case = wdLowerCase
End Sub

Sub SentenceCase()
    Selection.Range.Case = wdTitleSentence
End Sub

Sub WordCase()
    Selection.Range.Case = wdTitleWord
End Sub

Outlook 2013 creates a new storage folder for EACH account by default. Whether they can be combined or not depends on the type of email accounts they are.  What are they, POP, IMAP, Exchange ActiveSync (Outlook.Com/Hotmail/Live mail), Exchange server, something else?

hal hostetler - mvp-outlook say:
IMAP accounts REQUIRE separate folder sets which cannot be combined.  With earlier versions on Outlook, each IMAP account required two storage files, an .OST file for email and a .PST file for contacts, calendar, etc. With Outlook 2013, this has been reduced to one .OST file per account, but these cannot be combined.  If the two accounts were combined before, then you were likely using POP instead of IMAP, as POP accounts can be combined into a single .PST fie.
