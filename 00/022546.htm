graham mayor say:
The following takes account of the revised information. You can remove the lines marked optional if you prefer, but they simply add the date to the filename in the format yyyymmdd


Sub FileSave() 'Use only in a TEMPLATE
'from which you are creating new unnamed documents
Dim orng As Range
Dim strAsk As String
Dim strDate As String        'Optional
Dim strFname As String
Const strPath = "D:\My Documents\Test\Versions\"
With ActiveDocument.Tables(1)
    Set orng = .Cell(1, 2).Range
    orng.End = orng.End - 1
End With
strDate = Format(Date, "yyyymmdd")        'Optional
strFname = Trim(orng.Text)
strFname = strFname & "-" & strDate        'Optional
If Len(ActiveDocument.Path) = 0 Then
    If Dir(strPath & strFname & ".doc", vbNormal) = "" Then
Else
    ActiveDocument.Save
End If
End Sub


If you persist in using existing documents as the basis for new documents, then you need to use SAVEAS, so use the following macro

Sub FileSaveAs()
Dim orng As Range
Dim strAsk As String
Dim strDate As String        'Optional
Dim strFname As String
Const strPath = "D:\My Documents\Test\Versions\"
With ActiveDocument.Tables(1)
    Set orng = .Cell(1, 2).Range
    orng.End = orng.End - 1
End With
strDate = Format(Date, "yyyymmdd")        'Optional
strFname = Trim(orng.Text)
strFname = strFname & "-" & strDate        'Optional
End Sub

The macros save the documents as Word 2010 docx format. If you want to save as word 97-2003 DOC format as suggested by your use of DOC originally, change the relevant bold sections with

If Dir(strPath & strFname & ".doc", vbNormal) = "" Then
    ActiveDocument.SaveAs strPath & strFname & ".doc", wdFormatDocument97
Else
    strAsk = MsgBox("Filename Exists! Do you wish to overwrite?", vbYesNo, "Warning!")
    If strAsk = vbYes Then
        ActiveDocument.SaveAs strPath & strFname & ".doc", wdFormatDocument97
    End If
End If
