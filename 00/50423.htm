The “unknown publisher” -- If the program has an unknown publisher, it doesn't have a valid digital signature from its publisher. This doesn't necessarily mean the program is harmful—many older, legitimate programs lack signatures. However, you should use extra caution and allow a program to run only if you obtained it from a trusted source, such as the original CD or a publisher's website. If you’re unsure, look up the name of the program on the Internet to determine if it’s a known program or malicious software.
