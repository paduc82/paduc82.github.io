Looking at the minidump file the error points to an issue related to USB:

BugCheck 133, {0, 501, 500, 0}

Probably caused by : USBPORT.SYS ( USBPORT!USBPORT_ProcessNeoStateChangeList+3e )

The issue may be from the CMUSBDAC.sys as the CMUSBDAC.sys has been causing issues recently for some users such as in the following topic:

https://answers.microsoft.com/en-us/windows/forum/windows_10-performance/windows-kernel-power-crash-need-help/069b5a68-7c6a-4d54-bac2-6063ad062d6d

CMUSBDAC.sys is a driver for a C-Media USB device so try updating the driver for the C-Media USB device or uninstall and reinstall the driver or roll back to a driver without issue.

Also, see if there is an updated driver for the Logitech camera as the presently installed driver appears to be from 2012.
