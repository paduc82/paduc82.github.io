Office 2011 is quite inflexible with regard to Themes. IMHO, you're better off to stay with the Default Office Theme & use your own Paragraph & Character Styles to handle formatting.

You cannot select which fonts a Theme uses. The only option in Word, itself, is to choose one of the Themes selectable from the Themes button on the Home tab. Font pairings differ for each theme. Unfortunately, the only way to know which fonts are used by any given Theme is to select the Theme, then open the Fonts menu to see what's being used.

The only customization available must be done in PowerPoint. There you can create your own Custom Theme by assigning Theme Colors of your choice, but even there you are restricted to using only the built-in font pairings for Theme Fonts... you can't specify fonts of your own. Once the Custom Theme is saved it will be available in Word as well as Excel.

Full Theme customization has to be done in a Windows version of Office or by writing your own XML files.

Those are the most common contributors to something of this nature. See if you can correct it by:
  1. Save any open documents
  2. Use Word's Help> Check for Updates command
  3. Quit Word
  4. Apply any OS X updates not yet applied [Apple Menu> Software Update]
  5. Repair Disk Permissions using Disk Utility or comparable app
  6. Restart your Mac
