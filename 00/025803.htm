jay freedman say:

In each document (or in a template that you use to create documents like this), define a tab stop at the right margin, and specify a "leader" for the tab consisting of dashes.

To define the tab stop, modify the Normal style (and each other style that you expect to use). In the Modify Style dialog, click the option for "New documents based on this template" if you want to change the template; otherwise, leave "This document" selected.

Next, click the Format button and choose Paragraph from the menu. In the Paragraph dialog, click the Tabs button. In the Tabs dialog, enter the location of the right margin, choose the dashes leader, and click the Set button. Then click OK to close each of the dialogs.
