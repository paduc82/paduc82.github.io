For the 5 largest, entered in B2 and copied down to B6:

=LARGE(A$2:A$19,ROWS(B$2:B2))

For the 5 smallest, entered in C2 and copied down to C6:

=SMALL(A$2:A$19,ROWS(C$2:C2))

Sum of the 5 largest:

From a very recent similar post. Could be simplified but works

Sub redolistSAS_1()
Dim nr As Long
Dim R As Long
Dim i As Long
Dim j As Long
nr = Cells(Rows.Count, 1).End(xlUp).Row
R = nr + 2
For i = 2 To Cells(Rows.Count, 1).End(xlUp).Row
For j = 1 To Cells(i, "d")
Cells(i, 1).Resize(, 6).Copy Cells(R, 1)
R = R + 1
Next j
Next i
Rows(2).Resize(nr).Delete
End Sub

doug robbins - mvp office apps & services (word) say:

While it's simple matter to turn it off, either via the Review tab or by adding the track changes item to the status bar (Right click on the Status bar, you need to be a bit careful, because it is a toggle and if you double click, you will turn it back on.

 

Another thing that you could do would be create AutoOpen and AutoNew macros in your Normal.dotm template or in a template that you save in the Word Startup folder that contain the following code:

 

With ActiveDocument
    If .TrackRevisions = True Then
        .TrackRevisions = False
    End If
End With
