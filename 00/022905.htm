davidf2 say:

After opening an existing Publisher web publiction in Pub 2010, you will have a web tab in the menu > Navigation bar and select either a navbar that matches (or not) your existing navbar and insert. This will insert a single button without text. Right click the image part of the navbar button to find the option of Navigation Bar Properties. When you open that dialog you have the option of "Update this navigation bar with links to new pages...".  Uncheck that option and this navbar button will only appear on the one page. Click Add Link which opens the Add link dialog where you choose to link to another page in the document or an outside site. At the top of that dialog is where you choose the text to appear on the navbar button. Once you are finished you can drag the navbar button to where ever you want on the page including below the existing navbar if that is where you want it.
