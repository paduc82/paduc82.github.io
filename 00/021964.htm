  2. (quite possibly optional: Delete all the unused test identities via the Microsoft Database Utility)
  3. Move the MUD folder from ~/Library/Preferences to ~/Documents
  4. Pray this didn't break anything
  5. Start Outlook, search didn't work yet but all the mail and settings were still there
  6. Close Outlook
  7. Use "mdimport ~/Documents/Microsoft-Benutzerdaten" and confidently ignore the error messages about "mach-o, but wrong architecture)
  8. Start Outlook, search works for some older emails
  9. Close Outlook
  10. Use "mdutil -E /Volumes/Macintosh\ Data" to reindex the whole volume
  11. Wait
  12. Start Outlook, search now works for most emails (only the most recent ones do not appear yet)
  13. ???
  14. Profit!

jay freedman say:
Use the "Organizer" dialog. There are two ways to get there: If you have the Developer tab displayed on the ribbon (it's hidden by default) click the Document Template button; or press Alt+F8 to open the Macros dialog. In either case, click the Organizer button. The dialog has two tabs, and you want to go to the Styles tab.

On one side of the Styles tab, the "Styles available in" box should show your document. In the list above that box will be all the styles' names. You can select as many of those names as you need to, using Ctrl+click to add one to the selection, or Shift+click to add any continuous group. When you have a bunch of names selected, click the Delete button in the center of the dialog. You can repeat the select-and-Delete until you've gotten the list down to just the styles you want. Then click the Close button and save the document.
