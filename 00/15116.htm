alex wang msft say:

Hi Lonna,

OneDrive keeps Modified time of each files. The Modified column keeps the last modified time of the file. When anything happens to the file, such as renaming, moving, uploading, the Modified column value updates. It is not feasible to change the column to other time. There is also a Create column in OneDrive for business,  and the Created column keeps the created time of the file. 

Maybe you can add the Created column to the view in OneDrive Online, but it will take more steps to show the view:

1. click Return to classic OneDrive(at the bottom-left corner) then follow these steps:

1. Go to LIBRARY > Library Settings > scroll to the bottom and click Create view.  (If there is no LIBRARY at the upper-left corner, please click the Settings icon at the upper-right corner > Show Ribbon )

2. Choose a view type (Standard View), give a name to the view, then you can select which columns to be displayed in this view.

3. Back to OneDrive, under the LIBRARY tab, choose a Current View by selecting the customized view name created in step 2.

When you need this information, you can return to classic OneDrive and choose the view.

You can also make this view default in OneDrive, but there is only a out of box way that using CSOM powershell to make it: How to set new view as default view in OneDrive for Business default Document library?

Regards,

Alex
