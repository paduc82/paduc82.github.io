frx valak say:

There are multiple ways of going about this, depending on whether you want to simply turn off windows defender temporarily, or disable it entirely. I will provide a method for each.

1. Turning off Windows Defender:

Open the start menu and go to settings. Under settings, go to "Updates & Security". Click on the "Windows Security" tab in the upper-left side of the window. Click on the "Virus & Threat Protection" heading. This will open the Windows Defender window. From this window, click on "Virus & Threat Protection Settings". Under the "Real Time Protection" heading, turn the toggle switch to "off" and click Yes when prompted.

With this method, Windows Defender will turn itself back on the next time you restart your PC.

2. Disabling Windows Defender:

Open the start menu and go to the registry editor (type in regedit and click the blue icon at the top of the start menu, then click Yes when prompted). Find the "HKEY_LOCAL_MACHINE" folder and expand it by double-clicking (in some cases this folder may already be expanded, in which case you can skip this step). Expand the "SOFTWARE" folder, then scroll down and expand the "Policies" folder, then expand the "Microsoft" folder, and then click once on the "Windows Defender" folder. With this folder selected (by clicking on it once), you will then need to right-click it, which should cause a drop-down menu to appear. Select "New" near the top of the drop-down menu. Doing this should prompt a pop-out menu. Click the "DWORD (32-bit) Value" option in this pop-out menu. This should create a blue and white file in the "Windows Defender" window on the right side of the page. Enter "DisableAntiSpyware" as the file name. When the DWORD file appears, type in DisableAntiSpyware and press enter. Open the "DisableAntiSpyware" DWORD file by double-clicking it. A pop-up window will open. Replace the "Value Data" number with 1 (this will turn on the DWORD value), then click OK. After this is complete, close all of the windows and restart your PC, and Windows Defender will be disabled. 

If you decide to re-enable Windows Defender in the future, simply return to this same exact "DisableAntiSpyware" Value that was created, and change the "Value Data" from 1 to 0, then restart the PC. (You can also delete this value altogether if you wish). 

I hope this helps, but keep in mind that you must be careful when changing or editing things in the Registry Editor. If you have any further 

questions or concerns, please feel free to ask.

By "inserting" you mean as a printout (visible, so you can see the content of the PDF pages in OneNote)?

OneNote is no PDF reader, it can not render (display) PDFs at all. What it does is printing the PDF to OneNote.
