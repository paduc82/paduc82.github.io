microsoft barbie say:

Office Professional and Ultimate versions do include Access.  You can also purchase a stand-alone version of Access for about $229:  http://office.microsoft.com/en-us/access/default.aspx.  Access 97 is quite old, and is not recommended for Windows 7.  That being said, you can probably get it to work.

You can try to run it in Compatibility mode.  Click Start, type Compatibility in the search field, and click the Enter key. This will bring up the Program Compatibility troubleshooter.  Click Next and it do a quick check on your machine.  It will ask you to Select the program you're having problems with and click Next.  Click Troubleshoot Program, then check the box for The program worked in earlier viersions of Windows...  Click Windows XP (Service Pack 2) and click Next.  Click the box for Start the program.  It will attempt to open the program as if you were running XP. After the program attempts to open, click Next. Now it will ask if it worked for you.  If so, click Yes, save these settings for this program.  If you clicked Yes, every time you open the program now it will open as if it was running in XP.  You're done if this worked!

Sometimes that method doesn't work.  For those cases, there is a new feature called XP Mode, which will install a virtual machine for you to run your program  in.  It's not too hard to get configured; it actually installs a legal copy of Windows XP for you to run your program in.   Here's a Microsoft article on how to get it set up:  http://www.microsoft.com/windows/virtual-pc/.  XP mode will now work even if you don't have the chip installed for Hardware Assisted Virtualization Technology.

skier-hughes say:

Burning at a fast speed can cause problems with inaccurate burns, so changing to a slow burning speed is classified as a solution to the problem, rather than a way around it.

 

Another thing to try is lowering bitrates, lots of dvd players can have trouble with dvds burned at greater than 8,000 and this figure needs to include both video and audio bitrates.

 

DVD manufacturers can use and/or swap their dye manufacturers or the same dye manufacturer can upgrade their dye. This means although you are using the same make they may be totally different. Even some makes have several factories and one made in China may be different to one made in Taiwan, so I'd buy a different batch of dvds to trial. It could just be a faulty batch, it does happen.  I know, I burn hundreds of dvds every week, and have this problem even with the high quality taiyo yuden disks that I buy.

 

Multi tasking is a hard one, as it only needs something to be using the same data route as that which the dvd burner is using to cause a glitch. I'd turn off everything that you don't need to be running.

 

Does the issue show up at the same point of the movie in each dvd player?

Or

Does the problem occur on each dvd player but at different points in the movie?

 

The first tends to be bad disc, the second bad authoring/burn.
