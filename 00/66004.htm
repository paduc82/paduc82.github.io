I assume from your comment about "Other Devices" that there is an entry for the printer in Device Manager.  The following may work:

  1. On the Brother download page for your printer (http://support.brother.com/g/b/downloadlist.aspx?c=us&lang=en&prod=dcpl2540dw_us_as&os=93), select the proper version of Windows and then download the "Add Printer Wizard Driver" (http://support.brother.com/g/b/downloadend.aspx?c=us&lang=en&prod=dcpl2540dw_us_as&os=93&dlid=dlf100999_000&flang=4&type3=407).  It may be best if you save the downloaded file in its own new (temporary) folder.

  3. Right-click on the printer entry in Device Manager and select "Update Driver"
  4. Click "Browse my computer for driver software"
  5. Click "Let me pick from a list of device drivers on my computer" and click Next
  6. Click the "Have Disk" button
  7. Browse to the folder where you saved the extracted files in Step 2 and select the *.inf file.  It should be named brprxxxx.inf (you may not see the .inf if you have Windows Explorer set to hide file extensions for known file type).  Click "Open"

@echo off
goto start

::What this .bat does-

::Checks for administrator permissions by seeing if we can use `net session`
::then installs ScpVBus, HidGuardian, and HidCerberus.Srv in that order.
::Then ping localhost twice to wait for one second.
::Then the HidCerberus.Srv service is started.

::tell HidGuardian to block the Switch Procon's hardware ID.

:start

if %errorLevel% neq 0 (
    echo Administrative permissions denied. Launch as an Administrator!

)

cd ..\HidCerberus.Srv
echo Installing HidCerberus.Srv...

HidCerberus.Srv.exe install

ping 127.0.0.1 -n 2 > nul

net start "HidCerberus Service"

ping 127.0.0.1 -n 2 > nul

cd ..
wscript "add HidCerberus blacklist.js"

:exit
pause
