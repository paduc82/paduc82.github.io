vianyil say:

Wrapping text around an entire page

Hi all.

I am writing a document which requires images and figures to be on separate pages to text. That is, I can have a page of text where I refer to a figure, but that figure must be on its own on the next page. I have tried playing around with various breaks and margin tricks but to no avail. I want to try and wrap the text around the whole page so it flows naturally to the next available page after the figures. I want it to look something like this:

1. Page full of text with reference to Figure 1 and Figure 2

2. Page with Figure 1 and Figure 2

3. Page which continues original text

Any suggestions?

Many thanks.

suzanne s. barnhill say:

I'm surprised you haven't found an answer to this question, as it has been asked many times. What you may mean is that you haven't found an answer that tells you how to do it. That's because what you ask is not possible in Word. Wrapped elements must be anchored to a text paragraph on the same page, so it is not possible for them to be entirely divorced from the text flow.

 

That said, this is how you must accomplish what you need:

  1. Don't even try to insert figures until editing of the text is complete (or, if you do insert them, don't worry about final placement).
  2. When editing is complete, at the end of your page (1), insert a page break (more on this later).
  3. On the following page (2), insert your figures 1 and 2, followed by another page break.
  4. Your text will continue on page (3).
  5. If you are fortunate enough to be able to insert the page break at the end of a paragraph on page (1), then you're done. Or, if your paragraphs are ragged right and have no first-line indent, you're done.
  6. If your paragraphs are justified and/or have a first-line indent, then you have one or two more steps.
  7. At the end of page (1), before the paragraph break, insert a line break. This will justify the last line of text on the page, but the paragraph mark on the next line will overflow the page. Format it as Hidden (Ctrl+Shift+H), and it will disappear, and the text will again fit on the page.
  8. At the beginning of page (3), apply an unindented style (Body Text, for example, if you're using Body Text First Indent) for the indented paragraphs) to the first paragraph, so it will appear to continue from page (1).

Yes, this is very tedious, but it is the only way you can achieve this effect in Word. It takes longer to describe than to actually do, and provided you wait until editing is complete, you will only have to do it once.

jp ronse say:

Hi,


Try this:


=ROUND(((HOUR(OR(L15=24,L15=0)*1+TIME(+L15,+N15,0)-TIME(+H15,+J15,0))*60)+MINUTE(OR(L15=24,L15=0)*1+TIME(+L15,+N15,0)-TIME(+H15,+J15,0)))/60,2)


You can subtract times but it cannot be negative. Excel recognize 24 as 12:00 AM (or 0). If we pass over midnight, the end time will always be smaller then the start time. Therefore we must add 1 to make it greater. That is what OR(L15=24,L15=0)*1+ does.


You can simplify the formula to:


=IF(H15="","",ROUND((((L15=0)*24+L15-H15)*60+N15-J15)/60,2))


There is no need to convert it to time and calculate hours & minutes.


(L15=0)*24+L15-H15)*60 subtract the hours and convert it to minutes. (L15=0)*24 as workaround  for midnight if we use 0 iso 24.


Add the difference of the minutes: +N15-J15

Divide by 60.

[EDIT] For better understanding.

Dates & times are a format applied on the value. We have the freedom to interpret 24 as 24 hours, 24 days, 24 months, years etc. We only have to keep in mind that all values should have the same base.

So if we want to add 10 minutes to hours, we have to multiply the hours by 60 to have the same base.
