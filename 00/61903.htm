how to repair large dbx files in outlook express - Microsoft Community

darek.lesniewski say:

how to repair large dbx files in outlook express

I have some dbx files with mailboxes which exceeds 2GB size limit. I'am looking for any Microsoft tool to fix that. I found some third party tools but they are not very useful. Is there any tool from Microsoft?
steve cochran say:

No.  That's why the 3rd party tools were written.  The best one is mine (DBXpress -- www.oehelp.com/DBXpress/).  I know because I wrote it.

Seriously, you cannot repair such a file.  The only thing you can do is extract the messages from it, if it still contains messages.  Sometimes the files get wiped out and filled with zeroes and then nothing can extract messages from a file filled with zeroes.  You can try zipping the file and if it zips down to nothing, that's what's in the file:  Nothing.  If that is the case then my DBXpress program is the only one that has the extract from disk feature that will bypass the file system and analyze the entire disk drive directly for messages.   If they are still on the disk clusters somewhere, it will recover them.

But one should not let a dbx file get above a few hundred megabytes or else it gets subject to corruption.  The backup of the dbx files as bak files prior to folder compaction (see www.oehelp.com/OETips.aspx#2) was added just because of that issue -- so many people were letting their folders get too large and then during compaction the files would get loaded with zeroes (and antivirus software made it worse). 

So the best thing you can do in the future is to backup frequently (there are FREE tools on my website -- see www.oehelp.com/OETips.aspx#6) and keep the size of the dbx files under a few hundred megabytes.
