considerate_guy say:

One way is to save the given video and audio as its own movie. File > Save movie. That will create a new movie with everything synched the way you want. Now if you start a new project, add this movie and add your titles.

Adding transitions will always affect the timing and you'd need to account for that. For example, if you have two pictures and you are telling them to transition with say a 1.5 second transition, you should add 1.5 seconds to one of the photos so everything else in the storyboard is still in synch. But there isn't a way to make that happen automatically or to "lock" the audio to the video.

donna men say:

Hi Gordon-A,

This error message usually occurs when there is a software updater running on your startup. As an initial troubleshooting, I suggest you follow these steps:

  1. Open Task Manager by simultaneously pressing Ctrl + Shift + Esc.
  2. Go to Startup tab, and check if Software updater is on the list.
  3. If so, select it then click Disable.
  4. On Search/Cortana, type "mscofig", then Enter.
  5. On System Configuration window, go to Services tab.
  6. Tick the check box beside Hide all Microsoft Services.
  7. Check if Software updated is on the remaining list, then uncheck its box.
  8. Click Apply and OK.
  9. Restart PC.

The document output selection simply adjusts the scan output settings to be more accurate for a doc

A scan simply copies the document as an image (photo)
