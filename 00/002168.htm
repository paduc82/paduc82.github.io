kris ver say:

Hi Shinsuke_0124,

Yes, you can set up a rule to move the junk emails on your inbox to the Trash or Junk Email folder. Here are the steps on how to do it in Outlook 2016:

1. In Outlook, under the Home tab, on the Move group, click Rules. 

2. Then, select Manage Rules & Alerts.... Rules and Alerts dialog box will pop-up. 

3. In the Rules and Alerts dialog box, click New Rule... under the E-mail Rules tab. Rules Wizard dialog box will pop-up. 

4. In the Rules Wizard dialog box, click Apply rule on messages I receive. Then, click Next. Another Rules Wizard dialog box will pop-up. 

5. In the new Rules Wizard dialog box, in Step 1, put a check beside the option with specific words in the sender's address. Then, click specified words in Step 2. Search Text dialog box will pop-up.

6. In the Search Text dialog box, type the email domain in the Specify a word or phrase to search for in the sender's address box. Then, click Add. Then, click OK. 

Note: You can add multiple email domains as you need.

7. The Search Text dialog box will close and it will return to the Rules Wizard Dialog box. Click Next. 

8. In the new Rules Wizard dialog box, put a check  beside the move it to the Step 1. Then, click specified in Step 2. Rules and Alerts dialog box will pop-up.

9. In the Rules and Alerts dialog box, select the specified folder(e.g. Junk Email) which you want to move the messages to. Then, click OK. 

10. The Rules and Alerts dialog box will close and it will return to the Rules Wizard Dialog box. Click Next. 

11. Another Rules Wizard dialog box will pop-up. But this time, click Next without any settings.

12. In the last Rules Wizard dialog box, name the rule in the Step 1 section. Then, put a check beside the options Run this rule now on message already in “Inbox” and Turn on this rule under Step 2. Then, click the Finish button.

13. After clicking the Finish button, all emails that will meet the rule condition will be moved to the specified folder automatically. Just click Apply and OK once it returns to the Rules and Alerts dialog box. 

You can also create and use inbox rules in Outlook.com. However, currently, you can't run inbox rules on existing messages. A new rule is applied only to messages you receive after it was created. For more information about inbox rules in Outlook.com, kindly check this article: Use inbox rules in Outlook.com

If you have additional questions, or if the issue persists after the troubleshooting steps above, please let me know by replying to this thread. This way, I'll get notified and respond back to you as soon as I can.

The search function has stopped working in Outlook for Mac Version - Microsoft Community

bradwu say:

The search function has stopped working in Outlook for Mac Version 15.31

Split from this thread.

After upgrading to 15.31, I found outlook can just show the search results before 2017. Any results after 2017 are not displayed. how to fix it?

aaron zhu msft say:

Hi BradWu,

 

I understand that the search results are not as expected in Outlook 2016 for Mac. In most cases, Outlook for Mac search issues are related to the search database index and re-index it could solve the issues. As a result, please refer to the following steps to try it and see if the issue can be fixed.

 

1. Open Terminal by going to Applications --> Utilities --> Terminal

2. Type "mdimport -L"

3. If you see more than one instance of Microsoft Outlook Spotlight Importer.mdimporter, delete the Outlook application that you are NOT using, empty it from the Trash, restart your Mac, and go back to Step 1.

4. Now re-index your Outlook database using the following command, substituting your own <user_name>:

mdimport -g "/Applications/Microsoft Outlook.app/Contents/Library/Spotlight/Microsoft Outlook Spotlight Importer.mdimporter" -d1 /Users/<user_name>/Library/Group\ Containers/UBF8T346G9.Office/Outlook/Outlook\ 15\ Profiles/Main\ Profile 

 

Note in the command above the path after "-g" is the default path where your Outlook is installed. The path after "-d1" is the default path to your profile. You will have to substitute your actual paths if you have renamed your profile or installed Outlook in a different location.

 

5. After starting the command in Step 5 it will take some time to complete re-indexing. Please wait until the re-indexing completes. 

6. Quit and re-launch Outlook. 

 

If the issue persists after performing the steps above, I need to collect some information for investigation:

 

  1. The detailed version of your Outlook, such as 15.31 (170117). To get this, click Outlook > About Outlook.
  2. Can you manually find those emails which were received after 2017 in your Inbox?
  3. Which kind of account are you using, an Office 365 for Business account, a Gmail account or any other kind of account? Meanwhile, please check if the search function works fine using the web app of your mail server. For example, OWA(Outlook Web App) for an Office 365 account.

 

Any updates will be appreciated.

 

Best Regards,

Aaron
