a. user say:

So I "solved" the issue by plugging in three network cables into the network card of the PowerEdge and teaming them using the 'Adapter Fault Tolerance' type.

Now, one network adapter is active and the other two are on standby.

Once the active one fails, one of the standby ones immediately and fluently takes over, keeping network connections like VNC or FTP transfers going while doing so.

The best thing about this approach is that all network adapters share the same internal IP-address.

So whichever adapter is active, it will have the same IP as each other one, when they are active.

This is great for all my port-forwardings, which rely on internal IP-addresses.

a. user say:

Hi Robert,

 

We are sorry to hear that you are unable to use Windows Media Player since the day you have purchased Windows 8.

 

This problem occurs when Windows Media Player database gets corrupted.

 

So let’s try out these methods and check for the cause of the issue.

 

Method 1:

I would suggest you to run the Windows Media Player library troubleshooter which will tell you about the problems with Windows Media Player library.

Follow these steps to run Windows media player Library troubleshooter.

a.      Press Windows key + W and type troubleshooting.

b.      Select troubleshooting.

c.       Select View all on the top left corner.

d.      Run Windows Media Player Library troubleshooter.

 

Method 2:

To resolve the problem, we need to clear the Windows Media Player database.

Follow these steps to do so.

a.      Exit Windows Media Player.

b.      Press Windows key + R and type %LOCALAPPDATA%\Microsoft\Media Player and hit Enter.

c.       Select all the files in the folder, and then click Delete on the Home menu.

Note: You do not have to delete the folders that are in this folder.

 

Method 3:

If this does not resolve the problem, clear the Windows Media Player database cache files. To do this, follow these steps:

a.      Exit Windows Media Player.

b.      Press Windows key + R and type %LOCALAPPDATA%\Microsoft and hit Enter.

c.       Select the Media Player folder, and then click Delete on the Home menu.

d.      Restart Windows Media Player.

Note: Windows Media Player automatically rebuilds the database.
