jay freedman say:

There are a couple of possibilities, none of them related to the styles or language dialogs.

Probably the best, because it's saved in the document file and is effective when the document is opened on another computer, is to go to File > Options > Proofing and check the box for "Hide grammar errors in this document only". You'll need to do this in the template used to create these documents, or else once in each document. The drawback (if you consider it to be one) is that it suppresses all grammar-checking, not just capitalization.

The Capitalization check box does still exist in Word 2013 / Office 365. Thus, another possibility is to go to the Proofing options and click the Settings button next to the Writing Style dropdown. You can uncheck the box there:

However, this setting applies only on your computer (to all documents that you open while it remains unchecked) and is not saved in the document file. If another user opens the document on a computer where the box is checked, they'll see the underlines.

A similar option -- applicable only on your computer -- is to uncheck the "Mark grammar errors as you type" box on the Proofing options page. If you also don't want to see the capitalization errors when you do a manual (F7) spelling check, also uncheck the "Check grammar with spelling" option.

Finally, to avoid inadvertent capitalization of those headings, you can click the AutoCorrect Options button in the Proofing page, and uncheck the option for "Capitalize first letter of sentences".

Sub mcrCopyStuff()
    iNewRow = Sheets("Sheet2").Range("A" & (2 ^ 20)).End(xlUp).Row + 1
    Sheets("Sheet2").Range("A" & iNewRow) = Sheets("Sheet1").Range("F1").Value
    Sheets("Sheet2").Range("B" & iNewRow) = Sheets("Sheet1").Range("F5").Value
    Sheets("Sheet2").Range("C" & iNewRow) = Sheets("Sheet1").Range("B6").Value
    Sheets("Sheet2").Range("D" & iNewRow) = Sheets("Sheet1").Range("G21").Value
End Sub

This is an issue which may occur in Nokia N86 8MP phone software version 20.115. If some of the video clips cannot be played after updating the phone software to 20.115, the problem can in most cases be solved in the following way:

3. Delete the following 3 files by browsing to the file in the list and selecting Options > Delete:
