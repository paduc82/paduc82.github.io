___metalman___ say:

Publish Movie as AVI in Windows Movie Maker 6.0

I'm new to Movie Maker.  Apparently older versions has an easy way to Publish/Save a movie as AVI.  Is there a way to do this in version 6.0?   I don't see any drop down or other options to do this.  Thanks, I use Vista.

considerate_guy say:

It depends on what you mean by "AVI" format.  Yes, there is a way to save in DV-AVI format.  This is a lightly compressed codec intended for SD footage from a DV camera.  It uses up about 13.5 MB per hour of footage.

 

To use this format, choose File > Publish Movie.  Choose to save to your computer, give the movie a name.  On the next screen you should see 3 radio buttons.  Choose the last option which is "More Settings" and you should see the DV-AVI option.

 

Note: The AVI (Audio-Video Interleave) format can be used with other compression codecs, but WMM 6.0 can only generate DV-AVI files.  If what you want is something different (say MPEG4 encoded video in an AVI container), that can't be done directly out of WMM 6.0.  Instead, what you would need to do is convert the video after saving.  Save as normal in .wmv format then use a conversion program (like Freemake Video Converter) and convert to the desired format.

Welcome to the Microsoft Windows Vista Communities site.

 

 

Mouse options in Windows…

 

There are some options within Windows to adjust the scroll wheel behavior. However, your mouse most likely has its own device driver, and that driver will control this functionality. If when scrolling you get a “stuttering” or “jerking” effect the most likely cause is the device driver. You would want to try updating the driver after visit the manufactures website for the most up-to-date driver version.

 

If you aren’t experiencing any jerking or stuttering you can try modifying the settings within Windows.

 

Modifying the mouse settings within Windows…

1.     Click Start, click Control Panel, and click Mouse under the Hardware and Sound category.

2.     Click on the Wheel tab, and you can adjust the “number of lines” per wheel notch.

3.     You will want to decrease the number of lines for a “smoother” effect.

4.     Depending on your hardware there may be addition menu items or options.

5.     See if there are any options that would control your scrolling aside from the ones mentioned above.

6.     If so adjust the configuration until it suites your needs.

Step 1:

 

Disable and Enable the Webcam Driver

 

a. Shut down and restart the computer.

 

 

b. From the Windows 7 default Start menu, click Control Panel.

 

 

c. In Control Panel, click System and Maintenance, and then click Device Manager.

 

 

d. In the User Account Control dialog box, click Continue.

 

 

e. In Device Manager, click the plus sign (+) next to Imaging Devices.

 

 

 

f. Right-click Gateway USB 2.0 Webcam, and then click Disable.

 

 

g. In the Gateway USB 2.0 Webcam dialog box, click Yes.

 

 

h. Right-click Gateway USB 2.0 Webcam again, and then click Enable.

 

 

i. If this did not resolve the webcam issue proceed to step two.

 

Step 2:

 

Uninstall the Camera Assistance Software

 

1.   From the Windows 7 default Start menu, click Control Panel.

2.   In Control Panel, under Programs, click Uninstall a Program.

3.   In the Uninstall or Change a Program dialog box, click Camera Assistance Software for Gateway, and then click Change.

4.   In the User Account Control dialog box, click Continue.

5.   In the Camera Assistance Software for Gateway – Install Shield Wizard, click Remove, and then click Next.

6.   When prompted for confirmation, click Yes.

7.   In the Uninstall Complete dialog box, click Yes, I want to restart my computer now, and then click Finish.

8.   If this did not resolve webcam issue proceed to step three.

 

Step 3:

 

Reinstall the webcam driver from the Gateway Recover Center.

 

1.   From the Windows 7 default Start menu, click All Programs, Gateway Recovery Center, and then click the Gateway Recovery Center program.

2.   In the User Account Control dialog box, click Allow.

3.   In the Gateway Recovery Center dialog box, click Hardware.

4.   In the Hardware dialog box, click Device Driver Recovery.

5.   In the Device Driver Recovery dialog box, click Other.

6.   In the Other dialog box, click Chicony WebCam SW and Driver, and then click Next.

7.   In the Congratulations dialog box, click Finish.

8.   Shut down and restart the computer.

9.   If this did not resolve webcam issue proceed to step four.
