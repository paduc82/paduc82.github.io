UDF syntax- how to overwrite the source cell's formula? - Microsoft Community

kergga say:

UDF syntax- how to overwrite the source cell's formula?

I understand there are limitations with UDFs related to changing other cells, cell formatting, etc.

In my particular case, I can get the UDF to change the cell value as expected, but I have (so far) been unable to get the UDF to change the cell.formula (to update its own parameters). I don't know if this is a limitation of UDFs, or if my syntax is incorrect.

In the following example (still under construction), when the formula is first entered it needs to capture and store the value in the referenced cell, as the third parameter. For convenience, I also start with the 2nd parameter blank, so I can detect whether the UDF has captured the third parameter. That is not really necessary in this example, but my intent is to add further capabilities to this function and I'll be using that second parameter for additional conditions.

Any thoughts on how to have the UDF change the UDFs own parameters in the calling cell would be greatly appreciated!

Thanks,

Keith

Function UpdateOnce(TVal As Variant, Optional RepeatFlag As Variant, Optional FVal As Variant)
    'First parameter simply points at the cell that the user wants to monitor
   
    Dim TCell As Range
    Set TCell = Application.ThisCell
    TForm = TCell.Formula
   
    '"ThisFlag" is the 'stamp' to tell who changed the cell, and when
    Dim objNet As Object
    On Error Resume Next
    Set objNet = CreateObject("WScript.NetWork")
    ThisFlag = objNet.UserName & " " & Now()
    Set objNet = Nothing
   
    'RepeatFlag tells us if the user just entered this UDF (blank -> 1)
    '                    or if a change has been detected in the source cell (1 ->9)
   
    If IsMissing(RepeatFlag) Then
        FVal = TVal 'capture the current (baseline) value (e.g. value in target cell)
        RepeatFlag = 1 'default to only one update for now
        TCell.Formula = Left(TForm, Len(TForm) - 1) & ",1," & FVal '<-- ** Has no effect **
        UpdateOnce = "" 'don't insert a value until the first change /after/ the formula is entered
    ElseIf RepeatFlag = 1 Then 'update once only
        If TVal <> FVal Then
            TCell.Formula = ""
            UpdateOnce = ThisFlag
        End If
    ElseIf RepeatFlag = 9 Then
    'do nothing, this cell is already used
    Else
        UpdateOnce = "Invalid Parameter Call"
    End If

End Function
