tricky300 say:

Hi,

It may interest you to know that you do not need to use anything other than copy and paste and import and export in order to backup Outlook Express.

If you go to Tools, Options, Maintenance tab and then click Store Folder, it will show you where your folders are located (you can copy and paste this path if you want). There is also a button there to move this folder to a place of your own choice.

It doesn't happen that often these days, but when I have to set up Outlook Express again for customers, I always move this store folder to (for example) a folder in My Documents called Outlook Express. If the customer regularly backs up My Documents, Outlook Express is backed up by default.

Restore (of messages) is effected by either copying the folder back to My Documents or by using the Outlook Express import facility and pointing it to the backup location.

Back up of Contacts can be done by Import and export from the address book File menu.

To backup your account settings, go to Tools, Accounts where you will find Import and Export features.
