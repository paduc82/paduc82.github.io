adampride say:

It's easy to add a control to the right-click menu and then have that control run a subroutine which does the Paste Special... Transpose. Paste this code into a Module in the VB Editor:

Private Sub AddToCellMenu()
' Added July 2016: Adds the 'Fill Down' button to the Cell Context Menu
' This is the ony way to modify a context menu
  With Application.CommandBars("Cell")
    .Controls.Add Type:=msoControlButton, ID:=372, Before:=5    ' Fill Down Control
    .Controls("Pick From List...").Delete                         ' I've never ever used this !
    .Controls("Clear Contents").Delete                          ' Easier to use the keyboard
' Added Feb 2017. Paste + Values & Paste + Transpose on Right-click Cell Menu
    With .Controls.Add
      .Move Before:=4:  .FaceId = 526:  .Caption = "Paste Values":    .OnAction = "PasteValues"
    End With
    With .Controls.Add
      .Move Before:=5:  .FaceId = 1813: .Caption = "Paste Transpose": .OnAction = "PasteTranspose"
    End With
  End With
' Un-Rem this line to return the Cell context menu to default
'  Application.CommandBars("Cell").Reset
End Sub

This does a few other modifications to the right-click menu, i.e. adds the 'Fill Down' control, which I use quite a lot, and removes a couple of controls which I never use. Rem-out or delete these lines if you don't want these modifications. This moves the original 'Paste Special...' to the bottom of the list - change the '.Move Before:=' number to adjust where the controls go.

You then need to add this code:

Private Sub PasteValues()
  On Error Resume Next    ' In case there is nothing in the Clipboard to paste
  Selection.PasteSpecial Paste:=xlValues
End Sub

Private Sub PasteTranspose()
  On Error Resume Next    ' In case there is nothing in the Clipboard to paste
  Selection.PasteSpecial Paste:=xlValues, Transpose:=True
End Sub

Obviously if you want only the Paste Special... Transpose function then remove the Sub PasteValues and the code which adds this control. I prefer to paste values when I paste transpose but if you don't want this then remove 'Paste:=xlValues,' from Sub PasteTranspose().

You only run Sub AddToCellMenu() once to add the controls. This will only work in the workbook the code was pasted into. If you want it to work in any workbook the you'll have to make it an Add-In. There's loads of information on how to to this, e.g. http://www.cpearson.com/excel/createaddin.aspx

The .FaceID defines what the icon looks like. You can find what icons are available at http://skp.mvps.org/faceid.htm

This works in Excel 2000 but I assume it'll also work in later versions. Hope this helps !

There are two concepts to understand here 1) product activation 2) access to online documents

1) To activate the Office 2016 for Mac product, you need to use a license that has 'client rights'. In your case, this can be satisfied with the Home account. Once the product is activated, you get full functionality.

2) As part of activation, we automatically connect you to the OneDrive service associated with the account you used to activate (which is what most people want). In your case, you want to also add a 'Connected Service' (this is done by clicking on your account/name in the File|New dialog where you normally choose which template/file to open. You can add a new Connected Service of OneDrive for Business that refers to your Business Essentials document storage.

Technically, the Mac client will be just using the 'Home' license in your scenario.

Private Type LARGE_INTEGER
  LowPart As Long
  HighPart As Long
End Type

Private Type MEMORYSTATUSEX
  dwLength As Long                         'The size of the structure, in bytes. You must set this member before calling GlobalMemoryStatusEx.
  dwMemoryLoad As Long                     'A number between 0 and 100 that specifies the approximate percentage of physical memory that is in use (0 indicates no memory use and 100 indicates full memory use).
  ullTotalPhys As LARGE_INTEGER            'The amount of actual physical memory, in bytes.
  ullAvailPhys As LARGE_INTEGER            'The amount of physical memory currently available, in bytes. This is the amount of physical memory that can be immediately reused without having to write its contents to disk first. It is the sum of the size of the standby, free, and zero lists.
  ullTotalPageFile As LARGE_INTEGER        'The current committed memory limit for the system or the current process, whichever is smaller, in bytes. To get the system-wide committed memory limit, call GetPerformanceInfo.
  ullAvailPageFile As LARGE_INTEGER        'The maximum amount of memory the current process can commit, in bytes. This value is equal to or smaller than the system-wide available commit value. To calculate the system-wide available commit value, call GetPerformanceInfo and subtract the value of CommitTotal from the value of CommitLimit.
  ullTotalVirtual As LARGE_INTEGER         'The size of the user-mode portion of the virtual address space of the calling process, in bytes. This value depends on the type of process, the type of processor, and the configuration of the operating system. For example, this value is approximately 2 GB for most 32-bit processes on an x86 processor and approximately 3 GB for 32-bit processes that are large address aware running on a system with 4-gigabyte tuning enabled.
  ullAvailVirtual As LARGE_INTEGER         'The amount of unreserved and uncommitted memory currently in the user-mode portion of the virtual address space of the calling process, in bytes.
  ullAvailExtendedVirtual As LARGE_INTEGER 'Reserved. This value is always 0
End Type
Private Declare Function GlobalMemoryStatusEx Lib "Kernel32.dll" (ByRef lpBuffer As MEMORYSTATUSEX) As Long

Enum ByteGraduation
  Bb = 0    'bytes
  Kb = 1    'kilobyte
  Mb = 2    'megabyte
  Gb = 3    'gigabyte
  Tb = 4    'terabyte
  Pb = 5    'petabyte
  Eb = 6    'exabyte
End Enum

Function BytesDiv(ByVal Bytes As Variant, ByVal Grad As ByteGraduation, _
    Optional kPower As Integer = 1024) As Variant
  'Return number of bytes as Kb,Mb,Gb, etc.
  If Grad = Bb Then
    BytesDiv = Round(Bytes, 0)
  Else
    BytesDiv = Round(Bytes / kPower ^ CDec(Grad), 0)
  End If
End Function

Function BytesMul(ByVal Bytes As Variant, ByVal Grad As ByteGraduation, _
    Optional kPower As Integer = 1024) As Variant
  'Reverse of BytesDiv
  If Grad = Bb Then
    BytesMul = Round(Bytes, 0)
  Else
    BytesMul = Round(Bytes * kPower ^ CDec(Grad), 0)
  End If
End Function

Private Function LargeInt2Decimal(LargeInt As LARGE_INTEGER) As Variant
  'Convert LargeInt to Decimal
  Const DLong = 4294967296#
  Dim Low As Variant, High As Variant
  'API:
  '  LargeInt2Decimal = CDec(1)
  '  CopyMemory ByVal VarPtr(LargeInt2Decimal) + 8, LargeInt, LenB(LargeInt)
  If LargeInt.LowPart < 0 Then Low = DLong + LargeInt.LowPart Else Low = LargeInt.LowPart
  If LargeInt.HighPart < 0 Then High = DLong + LargeInt.HighPart Else High = LargeInt.HighPart
  LargeInt2Decimal = CDec(Low) + CDec(High) * DLong
End Function

Function MemoryLoad() As Long
  Dim MemStat As MEMORYSTATUSEX
  'initialize structure
  MemStat.dwLength = Len(MemStat)
  'retireve memory information
  GlobalMemoryStatusEx MemStat
  MemoryLoad = MemStat.dwMemoryLoad
End Function

Function MemoryTotal() As Variant
  Dim MemStat As MEMORYSTATUSEX
  MemStat.dwLength = Len(MemStat)
  GlobalMemoryStatusEx MemStat
  MemoryTotal = LargeInt2Decimal(MemStat.ullTotalPhys)
End Function

Function MemoryFree() As Variant
  Dim MemStat As MEMORYSTATUSEX
  MemStat.dwLength = Len(MemStat)
  GlobalMemoryStatusEx MemStat
  MemoryFree = LargeInt2Decimal(MemStat.ullAvailPhys)
End Function

Function MemoryPageFileTotal() As Variant
  Dim MemStat As MEMORYSTATUSEX
  MemStat.dwLength = Len(MemStat)
  GlobalMemoryStatusEx MemStat
  MemoryPageFileTotal = LargeInt2Decimal(MemStat.ullTotalPageFile)
End Function

Function MemoryPageFileFree() As Variant
  Dim MemStat As MEMORYSTATUSEX
  MemStat.dwLength = Len(MemStat)
  GlobalMemoryStatusEx MemStat
  MemoryPageFileFree = LargeInt2Decimal(MemStat.ullAvailPageFile)
End Function

Function MemoryVirtualTotal() As Variant
  Dim MemStat As MEMORYSTATUSEX
  MemStat.dwLength = Len(MemStat)
  GlobalMemoryStatusEx MemStat
  MemoryVirtualTotal = LargeInt2Decimal(MemStat.ullTotalVirtual)
End Function

Function MemoryVirtualFree() As Variant
  Dim MemStat As MEMORYSTATUSEX
  MemStat.dwLength = Len(MemStat)
  GlobalMemoryStatusEx MemStat
  MemoryVirtualFree = LargeInt2Decimal(MemStat.ullAvailVirtual)
End Function

Function MemoryExtendedVirtualFree() As Variant
  Dim MemStat As MEMORYSTATUSEX
  MemStat.dwLength = Len(MemStat)
  GlobalMemoryStatusEx MemStat
  MemoryExtendedVirtualFree = LargeInt2Decimal(MemStat.ullAvailExtendedVirtual)
End Function
