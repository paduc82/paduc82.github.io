john korchok say:

When you link a file to PowerPoint, the path is hard-coded into your presentation. If the Excel file is in your user folder when you do this, the link will never work on someone else's computer, because the path includes their user name and is always different. 

You can work around this by placing the Excel file in a place that will have the same path on all Windows computers, like C:\. Then tell your colleagues to place the Excel file in the same place on their computers and the link will work.

An alternative is to request your colleagues to add the Edit Links to Files command to their Quick Access Toolbar, then edit the links to the Excel file. But this is more work for everyone.
