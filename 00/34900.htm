captaincadmium say:

Ok, so after a lot of trouble, things finally fixed for me. I think I managed to trick it into activating the SSD install.

For any one interested in how to change the boot drive:

I will use the terms Old Drive and New Drive. The former refers to the existing drive, and the latter refers to the fresh drive that you wish to use, in my case an SSD.

1) Upgrade through the original Windows 7/8/8.1 install on your system. After this is complete, go to step 2.

2) Use the Media Creation Tool to create a USB installation drive. Plug it into a spare USB 2.0 port.

3) Enter the BIOS and go into boot settings. Change the priority to the New Drive > USB > Old Drive.

4) Install Windows 10 on the New Drive using the USB.

It won't activate, as it is a fresh install and the key it comes pre-keyed with is a temporary key. You will get the option to purchase a key and to enter a new key, however do not do either of these.

5) Restart the system, enter the BIOS and boot into the Old Drive with out changing boot priority. (Most recent motherboards will have this option, just refer to the manual, or go exploring in the settings. I found it in the UEFI BIOS under BOOT MANAGER, for my Asus Motherboard).

6) You should now be loaded up into the activated version of Windows 10, as this was done straight through upgrading your previous OS. Go to Start > Settings > Updates and Security > Recovery > Reset this PC (Get Started).

7) Once you click Get Started, you will be prompted to restart the PC, so do that. 

Because of the boot options, it will then load straight into the New Drive, and it effectively swaps the activation so that the New Drive is activated and Windows deactivates the Old Drive.
