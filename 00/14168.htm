ken sheridan say: I don't see how you'd do this without some VBA.  I would pass the ConsultantID or similar key value to the form when you open it:

DoCmd.OpenForm "frmConsultants", OpenArgs:=Me.ConsultantID

or you might want to open it in dialogue mode to force the user to close it before returning to the calling form:

DoCmd.OpenForm "frmConsultants", WindowMode:=acDialog, OpenArgs:=Me.ConsultantID

Then in the frmConsultants form's module add a function:

Private Function GoToConsultant()

   If Not IsNull(Me.OpenArgs) Then
       With Me.RecordsetClone
           .FindFirst "ConsulantID = " & Me.OpenArgs
           If Not .NoMatch Then
               Me.Bookmark = .Bookmark
           End If
       End With
    End If

End Function

Set the form's On Load event property to:

    =GoToConsultant()

If you wish the user to be able to return to the original consultant record at any time after navigating to other records then you could add a button to the form whose On Click event property is similarly:

    =GoToConsultant()

- Select the People tab on the left

- Under "Online status and photographs" uncheck "Show user photographs when available"

- Click OK

- Restart Outlook
