Hello Sabine,


If you would like your email apps to not affect one another if you delete an email message, you will need to set up your email account on both of your devices as a POP (Post Office Protocol) email account.

Please note that you will have to remove your email account on both devices for you re-add them as a POP email account. You do not have to worry about losing your messages that are still stored on your email server. The email apps that you are using will automatically download the messages from the server after successfully setting it up again.

You will need to use the following incoming mail server settings:

Incoming mail server for POP: outlook.office365.com
Port: 995
Encryption method: SSL

For sending email messages, use the following email server settings:

Outgoing mail server settings SMTP (Simple Mail Transfer Protocol): smtp.office365.com
Port: 587
Encryption method: TLS or STARTTLS

You may also ask your company's email administrator for assistance on how you can set up your email as a POP acount.

ronrosenfeld say:

The FormulaArray property has a limit of 255 characters (see VBA Help for that property).  Your formula exceeds that limit.  The R1C1 formula type is not an issue in more recent versions of Excel.

I would suggest either splitting things up, or using a Name.  Here is an example of using a Name:  Note that I also changed the references to absolute instead of relative.  This will ensure they remain the same when you enter them using the Name.

================================

Sub foo()
Dim WS As Worksheet
Dim V As Variant
Dim I As Long
Dim sForm1 As String
sForm1 = "=IF(ISERROR(SUM(OFFSET($A$1,MATCH(1,ABS($A$1:$A$100),0)-1,0,MATCH(-INDEX($A$1:$A$100,MATCH(1,ABS($A$1:$A$100),0)),$A$1:$A$100,0)-MATCH(1,ABS($A$1:$A$100),0)))),SUM($A$1:$A$100),SUM(OFFSET($A$1,MATCH(1,ABS($A$1:$A$100),0)-1,0,MATCH(-INDEX($A$1:$A$100,MATCH(1,ABS($A$1:$A$100),0)),$A$1:$A$100,0)-MATCH(1,ABS($A$1:$A$100),0))))"

Set WS = ActiveSheet
With WS.Names
    'Check to make sure formula not already there
    On Error Resume Next
        V = .Item("myFormula")
        If Err.Number <> 0 Then .Item("myFormula").Delete
    On Error GoTo 0
    .Add "myFormula", sForm1
End With

Range("b1").FormulaArray = "=myFormula"

End Sub

it's amazingly easy to make Nokia-format themes using Notepad

Maybe this is old hat to some readers, but...

Take a Nokia theme file, which will have suffix ".nth".
Change the suffix to ".zip", and you will find a commonpor-garden zipped folder.

When you extract or view, you'll find a load of .jpg files, and an .xml file called a "theme descriptor".
You can just open the latter with a text editor such as Notepad and alter to your heart's content; and of course insert different .jpg files

Then zip the whole lot and change the suffix back to .nth
