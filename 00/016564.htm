rich007a say:

Hi.

Try this:

 

Sub Test3()
    Dim rng As Range
   
    Set rng = Range("A50000").End(xlUp).Offset(3, 0)
   
    rng.FormulaR1C1 = "New Item"
    rng.Offset(1, 0).FormulaR1C1 = "1234"
   
    Set rng = rng.Offset(16, 0)
    ActiveWindow.SelectedSheets.HPageBreaks.Add Before:=rng
   
    Do
        Set rng = rng.End(xlUp)
        Set rng = rng.End(xlUp)
        Set rng = rng.End(xlUp).Offset(1, 0)
        ActiveSheet.HPageBreaks.Add Before:=rng
    Loop While rng.Row > 8
End Sub

 

Much better to work with a Range object rather than selecting cells.  If you want to see what/where the rng is pointing to, you can temporarily add:

 

   rng.Select

 

to your code while you step through it (press F8 while in the VBE).

 

Depending on your data, an alternative loop which might work better could be:

 

    Do
        Set rng = rng.End(xlUp)
        Set rng = rng.End(xlUp)
        Set rng = rng.End(xlUp).Offset(1, 0)
        If rng.Row < 8 Then Exit Do
        ActiveSheet.HPageBreaks.Add Before:=rng
    Loop

 

 

Hope that helps.

suzanne s. barnhill say:

You can't save to an encrypted file; you can only save to a specific folder, and I'm not sure a folder can be encrypted. That aside, Word will choose the folder as follows:

  1. When you use Save As to save an existing document, Word should default to the folder where the existing file resides.
  2. When you use Save As to save a new document, Word will default to the currently active folder, that is, the one from/to which you most recently opened/saved a document.
  3. When you save a new document without having changed the folder (for example, when you have just started Word and created a new document), Word will default to the folder specified as the "Default local file location" at File | Options | Save. This is (or should be) the same folder shown for "Documents" at File | Options | Advanced: File Locations.
