lemp say:

1.  There is no "direct upgrade" from Windows XP to Windows 7.  You would have to do a "clean install," which would require you to back up all of your data files to external media and then restore those files after Windows 7 was installed. 

In addition, you would have to install all of your application programs, including the "Bankruptcy Software," from original installation media (e.g., CDs, DVDs, or downloaded installer files). 

You would also have to install Windows 7 drivers for the various bits of hardware in your computer.

2.  There are 2 complicating factors to doing the above:  your existing computer may or may not work with Windows 7 (e.g., there may not be Windows 7 drivers for some of your computer's hardware) and it has become quite difficult to find a copy of Windows 7 from a legitimate source. 

Microsoft stopped selling Windows 7 on October 31, 2013.  Although you may find Windows 7 for sale from online vendors, there is an increasing chance that any copy you purchase will be counterfeit and fail immediately to activate or -- worse -- apparently activate only to fail some months later. 

Even if if you purchase from a seemingly reputable source such as Amazon, you may end up with counterfeit software, as some of the reviews here suggest -->

http://www.amazon.com/Windows-Professional-System-Builder-Packaging/product-reviews/B00H09BOXQ/ref=cm_cr_dp_qt_hist_one?ie=UTF8&filterByStar=one_star&showViewpoints=0

3.  Other than the issue of purchasing potentially counterfeit software, moving to Windows 10 raises all of the same problems as moving to Windows 7.  If I were you, I would ask your software vendor if the current version of the Bankruptcy Software is compatible with Windows 10.  Assuming that it is, I suggest purchasing a new computer with Windows 10 pre-installed and then installing the software on the new computer.

keith a. miller say:

You can untangle this mess.  I want to take a moment to explain the naming issue.  These folders get their Display name from the desktop.ini file within each folder.  This lets Windows "think" in English while displaying Dutch names.  This is why Media became Downloads --- the folder is still named Media, but its Display name is Downloads.  This can be verified by viewing your D:\ drive from a command prompt.

The trick to relocating the folders is to remember that you want to redirect the location all the way to the new folder, not just its parent:  the location you should have specified for Downloads is D:\Media\Downloads.  For Pictures, D:\Media\Pictures, etc.

Now, for untangling the paths, you'll need to edit the Registry.   There are two keys where these values are stored:

HKCU\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\Shell Folders

HKCU\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\User Shell Folders

Most of the names are self-explanatory.  However, the name used for Documents is Personal, and Downloads is {374DE290-123F-4565-9164-39C4925E467B}.

I'm happy to help with more detail at any step in the process, but this should be enough to get you started.

The first step is to make sure none of the folders are pointing to D:\Media, but instead D:\Media\FolderName.

We can sort out desktop.ini issues later.

Keith
