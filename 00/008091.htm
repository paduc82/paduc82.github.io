How you implement this in your code is something that will need working out but this is how you can do it in a test workbook and I suggest you do that first. Create a new workbook and in VB editor create a userform. There's no need for any controls on the userform because this utilises the 'Click' event of the userform.

 

Right click this userform, view code and paste all the code below in.

Run the userform.

Click it to make the Application invisible/visible. Note you can still see Excel in the taskbar but the taskbar icon does nothing

 

Private AppLeft As Long
Private AppTop As Long
Private WState As Long

Private Sub UserForm_Click()
    If (Application.Left + Application.Width) > 0 Then
        AppLeft = Application.Left
        AppTop = Application.Top
        Application.Left = -(Application.Left + Application.Width)
        Application.Top = -(Application.Top + Application.Height)
    Else
        Application.Left = AppLeft
        Application.Top = AppTop
    End If
End Sub
Private Sub UserForm_Initialize()
    WState = Application.WindowState
    Application.WindowState = xlNormal
End Sub
Private Sub UserForm_Terminate()
    Application.WindowState = WState
End Sub

    4.    Go to the folder that has your Skype Name, and locate config.xml and drag it to the trash.
