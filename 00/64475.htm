quintilianoizquierdo say:

what is DNS

network error (DNS_unrresolved_hostname)
cmdrkeene say:

domain name server, the system that automatically translates Internet addresses to the numeric machine addresses that computers use.

It's what converts something "friendly" like a name http://microsoft.com into a address (like a phone book converts a name to a phone number).  For example, you can also go to the same website by typing http://23.100.122.175 into your address bar.

If your DNS can't find a number for whatever name you typed, it will say it can't resolve that name.
