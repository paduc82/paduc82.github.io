shinmila h say:

Hi M.Ahrens,

Hereâ€™s a brief detail about what System image does. A system image is in essence a snapshot of an entire drive(s). The backup is done in block level (as opposed to file level) increments and includes all user and system files, configuration data and applications that are present on the drive, plus information regarding disk layout and boot entries.
When you create a system image using Windows 7 Backup and Restore, it creates a virtual hard drive vhd-file which contains all your files and folders, and some xml-files which contain the information needed to restore your system using this image.

Note: If you have more than one drive or partition on your computer, a system image is only made of the drive or partition that Windows is installed on.

This answers your first question.

Creating a System Images saves everything on the drives you select, and then compress them into an image file which means less space for backups. This answers your second question.

Windows Key+I > System > Power & sleep > Click Additional power settings to open Power options > Click Change Plan Settings > Click Restore default settings for this plan

Reach to Power options as in previous step > Click Choose what the power buttons do > Click Change settings that are currently unavailable > Scroll down to Shutdown settings and uncheck Turn on fast startup > Click Save changes

Shut down your computer > Detach power chord > Remove battery (if fixed no need to remove) > Attach power chord > Reboot your computer
Windows Key+X > Device Manager > Expand Batteries > Right-click Microsoft ACPI-Compliant System > Uninstall > Shutdown the computer > Remove the power chord > Attach your battery > Attach your power chord > Reboot the computer to automatically install the driver

Microsoft Software Shadow Copy Provider service causes 100% active - Microsoft Community

steveklt say:

Microsoft Software Shadow Copy Provider service causes 100% active time on hdd.

I am writing this to help other people to fix their issue with 100% active disk time.

I've been struggling to find the service that causes the issue on Windows 8.1. I had several random hungings with reboots in my pc recently. I noticed that before it happens, the disk in the task manager was showing a 100% active time and no read or write activity. I though it was my hdd that was dying but i tried disabling windows services till I found that the one that was causing the issue was the Microsoft Software Shadow Copy Provider (swprv.dll). So the solution is to start the Services application, find the Microsoft Software Shadow Copy Provider service/Properties/Startup type: Disabled.

This will fix the issue till Microsoft releases an update to fix it. Then you can re-enable it if you want.
