Hi,

The previous update you've installed could be the cause as to why you encounter this issue. With this, we suggest that you follow these methods on how to troubleshoot your camera:

Check your antivirus software

In your antivirus application, look for settings related to blocking access or permission to use your webcam. For help, visit your antivirus software company’s website.

Check for an outdated webcam driver

  1. Select the Start  button, type Device Manager, and then select Device Manager from the search results. 
  2. Find your webcam under Imaging devices or Sound, video and game controllers. 
  3. Press and hold (or right-click) the name of your webcam, and then select Properties.
  4. Select the Driver tab, select the Driver Details button, and look for a file name that includes stream.sys. If it’s there, your webcam was designed before Windows 7 and you'll have to replace it with a newer webcam.
  5. If you don't find a file name that includes stream.sys, try rolling back your webcam driver.

Roll back your webcam driver

  1. In Device Manager, press and hold (or right-click) your webcam, and then select Properties. 
  2. Select the Driver tab, select Roll Back Driver, and then choose Yes. (Note that some drivers don’t provide a rollback option.) 
  3. After the rollback is complete, restart your computer, and then try opening the Camera application again.
  4. If rolling back didn’t work or isn’t available, uninstall your webcam driver and scan for hardware changes.

Uninstall your webcam driver and scan for hardware changes

  1. In Device Manager, press and hold (or right-click) your webcam, and then select Properties. 
  2. Select the Driver tab, select Uninstall > Delete the driver software for this device, and then select OK.
  3. In Device Manager, on the Action menu, select Scan for hardware changes. Wait for it to scan and reinstall updated drivers, restart your computer, and then try opening the Camera application again. 
  4. If your webcam still doesn’t work, try looking for help on the webcam manufacturer’s website.

To be clear: There is no keyboard shortcut available, record a macro.

But do not select anything during the recording! Just right-click and delete the cells. So your macros can look like this 

Sub DeleteCellsLeft()
  Selection.Delete Shift:=xlToLeft

Sub DeleteCellsUp()
  Selection.Delete Shift:=xlUp
End Sub

ricw-nw say:

Hi Chad...

When you say 'all the files have been removed from my downloads' - do you mean the files have been deleted from your pc? If that is the case you will need to download them again... most download sites permit this.

A common error made with WMP is the incorrect belief that the music is stored there - it is just a player, and links to the original files which would usually be in the 'Music' folder. If those original files have been deleted or moved the 'restore deleted files' tool in WMP cannot help - it can only restore items that have been removed from the WMP library - not items that have been totally deleted.

If you still have the original files but they have been moved elsewhere, we can resolve this, so please update...

Thanks, Ric.

chadbrennan say:

thanks ric,

i guess il have to download them again is there anyway to prevent this from happening again though?

ricw-nw say:

My method is to choose a location in 'My Documents' for music downloads - the documents folder is not monitored by WMP, so you can check the files are named how you like them etc. before cutting and pasting them to the Music folder. There they will be picked up and included by WMP in its library although if this includes tracks you have previously deleted you may need to rebuild the WMP library. Get back if you need more detail or if you get stuck.

Cheers, R.

ps - you ask 'is there any way to prevent this happening again?' - the only way the files should have gone missing is if you deleted them yourself... If you do not recollect doing so there may be another underlying issue like the presence of a virus or malware, so just be aware of that.
