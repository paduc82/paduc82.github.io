ken sheridan say:

I need a group count of how many plans there are without the grouping of POs.

Here's an example using Northwind, which returns the number of instances of each status in the Order Details table:

SELECT[Status Name], COUNT([Order Details].[Status ID]) AS StatusCount
FROM [Order Details Status] LEFT JOIN [Order Details]
ON [Order Details Status].[Status ID] = [Order Details].[Status ID]
GROUP BY [Status Name];

The query returns the following result set:

Status Name    StatusCount
Allocated         6
Invoiced          50
No Stock         1
None              1
On Order        0
Shipped          0

Note the outer join to return zero counts where a status is not represented in Order Details.  You can base a subreport on such a query, or, rather inefficiently, have a set of unbound text boxes, each of which calls the DLookup function, with a criterion identifying the appropriate status value, e.g.

DLookup("StatusCount", "MyQuery", "[Status Name] = ""Invoiced""")

vijaykumar shetye say:

Dear JoscelynMilstone,

Select the cells in Column B, which are required to be conditionally formatted.

Go to Home - Conditional Formatting - New Rule - Use a formula to determine which cells to format -

Format values where this formula is true:

In Conditional Formatting, use the following formula for highlighting the cells Red (This Week)

=IF(AND(TODAY()>(B2-WEEKDAY(B2)),TODAY()<(B2+7-WEEKDAY(B2)),ISBLANK(C2)),1,0)

Format - Fill - Red

In Conditional Formatting, use the following formula for highlighting the cells Green (Next Week)

=IF(AND(TODAY()>(B2-WEEKDAY(B2)-7),TODAY()<(B2+7-WEEKDAY(B2)-7),ISBLANK(C2)),1,0)

Format - Fill - Green

I have considered Sunday as the first day of the week.

Ensure that the cells in column C are actually blank.

In case cells is column C, appear to be blank due to the result of a formula, then the above formulas will not consider the cells to be blank. In that case, you can change the below portion in both the formulas

from   ISBLANK(C2),

to    LEN(C2)=0

The formula can be shortened if required.

Do let me know if this was what you wanted to do, or if any changes are required.
