They don't appear to be able to recognize when a Metro App/Start Menu is open. These seem to default to the Intel HD 4000 graphics driver, which in turn is currently out of date with the latest Dell driver package. It appears that this has been an issue for the Graphics manufacturers (ATI and Nvidia) with the changeover from Win7 to Win8.

The way the drivers are handled for these cards is through an installation package provided by AMD/ATI, through Dell, which updates both the ATI card and the Intel card at the same time (E.g. the Intel drivers are a part of the ATI Installation). This installation package seems to ensure that the switchable graphics feature is preserved. If you look at your drivers page from the support section on Dell's site, you'll see this single package (For me, it is Version 8.982.6.0, A03).

I think that the problem is that this package from Dell has an old version of the Intel driver, which does not work with things like the Netflix App.

I seemed to be able to resolve this issue by finding newer Dell Intel drivers, and then a newer ATI Driver package directly from AMD (and NOT Dell).

NOTE: Ensure you have updated your BIOS to the most current version available from Dell

1. Download and Install the Latest Intel HD Graphics driver (Even though the 7520 isn't listed, it works): http://www.dell.com/support/drivers/us/en/19/driverdetails?driverid=6NJX6

Follow the prompts on the screen, install, restart. 

2. Download the AMD Mobility Radeon Driver Verification Tool, and run the application: http://support.amd.com/us/gpudownload/windows/Pages/radeonmob_win7-64.aspx#1

It is EXTREMELY important that you run this utility. It helps make sure you find the right driver for your laptop. It should prompt you to download and install the driver (13-4_mobility_win7_win8_64_dd_ccc_whql.exe). Ensure you put it somewhere you can find it, like your desktop.

3. Run the driver application the utility downloaded, and click uninstall when prompted to. Select Custom, and ensure every box is ticked on the next screen. Click through and complete the uninstall, restart the computer again.

4. When you have restarted your computer, run the driver application again, and install the drivers as normal. Restart.

...and that should do it! Hope this helps!
