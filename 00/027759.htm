jay freedman say:

I don't think it's possible to have multiple bibliographies that are still active as "citations" and "bibliographies". The only way I believe you can do this is to (a) prepare the single bibliography, (b) convert the bibliography to static text, and (c) cut and paste pieces of the static text to the separate sections.

When the cursor is in the bibliography, you can click the down arrow in the tag of the content control:

You'll find the "Convert bibliography to static text" command at the bottom of the menu. Once you use that command, the bibliography will no longer be able to update, so be sure all your citations and bibliographic data are complete and correct before you convert.

If you're using a numbered citation style such as IEEE and you need to change the numbering, you'll also find a "Convert citation to static text" command on the menu of the content control around each citation.
