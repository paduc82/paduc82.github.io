There are a number of things that can cause this.

Go to Tools > Design Checker and run it. It may find a design element overlapping the page and the scratch area or something else that is causing the problem.

It could be a corrupt image. Drag your images off the page into the scratch area one at a time and try to create your pdf. You can click the Edit > Undo to move it back after the test or just Ctrl+Z. If you find a defective image then delete it and reinsert a fresh copy.

Here are two threads that discuss various reasons why you cannot save as a pdf.
 
Reference: Publisher cannot save the file to PDF: http://answers.microsoft.com/en-us/office/forum/office_2007-office_other/publisher-cannot-save-the-file/31021a27-6183-4d81-ae21-83d5c646facd?page=1

If all else fails then a short term workaround is to download and install www.primopdf.com a good free pdf tool and use it to create your pdf instead of the built-in Publisher PDF tool. When you install it watch for the optional toolbars and such and opt out.

I am attempting to make a list that bolds one of the fields but only if there is something filled in in the adjacent cell.
jeeped say:
Let's say you want to have the text/number/date in column A bold if the cell in in the same row but in column B is not blank.
  1. Select all of column A by clicking the column header label then click the Home tab's Conditional Formatting, New Rule command.
  2. Select Use a formula to determine which cells to format from the Select a Rule Type: list and then paste this formula into the Format values were this formula is true: text box,
           =NOT(ISBLANK($B1))
  3. Click the Format button and select Bold from the Font tab's Font Style list. Click OK to save the format selection then OK to save the new Conditional Formatting rule.
The values in column A should now be in a bold font if there is a corresponding value in column B. If you have data labels in A1 (or somewhere at the top of your data) that you do not want to be bold, select only the cells that you want to receive the formatting rather than the entire column in step 1 and change $B1 in the formula to whatever cell is in the top row for column B (e.g. $B2 or $B3 perhaps).

luke moraga say:

Select the cell (I'll asume A2), then go to Format - Conditional Format. Change the first dropdown to "Formula is" then type:

=$B2<>""

(Where B is the column that we are checking if something is filled in)

Click the format button, and select bold. Ok out.

Copy this formatting to other cells as desired.
