    eos4answers say:

    Hello again.

    I am coming back to report that I have finally found a suitable solution for this particular problem and here it is:

    Since Microsoft was unable to help me I had no other option but to re-create the affected account.  However, there are a few steps that you might consider taking prior to disposing of the old account.  The following steps ensure that in the end you retain as many software settings as possible within your new account.

    1. Back up your old account by simply copying all files within the respective c:\Users subfolder.  Take heed of all special files such as NTUSER.DAT to be found at the root of your user's subfolder.  Do not forget to adjust your explorer's settings to be able to see hidden and system files.

    2. Log in using the broken account and run regedit or even better a more sophisticated tool such as TuneUp Registry Editor. Go to HKCU and export the entire Software key to a REG file.

    3. Delete your old account via System Properties > Advanced > User Profiles.  Be sure to log in as local Administrator before you do so, otherwise you will not be able to remove an active profile.

    4. Create a new user account giving it the name of the old one.

    5. Log in as local Administrator and go into c:\Users\<your_new_user>.  Copy all files you can see such as NTUSER.DAT to a temporary destination.

    6. Copy your back-up data from step 1 into the new user subfolder overwriting all new files.  I recommend using additional software such as Acronis True Image for example because Windows might not allow copying or overwriting certain files at this point.

    7. Take all files from step 5 and copy them back to your user’s folder overwriting the old ones whenever prompted.

    8. Log in as the new user and run TuneUp Registry Editor.  Use the REG file from step 2 to import it into your new clean HKCU branch.  Note: Windows regedit will most likely refuse merging your back-up REG file with the registry for some reason.  That is why I strongly recommend using different software for this particular purpose (I have successfully tested TuneUp Registry Editor as mentioned above).

    9. Once the merging is done log off and back in again to make sure your old software settings are applied properly.  Verify that everything (or at least the most part of it) works as expected.
