Try the steps given below to create a Gantt chart:
 
1.       On the Excel sheet, enter the data that you want to plot.
 
2.       Select the cells containing the data for your chart (for example, A1:C6)
 
3.       On the Insert menu, click Chart.
 
4.       In the Elements Gallery, click the Bar tab.
 
5.       Click the Stacked Bar thumbnail.
 
6.       In the chart, double-click the first data series (the blue portions of the bars), click Fill, and then click the Solid tab.
 
7.       On the Color pop-up menu, click No Fill, and then click OK.
 
8.       On the chart, click the legend, and then press DELETE.
 
9.       Double-click the vertical axis, click Scale, and then select the Categories in reverse order check box.

Open regedit from start. Go to HKEY_LOCAL_MACHINE -> Microsoft -> Windows -> Current version -> Policies -> System

jay freedman say:

There's a quick way that doesn't involve macros.

First, go to File > Options > Proofing > AutoCorrect Options and click the AutoFormat tab of the dialog (note: NOT AutoFormat As You Type). Clear all the check boxes in that tab except "Internet and network paths with hyperlinks". Click OK.

Next, go to File > Options > Quick Access Toolbar. Set the top left dropdown to All Commands. Select AutoFormat Now in the list, and click the Add button to copy the command to the right-hand list. Click OK.

Finally, with your document open, click the AutoFormat Now button. That will create all your hyperlinks.
