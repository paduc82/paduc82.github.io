It is quite common to learn that certain system files associated with the storage and distribution of updates have been corrupted. Delete the contents of the folder.
Restart your computer and make sure the behavior of your Windows Defender has improved.

cd / d "\ Program Files \ Windows Defender"
exe - signature update

Navigate to the individual services associated with Windows Defender.
Right-click on all services and open Properties.
Make sure they are all set to start automatically.

4- Manually upgrade Windows Defender
Like any other program, you can manually download the latest setting for Windows 10 and install it on your computer. Needless to say, you can also choose to manually upgrade Windows Defender.
