How do I change my skype name? - Microsoft Community

ar_12 say:

How do I change my skype name?

How do I change my skype name on a pre-existing account?

Transferred by Moderator from the Other Features: Skype Interviews message board to the Account Set-Up & Maintenance message board.

danida_u say:

Hi

It is not possible to change an account's Skype name. When Skype accounts were migrated to basic Microsoft accounts last October 2016, the option to create a custom Skype name is also no longer possible. The account creation process would prompt new users to create their accounts using either an email address or a phone number. 

As a workaround, you can change your display name or create a different account. Note that an account created using an email address will have a system-generated Skype Name that will show as live:whatevercomesbefore@.

Hope this answers your query!
