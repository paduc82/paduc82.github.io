This never was an option (as you couldn't turn it off) but a UI design.

In Outlook 2013, folders with unread items will not show in bold anymore but will only show the amount of unread items behind it.

Until such time as ALL browsers render the same HTML/CSS the same way, and ALL users use the same settings, there is no such thing as a WYSIWYG editor for web pages. The web is NOT print - WYSIWYG will work for print, where a sheet of paper is fixed in size, users cannot change text sizes, users cannot change fonts. In the world of the web, the author can only make suggestions as to the font used, the font size, and the size of the browser portal.

Changing your editor will not make any difference - if you want to use FrontPage extensions, you need a host that supports FrontPage extensions.  Microsoft abandoned the extensions in 2006 - four (4) years ago. Hosts that continue to supply them do so on old operating systems or support them themselves. The latest versions of Windows do NOT support extensions.  My host (webhost4life.com) has just dropped them, and other hosts are removing them, sometimes without warning.  It is time to move on, and that means learning server side technologies, such as asp.NET and PHP.
