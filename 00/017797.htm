 First of all -

You would need it in the FORMAT event of the section if you want it to happen in the Print Preview View or actual Printing.    If you want it to work in Report View, you need to also put it in the ON PAINT event of the section.  I've found this to be a bit of a pain but it won't work in Report View if you have it only on the Format event.  Nor will it work in Print Preview if you only have it in the On Paint event.

The Current event of the report? As it says in the Help file, "The Current event occurs when the focus moves to a record, making it the current record, or when the report is refreshed or requeried." That's not the event you want!
