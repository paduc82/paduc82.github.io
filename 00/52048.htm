On my installation, I needed to both take ownership and modify the permissions for every file and folder before I could delete them.

2. Right-click each file or folder and select properties -> Security tab
3. Click Advanced -> Owner tab
4. Click Edit -> Approve UAC prompt

6. This will return you to the Security tab -> select Edit to change permissions
7. Select your username and fill the checkbox to allow Full Control
8. OK -> OK

Now that you've taken ownership and have Full Control, you'll be authorized to delete the files.
