Any way to do this in Word?  Would this be easier in Excel?

pesach shelnitz say:

When you create the bookmark you need to highlight (select) the text that the bookmark will embrace. If you change the text after you create the bookmark, you must be careful not to delete the bookmark itself. To preserve the bookmark, never highlight the first or last character of the bookmarked text. If you need to delete them, use Backspace to delete the first character and Del to delete the last character after you make the other changes.

Besides the techniques for creating a REF field in the help topic, in my opinion, the easiest way to create a REF field is to type the name of the bookmark, highlight it, press Ctrl+F9, and then press F9. (It is not necessary to type REF).

You indicated that you will frequently change the bookmarked text and you want the REF fields to update automatically. The closest thing to automatic updating that I can suggest is to press Ctrl+A and then F9. (Ctrl+A selects the entire document, and F9 updates all fields in the selected text.) If you do this whenever you change the bookmarked text, your REF fields will always be updated.

Hope this helps,
