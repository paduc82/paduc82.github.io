Sub zap_narrations()
Dim osld As Slide

For Each osld In ActivePresentation.Slides

ben m. schorr -bschorr say:

It's possible but it's not a great idea. 

 

1. You can't guarantee how it's going to look on the other side. If the other person is reading their e-mail on a device that doesn't support HTML or that renders it differently because of the small screen, it could look pretty bad (or not at all) on the other side.

2. If you're not careful, it needlessly bloats the size of the e-mail. If you manage to get a low-res, highly compressed, image that is only a K or two that's one thing, but you could end up with an image file that's 100K or more - and if you put that on every e-mail you're going to fill up people's e-mail boxes pretty fast.

 

I get the desire to do a "handwritten" signature. It would look cool. But as a practical matter it's less than ideal.

 

Just my $.02.

Put the items that you want to appear only on the last page inside an If...then...Else field such as

{ IF { PAGE } = { NUMPAGES } "[INFORMATION FOR LAST PAGE]" "" }

You must use CTRL+F9 to insert each pair of the field delimiters { } and use ALT+F9 to toggle off the display of the field code.
