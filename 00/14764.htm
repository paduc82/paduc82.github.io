Install Font During Install of Access 2010 Runtime Front End - Microsoft Community

cevinmoses say:

Install Font During Install of Access 2010 Runtime Front End

I am using the Package Solution Wizard to install front ends on computers that don't have Access installed.  How can I install a particular barcode font as part of the install?  The package will install on both WinXP and Win7 machines.
max meng say:

My suggestion is to create a batch script to launch the setup for both the Access Solution and the Font.

e.g.

Copy and save the following script as AddFont.cmd.

##########################################################################

@ECHO OFF
TITLE Adding Fonts..

IF NOT "%*"=="" SET SRC=%*
ECHO.
ECHO Adding Fonts..
ECHO.
FOR /F %%i in ('dir /b "%SRC%*.*tf"') DO CALL :FONT %%i
ECHO.
ECHO Install Solution..
ECHO.
setup.exe
ECHO.
ECHO Done!
PAUSE
EXIT

:FONT
ECHO.
SET FFILE=%~n1%~x1
SET FNAME=%~n1
SET FNAME=%FNAME:-= %
IF "%~x1"==".otf" SET FTYPE=(OpenType)
IF "%~x1"==".ttf" SET FTYPE=(TrueType)

ECHO FILE=%FFILE%
ECHO NAME=%FNAME%
ECHO TYPE=%FTYPE%

COPY /Y "%SRC%%~n1%~x1" "%SystemRoot%\Fonts\"
reg add "HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Fonts" /v "%FNAME% %FTYPE%" /t REG_SZ /d "%FFILE%" /f
GOTO :EOF

##########################################################################

When package solutions, add fonts file and the AddFont.cmd in the same folder with solutions.

Let the user run the AddFont.cmd instead of the setup.exe.

 

NOTE: To use the fonts installed by the batch script, a system restart is needed.

We can also add the following parameter to reboot automatically.

======================================================================

shutdown -r -f -t 10 -c "Reboot required for Fonts installation"

======================================================================

Top and bottom page margins are honored unless there is text in the header or footer that takes too much room. This would shrink the text area toward the center of the page. If your problem is in the footer, go there through View > Header and Footer in the main menu bar. Then:

  • Click on the ¬∂ icon in the Standard toolbar to display nonprinting characters. (If you don't see the Standard toolbar, enable it through the main menu bar's View > Toolbars > Standard.)
  • Delete any extraneous material (for instance, a bunch of carriage returns, displayed as blue pilcrows, ¬∂)
  • If you have just one paragraph, make sure it's Space Before and Space After are set to zero. (To do this, use Format > Paragraph.)
  • It is also possible to have a large character, even if the paragraph is empty. (Fix this through the Ribbon's Home toolbar or select Format > Font.)
  • Lots of other stuff can also take up space and push the margin, like a text box.
