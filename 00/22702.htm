scottgem say:
The problem here is that when the first rows of your spreadsheet contain data that looks like a number, Access tries to import it as a numeric value. And since the datatype in your table is Text, this confuses Access.

The better way to do this is to link to your spreadsheet and use an Append query to add the data to your table. In this way you can convert the data in your Excel sheet to fit the datatype in your Access table. So, for example, the column in your query might be:

Code: CStr([columnname])

This insures that its importing the numeric values as Text data.
