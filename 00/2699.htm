jim gordon mvp say:
To amplify what Phillip said, everything depends on whether or not you ever saved the document in Microsoft Word format. If you did, then follow Phillip's instructions.

If the document was saved only in Rich Text Format (.rtf), then that's the only format that you will have. However, Microsoft Word can open Rich Text Format (.rtf) files, as can TextEdit. If you can open your RTF file in either program, you can use FIle > Save As and choose Microsoft Word (.docx) format as the format for saving and restore the file that way.

If your document was saved as a Word document, and you simply changed the file extension from (.docx) to (.rtf) manually in Finder, then you can restore the document by restoring its proper file extension in Finder by changing it back to (.docx).

Changing file extensions in Finder does not change the structure of the file. If you save a document in Microsoft Word format (.docx), it is still in Word format no matter what extension you tack onto the end. Putting the wrong extension onto a file may make it unreadable by the program that would normally be used to open it.
