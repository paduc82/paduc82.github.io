admin privileges - Microsoft Community

rubenramirez lozano say:

admin privileges

Hi,

Even since I bought my PC, I noticed that whenever I needed to make a change in setting I received a message saying I was not the administrator. In order to run CDM to try to make a change, it tell me that the operation requires elevation (run as administrator) Why is that?

a. user say:

That's a great question. Since long before anyone heard of Microsoft, the computer industry has been concerned about the security and privacy of information. The industry created the 'principal of least privileges' which states that every computer user should have only the amount of permissions that they require for their use of the computer, and no more.

Administrators have more permissions than Users. Administrators have the ability to make changes that affect the entire computer, like installing software. Users only have the ability to make changes that affect their own user accounts, like using software.

If someone was to successfully break in to a computer, it would be much more serious if they broke in to an Administrators account than if they broke in to a Users account, because the attacker can do much more damage from an Administrators account.

That's why we are supposed to sign in to a Users account for day to day computing. We only sign in to an Administrators account if we need to do something - like installing software - that requires an Administrators' elevated permissions.
