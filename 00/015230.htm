Are there any settings in powerpoint to ensure that they export without being Indexed?

john sr wilson say:
Don't think you can change the export mode but actually when I either export slides as PNG or Save As PNG I get RGB mode. Maybe it depends on the content of the PPTX?

steve rindsberg say:

This may vary across versions but some time back I noticed this same thing. 

As John's suggested, it depends on the content of the slide.  If nothing on the slide *demands* 24-bit color, PPT will export PNGs at lower color resolution.

One simple solution would be to go ahead and export the images then use a program that can make batch conversions to convert to 24-bit color.  IrfanView (www.irfanview.com) is free and does a nice job of this.  As I recall, it can even do this type of conversion from the command line or batch file in case this is something you need to do often.

Alternatively, include content on the slide that forces PPT to export 24-bit color.  JPGs are, by definition, 24-bit, so putting a tiny (possibly even 1 pixel) JPG somewhere on each slide (or possibly on each master) should do it.  At one time, the JPG didn't even need to be on the slide; it could sit off-slide, but they've "fixed" that.
