As to saving as PPTX, you have to do a weird little dance

To save a PowerPoint 2000 file as a PowerPoint 2007 file, follow these steps:

  1. Locate the file in Windows Explorer.
  2. Right-click the file, and then click Save As.
  3. In the Save As dialog box, click one of the following PowerPoint 2007 file formats in the Save as type box, and then click Save:
    • Microsoft Office PowerPoint 2007 Presentation (*.pptx)
    • Microsoft Office PowerPoint 2007 Macro-Enabled Presentation (*.pptm)
  4. When the save is complete, click OK.

Note When you Save As and specify the PowerPoint 2007 format, you can then open the document in PowerPoint 2010 too.

That's from:

http://support.microsoft.com/kb/924074

elbert_liu say:

DDE in mac os Excel

Hi may I get some help on how to enable DDE on my Excel macOS version. Many thanks!
andreas killer say:
That is not possible, because of the Apple Sandboxing feature.

The Sandboxing feature of a Mac completely encapsulates a task from the environment, he has no access to the system or other tasks.

DDE means Excel creates another Excel tasks and both tasks exchanges data.
This is no longer possible with the security concept of your Mac.

https://developer.apple.com/app-sandboxing/

Disclaimer: This is a non-Microsoft website. The page appears to be providing accurate, safe information. Watch out for ads on the site that may advertise products frequently classified as a PUP (Potentially Unwanted Products). Thoroughly research any product advertised on the site before you decide to download and install it.
