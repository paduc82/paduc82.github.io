To fix, first write down your password and username (usually the email address used to purchase office subscription). (Depends on whether you bought directly from Microsoft, or you bought through the Apple Store).

Now:

  1. Quit all Office Applications.
  2. Go to Utilities.
  3. Open Keychain.
  4. Click Passwords
  5. Look for any passwords with a modern date, and refers to Microsoft.
    (It might also refer to Word, Excel, PowerPoint, etc.)
  6. Delete those references.
  7. Now open any Office app and sign in and save in keychain.
    (be sure to use correct username and password or else you'll have to retry.)
  8. If you have entered everything right, you should be able to use applications without having to continually trying to sign in.

hansv mvp say:

Here is a modified version:

 

Function Base64To16(Base64 As String) As String
  Dim Base2 As String
  Dim i As Long
  If Len(Base64) Mod 4 > 0 Then
    Base64To16 = CVErr(xlErrValue)
    Exit Function
  End If
  For i = 1 To Len(Base64)
    Select Case Mid(Base64, i, 1)
      Case "A"
        Base2 = Base2 & "000000"
      Case "B"
        Base2 = Base2 & "000001"
      Case "C"
        Base2 = Base2 & "000010"
      Case "D"
        Base2 = Base2 & "000011"
      Case "E"
        Base2 = Base2 & "000100"
      Case "F"
        Base2 = Base2 & "000101"
      Case "G"
        Base2 = Base2 & "000110"
      Case "H"
        Base2 = Base2 & "000111"
      Case "I"
        Base2 = Base2 & "001000"
      Case "J"
        Base2 = Base2 & "001001"
      Case "K"
        Base2 = Base2 & "001010"
      Case "L"
        Base2 = Base2 & "001011"
      Case "M"
        Base2 = Base2 & "001100"
      Case "N"
        Base2 = Base2 & "001101"
      Case "O"
        Base2 = Base2 & "001110"
