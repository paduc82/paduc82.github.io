bradannapolis say:

Hide rows based on greater than/less than in another cell

Greetings Everyone,

I am hoping to get some help on changing the code in a worksheet that will hide rows 17-20 if the value in cell D16 is less than 328 and to show these same rows if the value is greater than or equal to 328. Any assistance would be greatly appreciated.

davem121 say:
Hi Brad, you cannot do that in a formula, your would need to use a VBA macro placed on that sheet . ..

Private Sub Worksheet_Change(ByVal Target As Range)
If Range("D16").Value < 328 Then
Rows("17:20").EntireRow.Hidden = True
ElseIf Range("D16").Value > 327 Then
Rows("17:20").EntireRow.Hidden = False
End If
End Sub
