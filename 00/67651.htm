waarhorse777 say:

Just stumbled across this and thought I'd chime in.  Looking to do much of the same thing but here is what I've found:

SurfacePro and Windows 8+ use UEFI vs. BIOS so they will not boot the same way.  SurfacePro used the GPT file system (Global Unique Partitioning Tables) vs MBR (Master Boot Record) like older style windos 7 and less.  So far, Ultimate boot hasn't created a EFI bootable version of their software (i've seen mention it's because many tools work in  DOS which isn't available with UEFI booting).

You can boot from CD or USB if you use a bootable iso like Ubuntu 12.0 or higher.  It needs to be EFI bootable though, without that it's just going to work with machines with BIOS.

A good tool to install ISO's on usb is Rufus as it lets you choose partition type (GPT vs MBR).  Won't let you force a non-bootable iso into a GPT however.

If you or anyone finds a EFI bootable version of UBCD, let me know!  It would be great.  World seems to be going UEFI vs BIOS.  My take: don't complain about it, but find a way to overcome and be useful in this new market.  It's the whiners that will be paying me to fix their computers when they give up creating a new tool kit. Don't be one of them.

**Feel free to correct me if I'm missing anything.  The more we share the better we become.**

ANY bootable usb flash drive used on the surface pro's MUST be Fat32! If the usb flash is UEFI enabled it should boot, if it doesn't, then it's being blocked by secure boot, another protection in the surface bios. Both the secure boot and UEFI option can be disabled if you use the advanced startup option in the troubleshooting section of the pc settings, and then restart your surface. Hi All,

Thanks,

jomel_g say:

Hi

Skype numbers are non-transferrable from one Skype account to another. Skype Numbers are associated with the Skype account used to make the original purchase. As a suggestion if you'd like to use the same number on a new account, please cancel the recurring payments for the Skype Number on your personal account. Then after the expiration date, you will need to wait for 90 days as Skype will reserve the Skype Number for your account during the said time frame. However, we cannot guarantee that the number will only be available for you since it will be added back to the Skype Number pool where other users purchase the service.

If you really need to have a Skype Number on your business account as soon as possible, you can cancel the recurring charges for your Skype Number on your personal account, then purchase a new one from the Skype Number pool on your business account.

I hope this helps.
