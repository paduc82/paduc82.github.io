correct procedure to insert intellicad object into docx - Microsoft Community

richardshoesmith say:

correct procedure to insert intellicad object into docx

I have created intellicad drawings and now want to insert a "snapshot" of each of them into a word docx. What is the step by step procedure for that, please.
jay freedman say:

Hi Richard,

Since you already have the drawings you want to put into a Word document, you should use the "Create From File" tab of the Insert Object dialog, not the "Create New" tab. In theory, "Create New" would embed an OLE object connected to IntelliCAD in which you can make a new drawing, but that isn't what you want to do. The "Create From File" tab just asks you to type the path and file name (or browse for it).

However, that inserts an icon rather than a display of the contents of the file -- again not what you want. If you click the Picture button on the Insert tab, and open the dropdown next to the "File name" box, does the file type of IntelliCAD files appear in that list? If not, you can try selecting "All Files (*.*)" in the dropdown, but then the import probably won't work because of the lack of an import filter (converter) for that file type.

If that also doesn't work, you'll have to ask someone who does support for IntelliCAD. The import filter for that object type is installed on your computer as part of the IntelliCAD package -- if there is one at all. If it doesn't exist, or it doesn't work with Word, or if it needs some setup/configuration to make it work, that will be a matter to take up with the IntelliCAD Consortium.

One other suggestion: Use IrfanView or other graphics program to open the IntelliCAD drawing and save it directly (NOT through PDF) as a TIF file. When the Save dialog opens, there will be a small box of TIF Save Options to the right of it -- choose "No compression" to retain the maximum resolution (it will probably create an enormous file). Word will be able to import that file, using Insert > Picture, NOT Object, but it will probably apply compression to its internal copy and will lose some resolution. Whether that's acceptable is up to you, but it's the best Word can do.
