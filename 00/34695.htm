the retopian say:

Decrypting EFS files

I am unable to open EFS files are maintained on external hard-drive.

recognize .uea program.

What should I do?  I have the userkey.psw file and a recent backup from my stolen laptop.  
divya r say:

Hi the retopian,

Do you have a backup of your encryption ceritificate?


The first time you encrypt a folder or file, you should back up your encryption certificate. If your certificate and key are lost or damaged and you do not have a backup, you won't be able to use the files that you have encrypted.


The user.psw file that you are referring to is the file to create a password reset disk.

1)  When you create this password reset "disk", all you are doing is creating a file named userkey.psw in the root of that media.

2) You can copy that userkey.psw file to the root of any other removable media and your computer will successfully use that file if the media can be

detected by the Operating System.

3) If userkey.psw is in a folder, it cannot be used to reset the password.

4) If at any time you think that your userkey.psw is in the wrong hands, you can create a new password reset "disk", which voids any previous userkey.psw files.

5) If you step away from your computer for a minute, someone else can create a new password reset "disk", thus locking out the most recent userkey.psw

you created.

 

Follow the steps mentioned in this link to recover the encrypted files.

http://windows.microsoft.com/en-us/windows7/Create-a-recovery-certificate-for-encrypted-files

http://windows.microsoft.com/en-us/windows7/Recover-encrypted-files-or-folders


So, if you have the certificate and key, you may be able to decrypt the files in Windows 7 Operating system.

Follow the below mentioned step to do the same:

To decrypt a folder or file

1.    Right-click the folder or file you want to decrypt, and then click Properties.

2.    Click the General tab, and then click Advanced.

3.    Clear the Encrypt contents to secure data check box, click OK, and then click OK again.

Refer the following link for more information:


Encrypt or decrypt a folder or file

http://windows.microsoft.com/en-US/windows7/Encrypt-or-decrypt-a-folder-or-file
