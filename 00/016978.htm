a. user say:

I think Daniel's answer + your \r switch for resets nails it, but just for clarification, when you are using the built-in Heading styles Heading 1 etc. and you use Heading 1 as your "chapter" heading, you can use 

{ SEQ EquationChapter \s 1 }

to restart the EquationChapter sequence at each new Heading 1 paragraph, regardless of whether Word's automatic numbering is being used to number the chapters or not. 

If you are actually using Word's automatic numbering for the chapters, the way Word itself does this kind of thing is to use

{ STYLEREF 1 \s }

for the chapter number instead of a SEQ field. In this case, the "1" is a shorthand for "Heading 1". which makes the document slightly more portable across different language versions of Word.

In fact when you have automated chapter numbering with the chapter-caption numbering option, that's what Word inserts, when you use References->Insert Caption. (Have a look at the Numbering: Format.... options and take note of the option to exclude the Caption label. Also worth knowing that the character it inserts when you specify a - separator is a special hyphen of some kind (non-breaking or some such, I forget). Captions in that format will be picked up by the ToC field code inserted using the References->Insert Table of Figures option (which AFAICR uses the \c option to specify the SEQ sequence name you want to list - e.g. if you use SEQ EquationNumber, you will need \c EquationNumber. 

But since you want ( ) around the number, you would probably be better off defining an autotext/autocorrect that inserted the entire caption. If that's all you want in the table of Equations as well, you can use the standard Insert Table of Figures ToC field. If you want something else, you would probably still need to follow Daniel's suggestion of using TC fields to insert the right thing in the ToC.

Hello Jesc,

Thank you for providing us the opportunity to assist you.

Once yammer is activated, the data cannot be erased for the whole network. 

In yammer basic you don't have administrator rights hence you cannot delete the content or the users on the network, However in yammer enterprise network you can manually delete the groups, users and the verified admin can also delete the posts.
If you upgrade your yammer network to enterprise network then you may delete the content manually.
