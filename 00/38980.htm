joeo sullivan say:

I was having a similar problem. Whenever I move or do anything to a music file its attributes change from 'content indexed' to 'not content indexed'.

Have a look at your music files in file explorer and see if they are being indexed by content. Go to a song >right click > properties > details, and scroll down to attributes and see if it's A or AI. (AI being not content indexed)

Or search * in the search box of your Music folder and see if song and album titles are showing in the results.

If some songs have an attribute of  AI  then just run the following:

attrib -I /s /d "D:\Music\*"

in the command prompt, with the correct music folder location eg "C:\onedrive\music\*".

It lets Cortana access album and song names etc, that are embedded in the mp3s.. It's worth running the command anyway as it only takes a few seconds..

That symptom could be a sign that you are looking at the second half of an extended display, with your Start Screen only rendered on the primary half (e.g. on a phantom monitor or on a second monitor if you hooked one up).  To test that idea you could press Win-P, CursorUp, Enter.  E.g. doing that would switch the Display Project dialog from Extend to Duplicate (and cause a screen refresh doing it.)

However, it could also be a sign that you don't have an explorer.exe running and you are just looking at your normal Desktop color without its wallpaper.  In that case you would press Ctrl-Alt-Delete to get into the Task Manager and use it to try to start explorer.exe (e.g. with its File New task... command).

Of course you could also have a composite of the two symptoms too with both the Start Screen and Task Manager only showing on the phantom monitor.  Then you would have to imagine seeing that case too and work around it using keyboard commands (which I have documented elsewhere for others with this symptom when it is described as a blank black screen.)
