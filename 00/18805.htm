Description is a reserved word in Access so its not recommended to be used for Object names. Since its a reserved word you need to wrap it in brackets [].

When comparing values, you need to use delimiters specific to the data type. Number datatypes do not require a delimiter. Text requires quotes. Dates require the octothorpe (#).
