saltgrass say:

An OS has to be installed as UEFI for it to work that way.  To install as UEFI you need to select the UEFI version of the install media, which must be available prior to boot.

During the install, a UEFI install will add four partitions and a Legacy install will only install two.  The correct install media will set a drive up as necessary if it has been cleaned and contains no partitions.  Configuring a drive as GPT prior to that time might put the MSR partition in the wrong location.

There are ways to convert back and forth and even Microsoft has the utility MBR2GPT.exe for that purpose.

A UEFI install has a EFI System partition, which is easy to spot in Disk Management.

In case you do not know, if you are installing, your can use Shift+F10 on the second Install window to open a command prompt and run Diskpart to clean the drive, which of course wipes out any prior data.  If you have a test drive you can play if you want, just be careful with any commands if another drive is attached.
