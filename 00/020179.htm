I assume that the variable strPW has been declared elsewhere in your code and assigned a value, otherwise it doesn't serve any purpose here.

As far as I can tell there is no need to unprotect/protect the BONA sheet, since you don't change any of its cells in the code.

The combined code could look like this:

 

Private Sub Workbook_BeforePrint(Cancel As Boolean)
    'Each time the Print button is clicked:
    Dim wsDst As Worksheet, wsSrc As Worksheet
    Dim rngFound As Range
    Dim lnRow As Long
    Set wsSrc = ActiveSheet
    Select Case wsSrc.Name
        Case "LC"
            Set wsDst = Worksheets("REG")
            wsSrc.Unprotect Password:=strPW
            Application.EnableEvents = False  'Disables events
            wsSrc.PrintOut Copies:=2          'Prints 2 copies of the worksheet
            Application.EnableEvents = True   'Re-enables events
            wsSrc.Range("F9").Value = wsSrc.Range("F9").Value + 1
            wsSrc.Protect Password:=strPW
            If wsSrc.Range("F7").Value = "" Then Exit Sub
