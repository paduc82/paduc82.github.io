srimadhwa b say:

Hi Paitin,

Encryption turns the data sent over the network to random characters that are impossible to decipher without a key, which both the computer and router have. Without the encryption key, any data intercepted from secured network transmissions is useless.

Encryption is the process of securing the contents of a file or message by encoding it. An encrypted message looks like gibberish, unless you can decode it using a key to translate those random characters back into real text or data.

You may contact the owner of the PDF file to get the password as the file is encrypted. Once you have the password you can open the file. If you do not have the password for the encrypted file then you cannot access the file unless you decrypt the file.

paitin say: Thanx  for d enlightenment but u actually didn't answer my question. I wanted to simply know if u could extract with command prompt and how to go about it.

srimadhwa b say:

Hi Paitin,

Thank you for the reply.

Follow these steps and check if this helps.

-          Type "cd x:\" into the command prompt, but replace "x" with the letter assigned to the local hard drive where you downloaded the "unzip" file. Press the "Enter" key.

 

-          Type "unzip [file_location]\[filename.zip]" into the command prompt, replacing "[file_location]" with the full directory path and "[filename.zip]" with the name of the ZIP file you want to extract. Press the "Enter" key. For example, if you want to unzip the file "sample.zip" that is located in the directory "c:\temp", you would type "unzip c:\temp\sample.zip" into the command prompt.

 

-          Close the command prompt window and launch the Windows Explorer.

-          Browse to the directory containing the ZIP file and locate the file folder with the same name as the unzipped file. Double-click the folder to verify that the files contained in the ZIP file have been extracted.

MacBook Air SuperDrive - Microsoft Community

hilarioitriago say:

MacBook Air SuperDrive

Necesito saber como instalar MacBook Air SuperDrive

andre da costa say:
La MacBook Air SuperDrive tiene un firmware único que impide que Windows la reconozca como una unidad USB óptica plug and play. Afortunadamente, Apple proporciona los controladores para ejecutar Windows en una Mac, como parte de su software BootCamp.

Si descarga la última versión de BootCamp del sitio de soporte de Apple, tendrá un archivo zip en el que la ruta bootcamp \ drivers \ Apple contiene el archivo "AppleODDInstaller64.exe".

Simplemente ejecute este archivo para instalar el controlador de disco óptico de Apple para Windows de 64 bits, ¡y listo! A continuación, puede utilizar el MacBook Air SuperDrive con su PC con Windows.
