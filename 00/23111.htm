siddyboy26 say:

output filename based on text box

Hi all

The form is called [report wizzard form] and the text box is called [current month start date].

Not sure how to go about it and cant find any threads with the same problem.

Hopefully it will be simple for someone to show me how.

Thanks heaps.

hansv mvp say:

Create a command button on your form.

Set its Caption property to an appropriate text, for example Export Report.

Set its Name property (in the Other tab of the Propety Sheet) to cmdExport.

Set its On Click property (in the Event tab) to [Event Procedure].

Click the builder dots ... to the right of the dropdown arrow in the On Click property.

Make the code look like this:

 

Private Sub cmdExport_Click()
    DoCmd.OutputTo acOutputReport, "NameOfReport", acFormatPDF, _
        "C:\Export\" & Me.[current month start date] & ".pdf"
End Sub

 

Replace NameOfReport with the name of the report, and modify the path as needed.

Switch back to Access.

hansv mvp say:

The reason is probably that you omitted the backslash after the path, and also that the default date format contains slashes /. These are not allowed in a file name. Try specifying a different format, such as yyyymmdd (which makes it very easy to sort documents):

 

    DoCmd.OutputTo acOutputReport, "report16", acFormatPDF, _
         "C:\Users\Office\Documents\" & Format(Me.[current month end date], "yyyymmdd") & ".pdf"
