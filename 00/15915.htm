                    Is there a macro to first link the headers and footers to previous sections and THEN go back and unlink them.  We get documents from time to time with section breaks, etc., that end up with a blank page at the end.  When removing the section break, it
 removes the footer/header from all the previous sections.  Since these are pleadings (the graphic lines and numbers down the side) it becomes a problem.
                



            



            



            
        

             doug robbins - mvp office apps & services (word)  say:

    
    

            
            
            
            
Is there a macro to first link the headers and footers to previous sections and THEN go back and unlink them.  We get documents from time to time with section breaks, etc., that end up with a blank page at the end.  When removing the section break,
 it removes the footer/header from all the previous sections.  Since these are pleadings (the graphic lines and numbers down the side) it becomes a problem.





The following code should do that:


 


Dim i As Long, j As Long

With ActiveDocument

    If .Sections.Count > 1 Then

        For i = 2 To .Sections.Count

            With .Sections(i)

                For j = 1 To .Headers.Count

                    .Headers(j).LinkToPrevious = True

                Next j

                For j = 1 To .Footers.Count

                    .Footers(j).LinkToPrevious = True

                Next j

            End With

        Next i

        For i = 2 To .Sections.Count

            With .Sections(i)

                For j = 1 To .Headers.Count

                    .Headers(j).LinkToPrevious = False

                Next j

                For j = 1 To .Footers.Count

                    .Footers(j).LinkToPrevious = False

                Next j

            End With

        Next i

    End If

End With




 


Alternatively, you could use the following to change the SectionStart attribute of the final section in the document to Continuous:


 


With ActiveDocument

    If .Sections.Count > 1 Then

        .Sections.Last.PageSetup.SectionStart = wdSectionContinuous

    End If

End With
