macropod say:

You can't easily import/export sources from bibliography text in Rich Text Format. The following four macros should allow you to achieve whatever you're after - provided you still have the sources stored in Word.

The first two macros allow you to backup and restore all the sources stored in Word. Add these two the macros to a new Word document that you'll dedicate to this usage:

Sub SourcesBackup()
Dim oSrc As Source, StrSrc As String
With ActiveDocument
  .Range.Text = vbNullString
  For Each oSrc In Application.Bibliography.Sources
    StrSrc = StrSrc & vbCr & oSrc.XML
  Next
  .Range.InsertAfter StrSrc
  .Paragraphs.First.Range.Delete
End With
End Sub

Sub SourcesRestore()
Dim i As Long
'In case the source is already present
On Error Resume Next
With ActiveDocument
  For i = 0 To UBound(Split(.Range.Text, vbCr))
    Application.Bibliography.Sources.Add Split(.Range.Text, vbCr)(i)
  Next
End With
End Sub

Another way of making backups would be to periodically make copies of Word's Sources.xml file, usually found in:

C:\Users\%Username%\AppData\Roaming\Microsoft\Bibliography\Sources.xml

A limitation of that approach, though, is that if you delete a source, then decide you want to restore it, replacing the Sources.xml file will also wipe out any sources you've added since then.

The next two macros automate the transfer of sources in the active document's source list to Word's master list and, respectively to add all of Word's master sources to the active document's source list.

Sub SourcesImport()
Dim oSrc As Source
'In case the source is already present
On Error Resume Next
With ActiveDocument
  For Each oSrc In .Bibliography.Sources
    Application.Bibliography.Sources.Add oSrc.XML
  Next
End With
End Sub

Sub SourcesExport()
Dim oSrc As Source
'In case the source is already present
On Error Resume Next
With ActiveDocument
  For Each oSrc In Application.Bibliography.Sources
    .Bibliography.Sources.Add oSrc.XML
  Next
End With
End Sub

Do note that documents in the .doc format do not support Word's reference sources, citations or bibliographies.

"pst file in SkyDrive" - Microsoft Community

bveagle7 say:

"pst file in SkyDrive"

Can I put my Outlook 2010 pst data file in SkyDrive and use it as my active Outlook pst file?
roady [mvp] say:

No, while you can place a copy there so you'll create an on-line copy, it is not recommended to use it as your active pst-file.

Due to the active connection and lock Outlook has on the file, SkyDrive will continuously try to upload the file and thus use a lot of network bandwidth. It is very likely that this will end up in an incomplete or otherwise corrupted copy which will sync to another device and/or back to you computer and result in a complete loss of the pst-file.

Depending on the size of your pst-file and your network limitation set by your ISP, on-line storage (SkyDrive, DropBox or something similar) might not be the solution at all. Instead, consider using a mail host which backups the mail for you such as Office 365.

palcouk say:
