hansv mvp say:

There is no built-in keyboard shortcut for that, but you could create a macro for this purpose and assign a custom shortcut to it, or add the command to move to the top to the already existing macro.

 

The instruction to move to the top of the column containing the active cell is

 

    Cells(1, ActiveCell.Column).Select

 

or if you have a range variable rngCell that refers to a cell, you can use something like this:

 

    Cells(1, rngCell.Column).Copy

 

to copy the cell at the top of the column to the clipboard without actually selecting it.
