graham mandeno say: Hello Emanuel

You could omit it from the list entirely by using a SELECT statement for your combo box's RowSource, and including a WHERE clause like this:

WHERE [Fieldname] <> "Item to omit"

Or do you mean that you want it to appear in the list, but the user is prohibited from selecting it?

If so, you could use the BeforeUpdate event of the combo box:

If Me.[ComboBoxName] = "Item to omit" Then
    MsgBox "You may not select that item"
    Me.[ComboBoxName].Undo
    Cancel = True
End If
