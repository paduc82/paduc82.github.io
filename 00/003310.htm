hansv mvp say:

One option is to make sure that the Excel window isn't maximized, and to enlarge the window to cover both monitors.

Also make sure that the workbook windows aren't maximized within Excel; you can then drag one workbook to the other monitor (still within the application window).

The advantage of this method is that both workbooks are open within the same instance of Excel, so while copying/pasting between them you benefit from the special features of Excel's internal clipboard.

The disadvantages are that the ribbon is displayed only on one monitor, and that it won't look good if the monitors have different resolutions.

 

Another option is to start two different instances of Excel (for example click your shortcut to Excel, then Shift+click it to start a second instance.)

The advantage is that you'll have two completely independent application windows, each with its own ribbon.

The disadvantage is that the each instance sees the other as a different application. When you copy/paste from one to the other, you lose the special options that are available within Excel.
