How do I disable Windows Explorer suggestions when i type in a file - Microsoft Community

brandonbyrnes say:

How do I disable Windows Explorer suggestions when i type in a file name in Windows 7 Home Premium?

How do I disable Windows Explorer suggestions when I type in a file name, in Windows 7 Home Premium? I know I can't access Local Group Policy Editor in this version of Windows so how do I disable the suggestions?

the larch say:

Are you comfortable with making changes in Windows Registry, that place where we get warned that we could accidentally ruin the computer with one false move?  If so, the fix isn't hard.  Anyway, here are my notes on the topic:

When you enter search terms into a Windows 7 Search box, they are stored in the Windows Registry at HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\WordWheelQuery and appear as suggestions for later searches.  To remove the stored terms, delete that entire part of the Registry.  (The search terms are stored in hexadecimal code and so aren't recognizable as ordinary text here.)  To prevent the storage of search terms, go to HKEY_CURRENT_USER\Software\Policies\Microsoft\Windows\Explorer, rightclick on Explorer, mouse to New and click DWORD (32-bit) Value, type DisableSearchBoxSuggestions, and press Enter.  Doubleclick DisableSearchBoxSuggestion, type 1 for its value, and click the OK button.
