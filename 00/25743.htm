Skype should start signed in even when the "Start Skype in background" option is selected. However, this may happen with some delay. Testing this on my Windows 8.1 laptop showed that Skype signed in with a delay of about 20 sec. after other applications and background processes were finally loaded. It may take even longer time if you have many applications/processes set to start when Windows starts.

Note also that when the "Start Skype in background" option is selected, Skype will start in background (i.e. without visible window) even when you start Skype manually by clicking on the desktop shortcut.
