graham mayor say:
The issue throughout this thread relates to the port settings used by SMTP (the outgoing mail server).

The default setting for this port is 25, which is reserved for the SMTP port of the home service provider. Other service provider SMTP servers are blocked. Thus to enable Outlook to 'travel' you must set the alternative port address for the SMTP server (often 587, but not exclusively so). Check the settings provided by the e-mail service provider.

When an alternative port is used, you will also have to check the option for authenticating the outgoing server, and provide the requisite log-in details.

The server may also require an encrypted connection -again check the settings provided by the e-mail service provider.

that is very similar and as you have only one value very simple, set Dest to C6 in the destination file first, then copy the value, then set Dest to the next cell below.

Sub Test()
  Dim Ws As Worksheet
  Dim Source As Range, Dest As Range
 
  'Set the first cell
  Set Dest = ThisWorkbook.ActiveSheet.Range("C6")
 
  'Visit each sheet
  For Each Ws In Workbooks("Find Last Book.xlsx").Worksheets
