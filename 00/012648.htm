It does not appear there's an option to remove a Skype username from the login screen using Skype for Windows 10. Though, it can be done on Skype for Windows Desktop or the classic version. Here's how:

  1. Quit Skype.
  2. Double-click the My Computer
  3. Go to Tools > (Tools) Folder Options (Folder Options).

  5. Press the Windows  and R keys on your keyboard at the same time.
  6. Type "%appdata%\skype" into the Run window and click OK. 
  7. Delete the folder with the Skype username you want to remove from the login screen.

The problems I see with your code are:

  1. In Office 365, instead of “GetObject” you have to use the “CreateObject” method.
  2. Instead of a Word document, what I do is to use a template (ie a *.dotx) and assign bookmarks which I use to populate the MS Access table data.
  3.  You can get the data directly from a table, but for complicated merges, as Scott stated, it’s easier to use a dedicated Access report for such a merge.
