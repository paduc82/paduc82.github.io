jay freedman say:

I have fixed the problem in the AutoCorrect.dotm template and posted the new version of AutoCorrect2007.zip at https://jay-freedman.info. I've also sent a copy directly to Marcelo.

The backup part of the macro writes the value of the Autocorrect.Entries(x).RichText property for each entry into the backup document. On a computer with English as the default language, the value appears as the word True or False. I didn't realize -- until I saw Marcelo's document -- that the word is internationalized, in this case to Portuguese as Verdadiero or Falso. The restore part of the macro looks for the literal "False"; if that doesn't match, the macro assumes that it's "True". This resulted in the attempt to add a plain-text string to the AutoCorrect list as a rich-text object, which failed.


The backup part now will write the English literals "True" and "False" in the backup document regardless of the default language or the language of the selected text. Plain-text entries are still restored to the .acl file appropriate for the language applied to the AutoCorrect's value.

The E-16 values are the result of rounding anomalies that are common with Excel arithmetic.  (Explained below.)

With your example, we can correct the rounding anomalies by array-entering =ROUND(MMULT(B3:F7,B10:F14),15) instead, with the following results:



Note:  Generally, I would round to something less than 15.  Choose a precision that fits your problem and expected results.

Is there a way to apply conditional formatting with one click? - Microsoft Community

mathdunce say:

Is there a way to apply conditional formatting with one click?

With many Workbooks to evaluate, is it possible to create a macro (or something else) which would allow selecting cells and then applying conditional formatting. Format Painter only works for the open workbook in which the Conditional Format is created.

bill manville say:

Certainly you can create a macro to apply conditional formatting to the currently selected cells.

And you could attach the macro to an icon on your QAT (quick access toolbar).

That would give you the one-click method you desire.

First we need to be clear about the effect you want to achieve:

- is it to replace any existing conditional formatting in the selected cells?

- or is it to add to that existing formatting, and if so, do the new rules come first or last in the sequence?

The easiest case is where you want to replace any existing formatting.

I would suggest you start by selecting some cells and then recording a macro (Developer > Record Macro > Name: ApplyCF; Store macro in: Personal Macro Workbook) while you do:

-  Home > Conditional Formatting > Clear Rules > Clear Rules from Selected Cells

- Home > Conditional Formatting > New Rule > ... and set up the rule(s) you want

Then stop recording (Developer > Stop Recording)

Then test it by selecting some other cells and Developer > Macros > ApplyCF; Run

If not satisfied you can change it in the VB editor (Developer > Visual Basic) or delete it and start again.

When you are satisfied with it you can add it to the QAT.  Right-click the QAT > Customize the Quick Access Toolbar > Choose commands from : Macros; click on ApplyCF > Add

Note: if you don't have a Developer tab on your ribbon, Office Button > Excel Options > Popular > Show developer tab in the ribbon

vinay, india say:
I was successful in creating a workaround solution using a CURVE instead of STRAIGHT LINE and drawing a Straight Line out of a Curve!

Select Insert > Shape > Select Curve.

Start at One Point Possibly at a GRID LINE intersection of some column and Row. Move Across the pointer horizontally to the next intersection point and CLICK once, move over to another intersection point and CLICK ONCE and continue this for 20 times. When you are through DOUBLE CLICK to complete the Shape.

Then Select the shape > Go To the Format TAB and select SHAPE OUTLINE to give it a Dotted effect.

Also Select the shape and Click on the EDIT SHAPE > EDIT POINTS option to check the points and correct them if required. You may also Right Click and ADD / DELETE / ADJUST a point.

Each of the points will act as Connectors.

Hi Deven,

Please find the answers below.

#2

I would presume you are trying to combine cloud hybrid search with hybrid federated search.

Please just note that, in cloud hybrid search configuration, we manage all search settings, except for on-premises content crawling, in search administration in SharePoint Online in Office 365, including the query rules and result sources.

#5

Since the search index used after setting up cloud hybrid search are in Office 365, it is not feasible for us to directly trigger a Index reset on the Office 365 search index.
