OK, as long as the Default Language is as it should be:

  1. Launch Word to create a new blank document
  2. Go immediately to Tools> Language
  3. Confirm that the correct language is selected
  4. Clear both check boxes
  5. Click the Default button
  6. Confirm Yes to updating the NORMAL template
  7. Quit Word - If prompted to save the Normal template again click Save [There's no need to save "Document1"]

Note that this is the default setting for new documents only & is not a per document setting.
