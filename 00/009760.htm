teylyn say:

Hello,

you don't need VBA to do this in Excel 2013. You can use conditional formatting with the option "use a formula to determine ..."

To set a format for a cell that has a value, not a formula, use thi: formula

=NOT(ISFORMULA(A2))

Adjust the cell reference as required.

To set a format for a  cell that has a (any) formula use

=ISFORMULA(A2)

To set a format for a cell that references another sheet use

=ISNUMBER(FIND("!",FORMULATEXT(A2)))

This will highlight cells that are on a different worksheet. To identify cells that are outside the print area, click the View ribbon > page break preview

There really is no point in coloring cells that are outside of the print area. Excel has tools to show the print or page borders on the screen.

graham mayor say:

I do something similar, and find it simpler to create a rule to move the messages that match the requirements to a dedicated inbox sub folder. In the macro example that sub folder is called Forms_In

The Outlook macro reads the most recent message in that folder, sets the read flag and extracts all attachments that have the extension 'doc' to a folder - Here "D:\My Documents\Test\" Make whatever adjustments you need to the macro to identify the attachment(s) you are looking for

Sub ExtractAttachments()
    Dim olItems As Outlook.Items
    Dim olItem As Outlook.MailItem
    Dim oAtt As Attachment
    Set olItems = Application.Session.GetDefaultFolder(olFolderInbox).Folders("Forms_In").Items
    olItems.Sort "[Received]", True
    Set olItem = olItems(1)
    On Error Resume Next
    For Each oAtt In olItem.Attachments
        If LCase(Right(oAtt.DisplayName, 3)) = "doc" Then
            oAtt.SaveAsFile "d:\My Documents\Test\Merge\" & _
            oAtt.DisplayName
        End If
    Next oAtt
    olItem.UnRead = False
    Set olItem = Nothing
    Set olItems = Nothing
End Sub
