Add a reminder to documents in Windows Explorer - Microsoft Community

cxc00 say:

Add a reminder to documents in Windows Explorer

I wish to add a reminder to various documents in Windows Explorer (excel and word files). Is there any way do to this in Windows 7 or is there a plug-in I can download that will enable this functionality.

Thank you.

jay freedman say:
You can use the Windows Task Scheduler to do this. On the Start menu, type  Task Scheduler  into the search box and start the program. Choose to create a basic task. Follow the prompts to set the time (and recurrence, if any, or one time only).

When it asks what program to start, use the browse button and select the document you want. Just like double-clicking the document's icon in Windows Explorer, "starting" the document through Task Scheduler will start the associated program with the document opened in it.

Make a separate task for each document.

An alternative, if you have Outlook, is to add the reminders to Outlook's calendar. That will pop up a message (if Outlook is running when the time occurs) but won't start the program; you would have to find the document and open it yourself.

hansv mvp say:

You want

                    If InStr(.Value, "P113") = 0 And InStr(.Value, "P114") = 0 Then .EntireRow.Delete

instead of

                    If InStr(.Value, "P113") = 0 Or InStr(.Value, "P114") = 0 Then .EntireRow.Delete

Otherwise, only rows that contain both P113 and P114 in column Y will remain.
