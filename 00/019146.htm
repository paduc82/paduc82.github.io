rick rothstein, mvp - excel say:

Just to show you what Shane meant when he said there is "a lot we could do to simplify the code", here is but one of many possibilities. All the code that you posted, and which Shane modified for you, can be reduced to this (which is not a perfect example because it does not have any error checking to protect the code)...

Sub CopyData()
  Dim X As Long, Source As Worksheet, Destination As Worksheet
  Set Source = Sheets(InputBox("What is the Source sheet's name?"))
  Set Destination = Sheets(InputBox("What is the Destination sheet's name?"))
  For X = 0 To 3
    Destination.Range("D466").Offset(12 * X).Resize(9, 22).Value = _
                Source("EO2").Offset(11 * X).Resize(9, 22).Value
  Next
End Sub

Yes, I know you said you were an absolute beginner; but I thought since this is a project which you are familiar with, when you had the time to learn more about VBA, you could use the above code as a template for your investigation. The key points about the code are...

1) It is almost always unnecessary to select a range of cell in order to work with the Selection that resulted from doing so... you can usually operate directly on ranges without having to select them beforehand.

2) When you have repetitive code in which the repeat intervals are fixed and regular in size, you can use a loop to iterate through them.

3) VBA can assign the values in contiguous ranges to other contiguous ranges elsewhere on the sheet or on another sheet without having to Copy and then PasteSpecial them.

4) Offset and Resize are very powerful constructs for referencing ranges.

petrib say:
Valentas wrote:

Is there no way to install it myself?

Not in any supported fashion. Nokia's tools for users (Nokia Software Updater, NSU) or the built-in over-the-air (OTA) updating only updates the firmware with the languages that were on the phone when it was originally manufactured.

 

If an NSU or OTA updated causes a language to be lost, then someone after the device was manufactured, changed the firmware to a language variant that was not originally on the device, and when you did the NSU or OTA update, the original language variant was installed "back".

 

Usually this happens with, so called "grey import" devices (devices originally manufactured and sold to some other country, and then is importend and - incorrectly/partially - updated by someone who sells them then in some other country).
