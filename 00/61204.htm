With the introduction of Windows 8.1 Microsoft reverted back to the old (Windows 7) licensing model. Windows 8 did not have a retail version only System Builder with OEM and Personal User terms, the latter has transfer rights.

Windows 8.1 has the retail version with a specific OEM release for System Builders. The latter does NOT have a transfer rights clause, it can only be sold with the PC it has been Licensed on.
