bill manville say:

Took me a while to work out what was going on.

Some of the charts don't have the value axis starting from 0.

 

Run this macro
Sub FixCharts()
  Dim CO As ChartObject

  For Each CO In ActiveSheet.ChartObjects
    CO.Chart.Axes(xlValue).MinimumScale = 0
  Next

End Sub
