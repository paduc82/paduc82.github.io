halima s say:

Hi nancywhit,


• Do you remember making any changes to the computer recently?

This may happen if the Windows Installer files that are on your hard disk are damaged or are missing.
You install or remove a program that uses the Windows Installer Microsoft Software Installation (MSI) package file (.msi). For example, this may occur when you try to install Microsoft Office on your computer.

Method 1
First I would suggest that you check if the Windows installer service is running on the computer.

1. Go to "Start" - "Run" type "services.msc" and press on "Enter" button.
 
2. Double-click on the service named "Windows Installer"
 
3. Check the value in the "Startup type" field. If it's currently set to "Disabled", then this is the problem. Change it by selecting "Manual" from the drop-down box.
Make sure that the service is started and set to Manual.

Method 2
If the above step fails then unregister Windows Installer, and then reregister Windows Installer

To do this, follow these steps:
 
1. On the "Start" menu, click "Run"
 
2. In the "Open" box, type "msiexec /unreg", and then press ENTER
 
3. On the "Start" menu, click "Run"
 
4. In the "Open" box, type "msiexec /regserver" and then press ENTER


If the above steps fail then I would suggest that you install the latest version of Windows installer on the computer, refer the below link.

http://www.microsoft.com/downloads/details.aspx?FamilyID=5a58b56f-60b6-4412-95b9-54d056d6f9f4&displaylang=en

anonym0use say:

Hello Kyle,

I had a similar issue with VLC Media Player on the Version 1709 of Windows 10 (Creators Update), with the "VLC is unable to open the MRL 'dvd:///<drive>:\." error.

Where <drive> is replaced with the drive letter of the media drive.

To work around this issue you can try setting VLC Media Player to 'Run as Administrator'. To do this:

  1. Right-click on the shortcut for VLC Media Player and select Properties.
  2. Select the Compatibility tab.
  3. Down at the bottom, select the 'Run as Administrator' check box. Click OK.

Notes: The 64-bit version did not work with this change and would silently exit. The 32-bit version did work when 'Run as Administrator'. This workaround was successful with the latest VLC version as of this writing, 3.0.1. My issue was with playing DVDs, but I suspect this workaround should work with Blu-Ray discs as well.

Please also note that you can play personal Blu-Ray discs with VLC Media Player, but not commercial discs without a couple of extra files. More information found here:

https://www.easefab.com/tutorial/play-blu-ray-with-vlc-media-player.html
