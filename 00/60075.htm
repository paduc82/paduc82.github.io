Here's how to check for an outdated webcam driver:

  1. Select the Start button, type Device Manager, and select it from the search results.
  2. Find your webcam under Imaging devices or Sound, video and game controllers.
  3. Right-click the name of your webcam, and then select Properties.
  4. Select the Driver tab, select the Driver Details button, and look for a file name that includes stream.sys. If it's there, your webcam was designed before Windows 7 and you'll have to replace it with a newer webcam.
  5. From the Driver tab, select Update Driver > Search automatically for driver software.
  6. After installing the updated driver, restart your device and see if your webcam is working.

Change the "Remote Desktop" port in Windows.

To change it, we must modify the Windows registry.

Before modifying the connection port it is advisable to make sure that this port is not being used by another resource and that you allow the passage through the Firewall for the new port.

Start the Registry Editor (Regedt32.exe).

Finally you have to exit the registry editor and restart the server.

arya s asok say:

Hi  BluebyrdM,

 

If you are facing low disk space while copying and pasting to a new user profile, you may try to use the option to cut and paste. Cut and paste will not use much disk space. The files in the old user account will be moved to the new user account.

 

If you copy and paste the files will still be there in the old user account and that will consume lot of space on the hard drive.

 

To Cut the files and paste it to a different location, you may follow the steps below:

 

a.    Open the location where you want to copy the files from.

b.   Select all the files.

c.    Press the Alt key on the keyboard.

d.   Click on Edit menu.

e.    Click on Cut.

f.    Navigate to the new user account where you want to paste it and Right click in the Windows explorer and click on paste.

 

This will move the files without consuming much space.
