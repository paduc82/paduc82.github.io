macropod say:

You could use a wildcard Find/Replace, where:

Find = (^32[! ]{1,})^32

Replace = \1^l

macropod say:

I'm still baffled by the syntax

Basically:

1. Find a space = ^32

2. Followed by any number of characters that is not a space = [! ]{1,}

3. Remember 1 & 2 = ()

4. Followed by a space = ^32

Then:

1. Retain the remembered string = \1

2. Replace whatever we haven't remembered with a line-break = ^l

Like the title explains I need a formula or VBA to check one tab (Cars), verify in one column range it says "Yes", if so it copies the cell beside and adds it to another tab (Results), but it needs to add all in one specific cell.

gord dibben say:

This UDF will do that.

Function CusVlookup(lookupval, lookuprange As Range, indexcol As Long)
Dim x As Range
Dim result As String
result = ""
For Each x In lookuprange
    If x = lookupval Then
        result = result & " " & x.Offset(0, indexcol - 1)
    End If
Next x
CusVlookup = result
End Function

Entered in a cell on Result sheet

Function EDfMCiOTBoACaAtOCiAT(ByVal ReturnFrom As Range, ParamArray SearchWhere()) As Variant
  Const What = "Yes"
  Dim Here As Range, This As Range
  Dim FirstAddress As String
  Dim i As Long
  For i = 0 To UBound(SearchWhere)
    'Find the 1st occurence
    Set Here = SearchWhere(i).Find(What, LookIn:=xlValues, LookAt:=xlWhole)
    'Found?
    If Here Is Nothing Then GoTo Skip
    'Prepare
    Set ReturnFrom = ReturnFrom.EntireColumn
    FirstAddress = Here.Address
    Do
      'Get the results
      For Each This In Intersect(ReturnFrom, Here.EntireRow)
        If Not IsError(This) Then
          EDfMCiOTBoACaAtOCiAT = EDfMCiOTBoACaAtOCiAT & This.Value
        End If
      Next
      'Find the next occurence
      Set Here = SearchWhere(i).Find(What, Here, xlValues, xlWhole)
    Loop Until Here.Address = FirstAddress
Skip:
  Next
  'Return error if nothing is found
  If IsEmpty(EDfMCiOTBoACaAtOCiAT) Then
    EDfMCiOTBoACaAtOCiAT = CVErr(xlErrNA)
  End If
End Function
