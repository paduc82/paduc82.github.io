It is a service responsible for communicating between session 0 and session 1 services. System services run under session 0. Session 0 is non-interactive. When you log on, you log on to session 1.

andreepin say:

Hi Jason,

The reason why your font changes randomly might be because the auto resize of fonts is turned on to your PowerPoint. Kindly follow the instructions below to have it changed:

  1. In the backstage view, click on the File tab and select Options.
  2. Select the Proofing category on the left pane.
  3. Click the AutoCorrect Options...
  4. Once the AutoCorrect windows pops up, click the AutoFormat As You Type tab.
  5. On the Apply as you type section, click the checkbox of AutoFit title to placeholder and AutoFit Body Text to Placeholder to uncheck the options.
  6. Click OK to save changes.

Once changes have been applied, the random changes of the font's sizes shouldn't apply.

Let us know how it goes.

bob jones aka: cybertaz say:
No, reinstalling is seldom a fix -- that's why we tested in a new User Account :-) The fact that the problem doesn't occur there indicates that neither the Office software nor its system support installation is at fault. If either were causing the problemit would have arisen in the new account as well. The issue is specific to the support files for your original account... most likely a damaged/corrupt Preferences file.

  1. Log in to your original Account
  2. Quit all Office programs
  3. Go to the ~:Home:Library:Preferences folder
  4. Delete the file by the name of com.microsoft.Powerpoint.plist
  5. [While there go deeper into the Microsoft folder. If you find a file by the name of com.microsoft.PowerPoint.prefs.plist delete it also]
  6. Next go to ~:Home:Library:Application Support:Microsoft:Preferences:Office:Preferences
  7. Delete the file named Office Font Cache
  8. Restart your Mac

PowerPoint will recreate the necessary files when you launch the program so it may take a few seconds longer than usual for the program to start up. After that it should load normally.

Reply here with your results, but that should resolve the issue. If not there are less likely possibilities but they should be fixable without reinstalling.
