I have created a worksheet with several tabs at the bottom for different materials we use in our company.  We are using each sheet to calculate the material we have vs. what is needed for a job, then it will highlight the resulting value and calculate the material needed to order.  Rather than going through each tab and copying each row, how can I have excel autopopulate a new sheet whenever the resulting column, which is E has a value greater than 0?

bernie deitrick say:

Try this macro - just update the name of the tab that is the PO template.

Sub TestMacro()
    Dim sht As Worksheet
    Dim rngC As Range
    Dim lngR As Long
    Dim strName As String
    
    strName = "PO Template"
    
    For Each sht In Worksheets
        If sht.Name <> strName Then
            For lngR = 2 To sht.Cells(sht.Rows.Count, "E").End(xlUp).Row
                If sht.Cells(lngR, "E").Value <> 0 Then
                    sht.Cells(lngR, "E").EntireRow.Copy
                    Worksheets(strName).Cells(Rows.Count, "E").End(xlUp)(2).EntireRow.PasteSpecial xlPasteValues
                End If
            Next lngR
        End If
    Next sht
End Sub

spectre-4 say:
In that case we can try removing the last of any user credentials info. If this is successful you will need to login one more time but that should be it.

To get started follow these steps:
1. Open Regedit (The Windows 10 Registry Editor)
2. Navigate to HKEY_CURRENT_USER\Software\Microsoft\Office\16.0\Common\Identity
3. Delete the Identity folder
4. Launch an Office program and sign in
5. Try launching a second and see if it has been resolved.

If this does not work then I suggest uninstalling office, rebooting the machine and reinstalling it. Other than that there is not a great deal else you can do as Credential Manager doesn't allow you to manually add accounts.
I hope that helps.

rohn007 say:

Yes he could install your paid for copy of Office for you. But "giving you a free" copy of Pro Plus is wrong.  It is possible for him to give you a "free trial" of Pro Plus, and use a little known, but still legal, technique to extend the trial to 6-7 months.  I don't know of any way they can give you a legal free trial that lasts more than a year.
