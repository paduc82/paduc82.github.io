The drive letters assigned to the partitions during a clean install of an OS are completely independent of any other OS in a multi-boot situation. tom van stiphout say:

That's by design. Here is a workaround:

In the Form_Open set the timer to 1 msec:

Me.TimerInterval = 1

In the Form_Timer event set:

Me.TimerInterval = 0  'stop the timer
MsgBox "Eureka!"

The reason this works is that Timer is a low-priority event, and it happens after all other events are settling down.

Protect locked cells in multiple sheets - Microsoft Community

a. user say:

Protect locked cells in multiple sheets

I would like to protect locked cells in MULTIPLE sheets in one workbook but it would take much to long to do each one individually.  How do I do it across all sheets in one go?
gord dibben say:

I don't run 2016 but in earlier version I would use a macro.

Sub ProtectAllSheets()
    Application.ScreenUpdating = False
    Dim N As Single
    For N = 1 To Sheets.Count
        With Sheets(N)
          .EnableSelection = xlUnlockedCells
          .Protect Password:="PASSWORD"
          .EnableOutlining = True
        End With
    Next N
    Application.ScreenUpdating = True
End Sub

Sub UnprotectAllSheets()
    Application.ScreenUpdating = False
    Dim N As Single
    For N = 1 To Sheets.Count
        With Sheets(N)
          .Unprotect Password:="PASSWORD"
        End With
    Next N
    Application.ScreenUpdating = True
End Sub


Sub Protect_Selected_Sheets()
    Dim ws As Worksheet
    Set MySheets = ActiveWindow.SelectedSheets
    For Each ws In MySheets
        ws.Select
        ws.Protect Password:="justme"
    Next ws
End Sub

Sub UnProtect_Selected_Sheets()
    Dim ws As Worksheet
    Set MySheets = ActiveWindow.SelectedSheets
    For Each ws In MySheets
        ws.Select
        ws.Unprotect Password:="justme"
    Next ws
End Sub
