jay freedman say:

What you're seeing are the codes of TC fields.

TC fields are one way of creating a table of contents, and they're automatically marked as hidden text. If you can see them, one of two things must be true:

  • You have all nonprinting characters displayed because the ¶ button on the Home tab is active (shaded). If that's true, you should also be seeing other kinds of things, such as dots in the spaces between words, and paragraph marks at the ends of paragraphs. If you see those things, click the ¶ button to turn it off, or use the keyboard shortcut Ctrl+Shift+8.
  • If the TC fields are the only nonprinting characters you see, then a particular option has been checked. Go to File > Options > Display and clear the check box for "Hidden text".
