Moving cursor to the next line - Microsoft Community

hamid shamsiyan say:

Moving cursor to the next line

Could you please suggest a macro which moves the cursor to the beginnig of the next line and shows the message "The next line exists", otherwise if the next line does not exist, it will show the message "The next line does not exist". Could you please suggest a macro which moves the cursor to the beginnig of the next line and shows the message "The next line exists", otherwise if the next line does not exist, it will show the message "The next line does not exist".
rich007a say:

Does this do what you need?:

 

Sub MoveDownOneLine()
    Dim rng As Range
    Dim lnCurrentRow As Long, lnLastRow As Long
   
    lnCurrentRow = Selection.Information(wdFirstCharacterLineNumber)
   
    Set rng = ActiveDocument.Range
    rng.Collapse (wdCollapseEnd)
    lnLastRow = rng.Information(wdFirstCharacterLineNumber)
   
    If lnCurrentRow < lnLastRow Then
        Selection.MoveDown Unit:=wdLine, Count:=1
        MsgBox "The next line exists"
    Else
        MsgBox "The next line does not exist"
    End If
End Sub
