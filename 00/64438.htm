mike hall... always say:
If it takes a long time to get to the Windows logon, the hard drive has a problem.
 
What happens is simple. Clusters on the hard drive start to fail, and the computer has to move data from bad clusters to good. In cases like this, you can either leave ity as it is, or consider replacing the drive and cloning the installation soon, or wait until it becomes a critical problem, in which case everything will have to be re-installed..
 
If the time it gets to logon is reasonably fast, but is slow getting to the desktop from logon, it may be that you have picked up a virus, or installed a program which places itself (needlessly in some cases) in Startup. The other answers that you have received cover the latter two options..

(Feature Request) Custom on-screen keyboard for artists. - Microsoft Community

mario baldi say:

(Feature Request) Custom on-screen keyboard for artists.

Many artists are looking to the Surface Pro 4 as a digital sketchbook, both because of its pen and because it can run pro softwares like photoshop, zbrush, etc.

The main problem with these programs though, is that they are almost never optimized for a tablet experience, as they heavily rely on keyboard shortcuts.

Wacom solved this issue with on-screen controls: http://101.wacom.com/UserHelp/en/OSC.htm

...basically they are similar to the standard on-screen Windows keyboard, but they allow to customize its layout and its mapping as well.

Thirdy party developers gave some solutions in the past, like the beautiful "Radial Menu", which sadly doesn't work properly on the Surface Pro 4.

A newer tool, called TabletPro tries to do the same, but it has some serious issues registering the buttons when using the pen at the same time, which defeat the whole purpose of using the tablet as a sketchbook.

In my opinion, such a customizable on-screen keyboard should be available by default in Windows 10, especially considering that the pen is a very important tool for the Surface Pro 4. Noone better than Microsoft could solve the problem of using both the pen and these on-screen controls at the same time.

The Type Cover is not practical when we want to use the Surface Pro 4 in tablet mode.

Thanks for listening.
