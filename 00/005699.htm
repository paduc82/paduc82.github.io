Hello Marc,

Unfortunately you cannot rearrange the group order in the left side menu in Yammer.

There really isn't an order so to stop the algorithm for Chronologically arrange the topics.

 

Feeds are based on algorithms which prioritizes active stream content and will be different for every user AND will be different on the mobile app than from the desktop.

 

It finds any result that contains at least one of the search terms and then orders the results using an algorithm that gives priority to (a) results containing more of the search terms than others (ex. search for the brown fox, (b) results that are more recent, and (c) results that are more likely to be relevant to you (i.e. result is in a group you're a member of).

We still support exact phrase search (i.e. only match "the brown fox", not "the fox brown") if you use quotation marks around the search terms.

argmarovski say:

It seems you misunderstood the system requirements posted for Windows 10. It says that you need free space of 16GB (for 32 bit systems) or 20Gb (for 64 bit systems) for the upgrade. Note that this is valid for upgrades only, e.g. in case you upgrade from Windows 7/8/8.1 you need the said amount of free space on your system directory.

If you perform a clean installation of Windows 10 you need 50GB of free space as a minimum, but I would suggest 150GB for better performance. There are some games from Windows Store that you cannot install on a different directory than your main one.
