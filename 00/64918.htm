It is entirely possible to get Win7 to recognise USB 3.0 devices but in order to do so you need to create a custom install ISO with the drivers slipstreamed in.

This means you need a valid Windows 7 disc or at least a valid product key (you can download Win7 from MS).

I know for a fact that Intel have USB 3.0 drivers for Win7 on their support site.

There are several guides on how to create a USB 3.0 enabled Win7 ISO, the method is fairly straight forward but apart from a valid Win7 image you also need a utility to create the new ISO with drivers slipstreamed in, also the drivers need to support the USB chipset on your motherboard. Another thing to watch is whether the USB 3.0 drivers are for 32 or 64 bit Windows.

So, even though it is possible, it may be too much trouble to do.

Anyway at the end of all that the only way to install the USB 3.0 drivers is at the Windows install time.
