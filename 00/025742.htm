dirk goldgar say:

You can set the caret (the insertion point) to the end of the text currently in the text box when the control has the focus, using the code lines:

 

    With Me.txtYourTextbox
        .SelStart = .SelLength
    End With

 

(replacing "txtYourTextbox" with the name of your text box).  But you can only do this when the text box has the focus.  You could, of course, first set the focus to the control, and then set .SelStart:

 

    With Me.txtYourTextbox
        .SetFocus
        .SelStart = .SelLength
    End With

 

I'm not sure from your description, though, what your command button is doing.  If you are always appending the same text string (with a time stamp), you don't need to set the focus to the control at all, nor set the insertion point.  You can just append the desired text to the control's current value; something like:

 

    With Me.txtYourTextbox
         .Value = (.Value + vbCrLf) & Format(Now, "m/d/yyyy h:nn:ss AMPM") & " - Your Text String"
    End With

 

But if you want the user then to type something in the text box after that, you might write something like this:

 

    With Me.txtYourTextbox
        .Value = (.Value + vbCrLf) & Format(Now, "m/d/yyyy h:nn:ss AMPM") & " - Your Text String"
        .SetFocus
        .SelStart = .SelLength
    End With
