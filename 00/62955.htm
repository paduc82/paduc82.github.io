Method 2: Transfer the user profile data from the old account to the new user account.

 

  1. Type File Explorer in Search.
  2. Click the View tab.
  3. Click Options at the top right corner.
  4. Select the View tab.
  5. Click Show hidden files, folders, and drives.

  7. Hit Yes.
