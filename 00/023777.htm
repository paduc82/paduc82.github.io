Linking _to_ the Table of Contents _from_ section headings - Microsoft Community

mrmitchell say:

Linking _to_ the Table of Contents _from_ section headings

I have a document with a Table of Contents linked to about three dozen different section headings and I want to link the section headings back to the Table of Contents, so that whenever the reader is done reading a section they can click the heading to go back to the TOC.

 

I've searched online but all I can find is help for creating TOCs, which is not what I need.  All the section headings are in the Heading 1 style but I can't find a way to modify the style to include the hyperlink.  Selecting all the instances of Heading 1 greys out the Ribbon button for hyperlinks so I can't do that either.

 

I'm thinking there simply has to be an easy way to do this for all the headings at once.  If it matters any the headings are also all Level 1 in Outline view.

suzanne s. barnhill say:

Perhaps someone can provide VBA to automate this, but the bottom line is that you would insert a bookmark above or around the TOC, then select each heading and use Insert | Links | Hyperlink (or Ctrl+K) to insert a hyperlink to that bookmark. The heading you selected will be the display text, so it won't change appearance except that mousing over it will product the Ctrl+click to follow hyperlink ScreenTip. I gather this is what you had in mind, but you can't do it for the headings en masse; you have to do them individually.

 

Alternatives: readers can use Alt+Left Arrow (the WebGoBack command) to return to the TOC, or they could display the Navigation pane and use that, or they could add the GoToTOC button to the QAT.
