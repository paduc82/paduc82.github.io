Short answer: While in Print Layout View, do Select All and then click on the highlighted shape.

Long answer: Add the missing tool

Actually, you don't need it, but it's a nice tool to have around. If you do Select All (Command A) in Print Layout View, all objects become framed with the selection color and you can click on them to select them. However, you cannot select several object at once, because shift-clicking does not work.

I don't recommend modifying either of the two built-in toolbars (Formatting and Standard), but you can skip step 1.

1) Create a new toolbar

  • At the top of the dialog box, select the Toolbars and menus tab.
  • Click on the New button at the top right. Give the toolbar a name and press OK. Pay attention, as the new toolbar will be hard to see it because it's small. You can move it around by grabbing it from the "colon" below the Close circle.

2) Add the missing selection tool
While still in the Customize... dialog box:
  • Click on the Commands tab at the top.
  • In the "Categories" list to the left, click on "All Commands" to select it.
  • In the list to the right, scroll to "SelectDrawingObjects."
  • Click on it and drag it to the new toolbar.

This tool lets you select several objects—click anywhere but on a picture and drag the selection rectangle.

What is difference between count and numerical count? - Microsoft Community

jcmu99 say:

What is difference between count and numerical count?

Hello, please see screenshot below. When I highlight a range of 3 cells and look at my taskbar and see "count" and "numeric count" they both show the same number "3". Even though the sum of values in the cells is not "3". I assume one of those options is for the total number of cells which is 3 so that would be correct. But why would they both say "3" ?

ben schorr say:

Count is the number of cells that aren't empty.

Numerical count is the number of cells that have numbers in them.

So you have 3 cells with content and all 3 of those cells have numerical values.

If you add a 4th cell with text in it to the selection your count would go to 4 but numerical count would remain at 3.


Hope that helps!
