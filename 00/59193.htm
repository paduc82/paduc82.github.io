Windows media center, overlapping recordings - Microsoft Community

heymike_s say:

Windows media center, overlapping recordings

Why, with multiple sources available does WMC insist on making back to back recordings from the same tuner?  I'm pretty much guaranteed that I'll have to watch all my recordings in order, since the first couple of minutes of the second recording is always tucked into the last couple of minutes of the first recording.
chris cupler say:

You're correct - it's not a hardware issue.  Rather, it's a long-standing (read:  it's been like this for years, and across multiple versions of Media Center) failing of the tuner-assignment logic that many of us over at thegreenbutton have complained about ad nauseum.

There is one thing that (sometimes) fixes it:
When you have two recordings, back to back, on the same channel, change the "stop after" option for the first recording to a hard value (e.g. "10 minutes"), rather than the normal "xx minutes, when possible."  That (sometimes, but not always) forces the first recording to go long enough that it forces MC to use a second tuner for the second recording.

You think it's annoying?  I have  14 tuners (4 analog, 6 Clear-QAM, and 4 CableCARD tuners in my main system (via 2 DualTV MCEs, 3 HDHRs, and an InfiniTV4)), and still I get this.  Even when there are several available tuners, it still does this.  :evil:  and ((shrugs))

Maybe it'll be fixed in the next version.....

HTH,
Chris

Do this from the command line:

Open Start, type: CMD
Right click CMD
Click Run as administrator

Type in at the prompt OR Copy and Paste these one at a time : (Hit enter after each)

TAKEOWN /F C:\ /R /D Y

Once you have taken ownership, then you have to set the permissions; for this, you use ICACLS:

ICACLS "C:\" /reset /T
