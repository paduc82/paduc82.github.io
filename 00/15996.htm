Copy and Paste column with no formula change - Microsoft Community

lisa weichold say:

Copy and Paste column with no formula change

How do I Copy and Paste a column to another column with no formula change? How do I Copy and Paste a column to another column with no formula change?
jeeped say:

You can easily accomplish this using an intermediary program that will strip any overhead or relationship that Excel has attached to the formulas.

  1. Press CTRL+~ to change the display to formulas rather than values.
  2. Click the alphabetic header label of the column to be copied. Press CTRL+C to copy the entire column.
  3. Open Notepad from the Start Menu and press CTRL+V to paste. Immediately press CTRL+A to Select All then CTRL+C to copy again. Close Notepad without saving.
  4. Return to Excel and click the alphabetic header label of the target column to select the entire column. Press CTRL+V to paste. Click OK to ignore the silly warning.
  5. Press CTRL+~ to toggle the display back to values from formulas.

Your column of formulas (and possible values) should have copied directly across without changing column cell references. It might seem like the long way around but it takes about a tenth of the time to actually do as it takes to write out the steps.

Or you could modify the column cell references in the original column by prefacing them with a $. This makes them an absolute reference that will not change when the formula is copied or moved laterally.

jhoyt say:

As far as I know, having four non-consecutive time references isn't exactly what Scatter Plots are designed for. You should just use a line graph because,

  1. You only have four points, and scatter plots are designed to show time intervals to give a visual idea of how frequent/infrequent occurrences are over time.
  2. Since you only have four points above, you probably want to see a trend. A line graph will still show you a trendline, and you can make your axis whatever you want.

Now if you would like to use the line graph option and type in the dates as you have above, the graph will initially add all these other dates to the axis. You can make the axis be exactly what you want by formatting the axis. Format x-axis, go to axis options, go to axis type and select Text axis.

Text axis isn't an option in scatter plot formatting, because it's outside of what a scatter plot was designed for.

mike middleton say:

The problem may be that Excel is not interpreting your dates as date values.

You may need to re-enter the dates into cells that have been pre-formatted as dates before you construct the chart.

If your data are dates on the X axis and numbers for the Y axis, you can use either a Line chart type or an XY Scatter chart type.

If Excel interprets any of your X values as text (instead of numbers or dates), Excel will default to the Line chart type with equally-spaced "text" entries on the horizontal axis.
