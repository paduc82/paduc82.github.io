MS continues to recommend people use 32bit office even on 64 bit windows unless they have an explicit need for 64 bit support.

Hi. I'm using Publisher 365 to layout books for binding. Using booklet templates for each individual signature (group of pages that will be stitched together.) Also need to know if it is possible to set auto page numbering to start at X. Currently manually numbering pages in individual signature files. I need to be able to set starting page number for each signature and let Publisher continue auto numbering from there. (eg) Sig1 = page 1-16, Sig2 = page 17-32, Sig3 = page 33-48, etc.

I'd appreciate any ideas or input you may have.

davidf2 say:

Publisher booklet templates must be be in 4 page increments, so your 16 page booklet group is fine in that regard.

As far as combining into one file and maintaining a 16 page "signature" groups, I don't think so. You can insert a new section ever 16 pages if that would help but I am not convinced that the goal of one large file is best as the bigger the Publisher file gets the more likely it is too become corrupt, especially if you are using a lot of inserted pictures. By the way, back up your backups...

Before you setup your book in this way, I would suggest you double check with your proposed book publisher as to what format they want to receive your book pages. It may be that they will want to do this final book layout themselves and want your pages in a more usable format. You don't want to make an assumption that results in your having to go back and change everything.

As to continuing the page numbering across multiple Pub files, that can be done. One way is to go to View tab > Show group and check Page Navigation. Then go to the left hand Page Navigation pane, right click page one > Page Numbers and choose where you want the page numbers to appear such as bottom left, etc. Then again right click the first page > Page Numbers > Format page numbers > in the Page Number Format dialog check Start this section with: and insert the next number after the last 'signature' or pages such as 17 if it is the second file or Signature group of pages.

One final suggestion is that you consider using Word for writing your book. In general it is considered the best tool for large page number files such as books.

DavidF

continue numbered list across header sections - Microsoft Community

__pc__ say:

continue numbered list across header sections

Hi,

I want to have a numbered list in a Word document. I have numbered headers, 1. Overview, 2. General Intro etc., but I want the numbering for these list items to continue between sections.

I also want to create a table of the list at the beginning of the document, something like a Table of Figures.

Not sure how I can do this, any assistance would be appreciated

TIA

eddiec :-)

suzanne s. barnhill say:
If I understand you correctly, you are using numbered headings for your sections. You may even have used outline numbering to number the top-level and lower-level headings (if you need help with this, seehttp://www.shaunakelly.com/word/numbering/OutlineNumbering.html). But you have a simple list whose numbering you want to be continuous throughout the document, and you want a list of these numbered items at the beginning of the document.
 
The key to this setup is to use styles for all the relevant paragraphs. Associate numbering with the heading styles as described in the article referenced above. Then create another style that is NOT part of this outline-numbered (multilevel) list and link it to the numbering style you want for the list. Its default behavior in some versions of Word will be to continue from the previous number, but if you find that the numbering restarts, right-click and choose Continue Numbering. (Alternatively, you could use a SEQ field for the numbering, but this cannot be defined as part of a style, so it must be added manually and updated manually.)
 
To create a list of the numbered items at the beginning, create a TOC or TOF based on the style you have used for the numbered paragraphs. A Table of Figures based on a single style is perhaps a little easier than a TOC, but either will work; in a TOC you will have to open the Options dialog and delete the numbers for the built-in styles that have outline levels (all headings, plus Title and Subtitle).
