Hi here2read,

 

Welcome to the Nokia Discussion boards!

 

You can have contacts in your SIM card and in your phone memory. You can view and use the contacts simultaneously from both options, or you can choose to use just one of the alternatives.

 

To adjust which contacts you want to see:

 

  1. In Contacts, select Options > Settings.
  2. Scroll to Contacts to display and select Change. Mark the desired contact in:
    • Phone memory
    • SIM memory
    • Service numbers
  3. Press OK.

You can also choose the default saving memory for saving phone numbers:

  1. In Contacts, select Options > Settings.
  2. Scroll to Default saving memory. Select Change. Choose from Phone memory or SIM memory.
  3. Press OK.

 

When you open the contacts list for the first time, the device asks if you want to copy names and numbers from the SIM card to your device.

 

  1. To start copying, select OK. If you do not want to copy the contacts from the SIM card to your device, select Cancel.
  2. The device asks if you want to view the SIM card contacts in the contacts directory. To view the contacts, select OK. The contacts list opens, and the names stored on your SIM card are indicated with the SIM card symbol.

 

You can copy single or several contacts from the SIM card to your phone memory and vice versa.
To copy contacts:
  1. Scroll to the contact you want to copy, and select Options.
  2. If you want to copy a single contact, select Copy. Now you can choose the destination you want to copy the contact to.
  3. If you want to copy several contacts, select Mark/Unmark and Mark to choose the contacts,or Mark all. The chosen contacts are displayed with a mark. Select Options >Copy again, and select the destination you want to copy the contacts to.

Hope this helps.

When you create a new scheme the scheme is applied to the template and it is also saved to your profile as an xml file. That is why you see the custom entry for it under colors.

 

It is stored here in your profile folder : %appdata%\Microsoft\Templates\Document Themes\Theme Colors

 

This profile is not accesible by another user on the same machine or another.

Public Function IsFileOpen(filename As String)
'Fm Ken Snell - 11/7/03
Dim Filenum As Integer, errnum As Integer
On Error Resume Next   ' Turn error checking off.
Filenum = FreeFile()   ' Get a free file number.
'Attempt to open the file and lock it.
Open filename For Input Lock Read As #Filenum
Close Filenum          ' Close the file.
errnum = Err           ' Save the error number that occurred.
On Error GoTo 0        ' Turn error checking back on.
'Ck to see which error occurred.
Select Case errnum
  'No error occurred.
  'File is NOT already open by another user.
  Case 0
    IsFileOpen = False
    'Error number for "Permission Denied."
    'File is already opened by another user.
  Case 52  'Bad Filename
   MsgBox "Please Notify Bob.", , "": Exit Function
  Case 53 'File Not Found
   'OK
  Case 70
    IsFileOpen = True
    'Another error occurred.
  Case Else
   MsgBox Err.Number & " " & Err.Description: Resume
   'Error errnum
End Select
End Function
