Thanks

Pirate

macropod say:

You could use an on-entry macro coded as:

Private Sub Document_ContentControlOnEnter(ByVal CCtrl As ContentControl)
With CCtrl
  If .Title = "MyDate" Then
    If .Range.Text <> Format(Now, .DateDisplayFormat) Then
    .Range.Text = Format(Now, .DateDisplayFormat)
  End If
End With
End Sub

but you'd probably want something a little more nuanced than that, as this also updates the displayed date any time the month/year changes.
