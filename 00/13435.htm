Insert last modified date in PowerPoint - Microsoft Community

travelined say:

Insert last modified date in PowerPoint

Hi - I want to put a date field/box in a PowerPoint presentation slide master that updates when the doc is modified, not every time it's opened. 

I'm open to using a macro somehow to do this, if that's the best route.  Maybe have an "update date" button in the title slide that runs the macro to update the date in the slide master.

Advice?
shyam pillai say:

PowerPoint has limited field functionality. An updatble inserted date will always update when the presentation is opened.

Since you are open to a macro based solution fired from a button. You could use this:

Sub UpdateModifyDateOnMaster()
Dim oShp As Shape
For i = 1 To ActivePresentation.Designs.Count
    With ActivePresentation.Designs(i).SlideMaster.Shapes
        For j = 1 To .Placeholders.Count
            If .Placeholders(j).PlaceholderFormat.Type = ppPlaceholderDate Then
                .Placeholders(j).TextFrame.TextRange.Text = "Last Modified: " & Format(Now(), "mm/dd/yyyy")
            End If
        Next
    End With
Next
End Sub

 

Sub UpdateModifyDateOnSlides()
Dim oShp As Shape
For i = 1 To ActiveWindow.Selection.SlideRange.Count
    With ActiveWindow.Selection.SlideRange(i).Shapes
        For j = 1 To .Count
            If .Item(j).Type = msoPlaceholder Then
                If .Item(j).PlaceholderFormat.Type = ppPlaceholderDate Then
                    .Item(j).TextFrame.TextRange.Text = "Last Modified: " & Format(Now(), "mm/dd/yyyy")
                End If
            End If
        Next
    End With
Next
End Sub

 

Assign a button to fire this when you want to update the modified date. You can use either of the two.

 

Macro one: The macro checks for the date placeholder on the slide master and updates it.

Macro two: checks for date placeholder on the selected range of slides and updates it.
