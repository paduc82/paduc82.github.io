paulcordts say: First thing I am noticing is that you are using the Breaks/Lunch Time value twice.  Once for setup start time and again for Scheduled completion time. You may need one value for set up and one for job run the total duration of breaks would not be the same for a 4 hour set up and 120 hour run.

I am assuming that the first 4 cells are entered data and you want to calculate the Job Start and Scheduled Completion

You need to separate the calculation for days and hours
                A                                  B 
1 Setup time (minutes)                1500

2 Job run time (minutes)             1800

3 Breaks/Lunch time (minutes)        60

4 Setup Start time                      3/1/12 1:00 AM  

5 Job Start time                         3/2/12 3:00 AM   

6 Scheduled Completion Time     3/3/12 10:00 AM


Cells B1:B4 are entered

B5=ROUND((B1+B3)/(24*60),0)+TIME(0,MOD((B1+B3),(24*60)),0)+B4

ROUND((B1+B3)/(24*60),0) give you the full days of B1+b3

MOD((B1+B3),(24*60) give you the remaining minutes and the TIME function allows you to add it as time to B4 plus the setup days


B6=ROUND((B2+B3)/(24*60),0)+TIME(0,MOD((B2+B3),(24*60)),0)+B5


Same logic just using B2+B3
