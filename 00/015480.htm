adamf---01 say: I hope this thread answers your question Ralph.

_kmv say: Actually, you can change the time. The time is held by the network but most handsets will allow you to change it.

On the 6300 do this...

First go the menu set that controls these settings, all following instructions will be relative to this menu:
Menu -> Settings -> Call -> Call divert

Now, be careful with these settings because it is easy to mess up your voice mail or even make yourself unreachable if you get this wrong. Consider yourself warned!

So, to minimise an error note down your mailbox number:
If busy -> Check status -> Number

WRITE THAT NUMBER DOWN - INCLUDING THE '+' SYMBOL

Now go back to the menu:
Back -> Back

Now we can change the actual time, the easiest way to do this is via a shortcut function the 6300 provides (which sets several divert settings at once):

If not available -> Activate -> To other number -> (enter the number you wrote down including the '+' symbol) -> OK -> (select the delay time you want) -> Select

And you are done!


NB: If you mess these settings up don't come crying to me when people can't reach you or leave you voice mail. If this should happen contact your network provider and tell them that you have messed up your "divert settings" and that they need to be reset on the network.

You shouldn't save to a subfolder of Program Files, but either to your user templates folder or to your workgroup templates folder.

The user templates folder is by default C:\Users\<username>\AppData\Roaming\Microsoft\Templates; to see it you need to show hidden files and folders in Windows Explorer.

  • Click the Office button.
  • Click Word Options.
  • Click Advanced.
