john mcghie say:
Well!  It's REALLY pooched this time, isn't it?

I will put that in as a bug.  In the meantime, to get your dictionary back, do the following:

1)  Quit Word.

2)  Locate your custom dictionary in ~/Library/Preferences/Microsoft/Office 2011

3)  Move it to the desktop

4)  Start Word

5)  Go to Word>Preferences>Spelling and Grammar>Dictionaries

6)  Click "Add".  Notice that Word suggests a file named "Custom Dictionary.dic" with an extension this time?

7)  OK that, and Word will now list TWO active dictionaries named "Custom Dictionary".  It will then enable you to use REMOVE to remove the non-existent one.

8)  Now click Edit to edit the real one (which should be named "Custom Dictionary.dic" and have no content).  You may have to try more than once, ignoring error messages.

9)  Open your old custom dictionary from the desktop in TextEdit.  Copy all.

10)  Click in your new Custom Dictionary.dic in Word, and Paste.

11)  Save.  IGNORE any message about losing formatting.  You CERTAINLY want to lose the formatting :-)  If there's formatting in there, it WON'T work.  The file must be saved in plain text, as Unicode 5.1 (Little Endian).  If you ignore the warning, that's what Word should do.

12)  Close the file.  Create a blank document, then  Word>preferences>Spelling and Grammar and turn your spelling back on.

It should work now, and contain the words you want.  If you check in Finder, you will find ~/Library/Preferences/Microsoft/Office 2011 folder contains a "Custom Dictionary.dic" file, this time with the correct icon for a custom dictionary.

Hope this helps
