Word is hopeless with custom paper sizes. It uses the printer driver to format the document and if that driver does not accept the custom size, a standard size, which may or may not be close to the requested size, is substituted. If this standard size is larger than your requested size the return address may be 'printed' off the page.

You have two options. Either deliberately use a larger size and fix the positions of the addresses so that when printed they are in the correct place, or create a template to do the same. E65 - how do I change the text alert?

I have an E65 from Vodafone UK. Great phone! I want to change the text alert tone to one I have on my memory card, but cannot find a way to do so. It is in the correct format. How do I do this?

Dysania

keffa---01 say: Go into the menu and then go into "Tools", then "Profiles".

Choose the profile you wish to change (The "General" profile is usually the one you keep your phone in), select it and choose personalise. Then scroll down to "Message alert tone" and provided the tone is of the correct type then it will be shown in the list, along with a picture of a little memory card if its stored on there. Select this and you will have changed your text message alert tone.

dysania say: Thanks Keffa. Unfortunately the message alert tone does not show any of my tones on my memory card. How do I make it show them to be selected?

keffa---01 say: Make sure they are located either in "Sound Clips\Digital Sounds" or "\Sound Clips\Tones" and reboot your handset.

If they still do not appear it might not be in the correct file format.

There is no issue with ccleaner and MSE. However, you should use caution when letting it "clean" your PC by "fixing" the registry. Make sure to use the backup feature as letting any Registry cleaner make changes that you don't explicitly understand can lead to serious problems on the PC.

Don't run the malwarebytes real time protection with MSE, though. The on demand scans can certainly be run as desired.

MSE does not or should not use excessive RAM and has been shown to be quite effective at protecting the PC. No solution is 100% effective, however, and safe computing practices should be observed. If you are seeing excessive RAM or CPU usage by MSE, it is usually caused by a conflict - which could certainly be the case if you are using the paid version of malwarebytes with real time protection enabled.

I'll add that the "someone" quote is very wrong about one thing that is implied - ccleaner does not scan for or protect against malware.
