ben m. schorr -bschorr say:

Insert a copy of the file actually embeds the Word file in OneNote. Insert a link merely creates a link to the file stored somewhere on your system.

Neither of those options gives you the document where you can edit it in OneNote. If you really want that then you have to copy/paste the text from Word to OneNote.

Be forewarned though - Word is a far more powerful editor than OneNote is. I'm not sure you'll like the results if you copy/paste a Word document to OneNote.

I always just use the link option if I need to edit the document.  The printout option if I want to search/read/annotate it.

karl timmermans [outlook mvp 2012-2018] say:

First, please don't use the term "Accounts" and "profiles" synonomously. The two are not the same. You can have multiple email accounts within a single profile which from your description, is the case - with the Yahoo email account being the default email account. Similarly, data files are also not the same as an Email Account.

What publisher is usng is the "default email address" when the mrege is created.

Either change the default "email" account to be whatever you want or "copy" your existing profile saving it as another profile (making the other email account the default for that profile) - whichever makes it easier for you to remember what is going on.

Which profile will be used via Outlook (whether opened via the UI or programmatically) will be governed by what is selected under the option is checked for "Always use this profile"

Publisher should allow you to select contacts from any contact folder within your profile regardless of what email account is set as the default but the email address used to designated which email addess is used as the "from" address is controlled by the default email account.

ikuri say:

'this seems to apply to my Microsoft Account'
If an e-mail address is showing on the Sign-In/Lock screen it's because of the Microsoft account.

If you don't want to show the e-mail switch off the 'Show account details (e.g. email address) on sign in screen' option.  It's in Settings > Accounts > Sign-in options under Privacy.

If all you wants is to have the address not show.  That would work.  Though it wont remove it.

For that you need to follow the guide above and create a new alias (Using the e-mail you want), then make it primary.  It will warn you at the top of the page afterwards about taking up to 48 hours for it to take effect and some other things.

Once this is done it would be possible to remove the old e-mail address here as well.  Do so with caution.

=========================

Quick steps.  Ultimately the same as the linked guide.
-----
Use Settings > Accounts > Your info > Manage my Microsoft account to open the account.microsoft.com website.  Logged into the account associated with Windows 10.

On that site you want Your Info > Manage how you sign in to Microsoft.  You will have to sign in again and the next page is one you want.  Look at the Account alias section near the top.

Select Add e-mail address and add the address you want to switch too.  You will need to confirm it.

Once added you will see a Make primary link.  Click and confirm.  Then read the message that will appear at the top once it reloads (the 48 hours message).
-----
If you wish to remove the old address all you need to do is click Remove and confirm it after reading the warning it will give you.

The change sign-in preferences link further down will allow you to stop the old address from being able to sign in if you don't want it removed.

=========================

If you want a screenshot of the login screen btw you'll have to use a Camera.  Windows wont screen shot the sign-in/lock screen.
