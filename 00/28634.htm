Leon

james yan msft say:

Hi Leon,

We can import the Excel spreadsheet to SharePoint as a list and then create a lookup column which get data from that list. Below are the detailed steps:

1. In IE, go to your site > Settings icon > Add an app > in the second page, click Import Spreadsheet.

2. Give a name to the list, click the Browse button > select the Excel file which contains the name column > click Import.

3. In the pop-up page, select Range of cells in Range Type drop down list, click the - button under Select Range, select the cells contain names and click the following button then click Import.

4. Go to Site settings > Site columns > Create > type a column name > select Lookup (information already on this site).

5. Scroll down to Get information from, select the list we just created using Import Spreadsheet > OK.

6. Go to the list where you want to add this column, click LIST > List Settings > Add from existing site columns > select Custom Columns in Select site columns from:, select the column we created in step 4 > OK.

7. Now when we create an item in this list, we can select the names from the column we added.

Regards,
James

How to stop Internet Explorer from automatically launcing or opening - Microsoft Community

luis-alejandro dorta say:

How to stop Internet Explorer from automatically launcing or opening at the start up of your PC/Laptop

This situation has occurred in many cases when I've had to reinstall/reset Windows 10

  1. Open Internet Explorer (IE) ->-> Tools ->-> Internet options (last menu item at the bottom of the "Tools" dropdown  
  2. Under the first tab "General", look for the "Tabs" section that reads "Change how webpages are displayed in tabs" where you will click the grey "Tabs" box.
  3. This will open the "Tabbed Browsing Settings" section.  Look for the section that reads "When a new tab is opened, open:"  where you will click the dropdown menu to select "The new tab page" option.
  4. Next, look for the section that reads "Open links from other programs in:" to select the option (2nd radio button) that reads "A new tab in the current window", click OK.
  5. In the Internet option window, Click on Apply > OK.
  6. Close any/all IE session if applicable, then restart your PC/Laptop in order to truly test the changes just made.

Please note that the steps above are not related to "Turning a feature on/off" (IE) which is done by way of the following:

  1. Go to Control Panel.

  2. Click/tap on Programs and Features, (look towards left/upper area to click/tap on "Turn Windows features on or off".

  3. Clear the check box for Internet Explorer to turn it off (this will turn Internet Explorer off completely, thus should not be mistaken for deleting the  feature permanently) until you follow the same steps in revers to "re-clicking Internet Explorer to turn it back on".

  4. Click/tap OK, and when prompted, restart your PC.
