                        net start cryptSvc

                        net start bits

                        net start msiserver

Now you may try running the Windows Updates and check if the above steps resolve the issue.

rich007a say:

Look at the ChartObject's .Chart property:

 

    Dim cht As Chart
   
    Set cht = ActiveSheet.ChartObjects("Chart 1").Chart
   
    With cht.PlotArea.Format.Fill
        .Visible = msoTrue
        .ForeColor.RGB = RGB(221, 221, 221)
        .Transparency = 0
        .Solid
    End With
 
Hope that helps.
