Set As Default Shape applies only to the current document. In order to make the change apply to all new documents, you must make the change in the Normal template itself, which means that you must open Normal.dotm for editing. The easiest way to do this is as follows:

 

In Word, press Alt+F11; the Visual Basic Editor opens. Press Ctrl+G to activate the Immediate window. In that window, type

     NormalTemplate.OpenAsDocument

and press Enter. Normal opens, but note that it may not be on top of all windows. You can close the Visual Basic Editor. Make the desired changes to the Normal file, and then save and close it. If the changes you have made have not been recognized as changes (the Undo button is not activated), press Spacebar, then Backspace to "dirty" the file, thus ensuring that the template will actually be saved.

If you are syncing your contacts, tasks, and appointments between Outlook on your computer and your phone, then you probably won't
You can't have more than one ActiveSync Exchange Server type of syncing relationship between your phone, but syncing to Outlook on your computer is not the same thing as an ActiveSync Exchange Server type of syncing.  When you set up an ActiveSync Exchange Server type of sync, you sync the device via the internet to a cloud-based service (like a Windows Live account) or to a corporate Exchange server.  When you set up a local Activesync connection, you use a wire connection (via USB cord) to sync the data to Outlook on your computer.

Using third party backup software is not the same thing as syncing data.  You will install the third party software to your phone, then you will use the third party application to back up the contents of your phone to the storage card that you use with the device.  You won't use Activesync and you won't sync anything to your computer when you back up the device, unless you decided to "sync" the backup file to your computer to serve as a "backup of your backup"  You can't do much with the backup file that gets created when you back up your phone other than restore the data in the file to your windows phone.
