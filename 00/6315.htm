I suspect in C5 you have the number 128697.  In column H, I suspect you have the string value "128697"

If you haven't changed the default formatting, then the number 128697 would be right justified and the string value "128697" would be left justified  (double quotes are added to represent a string - the value in the cell would probably not have the double quotes but would still be stored as a string).

andreas killer say:

Open all Workbooks in that folder and execute the macro below.

Andreas.

Option Explicit

Sub Test()
  Dim Wb As Workbook, Wres As Workbook
  Dim Ws As Worksheet
  Dim Dest As Range, Source As Range
  Set Wres = Workbooks.Add
  For Each Wb In Workbooks
    If Wb.Windows(1).Visible And Wb.Name <> Wres.Name Then
      For Each Ws In Wb.Sheets
        Set Dest = SheetLastCell(Wres.Sheets(1))
        Set Dest = Dest.Parent.Cells(Dest.Row + 1, 1)
        Ws.UsedRange.Copy Dest
      Next
    End If
  Next
End Sub

Function SheetLastCell(Optional Ws As Worksheet) As Range
  'Liefert die letzte gefüllte Zelle (Schnittpunkt Zeile/Spalte) der Tabelle
  Dim R As Range, c As Range
  If Ws Is Nothing Then Set Ws = ActiveSheet
  On Error Resume Next
  Set R = Ws.Cells.SpecialCells(xlCellTypeLastCell)
  On Error GoTo 0
  If R Is Nothing Then
    'Tabelle ist geschützt
    Set R = Ws.Cells(1, 1)
    GoTo FindCell
  End If
  If R.Count > 1 Then
    'Specialcells geht nicht in einem Event das durch ein Makro ausgelöst wird
    If Val(Application.Version) < 10 Then
      Set R = Ws.Cells(1, 1)
      GoTo FindCell
    Else
      'Find geht ebenfalls nicht in XL2000
      Set R = Ws.UsedRange
      Set SheetLastCell = R.Cells(R.Cells.Count)
      Exit Function
    End If
  End If
  If IsEmpty(R) And Not R.Address = Cells(1, 1).Address Then
FindCell:
    Set c = Ws.Cells.Find("*", After:=R, SearchOrder:=xlByColumns, SearchDirection:=xlPrevious)
    If c Is Nothing Then
      Set SheetLastCell = Ws.Cells(1, 1)
    Else
      Set R = Ws.Cells.Find("*", After:=R, SearchOrder:=xlByRows, SearchDirection:=xlPrevious)
      Set SheetLastCell = Ws.Cells(R.Row, c.Column)
    End If
  Else
    Set SheetLastCell = R
  End If
End Function

Office Home and Student 2010 has an install of Office Professional trial (including Access, Publisher and Outlook)<o:p></o:p>

 <o:p></o:p>

When you put in the Home and Student disk and start the install you will be directed to enter your 25 character product key.  Two screens later you will see the following boxes (Upgrade and Customize) and below those boxes you will see a checkbox (already checked) which states "Include a free trial of Microsoft Office Professional 2010."     You MUST uncheck the box or you will install the trial.<o:p></o:p>

 <o:p></o:p>

To uninstall the trial software installed with your retail H & S product, start your activated Word 2010 program.  Click File, Help and you should see a box that says Licensing Operations.  Click it.  You should see two items, purchase Professional Product or Remove Trial.  Click to Remove Trial. Answer the Yes to the "You are about to remove your trial product..." to remove the trial. Once completed ("Programs have been removed...") click OK to reboot the computer to finish the uninstall.  You should have the 4 programs left that were included in H & S.

 

Once you remove the trial software attempt to reinstall your Outlook 2007.  If it runs you are home free.  If it fails you need to totally uninstall Office 2010, reboot the computer, install your older Outlook software using a custom install, update Office 2007 fully and then reinstall Office 2010 but ensure that you remove the check mark prior to clicking Upgrade.<o:p></o:p>
