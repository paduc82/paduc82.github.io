net start TBS

net stop TBS

To change Startup Type:

Automatic:
REG add "HKLM\SYSTEM\CurrentControlSet\services\TBS" /v Start /t REG_DWORD /d 2 /f

Manual:
REG add "HKLM\SYSTEM\CurrentControlSet\services\TBS" /v Start /t REG_DWORD /d 3 /f

Disabled:
REG add "HKLM\SYSTEM\CurrentControlSet\services\TBS" /v Start /t REG_DWORD /d 4 /f

Automatic (Delayed Start):
REG add "HKLM\SYSTEM\CurrentControlSet\services\TBS" /v Start /t REG_DWORD /d 2 /f

Note:

  • When you change to Automatic (Delayed Start) a new key DelayedAutostart is created with value 1.
    REG add "HKLM\SYSTEM\CurrentControlSet\services\TBS" /v DelayedAutostart /t REG_DWORD /d 1 /f

    When you change toAutomatic from Automatic (Delayed Start), DelayedAutostart change value to 0.
    REG add "HKLM\SYSTEM\CurrentControlSet\services\TBS" /v DelayedAutostart /t REG_DWORD /d 0 /f

sraab say: I resolved my problem.  The clue was in the Event Log entry.  The permissions on the C:\Users\Default folder were messed up.
