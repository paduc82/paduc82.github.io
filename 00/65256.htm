After a lot of trouble with this issue myself I finally found the culprit - it is the IDE ATA/ATAPI Controller driver. 

Windows by default loads a driver called Standard SATA AHCI Controller

If you have this device driver loaded, that is the problem:

  1. Right click this device and choose properties, then click the Details Tab
  2. Change the Property drop down to Hardware Id's
  3. Right click the top item in the list and select copy
  4. Open Google and paste the device id
  5. This will tell you the exact make and model of your chipset
  6. Download and install the correct SATA/AHCI chipset controller
  7. Reboot your system

Hi slic,

Thanks for your response. If the computer is a Windows 10 device, no software would be needed to format the drive. Here's how to do so:

  1. Open File Explorer on your Windows 10 PC
  2. Select This PC
  3. Find the External Drive you wish to format
  4. Right Click the drive and Select Format
  5. Click the File system drop-down
  6. Select NTFS
  7. Click Start to being formatting the drive

I hope this information helps, thanks again!
