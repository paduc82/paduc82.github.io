jeeped say:
If I am interpreting this correctly, there are worksheets named GrantB, SothiaC, NathonD and JonD. You want the name of the worksheet to determine which column in RegularHours and Overtime to put the values from N31 and P31.

Sub mcr_Xfer_Hours()
    Dim emp As String
    emp = ActiveSheet.Name
    If LCase(emp) = "regularhours" Or LCase(emp) = "overtime" Then Exit Sub
    With Sheets("RegularHours")
        If IsError(Application.Match(emp, .Rows(3), 0)) Then _
            .Cells(3, Columns.Count).End(xlToLeft).Offset(0, 1) = emp
        .Cells(5, Application.Match(emp, .Rows(3), 0)) = _
          ActiveSheet.Range("P31").Value
    End With
    With Sheets("Overtime")
        If IsError(Application.Match(emp, .Rows(3), 0)) Then _
            .Cells(3, Columns.Count).End(xlToLeft).Offset(0, 1) = emp
        .Cells(5, Application.Match(emp, .Rows(3), 0)) = _
          ActiveSheet.Range("N31").Value
    End With
End Sub

That macro will also add the employee (aka worksheet name) to the list in row 3 if it is not found.

Edit: added error control, to exit if ActiveSheet is either RegularHours or Overtime.
