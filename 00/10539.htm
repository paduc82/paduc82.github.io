bob jones aka: cybertaz say:
For no more requirements than you've stated both issues can be easily accomplished by the use of Paragraph Styles to handle the formatting. Hopefully the template you have provides the Styles you're expected to use - does the template do nothing more than define the paper size? If starting each chapter at the top of a page is a requirement I would expect to find the Heading (chapter title) Styles defined accordingly. If they haven't been fully defined you should be able to modify them yourself. Otherwise, you can create your own. If you aren't familiar with the use of Styles it would be well worth your while to review the information about them in Word Help. Other topics are available, but start here:

Apply, change, create, or delete a style

The Heading Style used for the chapter titles needs to include the Page Break Before paragraph formatting property. It's available in in Format> Paragraph> Line & Page Breaks dialog. That will cause each chapter title to start at the top of a new page.

The other formatting of the text you apply it to is also controlled by the Style. That includes font, font size, line spacing, spacing between paragraphs, paragraph borders, etc. Literally any formatting attributes can be included, as Word is a styles-based program. I can't imagine wanting any chapter titles to start at a different vertical position than the others, though. (Is that what you mean by "...lower the title"?) Normally I'd expect to have all chapter titles start at the top margin or a uniform distance below it. To force the title lower than the margin include the Space Before attribute in the Style.

As for changes, you need only modify the Style you've used. I.e., If you modify the font size used by the Style any content to which that Style has been applied will be reformatted automatically. Of course, any text you go on to apply the Style to will take on the most current attributes specified by the Style.
