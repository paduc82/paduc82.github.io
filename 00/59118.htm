Hello John,

Thank you for posting your query in Microsoft Community Forum.

Were you using 3rd party software to get the count of opened applications or Microsoft?

As you use Windows 8, you may have noticed the total number of Apps (in count) and below it, total number of Background processes running under Task Manager.

Under Task Manager of Windows 7, you will see the total number Processes running (in count). However, the applications do not show in numbers.

You may run Process Monitor tool to check if that is what you are looking for.
