ossiemac say:

Not sure why you would want to add buttons to do what clicking on the tabs at the top of the multipage control will do but anyway the following code will move to the Next of Previous. Note that I have set up the subs to do the work as called subs and then each button should call these subs.

The following 2 subs are examples of calling the subs to do the processing.

Private Sub CommandButton4_Click()
    'Assumes CommandButton4 is for Previous page
    Call PreviousPage
End Sub

'Following sub is example of moving to the next page
Private Sub CommandButton5_Click()
    'Assumes CommandButton5 is for next page
    Call NextPage
End Sub

The following 2 subs are the called subs to do the processing.

'Following sub is called to move to the previous page
Sub PreviousPage()
    'Note that pages count from Zero to one less than total pages (tabs)
    If Me.MultiPage1.Value > 0 Then     'Zero is the first page value
        Me.MultiPage1.Value = Me.MultiPage1.Value - 1
    Else
        MsgBox "Already at first page."
    End If
End Sub

'Following sub is called to move to the next page
Sub NextPage()
   
    'Note that pages count from Zero to one less than total pages (tabs)
    'In the following line Pages.Count - 1 is the last page number _
     so need to subtract 2 for the 2nd last page.
    If Me.MultiPage1.Value <= Me.MultiPage1.Pages.Count - 2 Then
        Me.MultiPage1.Value = Me.MultiPage1.Value + 1
    Else
        MsgBox "Already at last page."
    End If
End Sub

Hi mmpexchange,

Thank you for contributing to the forum.

This is the information regarding changing ringtones on the phone from windowsphone.com

To change any other ringtone or alert

  1. In the App list, tap Settings > Ringtones + sounds.

  2. Tap the type of ringtone or alert you want to change, then choose the sound you want to use.

You can get more information from the link below:

http://www.windowsphone.com/en-gb/how-to/wp8/settings-and-personalization/ringtones-sounds-and-volume
