We're getting there. Highlights are different from background colors. Highlights are set by  clicking on the Home>Font>Highlight tool and choosing a color. Daniel and Richard are correct. I just want to emphasize:

* Dates are 8-byte values stored in some complicated structure we don't need to remember.

* AT THE LAST MOMENT POSSIBLE we format the date value to the format we want, by setting the Format property or perhaps by using the Format function.

In your query you have violated rule2 but for good reasons: you need to get the count per hour.

The Hour(CheckoutTime) value is going to be 0-23. If you and your boss (:-)) want this to be US time rather than military time, you have to realize you are now "casting" an integer number to a string.

0 => "12 am"

1 => "1 am"

2 => "2 am"

12 => "12 pm"

13 => "1 pm"

etc.

If you would like a vba solution, then one possible solution would be to assign the macro "SumToClipboard" to the quick access toolbar.

Sub SumToClipboard()
  If TypeName(Selection) <> "Range" Then Exit Sub
  ToClipBoard (WorksheetFunction.Sum(Selection))
End Sub


Public Function ToClipBoard(s)
'/  In VBA, set Tools / Reference to the Microsoft Forms 2.0 Object Library.
'/  http://www.cpearson.com/excel/Clipboard.aspx
    Dim DataObj As New MSForms.DataObject
    DataObj.SetText s
    DataObj.PutInClipboard
End Function
