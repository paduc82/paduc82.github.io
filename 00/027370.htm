stefan blom say:

If you want the company information in the first header and the document name on the other pages, all you have to do is enable the "Different first page" on the Header & Footer Tools Design tab (to activate it, double-click the header).

You can use the Next and Previous buttons on the Header & Footer Tools Design tab to move between headers (or between footers).

Note that if you are initially working in a one-page document you can't see all headers unless you add a temporary page break.

Second question first :-) Creating a document using Publishing Layout View in Word 2011 is not the same as .pub files created by MS Publisher on a PC. Documents created using Publishing Layout View are still Word documents (.docx). MS Publisher is a Windows-only program & AFAIK there is no other program which can access .pub documents [Mac or Windows].

As to the first issue, it's impossible to provide any specific explanation without knowing how the "nice formats" are ruined & how they were attained. There are numerous variables involved, some of which are related to Word, some of which are related to differences in OS, some of which are related to print drivers, some of which are related to Fonts... The Custom dictionaries for Office 2011 are stored in:
