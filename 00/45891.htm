aileen alf say:

May we know which version of Windows you're on? If you're using Windows 10 Version 1511 and earlier, your notes are stored in the stickynotes.snt database file located in the %AppData%\Microsoft\Sticky Notes folder. If you're using Windows 10 Anniversary Update Version 1607 and later, your notes are now stored in the plum.sqlite database file located in the %LocalAppData%\Packages\Microsoft.MicrosoftStickyNotes_8wekyb3d8bbwe\LocalState folder. To view the folder, press Windows key + R and copy and paste the file location.

Since Sticky Notes has become a trusted Windows Store app, its location is within the apps folder which is under the Packages folder. Copy and paste the folder location below into the address bar of File Explorer, and press Enter (make sure to change YourUsername with your user name):

C:\Window.old\Users\YourUsername\AppData\Local\Packages\Microsoft.MicrosoftStickyNotes_8wekyb3d8bbwe

Copy everything within that folder and paste it here:

C:\Users\YourUsername\AppData\Local\Packages\Microsoft.MicrosoftStickyNotes_8wekyb3d8bbwe

For your future reference, you can backup and restore your notes depending on the Windows version you're on. For Windows 10 Anniversary Update Version 1607 and later, you can follow these steps:

  1. To backup your notes, sign in to the user account you want to backup Sticky Notes for.
  2. Copy and paste the folder location below into the address bar of File Explorer, and press Enter:

    %LocalAppData%\Packages\Microsoft.MicrosoftStickyNotes_8wekyb3d8bbwe\LocalState

  3. Copy the plum.sqlite file to where you like as a backup of your Sticky Notes.
  4. To restore your notes, sign in to the user account you want to restore Sticky Notes for.
  5. Close all instances of Sticky Notes (Microsoft.StickyNotes.exe) in Task Manager.
  6. Copy and paste the folder location below into the address bar of File Explorer, and press Enter:

    %LocalAppData%\Packages\Microsoft.MicrosoftStickyNotes_8wekyb3d8bbwe\LocalState

  7. Copy and paste the backed up plum.sqlite file from step 3 into the LocalState folder.
  8. Click on Replace the file in the destination.
  9. Open the Sticky Notes app to have your restored Sticky Notes appear.

We hope this information is helpful. If you have further questions, don't hesitate to get back to us.

CUSTOMIZE KEYBOARD - Microsoft Community

jaime mac1961 say:

CUSTOMIZE KEYBOARD

In windows 10 is there a way to customize a few keys as opposed to switching between differenent language keyboards?

For example : pressing the letter plus "ALT" to get the letter I want.

a. user say:

Hi Jaime,

 

Thank you for posting your query on Microsoft Community.

Please follow below steps and check if it helps.

Step 1:

  1. Bring the Start Menu on your screen and you can see a gear symbol saying Settings. Click on Settings.
  2. After the Settings app window gets spread on your screen, choose Ease of Access category.
  3. Now look at the left pane of Ease of Access and you can see many areas of settings are present. Click Keyboard.    
  4. The right side of the Keyboard exhibits you many settings with a slider to easily handle them. So, let us see each of them separately!

 

Step 2:

On-Screen Keyboard

  1. Turns on the On-screen keyboard: If you prefer to use the on-screen keyboard, you can turn on the toggle. When you turn on the on-screen keyboard, wait for couple of seconds and then you can see the keyboard on your display screen. Take a look at the below screenshot. Switch off the slider and it will go off from the screen.

Step 3:

Sticky Keys.

  1. Press one key at a time for keyboard shortcuts: This feature is favorite among those who love to use keyboard shortcuts for every small activities they do. Usually, to copy a content you use two keyboard keys simultaneously (Ctrl and C). But after you enable this option, click Ctrl key, release it, then click C key and your selected stuff will be copied. There are four modifier keys that become “sticky” when you turn on this feature. They are Shift, Ctrl, alt and Windows keys. Turning it on makes your work easy when two keys are far from each other and hitting them simultaneously becomes irritated.
  2. Turn on Sticky keys when Shift is pressed five times: Instead on turning sticky key on from the above way, you can enable it by pressing Shift key five times continuously. After doing so, you can see a dialog box waiting for your confirmation to activate the feature.
  3. Lock modifier key when pressed twice in a row: This is the most useful option. By default it is turned on. When this feature is enabled you can carry out many copy and paste work. Just press Ctrl key twice and then use “C” or “V” keys the number of times to copy or paste number of items.
  4. Turn Off Sticky Keys by releasing a modifier key in combination with another key: To disable the sticky keys you can jointly press a modifier key along with any other key.
  5. Play sound when modifier key is latched, locked and released: You can hear a sound when you press the modifier key, when its gets locked and also when it is released. If you don’t prefer to hear the sound, disable this option.
  6. Display the Sticky Keys icon on taskbar: With this feature enabled you can see a Sticky Keys icon on the system tray of taskbar.

Step 4:

Toggle Keys.

  1. Hear a tone when you press Caps Lock, Num Lock and Scroll Lock: By mistake you can press caps lock or num lock or scroll lock key and later on realize it. So to avoid the confusion, you can turn this feature on and hear a beep tone when you press any of these keys.
  2. Turn on Toggle Keys by holding the NUM lock key for 5 seconds: If this feature is enabled, you can turn on the Toggle Keys by keep holding the Num lock key for 5 seconds of time.

Step 5:

Filter Keys

  1. Ignore or slowdown brief or repeated keystrokes and adjust keyboard repeat rates: If you are scared of pressing any keys number of times, enable this feature which ignores the activity. This is really useful when you kids at home and who might press any keys number of times and that might hamper on your work.
  2. Turn on Filter key when Shift is pressed for 8 seconds: Keep hold the Shift key for 8 seconds and you can see Filter Key is turned on.
  3. Beep when keys are pressed or accepted: You can hear a beep sound when a key is pressed and accepted by the system.
  4. Display the Filter Key icon on Taskbar: To know if the Filter Keys are on, enable this toggle to On and see its icon on Taskbar.
  5. Enable Slow Keys: Turn this feature on if you want a delay between hitting a key and the action taking place on your computer. After you turn it on, you can see a drop-down menu with a different time slots. Choose a time after whose expiration the computer will accept your keystroke.
  6. Enable Repeat Keys: Turn this feature on if you want to have the repeated keys. This again provides you two different time slot drop-down menus. First is to select the time for computer to wit for the first repeated keystroke and the second one is to how long the computer will wait for the subsequent repeated keystrokes.

Step 6:

Other Settings

  1. Enable Shortcut underlines: To underline the hotkeys by default, you can enable this feature.
