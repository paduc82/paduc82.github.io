ken sheridan say: You should be able to reference the combo box in qryRegister, but you'll need to test for it being NULL to return all rows initially:

WHERE (Requester = Forms!frmRegister!cboSCNumReq
OR Forms!frmRegister!cboSCNumReq IS NULL)

You'll find an example of this in the file ComboDemo2K.zip in my public databases folder at:

https://skydrive.live.com/?cid=44CC60D7FEA42912&id=44CC60D7FEA42912!169

This includes a form demonstrating how to drill down via three correlated combo boxes to progressively restrict the rows returned in the form.  If you take a look at the form's RecordSource property you'll see that it is a query which references the combo boxes in the above way.  To examine the property zoom in on it in the form's properties sheet by right clicking the property and selecting Zoom from the shortcut menu.  Don't click on the build button (the one with 3 dots) or you'll get it in query design view where it will not make a lot of sense.

Similarly, when amending your query do so in SQL view, and save it in SQL view.  If you do so in design view Access will move things around.  The underlying logic should be the same and it should work, but the logic will be far less apparent.

Adding up a list of elapsed times in Excel where the total is greater - Microsoft Community

compucaf say:

Adding up a list of elapsed times in Excel where the total is greater than 24 hours

I need to create a spreadshett where I have some elapsed times. I have a Start Time & a Finish time. I have successfully created a formula that adds them all up over a month, and that will display the answer as Hours and minutes, with the hours greater than 24.
joeu2004 say: CompuCaf wrote:
I need to create a spreadshett where I have some elapsed times. I have a Start Time & a Finish time. I have successfully created a formula that adds them all up over a month, and that will display the answer as Hours and minutes, with the hours greater than 24.

It is important that you show how your data are entered, i.e. in what form, and the formulas you use for determining elapsed time and summing them.

You should enter your start and end times in the form hh:mm or hh:mm:ss with or without AM/PM.  If your start and end times might "cross" midnight, you should include the date.  You can format those cells any way you wish.

If your start times are in A1:A30 and your end times are in B1:B30, the elapsed time is simply =B1-A1, for example.

You can compute the individual elapsed times in that manner in C1:C30.  Then the total elapsed time is =SUM(C1:C30).

Alternatively, you can compute the total elapsed time directly using =SUMPRODUCT(B1:B30-A1:A30).

If you did all that, then format the elapsed time (column C) and the SUM or SUMPRODUCT formulas as Custom [h]:mm.
