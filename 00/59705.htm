https: //support.microsoft.com/en-us/help/12415 / ...

Windows Key + i to invoke Configuration> Update and security> Click View history of installed updates. These KB should be from the following list. Do not uninstall any other KB

Uninstall

Block

When you run this troubleshooter, it will search for available updates and list them
