NULL values - Microsoft Community

leng56 say:

NULL values

I am creating charts. When I set a cell to "", it shows up as a 0 on the chart. If I use VB and set the cell to NULL, then the chart is not a zero value.  How can I set a NULL value using a formula rather than VB?
bill manville say:

The treatment of NA() is dependent on an option setting.

I can't find where the option can be changed in the 2007 user interface (it was on Tools > Options > Chart prior to 2007), but in VBA you can set the Chart.DisplayBlanksAs property to xlInterpolated, xlNotPlotted or xlZero.  Setting that property of the chart to xlInterpolated and setting the cell to NA() will give the desired result.

To set it: select the chart; Alt+F11 to the visual basic editor; Ctrl+G to the Immediate window and enter: ActiveChart.DisplayBlanksAs=xlInterpolated.

Further experimenting...
in both 2003 and 2007:
DisplayBlanksAs = xlZero               gives 0 for each of blank cell, ="" and =NA()
DisplayBlanksAs = xlInterpolated    gives 0 for ="" and interpolation for blank cell and for =NA()
DisplayBlanksAs = xlNotPlotted       gives 0 for ="", a break in the line for blank cell, and interpolation for =NA()

 [Later] with the help of MVP Andy Pope: the user interface option in 2007 is on the Select Data dialog, behind the Hidden and Empty Cells button.

Your computer is a member of a domain

    Collapse this imageExpand this image

    Collapse this imageExpand this image
    If you're prompted for an administrator password or confirmation, type the password or provide confirmation.

Your computer is in a workgroup
