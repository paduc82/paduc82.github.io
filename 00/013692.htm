john korchok say:

While it's true that Word for Windows has a Detect language automatically feature, it's also a common piece of advice in the Word for Windows forums to turn that option off, since it doesn't work very well and frequently detects the wrong language.

If your documents have mixed languages, a better idea is to create a separate set of styles in your template, with the only variation being that the alternate styles are set to the language of your preference. In real life this works much better, allowing you to select the text in a language and applying the appropriate style.

If you are creating new documents that are 100% in a different language, it works better to set the OS X language first.  Then in Word, choose Edit>Select All and set your document language to the same setting as OS X.

If you still have problem, use Tools>Language, select the language and click on the Default button to temporarily set Word's default language to your preference.

stefan blom say:

Page colors only print if you select the "Print background colors and images" at File tab | Options | Display category.

The reason that the page border doesn't print is probably that it is in the nonprinting area of the document (as determined by the printer driver). On the Page Border tab of the Borders and Shading dialog box, click Options. Try changing the "Margins" settings for your borders. Note that you may have to change the "Measure from" setting as well.

g north say:

In answer to the first part of the question from the Word help.

A background or a page color is primarily used to create a more interesting background for online viewing.

This means that the background will never print from Word.  However you have the ability to save as a PDF (Online viewing) so printing the PDF will work. 

The best alternative is probably set your picture as a Picture Watermark. For a page colour just use a rectangle with the fill colour you want to use and save it as a Picture (right mouse button) to use as your Picture Watermark. 

As for the border problem, when it prints three sided are the left and right borders the same distance from the edges of the paper?
