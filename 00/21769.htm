Repair your windows apps & store with Powershell command

1. Click start, scroll down to W
2. Arrow down the Windows Powershell folder
3. Right-click Windows Powershell, More, Run as Administrator
4. at PS > prompt copy & paste this command & press enter

set-ExecutionPolicy RemoteSigned

5. Type Y and press enter
6. at PS > prompt copy & paste this command line & press enter

Get-AppXPackage -AllUsers | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register "$($_.InstallLocation)\AppxManifest.xml"}

You will get a blue background and yellow writing as it works.

You might get a few red writing errors at the end, but ignore them.

7. exit Powershell
8. Reboot your computer

9. Login and try opening the windows store.

Right-click start, command Prompt (admin)

net user /add [username] [password] = without the [ ]
net localgroup administrators [username] /add = without the [ ]

Type these commands and press enter for a user called sam and password super.

You can change the name & password to anything you want

net user /add sam super

net localgroup administrators sam /add

Log out and login as sam & password super
