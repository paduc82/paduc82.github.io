Thanks for your updates.

 

The Office 365 group site is slightly different from the team site. When you create a new group, you can’t use the custom site template to create the group site and the group site is created automatically. So if you want to use the group features based on Exchange Online, you may need to manually upload some content template like a list template via the way I recommended above. However, for creating subsites in the group site, you can use the custom site template including the lists and columns you want, to avoid some issues, I suggest you create the same lists and columns as the team site template you use in the group site and then navigate to the group site>Site Content>Site Settings>save it as template first. Then you can navigate to the group site>Site Content>Subsite>Custom>use the group site template to create subsites. The content such as lists and site columns in the group site template will appear in the created subsites.
