guessed say:

To include the chapter numbers in the page number fields in the TOC you MUST use the built-in heading names. This means that you need to make use of the "Heading 1" style for your headings.

If you were using a French version of Word, I think the Heading 1 style name would automatically show up as Titre 1 if that is French for Heading.

But because you are using an English version of Word, you will need to add an alias of this on the actual Heading 1 style.  You do this by renaming the Heading 1 style to "Heading 1,Titre 1". You can then hide the built-in style name and just show the alias name by going to the Style Options (in the bottom of the Styles Pane) and ticking the option to "Hide built-in name when alternate name exists".

Note that you wont be able to add an alias with exactly the same name as an already existing name - so you can't make the alias Titre_1 unless you remove/rename that style first. If Titre_1 is based on "Heading 1", deleting the style will automatically set those paragraphs using it to Heading 1 so that simplifies the effort of restyling the paragraphs.

Flagging a cell when the specified date falls within a quarter. - Microsoft Community

reesech say:

Flagging a cell when the specified date falls within a quarter.

Hi,

I am afraid I do not have any formulas to present as a background for this as to be honest I have no idea where to start!

I have been asked to flag a cell red whenever it's due date aligns with a business quarter, the problem being that they do not wish to manually change the due dates in the cell of origin.

For example, if a cell has todays date, and it needs to be flagged every 15 months, but at the start of the business quarter.

15 months from today would be 09/09/2018 and the date the flag would need to be raised would be the start of that quarter, in that example 01/07/2018. The following flag would be for 09/12/2019, flagging on the 01/10/2019 and so on.

Is there any way to approach this without manually entering the new dates each year?

Any guidance would be greatly appreciated!

keratl say:

A1=today()

A2= DATE(YEAR(A1),MONTH(A1)+15,DAY(A1))  'gives you the 15 months. Note that the order of your DATE variables may differ, my computer is set up based on USA date format

A3=MONTH(A2)  'gives us the +15 month we are working with

A4=INDEX({1,1,1,2,2,2,3,3,3,4,4,4},A3) 'for just 12 values, this is an easy way to get the quarter

                                                           'this assumes your FY is the same as the CY

A5= INDEX({1,4,7,10},A4)  'gives you the first month of the quarter

A6= DATE(YEAR(A2),A5,1) gives you the date of the first of the month of that quarter

You can use helper columns (or rows) to check the formulas and adjust them if needed; once they are working to your satisfaction you can combine them by replacing each reference with the formula in the source cell:
