suzanne s. barnhill say:

If the customizations were saved in the Normal template (which they would be by default), then it would appear that your modified version of the template has been lost. Perhaps some add-in overwrote the file or deleted it. There's a slight chance of recovering it.

  1. Navigate to the folder where user templates are stored (by default, this will be C:\Users\<Profile Name>\AppData\Roaming\Microsoft\Templates) and see what the date of Normal.dotm is.
  2. If it is more recent than your most recent intentional changes, then that's the problem.
  3. If you have "Always create backup copy" enabled at File | Options | Advanced: Save, then you should find a file called "Backup of Normal.wbk."
  4. Open that file; if it contains your customizations, then close it.
  5. Delete or rename Normal.dotm.
  6. Rename the backup file to "Normal.dotm," ignoring any warning messages about changing file extensions.
  7. Reopen Word, and you should have your customizations back.

If you don't have "Always create backup copy" enabled, you can see why it might be a good idea. This is also why you should have "Prompt before saving Normal template" (in the same location) enabled as well. Provided you are alert and don't say yes to the prompt when you have not personally made changes that need to be saved, your template should not be overwritten without your permission.

Even if you don't have a backup of Normal.dotm, you might check the Recycle Bin to see whether there might be a deleted copy there.

I am looking to see if there is a formula that can be used in Excel that will auto hide 3-4 rows based on a row above. For example. I have a sheet that states "Direct or Transferred" based on the entry in the column by typing direct or transferred, if the answer above is Transferred, hide the following 3 rows (26-29). If it is direct, make them appear. Not sure this is possible without a script or not. Thanks.
bernard liengme say:

Formulas can only return values to the cells containing the formulas.

You would need a Worksheet Change VBA subroutine to do what you asked about.

Copy this code

Private Sub Worksheet_Change(ByVal Target As Range)
  If Intersect(Target, Range("C20")) Is Nothing Or Target.Cells.Count > 1 Then Exit Sub
  mytext = UCase(Target.Value)
  If mytext = "TRANSFERRED" Then
    Rows("26:29").EntireRow.Hidden = True
  ElseIf mytext = "DIRECT" Then
   Rows("26:29").EntireRow.Hidden = False
  End If
End Sub

Right click the tab for the worksheet in question; select View Code; past the code in the white space of the resulting module

Change Range("C20") to reference the cell in which you will enter direct or Transferred

If you just upgraded to Outlook.com interface, the forwarding email that you have set up on Hotmail will still be retained.
