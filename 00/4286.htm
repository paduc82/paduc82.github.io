a. user say:

Hello KarthikNandakumar:

 

Thank you for posting to the forum regarding your Web browsing issue.  I have had a hard time finding any information on the error you are reporting.  The only thing close I found was the following article, http://support.microsoft.com/kb/235567.  This seems to indicate this is an issue when trying to connect to a remote server.  Please review this article to see if it matches up with what you experience.

 

I do have a few general troubleshooting steps to suggest:

  • First, try connecting via alternate means and to alternate networks.  If you are connecting to Wi-Fi when you receive this error, try connecting to a different network.  After doing this try connecting to just cellular data, go to settings > cellular/mobile > turn setting ON.  Once this is set to on, turn Wi-Fi off, tap settings > Wi-Fi > OFF.
  • The next thing to try would be to clear your browser cache.  Go to settings < applications < delete history.  Once you have done this, reboot the phone and try again.
  • Next, soft reset your phone.  If you are able to remove the battery, please do so for about 2 minutes, replace and restart the phone.  If you are unable to remove the battery, consult your user manual or contact Nokia support for the manual steps.

Please let us know if you have any other questions or concerns.
