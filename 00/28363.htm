VLOOKUP #N/A Error - Microsoft Community

terri.reed say:

VLOOKUP #N/A Error

I have been trying for hours to get the following VLookup formula to work - I am trying to pull over data that from another spreadsheet...

=VLOOKUP(C2,'terminals'!A1:D1193,2,0)

 

C2 = 78.DCHE.045896

Data on spreadsheet - "terminals"

A1 = V0758

B1 = 7526359874

C1 = 78.DCHE.045896

D1 = 08/17/12

 

Based on my formula it should be returning 7526357874 but it keeps giving me #N/A errors --- what am I doing wrong.  I have sorted column B in my lookup table in ascending order and I have also tried it by sorting C in ascending and it still doesn't work.  I'm at a loss -- please help!   

kergga say:

vlookup requires that the matching data be in the first column of your range (terminals A1:D1193.

 

You have two choices; you can re-order your columns so the data in column C is the first column in your lookup range, or you can switch to index(match()). Once you switch, you will probably never go back to vlookup.

 

=match(C2,'terminals'!C:C,false)

tells you what row the match is on. Then wrap that in an index to get the corresponding value from any column, regardless of whether it is before or after your lookup column:

=index('terminals'!B:B,match(C2,'terminals'!C:C,false))

 

HTH

How do you easily type a macron in excel?

In Word, I hit "Symbol" then assigned a shortcut key to the ones I wanted.  Excel doe not allow you to assign a shortcut to the symbol.  Why not?  And what's the alternative?
cristin yan say:

Hi,

 

For example, we want to assign a shortcut key (Ctrl+B) to the symbol "@" in Excel.

 

Step 1:

Press Alt+F11 >> Insert >> Module, then paste the following code in the added module.

'#################################

'Callback subroutine when using Ctrl+D.

'#################################

Sub Test()
    Application.SendKeys "@"
End Sub

 

'#############################################

'Register a shortcut key (Ctrl+D) to run the macro Test.

'#############################################

Sub RegisterShortcutKey2Macro()
    Application.MacroOptions Macro:="Test", ShortcutKey:="b"
End Sub

 

Step 2:

Run the macro RegisterShortcutKey2Macro first, and then go back to Sheet1, try to press the registered shortcut key (Ctrl+B), the symbol "@" will show in the active cell.

Cristin

bob jones aka: cybertaz say:
For one thing, your installation of Office 2011 is seriously behind on updates... 14.3.1 is the most recent version. I'd suggest that you first go to Help> Check for Updates to get the software to a current state. Once it's updated run Disk Utility to repair disk permissions then restart your Mac.

If you continue to have the problem please don't hesitate to reply here, but you'll need to provide more detailed descriptive information than what you've supplied so far :-) Be sure to specify;
  • How the "watermark" was created
  • Exactly what steps you're taking
  • What does happen when you try [vague phrases such as "not working" & "unable to" don't disclose anything helpful :-)]
  • Which version of Windows Word you're comparing to
  • Any additional details that may be pertinent

The function of text Watermarks [as a feature of Word] is virtually identical in all versions of the program where the feature is included. The Watermark is inserted into the Header layer as a WordArt object. To change it requires going through either:
  1. The Insert> Watermark dialog to specify different text, or
  2. In the Header, right-clicking the WordArt object to select Edit Text

In both cases, copied content can be pasted into the Watermark/WordArt dialog but I believe only the Command+V keyboard shortcut can be used.
