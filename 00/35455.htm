Method 1: Run the SFC scan

Run the Microsoft Windows Resource Checker tool, also know as "System File Checker" or "sfc". This tool scans and verifies the integrity of all protected system files and replaces incorrect versions with correct versions. Here's how:

1. Press Windows key + X, click Command Prompt (Admin).
2. In the Command Prompt, type the following command, sfc /scannow and then press Enter.
3. Wait until the process completes. This usually doesn’t take longer than 20 minutes with a regular hard disk and

    takes much shorter when you have a fast SSD drive.
4. Check to see if the issue is resolved. If not, restart the system.

If the problem continues, use the Deployment Image Servicing and Management (DISM) tool to fix Windows corruption errors. Here's how:

Method 2: Run the DISM tool

Run the DISM Tool to fix any files that System File Checker can't repair.

1. Right click on Start, click on Command Prompt (Admin).

Type Dism /Online /Cleanup-Image /ScanHealth and press Enter.
Type Dism /Online /Cleanup-Image /CheckHealth and press Enter.
Type Dism /Online /Cleanup-Image /RestoreHealth and press Enter.

2. Restart the computer and check if the issue is resolved.
