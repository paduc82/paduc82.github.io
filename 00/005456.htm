I have a seen a non-array version of the formulas used to make this calendar so I've adapted them to give you the layout requested in your earlier thread - which I assume is your preferred layout?

Assuming you have a full date entered in cell C5, then you can put the following formula into cell C13, and copy it into the other target cells E13, G13 ... O13, and downwards into rows 16, 19 ... 28

=IF(MONTH($C$5)<>MONTH(DATE(YEAR($C$5),MONTH($C$5),1)-WEEKDAY(DATE(YEAR($C$5),MONTH($C$5),1),1)+(ROW($A3)/3-1)*7+(COLUMN(B$1)/2)),"",DATE(YEAR($C$5),MONTH($C$5),1)-WEEKDAY(DATE(YEAR($C$5),MONTH($C$5),1),1)+(ROW($A3)/3-1)*7+(COLUMN(B$1)/2))

jormajack say: Never mind, I figured it out.  If anyone else has this problem, here's the answer;

The 3 asterisks automatically created a border line.  That code (invisible) reproduced itself in the block and paste into the new document, so every time there was a new paragraph in my new document, a new border line appeared (ugly as dirt...)

I went to the very beginning of the text that I had pasted in, then selected that text through the vey end of the entire new document. Then I went to table - border - selected "none" and all the lines disappeared.   THANK GOODNESS.....
