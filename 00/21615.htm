Possible cause:  You need to select the smart art object itself, rather than any element within the smart art. Otherwise, PPT grays out the alignment options. 

As a test, press ESC. 

If a shape within the smart art is selected, ESC releases that and selects the smart art object itself (and the the align features should now be enabled). 

If pressing ESC leaves nothing selected (ie, the gray selection rectangle around the smart art goes away) then you had the smart art correctly selected in the first place.
