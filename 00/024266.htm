Dear all,

I have a range of numbers and I have applied conditionally formatting to highlight those numbers, say >5,000.

There are then a number of non-contiguous Cells being conditionally formatted to be the same color.

Wondering is there any VBA code that can allow me to select all of the conditionally formatted cells with same color within the range? As I'd like to see the same of those conditionally formatted cells by applying such VBA, rather than having to select those non-contiguous cells by clicking them one-by-one.

(I was managed to find a code that allows me to select Cells with same color within a selected range, however, it only applies to normal colored cells but not for those filled with conditional formatting)

Here is the code for selecting cells filled with the same color (other than through conditionally formatting)

Sub SelectCellColorCellbyCell()

Dim CellColorFormat
Dim RangeString
Dim c

CellColorFormat = ActiveCell.Interior.Color

RangeString = ActiveCell.Address

For Each c In Selection

If c.Interior.Color = CellColorFormat Then
RangeString = RangeString & "," & c.Address
End If

Next c

Range(RangeString).Select

End Sub
