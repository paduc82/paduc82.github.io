jay freedman say:

I'm not sure whether the Starter edition has this option, but in the standard edition you would go to File > Options > Advanced and check the box for "Show AutoComplete suggestions". But that should be turned on by default.

But even then there are rules for AutoComplete that Microsoft hasn't made well known (for instance, there's nothing about them in the Help):

  • AutoComplete happens only for AutoText entries stored in Normal.dotm, not for those in Building Blocks.dotx or any other template. You can use the Building Block Organizer to move entries.
  • AutoComplete happens only when you've typed enough characters (with a minimum of four) to uniquely identify a specific AutoText entry. For example, if you have two entries in Normal.dotm with names "FredAndWilma" and "FredAndBarney", the AutoComplete tag won't appear until you get to the "W" or "B" -- because the "FredAnd" part isn't unique.

hansv mvp say:

For example:

Sub HideZeroTotals()
    Dim wsh As Worksheet
    Dim r As Long
    Dim m As Long
    Application.ScreenUpdating = False
    For Each wsh In Worksheets(Array("Business Development", "Customer Success", "..."))
        wsh.Cells.EntireRow.Hidden = False
        m = wsh.Cells.Find(What:="*", SearchOrder:=xlByRows, SearchDirection:=xlPrevious).Row
        For r = 5 To m
            If wsh.Range("O" & r).Value = 0 And wsh.Range("AC" & r).Value = 0 Then
                wsh.Range("A" & r).EntireRow.Hidden = True
            End If
        Next r
    Next wsh
    Application.ScreenUpdating = True
End Sub

Expand the list of worksheets to be processed as needed.

If you want to apply the macro to ALL worksheets, change the line

   For Each wsh In Worksheets(Array("Business Development", "Customer Success", "..."))

to

   For Each wsh In Worksheets
