greg carmack - windows mvp 2010-2020 say:
Hi Mary. I'm Greg, an installation specialist and 9 year Windows MVP, here to help you.


You can run the Printer Troubleshooter at Settings > Update & Security > Troubleshoot.

But rather than spending too much more time fiddling I'd simply Clean Reinstall the printer by cleaning out the old drivers which always works, lasts longer and actually saves time:

1) Power up the printer, if wired unplug it's USB cable from PC.

2) Go to Settings>Apps>Apps and Features to find and Uninstall the printer software.

3) Next type CMD in Start Search, right click Command Prompt result to Run as Admin, copy and paste in the following command and press Enter: printui /s /t2 (Note that there is one space before each forward slash).

4 Open the Print Server properties, Remove the printer‚Äôs drivers for this printer and for any other printers you don't use presently.

5) If you get a message about access denied or the printer being in use and you cannot find the driver or any installed program that would account for this, you may restart the computer in a diagnostic startup and then use the printui command to remove the driver package(s): https://www.isunshare.com/windows-10/enable-dia...
You must restart back to normal mode before you will be able to install the printer again.

6) Now type Control Panel in Start Search, open CP, choose Icon View. In Control Panel open Devices and Printers, right click the printer to delete the Printer and any scanner or fax of the same name. Restart PC.

7) Go to the printer model‚Äôs Support Downloads webpage to download and install the latest full software package. Now is also the time to check for any newer printer firmware to install which matters a lot.

Older software may need to be installed in Compatibility Mode to run correctly: https://www.groovypost.com/howto/run-old-softwa...

8) Extract Printer software, right click Setup file to Run as Administrator. Follow the setup screens closely as you install the software. For USB it should prompt when to plug in the printer.

If you have further problems reply back with results of each step.


I hope this helps. Feel free to ask back any questions and let us know how it goes. I will keep working with you until it's resolved.

  1. Turn off your computer, then turn it back on.

lemp say:

Sure, there are other possibilities than a dead drive ... but the more information you provide about your system increases the likelihood that you'll get more or less focused help rather than a scattershot approach.

The usual "registry problem" with respect to CD/DVD drives is the upperfilter/lowerfilter issue -- which you've already tried.  If this had been the problem, the drive wouldn't have showed in My Computer ... which was not among the symptoms you have described.

The fact that the drive was able to read a photo disk from Target -- new information that is different from your original statement that the drive wouldn't readany disks -- suggests that in fact the drive isn't dead (or if there is a hardware problem, it's intermittent, which would be very difficult to deal with using forum help).  One possibility is that the laser lens is dirty.  Clean it very carefully with a cotton swab moistened with 95% ethanol or isopropyl alcohol.  See steps 4-6here .

Losing the ability to play movies suggests that the problem is either (a) the DVD laser is dead or (b) the application that plays movies is not working.  Either of these possibilities is consistent with being able to read data on a CD (presumably files that have been stored on the CD).

 The basic driver for CD/DVD drives is cdrom.sys, which is a Windows system file and (IIRC) protected by theWindows file protection feature .  On the other hand, you may have installed other media applications (e.g., iTunes, Sonic DLA, Nero, etc.) that have installed their own drivers for your CD/DVD drive.  These drivers should be compatible with the Windows basic driver, but they may have become corrupt.  By uninstalling and reinstalling the CD drive, however, you should have cleaned these drivers out.  Take a look at the "Drivers" tab of the Properties dialog for your CD/DVD drive in Device Manager and see what, if anything, is present besides cdrom.sys.  If you have any burning software, uninstall it and see if that solves the problem.
