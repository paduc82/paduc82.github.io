jay freedman say:

There is an option to control the automatic conversion of email addresses and internet addresses to hyperlinks. To disable it, go to File > Options > Proofing > AutoCorrect Options > AutoFormat As You Type and clear the check box for "Internet and network paths with hyperlinks".

Word does not automatically change telephone numbers, page numbers, or other text to hyperlinks. There are a few exceptions: tables of contents/figures/tables, footnote/endnote markers, REF fields, and (if an option is changed) screenshots.

The PAGE field you showed is not a hyperlink; you're seeing the field code instead of the field result, possibly because you pressed the Alt+F9 shortcut. Press Alt+F9 again to toggle back to field results.

If you really are getting hyperlinks in the kinds of text you mentioned, there may be an add-in that causes it. Look at the list in File > Options > Add-Ins and disable any that look suspicious.

macropod say:

Although Word does not turn phone numbers and the like into hyperlinks, there are/have been 3rd-party addins that render them clickable so they'll start a Skype session, for example.

By default, Tables of Contents and Cross-References behave like hyperlinks, taking you to the referenced location, but they don't change colour when clicked and, unlike hyperlinks, the 'display text' automatically updates to reflect edits in the destination's wording.

Hi Colin,

Moving notebooks from one account to another can be facilitated by the Export option in OneNote. To do this, you can perform the following:

Export pages and/or sections

  1. Launch OneNote 2016 application and sign in using old account.
  2. Open a notebook through File / Open / select notebook.
  3. Click File / Export / Page / OneNote 2010-2016 Section (*.one). Type a file name and save the section or page on your local machine.
  4. Click on the account name section on the right top corner and click switch account.
  5. The prompt will ask you to add a new account and will prompt you to enter an email address and a password.
  6. Close the notebook through File / Info / Settings / Close.
  7. Create a new notebook through File / New / type a notebook name.
  8. Open the document though File / Info / Open backups / locate and select the pages/section file in your local drive. 
  9. File / Info / View Sync Status / Sync All or press Shift + F9 to sync.

Export notebook

  1. Launch OneNote 2016 application and sign in using old account.
  2. Open the specified notebook through File / Open / select notebook. 
  3. Click File / Export / Notebook / Onenote package (*.onepkg). Type a file name and save it on your local machine.
  4. Close all OneNote notebooks and switch to your other account.
  5. Close the OneNote application.
  6. Open the folder where we saved the *.onepkg file.
  7. Double-click the file and it will open up in OneNote
  8. Create a name and path to save.
  9. Click on File / Info / View Sync Status / Sync All or press Shift + F9.

Let us know if you need further assistance.
