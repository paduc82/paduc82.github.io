hansv mvp say:

If the date format is dd/mm/yy, i.e. day before month, then 2/22/17 is not a valid date, since 22 is not a valid month number. With the date format dd/MM/yy, the 22nd of February must be entered as 22/2/17.

If you want to be able to enter the 22nd of February as 2/22/17, you must set your date format to MM/dd/yy or similar (month before day).

_mz say:

Sorry if I'm reviving a dead thread, but I think there may be a few issues here.

I'm not sure if you want to make the OneNote shortcut WIN+S or if you want to set it to a different shortcut key (as you said, perhaps something like WIN+Z).

If you want to reset the WIN+S shortcut from opening a search box to taking a OneNote screen clipping, then:

First need to disable the WIN+S search shortcut that Windows 10 defaults to. To do so, go to HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\Advanced and either edit "DisabledHotkeys" or, if it does not already exist, create a new string value named "DisabledHotkeys" and set it to "S."

Once you have disabled the search shortcut, you need to create or edit the OneNote screen clipping shortcut in HKEY_CURRENT_USER\SOFTWARE\Microsoft\Office\16.0\OneNote\Options\Other. (Normally, you should choose the highest available value. It's entirely possible that this is 14.0 for you.) Now, edit or create (as a DWORD (32-bit) Value) the "ScreenClippingShortcutKey" and set it to 53 (hex) if you want your shortcut key to be S.

If you just want to set the OneNote screen clipping shortcut to something OTHER than WIN+S (for instance, as you mentioned, WIN+Z), then:

All you need to do is edit or create the OneNote screen clipping shortcut in HKEY_CURRENT_USER\SOFTWARE\Microsoft\Office\16.0\OneNote\Options\Other. (Again, you should choose the highest available value. It's entirely possible that this is 14.0 for you.) Now, edit or create (as a DWORD (32-bit) Value) the "ScreenClippingShortcutKey" and set it to 5A (hex) if you want your shortcut key to be Z.

You mentioned that you set yours to 59. That would make the shortcut WIN+Y.

You can consult this page: https://blogs.msdn.microsoft.com/descapa/2006/11/17/changing-the-onenote-screen-clipping-and-new-side-note-keyboard-shortcuts/ for a list of hex (and dec) values for each letter.

Finally, restart your computer for the registry edits to take effect.

Hope this helped!
