suzanne s. barnhill say:

The Windows Clipboard is the main Clipboard that retains one copied or cut item. It works in all Windows applications. The Office Clipboard, which can hold up to 24 items, works only in Office applications. There is a setting to "Collect without showing Office Clipboard," but ordinarily you would have it displayed if you were using it.

The most recent cut or copied item will be on the Windows Clipboard. It will also (normally) be listed first on the Office Clipboard (items copied earlier will be moved down). If you're not seeing the most recent item at the top of the Office Clipboard, then obviously something is wrong. Does Paste All work?

Since I have minimal experience with the Office Clipboard (just never got into the habit of using it, though I'm sure it would be very useful), I can't help much more, but there is an article at http://office.microsoft.com/en-us/word-help/troubleshoot-copying-and-pasting-with-the-office-clipboard-HP005254778.aspx that may offer a solution.

v_agarwal say:
Is there a way to set Excel to print column headings on every page exclusive of the header/footer function? 


In Page Set up, go to Sheet Tab (Next tab to header/footer)

and give the range in Rows to repeat at Top

Or columns to repeat at left.

 

So the selected range will be printed in everypage.


One more thing, If you want to print row number 1,2,3,4 .......  and  column heading like A, B, C, D .... then there in sheet tab itself 'CHECK' the row and column headings.

 

In Page set up you have to go through ribbon, not through print preview otherwise it will show it grey out and you will not able to change the settings.
