The range of paper sizes that can be printed is determined by the printer driver. Most are useless with custom paper sizes and Word, which uses the printer driver information to format the document, adopts a standard size which may or may not be close to the required size.

There are two potential solutions. The first is to use a standard paper size larger than that required and make up the difference with margins so that the document prints only on the available area.

The other is to configure a new form for the driver.

Run Windows Control Panel > View Devices and Printers.

Select the printer. From the menu bar select Print Server Properties and create a new form. That form will then appear as an available paper size in Word.

mike middleton say:

I suggest you let the Office installer take care of installing the Analysis ToolPak.

Close Excel. In the Windows 8.1 Control Panel, choose Programs and Features, select Microsoft Office 2013, click Change, click Add or Remove Features > Continue, select Office > Excel > Add-ins > Analysis ToolPak > Run from My Computer > Continue > Close.

Launch Excel 2013. Choose File > Options > Add-Ins > Manage Excel Add-ins > Go..., check the box for Analysis ToolPak, and click OK.

The Data Analysis item appears at the right end of the Data ribbon.
