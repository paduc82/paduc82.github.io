chirag. say:

You can use a function that returns you a range in an array like the following:

    Dim A() As Long
    Dim I As Long

    For I = StartIndex To StopIndex: A(I) = I: Next
    MyRange = A
End Function

And use that function as follows:

Sub RemoveUnwantedSlides()
    ActivePresentation.Slides.Range(MyRange(94, 101)).Delete
    ActivePresentation.Slides.Range(MyRange(85, 92)).Delete
    ActivePresentation.Slides.Range(MyRange(76, 83)).Delete
End Sub

Note that you should go from higher slide range to lower so that you don't have to rework out the slide ranges after the slides have been deleted.
