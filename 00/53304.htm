cnguinn say:

I also have a 13" laptop with a high resolution screen (3200 x 1800).

For my laptop to be usable, I first had to change the resolution to 1600 x 900.  I have also tinkered with the scaling option described above.  I tried 125% but that was too large.  100% was too small.  I eventually did custom scaling (which they don't recommend but do provide as an option) and set it to 110%.  Just right, for the most part.

I needed to do some tweaking, but discovered that Microsoft has removed the Display / Control Panel which contained more granular controls for text sizing of specific items such as Menus, Display Boxes, Icon Text, and others.  There are several discussion threads about the removal of Display / Control Panel.

There are 3rd party tools which provide this important resizing capability.  I recommend Winaero Tweaker as it has these controls and others.  Another, simpler tool is System Font Size Changer.  

A BIG issue with the install of Windows 10 Creators is that it resets all your custom text size settings, and specifically reduces the Icon Text size to quite small.  Icon Text size controls the size of text inside Windows File Explorer, and possibly is responsible for the tiny text in your Select File dialog box.

So get one of these 3rd party tools and adjust as necessary.

Best wishes.

These 3rd party programs work the same as the old Display Control Panel setting worked.  They affect specific user interface components (menu text, icon text, window title text) for everything on your machine.  You do not need to apply these to each app.

I think that Microsoft made terrible decisions to remove the Display Control Panel and then to reset all the display settings.  They have made a mess.  Fortunately the settings still remain in the Registry, they just removed the User Interface to easily adjust these settings.  The 3rd party tools give us back the UI that Microsoft removed.

spiritx say:

Hi,

You really can't repair the 2nd drive's Windows installation. You can run checkdisk on it.
And you should be able to copy your files to the new drive. Without some (unknown to
me) 3rd party software you will need to re-install your programs on the new drive. An
added caveat is that the programs, registry, and other files on the old drive might be
corrupt/damaged.

Check with a real computer shop (not leeks and peeks at a BestBuy or other BigBox Stores)
or the System maker's Support to see if the old drive installation can be saved. Of course
that depends a lot on the methods you might have already tried.

Hope this helps.

andre da costa say:
The command line supports several types of copy commands:

Copy: If you just want to copy files or folders from location to the other.
Xcopy: A robust copy option suitable for copying the contents of a folder with subfolders to a destination folder. Xcopy is being superseded by Robocopy which provides the same functionality.

Here are some examples how to use these copy commands.

I want to copy the PowerPoint presentation named Evolution of the Settings interface.pptx to the Work folder located in the Groovy folder we created earlier. First, change your directory to the location of the file.

In my case, the PowerPoint file is in my User folder under Documents. First, type the copy command, followed by a file name with open and close quotes then the file path destination, example: copy “Evolution of the Settings interface.pptx” c:\Users\Andre\Documents\Sample\Groovy\Work

Since Robocopy has superseded Xcopy, we are just going to focus on using that command for this example. Robocopy includes a number of arguments you can use to copy lots of folders thoroughly. It’s particularly great for creating backups. For instance, if I want to make a backup copy of my Downloads folder to a folder on my thumb drive named Backup, this is how you do it.

robocopy “D:\Users\Andre\Downloads” “F:\Backup”

Learn more: https://www.groovypost.com/howto/10-command-lin...

To unblock a software publisher please follow the steps below:
1. In Internet Explorer, click the Tools button, and then click Internet Options.
2. Go to the Content tab, Look under the Certificates section.
3. Click on Publishers button.
4. Select the Untrusted Publishers tab.
5. Select the Publisher you wish to unblock and click the Remove button.
