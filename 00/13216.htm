  • Outlook 2010 on Windows XP

  • Outlook 2010 and Outlook 2013 on Windows Vista, Windows 7 or Windows 8
    C:\Users\%username%\Documents\Outlook Files

Characters not Allowed In Sheet Names - Microsoft Community

faraz a. qureshi say:

Characters not Allowed In Sheet Names

In one of my macros I need to update the sheets names based on some speciall cells values. However, any idea of developing a list of unacceptable characters or not fulfilling the acceptable rules for being applied to a sheet's name, so as to replace the same with underscores if any of the same is found in the cells?
ossiemac say:

You should be able to create a list with code like the following where it attempts to use the characters in a worksheet name and if errors then unacceptable character.

 

Chr(39) which is a single quote does not show in the cell but you can see it in the formula bar.

 

Sub SheetIllegalChars()
    Dim r As Long
    Dim varErrNumb As Variant
    With Worksheets(1)
        .Select
        .Columns("A:B").Clear
        .Cells(1, 1) = "ASCII"
        .Cells(1, 2) = "Character"
        For r = 0 To 255
            varErrNumb = 0
            On Error Resume Next
            ActiveSheet.Name = "Sheet1_" & Chr(r)
            varErrNumb = Err.Number
            On Error GoTo 0
            If varErrNumb > 0 Then
                .Cells(.Rows.Count, 1).End(xlUp).Offset(1, 0) = r
                .Cells(.Rows.Count, 1).End(xlUp).Offset(0, 1) = Chr(r)
            End If
        Next r
    End With
End Sub
