ken sheridan say: Your reference to the subform seems to be missing any reference to the subform control.  As it stands the code is referencing the parent form, not the subform.  Only parent forms are members of the Forms collection.  The full syntax is:

    Forms!NameOfParentForm!NameOfSubformControl.Form

NameOfSubformControl is the name of the subform control in the Controls collection of the parent form, NameOfParentForm, i.e. the control which houses the subform.  This might or might not have the same name as its source form object.

If the code is executed in the parent form's module it can be shorted to:

    Me!NameOfSubformControl.Form

Similarly the syntax for your first Case statement, Me.FilterOn = False, should be.

    Me!NameOfSubformControl.Form.FilterOn = False

brightwebs say: Never mind, it's in the same screen, a little further down in the Print section - Print field codes instead of their values. That means if you actually want to show the hyperlink for a course handout, you can.

So ALT-F9 toggles the code view on and off, but to print the codes you have to go into the Options/Advanced/Print section menu and change it.

robertlo9 say: I found out the solution! It was hidden in the options. I discovered that the term for those hyperlinks is "field codes." To change the settings so it shows the value instead of the field code, which is what I was trying to do, you click the office button on the top left-hand side of the Microsoft Word 2007 page. It's the circle with the Office logo. Click "Word Options" on the bottom of the dropdown menu. Click the "Advanced" tab and scroll down to the "Show document content" section. Right above the "Field shading" box, there is an option that says, "Show field codes instead of their values." If that has a check next to it, unclick it and hit OK, and your hyperlinks will return to normal in your Word documents. There you have it! Problem solved.

Hi Max,

Sorry for the inconvenience it has caused.

To change the load behavior, you can edit the registry key of this add-in:

1. Close OneNote and open Registry Editor: Windows key + R > input regedit > Yes

2. Under this location:

Computer\HKEY_CURRENT_USER\SOFTWARE\Microsoft\Office\OneNote\AddIns\Microsoft.ClassNotebooks.AddIn

change the LoadBehavior's value to 3
