hpholm say:

The system backup does not contain personal files, it's basically a list of installed apps, a list of mail and other accounts (without passwords) and the start screen layout. Only very few WP8 apps and games were upgraded to include user data in this backup.

Also please note the separate SMS and pictures/videos 'backup' is not retroactive, it only syncs new items after the settings are toggled On.

I would rather suggest copy all pictures, videos, music, documents and other user files via USB to a PC. Then do a backup of the contacts from the Manage option in the top bar at Outlook People - It's a standard .CSV file that can be imported elsewhere later (and good to have when things get messed up).

- If some of the attachments in MMS are important make sure to store a local copy.
- Important SMS texts can be sent to yourself via email.
- If data from third-party apps are important, export the user data via the save to OneDrive or other options in those apps, if possible.

Then set up the new phone from scratch with a freshly formatted or a brand new SD memory card (if any).

- Setting up a Windows 10 Mobile phone on the same Microsoft account where WP8 phones are still actively used can potentially cause all sorts of interesting issues.
- Some apps are not updated to Windows 10 Mobile. For example HERE maps and HERE Drive are not available.

I hope you get it sorted out, one way or the other.
