Counting multiple occurrences of multiple words in a range - Microsoft Community

jarreddaw say:

Counting multiple occurrences of multiple words in a range

Hi Team,

I am looking for a formula to count the occurrences of multiple words or multiple criteria you could say.

I have also tried using an IF formula to constantly cycle through IF ELSE statements and counting with no prevail.

The COUNTIFS formula only counts the first occurrence and doesn't a second occurrence in that cell or another word.  

For example if cell A1 has example1 and example2 and A2 has test1 and test2.

amit tandon say:

One option is:

=SUMPRODUCT(LEN($A$1:$A$2)-LEN(SUBSTITUTE($A$1:$A$2,"example","")))/LEN("example")+SUMPRODUCT(LEN($A$1:$A$2)-LEN(SUBSTITUTE($A$1:$A$2,"test","")))/LEN("test")

Regards,

Amit Tandon

jarreddaw say:

Excellent, and I assume just keep nesting the formula for more strings. 

Quite a large formula, I am wondering if I have perhaps gone about this the wrong way?

Would there be a better way of going about counting multiple occurrences of multiple words?

- Jarred

amit tandon say:

The formula basically repeats for each string:

=SUMPRODUCT(LEN($A$1:$A$2)-LEN(SUBSTITUTE($A$1:$A$2,"example","")))/LEN("example")

plus

=SUMPRODUCT(LEN($A$1:$A$2)-LEN(SUBSTITUTE($A$1:$A$2,"test","")))/LEN("test")

It adds both these - formula repeated for each string ("example" & ""test"). So for each extra string this will need to be repeated.

Note: Note that the Excel Substitute function is case sensitive. So you may use below in case the cells contain both Upper / Lower case - it just adds UPPER in the formula:

=SUMPRODUCT(LEN($A$1:$A$2)-LEN(SUBSTITUTE(UPPER($A$1:$A$2),"EXAMPLE","")))/LEN("EXAMPLE")+SUMPRODUCT(LEN($A$1:$A$2)-LEN(SUBSTITUTE(UPPER($A$1:$A$2),"TEST","")))/LEN("TEST")

Amit Tandon

c.greaney say:

styleref in header conditional logic

Hi,

I'm trying to use the styleref field code to dynamically populate a header in a Word template.

My challenge is that the appropriate heading text will be be in one of two styles; depending on the intended audience for the document. How can I have Word look for style1 and if it doesn't find it, look for style2 and use that instead?  

Thanks in advance for your help!

Christa

***Post moved by the moderator to the appropriate forum category.***

macropod say:

Basically, you'd use a field coded along the lines of:

{IF{STYLEREF "Heading 1"}= "Error!*" {STYLEREF "Heading 2"} {STYLEREF "Heading 1"}}

where 'Heading 1' and 'Heading 2' are the Style names.

Note: The field brace pairs (i.e. '{ }') for the above example are all created in the document itself, via Ctrl-F9 (Cmd-F9 on a Mac); you can't simply type them or copy & paste them from this message.
