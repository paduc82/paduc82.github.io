To test for the above when the code will not run:

  1. Open the VBA editor.
  2. Open the Immediate window (Ctrl and G)
  3. In the immediate window type the following line and press Enter (ensure you include the leading question mark)

         ? Application.EnableEvents

If it returns True then the above is not the problem. However, if it returns False then events are disabled so type the following line in the Immediate window and press Enter to re-enable events.

Application.EnableEvents = True

If you establish this is the problem then your code needs to be like the following example to so that if an error occurs then the Events are automatically re-enabled.

Private Sub Worksheet_Change(ByVal Target As Range)

    On Error GoTo ReEnableEvents    'Include in line before disable events
    Application.EnableEvents = False
   
    'Remaining code codes in this section.
    'Do not use Exit sub in the code.
    'Use GoTo ReEnableEvents in lieu of Exit Sub
   
   
   
    'All of the following code is the last code before the End Sub
ReEnableEvents:         'Note the colon suffix
    If Err.Number <> 0 Then
        MsgBox "An error occurred in Private Sub Worksheet_Change"
    End If

    Application.EnableEvents = True
