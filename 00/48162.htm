Windows Store Apps Screw Up Regularly - Fix Them With A Windows Repair Install

I really like Windows 10 but it is very frustrating that Windows Store Apps just seem to get screwed up on one account or another and there seems to be no way to fix them.  I have two virtually identical PCs both running Windows 10 each of which has two main accounts for myself and my wife.  Hardly a week goes by when the windows store apps screw up in some way on one or other of the PCs/accounts.  Right now Im having a problem with the Photos and OneNote app on my account on my second PC, the store says there are updates but they wont install.  It gives me some error code (cant remember it right now).  Last week the windows store app icon mysteriously disappeared on my wife's account on my PC.  Anyway as usual I have googled these problems and there are the usual bunch of posts covering powershell commands, troubleshooters (store apps, microsoft account & windows update) and other such things - Ive tried them all on various problems and nothing works.  

So Ive got to the stage now when as soon as I see this kind of problem i just KNOW that trying to fix it is hopeless.  There is only ONE solution which works, which is doing a windows repair install keeping all apps, files and settings.  Now a windows repair install may sound drastic (and it is) but Ive done it several times now and though it takes a while, it DOES seem to solve these problems - at least it has for me anyway.

A detailed guide for doing a repair install may be found here:-

http://www.tenforums.com/tutorials/16397-repair-install-windows-10-place-upgrade.html

Now there are some tedious aspects to it, like:-

  1. It takes quite a long time to do
  2. You have to uninstall your anti-virus first and re-install it afterwards
  3. That it does uninstall some desktop apps that you have to then re-install afterwards eg ccleaner maybe others ?
  4. That it uninstalls my canon printer and scanner driver and I have to re-install these afterwards
  5. That it screws up my outlook / icloud integration and I have to logout and login again for each account and set up the icloud integration again.

Obviously on a different build with other apps installed then other things might also be affected.

So do this at your own risk obviously AND definitely make sure to select the "keep personal files and apps" setting, or you will nuke your machine.  But all that said its very similar to doing a major version upgrade of the operating system and it has worked for me several times now.  

Hope someone else finds this useful.
