Are you saying to copy the letter head and watermark separate and then copy the letter body afterwards ?

A document contains several independent 'layers', in effect like a pile of transparencies which when stacked produce the finished document. The header is in a different layer from the body text of the document, so you cannot copy them together.

shane devenshire say:

There are times when you may want to change and extension, for example some firewalls won't allow exe, dll, xlsm files through for obvious security reasons.  Sometimes renaming the file with a different extension will get it past the firewall.  Of course the user on the other side should change the extension back to the correct one.

If as is more likely in your case you are change the extension of a xls file to xlsm or xlsx, that is not a good idea.  The file structures of 2003 and earlier versions of Excel (xls) is completely different than 2007 file structures.  You may get away with it but its not a good idea.  If you want to change the file type from 2003 to 2007 open the file in 2007 and do a Save As and pick either xlsx or xlsm.  If you are trying to change an xlsx or xlsm file to a  xls file, changing the extention is not going to do what you want.  Again, you should open the file in 2007 and Save As a 2003 (xls) file type.

graham mayor say:

The aim of my reply is to understand what you are doing in order to provide the best method of achieving the end result. It was not even clear where the formatted text was coming from.

If you want to add to a selected cell then you need to set a range at the end of the cell and insert the formatted text in the range. The following covers two methods with annotations:

Option Explicit

Sub Macro1()
Dim oCell As Range
Dim oRng As Range        'required with the second option
    If Selection.Information(wdWithInTable) Then
        Set oCell = Selection.Cells(1).Range
        oCell.End = oCell.End - 1        'Lose the end of cell character
        oCell.Collapse 0        'collapse the range to its end
        oCell.Text = vbCr        'optionally add a paragraph break
        oCell.Collapse 0        'collapse the range to its end again
Option1:
        'add the formatted text e.g.
        'oCell.Paste 'Paste from the clipboard
        'OR
Option2:
        Set oRng = ActiveDocument.Tables(1).Cell(2, 1).Range 'The range to be copied
        oRng.End = oRng.End - 1 'It's a cell so remove the cell end character
        oCell.FormattedText = oRng.FormattedText 'duplicate the formatted text
    End If
lbl_Exit:
    Set oRng = Nothing
    Set oCell = Nothing
    Exit Sub
End Sub

A possible solution was found..... If you are using a USB microphone, you need to start OneNote BEFORE you plug in the microphone.

Apparently when OneNote starts it takes the audio settings in effect as of when it launches and if you plug in a USB microphone after launching OneNote it isn't detected by OneNote as an audio input source even though Windows may say it is so.

When the first recording in OneNote is blank and the second one works it is because OneNote also refreshes the audio settings when you create a new page.

The moral here is to launch OneNote first and then plug in your USB microphone and THEN create your new page to 'house' the recording on.

Seems like standard line in microphones are automatically recognized but USB devices aren't.

Either way, test - test - test and test again and then pray because the only sure fire solution is to wait until the recording is finished and listen back to it.

Repeating Table of Contents title on each page - Microsoft Community

afairshake_wv say:

Repeating Table of Contents title on each page

Is there a way to repeat "Table of Contents" at the top of each page of a TOC that is multiple pages long?
jay freedman say:

I'll assume for starters that you've inserted the TOC by clicking the Automatic Table 2 thumbnail on the Table of Contents button. In that case, the words "Table of Contents" are formatted with a style named TOC Heading. Then:

  • If you don't already have a section break between the TOC and the start of the document body, insert one.
  • On the first page of the document body (the second section), open the header pane so the Header & Footer Tools tab appears on the ribbon. Click the Link to Previous button in the ribbon to turn it off.
  • Go to the first page of the TOC and open its header pane.
  • Check the box for "Different first page".
  • On the same ribbon tab, click the Next button in the Navigation group to go to the header on the second page.
  • If there is already text in this header, edit it or remove it as needed. Then press Ctrl+F9 to insert a pair of field braces. Between them, type StyleRef "TOC Heading" and then press F9. This field will repeat the "Table of Contents" wording on every page of the TOC (except the first).
  • Click the Next button again to go to the first page header of the next section. Verify that it's the correct header for that part of the document.

If the heading on the Table of Contents has some style other than TOC Heading, put its name in the StyleRef field instead.
