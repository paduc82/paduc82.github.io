Wellllll ... I can't really test it from my printer at home because it only has one bin, but the Immediate window results suggest that it might work.

Private Sub PageHeaderSection_Print(Cancel As Integer, PrintCount As Integer)
Dim MyPage As Variant
MyPage = Me.Page

For MyPage = 1 To Me.Pages
Debug.Print "Page : " & MyPage


Select Case MyPage
    Case Is = 1
        Printer.PaperBin = acPRBNLower 'The first page should print on 8.5" x 14"
    Case Else
        Printer.PaperBin = acPRBNAuto 'All of the rest of the pages should print on standard 8.5" x 11"
End Select

Debug.Print "Print from : " & Printer.PaperBin

Next MyPage

End Sub

Note that Debug.Print Printer.Paperbin returns the numeric constant.

1= Upper Bin

2 = Lower Bin7 = Default (or Auto) Bin

Presume your range is A1:C15. Cell A1 s the heading "imType" & C2:C14 have data - "Cesarean" to "Vaginal". Cell B2 s the heading "inDrop2" & B2:B15 have data - "Primary" to "Readmits w/in 14 days". =IFERROR(INDEX($C$2:$C$15,SMALL(IF(($A$2:$A$15=$E$1)*($B$2:$B$15=$E$2),ROW($2:$15)-ROW($1:$1)),ROW($A1))),"")

Conditional formatting if the cell contains a formula - Microsoft Community

edenton say:

Conditional formatting if the cell contains a formula

Can you use conditional formatting to highlight a cell on the logic of highlight if cell contains a formula?
mike h.. say:

For example if a cell was changed from containing "10" to now containing "=10-3"

I am wanting to use conditional formatting to highlight that cell when it was changed from just a number to a formula.  Is that possible?


Hi,

 

It's possible to highlight a range if the cell contains a formula. Here's one way.

 

ALT+F11 to open vb editor, right click 'ThisWorkbook' and insert module and paste the code below in and then close VB editor.

 

On the worksheet aplly conditional formatting to the cell using this formula

 

=isformula(A1)

 

Where A1 is the cell being tested.

 

 

 

Function isformula(rng As Range) As Boolean
If rng.HasFormula Then
    isformula = True
Else
    isformula = False
End If
End Function
