Hi Vipervik - it was probably my post that you read :)

It all comes down to where the messages that were deleted are stored - I may be wrong here but I think there's three locations you can store them :

1) On the phone itself
2) On the memory card
3) On the SIM (Not 100% sure about this one, I know you can save contacts on the SIM, but not sure about messages)

On my N95, I'm presented with options to save text messages on either phone memory or memory card.

The principles of data recovery rely on the fact that no new data is written over the data that has been deleted. If you examine what happens when an Operating System (the software that runs your phone or computer) deletes a file, on older OSs what used to happen is that the file actually just got renamed with a question mark as the first character of it's filename. The OS then recognised this as meaning that the file was deleted and marked it as free space ready to be reused. If the space that the deleted file was reused, then there is no way to recover the data. (Technically this isn't strictly true if the two different files are different sizes and live in different clusters, but this is WAY beyond the scope of this discussion).

Military (and financial) institutions ensure that data can not be recovered by deliberately taking advantage of the above fact and writes gibberish data continuously to every location on a disk/device before deleting it finally. This ensures that the data recovered would be gibberish. Often if the material in question is VERY sensitive, they will often physically destroy the media rather than risk its reuse.

Going back to your phone example, yes it is capable of recovering data on your phone. There are two ways of doing so :

1) Taking it to a qualified data recovery company and paying a lot of money to a guy in a white coat and probably a clean room to open up your phone and perform data recovery forensics on it to get the data back
2) There MAY (I don't know as I've never researched it) be software out there which you can install onto your phone to check for deleted messages, data etc. However anyone installing the application will have the same quandry that I outlined above - in order to install the application, they MAY be overwriting exactly the information that they want to save.

The memory card situation was covered in my other post, but essentially it's a lot easier. You take the memory card out of your phone, and plug it into a laptop/PC, install some data recovery software on your PC, and start looking for your data - easy - if you know what you're doing. It would pose a challenge to a non techie.

Hope this helps.
