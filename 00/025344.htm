It is. 

A couple of things to check:

In the past, if you had an empty content or picture placeholder on the slide when you inserted a picture, the picture would drop in to the placeholder AND would be compressed, regardless of your compression settings.  This unwanted compression seems to have been corrected in the most recent version/update of 2016, at least.  To be sure you have this:

  • Start a new presentation (blank is fine)
  • Add a new title & content slide
  • Reduce the content placeholder to a small size
  • Add a picture to it
  • Now enlarge the placeholder

Does the picture become blurry?  If so, you might want to force an update and try again. If not, we've eliminated one possible cause.

PPT's compression settings don't affect the compression that Adobe Acrobat's PDF printer driver applies.  Try using PPT's own File | Save As | Save as type:  PDF and be sure to choose the "Standard (publishing ...)" option. 

How does that compare with the result from Acrobat?

I find PPT's PDF save feature to be WAY faster than Acrobat's and generally prefer the result, but there are times when you need the extra control that Acrobat gives you.  If so, yell.  One of us will explain how to make it happen.

It sounds like your hardware changes were large enough to cause the activation code to trigger. It is possible that the re-activation is blocked by the same problem that is causing the instability. Your best best at this point would be to contact customer support directly, as installation issues are no charge to you, and working with a technician on the phone is a more appropriate way to attack the problem and get your system fixed.  You can call 1-800-Microsoft and request support assistance to get routed properly.

You may find that fixing the instability still leaves the product un-activated.  In that case, try requesting the program to activate "By Telephone" instead of over the internet.  You will dial into a voice response system, and if it cannot activate you immediately, you can then talk with an activation person, explain that your hardware failed and had to be replaced, and they will be able to provide you with a code to get going again.

Good luck!
