User accounts CANNOT be added through Change PC Settings - "An account using this Microsoft account already exists on this PC".
In Local Users and Groups management console, the Users do appear.

Windows 7 : Click Start --> Run... --> (and type) lusrmgr.msc
Windows 8 / 7: (press the) START (key and) R (at the same time) --> (and type) lusrmgr.msc

intensebutlovbl say:

aMillion, you got me on the right track, but I think this is a quicker way to fix the problem:

Once lusrmgr.msc is running, double-click on the Users folder.

When the list of users appears, right-click on the user that you want to add to the Users group and choose Properties from the drop-down menu that appears.

In the window that appears, click the "Member of" tab.  All of the groups that the user is a member of will be displayed.  To add the user to an additional group (such as Users), click the Add button.

When a new window appears, click the Advanced button.  After the window changes, you will see that it allows you to search for objects.  The object type Groups will be selected in the top box.  So when you click "Find Now", there will be a list of all of the groups on the computer.

Click Users (or any other group you want) and click OK. When that window closes, click OK on the window that was behind it.  Now you should be back to the "member of" tab and the group you added should have appeared in the list.  Click OK to that and you are done.
