How to secure the phone contacts of Asha and other Nokia Series 40 and - Microsoft Community

srikanthsd say:

How to secure the phone contacts of Asha and other Nokia Series 40 and Symbian smartphones.

Technical Level : Intermediate

Summary

Details

Phone contacts are very important assets. It is always advisable to take regular backup of important data for safety.

 

Here are the different methods to secure your contacts and solutions to solve common contacts problems.

 

1. Using your phone's sync-and-backup option

 

To backup the phone contacts using your phone,
go to settings, sync-and-backup, create backup. Mark the contacts and neccessary items you need for backup.

 

Note: Creating backup doesn't work without memory card.

 

2. Using Nokia suite

 

Install the latest version of Nokia suite from here.

 

Connect your phone to PC using data cable(set Nokia suite mode on phone) or via bluetooth.

 

Launch the Nokia Suite, open contacts and follow these:

 

2A. Using drag and drop

 

You can simply drag and drop the contacts to a folder of your computer as shown:

 

 

You can also drag and drop contacts from the PC folder back to Nokia suite.

 

2B. Using Backup/Restore option of Nokia Suite

 

To backup or restore using Nokia suite, go to tools-backup/restore.

 

 

2C. Using the sync option

 

You can sync your phone contacts to Nokia suite by default, as well as with Microsoft Outlook.

 

To sync your contacts to Outlook, open Nokia suite, go to sync-sync options and select Microsoft Outlook in contacts as shown:

 

 

After syncing, the contacts of your phone will be copied to both Nokia suite and Microsoft Outlook:

 

 

Note: Creating or deleting any contacts in any one of the locations like either phone or Nokia suite or Outlook,  updates the same changes in the remaining two places.

 

It is highly recommended to use drag and drop method to backup contacts instead of sync for simplicity.

 

3. Common problems with contacts and solutions

 

3A. Avoiding repeated contacts.

 

You might have copied all the phone conatcts to SIM card as a backup, but the contacts will be shown twice.

When you have similar contacts in both the SIM card as well as phone, you can change the settings such that either of the SIM contacts or phone contacts are only visible, for this, go to contacts-settings-memory in use and change it to either phone or SIM card.

 

3B. Deleting the either only SIM or phone contacts.

 

You can always delete either SIM or phone contacts. Go to contacts-delete all contacts and select either SIM card or phone.

 

3C. Deleting repeated contacts using PC.

 

Sometimes the contacts will be repeated in the phone memory itself. To fix this, drag and drop your contacts to a folder and delete all the contacts having parenthesis () as shown:

 

 

Now you can delete all the contacts of the phone and copy(drag and drop) the non repeated contacts from the PC back to phone.

 

4. Recovering the contacts from a corrupted backup file of the phone using PC

 

There are situations when restoring data fails due to corrupted backup file.

To solve this,

 

A. Connect the memory card to PC using a card reader.

 

B. Enable the hidden folders and extension in view or in the folder options, go to memory card/backup folder and copy the .NBF file to your PC.

 

C. Change the extension of the backup file from .NBF to .ZIP.

 

D. After renaming, the backup file can be extracted using any compression applications like WinRar or default ZIP manager.

 


E. The contacts of the backup file can be found at

\Backup000\predefhiddenfolder\backup\WIP\32\contacts.

 

These contact files can now be copied back to your phone using Nokia suite(using drag and drop method).

cause more problems than they are worth.  There is usually a workaround
for them so you don’t need them.  Not all of them have things that you can
use besides them but you are at the mercy of versions if you do use some (like
the Windows Media control and the IE Web Browser).



So, some alternatives which you can use:

For the Rich Text control:You can use Rich Text in 2007 or above by simply using a normal
text box and setting the field in the table to MEMO and then changing the
control’s TEXT FORMAT, located on the DATA tab within the Properties dialog, to
Rich Text from Plain Text.
For the CommonDialog:

You can use the built-in File Scripting Object (just need to set
a reference to Windows Scripting Host) or the Office File Dialog which you can
call if you have the Microsoft Office 14 Object Library checked.  And last
you can use this API here: http://www.mvps.org/access/api/api0001.htm
