3- Open Task Manager and 'End task' on your web browser. You'll lose all open tabs, but that's better than being hacked.

4- Don't try to visit that web page again, at least not until the web page owner has fixed the page.

Method 1:

If you are not facing any internet connection issues, then you can ignore the error message.

On the other hand, if you are facing issues with the network connection, then you need to uninstall the adapter and restart the computer.
 
Uninstall the Microsoft ISATAP adapter from device manager.
 
a.    Click on Start; in the start search type devmgmt.msc and hit Enter.
b.    Choose the Microsoft ISATAP adapter.
c.     Right click on it and choose uninstall.
d.    Restart the computer once it is uninstalled.
Method 2:

Code 31: This device is not working properly because Windows cannot load the drivers required for this device.


You may be prompted to provide the path of the driver. Windows may have the driver built-in, or may still have the driver files installed from the last time that you set up the device. However, sometimes, it will open the New Hardware Wizard which may ask for the driver. If you are asked for the driver and you do not have it, you can try to download the latest driver from the hardware vendor’s Web site.

 

I would suggest you to uninstall the corrupted driver, if you have any previous installed adapter drivers and then install the drivers in Clean boot mode.

http://support.microsoft.com/kb/929135

After you have finished troubleshooting, follow step 7 in the article to reset the computer to start as usual.
