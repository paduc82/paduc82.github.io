Random number generation in excel - Microsoft Community

john smithzx say:

Random number generation in excel

I need to generate a random number in excel. The first 5 digits are defined but the last 5 are not. How do I use the rand function to do this while also eliminating numbers that already exist in the list of numbers i have? Im looking for remaining combinations basically. any help would be awesome  :)

vijay a. verma say:

You will have to use a Macro for this purpose. This will generate all possible combinations between 0 to 99999 (5 digits) with a 5 digit prefix and will remove duplicates if already existing in your list.

1. Make a backup of your workbook. 
2. Open your workbook and ALT+F11
3. Locate your Workbook name in Project Explorer Window
4. Right click on your workbook name > Insert > Module 
5. Copy paste the Macro code given and change bold lines as per your requirement.
6. Go back to your Workbook and ALT+F8 to display Macro Window
7. Run your Macro from here
8. Delete you Macro if the Macro was needed to be run only once.
9. Otherwise save your file as .xlsm if you intend to reuse Macro again.

'**** Vijay A Verma (eforExcel.com) ****

Sub GenerateRandoms()
    Const Min As Long = 0
    Const Max As Long = 99999
    Const Prefix As Long = 12345
    Dim LastRow As Long
    Dim Ws As Worksheet
    Dim i As Long, j As Long, Temp As Long, Number As Long
    Dim Arr
    Set Ws = Worksheets("Sheet1")
    Application.ScreenUpdating = False
    Application.DisplayAlerts = False
    Number = Max - Min + 1
    ReDim Arr(1 To Number, 1 To 1)
    For i = 1 To Number
        Arr(i, 1) = Prefix & Format(i, "00000")
    Next i
    Randomize
    For i = 1 To Number
        j = Int((Number - i + 1) * Rnd) + i
        Temp = Arr(i, 1)
        Arr(i, 1) = Arr(j, 1)
        Arr(j, 1) = Temp
    Next i
    LastRow = Ws.Range("A" & Rows.Count).End(xlUp).Row
    Ws.Range("A" & LastRow + 1 & ":A" & Number + LastRow) = Arr
    Ws.Range("A1:A" & Number + LastRow).RemoveDuplicates Columns:=1, Header:=xlNo
    Application.ScreenUpdating = True
    Application.DisplayAlerts = True
End Sub
