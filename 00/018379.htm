ronnie vernon say:

Hi Win8to10

That 'more space' dialog box is already aware of all the space that is available on your hard drive, including the 6.60GB on the C: Drive. It is saying what you need is another 10GB in addition to that existing free space.

An Upgrade install as opposed to a Clean Install always requires a lot more disk space. A Clean install simply installs a new system, an Upgrade must replace an existing installation, while it is still running.

So when upgrading Windows must have room to create all of the files that will be used to replace the existing installed files. These files will be created in Temporary folders and staged there until they are needed to replace the existing files.

There is also a folder named Windows.old that contains a large percentage of the files from the previous installation of Windows. The installation must also include enough disk space to create another Windows.old folder for the files in the current installation.

"Also what in the world do they actually mean when they say "Choose another drive or attach an external drive with 10GB available?"

The Windows installation program can use External Storage Space to stage temporary files until they are needed, such as the new Windows.old folder.

Let us know if you have more questions.
