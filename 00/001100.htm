kosh vorlon say:

Hi Dawn,

 

To clear cache (AKA Browser temporary file history):

 

Tools | Internet Options |  General | Browsing History | Delete.

 

Check ONLY "Preserve Favorites website data" and "Temporary Internet Files".

 

Click Delete and wait.  It will notify you when done.  Your cache is now cleared.

 

To increase cache (AKA Browser temporary file history size)

 

Tools | Internet Options | General | Browsing History | Settings

 

Set Temporary Files to Automatic

 

Increase the Disk Space to use from what it is to at least the recommended maximum or up to the stated maximum above that.  I use 250MB.

 

Decrease the number of pages of History to store - I had troubles set at 180 days (performance issues - not same error as you), so I now use 90 and am considering reducing it to 60 or even 30.

 

********************************************

 

When done, reboot and see how things go.  If the problem persists, try running IE in no-add-ons mode.

 

Start > All Programs > Accessories > System Tools > Internet Explorer (No Add-ons)

 

Alternatively, you can run iexplore.exe -extoff

 

This will start IE with no add-ons.

 

Then through Tools | Internet Options | Manage Add-ons, you can enable/disable each add-on to identify which, if any, add-on is creating problems and deal with the culprit.

 

*********************************************

 

I hope this helps.

 

Good luck!

If you have been using Windows long enough, you know that there are tons of problems with the existing implementation of Windows Update in previous versions.

These existing versions used the old (and outdated) Direct UI code which is a set of libraries used by the Microsoft developers to build Shell Code in Windows.

In this current 9926 Build, Windows Update (and many more of the old Control Panel components) are now using a XAML code base to build these components. XAML gives the developers many more advantages such as performance, stabilitiy, and is much less prone to vulnerabilities.

Remember that Build 9926 is still very early pre-release code and many more features will be added to all of these new components.
