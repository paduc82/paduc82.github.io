vijay a. verma say:

Since, your Worksheets are getting incremented by 1 only, then VBA is not needed. Use following formula as first formula to refer to previous sheet.

=LOOKUP(10^307,INDIRECT("'A-"&TEXT(SUBSTITUTE(REPLACE(CELL("filename",$A$1),1,FIND("]",CELL("filename",$A$1)),""),"A-","")-1,"00")&"'!"&"A:A"))

karl timmermans [outlook mvp 2012-2018] say:
If this is the only account configured in Outlook, the cause would be that the data file associated with the account is set as the 'default data file" for the profile which is where the default contact/calendar files are stored.

Since you cannot have an Outlook profile without an Outlook data file, you will need to create a new PST data file. From the Home ribbon,
1) select <New Items> -->
2) select <more Items> -->
3) Outlook Data file
when done
4) go to File --> Account Settings --> Data Files tab --> select the newly created PST file and set that as the "default data file"
5) when done, switch to the Emails tab --> highlight the email account --> Remove

If the account was configured using IMAP, no further action required. If it was configured using POP3, return to the Data Files tab, highlight the PST file associated with the account --> Remove
