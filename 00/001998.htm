    suzanne s. barnhill say:

    Pam has suggested a simple F&R operation that is likely to cause as many problems as it solves (and it doesn't address sentences ending in question marks or exclamation points). Since you're going to have to examine the document closely for incorrect replacements (two spaces after initials, for example, as Pam mentioned), it might be easier to take a different approach.

    Even if you are not using the grammar checker for anything else, you can enable it for one single issue: number of spaces between sentences. Go to File | Options | Proofing and check the appropriate boxes for checking grammar either with spelling (explicitly) or checking it as you type. Then click Settings. Under Require, select 2 for "Spaces required between sentences." You can then uncheck every other box in the dialog, if desired, so that the grammar checker won't bother with anything except the spaces. If you enabled "Check grammar as you type,"  you'll be able to scan the document visually for marked instances of a single space between sentences and correct them.

    I have to say, though, that I'm surprised that APA 6 requires two spaces between sentences, as modern typographical practice prefers just one (unless a monospace font is being used).

    How to reset settings on Outlook 2010? - Microsoft Community

    rutherford morais say:

    How to reset settings on Outlook 2010?

    How to reset Outlook 2010 completely?
    brian tillman [outlook mvp 2007-2019] say:
    Depends on what you mean by "reset".  If you want to start Outlook with a clean set of folders, create a new PST.  If you want to start Outlook over with not only clean folders but with having to add accounts again, create a new mail profile and PST.  If you want Outlook to behave as if it's running for the first time, start it using the /firstrun command switch (it's not listed in Outlook's Help, but it was there for Outlook 2007, so I assume it's there for Outlook 2010 as well.  I can't check right now).  Finally, you can completely uninstall Outlook, delete all mail profiles, and reiinstall Outlook if you want to start completely fresh.

    a. user say:

    This works in Word/Excel/Powerpoint:

    Click on File > Options

    Click on 'Quick Access Toolbar'

    Choose commands from:

    Select - 'All Commands'

    Scroll down to 'Enable Editing'

    Select it and Click on 'Add'

    An icon will appear at the very top left of Excel/Word/Powerpoint.

    Open a document that has the message 'Protected View This file originated from an Internet location and might be unsafe.....'

    Press Alt button a maximum of 4 times. This enables you to access the main menu items by typing a letter - but also allows you to access the 'Quick Access Toolbar' using numbers.

    The 'Enable Editing' icon looks like a notepad with a pencil, it should have a 2 next to it.

    Therefore the full command is Alt x 4 then press '2' for the pop up to go away.
