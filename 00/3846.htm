Adding Contacts

aljon_t say:

Hi

There are three available Skype applications for Windows 10 devices; classic Skype, Skype for Windows 10 and the new Skype.

See the steps below on how to add contacts using these Skype clients.

Classic Skype (7.40):

Adding a Skype account as contact.

  • Go to Contacts.

  • Select the add contact icon or click directly in the search text box and type the name, Skype name or email of the person you want to add. Click  Search Skype.

  • Select your friend from the search results and click Add to Contacts.

  • Type a quick note to introduce yourself and then click Send.

Adding a phone number as contact.

  • Go to Contacts and click Add contact, click Save a Phone number then Add number.
  • Select Call phones  > Dialer . Type the number. Click Save phone number to add it as a contact.
  • Save a number as a contact by calling it first. Click Call phones, dial the number and click the Call button. When the call has ended, click Add to Contacts.

The new Skype (8.xx):

Adding a Skype account as contact.

  • In the Chats screen, tap Search Skype.
  • Type the name, Skype name or email of the person you add as contact.
  • Right click on the account, then click "View Profile".
  • Scroll down, click "Add contact".

Adding a phone number as contact.

  • Call the phone number. (user does not need to answer the call)
  • On the recent view, right click the phone number then select View Profile.
  • Select Add contact.

Skype for Windows 10 (12.xx)

Adding a Skype account as contact.

  • In the Chats screen, tap Search Skype.
  • Type the name, Skype name or email of the person you want to chat with.
  • Select your friend from the search results and you'll be taken to a conversation window.
  • Click Start conversation, it will serve as a contact request.

Adding a phone number as contact.

  • Dial the number.
  • After that, the number will show on the recent activities.
  • Click the Phone number.
  • Under it, click "Save to local address book".
  • Then make the changes you want.

Hope this information helps. Feel free to post back if you have questions and clarifications.

suzanne s. barnhill say:

It would appear that you are creating the TOC manually. You may want to consider generating it automatically (see http://www.shaunakelly.com/word/numbering/tableofcontents.html), in which case Word sets the tab stop automatically.

For a manual TOC, you should not be using spaces to position the page number. You should set a right-aligned tab stop where you want the page number to be. You then press Tab to insert a tab character between the TOC entry and the page number. If you want period leaders, you can specify this in the Tabs dialog.

You might also want to start with the Manual Table of Contents from the Table of Contents gallery on the References tab. For more on this subject, see http://wordfaqs.mvps.org/ManualTOC.htm and http://WordFAQs.mvps.org/NumberAlignment2007.htm.

suzanne s. barnhill say:

If you neglect to select "(none)" as the tab leader when you initially insert the TOC (in the Table of Contents dialog), then you'll need to modify the styles. Since TOC styles all update automatically, all you [should] have to do is modify one entry at each level. Right-click in, say, a TOC 1 entry and choose Paragraph... from the context menu. In the Paragraph dialog, click the Tabs... button in the lower right corner.

In the Tabs dialog, select the tab stop (if there is more than one) that represents the page number position. Under Leader, select None, and click OK. This should correct all the TOC 1 entries. Repeat for the remaining levels.

If you find that the entries don't update automatically (the example document I was working in is being balky), then use Ctrl+Shift+S to open the Apply Styles dialog. This should display TOC 1 when you're in a Level 1 entry. Click the Modify... button, and use Format | Tabs to modify the tab leader setting directly in the style.

Hello Martin,

Thank you for choosing Microsoft Community and posting your query.

Provide additional information:

1) Are you able to download/save and then open the attachment.
2) Is any previewer is disabled in Outlook 2013? Check for the same by doing the following:
Open Outlook > click on File > Options > Trust Center > Trust Center Settings > Attachment Handling > Attachment and Document Previewers and check if any previewer is disabled.

Microsoft does not support installation of the MSI version of Outlook 2013 side-by-side with an earlier version of Outlook. Check the following Tech Net article for more information regarding the same:
http://technet.microsoft.com/en-us/library/cc179110.aspx (Outlook coexistence)

Hence existing of Outlook 2007 along with Outlook 2013 may cause conflict. Let's disable Outlook 2007 from Office installation and then check if the attachments work as expected in Outlook 2013. Refer to the following link for instructions on how to disable Outlook from Office installation:
http://office.microsoft.com/en-in/help/install-or-remove-individual-office-programs-and-components-HA001234067.aspx

If issue still persists, you may try the following instructions and see if it helps:

1) Start Outlook in safe mode and verify the result.

* Hold Windows key + R.
* Copy and paste, or type the following command in the Open box, and then press Enter: Outlook /safe
Note: There is space between Outlook and '/'.

If Outlook safe works, disable the add-ins following steps mentioned below and determine the problem causing add-in:

a) Click on File menu, click Options, Add-ins, Go button besides Manage: Com-in Add.
b) Check if there are add-ins listed, clear the check box.
c) Close the Office program and restart it.
d) Disable each add-in one at a time, restart the Outlook, and repeat the above procedure. If the issue reappears again, it will help you to determine which add-in is causing this issue.

Note: Check if you're able to download/save and then open the attachment.

2) If issue still persist, create new profile and verify the result. You may refer to following Microsoft article for instructions on how to create new profile in Outlook: http://support.microsoft.com/kb/2752583 (Method 1 till 6th step)

Hope it helps. Please take your time to try the suggestion and let me know the results at your earliest convenience.

Thank You.
