andre da costa say:

Unfortunately, 64 bit versions of Windows, including Windows 8 64 bit cannot run 16 bit applications because it does not have a 16 bit subsystem. You will need to install a Virtual Machine program and run the software on a 32 bit version of Windows, preferably Windows XP or an older version such as Windows 98.

You can use alternatives such as Oracle VirtualBox or VMWare Player.

You need to shrink the D partition and extend C into the free space. Windows 7 has a built-in utility to do this HOWEVER, it is severely limited in that it won't shrink a partition left to right, so that the free space is next to the C drive (which is a requirement to extend C), nor will it MOVE the D partition in order to accomplish the same task.

You will need a third party Partition Utility to do this.

There are several around - my preference is for GParted which has served me well and is free.
