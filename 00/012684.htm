henry van der wielen say:

How do I add SharePoint as the storage account in Outlook?

Split from this thread.

I also have this problem. In the desktop version of Outlook it's no problem to access documents in libraries in a SharePoint Online Team Site and add them as a link or attachment to a new email.

But in Outlook Online the only Storage Account available is OneDrive and there is no option to add a SharePoint Team Site. Only third party cloud storage services like Dropbox can be added.

There are requests from users about this problem for about two years now.

It's very strange Microsoft does not address this issue because more and more customers are migrating to SharePoint Online.

OneDrive uses the same backend as SharePoint so it should not be very difficult to implement.

Microsoft is making a fool of itself by promoting third party storage solutions over their own products and let this bug drag on.

Please act on it and add SharePoint Team Sites to the storage accounts!
