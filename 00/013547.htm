You should split your database into a backend (with the tables in) and a front end (with all the other objects in and linked tables to the backend).

I don't think you can suppress that prompt. If you split the database so the tables are in the backend and make that read-only and take the read-only off the front end I don't think you will get that prompt.

Actually, splitting the DB is a good idea anyway when you have multiple users. After splitting, give each user his own local copy of the front end.
