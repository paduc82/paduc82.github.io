How do you move files to a folder in One Drive using a Mac - Microsoft Community

jerry_1953 say:

How do you move files to a folder in One Drive using a Mac

I am using Word 365 on my Mac and I can not figure out how to moved files that I have saved to newly created folders on the One Drive.

a disciple of christ say:
  1. From your computer, click File > Share > Save to OneDrive.

  2. If this is the first time that you have tried to access OneDrive, type your Microsoft account and Password, and then click Sign In.

    If you use Hotmail, Messenger, or Xbox Live, you already have a Windows Live ID. If you don't have one, click Get a Live ID to create a new Windows Live ID.

  3. In the Save As box, enter the name of your document.

  4. Under Personal Folders or Shared Folders, click the folder where you want to save the document, and then click Save.
    https://support.office.com/en-in/article/save-a-document-to-sharepoint-or-onedrive-115cf002-4a39-4c5c-8103-fbd53a3f166b

john mcghie say:

As an added note, you can't "move" files, or conduct any file operations other than Open and Save, from within a Mac application.

On the Mac, you must use the Finder for file management operations.

If you look in your Applications folder, you should find a OneDrive application.  If you haven't got it, get it from the Mac App Store.

If that is running, your OneDrive should appear in the Finder, enabling you to drag files around as you wish.
