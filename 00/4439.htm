1.Declare a private Variable of type boolean for your whole Form

2.Set its Default Value to False by using the Form_Current event

3.Use the Before Update Event to check if mSaved = true or false and react

4.Use the Saved Button to change the value of mSaved!

 

Use this Code in Your Form: 

Private mSaved As Boolean

Private Sub Form_BeforeUpdate(Cancel As Integer)
If mSaved = False Then
    Cancel = True
    Me.Undo
    Cancel = False
End If
End Sub

Private Sub Form_Current()
mSaved = False
End Sub


Private Sub YourButtonName_Click()
mSaved = True
MsgBox "Changes saved!"
End Sub

 

Instead of YourButtonName you Have to use the name of your button ;)

How to rank with missing data? - Microsoft Community

jdonnenfi2 say:

How to rank with missing data?

I have 180 subjects that I'd like to rank based on their performance in a variety of categories. However, not every subject is present in each category. Only one of the categories has every subject present while others have only 80, 50, or 10. Is there a reasonable way for me to rank these 180 subjects based on their overall performance across the categories?

bernie deitrick say:

Let's say that you have four scores for your categories, in column B to E, with subjects listed in column A.

The first option is a simple equal weight to each the scores (a simple average)

=IFERROR(AVERAGE(B2:E2),"")


The second option is to weight the categories - with weights given in row 2 - and use a formula like this to get a weighted average for the subject

=SUMPRODUCT(Scores,Weights)/SUMIF(Scores,"<>",Weights)

Like this

=IFERROR(SUMPRODUCT(B3:E3,$B$2:$E$2)/SUMIF(B3:E3,"<>",$B$2:$E$2),"")
