a. user say:

Only Groups created in Outlook or the Admin Center show up in Outlook Desktop or OWA.

Groups created with Teams should show up in the Teams app or website.

a. user say:

The Office 365 groups created by teams are hided in the Exchange Online address book by default, so you cannot see them in the Outlook Client or Web.  However, you can try to use Exchange Online PowerShell to make them available in the Outlook.  Use powershell run  set-UnifiedGroup -identity "Office365 group name" -HiddenFromExchangeClientsEnabled:$false    replace the Office 365 group name with your group email address.  For more details, you can refer to https://techcommunity.microsoft.com/t5/Microsoft-Teams/Teams-do-not-create-365-group/m-p/210239#M14980.
