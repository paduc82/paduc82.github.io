somasundaram pandiaraj say:

Try the following steps.

1)     Click Start > All programs > Accessories > Run

2)     Type in C:\Program files\Microsoft Office\Office14

3)     Locate the file Winword.exe

4)     Right click on the file and choose Properties > Compatibility

5)   Enable the option Run this program as an Administrator

 

If the above step does not work then you could create a new windows user account using the following steps.

1)     To open User Accounts, click the Start > Control Panel > User Accounts and Family Safety > User Accounts

2)     Click Manage another account.  If you're prompted for an administrator password or confirmation, type the password or provide confirmation.

3)     Click Create a new account.

4)     Type the name you want to give the user account, click an account type, and then click Create Account.
