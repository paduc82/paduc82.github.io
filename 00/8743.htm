pamcaswell say: First , please do this.  In your test document, go to  Styles pane > Manage Styles button > Set Defaults tab.  There make these settings: 11 point Arial font, left aligned, 9 points space before paragraphs, 0 points space after.  When you OK out of the dialog, you should see the fonts you set in the table style.

Just a little background (revised a little from a previous post).  By design and initially, the defaults, the table normal (and table grid, which is based on it) table style, and the normal paragraph style have the same font settings.  So if the font size is 12 points in the defaults, it will be 12 points in table normal and in the normal paragraph style.  In other words, every table you create will be in 12-point type.

If  tables must appear in a different font size (or font, etc.),  you can do that by 1) applying a paragraph style to the table text and not change the font settings in the table style (this is what I still do because many of the business documents I prepare must be in W2003 format and it's the only way  to make table styles work in W2003) or 2) modifying  table grid or creating a custom style that specifies the font size needed. 

**Everything works the way I described if we change the document's regular text fonts, style sets, line spacing and such either from the change Styles icon on ribbon--as MS intends us to do for W2007-13-- or by changing the defaults manually.

But many of us learned Word when the typical/recommended way to change formatting was to  modify normal. Big problems in W2007-13...because changing normal does not also change the defaults (probably for good reason) and when normal is different from the defaults, its paragraph settings override the font settings in the table styles.  Even changing the font with direct formatting can fail.  Read more about styles hierarchy here. 

So the most important thing to know about tables is that the styles in the document must be set up for W2007-13.  From there, learning table styles by doing is pretty  straightforward.  Another Word blog explains about  table styles and conditional formatting.   Please post back if you have more questions.

jay freedman say:

After changing the placeholder text as Suzanne described, turn off the Design Mode button.

To change the appearance of the user's response, click in the field (in Microsoft-speak, it's a content control rather than a field) and click the Properties button on the Developer tab. In the Properties dialog, check the box for "Use a style ..." and then click the New Style button.

In the Create New Style dialog, enter a name for the new style (here I've typed "Response") and select the Italic and the desired font color. Click the "New documents based on this template" button and click OK.

Click OK in the content control's Properties dialog and save.

You'll have to go to the Properties dialog of each content control, open the dropdown, and select the style you created.

There's one thing you need to be aware of: If you're creating the form in a template (*.dotx or *.dotm), the "based on this template" option causes the new style to be saved in that template. If you're working in a regular document based on the Normal.dotm template, though, the style will be saved in your Normal.dotm template -- which means that it won't be available to users on other computers (or even in other profiles on the same computer). And if you choose the "Only in this document" option, the style will be saved only in that document's file. In general, anything you expect to send to other computers should be based on a specific template from which new documents are created and then sent out.

phillip pio say:

Thank you for your response Sherri,

We are thankful that the article that we provide to you was very helpful. About your second query on how do we get the contents of an adjacent cell to automatically change the color of cells around it. In order for us to help you with this, let's say that the value in cell A1 in the "Status" column is changed to "Complete". Therefore, the entire row 1 should be highlighted in green. To achieve this, please follow these steps:

  1. Select row 1 or all the rows where you want to apply the conditional formatting.
  2. Click Home > Conditional Formatting > New Rule > Use a formula to determine which cells to format.
  3. Under Format values where this formula is true, type $A1="Complete".
  4. Click Format > Fill > Select the color green.
  5. Click OK > OK > Apply.

You can also check this article: Use formulas with conditional formatting. It contains troubleshooting steps on how to use formulas with conditional formatting.
