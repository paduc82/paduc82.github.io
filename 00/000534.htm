Hi, I was wondering if there was a way to set a tab order in an Excel worksheet, so cells that are rarely used can get skipped over. hansv mvp say:

One option is to protect the worksheet.

Step 1:

Select the cells the user should be able to edit. You can hold down Ctrl to select non-adjacent cells.

Press Ctrl+1 to activate the Format Cells dialog.

Activate the Protection tab.

Clear the check box labeled Locked. (By default, all cells in a worksheet are locked, but this only becomes effective when you protect the sheet).

Click OK.

Step 2:

Activate the Review tab of the ribbon.

Click Protect Sheet in the Changes group.

Use the check boxes to specify what the users will be allowed to do.

Make sure that you leave the check box "Select unlocked cells" ticked!

If you wish, specify a password that will be needed to unprotect the sheet. If you do, make sure you remember the password!

Click OK. If you specified a password, you'll be asked to enter it again as confirmation.

Pressing Tab will now cycle through the unlocked cells only.

gord dibben say:

Another method using VBA Event code in the worksheet module.

Private Sub Worksheet_Change(ByVal Target As Range)
'Anne Troy's taborder event code
    Dim aTabOrd As Variant
    Dim i As Long

     'Set the tab order of input cells
    aTabOrd = Array("A5", "D9", "C1", "A10", "G13", "E12", "H5")

     'Loop through the array of cell address
    For i = LBound(aTabOrd) To UBound(aTabOrd)
         'If the cell that's changed is in the array
        If aTabOrd(i) = Target.Address(0, 0) Then
             'If the cell that's changed is the last in the array
            If i = UBound(aTabOrd) Then
                 'Select first cell in the array
                Me.Range(aTabOrd(LBound(aTabOrd))).Select
            Else
                 'Select next cell in the array
                Me.Range(aTabOrd(i + 1)).Select
            End If
        End If
    Next i

End Sub

dale howard [mvp] say:

Stormy Girl --

I have not answered a question for you in a long time.  The answer to your question is that you can create an Excel visual report to show resource availability.  Here is what you would need to do:

  1. Create the new Visual Report template for Excel using the Resource Usage data.  The fields you need for your report will already be in the template (Work Availability, Work, and Remaining Availability).
  2. Add the Resources field to the ROWS drop area.
  3. Add the Weekly Calendar field to the COLUMNS drop area.
  4. Add the Work Availability, Work, and Remaining Availability fields to the VALUES drop area (in that exact order).
  5. Expand your time dimension in the PivotTable to the level of detail you want to see.
  6. Save the Excel workbook as an Excel template.

Keep in mind that this report will only display the time span that matches the Duration of the project you have open, and the column headers will show items like Week 14 and not the actual dates.  Those are limitations of using Excel Visual Reports for this purpose.  Hope this helps.
