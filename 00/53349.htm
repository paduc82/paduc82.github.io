First, TITTAYCSIMHC, if you're expecting the SYSTEM_THREAD_EXCEPTION_NOT_HANDLED BSoD to appear on your computer, this indicates that a system thread generated an exception which the error handler did not catch. According to the BSoD information you provided, 0xFFFFFFFFC0000005 in Parameter 1 more likely indicates an access violation (or segmentation fault), meaning that the system thread is trying to write to memory where it is not allowed to write to.

If you're trying to boot and/or install Windows 7 for the first time on a very modern computer, and are experiencing problems, you'll have to do a clean boot. To do so, follow the steps in this Microsoft Support article:
