hansv mvp say:

It's normal and to be expected that Drawing tools appears - each control is also a shape, and Drawing Tools is displayed when a control is selected.

 

I think that you have created the controls from the ActiveX section of the Insert button on the Developer tab of the ribbon. If so, you *must* turn on Design Mode (the button next to Insert) in order to edit the controls. When Design Mode is on, the right-click menu of the controls displays items such as Properties and View Code. When Design Mode is off, the controls don't react to a right mouse click.
