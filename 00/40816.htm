Windows stored the Wifi password in an encrypted form, but for some reason (security?) deletes it each time the PC reboots.

Find the folder below

C:\ProgramData\Microsoft\Wlansvc\Profiles\Interfaces

When windows reboots the files can't be deleted and the pc should automatically connect to your WiFi

Windows Store Process wsappx.exe Causes High CPU Usage / High Disk Usage – Solved

After March 2018 service update for Windows 10 1709, Windows Store was stuck updating causing permanent high CPU usage around 18% associated with permanent fan activity.

Booting into save mode would bring the process back to normal levels until the affected account was logged in again. 

Ive tried all of the following solutions just to find out that they didnt work for me. Only creating a new user account as a last-resort solution would fix the problem on my system.

There are some 20+ threads on Microsoft Answers concerning this issue thats why I want to share my experience to save other users some time.

Solution that worked:

  • Creating a new user account from scratch

Steps that didnt work:

  • Clear Windows Store cache (no effect)
  • Run Windows Update troubleshooter (no effect)
  • Run Windows Store Apps troubleshooter (no effect)
  • Kill process from task manager (not possible)
  • Disable process under Group Policy Objects (not possible)
  • Running Check Disk (unrelated errors found and fixed)
  • Running Deployment Image Servicing and Management (DISM) (unrelated errors found and fixed)
  • Installing Windows Store from PowerShell from affected account (impossible, stuck at 0%)
  • Installing Windows Store from PowerShell from another admin account (successful but to no avail)
  • Uninstalling Windows Store from PowerShell from affected account (made it worse, uninstalled only half way)
  • Repair upgrade of Windows 10 from ISO (problem persisted)

The lesson of this episode: There should be an option for Windows Pro users to get rid of Windows Store and pre-installed junk apps and games altogether. The only apps I ever use is Calculator, Weather and Sports, but I could do without them.

Good luck!
