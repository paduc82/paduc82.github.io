andreas killer say:

Unfortunatelly this is not the only problem, if the placement is set to xlMove and a row/column that touch the control is hidden, the control moves after reopening/unhide the rows/columns.

The only way out of this is to set the placement to xlFreeFloating and hide the affecting control when you hide rows/columns.

Unfortunatelly we can not detect if somebody hide rows/columns manually, so the only way is to hide/show rows/columns with a macro.

But as you said you hide your rows with a button, this routine may a solution for the interim.

Andreas.

Option Explicit

'if SavePlacementState then the current placement is re-/stored in the AlternativeText property of the shape object
#Const SavePlacementState = True

Private Sub Example_Hide()
  HideCells Range("7:11")
End Sub

Private Sub Example_Show()
  HideCells Range("7:11"), True
End Sub

Sub HideCells(R As Range, Optional ByVal Show As Boolean = False)
  'Hide or show rows/columns and intersecting OLEobjects too
  Dim O As OLEObject
  Dim DoRows As Boolean, DoCols As Boolean
  Dim C As Range
  DoCols = R.Rows.Count = Rows.Count
  DoRows = R.Columns.Count = Columns.Count
  If Not DoCols And Not DoRows Then
    'Hide/show both
    DoCols = True
    DoRows = True
  End If
  If Show Then
    'Show rows/columns
    If DoRows Then R.EntireRow.Hidden = Not Show
    If DoCols Then R.EntireColumn.Hidden = Not Show
  End If
  'Visit each OLEobject inside the parent sheet
  For Each O In R.Parent.OLEObjects
    'Did it touch our rows/columns?
    Set C = Range(O.TopLeftCell, O.BottomRightCell)
    If (DoRows And Not Intersect(R.EntireRow, C) Is Nothing) Or _
        (DoCols And Not Intersect(R.EntireColumn, C) Is Nothing) Then
#If SavePlacementState Then
      If Show Then
        'Restore previous state
        If IsNumeric(O.ShapeRange.AlternativeText) Then _
          O.Placement = O.ShapeRange.AlternativeText
      Else
        'Save current state inside the object
        O.ShapeRange.AlternativeText = O.Placement
        'Only this state makes a restoring possible!
        O.Placement = xlFreeFloating
      End If
#Else
      O.Placement = xlFreeFloating
#End If
      'Show or hide the control
      O.Visible = Show
    End If
  Next
  If Not Show Then
    'Hide rows/columns
    If DoRows Then R.EntireRow.Hidden = Not Show
    If DoCols Then R.EntireColumn.Hidden = Not Show
  End If
End Sub

drewharrison say:

Outlook 2010/2013 From Field Drop-Down List Limit

There appears to be a limit to the number of addresses that the Outlook 2010/2013 From: field drop-down list will hold (10). If you want to send from an address not already in that list, you are forced to select the 'Other E-mail Address' option and then browse for the alternate address. Is there a way to increase the number of addresses held in the drop-down list?

andy_ji say:

Hi Partner,   

 

Thank you for posting in Partner Support Community. This is Andy and I will be assisting you with this technical issue.

 

From your description I understand that you’d like to know if there a way to increase the number of addresses held in the drop-down list If my understanding is not correct please let me know.


Based on my research we can change the limit for the AutoComplete list by the following steps:
1. Exit Outlook.

 

2. Start Registry Editor.

In Windows 8.x, press the Windows key, type regedit and then press Enter.

In Windows 7 and Windows Vista, click Start, type regedit in the Start Search box, and then press Enter.

 

3. Locate and then select the following registry subkey:

HKEY_CURRENT_USER\Software\Microsoft\Office\x.0\Outlook\AutoNameCheck

 

Note The placeholder x.0 in this registry subkey represents your version of Microsoft Office. Use the appropriate value from the following list.

 

Outlook 2016 = 16.0

Outlook 2013 = 15.0

Outlook 2010 = 14.0

Outlook 2007 = 12.0

Outlook 2003 = 11.0

 

4. On the Edit menu, point to New, and then click DWORD value.

 

5. Type MaxNickNames, and then press Enter.

 

6. On the Edit menu, click Modify.

 

7. Type the new value for the limit, and then click OK.

 

Notes

Make sure that you type the number in decimal form. That is the correct form in which to type the number.

Try increasing the limit by only a marginal amount to test the new limit. For example, to create a 20 percent increase in the limit in Outlook 2013, you would specify 1200 for the MaxNickNames value.

 

8. Exit Registry Editor.

 

9. Start Outlook.


For more information about it you can refer to the following link:
Information about the Outlook AutoComplete list
https://support.microsoft.com/en-us/kb/2199226

vincentchoy say:

MS Office is a separate product from Microsoft Windows 10 and is priced separately.

Windows 10 is a operating system whereas MS-Office is a suite of application software.

A operating system is a piece of software that makes your PC/Notebook work.

An application software sits on top of the operating system delivery specific functions  (like word processing, spreadsheet..etc)

MS Office 2010, 2013 and 2016 will work with Microsoft Windows 10

https://support.office.com/en-us/article/Which-versions-of-Office-work-with-Windows-10-0fc85c97-da69-466e-b2b4-54f7d7275705?ui=en-US&rs=en-US&ad=US

If you own one of the above mentioned MS-Office licenses there is no need to purchase a new one.

However, since you are changing the underlying operating system to your PC/Notebook you might need to re-install your existing MS-Office, which means the CD and license keys need to be available.

Best have a discussion with a experienced IT technician about this exercise.

garym123 say:

Phone backup started including app data. The developer of the app had to include this feature in the app. Some apps that have not been updated with this feature or incorrect implementation when the app was updated causes this problem. Social app backup is important to those users. For other apps the app data is just the preferences you setup and your location, maybe app passwords, etc.

Phone backup includes phone settings, app list and app data. SMS message backup is a separate feature that happens every time you receive a text message.

When you get to about 95% complete and it fails it is because of app data.

You can see that your phone backup is stored on your OneDirve at https://onedrive.live.com  gear icon, options, device backups.  You see the device name and date of the backup.

If your phone backup is listed it will work with the exception of the affected data for one or more apps, note the date it was stored.

You can delete the backup and try again. If you get the same result you know the app data is causing the problem.

jomel_g say:

Hi DanSussman,

When you have an active subscription on your Skype account that is attached to the payment method you want to remove, you will really get that kind of error. For you to remove the card, you need to change the way you pay for the recurring charge. You can follow the below steps on how you can update your payment method for a recurring plan:

  1. Under your profile picture, click the subscription you want to change the payment method for.
  2. Click Change payment method.

After successfully updating your payment method for a recurring subscription, you can use the below steps to remove your card from the account:

  1. Sign in to your account at skype.com.
  2. Scroll to the bottom of the page and, under Account details (Billing and Payments), select Stored payment details.
  3. Click Delete next to the payment method you want to remove, and then click Delete on the confirmation page.

For further details, you can utilize this support article: https://support.skype.com/en/faq/FA10486/ 

techfreak_ say:

Go to all your subscription/auto-recharge details and set it to the new card, not just your stored payment details. Once you do, you should be able to delete the old card. If it still will not remove, then you may have to wait one billing cycle until the new card is charged for each of your subscriptions. Once it is, there should be nothing left tied to the card which should not allow you to remove it completely.

All your calling subscriptions will be in the left blue pane below your profile picture in your Skype account. Clicking on each subscription should allow you to change the payment method to the other card. 

If you have a Skype number, you can change the payment method in the Skype Number area of your Skype account.

If you have auto-recharge enabled, you can either disable it or assign it to your new card in the auto-recharge area of your Skype account.
