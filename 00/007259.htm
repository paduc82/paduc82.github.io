Hi JORGEALBERTOGARCIA

Most of the time this occurs because the account is synchronized to a mobile device that creates folders with the same name. For this reason, it is necessary to unlink the device account to remove the folders or change the name.

It can also occur when there has been a sudden shutdown of the computer.
if that is the case The first step is to access the folder where Outlook Express emails are physically stored. If we do not know where they are stored, the steps to follow would be the following:

1. In Outlook Express, go to menu "Tools>" Options ... "> tab" Maintenance "> button" Warehouse folders ... "
2. It indicates the route where the messages are stored. We have to copy that route, so with the mouse we click from the beginning (usually starts with C: \ Documents and Settings \ ...) and drag to the end (usually ends in ... \ Outlook Express). Once we have the entire route selected, press Ctrl + C to copy the content.
3. We click on Accept and Accept again, and close the Outlook Express.
4. We go to Windows Start button, we choose the menu option "Run ..."
5. And in the Open field, if there is something written, we delete it, and then press Ctrl + V, in order to paste the content that we have copied in step 2.
6. Click on OK, and we open the folder where all the emails are stored. Here you will find files that are called as the Outlook tray, "Inbox.dbx", "Outbox.dbx", "Sent Items.dbx", ... (the .dbx extension may not be visible , since it depends on how the folder options are configured)

The second step is to analyze the damaged file and rename it. The steps to follow are these:

1. Select the file "Inbox", click on the right mouse button and choose the option "Properties". In the information that shows us in the "General" tab, it indicates the size of the file. If the size is 1.99 GB then we have reached the maximum limit of the file, as we have already mentioned. If the size is lower, we simply do not see the emails or we can not move them or delete them, because the file is damaged or corrupted.

2. Close the Properties window, and select the "Inbox" file again, which we suspect is damaged, and change the name. To do this, press the F2 key, and let us edit the name of the file. We can simply add a number, for example, "inbox2.dbx".

Finally, you can only open Outlook Express again. The program no longer finds the tray (because we have changed its name), and generates a new one, although empty. As this tray that is generated is no longer damaged, the problem is solved and we will be able to receive emails again
