The simple solution is to move the taskbar, possibly auto-hide it, and drag the email up from behind the taskbar area back on to the normal desktop area.  Magically, everything works normally again.

jay freedman say:
I can't reproduce the disappearing tab leaders, so I'm not sure what that's caused by.

It may help in the diagnosis if you take a look at the field code created by the Insert Index dialog. To see it, click in the index and press Shift+F9, or right-click it and choose Toggle Field Codes.

After choosing the right-aligned page numbers, the field code should look like

{ INDEX \e "    " \h "A" \c "2" \z "1033" }

and if you click the button on the Home ribbon, there will be an arrow between the first pair of quote marks to represent a tab character. (When page numbers are not right-aligned, there are spaces between those quote marks.) The meanings of the "switches" are given at http://office.microsoft.com/en-us/word-help/field-codes-index-field-HP005186209.aspx. For some reason the presence and type of the tab leader aren't part of the field code; apparently that information is hidden inside the tab code. (The Modern format gets translated into changes in the definitions of the styles named Index Heading and Index 1 through Index 9, and aren't part of the field code at all.)

If your index template works except for the absence of index headings, verify that the field code includes the switch \h "A" and that the Index Heading style is defined.

Create a user account

If the issue does not exist in new user account, you may have corrupted user profile. You may follow the steps to troubleshoot a corrupted profile.

a) Press Windows and x together and select Windows Explorer.

b) Click on View tab and Click on Options at the top right corner.

c) Click the View tab, and then click Show hidden files, folders, and drives.

d) Uncheck the box which states Hide protected operating system files.

e) Click Yes to confirm, and then click OK.

f) Locate the C:\Users\Old_Username folder, where C is the drive that Windows is installed on, and Old Username is the name of the profile you want to copy files from.

g) Select all of the files and folders in this folder, except the following files:

Ntuser.dat

Ntuser.dat.log

Ntuser.ini

h) Right-click and Select Copy.

i) Locate the C:\Users\New_Username folder, where C is the drive that Windows is installed on, and New Username is the name of the new user profile you created.

j) Right-click inside the folder and select Paste.

k) Restart your PC and check if you are able to log in to default profile.
