And Jim G. joined Microsoft and had over 1000 words in it. 

What I did to solve the problem was: 
  1. create a new custom dictionary using the options button in the spelling set up..  
  2. Added a word just to make sure it was working.
  3. Navigated to  ~/Library/Application Support/Microsoft/Office/Preferences/Office 2011 
  4. Opened the old non-working dictionary and the new dictionary in a text editor like TextEdit
  5. copied and pasted all of the words from the old file into the new file (replacing the new word because it appears everything needs to be in alphabetical order.  Note that if my old dictionary was corrupted this would have copied the corruption.
  6. Saved and quit
  7. Relaunched Word and all is well.
