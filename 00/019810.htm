Try a table-level validation rule (in the Property Sheet for the table):

 

[Field1]="Other" And [Field2] Is Not Null Or ([Field1]<>"Other" Or [Field1] Is Null) And [Field2] Is Null

 

Or handle it in the Before Update event of a form based on the table.

THANKS to all of you for steering me in the right direction. Here is what finally seems to work:

 

Private Sub CourtDisposition_AfterUpdate()
If Me.CourtDisposition.Value = "Other" Then

Else

End If
If Me.CourtDisposition.Value <> "Other" Then
    Me.CourtDispositionOther.Visible = False
End If

End Sub
