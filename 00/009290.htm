scott help say:

Time Format Minutes Seconds Decimal nn:ss.00

In MS Excel I have a time format of mm:ss.00 to display a time in minutes, seconds and the seconds decimal point for recording running times.  In MS Access I formatted the cell with nn:ss.00 to display the time but it won't return the decial point for the seconds.  How do I get MS Access to return the seconds with the decimal point?

For example, for a time of 17 minutes and 5.22 seconds for a 5K race I get the following.

MS Excel formatted with mm:ss.00 I get 17:05.22

MS Access formatted with nn:ss.00 I get 17:05.00 and the format is changed to nn:ss".00"

How do I define the format to get MS Access to return 17:05.22?

Thank you!

john w. vinson say:

Access, unlike Excel, records Date/Time values only to the second. Fractional seconds are not stored.

In fact, Date/Time fields are best used to store specific points in time, not durations. Not only can you not store sub-second values, but it's hard to display times exceeding 24 hours.

For race durations, I'd suggest storing the run time in a Double Float count of seconds - e.g. 17:05.22 would be stored as 1025.22. You can display this on a form or report with a calculated field such as

[racetime] \ 60 & Format([racetime] - 60*([racetime] \ 60), "00.00")

Important The below instruction involves in modifying registry. Hence make sure to back up the registry before modifying it.


1) Open Registry

a) Hold Windows key + R.
b) Copy and paste, or type the following command in the Open box, and then press Enter:
regedit

2) Back up the registry following below mentioned instruction:

Click on File, Export, name the backup and then click on Save.
Note: Ensure that All is selected in Export range.

3) Now navigate to HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Office\Excel\AddIns

Select Master Data Service add-in and in right pane double click on LoadBehavior and change Value data to 3

3) Now open Excel and verify whether Master Service Data add-in is enabled. If disabled, enable it and restart Excel and verify the result.
