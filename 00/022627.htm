Hi Leopold,

The email alias within this domain @xxx.onmicrosoft.com is not synced by Directory Synchronization service. Moreover, the user is synced with on-premise AD, and we cannot directly remove or add email alias.

We need to temporarily disable/deactivate Directory sync, remove the alias and add a new alias. Then, please enable directory sync again.
