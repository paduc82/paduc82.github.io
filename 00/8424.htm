tom van stiphout say:

john w. vinson say:

253 fields isn't "not ideal". It's DREADFUL!

I've needed as many as 60 fields in a table (once, kicking and screaming). I'm all but certain that you have some (probably several) sets of repeating fields and that your data is in need of normalization!

That said... a completely blank Form (or Subform) means that the form's Recordsource query doesn't return any records (so it's not showing existing data) and is not updateable (so it's not showing the new blank record either).

What are the Recordsources for your Form and Subform? What's the Master/Child Link? And what are a few representative fieldnames?

john w. vinson say:

You'll have to use the table with the Autonumber primary key as the mainform's recordsource, and use a Long Integer primary key for the second table (the one in the subform). An Autonumber cannot be written to - and the Master/Child Link of a Form/Subform will try to do just that.

The subform will be able to link to existing records with a matching ID, or to add a new ID by copying the parent form's Autonumber value - but not the other way around.

bellow is new code. I added the option to do all the clients in master sheet at ones or do only the selected one.

code:


Sub new_sheet_client_2()

Dim clients_() 'clients name series
Dim header_() 'header series
Dim i As Integer 'series label
Dim y As Integer 'another label
Dim f As Integer 'another...

all_or_single = InputBox("selected client only = 1 , all clients in master sheet = 2")


'disable any alerts temp and screen updating
    Application.DisplayAlerts = False
    Application.ScreenUpdating = False


If all_or_single = 2 Then
master_sheet = ActiveSheet.Name 'defines which is the master sheet
columns_ = Application.WorksheetFunction.CountA(Range("1:1")) 

ReDim header_(1 To columns_)
For f = 1 To columns_
header_(f) = Cells(1, f).Value
Next f

rows_ = Application.WorksheetFunction.CountA(Range("A:A")) 'defines how many rows there are
Range(Cells(2, 1), Cells(rows_, 1)).Copy ' copy all clients names
Sheets.Add , ActiveSheet 'adds sheet
temp_sheet = ActiveSheet.Name
Cells(1, 1).Select
ActiveSheet.Paste 'paste header
ActiveSheet.Range(Cells(1, 1), Cells(rows_ - 1, 1)).RemoveDuplicates Columns:=1, Header:=xlNo 
how_many_clients = Application.WorksheetFunction.CountA(Range("A:A")) 

ReDim clients_(1 To how_many_clients) 'defines clients name series
For i = 1 To how_many_clients
clients_(i) = Cells(i, 1).Value
Next i

ActiveSheet.Delete

For i = 1 To how_many_clients
Sheets.Add after:=Sheets(Sheets.Count) 'adds sheet
ActiveSheet.Name = clients_(i) 'rename sheet to client's name
For f = 1 To columns_
Cells(1, f).Value = header_(f)
Next f
Cells(2, 1).Select

Sheets(master_sheet).Select
For y = 2 To rows_
If Cells(y, 1).Value = clients_(i) Then
    Range(Cells(y, 1), Cells(y, columns_)).Copy
    Sheets(clients_(i)).Select
    ActiveSheet.Paste
    ActiveCell.Offset(1, 0).Select
    Sheets(master_sheet).Select
    End If
Next y
Next i

End If

If all_or_single = 1 Then
client_name = ActiveCell.Value 'defines client's name
client_location = ActiveCell.Row 'defines in which row from the master sheet is the client
master_sheet = ActiveSheet.Name 'defines which is the master sheet
columns_ = Application.WorksheetFunction.CountA(Range("1:1")) 'defines how many fields there are
rows_ = Application.WorksheetFunction.CountA(Range("A:A")) 'defines how many rows there are
client_rows = Application.WorksheetFunction.CountIf(Range(Cells(2, 1), Cells(rows_, 1)), client_name) 
Range(Cells(1, 1), Cells(1, columns_)).Copy ' copy the headers to be pasted in new sheet
Sheets.Add , ActiveSheet 'adds sheet
ActiveSheet.Name = client_name 'rename sheet to client's name
Cells(1, 1).Select
ActiveSheet.Paste 'paste header
Sheets(master_sheet).Select 'back to master sheet
Range(Cells(client_location, 1), Cells(client_location + client_rows - 1, columns_)).Copy 
Sheets(client_name).Select 'back to client's sheet
Cells(2, 1).Select
ActiveSheet.Paste 'paste client's info
Cells(1, 1).Select
End If


'enable any alerts temp and screen updating
    Application.DisplayAlerts = True
    Application.ScreenUpdating = True



End Sub
