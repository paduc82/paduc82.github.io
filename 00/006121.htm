jay freedman say:
Hi Bob,

I don't know a whole lot about the Excel object model, so it took me a little while to get this far. It probably isn't the best way to do it, so you should probably ask in the Office Programming forum (at the top of this page, set the topic to "Microsoft Office Programming") and see who thinks I'm out of my depth...

I made a Word template named "bookmarked.dotx", containing some text and a bookmark named "bk1". I put some data into A1:B5 of Sheet 1 in a workbook named "data.xslx". These files are stored in the default locations for a Word template and for an Excel workbook, respectively. This much is hard-coded in the macro and would need adjustments for your setup.

The central operation is (a) copying the source range from the worksheet to the clipboard and (b) using the PasteExcelTable method of the bookmark's range. That method gives you arguments to control linking and formatting. A more complex alternative would be to use the PasteSpecial method.

Before you try to run this macro, click Tools > References and set a reference to the Microsoft Word Object Library.

Sub CopyRangeToWord()
    Dim wd As Word.Application
    Dim doc As Word.Document
    Dim templ As String
    Dim srcWkBkName As String
    Dim srcWkBk As Workbook
    Dim srcRg As String
   
    srcWkBkName = Application.DefaultFilePath & "\data.xlsx"
    srcRg = "A1:B5"
   
    On Error GoTo bye
   
    ' open Word and make a document based on the template
    Set wd = CreateObject("Word.Application")
    wd.Visible = True
    templ = wd.Options.DefaultFilePath(wdUserTemplatesPath) & "\bookmarked.dotx"
    Set doc = wd.Documents.Add(Template:=templ)

    ' open the source workbook and copy the data range (could be a table)
    Set srcWkBk = Workbooks.Open(Filename:=srcWkBkName)
    srcWkBk.Sheets(1).Range(srcRg).Copy

    ' paste the data into Word at the bookmark
    ' set the arguments as needed
    doc.Bookmarks("bk1").Range.PasteExcelTable LinkedToExcel:=False, _
        WordFormatting:=True, RTF:=False
   
    doc.Save
    srcWkBk.Close
bye:
    If Err.Number Then MsgBox Err.Description
    If Not wd Is Nothing Then
        wd.Quit
        Set wd = Nothing
    End If
End Sub

jay freedman say:
I didn't expect there to be anything at the bookmark before the Excel data was pasted in, so I didn't try that. It looks like both the PasteExcelTable method and the PasteSpecial method ignore the usual option in which typing or pasting replaces the content of the range, so I had to delete the existing table and then paste in the new content. This would make it very difficult to replace only part of an existing table.

Anyway, this version seems to work. Feel free to tell me how it fails for you. ;-)

    Dim wd As Word.Application
    Dim doc As Word.Document
    Dim templ As String
    Dim srcWkBkName As String
    Dim srcWkBk As Workbook
    Dim srcRg As String
    Dim destRg As Word.Range
   
    srcWkBkName = Application.DefaultFilePath & "\data.xlsx"
    srcRg = "A1:B5"
   
    On Error GoTo bye
   
    Set wd = CreateObject("Word.Application")
    wd.Visible = True
    templ = wd.Options.DefaultFilePath(wdUserTemplatesPath) & "\bookmarked.dotx"
    Set doc = wd.Documents.Add(Template:=templ)
   
    Set srcWkBk = Workbooks.Open(Filename:=srcWkBkName)
    srcWkBk.Sheets(1).Range(srcRg).Copy
   
    Set destRg = doc.Bookmarks("bk1").Range
    If destRg.Tables.Count > 0 Then destRg.Tables(1).Delete
   
    destRg.PasteExcelTable LinkedToExcel:=False, _
        WordFormatting:=True, RTF:=False
    doc.Bookmarks.Add Name:="bk1", Range:=destRg.Tables(1).Range
   
    On Error Resume Next
    doc.Save
    srcWkBk.Close SaveChanges:=xlDoNotSaveChanges
bye:
    If Err.Number <> 0 And Err.Number <> 4198 Then MsgBox Err.Number & "  " & Err.Description
    If Not wd Is Nothing Then
        wd.Quit SaveChanges:=Word.wdDoNotSaveChanges
        Set wd = Nothing
    End If
End Sub

I think this'll do it:

You need to modify the registry:

HKEY_CURRENT_USER\Software\Microsoft\Office\11.0\PowerPoint\Options

Background spell checking = 0 to turn it off, 1 to turn it on

You'll need to change the registry BEFORE starting PowerPoint.  Changing it after PPT has started will have no effect.
