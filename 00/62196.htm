Hi Kelvin11,

I'm Stefano an Independent Advisor, here to help you.
If Windows comes preinstalled on your laptop, I think your license is an OEM one, so it isn't transferrable. You can check if your license is an OEM one, following this procedure:

1. Type "cmd" in windows search bar
2. Right click on "Command Prompt"
3. Select "Run as Administrator"
4. Type "slmgr /dlv" without quote and press ENTER

Reade the uppercase word on Description line. If it is OEM your license isn't transferrable, if it is RETAIL, you can transfer the license.

If you need to buy a new license, however, I suggest you to buy it directly from Microsoft Store, that's the safest way to buy a proper license.
