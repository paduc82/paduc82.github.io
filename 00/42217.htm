zaki j say:

Hi Ironoxide,

There are a few stepa that we can try here which will remove the restart message coming up again and again. And whenever you restart it will install your updates.

Step 1:

You can set the settings such that you choose when to Install your updates. Whenever you complete your work you can Install the updates and it will restart on its own without bothering you.

Set Automatic Updates to “Download updates for me, but let me choose when to install them”. Go to Control Panel

> System > Automatic Updates tab.

Step 2:

Stop the "Automatic Updates" service.

Navigate to Control Panel | Administrative Tools | Services:

Right click the Automatic Updates service and stop it. You can also do the same thing at the command line by typing:

Net stop wuauserv

Or you can type this, which does the same thing, and is a little easier to remember:

Net stop "automatic updates"

After the service is stopped, the message stops, too. Then you can reboot when you have time. The service will restart when you reboot.

Modify Group Policy settings.

•          Go to Start Menu -> Run type “gpedit.msc” and press Enter

•          This opens the Group Policy editor

•          In this window go to: Computer Configuration -> Administrative Template -> Windows Components -> Windows Update

•          Double click on No auto-restart for scheduled Automatic Updates installations

•          In the settings window Choose Enabled and click OK

•          Close Group Policy Editor

The next time Windows Update updates your system it will not restart the computer automatically, but will notify you that restart is needed in order to finish the installation.
