hansv mvp say:

If you clear the "Set Intercept" check box, Excel will determine the line that best fits the data, in this case taking 1, 2, 3 as X values. "Best fit" means that the sum of the squares of the vertical distances from the line to the data points is minimized.

If you tick the "Set Intercept" check box and provide a value for the intercept, Excel will force the line to pass through the point with X = 0 and Y = intercept. This is used if you have theoretic reasons for setting a fixed intercept.

As far as I can tell, there is no reason to do so here, so you should leave "Set Intercept" cleared.

rich michaels - mvp (greatcirclelearning.com) say:

Control + Alt + M is most definitely a shortcut key assigned by Microsoft Word. In other words, you did not assign it.

The Control and Alt keys are modifier keys and they do not register as a keystroke unless pressed in conjunction with another key... that is why they cannot be assigned by themselves as a keyboard shortcut.

As to why the combo of Control + Alt + M stopped working becomes just a guess but here are a few options for you to consider:

  • If it's been quite awhile since you restarted your system and you have left Word open for many days, you could simply try restarting the computer and retesting the shortcut.
  • Keyboard shortcuts are stored in Word's Normal template. If the template has become corrupted you could rename the existing one and let Word regenerate a new Normal template. Do not delete the existing Normal template, just rename it because you'll probably want to copy over Styles, Custom Macros, etc from the old template to the new one. If you need more instructions about this, let us know.
  • The commands InsertAnnotation and InsertNewComment perform the same thing so you could simply assign the shortcut Control + Alt + M to the InsertNewComment command and that should work just fine... I tried it and I don't see a problem.

I hope this helps
