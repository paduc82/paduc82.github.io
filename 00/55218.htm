a. user say:

If you're talking about Windows Explorer, there's the option in Folder Options to open new folders in a new window (but I'm sure you've already checked that).  I'm not sure why it is not working on your system. However, you can add "Open Folder in a New Window" to the right click context menu so all you have to do is right click on the folder and then click that option (two steps instead of one or automatic, but it works).  Here's the procedure: http://www.askvg.com/add-open-in-new-window-option-in-folders-and-drives-context-menu-in-windows/.  Since this involves working with the registry, I suggest you first make a system restore point and then backup the registry as follows: http://www.adamsdvds.co.uk/tutorials/vista/regedit/backup.php.

Let's check some of your system files and your hard disk to see if there is any corruption:

Go to Start / All Programs / Accessories / Command prompt and right click on command prompt and click run as Administrator.

Type sfc /scannow and enter and let it run.  It will scan and try to fix some of your system files.  Hopefully it will complete with no corruption it could not repair (if there is such corruption post back here or try to analyze it to find the problem file(s) using http://support.microsoft.com/kb/928228.  Try to post any corrupted files here so we can see if they can be repaired with good copies from the installation disk (unless there are too many).

While in Command Prompt, type chkdsk /f /r and enter and let it run.  It may want to schedule itself to start at the next startup.  Agree and then reboot so the program can run. It will scan and try to fix any corruption or bad sectors on your hard drive and mostly remove that as a potential cause.

If SFC comes up with corruption it cannot repair, we'll need to do a system repair/upgrade using the genuine Windows Vista Installation Disk (one you own or one you can borrow from ANYONE).  Here's the procedure: http://www.vistax64.com/tutorials/88236-repair-install-vista.html.  Although this will not affect your data, settings, or programs, you should still backup your data before starting just to be on the safe side. If the other procedures don't work, THIS one is almost certain to work.  You may have a lot of updates to re-install (including any service packs you had to remove).  If the version on the system came with SP1 or SP2 pre-installed and the disk is an earlier version, then you'll need to make a slipstream disk as follows: http://www.vistax64.com/tutorials/151606-vista-sp1-slipstream-installation-dvd.html. 

====================================================================================================================

To open new folders in a new Window in Internet Explorer here are the general procedures: http://www.vistax64.com/tutorials/91587-open-folder-same-separate-new-window.html (though I have a feeling you've already done this).  If that doesn't work, then go to Tools / Internet Options / Tabs and disable Tabbed Browsing (then close all open IE windows).  This may actually make things worse for you if you're now using Tabs (as I am) but it might clear up the other problem if it sees no alternatives (I'm just guessing here).  If that doesn't work or makes things worse, then turn it back on, close all open IE windows, and the next one you open will have tabs back.

If it is Internet Explorer, here are two other options you can try (but be forewarned that the latter will delete your history and a lot of other stuff from your system - but it does give a warning message telling what it is going to delete and giving you the option to continue or cancel - so the decision is yours.  You can try them or you can go directly to the IE Forum to get their advice. 

Troubleshooting and Internet Explorer’s (No Add-ons) Mode [Applies to IE7 & IE8 in all OSS] - the issue may be caused by some add-on and this will help rule that out.
http://blogs.msdn.com/ie/archive/2006/07/25/678113.aspx

Does the behavior persist if you Reset IE Advanced settings (RIES)?
http://support.microsoft.com/kb/923737 

If the issue is with Internet Explorer (I'm not clear from your post), and the above doesn't work then I suggest you repost in the Internet Explorer Forum at: http://social.answers.microsoft.com/Forums/en-US/InternetExplorer/threads where people who are experts at IE will be more than happy to assist you with your problem. Be sure to include your version of Windows (for example, 32-bit Vista Business SP2) and your version of IE8 and everything you've tried to resolve the problem (so they don't give you advice you've already received or tried).

I hope this helps.

Good luck!

Hi Bill,

Sorry for the delay in response.

I'm leery about using System Restore points older than a week or so, though a quick backup will give you the ability to experiment without the threat of losing any data.

I don't think reinstalling the Norton after a restore will qualify as a second installation instance of the product. You may have to contact Norton to get it reactivated.

I think the license is pretty much tied to the motherboard, so a reinstall on the same hardware should be OK.

You're doing a great job troubleshooting this issue.

Before trying the System Restore, you might try creating a new profile to see if the issue exist there:

 

http://windows.microsoft.com/en-US/windows7/Fix-a-corrupted-user-profile

They are the drivers for the motherboard that cover the on-board stuff, like the card readers and the PS2 port.

I suggest checking with the computer manufacturer to see if there is a version on their website to download and install.

You might have a look for a BIOS firmware update while you're over there.
