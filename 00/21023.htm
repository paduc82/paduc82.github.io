jeeped say:

..  I've tried multiple different formulas (IF, SUMPRODUCT, RANK, etc), but can't seem to come up with the right combination ...


You need the equivalent of a RANKIF() formula which can be created with SUMPRODUCT().

               

The formula in D2 is,
=SUMPRODUCT(($B$2:$B$21=$B2)*($C2<$C$2:$C$21))+1
... and that can be filled down to D21. If it needs to be carried down further, the ranges will have to be extended.

Lock field for single record - Microsoft Community

steveinsf say:

Lock field for single record

Is there a way in Access 2007 to enter data in a blank field and then have that field be read/only (i.e. locked) so it can’t be changed, but only for that record, i.e. for a new record you would still be able to enter data in that field. Once again, though after you did that, the field would become locked for that record as well, although still open for the next new record?

 

Thanks Thanks
ken sheridan say: Firstly data must be entered via a form (which it should always be in any case, never in a table's datasheet).    Assuming this is a 'required' field, so cannot be left Null when a new record is saved, and returned to later to insert a value.  In the form's Current event procedure put:

' first move focus to another control so that
' relevant control can be disabled
Me.[SomeOtherControl].SetFocus

' lock and disable control if not at a new record
Me.[YourControlName].Locked = Not Me.NewRecord
Me.[YourControlName].Enabled = Me.NewRecord

Setting both properties prevents the user from even moving focus to the control at all by disabling it, but keeps its normal appearance rather than greying it out, as is the case if it is only disabled.

If you want to disable the control immediately after a value is entered then you can put similar code in the control's AfterUpdate event procedure.  I'd caution against this, however, as it gives the user no opportunity to correct a mistake before saving the new record.

If the field is not 'required' and you want to be able to return to the record and insert a value if it's Null, amend the code to:

' first move focus to another control so that
' relevant control can be disabled
Me.[SomeOtherControl].SetFocus

' lock and disable control if not at a new record or control is Null
Me.[YourControlName].Locked = Not Me.NewRecord And Not IsNull(Me.[YourControlName])
Me.[YourControlName].Enabled = Me.NewRecord Or IsNull(Me.[YourControlName])
