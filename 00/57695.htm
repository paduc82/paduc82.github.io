wpcoe say:

I just realized the easier way might not work for the original poster, if the saved networks are not showing in the pop-up column.  Mine all did because they were all multiple entries for the same (home) WiFi network named like:  Home Wifi, Home WiFi 2, Home WiFi 2 3, Home WiFi 2 3 4 ... Home WiFi 2 3 4 5 6 7 8 9 10, etc.


This is not the only way to use netsh, but it's the way I find easier.

1) Open a Command Prompt with Administrative Privileges -- if you don't know how, just press the <Windows Key+X> and choose "Command Prompt (Admin)"

2) type "netsh" (without the ") and press <enter> -- your prompt should now read "netsh"

3) type "wlan" and press <enter> -- your prompt should now read "netsh wlan"

4) type "show profiles" and press <enter> -- this will display all saved WiFi profiles

5) type delete profile name="xxx" (use the " around the profile name) where xxx is on the list from "show profiles". 

Hint: You can use wildcards -- e.g. in my example above I could enter

netsh wlan>delete profile name="Home*"

and it deletes all of them in one entry

or, I could delete them one at a time:  e.g.   netsh wlan>delete profile name="Home WiFi" and <enter> to delete the first one only.
