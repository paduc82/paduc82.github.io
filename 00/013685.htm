Option Compare Database
Option Explicit

Private Sub Form_AfterInsert()

     SetFormProperty Me.Name, "LastRecord", Nz(Me.CityID, 0)

End Sub

Private Sub Form_Load()

    Dim rst As DAO.Recordset
    Dim lngCityID As Long
    
    lngCityID = GetFormProperty(Me.Name, "LastRecord")

Timesheet with overtime and doubletime - Microsoft Community

deboradarden say:

Timesheet with overtime and doubletime

Original title: TIMESHEET WITH OVERTIME AND DOUBLETIME

I need a timesheet that calculates overtime on a day when more than 8 hours are worked, and double time when more than 12 hours are worked. The formulas are giving me grief. Can someone help? Thanks.

ossiemac say:

Hello deboradarden,

After starting on this I realised that I should have asked some more questions on whether you want the calculation single time, time and half and double time separated in separate cells or if you want one calculation for the total of all 3 or just the overtime component (time and half plus double).Therefore I have provided formulas for all.

For the formulas $A$1 holds the single hourly rate and A6 holds the hours worked in time format.

Calculate single time component only irrespective of actual hours worked:

=IF(A6*24<=8,A6*24*$A$1,8*$A$1)


Calculate time and half component only irrespective of actual hours worked. Returns zero if not applicable.

=IF(A6*24>8,IF(A6*24<=12,(A6*24-8)*$A$1*1.5,4*$A$1*1.5),0)


 Calculate double time component only irrespective of actual hours worked. Returns zero if not applicable.

=IF(A6*24>12,(A6*24-12)*$A$1*2,0)

Sum the above for total for the shift.

 

Calculate Time and half plus double time components (All overtime). Returns zero if not applicable.

=IF(A6*24>8,IF(A6*24<=12,(A6*24-8)*$A$1*1.5,IF(A6*24>12,4*$A$1*1.5+(A6*24-12)*$A$1*2)),0)

 


Calculate total for shift in one formula. (Includes single, time and half and double)

=IF(A6*24<=8,A6*24*$A$1,IF(A6*24<=12,8*$A$1+(A6*24-8)*$A$1*1.5,IF(A6*24>12,8*$A$1+4*$A$1*1.5+(A6*24-12)*$A$1*2)))

 

Note: An easy way to replace $A$1 and A6 to match your cell addresses for the single hourly rate and the hours worked.

Copy the formula from this post and before pasting into to a cell, type a single quote so that it goes in as text and does not calculate.

Then select the cell plus at least one adjacent cell (preferably an empty one) so that you have a range of more than one cell selected. Then use Find/Replace to replace the cell references with the references required. Don't forget to include the $ signs with whatever address you are replacing $A$1 because I am sure it will need to be absolute.

Delete the single quote (in the formula editing bar) when finished to return to a working formula.

Selecting more than one cell for Find and Replace restricts the replace to the selected range and does not replace throughout your worksheet. If only one cell selected it thinks you want to replace thoughout the worksheet.


Not really essential to use the single quote paste in as text only because Replace will work in a formula but I find it easier because I can see what is occurring.
