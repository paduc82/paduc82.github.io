    hansv mvp say:

    Track Changes has been turned on. The person who created the document may have done this intentionally, so that it's clear what changes have been made to the document and by whom.

    Later on, the changes can be accepted or rejected.

     

    If you're sure that this is not the intention:

    • Activate the Review tab of the ribbon.
    • In the Tracking group, the Track Changes button should be highlighted.
    • Click the top half of this button to turn it off.
    • You can accept changes made before you turned off Track Changes by clicking the lower half of the Accept button in the Changes group of the ribbon and selecting Accept All Changes in Document from the dropdown menu.
