How can I link an existing OneNote notebook to a new Planner plan? - Microsoft Community

cthijssen say:

How can I link an existing OneNote notebook to a new Planner plan?

Just discovered the Planner app in Office 365. I'm rolling this out to my team to be used on a project. The project already had a onenote set up for information sharing. How do I link the existing onenote with the new planner page?

jiaxing bian msft say: Hi CThijssen,

Each plan in Planner will have a corresponding OneNote notebook. To make your existing notebook as the default one for the plan, please follow the steps below:

1. Open both your existing Notebook and the one for the new plan in the OneNote desktop application. First open it in OneNote Online, and then click EDIT IN NOTEBOOK.

2. Navigate to the notebook of the plan, click the down arrow beside the Notebook Pane, then right click the notebook name and select Properties.

 

3. Click Change Location on the Notebook Properties window.

 

4. Find the URL for the notebook in the address bar, remove the last part of the URL (Plan Notebook), and then copy the URL to the clipboard. The URL should be like this: https://tenant.sharepoint.com/sites/plan/SiteAssets/

 

5. Navigate to your existing notebook, repeat the above steps to the Change Location stage.

6. Paste the URL you copied in Step 4 to the Folder name section, and click Select to save the notebook to the new location.

7. Open the URL you copied in Step 4 in a browser to open the library that contains the notebook. You will find 2 notebooks here, the build-in one named with the plan name and the other one you just saved.

8. Rename the build-in one, for example: add an underline, and use the name plan Notebook for your existing notebook.

 

9. When you click Notebook from the plan page, you will be brought to your existing notebook.

Regards,
Jiaxing
