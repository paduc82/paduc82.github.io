greg maxey say:

In the document itself, I don't think there is a solution at all.

You could use a userform for data entry and then use a listbox and it mouse move event to display amplifying information in an adjacent label:

Private Sub UserForm_Initialize()
Dim strChoices() As String
  strChoices = Split("A,B,C", ",")
  ListBox1.List = strChoices
End Sub
Private Sub ListBox1_MouseMove(ByVal Button As Integer, ByVal Shift As Integer, ByVal X As Single, ByVal Y As Single)
  Select Case True
    Case Y <= 10: Label1.Caption = "Apples"
    Case Y > 10 And Y <= 20: Label1.Caption = "Blueberries"
    Case Y >= 20 And Y < 30: Label1.Caption = "Cherries"
    Case Else: Label1.Caption = vbNullString
  End Select
End Sub

You may have to tweak the Case statements.
