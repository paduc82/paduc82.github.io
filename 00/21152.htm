To upload screenshots, click Use rich Formatting in this thread and select Insert/Edit Media.

Generally, if you are using default retention policy and didn’t assign a customer one to users, the emails of these users in default folders except Junk Email and Deleted Items will not be deleted by default retention policy. If you delete emails manually, then these emails will be moved to Deleted Items. Emails in Deleted Items and Junk Email will be retained for one month by default. In some Office 365 plans, we can use litigation hold feature to hold the emails, even though the emails are deleted manually, these emails can still be held in our Office 365 server. Open Outlook client > File > Open & Export > Import/Export > select Export to a file > click Next > select Outlook Data File (.pst) > click Next > select corresponding folder > click Next.
After exporting the .pst file, you can open it as the steps below:
Open Outlook client > File> Account Settings> Account Settings > Data Files > Add

Or if you don’t mind, you can delete the corresponding retention tag so that retention policy will not influence your emails any more. Please let me know if this workaround meet your requirements and I will provide detailed information about it.

Question 3: If you configured your account into the Outlook client via Exchange, the answer is no.

Question 4: Regarding this question, may I know if you want to assign retention policy in the Outlook client, or you want to prevent specific emails from deleting because of retention policy? We can give more specific suggestions once we know your root concerns. Thanks for your understanding.

Hi Ande0531,

The correct term is to CONCAT data for your query. We will have to write some formulas across 3 sheets to achieve what you are looking for.

You can check this workbook in detail

Sheet 1 – I have assumed the data based on your example in Column A to Column D.

  1. Column H: Cell H2 = 1 , Cell H2 = “=H2+1” and fill down the formula till you have data in this sheets.
  2. Column I – We will CONCAT each row data which will simplify our later formulas. I2 = “=CONCATENATE(A2&","&B2&","&C2&","&D2&","&E2&",")”, again, please fill it down

Sheet 2 – I have assumed 8 locations.

  1. Column A – I have assigned numbers 1 to 8 to each location
  2. Column B – Each location is listed against each number.


Sheet 3 – We will pull up values from Sheet1 and Sheet 2 and CONCAT as per your requirements.

  1. Column A: Cell A2 = “=ROUNDUP((ROW(A2)-1)/8,0)”.We are using ROW function to get correct row number minus 1 and divide by 8. Using ROUNDUP of 0 we are getting repetition for each row exact 8 times.
  2. Column B – From B2 to B9 simply type 1 to 8.  B10 = “=B2” and fill down in total rows that you wish to repeat your data. (Number of data rows in Sheet1 1 x 8)
  3. Column C – We will use VLOOKUP from Sheet 1 using Column A. C2 = “=VLOOKUP(A2,Sheet1!H:I,2,FALSE)” Since A2 to A9 has value of 1 , it will pull same value 8 times from Sheet1 and same will repeat for lower rows as well.
  4. Column D - We will use VLOOKUP from Sheet 2 using Column B. D2 = “=VLOOKUP(B2,Sheet2!A:B,2,FALSE)”. Its simply pulling up locations 1 to 8 which we CONCAT to each Sheet 1 Data we pulled.
  5. Column F – Now here we will simply CONCAT these 2 LOOKUP value to get for final expected result.F2 = “=C2&D2”. It simply CONCATS 2 cell.

Please note that you will have to fill down all formulas in Sheet 3 for Data row in Sheet 1 x 8 times.

You can always hide the columns later once you have your result in Sheet3.

Neha
