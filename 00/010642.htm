This is not normal behavior.

Try starting Word while holding down the Ctrl key, and answer Yes to the prompt about starting in Safe Mode. How to automatically use filenames as document titles? - Microsoft Community

jingxuan say:

How to automatically use filenames as document titles?

I have a large number of doc files (a few hundred-ish) that have their filenames also intended as their titles.

Is there a way of automatically adding titles to documents using their filenames? I have to do this for an enormous number of files, so I suppose making a macro would be easier? I have little or no knowledge of macros and the language involved but I have just read Merge multiple word 2010 documents and am quite confident of carrying it out.

Please provide me detailed and easy-to-follow instructions. Thank you (=
hansv mvp say:

Place a bunch of documents that you want to process in a single folder with no other Word documents. The documents should not be password-protected.

Start Word.

Press Alt+F11 to activate the Visual Basic Editor.

Select Insert | Module to create a new code module.

Copy the following code into the module:

 

Sub AddTitles()
    Dim doc As Document
    Dim strPath As String
    Dim strFile As String
    Dim strTitle As String
    ' Ask the user for a folder
    With Application.FileDialog(4) ' msoFileDialogFolderPicker
        If .Show Then
            strPath = .SelectedItems(1)
        Else
            MsgBox "You didn't select a folder!", vbExclamation
            Exit Sub
        End If
    End With
    ' Add trailing backslash
    If Right(strPath, 1) <> "\" Then
        strPath = strPath & "\"
    End If
    ' Hide updates
    Application.ScreenUpdating = False
    ' Loop through the documents in the folder
    strFile = Dir(strPath & "*.doc*")
    Do While strFile <> ""
        ' Open document
        Set doc = Documents.Open(FileName:=strPath & strFile, _
            AddToRecentFiles:=False)
        ' Strip extension from filename
        strTitle = Left(strFile, InStrRev(strFile, ".") - 1)
        ' Insert paragraphs
        doc.Content.InsertParagraphBefore
        doc.Content.InsertParagraphBefore
        ' Format as title and set text
        With doc.Paragraphs(1)
            .Range.Text = strTitle
            .Style = doc.Styles(wdStyleTitle)
        End With
        ' Close & save document
        doc.Close SaveChanges:=True
        ' On to the next one
        strFile = Dir
    Loop
    Application.ScreenUpdating = True
End Sub

 

With the insertion point anywhere in the code, select Run | Run Sub/UserForm or press F5.

The macro will ask you to select the folder to process. When you've done that, the code will open each document in turn, add a title, then save and close it.

Try it out on a folder with a small number of documents first (preferably copies of the original documents), so that you can inspect the result before processing larger numbers of documents.
