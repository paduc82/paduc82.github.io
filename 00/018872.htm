Hi Ronnie,

Thank you for your interest in Windows 10 Technical Preview.

RAW pictures are high-quality, uncompressed files that differ by camera manufacturer. Some common file name extensions for RAW picture files include: .cr2, .crw, .dng, .nef, .orf, .rw2, .pef, .arw, .sr2, .srf, and .tif.

You can view RAW pictures in Windows Explorer or Photo Viewer if you have the right codec for your specific camera installed on your PC. If youre having problems seeing your RAW files, download and install the Microsoft Camera Codec Pack. If youre still having problems, your cameras format might not be supported. Check the list of supported formats provided on codec packs download page for more info.

Common graphics file types include JPEG (.jpg), TIFF (.tif), and sometimes bitmap (.bmp). In addition, some digital cameras can save pictures in RAW format, which is uncompressed and hasn't had effects like white balance or sharpening applied. There are many types of RAW picture file formats which can vary by camera manufacturer. RAW picture files are typically the highest quality picture file formats.

RAW Advantages:

High-quality images aren't compressed and preserve finer details than JPEG pictures.

RAW images are less likely to result in blown-out highlights, in which detail is lost in areas that are too bright, such as the sun shining on a person's face.

You've got more control when editing RAW pictures in a photo-editing program. Edited pictures can be saved at a higher-quality level, too (compared to JPEGs).

RAW Disadvantages:

Many programs, including most web browsers, can't display RAW pictures.

Specific codecs, which vary by camera manufacturer, need to be installed to view RAW pictures in Windows.

RAW picture files are usually much larger than the same picture that's saved as a JPEG. As a result, RAW pictures consume hard disk space far more quickly than JPEGs.

RAW pictures often require more advanced photo-editing programs than those used for editing JPEGs.

There is no way to add support for RAW files in Photo Viewer. Instead I would suggest you to use the Photo App and check if this helps.

john korchok say:
  1. Create a new blank presentation.
  2. Choose File>Options>Customize Ribbon in PowerPoint
  3. Check Developer in the right-hand column of Main Tabs. OK out and the Developer tab becomes visible. 
  4. Click on the Developer tab and choose Macros.
  5. In the Macro Name field, type a name like UpdateLanguage. then click on Create. The VBA editor window opens.
  6. Paste in the following code: This shows the final result, make sure that the first and last lines are not duplicated:
  Dim Slides%, Shapes%, SlideCount%, ShapeCount%
  SlideCount% = ActivePresentation.Slides.Count
  For Slides% = 1 To SlideCount%
    ShapeCount% = ActivePresentation.Slides(Slides%).Shapes.Count
    For Shapes% = 1 To ShapeCount%
      If ActivePresentation.Slides(Slides%).Shapes(Shapes%).HasTextFrame Then
        ActivePresentation.Slides(Slides%).Shapes(Shapes%).TextFrame.TextRange.LanguageID = msoLanguageIDEnglishUS
      End If
    Next Shapes%
  Next Slides%
End Sub

  1. In the above code, change the msoLanguageIDEnglishUS to the language variant you prefer. Here's the complete list: MsoLanguageID Enumeration
  2. Save the presentation, changing the Save as type: dropdown to PowerPoint Macro-enabled Presentation (*.pptm)
  3. Leave the presentation open.
  4. Open the deck where you want to change the language.
  5. Choose Developer>Macros. 
  6. Change the Macro in: dropdown to the name of the .pptm file you saved.
  7. Select the UpdateLanguage macro and click on Run. The language of all placeholders and text boxes is updated to your preference.

Please note, this will not set the language of all elements, like tables or notes pages. To do a thorough language update, I recommend buying an add-in that does a more complete job. Here are 2 good ones:

Lingo10 Add-in

PPTools LanguageSelector
