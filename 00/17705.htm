Hi Riordan

It is not feasible to do that from the Exchange admin portal. However, you can achieve it by using PowerShell commands. Below are the steps:

2. If you want to configure Automatic Replies for someones mailbox(e.g. Tony) to be set between the specified start and end dates and include an internal message, please run the following command:
    Set-MailboxAutoReplyConfiguration -Identity tony -AutoReplyState Scheduled -StartTime "7/10/2012 08:00:00" -EndTime "7/15/2012 17:00:00" -InternalMessage "Internal auto-reply message"
3. If you want to configure Automatic Replies for someones mailbox(e.g. Tony) to be sent and include an internal and an external message, please run the following command:
    Set-MailboxAutoReplyConfiguration -Identity tony -AutoReplyState Enabled -InternalMessage "Internal auto-reply message." -ExternalMessage "External auto-reply message."
