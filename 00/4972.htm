The Data Entry property is the most misnamed property in Access. Setting Data Entry to Yes means the form can ONLY be used the enter new records. Therefore you can't use it to find existing records which is why your find record button fails.

I generally use button on a menu form to open a form. One to open in Add mode, the other in Edit mode. When opened in Add mode the form can only be used to enter new records. When opened in Edit mode you can search for existing records.
