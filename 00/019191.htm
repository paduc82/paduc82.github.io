The error message Personalized Settings Not Responding appears while you sign in due to unwanted registry entry. Kindly follow the below steps and check if it resolves the issue.

Step 1: Disconnect your Devices.

Windows Update will prevent your Windows to load the device drivers. So, rebooting the PC without any devices connected will help. Disconnect all the devices including keyboard and mouse before rebooting. Once after rebooting, plug in keyboard and mouse once the system is done booting.

Step 2: Restart Windows Explorer.

Right click on Taskbar and select Task Manager > Expand the windows by clicking on More details > Now locate Windows Explorer and right click on it, to select Restart.

Step 3: Remove the Registry Key.

Once when the Windows Update key is deleted, the key will be rebuilt again which will resolve the issue. To do that:

  • Hold CTRL+SHIFT+Esc keys to open Task Manager.

  • Click File and select Run new task.

  • Check the box Create this task with admin privileges and type regedit and press Enter.

  • Locate to the following location: Computer\HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Active Setup\Installed Components\{89820200-ECBD-11cf-8B85-00AA005B4340} > right click on the key and select Delete.

Registry Disclaimer: The registry is a database in Windows that contains important information about system hardware, installed programs and settings, and profiles of each of the user accounts on the computer. Windows often reads and updates the information in the registry.

Normally, software programs make registry changes automatically. You should not make unnecessary changes to the registry. Changing registry files incorrectly can cause Windows to stop working or make Windows report the wrong information.

Please take a backup of the registry. Follow the steps given in the link below:

Link: https://support.microsoft.com/en-us/help/322756/how-to-back-up-and-restore-the-registry-in-windows

The mailmerge preview does not show what would be output for page 2 unless, in your scenario, you scroll through to record 5. You need to complete the merge, via Finish and Merge, to get the actual output.

If the output generated via Finish and Merge>Edit Individual Documents is repeating the same 4 records over and over, that suggests you've configured it to merge only a limited number of records and you're simply repeating the Finish and Merge process for that same limited set of records instead of doing it once for all records.
