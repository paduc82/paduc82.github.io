    Hello,

    All Skype accounts (new and existing) are now Microsoft accounts and users can use their Skype credentials for logging in to any Microsoft services. This migration provides the following benefits:

    • A single identity system across Microsoft services, allowing users to sign in and use other Microsoft services seamlessly.
    • Improved security and better experiences for forgotten password and account recovery scenarios.

    With that being said, linking and unlinking of your old Skype account to a Microsoft account created using your email address is not possible.

    smokinjo54 say:

    Ok, I was able to finally get 10547 to install successfully and everything back that I lost after I chatted with a very helpful Microsoft tech named Patrick.  Below is what it took to get the new build to install without issue.

    1. Open an elevated command prompt. Press WinKey + X, click Command Prompt (Admin).
    Type the following command, and then press Enter. It may take several minutes for the command operation to be completed.
    dism.exe /online /cleanup-image /scanhealth
    After you get message The operation completed successfully type second command and press enter. It will take some time to finish.
    dism.exe /online /cleanup-image /restorehealth

    2. Reset Windows Update Components from http://support.microsoft.com/kb/971058

    I did receive an error at this point that the service was not running or corrupt but I guess it didn't matter.

    3. Run Windows Update Readiness Tool from http://support.microsoft.com/kb/947821

    4. Perform a clean boot and try to run Windows Update again.

    (I didn't have to do a Clean Boot, Windows update immediately kicked off the 10547 update as soon as I completed the above steps.  At that point I shut off both monitors since I have a dual monitor setup and thought that that might be part of my problem since the video driver and video anomalies were evident during the previous attempts at the update.  I walked away and when I came home from work all was well.
