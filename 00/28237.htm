"~ Land-Line Phone Calls Within Mexico:
Land-line to local land-line: Dial the seven digit number (i.e. 924-1234)
Land-line to long distance land-line: Dial 01 & the ten digit number (i.e. 01-999-924-1234)
Land-line to local cell phone: Dial 044 & the ten digit number (i.e. 044-999-924-1234)
Land-line to long distance cell phone: Dial 045 & the ten digit number (i.e. 045-999-924-1234)    Cell-Phone Calls Within Mexico:cell phone to local or long distance land-line: Dial the ten digit number (i.e. 999-924-1234).  Cell phone to local or long distance cell phone: Dial the ten digit number (i.e. 999-924-1234)"   So let's not make Carlos Slim any richer.  Let the money go to Skype and Microsoft and let us be able to complete calls in Mexico by not always inserting +52 where it won't work.  Thank you.

a. user say:

Hi Carlz,

To verify if the issue is hardware related. We suggest that you perform a soft reset. Press and hold the volume down and Power button until your phone vibrates and restarts. Note that you should not touch the device screen during this process (not even when the 'slide down to power off' prompt appears). After performing a soft reset, try to check if the issue persists.

If a soft reset didn't work, we suggest that you perform a hard reset instead. You can reset your phone under Settings: Settings  > about > reset your phone. Make sure to back up all of your files before performing a hard reset. For Windows 8.1, go to Settings > Backup > Apps + settings > Back up now. For Windows 10, go to Settings  > Update and security > Backup >More options > Back up now.

Let us know the results.

Regards.

How to boot into Safe Mode

Press Windows Flag key + R

Type msconfig

Press Ok

Click Boot Tab

Check Safe Boot


You can also choose the type of Safe Mode you want to boot into:

  • Minimal is normal safe mode.
  • Alternate Shell is safe mode with command prompt
  • Network is safe mode with networking.


Remember to uncheck Safe Boot when you are finished or you will continue to boot into safe mode
