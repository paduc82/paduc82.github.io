lemp say:

If the laptops have a COA sticker -- and they should -- you will be able to do a "clean install" of the same edition of Windows that is shown on the sticker.  These labels should be on the bottom of the laptop (where hopefully the information will still be legible) or perhaps inside the battery compartment.

Assuming that all of the laptops have COA sticker for the same edition of Windows 7 (for example, Windows Home Premium), you will need one Windows 7 installation DVD.  Otherwise, you will need one Windows 7 installation DVD for each different edition of Windows 7.

Your options for obtaining this DVD are, at least:

  1. Borrow one from a friend or colleague
  2. Buy one of the "recovery disks" that explicitly require use of your own Product Key.  I have not personally used one of these, but they should be safe to use.  In my view, the fact that they don't pretend to permit a valid installation unless you have your own product key makes them far less likely to be a scam.
  3. Buy one "OEM System Builder" installation DVD.  These are still available from Amazon and other legitimate sources for about $150 or so.
  4. Ask HP if they will send you recovery media.  See, e.g., https://www.amazon.com/s/ref=nb_sb_noss_2?url=search-alias%3Daps&field-keywords=windows+7+home+premium
  5. Download a legitimate copy of a Windows 7 ISO file (from which you can create an install DVD or USB flash drive).  At the moment, I'm aware of only two sites that have these files:
    • http://windowsiso.net/ -- the catch with this site is that it only has links to "torrents."  Although torrents are often associated with pirated software and/or malware, as far as I know, this site is legitimate.  You will need a "torrent client" in order to download the file.
    • https://tb.rg-adguard.net/public.php - the catch with this site is that the only English-language Windows 7 ISO files available are for Windows 7 Ultimate.  You can use a Windows 7 Ultimate ISO file to install any version of Windows 7 by doing a little editing.  There are various "How to" instructions available by search for, e.g., <<install any version windows 7>>.  See, e.g., https://answers.microsoft.com/en-us/windows/forum/windows_7-windows_install/how-to-create-a-universal-windows-7-installation/15183b74-4478-4647-84dd-6d6431f1cfa4 (none of the download links in that article work any more and the "SHA1 hash values" listed may or may not be the same for ISO files downloaded from either of the links above)

You'll also need the drivers for the laptops.  Check the HP support website for the particular model(s).  It will be easier if all of the computers are identical.  Otherwise, you may have to download multiple sets of drivers.  Many of the "recovery disk" mentioned in #2 above say that the include drivers.  Those drivers may or may  not work with your laptops and they may or may not be the correct versions.  In your position, I would make an effort to get the drivers from the HP website.

Once you have the requisite drivers and installation medium, see this for doing the installation:  https://answers.microsoft.com/en-us/windows/forum/windows_7-windows_install-wininstalls/clean-reinstall-windows-7/0ae269fd-167e-423b-94ef-26133f89e10c

If all of the laptops are truly identical, you may be able to save some time if you can connect them all to a local network.  See  https://www.google.com/search?q=how+to+install+windows+7+on+multiple+computers

lemp say:

I forgot to mention that after you do a clean install of Windows 7, you will not, of course, have any productivity software installed on the computer.

You can install a free open source suite of productivity applications (word processor, spread sheet, presentation, etc.).  Both of the following are very similar to Microsoft Office.  You can research the difference between the two --

https://www.libreoffice.org/

https://www.openoffice.org/

Alternatively as a "high needs, high risk Title I school," your organization probably qualifies to purchase heavily-discounted copies of Microsoft Office.  See https://www.techsoup.org/  Unfortunately, TechSoup only has an "upgrade" version of Windows 10 available.

a. user say:

For future reference - in case this happens again (hopefully never!)

The only firmware your computer has is the firmware for the motherboard, which is included in the BIOS, if you have an older computer, or the Unified Extensible Firmware (UEF.) Dell can't update your firmware - they have no ability to do this. Only you can update the firmware, which is a good thing to do if Dell has notified you that there's an update available.

The terms firmware and software are easily confused because the distinction is subtle. Firmware is software, but it's software that is stored inside a hardware device - rather than on your hard drive - and is intended for use only by that device.

Other hardware devices, notably printers, also have firmware and get firmware updates. The updates are almost always created to solve a problem, which is why you should update the firmware when you have the opportunity.

Since it's designed for a hardware device, rather than for Windows, firmware has no ability to affect your Windows password.

One last bit of (unsolicited) advice: There's no do-over to a firmware update, and making a mistake will turn your device into a paperweight. So follow the manufacturer's instructions like it's the Bible.
