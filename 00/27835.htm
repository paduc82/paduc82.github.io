mark [excel off the grid] say:

The only way to do this is with a Macro.  I see you've got the Developer Tab, so I will assume you know how use macros a bit.

Copy the following code into the Worksheet Sheet Module:

Private Sub Worksheet_Change(ByVal Target As Range)

Dim firstRow As Integer
Dim lastRow As Integer

firstRow = 28
lastRow = 40


Rows(firstRow & ":" & lastRow).EntireRow.Hidden = False

If Target = Range("Y6") Then

Rows(firstRow + Target - 1 & ":" & lastRow).EntireRow.Hidden = True

End If

End Sub
