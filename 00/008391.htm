Office 2011 has a feature to "Always Create a Backup." With this feature enabled every time you click save a copy of the previous version of the file (as it was prior to the last save) is created in the same folder as the original file. The file name prefix of the previous version file is "Backup of". 

To turn this feature on go to Word > Preferences > Save

Another recovery feature to have turned on is Mac's Time Machine. With that enabled you'd have the file as it was an hour previous.

The executable for IE 64-bit is located here:

  •  C:\Program Files\Internet Explorer\iexplore.exe

Browse to the proper Program Files folder and create a shortcut to that version of iexplore.exe

    How to change the time Windows notification dialog boxes stay open (aka Hover Over)

    Technical Level : Intermediate

    Summary

    Internet Explorer 11 (IE11) uses a feature of Windows 8 to determine the amount of time hover over text needs to show up. This applies to windows tool-tips and notification dialogs.


    On the Microsoft Community platform, this will allow you to adjust the time tool-tips and preview text stay active on your screen.


    Note: Chrome and Firefox do not use the same value at the time of this article.  


    Details

    In Internet Explorer 11 (IE11) you may use a feature of Windows 8 to adjust (customize) the amount of time hover over text & notifications show in your browser window. 

     

    Here are steps to change that:

     

    Step 1 - On Windows Navigate to Control Panel

    Step 2: Click on Ease of Access


     Step 3: Click on Ease of Access Center


    Step 4: Click on Make it Easier to focus on tasks



    Step 5:  Scroll down to find the option – How long should windows notification dialog boxes stay open. Open the dropdown and update as needed and click Apply.


    Updating this value would mean the regular windows notification boxes would adhere to the same timing.


    Update this value for the amount of time you'd like to have alerts, notifications and hover-over text to stay visible on your machine.

    Be sure to click OK and APPLY before moving on.

    Application.Dialogs(xlDialogFormulaFind).Show

    Application.Dialogs(xlDialogFormulaReplace).Show

    VBA commands that follow either of the above lines of code is halted until the dialog boxes are closed.

    In later versions of Excel enter "XlBuiltInDialog Enumeration" (without the double quotes) in the Help search.
