john john __ say: You have an OEM license, you can install a second copy on the same computer but the terms of the EULA does not permit you to install more than one copy of Windows on the same machine or on other machines while installed on another computer.  There are no technical problems installing a second copy on a different disk or on another partition on the same disk, Windows will install and run without problems.

However, note that Windows installation is not supported on removable drives and the Windows setup program does not support installation on removable drives (such as USB).  Installing and running Windows on USB requires changes to the setup system registry to change the load order of the USB stack, on Windows XP the USB stack is loaded after the kernel is loaded and after the Session Manager is started, ergo Windows cannot start off a USB drive unless the load order of the USB stack is changed.  Also note that USB drives are slower than internal drives and Windows performance on USB is lackluster at best.

Why are you wanting to do this?  Do you need a bigger drive?  Or is your present drive failing?

John
