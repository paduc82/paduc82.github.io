jay freedman say:

Assuming the merged document is the active document (has the cursor / focus) when the macro starts, you can surround the code that does the changes with a loop of the form

For Each objSection in ActiveDocument.Sections

   ' do something to objSection.Range

Next objSection

If the changes involve a find/replace that does the same thing in each section, you don't even need that loop; just run it against the whole document.
