{ TOC \t "Caption" }

Thank you all for your replies.

It turns out that it depends on how I create the cross-references:

When I insert a cross-reference via the Cross-Reference dialog, Word automatically creates a bookmark and uses a REF field for the cross-reference. In this case, the cross-reference does not appear in the list of tables. Obviously, this is the correct way of doing it.

jay freedman say:

Because the citations are placed within content controls (and a special kind of control at that, since they don't appear in the ActiveDocument.ContentControls collection), Word won't allow them to be formatted with the Highlight tool. You can see this when you try it manually with the button on the Home tab -- the status bar says "You can't make this change because the selection is locked". VBA gets an error, "The HighlightColor statement is currently disabled".

What can be done is to change the font color to make the citations stand out, for example to red. This macro changes all the citations to red font:

Sub ColorCitations()
    Dim fld As Field
    For Each fld In ActiveDocument.Fields
        If fld.Type = wdFieldCitation Then
            fld.Result.Font.ColorIndex = wdRed
        End If
    Next
End Sub

This is best done on a copy of the document, not on the original.

Editing the subject line requires that the email header is expanded.

Open any email message, to the extreme right bottom of the header. If the email header is not expanded, you'll see a <down> arrow carat. Click on that to expand the header - arrow carat will now appear as an <Up> arrow carat. You will then be able to edit the subject line.

Custom Views is the best long term, easily repeatable fix.

Onetime, "hiding" columns is the answer:

  • select Hide

  • right click in the selection
  • Select Unhide
