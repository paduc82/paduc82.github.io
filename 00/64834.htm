songoku'sson say:
Hi there Nancy! Below I have the reason why this is happening, and a potential fix.
"The problem is with the Windows 10 drivers for the graphics cards. Windows 10 drivers don't support Fixed Graphics Mode (for which your computer was designed), only Dynamic Graphics Mode (which won't support HDMI for computers designed with Fixed Graphics). You will need to use your old drivers from your manufacturer and turn off the automatic driver updates.



Here are the steps I used to fix the problem based on the best advice I've found on the internet.



1) Turn off automatic driver updates in the "device installation settings" menu (walkthrough here: http://www.isunshare.com/windows-10/change-devi...)

2) Download and run display driver uninstaller (DDU) and boot into safe mode - DDU should reboot automatically (I used http://www.guru3d.com/files-details/display-dri..., latest version at the bottom)*

3) Uninstall all graphic drivers with DDU (in my case, AMD and INTEL drivers) - DDU will reboot

4) Download and install your latest working graphics driver(s) from your computer manufacturer* (in my case, Windows 7 drivers from HP); once finished, reboot

5) Block windows updates using the "Show and Hide Windows Update" tool from Microsoft (download it here, https://support.microsoft.com/en-us/kb/3073930); run the program and block all updates related to your graphics card (in my case AMD)"
(From Jman-West in the community help forum)
