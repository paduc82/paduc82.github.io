  6. !! Notice how Word creates a 3rd box for the new bullet.

suzanne s. barnhill say:
Unfortunately, there is no way to do this with an ordinary border. What you can do is put the text in a text box or frame and apply a border to the text box/frame. Either can be formatted as In Line With Text so that it behaves pretty much the same as ordinary text except that it can't be broken across pages. Another, perhaps better approach would be to use a table cell, which will be inline by default. The primary disadvantages of working in a table are that you have to use Ctrl+Tab to insert a tab character, and it is sometimes more difficult to work out indents, but a table can break across pages.
