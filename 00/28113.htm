Is there a way in VBA to preserve bookmarks when unlinking fields?

Thanks in advance for any helpful answers.




jhac say: Paul, thanks for your reply.

In my application, the field is often larger than the contained bookmarks, so I've adapted your idea :

Sub UnlinkFieldPreserveBmk(fld As Field)

Dim bmk As Bookmark, newBmk As Bookmark, rng As Range, newBmkStart As Long, newBmkEnd As Long

If fld.Code.Bookmarks.Count = 0 Then
  fld.Unlink
Else
  Set rng = ActiveDocument.Range(fld.Result.End + 1, fld.Result.End + 1)
  ' --- copy the field content to behind the field
  rng.InsertAfter fld.Result

  For Each bmk In fld.Code.Bookmarks
    ' --- get the bookmark start and end position
    newBmkStart = InStr(rng.Text, bmk.Range.Text) - 1
    newBmkEnd = InStrRev(rng.Text, bmk.Range.Text)
    ' --- create the new bookmark
    Set newBmkRange = ActiveDocument.Range(rng.Start + newBmkStart, rng.End - newBmkEnd)
    ActiveDocument.Bookmarks.Add Name:=bmk.Name, Range:=newBmkRange
  Next bmk
  ' --- delete the field
  fld.Delete
End If

End Sub

This macro does not yet consider if the field contains the bookmark text more than once. I hope I can find a solution for this.

EDIT: It doesn't work properly with nested fields either.

Sub MySave()
Dim strName As String
    ActiveWorkbook.Save
    strName = ActiveWorkbook.name
    strName = Left(strName, InStrRev(strName, Chr(46))) & "csv"
    ActiveWorkbook.SaveAs FileName:=GetFolder & strName, _
                          FileFormat:=xlCSV, _
                          CreateBackup:=False
    ActiveWorkbook.Close 0
End Sub

Private Function GetFolder() As String
Dim fDialog As FileDialog
    Set fDialog = Application.FileDialog(msoFileDialogFolderPicker)
    With fDialog
        .Title = "Select the folder to store the CSV file and click OK"
        .AllowMultiSelect = False
        .InitialView = msoFileDialogViewList
        If .Show <> -1 Then
            MsgBox "Cancelled By User", , _
                   "List Folder Contents"
            Exit Function
        End If
        GetFolder = fDialog.SelectedItems.Item(1) & Chr(92)
    End With
End Function
