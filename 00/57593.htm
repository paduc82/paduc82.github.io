Hi

For general knowledge.

IPv6 is Not the mainstream Internet protocol and thus play No roll in Internet connection.

IPv6 can be use for local network sharing, but even there it is Not necessary. I.e. File and Printer Sharing can be done just the same with the IPv4 protocol that is also used to access the Internet.

In many cases when using peer-to-peer Network, and the user is Not a network expert, to avoid confusions it is probably better to Disable it.

Just log to the Wire, and or Wireless Connection properties. Uncheck the IPv6. Make sure that the IPv4, and File and Printer Sharing is checked.

http://www.ezlan.net/Win7/net_connection_tcp.jpg
