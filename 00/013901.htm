jacques raubenheimer say:

Hi Lydia

I discuss this at length in my book on using Word for doing a dissertation, but I will give you the short answer here.

1. No, Word, disappointingly, makes no distinction between a bibliography and a reference list, other than giving you two different titles for a

However, your understanding of what Word does is not correct, not, I think, through any fault of your own, but because Word is not doing what any sensible person doing some academic writing would reasonably expect it to do.

So let me first explain the process.

If you click on Manage Sources, you will find two lists: Master and Current.

Master is all sources that have ever been added to all Word documents on your PC. you make want to make a backup of that file, by the way (look for C:\Users\YourUserName\AppData\Roaming\Microsoft\Bibliography\Sources.xml).

Current is all sources you have added to that document (you can add them manually, or copy them over from Master if you have already added them in some other document).

Next, note that Current list will have some sources with a tick next to them, and some not. Those with ticks are the ones you have actually cited (i.e., those you would expect to see in your reference list).

When you add any of the Bibliography options, you get a full bibliography--i.e., all the sources in current list, regardless of whether they have been cited or not (just with different headings). Word does not give you a reference list.

2. What to do? You will, unfortunately, have to do some manual work. You can add multiple bibliographies to a document, so do that, just making sure that there are about two blank paragraphs between the two (you don't want to get them mashed up). Then select and Copy the second one, add another two blank paragraphs after it, and then use Paste Special (Ctrl+Alt+V) to paste the Unicode text. This will give you a "static" list. Now you will have to delete the sources not cited to get your reference list.

Note that this is:

1. Not what you would expect to have to do (I agree!)

2. A lot of unnecessary manual work (I agree!)

3. if you add more sources, you will not find them in the static list, so you should do this right at the end of your work, or either repeat the whole process, or manually add references for new sources to the static list.

In Microsoft's defense, we should note that Word is a Word processor. Add-on features like citation tools, indexing tools, charting tools (handled via Excel), picture editing, etc., are not its primary focus. So it gives us those features in a good-enough-for-most version, but for truly professional purposes, you would want to look for specialised software. So there are specialised indexing programs, and there are many specialised citation programs. The one I have been using for the last few years is Mendeley (I used ProCite before that).

jacques raubenheimer say: Hi Steve

Yes, you are right, but there is a workaround for that too, and it's a lot easier than the one for Word. At the risk of sounding pretentious, but that issue is discussed in my book on Mendeley, but I will again give a short answer.

Mendeley starts at the right place--with a reference list. It's a lot easier to add, than to take away.

1. Add two page breaks with an empty paragraph between them. I normally do this just before the reference list. The page breaks are optional (and will, as you will see, become invisible), but I like adding them just to keep things organised.
2. Select these and make them all hidden.
Steps 1 and 2 can be combined from the keyboard like so: Ctrl+Enter, Enter, Ctrl+Enter, Shift+Up arrow three times, Ctrl+Shift+H.
3. Now, in that blank paragraph, add the citations, with Mendeley's tool, to the sources you want in the bibliography that have not already been cited in the document (unfortunately, here Mendeley doesn't give you an easy to tick list, but you should know which sources those are--after all, you wrote the thing!).
4. Now, when you update, those sources will be added to the list.

You are essentially tricking Mendeley into including the additional sources, but because they are hidden, the citations are not visible to readers of your printed (or PDF-ed) document.

Again, some caveats:
1. Unlike before, this method allows you to update, but see below.
2. As before, you can add multiple reference lists with Mendeley's tool. But now both lists will contain all the sources. So you will have to follow the same steps as with Word's citation tool (copy, paste as text), to trim down the bibliography into a reference list.
3. One thing I didn't mention earlier, that should be pointed out, is that style-specific formatting (e.g., journal title in italics) may get lost in this process. One advantage to Mendeley is that you can export the document as a static copy, which then loses all the links, but keeps all the formatting. It may be easiest to do that: create your document, add the two reference lists (both of which will, then, be bibliographies--sorry for the confusing-sounding terminology), and then export it, and trim one bibliography down to a reference list.

Yes, this is still a workaround, but it is a bit cleaner and easier than Word's version. And there are numerous other reasons why professional software like Mendeley (there are others, too), are a good option for citations.

This one is a bit more work while you work on the document, but a lot less afterwards.
1. Add only one reference list. And forget about all the hidden citations in Option A.
2. Mendeley allows you to create folders into which you can organise your sources. These folders, though, are not like folders on a hard drive--they merely store references to sources. This means that sources can be added to multiple folders. So, create a folder for your thesis in Mendeley, and add every source that you want in your bibliography, to that folder. These should obviously include the sources you have cited, and any additional ones you want in the bibliography.
3. When you are done, select all the sources in that folder, and press Ctrl+Shift+C (note that your display citation style should be set to the style you are using in Mendeley, but that would pretty much be the case anyway).
4. Now go to Word, and paste your completed bibliography as static, but formatted, text. If you need to add something, you can just overwrite the old one with the new one.

I would probably actually recommend B--although it is more work while you are writing, it is less work at the end. And it allows changes to be made with less fuss.
