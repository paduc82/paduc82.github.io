jeeped say:
Say Sheet1 has the MemberID in column A and the LastActiveYear values in column G as either dates or numbers representing years.

Say Sheet2 has the MemberID in column A and you want a matching LastActiveYear value in column D.

In Sheet2's D2 use this to get the most recent matching date from Sheet1, 

=MAX(INDEX((Sheet1!$A$2:$A$9999=$A2)*(Sheet1!$G$2:$G$9999),,))

Non-matching MemberIDs will return zero and you can remove their display with custom number fortmatting. If the LastActiveYear values in Sheet1's column G are dates you will have to format the values in Sheet2 as dates. More information on this method can be found in either of my following Wiki articles.

MINIF, MAXIF and MODEIF with Array Formulas
MINIF, MAXIF and MODEIF with Standard Formulas

jeeped say:
You can base a conditional formatting rule upon a formula; specifically, the formula that you are using n E2.

Select C2:C999 with C2 as the active cell. From he Home tab, choose Conditional Formatting, New Rule. When the New Formatting Rule dialog opens, choose Use a formula to determine which cells to format and supply the following for Format values where this formula is true: 

=AND(ISNUMBER($D2), $D2<2013, $C2=1)

Click Format and supply a red bold font. Click OK to accept the formatting, then OK again to create the new rule.

Repeat for green formatting with,

=AND(ISNUMBER($D2), $D2<2013, $C2=0)

If the cell's normal font is black, there is no need to create another rule.

How do I uninstall Silverlight from Mac OS X? - Microsoft Community

silvercoyote say:

How do I uninstall Silverlight from Mac OS X?

(My question doesn't fit any of the provided Forum options, so I've slotted it into the only Mac-related area.)
jim gordon mvp say:
Your question is in the right forum. Close enough for me, anyway.

Silverlight is a web browser plug-in. You might find it in various places on your Mac. It's pretty easy to do a Spotlight Search to find Silverlight on your Mac.

The Sliverlight.plugin file is the actual plug-in. Put that file into the trash, empty the trash, and Silverlight is no longer available to run on your Mac. You may have a Silverlight installer and/or Silverlight Configuration Utility application somewhere on your Mac, which you can keep or trash. It's easy enough to download and install the plug-in again from Microsoft's Silverlight web site, should you change your mind and want Silverlight again.
