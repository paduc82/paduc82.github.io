FirstName & (" " + MiddleName) & " " & LastName

Junk Email Protection in Outlook for Mac applies only to POP and IMAP accounts. They won't affect Exchange accounts.

I'd first test whether these messages are getting moved by the server by quitting Outlook for Mac altogether and sending a few messages.

If everything appears correctly in OWA then launch Outlook and see if things begin to move in OWA.

Likewise, you can test with Apple's Mail application by connecting it to the same account and seeing if the same results occur.

If you deem the problem to be Outlook for Mac then I suggest launching Outlook with the Option key held down. This will launch the Database Utility application.
