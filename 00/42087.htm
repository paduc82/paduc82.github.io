Hi Imran,

Many times Windows 8/8.1 starts showing weird characters such as vertical rectangles instead of new Metro icons in several places such as Start Screen, Login Screen, several Metro apps, etc.

 

 

This problem occurs if a system font "Segoe UI Symbol" is missing or corrupt in Windows 8. This font contains several metro icons which are shown in the places mentioned above and if the font gets deleted from Windows 8 or if the font becomes corrupt, then Windows 8 starts showing these weird characters instead of metro icons because Windows can't find the required icons to show.

 

 

Sometimes 3rd party software also replace the default "Segoe UI Symbol" font in Windows 8 with an older version of the same font which can also cause this issue. Follow these methods and check if it helps:

 

 

Method 1: I would suggest you to run the SFC scan. System File Checker tool (SFC.exe) scans for missing or corrupted system files and repairs them.

 

Use the System File Checker tool to repair missing or corrupted system files

http://support.microsoft.com/kb/929833/en-us

 

 

Method 2: Try changing the Windows display language on Windows 8.1 and reverting back to English language and check:

 

Add a language or keyboard

http://windows.microsoft.com/en-in/windows-8/add-language-keyboard

 

 

Method 3: If you have access to another Windows 8.1 computer, you can copy "Segoe UI Symbol" font from that system and paste the font in your Windows 8 computer. You just need to type Fonts in RUN dialog box and press Enter. It'll open Fonts folder. Scroll down and you'll see "Segoe UI Symbol Regular" font in the list. Right-click on the font and select copy option and then you can paste the font file in your USB drive. After that copy the font file in your Windows 8.1 system.

 

Also make sure to install all the latest Windows Updates on the computer:

http://windows.microsoft.com/en-IN/windows-8/windows-update-faq

vijay a. verma say:
Option 1 - General troubleshooting steps
1. Try a different USB port
2. In case of wireless mouse, pair it again with the computer
3. In case of wireless mouse, check if batteries are not weak.

Option 2 - Use the Hardware & Devices Troubleshooter
Type Control in Windows Start Search box
Click Control Panel
Click Troubleshooting > View all
Select Hardware and Device
Click on Advanced on the Hardware troubleshooter dialog box and click on Run as administrator (Enter the credentials if required).
Click on Next and follow onscreen instruction complete the Hardware troubleshooter

Option 3: Update mouse driver
Windows Key+X (Or right click on Start)
Click Device Manager
Expand Mice and Other Pointing Devices
Right click on the mouse listed there (You will have to repeat the steps for all entries listed including Synaptics)
Click Update Driver Software
Click Browse my computer for driver software
Click Let my pick from a list of device drivers on my computer
Choose the driver
Click Next to update
If this does not work, select Search automatically for the updated driver in place of Browse my computer for driver software
Restart your computer

Option 4: Update driver from manufacturer's website (You will have to repeat the steps for all entries listed including Synaptics)
Go to the website of the Mouse's manufacturer or computer's manufacturer
See if some driver is released for your model and OS
If yes, download that
In Option 2, you click Update Driver Software, in place of this, click Uninstall
Install the downloaded driver
Restart your computer

Option 5: Roll back the driver (You will have to repeat the steps for all entries listed including Synaptics)
Windows Key+X (Or right click on Start)
Click Device Manager
Expand Mice and Other Pointing Devices
Right click on the mouse listed there
Click Properties
Under Drive tab, if Rollback Driver is enabled click it

considerate_guy say:

The first question is what is on the flash drive?

If you just have the project file (.wlmp) that that isn't a movie. What you instead have in the project file is the saved editing state of your project. It is a text file that remembers the names of all the photos, videos, songs, etc. and how you have put them together.

But the project file is only useful within Movie Maker. Movie Maker will read the project file and reload all the pictures and let you continue editing.

When you are all done with the editing, you want to create a movie. To do this you choose File > Save movie. The resulting output is a .mp4 file (or a .wmv file in older versions of movie maker).

Some SmartTVs are able to play .mp4 files, so you should be able to copy that to a flash drive and play it on the TV. Some SmartTVs will also play .wmv files, but not as many. If you are using an older version of Movie Maker, you should upgrade to Movie Maker 2012 so you can create .mp4 files.

Finally, if you were just trying to "play" the project file (.wlmp) from the flash drive, that will never work. First there is no video/photos/music within a project file, just the *names* of those files to let you edit. Second, that format is only understood by Movie Maker and not by any consumer devices like TVs.

sumit (independent advisor) say:
Hi,
I am Sumit, an Independent Advisor and a 2-Year Windows Insider MVP here to help.

Activation depends on what type of license you have of Windows 7/8/10.

If Windows 7/8/10 came preinstalled with the computer, that is called an OEM license. For an OEM license, the license dies as soon as Motherboard dies. So essentially it is non-transferable. If that is the case you need to get a new license.

However, this is not same for retail licenses i.e. you purchased the license through Microsoft store. That does carry transfer rights. If that is the case, please see the below steps:

Run the activation troubleshooter on the new device using the below steps:
Go to Settings>Update and security>Activation and run the troubleshooter. and select I recently changed hardware on this device.

Please note that Activation troubleshooter requires you to have digital license linked to Microsoft Account.
