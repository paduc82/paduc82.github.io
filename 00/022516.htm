jay freedman say:

Because the pictures have in-line-with-text wrapping (as can be seen by the text cursor to the right of the histogram), they behave like big characters. Alden Rey's suggestion is almost correct. However, you have to position the cursor between the pictures, rather than clicking on one of them, before you press the space bar; otherwise the spaces will replace the selected picture (assuming you have the default option that typing replaces the selection).

For more precise positioning, you can set a tab stop at the desired horizontal position of the left edge of the histogram, and insert a tab character between the pictures instead of spaces.

ken sheridan say: Contractors and suppliers are sub-types of type clients.  The way to model a sub-type is by a one-to-one relationship type in which the primary key of the table modelling the sub-type is also a foreign key referencing the primary key of the table modelling the type.

A sub-type is characterized by sharing all attributes of its (super) type but not those of other sub-types.  Consequently all attributes which are common to all clients are represented by columns in the Clients table.  Those which are specific to a sub-type (the vastly different ones to which you refer) are represented by columns in Contractors and Suppliers respectively.  Do the same for other sub-types.

In a combo box to select a client which is correlated to another combo box in which a sub-type is selected the RowSource property can be set in the AfterUpdate event procedure of the first combo box to an SQL statement in which Clients is joined to the relevant sub-type table by means of an INNER JOIN.  Only those rows in Clients for which a matching row in the relevant sub-type table exists will be returned in the combo box's list.

david posso say:

Hi agr8leo,

To open a PDF file without converting it to a Word document, open the file directly wherever it's stored (for example, double-click the PDF file in your Documents folder).

However, if you want to edit the PDF file, go ahead and open it in Word. Word makes a copy of the PDF, converting it to a Word document and attempting to match layout of the original PDF. You always have the original PDF file, in case you don't want to keep the version that Word converts.

Converting from PDF to Word works best with files that are mostly text—for example, business, legal, or scientific documents. But when you open a PDF file in Word, it might not look exactly the way it looked as a PDF.
