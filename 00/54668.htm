briguy_sc say:

This most likely has to do with the display scaling on high DPI displays, such as the Surface Pro 4.  Its not an actual hardware issue, but an issue of applications developed for older (lower resolution) displays.  Windows tries to scale the resolution up (by basically just using multiple pixels for each one) in order to make the text and windows large enough to see on the small, but very high resolution screen.

Newer applications use newer Windows APIs that smoothly scale up the text fonts, though even some of the built in utilities in Windows 10 can still appear to have blurry fonts as MS obviously has not updated them all.  

There are a few things that can be done to help with this including changing the screen scaling, clicking on the app executable and changing the scaling options, etc.  

Take a look at a few of these links:

How to Make the Windows Desktop Work Well on High-DPI Displays and Fix Blurry Fonts

Windows 10 Feature Focus: Display Scaling

Some desktop applications may appear blurred on high-DPI displays

And many others.  You can also Google (or Bing :-)) for "windows 10 display scaling blurry" in order to find more links.

BTW - while these articles will show some ways to alleviate this, the RIGHT answer is for the application vendor to update their application for direct support for High DPI displays.  

Hope this helps!
