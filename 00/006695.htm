hansv mvp say:

Normally, it's an advantage that each user will see numbers displayed according to his/her system settings.

But you can force Excel to use specific settings - just don't forget to restore the user's original settings at the end!

For example:

Dim strDecimal As String
Dim strThousands As String
' Store current settings

strDecimal = Application.International(xlDecimalSeparator)
strThousands = Application.International(xlThousandsSeparator)
' Change settings
Application.International(xlDecimalSeparator) = "."
Application.International(xlThousandsSeparator) = ","
...
...
' Restore original settings
Application.International(xlDecimalSeparator) = strDecimal
Application.International(xlThousandsSeparator) = strThousands

Sorry, my mistake. Use Application.DecimalSeparator and Application.ThousandsSeparator instead. Those are read/write.
