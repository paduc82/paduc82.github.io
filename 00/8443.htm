To edit, the physical style used for email you need to follow these steps:
  1. Application unlock your device (see: http://www.mobilejaw.com/articles/2009/09/removing-application-lock-on-windows-mobile-standard-devices/ )
  2. Connect the device to your computer and browse to the \Windows folder and look for the following files - read_style.0409.css, read_sms.0409.css, and compose_style.0409.css
  3. Copy this files to your desktop and make a backup copy
  4. Open them in Notepad
  5. Change the font tags as you require
  6. Save them, and copy back to the mobile device
  7. You may have to reboot after copying them back to the device
One thing to note, the "0409" part of the file is for the region.  North America typically uses 0409. If you're not located in NA, then you might have to modify the files with 0c0a in the name.

What about

Public Function Decrypt(ABCin As String) As String
    Dim strOut                As String
    Dim Keynum                As Integer
    Dim lHiderLen             As Long

    lHiderLen = Len(CStr(Hider))
    For Keynum = 1 To (Len(ABCin) / lHiderLen)
        strOut = strOut & Chr$(Val(Mid(ABCin, ((Keynum * lHiderLen) - (lHiderLen - 1)), lHiderLen) Xor Hider))
    Next Keynum
    Decrypt = strOut
End Function
