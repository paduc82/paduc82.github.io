anula d'souza say:

Hi,

Thank you for posting in Microsoft Community.

 

This error may occur if the Click-To-Run registry key is corrupted. In order to fix this issue, you will have to delete this registry key using Registry Editor.

 

  • On the keyboard, press the Windows key + R.
  • Type regedit in the Open box, and then click OK.
  • In the Registry Editor, go to the following registry location and delete the Click-to-Run registry key.

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Office\15.0\Click-To-Run

 

Once this key is deleted, you may start the installation of Office 2013, check if the installation completes successfully.

 

Note:  Ensure to backup the registry before editing or deleting any key from the registry.
