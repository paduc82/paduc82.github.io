You can try steps given in the article below and clear temparory folder:-

http://support.microsoft.com/default.aspx/kb/260897

 

1.    Click on Start and then Run. Type %Temp%

2.    All of the folders and files you see in this Temp folder are no longer being used by Windows XP and can safely be deleted.

3.    Choose Edit and then Select All from the menu.

4.    Note: If you're prompted that there are hidden files in this folder, just click on OK to bypass the message. A few hidden files hanging out in the Temp folder probably aren't important enough to worry about.

5.    Choose File and then delete from the menu and click on Yes

6.    Confirm that you want to delete the files by clicking yes on the Confirm Multiple File Delete window that opens.

7.    After all of the files have been deleted you can close the window and empty your Recycle Bin, permanently removing the files from your PC.

Tips:  You may receive a Error Deleting File or Folder message while the files are being deleted. This just means that one of the files is in use by a program right now. Click OK, close all open programs, and repeat the steps above. If you still receive the message, try restarting your PC and repeating the process again.

 

You can even try disconnecting mapped network drive.

1.    Click Start, and then click My Computer.

2.    Right-click the icon for the mapped drive.

3.    Click Disconnect.
