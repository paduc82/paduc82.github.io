eyassh say:

Does Windows 10 have an articulate, reasoned response to Sinofski's Start Screen argument from 2011?

We all know the user feedback and general response to the start screen, and we all know it is mostly negative (at least for traditional users).

That said, Steven Sinofski's arguments on Building Windows in support of the Start Screen were quite reasoned:

http://blogs.msdn.com/b/b8/archive/2011/10/11/reflecting-on-your-comments-on-the-start-screen.aspx

The argument effectively says that the start screen is better for *all* users, not just tablet users. It uses a number of facts:

  • Eye-tracking for Windows 7 users should that their vision is restricted to the small area of the Start Menu when using it; they aren't looking anywhere else -- giving them more space is strictly better
  • Windows 7 users mostly rely on search to launch programs, in addition to search (which still works on Windows 8), Windows 8 users can rely on position, shape, co-location, and color to find apps in a very visual way
  • Fitt's law works better for Windows 8 given the size of the tiles and the fact that its a 2D grid

I understand the user feedback says otherwise. But these points show clear advantages to the start screen even for desktop users.

Is Microsoft simply ditching these improvements based on feedback? Or will Microsoft be providing a no-compromises solution?

The current start menu seems like it mainly ditches these improvements:

  • The space is generally smaller (yes, you can make it bigger and taller), thus restricting the user to less space
  • Users relying on position, shape, co-location, and color feel rather lost (myself included) when navigating the smaller menu... because there is less room to put all that you need on there
  • Fitt's law doesn't work well at all since the tiles are far away from the mouse location when pressing start

I could see forgoing the fitt's law advantage as an engineering trade-off. I am more uneasy about the first two.

Finally, Start Screen Tile Groups were very important for position and co-location. Even if a user makes their start screen bigger and adds more tiles.. it can quickly become overwhelming.

Again, the question is: are there reasoned, logical, counter arguments to Sinofski? Or is Microsoft simply saying "I guess he is empirically wrong, so that's the end of the argument."?

tooolnut say:

The problem is now fixed. I don't know exactly which one of these fixed it, but here they are:

  1. I didn't know that I had inadvertently uploaded thousands of emails onto the Comcast server, in the process of haphazardly copying, cutting, and pasting emails from my old POP account to my new IMAP one, because I was told that these would be lost, once I deleted my POP account. What changes you make to one device's IMAP account get reflected in all others accessing the same server in the same way: I had no idea that was the case when I deleted and purged the whole bunch in Webmail, thinking it would help Outlook get "synchronized" quicker, something it said it was trying to do, but it didn't work that way.
  2. Changing the encryption of SMTP to TLS.
  3. Restarting Outlook.

Again, not knowing better, I deleted and purged these thousands of emails in Webmail, before I realized that the copies of these in Outlook would also go away, once synchronization occurred upon restarting it: Outlook, the IMAP client, only downloads what is on the server, temporarily buffering that content on the PC. Fortunately, after calling Comcast for technical assistance with recovering these emails, I learned that you can do so by right-clicking the Trash folder, in Webmail, and in the pop-up menu, selecting "Recover deleted items." In this one aspect, IMAP works like Microsoft Exchange and One Drive: all the storage is in one place and accessed and modified by multiple devices from multiple locations; no explicit synchronization is necessary.

nexgraphics888 say:

Grey arrow in Outlook calendar view (month view)

SOme of my months in Outlook CALENDAR do not display all my appointments, instead there is s small grey arrow in which you have to click to show all appointments.  I have months which will show me 6 items, others only 4.  I hate this, how do I get it to default to 6 items, thanks.
roady [mvp] say:

There is no technical issue in this case and Outlook works as it should. It all depends on how much vertical screen space is available. When there are more appointments than there is place to show them, you'll get the arrow to indicate that there are more appointments for that day. Click on it to see them.

 

The fact that some months show 4 and others show 6 is caused by how many weeks that month spans.

 

If you can't increase your vertical screen real estate (enlarge window, minimize Ribbon, higher resolution monitor, etc...) and regularly have more than 4 appointments on a day, consider using the Week view instead.

 

Note that you can select multiple weeks in the week view by holding CTRL and clicking in front of the week (or on the week numbers if you have that enabled). This basically takes you into a month view but you can decide how many weeks are visible.
