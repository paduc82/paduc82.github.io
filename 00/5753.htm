bryan roosien say:

How can I read and write csv files using a word macro?

I would like to be able to read a csv file into an array in a word macro so I can do sort and replace on the contents of the array.  For example I would like to import a list of journal names and abbreviations so I may run a macro replacing journal names with abbreviations.  I would also like to be able to find words in a word doc and record them in a csv or text file for later us.

hansv mvp say:

Try something like this:

 

Sub ImportCSV()
    Dim strFile As String
    Dim strLine As String
    Dim arrWords() As String
    Dim arrParts() As String
    Dim lngNum As Long
    Dim lngCount As Long
    ' Modify as needed
    strFile = "C:\Test\Journals.csv"
    lngNum = FreeFile
    Open strFile For Input As lngNum
    Do While Not EOF(lngNum)
        Line Input #lngNum, strLine
        arrParts = Split(strLine, ",")
        lngCount = lngCount + 1
        ReDim Preserve arrWords(1 To 2, 1 To lngCount)
        arrWords(1, lngCount) = arrParts(0)
        arrWords(2, lngCount) = arrParts(1)
    Loop
    Close #lngNum
    ' Do something with the arrWords
    ' ...
End Sub

 

At the end of the loop, arrWords will be a two-dimensional array with two rows, and many columns, the first row containing the journal names and the second row the abbreviations.

For account recovery process, Skype will always use the primary email address that is associated with the account. If you no longer have an access to the primary email address, unfortunately, you cannot recover the account as well.

In this case, you will need to retrieve the email address or fill out the recovery form and pass the verification process. To help you answer the form, you may refer to these tips:

  • The chances of passing the validation increase if you fill out this form from a computer that was previously used to sign in to that account, and was successful.
