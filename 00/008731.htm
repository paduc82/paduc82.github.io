john - project say:
Maria,
First of all, do you have SP1 installed? If not, go to: http://technet.microsoft.com/en-us/office/ee748587, download and install it.

With regard to the overallocation. Even though it may appear that a resource is not overallocated, the red flag is determined by the setting under Resource/Level group/Leveling options. If the setting is on an hour by hour basis, the resource could have less than 8 assigned hours in a given day but yet be assigned to work more than 60 minutes during a given hour. Note: this option setting determines when the overallocation red flag will appear regardless of whether the user has resources leveled.

First check the option setting. The default is day by day. Then set the timescale in the Resource Usage view to have the same units (e.g. day) and see where the overallocation occurs.

Hope this helps.

    want a formula that countdown to a specific date, then turn Red with - Microsoft Community

    _782 say:

    want a formula that countdown to a specific date, then turn Red with 'expired'

    I want a formula that countdown to a spisefic date then when it's reach that date turn's the cell automaticlly to red color with word Expired.

     

    For Example: i want the cell to count from today to 12/5/2014, and when it's reach it, i want the cell automaticlly turn to red color with word Expired.

    hansv mvp say:

    Enter the formula

    =IF(TODAY()>=DATE(2012,5,12),"Expired","")

    You can use conditional formatting to change the background color:

    Select the cell.

    On the Home tab of the ribbon, click Conditional Formatting > New Rule...

    Click 'Format only cells that contain'.

    Select 'Cell value is', 'equal to' and enter Expired in the box next to it.

    Click Format..., activate the Fill tab and select red as fill color.

    Click OK twice.
