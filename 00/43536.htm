pa bear - ms mvp say:

Well, it's extremely unlikely for Windows Update to offer an update if the qualifying application isn't installed. That being said, you could hide the update.

1. Stop the Automatic Updates service:

Start > Right-click on Run and select Run as Administrator > (type) services.msc > [OK]
Double-click Automatic Updates > Click on Stop
(Stopping the service will take a moment)

2. Delete the contents of the Download folder:

Start > Right-click on Run and select Run as Administrator >
(type or copy/paste) %windir%\SoftwareDistribution > [OK]
Open the Download folder and delete its contents
Close the window.

3. Start | Control Panel | System And Maintenance (If you are using Classic View, skip this step) | Windows Update |  View available updates (or Select updates to install): Right-click on Security Update for Visio Viewer 2007 (KB973709) | Click Hide Update | Confirm the UAC (User Account Control) prompt, if appropriate, & OK your way out.

4. Start the Automatic Updates service:

Start > Right-click on Run and select Run as Administrator > (type) services.msc > [OK]
Double-click Automatic Updates > Click on Start
(Stopping the service will take a moment)

=====================================
To restore/unhide updates in Vista: Start | Control Panel | System And Maintenance (If you are using Classic View, skip this step) | Windows Update | Restore Hidden Updates | Find and select the update you wish to install | click on Restore. You can now reselect it and install through the normal process.

danzizme say:

Battery charging isn't handled by the OS it is done by manufactures software/firmware. If you have any manufacture software related to the battery remove it.

Did this happen right after an update? If so what are the KB#s there was a kernel update that was removed recently because it caused unexpected results in a view computers but I haven't seen any battery related one. If you got a update Microsoft has been sending emails to your Microsoft account to inform people to uninstall it.

After you remove battery software remove the battery and unplug the PC. Hold the power button down for a min. And then try it again.

If the battery won't charge while the PC is off then it is a hardware failure and most likely the battery controller is bad.

Yes different warranty for a battery but usually a year. Sometimes longer. Permanent failure are different though most warranties don't cover wear on them but do cover a short.

Edit: I forgot you can try uninstalling any battery driver in device manager. Winkey + x > device manager > expand batteries > update and if none are found reinstall. Batteries can get very hot and can also explode. Don't let it sit plugged in with the battery in til you solve the issue. If the battery has more than positive and negative terminals make sure they haven't been corroded. If you commonly remove the battery make sure the PCs battery terminals aren't damaged. Batteries have one or two extra ports the first is for temp and the second is for data. If either of those has gone bad that could be the issue.

If the battery posts are green fluid has gotten to the battery. Humidity can do this but not as much as direct fluid. Heads up, the battery will also have a small sticker that will either add color or remove color to indicate that fluid has gotten on the battery. Either green color on the post or a sticker that has been active hated will void the warranty. I have had batteries that have never been close to fluid be green because of humidity. If you take the battery out to send it in store it away from fluids so they don't void it
