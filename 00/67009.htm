You have to sort out the problem which leads to the lock up. Some of the gadgets you have running will add to your problem and are maybe the cause as they sap vital resources too. I was very careful about gadgets used, and never had any running which constantly updated the information being shown.

 

CPU, Memory and Network gadgets are among the worst, and utilities in the past which showed a machine slowly buckling under the weight were actually the deadweight.

 

Shut them all down, and note how much longer your computer runs before it starts to freeze. Use Task Manager to determine what is causing slow downs.
