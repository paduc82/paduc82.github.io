hansv mvp say:

Like this:

Private Sub Worksheet_Change(ByVal Target As Range)
    If Target.Count > 1 Then Exit Sub
    If Not Intersect(Target, Range("B:H")) Is Nothing Then
        Application.EnableEvents = False
        Me.Unprotect 'Password:="Secret"
        Cells(Target.Row, "M").Value = Environ("username") & ", " & _
            Format(Now, "yyyy/mm/dd  hh:mm:ss")
        Me.Protect 'Password:="Secret"
        Application.EnableEvents = True
    End If
End Sub

If you applied a password, replace "Secret" with the password that you used, and remove the apostrophe ' from before both occurrences of Password:=...
