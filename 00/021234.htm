suzanne s. barnhill say:
Highlighting cannot be made part of a style, but text shading can. In the Modify Style dialog, click Format, then Border... Select the Shading tab and select the desired color. If the style is a paragraph style, you will be able to apply the shading only to Paragraph, but in a character style, Apply to will be set to Text. Paragraph shading will extend the full margin width, but you can apply a shaded character style to selected text.
