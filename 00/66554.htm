tryx3 say:

The Product Key to use is the one on the CoA [Certificate of Authenticity] label on her PC.  The key found by the keyfinder could be a default key which is not a valid key for activation.


While you are checking the CoA you can check that you are trying to install the same Edition of Windows 7 [Home Premium etc] as that stated on the CoA.  The Product Key will only work for the same Edition.  It doesn't matter if you are installing 64 bit or 32 bit, the Product Key is valid for both.


You didn't mention where you got the ISO file to make the installation disk.  Digital River is the correct source [the authorised MS distributor].  Digital River has no index available to us mere mortals but there is a good index at Official Windows 7 SP1 ISO Image Downloads.
