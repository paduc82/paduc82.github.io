In order to carry out the first part of this task, my code needs to detect which text box is currently selected. Does anyone know how I can do this, please? 
ossiemac say:

Hello Don,

Is there a particular reason to delete the textboxes rather than set visible to false?

I think that the only way to detect which textbox has been selected is to use the GotFocus Event. GotFocus is not a property that can be tested for True or False and therefore you will need Event Code for each of the TextBoxes. This is the reason I suggest not deleting but setting Visible to False so the the code remains and all you will need is a small sub to return the textboxes to visible when required instead of re-creacting.

Following is some example code with the sub to be called each time a textbox is selected. I am assuming that you are using TopLeftCell to identify the cell where the textbox is located.

 

Private Sub TextBox1_GotFocus()
    Call ChangeCellFormat("TextBox1")
End Sub

 

Private Sub TextBox2_GotFocus()
    Call ChangeCellFormat("TextBox2")
End Sub

 

Private Sub TextBox3_GotFocus()
    Call ChangeCellFormat("TextBox3")
End Sub

 

Sub ChangeCellFormat(strTxtBoxName)
    Dim rngCell As Range
    Set rngCell = Me.OLEObjects(strTxtBoxName).TopLeftCell
    Me.OLEObjects(strTxtBoxName).Visible = False
    With rngCell
        .Interior.Color = vbYellow
    End With
End Sub

 

Following is example code to return all textboxes to visible and remove interior color from underlying cell.

 

Sub MakeTextBoxesVisible()
    'Also resets cell format
    Dim objTxtBox As OLEObject
    With Sheets("Sheet1")
        For Each objTxtBox In .OLEObjects
            If TypeName(objTxtBox.Object) = "TextBox" Then
                objTxtBox.Visible = True
                objTxtBox.TopLeftCell.Interior.Color = xlNone
            End If
        Next
    End With

End Sub
