rajesh govind say:

 

Hi Jos,

 

HP Universal Print Driver (UPD) is an intelligent print driver that supports a broad range of HP LaserJet printers and Multi-Functional Printers. This advanced print driver has the ability to discover HP printing devices and automatically configures itself to the device capabilities. HP Universal Print Driver provides two modes of operation: 1) Dynamic Mode and 2) Traditional Mode.

 

The reason why you are getting the prompt to choose which network printer to be used is because; you have opted for the Dynamic Mode of operation while installing the driver. In Dynamic Mode, it automatically discovers, manages and configures all the available printers on the network. But, in Traditional Mode of its operation, the universal printer driver is bound to a print queue and behaves like a normal printer driver and functions exactly like the product-specific drivers. In this mode, it bypasses the universal printing dialog box (where it asks to choose the printer) and goes directly into the typical print setup dialog box.

 

Both the Traditional and Dynamic modes can be used on the same PC to support both office and mobile printing and it is also possible to switch easily between them. In Dynamic Mode, a permanent instance of this driver can be created by simply clicking the option “Add this printer to my Printer and Faxes folder”. When you select this, the driver will function in Traditional Mode and users can select print devices from an application’s “Print” menu.

HP Universal Print Driver - Solution and Feature Guide

 

HP Universal Print Driver Series for Windows - overview and features

Contact HP
