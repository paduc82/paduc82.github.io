Each time you create an IE object, you create a new instance of IE in its own window. It also looks as if you should be using ie.Navigate2 to make it work with the constant that tells IE to open a new tab. This seems to do what you want:

Sub Weather()

     Dim strURL As String
     Dim objIE As Object
     Dim arrSites(4)
        
     arrSites(0) = "http://cleardarksky.com/c/AnaheimCAkey.html?1"
     arrSites(1) = "http://www.goes.noaa.gov/GSSLOOPS/wcir.html"
     arrSites(2) = "http://www.weather.com/weather/hourbyhour/l/92886:4:US"
     arrSites(3) = "http://www.wunderground.com/cgi-bin/findweather/getForecast?query=92886"
     Set objIE = CreateObject("InternetExplorer.Application")
     For i = 0 To 3 Step 1
        strURL = arrSites(i)
        If i = 0 Then
            objIE.Navigate strURL
        Else
            objIE.Navigate2 strURL, 2048
        End If
     Next i
     objIE.Visible = True
     Set objIE = Nothing

ekw590 say:

Cell-Fill Color based on Cell Value

I have a Column whose rows are coded "A", "B", or "C".  I'd like to color-code the cell-fill color: blue for value "A",  Yellow for "B"  and Green for "C".  How would I do this?   Thanks for you help.  EVan
tom-s say:

ekw590, how about this:

Sub ColorCode()
'
Dim i As Integer
'
Sheets(1).Activate
'
For i = 1 To Range("J1").End(xlDown).Row
'
    Select Case Cells(i, "J")
    '
        Case "A": Cells(i, "J").Interior.Color = vbBlue
        Case "B": Cells(i, "J").Interior.Color = vbYellow
        Case "C": Cells(i, "J").Interior.Color = vbGreen
    '
    End Select
'
Next i
'
End Sub

This assumes that your data is in column J of sheet 1, starting at J1 and having no blank cells before the very end of the data column.
