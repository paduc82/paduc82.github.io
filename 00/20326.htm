2- in the subform (="submask2") I have another combo box, necessary for me to filter again the values in the continuous form (using another field ="nameoffieldtofilter") and the combo box is in the mask header area: after updating this combo box ("ComboSubFormName"), I would filter the continuous mask.

I used these VB lines, but surely I made some syntax errors, since they don't work correctly:

    Me.Filter = "Me.mask1!submask2.Form![nameoffieldtofilter] = " & Me.ComboSubFormName
    DoCmd.RunCommand acCmdApplyFilterSort

graham mandeno say:

Hi Davide

The correct syntax is:

Me.Filter = "[nameoffieldtofilter]=" & Me.ComboSubFormName
Me.FilterOn = True

If nameoffieldtofilter is a text field then the value must be enclosed in either single or double quotes:

Me.Filter = "[nameoffieldtofilter]='" & Me.ComboSubFormName & "'"

This code should be in the AfterUpdate event procedure of your combo box.

If you sometimes delete the value from the combo box to renove the filter, then your code should allow for that also:

If IsNull(Me.ComboSubFormName) Then
    Me.FilterOn = False
    Me.Filter = ""
Else
    Me.Filter = "[nameoffieldtofilter]=" & Me.ComboSubFormName
    Me.FilterOn = True
End If

Best wishes,
Graham

In economics, a production–possibility In economics, a production–possibility frontier (PPF), sometimes called a production–possibility curve, production-possibility boundary orproduct transformation curve, is a graph that compares the production rates of two commodities that use the same fixed total of the factors of production. Graphically bounding the production set, the PPF curve shows the maximum specified production level of one commodity that results given the production level of the other. If you want the value of a cell to be the sum of a range, but not greater than a certain value, you can use MIN:

          =MIN(SUM(C1:G1),25000)

 

If you want to calculate the amount by which a sum exceeds a threshold, use:

          =IF(SUM(C1:G1)>25000,SUM(C1:G1)-25000,0)

          =MAX(SUM(C1:G1)-25000,0)
