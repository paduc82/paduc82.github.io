Private Sub Worksheet_Change(ByVal Target As Range)
    If Not Intersect(Target, Columns(3)) Is Nothing Then
        On Error GoTo FallThrough
        Application.EnableEvents = False
        Dim c As Range
        For Each c In Intersect(Target, Columns("C"))
            c = Application.Trim(Replace(c, Chr(160), Chr(32)))
            If CBool(InStr(1, c, Chr(32))) Then
                If Left(c, 1) = Chr(35) And IsNumeric(Mid(c, 2, 1)) Then
                    c.Offset(0, -1) = Left(c, InStr(1, c, Chr(32)) - 1)

teylyn say:

Hello,

if you have set up the conditional formatting with a formula, like

=A1=1

first make sure the formula does not have a $ sign in front of the 1 in A1.

Now there are several ways to copy the format.

Using Applies To

Then select the cell and click Conditional formatting > Manage rules. In the dialog click into the "Applies to" box and select the range that you want to apply the formatting to.

Using the Format Painter

Click the formatted cell, then click the Format Painter on the Home ribbon. Select the cells below the formatted cell and release the mouse. The format, including conditional formatting, has been applied to these cells.

Using copy and paste special

Select the cell and copy it.

Select the cells you want to apply the format to. Click the Paste dropdown on the Home ribbon and select Paste Special at the bottom, or use the keyboard shortcut Alt-e-s. Then click "Formats" in the dialog and select OK. Only the formatting has been copied to the selected cells.
