ken sheridan say:

You should be able to parse the string with the following function:

Public Function GetListItems(varItemList As Variant, _

                    strDelimiter As String, _

                    intStart As Integer, _

                    intNumber As Integer) As String

    Dim strItemList As String

    Dim intDelimLength As Integer

    Dim intSubStart As Integer

    Dim intSubEnd As Integer

    Dim n As Integer

    strItemList = "" & varItemList & strDelimiter

    intDelimLength = Len(strDelimiter)

    intSubStart = 1

    ' get start of substring

    If intStart > 1 Then

        For n = 1 To intStart - 1

            intSubStart = InStr(intSubStart, strItemList, strDelimiter) + Len(strDelimiter)

            If intSubStart > Len(strItemList) Then Exit For

        Next n

    End If

    intSubEnd = intSubStart

    ' get position immediately after end of substring

    For n = 1 To intNumber

        If InStr(intSubEnd + 1, strItemList, strDelimiter) > 0 Then

            intSubEnd = InStr(intSubEnd + 1, strItemList, strDelimiter)

        End If

    Next n

    ' return value is substring starting at intSubStart

    ' and of length intSubEnd - intSubStart

    GetListItems = Trim(Mid(strItemList, intSubStart, intSubEnd - intSubStart))

End Function

Taking your sample string as an example you can see how it works in the debug (aka 'immediate') window:

strList="a03.qwerty.qwerty.in.school.date/current.passed.willnot.parse.in.school.date/spon.atclass.loadfromdbase.onload.run.in.school.date/lapsed.over.parsethrough"

First parse the string to return the substring between the first instance of the delimiter "in.school.date/" and the next instance, if any:

strItem = GetListItems(strList,"in.school.date/",2,1)

Then parse the value returned by this to return the two substrings before the second instance of the  of the delimiter "."

? "in.school.date/" & GetListItems(strItem,".",1,2)

in.school.date/current.passed

Then parse the full string again to return the substring between the second instance of the delimiter "in.school.date/" and the next instance if any:

strItem = GetListItems(strList,"in.school.date/",3,1)

Again parse the value returned by this to return the two substrings before the second instance of the  of the delimiter "."

? "in.school.date/" & GetListItems(strItem,".",1,2)

in.school.date/spon.atclass

Repeat this process, returning the substring between the subsequent instances of the delimiter "in.school.date/" and the next instance if any:

strItem = GetListItems(strList,"in.school.date/",4,1)

? "in.school.date/" & GetListItems(strItem,".",1,2)

in.school.date/lapsed.over

Once the last substring has been returned a zero length string will be returned if any further substrings are sought.  You can see this in the debug window.

? Len(GetListItems(strList,"in.school.date/",5,1))

 0

So, for each row in the original table you'd need to put the two function calls in a loop, incrementing the value of the intStart argument in the first of the two function calls by 1 each time, and terminate the loop once the first of the function calls returns a zero-length string.  At each iteration of the loop you would insert a row into the table with the value returned by the second of the two function calls.

ztuka---01 say:

And this answer I received from th devoloper of myExplorer in March;

 

Hello!


 Unfortunately, myExplorer does not support WebDAV. However you can still connect remotely to your NAS or other file server in two ways:

    A. By forwarding the SMB port in your router.This is easy to setup but it is not very secure. While authentication information (user name and password) is protected, as they are never transmitted unencrypted, the rest of the traffic will go over unsecured connection. Here is what you have to do for this to work:

    1. Forward port 445 (or 139) on the file server (or NAS) in your router to another port (for example 22678 - the number is not important, what is important is to write it down for later).

    2. Create a new network share in myExplorer and type the external IP address of your router in the field "Computer IP address). If your external IP address is dynamic and you are using some kind of dynamic DNS, instead of IP address, please fill the DNS name of the router in the field "Computer DNS name".

    3. Fill in the port number from step 1 in the field "Port number"

    4. Select "Always ask" as access point because you may need to use different access points depending on your current physical location (3G on the road or Wi-Fi in the office, etc.).

    5. Please note that you will have to add exception to your firewall for port 445 or 139 as most firewalls block the ports 445 and 139 when the other IP address is not local. This is true even for the built-in firewall in Windows.

    B. By using VPN network.  This method is better from security point of view but it requires more work to setup. First you need to setup a VPN network from your phone to the LAN where the NAS is located. This can be done in few different ways and the concrete steps depend on what kind of VPN you are using. After you have working VPN, all you need to do is this:

    1. Fill in the NetBIOS name or the (local, not external) IP address of the computer.
    2. Select the created VPN access point as the access point for this network share.

    Please note that both methods above require some knowledge about the networks. We are planning to create a tutorial, which explains the needed steps but it is not ready yet.

Best regards!

 

myExplorer
