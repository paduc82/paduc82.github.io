    Can I drag and drop emaiil messages into the calendar? - Microsoft Community

    tlcarlson say:

    Can I drag and drop emaiil messages into the calendar?

    I would really like to be able to drag/drop emails to my Calendar.  Does anybody know how to accomplish this?

    [New question split by moderator from different question]
    william - mbpro say: I used the "drag-email-to-calendar" feature of Outlook 2010 for Windows all the time, so was disappointed when Mac Outlook 2011 did not have this feature.  I was able to replicate this ability by creating an Outlook Script in AppleScript.  I don't see a way to upload a ZIP of the AppleScript file to this message, or would have posted it.  Hopefully it is OK to copy/paste the script into this response (below).

    Once you have created this script in the "Outlook Script Menu Items" folder, it will be available to you from the Outlook 2011 script menu.  It will create a new event for your Calendar for the selected email Message when you run the script, which I named "Create Meeting from Message".  The script creates a Calendar "Appointment" event, since a "Meeting" requires you to invite other people and send as an email to actually save it to your Calendar. 

    Speaking of which... I wish there was a way to include an attachment on an "Appointment" Calendar event.  You can add an attachment to a "Meeting" Calendar event, but that requires you to email the Meeting to others to save it to your Calendar.  Why not allow an "Appointment" to include attachments???  I've submitted this request to Microsoft, and hopefully others will make the same request.

    Thanks... William W.

    ================================================
    Here is the script I created: "Create Meeting from Message"
    ================================================

    (*
        Create Meeting from Message
    *)
    tell application "Microsoft Outlook"
       
        -- get the currently selected message or messages
        set selectedMessages to current messages
       
        -- if there are no messages selected, warn the user and then quit
        if selectedMessages is {} then
            display dialog "Please select a message first and then run this script." with icon 1
            return
        end if
       
        repeat with theMessage in selectedMessages
           
            -- Store data from teh Message in variables
            set theName to subject of theMessage
            set theCategory to category of theMessage
            set theDateRecvd to (current date) + 1 * days
            set theHTMLStatus to has html of theMessage
            set theAttachment to attachment of theMessage
            set theEndDate to (current date) + (1 * days)
            set theEndDate to theEndDate + (1 * hours)
           
            set thePriority to priority of theMessage
            set theContent to content of theMessage
           
            -- Create a new Calendar Appointment event with the information from the Message
            set newAppt to make new calendar event with properties {subject:theName, start time:theDateRecvd, end time:theEndDate, has reminder:true, reminder time:5, content:theContent}
           
        end repeat
       
        -- If there was only one message selected, then create the Appointment.
        if (count of selectedMessages) = 1 then open newAppt
    end tell
