Hi,


Right click the worksheet tab, view code and paste the code below in on the right, close VB editor.


Whenever you change a1 then the filter on the table changes. You must now save your workbook as macro enabled with a .xlsm extension.



Private Sub Worksheet_Change(ByVal Target As Range)
If Intersect(Target, Range("A1")) Is Nothing Then Exit Sub
  ActiveSheet.Range("$A$5:$M$250").AutoFilter Field:=2, _

how do i avoid downloading old yahoo emails into outlook?

I recently signed up to be able to use outlook with my yahoo email account.  I have many years worth of emails on my yahoo account that i would prefer not downloading to my new computer.  Is there a way to exclude old emails without deleting them or putting them into non-inbox folders?  Thanks for any help! 
brian tillman [outlook mvp 2007-2019] say:
No, there's no way to do what you ask.  When you create a new mail profile, either on the same or a different computer, you no longer have the information the old profile has about what you've downloaded.  Outlook believes everything in the Inbox on the server is new.  The best solution, in my opinion, is to log into your Yahoo account via a web browser, create a new folder there, and move the Inbox messages you don't want downloaded again to that folder.  If they're not in the server-side Inbox, Outlook can't see them and won't download them.
