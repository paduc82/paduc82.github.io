Use the following code in the ThisDocument object of the template

Private Sub Document_ContentControlOnExit(ByVal CC As ContentControl, Cancel As Boolean)
For Each CC In ActiveDocument.ContentControls
    If CC.Checked Then
        CC.Range.Font.ColorIndex = wdRed
    End If
Next
End Sub

A formula can only copy data from the same Task, so to copy to all tasks requires a VBA macro. For example if you want to copy Text1 on task 1 to all text2 rows then the following works:

Sub Test()
Dim Tsk As Task
Dim Data As Variant
    Data = ActiveProject.Tasks(1).Text1
    For Each Tsk In ActiveProject.Tasks
        If Not Tsk Is Nothing Then
            Tsk.Text2 = Data
        End If
    Next Tsk
End Sub

conditional formatting based on number of characters in field - Microsoft Community

brirand say:

conditional formatting based on number of characters in field

Account numbers are either 6 or 8 characters in length.  In the merge, I need the 6 character account #s formatted 00 00 00 and the 8 character formatted 000 000 00.  How can I count the number of characters in the field?

a. user say:

1. Can any of the characters in these numbers be 0? even the leading characters (e.g., can you have 012345 ?)

2. If so, when the numbers come across to Word, are they being treated as numeric (in which case 

{ MERGEFIELD accountno } would insert 12345, and you would not be able to tell the difference between 012345 and 00012345) and you will have to fix the problem in Excel.

3. If the data is always reaching Word as a 6-character string or an 8-character string, then you may be able to use

{ IF "{ MERGEFIELD accountno }" = "????????" "{ MERGEFIELD accountno \#"000 000 00" }" "{ MERGEFIELD accountno \#"00 00 00" }" }

where all the {} are the "special field code brace pairs" that you can insert using ctrl-F9. 

 





jeeped say:
Read the instructions and I don't see this option in Word. I see them in Excel though.  I need to do this in Word.

I usually recommend that people format their data at the target (e.g. Word) rather than the source (e.g. Excel), but your situation was a special case where I believed that Excel would provide a much easier formatting method.

If it absolutely has to be done within Word then it can be done if the data is carried across as a numerical field. Tap Alt+F9 (e.g. show field codes) and modify your merge field to this,

{ IF { MERGEFIELD Accnt_Nos}<1000000 { MERGEFIELD Accnt_Nos \# "00 00 00" } { MERGEFIELD Accnt_Nos \# "000 000 00" } }

This will format account numbers that are numerically less than 1 million (e.g. 6 digits) as 00 00 00 and anything else as 000 000 00.
Note that you cannot just type in the field braces (e.g. { and }). You need to tap Ctrl+F9 then type inside the inserted braces.
