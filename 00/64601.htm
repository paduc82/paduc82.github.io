Hi Galtom,

 

Were there any changes made on the computer prior to the issue?

 

You might face this issue if the Windows Search Service is not configured correctly or if the user profile is corrupted. Follow these methods to troubleshoot the issue:

 

Method 1:

I would suggest you to run the Search and Indexing troubleshooter and check if the issue persists:

a. Press Windows key to launch Start screen.

b. Type “Troubleshooting” without quotes and click on Troubleshooting.

c. In the new window, click View all on the left pane.

d. In the list of options, click on Search and Indexing.

e. Click Next and follow the on-screen instructions.

 

Method 2:

I would suggest you to follow these steps and start the Windows Search service:

a.     Press Windows key + R on the keyboard.

b.    Type “services.msc” without quotes and press Enter.

c.     Scroll down to find Windows Search.

d.    Right click and select Properties.

e.    Set the Startup type of Windows Installer to automatic.

f.      Click Start to start the service. Note down the error message, if any.

 

Method 3:

I would suggest you to follow these steps and configure the Windows Search Service to log on as local system account:

a.     Press Windows key + R on the key to bring up Run.

b.    Type “Services.msc” without quotes and press Enter.

c.     Note: If you are prompted for an administrator password or confirmation, type your password, or click Continue.

d.    Right click the "Windows Search" service and choose Properties.

e.    Click on the "Log On" tab.

f.      Make sure “Local System Account” is checked.

g.    Click on Apply.

h.    Under General tab, click on Start to start the service.

 

Let us know if you need further assistance and we’ll be happy to help.
