james vinyard say:

Okay, I think all you wanted was to turn off the "live" tile and without keyboard, simply press and hold down on a tile and when you see a square shape appear, release and a mouse menu will appear and you can shut off the "live" mode.

For any running apps, usually, like what LCRT had said, the Metro UI apps don't really eat up much in resources.  When not displayed when you switch apps, they usually eat up about as much in resources as a RSS news feed which is nothing.  Some apps like the Metro UI version of Internet Explorer can cause the tablet not to shut down by not closing when you power off.  That is also nothing and you just force it to close.  That doesn't happen all the time.

Any Metro UI app you want to just plain exit and not run in the background, swipe from the left side of the screen and the Metro UI task manager (slide out bar) showing the running apps and you press and hold down on the app you want to exit and then drag it to the right and then down to the bottom of the screen and then back to the left, far left corner and you will see the app 'flip' and then it is gone.

An alternative to that if you have the keyboard attached is CTRL + ALT + DEL and select the Task Manager and those apps should display in the list and click on "End Process".

It all may be goofy but welcome to a Microsoft version of Android!  You have two systems instead of one - an Android like system and a super modified Windows 7 system.  Metro UI is the "Android" like system and is why those apps will not run on the "Desktop".  That changes in Windows 10 where the apps run in a "run time window".  That way, they can run anywhere.

When you upgraded from a previous version of Windows or receive a new computer preinstalled with Windows 10, what happened is the hardware (your PC) will get a digital entitlement, where a unique signature of the computer will be stored on Microsoft Activation Servers. Anytime you need to reinstall Windows 10 on that machine, just proceed to reinstall Windows 10. It will automatically reactivate. You will prompted to enter a product key a couple times through the installation, click I don't have a key and Do this later.

Windows 10 - Disable click to tap and gestures (ELAN) - Microsoft Community

lancetti say:

Windows 10 - Disable click to tap and gestures (ELAN)

I just wanted to share this information that I found after a lot of searching for answers.

I just purchased a new laptop with Windows 10 Home edition. I was able to figure out that I had an ELAN touchpad, but no controls to modify the settings beyond sensitivity in anyway. So if I wanted to turn off scrolling, or make the touchpad sensitive to my palm, or stop the click to tap feature, I was unable to. A lot of the search answers involved either drivers or registry editing, which I didn't want to do a registry edit. My laptop kept erroring on installing the Synaptics controls from the Microsoft Store, and the Dell touchpad controls didn't work and kept forwarding me to the "mouse properties" window. I did finally find a solution!

TL;DR: Windows 10 Home with an ELAN touchpad does not come installed with the ELAN software. You'll need to install it yourself.

  1. Make a System Restore point.
  2. Make sure you have allowed applications to be installed NOT from Microsoft Store.
  3. Search up "Mouse Settings". On the right, click "Additional Mouse Options." It'll open a WinXP looking window titled: Mouse Properties. You're looking for the red ELAN logo on the tabs (should be on the right). [In this process I did find that there's two different Mouse Properties windows, one will show the ELAN icon and one will not.]
  4. Click the ELAN tab. At the "enable" area of the window, it should be "enable"d. Double-click the word "ELAN smart-pad Enable" or whatever is listed, and Windows 10 should prompt to ask you to install a software. Click confirm.

    If you want to make sure your touchpad is working, select for it to "Display Dynamic/Static Icon" and the icon will appear in the task area in the start bar.
  5. Once the software is installed, it should automatically open the ELAN touchpad software.



    This software is FREE.

The laptop I bought should've been updated with this software but for some reason it wasn't.

It seems that upon using this software I've been able to turn off the "gestures" that came with Windows 10.

The only follow-up you might need to do once you've installed and set-up the ELAN software is restart. I didn't need to, but I need to update the ELAN mouse drivers to a version from July 2018 (auto-found through the internet search).

I really hope this solution helps people who are having issues with their touchpads!
