At the time that you’ve changed the password of your Microsoft account using a different PC, the old password from the computer that you don’t have access to is still remaining in there.

Now, to have this password removed and changed it to the new password, we’ll need to enable the built-in Windows 10 administrator account to perform this action. Here are the steps:

  1. Create a bootable media tool using a USB drive from a working computer. Click here to download the Windows 10 Disc Image (ISO File)
  2. Insert the media creation tool.

    Note: Make sure that the USB drive is in the boot priority. If you're unfamiliar doing this, we suggest contacting the manufacturer of your device for further assistance.
  3. Once the Install window appear, select Repair your computer.

  4. Select Command Prompt from the Advanced options window.

  5. Type the command: net user administrator /active:yes and press Enter.
  6. Close the window, reboot your computer, and then immediately unplug the media.
  7. Once the system restarted already, you’ll see the list of profiles, then log in to the user named Administrator.
  8. When you’ve reached the desktop view of Windows, type Computer Management in the search box and click on the result.

  9. Go to Local Users and Groups and double-click Users.
  10. Select the problematic account/profile, right-click on it and select Set Password > Click Proceed > Enter the new password of the Microsoft account that you’ve set from another computer > Click OK.

  1. In the Search box, type cmd, and then press CTRL + SHIFT + Enter.    
  2. If the User Account Control dialog box appears, confirm that the action it displays is what you want, and then click Continue.    
  3. On the Command Prompt, type net user administrator /active:no and press Enter.
