There are no settings that need to reset.

Search your hard disk for PERSONAL.XLSB

If you can not find this file continue Step 2 below.

If you found this file:
- Open Word

- Search the PERSONAL.XLSB in the project explorer (press CTRL-R if not visible)
- Double click it.
- Explore any module (click the + to view the tree, double click any module name)
- If you see some text on the right side:
  - Mark all text
  - Right click it
  - Select Copy
  - Switch to Word
  - Paste the text

repeat until you have copied anything to Word. Save the Word file anywhere.

- Close Excel
- Delete the PERSONAL.XLSB


- Open Excel
- Start record a macro, select "Personal Macro Workbook" as destination
- Do anything (select some cells), we need an action to create a macro.
- Stop record the macro
- Open the VBA editor
- Search the PERSONAL.XLSB in the project explorer
- Double click it.
- Search the created module, double click it.
- Remove the macro
- Click the menu Tools\Options
  - Check any item in the Editor tab except the first "Automatic syntax checking"
  - Click OK

If you have saved some macros:


- Open the save Word file
- Copy all text into that module
- If you see some red text in that module:
  - Remove that macro that includes the red text, it's damaged.
- Click the menu Debug\Compile VBAProject
- If an error is shown:
  - Remove that macro where the error is shown, it's damaged.
  - Repeat Debug\Compile until no error is shown
- Close the VBA editor
- Test the rest of your macros if any

FIXED ON SAMSUNG GALAXY S2?
 
I had the same problem with hundreds of duplicated emails in my outlook sent mail folder. I have a Samsung Galaxy S2 and I use the native email app (which I prefer after trying many alternatives). I cleared the problem using the native email app on my phone.  I deleted the outlook account [open app; settings; bin in top right corner; delete] then added it back selecting 'Manual Setup'.  When the app then asks 'What type of account?' I selected 'Microsoft Exchange ActiveSync'.

The settings needed are as follows...
 
Domain\user name: \[Your outlook email address]
Password: [Your outlook password]
Exchange Server: s.outlook.com
Use Secure connection (SSL) [Tick yes]
Use client certificate: [Leave blank]

All your emails will be back in place on the phone within a few minutes once you have worked through a few more settings which are all ok at their default values.

I can now send emails from my Samsung with only one instance appearing in my outlook sent folder :)

Basically,
- You start by emptying the “Favorites” tracklist.
