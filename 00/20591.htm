a. user say:

Ran into this same issue with one of my systems.  This is how I fixed it:

1)  Download the latest driver from the manufacturer's website.

2)  Download the MSFT tool that allows you to block certain driver updates through Windows Update.

3)  Disable all network access (enter Airplane Mode and disconnect network cables).

4)  Uninstall your touchpad driver.  Click "delete the software for this device" if the checkbox is there.

5)  Restart your system.

6)  When Windows reboots, install the driver you downloaded but do not reconnect to the Internet.

7)  Verify that your touchpad gestures are working as you want them to.

8)  Go back to Device Manager and see what the driver version number is that's installed.

9)  Restart your system again.

10) When your desktop reappears, disable Airplane Mode and reconnect to the 'net.

11) Run the MSFT tool you downloaded; if it spots anything for your touchpad, tell it to ignore that update.

12) Make sure the tool does everything it needs to do, then close it, and run Windows Update.  If it downloaded a new driver anyway, let it install, but then when it does, go back to Device Manager and roll the driver back to the previous version.

It's convoluted, I know, but it's proven effective for me.

* Restart Outlook.

- Go to the "Control Panel" (as mentioned above).
