brian tillman [outlook mvp 2007-2019] say:

There are third party tools you can buy that will do the job.  See these:
http://www.slipstick.com/outlook/sync-outlook/synchronizing-outlook-on-two-machines/
http://www.slipstick.com/outlook/sharing/sharing-outlook-calendar-and-contacts/

Another, unsupported method can be used if you are willing to take some chances.  You can put a PST on a network share and reference that same PST from two systems.  If you elect to do this, be sure to make frequent backups because, while the method usually works, you do run the risk of corrupting the PST if something happens to the network connection.  The PC that hosts the network share must be on when the remote computer accesses the PST and only one instance of Outlook can be open at any one time; you can't access the PST simultaneously from multiple PCs.  Furthermore, if using POP accounts, neither PC will know that the other PC has downloaded messages so if you leave copies of messages on the server.  Each PC will download anything it sees as new regardless of whether or not the other PC has downloaded it, so you can get duplicates.  If you've interested in sharing mail only, then an IMAP server is better because it's designed for access by multiple clients.

You can also use the Outlook Hotmail Connector and access the same Live/Hotmail mailbox, with its attendant mail folders, calendar, and contacts.  If you make this mailbox the default data file, then both PCs will be synchronized automatically without the dangers of sharing a PST.
