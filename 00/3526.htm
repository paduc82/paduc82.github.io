bob jones aka: cybertaz say:

Admittedly, a bit confusing, but in regard to the Locked property: In a new Excel worksheet (PC or Mac), the status of all cells is for the Locked property to be enabled [i.e., the box is checked]. However, the property does not go into effect unless Protection is applied to the worksheet. When Worksheet Protection is applied, Locked cells no longer are editable... editing is restricted to only those cells for which the Locked property has been removed [i.e., the check has been cleared].

Re the new issue: Go to Excel> Preferences - Edit, then clear the check for Display this number of decimal places. This Preference is not a formatting tool. It effectively intercepts & alters the entry to comply with whatever its value is set to. Formatting simply determines how many decimal places of the entered value is displayed, it doesn't change the value. The Preference should be used only if there is a specific & valid reason for having all entry comply with what it dictates.

jay freedman say:
Yes, you can do this, but it's more a function of Windows 7 than of Word or Office.

Click in one of the programs (Word or Windows Explorer), hold down the Windows key, and tap the left arrow key to make that program occupy the left half of the screen. Repeat for the other program, but tap the right arrow key instead, so that program occupies the right half of the screen. Then, if necessary, you can drag the vertical edges of the two programs so one is less than half and the other is more than half.

Your contacts are backed up to Windows Live. So to edit your contacts on the computer, visit contacts.live.com and sign in with the Windows Live ID you use on the phone.

Windows Phone cannot be used as a mass storage drive, so it won't show in Explorer.

entegy say:

www.windowsphone.com/help has a lot of great resources. There should also be a Help + How-To app on your phone.

As for contact sync, it will happen automatically and over-the-air. So just edit away on the website and you'll see the contacts on your phone.
