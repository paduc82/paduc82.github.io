joeu2004 say:

Since the VALUE function returns an error, the data is not recognized as numeric text.  There can be many reasons for that.  Since you copied the data from website, perhaps it contains non-breaking spaces (HTML &nbsp).  Try the following paradigm:

=--TRIM(SUBSTITUTE(A1,CHAR(160),""))

The double-unary negative (--) has the same purpose as calling the VALUE function.
