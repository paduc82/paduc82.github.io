We develop software for use with Zebra printers, and have tested these printers on Windows 7 and Windows 8. Do the following to get it up and running...

Update Firmware
You will need to update the Zebra LP2844 firmware, you'll need it connected to a parallel cable to update the firmware. 

Note: The LP2844 uses EPL, the LP2844-Z uses ZPL, you need the EPL Firmware, they physically look the same but speak a totally different language... don't get them mixed up.
Download the Zebra Firmware Here

Download and Install the Zebra Drivers
Download the Official Zebra LP2844 Printer Driver Here (Working - These drivers aren't available on the Zebra website anymore.) There are three different versions of the LP2844 driver in this zip file, if one doesn't work, try the next one.

You MUST install the driver in Windows XP Compatibility Mode. The LP2844 is old, and so are the drivers. The new Z-Designer drivers on the Zebra website aren't so good.

When you run the file, it will extract the exe to your machine. Go to the location the files were extracted to and right click on setup.exe then click Properties > Click on the Compatibility tab and select Run this program in compatibility mode for: Windows XP. Then click Run as administrator and click Apply and OK.

Now run the setup.exe file
Select Language: English
Click Next
Select Zebra EPL and Select Zebra LP2844
Click Next
Click Add a new printer
Click Next
Now select the parallel port LPT1/LPT2/LPT3
Click Next
Assign the Printer A Name or leave default Zebra LP2844
Click Next
Leave default dimensions
Click Next
Dispense Mode: None
Click Next
Media Type: Direct
Click Next
Media Tracking: Use Label Gaps
Click Next
Then Click Finish

These aren't digitally signed drivers for Windows 7, if it prompts you, allow the install still.

Now to test the printer, click on Start > Devices and Printers
Double click on the Zebra LP2844 printer
Then click Printer > Properties
At this point if it says, the driver isn't installed fully, Add the Printer Driver by clicking on Have Disk on the Add Printer Driver Wizard
Then click Browse, and browse to the location the Zebra drivers were extracted to... and select the USBPRINT.INF file and click Open.
Then press OK
Now you can see the Zebra drivers... Under Manufacturer select Zebra EPL > and under Printers select Zebra LP2844
Click Next > Click Finish
Now the Printer is Installed!
 
If this didn't work for you, use this same method but with the Z-Designer Drivers from the Zebra Website instead.
