pamcaswell say: Are you using the same paragraph style for all list levels?  If you modify one of the paragraph indents and then update the style, the settings for that level will apply to all levels.  Instead, adjust the indents for each level-- or change the number/bullet style-- in the define new multilevel list dialog and leave the paragraph style settings alone (zero indent).

You don't mention autonumbering in your post, but if that's what you want, instead of bullets, linking a different style to each number level gives more reliable results.

You also have the alternative of vertical justification. Try it with different parameters (Space before, Space after, Line spacings). You'll find it in Format > Document > Layout tab, and then Vertical alignment. The dropdown menu to its right controls the range where you'll apply it, so to do it only on select portions, you'll have to use Sections (see Insert > Break for various options).

Are you up to installing macros? These will work with single paragraphs only. If you like them and want them to work on several paragraphs, its a simple fix. Let me know.

Sub IncreaseLS()
Selection.ParagraphFormat.LineSpacing = Selection.ParagraphFormat.LineSpacing + 1
End Sub

Sub DecreaseLS()
Selection.ParagraphFormat.LineSpacing = Selection.ParagraphFormat.LineSpacing - 1
End Sub

Once you paste them in your macro editor (Tools > Macro > Macros > Edit), you can assign them keyboard shortcuts through Tools > Customize Keyboard, choosing Macros on the left list and the above macros on the right list.
