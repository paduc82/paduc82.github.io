wunders say: To begin with, you have XP Home which means that "Simple File Sharing" is enforced on that machine.  Further, that means that any network connection to your XP Home machine from any other machine is forced to authenticate as "Guest" user, no matter what privilege that the actual user may have.  Going from here, it would appear that the "Guest" user does not have privilege to the files or folders that you are sharing.  To fix this, you need to change the access permissions to these files.

If you drag (move) files from elsewhere on your computer into your share folder, the original permissions on the file will be maintained.  Rather, I would suggest that you copy the file into the share folder by dragging with the right mouse button and select "Copy" instead of "Move".  A copied file will end up with permissions matching the destination folder which usually will be more suitable for sharing.

For files and folders that are already in this shared folder, to change permissions you will need to access the "Security" tab under the properties screen.  In XP Home, this is only possible if you log in as an administrator in "Safe Mode".  So start up your computer in Safe Mode (repeatedly tap the F8 key while booting), log in as an administrator user, navigate to the file or folder with access problems, right-click on it, select "Properties", select the "Security" tab, and set the security such that "Everyone" (or at least  "Guest") has the access privilege that you desire.
