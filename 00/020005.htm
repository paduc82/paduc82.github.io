zorvek say:

Normalized scientific notation states that only one digit is displayed to the left of the decimal point. Any other representation is therefore open to confusion and should be avoided, especially if you intend to show the results to anyone else. The problem with trying to manipulate a "standard" is that the observer, operating under generally accepted standards, will be prone to assume certain aspects of what is observed and you will therefor be creating a situation where either 1) the assumption will be accepted without further scrutiny and the observer left with incorrect knowledge, or 2) the observer will notice something odd about the presentation and will then move into a position of having to think about the displayed value and do more mental math to know the actual value. Neither of these is considered good practice, especially when presenting information to others.

I want to point out that the real definition of the pound sign is that it is a placeholder only, it does not instruct Excel to push more or less digits into the resulting formatted text. The pound signs are used only to designate the thousands comma position if and only if there are more than three digits to the left of the decimal point.

From Excel's help on the use of the pound sign in a custom format: "This digit placeholder follows the same rules as the 0 (zero). However, Excel does not display extra zeros when the number that you type has fewer digits on either side of the decimal than there are # symbols in the format. For example, if the custom format is #.##, and you type 8.9 in the cell, the number 8.9 is displayed."

chris trabing say:

There are two methods in which a WP8 gets data.

 

Firstly, through your Mobile Telecommunications provider.  Typically your Mobile Connection will have a Voice component and a Data Plan.  If you were on a pre-paid plan with no Data, then you could go to Settings, Mobile Network and set the Data Connection to off.  If you did this, your phone would never be using data - you wouldn't be able to get e-mail or access bing or browse to an internet site.

 

Secondly, through a WiFi Access Point - settings; WiFi; Turn on and connect to a Wireless Access Point.

 

Many people use both connections.  If WP8 has access to a WiFi Access point, it will use this prior to using the Mobile Connection for Data.  For example, I have a Wireless Access point at home which my WP8 uses when I'm at home.....and when I'm out and about it uses my Data Plan.  When I arrive at work, it uses the Wireless Access Point at work for my data requirements.

 

If the phone is in locked mode and put away.....it is still accessing data.  You realise this when you see updates available in the Store when you look at your phone first thing in the morning.  There are some things you can do to minimise data use.  For example, Settings, System, phone update - ensure that "Automatically download updates if my data settings allow it" is deselected.
