karl timmermans [outlook mvp 2012-2018] say:
I don`t have .bak file. Where can I find it?

If the option to create the backup file was manually unchecked for some reason (it is checked by default), then no .bak file will be created. If it was checked, then the default location for the backup file would be in the same folder where the PST file existed but it can be optionally changed to another location. In short, if no changes were made to ScanPST after selecting the PST file to be scanned, a file with a .bak extension will be created.

While unlikely, but just to cover all the bases, if you happened to run a disk clean-up program, it's quite possible that files with a .bak extension were deleted.
