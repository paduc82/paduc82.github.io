patc---01 say: You can either use the PC Suite "Store images" tool or you can simply use Windows Explorer to drag and drop the photos to your PC.

In the first case, you will need to install PC Suite (obviously, given the name, you computer has to be a Windoze PC for this), then connect the phone via cable, bluetooth or IR in PC Suite mode and select "Store images". The rest is pretty much automatic. Note that you may need to check your settings if you want to keep the pictures on the phone as well as copy them to the PC.

Using drag & drop is more straightforward (and way faster): connect the phone with the cable (no need to install any software), select data transfer mode, navigate to [mass memory]:\Images in explorer and simply copy the files to wherever you want them on the PC. Of course, if you have pictures in phone memory, you'll first need to use File Manger on the phone to copy the pictures to mass memory.

Alt Text for pictures not displaying in IE 8

alexander tarvid say: Hello, ScottSven.

The following comes from MSDN (http://msdn.microsoft.com/en-us/library/cc288472.aspx):

The alt attribute is no longer displayed as the image tooltip when the browser is running in IE8 mode. Instead, the target of the longDesc attribute is used as the tooltip if present; otherwise, the title is displayed. The alt attribute is still used as the Microsoft Active Accessibility name, and the title attribute is used as the fallback name only if alt is not present.

Hope this helps to clarify the issue.
