Macro I used:


Sub SetColumnWidthMM(ColNo As Long, mmWidth As Integer)
' General formula to change the column width to mmWidth
Dim w As Single
    If ColNo < 1 Or ColNo > 255 Then Exit Sub
    Application.ScreenUpdating = False
    w = Application.CentimetersToPoints(mmWidth / 10)
    While Columns(ColNo + 1).Left - Columns(ColNo).Left - 0.1 > w
        Columns(ColNo).ColumnWidth = Columns(ColNo).ColumnWidth - 0.1
    Wend
    While Columns(ColNo + 1).Left - Columns(ColNo).Left + 0.1 < w
        Columns(ColNo).ColumnWidth = Columns(ColNo).ColumnWidth + 0.1
    Wend
End Sub

teylyn say:

Hello,

The setting to show "categories in reverse order" is applied to the X axis. When you send one series to the secondary axis (which is NOT called the Z-Axis, by the way, but the secondary Y-Axis), Excel also generates a secondary X-Axis, which is hidden by default.

In order to reverse the categories on the series that are plotted on the secondary axis, you need to show the secondary X axis. It will be displayed at the top of the chart.

Select the axis and format it with the setting to plot "categories in reverse order", just like you did with the primary X axis.

After that, you can hide the secondary X axis again or leave it in the chart and set it to no line and no labels to make it invisible.
