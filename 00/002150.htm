cjlim say:

Your phone could have been asking for 1 of 2 codes.

 

The first is the SIM card PIN. This PIN should have been given to you with the SIM card. If you enter the wrong PIN more than 3 or 5 times (can't remember which) it locks it permanently and you need the PUK code available from your network to unlock it. 

 

The second is the phones Lock Code. It is a 5 digit number and is usually 12345 by default. You have already tried this code and it doesn't work. Sometimes phones are shipped with a different code but when this happens the code is usually printed on the label on the box. If you can't find the code or if you changed the code but cannot remember it, your only option is to get it reset at a Nokia Care Point. Most Nokia Care Point will only reset it for you if you can prove ownership so bring proof of ownership with you.

 

From your description, it sounds like the Lock Code to me.
