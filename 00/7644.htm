rich michaels - mvp (greatcirclelearning.com) say:

Your VBA routine does not work because the "boxes" around those paragraphs are not shapes, they are borders.

Try this code ...

Sub RemoveParagraphBorders()
    Dim para As Word.Paragraph
    For Each para In ActiveDocument.Paragraphs
        If para.Borders.Enable = True Then
            If para.Borders(wdBorderLeft).LineStyle = wdLineStyleNone And _
                para.Borders(wdBorderTop).LineStyle = wdLineStyleNone Then
                'let pass
            Else
                para.Borders.Enable = False
            End If
        End If
    Next
End Sub

It is constructed the way it is because your documents also have a vertical right border on paragraphs where changes have been made, so the above routine bypasses those.

Assuming this works for you an an actual document and not just the test document you supplied, if you want to run the routine as a batch process on all of the documents you have ... take an look at this Wiki article that provides the code you will need to make the above routine process multiple documents. hansv mvp say:

You could create a macro in your Normal.dotm template:

 

Sub InsertStyleSeparatorNoSpace()
    Selection.InsertStyleSeparator
    Selection.TypeBackspace
End Sub

 

and assign it to Ctrl+Alt+Enter (the keyboard shortcut for the style separator).

a. user say:

Hello Theodore,

You may use Autofilter to find values, show or hide values, in one or more columns of data. You may filter the data based on the choices you make from the list, search to find the data that you want to see. When you filter data, entire rows are hidden if values in one or more columns don't meet the filtering criteria. Here's how to use the Autofilter:

  1. Select the data you want to filter.
  2. Click Data > Filter.
  3. Click the arrow in the column header and do one of the following:
    • Select specific values: Check (Select All) to clear all of the check boxes, and then select only the values you want to see.
    • Search for values: In the Search box, type the text or numbers you want to see.
  4. Click OK to apply the filter.

If you want to apply a predefined or custom filter to your data, see Filter data in a range or table.

carolejeanx say:

You could use this macro with some tweaking, I think.  I got it from OssieMac today.

Sub HideR()
   
    Dim lngLastRow As Long
    Dim r As Long
   
    With Worksheets("Sheet1")
        lngLastRow = .Cells(.Rows.Count, "A").End(xlUp).Row  'Last used row in column A
        For r = lngLastRow To 2 Step -1     'If deleting rows then must work backwards
            If .Cells(r, "B") = .Cells(r, "A") + 30 Then
                .Rows(r).Hidden = True
                '.Rows(r).Delete     'Uncomment to delete row and comment out previous line
            End If
        Next r
    End With
   
End Sub
