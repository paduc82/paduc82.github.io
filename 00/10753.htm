Hi there,

 

Without knowing more, such as where your data is, here is a somewhat generic piece of code you could use from Excel to send information to an appointment item of a specific calendar (works in 2010 32bit, haven't tested in 64bit, but I think it should work)...

 

Option Explicit

Sub MoveToOutlook(rMove As Range)
    Dim OL As Outlook.Application
    Dim NS As Outlook.Namespace
    Dim olFolder1 As Outlook.Folder
    Dim olFolder2 As Outlook.Folder
    Dim olAppt As Outlook.AppointmentItem
    Dim bOLOpen As Boolean
    'If you want to check if the 'rMove' is a valid range, this
    'would be the place to put your checks in place for it.
    On Error Resume Next
    Set OL = GetObject(, "Outlook.Application")
    bOLOpen = True
    If OL Is Nothing Then
        bOLOpen = False
        Set OL = CreateObject("Outlook.Application")
    End If
    On Error GoTo 0
    Set NS = OL.GetNamespace("MAPI")
    Set olFolder1 = NS.GetDefaultFolder(olFolderCalendar)
    Set olFolder2 = olFolder1.Folders("YOUR FOLDER NAME GOES HERE")
    Set olAppt = olFolder2.Items.Add("IPM.Appointment")
    With olAppt
        .Start = YOURSTARTTIME
        .End = YOURENDTIME
        .Subject = YOURSUBJECT
        .Location = OPTIONALLOCATION
        .BusyStatus = olBusy 'BUSYSTATUS
        .ReminderMinutesBeforeStart = 240 'REMINDER TIME IN MINUTES
        .ReminderSet = True 'REMINDERORNOT
        .Save
        .Close olSave
    End With
    If bOLOpen = False Then OL.Quit
End Sub

Public Sub TOGGLEEVENTS(blnState As Boolean)
'Originally written by Zack Barresse
    With Application
        .DisplayAlerts = blnState
        .EnableEvents = blnState
        .ScreenUpdating = blnState
        If blnState Then .CutCopyMode = False
        If blnState Then .StatusBar = False
    End With
End Sub

For Excel 2007 and later

SUMIFS(SumRange,Criteria1 Range, Criteria1,Criteria2 Range, Criteria2)

Work like the SUMIF but this will allow you to specify multiple criteria.  One difference to note it that for SUMIFS the sum range is the first item.  For SUMIF if you were using a sum range different than you criteria range it was the last range.

Yes, that happens because the web site icons are stored with your web surfing history in the IE History folders.

If you have set IE to delete your history on exit or only maintain a few days history the icons on the desktop will be deleted when the history that contains the icon is deleted.

You can stop this from happening by right clicking on the desktop shortcut and choosing Properties, then on the Advanced tab, 'Change Icon" and select an icon from your hard disk and not from your Internet History cache.
