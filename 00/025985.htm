jay freedman say:
I think you got close to the correct explanation in your last sentence -- the click event handler code isn't connected to the event because the control doesn't exist when the userform object is created.

The solution is to create all three option buttons in the designer and write their event code. In the initialize event, set the .Visible property to False for each option button that you don't want shown. You can do the same with the option buttons in the second frame. When the code sets a button's .Visible property to True, its click event will become active.

yuan12 say:

how to find the specific sheet (at the bottom of excel) among hundreds of sheets in Excel (Mac)?

As show above, there are hundreds of different sheets in excel but how can I find "the specific sheet name" among them? I tried to search sheet but this function works in one sheet only. Could anyone give me some advice on how to deal with this basic question in Mac excel? thanks.

ossiemac say:

Not sure if this works in Excel for Mac but I am presuming that it is not much different in the user interface to Excel for PC.

Towards the Top left of the active window (to left of formula bar) there is a field that displays the currently selected cell.

Screen Shot displaying the field referred to.

Click in the field and replace the currently selected cell with the required worksheet name followed by exclamation mark and A1 (eg. Sheet1!A1 similar to the screen shot)

Press Enter and you should go to cell A1 on the worksheet.

Alternatively on the Home ribbon, select "Find & Select" -> GoTo and enter the sheet name, exclamation mark and A1 (eg.  Sheet3!A1) and click OK.

If you want to have a DropDown with the sheet names then the field towards top left (left of the formula bar) displays all of the Defined Names in the workbook by clicking the DropDown at the right.

Therefore by using VBA code to create defined names for cell A1 in each of the worksheets then you can select the defined name representing the worksheet. I suggest prefixing the sheet name with "Sht_" or similar so that they all sit together in the DropDown and avoid confusion for the user with the other defined names in the workbook.

The following VBA code will create the Defined Names for the DropDown. If you require more information to install the code in a standard module then see the end of this post.

Sub CreateSheetList()

    Dim ws As Worksheet
   
    For Each ws In ThisWorkbook.Worksheets
        ws.Range("A1").Name = "Sht_" & ws.Name
    Next ws
   
End Sub

To install the VBA code: (Applies to PC. Might be slightly different on Mac)

  • Alt and F11 to open the VBA editor window
  • In the VBA editor select menu item Insert -> Module
  • Copy the VBA code above and paste into the VBA editor.
  • Close the VBA editor (Cross very top right of VBA editor window)
  • Save the workbook as Macro enabled.
  • Ensure macros are enabled as follows.
  • If Developer ribbon not displaying then right click anywhere in one of the ribbons and select "Customize the ribbon".
  • On the right side of the dialog check the box against "Developer" and click OK.
  • Select Developer ribbon.
  • Select Macro Security (In the Code block at left end of Developer ribbon)
  • The dialog that displays should default to Macros in left column.
  • Select required security option button. (Option to "Disable all macros with notification" should be OK.)
  • Click OK to close the dialog.

The 400 Bad Request error is an HTTP (Hyper Text Transfer Protocol) status code that means that the request you sent to the website server (i.e. a request to load a web page) was somehow malformed therefore the server was unable to understand or process the request.

Check to see if the following method helps.

Method 1: Check your URL (Uniform Resource Locator). If you typed the URL wrong, you will get Error Code 404. Even a single letter or number in the URL can make a difference.

Method 2: Turn off friendly HTTP error messages in Internet Explorer

a)       To do these go to Tools -> Internet Options -> Advanced.

b)       Uncheck show friendly HTTP error messages.

Once you have done this, you should get a more detailed message that will point you to the real problem.

Method 3: Clear Internet Explorer cache

a. Open Internet Explorer.
b. Click Tools and then Internet Options.
c. Click “Delete” under “Browsing History”.
d. Check all the options and click “Delete”. Click OK.

 

Method 4: Add to trusted websites.

a. Open Internet Explorer.

b. Click on Tools, select Internet Options.

c. Click on the Security Tab.

d. Select the Trusted Site option

e. Click on Sites. You can add the sites to the zone and all the websites in this zone will use the zone’s security settings.

f. Click on Ok.

suzanne s. barnhill say:

Paper size can be set in Word 2013 in the same place as in any other version: the Paper tab of Page Setup. This can be accessed in several ways:

  1. The easiest way is to double-click at the top of the horizontal ruler. This opens the Page Setup dialog; select the Paper tab.
  2. On the Page Layout tab, click the dialog launcher (tiny arrow) in the bottom right corner of the Page Setup group. This opens the Page Setup dialog; select the Paper tab.
  3. In the Page Setup group on the Page Layout tab, click Size and choose More Paper Sizes... This opens the Page Setup dialog with the Paper tab selected.

On the Paper tab, as shown below, you can choose "Custom size." Or, if you enter measurements that are different from those of any standard size, Word will change the setting to "Custom size."

How do I remove my initials from the comment bubbles? - Microsoft Community

janiceleuschen say:

How do I remove my initials from the comment bubbles?

I've been asked to remove my initials from the comment bubbles in a document I edited.  How do I do that?

suzanne s. barnhill say:

Go to Office Button | Prepare | Inspect Document. Clear all the boxes except "Document Properties and Personal Information," then click Inspect. Word will inform you that it has found some metadata. Click Remove All. When you save, "Author" will be substituted for your name and "A" for your initials.
