I finally figured out what  have to do to share my Windows 7 external hard drive. I stumbled on this totally by accident; I have not seen it documented in any of the places OI have looked for help.

What I had to do was:

1. Un-share the drive.

2. Create a master folder on the drive (I called mine X Drive.)

3. Share the new X Drive folder with everybody, giving full read-write permission.

ken blake say:
On Mon, 7 Jun 2010 19:24:25 +0000, John A. Roberts Jr. wrote:
 
 
> How do I print all the NAMES of the files in a file folder? I do not want to print what is in the actual file, just the names.
>
> i.e. When I open a folder it gives me the names of all the files in that folder. I just want to print this lists of names.
 
 
 
Here are four ways:
 
1. Go to a command prompt and issue the command
 
dir [drive:folder] > c:\tempfilename (you can use any name and put it
in any folder you want)
 
Then open notepad, open tempfilename, and print it from there.
 
2. Write (for example in Notepad) a 1-line text file:
DIR %1 /O >LPT1:
 
Save it as "printdir.bat" in the "Send To" folder.
 
Then, to print list of files in any folder, right-click that folder
and select Send to | printdir.bat
 
To include subfolders, change the command to DIR %1 /O/S >LPT1:
 
3. Go to
http://support.microsoft.com/default.aspx?scid=kb;EN-US;Q321379 and
follow the instructions there.
 
4. Download and use any of the several freeware/shareware utilities
that can do this, such as the popular
http://www.karenware.com/powertools/ptdirprn.asp
