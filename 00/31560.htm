cameron parrish say:

If you would prefer to use a tool built into Windows, there is a way to change things with an extremely complicated tool called bcdedit.exe. I played around with it and I found out that it is way easier if it was done from Windows 8.1.

 

So, start the computer into Windows 8.1 and paste this command into an administrator command prompt:

 

bcdedit /set description "windows 8.1"

 

Why a command prompt? It will stay open and tell you that it completed successfully. But if you want, you can paste it into the search charm and press ctrl+shift+enter.

 

But, please be careful with this command. Bcdedit is capable of rendering a system inoperable, but the instructions I provided you with should not cause this.
