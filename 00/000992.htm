andreas killer say:

The apostrophe char is a reserved char in Excel that is used to mark an input as text, e.g. if you enter 001 into a cell, Excel convert that into a number 1, but when you enter '001 Excel accepts 001 as text.

The first apostrophe is still in the cell, you can see it in the formula bar. If you want to see the first apostrophe in the cell itself you have to write two apostrophes.

Don't worry about the first (of the two) apostrophes, formulas and VBA ignores that char.

B1:  =MID(A1,1,99)
