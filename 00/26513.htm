we don't have to be a site admin to see the people and groups option, we can assign the group owner(s) a customized permission level:

pj98rx say:

As noted by metalfan above, it would be easiest if you check the required WLAN settings from your WLAN administrator. There are various authentication methods that you can configure in E71 and it can be hard to find out the right one by just testing.

 

You could start by checking with Connectivity > WLAN wizard the WLANs that your device detects. Select Options > Details to check the security mode that is used with the WLAN. This tells whether it's an open network or if it requires WEP or WPA/WPA2 authentication. If it's WPA/WPA2, you should try to check with your WLAN administrator what EAP plugin settings need to be selected.

 

You can create the WLAN access point by selecting Options > Define access point in the WLAN wizard but you need to complete the configuration in the Settings > Connection > Access points menu.

 

If your WLAN network requires username and password to be configured, it could be e.g. WPA/WPA2 with EAP-LEAP but there could be also some other alternatives. If EAP-LEAP is used, select in the "WLAN security settings" menu WPA/WPA2 = EAP and in "EAP plug-in settings" menu enable just the EAP-LEAP authentication and define the user name and password in the EAP-LEAP menu.
