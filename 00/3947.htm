jetta48 say:

Hi

In IE11 Desktop, there is no specific option that allows the tab sizes to be customised. As the no. of tabs increases to the point that there is no more horizontal space for them, all the tab sizes widths decrease to fill the space to exactly fit the available space to the right of the "address bar" (where the URL is shown). If you close enough tabs, their widths expand until they reach the same size as when there is only one tab open.

You can drag* the right edge of the "address bar" (where the URL is shown) to increase it's width and this will reduce the available space for tabs. The tabs will still behave as described above though. Similarly, you can drag to reduce the width of the "address bar".

"Provided that you do not gave tabs set to show on a separate row.

Hi,

 

Thank you for the reply and sorry for the delay in response.

 

POP3, IMAP and EAS (Exchange Active Sync) are the type of protocols which your email account can be configured on Outlook.

 

Outlook supports these protocol formats:

POP3- Post Office Protocol 3

IMAP - Internet Message Access Protocol.

EAS - Exchange Active Sync ( This setup can be done only on Outlook 2013 for the Live accounts like Hotmail, Live, Outlook.com or MSN).

 

Since you are using Outlook 2013, your Outlook.com account is setup as Exchange Active Sync (EAS).

Creating Contact groups in this account type is not available. If you had any groups created in your contacts list, if the email account is setup in EAS protocol, it will not display those contacts or you will be able to create new contact groups.
