barry xie msft say:

Hi Sal_mm,

 

I’d like to say that the expanding and collapsing behavior of Outlook is random. But there are two basic rules:

1. Outlook should remember which folders were expanded or collapsed when you closed Outlook and should also start with this folder state again.

2. If a folder is collapsed and a new message is delivered to one of its subfolders, the folder will automatically expand to that subfolder.

 

Since you don’t want your mailbox folders collapsed, please add your Inbox and other folders that receive mail to Favorites list. Also, navigate to File > Options > Advanced > Outlook start and exit > Start Outlook in this folder > Select an email folder which do not receive emails.

 

If the problem still persists, please provide us the following information:

1. What version of Outlook client do you use?

2. Did this problem occur before? If not, do you install any application or add-ins which may cause this problem?

 

Regards,

Barry
