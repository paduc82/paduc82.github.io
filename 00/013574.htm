Dim oRF As RecentFile
Dim WSHShell As Object
Dim strRegKey As String, strKeyWord As String
    Select Case Application.Version
      Case "12.0"
        strRegKey = "HKEY_CURRENT_USER\Software\Microsoft\Office\12.0\Word\File MRU\"
      Case "14.0"
        strRegKey = "HKEY_CURRENT_USER\Software\Microsoft\Office\14.0\Word\File MRU\"

    End Select
    Set WSHShell = CreateObject("WScript.Shell")
    ActiveDocument.Range.Delete
    For Each oRF In RecentFiles
      strKeyWord = WSHShell.RegRead(strRegKey & "Item " & oRF.Index)
      ActiveDocument.Range.InsertAfter strKeyWord & vbCr
    Next oRF
    ActiveDocument.Range.Paragraphs.Last.Range.Delete
End Sub

This will extract registry values and create a list of all recent files in the document.  Save and close the document.

Open the document in Word 2013.  Using the VB Editor, paste in the following code:

Sub AppendItems()
Dim oRng As Range
Dim arrItems() As String
Dim WSHShell As Object
Dim strRegKey As String, strKeyWord As String
Dim lngIndex As Long

  Set oRng = ActiveDocument.Range
  oRng.End = oRng.End - 1

    Case "14.0"
      strRegKey = "HKEY_CURRENT_USER\Software\Microsoft\Office\14.0\Word\File MRU\"
    Case "15.0"
      'Navigate to the following key in the registry:
      'HKEY_CURRENT_USER\Software\Microsoft\Office\15.0\Word\User MRU
      'Then expand it and then expand your unique LiveId key.

      strRegKey = "HKEY_CURRENT_USER\Software\Microsoft\Office\15.0\Word\User MRU\LiveId_4R6FC9D843C9DA15F07724077E72FDB43FD7AAF7CF282A95C68DE18A179A858A\File MRU" & "\"

hansv mvp say:

A (sub)form in Datasheet View cannot display pictures; it only displays text.

A (sub)form in Continuous Forms view can display pictures. There are two ways to do this:

1) Create an OLE field in the table; populate the field with the pictures.

Create a Bound Object Frame control on the form and set its Control Source to the OLE field to display the pictures.

2) Create a Short Text field in the table and populate it with the path and filename of pictures stored outside the database.

Create an Image control on the form and set its Control Source to the text field to display the pictures.

The address book in Outlook contains nothing.  It is just a view of those Contacts folders that you have enabled as address books.  All your contacts are in Contacts folders.  Right-click Contacts, choose Properties, select the Outlook Address Book tab, and check the box labeled "Show this folder as an email Address Book".  Click OK.  If the box is inactive and cannot be checked, then your mail profile is damaged and you need to create a new one.  http://support.microsoft.com/kb/829918
