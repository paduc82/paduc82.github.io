Its not that the Xbox 360 S can not play original xbox games, its that the 4GB flash memory does not have the emulator required to play the original xbox games. This is how,

System Settings > Memory > Left Bumper, Right Bumper, X, Left Bumper, Right Bumper, X > Yes

Then once you start your xbox and connect to xbox live, you will be prompted to re download the system update. If this does not help or if you have any questions, let us know.

DMZ

Login Information

  • Gateway - 192.168.1.1 or 192.168.0.1
  • Username - admin
  • Password - password

DMZ

  1. Open a web browser.
  2. Enter http://192.168.1.1 or http://192.168.0.1 into your address bar. (Or click either link)

  3. If prompted, enter "admin" for the user name and "password" for the password.

  4. From the menu at the top, click on Security.

  5. Select DMZ Hosting from the menu on the left.

  6. Select the radio button for "On".
  7. Click on the drop box labeled "DMZ Host PC Name" and find the xbox in the list either by IP address or MAC Address.
  8. Click Apply at the bottom of the page.

 

Disable DMZ

  1. Open a web browser.
  2. Enter http://192.168.1.1 or http://192.168.0.1 into your address bar. (Or click either link)

  3. If prompted, enter "admin" for the user name and "password" for the password.

  4. From the menu at the top, click on Security.

  5. Select DMZ Hosting from the menu on the left.

  6. Select the radio button for "Off".
  7. Click Apply at the bottom of the page.
