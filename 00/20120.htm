mike h.. say:

Hi,

Excel has no inbuilt functionality to do that so we can use a User Defined Function and below are 2. One to sum bold cells and one to sum none bold.

ALT+F11 to open VB editor. right click 'ThisWorkbook' and insert module and paste the 2 modules below in

Back on the worksheet call with

=sumbold(B1:B6)

or

=sumnotbold(B1:B6)

Note bolding a cell  does not  cause Excel to recalculate but these will recalculate whenever Excel does

Function SumBold(rng As Range)
Dim C as range
Application.Volatile
For Each c In rng
    If c.Font.Bold = True Then
        SumBold = SumBold + c.Value
    End If
Next
End Function


Function SumNotBold(rng As Range)

Dim C as Range
Application.Volatile
For Each c In rng
    If c.Font.Bold = False Then
        SumNotBold = SumNotBold + c.Value
    End If
Next
End Function
