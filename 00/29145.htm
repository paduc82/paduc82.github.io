Hi,

 

Why do I receive this message from Windows?
You experience a message as this one when a couple of PCs are connected
to a network. Each PC has an IP-address, however, when two PCs have each one
IP-address that is identical to the other, and then there may be a problem. Windows
then notifies one of the clients about this, and requires that the administrator make some changes on the network settings. This can be an issue with the DHCP on the router.


Resolution:
Check Dynamic Host Configuration Host Protocol (DHCP) settings on the router.
Check each IPv4 address on each PC connected to your network.

Follow the methods.


Method 1

1. Click Start Menu and in the Instant Search type “Command Prompt” then
right-click and left-click Run as administrator.
2. Type the first command ipconfig /release and pressEnter.
3. Type then the second command ipconfig /renew and pressEnter.


Explanation:
The fist command will send a DHCPRELEASE message to the DHCP server to release the current
DHCP configuration and discard the IP address configuration for either all adapters, the second
command Renews DHCP configuration for all adapters.


For more details see:
http://technet.microsoft.com/en-us/library/bb490921.aspx

Method 2
Run the network troubleshooter to diagnose and repair any problem related to your network. Access the link mentioned below for steps:
http://windows.microsoft.com/en-US/windows7/Using-the-Network-troubleshooter-in-Windows-7

 

Method 3
Uninstall the adapter and then reinstall the latest driver.

a) Click Start, type Device Manager in the Start search box and hit Enter.
b) Locate Network adapters and expand the same.
c) Select the wireless connection, right-click and select Uninstall.
d) Restart the computer, if prompted.
e) Drivers should get installed automatically after restarting the computer.
Now, you may install the latest wireless drivers and check if that fixes the issue.

Update a driver for hardware that isn't working properly
http://windows.microsoft.com/en-us/windows7/Update-a-driver-for-hardware-that-isnt-working-properly

 

Method 4
For a full list of steps to try, see the Wired and Wireless tutorial on this page.
Wired and wireless network connection problems in Windows
http://windows.microsoft.com/en-US/windows/help/wired-and-wireless-network-connection-problems-in-windows

 

Also refer the link:

Get help with "There is an IP address conflict" message
http://windows.microsoft.com/en-US/windows7/Get-help-with-There-is-an-IP-address-conflict-message

  • booting from recovery disk (or Install USB Disk)
    • Startup Repair
    • Command Prompt
      • "bootrec /rebuildbcd"
      • Bcdboot C:\Windows
  • booting from another OS (Win7 on another hard drive)
    • Command Prompt
      • sfc /scannow /offbootdir=d:\ /offwindir=d:\windows
  • An more that I won't enumerate here
