You may wish to try removing the scanner software and reinstalling it using the procedure below; the scanner should be connected to the PC and powered on.

5) Close DEVICE MANAGER and turn the scanner off.

http://windows.microsoft.com/en-US/windows-vista/Add-or-remove-a-scanner

1) Click on the Windows Orb, and then CONTROL PANEL.

2) In Control Panel, double-click on Programs and Features

3) Remove EPSON Scan. This is the only item you need remove here, if asked to restart the PC at any point, un-tick the box that might make reference to this option and choose not to restart. Close the window.

4) Now hold down the Windows Logo key + "R" key on your keyboard. This will open the 'Run' dialogue box. In the Run box, next to where it says Open type inf and click OK. This will open up a list of files; we need to locate the files beginning with OEM. If the Run box does not appear, type inf into Start Search and open the yellow inf folder that will be amongst the search results.

5) There may be several files beginning with OEMxx where xx is a number. You can ignore those with the .pnf extensions. The OEMxx files are in fact the original INF files located on the drivers CD (any manufacturer) that Windows has copied to the INF folder and then renamed. You need to open each OEMxx.inf file. Do this by double clicking on it. Please note that before proceeding to step 14, you MUST NOT delete any OEMxx.inf files that do not have "SEIKO EPSON" in them.

6) If the .inf file reads SEIKO EPSON at the top and the Scanner details further down, then close the file and delete the corresponding file(s) i.e. if OEM1.INF is the correct file, then delete oem1.inf and oem1.pnf. Repeat until all EPSON OEMxx.INF files have been deleted. Please also look for any files (not OEM) that have 'E' and 'Scan' in the file name. Open the .INF files for any found matching this description and check if the scanner is mentioned here. Remove if the scanner name is found.

7) Click the Windows Orb menu and click on Computer.

8) Click on Local Disc C drive (where ‚ÄòC‚Äô is the drive letter on which Windows is installed). Click on Windows folder and then double-click on Twain_32. In the Twain_32 folder there should be a folder called ‚ÄòESCNDV'. Right-click this folder and Delete, then close back to the desktop.

Once removed, restart the computer.

Now get the latest software from Epson and install it. This is all you need to do to get the Epson all-in-one to work like charm.

cturn-d99a90fe-2031-444a-95e7-b7bbe7392604 say:

Controller Freeze / Lockup Fix (Microsoft Tech Support)

Hello All,

I've read plenty of post about controller freezing and not reconnecting at times. I spoke with Microsoft Tech support rep: Kristin J and these are the steps she gave me to resolve my issue.

How the Issue Occurred:

My problem occurred when I turned on my Xbox One controller with the headset adapter and my headset plugged in. I noticed that the headset adapter was not outputting sound so I disconnect the headset adapter and re-plugged it into the controller in attempt to get sound. This put my controller in the FREEZE/LOCKUP state at which I was unable to use the controller with the console. Below are some steps I had tried before contacting Xbox support. 

Methods tried for getting controller out Frozen State:


Took out the battery and tried plugging it back in. (Result: Controller still inactive, Xbox one Guide button will remain lit but no function)
Took out battery and plugged in controller via USB cable "Play n' Charge cable or Headset adapter cable" (Same result as before)
Took out battery, plugged it back in and tried to reconnect the controller via IR blast to the Kinnect (Same result as before)

*START HERE FOR FIX*

1. Remove AA batteries or rechargeable battery from  controller

2. Turn off your console and check to see if the power supply light is ORANGE

3. If the power supply light is ORANGE as the console is off, unplug the power supply from the wall / surge protector
ORANGE as the console is off, unplug the power supply from the wall / surge protector

4. Wait for the ORANGE light to go away on the power supply once you've unplugged the power supply.

ORANGE light to go away on the power supply once you've unplugged the power supply.

5. After the light has gone away, wait 30 seconds and plug the power supply back in.

6. Plug the batteries back into the controller and turn the controller on. (This should turn on your Xbox and you should be OK)

If this did not solve your issue, I'm sorry and please contact Microsoft Support with your issue. I'm just posting things I've tried to solve this issue and the steps given to me from the Microsoft Rep that resolved my problem. Hopefully this will help someone and I hate to say this but, "When your console is in doubt, power cycle it out!) It's fixed most of my problems in the past.
