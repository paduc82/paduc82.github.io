bill manville say:

One way is to simply filter the row or column field to show only the value you want in the chart.

You can do this within the pivot table or in the pivotchart if you have added controls to it.

But if you want to see the entire table but only chart a portion of the data you would be best creating a range containing the data to be charted, with the cells referencing the pivot table data, and create a normal chart from that range.

By using the GETPIVOTDATA function you can make the range pick up the data for a specific combination of items (e.g. Sales in North in Quarter 1)

=GETPIVOTDATA("Sum of Sales",$A$3,"Quarter","Q1","Region","North")

and it will not matter if the layout of the pivot table changes on refresh or on user action as long as that information is still visible in the pivot table.  Also you can use cells to contain the values you want to get the data for (e.g. "North" in my example) can be referenced from the GETPIVOTDATA function.

=GETPIVOTDATA("Sum of Sales",$A$3,"Quarter","Q1","Region",$C$1)
where C1 contains "North"

hansv mvp say:

As you have found, {...} only works with constant values, not with cell references.

You can use the following formula instead:

=SUMPRODUCT(SUMIF(A2:A7,D2:D3,B2:B7))

or as an array formula confirmed with Ctrl+Shift+Enter

=SUM(SUMIF(A2:A7,D2:D3,B2:B7))

But you could also use

=SUMIF(A2:A7,D2,B2:B7)+SUMIF(A2:A7,D3,B2:B7)

jamesyan_0708 say:

hi andre,

we can save the folder in one of the user's onedrive for business library then share the folder with each user with specific permissions (can edit or can view).



the user can go to his own onedrive for business page > shared with me section to check the shared folder.
as the owner, he can also go to share > shared with to change the user's permission to this folder or stop sharing with the user.

the user can also sync the shared folder to local via clicking the sync button.

about “we want to limit there storage space usage as well”, my understanding is you want to limit their own onedrive for business storage, if so, this is not feasible. the onedrive for business storage limit is 1tb for each user. you can refer to onedrive for business limits part in this article.
if there is any misunderstanding, please post back.
