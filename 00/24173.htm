john w. vinson say:

If you just want to display the contents of the pdf file for the user to see, then you just need to open the file in one of the numerous applications which display .pdf files. Gina's FollowHyperlink suggestion will use whichever app you have defined as the default pdf viewer on your system.

If you want to link or import information IN the pdf file into your database, that's a whole lot tougher. Generally .pdf files don't contain data in computer readable form; the file is more like a graphical image of the data, and you would need to use Optical Character Recognition software to translate the dots and background into text strings.

It appears that a small number of Windows 10 or 8.x systems have experienced this issue, usually those have been upgraded or reverted to the earlier operating system version, though that's not consistent.  However, the common issue is that they have apparently experienced registry corruption for some reason.

The following will take you to an unsuccessful attempt to fix this on Windows 10, but that also contains a link in the first post to a similar thread for Windows 8.1, where at least a couple people appear to have figured out how to fix it for themselves.

https://answers.microsoft.com/en-us/windows/forum/windows_10-win_cortana/settingcontent-ms-filetype-no-longer-recognized/4509f704-6512-490b-ab4d-99f6b0590222

In any case, the corruption issue is complex enough that none of the Microsoft support personnel appear to have the skill to aid in fixing it.  So if you're not skilled enough to solve the problem using the information from the 8.1 thread, which of course might be different for a Windows 10 system, then performing a Windows 10 Restore operation might be the best course to fix it permanently.

Of course, you'll need to backup all your personal files before starting and insure that you can reinstall all of the applications, especially older ones if this was an upgrade once the process is completed.  If you're also having other issues since an operating system upgrade this should fix those as well, but it's not an easy process if you have either lots of older data or programs you'll need to recover.

Sorry there isn't better news, but simply understanding what you're up against is the most important thing to understand before beginning any type of repair.
