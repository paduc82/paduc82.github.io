There is no real explanation why the account stopped working after several years of working fine, but it came down to the email account being an ‚ÄúEASI‚Äù Account. A way to test if this the case for you, is to check if you can access the mail using ‚Äúprivate‚Äù mode on a browser. For me, this failed.

 

To resolve the issue, I had to create a new email alias for the account (*** Email address is removed for privacy ***) using the Microsoft account management tool (http://account.microsoft.com), and make that the primary account for login (don‚Äôt delete the old, leave it as secondary). After I did that, and created the outlook account on the phone once more but now with the new email as login, the email started working again.

 

Also, had to ensure from the mailbox (under accounts/connected accounts) that the old email is still the default send from address (and not the new email that is just for login purposes).

C:\Program Files\Zune\

Click Run
