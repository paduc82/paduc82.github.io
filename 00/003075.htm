yorickdowne say:

Edge rendering of roll20.net

Edge is struggling with roll20.net , up to current build 10130. I'll detail some of the things I've noticed, and make the case as to why to pay attention.

Why care: roll20.net is very much "the modern web." If it's in HTML5, it probably uses it. From GPU-accelerated canvas to clever scripting to WebRTC, it's all there. As such, it's one good test case for support of feature-rich sites. It renders well on Chrome and Firefox, not at all on IE, and from what I hear okay-ish but not great on Safari.

How to test: You do need a roll20.net account, and need to create a campaign to test. I'm hoping that's not a deal breaker for the dev team. Accounts are free.

Alternative: I'd be happy to invite a tester into a campaign so they can test without needing to create their own campaign. I'm thinking that's not how testing works within MS, but I'll throw the option out there just in case.

Issues I have seen in build 10130, testing with an AMD 7970 GE GPU:

- Background map canvas goes blank (white) after clicking on the Journal tab. To test, create a campaign and create a map with a picture (JPEG) as a map background

- 3D dice do not show even when that option is selected. To test, enable 3D dice in options tab to the right, then roll dice from chat (/roll 1d20) or from dice roller in side bar

- None of the macro buttons within a character sheet work. They will highlight blue, but not execute. To test, create for example a D&D 5e campaign, create a character in the Journal tab, and try one of the "Save" macros in the first General / stats tab of the character sheet.

Sub Test()

  Dim FilePath As String

  On Error Resume Next
  Set WB = Workbooks("QG Add-in.xla")
  On Error GoTo 0
  If WB Is Nothing Then
    'Add-In is not loaded!?!
    FilePath = "\\server\path\QG Add-in.xla"
    Stop 'Correct the path and remove this stop!
  Else
    'Get the path
    FilePath = WB.FullName
  End If
  'Unload it
  WB.Close False
  'Change Attribut
  SetAttr FilePath, vbNormal
  'Open it again
  Set WB = Workbooks.Open(FilePath, False, False)
  'Write and save it
  WB.Sheets(1).Cells(1, 1) = "Hello World"
  WB.Save
  'Close it again
  WB.Close
  'Change Attribut
  SetAttr FilePath, vbReadOnly
  'Open it again
  Set WB = Workbooks.Open(FilePath)
End Sub
