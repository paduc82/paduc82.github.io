Details of Every Room Calendar

I've added many room mailboxes to my organization. I need to have all the details shown for every single room so the delegate can see it on the calendar. I know how to do it one by one for each mailbox but that would be extremely time consuming. Is there a command that will do all rooms at once?

thanks

vasil l. michev say:

There are few factors in play here. First, to make sure that the Calendar info for all rooms is visible for all users in your tenant, run these cmdlets:

$rooms = Get-Mailbox -RecipientTypeDetails RoomMailBox

$rooms | % { Set-MailBoxFolderPermission -AccessRights LimitedDetails  $($_.PrimarySmtpAddress + ":\Calendar")   -User default -whatif }

Next, you have to make sure that the Room mailboxes do not automatically replace the event information:

$rooms | % { Set-CalendarProcessing $_.PrimarySmtpAddress -AddOrganizerToSubject $True -DeleteComments $False -DeleteSubject $False }
