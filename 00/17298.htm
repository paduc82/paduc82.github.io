When making a section in OneNote 2016 into a notebook, you have to manually create a new notebook first and the desired number of sections. Once you've created both, you may just copy or move the files or information from the specific section that you would like to convert as a notebook. You may follow the steps below:

  1. Create a new notebook by clicking File > New to display the New Notebook options.

Formula for adding a date/timestamp to a cell if text is present - Microsoft Community

patsve say:

Formula for adding a date/timestamp to a cell if text is present in another cell.

Hi all,

 

I write small notes in Excel. For each new note I use a new row (A1, A2, A3.. and so on). It would be very useful to me if I could add a date or time stamp to the next cell each row (B1, B2, B3... and so on).

 

If i write some text in cell A1 then I would like the date / time to appear in B1.

 

Can you help me with a handy formula which I can add to B1 that checks if there's text in A1 and if true then it adds the date / time stamp.

jeeped say:
You can accomplish this with an event triggered sheet macro. Right-click the worksheet's name tab and choose View Code. When the VBE opens, paste the following into the pane titled something like Book1 - Sheet1 (Code), 

Private Sub Worksheet_Change(ByVal Target As Range)
    On Error GoTo Fallthrough
    If Not Intersect(Target, Columns("A")) Is Nothing Then
        Application.EnableEvents = False
        Dim b As Range
        For Each b In Intersect(Target, Columns("A"))
            If CBool(Len(b.Text)) Then
                b.Offset(0, 1) = Now
            Else
                b.Offset(0, 1).ClearContents
            End If
        Next b
    End If
Fallthrough:
    Application.EnableEvents = True
End Sub

Tap Alt+Q to return to your worksheet. Anything typed into column A will immediately receive the current date/time in column B. If you delete the content from column A, column B will likewise be cleared.

ken sheridan say: Try something like the following procedure, which accepts the object name as a string and the object type as an integer (for which there are built in constants).  It opens the object transparently in design view and changes properties of its controls, changing FirstName and LastName controls' FontWeight properties  to Bold in this example:

Public Sub ChangeControlProps(objName As String, objType As Integer)

    On Error GoTo Err_Handler
   
    Dim obj As Object
    Dim ctrl As Control

    Application.Echo False
   
    Select Case objType
        Case acForm
        DoCmd.OpenForm objName, acDesign
        Set obj = Forms(objName)
        Case acReport
        DoCmd.OpenReport objName, acDesign
        Set obj = Reports(objName)
    End Select
   
    Set ctrl = obj.Controls("LastName")
    ctrl.FontWeight = 700
    ' code to change more properties of control
   
    Set ctrl = obj.Controls("FirstName")
    ctrl.FontWeight = 700
    ' code to change more properties of control
   
Exit_Here:
    DoCmd.Close objType, objName, acSaveYes
    Application.Echo True
    Exit Sub

Err_Handler:
    MsgBox Err.Description & " (" & Err.Number & ") ", vbExclamation, "Error"
    Resume Exit_Here
   
End Sub

Call it like so:

ChangeControlProps "frmContacts",acForm
or:
ChangeControlProps "rptContacts",acReport
