argel obm say:

Hi Rejitha,

You can perform multilevel figures in Word by using chapter headers. You can divide your images into chapters to make a multilevel figures. For example, If you have "Chapter 1" and insert some images under it, you can set the caption to include the chapter number. So, if you have five images under "Chapter 1" all of your images will have a caption like "Figure 1.1", "Figure 1.2, and so on. For more information, you can visit this article. If you want a letter to your captions like Figure 1.2a or Figure 1.2b, there is no function yet to do this in Word unless you do it manually. You can post this function as a suggestion by accessing this link. Post your suggested function there and Microsoft may include it on the future update of Word.

If you have any questions, feel free to reply on this thread.

suzanne s. barnhill say:

As Argel says, there's no way to add the a and b except manually, but here's how to repeat the number:

  1. When you use References | Captions | Insert Caption, click the Numbering button in the Caption dialog; in the Caption Numbering dialog, check the box for "Include chapter number" and select the heading level of the autonumbered heading you want to use to provide the "chapter number." Select "period" as the separator.
  2. If you have selected Heading 1, you will get Figure 1.1. If you examine the field code for this caption (press Alt+F9), you will see that it is in this form:
    Figure { STYLEREF 1 \s }.{ SEQ Figure \* Arabic \s 1 }
  3. By default, the SEQ field will increase by 1 with each caption and restart after each Heading 1. In order to repeat a number, you must add a \c switch. So, for example, in the caption you want to be 1.2b, you add the switch to get this result:
    Figure { STYLEREF 1 \s }.{ SEQ Figure \c \* Arabic \s 1 }
  4. When you update the fields, you will have two captions numbered 1.2, and you can manually add a and b to the end of the caption label.

steve rindsberg say:

yes one person will be in control of the game, 

Perfect.  So here's what you do:

For squares whose color you want to control, name them Question 1, Question 2 and so on.

Exactly as shown re caps/lowercase and a space before the number.

You can use the Selection Pane to name them.

Then for each square, add two other shapes, named Red 1 and Green 1 (then Red 2, Green 2 and so on).  The red and green shapes for each question can be atop the corresponding Question shape and made nearly transparent if you like or can be to wherever else on the slide you like.

After you've created all the RedX and GreenX shapes, add this macro to the file

Sub Recolor(oSh As Shape)

    Dim sShapeNumber As Long
   
    If Left$(oSh.Name, 3) = "Red" Then
       
        sShapeNumber = Mid$(oSh.Name, 5)
        With oSh.Parent.Shapes("Question " & sShapeNumber)
            .Fill.ForeColor.RGB = RGB(255, 0, 0)
        End With
   
    Else
       
        sShapeNumber = Mid$(oSh.Name, 7)
        With oSh.Parent.Shapes("Question " & sShapeNumber)
            .Fill.ForeColor.RGB = RGB(0, 255, 0)
        End With
   
    End If

End Sub

The reason for the fussiness with naming is that the macro will sort out which shape to make what color automatically. Otherwise you'd have to write different macros for each shape.
