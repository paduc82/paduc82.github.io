amit tandon say:

Array formula (Ctrl+Shift+Enter) in cell K1:

=SUM(IFERROR((FREQUENCY(IF((B1:E7=1)*(ROW(B1:E7)-ROW(B1)+1<=4),COLUMN(B1:E1)-COLUMN(B1)+1),COLUMN(B1:E1)-COLUMN(B1)+1)>0)*TRANSPOSE(F1:I7),0))

The formula considers the column (B:E) only if 1 is present and in the 1st four rows only. If the column is considered (in the above example columns B,C & D are considered) then corresponding columns from F:I are considered to SUM (ie. columns F, G & H are considered).

If 1 is present in any column, in the first 4 rows, then that column is referenced to SUM corresponding column(s) in F1:I7

Note: there could be multiple 1s in a column - need to be considered once only.

In case this is your requirement.

peter atherton say:

Anna

I had another look at this and wrote a Visual Basic function for this. It is tested with a couple of data ranges and seems to give the correct answer

Function SumDataIf(ByVal Crit As Range, Data As Range)
Dim tmpCrit As Range, tmpData As Range
Dim cfRow, cfCol, cRowCount, cColCount
Dim sCrit As String, sData As String
Dim dfRow, dfCol, dRowCount, dColCount
Dim i As Long
Dim mysum As Double

'Get criteria details for counting rows
cfRow = Crit(1).Row
cfCol = Crit(1).Column
cRowCount = Crit.Rows.Count + 1
cColCount = Crit.Columns.Count + 1
'Get data details for summing rows
dfRow = Data(1).Row
dfCol = Data(1).Column
dRowCount = Data.Rows.Count
dColCount = Data.Columns.Count
mysum = 0
'You can delete this if you like!

If dRowCount <> cRowCount Then
    MsgBox "Row count must be the same in  both " & _
    "criteria and data ranges!", vbCritical
    Exit Function
End If

For i = cfRow To cRowCount
    Set tmpCrit = Range(Cells(i, cfCol), Cells(i, cColCount + (cfCol - 1)))
    sCrit = tmpCrit.Address
    Set tmpData = Range(Cells(dfRow, dfCol), Cells(dfRow, dfCol + dColCount))
    sData = tmpData.Address
    With WorksheetFunction
        If .CountA(tmpCrit) > 0 Then
            mysum = .Sum(Range(sData)) + mysum
        Else
            'do nothing
        End If
    End With
    dfRow = dfRow + 1
    'MsgBox tmpCrit.Address
Next i
SumDataIf = mysum
End Function

A lot of the code is just getting the ranges for criteria and data ranges.
