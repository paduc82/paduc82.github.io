yangzheng_msft6300 say:

Hi Myles,

Firstly, whether a message is determined as spam could be related to multiple factors including spam filtering, transport rules or some other settings. So there are no exact tools for an Office 365 admin to achieve your requirement.

Then about whether a message is determined as a spam in Office 365 Exchange Online, there is a SCL (Spam Confidence Level) rating assigned to each message when it goes through spam filtering. This SCL rating helps determine whether a message is spam. If the SCL value is 5 or 6, this message is considered as suspect spam. If the SCL value is 7, 8, or 9, this message is considered as high confidence spam. Users can view the SCL rating of a message by checking Internet header information for an email message in Outlook client. And here is an article about detailed SCL value for your reference.
https://technet.microsoft.com/en-us/library/jj200686%28v=exchg.150%29.aspx

If you want these legit emails from the specific sender not to be marked as spam, you can use transport rules in EAC to bypass spam filtering or add this sender to the allow list of the spam filtering.
