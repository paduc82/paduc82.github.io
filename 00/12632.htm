pesach shelnitz say:

Hi

 

You can have what you are requesting today by using macros, to which you can easily assign a shortcut key. The following macros automate the processes that Suzanne described. The first macro (Overbar) is intended for a single character, and the second macro (OverbarLong) is intended for a longer expression. Simply type the single character or expression and then run the appropriate macro to add an overbar.

 

Sub Overbar()
    Dim myField As Field
    Dim myChar As String
   
    With Selection
        .MoveStart Unit:=wdCharacter, Count:=-1
        myChar = .Text
        .Delete
        Set myField = ActiveDocument.Fields.Add(Range:=.Range, _
        Type:=wdFieldEmpty, PreserveFormatting:=False)
        myField.Code.Text = "EQ \o(" & myChar & "," & ChrW(&HAF) & ")"
        myField.ShowCodes = False
        .MoveRight Unit:=wdCharacter, Count:=1
    End With
    Set myField = Nothing
End Sub


Sub OverbarLong()
    Dim myField As Field
    Dim MyWord As String
   
    With Selection
        .MoveStart Unit:=wdWord, Count:=-1
        MyWord = .Text
        .Delete
        Set myField = ActiveDocument.Fields.Add(Range:=.Range, _
        Type:=wdFieldEmpty, PreserveFormatting:=False)
        myField.Code.Text = "EQ \x\to(" & MyWord & ")"
        myField.ShowCodes = False
        .MoveRight Unit:=wdWord, Count:=1
    End With
    Set myField = Nothing
End Sub

 

Hope this helps,

Pesach

The following modified code is untested but should do what you want.

Sub EmailWithOutlook()
    'Variable declaration
    Dim oApp As Object, _
    oMail As Object, _
    WB As Workbook, _
    FileName As String
   
    'Turn off screen updating
    Application.ScreenUpdating = False
   
    'Make a copy of the active sheet and save it to
    'a temporary file
    ActiveSheet.Copy
    Set WB = ActiveWorkbook
   
   'New workbook will have data on worksheet(1).
    'Cells is the entire worksheet range of new worksheet
    'Copy and PasteSpecial -> Values for entire worksheet
    With WB.Worksheets(1).Cells
        .Copy
        .PasteSpecial xlPasteValues
       
        'Entire worksheet will remain selected so select just first cell
        .Range("A1").Select
    End With
   
    FileName = "Name of the workbook.xls"
    'On Error Resume Next
    'Kill "C:\" & FileName
    'On Error GoTo 0
    WB.SaveAs FileName:="C:\" & FileName
   
    'Create and show the outlook mail item
    Set oApp = CreateObject("Outlook.Application")
    Set oMail = oApp.CreateItem(0)
    With oMail
    'Uncomment the line below to hard code a recipient
    '.To = "*** Email address is removed for privacy ***"
    'Uncomment the line below to hard code a subject
    '.Subject = "Look at my workbook!"
    .Attachments.Add WB.FullName
    .Display
    End With
   
    'Delete the temporary file
    WB.ChangeFileAccess Mode:=xlReadOnly
    'Kill WB.FullName
    'WB.Close SaveChanges:=False
   
    'Restore screen updating and release Outlook
    Application.ScreenUpdating = True
    Set oMail = Nothing
    Set oApp = Nothing
 End Sub

bernie deitrick, excel mvp 2000-2010 say:

If your data of Column A and B is on a sheet named "Data Sheet" and you enter A, B, C, etc. into cell A1 on the separate sheet, in cell B1, array enter the formula (enter using Ctrl-Shift-Enter)

=IF(COUNTIF('Data Sheet'!$A:$A,$A$1)>=ROWS($A$1:A1),INDEX('Data Sheet'!B:B,LARGE(('Data Sheet'!$A$1:$A$999=$A$1)*ROW('Data Sheet'!$A$1:$A$999),COUNTIF('Data Sheet'!$A:$A,$A$1)-(ROWS($A$1:A1)-1))),"")

Then copy down until you get blanks - or for as many rows as your most frequent letter.

If your key cell is not cell A1, change only the bolded instances of $A$1 to the actual cell address - leave the other A1 addresses alone.

jay freedman say:

When you open a document directly from an attachment without saving it first, Windows silently saves it into a temporary file. When you edit that document, the changes go into the temporary file. Word cannot use that file as an attachment. Even worse, once you close Word the file can be deleted -- see http://www.gmayor.com/outlook_attachments.htm.

Reopen the original email and SAVE the attachment as a local file. Make the changes there and save the document. Then you can send it back as an attachment.
