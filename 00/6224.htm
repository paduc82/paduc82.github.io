In order to email documents from Word, you must have a local email application (such as Outlook or Windows Live Mail) set as your default mail client. If you use only webmail (such as Gmail or Yahoo Mail accessed via a website, opposed to downloaded to your local mail program), you must use that mail service's file attachment function.
