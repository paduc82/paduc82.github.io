Windows Vista + SITECOM Bluetooth + Nokia 6230i = Connectivity

I did it :-)

- Uninstall the SITECOM devices (mark the checkbox Delete the driver software for the device.
- Right click in the Device Manager on the USB Bluetooth device.
- Click 'Browse my computer for driver software'
- Click 'Let me pick from a list of device drivers on my computer'
- Choose 'Bluetooth Radios' from the list and click 'Next'
- Choose 'Toshiba' as manufacturer and as device 'Toshiba Bluetooth Adapter'
- Click OK and let Windows install the drivers.
- Right click the new Bluetooth Tray Icon and click ‘Add a Bluetooth device’.
- Walkthrough the device wizard.
- Once the connection is established I was able to add the connection in the Nokia PC Suite!!

Notebook Dell Latitude D610

karl timmermans [outlook mvp 2012-2018] say:

Really simple answer (length of response is going to make it appear much more complicated then it is - tried to make no assumptions about what you do or don't know in terms of steps that need to completed aside from the creation of an actual Outlook profile itself)

-------------------------------------------
Part #1

For your items #1 thru 5 - just create 4 profiles each with one account. Make sure you can send/recieve emails from each of the accounts. No more - no less.
-------------------------------------------

Part #2

To have the ability to share a common contact list between the 4 profiles

a) in any of the 4 profiles above - create a new PST File
      Outlook menu --> File --> New --> Outlook Data File
    Give it a name that will be relevant to you (i.e. CommonContacts.pst) and remember where  
    you save that file.- you'll need it for the other 3 profiles

b) In the newly created PST file - create a new "Contact" folder - again given that folder a relevant name - something other than "Contacts" for ease of recognition. If you really want to refine this process - create one contact folder for each of your 4 businesses so that you can keep each company's contacts separate but yet still accessible from all profiles

In O'2007 - that folder should be set to appear in the AddressBook for that profile - right click on the folder --> Outlook Address Book --> ensure that "Show this folder as an e-mail address book is checked

In this profile - you will now have 2 contact folders included in your AddressBook - the default contact folder for the profile AND the one you just created.

Now in the AddressBook <for this profile> - the default contact folder for the profile is likely going to be the first contact folder that is listed when selecting contacts from the AddressBook. If you want the newly created contact folder to be the first one that is used for the selection of contacts - open the AddressBook --> Options --> change the contact folder order for the AB

That take cares of Profile 1 of 4

For each of the other 3 profiles

- Open Outlook using the applicable profile

- Add the newly created PST file above to this profile - from Outlook menu --> File --> open --> Outlook Data File

- Ensure that the contact folder is still designated to be shown as an Email Address Book (it should be).

- Rinse and repeat whatever chocies you made for the AddressBook in the first profile to be the same in this profile (depending on whether you opted to create one contact folder in the new PST file or 4 folders - if you opted to have one contact folder for each of your companies)

Once the above is done for all profiles - you are good to go and have the equivalent of what you have been referring to as a ":common addressbook" across all profiles. Just one caveat, make sure that you add new contacts to the contact folder(s) located in the separate PST file and NOT the default contact folder for the profile.

As for "dedicated folder sets" - nothing special about them. Each of your profiles represents a "dedicated folder set" - only refers to default folders for any infostore (i.e. infostore is a PST file in your case).

Just a friendly piece of advice (and I do mean that sincerely) - going forward, you will save yourself a great deal of angst getting an answer to any question/problem/issue by using the term(s) that accurately represent what you are referring to. In this particular case, AddressBook and Contact folder(s) are not synonmous terms and there is no such thing as a "common addressbook" so no one will know that the above is what you are referencing unless you are explicit about it.
