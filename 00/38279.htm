anoop p k say:

Hi,

 

By default, Windows Update automatic restart is enabled in Windows 7 and Vista.

Even though if you disable automatic restart for Windows Update, you will still be required to restart the computer to finish installing the Windows Update. However, you will be able to do this when you some updates cannot be configured while the computer is switched on, which is why it shuts down then restarts.

If you check the screen during this period you will see a message when updates are being configured, which can be both before and after the shutdown and restart.

You usually find that it is the security updates that need to undergo this routine.

You will also notice that you are requested to shutdown then restart your computer when installing or uninstalling certain software. This is so that your computer can be reconfigured also.

Updates only request reboots when System Files that were in use at the time of the install are affected - and for no other reason.
The number of updates requiring reboots has dropped significantly since the days of Win2K (which required reboots for most updates), and will depend on what else the system is doing at the time of the install - If the system has a number of open apps it's more likely to require a reboot than if it's just been started.

michael_440 say:

Hi

Finally came up with a fix for (this) folder shortcuts that won't open after doing a clean re-install and then a restore of Windows 10. I noticed that I could initially open folders and subfolders under Quick Access and This PC, however after only a day or two I couldn't open subfolders anymore.

My fix was (Summary - recreate where each folder points to):

  1. right-click on one of the folders under 'This PC' and select 'Properties'. In this case I selected the 'Music' folder but any folder should work
  2. click on the 'Location' tab
  3. pay attention to the current path to your folder, and write it down. You will need it later
  4. click the [Move...] button
  5. select any other folder on your local disk, preferably one which is empty so it won't matter if you make a mistake.
  6. Click the [Select Folder] button
  7. Click the [OK] button
  8. When asked whether you want to move all of the files from the old location to the new location select [No]

If you click on your folder now you will find it empty, which is to be expected. Now you need to point it back to the original location which is where all your files are. ie Repeat steps 1 to 8 but point the folder back to the original location. Here are the steps:

  1. right-click on the folder under 'This PC' you moved earlier and select 'Properties'. In this case I selected the 'Music' folder but any folder should work
  2. click on the 'Location' tab
  3. click the [Move...] button
  4. enter the original path to your folder you wrote down earlier or navigate to the original location
  5. When asked whether you want to move all of the files from the old location to the new location select [No]
  6. Check that you can open folders and subfolders. Other folder shortcuts should work now too.
