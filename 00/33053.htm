A local account can only be used on the computer that created it, or as your daughter might say, it lives on the computer. A cloud account can be used on any computer that can get on the internet because it lives in the cloud (which is just a fancy name for the internet.)

A Microsoft Account (MSA for short) is simply Microsoft's version of a cloud account, just like a Google Account is Google's version of a cloud account, and iCloud is Apple's version of ... you get the idea.

To use a cloud account you need an email address, because that's your username for the cloud account.
