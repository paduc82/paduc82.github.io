Can you explain why it is behaving that way?

philipradler say:

From what you're describing, Word is behaving normally. That first jump is from the end of your current line (the last character typed) to the next default tab stop. So assume you typed a line such as "A quick tab test" in 10-point Arial (but leave out the quotes).  That line is about 0.94 inch long. If you're at the end of that line and press the tab key, the first default tab (at 0.5 inch) will be ignored because you're already past it. The cursor will instead jump to the next tab stop at 1.0 inch, traveling a distance of just 0.06 inch (from 0.94 to 1.0). The next press of the tab key will take you to the next default stop at 1.5 inches, and on each subsequent press the cursor will travel that same 0.5-inch distance to the next default tab stop.

 

Of course, if you set any tab stops yourself (in a paragraph or a whole document), they will override the default stops. So if you set tab stops at 1.25 inches and at 3.00 inches, type that same sample line ("A quick tab test"), and press tab, the first "jump" will be to the 1.25 tab you just set (a distance of about 0.31 inches), and the next will be to the 3.00-inch tab (a distance of 1.75 inches). After that, since you haven't set any later stops, the default tabs will once again be honored at every 0.5-inch interval. 

 

Hope this helps.
