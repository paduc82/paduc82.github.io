This is using latest version of Office 365 for business and latest outlook application. I use the same O365 subscription on both my work laptop and iPad on which I would also to be able to access the offline data files when away from home.

    The PST files are less than 3GB in size each. I will never be in the position that both the home and work computers will be on and accessing the PST files at the same time.

    Am I able to use either OneDrive or DropoBox (for business) to host a cloud copy of the OST file that can then be accessed by both computers?

    I can see that a number of people have tried but there are caveats and some horror stories of corrupting the files.

    If possible what are the steps for best practice implementation please?

    karl timmermans (outlook mvp 2012-2018) say:
    >>
    Am I able to use either OneDrive or DropoBox (for business) to host a cloud copy of the OST file that can then be accessed by both computers?
    <<

    Having PST files on any kind of network drive (whether a local network or cloud location) is not supported and can definitely lead to data file corruption especially those connected to a POP3 email account.

    With that statement out of the way, Outlook will not prevent you from having your PST files on OneDrive. Given that these are PST files that are not associated with a n email account - meaning that there is no regular send/receive process going on that will cause constant sync'ing, you are less likely to incur data file issues if the file is not heavily used in terms of constantly writing to the file.

    So bottomline, is it recommended, no. Can it be done, yes. Essentially, if the purpose of maintaining these PST files on OneDrive is for "reference" purposes, you should be OK. Biggest word of advice is that you maintain a regular backup of any PST file that is open in Outlook. A backup in this case means making a copy of the file with Outlook closed.
