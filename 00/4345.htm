Symbol subset "Combining Diacritical Marks" - Microsoft Community

lrelliott say:

Symbol subset "Combining Diacritical Marks"

I cannot find the "Combining Diacritical Marks" in my subset list. I am using Word 2016, Office 365 Pro Plus. Anyone know how I can fix this please?

Thanks!

jay freedman say:

The first thing to look at is what is selected in the Font box in the upper left corner of the Insert > More Symbols dialog box.

  • If the Font box says (normal text), the dialog box shows the characters available in the font used by the Normal paragraph style.
  • If a specific font's name appears in the box, the dialog box shows the characters in that font.

Some fonts (such as Arial Unicode MS) contain a lot of combining diacritical marks, while others (such as Symbol) don't contain any.

After selecting a font, look at the box in the lower right, just above the Cancel button. If that dropdown's list contains "Unicode (hex)", select it. If that doesn't appear in the list, then the selected font doesn't contain any characters other than the 256 ASCII characters.

If you get as far as choosing to show Unicode character codes, then the Subset box in the upper right should allow you to select the Combining Diacritical Marks subset. These characters start with character code 0300 and continue for as many as the selected font contains.

hansv mvp say:

1) Make sure that you have selected "(normal text)" or a font such as Arial, Times New Roman or Calibri as font. Some fonts don't include "Combining Diacritical Marks".

2) Make sure that you have selected "Unicode (hex)" from the dropdown above the Insert and Cancel buttons. Special characters are not available if you select one of the ASCII options.

pdf to word doc

Please help!
charles kenyon say:
You may need to run OCR on the pdf file.
A pdf can be created from a Word document. Those can easily be opened and edited by Word.

It can be created from a web page. Those can easily be opened and edited by Word.

It can be opened from a scan or a picture taken by a phone camera. Those are pictures of words saved as pdfs. Just as you can have a picture of a car. You can see the car in the picture, but you can't change the timing of the engine in that picture. You can't change the order of text or otherwise edit it with a picture of text. Word can open such a file, but it can't edit it.

There is an intermediate kind of program called OCR. That is optical character recognition. If run on a picture of text, it gives text, more or less accurately (but seldom perfectly). Most scanners come with such software. It is built into Adobe Acrobat (but not Acrobat Reader). It is also in OneNote 2016.

Once translated into text, it can be edited in Word.

  2. Check each Slide Layout to make sure each one has a footer placeholder. For any Layout missing the Footer, click on Slide Master>Footers even if it's already checked. The Footer should reappear.
  3. Close Slide Master view, return to your slides and click on the Home>Reset button for each slide to make it re-link to the Layout.
  4. Now edit your Footer text and click on Apply to All.
If this procedure doesn't add the footer text, your presentation may have broken the inheritance between the master and layouts or layouts and slides. In this case, you should re-create the file.
