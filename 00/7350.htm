Using one cells data to display data from another cell

I am trying to display data in the next column when I type in data in the column before it.  Example:

I have multiple tabs in my document, each tab refers to a year and they all have columns in them that have consecutive numbers in them (ie: 14001,14002,14003).  And next to each one of these numbers I have text that describes that number (ie: 14001 in the column next to it would say Landscape Project 1, 14002 - Landscape project 2...)Now on the first tab in my document  I have a column where I enter 14001, then in the next tab I want it to look up the tab that has the 14001 and then I want it to display Landscape Project 1. 

I hope this is clear.  Please let me know if I need to elaborate. 

Thanks!

jeeped say:

Let's take a look at one of your formulas.

VLOOKUP(B21,'FLLG-11'!C18:E157,3,FALSE)

VLOOKUP(lookup_value, table_array, col_index_num, [range_lookup])

VLOOKUP() will always look for its lookup_value in the first column of the table_array and return the value from the corresponding column indicated by col_index_num. That si to say, the returned value will always be the same column or in a column to the right of the first column.

If you want to return information from a column to the left of the column you are looking up then you need to use an INDEX(MATCH()) pair. From your example above, if you wanted to return the value from column A after looking up a value in column C then it would be,

=INDEX('FLLG-11'!A18:A157, MATCH(B21,'FLLG-11'!C18:C157, 0))

That formula will nest withing a series of IFERROR(IFERROR(...)) functions the same way that your original VLOOKUP() did. You might want to 'lock' the lookup and return ranges down like the following,

=INDEX('FLLG-11'!$A$18:$A$157, MATCH(B21,'FLLG-11'!$C$18:$C$157, 0))

With the ranges locked in as absolute they will not change if you copy the formula down to subsequent rows.

INDEX function Uses an index to choose a value from a reference or array
MATCH function Looks up values in a reference or array
VLOOKUP function Looks in the first column of an array and moves across the row to return the value of a cell

hansv mvp say:

Select A2:A8 (or however far down as you want. I'll assume that A2 is the active cell within the selection.

On the Home tab of the ribbon, click Conditional Formatting > New Rule...

Select 'Use a formula to determine which cells to format'.

Enter the formula

=AND(A2<>"",A2<>"X",F2="")

Click Format...

Specify the desired formatting.

Click OK, then click OK again.

Please note: the formula is based on your description, where you mention "the cells in column F are blank".

In your sample data, Date Paid is column D instead of column F, so there the formula should be

=AND(A2<>"",A2<>"X",D2="")

teylyn say:

You talk about column F in your description, but in the data sample the Date Paid is in column D. Change the formula accordingly.
