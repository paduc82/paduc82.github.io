</script>

matthewapplebaum say:

I solved it!

I've accomplished this task by the following:

Create specific library views for each of the top level folders - start from an existing view but add filter Name = your folder's name.

Create a new page for each folder, add Web Part, choose the document library, then change the Selected View under List Views to the view you just created.

Below that Web Part, Insert another Web Part.  Media and Content > Script Editor, Edit Snippet, insert the code you've extracted from your Yammer group.

Hope this helps someone else!
