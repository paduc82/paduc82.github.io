The second point might be the root cause.

Since the items of archive mailbox permanent hold in there unless we disable the online archive feature, they are stored in different date center.

OK after working with Microsoft we did figure out that it was in fact Drop Box. It was causing ALL Office Programs to refresh every +/-5 seconds.

The way we resolved this was as follows:

  1. In the command prompt type "msconfig"
  2. Within "System Configuration" do the following:
    1. Under the Tab titled "General" select "Selective startup"
    2. Under the Tab titled "Services" on the bottom there is a check box that reads "Hide all Microsoft services"
    3. Under the Tab titled "Startup" unselect the startup item "Dropbox"
    4. Click Apply
    5. Click OK
  3. Restart your computer and see if that fixed it.
  4. If it did you need to turn off Drop Box from opening on start up.
  5. Once you have turned off Drop Box make sure you repeat the above and when you get to 2.1 above make sure you select "Normal" Start up then skip to 2.4 so your computer starts up properly from there on out.

Opening an HTML File in Word:

  1. Click the “File” tab on the ribbon. Click “Options” at the bottom of the drop-down menu.
  2. Click "Advanced" in the left pane. In the right pane, scroll down to the General area. Select the "Confirm File Format Conversion on Open" check box and then click "OK."
  3. Click the “File” tab and then click “Open.” Browse to the Web page file to open and double-click the file name.
  4. Click "Plain Text" from the Convert File dialog box and click "OK." Your Web file opens as a plain text file showing the HTML code. Once you open the file, you can edit it and save the file as an HTML file.
