jim gordon mvp say:

The ability to edit a web query is not provided in Excel's interface. But editing a web query isn't all that hard to do.

First, make a copy of the Excel workbook, just in case things go bad you won't wreck the only copy you have. Make sure neither the original nor the copy are open in Excel.

In Finder, change the file extension of the copy from .xlsx to .zip.

  1. Double-click the Zip file
  2. Open the resulting folder, and inside look for and open the xl folder
  3. Open connections.xml in an app that handles xml (such as TextEditor)

Here's what I see when I examine a web query for Yahoo 

<?xml version="1.0" encoding="UTF-8" standalone="yes"?>
<connections xmlns="http://schemas.openxmlformats.org/spreadsheetml/2006/main"><connection id="1" name="Connection" type="4" refreshedVersion="0" background="1" saveData="1"><webPr url="http://finance.yahoo.com/d/quotes.csv?s=ORCL&amp;s=CSCO&amp;s=COHR&amp;f=sl1" htmlTables="1" htmlFormat="all"/></connection></connections>

You can simply edit the URL and save the connections.xml file. Then back out of the folders and change the file of the the outermost folder back to .xlsx

Your edited query is now ready to go to work in Excel.

  • look through installed list of programs.
  • if you find Office 365 as well as your purchased Office 2016,
  • UNINSTALL Office 365
  • Office Repair remaining Office 2016 should fix problem.
