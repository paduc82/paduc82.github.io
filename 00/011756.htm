psu curt say:

After about 2 hrs in chat with Verizon, I finally got it fixed.  Try this:  

> Make sure your system is CLEAN - run a Registry Cleaner as well as up-to-date Anti-Virus & Malware Security.  I use Microsoft Security Essentials & Malware Bytes (free), but there are others on the market.  

> Re-Boot your system in Safe Mode with Network

> Turn-off your Firewall & Security protocols  

> Go into Verizon and change your Password

> Go into Outlook and delete your account

> Re-build the account with the new Password

     > Manually enter the Server Access Information - Auto-Detect of the Server Setting won't work

> On the Internet Email Settings screen fill in the information and select More Settings

> On the Advanced Tab

     Incoming POP3 Port is 995

     Check the Server Requires Encrypyted Connections (SSL)

     Outgoing SMTP port is 465

     Use the following type of Encrypted Conncection - pick SSL from the drop-down box

     Check Leave a copy of the Messages on the Server

> Click OK - Next - Finish 

> Close & re-start Outlook

> Go into Verizon Webmail & send yourself a test message to make sure it works

>  Re-boot your system in normal mode and restore the Firewall and Security  

 

     After much trial-&-Error, this is what finally worked for me.  If it's still doesn't fix your problem, get into a Chat with Verizon Suport.  The problem is that Outlook will not recognize or cannot find the Verizon Server settings.
