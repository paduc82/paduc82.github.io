ossiemac say:

Following code example for Setting the RETURN key to run the macro and re-setting the RETURN key to default behaviour. Note that I have used a populated and blank cell but you could be more specific as to which cell to select and even test for a specific value in it to set the OnKey. You could also maybe use the worksheet deactivate to cancel the OnKey.

Private Sub Worksheet_SelectionChange(ByVal Target As Range)

    If Not IsEmpty(Target) Then   'Select a cell that is NOT empty
        Application.OnKey "{RETURN}", "SearchBox"   'Return key runs the sub
    Else
        'Use Mouse to select an empty cell
        Application.OnKey "{RETURN}"    'Returns key to Default setting
    End If
   
End Sub

=TEXT(A1,"#.00")
