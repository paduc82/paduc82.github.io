Can you change the subscript preferences? - Microsoft Community

wasie say:

Can you change the subscript preferences?

Is there a way to change the preferences for the subscript? I would like to make it a little larger and down a little farther. I know that I could select a  larger type size which ends up making the number larger, but it is barely subscripted. I am writing molecular formulas. Thank you.
daniel g. grau say:

If I had to do what you need, I would do it this way:


A. Using AutoCorrect (not recommended for numbers, but for whole formulas)

I will explain how to do this with numbers, although the real advantage would be using it for complete expressions.

This is how it looks with super/subscripts in Arial 12 pt (zoomed at 150%):

So we start with regular characters:

Change the numbers’ size to 9 points (down from 12). In addition, select each number and Format > Font > Advanced > Position, then choose Lowered by 4 points and Raised by 5 points.

Add them as AutoCorrect entries:

  1. Select the newly formatted subscript “2” (or formatted expression in case of a formula)
  2. Go to Word > Preferences > AutoCorrect.
  3. Enable the checkmark at the top.
  4. Click on “Replace text as you type.”
  5. Click on “Formatted text.”
  6. In the “Replace” field type 2d\ for “2 down.” The backslash is just a safeguard to build a unique sequence. You can actually use any sequence.
  7. Repeat for the subscript with 2u\.

From now on, type that sequence to obtain the modified numbers. For instance, typing H2d\O will produce:

as compared to the "conventional" form:

 

The problem is, you should repeat the process for other super/subscript digits. Note that the format embedded into the sequence won't change, so in this case we are stuck with "Arial 9 points lowered 4 pt." (Likewise for a formula—the original format will always be followed.)

Also note that if you need to type a subscript "22," the AutoCorrect sequence 2d\ won't work twice in a row (unless you add a space, for instance). However, you can instead use \2d\ as the AutoCorrect sequence and that will work twice in a row, because the two backslashes are clearly identified as separators.

And this is the real timesaver: If you define an AutoCorrect entry for a complete formula, then you could type, for instance, hexa\ and get (C6 HCl3 OH)2 CH2, except that subscripts would be properly formatted.


B. Using character styles

  1. Select the newly formatted subscript “2”.
  2. Go to Format > Style 
  3. Click on New
  4. Select Character style in the dropdown menu next to "Style type"
  5. In the bottom dropdown menu, select Shortcut key.
  6. Type a key sequence. For instance, Control D (for "down")
  7. Repeat for a superscript number using Control U (for "up")

To use this feature, type Control D before typing the number (although it would also work on a selection), type the number, then restore with Control Spacebar or Command Shift Z (either of these will reset the character style to that of the current paragraph style).

patc---01 say:

Unfortunately, as Krizanand says it is possible.

 

For future reference, your best option for built in memory - eg: the 8GB on the N95 - would probably be to try and run one of the disk shredders over it.  These basically write over the entire disk with random 1s and 0s a few times to remove any vestige of a data image from there.

 

The phone memory - ironically, where most of your private data is likely to be - is a bit more difficult.  You're probably not going to be able to run a shredder on this - if you did it would probably kill the phone competely.

 

Your best bet is probably to delete all of your personal data (eg: do a full reset), then copy the biggest file you can fit to phone memory, followed by lots of small files until the memory is completely full.  Delete these files then repeat a few times.  This is essentially doing what a shredder does, but only in the areas of memory you can access.

 

Not 100% safe by any means, but better than nothing.

Create Shortcut on desktop for new meeting - Microsoft Community

a. user say:

Create Shortcut on desktop for new meeting

How do I create a Shortcut on the desktop for new meeting? 

What I really want is a hotkey to create a new meeting, so I thought I'd create a shortcut on the desktop to a new meeting and assign a shortcut key to it.  

diane poremsky mvp say:

If you are working in outlook, you can use Ctrl+Shift+Q.

For a desktop shortcut, you'd use the path to outlook, then the /c switch and message class.
outlook /c ipm.schedule.meeting.request  

(that switch is not opening as a meeting request though - using /c ipm.appointment and clicking the button to invite people will work)  

More information on switches:
https://www.slipstick.com/outlook/command-lines...
