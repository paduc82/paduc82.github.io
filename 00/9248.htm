hansv mvp say:

Try this:

Sub SetFooters()
    Dim oSection As Section
    Dim oFooter As HeaderFooter
    For Each oSection In ActiveDocument.Sections
        Set oFooter = oSection.Footers(wdHeaderFooterPrimary)
        Select Case oSection.Index
            Case 1
                oFooter.Range.Text = "This is the first section"
            Case 2
                oFooter.LinkToPrevious = False
                oFooter.Range.Text = "This is a later section"
            Case Else
                oFooter.LinkToPrevious = True
        End Select
    Next oSection
End Sub

The only thing that changes in that address is the "0" from "0" to "100".  So I thought I should be able to use an array formula with INDIRECT to iterate between 0 and 100, have INDIRECT return a reference to "AvgSales_xx!A:A" and COUNTA would just sum them all up.  But I get a #VALUE error.

I've confirmed that this single-step does in fact work:

=COUNTA(INDIRECT("AvgSales_" & 0 & "!A:A"))

{=SUM((COUNTA((INDIRECT("AvgSales_" & (ROW($A$1:$A$101)-1) & "!A:A")))-1))}

Hi,

 

Are you trying to delete the Internet Explorer browsing history when you close Internet Explorer 10?

 

If you are trying to delete the Internet Explorer browsing history on exit, then you may try the following steps.

a. Open Internet Explorer 10.
b. On the Tools menu, click Internet Options. If you can't see the Tools menu, press Alt to display the menus.
c. Click the General tab.
d. Under Browsing History, click Delete.

Note: You may also check the option for Deleted Browsing History on exit.
e. In the Delete Browsing History dialog box, click to select the following check boxes, and then click Delete:
Temporary Internet Files
Cookies
History

f. Click Apply and ok.

Internet Explorer 10 privacy statement for Windows 7
