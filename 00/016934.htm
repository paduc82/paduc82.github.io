    Dim sh As Shape
    For Each sh In ActiveDocument.Shapes
        If sh.Type = msoPicture Then
            sh.PictureFormat.ColorType = msoPictureGrayscale
        End If
    Next

To find this, in the VBA editor click View > Object Browser (or press F2). Type  gray  into the search box and hit Enter, then look through the list of items in the Member column for a likely value... which is msoPictureGrayscale. bob jones aka: cybertaz say:

Nobody can answer the question abut Microsoft's "thinking" - We often wonder, ourselves :-)

For some degree of insight, though, the concept of Themes is to provide consistency of colors & fonts used... not just within a document, but in any Office document regardless of the Office program used to create it... IOW, all Office Themes are universally available in Word, Excel, PowerPoint, Outlook, etc.

There are only 12 main colors in each Theme but each offers 6 variants for a total of 72 shades. Each main colors & their variants in a Theme are associated with a particular element (SmartArt Objects, Borders & Shading, Styles, Fonts, Lines, Fills, etc.) that can be included in a document. There are 12 such elements which are identified in the Create Theme Colors dialog which in Mac Office can be accessed only in PowerPoint (where you can create your own sets of Theme Colors). The Preview in the dialog gives an indication of which elements are affected by which colors;

However, you are not restricted to using only the colors provided by the current Theme. You can select any of the 10 Standard Colors or use the More Colors dialog available in any of the color dropdowns to select or create virtually any color you want... There are several different Color Pickers available.

Theme Colors [along with their variants] applied in a document change when a different Theme or set of Theme Colors is selected. The change is based on the Theme Color in the newly applied Theme that is associated with each of the 12 elements. Further, all components to which the original Theme Color is applied will update as well. To better see what I mean, create a SmartArt Object or apply a colored Heading Style in a new blank document. Note the coloring of it, then just select a different Theme or different set of Theme Colors & observe the effect on the object.

The difference is that if you choose a color that isn't used in a Theme it "breaks the link" between the Theme & the component to which the color is applied. As a result, those components will no longer update when a different Theme is selected.

Unlike the very "old" DOS days when there was a signature file on the floppy, there is nothing unique on the CD to identify it. The uniqueness is provided via the Product key you use to activate it. So, you can borrow a CD without threatening their installation/activation.
  2. Ask at work if they have CD you can borrow
  3. Try to find a local "PC Computer Club". There is almost certainly a geek or three with disks you could borrow.
  4. Find a small local computer store or repair business. They often have old versions of software because they are still supporting customers using them.
  5. You could buy it online. Be very careful who you give your money to
  6. Try websites like Kijiji or Craigs List to see if you can borrow or buy it
  7. Try getting one from MS (if it works, please post back here, there are many interested people who would like to know that MS is still providing that kind of support) http://support.microsoft.com/kb/326246/ 

Once you get the installation file, make sure you burn a copy for yourself before returning the disk if it is borrowed.

Hardware Compatibility Testing - Microsoft Community

bilo cape town say:

Hardware Compatibility Testing

Follow this procedure to check the compatibility of any PC's hardware with Windows 10 Preview.

If you have already succeeded in installing the system on one PC- the hard drive can be removed from that one - and fitted to any other PC for testing.

Testing can be done with IDE or SATA Hard Drives - but it is recommended to use SATA3 6Gb/s to take advantage of the best performance and backward compatibility. 

Since the first preview of Windows 8 arrived on the scene - I have been using this method of checking the compatibility of other hardware configurations.

Windows 8 / 8.1 / 10 all have the ability to reconfigure their settings according to the new hardware found.

Initially it may install generic drivers for some components. But if the PC is connected on-line - it will even go and get the correct drivers and install them automatically.

NB: Never use this method to Dual Boot a pc - as Windows 8 and above have the ability to take control of everything connected on a PC.

After disconnection of the test system - ALL other systems and data on the other systems may get wiped out.

There is a way to prevent this - using Easy BCD - but is not worth the extra effort

A 2.5 inch Laptop Hard Drive is recommended to be used for the initial installation. Then it can be connected to any desktop / laptop for testing compatibility of that systems hardware.

Considering that laptop hard drives generally run at 5400 rpm - a Solid State Hard Drive will provide the best performance and speed up the process.

However, I am using Seagate Hybrid SSD Hard Drives with 500GB capacity - which also perform very well - and are far more cost effective. They also provide storage space for utilities and diagnostics which might be needed when checking out each system. But a USB Flash or External Drive can also be used for storage. 

(The Hybrid drives are available in sizes ranging from 500GB to 2TB already - but the 500GB size is big enough and very affordable. )

Some of the PC's I have successfully tested as compatible - have only 2GB of RAM running the 64 Bit OS.

My testing was also done on a 10 year old dual core Fujitsu Laptop and even a P4 running the 32 Bit OS.

Also see my post on Upgrade Win8.1 to Win10 which I have tested.
