Outlook notes section states in brackets (this computer only) 

 Means that one of the following applies

#1 - Your Outlook.com account has not been migrated to the Office 365 backend servers and is configured using the Exchange ActiveSync protocol (which if the case, you can't sync anything from a folder labeled with "This Computer Only"

To determine if your Outlook.com has been migrated - log into it via your browser and if it shows <Outlook.com> - it hasn't been migrated (otherwise it will show <Outlook Mail> or something similar depending on what region your in)

#2 - If the account has been migrated, you need to re-add the account to Outlook which will then configure the account using the <Exchange> protocol. 

Note: don't delete the original account UNTIL you have copied all the items from the <Notes (This Computer Only)> folder to the correspnding folder in the newly configured account, otherwise you're going to lose all that info

Now I want to create another contact folder from scratch. Not so intuitive.. I went to "People" tab  and under my contacts opposite clicked and asked to create a new folder. Which it did, but not where it is a useful contact folder.

When you create a new folder, Outlook needs to know what type of folder it is to be.  Right-click the root of the data file where you want the folder to reside and choose "New Folder" from the contact menu.  Enter the name you want the folder to have in the "Name" field, and change the "Folder contains" drop-down to "Contact Items".  Click OK.  It is the select of the folder contents type that I believe you neglected, since new folders default to mail folders.  What view of the Navigation Pane you have chosen (you said you clicked "People") isn't relevant when creating the folder this way.

 

However, if you were to click the Folder tab and then click the "New Folder" button, the default "Folder contains" value should already be "Contact Items" and your folder should work as you expect.
