How to upgrade Windows 7 Home Premium with a volume license?

We have been using all win7 home premium 64bit computers for several months in a new startup conpany. We have the need now to create a domain and join these computers but we do not want individuals to lose any of the data they have on these computers - just move their document folders to the server. Problem is we have purchased a number of volume licenses only to now find out we cannot use them to upgrade Home Premium. My question is:  how do you upgrade windows 7 Home Premium with a volume license, and which edition should we choose?

andre da costa say:
Windows 7 volume Licenses are upgrade licenses, which means, the computers must be running a qualifying license such as Windows 7 Professional or Ultimate. Consumer editions of Windows 7 such as Starter and Home Premium do not qualify for volume license upgrades.

Your option to remain compliant with the license is to upgrade the Home Premium licenses to Windows 7 Professional, then you will qualify for the Volume license upgrade. If you don't have a qualifying license installed, then you will need to get a Genuine Windows License Agreement from Microsoft. An audit might be required.

Even when you get the Genuine Windows License, you will still have to do a custom install, because there is no upgrade path from Home Premium to Windows 7 Volume License Upgrade. The best work around is to upgrade licenses to Windows 7 Professional using Anytime Upgrade, then upgrade to Windows 7 Volume License upgrade.

From Microsoft

Sometimes companies will use volume licensing to install Professional edition software on more PCs than are allowed.  For instance, Windows volume licenses require an existing qualifying operating system pre-installed as a pre-requisite on each PC. In most cases, this means a Professional-level operating system is required. Companies often acquire PCs with Windows Starter or Home editions pre-installed and mistakenly then apply a new Windows volume license for Windows 7 Professional.
 
If you think this might be happening within your business or organization, Microsoft and our trusted partners offer easy, economical ways to help ensure that the software installed on your PCs is properly licensed.
 
Learn more about: qualifying operating systems

Source http://www.microsoft.com/en-us/howtotell/Licensing.aspx
