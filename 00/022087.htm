macropod say:
When you create a link from Word to Excel, the LINK field path is always absolute. As such, you can move the Word file but not the Excel workbook. If you do move (or rename) the Excel workbook, you will need to edit the link(s) to point to the new filename/location. You can do that via File>Edit Links or by pressing Alt-F9 to expose the LINK field codes and editing the links. If there are multiple links to the same file, this could be done via Find/Replace. Do note the need for \\ as the path separators.

jebklb say:

dianeoforegon say: There is no option. Originally the option was there but it did not work so it was removed.

What type of accounts do you have? (POP, IMAP, Exchange)

For your IMAP and Exchange accounts, you can just setup accounts in Outlook and your mail will download from the server. Contacts and events can be synced to Outlook via Sync Services.
Apple Mail exports a non standard .mbox file so you would need a third party app to convert to a format that Outlook can read or you can create a free IMAP account (iCloud, Gmail) to upload your messages from Mail to the server then enter account in Outlook. You can then move the messages from the IMAP server to Outlook's "On My Computer" folders and delete the IMAP account if you no longer wish to keep it.

Don't move too many messages at a time. Verify by comparing number of messages in folder to messages on the server. I would start with a smaller number in the hundreds and see how long it takes. Verify the number moved was actually uploaded and is now on the server before starting the next batch.

Note: some ISPs will not allow too many messages moved in a day.

EmailChemy offers a conversion tool for mbox files.
http://www.weirdkid.com/products/emailchemy/

Note: If you do want to use iCloud to sync messages in Outlook, I suggest your turn off contacts and calendars in System Preferences > iCloud and/or don't check Outlook's Sync Services to sync to these items.

Hi Anurag,

 

Welcome to Microsoft community forum. Let me assist you with your concern.

 

The simple way to hide and unhide Disk Drive Partitions on Windows 10 Technical Preview is by following this steps:

 

To hide a partition in Windows

  • Press Windows key + X, Click on Disk Management
  • All your hard disk and its partitions will be show in the right hand side
  • Right-Click on the partition that you want to hide and select "Change Drive Letters and Path"
  • Click on "Remove" and click "Yes"
  • Your drive will now be hidden in my computer

To unhide the drive:

  • Go to Disk Management Right-Click on the hidden partition [there will not be a drive letter on the hidden drive] again select "Change Drive Letters and Path"
  • Click on add and select an appropriate drive letter.
  • Click Ok
  • Now the drive is un-hided.
