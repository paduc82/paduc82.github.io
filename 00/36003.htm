Most ntdll.dll errors result from a corrupt or damaged version of the ntdll.dll file itself, corrupt hardware drivers, or issues between Windows and other programs.
If the software program has any updates or service packs available, install them too. The software's programmers may have identified an issue with the program that caused the ntdll.dll error and then issued a patch for it.

Note: Third party software programs that have been installed on your computer are almost always the cause of ntdll.dll errors.

You can reset WMP to default settings at Control panel, Troubleshooting, Programs, WMP settings [Reset WMP to default settings]

volunteer j say:

What is the format (extension) of the attached file?

If you select (highlight) the email message and go to...
File / Save Attachments...you should be able to copy
the file to your desktop.

It sounds like the file you received could be a .mswmm
Movie Maker Project file instead of a Video file. (that would
explain why it is looking for Movie Maker).

Ask the sender to Publish (save) the file in the .WMV
movie format and resend it.

Just a little info on .MSWMM files:

MSWMM is not a video file...it is a Movie Maker project
file and is only useful for re-editing in Movie Maker.

A Movie Maker .mswmm project file does not include the
source files...(Photos, Video Clips, Music files)...it just
records the location they are saved in on your hard drive.
Basically it's just a recipe for a movie.

Hi,

This issue might be due to a corrupted Windows Store cache. To resolve this, we suggest following the methods below, one after the other, and check if it helps:

Method 1: Check and then install for updates. Click here for information on how to update Windows 10.

Method 2: Clear and reset the Windows Store cache.

a. Open Run (Win+R).

b. Type WSReset.exe, and then click OK.

Method 3: Re-register the Windows Store app.

a. Search for Windows PowerShell, and then run it as administrator.

b. In the Windows PowerShell window, type the following command and then press Enter:

Get-AppXPackage | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register "$($_.InstallLocation)\AppXManifest.xml"}

c. Restart your PC.
