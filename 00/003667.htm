steve rindsberg say:

This has become wildly complicated in Office 2007 and anon.

Rightclick the chart and choose Linked Excel Object | Open

That'll open the linked file in Excel.

Copy the chart from PPT into the open worksheet.  That gives you a chart on the worksheet with data linked to the same range as the original chart in PPT.

Rightclick the chart and choose Move; move it to a new chart sheet.  Make sure that the chart sheet is the active view.

Save the file; remember where you put it.

Now back in PPT, choose Insert | Object.  Choose Create from file, choose the file you just saved.  Don't choose Link.

Now you have an EMBEDDED chart plus the data needed to create it.

2 - Use 'Boot Camp' on your MBP to create another boot partition that will let you choose whether you boot into OSX, or Windows. If you choose to boot into Windows you can then use Office 365 with its familiar interface. This is how I have my MBP set up. It lets me run OSX, while giving me access to Windows and the applications that I can't run on OSX.

While setting up Boot Camp is pretty straightforward, you will need a copy of a Windows CD for the installation (you may be able to use a Windows image on a USB drive, but I've not tried it). This may be a deal-breaker for you if you don't have a Windows CD and licence key. But if you are new to OSX then having the option of booting to either operating system is a good way to learn the Apple/OSX approach to things, while also having Windows to fall back on when you need it.

With so many people switching to the Mac, having the ability to turn that toolbar on might be a desirable one.
