jllatham say:

I presume this is for conditional formatting of a cell?

If it is NOT, then this formula would do the trick:

=IF(AND(A1>=350,A2>0),"Gold",IF(AND(A1>250,A2>0),"Silver","Neither"))

If you want to use it for conditional formatting, it has to be split into the two AND() pieces.  Select the cell and choose Conditional Formatting, and choose new formula and then "Use formula..." option.  Enter this formula

=AND(A1>250,A2>0)

and set up the silver formatting

Then add another condition, again using a formula, but this time it would be

=AND(A1>=350,A2>0)

and set up the gold formatting.

charleskenyon (ia) say:
Hello Annie,

The information is from personal experience. I did not find it. I've been using Word for more than 20 years. Although I really do not care for Reading Mode, I have used it. It is a relatively new feature, coming with Word 2013. The idea is to make the Word document more like a web page and stop worrying about what the printed page looks like.

Workarounds:
You can insert the text as a link instead o as an object and it will show up in Reading Mode. You can also switch to Print view and open the object and then switch back to Reading Mode.

I presume that Word sees opening the object as being a type of editing.

Since Reading Mode can allow comments, I don't know why they could not allow opening of objects. You can certainly request this through feedback or UserVoice.

Archive all text messages at once? - Microsoft Community

riggo44 say:

Archive all text messages at once?

Is there a way to do this without going through each page one at a time for 4k messages?
richie weldon - msft say:

Hi MrMees -

Marco did answer your question, but perhaps he wasn't clear enough in this steps on what needs to happen, so let me give it a try!

When you load up the messages page on the My Phone portal, you will notice there are check boxes next to each of the 15 messages you can view on the screen.  The line above the first message listed will also have a check box.  If you select that check box, you will notice a new link will appear that says "Select all text messages (Inbox, Drafts, Sent). ".  This appears above the check box that you just selected.  If you click on that link you will then be able to select ALL of you messages at once and opt to move them into the archive.

Warm regards,

Richie Weldon

marco - msft say:

Hi, riggo44.

Yes, there is. In order to archive all messages at once, please do the following:

1. Go to the Messages page.

2. Click on the "Select All" icon on the message table header. This will select all messages in the page that you are on.

3. Once you select all messages in a page, you'll see just above the message table the following message: "All text messages on this page are selected. 

4. Click on "Archive".
