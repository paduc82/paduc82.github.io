The compass is a vector magnetometer. It measures magnetic field direction.. Think of this problem..

You're blind, in a field on a clear day with a directional thermometer..

 

You want to know where the sun is..

 

You point the thermometer up and take a reading.. You'd need to move the meter around to find the highest temperature.. that would be when you're pointing directly at the sun..

 

If you repeat the experiment the next day (when it might be cloudier) you'd need to "recalibrate" maximum since there might be a bit more cloud

 

Same with the compass.. the magnetic field strength changes depending on where you are.. and what you're near..
