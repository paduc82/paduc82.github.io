4.    Make sure DHCP server is running in network and reachable. If you know IP address of dhcp server, ping it from some other machine in network. Check other machines on same network are getting ip address from DHCP server.

8.    Is it issue with your router? Try resetting your router and see if it solves your issue.

http://support.microsoft.com/kb/928233

http://support.microsoft.com/kb/933340

a. Open Internet Explorer.

c. Click on the Connections tab.
d. Click on the LAN Settings button.
e. Then uncheck the option Use a proxy server for your LAN.
f. Click Apply.

d. In Start-up type, select Enable.

e. Click Start under Service status.

f. Click OK.

g. Repeat the steps c to f for the following services also:

Background Intelligent Transfer Service & Cryptographic Service.
