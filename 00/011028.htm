Function GetFileCreationDate(FileName As String) As String

Is there any way of extending the length of time it rings or is it something he is stuck with??!!

 

Many thanks for reading and hope someone can help :)

grschinon say: Menu > Settings > Calling > Call divert > Voice calls > If not answered > Activate > To voice mailbox

Then select the duration, from 5 seconds to 30 in 5 second increments.

Note that this is in fact a network function, not a phone function. It's the network that decides to stop your phone ringing after so long and then diverts the call to voicemail (which is also a network function - voicemail messages are not stored in the phone but on the network operator's systems).

The action above sends a command to the network saying that it should wait so many seconds before diverting the call to the voice mailbox. Some networks' systems ignore this command. If your husband's does then you or he will have to call the network asking if it is possible to extend the ring time.
