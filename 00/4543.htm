Ok, the MATCH function lookup_array MUST be a 1 dimensional array (a single row or column).

I guess that complicates things even more!

So, you'd have to wrap the INDRECT inside an INDEX function and tell it what column of the named range to use.

MATCH($B2,INDEX(INDIRECT(VLOOKUP($M$3,$T$2:$V$27,2)),0,1),0)

In the above we're using column 1 as the lookup_array in the MATCH function from the named range being returned by VLOOKUP.
