john_barnett say:

Hi n4ctf,

Take a look at this link http://support.microsoft.com/kb/314481

To remove an entry will require editing the registry. This link refers to Windows XP but i've checked it with Windows 7 and it works so it should work with Vista.

Once you have access to HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall  you will see a whole list of numbers. You will need to go through each of these numbered editions to find the program you are trying to remove. Click on a number in the left hand column, for example  {08D605B4-DCD1-451F-ABD7-52E6BB868E4E} and you will then see it's details in the right hand column of the registry editor. Look for the 'display name' to check to see what program the number refers to.

I would normally recommend that you try to re-install the application rather than tinker with the registry, but as you have already tried without success to re-install, editing the registry might be the last option. Incidentally, before making any changes to the registry, make sure you backup the registry first. This link from my website will show you how to back up the registry:http://vistasupport.mvps.org/backup_the_registry.htm
