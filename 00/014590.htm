aljon_t say:

Hi

Question 1: When I am signed into Outlook, am I also signed into Skype?

Answer: At Outlook.com, there is an integrated Skype that you can activate within the website. This will make your Skype account automatically sign in once you logged in to your Outlook account.

Question 2: Is the past Skype password moot today?

Answer: Since you are using a Microsoft account (Outlook) as log in to Skype, you need to use your current Microsoft account's password.

Question 3: I don't like the name that I have used with Skype.  I desire a name that is neither that nor my email address.

Answer: With Skype accounts generated using an email address, the Skype username will have the prefix "live:" plus the word comes before "@". This Skype name is not possible to be changed or modified. On the contrary, what you can edit or customize is your Skype display name.

Question 4: Can I activate my past Skype contacts?

Answer: Your old Skype contacts will not automatically populate since it is a new Skype account. If you can still access the old Skype account, then it is possible to transfer them to the new Skype account. If not, you need to manually add the contacts using the current account.
