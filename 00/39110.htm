bossdweeb say: Shekhar S shows one of the options, but there are two ways to setup front panel headphones -

(The older versions of Realtek HD audio drivers may have a slightly different format, but the
basics should be the same).

1) In Realtek sound manager go to Device advanced settings  in the top right corner of the window.
Tick the circle next to  Mute the rear... plugged in.
When done this was you also need to click on the yellow folder icon below the
Device advanced settings and verify that 'Disable front panel jack protection'  is clear.
-A disadvantage to this method is you don't get a separate settings option for the headphones,
so they use the same EQ and Environment settings as your speakers, but you can use both
speakers and headphones if desired.
- Note that with this option the headphones do not show in Window Sound Manager.

2) In Realtek sound manager go to Device advanced settings.
Tick the box next to Make front... simultaneously.
This will disable the speakers when the headphones are plugged into the front panel jack,
and revert to speakers when the headphones are unplugged.
Also tick one or more of the boxes below, depending on your need.

-With this setting there will be an option for the headphones in both Windows Sound manager
and Realtek HD Audio Manager.
- With this setting it does not matter which option you use in the yellow folder icon as the setting
is 'either/or' . You cannot run the speakers and headphones simultaneously.

- Note that with games the headphone needs to be plugged before the game is launched, or
the game will default to the speakers.
