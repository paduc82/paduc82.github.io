Deleting is always risky - it's forever, there is no "undelete" button; and the data might have oddities you haven't thought of. What if there are THREE records, one with +200 and the other two with -200? Delete one pair? delete all three? What if there are two pairs? Do you have any way to UNIQUELY identify one record?  

My first guess was to use a Self Join query joining two instances of the table by Document Number and putting a criterion on the Amount field:

SELECT A.[Document #], A.Amount, A.[Line#], A.[Posting Date], B.[Line #], B.[Posting Date]

FROM [Table 1] AS A INNER JOIN [Table 1] AS B

ON A.[Document #] = B.[Document #]

WHERE A.Amount = - B.Amount;

This will find all the "duplicates" (which aren't duplicates, 620 is not equal to -620); but since you don't have (that I can see) a Primary Key that would uniquely identify a record, I don't think you'll be able to change this to a Delete query.

DELETE *
FROM [Table 1] AS T1
WHERE EXISTS
    (SELECT [Document #]
     FROM [Table 1] AS T2
     WHERE T2.[Document #] = T1.[Document #]
     GROUP BY [Document #]
     HAVING SUM(Amount) = 0);

Your Microsoft ID is the address you use to sign into Outlook.com and, typically, Windows 8 or 10. If you log into Windows 8 or 10 with your Microsoft ID, the built-in Mail app will connect to Outlook.com and show you the mail in that mailbox and the People app and Calendar app will connect to that mailbox as well and show you your contacts and appointments.

You don't need to connect Windows 8 to a Microsoft ID, but you don't get access to the Windows Store. I haven't tried it yet, but I understand it's more difficult in Windows 10 to keep it disconnected from the Microsoft ID.

Please consider the following troubleshooting steps:

  1. Make sure that Skype is no longer running.
  2. Press the Windows  and R keys on your keyboard at the same time.
  3. Type "%appdata%\skype" into the Run window and click OK.
  4. Find and delete the shared.xml file. If file extensions are not displayed on your computer, the file will just be called shared.
  5. Open the folder that has your Skype name.
  6. Locate and delete the config or config.xml file.
  7. Uninstall Skype.
  8. Restart your Windows computer.
  9. Download the most up-to-date version.
