Hello John,

I don't know if it is really a bug or just badly worded help because the code does not return the positions of multiple shapes in a range.

However, all is not lost because the following workaround should do the trick to reposition all of the shapes in the ShapeRange.

Sub RePositionShapes()
    Dim shpRange As ShapeRange
    Dim lngLeftAdj As Long
    Dim lngTopAdj As Long
    Dim i As Long
   
    Set shpRange = SldOrig.Shapes.Range(Array(1, 2, 4))
   
    lngLeftAdj = 50     'Can be positive or negative adjustment for the position
    lngTopAdj = -50
   
    For i = 1 To shpRange.Count
        With shpRange.Item(i)
            .Left = .Left + lngLeftAdj
            .Top = .Top + lngTopAdj
        End With
    Next i

End Sub
