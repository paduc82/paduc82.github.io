Why are you trying to update the driver in the first place? 

Usually, drivers are best left alone unless you need a specific feature added by a new driver version or you have a specific problem addressed by a new driver version.

Where did you get the file that you are trying to use to update the driver?

Generally, you can safely ignore messages about unsigned drivers or inability to verify the publisher and "install anyway" IF you obtained the driver file from a trusted source, such as the manufacturer of the hardware in question. A web site for "free driver updates" is NOT a trusted source.

If you want to update the driver because some "free driver updater" utility told you to do so, I suggest uninstalling that utility and forget about updating the driver.

davem121 say:

This is a common enough occurrence, here is how to put your My Documents back into a folder:

  1. Open Windows File Explorer, create a folder called My Documents on your D Drive
  2. Put whatever files and folder should be in your My Documents folder into that folder
  3. Click on Quick Access on the left hand side of File Explorer
  4. On the right hand side right click the My Documents Library Folder and select Properties
  5. Select the Location Tab and Type D:\My Documents as the location
  6. When asked if you want to Merge Contents - select No

I hope this Helps!
