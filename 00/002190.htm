Word 2013 and later can open a PDF file and try to convert it to an editable document. Results may vary, depending on how the PDF was created.

Word 2010 doesn't have this feature built in. You can insert a PDF file in a document, but it will be inserted as a picture, displaying only the first page. To convert a PDF file to an editable document, you need third-party software. Some PDF readers let you copy the text from a PDF file, ready to be pasted into Word, and OCR programs can 'scan' the text in a PDF file and convert it to an editable document.
