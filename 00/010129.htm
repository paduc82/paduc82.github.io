I'm sure the solution is nesting the formula in IFERROR or COUNTIF something similar, tried many recommendations but failing to return correct values.

=STDEV(A1:A10)/SQRT(COUNT(A1:A10))

Let's say I have room for 10 experiments, did 5 that returned the numerical values, and 5 returned #DIV/0!. Trying to get correct SEM of those five values while ignoring the error of the remaining 5 cells.

mike h.. say:

Hi,

This an ARRAY formula, see below for how to enter it.

=STDEV(IF(ISNUMBER(A1:A10),A1:A10))/SQRT(COUNT(A1:A10))

This is an array formula which must be entered by pressing CTRL+Shift+Enter
and not just Enter. If you do it correctly then Excel will put curly brackets
around the formula {}. You can't type these yourself. If you edit the formula
you must enter it again with CTRL+Shift+Enter.

For example, if you download a program and it also installs another program, the program that you did not explicitly install is considered a PUP, even if it is a completely legitimate program, like, for example, Google Chrome.
