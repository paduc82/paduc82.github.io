jay freedman say:
To clarify, my old post was in response to someone whose document wouldn't stop continuously repaginating, which took up so much CPU time that editing was impossible. That situation indicates document corruption.

If you see a message about repagination only briefly and occasionally, that's normal. Every time you make any kind of change in the document, Word recalculates the line endings and page breaks to keep up with it. It is possible to turn off repagination only when you're using Normal view or Outline view, by going to Tools > Options > General and clearing the check box for "Background repagination". As soon as you switch to any other view, that check box will automatically be checked and grayed out.
