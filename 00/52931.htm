danieledm say:

Eventually I figured out what was the problem.

In order to force a workstation to have a program in status Enabled in the Startup, is necessary to properly set the following Registry Key:

HKCU:\Software\Microsoft\Windows\CurrentVersion\Explorer\StartupApproved\Run

with a value correspondant to the name of the application to the value ([byte[]] 0x02,0x00,0x00,0x00,0x00,0x00,0x00,0x00,0x00,0x00,0x00,0x00).

For instance for OneDrive I entered the following powershell command:

Set-ItemProperty HKCU:\Software\Microsoft\Windows\CurrentVersion\Explorer\StartupApproved\Run -Name "OneDrive" -Value ([byte[]](0x02,0x00,0x00,0x00,0x00,0x00,0x00,0x00,0x00,0x00,0x00,0x00))

HTH,

Issues Connecting to Xbox Live With AT&T - Microsoft Community

savageoncabbage say:

Issues Connecting to Xbox Live With AT&T

Hello Everyone!

Recently, there have been a lot of people having issues connecting to Xbox Live with AT&T as a provider.

If you are having an issue connecting your console, please follow these instructions that may resolve the issue:

Go to Settings < Network < Network Settings.

Choose Advanced Settings < DNS Settings < Then Manual.

For Primary IPv4 DNS please enter: 1.1.1.1

For Secondary IPv4 DNS please enter: 1.0.0.1

Once that happens, hold the power button on the Xbox console for 10 seconds and power it back on.

This should connect you back to the WIFI and resolve the issue.

Best,

Savageoncabbage
