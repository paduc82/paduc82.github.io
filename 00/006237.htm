charles kenyon say:

The following displays the character count.

Sub NumberCharactersStatusBar() ' less paragraph marks
    Dim strResult As String
    Dim strParaNumbers As String
    strResult = ActiveDocument.Characters.Count - ActiveDocument.Paragraphs.Count
    StatusBar = "Characters = " & strResult
End Sub

You could attach it to a keyboard shortcut or to the QAT. It supplants the status bar information, it does not add it to the status bar. Spaces and carriage returns count as characters. (So do tabs and manual line or page breaks.) This macro discounts the paragraph marks.
