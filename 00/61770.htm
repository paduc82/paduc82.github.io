Easy!

Highlight all of your images that you would otherwise rename. Decide on an overall name (Say "Image" or something) and click the rename button or right-click and click rename. When you are prompted to name the files, type in the name that you decided upon (In this case, "Image") and name the first JPEG "Image" (make sure while all this is happening, you have all the files you want to sort selected). When you rename that one file, it renames the others like this:

Image (1)

Image (2)

Image (3)

...

This will store the files sequentially making them easier to access, without having to rename them all individually. Other options are to place the images in a folder (Keep those files organized!), or on another drive (Such as a USB thumb drive)

Hi,

There are two types of command prompts. These are a standard User one and an elevated one, the latter of which runs with Administrator priivileges.

You can tell them apart by the prompts. A standard one looks like this:

C:\Users\Username>

An elevated command prompt looks like this:

C:\Windows\System32>

Both types can be found by right clicking the Start button which opens the power users menu. The one I've arrowed in the screenshot is the elevated type.

A code 87 is generated when a command requiring Administrator privileges is run from a standard Users command prompt.
