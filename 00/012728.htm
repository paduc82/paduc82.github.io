diane poremsky mvp say:
>>
It looks like they all end up unorganized in an "archive" folder. That sounds a lot more like a trans bin than an archive to me.
>>
You're close. The Archive folder is a replacement for the Trash bin. For years, many people would hit Del key to quickly file mail they wanted to keep... yep, filing it in the Deleted Items folder.

Microsoft added Quick steps to address this, but Del key was easier to use. So they added the Archive folder and the Backspace key for a one-click option to clear mail from the Inbox without deleting it. It's especially handy on a smartphone or mobile device - swipe to archive rather than trying to use the move to folder command.


While its unorganized, its also mail you need to keep but are done with. Instant Search or Search folders can find the mail if you need it.

Hi Huba0018,

Usually, the Undo feature acts on the way text was deleted i.e. if you select an entire paragraph and then you delete it then Undo requires only one action to undo the delete.

If, however, you have deleted the characters one by one (e.g. using the delete key repeatedly or holding down the delete key) then Undo will take each deletion as a separate action, thus requiring the same number of key strokes to undo the delete.

Could this, perhaps, be the case for you?

ADD-INS PROBLEMS
Start in Safe Mode (Hold CTRL key and click on application icon and don't release the CTRL key till it asks you for Safe mode confirmation).

Then start application normally and disable add-ins one by one and start application again every time you disable an add-in to determine the culprit add-in.

To disable add-in > File > Options > Add-ins > (scroll down below up to Manage: Com Add-in) > Press Go > Untick the Add-ins to disable and tick to enable them

OFFICE REPAIR
First try a quick repair install, If it doesn't work, please try Online Repair:

https://support.office.com/en-us/article/Repair...

FULL OFFICE UNISTALL (DIFFERENT FROM STANDARD ONE)
https://support.office.com/en-us/article/Uninst...

Then restart your PC and reinstall Office. Restart again and check if it works.

FIX CORRUPT USER PROFILE OR CREATE A NEW ONE
https://www.techbout.com/fix-corrupt-user-profi...

DEFAULT PRINTER NOT ACCESSIBLE
Close all office applications and change your default printer through Start > Devices and Printers to another printer. It may be some physical one or One Note or XPS or PDF or anything else. If this is the problem, then you will need to update your printer driver for default printer.
