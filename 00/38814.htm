Ok, then try following this steps:

1. Close any open office products
2. Type "cmd" in windows search bar
2. Right click on "Command Prompt"
3. Select "Run as Administrator"
4. Type "cd C:\program files\Microsoft office\office16" (or program files (x86), whatever's relevant) without quotes and press ENTER

7. Type "cscript ospp.vbs /unpkey:*****" (whatever those 5 chars where) without quotes and press ENTER
8. Repeat point 7 for every license key installed.
9. Once they've all been removed. Open an office program, sign back in and you're done

diana d say:

Hi Georgina,

Try creating a folder on the USB card. Then copy the videos to the folder. There are sometimes limits to the number of files that can be in the root folder.

If that does not help, convert the USB Key to NTFS format and check if you are able to copy Files.

Note: Make sure to take a back up of the data before you format the USB key.

3. Click "Start" and let the drive format. Your drive is now ready for use.
