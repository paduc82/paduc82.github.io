brian tillman [outlook mvp 2007-2019] say:
The URL references a service named DeltaSync that had been used to connect Windows Live Mail to what was known as Hotmail. Microsoft warned about removing DeltaSync from Outlook.com over a year ago and also warned that Windows Live Mail would no longer be able to connect with that protocol. They published an article last year stating that the Windows Essentials suite of products would no longer be supported after January 10, 2017. in this article: https://support.office.com/en-us/article/Window...

Even though WLM can no longer use DeltaSync, you can switch to using IMAP to connect to Outlook.com. The settings are here: https://support.office.com/en-us/article/POP-IM...

In my opinion, you'd be better served by employing a more modern mail client. You can use Thunderbird, which can use POP or IMAP, or you can use clients like Office Outlook or eM Client (http://emclient.com/), both of which can employ Exchange to connect to the Outlook.com mailbox. Office Outlook is available if you subscribe to Office 365 Home or Personal (or Business) or if you buy it standalone. eM Client is free.

john - project say:

AyshaJaleel Ahamed,

It sounds like someone edited the file using copy/paste or drag/drop while the file was sorted or filtered. As Vijay suggested, try sorting by ID, you may get lucky and that will clear it up.

If sorting by ID does not work, try sorting by Unique ID. If you did not knowingly insert blank task lines, (some users use that method to improve readability), blank lines could be the result of an inadvertent scroll and mouse click. Sorting by Unique ID will reveal those blank line which can then be deleted.

If neither of the above helps, there are ways to restore the file structure but the best approach depends heavily on how bad the structure is "jumbled". For example, one method involves going through the file task by task and creating a new ID sequence by using an extra Number field (e.g. Number1). Then sort on that Number field and select the sort option to "permanently renumber tasks". You may also need to outdent/indent to restore the correct outline structure.

Hope this helps.

John
