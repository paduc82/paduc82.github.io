The good news is that the same post also gives an applescript that can be run with automator (creating a droplet) on which you can drag all your files to be converted and they will be converted and the converted copy placed in the same folder as the original. This means you can use search in Finder and drag all of the docs on the droplet and the automator will chew through them. I should add that for 1300 docs it took one hour on my macbookpro.

Please note that the link given above is for converting to PDF. I changed the code slightly so it would convert to docx. The applescript code I used is as follows:

on open of theFiles

tell application "Microsoft Word" to set theOldDefaultPath to get default file path file path type documents path

repeat with x in theFiles

set theDoc to contents of x

tell application "Finder"

set theFilePath to container of theDoc as text

set theFilename to (name of theDoc) & ".docx"

end tell

tell application "Microsoft Word"

set default file path file path type documents path path theFilePath

open theDoc

set theActiveDoc to the active document

ConvertDocument theActiveDoc file format format document file name theFilename

close front document of application "Microsoft Word"

end tell

end repeat

tell application "Microsoft Word" to set default file path file path type documents path path theOldDefaultPath

end open


Paste this into automator, save as an app and drag you files on the app. Preview works fine on the converted docx files so I assume all is ok.

Menu Bar Missing - Microsoft Community

nancy7105 say:

Menu Bar Missing

The menu bar that says "file, edit, view, insert, etc" across the top is missing.  This is in Excel 2003.  The worksheet menu bar under customize is checked and the menu bar is still not showing up.  We have restarted the computer and still had no luck. 

hansv mvp say:

Press Alt+F11 to activate the Visual Basic Editor.

Press Ctrl+G to activate the Immediate Window.

Type the following, then press Enter:

Application.CommandBars("Worksheet Menu Bar").Visible = True

Then type the following and press Enter:

Application.CommandBars("Worksheet Menu Bar").Position = 1

Switch back to Excel to see if the menu bar has returned.

If not, right-click a toolbar and select Customize... from the popup menu.

Select the menu bar in the Toolbars tab, then click Reset.

If that doesn't help either, see http://www.jkp-ads.com/Articles/StartupProblems.asp.
