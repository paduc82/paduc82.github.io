Here is an example of how to do that:-

 

1. Go to:-

 

http://www.pierrefondes.com/

 

 - item 171.

 

Open the Workbook.

 

2. In the Worksheet called:-

 

sort_worksheet

 

 - highlight the range:-

 

A2:C21

 

 - then:-

 

Data tab

 

Sort & Filter group

 

Click on the button called:-

 

Sort

 

The:-

 

Sort

 

 - window should open.

 

3. In the:-

 

Sort

 

 - window make the following 2 changes:-

 

Sort by - change to Name

 

Order - change to Z to A

 

 - then click on:-

 

OK

 

You should now find that the range:-

 

A2:C21

 

 - is sorted into descending alpha order of Name (and the rows are kept together).
