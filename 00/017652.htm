pmccrory say:

There is a sort of workaround that will fix most cases of this problem.

1. Go to Settings > Connectivity > Settings > Network Destinations

2. On the menu option on the bottom right, press & select "New Destination"

3. Give a name to this new destination, any name other than "Internet" or WAP Services" will do.

4. Move your used wifi networks/access points into you new category you have just made along with your mobile AP if needed (or you can just copy them if you want).

5. Ensure that your new destination is set to be the "Default Connection", which can be chosen in the same menu as that for "New Destination"

You should now find that you wifi connection should no longer drop into an uncategorised destination that you need to keep moving back to "Internet". The wifi access point may still come up as having limited activity in your new destination, but it should not drop the wifi connection. With some applications, you might have to change in their individual settings its default network destination, but should be fine once this is done.

There might be a few apps that might not work with this workaround - it seems these were written to automatically get an internet connection from the "Internet" destination and there is nothing in their options to change that. :(

It seems to be the problem that when a wifi AP is being used for an internet connection, it tries to connect to a Nokia server to check that the connection is working properly. But the web address for this connection seems to have suddenly gone offline and because the 808 (and seemingly other Symbian phones with Belle firmware) is dependent on receiving something back from this server, when it doesn't it assumes that the internet connection on your AP is not working properly, hence showing "limited connectivity".

hansv mvp say:

Create a macro in your Normal.dotm template:

Sub ConnectLetters()
    Selection.TypeText Text:="35c"
    Selection.ToggleCharacterCode
End Sub

You can assign this macro to a custom keyboard shortcut, a custom Quick Access Toolbar button or a custom ribbon button (you have to create a custom group on the ribbon for the latter).

Usage:

- Place the insertion point after the space between the letters you want to connect.

- Run the macro.
