willy denoyette say:

Check your power settings, run powercfg /a from a command prompt.

If the output shows Fast Startup, your system enters a hybrid standby when you shutdown.

Waking from hybrid state is different than waking from a non hybrid state.

Waking from hybrid state loads the saved kernel (including loaded drivers) from hibernate.sys. Some (faulty) drivers do not handle this transition as it should and fail to initialize.

So what you can do is turn this Fast Startup off and check your device drivers for updates.

I have searched and searched, but can't come up with a solution. I have a basic table.

620 { IF A1=610 "Yes" "No" }

I used CTRL+F9 to create the function and I updated the field.

No matter what I put in A1, B1 always returns "Yes".

macropod say:

You could use a field coded as either:

{IF{=A1}= 610 "Yes" "No"}; or

{=A1-610 \# "'No';'No';'Yes'"}

Note: The field brace pairs (i.e. '{ }') for the above examples are all created in the document itself, via Ctrl-F9 (Cmd-F9 on a Mac); you can't simply type them or copy & paste them from this message. The spaces represented in the field constructions are all required.
