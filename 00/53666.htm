john_barnett say:

Hello Hungarian Paprika,

The reason I prefer CCleaner to the Windows Disk Cleanup utility is that CCleaner clears out more than DiskCleanup does and, more importantly, for me, CCleaner also deletes all Windows.dat files. This is particularly important for security.

When you visit internet sites (any site) files are cached in the Temporary Internet Files folder, along with the History Folder. Both CCleaner and the Windows Disk Cleanup utility remove these files when asked to do so by the user. However, these temporary internet files etc., are also stored in a Index.Dat file.

Assuming you have cleared out all of the temporary internet files and history folders using Disk Cleanup. The Index.Dat files will still contain information relating to your browsing habits. Now assume that someone got access to your PC, by using the information in the Index.Dat files they could trace all of your movements while you are on the web.

Removing the Index.Dat files with CCleaner eliminates this source of vital information, thus making your PC more secure from prying eyes. Okay removing the Index.Dat files doesnt prevent law enforcement agencies gaining access to this vital information. They have specialised software that will trawl through your hard drive and obtain all the information they need for criminal activity. But then the average computer user isnt doing anything illegal, all they are trying to do is prevent any unauthorised person from having access to this information. So my main reason for using CCleaner is added security.

I can understand why Electronic Arts has suggested that you use CCleaner, especially to address the list of items you have supplied. These of course, with the exception of DNS Cache are enabled by default within CCleaner so CCleaner would automatically clean these out whenever you use the application.

The Scan for Issues and Fix Selected issues options refer to the registry cleaner option of CCleaner. This, as I stated in my previous post, I would suggest that you do not use. Registry cleaner have been known to delete legitimate registry entries and, as a consequence, render the PC unbootable. Microsoft, incidentally, do not recommend that you use registry cleaners. Registry cleaners are used at the owners risk.

Keep an eye out for firewalls is an intriguing statement. Window has a built in firewall and it is essential for the well-being of your system. It should not, under any circumstances, be disabled. If an application require permission to make changes to the firewall a pop up message should usually appear asking for your permission to make the changes. Yes, by all means check what application is asking for permission and, if that program alien to you then refuse permission.  

I hope this explanation helps.
