chuky_e65 say:

Hi Tommy,

I was having the same issue, and I was able to fix, you need to copy NTLDR, boot.ini and NTDETECT.com from your Windows XP partition to the root of boot partition (in this case Windows 7)

After that, you need to edit your bcd file with the following:

bcdedit -set {ntldr} device boot

This means that when you select Windows XP at your menu, it will go to your active partition (which is Windows 7) and then from there with the help of NTLDR and boot.ini, it will know that it needs to go to partition 2.

At least that worked for me.
