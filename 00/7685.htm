There's no setting to turn Compression off because there's no Compression involved :-)

The only Compression feature in PPT has to do with compression of photos inserted into the presentation file itself. That's regulated through the File> Reduce File Size command or in the Adjust group of the Format Picture contextual tab. It doesn't apply to saving slides or drawing objects as pictures, though.

AFAIK, objects saved as pictures are fixed @ 150 ppi (regardless of image file format used) & there is nothing in the program to change that. PPT simply is not a graphics design program, so the accessory tools it provides are limited.

The pixel dimensions of slides saved as pictures can be adjusted in PowerPoint> Preferences> Save.

I am currently testing software that records both phone conversations and the FM radio output on my E73.  There may not be a free program for this, but most aren't very expensive.  I have found that programs marketed as "call recorders" generally also will record from the radio, provided there is a manual start and stop option.  So maybe they will record other sources, too?  You might start with the demo version of Total Recall from Killer Mobile (http://www.killermobile.com), and if that works, either register it ($15 US) or look for other similar programs.  I prefer a program that records in .mp3 format, as .amr files aren't meant for recording music, and .wav files can be too big. So far I have found just one program with this feature.
