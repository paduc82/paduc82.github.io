idolno1 say:

As I was reading I was wondering if you were mistakenly thinking your "Switch" was a "router". In that case, the switch does nothing but handles frames of packets and does not DHCP or routing. So whatever computer would come up first get the IP from the modem in which case, the modem will only give out one IP address. That is where the router comes in. The router takes that "one" IP from the modem and uses it as it's internet IP. Then it assigned internal (or private) IP's to your home network devices (computers, iPads, Laptops, etc,..).

 

Yep, get yourself a router problem solved.

idolno1 say:

Don't need no blasted CD!

 

Turn everything off, then power up modem, allow it to come online first so that it'll see the router when it boots up. When the "Internet" light comes on the modem, then boot the router. When the "Internet" light comes on the router, then you can start booting your computers, or plugging them into the router. Best to have them plugged in first, then boot them up one at a time. Usually you can just start turning them on then the router will assign the IP's to each devices as the request an IP from the router.

 

You should be good to go from there as long as you don't have any custom settings on your computers such as static IP's. "Obtain IP address automagically" for DNS "Obtain DNS automagically". lol

 b. Click on User Accounts.
 c. In Make changes to your user account area of the User Accounts window.
 d. In the text box on the next screen, enter your current Windows 8 password.
 e. Click the Remove Password button to confirm that you'd like to remove your Windows 8 password.
 f. You can now close the User Accounts window.
 g. Now that your Windows 8 account password has been deleted, you will no longer need to log on when your computer starts.

Ei.cfg Removal Utility Lets You Use Any Product Key
http://lifehacker.com/5438005/eicfg-removal-utility-lets-you-use-any-product-key-with-your-windows-7-disc

 

we will edit the image and remove the ei.cfg file so we can reveal all editions of Windows 7 during the installation and select Windows 7 Starter. Lets take a look at doing it step by step.

 

After downloading the eicfg removal utility, you need to extract, since its in a .zip file. Just right click it and click Extract All.

 

Double click the eicfg_remover.exe file to launch it.

 

Browse to where the Windows 7 Image (.ISO file) is stored, select it and click Open.

 

Thats it! the ei.cfg file is now removed. The next step now is to reinstall Windows 7 Starter. Because most Netbooks do not include a DVD drive, you will have to use a tool such as the Windows 7 USB/DVD Download Tool (http://store.microsoft.com/Help/ISO-Tool)

Select the USB device where you want to have the backup stored to for installation.

 

Once the USB device has been identified, click the Begin copying button.

 

And thats it, the next step now is to boot your Netbook from the thumb drive, make sure the thumb drive you will be installing Windows 7 from is plugged into the USB port. This might vary by manufacturer, but the usual one to try is F12, start your Netbook and press F12 repeatedly until a screen is displayed giving you the option to boot from a USB Memory device, select that and press Enter on your keyboard.

 

Windows 7 will boot as it normally does into the setup environment, the only difference this time is, you will be given a choice to select the edition of Windows 7 you want to reinstall. Carefully click through during the setup wizard.

Click Run as adminstrator

slmgr.vbs -ipk xxxx-xxxx-xxxx-xxxx (allows you to replace the current product key with the specified)

xxxx-xxxx-xxxx-xxxx - represents your product key
