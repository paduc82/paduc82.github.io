slickrcbd say: Generally on a support forum, you shouldn't hijack somebody else's thread to ask an unrelated question even if it is somewhat similar. Asking a related question is SOMETIMES acceptable, but this one is not related.

That said, generally with HP printers, you have to either use the CD that came with the printer or download the driver's installer from the HP driver download website.
They don't usually distribute drivers in a from the "have disk" option will accept because they like to bundle other junk with their printer drivers.

With older HP printers that were out for at least a year before whatever version of Windows you are using was released (or sometimes a year before the service pack included on the installer was released) you might be able to let Windows handle installing things, but with most current printers you need to download the drivers.

I do see an "HP OfficeJet J4500 Series" on the Windows 7 list however. I have no specific knowledge of this driver or the model of printer, so I'm unsure if it is the same all-in-one.

That said, my experience using the built-in drivers even when they are present is that you generally only get the most basic functions and you may not be able to use all the features of your all-in-one with just Microsoft's drivers. You may only be able to print with the most basic print functions.

If you wish to try the Microsoft "HP OfficeJet J4500 Series" driver, follow these instructions below. However, you are better off downloading HP's drivers for your specific model from here.
http://www8.hp.com/us/en/support-search.html?tab=1#/qryterm=OfficeJet%204500&searchtype=s-002

I'm unsure if the following instructions will let you do anything but print, since I've never installed an all-in-one printer using this method. I've always used the CD that came with the printer.
To try the Microsoft Drivers
  1. Log in with an administrator account.
  2. Go to Start->device and printers
  3. Click on "Add a printer" 
  4. Click "add a local printer"
  5. Click the button for "use an existing port" (should already be selected), and choose either LPT1 or something that starts with USB depending on if it is a parallel printer (big wide connector) or USB (small rectangular connector with cropped corners on one side that is the same as a flash drive on the end that goes into the computer, a smaller squarish connector with the same cropped corners on the other that goes into the printer) respectively.
  6. click next
  7. In the next screen, in the left pane under "manufacturer" find "HP" and click on it. See step 8 before clicking next.
  8. In the right pane, scroll down until you find the appropriate model, in your case "HP OfficeJet J4500 series" (I hope). It's in alphabetical order, and to the computer letters come after numbers, so the HP OfficeJets without the J in front will come first.
  9. Click next.
  10. Enter in a name for the printer or keep the default and click next.
  11. Wait. Then answer the question on if you want to share the printer.
  12. Click next, and do a test page. If the test page works, then click finished. Otherwise cancel and go download the drivers from HP.


Yes, I know I keep harping on downloading the drivers from HP, but with all-in-ones that aren't automatically being correctly recognized, that is usually the correct solution 99.999998% of the time.

Then, Boot your PC from the Installation Media you just created (change Boot Order in your BIOS) to begin installing Windows 10

If you have problems booting from a boot disc, you may have UEFI BIOS:
Insert the Bootable Installation Media, then go into your BIOS and make the following changes:
1. Disable Secure Boot
2. Enable Legacy Boot
3. If Available enable CSM
4. If Required enable USB Boot
5. Move the device with the bootable disc to the top of the boot order
6. Save BIOS changes, restart your System and it should boot from the Installation Media
