jllatham say:

Try rewriting it this way:

Private Sub Copy_Click()


        DoEvents ' allow system to process background tasks (like 'seeing' [Break] key)

        Set c = SrchRng.Find("not", LookIn:=xlValues)

            c.EntireRow.Copy

'this next statement is wrong, will show how to fix it later
            c.EntireRow.PasteSpecial xlPasteValues

       End If

OK, that will kind of take care of some of it.  First - I added the DoEvents statement inside of your Do ... Loop block so that if it does get 'locked up' it should still see you hit the [Break] key.

Next I put everything you want to do when 'c Is Nothing' within an IF ... End If block so that we only do the sheet .Select and paste operation when we copy the source row.  As you how have it, it will always try to do that regardless of whether or not c Is Nothing.

Problem with the c.EntireRow.PasteSpecial xlPasteValues is that you aren't really telling it where to paste on Sheet3 - so it's probably going to overwrite one line on that sheet forever.

Another potential problem is that in a .Find operation, Excel will start over looking from the top of the search range after it finds the last entry in that range.  This leads to an infinite loop.  You have to 'remember' where the first match was found, and if it loops back to that address, then stop the loop just as if c is nothing (no match found).

See this page for a pretty good example which we will use and modify slightly:

https://msdn.microsoft.com/en-us/library/office/ff839746.aspx

Try this version of it and see if it works better for you.

Private Sub Copy_Click()

  Dim c As Range
  Dim SrchRng
  Dim firstAddress As String
  Dim nextSheet3Row As Long
  
  'using .Range("A" & Rows.Count) lets it work with any version of Excel
  Set SrchRng = ActiveSheet.Range("A1:" & ActiveSheet.Range("A" & Rows.Count).End(xlUp).Address)
  Set c = SrchRng.Find("not", LookIn:=xlValues)
  If Not c Is Nothing Then
    'we found at least 1 match, save first address
    firstAddress = c.Address
    'NOW start looping through
    Do
      DoEvents ' allow system to process background tasks (like 'seeing' [Break] key)
      'find next empty row (based on column A) on Sheet3
      nextSheet3Row = Sheets("Sheet3").Range("A" & Rows.Count).End(xlUp).Row
      c.EntireRow.Copy
      'no need to actually .Select the other sheet
      Sheets("Sheet3").Range("A" & nextSheet3Row).PasteSpecial xlPasteValues
      Application.CutCopyMode = False
      Set c = SrchRng.FindNext(c)
      If c.Address = firstAddress Then
        Exit Do
      End If
    Loop While Not c Is Nothing
  End If
  MsgBox "Required copy complete"
End Sub
