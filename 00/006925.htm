Here's a simple bit of code to do it. Change the column letter in bold if you need to look at a different column.


Sub insertrows()
Dim MyColumn As String, x As Long
'insert a row at every name change in a column
MyColumn = "A"
For x = Cells(Rows.Count, MyColumn).End(xlUp).Row To 2 Step -1
    Rows(x).Resize(3).Insert
Next x
End Sub

in your loop, then after the loop reset it with

application.statubar=false

This only works as long as Excel has the focus (which it might lose somewhere along the way, and appear to have stopped when it is actually still processing), but it generally works for me. Function ElapsedTime(endtime As Date, starttime As Date)
    Dim strOutput As String
    Dim Interval As Date

    Interval = endtime - starttime

    strOutput = Int(CSng(Interval * 24 * 60)) & ":" & Format(Interval, "ss") _
        & " Minutes:Seconds"
    ElapsedTime = strOutput

End Function
