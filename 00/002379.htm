scott .. b say:

Internet Explorer 11 has the ability to emulate earlier IE versions, such as IE9, on a given webpage.  Its a work-around only, but maybe you can use it.

Open Internet Explorer on the Desktop.

Go to the webpage that you would like to open.

Press the Windows Key and f12 .

On the left side, scroll to "Emulation"

Click on "User Agent String" and select "Internet Explorer 9....etc"

Your screen will change immediately.

NOTE:  The emulation switch only works on the currently opened webpage, so if you change pages, you will have to set it again.  But, you can pin the emulation window to the bottom of any page, for easier access.
