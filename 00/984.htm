Hi,

First I would ask are the cells empty or do they contain 0?  If they are empty then you should test for

.c=""

Second your code is not for Conditional formatting, here is what that code would look like:

    ActiveSheet.PivotTables("PivotTable1").PivotSelect "", xlDataOnly, True
    Selection.FormatConditions.Delete
    Selection.FormatConditions.Add Type:=xlCellValue, Operator:=xlEqual, Formula1:="0"
    Selection.FormatConditions(1).Interior.ColorIndex = 3
