¡firedog say:

Please note that this feature doesn't subscribe you to anything. All that happens is that Outlook scans your mail as it arrives looking for indications that this is a subscription to this and future messages from the same sender. This evidence might simply consist of an unsubscribe link in the body of the message. Legitimate messages - and probably some illegitimate ones - will also contain a header line starting with List-Unsubscribe: and giving the link to the unsubscribe page.

Spammers, particularly those sending phishing mail, will often include an unsubscribe link among others in the body of the message to help convince you that it is a genuine message from someone else - a bank, a shopping site or an email service provider, for example. Clicking on such a link will, even though you go no further, prove that the email address the message was sent to is a genuine one - yours - making it more valuable to the spammer, who can sell it to other spammers and thus make you a target for more spam.

The best advice I can give is simply ignore these listings in the Manage subscriptions dialogue. Use it only when you do want to unsubscribe from a list you actually did subscribe to. If messages from the fraudulent senders are being delivered to your junk folder, they will automatically be deleted after 30 days; otherwise, you can simply delete them yourself. If they are being delivered to the inbox, mark them as junk. This will both remove them to your junk folder and send a signal to Outlook and its partners. If a large number of users mark these messages as junk, they will eventually be stopped by spam filters everywhere and not even make it to users' junk folder.

It is likely that Outlook.com will at some stage make it possible to delete bogus entries from the list of subscriptions. Until then, ignore them.

Hi,

 

To date there is no version of Excel that can do that so we need to write out own function to do it.

 

ALT+F11 to open VB editor. Right click 'ThisWorkbook' and insert module and paste the code below in on the right. Close VB editor.

 

Back on the worksheet you call it like any other function like this

 

 

=CountColour(C23:DA23,D23)

 

Where C23:Cz23 is the range to count and D23 is another cell with the same fill colour you want to count.

 

 

 

Function CountColour(rng As Range, clr As Range)
Application.Volatile
Dim c As Range
For Each c In rng
    If c.Interior.ColorIndex = clr.Interior.ColorIndex Then
        CountColour = CountColour + 1
    End If
Next
End Function

gord dibben say:
I want to a formula or macro to return the number of cells in range C23:CZ23, that have a red shaded background.


This UDF can do that.

 

Function CountByColor(InRange As Range, _
    WhatColorIndex As Integer, _
    Optional OfText As Boolean = False) As Long
Dim rng As Range
Application.Volatile True

For Each rng In InRange.Cells
If OfText = True Then
    CountByColor = CountByColor - _
            (rng.Font.ColorIndex = WhatColorIndex)
Else
    CountByColor = CountByColor - _
       (rng.Interior.ColorIndex = WhatColorIndex)
End If
Next rng

End Function

 

=CountByColor(range,colorindexnumber)    red is index 3

 

If you don't know the colorindex number you could use this to find that.

 

Function CellColorIndex(InRange As Range, Optional _
        OfText As Boolean = False) As Integer
    Application.Volatile True
    If OfText = True Then
        CellColorIndex = InRange.Font.ColorIndex
    Else
        CellColorIndex = InRange.Interior.ColorIndex
    End If
End Function
