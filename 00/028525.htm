j.e. mcgimpsey say: Homework question?

Fortunately, this seems trivial...

    3.11 - 0.000137v^2 = 62.1 / v  - 0.046 - 0.000137v^2

obviously reduces to 

     v = 62.1 / (3.11 + 0.046)  

     v = 19.678060836502

and therefore

     dv/dt = 3.05695679242146

On something non-trivial you could put the right side of each equation into separate cells (substituting a third cell reference for v), put the difference into  a fourth cell, then use Goal Seek or Solver to set the value of the fourth cell = 0 by changing the third...

I believe I responded to a question like this many years ago with a UDF I wrote and kept, does this do what you want. It goes in a general module.

It will calculate the Nth prime if you call it with

=NthPrime(a1)

where a1 contains the number of the prime you want.

or

=NthPrime(99) for the 99th prime number.

It gets a bit slow for very large numbers but for the first 100k primes it's not bad. Looking at other responses I see you are referred to Chip pearson's site, I think Chip's formula for determining a prime is good but wrong in that it identifies 1 as the first prime number, I disagree with that, 1 is not prime.

Also as another respondent has said, no formula has yet been divised for finding the nth prime and I agree with that, to do so would guarantee anyone who did it a place in history alongside the worlds greatest mathmaticians, the only way; to date, is to test every prime until you find the one you want.

Function NthPrime(ReqPrime As Long)
Dim i As Long, x As Long
Dim GetMeOut As Label
On Error Resume Next
For x = 2 To 10000000
    If (x <> 2 And x Mod 2 = 0) Or x <> Int(x) Then GoTo GetMeOut
        For i = 3 To Sqr(x) Step 2
            If x Mod i = 0 Then GoTo GetMeOut
        Next
        foundprime = foundprime + 1
        If foundprime = ReqPrime Then
            NthPrime = x
            Exit Function
        End If
GetMeOut:
Next x
End Function
