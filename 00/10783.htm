deleting email from one device - Microsoft Community

donnaurbanovsky say:

deleting email from one device

Is there a "Rule" that can be created so that if an email is deleted from my phone, it is deleted from the inbox on the computer?

karl timmermans (outlook mvp 2012-2018) say:
>>
Is there a "Rule" that can be created so that if an email is deleted from my phone, it is deleted from the inbox on the computer?
<<

No such capability exists - how that particular function is controlled is via the protocol (POP3, Imap, Exchange etc) used when the account is configured on each device.

The following applies to anything (computer or mobile device) that connects to an Ouotlook.com account using an <email client> versus a browser.

POP3 - is a one-way one-time download of the <Inbox>. Nothing that gets deleted on the computer/device gets reflected on the server

IMAP - is a bi-directional sync of email folders. What gets deleted on the computer/device is then removed from the server and vice-versa. In short, if you delete an item on your device - there is no way to selectively control what gets removed from the server.

Outlook.com/Exchange - the same as IMAP except it includes all item types (email/contacts/calendar) etc

Outlook does not connect to/sync email accounts directly with other devices but rather connects with the server. Each device does the same thing.

Since you have posted the question in the Outlook.com forum (the forum dedicated to accessing your Outlook.com account via a browser) - the above information is not relevant. When connecting via a browser, nothing is specific to a given device since it reflects what's on the server.
