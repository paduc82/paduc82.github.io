Are you sure the card is good?  Does it work in any other device?  It has probably become corrupted and needs reformatting.  If you have a card reader on your PC, you might try it in that, right click on the reader's drive letter in Explorer, select properties and try to repair the card.

 

My old AT&T Tilt WinMo phone used to corrupt its microSD card every few months.  I used to back up the card to my PC with a reader every week or so, so I was able to reformat it and reload the data whenever the Tilt decided to nuke the card.

a. user say:

Hi SM,

I'd like to confirm whether after user mailboxes are moved to Office 365, only some users are asked for their password after restarting Outlook client in a Hybrid with Office 365. For Outlook continues to prompt for credentials, please try the following steps:
1. If this issue occurs to Outlook 2010, please run Desktop setup from the Office 365 Portal.
2. Remove stored Outlook credentials from Control Panel > Credential Manager > Windows Credentials.
3. Run SARA tool (Office 365 Support and Recovery Assistant).

After doing the steps above, Outlook still prompt for credentials. Please start the Test E-mail AutoConfiguration tool, follow these steps:
1. Start Outlook.
2. Hold down the Ctrl key, right-click the Outlook icon in the notification area, and then click Test E-mail AutoConfiguration.
3. Verify that the correct email address is in the E-mail Address box.
4. In the Test E-mail AutoConfiguration window, click to clear the Use Guessmart check box and the Secure Guessmart Authentication check box.
5. Click to select the Use AutoDiscover check box, and then click Test.
6. Send the results to me via Private Messages by clicking on the link below:
https://community.office365.com/user/conversations

Thanks,
Edward
