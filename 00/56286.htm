mike--- say:

hpuszyns,
    If you are able to boot to Safe Mode with Command Prompt then you can use the DISM tool to remove the most recent updates.  Then you can see if you are able to log in.

DISM to remove updates.

Boot to System Recovery Options and select Command Prompt

From the command prompt run the following command:

Dism /online /get-packages

This will produce a list of installed updates and packages.  They will look like this:

Package Identity : WUClient-SelfUpdate-Core-TopLevel~31bf3856ad364e35~x86~~7.4.7600.226

State : Installed

Release Type : Feature Pack

Install Time : 9/25/2009 12:02 AM

Find the most recent one and then make a note of the package name.

Then run: dism /online /remove-package /PackageName:WUClient-SelfUpdate-Core-TopLevel~31bf3856ad364e35~x86~~7.4.7600.226(name of update from previous step)

Then reboot your PC.

You will want to start with the most recent and remove them one at a time to find the problem update.

For more information about the DISM command, go to this link.
