Converting PowerPoint Shows to Video Files - Microsoft Community

joe kalinowski fv say:

Converting PowerPoint Shows to Video Files

I created a narrated PowerPoint Show file (.ppsx) using PowerPoint for Mac 2011 (Version 14.6.1). I would like to convert that file to an audio-narrated video file for posting to YouTube. My version of PowerPoint does not support that function.

If I open the existing file on a Windows PC running a later version of PowerPoint (say 2010, 2013 or 2016), will that PC allow me to generate the desired audio-narrated video file? If so, what steps do you recommend?

john korchok say:

You can export to video in a Windows version of PowerPoint and the narration will be captured.

There also are 2 ways to do this on a Mac. You can extract the narration and add it as a sound file to the video export, or you can record the presentation in QuickTime Player.

Method 1:

Please follow these steps to extract the audio:

  1. Close PowerPoint. Make a copy of your presentation and rename it, adding .zip to the end of the file.
  2. Right-click on the file and choose Open With>Archive Utility. The file is unzipped.
  3. Archive Utility creates a folder with the same name as the presentation. Open it.
  4. Open ppt>media.
  5. Your narration will be in a WAV file. If you only have one narration and no music, then there will only be one file. If there are more, play the files to find the narration.

Now you can re-export the movie:

  1. Open the presentation in PowerPoint.
  2. Choose File>Save as Movie.
  3. Click on the Movie Options... button
  4. Beside Background soundtrack, click the dropdown and choose Select soundtrack.
  5. Choose the WAV file that holds your narration.
  6. Click OK, then Save.

Method 2:

  1. Open your presentation in PowerPoint and resize the window to the size you want your video to be (i.e. 640x480 pixels, etc.).
  2. Select Slide Show>Set Up Show...
  3. Under Show type, choose Browsed by an individual (window), then click on OK.
  4. Start your Slide Show.
  5. In QuickTime Player, choose File>New Screen Recording.
  6. On the Screen Recording dialog, click the red button in the middle.
  7. Drag your mouse to select the PowerPoint window.
  8. Click on Record.
  9. Run your slideshow. When you're finished, click on the square dot inside a circle on the OS X menu bar to stop QuickTime. Save the video file.

The browser needs certain amount of video buffered into memory before it can play. If it doesn't have enough network bandwidth to buffer as much as it needs to, the browser will keep trying but never succeed at playing the video. Issues that could keep you from having enough bandwidth for buffering include: 
 
a. A slow network connection (such as a dial-up modem).


b. Too many services or applications using your local network (e.g., watching video while downloading files and surfing the Internet).


c. Too many people using the local network.
