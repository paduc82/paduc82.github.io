Install this AddIn. It has no user interface nor any settings.

### How to install:

Save the file in a trusted location.
Click File\Options\Add-Ins\Dropdown Manage: Excel-Add-Ins\Button: Go To\Button: Browse
Select the AppCloser.xlam file
Click OK\Ok


### How it works:

It tests with some Windows API codes if the mouse cursor is positioned on the title bar if a workbook is closed,
that happens if you click the close button in the upper right edge of Excel.

If this is the case, it tries to close all opened workbooks, one by one.

If a file isn't saved, Excel asks you to save it or not. If you cancel the close, the AddIn stops closing remaining files.

If all files are closed successfully, it closes Excel.
