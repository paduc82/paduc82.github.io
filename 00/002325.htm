bob jones aka: cybertaz say:

I'm not an authority on the subject but this is my understanding:

The key question is whether the PDFs actually were embedded using a Windows version of Word or whether they were inserted using a Mac version.

  • In the first case there should be an embeddings folder in the unzipped Word document. That folder should contain the complete PDFs in PDF format.

  • In the second case: Object Linking & Embedding [OLE] is a Windows technology which never has been fully implemented on the Mac operating system. It is supported in Mac Office only for Word documents & Excel files... PDF is not a supported OLE file type.

In Mac Word a PDF is inserted as a picture displaying only the first page of the PDF, which is processed as an EMF image file. The inserted EMFs are what you are finding in the media folder. Unfortunately, EMF is not native to Mac so they can be opened only by certain graphics programs which support that file type, Photoshop among others. Even then, only the inserted page will be included if the original PDF was a multi-page document. Also, the EMFs can be inserted into another Word document {Insert> Pictures - Picture from File] but will present only as a picture displaying the page that was inserted originally.

Hi,

Try this

1. Suppose your time entries are in range B2:J2

2. Select B2:J2 and press Ctrl+H

3. In the Find What box, type m i.e. m and space

4. Click on Replace All

5. In the Find What box, type s

6. Click on Replace All and close the dialog box

7. Type 60 in any cell and copy it

8. Select B2:J2, right click > Paste Special > Divide > OK

9. Format B2:J2 as hh:mm:ss

If your data are in B2:C2, then:

=SUBSTITUTE(SUBSTITUTE(TEXT(SUMPRODUCT(

[EDIT] The double negate (--) has the effect of converting text in the form of m:s.0 to numeric time.

Sub Splitter()
    ' declare some variables we are going to use
    Dim mask As String
    Dim originalDocument As Document
    Dim docTemplate As String
    Dim counter As Long

   
    ' get the document's template and store a copy of the original
    docTemplate = ActiveDocument.FullName
    Set originalDocument = ActiveDocument
   
    'Letters = Selection.Information(wdActiveEndSectionNumber)
    mask = "yyyyMMdd"
    counter = 1
   
    ' save the split parts of the document one at a time
    For Each mergeSection In originalDocument.Sections
           
        ' this is the location where the files will be saved
        DocName = "W:\Mail Merging\" & Format(Date, mask) & "split\" & Format(Date, mask) & " " & LTrim$(Str$(counter))
       
        ' copy the current section
        mergeSection.Range.Copy
       
        ' store the new doc so we can set the template
        Dim newDoc As Document
       
        ' add the new doc using the normal.dot
        Set newDoc = Documents.Add(, , , False)
       
        ' set the style
        newDoc.CopyStylesFromTemplate (docTemplate)
       
        ' copy the section into new doc
        newDoc.Activate
        newDoc.Range().Select
       
        Selection.Paste
       
        ' cut the last section break if it's in there
        If (newDoc.Sections.Count > 0) Then
            RemoveSection
        End If
       
        ' save and close
        newDoc.SaveAs DocName
        newDoc.Close
        Set newDoc = Nothing
        counter = counter + 1
    Next
   
    originalDocument.Activate
   
    Set originalDocument = Nothing
    MsgBox "done"
       
End Sub
