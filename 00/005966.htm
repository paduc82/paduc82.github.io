What do you mean by "...word keeps turning it back"?

If you launch Word then in the new blank document go to Tools> Language, what is indicated? That tells you what your Default Language is set to for new documents yo create.

If what you're actually referring to has to do with existing documents, your Default Language has no bearing on them or on the
Those are Formatting Attributes of the text in the document. IOW, Word does not arbitrarily change the Language formatting of the documents you open to match the Default Language you use for creating new ones any more than it changes the Bold, font size or any other formatting in the document to match those defaults. It recognizes what the content of the document is marked to be.

If an existing documents [or parts of them] are formatted for the improper language you need to select that content then specify what should be used instead. That can be done on a direct basis or [preferably] by modifying the Styles use to format the text. See this web page for more:

Spelling, Language, Dictionary
