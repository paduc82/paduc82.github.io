Word 2003's AutoText feature always entered plain text which would assume the formatting that I had selected for my current project. How can I make F3 key enter plain text in Word 2007?

stefan blom say:

As you already know, AutoComplete does not work for AutoText in Word 2007 (the functionality has been restored in later versions of Word).

With regards to formatting, note that AutoText entries (Building Blocks) are never plain text in Word. This applies to all versions of Word (since at least 1997). What you have to do is prevent direct formatting from being added to your entries as you create them. When you are in the process of creating an AutoText entry, be sure to select your example content and press Ctrl+SpaceBar (to clear any direct font formatting) and then create the entry. This prevents direct font formatting from being added.

In addition, if you have selected a whole paragraph (up to and including the paragraph mark, the current paragraph style will be saved with the entry. If that style does not exist in documents where you insert the AutoText, formatting will change in the target document.

You may want to make use of AutoCorrect entries instead (File | Options | Proofing | AutoCorrect Options | AutoCorrect). These are plain text by default and they will be triggered by specific text strings. For example, you can have "BJ" autocorrect to "Ben Jones."

I don't fully understand the question...

If you're referring to the interface language, it should be rather obvious. Mac Office is language-specific, so if you have the French version the menus, dialog windows & other text elements of the user interface will be in French. If you have an English version, all will be in English, etc. Other than that, Mac Office doesn't have a 'language'.

Language is a property of the text used in the document, regardless of whether it's an Excel spreadsheet, a PowerPoint presentation or a Word document. The initial Default language for text typed into an Office document is determined by the software version. However, any portion of any document can be formatted as being in virtually any language.

The easiest way to determine what the current Default language is set for is to open a new blank document in Word, then go to Tools> Language to see which language is highlighted in the list. The Default can be changed to whatever you prefer & the choice will affect Excel as well. PowerPoint has its own Default language setting.

Maybe a simple macro like this
 
Public Sub ProcessData()
Dim LastRow As Long
Dim sh As Worksheet
 
    Set sh = Worksheets.Add(after:=Worksheets(Worksheets.Count))
    With sh.Range("A1")
   
        .Value2 = "Item Name"
        .Font.Bold = True
    End With
   
    With Worksheets("Stock")
       
        LastRow = ..Cells(.Rows.Count, "A").End(xlUp).Row
        ..Rows(2).Resize(LastRow - 1).Copy sh.Range("A2")
    End With

        LastRow = ..Cells(.Rows.Count, "A").End(xlUp).Row

        ..Rows(2).Resize(LastRow - 1).Copy sh.Range("A1").End(xlDown).Offset(1, 0)
