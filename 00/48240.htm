Create a template for mail messages - Microsoft Community

¡firedog say:

Create a template for mail messages

Technical Level : Basic

Summary

If you regularly send messages to the same group of people and/or with the same or largely unchanged content, you can save time and effort by using a template to store all the details that don't change from message to message. A template is simply a draft message which can be opened, completed and sent. You can create as many templates as you like for different purposes.


Details

Users frequently ask how to include a CC: or BCC: address in all outgoing messages automatically. There is no function within Windows Live Mail for doing this, but it is possible by using the method in this article.

Another scenario that causes problems for many users of Windows Live Mail 2012 is the unreliability of the group email feature, which relies on addressing a message to a category (a group of contacts) rather than to individual contacts. While there is a workaround for this fault in the program, it doesn't help if the category itself doesn't always return all the contacts it's supposed to. Again, using a template automates the group email process.

Create a template

  1.  Start a new mail message (e.g. by pressing Ctrl-N) and enter the bits that don't change, or don't change much, e.g. addressees, subject, body, pictures, signature and attachments. Format the message to suit.

    This is a simple template that will include a BCC address in the outgoing message:

      

    Here's a more complicated template for a monthly     newsletter:

                  

  2. Press F12 to save the message as an .eml file in a convenient place, like the desktop.
    NB: the     Save message As dialogue will by default apply the subject line as the filename; edit this to suit. 
     

Double-clicking on this .eml file will open a new message ready for completing and sending. The program doesn't have to be running when you start a new message in this way. The template itself will stay unchanged where it is, ready to be used again next time.

Put a descriptive name for the distribution list in the To... box. When sending to a distribution list, it's polite to put the addressees in the BCC field so that none of the recipients will see who else has received the message. A useful trick to ensure that the message makes it to its destinations is to put the sender in the To... box. In the example newsletter template above, a new contact has been created with WLMail as its first name, Newsletter as its last name and with the sender's address as its email address. If the To... box is left empty, the message will show as being sent to Undisclosed recipients, which isn't very helpful to them or to you when you need to find a specific message in your Sent items folder.

the c. say: Step 1
Dell Dock - Advanced settings

Right click anywhere on the Dell Dock. This will bring up an options selection menu. Select "Advanced Settings...".

NOTE: There is an option to "Close the dock". This will only shut the Dell Dock down for that one session. It will start up, again, the next time you log in, so don't select that option.

  • Step 2
    Dell Dock Options

    Left click on the arrow next to "Dock options". Under "Dock options", clear the check box by "Run at startup".

    This will keep the Dell Dock from loading the next time we log in.

  • Step 3
    Unloading the Dell Dock

    Now, we will shut the the Dell Dock for the current session so we don't have to log off and back on.

    Click the arrow next to "Unloading the dock".

    There are two options here: "Close the dock" and "Turn off the dock and switch back to desktop icons".

    Selecting "Close the dock" will simply close the dock, but selecting "Turn off the dock..." will export the icons on the dock to your desktop. If you like some of the functionality of the dock, you might want to go with the option to switch back to desktop icons.

  • Step 4
    Close Dell Dock Prompt

  • You'll be prompted to confirm that you really want to close the Dell Dock. Click "Yes" and you are all done.



    Hope this helps you. Let us know either way. Make it a great day!

    a. user say:

    Hi Gary,

    Thank you for posting your query on Microsoft Community. Let me assist you.

    The touch pad can be disabled by you on the Control Panel of your computer.

    Refer the steps below to disable the touchpad on your computer.

    1.  Right click on Start, and then click Control Panel.

    2.  Click Hardware and Sound.

    3.  Click Keyboard.

    4.  In the Mouse Properties window:
      • Click the Device Select tab to disable the touch pad.
      • Click the Touch Pad tab to adjust touch pad settings.

    5.  Click OK to save the settings and restart the system and check.

    Try updating the Touchpad drivers from the manufacturer's website and check if the issue persists.

    Follow the steps to update the drivers:

    a. Press Windows key + X key and click Device Manager.
    b. Expand the Touchpad tab.
    c. Right-click and click on Update driver software.
    d. Restart the computer and check.

    Hope this information is helpful. Please do let us know if you need further assistance, we’ll be glad to assist you.
