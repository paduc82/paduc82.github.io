Three example subs below. Note that if you dimension the correct type of variables then much of the code can be found using the intellisense dropdowns when writing the code.

If you don't know the type of variable to dimension then dimension it as a variant and use TypeName function to find the correct type of variable to use and then go back and edit Variant to correct type. If Variant does not work (ie. errors)  trying to get the TypeName then try dimensioning as Object (eg. Dim optBtn as Object).

Example code for above.

Sub IdentifyVariableTypes()
    Dim optBtn As Variant
    Set optBtn = ActiveSheet.OptionButtons("Option1")
    MsgBox TypeName(optBtn)
End Sub

The example code will return variable type OptionButton so edit the Dim statement change from Variant to OptionButton. Then in the following example code, use the With statement and in the subsequent lines under With OptBtn when you type a dot you will get a dropdown with many options and it is usually possible to identify the command you require.

    Dim optBtn As OptionButton
    Set optBtn = ActiveSheet.OptionButtons("Option1")
   
    With optBtn
        .    (Type the dot here and you will see a dropdown with options for the next command)
       
    End With

Code examples for your project

Sub to create an Option Button

Sub to Find Option Button and create if not exist

Sub to Delete Option button.

Sub CreateOptBtn()
    Dim optBtn As OptionButton
    Dim rCel As Range
    Dim dblLeft
    Dim dblWidth As Double
    Dim dblHeight As Double
    Dim dblTop As Double
   
    Set rCel = Range("A5")  'Edit "A5" to cell where option button required
   
    With rCel
        dblLeft = .Left
        dblWidth = .Width * 2   'Double width of cell
        dblHeight = .Height
        dblTop = .Top
    End With
   
    Set optBtn = ActiveSheet.OptionButtons.Add(dblLeft, dblTop, dblWidth, dblHeight)
   
    optBtn.Name = "Option1"
   
    optBtn.Caption = "Option 1"
   
End Sub

Sub FindOptionButton()
    Dim optBtn As OptionButton
    Dim bolShpFound
   
    For Each optBtn In ActiveSheet.OptionButtons
        bolShpFound = True
        Exit For
    Next optBtn
   
    If bolShpFound = False Then
        Call CreateOptBtn
    Else
        MsgBox "Option1 found at cell " & optBtn.TopLeftCell.Address(0, 0)
    End If
   
End Sub


Sub DeleteOptBtn()
    Dim optBtn As OptionButton
   
    With ActiveSheet
        On Error Resume Next    'In case the Option button does not exist
        Set optBtn = .OptionButtons("Option1")
        On Error GoTo 0
    End With
   
    If Not optBtn Is Nothing Then
        optBtn.Delete
    End If
End Sub

SMTP mail supports a command that can verify the existence of a mail address but most SMTP services ignore the command.  If they didn't, spammers could verify the existence of the addresses on the lists they exchange with each other.  So, it's not likely that you can verify an address is real.
