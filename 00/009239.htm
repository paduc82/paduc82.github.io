How do you add accessibility (For a program like JAWS) to - Microsoft Community

bdm_dbs say:

How do you add accessibility (For a program like JAWS) to checkboxes in Word 2010?

I'm attempting to create a form that has checkboxes that I would like to have some form of accessibility for individuals who use programs such as JAWS.  Currently, the program won't even tab to the checkboxes when I attempt to protect the form, but that is a whole issue unto itself.  Any help is appreciated.

jay freedman say:
As I said in my reply to your other question, it appears you're using content controls rather than legacy form fields, so you should not use forms protection. That accounts for not being able to tab to the checkboxes.

Unlike other content controls, checkboxes don't have any placeholder text, nor can they ever contain any text (the 'x' in a checked box isn't text). That means the screen reader programs won't ever see anything there to read. Of course, you can and should put some descriptive text next to each checkbox, but that isn't going to let the screen reader know whether the box is checked or unchecked.

One alternative would be to replace a checkbox with a dropdown list content control, in which you set the list (in the control's Properties dialog) to the items Yes and No (or some other pair of words). The screen reader will be able to read the selection.

If you have a little bit of experience in working with VBA macros, you may be able to adapt the technique that Greg Maxey shows at http://gregmaxey.mvps.org/word_tip_pages/add_interactive_toggle_objects_to_document.html to replace the checkboxes with fields containing text that would be visible to a screen reader.
