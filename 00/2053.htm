When printing I use booklet format with two pages per A4 sheet which at the end yields B5 booklet. Everything is ok except page numbers. For consistency reasons, I need same page number on the left and on the right side. In other words, when the booklet is folded cover page does not have page number; first page of the original text should have page number 1, corresponding text on the right side page number 1 as well; and so on. As obviously different odd and even pages function does not reproduce different sections ("odd section" and "even section") the document is consisting of only one section (except different first page) and pages are numbered consequently.

 

Is this possible to achieve while retaining the different odd and even page format (which is essential for the consistency of headers and footer of odd pages on the one side, and even pages on the other side)? By applying only section breaks I would loose the advantage of the same headers and footers for odd and even pages.

 

I am using word 2010.

 

Thanks a lot! Every input is appreciated.

doug robbins - mvp office apps & services (word) say:

If you put a Section Break after the title page and set the page numbering in the second section to start at 1, you can use the following field construction for the Odd page header

{ = { PAGE } / 2 +0.5 }

and for the Even page header use

{ = { PAGE } / 2 }

You must use Ctrl+F9 to insert each pair of field delimiters { }

If you do not use a Section Break to separate the title page then you would use

{ = ( { PAGE } - 1 ) / 2 +0.5 } for the Even page header, and

{ = ( { PAGE } - 1 ) / 2 } for the Odd page header

In this case, you would not modify the starting page number.

 

suzanne s. barnhill say:
My guess, however, is that Word will not let you put the number 1 on a verso (even) page (it will insert a blank even page), so this can be best handled without a section break. The even page header can still be { = { PAGE } / 2 }, but for the odd header you'll need { = ({ PAGE } -1) /2 }. There will be no page number on the cover page, which will be actual page 1.
