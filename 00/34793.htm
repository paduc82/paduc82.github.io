bossdweeb say:

- Generally, the best way to get rid of Windows.old is to use Disk Cleanup. Click on the Clean up System Files option after it does the initial scan.

Do not delete Windows.old until you are certain a recent version update has not caused serious problems (1809 is the worst update yet

and people are discovering new problems almost daily).

- Copying is not a safe method of dealing with the Programs folders. Partly because the three Program folders are system protected

(linked to Windows) which can cause various issues such as some files not being copied, or having to stuff around with

permissions to do the copy.


Best practice when using a new HDD/SSD is to clone/image/migrate (different terms used by cloning software) your C:\ drive to the

new HDD/SSD. Cloning copies everything on C:\ exactly how it is on the original drive and keeps Windows, and anything on C:/

from getting confused, corrupted; borked.

Macrium Reflect and AOEMI Partition Assistant are two popular programs for this task.

Free Partition Manager - AOMEI Partition Assistant Standard

Macrium Software | Macrium Reflect Free

- An exception is if you install your software to another HDD/Partition. In this instance if you clone C:\ to a new drive you

can copy the software/game installs that are on another partition/HDD to the new HDD/SSD.

Just be sure to use the same drive letter on the drive you copy to and the software will have no problems from being copied

to a new drive because the file path will be retained.
