The problem is that the ModernUI (Metro) Apps use a completely different programming model than the traditional Win32 (Desktop Programs) use.

Since Windows 8x was first released the only way to completely close these apps was to either use the 'drag to the bottom of the screen' function, Press the Alt+F4 keyboard shortcut, or open Task Manager and use the End Task function to completely exit the app.

The current implementation of adding the Title Bar with the 'X' button to the Apps follows the previous behavior for these Apps, if you switch away from an App to another app or program, the App will automatically Suspend after 5 Seconds.

This behavior may be changed in the next version of Windows.

  • If the "Encryption Method" states something other than "Hardware Encryption", such as "AES-128" or "XTS AES-256", then BitLocker is using software-based encryption.

    BitLocker's default encryption method can be controlled with Group Policy settings. Configure these settings to force BitLocker to use software-based encryption by default. Once these policy settings have been changed, BitLocker needs to be disabled and re-enabled to re-encrypt the drive with software-based encryption (if not already using software-based encryption).

    arthur rui say:

    Hi James,

     

    Recovery Partition is the part of the drive to reset the machine back to how it was when you bought it. Windows creates a System Reserved partition when you install it on a clean disk. Windows doesn’t assign a drive letter to these partitions, so you’ll only see them when you use Disk Management or similar utility.
