Is it possible a macro to toggle automatic/text font color? - Microsoft Community

rxaverius say:

Is it possible a macro to toggle automatic/text font color?

I wonder if something that it is indeed possible with highlighting text is possible when a word appears at a certain color.

Imagine I color a word with the "red color" and I use a shortcut for that.

Is it possible to use the same shortcut to retrieve that word into the "automatic color" ?

I can actually do that for a highlighted word.

thanks

graham mayor say:

If you move the selection point to the end of the word, as requested, the original word is no longer in the selection, so the toggle works on the new selection point. You would have to return the cursor to the word you want to toggle. The macro cannot predict whether you would want to toggle the previous selection or the new location. The best you can achieve is to return the cursor to its start position:

Sub ToggleRedFont()
Dim orng As Range
    If Selection.Type = wdSelectionNormal Or _
        Selection.Type = wdSelectionIP Then
        Set orng = Selection.Range
        If Selection.Type = wdSelectionIP Then
            Selection.Words(1).Select
        End If
        
        If Selection.Font.ColorIndex = wdAuto Then
            Selection.Font.ColorIndex = wdRed
        Else
            Selection.Font.ColorIndex = wdAuto
        End If
        orng.Select
    Else
        MsgBox "First select the text to change.", , "Toggle Red Font"
    End If
End Sub

jay freedman say:

Highlighting doesn't work that way either -- if you want to highlight a whole word, you have to select the whole word first (most easily by double-clicking it) or drag the mouse cursor across it with the highlight tool.

Just for you, though, this version of the macro will select the whole word if you just single-click in the middle of it or to the left of the first character of it.

Sub ToggleRedFont()
    If Selection.Type = wdSelectionNormal Or _
        Selection.Type = wdSelectionIP Then
        
        If Selection.Type = wdSelectionIP Then
            Selection.Words(1).Select
        End If
        
        If Selection.Font.ColorIndex = wdAuto Then
            Selection.Font.ColorIndex = wdRed
        Else
            Selection.Font.ColorIndex = wdAuto
        End If
    Else
        MsgBox "First select the text to change.", , "Toggle Red Font"
    End If
End Sub

huni shen msft say:

Hi Calum Morrison,

 

App Passwords are designed for applications that do not natively support MFA. Instead of using a password that is set by the user, which is likely used in other locations, a random password is generated for us for a very specific application. MFA is enabled per user. This means that if a user has MFA-enabled, they won't be able to use a non-browser client until they create an app password.

 

When using app passwords, it's important to remember:

 

  1. App passwords are auto-generated and only entered once per app.
  2. There's a limit of 40 passwords per user. If you try to create one after that limit, you'll be prompted to delete an existing password before being allowed to create the new one.
  3. Use one app password per device, not per app. For example, create a single password for all the apps on your laptop, and then another single password for all the apps on your desktop.

 

If your app passwords aren't working properly, you can try to create a new app password if your admin allow MFA users to create App passwords for Office client apps.

 

For more detailed information, here is an article for your reference: https://docs.microsoft.com/en-us/azure/active-directory/user-help/multi-factor-authentication-end-user-app-passwords#create-and-delete-app-passwords-using-the-office-365-portal

 

Best Regards,

Huni
