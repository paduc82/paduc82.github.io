If when you type the text goes up as well as down on the page, you have your page centered vertically (center aligned).

Open the normal template and change this. See Normal Template in Microsoft Word - How to Open if you have trouble finding this template.

charleskenyon (ia) say:
Hi Mark,

I strongly urge you to read the series of short articles by Shauna Kelly before you do this.

Here is a macro that will change the normal style in the normal template to single space.

[Code]
Sub SingleSpaceNormal()
'
' SingleSpaceNormal Macro
' Charles Kenyon
' Read the series of articles by Shauna Kelly before using this!
' http://www.addbalance.com/usersguide/BasicConce...
'
'
Dim mContinue As VbMsgBoxResult
mContinue = MsgBox(prompt:="This changes defaults to single-space." & _
" It cannot be undone." & vbCr & vbCr & " Continue?", _
Title:="Warning! This cannot be undone!", _
Buttons:=vbYesNo)
If mContinue = vbNo Then Exit Sub
Application.ScreenUpdating = False
Application.NormalTemplate.OpenAsDocument

With Selection.ParagraphFormat
.SpaceBefore = 0
.SpaceBeforeAuto = False
.SpaceAfter = 0
.SpaceAfterAuto = False
.LineSpacingRule = wdLineSpaceSingle
End With
With ActiveDocument.Styles("Normal").ParagraphFormat
.SpaceBefore = 0
.SpaceBeforeAuto = False
.SpaceAfter = 0
.SpaceAfterAuto = False
