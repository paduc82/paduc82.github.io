bill manville say:

That function is counting the number of cells whose interior is a light blue colour.

If you want to count cells which contain the text in C1 and the colour in C2 you could modify the function to

Function CountByCellColorAndText(rng As Excel.Range, SearchText As String, rngColour As Range) As Long
    Dim cell As Excel.Range
    Dim CellCount As Long

    For Each cell In rng.Cells
        With cell
            If .Value = SearchText Then 
                If .Interior.Color = rngColour.Interior.Color Then
                    CellCount = CellCount + 1
                End If
            End If
        End With
    Next cell
    CountByCellColorAndText = CellCount
End Function

and you could call it by

=CountbyCellColorAndText(A1:A8,C1,C2)

Note that changing the colour in C2 will not automatically cause the formula to recalculate.  If you want to be able to have it recalculate on a change of colour add

  Application.Volatile
as an extra line in the function before the For... line and press F9 after changing the colour.
