Try the following. It might or might not resolve the issue. If it does not work then I don't have any other suggestions.

  1. Close the problem workbook (this first step is the essential)
  2. Using Windows Explorer, Open the Folder where the Excel file is saved.
  3. Select the menu item "View" at the top of the screen.
  4. Check the box against "Hidden items" (this allows normally hidden files to be visible)
  5. Look for a file that is prefixed with a tilde and dollar sign (~$) and the remainder of the name is same as the problem workbook.
  6. This file that is prefixed with ~$ is a temporary file that is created when a workbook is opened and it should delete when the workbook is closed. If the file exists after the workbook is closed then that can occur when the workbook is improperly closed and it can cause the workbook to be locked when re-opened.
  7. Delete the file that is prefixed with the tilde and dollar sign.
  8. Open the problem workbook and see if problem resolved.
