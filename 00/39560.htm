jienhowe say:

Upgrade to windows 10 from windows 8.1 free

Hi,

I had a windows 8.1 laptop and upgraded to windows 10.

I purchased a new windows 8.1 tablet 2-in-1, can i upgrade or transfer the license of windows 10 for free?

neilpzz say:

You can't transfer the licence as it was probably pre-installed and an OEM licence, those are tied to the device.

The free upgrade period finished in July 2016, so the official answer is no. However some have found a new device will activate after an upgrade. You can try but you may end up having to call support to ask to activate and they may say no.

jienhowe say:

Thanks for answering,

i factory reset a device upgraded to windows 10 for free from windows 8.1, it is now windows 8.1.
Do i get to upgrade it back to windows 10 and how?

The previous owner of the device is not me(or not my microsoft account)

neilpzz say:

Once a device has a digital licence for Windows 10 following the upgrade then the licence is for the life of the device. (this assumes the device was activated after the upgrade and the motherboard is the original)

To upgrade to Windows 10 again go to http://www.microsoft.com/en-us/software-download/windows10 and download the Media Creation Tool. This can create DVD/USB installation media, once created you can use this to upgrade. If you wish you can select to upgrade this PC now but the better option is probably to create the media then you will have it for the future.

If asked for a key select Skip, Do this later or I'm re-installing windows and proceed through the upgrade. A clean option would be to clean install Windows 10 but this will wipe away Windows 8.1 completely. Activation should complete once connected to the internet.

You can then use a MS account to sign-in. I would create a local account during the install and an MS account after you have verified the install is successful. The link between activation and the MS account is to help activation should there be a problem and it doesn't mean you need the same MS account to activate a device.
