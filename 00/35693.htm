No.

mark ole say:

See if you can remove the "read only" attributes of the hard drive. Please try the steps below:

  1. Open command prompt.
  2. Type in Diskpart.
  3. Then type in list volume.
  4. Select the volume number of the local disk drive. (ex. disk 2)
  5. Type in the command: attributes disk clear readonly
  6. Once done, close the command prompt.

Open the command prompt again and type the following commands and press Enter:

bootrec /RebuildBcd
bootrec /fixMbr
bootrec /fixboot

These commands will try to repair the Master Boot Record. After running the commands, see if you'll be able to boot the system in Windows.

If still you're unable to boot the system in Windows, run chskdsk /r. This will check the entire disk surface for bad sectors and will attempt to repair it finds any.

In case non of the troubleshooting steps worked, we suggest that you refer to this link on how get service for your Surface.
