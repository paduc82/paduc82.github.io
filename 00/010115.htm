Excel, Is there a way to blank a drop down menu cell if the preceding - Microsoft Community

sbxu say:

Excel, Is there a way to blank a drop down menu cell if the preceding cell is blank?

Is there a way to blank a drop down menu cell if the preceding cell is blank?

hansv mvp say:

Let's say that you want dropdowns in E1:E50, and that the source range for the dropdown is A1:A4.

Select E1:E50, then select Data Validation on the Data tab of the ribbon.

Select List from the Allow dropdown.

In the Source box, enter the formula =IF(D1="","",$A$1:$A$4)

D1 is the cell to the left of E1, the first cell in the selection; because the cell reference D1 is relative, Excel will adjust it automatically for E2, E3 etc.

hansv mvp say:

That's not what you asked. The dropdown list in E1 will be empty when you clear D1, as per your original request.

If you also want the value of E1 to be cleared if D1 is cleared, you need a bit of VBA code:

  • Right-click the sheet tab.
  • Select View Code from the context menu.
  • Copy the following code into the worksheet module:

Private Sub Worksheet_Change(ByVal Target As Range)
    If Not Intersect(Range("D1"), Target) Is Nothing Then
        If Range("D1").Value = "" Then
            Application.EnableEvents = False
            Range("E1").ClearContents
            Application.EnableEvents = True
        End If
    End If
End Sub

Uninstall all Office installations using the appropriate MS FixIt(s) from this article: http://support.microsoft.com/kb/290301
