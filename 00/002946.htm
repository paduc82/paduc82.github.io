Hi everyone,

So is there a way to insert these pdf files and have them locked to the top and bottom sections so that they won't move?

Thanks in advance for any help.

bob jones aka: cybertaz say:

Your understanding is correct.

Graphic objects in a Word document are always 'connected' to the text in 1 of 2 ways;

  • In Line with Text, where the object is treated as though it were a character in a paragraph,

or

  • Floating, i.e. having some form of Text Wrapping applied to it

In Line graphics are contained in the paragraph into which they are inserted. Floating graphics, however, are Anchored to a paragraph, but can be positioned anywhere on the 'page' containing that paragraph. By default, when the graphic is inserted, converted to Floating or moved to a different location it becomes anchored to the closest paragraph. Using the Lock Anchor setting causes the object  to remain anchored to the same paragraph when the object is repositioned -- IOW, the setting pertains to locking the anchor [to a specific paragraph], not the position of the object. It also prevents the object from being moved to a different 'page' than the one containing the paragraph to which it is anchored.
