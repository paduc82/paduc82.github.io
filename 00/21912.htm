andreas killer say:

So far, I can only print was is on the screen, but there is additional information/timeline if you scroll. 

For this special template the answers is yes (you can only print what you can see) and no, there is no additional information/timeline.

What you see on screen is a chart, and it shows only a part of the data.

When you use the scrollbar below the chart, you think that something that just not visible moves into the view, but in fact a value of a cell is changed.

Some short technical details:
The name of the cell that is modified by the scrollbar is "WindowOffset" and the cell is D26 in sheet "calcs". You can't see that sheet, because it's hidden. In sheet "calcs" are many formulas, also within the name manager. Those formulas recalculates the data for the chart, from which point (WindowOffset) and how many days (WindowDays) should be shown.

At the end, all that means it is not possible to print the whole timeline with this template.

alpac1t say:

Thanks for all the excellent suggestions and commentary, y'all. I had many images linked to a user\documents directory, and then moved all the images to SkyDrive. By saving Word file back to 2003 format, doing a batch replace of "user\documents" with "user\SkyDrive," and then saving the Word file to docx format but with "compatibility mode" set, I accomplished everything needed. All image references to old folder have now been updated/relinked to new folder, all with just a few minutes work and NO handling of individual images.

Thanks much!

mike plant say:

thank you, this is at least a good workaround to find the picture's path and file name in an easy way.

Another way is to work in compatibility mode by:

  1. Save my docx (2010) document to doc.
  2. Save the doc back to docx while activating the option "Maintain compatibility with previous versions of Word".

This way you still have INCLUDEPICTURE fields for all linked pictures.
