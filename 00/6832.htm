Public Function SplitCSVRecord( _
      ByVal Record As String, _
      Optional ByVal QuotingCharacter = """" _
   ) As Variant

' Split a CSV (comma separated values) record into an array of field values and
' return those fields as a variant array.
'
' Record - Valid CSV record. Fields are delimited by commas. Fields can be
'   enclosed in double quotes. Fields must be inclosed in double quotes if they
'   contain a double quote or a comma. Double quotes contained in a field are
'   escaped by adding a second double quote.
'
' QuotingCharacter - The character used to quote fields containing the quoting
'   character or a comma. Optional, If omitted then a double quote is assumed.

    Static RegExp As Object
    Dim Pattern As String
    Dim Fields As Variant
    Dim Index As Long

    If RegExp Is Nothing Then Set RegExp = CreateObject("VBScript.RegExp")
    RegExp.IgnoreCase = True
    RegExp.Global = True
    Pattern = ",(?=([^""]*""[^""]*"")*(?![^""]*""))"
    If QuotingCharacter <> """" Then Pattern = Replace(Pattern, """", QuotingCharacter)
    RegExp.Pattern = Pattern
    Fields = Split(RegExp.Replace(Record, vbTab), vbTab)
    For Index = LBound(Fields) To UBound(Fields)
        If Left(Fields(Index), 1) = QuotingCharacter And Len(Fields(Index)) > 1 Then
            Fields(Index) = Replace(Mid(Fields(Index), 2, Len(Fields(Index)) - 2), QuotingCharacter & QuotingCharacter, QuotingCharacter)
        End If
    Next Index
    SplitCSVRecord = Fields

End Function

a. user say:

Hi Griffonman,

I understand your concern. Let me help you with the issue.

If the first Network adaptor driver fails to properly install, Windows does not start the WLAN service and even though the subsequent Wireless network adaptor drivers install and goes through fine, network is not available to the system. We must check if there is any failed Wireless network adaptor and disable it in order to not conflict with the actual drivers. Follow the steps below.

  1. Right click on the Windows logo and select Device Manager.
  2. Check if there are any Unknown device or network controller with a yellow exclamation.
  3. Right click and select properties.
  4. Select driver tab and hit disable.
  5. Apply changes.
  6. Back to Device manager and expand Network Adaptors.
  7. Right click on the wireless driver and select uninstall.
  8. Apply changes and hit OK.
  9. Restart the computer which will load the generic drivers.
  10. Check if the issue persists. 

Hope it helps. Please reply to us with the status so that we can help you further.
