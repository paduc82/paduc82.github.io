a. user say:

Hi Felipe,

I will be glad to help you with the issue you have with the computer. I understand the frustration when things don‚Äôt work the way it should.

Fast startup is an innovative feature introduced in Windows 8.

The possible causes fast boot failure would be

-hardware failure;

-power loss;

-bad drivers;

-firmware unknown issues;

-simply opening boot menu to boot from another device.

To test if it is truly a fast startup problem open Event logs and look for critical reports.

There should be a matching error report for the critical reports as well.

Steps to repair corrupted fast startup image:

1. In Control Panel open 'Power Options', go to 'Choose what the power button does' , click / tap "Change settings that are not currently available" if you are not allowed to disable fast startup which is your goal.

Save changes and close Control Panel;

2. Open an elevated command prompt and type:

powercfg /h off

Also type

dir c:\ /a /p

to check if the hibernation was turned off. This will list all the files and folders on your windows partition root. You need to look for hiberfil.sys. If it's missing then the hibernation was successfully turned off.

3. Now lets put everything back. Re-enable hibernation via

powercfg /h on

Check for hibernation file presence (hiberfil.sys). It should be there now:

dir c:\ /a /p

Finally re-enable fast startup via Control Panel. This should sort it out until the next corruption incident.

Hope this information is helpful. Please post in case you have any further issues. I will be glad to help you further.

Thank you.
