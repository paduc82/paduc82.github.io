macropod say:
Word links to Excel files use an absolute path. Consequently, if you move the Excel file to a different folder the link will break. Simply copying the Word file to another folder won't affect the link - it'll still point to the original Excel file. After moving the files, you therefore need to go to File>Edit Links to change the source. Adding the Edit Links button to Word's QAT makes this a bit quicker.

ryanlv_msft1519 say:

Hi Keith,

The root site collection can be recreated as a publishing site using the workaround below. Please strictly follow the steps and test this before taking actions on your own tenant.

1. Go to SharePoint Admin Center > select the root site collection > Delete.

2. Use SharePoint Online Management Shell to permanent remove the root site collection from the recycle bin. To do this, follow the steps below:
    a) Enter Connect-SPOService, input the SharePoint Admin Center Url https://<tenant>-admin.sharepoint.com/ and your global admin account credential
    b) Enter Remove-SPODeletedSite -Identity https://<tenant>.sharepoint.com/ (your root site collection Url), input Y to confirm removal.
    To set up the SharePoint Online Management Shell environment, please check the article here https://support.office.com/en-us/article/Set-up-the-SharePoint-Online-Management-Shell-environment-7b931221-63e2-45cc-9ebc-30e042f17e2c.

3. Go back to SharePoint Admin Center, check if the root site collection is removed from recycle bin.

4. In SharePoint Admin Center, create new site collection by clicking New > Private Site Collection.

5. Enter root site collection Url https://<tenant>.sharepoint.com/ in Web Site Address and other information. Choose <select template later> in Custom as a template. Click OK.

6. Wait until new root site collection is provisioned, access it via Url.

7. Select Publishing Portal template as the template in the first page and follow the instructions next to complete the settings.

Note: Do not directly use Publishing Portal template to recreate root site collection in SharePoint Admin Center. The site collection will use Enterprise Wiki template instead of Publishing Portal template.
Reference: https://support.microsoft.com/en-us/kb/2674732
