Hopefully it won't get to this point, but if your drive truly is empty, and you don't have a backup, it may be possible to retrieve the contents but that requires special equipment and knowledge and it's very expensive.

Windows Media Player cannot play the file. The Player might not support the file type or might not support the codec that was used to compress the file. You might encounter this error message for one of the following reasons:

Dual Boot Setup License

If you install Windows 7 on the same partition that XP or Vista is on you can buy the Upgrade. Windows 7 RC does not qualify.
If you install Windows 7 on a different partition and dual boot with XP or Vista you need the Full version.
