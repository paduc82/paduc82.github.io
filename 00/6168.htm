john sr wilson say:

 Raif

First you are using legacy code for 2000. You should use the timeline object for all versions after 2000.

See if this is close:


Dim oshp As Shape
Dim curS As Slide
Dim oeff As Effect
Dim oeff2 As Effect

Set oshp = curS.Shapes.AddTextbox(1, Top:=iTopMarg, _
Left:=tbLeft, _
Width:=tbWidth, _
Height:=tbHeight)

oshp.TextFrame.TextRange.Text = sSlideText

'Animate shape
Set oeff = curS.TimeLine.MainSequence.AddEffect(Shape:=oshp, _
effectid:=msoAnimEffectBlinds, _
Level:=msoAnimateLevelNone, _
Trigger:=msoAnimTriggerWithPrevious)

'animate text
Set oeff2 = curS.TimeLine.MainSequence.AddEffect(Shape:=oshp, _
effectid:=msoAnimEffectBlinds, _
Level:=msoAnimateTextByFirstLevel, _
Trigger:=msoAnimTriggerOnPageClick)

'change first to with previous
curS.TimeLine.MainSequence(oeff2.Index).Timing.TriggerType = msoAnimTriggerWithPrevious

Try this...

Use 2 cells to hold both the lower and upper weight limits.

Criteria:

  • A1 = cow
  • A2 = lower weight limit = 1000
  • A3 = upper weight limit = 1200

Enter this formula in A4. This will return the count of records that meet the criteria:

=COUNTIFS(C2:C5,A1,D2:D5,">="&A2,D2:D5,"<="&A3)

Enter this array formula** in E2. This will extract the results.

=IF(ROWS(E$2:E2)>A$4,"",INDEX(B:B,SMALL(IF(C$2:C$5=A$1,IF(D$2:D$5>=A$2,IF(D$2:D$5<=A$3,ROW(B$2:B$5)))),ROWS(E$2:E2))))

** array formulas need to be entered using the key combination of CTRL,SHIFT,ENTER (not just ENTER). Hold down both the CTRL key and the SHIFT key then hit ENTER.

Copy down until you get blanks.

Note that this may be "slow" to calculate if you have 1000's of rows of data.
