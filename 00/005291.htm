Hi JZ,

Sorry for the miss. Another thing to try involves observing the Registry. Use caution in that process. Do not modify any more than I specify.

Go to regedit by selecting "start" and type "regedit". Select "Run as Administrator".

On the Registry Editor page, navigate through the Registry by selecting the >

before each of the following:

>Hkey_Local_Machine>System>CurrentControlSet>Services and select

SecurityHealthService.  In the right pane find the Key labeled "Start". Observe

the Reg_Dword value for the key. If it is set to "4", modify the value to "3".

Do that by right clicking "start" and selecting Modify. Change the highlighted

value of "4" to "3" and select "OK".

If you found the value to be "4", we are on the right path. In either case, to keep

the Registry neat, back out of it by selecting the > in the reverse order, until you

reach HKLM. From there select "file" and select "exit" to leave regedit.

Select "start" and type "services". Select "Run as Administrator". When the "Services" screen appears, scroll down to "Security Center". It should be running

and set for Automatic (Delayed Start). If it is not, double click it, choose

Automatic (Delayed Start). Select "Start" select "OK".

While still on the "services" panel, scroll down to the "Windows Defender" services and ensure that they are all running. The "Antivirus and Firewall" should be set for "Automatic" start.

If this process has been unsuccessful, you might try to use Command Prompt

(admin) to enable Windows Defender. Enter Command Prompt (admin) by

selecting "start" and type "cmd". Select "Run as Administrator" and you will be taken to the Command Prompt screen.  Enter the following commands, one at

a time, and press enter after each.

CD\     <enter>

CD Program Files\Windows Defender     <enter>

MpCmdRun -wdenable     <enter>

Exit     <enter>

Check Windows Defender to see if is now enabled.

Good Luck,  Glen

rich michaels - mvp (greatcirclelearning.com) say:
Also in earleir versions of word it was fairly simple to insert a powerpoint slide into a word document. Now it doesn't seem to be possible, can it still be done and if so how?.

While in PowerPoint use either the Normal View with the Outline Panel showing and the Slides button selected, so you see thumbnails of each slide, or Slide Sorter view.  You can Option + Click (a.k.a. right click) on any thumbnail, select Copy and then Paste the slide into your Word document, where there it will be treated as a picture.

Unfortunately with the above method you can only copy/paste one slide at a time.  It teases you into believing that you will be able to select and then paste multiples but actually only the select multiples is implemented.

Another option is to go to Print and choose one of the Handout options.  This will build a Word document of the slide images and you can then move them around from there.

Finally, your last option is to choose File > Save as Pictures and this will convert each slide into an image file and place them into a folder.  From there you can Insert the pictures into your Word document.

Hope this helps

jared cheng msft say:

Hi Karl,

 

Thanks for your information in PM.

 

Since OneNote has its unique sync mechanism, it's not suggested to copy OneNote Files directly from SharePoint library to local place, nor from local place to SharePoint library. Unlike other app files like Word, Excel which can work this way.

 

It's suggested that OneNote notebooks should always be synced and moved by the OneNote application, and not by file based synchronization systems such as a client application (such as the OneDrive or DropBox local client) or offline folders. For detailes, please refer to: OneNote syncing best practices.

 

Therefore, I suggest you try the following methods to sync OneNote files:

#1. Create local OneNote Notebooks and sync it from local to SharePoint library.

a. Open OneNote > New > Computer > enter name > Create Notebook. Then the new notebook will be default saved in C:\Users\Administrator\Documents\OneNote Notebooks.

b. File > Share > SharePoint library > SharePoint folder > in the pop out window, click the specific library > Move. Then the local OneNote file will be moved to SharePoint library.

#2. Create online OneNote files and open it in OneNote desktop app. (make sure the file is created online directly or used method 1 to move to SharePoint library)

a. OneNote > File > Open > under Recent, click the SharePoint library > click folder > navigate to the SharePoint OneDrive library > Open

Please check if you can use the methods above to open the OneNote file on OneNote 2016 properly.

 

Regards,

Jared
