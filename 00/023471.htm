merlijndonk say:
Hi,

I am a Microsoft Community expert here to help you.

You can try something like this:

Sub CopyToMaster()
ShtCount = ActiveWorkbook.Sheets.Count
For i = 2 To ShtCount
Worksheets(i).Activate
LastRow = ActiveSheet.Cells(Rows.Count, "D").End(xlUp).Row
Range("A2:Z1000" & LastRow).Select
Selection.Copy
Sheets("Master").Activate
LastRow = ActiveSheet.Cells(Rows.Count, "D").End(xlUp).Select
'Required after first paste to shift active cell down one
Do While Not IsEmpty(ActiveCell)
ActiveCell.Offset(1, 0).Select
Loop
ActiveCell.Offset(0, -3).Select
Selection.PasteSpecial
Next i
End Sub

alexander torbet say:
Hello tkhaw,

You can use VBA to do that.

The code below copies all values in "Sheet31". You only need to add the column names after the macro has finished.

To use the macro, simply open the editor (Alt + F11) and then copy paste de code below then launch it. 

Sub addsheet()
    Sheets.Add(After:=Sheets(Sheets.Count)).Name = "Sheet31"
    
    For Each sh In Sheets
    
    sh.Select
    
        If ActiveSheet.Name = "Sheet31" Then Exit Sub
        
            LastColumnNumber = Cells.SpecialCells(xlCellTypeLastCell).Column
            LastRowNumber = Cells.SpecialCells(xlCellTypeLastCell).Row
            
            Range(Cells(2, 1), Cells(LastRowNumber, LastColumnNumber)).Copy
            
            Sheets("Sheet31").Select
            
            LastRowNumber2 = Cells.SpecialCells(xlCellTypeLastCell).Row
            
            Cells(LastRowNumber2, 1).Select
            ActiveSheet.Paste
        
        
    Next sh
End Sub


Hope this helps,

garym123 say:

The phone does not have an easy way to make a playlist.

Doing it on a PC is easy and provides a way to manage the playlist as well.

 

Start by making a playlist using the phone. Play a single song on the phone. Tap the bold song name below the album art, tap the save icon at the bottom of the page, name the playlist. Pause the player.

 

Connect the phone to the PC. In windows explorer find the music folder on the phone. Open the playlist folder in music and right click the playlist you just created and select properties, recources tab. The recources tab has different names depending on the operating system. It will have a window.  Drag and drop songs that are on the phone into the window to add songs to the playlist. Using the window you can delete songs from the playlist and arrange the play order.

Explaining it takes a lot more than actually doing it. Once you have it figured out it is very easy. Remember, the songs you drag and drop have to be on the phone.

You will need to move to an array formula to pick up the 2nd, 3rd and 4th MAX() match, and that can be tempered with an IFERROR().

     

The standard formula in F2 is the same as before,
=INDEX(A$1:D$1,MATCH(MAX(A2:D2),A2:D2,FALSE))
The array formula in G2 is,
=IFERROR(INDEX($A$1:$D$1,SMALL(IF($A2:$D2=MAX($A2:$D2),COLUMN($A:$D)),COLUMNS($A:B))),"")
... which requires Ctrl+Shift+Enterâ†µ rather than just Enterâ†µ. Once entered properly, it can be filled right to I2, then F2:I2 can be filled down as necessary. If your actual report requires more than four columns, the array formula has been written so that it can be filled further to the right to cover more columns of data but internal cell range references will have to be adjusted.

I used two conditional formatting rules based upon formulas to highlight the MAX() value(s) in each row. The Format values where this formula is true: for a single MAX() value is,
=AND(ISNUMBER(A2),A2=MAX($A2:$D2),COUNTIF($A2:$D2,A2)=1)
... and for multiple MAX() values,
=AND(ISNUMBER(A2),A2=MAX($A2:$D2),COUNTIF($A2:$D2,A2)>1)
I chose two rules in order that a different highlight could be applied to single MAX() values and multiple MAX() values. I doubt whether you will have anywhere the number of duplicates I created in my sample, but multiples should probably be visual distinguishable from singles.

pamcaswell say: It's a two-step process.   First make the paragraph bold. Then select the paragraph mark only and press Ctrl+b, which will unbold the number only.

Added later:

Or  in the multilevel list dialog for level 2, you could set the font to regular.  After that, making the text bold will not also make the number bold.


Pam

pamcaswell say: The simplest way to get numbered lists without errors is to use one of the built-in lists. 

---You only need to  create a list from scratch if you have customizations (say, A, A.1, A.1.1, etc, or you prefer spaces instead of tabs after the numbers, or you need  to use one of the longer number formats  like 00001,00002, 00004).  From scratch, it's way too easy to  get the number fields twisted, or forget the restart afters, ---
