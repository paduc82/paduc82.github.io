ron sommer mvp 2009-2017 say:

If you have a @external.com email address, all emails sent to that address will go to the external.com server.  Emails cannot go to two servers.

You add the @external.com address to your Microsoft account as an alias or create a new Microsoft account using the @external.com address.  Any emails sent to that address will still go to the @external.com server.

When you send using the external alias address, you will get on behalf of unless you use an external SMTP server.  My guess is that the outlook_<some_sort_of_alpha-numberic_id>@outlook.com address will get replies back to your Microsoft account.

The primary alias determines what address is in From when you compose (new or reply) an email.

Make your @outlook.com address as the primary alias so it will be used in the From field.

When you are using a browser, the address showing in From when you compose and send will be the address that the recipient sees.

If you send using an email program (Outlook 2007), the recipient will see the Default "From" address from your account settings.  Click the Cog icon > Options > Your email accounts > Default "From" address.

shane devenshire say:

Here is how you used to do it and it still works in 2007:

Choose Home, Number Format, More Number Formats, Custom, and enter the following on the Type line

[Blue]#,##0;[Red]-#,##0

You can now do this with conditional formatting also, suppose your numbers are in F2:F6:


In 2007:
1. Highlight all the cells on the rows you want formatted
2. Choose Home, Conditional Formatting, Manage Rules, New Rule
3. Choose Use a formula to determine which cell to format
4. In the Format values where this formula is true enter the following
formula:

=$F2<0

5. Click the Format button and choose a format. (Red Color on the Font tab)
6. Click OK twice
7. New Rule
8. Choose Use a formula to determine which cell to format
9. In the Format values where this formula is true enter the following
formula:

=$F2>0

10. Click the Format button and choose a format. (Blue Color on the Font tab)
11. Click OK tree times.

If you want into situation where you icons do not fit into taskbar, and you have to scroll them.. Then this feature is for you.

How does it work.

You drag icon on taskbar, and drop it over another icon.

Then those two icons are automatically grouped together. 

If you click on the group, new menu will unfold just above where the group is on taskbar, showing all grouped icons.

You can group any kind of shortcuts, to applications, folders (would be so cool, to have single place where I can keep shortcuts to all my frequently used folders).

Multiple taskbars.

Make all the functionality of taskbar widgets. Start Menu, New icons (search and multi desktop), tray, time and date, even application shortcuts.

Of course if everything is widget, there should API for users to create custom ones.
