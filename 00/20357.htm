somasundaram pandiaraj say:

The option to create Concordance or Automark file is still also available in Word 2007.

 

 

Try the following steps to create the Automark file.

 

1)     In a new document, insert a two-column table.

2)     In the left column, type the words in your document that you want to mark for indexing.

                (Note that these entries are case sensitive)

3)     In the right column, type the corresponding text that you want to appear in the index.

4)     Save and close this document.

5)     Go to References > Insert Index; click the AutoMark button in the dialog that opens.

6)     Select the file you created in steps 1 through 4 and click Open.

 

Word will now automatically mark the words in your document.

sanwin2 say:

Defaulting to Free on all day events is by design.  I don't believe there is any way to change this default action.  I think it is because most people use all day event to record birthdays, anniversarys etc.  and there is an expectation that meetings will not normally last 24 hours.

kevin mcnamee say:


> Defaulting to Free on all day events is by design.


Yes, but the user is not creating an event to begin with. The user starts by choosing "New Item" -> "Meeting" because they want to create a meeting. Then rather than having to fill in the start and end times, the user is helpfully provided with an "All day event" checkbox because they are going to be hosting a workshop or a training course or whatever. This time should be marked as "Busy"

> there is an expectation that meetings will not normally last 24 hours.

This depends on the definition of "All day". As a product, Outlook Professional is intended for company users and as such "All day" would refer to the user's working hours and not literally the 24 hour time period. If someone says they were in a meeting (event?) all day, then it means they spent the whole working day in a meeting, nothing else. In all my years using Outlook professionally, that is how people who book meetings use this functionality.

So, what I am arguing for is that the default "Show As" should be set to "Busy" for all day meetings. The main reason for this is that "Free" will not show up in other people's calenders (unless all the meeting details are exposed (not an option)) which causes a lot of pain.
