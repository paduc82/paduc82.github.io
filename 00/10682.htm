Backing up your important Outlook.com email data

1. Messages were manually deleted either due to accidental deletion or were moved or deleted due to your account filter settings.

macropod say:

Office 2000 is not compatible with Windows 7 or any later Windows OS. Although some people have been able to get it running on Windows 7, there are components required for a fully-functional Office installation that neither Windows 7 nor any later Windows OS provides. Given how old Office 2000 is, plus the fact it hasn't been supported for years (leaving security flaws unfixed and thereby potentially compromising the rest of the OS), you really upgrade to a currently-supported version.

As for Windows 10, if you've upgraded from Windows 7 in the last month, you should still be able to roll back to the Windows 7 installation.

Try this formula in cell D2:

=IF(AND(B2<>"Owner",SUMPRODUCT(--(A$2:A$10000=A2),--(C$2:C$10000=C2),--(B$2:B$10000="Owner"))>0),"DUPLICATE","")

Copy cell D2 down as far as you have data in columns A to C. Adjust 10000 to suit the size of your data.

In column D, the word "DUPLICATE" will now appear where the condition for removal is fulfilled.

hansv mvp say:

Select A2:C10000 (or wherever the data end). I'll assume that the active cell within the selection is in row 2.

On the Home tab of the ribbon, click Conditional Formatting > New Rule...

Click 'Use a formula to determine which cells to format'.

Enter the formula:

 

=AND($B2<>"Owner",COUNTIFS($A$2:$A$10000,$A2,$B$2:$B$10000,"Owner",$C$2:$C$10000,$C2)>0)

 

Click Format...

Activate the Fill tab.

Select a highlight color, e.g. red.

Click OK twice.
