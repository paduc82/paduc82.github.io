a. user say:

You would have to be really lucky to be able to take a HDD/SSD from a working system and plug it into a different system and expect it to work - unless the motherboards are exactly the same.  Expectations like that exceed reality.

When Windows is installed and after the installation you really need to visit the motherboard manufacturers WWW site and download/install on the HDD/SSD the drivers (chipset, video, audio, network, etc.) specific to that motherboard.

If you take that HDD/SSD and install it in a different system, unless the motherboards are exactly the same it is unlikely to work.

If you reinstalled Windows on your SSD it is extremely unlikely that you will be able to get online since the network drivers that come on a Windows 7 SP1 installation CD/DVD are generic.  To get online you need network drivers specific for that hardware.

After the clean install of Windows on your SSD you need to from a different working system visit the motherboard manufacturers WWW site and download the drivers (chipset, video, audio, network, etc.), copy them to something like a USB thumb drive and carry that USB thumb drive with the drivers to the new installation (using Sneakernet) and install the drivers on the new system,

You might be able to just install the network drivers so you can at least get online, then visit the motherboard manufacturers WWW site and get the rest of the things you need to get audio, USB and other things working - those things you need are not going to be on the Windows 7 SP1 installation media.

It is not a problem with Windows - this is the way things work.

The Windows ISO download site only accepts Retail and System Builder 25 character Product Keys and if you try to use an OEM product key that comes with a store bought system (Dell, HP, Lenovo, etc.) you will not be downloading any ISO file.

If you need a Windows 7 SP1 installation ISO (everybody should have one) you can click my profile link and send me an email which should include the Windows Edition (Home, Pro, Ultimate, etc.) and Architecture (32 or 64-bit) of your desire and I can probably hook you up with that.

For the past 5 years Microsoft made it quite easy for users of the Windows 7 operating system to obtain reinstall media. If you lost your retail installation disc or recovery media; either it was damaged or faulty; you could download a copy of the edition you have a license for from an affiliate website named Digital River. Most recently, Microsoft ended availability of reinstall media which you could download as a .ISO file from Digital River, which is a digital replica of a optical disc.
