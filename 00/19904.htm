Microsoft Word will let you resize the image up or down without any problem. However, something happens to an image when you try to make it larger than the original size. As in any graphic design application, Word has to guess how best to display a larger image by choosing the correct pixel.

brian tillman [outlook mvp 2007-2019] say:

I moved your message to the Outlook.com "Email" forum because, as Bruce Hagen says, it does not appear you're referring to Windows Live Mail, but the web-based Outlook.com mail (formerly Hotmail).

 

If you want to delete everything in a folder, right-click that folder on the left-hand side of the windows and choose "Empty folder" that should appear in the pop-up context menu.

 

Alternatively, check the box to the left of the top-most message in the folder, scroll down to the bottom-most message and while holding Shift, check that one.  All the intervening messages shold become checked and in the Reading Pane should appear a link that says "Select everything".  Click that link and then click Delete in the Menu bar.

I figure it out: For a reason that I cannot explain there was a recurrent appointment "hidden" which I could not see when using the Outlook application.
