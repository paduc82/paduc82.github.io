nooooooooooooooooooooooooo say:

MS Word 2010- Templates. How to get text placeholders. ex: '[Your Name]'

Trying to make a template where people can just click on things and replace the placeholder text.  Can't find it in word 2010.  Will this be complatible with 2003/97?
suzanne s. barnhill say:

My preference for doing this in all versions of Word is to use MacroButton NoMacro fields. The instructions at http://word.mvps.org/FAQs/TblsFldsFms/UsingMacroButton.htm apply generally to all versions. If you want to insert this through the Field dialog,  you'll find it at Insert | Text | Quick Parts | Field, but it is much more efficient just to type the field syntax, select it, and press Ctrl+F9 to convert it to a field.

So, for example, you'd type:
     MacroButton NoMacro [Click here and type name]

Then select that entire string and press Ctrl+F9, which produces:
     { MacroButton NoMacro [Click here and type name] }

Click on that and press F9 or Alt+F9 to update/toggle the field, and you'll just see:
     [Click here and type name]

When you click on that, the entire field is selected, so anything you type there replaces it. This is backward-compatible with all previous versions.

SELECT AccountName, OpeningBalance,

OpeningBalance - NZ(SUM(Payment),0) As CurrentBalance
FROM Accounts LEFT JOIN Payments
ON Accounts.AccountID = Payments.AccountID
GROUP BY AccountName, OpeningBalance;

About all you can do is to create a separate Normal.dotm template under a different name, and use it to overwrite the company template every time you start to use Word. To do that:

  • Shut down Word.
  • Go to %appdata%\Microsoft\Templates and rename the company Normal.dotm to something else, or move it to a different folder.
  • Restart Word. The new blank document will be based on the program's internal defaults. Make whatever customizations you need, modifying styles and so forth. Use the option in the Modify Style dialog for "New documents based on this template" to have the changes applied to the Normal template. If there are styles or macros in the company template that you want in your version, use the Organizer.
  • Close Word, which will cause the new Normal.dotm file to be written.
  • Make a copy of this new Normal.dotm and store it in a different folder, where it won't be touched by the company's interference.
  • Each time you're about to start a Word session, first copy your private Normal.dotm to the %appdata%\Microsoft\Templates folder, overwriting the file that's there. This can be done with a batch file on your desktop.
