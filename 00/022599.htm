joel ivory johnson say: It's recommended that you run the 32-bit version of the Office applications even if you have a 64-bit computer. With the exception of unusually large Excel documents there you won't encounter any benefits or performance enhancements from running the 64-bit versions of the applications. 64-bit applications won't integrate with 32-bit components and in the case of Outlook 64-bit it will not integrate with some of the 32-bit Windows Mobile related components.  So it cannot directly sync with Windows Mobile. 

Your choices would be to either uninstall Outlook 64-bit and install Outlook 32-bit (You don't need to uninstall the other office applications) or you have the choice to use an online account as an intermediate syncing point. For example, if you sync your Outlook installation with a Hotmail or Gmail account and also sync your phone with the same account then any time you make a change to your contacts or calendar on one device it will automatically sync to the other.
