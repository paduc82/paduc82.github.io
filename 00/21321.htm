[Original title: Word 2010]

suzanne s. barnhill say:

With Word in Insert mode, when you start typing in the space that you have, I assume, created with empty paragraphs (by pressing Enter multiple times), you are adding text that pushes the existing empty paragraphs down. There are two possible approaches, either of which will work better.

  1. Add only a single empty paragraph between the tables. Text inserted there will make room for itself. If it exceeds the space you intended, it will push the bottom table to the next page, but at least the space between the tables will be filled with actual text and not empty paragraphs.
  2. Instead of splitting the tables with normal text, make one single-column row to allow for the text. You can give it an exact row height so that the space will remain constant (any excess text entered in the cell will disappear).
