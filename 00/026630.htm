hansv mvp say:

Formulas that are enclosed in { } are so-called array formulas. To turn a formula into an array formula, confirm it with Ctrl+Shift+Enter instead of just enter.

For example, the following array formula, confirmed with Ctrl+Shift+Enter, will return the number of distinct entries in A1:A6:

 

=SUM(1/COUNTIF(A1:A6,A1:A6))
