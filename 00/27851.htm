johnlynch_742 say:

Page split into jpeg (top half) and columns (bottom half)

I want to create a document in Word 2010 in which each page consists of two halves -- the top half will contain a jpeg (different on each page) and the bottom half will have two columns of text. How can I do this?
jay freedman say:
You have essentially two choices.

One way is to create a 2x2 table on each page, and merge the top two cells into one. In the Table Properties dialog, set the row heights to slightly less than half of [the height of the page minus the heights of the header and footer]. Put the picture in the top cell and the text in the two bottom cells. If the text needs to flow from one column to the other in this method, you can put a text box in each bottom cell and then link the left text box to the right text box (click the Create Link button in the Text group of the Drawing Tools tab). The entire table can be stored as a building block so you can insert one on each page with just a couple of clicks.

The second method is to insert a continuous section break between the picture and the text, and a Next Page section break at the end of the text. With the cursor in the text area, click the Columns button in the Page Setup group of the Page Layout tab and choose two columns. Because of the section breaks, that won't affect the picture areas before or after the text. You'll have to repeat this setup on each page as you get to it.

While Sheen V. Is correct, there is no such thing as AutoSave (It's called Auto Recover).

Word does have a feature that can be turned on in Preferences called Save Backup (it's in the Save Preference.

How it works Is that each time you hit "Save" or "SaveAs...", it creates a "Backup file inside a Folder placed in the same location as the original file. It depends on you taking the time to hit Save or "SaveAs..." to work. The only reason it is a preference choice instead of being required is that it takes extra space and most uninformed people find it a nuisance. If The document is important to you, turn it on.

The way Auto Recover works (if and when it works) is that it retains in cache memory all typing from the point of the last save until the point of Crash. If auto recover is setup right the next time you open which should be as soon as the crash report is sent, you open the document. It should ask if your wish to recover. If it does, do so it will read that saved file from the cache setup to retain it. Then immediately save.

To set up Auto-Recover:

  1. Open Word menu choose Preferences.
  2. Locate and open File Locations

Make AutoRecover is set if Blank the AutoRecover files will automatically be dumped in the trash.

Install Office 2013 to D: drive? Error Code: 30179-4

I have a Windows 10 computer with a small SSD C: drive and want to install Office 2013 Professional onto my large D: drive.  My C: currently has 2.8GB free and the D: has 370GB free.  When starting the installation I get the error you must have 3419MB of space to install.  Then there is nothing else I can do.  The only options it gives is Go Online For Additional Help Error Code: 30179-4 or Close?  So here I am looking for support.

diane poremsky mvp (slipstick.com) say:
Did you empty the recycle bin and run disk cleanup?

Installing click to run on another drive is not supported. You can try change the programs folders in the registry but it is not supported. Change the path for program files & program files x86 at HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion

bobstrong1 say:

650 no longer puts txt msg notification on lock screen - Microsoft Community

tirgoman say:

650 no longer puts txt msg notification on lock screen

Windows 650 with 10 has stopped putting Text msg notifications on lock screen. Did originally and then stopped about a week ago. Checked settings and all seems correct. Ideas?
stranger_hanyo say:

Make sure phone and apps are up-to-date. Now try these-

1. Open settings>personalization>lock screen and make sure messaging app is set to display notifications. If its already there, re-add it again.

2. Open settings>system>battery saver and make sure messaging app is allowed to run when battery saver is enabled.

3. Open settings>privacy>background apps and make sure messaging app is allowed to run in the background.

4. Open settings>system>notifications and check if messaging app notifications are enabled and allowed to display on lock screen.
