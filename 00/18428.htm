How to use  pictures from Apple's Photos Application in Office 2011:

  1. In Finder create a blank Folder on Desktop. (Name it something easy to remember).

  2. Next, Open Photos.

  4. Click on First Picture.

  14. Save file in Folder You created.

  15. After you exported your files, open your Word Document
    Go to Insert Menu
    Select From File

Repeat steps for each picture you want to use in your document.  Although I've broken down into a lot of baby steps, after getting the hang of doing the Procedure, doesn't really take too long.
