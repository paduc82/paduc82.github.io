Restting network settings - Microsoft Community

robertthomas54 say:

Restting network settings

Why do I need to reset the network settings every day to get any good internet connection?

johann eva say:

Hi Robert,

Resetting your Wi-Fi router is one of the maintenance procedures that help refresh your network connectivity. In this way, it could eliminate latency and will return your network status like when you started using the Internet.

Explanation: It's also a standard way to prevent lag when receiving online data. When you restart your router, it will detect current devices that require Internet. All other devices that are now turned off will be temporarily forgotten and your WiFi will prioritize the current ones that are in use.

Let us know if you have other concerns.

johann eva say:

It's for the hardware side. However, network connection also requires correct setup before users will be able to share, manage and access online resources consistently. If you want to get a stable connection to a local network or Internet, you need to check out these few things to make sure that you will have an established a stable connection:

  • Check your WiFi router if the green light is always blinking.
  • Check your network icon in the system tray if there’s a red X mark on it.
  • Make sure that your Ethernet network driver is updated.
  • Check the status of your Internet connection. Ping www.microsoft.com server

1. Press Windows key + R to open the Run window.

2. Type cmd, then press Enter. Type ping www.microsoft.com and press Enter.

  • If you receive 4 replies, it means that your connection with the site is working properly.       
  • If you got a request time out, it means that your connection has issues.

In addition, you can visit these links to perform all other workarounds that will help you connect to the network or Internet.

  • Fix network connection issues in Windows 10
