Filter by team - Microsoft Community

ramzayi say:

Filter by team

I hope somebody would suggest a solution for the following challange.

 

I have a spreadsheet containg rows for employees.  Each row contains employee name, contact information and the team.  I need to be able to filter employees by team.  The challange is that each employee may belong to multiple teams.  So, the question is:

How to best organize the data, so on one hand I can list all employees and teams they are on, and on the other hand be able to filter only people belonging to a team i can select?

e.j. gun say:

You can add those other items to the Row Labels after the team and employee name and then each row will contain that information.  You cannot add it to the Value area, since the purpose of that area is to aggregate the values of common items (i.e. the same name and team).

 

You might notice a bunch of sub-totals start showing up.  Click on each item in the "Row Labels" area, select "Field Settings..." and select "None" for Subtotals.

 

To remove Grand Totals, go to PivotTable Options ->Totals & Filters tab, uncheck the "Show grand totals" options.

 

HTH,

 

Eric

e.j. gun say:

I would suggest creating a pivot table using your existing data as the source.  Select all your data.  From the Insert tab on the Ribbon, select "Pivot Table...".  Click OK and the pivot table is created on a new worksheet.

 

Now, drag Employee Name and Team to the Row Labels area.  This will show you what team each employee is on.  If you switch the order (Team first, then Employee Name), you can show which employees are on each team.

 

Click somewhere in the pivot table.  From the "PivotTable Tools"/Options tab on the Ribbon, select Options.  On the dialog box, go to the "Display" tab and select "Classic PivotTable layout".  Now you can filter each of the Row Labels as desired.

 

If you drag any other field into the "Values" area of the pivot table, and change "Sum of ..." to "Count of ...", the pivot table will show you counts for how many employees are on each team, or how many teams each employee is on, depending on how you order the Row Labels.

 

Give it a try and let us know if this meets your needs.

 

HTH,

Eric
