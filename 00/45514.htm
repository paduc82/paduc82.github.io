The JBL Clip 3 uses Bluetooth 4.1 (see spec here).  This is a problem because the built-in Bluetooth support in Windows 7 does not support Bluetooth 4.

Open Device Manager on your computer (click Start, type Device Manager in the Search box, and press Enter)

Double-click the Bluetooth category to expand it

Right-click your Bluetooth adapter (receiver, transceiver, etc.) and select Properties

Select the "Advanced" tab

Check the LMP version

  • LMP 0.x - Bluetooth 1.0b
  • LMP 1.x - Bluetooth 1.1
  • LMP 2.x - Bluetooth 1.2
  • LMP 3.x - Bluetooth 2.0 + EDR
  • LMP 4.x - Bluetooth 2.1 + EDR
  • LMP 5.x - Bluetooth 3.0 + HS
  • LMP 6.x - Bluetooth 4.0
  • LMP 7.x - Bluetooth 4.1
  • LMP ‚Äã8.x - ‚ÄãBluetooth 4.2
  • LMP 9.x - Bluetooth 5.0

If you don't have LMP 6.x or better, that's your problem.

To confirm that the speaker is not the problem, see if you can pair it with your smartphone and hear sound.  Most smartphones do support Bluetooth 4.

Go to the Dell support web site (https://www.dell.com/support/home/us/en/04), enter your laptop's Dell Service tag, and determine if you have the latest driver for the Bluetooth module in your laptop.  If you do not have the latest driver, download the driver from the Dell site and install it.  Then check Device Manager again.

If this does not solve your problem, or if you already had the latest driver, you can buy an inexpensive Bluetooth USB transceiver that will work with Windows 7.  For example:

If you do get a Bluetooth transceiver, disable the onboard Bluetooth in your computer.
