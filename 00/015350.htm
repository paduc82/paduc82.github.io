jeeped say:

It would appear that something (a command modifying startup macro perhaps?) has set the number of 'undo levels' to zero. There is a Microsoft support article on this problem and how to resolve it. For Excel versions 2000 and above:

http://support.microsoft.com/?kbid=211922


The Undo command is Ctrl+Z and the Redo is Ctrl+Y. The Redo button is tied to the Undo sequence. If one or more operations have been undone, they can be redone. This behavior closely duplicates an internet browser's History where you can go back and then forwards but never further than the newest list entry in history. The Ctrl+Y keystroke shortcut is tied to the Redo operation, but it allows to continue forward, as in Do Again.

F2 is the command to edit initiate an edit of a cell's contents. F3 displays the Paste Name dialog. More on Excel shortcut keys in this Microsoft article: Excel shortcut and function keys.
