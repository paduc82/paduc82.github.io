 You might encounter this error message for one of the following reasons:

Hi Mateenn,

Share files using Windows Home Groups:

Home groups were introduced in windows 7 and are designed to simplify the process of sharing files between the various PC. A home group is created initially on one PC and then each of your other computers connect to it in order to link them all together. Once you have joined the home group, you can then decide which libraries and file types you want to share such as pictures, Documents and music.

To create a home group:

  1. Open up the Start menu and search for home group.
  2. Click on home group from the list of result that comes up.
  3. Your PC will now perform a quick search for any existing home group. But in this case we need to create one. After a moment or two, you should see a button to create.
  4. Tap this to create your home group.
  5. You will now be walked through a quick guide to get the home group setup.
  6. Tap next on the Windows that pops up to get started.
  7. Now choose what types of files you want to share with others who join your home group and tap next to continue.
  8.  

    Joining a Home group:

    1. Open up the start menu and search for home group.
    2. Click on home group from the list of result that comes up.
    3. Your PC will now perform a quick search for any existing home group and should find the one we just created. Tap Join to continue.
    4. Type in the password you got from the setup previously.
    5. You will now be asked what files you want to share within the home group.
    6. Press Finish, and that’s the second PC now added to the home group.
    7. Now simply repeat that process with any other PC’s you want to join to the home group too.

    Accessing files in the Home group:

    1. Now that you have connected your PC’s together, the next step is to actually find the files that you have shared within your home group. This is pretty easy as windows will show you the home group on the left hand sidebar within Windows Explorer.
    2. So to access the files.
    3. Open up Windows Explorer by clicking the Folder icon on your taskbar.
    4. Look for home group on the left and click this.
    5. You will then be shown all the computers in the home group, so click on the one you want to view.
    6. Now you will see the shared files from that PC, so again simply click on the folder or file you are after.
