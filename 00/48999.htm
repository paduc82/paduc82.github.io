d m say:

Is there real difference between ad-hoc and softAP WiFi networks?

As both network types support the same connection speeds and WPA2, is there any difference between them? (when used to connect two PCs)

samhrutha g say:

Hi d m,

 

An ad hoc network is a temporary connection between computers and devices used for a specific purpose, such as sharing documents during a meeting or playing multiple-player computer games. You can also temporarily share an Internet connection with other people on your ad hoc network, so those people dont have to set up their own Internet connections. Ad hoc networks can only be wireless, so you must have a wireless network adapter installed in your computer to set up or join an ad hoc network.

 

A software-based wireless access point (AP) sometimes referred to as a SoftAP that uses a designated virtual wireless adapter.

 

SoftAP and ad hoc networking are mutually exclusive on the same physical adapter. If SoftAP is running on the AP adapter and a user or application starts ad hoc networking on the STA adapter, SoftAP will be shut down. If ad hoc networking is running on the STA adapter, an attempt to start SoftAP on the AP adapter will fail.

toondog say:

FIXED!!! My laptop had this exact problem from Day One.  I found a thread where some people had fixed it by going into mouse properties, and either enabling trails (shortest trail setting was OK and hardly noticeable) or by changing the cursor display from standard to magnified (or, to black).  But that did NOT work for me.  Here's what absolutely worked:

I turned off the touchscreen.  Who needs it -- it was a stupid Windows 8 idea, anyway, and the trade-off for fixing that "ghost cursor" issue was WAY worth it.

Go to Device Manager.  Under Human Interface Devices there are 2 lines for USB input device.  (If there are more, temporarily unplug wireless mouse receiver or other USB input devices).  Disable the first one, by right-clicking on it and selecting Disable.  And the problem is GONE!  Hope this helps.

Possibly the screen had a defect that made it act like an invisible finger was tapping it, or more likely it's a Windows 8 bug that affects some machines.  Anyway, it's a huge relief to see it fixed, as the laptop was horrible when this happened.  Good luck!
