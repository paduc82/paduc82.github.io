Sending multiple email messages to Microsoft OneNote folder>tab - Microsoft Community

would like to learn say:

Sending multiple email messages to Microsoft OneNote folder>tab

Hello,

Open the folder that contains the messages you want to send to Microsoft OneNote.

Open Microsoft One Note and Navigate to the folder and tab section you want to have those multiple messages go to.

Keep in-mind that even if you place a message(s) in the same folder and you want to place them in different Folders>tabs, you can easily do this after you have them moved into OneNote.

You can navigate between windows in many ways, but in order to do this we simply are going to select multiple messages in Outlook 2010 and drag them into OneNote (The tab you have open in OneNote).

This means you may want to grab the top border of you outlook window while holding the mouse button down and "bump" the mouse to the left or right side of the screen so that window you grabbed then occupies half of the actual screen. Then do the same to the OneNote window.

As I said there are many solutions, but I am writing this for those that are unfamiliar with other methods and this is a simple solution.

Important: When selecting your messages in Outlook, please be sure to order the massages by date, oldest (on top) to most current (at the very bottom). Then, select the 1st oldest message by highlighting it with a mouse-click, scroll down to the newest, hold the shift-key and select that message in the same manner. 

All messages in-between will now be selected. This method will give you a chronological order and for the moment, it is the best way I have found to get them to stay orderd by date when they appear in the tab.

Now grab your highlighted messages in the open Outlook folder and drag and drop them into the OneNote 2010 tab you have open and waiting on the other side of the screen. Again there are other ways to do this, but I am offering a simple solution to show you the concept and then yo can innovate from there.

Yes, the messages will display as an envelope message, but no worries here because you can right-click the message and select "Insert as printout". The problem is that OneNote will only let you right clock one message at a time.

I do have the satisfaction of the drag and drop for removal, but lack the same behavior as selecting one message in Outlook and clicking the OneNote button in Outlook and having the message be formatted as an editable message/Printout.

The option above is to build a full solution when others read this and ad to it. I hope to have a real solid solution that gives us all what we want in time.

charles kenyon say:

Strong is also a character style. I regularly use it when I want bold text.

You cannot merge two character styles, they overwrite each other.

Your style is first applying the default paragraph font (which is not bold) and then adding underlining.

Even if you base your style on underlying properties, the bold is not an underlying property.

A linked style applied to only part of a paragraph has the characteristics of a character style.

Workaround:

Use a macro.

I am a big proponent of using styles for your formatting. However I can't see an easy Style solution for this.

Record a macro that underlines selected text and assign it to a keyboard shortcut.

Or, use your character style and go back and apply bold (not Strong) to the text in your character style.

Or, create an underlined bold character style to apply to words that are already bold.

Or, use a paragraph style as the underlying style.
