teylyn say:

Hello,

Exit function will stop the execution of the function. In the first example, the Exit function ensures that the routine does not run through the lines with the error handling. 

Resume next is used at the end of error handling code to return the execution of the code to the row after the one that triggered the error handler.

The Exit function in the error handler of the second example is strictly speaking not required. It is more typically used when other lines of code follow, because Exit Function will keep global variables, whereas End Function will clear variables. In this case it may have been copied and pasted from another module and simply not been removed.
