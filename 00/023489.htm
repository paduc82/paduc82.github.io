It seems that we have been unable to verify your correct MX record (used only for mail-enabled accounts) or CNAME (for accounts that chose not to use mail). To confirm that you own the domain, follow the steps below, depending on how you configured your domain.

Configure your MX record if you chose to use Windows Live Hotmail

1.  Sign in to http://domains.live.com by using your domain administrator if you have not already done this. Then, select your domain in the list of domains that you control.

2.  On the Domain settings tab, locate the MC server, and then copy the entry that appears there in bold letters. The entry has the format of xxxxxxx.pamx1.hotmail.com. The x‚Äôs represent numbers.

3.  Visit your domain registrar, and then create a new MX record. To do this, use the information that is provided in the MX record section and the MX server information that you copied.

Configure your CNAME if you chose not to use Windows Live Hotmail

1.  Sign in to http://domains.live.com by using your domain administrator if you have not already done this. Then, select your domain in the list of domains that you control.

2.  On the Domain settings tab, locate the CNAME record, and then copy the numeric value in the Host field.

3.  Visit your domain registrar. Then, create a new CNAME record that has the Host field set to the number that you copied from Admin Center, and that has the Points to field or the Value field set to domains.live.com.
