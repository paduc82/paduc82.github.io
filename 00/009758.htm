Windows 10 Device Drivers: The hidden gold in Windows 10 - Microsoft Community

j w stuart say:

Windows 10 Device Drivers: The hidden gold in Windows 10

Technical Level : Basic

Summary

All PCs, Tablets and other devices supported by any version of Windows need to access the various devices that are built into the hardware or motherboard. Without a device driver or drivers your computer may not function properly.


Details

Without the proper device drives things like no access to the Internet (No LAN Driver), some or all of your USB ports (Typically USB 3 ports) may not function, lack of sound (No Audio driver), SATA / RAID drivers, all thought with the rapid drop in prices for Solid State Drives (SSD) which perform anywhere from 3 to 6 times faster than the best hard drives the need for RAID may no longer be required by most of the users who currently use RAID as a means of gaining performance or data protection. Also let’s not forget your display graphics.

I have installed Windows 10 (various builds) on motherboards that have Intel X38, P55, P8Z68 and Z97 chipsets (sorry no Z99 which I know was a problem with some of the later builds) and found that Windows 10 has included all the necessary drivers for each of these computers.

Most likely the only hardware that will require you to go to the manufacture’s support web site for drivers are the “do it yourselfers crowd” using the latest Intel chipsets scheduled to be released later this year, most likely after Windows 10 is Released To Manufacturing.

If you just finished installing the Windows 10 preview build sometimes (especially with getting the graphics driver installed) is to reboot the computer and then run Windows Update. You can also check your history on what Windows updates were installed.

Report Drivers Issues:

I realize that everybody may not share this experience so if you do have a device driver issue, report it to Microsoft by clicking on the Windows Feedback tile in the Start menu or on the Start screen.

Notes:

Sometimes the latest build may introduce problems you did not have with the previous build. You can Roll Back to the previous build if necessary by following the steps in this Wiki Article:

How to recover / restore your previous version of Windows using the Rollback feature

Also a Clean / Custom install of the latest build may solve your driver problem.
