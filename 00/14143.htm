andrew simpson uk say:

You mention "clutter" being an issue, but not file size.

If you're finding file size isn't too onerous, you could simply use the filter for incomplete tasks.

  • Project 2007, select from the drop down menus Project > Filtered for > Incomplete Tasks
  • Project 2010, select from the ribbon View > Filter > Incomplete Tasks

This will hide anything which is 100% complete from the view, and is available in the Gantt view and the Task Usage view.

There isn't an inbuilt view in the Resource Usage view, but you could build one yourself, filtering for tasks where "% Work Complete" is less than 100.

If you are finding file size a problem, then you'd need to look at a more advanced solution, like John's, or find a mechanism of moving completed tasks over to a different MPP file.

rohn007 say:

Yes, there is a way to do that in Office.

MS calls them "Themes".   The concept of “Themes” was introduced in Office 2007. It was implemented in most of the core Office applications: Word, Excel, PowerPoint, Outlook.. Since Themes are a new feature that was implemented at the same time in all of the apps the implementation is fairly consistent. Therefore, when you read a tip written for a specific application, there is a very good chance the tip is applicable to the other apps too.

The idea is that you can use one of the built in themes, or create and distribute a custom corporate theme.  The general idea is that if you use a single theme, ie corporate theme, in all Office files you will have a consistent look and feel through all of the applications

The best place I've seen for learning about Office themes is RedCape Co. They have a US$200, 1 hour video course that is worth the money to get a clear, concise lesson in setting up themes: https://jb121.infusionsoft.com/app/storeFront/showProductDetail?productId=85  

(Look around the site for other free stuff you can watch or download!)

RedCape makes one point I have not seen mentioned anywhere else. If you are creating a custom theme, you should start it in PowerPoint. Apparently if you start in another application there are a couple of PPT features that are not initialized properly and therefore will not ever get setup properly in the theme!
