Macro to clear subtotals in row

Sub NoSubtotals()
'turns off subtotals in pivot table
'.PivotFields could be changed to
'.RowFields or .ColumnFields
Dim pt As PivotTable
Dim pf As PivotField
On Error Resume Next
For Each pt In ActiveSheet.PivotTables
For Each pf In pt.PivotFields
'First, set index 1 (Automatic) to True,
'so all other values are set to False
pf.Subtotals(1) = True
pf.Subtotals(1) = False
Next pf
Next pt
End Sub

For the sake of this response - assumption is that the following applies

#1 - Imported your data from an external source (CSV, XLS)

#2 - Your address info is contained in one field

#3 - You mapped this single field (item #2) to the single Address Field (Home, Business or Other) which is comprised of street address, City, Sate/Prov, Zip/Postal Code, Country etc

I have no problem sharing audio from that soundboard site using my Creative What U hear component which is equivalent to Stereo Mix. If your Stereo Mix option does not transmit audio due to Skype blocking it, you will need to use the Share system sound option in Skype. 

In the current versions of Skype you can only use the Share system sound option on Skype to Skype calls, when sharing a desktop/application window. The phone user will only hear the audio sent.
