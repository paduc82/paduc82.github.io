Could be 4, but there is not T in column C

5 T

hansv mvp say:

As an array formula confirmed with Ctrl+Shift+Enter:

=INDEX(B1:B28,SMALL(IF(C1:C28="T",IF(H1:H28<>"",ROW(B1:B28)-ROW(B1)+1)),2))

Adjust the ranges as needed. If you wanted to find the 3rd value, change the 2 at the end of the formula to 3, etc.

Perhaps I'm overcomplicating it, but try this (still an array formula):

=INDEX(B1:B28,MIN(IF((C1:C28="T")*(H1:H28<>"")*(ROW(B1:B28)-ROW(B1)+1>MIN(IF(H1:H28<>"",ROW(B1:B28)-ROW(B1)+1))),ROW(B1:B28)-ROW(B1)+1)))

Another option:

graham mayor say:

With Outlook on Windows, it is certainly possible to save the recipients and subject in a message template (save as OFT format).

It would also be a relatively simple task to use a macro to create a new document using your template and add whatever variable text is required. e.g. the following will create a message using the indicated template, It will add the text and date to the start of the message and leave the message open for you to add variable text.

Don't add your signature to the template or it will be duplicated. It might also be possible to automate the addition of the status text also, but you haven't provided enough information about that.

Sub CustomMessage()
Dim olItem As MailItem
Dim olInsp As Inspector
Dim wdDoc As Object
Dim oRng As Object
    Set olItem = CreateItemFromTemplate("D:\Word 2016 Templates\test.oft")
    With olItem
        .BodyFormat = olFormatHTML
        Set olInsp = .GetInspector
        Set wdDoc = olInsp.WordEditor
        Set oRng = wdDoc.Range
        .Display
        oRng.collapse 1
        oRng.Text = "System Status for Tuesday, "
        oRng.collapse 0
        oRng.Text = Format(Date, "dddd, mmm, d, yyyy") & vbCr & vbCr
        oRng.Font.Bold = True
    End With
lbl_Exit:
    Set olItem = Nothing
    Set olInsp = Nothing
    Set wdDoc = Nothing
    Set oRng = Nothing
    Exit Sub
End Sub

The Resource Names field will wrap text as long as there are spaces in the text string. Multiple names can be displayed in the Gantt Chart view by increasing the row height. You can also see the complete list of names by hovering your mouse over the Resource Names field cell for any given task.
