mike h.. say:

Hi,

How are you with macros? ALT+F11 to open VB editor. Right click 'ThisWorkbook' and insert module and paste the code below in and run it.

Sub Clear_Dead_Files()
Dim i As Long
For i = Application.RecentFiles.Count To 1 Step -1
    If Dir(Application.RecentFiles(i).Name) = "" Then
        Application.RecentFiles(i).Delete
    End If
Next
End Sub
