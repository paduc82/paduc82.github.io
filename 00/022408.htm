doug robbins - mvp office apps & services (word) say:

To set\modify the language of a Style, right click on the Style in the Styles Gallery or the Styles pane and the select Modify and then in the Modify Style dialog, click on Format and then Language.

Language is an attribute at the word level so if you copy and paste text from another source, you are most likely going to bring the language that was assigned to the text with it.  One way to overcome that, is to use CTRL+SHIFT+s to display the Apply Styles dialog and with the text selected, click on the Reapply button to re-apply the Style to the text.

stefan blom say:

The way I understand it, language formatting exists on three levels in Word: (1) as part of document defaults, (2) as part of an individual paragraph style or character style, and (3) as direct formatting applied to a text range (note that a range of text consists of one or more text characters).

The document default language setting can only be modified for a template. This happens when you select a language in the Language dialog box (Review | Language | Set Proofing Language) and then click the Set As Default button. Each document inherits the default language from its template but you can't change the default unless you edit the underlying XML structures for the document. 

To change the language of (part of) a document you can apply a style with a different language (method 2 above). Sometimes, all you can do is (3): apply language to the selection, that is, first select text, then pick your language in the Language dialog box, and finally click OK.

When you are dealing with text from other sources, pasting as unformatted text (for example by using the "Keep Text Only" Paste Option) is the easiest method. However, once you already have text in the document with incorrect language, applying the correct language as direct formatting via the Language dialog box is usually required.

How do I take an average excluding high and low numbers?

I have a list of physicians.  In a different column is procedure.  In yet another column is the duration (number of minutes the procedure took.)  I need to get an average for each physician by procedure, but I want to throw out the high and low durations so they don't skew a projected time the procedure will take.  (Each physician may take a different average time to complete a procedure.  One physician may take an hour to complete a knee surgery while the another physician takes 2 hours.)  I may even want to exclude the top 3 highs or bottom 3 lows if there is a higher quantity of procedures to make an average with.  How can I do this?  It's like I need to combine a "SUMIFS" with a "LARGE" function, but I don't know how.  Any ideas?  Would love some help. 

bernie deitrick, excel mvp 2000-2010 say:

Note too that the formulas given will fail if there are two or more values that equal the max or min. It is much better to do the conditional sum and then subtract the large and small values, for however many you need. For example, to remove the three high and low values from the calculation, array enter this formula, where the names are in column A, procedures in column B, and the times are in column C, with the name of interest in cell D2 and the procedure in E2:

 

=(SUMIFS(A:A,D2,B:B,E2,C:C)-SUM(LARGE(IF($A$2:$A$1000=D2,IF($B$2:$B$1000=E2,$C$2:$C$1000)),{1,2,3}))-SUM(SMALL(IF($A$2:$A$1000=D2,IF($B$2:$B$1000=E2,$C$2:$C$1000)),{1,2,3})))/(COUNTIFS($A$2:$A$1000,D2,$B$2:$B$1000,E2)-6)
