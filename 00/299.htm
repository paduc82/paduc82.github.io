I have never encountered this error. Based on your description, I have a theory on what this might be.

 

You are downloading map data, which are huge files. When you ask the phone to start downloading something, it estimates the time to download. If you are getting this error even after connecting the charger, wait for the battery to fill up and then try the download.

Windows Phone and OS X - Microsoft Community

hayden_esq say:

Windows Phone and OS X

A huge barrier to my adoption of Windows Phone is my use of OS X as a desktop OS for 10+ years. I dual-boot Windows for gaming but have no intention of switching to it as my primary OS.

A OneDrive app for OS X makes it possible to sync files and photos. An outdated Windows Phone sync app makes it possible to sync music, ringtones, and other media from iTunes. However, this is only a partial solution to my syncing needs.

As of this writing, there is no free or easy way to sync reminders, notes, bookmarks from Safari, or internet passwords from Keychain to Windows Phone. Furthermore, third-party options such as LastPass and Xmarks, for passwords and bookmarks respectively, are woefully inadequate on the Windows Phone platform.

My only option is to use more Google services, subscribe to Office365 to get otherwise free features, or to resort to keeping my bookmarks and passwords in OneNote. None of these are ideal.

Syncing Options for OS X users:

  • Mail: Any IMAP, including iCloud, Google, and Outlook.com
  • Calendar: iCloud, Google, Office365 ($7/month), third-party Exchange host, a few others
  • Contacts: iCloud, Google, Office365 ($7/month),, third-party Exchange host, a few others
  • Notes: Office365 ($7/month), or third-party Exchange host
  • Reminders: Office365 ($7/month) or third-party Exchange host
  • Bookmarks: partially via Xmarks (cannot edit bookmarks, $12/year) or by installing iCloud for Windows and booting Windows every time to sync
  • Keychain/password: Lastpass ($12/year)
  • Files: OneDrive
  • iTunes Media: Windows Phone for OS X app

This is an incredible round-about and expensive way to get anything close to the seamless syncing available between Google and Apple iCloud services and OS X.

Apple released a iCloud for Windows that makes it easy to sync your Windows computer to iCloud and iPhone. Microsoft needs to release a Control Panel for OS X that makes it easy to sync your OS X to your Microsoft account and Windows Phone, or, in the alternative, work with Apple to make Microsoft services an option on OS X alongside Google and Yahoo.
