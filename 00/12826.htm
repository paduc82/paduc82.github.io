The screen location of the Word window is supposed to be saved in the registry each time you shut down the program. It's part of the binary blob in the Settings value under the key HKEY_CURRENT_USER\Software\Microsoft\Office\16.0\Word\Data.

It's possible that the Data key in your registry is damaged or that the permissions governing writing to it have been set wrongly. To try to fix it, follow Method 3, "Delete the Word Data registry subkey", in https://support.microsoft.com/en-us/help/921541/how-to-troubleshoot-problems-that-occur-when-you-start-or-use-word. Be sure to take a backup of the key first so you can restore it if the attempt doesn't work.
