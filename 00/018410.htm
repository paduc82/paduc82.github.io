Try this array formula**.

Let's assume your data is in the range A2:D4.

Lookup criteria:

  • F2 = B
  • G2 = YYY

Enter this array formula** in H2:

=INDEX(C$2:C$4,MATCH(1,IF($A$2:$A$4=$F2,IF($B$2:$B$4=$G2,1)),0))

** array formulas need to be entered using the key combination of CTRL,SHIFT,ENTER (not just ENTER). Hold down both the CTRL key and the SHIFT key then hit ENTER.

Copy across to I2.

elrendhel say:

Greetings CrossfireLTD!  Welcome to the Windows Mobile Total Access Forums!

 

Since all of your other devices are able to connect, then your Router is not the problem.  This presumes that you have the following settings on your Router:
1.) Number of available IP addresses exceeds the number of devices that you have connected by at least two.

2.) That you do NOT have MAC Address Filtering enabled.

 

Assuming these issues are confirmed, then your device not being able to accept/recognize the WEP key sounds like either the device driver is not loading properly, or the device and the Router speak different languages.  Confirm that the key is in either Hex (Hexadecimal) or ASCII format.  Linksys Routers usually talk in ASCII format.   ASCII should be the same format used by your downstream devices that go to connect to the router, but occasionally a device may invite you input the key in HEX format.

 

Let us know how you fare!

bill mosca say:

Set a variable to the DBEngine so you keep the object open like this:

Public Function dev_TbrFixZLS()
'Purpose  : Disallow Zero length strings in all tables.
'DateTime : 8/14/2006 10:34
'Author   : Allen Browne
'Modified By: Bill Mosca to skip linked SQL tables on error.
    Dim db As DAO.Database
    Dim tdf As DAO.TableDef
    Dim fld As DAO.Field
    Dim prp As DAO.Property
    Const conPropName = "AllowZeroLength"
    Const conPropValue = False

    On Error Resume Next

    Set db = DBEngine(0)(0)
    For Each tdf In db.TableDefs
        If (tdf.Attributes And dbSystemObject) = 0 Then
            For Each fld In tdf.Fields
                If fld.Properties(conPropName) Then
                    fld.Properties(conPropName) = False
                End If
            Next
        End If
    Next

    MsgBox "'Allow Zero Length' has been set to No for all Access tables."

    Set prp = Nothing
    Set fld = Nothing
    Set tdf = Nothing
    Set db = Nothing

End Function
