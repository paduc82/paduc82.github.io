In most cases for messages stuck in the Outbox that can *not* be deleted the Compact option resolves the issue.

 

There are reported cases where the user can not delete the messages in the Outbox. In this case the recommendation would be go off-line in WLM  (temporary disconnecting from the internet) using the WLM provided 'Work Offline' option.

 

Click on Work offline (found on the Home tab) on the 2011/12 WLM Ribbon
Move the emails out of the Outbox to Drafts or Delete them
Click Go online
Check if you can send an email.

 

Other solutions that have been known to work (since not everyone's issue is identical or having the same root cause)

 1. Repair WLM (available as an option in Programs&Features or Add-Remove Programs by clicking on the Windows Live Essentials installed program item

 or

 2.  Delete any messages in the account Outbox folder on the hard drive (requires navigation to the account Outbox folder using Windows Explorer)

 or

 3. Remove the account and recreate the account anew (Note: Backup any messages to a user created folder in WLM's storage folders prior to removing any account - especially if the account  is configured to use the POP3 protocol since removing a POP3 account will delete the account and messages in WLM and **also** on the user's hard drive.

 

After performing any repair, deletion, or account removal/re-setup retest the ability to send and receive mail by sending a test message to oneself or another contact.

 

In a small percentage of cases the problem may not be within WLM but a corrupt Windows user profile and associated registry key necessitating creation of a new Windows user profile.

 

Some of the possible root causes of the 'stuck message in the Outbox' are:

 1. ISP temporarily disconnecting the user from the internet corrupting WLM's index

 2. User aborting WLM while the program is still sync'ing email, contacts, or calendar

 3. Sending and Receiving mail while downloading a large volume of newsgroup messages impacting WLM's ability to index both mail and news

 4. 3rd party resident Antivirus or Antimalware programs running in the background (especially suites or programs that are configured to scan incoming and outgoing email)

 5. Logging off a Windows profile or shutting down/restarting the pc while WLM is still indexing.

 6. Other intermittent connectivity problems (router, wifi connection, etc)

a dual jack is the 1/8 inch jack port that is both input and output on most PC's and Laptops these days.

I have 2 laptops with them. When I plug in speakers or a mic, the app pops up and you select which you are plugging in.

Double check in you all apps section for any folders or apps pertaining to sound or your sound card to make sure.

Or you can go to your PC OEM website and look at the specs for your PC and find out there.

But have you checked in device manager and made sure the drivers are properly installed.

Control Panel\All Control Panel Items\Device Manager>>View Tab>>show hidden items

Make sure there are no RED or YELLOW flags next to any item.

Try to uninstall the drivers for all sound items, restart PC and windows will reinstall them. restart pc again.

Here you can also try to roll back drivers to older version, but I have never seen this really work.

You can run the sound troubleshooter in

Control Panel\All Control Panel Items\Troubleshooting

troubleshoot audio recording
