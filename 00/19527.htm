No, your @Hotmail.com account cannot be changed to @outlook.com account but you can use an alias instead.

We suggest to use the Windows Device Recovery Tool. Start by downloading and installing Windows Device Recovery Tool (for Windows 10, 8.x and 7).

The procedure to restore your phone using Windows Device Recovery Tool is as follows:
