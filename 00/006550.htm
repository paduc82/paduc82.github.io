As HansV said, there is no event that will react to a change in a cell produced by a calculation.  There is a calculation event, but it does not identify what cell triggered the calculation.  It just says a calculation occurred.  You would have to update all your shape colors on each calculation whether related or not.   As far a copy and paste, if you only paste one cell then It should work.  A paste triggers a change event, but your code is written to only process a change event triggered by a single cells. 

If you wanted to process multiple cells you would do something like

Private Sub Worksheet_Change(ByVal Target As Range)

    Dim sh As Shape
    Dim r As Range, cell As Range

'    If Target.Count > 1 Then Exit Sub
    If Not Intersect(Range("AW4:AW161"), Target) Is Nothing Then
       With Sheets("Visual")
          Set r = Intersect(Target, .Range("AW4:AW161"))
          For Each cell In r
            Set sh = .Shapes(.Cells(cell.Row, "UH").Value)
