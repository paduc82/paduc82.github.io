You normally only replace/reinstall drivers if you are having a problem with the existing ones or you require newer features that a newly released driver for a particular device would give you.

Black Screens can be very difficult to resolve since there are no error messages or systematic files to help determine anything to help.

Boot in to Safe Mode - Start Search type -->Device Manager            (find at top)

Reboot the computer and hopefully the BlackScreen will be history.
