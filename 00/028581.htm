Run the following macro after starting a new Word session:

 Sub StyleEnglishCanadianNormalTemplate()
'   Written 24 April 2019
'   Charles Kenyon
'   Intended to set all styles in Normal template to EnglishCanadian, proofing, not automatitically update
'   Use right after opening Word
'   Language IDs https://docs.microsoft.com/en-us/office/vba/api/word.wdlanguageid
'
    Application.ScreenUpdating = False
    Application.NormalTemplate.OpenAsDocument
    Dim aStyle As Style
    On Error Resume Next ' Some styles have no language attribute and will give an error
    For Each aStyle In ActiveDocument.Styles
        Select Case aStyle.NameLocal
            Case "TOC 1", "TOC 2", "TOC 3", "TOC 4", "TOC 5", "TOC 6", "TOC 7", "TOC 8", "TOC 9"
                Let aStyle.AutomaticallyUpdate = True
            Case Else
                Let aStyle.AutomaticallyUpdate = False
        End Select
        aStyle.LanguageID = wdEnglishCanadian
        aStyle.NoProofing = False   ' also turn on spelling and grammar checking
    Next aStyle
    ActiveDocument.UpdateStylesOnOpen = False ' For information on using this line, see:
'       http://www.shaunakelly.com/word/sharing/willmyformatchange.html
    ActiveDocument.Close SaveChanges:=True
    Application.ScreenUpdating = True
    Application.ScreenRefresh
    MsgBox Title:="All done!", Prompt:="Proofing Language in all styles in the Normal template set to EnglishCanadian."
    On Error GoTo -1
End Sub

Close Word.

stefan blom say:

If you mean that you want more than nine heading levels in Word, the answer is that it can't be done. Only nine levels exist. It should suffice for most documents, though.

jay freedman say:

Not only are there only nine heading levels, but the TOC field supports only nine TOC levels, and there are only nine TOC n styles that can be used by the TOC.

You can click the Options button in the Custom Table of Contents dialog and select more than nine heading styles to put into the TOC, but the additional ones will have to be assigned to TOC levels within the range 1 to 9 (that is, having the same level as one of the other heading styles).

stefan blom say:
Indeed, you can add as many styles as you wish to a TOC, but you you can't add more levels.

suzanne s. barnhill say:
A document with more than nine heading levels is scary to imagine (though I am aware that some government specifications documents do have many, many levels). Just in case, be aware that Heading 1, Heading 2, etc., are for different levels in a document, not for different headings. That is, you don't use Heading 1 for your first heading, Heading 2 for the second, etc. You use Heading 1 for all the major headings, Heading 2 for subheads, etc. If you already understand this, please don't be offended by my pointing this out; you would be surprised how many people do misunderstand.
