stefan blom say:

There are basically two folders where Word places the *.asd files: one is the autorecovery location whose path can be retrieved from the File Locations dialog box (click File Locations at File | Options | Advanced), and the other is the location for unsaved files (go to File | Options | Save to find out and/or change that location).

Note that if you have found an *.asd file by searching for it in File Explorer, you can open the file if you browse to that location in Word's Open dialog box. From within Word, click File | Open | Computer | Browse. In the Open dialog box, browse to the folder where you found the *.asd file, click to select the file, and click Open in the dialog box.
