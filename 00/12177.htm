    readyohead say:

    To be clear, I am looking to create my own Font themes that are not a part of the preinstalled combinations such as Calibri LIght, Calibri.  

    Extra credit: If no custom fonts are now able to be created, would previously created Font themes be honored if they were copied into the resource folders?

    john korchok say:
    BBEdit or the free TextWrangler makes the job a little easier.

    Here's what's in a Font Theme:

    <a:fontScheme xmlns:a="http://schemas.openxmlformats.org/drawingml/2006/main" name="BW">
      <a:majorFont>
        <a:latin typeface="Myriad SemiBold"/>
        <a:ea typeface=""/>
        <a:cs typeface=""/>
      </a:majorFont>
      <a:minorFont>
        <a:latin typeface="Myriad"/>
        <a:ea typeface=""/>
        <a:cs typeface=""/>
      </a:minorFont>
    </a:fontScheme>

    The a:ea tag is to spec an East Asian font alternative and a:cs id for Complex Scripts such as Arabic.

    Title the XML file as you want it to appear in PowerPoint.

    Hold down Alt while clicking on the Go menu, then choose Library. Copy the file to /Library/Group Containers/UBF8T346G9.Office/User Content/Themes/Theme Fonts. Apply it in Preview by choosing Design, dropping down the Theme Gallery, choosing Fonts>Custom>YourThemeName.
