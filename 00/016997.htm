jackpot. say:

Try the below...which adds up the figures in B1:B10 if there is a "no" present in ColA

=SUMPRODUCT(--ISNUMBER(SEARCH("no",A1:A10)),B1:B10)

joeu2004 say:
The formula is like =SUMIF(A1:A10,"*123*",C1:C10) and I have 2nd criteria in colB which I have to include as "B1:B10>=date(2010,06,01)".

Sorry about all the misdirection.

Building on Jackpot's solution, which was what you needed after all:

=SUMPRODUCT(ISNUMBER(SEARCH("123",A1:A10))*(B1:B10>=DATE(2010,6,1)),C1:C10)

You do not need wildcards in this case when using SEARCH.  See previous postings for the explanation of ISNUMBER(SEARCH).

Caveat:  SEARCH works best if A1:A10 contain text.  If they contain numbers, SEARCH usually works.  But since the actual cell value is not always what is displayed, SEARCH might fail unexpectedly.  For example, 98122.5 might be displayed as 98123, but SEARCH will fail.
