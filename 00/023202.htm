Word has no provision for putting any info re the printer into a Header or Footer. You can include the date printed, but not anything about the printer, itself. Even the built-in Headers/Footers [Document Elements] don't include anything of that nature. Likewise, the program cannot arbitrarily insert any User info. You have to insert fields in order to include those details.

Since you didn't provide any specific details, start with the basics:

  • Make sure your installation of OS X is current
  • Confirm that Office is fully updated [12.3.6 was the final update]
  • Check the mfr's web site to see if there is an updated driver for your printer
  • Restart your Mac
