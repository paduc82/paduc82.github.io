Sub ScratchMacro()
Dim oDoc As Document
Dim oRng As Range
Dim oFooterRng1 As Range
Dim oFooterRng2 As Range
    Set oDoc = ActiveDocument
    Set oRng = oDoc.Sections(1).Footers(wdHeaderFooterPrimary).Range
    With oRng
        .Text = "ETHICON CONFIDENTIAL use pursuant to Company Procedures page PAGE of NUMPAGES "
        Set oFooterRng1 = oRng.Words(9)
        Set oFooterRng2 = oRng.Words(11)
    End With
    'Insert the field code around PAGE expression
    fInsertFields oFooterRng1, "PAGE"
    'Insert the field code around the NUMPAGES expressions
    fInsertFields oFooterRng2, "NUMPAGES"
    oRng.Fields.Update
lbl_Exit:

End Sub

graham mayor say:

'Lines' are a difficult concept in Word; however if you mean 'paragraph' then that is easy enough to arange:

Sub SSHeaderMacro2()
'Graham Mayor 28 May 2015
Dim oDoc As Document
Dim oRng As Range
Dim oPara As Range
Dim vFindText As Variant
Dim vReplace As Variant
Dim i As Long, j As Long
    vFindText = Array("**ADAPTIV**", "**PRODUCT NAME**", "**STUDY DIRECTOR**")
    Set oDoc = ActiveDocument
    Set oPara = oDoc.Paragraphs(1).Range
    oPara.End = oPara.End - 1
    vReplace = Split(oPara.Text, Chr(44))
    'Establish whether the first paragraph has the requisite pieces of text
    j = Len(oPara.Text) - Len(Replace(oPara.Text, ",", ""))
    If Not j = 2 Then
        MsgBox "The first paragraph must have three sections separated by commas"
        GoTo lbl_Exit
    End If
    'end of check
    Set oRng = oDoc.Sections(1).Headers(wdHeaderFooterPrimary).Range
    With oRng
        .Collapse 1
        Application.Templates("Normal.dotm"). _
                BuildingBlockEntries("COE Package").Insert _
                Where:=oRng, _
                RichText:=True
    End With
    For i = LBound(vFindText) To UBound(vFindText)
        Set oRng = oDoc.Sections(1).Headers(wdHeaderFooterPrimary).Range
        With oRng.Find
            Do While .Execute(vFindText(i))
                oRng.Text = vReplace(i)
                oRng.Collapse 0
            Loop
        End With
    Next i
    Set oPara = oDoc.Paragraphs(1).Range
    oPara.Delete
lbl_Exit:
    Set oDoc = Nothing
    Set oRng = Nothing
    Set oPara = Nothing
    Exit Sub
End Sub

Basics: End your Windows Session - shutting down your PC - Microsoft Community

andre da costa say:

Basics: End your Windows Session - shutting down your PC

Technical Level : Basic

Summary
Today I start a new series of articles that looks at the basics of using Windows 10. This is an everyday task every Windows user is likely to know, right? Yes, but if you have moved from various versions of Windows to newer versions, there might be some initial disorientation. Powering down your computer is an important task and it must be done properly to ensure your information and settings is properly written to disk and saved so you can access them them next to time you use your PC. In this article, we take a look at several ways you can do so.
Details

Method 1 - Shutdown from the Start menu

The quickest and easiest way to Shutdown Windows 10 is by using the Start Screen. Depending on the layout, your Start menu might look different from mine, but the Power Options is always present at the bottom left hand corner of the screen. Just click Start > Power > Shutdown.


Method 2 - Using the Windows key + X power user menu

Introduced in Windows 8, Windows 10 retains the powerful and easy to access hidden menu that you can invoke by pressing the Windows key + X command. From there you can select the Shutdown or sign out menu and click Shutdown

Method 3 - Assign hardware power button shutdown command

You can make adjustments to the Power button your computer from the Power Options item and set it to Shutdown when then the power button is pressed. Press Windows key + X > click Power Options > click Choose what the Power button does. Click in the list box for When I press the power button and select Shutdown. Click Save Changes.

Method 4 - Shutdown dialog

Windows 10 retains the classic shutdown dialog. If your screen is at the desktop, you can quickly discover this dialog by pressing ALT + F4 on your keyboard and select Shutdown.

Method 5 - Shutdown using the Command Prompt

If you want to learn a power user command for shutting down your Windows PC, this should do the trick. Launch the Command Prompt, press Windows key + X > click Command Prompt (Admin).

Type: shutdown -s -t 01

Hit Enter on your keyboard

Bonus tip: if you want to Restart, type: shutdown -r -t 01

There you go, 5 quick ways, you can shutdown your Windows PC safely.

Please note, most of these commands will work in older versions of Windows too.
