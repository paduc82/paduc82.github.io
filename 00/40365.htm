Gateway ZX6900 series Windows 10 Install Issues Solved! - Microsoft Community

mrsparkle63 say:

Gateway ZX6900 series Windows 10 Install Issues Solved!

Gateway is now owned by Acer.

Original Problem:

I have been having issues installing Windows 10 through the Windows Update on the Gateway ZX6900 Series Computer which is an all-in-one computer.  The computer gets to a certain point in the update and then restores to the previous edition of Windows.

Solution:

After reading an article by Mina at Win10 FAQ, (Windows 10 update stuck - What to do?) I decided that the only way to resolve the problem (after trying everything else) was to physically remove the wifi card from the computer.  The Gateway ZX6900 allows you to disable the LAN in the BIOS, but it does not allow you to disable the wifi card in BIOS.  There is likely a driver issue with the wifi card that causes an update failure which crashes Windows Update and restores you to the previous version of Windows.

The Gateway ZX6900 uses the Ralink RT3090 wifi card.  (This card has also caused issues in older laptops I am told, so this may be an issue for you if you are not using the Gateway ZX6900 and having a similar update issue).

This is the culprit

You will need to open your computer case.  The following is a link to a youtube video from Korex Trk dated Nov 5, 2015 on how to open your Gateway ZX6900 to access memory/hard drive/wifi card:

https://youtu.be/gEITeI7ePg8

Once you have removed the back cover, there is an aluminum guard covering the CPU/GPU/wifi card, etc.  The location of the wifi card beneath the guard is circled.  With the shield installed, it looks like this:

Aluminum Cage with location of wifi card indicated

Next, unscrew the aluminum guard and put it aside, revealing the wifi card (circled in yellow).

wifi card exposed, unscrewed from board and aerials disconnected

Pull the wifi card from the socket and wrap aerial leads so the don't pose a short circuit issue if they touch the mother board.

Reconnect the aluminum guard.

Reassemble the case as per the Youtube video.

Ensure that the BIOS has the built in LAN enabled.

At this point, plug in your Ethernet cable to your router and run a Windows Update.

You should have no problems updating to the latest version of Windows 10.

As indicated this fix can work for notebooks with a similar configuration.

BTW, this Gateway has been updated with a SSD, and added another 4 GB of RAM for a total of 8 GB.  This computer is fast and still functional with touch screen to boot!

Thanks to all those on the web who have contributed to this happy ending.

Hope you find your solution soon.

If Windows 10 Professional was activated on your computer, it remains activated for the life of the computer. The activation does not expire and does not need to be renewed.
