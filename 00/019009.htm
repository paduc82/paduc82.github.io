Hi,

There are three ways to add a vertical line in Word: keyboard shortcut, ASCII code shortcut, and Symbols in Word. To add a vertical line, see the methods below:

Method 1: Keyboard shortcut. Press and hold the Shift key, then press the Backslash key.

Method 2: ASCII code shortcut. Press and hold the Alt key, then on the numeric keypad push numbers 1, 2, and 4.

Method 3: Symbols. Click the Insert tab in the Word app, then click the Symbol and select More Symbols. The vertical line is below the small letter L.
