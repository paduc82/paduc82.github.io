Some options, such as safe mode, start Windows in a limited state where only essential programs are started. However, to access internet you have to enable safe mode with networking. It starts Windows in safe mode and includes the network drivers and services needed to access the Internet or other computers on your network. Wireless drivers would not load and services required for wireless connection would not run in safe mode in networking so wireless internet connection would not work in safe mode with networking.

You cannot use a Standard (Local) account to change the account to Administrator. You can only use an existing Administrator account to do this.

net user administrator /active:yes

e) Log off from the current temp account and then login to the built-in administrator account. (You’ll now see the built-in administrator account as a choice on the login screen).

f) Now, you may logout from the current user account and then log into the user account you have just given admin access.

net user administrator /active:no
