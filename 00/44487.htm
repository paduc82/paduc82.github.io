Check for a newer Display driver on the PC or Display adapter Support Downloads webpage. While there check if there are other newer drivers to Update. You can take your time until they are all updated.

Ciao,

Devi trasferire le licenze dalla vecchia console alla nuova.

Se è stata acquistata una nuova console o la console è stata riparata, utilizzare l'opzione di trasferimento delle licenze sulla nuova console. A tale scopo, attenersi alla procedura indicata di seguito.

  1. Sulla nuova console, accedere a Xbox Live con il gamertag utilizzato per acquistare il contenuto.
  2. Andare a impostazioni, quindi selezionare Account.
poi
  1. In Opzioni di fatturazione selezionare Trasferimento licenze.
  2. Seguire le istruzioni visualizzate per trasferire le licenze del contenuto.
  3. Dopo avere trasferito le licenze, scaricare di nuovo il contenuto per aggiornare le licenze locali. Per informazioni su come effettuare questa operazione, vedere Come scaricare di nuovo nella console Xbox 360 contenuto già acquistato o scaricato in precedenza.

Il contenuto potrà essere utilizzato da altri profili sulla console. È inoltre possibile utilizzare il contenuto quando non si è connessi a Xbox Live.

I could be wrong, but I don't think you
need the proprietary software.

If you simply copy your JPEG photo files
to the memory card using an internal or
external USB media reader...and then
insert the memory card in the correct
slot on the frame you should be able to
view the photos.

It's best if copy your photos into a folder
and then copy the folder to the memory
card. Reason being...there is a limit to
how many files can be copied toi the root
directory of a memory card.

The following tutorial may offer a little more info:

If you insert the SD card in your media reader or
connect the frame to the PC via USB cable and
go to...Start / Computer...the media reader or
frame should be assigned a drive letter...probably
something like...Removable Disk (E:)...

If there are several removable disks you can right
click them and choose Open or Explore from the
menu to see the directory...the one that displays
your photo names will be the memory card..

Once you have the identified the correct drive letter
for the SD card...try the following...

Go to your Pictures folder...right click a thumbnail
or a selected group of thumbnails and from the menu...
choose...Send To...(choose the SD card drive letter)...
For example...Removable Disk (E:)...

The Send To option may not appear in Windows Photo
Gallery so if you prefer to work from there...you'll need
to copy the selected photos and paste them in a new
folder on your desktop...then you can use the Send To
option. Or you could navigate to Start / Computer and
right click the SD card drive letter and from the menu
choose Paste.

TIp: Only a limited number of files can be copied to
the Root Directory of the memory card...you may see
a dialog that says: "The directory or file cannot be
created"...to correct this issue create folders on the
SD card to sort your photos in.
