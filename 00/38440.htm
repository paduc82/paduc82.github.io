considerate_guy say:

When you film something sideways, you end up with vertical video. In other words, the video is much taller than it is wide. You can rotate that so it is standing upright, but it will still be vertical video.

There isn't anything that Movie Maker can do to add back any video on the sides that wasn't recorded. And since Movie Maker is designed to make horizontal movies (either in 16:9 or 4:3 aspect ratio), all it can do is set it in the middle with black on either side.

What you might want to look into is a separate tool for rotating the video, but keeping the original dimensions. It will still be vertical video and it won't play too well on horizontal devices like a TV, but at least it won't have black on the side. I've used Freemake Video Converter for tasks like this. But if you do install Freemake, always choose the custom install option and uncheck all the add-ons it wants to install.
