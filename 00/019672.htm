malte ahrens say:

While I don't see why clicking-and-draging the OneNote window doesn't work, there is an alternate way to set the dock location - using the registry.

 

  1. Open the run window (Windows Key + R)
  2. Type 'regedit' and hit enter
  3. In the registry editor, scroll to: [HKEY_CURRENT_USER\Software\Microsoft\Office\14.0\OneNote\Side Note]
  4. Open the 'DockedLocation' value (double-click it)
  5. Change the value to 101 (if you want it docked in the left part of the screen), 102 (right), 103 (top), or 104 (bottom)
  6. Hit OK, and back in OneNote, go into 'Dock to Desktop' mode again and it should work now!

bob jones aka: cybertaz say:
The feature you're triggering is a part of the AutoCorrect family. Specifically, Tools> AutoCorrect> AutoFormat As You Type - Headings. It applies a Heading 1 Paragraph Style if there's no end punctuation & the paragraph is no more than one line. You can clear the check to turn the feature off altogether.

If you'd prefer to leave the feature on you can do so -- just use the Undo command [Command+Z] when the Heading Style is applied.
