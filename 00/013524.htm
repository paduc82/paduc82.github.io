TOC: Turning off level 2 page numbering - Microsoft Community

stabes say:

TOC: Turning off level 2 page numbering

In my TOC, how can I turn  off "Show Page Numbers" on TOC Level 2? I only want page numbers showing in TOC Level 1.

Thanks,

Mart

stefan blom say:

Press Alt+F9 to reveal field codes. Add the \n switch where you specify the levels for which you don't want page numbers, as follows:

{ TOC \O "1-9" \n "2-2" }

The above example omits the numbering from level 2 only; if you only want page numbers for level 1, use \n "2-9" instead.

When you are done, press F9 to update the TOC; choose the option to update the whole table if prompted. Finally, press Alt+F9 again to hide field codes.

mike h.. say:

Hi,

 

Try it this way.

 

 

Sub insertrows()
Dim MyColumn As String, x As Long
MyColumn = "A"
For x = Cells(Rows.Count, MyColumn).End(xlUp).Row To 2 Step -1
    If Cells(x, MyColumn) <> "" And Cells(x - 1, MyColumn) <> "" Then
       Rows(x).Insert
    End If
Next x
End Sub
