daniel g. grau say:
You need to use section breaks, which let you define separate headers and footers, and even reset the numbering. See this:

  • Insert, delete or change a section break

To unlink the headers and footers (and thereby make then independent), once you have your section breaks, go to the header (and/or footer) and the Ribbon will show an extra "Header and Footer" tab. In there, find "Link to Previous" and uncheck it. Then you can delete the first footer.

Finally, change the page number. The instructions in Start page numbering with a number other than 1 say to "click Format" but I don't see any "Format" icon in my Ribbon. If that's your case as well, do this: select Insert > Page Number > Format and set the number to 1.
