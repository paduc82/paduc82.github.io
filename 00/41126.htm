Hi Mike,

This issue occurs because the drivers have no assigned letters yet. To assign drive letters in diskpart command, we suggest that you type the following in Command Prompt:

  1. Open Command Prompt.
  2. Type diskpart and press Enter.
  3. Type list volume and press Enter.
  4. Type select volume and the number you'll want to modify and press Enter. Example: select volume 0.
  5. Type assign letter = " " the letter you want for that particular drive and press Enter. Example: assign letter=C.

  2. Type bcdedit /set {default} bootmenupolicy legacy and press Enter.
  3. Type exit or you can close the Command Prompt.

What’s the deal with licenses?
When you purchase content on the Xbox LIVE Marketplace, you are buying licenses. These work in two ways:

  • Console License: Anyone can use the content as long as it is on the console it was purchased on. 
  • Profile License: You can use your content on any other console as long as you are logged in to LIVE. 


My content won’t work offline or for other profiles
If you have recently started using a new console, you will need to transfer the console licenses of your games and add-ons to this new console to avoid these issues. The process is simple and starts at http://xbx.lv/360DRM. It only takes a few minutes, but does require your console to be connected to the LIVE service during the transfer. Just make sure to follow the instructions carefully because you can only perform a license transfer once every 120 days.
