teylyn say:

Hello,

Excel as the original spreadsheet system does not support relationships between data tables (like Access).

Power Pivot, which is an add-in to Excel 2010 and built into Excel in later versions, has the so called "Data Model". Inside the Data Model, tables can have relationships, which can be used in the DAX formula language to create reports on the data. But the Data Model is not the same as an Access database with forms and views. It is just for storing the data. 

So, yes, the relationships in the Excel data model provided by Power Pivot can be compared to the relationships in Access, since they both link data tables.

edmundjohn_m. say:

Hi SaraCollins1980,

 

Welcome to Microsoft Community! Let me assist you on how to update the permission settings for a shared calendar.

 

Please do the following steps below:

 

  1. Sign in to https://calendar.live.com using the credentials of your account.
  2. Click on Share.
  3. Select the calendar that you shared.
  4. Under Permissions, select the contact you want to update the permission settings.
  5. There will be a drop down menu for the settings that you can configure for the person. Note that this will automatically save the settings.
  6. Click on Save.

 

Hope this resolves your concern. Feel free to contact us again if you have other concern.
