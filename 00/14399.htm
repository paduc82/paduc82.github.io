prashantkumar96 say:
Windows Defender keeps deleting my gaming files because they are .exe extensions.

Nearly every program running in your system has EXE extension. So, having an EXE extension should not be a problem. They are most probably malicious (maybe cracked?). Make sure you have genuine copy of your game, otherwise you can get into trouble.

So I would like to know how to prevent Windows Defender from deleting stuff without asking me for permission prior to.

A malware can do a lot of damage in few seconds. So taking permission before removing them seems a threat in itself to me.

I also don't want to go to add exclusions. I want it the exact opposite I don't want to have to add something to not be deleted, I want to not have my stuff being deleted without me wanting it so.

Windows Defender won't delete your stuff, unless it has nothing but malware, it moves your stuff to a safe vault, a quarantine, and you can restore your stuff anytime you want (but do it only when you are 100% sure your stuff is not malicious). Adding exclusions makes sense, exact opposite don't. As I said before, if Windows Defender started waiting for your approval, malware can do a lot damage between that time.   1. Press Win key + R. This will open Run dialog.
  2. Type 'gpedit.msc' and hit enter. This will open Local Group Policy Editor.
    Note: This won't work with Windows 10 Home edition.
  3. Navigate to Computer Configuration -> Administrative Template -> Windows Components -> Windows Defender -> Threats.
  4. The options there will give you some sort of control over threat removal.

Experiment at your own risk!
