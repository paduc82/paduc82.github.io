Hi Cincyboy07,

Windows 7 keeps copies of folders stashed away on a hidden sector of your hard drive. This feature is called "Shadow Copies". It's enabled by default, but some manufacturers turn it off so you can squeeze a little more disk space out.

To see if it'll do you any good, you need to find the root folder first. For example, let's say you stored your files in the directory Documents > Dogs > 8-11-07. In Previous versions, you might be able to find a copy or two from a while ago. You can double click the copy that comes up to look at the copy, and once you decide which one you want, you can merge it back to the previous location.

"How to Refresh Windows 8 and Windows 8.1"

  • The PC boots into Windows RE (Recovery Environment).
  • Windows RE scans the hard drive for your data, settings, and apps, and puts them aside (on the same drive).
  • Windows RE installs a fresh copy of Windows.
  • Windows RE restores the data, settings, and apps it has set aside into the newly installed copy of Windows.
  • The PC restarts into the newly installed copy of Windows.

"How to Reset Windows 8 and Windows 8.1"

"When you reset your PC, the drive will be completely reset, and all your apps and files will be removed from it. Windows 8 will be reinstalled, and you'll need to change any settings and reinstall any apps that you want to get back after resetting. If you know if any files that you don't want to lose, then you should back those files up before starting the reset"

There are three ways to update your driver:

  • Use Windows Update. Windows Update might have an updated version of your printer driver. To update your drivers using Windows Update follow these steps:

  1. Press and hold (or right-click) the Start  button, select Device Manager. You might be asked for an admin password or to confirm your choice.
  2. Expand Printers, find your device, right-click it, and then select Update driver.
  3. Select Search automatically for updated driver software.

  2. Select the Start button, then select Settings  > Devices > Printers and Scanners.
  3. Find the printer in the list under Printers & scanners, select it, and then select Remove device.
  4. In the search box on the taskbar, type Print Management, then select the search result of the same name.
  5. Select All Printers, find your printer in the list, press and hold (or right-click) it, and then select Delete.
  6. Restart your PC.
  7. Reconnect your printer's cable to your PC and try installing the software and driver again.

Note: Some printer software asks you to keep your printer disconnected until its ready. See your software or driver documentation to determine when to connect your printer.
