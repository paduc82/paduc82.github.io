No, this isn't possible in Windows Phone 7 like it was in Windows Mobile. Without a user-exposed file system, I suppose the designers figured there wasn't much point in adding the ability to browse the file system on a connected network.

 

In desktop IE9, using an UNC path opens an instance of File Explorer pointed at that path.  There's no file explorer on the phone to open with IE9 Mobile.  :(

how do get previously used email addresses to auto-complete within the - Microsoft Community

mikesnline say:

how do get previously used email addresses to auto-complete within the address lines

I used to be able to begin typing someones email address and it would auto-complete. my new computer doesnt do it though. how do i make it so it does it again

brian tillman [outlook mvp 2007-2019] say:

This question doesn't seem to be related to Office customization and programming, so I'll be moving it to the proper Outlook forum after answering.  The forum will be Using Contacts, Calendar, Notes, and Tasks.

The autocompletion cache does not reference your Contacts in any way whatsoever.  It simply remembers the last 2,000 addresses to which you've sent mail.  Since this is, as you say, a new computer, the cache will be empty.  As you continue to send, more addresses will be added.

There are two ways to populate the cache with your contacts.  One is to open your Contacts folder, press Ctrl-A to select all your contacts, then drag the selection to the Inbox to open a new compose window with all of your contacts in the To field.  Press Ctrl+K to resolve all the addresses.  Then close the compose window and  cancel the sending.  Close Outlook to write the cache to disk.

The other way, if you're using Outlook 2007 or earlier, is to copy the cache file from the old computer.  It will be in the (usually hidden) folder %AppData%\Microsoft\Outlook, it's name will match the name of the mail profile you were using on that computer, and its file type will be ".NK2".  With Outlook closed, copy that file to the new computer, put it in %AppData%\Microsoft\Outlook, and name it to match the current mail profile, keeping the .NK2 file type.  WHen Outlook starts, it should be able to use it.

a. user say:

Hi MRE-1,

 

Thank you for reporting this issue to Microsoft.

 

I understand that you want to block someone who constinually sending annoying emails to you. There are many ways to block a specific sender. Please follow the steps below:

 

A. Block a sender or a domain name

1. Go to your Inbox page.
2. Click the gear icon at the upper-right side of the page.
3. Select More mail settings.
4. Under Preventing junk email, click Safe and blocked senders.
5. Click Blocked senders.
6. Enter the email address or domain in the box labeled Blocked email address or domain: and click Add to list.

B. Set the filtering level of the messages

Please follow these steps to setup your junk e-mail filter in Outlook.com.

 

1. Sign in to http://mail.live.com.
2. In the upper-right corner of the page, click the gear icon  then More mail settings.
3. Under Preventing junk email, click Filters and reporting.
4. Under Filters and reporting, select the options that you want, and then click Save.

 

Notes:

 

* If you choose Exclusive for your junk email filter, only messages from email addresses on your Contacts or Allowed senders lists appear in your Inbox. Messages from senders not on those lists are automatically sent to the Junk e-mail folder.

* You should occasionally check your junk email folder to make sure that good messages are not sent there by mistake.


C. Sweep function - This will allow all received or even future emails coming from the specific sender to be deleted. Here are the steps to activate the Sweep option:

1. Go to your Inbox page.
2. Check the specific sender you would like to filter.
3. On the upper part, click Sweep
4. Choose Delete all from
5. On the pop-up box, just click Delete all.

Note: You can also block future messages from that sender by checking the box beside Also block future messages.

D. Rules - we can create a rule that can block a specific word from the subject of the spam email.

 
1. Sign in to http://mail.live.com.
2. In the upper-right corner of the page, click the gear icon  then More mail settings.
3. Under the Customizing Hotmail section, click on Rules for sorting new messages.
4. Click on the New button.
5. Under Step 1: Which messages do you want this rule to apply to,
a. First drop-down menu - choose Subject
b. Second drop-down menu - choose contains           
c. Third field - input the word you would like to block
6. Under Step 2: What action do you want to apply, click on Delete these messages
7. Click Save button.

 

Note: Please be informed that the blocked sender will not be notified that he/she was being blocked by his/her recipient.


I hope I was able to help you. Let us know if you're still receiving emails from this sender after doing this steps.

 

Thank you.
