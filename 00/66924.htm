brian-- say: From: Colin Barnhorst


When you install Windows Vista by booting with the dvd Setup will enumerate drives based on the BIOS ordering.  When you start Setup from within existing Windows Setup will use the lettering already in place on the existing Windows.

When I set a purely Windows Vista multiboot box I start with the x64 version by booting with the dvd.  Then I start the x86 Setup from the x64 desktop so that the second Windows Vista will use the same drive letters as seen from the x64 edition.  The reason I start with x64, of course, is that while x86 Setup will run in an x64 desktop, x64 Setup cannot run in the x86 desktop.  So I start with x64.

First make a manual System Restore point because you need to make a registry adjustment.

Click the Start button/orb to open the Registry editor like this:

HKEY_CURRENT_USER\Software\Microsoft\Office\12.0\Common\General

That means that the next time you open an Office application (like Word) you will be asked if you want to get Office Updates through Windows Update.

On the next screen I will choose to download updates from Microsoft Update and click Finish.
