Removing dual-boot leaving Windows 10 - Microsoft Community

rvwilliams say:

Removing dual-boot leaving Windows 10

I initially installed Windows 10 Tech Preview on a separate hard drive but as a dual-boot with Windows 8.1.  I've now used Windows 10 for a sufficiently long time that I no longer needed Windows 8.1 and I wanted to remove its hard drive.  But there was a problem due to the dual-boot.  First I tried booting the Windows 10 installer and using the Repair -> Command Line to run bootrec etc.  This did no good.  Then it occurred to me that the Windows 10 disk didn't have a System partition.  This is what I did (with the help of this article https://technet.microsoft.com/en-us/library/ee829683%28v=ws.10%29.aspx).

1.  I booted Windows 10 using my dual-boot configuration.

2.  Used Disk Management to create a 100MB partition on the Windows 10 disk, formatted it as NTFS and assigned a drive letter (F:, say).  I then marked it as Active.

3.  Opened up Command Line as Administrator.

4.  Typed the command bcdboot c:\windows /s F:.Bcdboot.

I then Restarted and changed the BIOS to boot from the Windows 10 disk and Windows 10 started normally without any evidence of a dual-boot.  Later I shutdown the PC and removed the Windows 8.1 disk.

I hope this may help others in a similar situation.

Networked Computers Description

I've posted this in 2 other forums.  If you have read it there my apologies for using your time.

In Windows 8 (perhaps in Windows 7 also) the description of the computers on your network could be obtained by creating a folder, usually on your desktop, named Network.{208d2c60-3aea-1069-a2d7-08002b30309d}
