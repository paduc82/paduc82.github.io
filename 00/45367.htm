Register the File Manually

  3. In the Open window, type the following command, and then hit Enter.

RegSvr32 <filename>

You should receive a message that the file is now correctly registered. If you receive a message that the file is not registered, try the other workarounds.

Clean-Start the Computer

This might be happening if any of the background programs is confliction with the installation process. This kind of startup is known as a "clean boot." A clean boot helps eliminate software conflicts.

 

The steps to be performed in clean boot are given below:

1. Log on to the computer by using an account that has administrator rights

2. Click Start, type msconfig.exe in the Start Search box, and then press ENTER to start the System Configuration Utility.

3. On the General tab, click Selective Startup, and then click to clear the Load startup items check box. (The Use Original Boot.ini check box is unavailable.)

Note: Following this step lets Microsoft services continue to run. These services include Networking, Plug and Play, Event Logging, Error Reporting, and other services. How to troubleshoot a problem by performing a clean boot in Windows Vista or in Windows 7

Computer dies at 39 percent battery life with no warning

For the past few weeks my laptop, which I installed Windows 7 on a couple months ago, has gone dead around 39 percent of battery life with no warning.  I don't get a low or critical battery warning, and my battery action prompts are set for 10 and 5 percent.  I've looked at other forum posts here with similar problems but nothing works for me.  This never happened with Vista - I always got a warning when I was supposed to.  I don't think the battery is the problem, but rather Windows' interpretation of the battery level.  How can I fix this?
frederik long say:

While it is tempting to assume that Windows 7 is the culprit, you need a little more evidence to make sure. I would do this:
1. Run the machine down to 42% of battery charge.
2. Reboot it into the BIOS setup mode.
3. Run it in this mode for a period that would cause the battery charge to drop to 30%.

If the computer shuts down well before reaching the 30% level (which you have to guess based on prior experience) then the problem is not with Windows. Otherwise you need to download the latest power management driver for your particular laptop from the manufacturer's site.
