suzanne s. barnhill say:

To create text on a path, you really need an illustration program such as CorelDRAW or Adobe Illustrator. The closest you can get in Word is to use WordArt..

  1. To create this WordArt object, use Insert | Text | WordArt.
  2. Select any one of the A's (you can change the formatting later).
  3. In the WordArt box that is created, type your text in a single paragraph.
  4. On the contextual Drawing Tools | Format tab, in the WordArt Styles group, click on the Text Effects button.
  5. Select Transform.
  6. In the Follow Path section, choose Circle.

For this to be actually circular, you will need to make sure that the box itself is square. You'll also need to change the box size to make the text complete the circle. Changing the font size doesn't affect the ultimate display, so keep it small enough so that you can see it when you click in the box. You'll probably want to format the fill color as Automatic and the line color as No Line to get something that looks like ordinary text.

The above will create text in a circular shape. If you also need to have an actual circular line (inside or outside the text), you'll need to create a circular AutoShape and align it with the WordArt object.
