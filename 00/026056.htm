julie fka profjulie say: If you set up the phone with a windows live id and now want to change it to a different windows live id, you'll need to reset the phone.  The contacts you've entered into the phone under the original windows live id should be backed up to the windows live account that is associated with that windows live id and you can verify that the contacts are there by logging into the windows live account from your computer browser.

After you perform a hard rest of the phone and set it up to use a different windows live id as the primary Microsoft account on the phone, none of the contacts from the original windows live id will sync back to your phone unless you either:

  • Set up a second windows live account that uses the original windows live ID; OR
  • Export the contacts from the original windows live account and then import them to the one you set up as the primary Microsoft account that you are now using on the phone.
