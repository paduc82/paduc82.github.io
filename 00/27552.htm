doug robbins - mvp office apps & services (word) say:

Assuming that you are using ActiveX radio buttons named optAlpha, optBeta, optCharlie and optDelta. then the following code will insert the caption of the button whose value is true in cell A1 of Sheet 2

Private Sub optAlpha_Click()

If optAlpha.Value = True Then

    Sheets(2).Range("A1") = "Alpha"

End If

End Sub

Private Sub optBeta_Click()

If optBeta.Value = True Then

    Sheets(2).Range("A1") = "Beta"

End If

End Sub

Private Sub optCharlie_Click()

If optCharlie.Value = True Then

    Sheets(2).Range("A1") = "Charlie"

End If

End Sub

Private Sub optDelta_Click()

If optDelta.Value = True Then

    Sheets(2).Range("A1") = "Delta"

End If

End Sub

You might however want to consider just using an ActiveX combobox that contains the list Alpha, Beta, Charlie and Delta and then all that you have to do is set the value of the cell to the value of the combobox.

norman david jones say:

Hi Seth,

I am trying to reference the value (checked or unchecked) of an Option (radio) button in a formula.

If the value of the radio button is checked, I want to be able to assign a label to it on another worksheet in the same workbook. For example:

  • I have four Option/Radio buttons on Worksheet 1 labelled Alpha, Bravo, Charlie, and Delta. A user may choose only one of these.
  • On Worksheet 2 I want to reference what the user had chosen. I thought I would do this via a set of nested =IF( ) formulas.

I am trying to figure out how to assign a value to the Checked or Unchecked status of an Option/Radio button on the other worksheet.

Many thanks for any assistance you can provide!

  • Right-click the sheet tab
  • Select the View Code option from the resultant context menu
  • Paste the following event code: 
'=========>>
Option Explicit

'--------->>
Public Sub opt_Click()
    Dim vCaller As Variant

    vCaller = Application.Caller
    If TypeName(vCaller) = "String" Then
        With Me.OptionButtons(vCaller)
            If .Value = xlOn Then
                Sheets(2).Range("A1") = .Caption
            End If
        End With
    End If
End Sub
'<<=========

  • Alt+Q to close the VBA editor and return to Excel
  • Right-click each of the option buttons
  •  Assign Macro
  • Select the opt_Click macro
  • OK

===

Norman

rickytaylor2 say:

Ok managed to resolve this not only does this get the CMID sorted but the rearm count too!!

I've not done full testing of each step but this is exactly what I did and it works

1. Kbfix it ran to uninstall office 2010

2. Delete the HKLM\software\Microsoft\office & HKLM\software\Microsoft\officesoftwareprotectionplatform (had to mess about with permissions to do this). This is the main point, something in one of these registry keys must hold both CMID and rearm count. Nothing obvious thou.

3. Reboot and then reinstall office

4. run ospprearm and windows rearm. (office rearm was required for me as I tested this after step 3 to check new CMID was created first). NOTE CMID for office is generated on starting office or anything office related (including ospp.vbs to check CMID). If you didn't do this , just windows rearm is required.

If like me you are using Xendesktop and this is a golden image, this is point where you want to power it down, snapshot it and then provide this snapshot to the virtual PC's so they pick up unique CMID's for both windows and office on next boot up :D
