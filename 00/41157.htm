andre da costa say:

Because you upgraded from a retail Windows 8 license, you have a one time transfer right.

Because the base license remains Windows 7, as long as the edition of Windows 10 installed is Windows 10 Home - which would be equivalent to Windows 7 Starter - then you would be free to transfer the Windows 10 on a new computer using the Windows 8 key.

If you upgraded to Windows 8 Pro then to Windows 10 Pro, then you would have to reset the netbook back to Windows 7 in order to relinquish the license for reactivation on another computer.

How to Transfer your Windows 10 License to a New Computer

Error 1326 when installing TurboTax
Updated: 3/03/2011
Article ID: SLN46781

If you don't have permissions to a certain folder required by TurboTax, Error 1326 will occur when you try to install TurboTax.

Solution
  1. Using Windows Explorer (not Internet Explorer), locate the following folders.
    • C:\ProgramData\Intuit
    • C:\ProgramData\Intuit\Common
  2. Right-click the folder itself and choose Properties.
  3. Click the Security tab, and then click the Edit button.
    • Click OK if a UAC window appears.
  4. In the Permissions window, in the Group or user names section, select your Windows user name. If your user name is not listed:
    1. Click the Add button.
    2. Type your user name in Enter the object name to select, click Check Names, and then click OK.
  5. In the Permissions for Users section, select the Full Control checkbox in the Allow column.
  6. Click Apply, and then click OK twice to close the Permissions window.

It is not important if you use IE9 as your default browser (I use Firefox ESR v52.9.0).  If IE9 is not patched with KB4014661 (Cumulative Security Update for Internet Explorer 9: released April 11, 2017), which is the last IE9 update released before extended support ended for Vista SP2, then your Windows Update has been broken for more than 2 years (these "Checking for updates..." hangs were first reported in August 2015) and needs to be fixed to patch Vista SP2 to April 2017 and free up CPU, RAM and other system resources.
