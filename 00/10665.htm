Inverse Selection

How do you select the inverse of a section of cells in excel?

bernie deitrick say:

You can use a macro:

Sub InvertSelection()
    Dim r As Range
    Dim c As Range
    Dim s As Range
    Dim n As Range
    
    Set s = Selection
    Set r = s.CurrentRegion
    For Each c In r
        If Intersect(c, s) Is Nothing Then
            If n Is Nothing Then
                Set n = c
            Else
                Set n = Union(n, c)
            End If
        End If
    Next c
    
    n.Select
End Sub

Hi blurrt, mwthrane and AndreasJohansson79,

 

By upgrading to Outlook.com, you are only upgrading the user interface of Hotmail to the new user interface; hence all your emails should still stay in your Inbox. However, if your emails were over 270 days old, you may have lost them due to inactivity in account even before you renamed the account. If you had Self-recovery of emails turned on in your account before rename, you can try to recover some of the deleted messages yourself by referring to this link. 

 

If you chose to rename your account to @Outlook.com, a prompt would have asked you Where do you want to save your emails? You will have the option to create a new folder for your old account or save the emails in the Inbox of your renamed account.

 

If you chose to create a new folder, the default name of the folder will be the name of your old account and the folder will appear in the left pane under Folders. And all your old emails will be stored in this folder.

 

Another reason why some users arent able to see old emails after rename is when their account was hacked and the hacker had already deleted emails in their account before their renamed the account.
