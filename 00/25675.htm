#macro say:

Import Macros (*.bas file) from Word for Mac 2011 into Word for Mac 2016

Import Macros (*.bas file) from Word for Mac 2011 into Word for Mac 2016

Please provide a checklist or "How To" as to how to select a *.bas Macro file created in Word for Mac 2011 and import it into the new Word for Mac 2016.

This would seem to be a high-use item for most users, so I would imagine this checklist should already be published and available. Thank you.

#macro say:

So, I did figure out how to successfully import macros into Word for Mac 2016, from Word for Mac 2011.

It does help that you can install the 2016 and not uninstall the 2011 version prior to install.

Here is the main set of steps:

1. Know where your macros are stored, for example, New Macros subfile saved within the Normal.dotm file of Word for Mac 2011.

2. Open up a blank document in Word for Mac 2016.

3. Select from the top menu area (outside of Word via Finder bar), Tools -> Macros -> Macro

4. Select Organizer, in the right area browse to your macro file, within your old Normal.dotm file

5. Migrate the file over the the left area, saving it to your new Normal.dotm file (not Document1, etc.)

6. Save the file and exit.

7. Reopen a new blank document, ensure macros are still there, via Tool -> Macros -> Macro, they should show up right on that front page of the macros [Note: However, they will not work, just yet...]

8. You now need to finalize this via mapping your keyboard commands, they ones you prefer and are already use to by work habit, to the macros. You do this again from the Finder bar menu, Tool -> Customize Keyboard. 

9. In this Customize Keyboard area, left window for Categories, select Macros, then individually re-map keystroke combinations to match your prior habit. (Remove/Assign buttons as needed). Double check your data-entry, save with OK.

10. Save the file and exit.

11. Open a new file, and verify the macros now are successfully transferred and also are mapped to the keystrokes you are familiar with from your prior work habit.

This should work, notice it is a two phase process, basically importing the macros, and then reworking the keyboard mapping.

Someday this may be uploaded to the actual Help file. That would be really useful. :)
