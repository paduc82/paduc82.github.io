The @ means "For this row within the table"   So you want the entire column within the table:

=COUNTIFS(Table1[Contract Deliverable],"Yes")

=COUNTIFS(Table1[Contract Deliverable],"It is")

jp ronse say:

Hi,

Try this VBA:

Function GetKeyWords(Phrase As Variant, keywords As Range) As String
   Dim C As Range, strResult As String
   For Each C In keywords
      If InStr(1, Phrase, C) > 1 Then strResult = strResult & " " & C
   Next C
   
   GetKeyWords = Replace(Trim$(strResult), " ", ", ")
End Function

Usage: =GetKeyWords(H1;key_words)

With the phrase in H1.

doug robbins - mvp office apps & services (word) say:

You should define Styles for each character with the desired font and font color in each style.

 

Then if you press Ctrl+Shift+s to display the Apply Styles dialog, when you use that combination, the focus will automatically move to the Style name control and when you type the first letters of the style that you want to use, the name will autocomplete and if you press Enter, the focus will move back to the document and that style will then be applied to the text that you type.

 

See the article "Creating a Template (Part II): Styles” at:

http://www.word.mvps.org/FAQs/Customization/CreateATemplatePart2/Styles.htm

suzanne s. barnhill say:

To add to what Doug has said, although you can assign keyboard shortcuts to the styles, you can save some time by making them alternating: In the Gina style, set the "Style for following paragraph" to Stephen, and make Gina the following style for Stephen. Then you won't need to use the shortcuts unless you want to switch to narrative or from narrative to one of the dialog styles.
