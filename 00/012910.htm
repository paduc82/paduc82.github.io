andre da costa say:

Free hard disk space recommended by Microsoft: 16 GB

 Depending on the architecture of Windows 10 you want to try. If you are using Windows 10 32 bit, you can allocate 16 GBs or 20 GBs if you using the 64 bit version for light testing. Another factor is disk space for applications, page file, drivers, accumulation of data over time. So make sure you make consideration for these factors. My personal recommendation is to go with a minimum 60 to 100 GBs of space for Windows 10 testing.

Also, you should take into account future builds that might need addition disk space.
