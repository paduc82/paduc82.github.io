Generally, SharePoint Online indeed has out-of-the-boxed features to back up files automatically for the cloud files such as the version history, the recycle bin and the second-stage recycle bin. brian tillman [outlook mvp 2007-2019] say:

MAPI is not limited to the Outlook Connector. EAS accounts use MAPI (Messaging Application Programming Interface) and so do Exchange accounts.

The "Outlook Mail" platform, hosted on Office 365 systems, is replacing Outlook.com's current platform. If the Outlook Connector no longer works, you don't want to switch to IMAP. You want to use a full Exchange account to connect Outlook to the server. That will allow to you sync contacts and calendar as well as mail. To create the Exchange account, create a new account, supply your MSN email and password and allow Outlook 2010 to configure the account automatically.
