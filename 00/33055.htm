How to enable the Guest account in Windows 8 and Windows 10 (even Home edition):

  1. Aquire ntrights.exe from the Windows 2003 Resource Kit.  As of this writing, the easiest way to get it is to and click the download button under Step 1 there.
  2. As stated on that page, you need to extract and place ntrights.exe into your Windows system32 folder.  I recommend opening the ZIP file from your web browser, dragging the ntrights.exe onto your desktop, right-clicking it, clicking Cut, opening File Explorer, navigating to C:\Windows, scrolling down and right-clicking system32, and clicking Paste.
  3. Right-click the Start button and click Command Prompt (Admin).
  4. Type/copy the following text and press [Enter]: net user guest /active:yes
  5. Type/copy the following text and press [Enter]: ntrights -u "Guest" -r SeDenyInteractiveLogonRight
  6. Close the command prompt window.
  7. Press [Ctrl]+[Alt]+[Delete] and click Switch user.  You should see the new Guest option in the lower left-hand side of the screen.

    If you have Windows 8/10 Professional and would prefer to use the built-in tools, here's another way to enable the Guest account:

  1. Click the Start button, type the following command and press [Enter]: LUsrMgr.msc
  2. Navigate to Users and double-click on the Guest account.
  3. Untick Account is disabled and click [OK].
  4. Click the Start button, type the following command and press [Enter]: secpol.msc
  5. Navigate to Local Policies -> User Rights Assignment.
  6. Double-click Deny log on locally.
  7. Remove Guest from the list and click [OK].
  8. Close the two management windows that we opened.
  9. Press [Ctrl]+[Alt]+[Delete] and click Switch user.  You should see the new Guest option in the lower left-hand side of the screen.
