doug robbins - mvp office apps & services (word) say:

If the Option Buttons are named OptionA and OptionB and you have data in the workbook, which is to be loaded into the combobox if OptionA is selected in a range to which the name OptionA is assigned and that which is to be loaded if OptionB is selected in a range to which the name OptionB is assigned, the following code will populate a combobox (Combobox1) with the relevant data based on which option button is selected.

Dim ListArray As Variant
Private Sub OptionA_Click()
If OptionA.Value = True Then
    ListArray = ActiveWorkbook.Names("OptionA").RefersToRange.Value
    With ComboBox1
        .ColumnCount = UBound(ListArray, 2)
        .Clear
        .List() = ListArray
    End With
End If
End Sub
Private Sub OptionB_Click()
If OptionB.Value = True Then
    ListArray = ActiveWorkbook.Names("OptionB").RefersToRange.Value
    With ComboBox1
        .ColumnCount = UBound(ListArray, 2)
        .Clear
        .List() = ListArray
    End With
End If
End Sub

I entered "recent" in the Help search field and found the following instructions.

Customize the list of recently visited places

Clear the list of recently visited places:
1. Click the File tab.
2. Click Recent.
3. Right-click any location in the Recent Places list
   and select "Clear unpinned places".
4. Click Yes to clear all unpinnned places from the list.
   Any pinned places will still appear.

My note:  Refer to the section "Keep a file on the Recent
Places list" to learn how to pin and unpin locations.


Customize the list of recently used files

Clear the list of recently used files:
1. Click the File tab.
2. Click Recent.
3. Right-click a file in the list
   and select "Clear unpinned items".
4. Click Yes to clear the list.

Note:  Setting the number of recent files to zero also clears
the list.  To turn the feature back on, set the number to
something other than zero.

My note:  Refer to the section "Keep a file on the recently
used file list" to learn how to pin and unpin files.

lordalukard say:
Dear Tom,

I am an independent advisor, and I am trying to help other users in Community with my experience in Microsoft products.

I understand that this is frustrating to receive lots of spam and unwanted emails. It is even more frustrating to receive scams, spoofing and phishing emails. Scammers nowadays are using multiple methods to intimidate users.

There are couple of ways to deal with scammers and unwanted emails ( I am sure that you are aware of some and you are using them, but maybe additional information in support articles will help you ):

1. First of all check your spam filters and see suggestions on how to keep spam out of your inbox https://support.office.com/en-us/article/help-k...
2. Report the phishing, spam and unwanted emails https://support.office.com/en-us/article/deal-w...
3. Add emails or domains to block sender list https://support.office.com/en-us/article/block-...
4. Create specific rules for subjects or common phrases that are present in those emails https://support.office.com/en-us/article/use-in... Also you can set up a rule to deleted all emails from that address or domain

Some of the scammers figured how to "mask" their addresses and domains. So in order to check if that email address that you see is accurate, you can view the message source. It has a lot of data in it, and you should look for a part where it says from ( that will be the scammer ) and to ( your address)

In order to view message source please check this article
https://support.office.com/en-us/article/receiv...
Also you can add customize action that will enable an icon directly in the message window. Go here https://outlook.live.com/mail/options/mail/surf... and Message Surface > tag View message source.

Also read this information about fighting junk emails from Postmaster for Outlook https://sendersupport.olc.protection.outlook.co...

You can also check tips on how to stay safe online from Microsoft https://www.microsoft.com/en-us/digital-skills/...

Hope this will help your email free from Junk!
