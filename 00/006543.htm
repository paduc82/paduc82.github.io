doug robbins - mvp office apps & services (word) say:

Modify the Paragraph Format in the Envelope Return Style so that it is centre aligned.   In addition, you will probably want to set a Right Indent for the paragraph as otherwise, the return address will be located at the centre of the envelope.

As the amount of the right indent that is required to locate the address in the desired position will depend on the envelope in use, if you commonly use more that one size of envelope, you would need to change the right indent for each.  In that case, it would be best to create a template for each of the envelope sizes and then use the respective template when you want to create an envelope.

nayana s m say:

Hello Steve,

 

You need to set the data file which has all calendar and contact entries associated with it as default and associated it with the email account.

 

In order to know which data file has all the entries, right click on the Calendar folder which has all the entries and check the name of the data file associated with it.

 

Then click on File > Account Settings > Data File tab > highlight the data file which needs to be set as default > Click on Set as default button.

 

To set new mails to deliver to this data file:Open Outlook > File > Account Settings > click on Email tab > highlight the email address > Click on change folder at the bottom > then select the pst file which you want your mails to be received.
