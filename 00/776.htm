jay freedman say:

It is not possible to embed a different icon in the file. Even ordinary Word documents don't have any icon embedded in the file; instead it's Windows that knows (because of a registry entry) what icon to apply to all Word documents. The icon is associated with the file's extension (or, more correctly, with its file type).

However, you can change the icon displayed by any shortcut. Right-click the shortcut's current icon and click Properties. In the Shortcut tab of the Properties dialog, click the Change Icon button. In the next dialog, you can choose from among the icons shown (by default, those are from the Shell32.dll file that comes with Windows) or use the Browse button to open any .ico file or most .exe or .dll files. The icon you choose will appear on the desktop and in Windows Explorer (and dialogs such as File Open that are based on Explorer).

You will have to change the icon of each shortcut individually.
