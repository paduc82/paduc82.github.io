gerry c j cornell say:

Contrary to what Divya R says I would run Disk Defragmenter weekly notwithstanding that you see  "You do not need to defrag this volume". The report is based on a statistical analysis and figures produced in this way on a global basis can often mislead. You can manipulate figures to present the result you wish to see.

An obvious defect is that the analysis is almost certainly done on all files. However, the files you constantly need to access are only a small proportion  of the whole. You want to be able to access the files you use not the large archive of files you rarely use.

The Disk Defragmenter provided with Windows XP is a utility written over 10 years ago You might want to try Defraggler which is freeware and a more flexible tool. I used it with success for the last 2/3 years before I migrated from Windows XP to Windows 7.
