As in a photo album, each slide only has one object which is the image.

I tried messing with the slide master, but no luck. It would be nice to be able to copy the custom animation and then select all the objects in all the slides and hit paste. Seems like an easy solution but it doesn't seem to work that way. Any help would be appreciated.
Jon
john sr wilson say:
See if this helps:

Sub albumAni()
Dim osld As Slide
Dim oshp As Shape
Dim oeff As Effect
'This code is for pre 2007 only
For Each osld In ActivePresentation.Slides
For Each oshp In osld.Shapes
If oshp.Fill.Type = msoFillPicture Then
Set oeff = osld.TimeLine.MainSequence.AddEffect _
(oshp, msoAnimEffectFade, , msoAnimTriggerWithPrevious)
With oeff
.Timing.Duration = 0.1
.Timing.TriggerDelayTime = 2
End With
Set oeff = osld.TimeLine.MainSequence.AddEffect _
(oshp, msoAnimEffectFade, , msoAnimTriggerWithPrevious)
With oeff
.Exit = True
.Timing.TriggerDelayTime = 10
.Timing.Duration = 0.1
End With
End If
Next oshp
Next osld
End Sub

jay freedman say:

The text that you cut to the clipboard is irretrievably destroyed, because a restart clears the computer's memory. Your only hope is to find a file containing some or all of the document as it was before the cut command.

If you have no known backup copy, use the Search box in the File Explorer to look for files that contain some text that you know is in the document. It will help to start the search term with the prefix content: to avoid looking only at file names.

If you haven't set up Windows 10's File History feature yet, do so. It won't bring back anything that's already gone, but it will provide automatic backups of documents as you edit them.

Counting Checkboxes in a Word Form - Microsoft Community

michaelcip say:

Counting Checkboxes in a Word Form

I'm putting together a report card that has 3 columns @ the end w/ a checkbox in each.  Each column/checkbox represents a letter grade that has points attributed to 'em.  The user will select on of the 3.  I want a calculation @ the bottom that tallies up the checkboxes in each column & sums their values.  Can anyone help me???  Thanks in advance, M.C.
doug robbins - mvp office apps & services (word) say:

Run a macro containing the following code on exit from each of the checkbox formfields (it assumes that the table is the first table in the document and that it contains a header row, so that the first row containing check boxes is the second row of the table):

Dim i As Long, j As Long, k As Long, m As Long, n As Long
k = 0
m = 0
n = 0
With ActiveDocument
    .Unprotect
    With .Tables(1)
        j = .Columns.Count - 2
        For i = 2 To .Rows.Count - 1
            If .Cell(i, j).Range.FormFields(1).CheckBox.Value = True Then
                k = k + 1
            End If
            If .Cell(i, j + 1).Range.FormFields(1).CheckBox.Value = True Then
                m = m + 1
            End If
            If .Cell(i, j + 2).Range.FormFields(1).CheckBox.Value = True Then
                n = n + 1
            End If
        Next i
        i = .Rows.Count
        .Cell(i, j).Range.Text = k
        .Cell(i, j + 1).Range.Text = m
        .Cell(i, j + 2).Range.Text = n
    End With
    .Protect wdAllowOnlyFormFields, NoReset
End With

-- Hope this helps.
