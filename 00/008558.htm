constant314 say:

I have a method that works approximately. 

 

I add two columns to the chart.  Let's call them corner_x and corner_y or just corners when talking about the two columns collectively.  They contain two points.  The coordinates of one point is the overall maximum x value and the overall maximum y value.  The coordinates of the other point is the overall minimum x value and the overall minimum y value.  By making the x span and y span of these corner points identical, the aspect ratio of the chart is constrained.  It may not be perfect, but it prevents things like 10:1

 

So,

let C_x (i) be the i'th column that contains x coordinates. 

          It is ok if C_x (i) is undefined for some i's.

 

let C_y (i) be the i'th column that contains y coordinates. 

          It is ok if C_y (i) is undefined for some i's.

 

          MAX_x (i) =   Max {C_x (i)}           /* The maximum entry in column C_x (i) */

          MIN_x (i) =   Min {C_x (i)}            /* The minimum entry in column C_x (i) */

          MAX_y (i) =   Max {C_y (i)}           /* The maximum entry in column C_y (i) */

          MIN_y (i) =   Min {C_y (i)}            /* The minimum entry in column C_y (i) */

 

          Max_x =        Max {MAX_x (i)}       /* overall maximum x entry */

          Min_x =        Max {MIN_x (i)}       /* overall minimum x entry */

          Max_y =       Max {MAX_y (i)}      /* overall maximum y entry */

          Min_y =        Max {MIN_y (i)}       /* overall minimum y entry */

 

Let R be the desired resolution.  For example R=5

 

Round the maximums up.

          Xmax = INT (Max_x /R)*R + IF (INT (Max_x /R)*R >= Max_x, 0, R)

          Ymax = INT (Max_y /R)*R + IF (INT (Max_y /R)*R >= Max_y, 0, R)

 

Round the minimums down

          Xmin = INT (Min_x /R)*R - IF (INT (Min_x /R)*R <= Min_x, 0, R)

          Ymin = INT (Min_y /R)*R - IF (INT (Min_y /R)*R <= Min_y, 0, R)

 

Find the maximum span

          Span = Max (Xmax-Xmin, Ymax-Ymin)

 

in the 1st row of the "corners" columns, put the point (Xmax, Ymax)

in the 2nd row of the "corners" columns, put the point (Xmax-Span, Ymax-Span)

 

You now have a line that spans the plot diagonally which tends to keep your plot with a fixed ratio.  I then change the format to remove the line and then color the markers slightly darker than the background, so I can find them but they are not distracting.

 

Now all you have to do is adjust the height and width of you chart to give you a 1:1 aspect and it will hold to that fairly well.

 

Logistically, assuming n columns, I reserved a 4 row x n column block above the columns that make up the chart.  For each column that has x values, the 1'st row is used to hold the calculation of the maximum of that column and the 2'nd row is used to hold the minimum of that column.  For each column that has y values, the 3'rd row is used to hold the maximum of that column and the 4'th row is used to hold the minimum of that column.  Then, to the left of the block I have a 4 row x 1 column block.  The top cell holds the maximum of the 1'st row of the 4 x n block (Xmax).  The second cell holds the minimum of the second row of the 4 x n block (Xmin). The third cell holds the max of the third row (Ymax) and the fourth cell holds the minimum of the last row (Ymin).
