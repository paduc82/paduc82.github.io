superfranky say:

hi jim,

there’s no quicker way other than create a new outlook profile. once the email address is changed, you can create a new outlook profile for the user.

here are the steps:
1. close outlook.
2. open control panel.
3. click mail.
4. click show profile.
5. click add to create a new profile and complete the auto account setup with the new email address.
6. change the default profile when starting outlook.

since you are using office 365 email services, all the old messages will be downloaded to the new profile automatically.
