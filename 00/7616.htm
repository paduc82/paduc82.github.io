When a volume license version of Office Standard or Office Professional Plus is installed on a new computer, a Start screen may appear each time you start Office. This can occur if you do not uninstall the preinstalled version of Office on the new computer before you install a volume license version of Office.
Check that your version of Office uses licenses by volume and take a screenshot, anyway I share how to validate.

In case it is not a volume license version, I recommend updating the Registry to remove the activation request for Office 365.

Close the activation window and all Office applications.

Right-click on the Start button Windows Start button in Windows 8 and Windows 10 (in the lower left corner of the screen) and select Run.

Type regedit and press Enter. Select Yes when asked to open the Registry Editor.

On the left side of the Registry Editor, on PC, navigate to the following registry key:

HKEY_LOCAL_MACHINE \ SOFTWARE \ Wow6432Node \ Microsoft \ Office \ 16.0 \ Common \ OEM

Right-click on the OEM value and click File> Export.

Save the key.

After backing up the key, select Edit> Delete.

Repeat steps 3 through 6 for the following key:

HKEY_LOCAL_MACHINE \ SOFTWARE \ Microsoft \ Office \ 16.0 \ Common \ OEM

Close the Registry Editor and restart Office.

daniël van den berg say:

It's interesting that there are four different way's you can set the color in Access, below example on setting the Background of a Textfield to Blue.

1. Using the Hex code # (converted by HexColor function)

2. Using the RGB function (color code)

3. Using the build-in Access color code

4. Using the VBA color constant

 

    ' source HexColor function
    'http://blogs.office.com/b/microsoft-access/archive/2009/10/26/convert-hexadecimal-color-codes-so-you-can-use-them-in-code.aspx
    Dim strColor As String
    Dim strR As String
    Dim strG As String
    Dim strB As String
    Dim HexColor As Long
   
    ' set Hex code for blue
    strHex = "#0000FF"
      
    'strip the leading # if it exists
    If Left(strHex, 1) = "#" Then
        strHex = Right(strHex, Len(strHex) - 1)
    End If
   
    'reverse the first two and last two hex numbers of the R G B values
    strR = Left(strHex, 2)
    strG = Mid(strHex, 3, 2)
    strB = Right(strHex, 2)
    strColor = strB & strG & strR
    HexColor = CLng("&H" & strColor)
   
     ' 1. Set blue with Hex code converted by HexColor function
    Me.txtHEX.BackColor = HexColor
   
    ' 2. Set blue with RGB function
    Me.txtRGB.BackColor = RGB(0, 0, 255)
   
    ' 3. Set blue with Access color code
    Me.txtAccess.BackColor = 16711680
        
    ' 4. Set blue with VBA color constants
    Me.txtVbColor.BackColor = vbBlue

 

Then for an overview of the color codes check below thread:

http://www.endprod.com/colors/

 

Note: the VBA color constants are limited to below row:

http://msdn.microsoft.com/en-us/library/x3h2c4d9(v=VS.85).aspx

You can also use System Color constants listed below:

http://office.microsoft.com/en-us/access-help/HV080755556.aspx
