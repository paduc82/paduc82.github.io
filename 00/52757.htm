Hi Shmuel,

Click this link:
https://www.microsoft.com/en-us/software-downlo...
to download the Media Creation Tool (Click on Download Tool Now), with that you can download the latest Windows 10 ISO (Select Create Installation Media for Another PC), you can create a bootable USB flash drive (min 8GB) using that tool

Then, Boot your PC from the Installation Media you just created (change Boot Order in your BIOS) to begin installing Windows 10

Since you previously had Windows 10 installed and activated on that PC during the installation process skip the steps which ask for a product key and select the option 'I am Reinstalling Windows 10 on this PC', and activation will not be an issue, your PC will have a digital entitlement stored on the Microsoft Activation Servers


If you have problems booting from a boot disc, you may have UEFI BIOS:
Insert the Bootable Installation Media, then go into your BIOS and make the following changes:
1. Disable Secure Boot
2. Enable Legacy Boot
3. If Available enable CSM
4. If Required enable USB Boot
5. Move the device with the bootable disc to the top of the boot order
6. Save BIOS changes, restart your System and it should boot from the Installation Media

tryx3 say:

Jannes,

The MS website [www.microsoft.com/en-us/software-download/windows7] only provides ISO downloads for "retail" licences not OEM licences such as ours.  Their error message normally explains this but it might have gone wrong when you used it.  There is no point trying again, it will not work for your OEM product key.

For retail licences, the MS website ought to provide ISO downloads but there are constant complaints in this forum that it does not work correctly and only provides downloads for a small number of languages [often just Korean, so if you want to see user reports then search the forum for Korean and you'll find them].

Amazon reinstallation disks

I have just been warned that

-  These reinstallation disks might not be made under MS licence.  Potential counterfeiters might not just be stealing from MS, they might also be planting carefully crafted & extremely well-hidden malware on your computer.

-  At least some of the reinstallation disks offered for sale are not reinstallation disks but merely images of a Windows installation coupled with some sort of automatic image restoration utility.  Such disks would not, could not, run the Windows installation section during which hardware is identified & the most appropriate hardware drivers installed [users would have to check for updated drivers manually for every hardware item].  Such disks could not be used as System repair disks either.

Before starting to post links to these disks in the forum, I checked through dozens of examples of disks for sale & I studied hundreds of customer reviews.  However, I did not consider checking for any evidence of MS licensing or that reinstallation disks might not be reinstallation disks.

I will update this notice when I find out one way or the other. 

If you want to keep things simple, you can buy a reinstallation disk [instead of downloading an ISO file then making your own]. 

  1. Amazon now stock reinstallation DVDs for only a few quid.  Amazon search results links - Amazon UK,  Amazon USA.  Their other international sites also stock them]. 
  2. These appear to be exactly what they claim to be as they do not include a product key and they are specifically described as being for counterfeit Windows.  The term reinstallation DVD is not formally defined but it is generally used to refer to an installation DVD that is supplied without any licence of its own [and hence without a product key].
  3. You would use the Product key on your Certificate of authenticity [CoA] for activation.  Your product key is valid for both 32 and 64 bit installations.  For computers with preinstalled Windows, the CoA can be found on the computer casing or inside the battery compartment.  Reinstalling from these generic reinstallation disks will not allow self-activation [self-activation can only be achieved using a preinstalled restore-to-factory facility or branded 64 bit Windows by checking your original purchase documentation, by looking for the computer specs on the computer maker's website or by running one of the Windows upgrade checkers because they report on this subject - see

Note that you can transfer your installation DVD to a USB if you want to - see Transferring an installation DVD to USB
