Unfortunately, formulas don't really work that way. It should be like this:

hansv mvp say:

RAND() generates random numbers between 0 and 1. Joeu2004 only uses them in his 'recipe' to sort the rows and pick the first 2000. It doesn't matter what size these numbers are.

If you want to return a random number between 3 and 7 that is a fraction such as 5.2939 or 3.1415, you can use (7-3)*RAND()+3

If you want to generate a random number between 3 and 7 that is a whole number, you can use =RANDBETWEEN(3,7)
