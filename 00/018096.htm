ken sheridan say: You should not need to manipulate the value of the section's Height property in code at all.  Merely setting the CanShrink property of the detail section and the COCCode control to True (Yes)  should be sufficient.  Note, however, that 'white space' does not shrink, so the controls should be vertically contiguous.  Also the section will not shrink if there are other non shrinking controls alongside the COCCode control, including any graphical controls such as  lines rectangles.

You need the control's Visible property to be set both ways, so all you should need is one line of code:

Me.COCCode.Visible = (Me.COCCode <> "1")

An alternative approach is to concatenate the values of the controls in an expression as the ControlSource property of a single text box.  There is a demo file Concat.zip in my public databases folder at:

https://skydrive.live.com/?cid=44CC60D7FEA42912&id=44CC60D7FEA42912!169

In the demo, if you select the option to concatenate values as a delimited or numbered list, and in the form which opens change the ControlSource of the txtConcat1 control on the left to:

=ConcatValues(Chr(13) & Chr(10),[txt1],[txt2],[txt3],[txt4],IIf([txt5]="Pears",Null,[txt5]))

this will separate the values with carriage returns/line feeds and if, as in the sample data, the final value is 'pears' will return this as a Null, so the final line will be suppressed.  If, having added the ConcatValues function to a standard module in your
=ConcatValues(Chr(13) & Chr(10),[YourFirstLineField],IIf([COCCode]="1",Null,[COCCode]))

You then simply need to make the control one line deep and set its CanGrow property, and that of the detail section to True (Yes).

You've enclosed the value of COCCode in quotes,"1", so I assume it's a text data type.  If it's a number data type the quotes should be omitted.

don cuthbert say:

Try installing Windows 7 to a small partition (or to a VHD file -- Native Boot) on the new motherboard device. Activate it with the Product Key. Windows 7 does not take very long to install and activate. Sometimes you have to use the automated phone activation to move the Product Key to the new device.

Then find gatherosstate.exe in the Windows 10 install media \Sources folder and copy it to a USB stick, or to the activated Windows 7 Desktop.  If you run gatherosstate from within the Activated Windows 7, it will create a small XML file named GenuineTicket.xml in the same location as the gatherosstate.exe.  Save that file for use in Windows 10.

Copy the GenuineTicket.xml file to the C:\ProgramData\Microsoft\Windows\ClipSVC\GenuineTicket\ folder of Windows 10 Home and restart. See if this will activate Windows 10 Home. Sometimes this has to be done more than once to work.  But unfortunately, there may be times when only a clean install will enable Windows to activate. Still, the XML file may help.

This GenuineTicket.xml method only works if the previous version of Windows (in this case, Windows 7) was activated on the same device when gatherosstate.exe was run. You do not have to do any Windows 7 updates, just activate Windows 7 Home. Windows 10 has to be Windows 10 Home edition to match the Windows 7.
