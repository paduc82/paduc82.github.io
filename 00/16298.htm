Word 2016 Page Number Issues

I cannot add page numbers to my documents in Word 2016 because the program keeps telling me to go to Office.com to get them.  I am definitely connected to the internet (because I'm posting this) and Word is updated to the latest version. Any ideas? Please help. 

doug robbins - mvp office apps & services (word) say:

You can create any sort of page numbering that you want without going to Office.com

Once you are in the Header\Footer pane, use the Tab key to put the selection where you want the page number to appear and then you can either

  1. Press ALT+SHIFT+p
  2. Press CTRL+F9 to insert a pair of field delimiters { } inside of which you type the word {page} and then press ALT+F9 to toggle off the display of the field codes in the document
  3. Go to the Insert tab of the ribbon and then to Quick Parts>Fields and select the Page field item.

If you want Page x of y page numbering, you can type the Page and of in the appropriate places and then either using method 2 above, but this time type {numpages} inside the { }

or use Method 3 and select the Numpages field.

suzanne s. barnhill say:

While it is true that you can insert a simple page number without using the built-in building blocks, the reason you are not seeing a gallery of building blocks is that the relevant file is damaged. Go to C:\Users\<UserName>\AppData\Roaming\Microsoft\Document Building Blocks\1033\16 and look for Built-In Building Blocks.dotx. Delete this file, and Word will recreate it from within the program.

If you don't find this file in this path (it may be different in Windows 10, search for Built-In Building Blocks.dotx. Be sure the one you delete is the one in your user profile and not in the Program Files folder.
