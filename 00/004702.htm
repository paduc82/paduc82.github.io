There must be something else at play...

The OLE engine in Mac Word/Office does not support linking to the PDF file type regardless of whether inserted as Pictures or as Watermarks. Links can be used only when inserting actual graphic file types [PNG, JPG, TIF, etc.] or a limited number of others, but not PDF.

Watermark pictures are embedded by default. When you insert a picture as a Watermark, after clicking the Select Picture button click the Options button in the lower left corner of the dialog window. Note that if you select a PDF file type the Link to File check box at the bottom of the dialog window becomes inactive.

The first suggestion I'd offer is that you confirm that the recipients are saving the document to their local drive. If they're opening the attachment in their email program it could explain why the image doesn't display.

If the image appears when you reopen the document it further suggests that it may be an issue on the receiving end involving the user installation. It also would be a good idea to confirm that your installation is fully updated by going to Word> About Word to determine the present Version which should be 16.13.1 or later.
