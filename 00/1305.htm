Mail Merge displays light grey in Compatibility Mode

I have a form which comes up in compatibility mode that i need to use mail merge in.  What do i need to do to this form.

jay freedman say:

Whether a document is in Compatibility Mode should not have anything to do with it displaying in light grey. Compatibility Mode simply means that the document was created in an earlier version, so features that can't be saved in that file format are disabled. If you want to get out of Compatibility Mode for the currently open document, go to File > Info and click the Convert button. When the conversion is complete, save the document; it will now be in .docx format, the native format for Word 2013. I suspect, though, that the text of the document will still be light grey.

The display of grey text may be the result of having the header or footer pane open. Word displays the header or footer in low contrast when the body text is active, and it displays the body text in low contrast when the header or footer is active. If the Header/Footer Tools tab of the ribbon is visible, click it and then click the Close Header and Footer button at the right end of the ribbon.

The text may be light grey if the style applied to the text is defined to use that color, or if the color has been applied as direct formatting (with the color button on the Home ribbon). Press Shift+F1 to open the Reveal Formatting pane, and at the bottom of the pane check the box for "Distinguish style source". Look in the Font section to see whether there's any setting of the font color.

There is at least one other place to look, but it's very obscure and would apply only to your computer. If the other items don't check out, post back for more.
