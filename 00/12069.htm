jeeped say:
I'm not sure I understand the term "freeze the pane".

If you want to freeze a row and/or column while you scroll the remainder of the worksheet, then select a cell that is one row down and/or one column to the right of the rows and/or columns you want frozen and use the View tab's Freeze Panes, Freeze Panes command.

If you want to have the two left columns frozen, select C1 and use the View tab's Freeze Panes, Freeze Panes command.

If you want the top row and the left column frozen, select B2 and use the View tab's Freeze Panes, Freeze Panes command.

jay freedman say:
The ProtectForm() macro will still intercept the Lock icon, which isn't on the Quick
Access Toolbar by default but can be added to it through the Customize dialog.
 
The ToolsProtect() macro needs to be renamed as RibbonReviewRestrictFormatting(), and it
will then intercept the "Restrict Formatting and Editing" choice on the Protect Document
button in the Review and Developer tabs of the ribbon.
