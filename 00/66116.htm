maria fab say:

Hello Jake,

It's true that it isn't possible to play sounds both from the speakers and the headphones at the same time, because only one can be set as a default. To verify, have you tried using a different headphones? First, let's see if we can locate the headphones by enabling the option to show disabled and disconnected devices on the Sound settings. Go to Sound> Playback, right-click on the default speakers, and then put a check mark on Show Disabled Devices and Show Disconnected Devices. The headphones should now appear. Right-click on it, and then click Enable. See image below.

Also, please let us know if there were any changes made on your computer prior to the issue. If it happened after an update, we suggest that you contact your manufacturer (OEM), or visit their website to download the latest drivers. Another method you can try is to use the Microsoft easy fix solutions. It will help diagnose and resolve software issues. Click on Find and fix problems playing sounds, or Find and fix problems with devices and hardware. Follow the on-screen instructions.
