Automatically capitalizing start of sentence

tjgarland say:

You may not have the correct AutoCorrect settings switched on.

Go to: FILE > OPTIONS > MAIL - then in the COMPOSE MESSAGES section select SPELLING AND AUTOCORRECT. Under AUTOCORRECT OPTIONS, select AUTOCORRECT OPTIONS to specify how Outlook corrects and formats text as you type. Check the box CAPITALISE FIRST LETTER OF SENTENCE. Go OK to everything and it should work on all new messages.

Sometimes it will not let you press the final OK due to a bug in Outlook 2010 that if you have PLAY REMINDER SOUND checked it will sometimes remove the reference to the sound so will not accept any changes to the options until this has been resolved. If you get an error message when pressing the final OK, then: go to OPTIONS > ADVANCED > REMINDERS and insert a sound reference (ie a WAV file). These can be found at C:\WINDOWS\MEDIA.
