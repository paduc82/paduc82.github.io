heather ackmann say:

There are a few ways to do this:

 

 If you want to keep it as one audio file played across all slides (it works, but is tedious):

1. Insert Tab-->Audio-->Audio from File

2. Select the audio file and click insert.

3. On the Audio Tools | Playback tab, select "Play Across Slides" from the drop-down menu in the Audio Options group.

4. Now,  you'll have to go through and calculate how long you want to spend on each slide and manually adjust the Transitions timing (Transitions Tab-->Check "after 00:00:00" box, but enter the time you want to spend on each slide in the appropriate box. This can be tedious depending on how long your presentation is).

 

If you don't mind splitting the audio files outside of PowerPoint (my preferred method):

For each slide...

1. Insert Tab-->Audio-->Audio from File

2. Select the audio file and click insert.

3. On the Audio Tools | Playback tab, select "Automatically" from the drop-down menu in the Audio Options group.

4. Now, go to the Transitions Tab and check the After 00:00:00 box, but leave the timing as "00:00:00". That way, when the audio is finished playing, the slide will advance automatically upon completion of the audio narration.

 

If you don't have any third-party way of editing the audio outside of PowerPoint, you can do this (though it is overkill):

1. Insert the full audio file (as discribed above) to each slide.

2. On the Audio Tools | Playback tab, click on Trim Audio.

3. Trim out the parts of the audio you don't want played on the slide.

4. Go toe the Transitions Tab, and check the After 00:00:00 box, but leave the timing as "00:00:00". That way, when the audio is finished playing (the trimmed part, that is), the slide will advance automatically upon completion of that portion of the audio narration.

 

steve rindsberg say:

In case you want to go with Heather's "edit sound outside PPT" method, Audacity is a very capable (and free!) audio file editor.  Google "audacity" and you'll find it right away.

To use it with MP3s, you'll need to install something called a LAME decoder.  The documentation describes how to do this and IIRC, there are links to the decoder on the Audacity site.

scoobyman---01 say:

Hi Zestje

 

Sorry to hear that you are disappointed with GPS experience here. A point to remember with use in different cars is that some modern cars are fitted with a special windscreen that cuts down the amount of UV (Ultra Violet) rays that will be transferred through the windscreen which help cut down on glare. These windscreens are called Athermic, or UV Coated and will dampen/reduce GPS signals coming through the windscreen, so if you have an Athermic windscreen, it means you will not receive the same quality of signal as you would do by taking the GPS outside of the car. A dedicated GPS unit has room for a larger satellite antenna than is possible within the confines of a feature packed phone.

 

You might say that it defeats the object but I always use a Bluetooth GPS SIRF Star III receiver with OVI Maps whilst in a car for this reason.

(I was somehow was off by one column in my original post. I''ve deleted and corrected it here.)

I've inferred a specified maximum of three tasks were to be assigned so I hard-coded the reiterative routines rather than loop them.

Right-click the worksheet's name tab and choose View Code. When the VBE opens up paste the following into the pane titled something like Book1 - Sheet1 (Code),


Option Explicit

Private Sub Worksheet_Change(ByVal Target As Range)
    If Target.Count > 1 Then Exit Sub
    If Target.Row > 1 And Target.Column > 3 And (Target.Column Mod 3) = 2 Then
        Dim v As Variant, r As Long
        On Error GoTo FallThrough
        Application.EnableEvents = False
        r = Target.Row
        If CDate(Cells(r, 11)) < CDate(Cells(r, 8)) Then
            v = Cells(r, 9).Resize(1, 3).Value
            Cells(r, 6).Resize(1, 3).Copy Destination:=Cells(r, 9)
            Cells(r, 6).Resize(1, 3).Value = v
        End If
        If CDate(Cells(r, 8)) < CDate(Cells(r, 5)) Then
            v = Cells(r, 6).Resize(1, 3).Value
            Cells(r, 3).Resize(1, 3).Copy Destination:=Cells(r, 6)
            Cells(r, 3).Resize(1, 3).Value = v
        End If
        If CDate(Cells(r, 11)) < CDate(Cells(r, 8)) Then
            v = Cells(r, 9).Resize(1, 3).Value
            Cells(r, 6).Resize(1, 3).Copy Destination:=Cells(r, 9)
            Cells(r, 6).Resize(1, 3).Value = v
        End If
    End If
FallThrough:
    Application.EnableEvents = True
End Sub

Tap Alt+Q to return to your worksheet. Changing (or adding) a date in columns F, I or L will result in a horizontal sort of the grouped data for that row.
