That's two questions.

The answer to the first question is: 

It is never a good idea to allow Windows Update to download/install any hardware drivers for your system of unspecified make and model with unspecified operating system and Service Pack.

Microsoft doesn't know what the best drivers are for your system and will often send you old or incompatible drivers which is one reason there are so many topics in the community that start with "Windows Update Broke my Computer...".

You should only get hardware driver updates from the manufacturers WWW Support site for your system of unspecified make and model with unspecified operating system and Service Pack.

If you system is working and you want it to keep working it would be wise to configure Windows Update to never offer hardware drivers from Microsoft and this link tells you how to do that:

https://support.microsoft.com/en-us/kb/2500967

Since it is also wise to look at every update you see offered (never just blindly install updates) if you recognize anything that looks like a hardware driver you should right click and hide that update and hope you never see it again.
