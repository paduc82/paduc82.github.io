    In the releases that followed the first one thing have become much clearer:

    Windows 10 is the new name of Windows. It's not that much of a version number anymore. Windows is now a service.

    Quality update are rolled out every month and feature updates twice a year (mostly).

    what vendors of registry cleaning software try to convince you of,

    bossdweeb say:

    "FPS which makes gaming more realistic."

    FPS = frames per second. This does not make games more realistic as such. A low fps makes games

    less smooth, and a very low fps will show as lag, both in the game and mouse movement.

    Higher fps also helps reduce blur when panning, particularly if over 60fps (when having a monitor with

    a 120 Hz refresh rate also helps)

    30 fps would be the minimum desirable frame rate, but some people are not happy with a frame rate below 60 fps.

    The game in question also factors into what the lowest desirable frame rate is. Some games look smoother

    at 30-40 fps than others, but again, different people have different perception.

    I'm generally happy with a low around 45+ fps, but not too fussed  if it drops to 35fps now and then,

    depending on the game.


    "6GB of RAM"

    Laptop mfg's tend to throw in poor RAM configurations. They do this to make the machine specs look better,

    but it is best to keep the RAM balanced, so if using four sticks, as you would have to do with 6GB

    = 2x2GB + 2x1GB, it would be better to have 4X2GB .

    Or worse -  Three sticks of 2GB. Three sticks will make the RAM run in the less efficient single channel mode

    vs  dual channel mode. To run in dual channel mode the machine needs either 2 or 4 sticks of RAM.

    Using two 2X4GB sticks is better than using four, as this puts less stress on the memory controller.

    - Having said that, FSX may get by on 4GB total ?

    - I've never used a modern flight sim (CFS2 & IL2), but I understand that flight sims are somewhat cpu bound.

    That is to say they rely heavily on the cpu for realistic flight calculations, so yes an i5 would be better

    than an i3. An i7 would only be better if the flight sim is capable of using Hyper-Threading.

    - However, if you are using  high resolution scenery, and other addons/mods  you may also need a high end gpu.

    Laptop gpu's are down-clocked vs their desktop equivalent (same model number with a M at the end) to

    help keep power use and temp down  (laptops are notorious for overheating when playing games or sims

    with high system requirements for long periods of time, ie- over an hour).

    Therefore you would need a laptop gpu one step higher than what you would use for the same level of

    graphics 'eye candy', and realism than someone using a desktop PC would achieve if you plan on connecting to

    a 24" monitor.

    This is not a recent game, but because of the addons the specs could be a fair bit higher than when first released,

    so the various flight sim  web site forums would be the best place to find out exactly what this game requires when using the addons you mention.

    - You can also compare the AMD and Nvidia options by reading reviews for their current model laptop gpu's.

    While the reviews are not likely to include a flight sim in the gaming benchmarks, you should still be able

    to extrapolate from the games they do benchmark when comparing the 'recommended' system specifications

    for those games with the FSX requirements (after factoring in the potential added stress of 'modern' addons).

    The Nvidia GTX 970m is the current model below the top end Nvidia gpu (GTX 980m).

    You may still be able to get by with a  GTX 960m, or even  a GTX 950m, but I'll start you with the

    review for the GTX 970m (just to tease you :D ).

    notebookcheck.net/NVIDIA-GeForce-GTX-970M

    If you have not already been here -   http://www.flightsim.com/vbfs/content.php
