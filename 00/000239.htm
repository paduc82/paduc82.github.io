vanderghast say:

If you are fine with working with VBA, you can, with JET ( + Access) define a VBA function, in a standard module:

 

Public Function IsPartialMatch( stringa AS string, stringb As String) AS Boolean

... write code which returns true if stringb is a 'partial match' to stringa, false otherwise

End Function

 

and use an SQL statement like:

SELECT *

FROM table1 AS a INNER JOIN table1 AS b

WHERE IsPartialMatch( a.title, b.title)

 

Note that the difficulty, in this case, is to define YOUR logic about what is a partial match, and what is not.
