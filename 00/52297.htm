big and tasty say: Okay, here's what I did and learned:

1.  I replaced my DVD burner with my original CD-R that came with machine and disconnected the 2nd CD-R.

2.  I disconnected my 2nd hard drive, partly to minimize current draw from the power supply, but also to keep the bootup clean and simple between a C: (hard) and D: (CD-R) drives.

3.  I re-tested my SDRAMs using the thorough, deep test mode.  To my surprise I found not 1 but 2 bad SDRAMs!  I replaced them with the original 128MB SDRAMs, and re-ran the memory testing using both the Windows Memory Diagnostocs and Memtest86+.  All of the 512MB RAM is perfect now.

4.  I booted up WinXP again and got to the "Setup is starting Windows..." msg.  This time, I waited for about 15 mins and to my surprise the setup screen came up!  There is a long time period between when the "Setup is starting Windows..." msg starts and when the setup menu actually comes on-screen.  I was able to partition and format my hard drive.

Lessons learned:
- Keep the number of peripherals down to a minimum so that it keeps the unknown hardware failures from confusing the troubleshooting issues.
- Testing the RAM in deep testing mode is important, and testing it multiple times.
- It could be that more than 1 RAM bank can be bad so test RAM thoroughly.
- Wait a long time for the Windows Setup menu to come up.  It won't be instantaneous, but it will come on sooner or later.
