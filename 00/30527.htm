a. user say:

Some of them were pdf files and other important word documents ... They were on desktop itself. 

That's not where you're supposed to store your personal files ... even though everyone does it, including me.

We're supposed to store our personal files in our personal folders, i.e.:

C:\Users\your user account name\Documents

C:\Users\your user account name\Music

C:\Users\your user account name\Pictures

C:\Users\your user account name\Videos

The desktop is for shortcuts to these files. That's how Windows was designed to be used, which is why Microsoft is careful to not touch those folders during an update.

(BTW - You  have a personal desktop folder, too. It's at

C:\Users\your user account name\Desktop

What you see on your desktop is actually a combination of four different folders, one of which is above.)

Like I said, despite what Microsoft intends, everyone stores files on their desktop anyway. And the truth is, 99% of the time it makes no difference. The 1% of the time where it does make a difference is when your computer is updated.

Sometimes, in order to update your computer, system files and folders have to be erased or replaced. If you store files in a system folder, they could get erased by an update. I would explain how I discovered this, but the memory is still painful.  ðŸ˜¥

Here's where you files might be:

  • in the Recycle Bin - unlikely, but it doesn't hurt to check
  • inside the windows.old folder - if you have one
  • search for your files by name and you might find them - also unlikely, but doesn't hurt to check

If neither of those provide results, you might have success with file recovery software, like Recuva, which is trustworthy, popular and free.

When you upgraded from a previous version of Windows or receive a new computer preinstalled with Windows 10, what happened is the hardware (your PC) will get a digital entitlement, where a unique signature of the computer will be stored on Microsoft Activation Servers. The Windows 7 or Windows 8 genuine license you were previously running will be exchanged for a diagnostics key.

Anytime you need to reinstall Windows 10 on that machine, just proceed to reinstall Windows 10. It will automatically reactivate. You will prompted to enter a product key a couple times through the installation, click I don't have a key and Do this later.

There is no need to know or get a new key, but if you have issues with Digital Licensing or the Activation Troubleshooter, you can utilize an existing Windows 7 or Windows 8/8.1 product key you already own to reactivate if necessary. or use the reset function in Windows 10.

Go to a working computer, download, create a bootable copy, then perform a clean install.
