BCCode: 50     0x00000050

Cause

The BCCode 50 usually occurs after the installation of faulty hardware or in the event of failure
of installed hardware (usually related to defective RAM, be it main memory, L2 RAM cache, or
video RAM).

Another common cause is the installation of a faulty system service.

Antivirus software can also trigger this error, as can a corrupted NTFS volume.

Resolving the Problem

Resolving a faulty hardware problem: If hardware has been added to the system recently,
remove it to see if the error recurs. If existing hardware has failed, remove or replace
the faulty component. You should run hardware diagnostics supplied by the system manufacturer.
For details on these procedures, see the owner's manual for your computer.

Resolving a faulty system service problem: Disable the service and confirm that this
resolves the error. If so, contact the manufacturer of the system service about a possible
update. If the error occurs during system startup, restart your computer, and press F8 at
the character-mode menu that displays the operating system choices. At the resulting
Windows Advanced Options menu, choose theLast Known Good Configuration option.
This option is most effective when only one driver or service is added at a time.

Resolving an antivirus software problem: Disable the program and confirm that this resolves
the error. If it does, contact the manufacturer of the program about a possible update.

Resolving a corrupted NTFS volume problem:  Run Chkdsk /f /r to detect and repair disk
errors. You must restart the system before the disk scan begins on a system partition. If
the hard disk is SCSI, check for problems between the SCSI controller and the disk.

Finally, check the System Log in Event Viewer for additional error messages that might help
pinpoint the device or driver that is causing the error. Disabling memory caching of the BIOS
might also resolve it.

dominicp say: WMC scans all frequencies for your country, not just the frequencies from your regional transmitter as Hauppauge's WinTV (and most proprietary TV software) does. That is why it detects many transmissions you may not want. There is no way to limit WMC's scan.

Look carefully at your surroundings. You may have blocked and/or reflected signals. A good local aerial installer should be able to spot problems and position your aerial for best reception.

You can manually add any tuners the initial tuner scan does not detect. This must be done before the initial scan for channels.

Channels detected can be disabled, and you can adjust the sources selected for each channel.

If WMC has selected more than one frequency for a channel you need to change the selection and make sure that the tuners selected match for each channel broadcast from a mux. This can be tedious.

You need to know which channels are broadcast (and the mux frequencies) from each tower/MUX combination that WMC detects.

You will want to disable the regular scan for new channels. See http://www.thegreenbutton.tv/forums/viewtopic.php?f=5&t=826 .

dominicp say:

I know nothing about your location, expected channels to receive, nor any permanent or temporary issues which may affect your reception. You need someone with local knowledge of actual and potential reception problems. 


A clear line of sight to the transmitter - master or relay - does not necessarily mean there are no local problems affecting your received signal. With multiple transmitters in range for channels there can be additional problems with channel setup.


If WMC reports a channel, it must have detected a signal for that channel. It may not have set up the channel correctly: if not, you will probably see "No signal" or have intermittent issues with reception or recording. In that case you will have to correct it in Tasks-Settings-TV-Guide-Edit Channels then Edit sources for each channel. For that, you will need the technical details for each channel and an understanding of how muxes work.


If you explore the Settings menus in WMC you will find how to scan for additional channels - and may well find other customisations you will like.
