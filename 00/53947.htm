Here are the steps according to Microsoft:

In Windows 8.1, the Windows update notifications are not available in the start screen. This is because most of the new Windows apps run in full screen and users might be interrupted by these Windows update notifications. So in Windows 8.1, apart from update notifications in the log on screen, the only other way to check if updates are available to check for them manually in Windows update section of control panel.
