miguel.sanchez17 say:

I had the same problem and fixed it by creating a new profile. Here's what I did in more detail:

  1. Log off the PC if you're already logged in.
  2. Log into the PC using another account (my wife's account, also an admin on the PC)
    • It's important that the other account be an admin account to do what you need for this fix
  3. Check if OneNote works for this user. If so, then it's your profile that's the problem (as it was for me) and not your PC, Windows 10, or OneNote (phew!)
  4. I went to c:\users and copied the folder belonging to my profile (eg. c:\users\migu_000) to another parent folder (eg. c:\temp)
  5. Searched for the Settings to manage and edit other users on this PC
  6. Found my login and chose the option to remove/delete
    • There's a warning pointing out that all your files and settings that are part of this user's profile will be deleted. This is no joke, so make a backup if you haven't done already per step 2.
  7. Then re-create your account on the PC (using the same tool from memory)
    • You'll just be asked for an email address, if you're using Microsoft accounts as recommended
    • Once created, be sure to promote the account from a standard user to an administrator if that's what you want
  8. Now you can log into the PC with your own account.
    • It will take a while to create a new profile for you, and you'll need to take the time to restore your web links and other application settings...but OneNote worked for me at this point. :) You have the copy of your original profile, too, if you need to get files or links or whatever from that.

All the best.
