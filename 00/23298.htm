On the form when the control has focus (activiecontrol) the DatePicker icon displays next to it.

Is it possible, when the control has the focus:

to expand the DatePicker and set the focus to the expanded DatePicker

hansv mvp say:

You can use the On Got Focus event of the control. Let's say the control is named BirthDate. The code would look like this:

 

Private Sub BirthDate_GotFocus()
    RunCommand acCmdShowDatePicker
End Sub
