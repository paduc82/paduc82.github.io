Should you?  Tell us why you want to.

MS defaults to installing 32 bit Office, even on 64 bit Windows. They do this because many software makers, INCLUDING MS!, have simply not bothered to create 64 bit versions of their addon tools. So if you switch to 64 bit you could potentially lose functionality!

In real world tests, no significant (user perceivable) performance improvements are visible after upgrading to 64 bit.

MS only recommends upgrading to 64 bit when:

  • file size is approaching 2GB on disk
  • application is using approaching 2GB of RAM when file is open

jasperp say: This is the English Excel forum. The German version can be found at

To find the last date of 2012, use a array formula (Matrixformel) like:

EN:   =MAX(IF(YEAR($A$1:$A$99)=2012,$A$1:$A$99,""))
DE:   =MAX(WENN(JAHR($A$1:$A$99)=2012;$A$1:$A$99;""))

Enter with Ctrl+Shift+Enter (STRG+UMSCHALT+EINGABETASTE).

You can combine this, for example, with a VLOOKUP (SVERWEIS).
