Though I am not able to understand the question completely, let me attempt to answer -

1.  If you are entering a date in D4, the cells in a column should change and date should not be visible but hidden. It has to be done in 2 steps

bob jones aka: cybertaz say:

Language is an attribute of the individual characters in each document... If the documents you receive are indicated as being US English, it's because that is the Language formatting used in those documents. It isn't because the settings on your installation are reverting or being changed... Nor will changing you Default settings have any effect on existing documents.

A user easily can specify the default Language to be used for creating new documents, but there is no built-in function to automatically change the Language formatting of existing documents. That would have to be done programmatically, but doing so could create other problems if a document's Language shouldn't be changed or intentionally contains text in a mixture of languages.

Which Spelling & Grammar Tools are used for checking is determined by the Language(s) used in the document. Word recognizes the Language formatting as it checks & switches to the appropriate set of Proofing Tools on the fly. IOW, you can tell Word which Language to apply to the text, but you can't tell it to use Proofing Tools that don't correspond to the Language formatting.

No judgement intended, but if the documents you receive should be in a Language other than what is being used, it might be helpful to have the originators trained to use the correct language when producing those documents.
