pesach shelnitz say:

Hi,

In the Word 2007 user interface there is no option to enable or disable the appearance of pop-ups for the automatic completion of dates, etc., but, as you said, it can be done in the Visual Basic Editor. To enable auto-completion, press Alt+F11 to open the Visual Basic Editor. In the Visual Basic Editor, press Ctrl+G, which opens the Immediate pane. Then, in the Immediate pane, type , and press Enter.

To disable auto-completion, follow the same procedure, but type Application.DisplayAutoCompleteTips = False.
