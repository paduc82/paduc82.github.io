Assuming that you are using Word's outline (multilevel) numbering feature, you can insert a LISTNUM field to add more outline numbers in the same paragraph.

To insert a LISTNUM field at the insertion point, you can do the following: Click Insert tab | Quick Parts | Field. Locate LISTNUM and select it. Make sure that "List name" is set to "(none)" and then choose the desired level in the "Level in the list" box. When you are done, click OK.

ruwim.b say:
In reply to deleted message

Try first to reset all local Skype settings.

Quit Skype or use Windows Task Manager to kill any Skype.exe process. Go to Windows Start and in the Search/Run box type %appdata%\skype and then press Enter or click the OK button. The Windows File Explorer will pop up. There locate a folder named “Skype for Desktop”. Rename this folder to something different, e.g. Skype_old.
