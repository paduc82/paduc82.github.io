Make sure that the tablet screen button is turned on...tiny switch on left side of PC keyboard adjacent to the left-hand shift key... you should see a red dot if tab screen is turned on. 

N.B. The following list of driver updates is valid as of 8/24.  Future updates (IF ANY???) will alter this list.

Go to ASUS support site and download the following and be certain that you are downloading the file with the highest rev number and latest creation date -- both are on the menu screens.

Download:

ATKPackage_81_64_VER100038

Audio_Realtex_Win7_Win8_ VER6016870

Bluetooth_Intel_Win8_64_VER3113307362

Chipset_Intel_Win8_64_VER_Z9301021

Closelid_Win8_64_VER118 (this is an important one)

DPTF_Intel_Win7_8_VER118 (also important)

LAN_Asix_Win_8_Ver1010

Power4GearHybrid_Win8_64_VER 308  (extremely important)

SmartGesture_Win8_64_VER224

VGA_Intel_Taichi_Win7_64_Z91702857

Unzip and run setup in each instance... reboot upon each request to reboot.

OK -- here is the critical part.

Go to Power4Gear and establish settings for battery and wall power.  That was the easy part.

Go to advanced settings and methodically choose each option and when it asks what function you wish to attach to closelid -- select 'do nothing'.  Each other setting that has the word 'lid' in it should be set to 'off' or 'do nothing'.  I recommend confirming these settings after exiting Power4Gear by going to Windows Settings and go through each setting carefully.  I recommend ditching sleep and hibernate...has caused issues for me.

N.B. Various upgrades and installations can change these sensitive lid settings.  The foregoing method has worked for me and I have had stable results since March.

I strongly recommend that you also download and install TouchFreeze, a shareware program eliminates unexpected cursor movement while using PC keyboard.  It locks out the touchpad when you are typing and restores function when keyboard activity ceases.

I keep a copy of the zip files on my SSD as they get used a bit more frequently during upgrades.  Also check the Device Manager for yellow warning triangles....particularly with the ATK driver, the HID driver and occasionally the USB drivers.
