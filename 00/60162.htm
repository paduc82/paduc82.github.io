a. user say:

Hello,

To assist you better, I would like to ask if you are making the changes as an Administrator.

I suggest you to follow the methods provided below and check if it helps.

Method 1: Change the Background form File/Windows Explorer.

  1. Login as Administrator and copy the two images which you want to set as Background and paste it in the path: C:\Windows\Web\Wallpaper\Windows
  2. Select both the images, right click and select the option Set as desktop Background.
  3. You would now get separate backgrounds for each of your monitors. These images are saved in %USERPROFILE%\AppData\Roaming\Microsoft\Windows\Themes before they are applied as desktop background. The names of these files are Transcoded_000 and Transcoded_001 without any extension. So let’s say, you want to swap the images on the monitor, simply rename 0 to 1, 1 to 0 on the file name. Finally sign out and sign in back from Windows to reflect the changes.

Method 1: Run SFC (System File Checker) scan.

  1. Press Windows + X keys and click command prompt(admin).
  2. Type “sfc /scannow” and press Enter.
  3. After the scan is complete, type “exit” and press Enter.

For more information about SFC scan, click on the link below:

https://support.microsoft.com/en-us/kb/929833

Method 2: Run the DISM TOOL:

  1. Press Windows + X keys and click command prompt(admin).
  2. Type the following commands in command prompt one by one and press Enter after each command.

Dism /Online /Cleanup-Image /ScanHealth
Dism /Online /Cleanup-Image /RestoreHealth

Also, suggest you to run a Complete system Scan for virus using Windows Defender or any other antivirus software (if any) installed in your computer.

Hope this helps. Keep us posted on the status for further assistance.
