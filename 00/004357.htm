Sub bluesky()
With Application
.ScreenUpdating = False
.DisplayAlerts = False
.EnableEvents = False
Dim area As Range, cell As Range, xRow&, strSheet$, asn$, iCol%
asn = ActiveSheet.Name
strSheet = "zzzCompilation"
On Error Resume Next
Sheets(strSheet).Delete
Err.Clear
Sheets.Add(After:=Sheets(Sheets.Count)).Name = strSheet
Range("A1:E1").Value = Array("Last Name", "First Name", "Address", "Title", "Group Name")
xRow = 2
For Each area In Sheets(asn).Columns(1).SpecialCells(2).Areas
iCol = 1
For Each cell In area.Cells(1, 2).Resize(area.Rows.Count, 1)

iCol = iCol + 1
Next cell
xRow = xRow + 1
Next area
Columns.AutoFit
.EnableEvents = True
.DisplayAlerts = True
.ScreenUpdating = True
End With

hi helpstar55,

 

there are two way to delete junk email retention tag in office 365 admin center: remove it from “default mrm policy” in retention policy or delete it in retention tag.

1. if you remove the tag from “default mrm policy” in retention policy as the screenshot below, the junk email folder will still use the junk email retention policy.

because when a retention tag is removed from the retention policy applied to a mailbox, the tag is no longer available to the user and can't be applied to items in the mailbox. but the existing items that have been stamped with that tag continue to be processed by the managed folder assistant based on those settings and any retention action specified in the tag is applied to those messages.

2. if you delete the tag in retention tag as the screenshot below, there is no retention policy tag in the junk email folder so it will use default policy tag (its type is default in the “default mrm policy”).

because when you delete a tag, the tag definition stored in active directory is removed. this causes the managed folder assistant to process all items in a mailbox and restamp the ones that have the removed tag applied. depending on the number of mailboxes and messages, this process may significantly consume resources on all mailbox servers that contain mailboxes with retention policies that include the removed tag.
