samir gunic say:

Unless someone proves otherwise, I think it's safe to say that you cannot do advanced search queries in the new Outlook.com.

The best you can do is switch back to the old Outlook.com by opting out of the Beta program.

You can then do a search for a keyword and then specify a date range. The old Outlook.com interface has UI elements that allow you to visually specify the range.

Sadly, you cannot only specify a date range, you must also do a search for some keyword. I tried to make my keyword all-inclusive by searching for commonly found words such as "and" or "I" or "hello". You cannot use wild card characters such as * (any).

Guess what? You can use the same syntax on the old Outlook.com as you would normally use with the Outlook desktop application. Undocumented side effect! So you can use any one of the following.

received:>=2015

received:1/1/2015..12/31/2015

The former will give you an open-ended range, including everything from year 2015 and later. The latter allows you to specify specific lower and upper limit. There are a couple of additional ways to specify a range, see the link above for those.

If you are using these search parameters and operators to specify the range, then you can ignore my comment about having to do a search for a keyword in order to be able to specify the range. That is only true if you want to specify the range using the UI elements on the side panel. There is no other way to display it.
