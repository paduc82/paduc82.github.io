In this Windows 10 version of IE, apparently the only way to determine this information is to check the source of the Shortcut that you use to open IE 11.

Right click the shortcut for IE and select Properties / Shortcut tab.

Look at the Target box.

If it shows "C:\Program Files\Internet Explorer\iexplore.exe" then it is the 64bit version.

If it shows "C:\Program Files (x86)\Internet Explorer\iexplore.exe" then it is the 32bit version.
