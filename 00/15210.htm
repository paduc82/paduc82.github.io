jomel ber say:

Hi Rehablftikhar,

I understand that you're having issues with Outlook application. With regard to your concern about Gmail messages that weren't transferred, once you've added your Gmail account in Outlook, a copy of your Gmail messages will be stored in a file that Outlook creates on your desktop. The file is called an Outlook Data File (.pst). If you receive more email to your Gmail account after you added the account, those additional messages won't be copied to your .pst file, but you can always repeat these steps to copy them:

  1. In the Outlook app, choose File > Open & Export > Import/Export.

  2. Choose Export to a file, and then choose Next.

  3. Choose Outlook Data File (.pst), and then choose Next.

  4. Select the account you want to export by choosing the name or email address of your Gmail account, and verify the Include subfolders box is checked (this ensures all your email is copied), and then choose Next.

  5. Choose Browse to select where to save the Outlook Data File (.pst) and to enter a file name. By default, the name is "backup," so choose another name that's easy to remember, such as Gmail. If you are exporting to an existing Outlook Data File (.pst), under Options, specify what to do when exporting items that already exist in the file. Choose Finish to continue.

  6. In the Add an optional password dialog box, add a password if you want. Choose OK to save the data file even if you leave the password boxes empty. Without further notice, Outlook begins exporting the data immediately. It doesn't give you a message when it's done, either.

  7. Go to the location where you saved the .pst file to confirm it's there.

Regarding with the name on your sub-folder (Sent Items), you can sort the arrangement from From field to To field to show the recipient name instead of the sender name. Here's how:

  1. Select the sub-folder you want to manage.
  2. Click the View tab.
  3. In the Arrangement column, select To.

If you have questions, please don't hesitate to post back.
