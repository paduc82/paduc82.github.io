Opening a .wiz file - Microsoft Community

juliapatterson_688 say:

Opening a .wiz file

I tried opening the following document which is identified as a Microsoft Word Wizard type.  My PC says I'm

missing something to view it.

Administrative_Benefit_Information_6/12/09

Where do I look and download this file?

jay freedman say:

A file with a .wiz extension is exactly the same format as a Word 97-2003 template file, and Word 2016 should open it without any errors or messages. What makes it a "wizard" is macro programming that leads you step-by-step through creating a specific kind of document.

There are several possible reasons that Word could balk at opening a .wiz file:

  • The file was downloaded from another computer, so Windows has marked it as a "foreign" file. In the folder where the .wiz file is stored, right-click the file, choose Properties, and click the Unblock button (if there is one).
  • The file is stored in a folder that isn't a "trusted location", and your macro settings block the execution of macros from untrusted files. If you're certain that the wizard is trustworthy, move it to your Templates folder (which is usually %appdata%\Microsoft\Templates) and double-click it there.
  • Your settings explicitly block opening Word 97-2003 templates. Go to File > Options > Trust Center > Trust Center Settings > File Block Settings. By default, only the check boxes in the Open column for binary documents and templates from Word 2, 6.0, and 95 are blocked. If any other boxes in that column are checked, especially Word 97, 2000, or 2003 binary documents and templates, uncheck them and try again to start the wizard.
  • It's possible that Visual Basic for Applications (VBA) is not installed or not enabled in your Office installation. Normally this occurs only in enterprises where the IT department uses special tools to lock down Office. If you have a personal version of Office, try running an online repair of the installation.
