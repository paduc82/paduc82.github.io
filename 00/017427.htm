it's not actually a question of LibreOffice compatibility with Word but with the document format the data are saved to.

DOCX is part of Office Open XML format standardized as ISO/IEC 29500. The purpose of standardization is to achieve interoperability: if two applications implemented that standard correctly then documents interchanged between them would look the same.

    Put the code below in the module "ThisOutlookSession".

    Every time you open an appointment outlook is minimized. If you did not want to have this behavior, press the CTRL key before open an appointment.

    Andreas.

    Option Explicit

    Private Declare Function GetAsyncKeyState Lib "user32.dll" (ByVal vKey As Long) As Long
    Dim WithEvents Insps As Inspectors

    Private Sub Application_Startup()
      Set Insps = Application.Inspectors
    End Sub

    Private Sub Insps_NewInspector(ByVal Inspector As Inspector)
      If CtrlPressed Then Exit Sub
      If TypeOf Inspector.CurrentItem Is AppointmentItem Then
        Application.ActiveWindow.WindowState = olMinimized
      End If
    End Sub

    Private Function CtrlPressed() As Boolean
      'True wenn die Strg-Taste gedrückt wurde
      Const VK_SHIFT = &H10 ' Shift Taste
      Const VK_CONTROL = &H11 ' STRG Taste
      Const VK_MENU = &H12 ' Alt Taste
      CtrlPressed = GetAsyncKeyState(VK_CONTROL) <> 0
    End Function

    EDIT: 
    We may have found a possible solution although it is a bit confusing. 

    There seems to be 3 ways to add additional mailboxes/accounts to outlook. 

    1. Right click on the account name active in outlook, select Data File Properties -> Advanced-> Advanced and ADD...to add account. 
    2. File menu -> Account settings button->Account Settings->E-mail tab ->NEW
    3. Close Outlook and go to Control panel-> Mail(32bit)->"E-mail accounts"->NEW

    Option 3 seems to allow for additional SEPARATE accounts to be added. 
    Option 2 allows for shared accounts to be added from within Outlook, but difference between that and option 3 is that option 3 seems to separate the accounts more, and option 3 does not work if Outlook is running.
