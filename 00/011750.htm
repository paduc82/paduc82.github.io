bernd p. say:

I dare to disagree.

Clipboard

The clipboard is used to transfer data between applications that DON'T use or understand the same formats. The source application adds data to the clipboard in the formats it understands. It's the source's responsibility to put as much data in the clipboard as possible to ensure the content can be reproduced on the other end.

Target applications request the clipboard data in the format they understand.

Example for a Clipboard content by Word:

Example for a Clipboard content by OneNote:

Two differences worthwile to mention:

OneNote creates some versions of the Bitmap type.

OneNote doesn't create Rich Text Format, which is IMO the hidden reason for your problem.

BUT Word and OneNote both create the simple Text format.

And MS Office doesn't use Rich Text Format as its standard, it uses HTML (which is produced by OneNote too).

You can see this by using Paste Special in Word.

So a text paste from OneNote to Word works.

So it looks like your messengers should request just Text (or HTML) instead of requesting Rich Text Format.
