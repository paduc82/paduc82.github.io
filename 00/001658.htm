Microsoft: Please do not Force Driver Updates Through Windows Update

Recently, you have started pushing proprietary driver updates through Windows update. This is good! But there are a number of laptops and other PCs that require specialised drivers obtained from the manufacturers websites for certain components to work correctly. Often, these are not entirely up to date, which means that Windows Update will constantly try to overwrite the working driver with one that fails to work. I have this problem with my Realtek sound card, which works both with no drivers installed, and my laptop's manufacturers drivers installed, but not with the drivers Windows Update provides. Others are experiencing similar problems with their graphics drivers, which is a more serious issue.

Please allow us to turn off driver updates through Windows Update so that we can have the choice of exactly which drivers we install on our computers.

suzanne s. barnhill say:

I'm not sure exactly what you mean but I think what you are asking is to have the text fill the text box without being reduced. The controls that used to be in the Format Shape dialog have now been moved to a task pane that (in my opinion) makes them harder to find and use, but they're all still there, I think.

  1. Right-click on the text box and choose Format Shape.
  2. In the Format Shape task pane, under Shape Options, the paint bucket (Fill & Line) will be selected.
  3. The other two icons are for Effects and Layout & Properties.
  4. Select the latter, in which you will find the margin settings and the controls for sizing the text box, wrapping text, etc. These are the options that were on the Text Box tab of Format Shape in earlier versions.

There is no option in Word (as there is in PowerPoint) to automatically resize text to fit a text box of a specific size. You will need to manually change the font size.
