ossiemac say:

You are working with times in the formula on the worksheet and therefore you have to work with time values in the VBA code.

170000 is not the same as 17:00 hours

90000 is not the same as 9:00 hours

10500 is not the same as 01:05 hours.

Also, Application.Sum is not the code to use. Just perform the calculation and assign the answer to a variable like the following. You can then assign the value in the variable to the textbox.

Note: By including the seconds in the TimeValue, there is no confusion in the code as to whether it is hours and minutes or minutes and seconds.

Dim dblAnswer As Double

dblAnswer = 500 / 3.5 / ((TimeValue("17:00:00") - TimeValue("09:00:00") - TimeValue("1:05:00")) * 24) + 3

You should be able to reply to a selected message using an account of your choice with an Outlook macro e.g. as follows. Put the name of the account that you want to use in place of 'account name' using the spelling shown under 'name' in File > Account Settings.


Sub ReplyUsingAccount()
Dim oAccount As Outlook.Account
Dim objItem As Outlook.MailItem
Dim oMail As Outlook.MailItem
Dim i As Long

Const strAcc As String = "account name"

Set objItem = ActiveExplorer.Selection.item(1)
For Each oAccount In Application.Session.Accounts
    If oAccount.DisplayName = strAcc Then
        Set oMail = objItem.Reply
        With oMail
            .SendUsingAccount = oAccount
            .Display
        End With
        Exit For
    End If
Next oAccount
Set oAccount = Nothing
Set objItem = Nothing
Set oMail = Nothing
End Sub
