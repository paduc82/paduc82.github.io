"Even though I have set the language as "Australian English" the spell checker auto-corrects to US English and I cannot "Add" the correct spelling to my dictionary."

This can be confusing. The definitive article on this is: Mastering the Spelling Checker by Suzanne Barnhill, MVP

The thing is, the proofing language setting is NOT a Word setting, a Windows or Mac setting, or even a document setting. It is set at the character level. A single document can have multiple proofing language settings. This language setting can be copied from one document to another or from one place in a document to another.

For a particular document, select all the text (Ctrl+A) and set the proofing language under Review > Language > Proofing Language.

If this does not solve it for you, chances  are good that it is in youro nomal template's styles. If so, write back and someone will help you change that.
