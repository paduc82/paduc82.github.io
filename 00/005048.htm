Placing a character in front of a number? - Microsoft Community

davejc say:

Placing a character in front of a number?

Hi, I have a long list of Car Numbers (#) that doesn't have the #-sign which is what I need.  Rather than having to  manually type in the #-sign in every Car Number cell is there any trick in Excel whereby I can add the #-sign in front of numbers? 

 

Thanks in advance

chip pearson say:

There are two ways to do this, one with code and one without. With code, use the following:

 

 

Sub AddNum()
    Dim R As Range
    On Error Resume Next
    If Selection.Cells.Count < 2 Then
        Exit Sub
    End If
    For Each R In Selection.SpecialCells(xlCellTypeConstants)
        If Err.Number <> 0 Then
            Exit Sub
        End If
        If R.HasFormula = False Then
            If StrComp(Left(R.Text, 1), "#", vbBinaryCompare) <> 0 Then
                R.Value = "#" & R.Text
            End If
        End If
    Next R
End Sub

 

Select the range of cells to change and run the code.  You must have at least two cells selected.

 

Without code, do the following. Create a new column next to the column containing your data. Enter the following formula in the first cell.

 

="#"&A1

 

where A1 is the starting cell. Now, select B1 (or wherever you entered the formula above), select down to the last cell of data and press CTRL D to fill down the formula into the selected range. Now, select the full range of the cells with formulas. Copy that and then Paste Values over the original range.

  1. Press Windows + I keys on the keyboard to open settings menu.

  2. Click on Time and language and select region and language.

  3. Under Language option, check which language is Set as default.

  4. If you find different language, you can change it by clicking on Add a language option.

  5. Select your preferred language (English Canada) and click on Set as default under English UK language.

  6. Now, click on English Canada and click on Options.

  7. Download the language, handwriting and speech packs.

  8. Now, click on Speech from the Time and language settings menu.

  9. Check if you are able to select the speech settings to English Canada.

  10. After performing above steps, check if you are able to use Cortana.
