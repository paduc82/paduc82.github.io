Hi. The answer about options available for lines, shapes, tables, etc. in iOS Word is: there are none.

I guess this is why we we are talking past each other.

In iOS Word there is a library of shapes. A straight line is one of those shapes. When added to a page it appears with a control point on each end.

The height of the line cannot be specified. I suppose it is a pixel.

You can touch and drag the line to move it. As you do, it moves content out of the way. The margins or space above and below the line cannot be controlled although the line-height of surrounding text can be controlled somewhat which can help with layout.

The control points on either end can be used to angle the line.

There are no other line controls or refinements. The bug we're referring to is that it is impossible to use the control points to produce a flat, horizontal line. charles kenyon say:
The default is  1.15 spacing. This is part of the normal style.

Modify that style or set a different style as your default. (I use Body Text.)

It's default settings are for:
Font: (Default) +Body, Left, Line spacing:  Multiple 1.15 li, Space After:  10 pt, Widow/Orphan control, Style: Quick Style

To change this in your normal.dotm template so that it will be the default for you in the future:
  1. Start a new document.
  2. Click the styles pane drop down (bottom right on styles group of home tab)
  3. Right click on normal and choose Modify.
  4. You can click on the icon for single spacing.
  5. Then click the radio button at the bottom of the Modify dialog box to have this apply to "New documents based on this template."
  6. Click OK
  7. Close Word.
  8. If you are asked if you want to save changes to the document, say no.
  9. If asked if you want to save modifications to Normal.dotm, answer yes.
Restart Word and see if this worked. If it didn't, write back, because you have something preventing changes to normal.dotm.
