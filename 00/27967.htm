karl timmermans [outlook mvp 2012-2018] say:

The Outlook Address Book is simply the representation (view) of one or more contact folders designated to be used as an "e-mail address book".

If you want to segregate your contacts by business / private (using your example) - you can create a secondary contact folder (one containing your business contacts - one containing your private contacts) and designate the new contact folder to be part of the OAB (right click on the folder --> properties --> select Outlook Address Book tab --> check the "Show this folder as an e-mail address book"

Note that if there is any overlap of contacts (i.e. a person is in both your business and personal contact folders, you will have to update each contact item separately for any applicable changes).

To can select which contact folder is to be shown as the "primary" list via
Outlook menu --> Tools --> Address Book --> Tools --> Options
