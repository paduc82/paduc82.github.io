arcane technology solutions say:

Hi,

 

For anyone who stumbles across this post, here is what worked for me:

 

1. Add Google.com as a search provider if you haven't already

 

2. Go to: HKEY_CURRENT_USER\Software\Microsoft\Internet Explorer\SearchScopes

 

3. Select the Key related to the Google.com search provider. (You'll know by the values within it)

 

4. Open the URL key. It should read something like:

http://www.google.com/search?q={searchTerms}&sourceid=ie7&rls=com.microsoft:{language}:{referrer:source}&ie={inputEncoding?}&oe={outputEncoding?}

 

Change the .com to .ie. In my case, here was the end result:

http://www.google.ie/search?q={searchTerms}&sourceid=ie7&rls=com.microsoft:{language}:{referrer:source}&ie={inputEncoding?}&oe={outputEncoding?}

 

Restart Internet Explorer and when you search from the address bar, it should search using Google.ie. Assuming Google was your default search provider.

 

All the best,
