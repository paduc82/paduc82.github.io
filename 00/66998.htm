rojo habe say:

What determines to which folder type a particular icon is assigned?  (The shortcut properties seem to indicate that ownership may have something to do with it.)

It's normally determined by the installer that created the shortcut in the first place.  Most software vendors are lazy and will place a single shortcut in %PUBLIC%\Desktop, which will appear on everybody's desktop if you have multiple users.  Unfortunately, if any one user deletes this shortcut it will disappear from everybody else's desktop too.  The solution is to manually copy it into everybody's individual Desktop folder and then delete the original.

 

Any shortcuts you manually make yourself (for example, by right-dragging a file onto the desktop)  will appear in your %USERPROFILE%\Desktop folder and hence appear only to you and not to any other users.

 

Of course, if you are the only user on this machine, none of the above is of any interest to you.  In this case, if you want everything sorted alphabetically, choose one of the two locations and manually move everything from the other one.  Since %PUBLIC%\Desktop is the one most installation software uses, it'll probably be the best choice, remembering that if you manually create shortcuts you'll have to manually move them yourself or they'll be out of order again.

 

Another thing to watch out for is software installers that ask if you want to install for "all users" or "just me" (or words to that effect).  The All Users option will put the shortcut in the Public folder; the Just Me option will use your Userprofile folder.
