Hello, onajtamo.

 

You can reboot the phone. If that will not work, you can also do a factory reset on the phone wherein no files will be deleted. To do that, go to Settings > Phone > Phone Management > Factory Settings > Restore and it will ask for a security code. You can use the default security code which is 12345 unless you changed it.

 

Is the phone's software up- to- date? You can  check it by dialling *#0000# . You can update the software using the phone itself by going to Menu > SW update. You may also use the Nokia Suite application. Check this link: http://www.nokia.com/global/support/nokia-suite/

 

 If the phone is up-to-date, you can consider re-installing the software of the phone by using the Nokia Suite.

 

Note: Nokia Suite is only compatible for computers with Windows Operating System

 

You can also back up and restore your files using Nokia Suite application to avoid data lost. 

 

To backup your device:

 

1.Start the Nokia Suite application > select Tools > Backup.
2.You will get a series of recommendations, click on Next.
3.Select the content you want included in the backup (Phone settings and Applications, Photos, Videos, Music and recorded sound clips, documents, backup of selected content also in the memory card). The contacts, calendars, notes and messages are stored by default.
4.Choose the folder where you want to save your backup, if you want to use the Nokia Suite folder created by default, select Next. To change the folder, select Browse.
5.Select Start to begin saving the backup.
6.Once the process is complete, select Finish.

 

To restore the backup on the device:

 

1.On the Nokia Suite application, just select Tools > Restore.
2.You will get a series of recommendations, click on Next.
3.Select the content you want to restore.
4.Click Start.


Note: If the contacts does not show in the backup, you have to copy the contacts in the phone memory from the SIM card.
