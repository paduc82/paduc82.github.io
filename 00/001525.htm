What operating system and service pack?

 

Prior Security Software?

 

Any security software still installed?

The following will find the Min and Max of a range - for this example, the range of values to be checked is in A1:A10. The Min value will be entered on B1, and the Max value on B2.

tushar mehta say:

Basically I have a spreadsheet - one part of it involves adding a bunch of rounded percentages to 1 to see if they add up properly. Sometimes in the rounding process, they will round to 0.99 or 1.01, or in rare cases higher or lower numbers. The program I enter these values in cannot accept values not totalling 1. What I need is a macro that will identify the maximum and minimum values in range. It will than take the difference of the percentage sum and 1, and add the difference to the smallest value in the range if the difference is negative, or subtract from the largest value if the difference is positive.

I realize this is likely quite a task.. any help/advice/anything to help be achieve this would be greatly appreciated.


In general you should adjust the largest number(s) rather than the smaller ones.  That way the percentage impact of the correction is minimized.

Here's one example.  Suppose your smallest number is 1% and the largest is 80%.  If your total is 99%, then if you add 1% to the smallest number, you are changing it by 100%!  And, what if the smallest number was 0%?

By contrast, adding (or subtracting) 1% from 80% will change it by 1.25%.  Even if the largest number were, say 25%, changing it by 1% would change it by only 4%.

 

ronrosenfeld say:

Whether the "difference of the percentage sum and 1" is positive or negative depends on the order of evaluation, which you do not explicitly state.

 

The macro below makes the appropriate adjustment, but I may have reversed the cases in which you want to subtract from the largest vs add to the lowest.  In any event, you should be able to make the appropriate changes if it is not correct.  Note that this macro alters your original data, so when testing, use copies of your original data.

 

====================================

Option Explicit
Sub AdjustPercent()
  Dim rSrc As Range
  Dim Percents As Variant
  Dim MX As Double, MN As Double
  Dim rToChange As Range
  Dim Adjustment As Double
 
Set rSrc = Application.InputBox("Select Range to Sum: ", Type:=8)
With WorksheetFunction
    MX = .Max(rSrc)
    MN = .Min(rSrc)
    Adjustment = 1 - .Sum(rSrc)

Select Case Adjustment > 0
    Case True 'subtract from smallest
        Set rToChange = rSrc(RowIndex:=.Match(MN, rSrc, 0))
        rToChange.Value = rToChange.Value - Adjustment
    Case False  'add to largest
        Set rToChange = rSrc(RowIndex:=.Match(MX, rSrc, 0))
        rToChange.Value = rToChange.Value + Adjustment
End Select
End With

End Sub
