WPA 2 is a Wi-Fi Protected Access version 2 (WPA2) encryption which is currently the best encryption method.  There are two encryption methods that WPA2 adds: one called Advanced Encryption Standard (AES) and one called Temporal Key Integrity Protocol (TKIP). If that doesn't help, try these troubleshooting steps.

Start with Network troubleshooter:


Refer:

Using the Network troubleshooter in Windows 7
http://windows.microsoft.com/en-US/windows7/Using-the-Network-troubleshooter-in-Windows-7
 
Make sure that Windows built-in Firewall is turned off as well as any anti-virus programs, anti-spy-ware programs and the like are de-activated before doing these steps and/or before attempting to connect to any Wireless Networks. If this helps in connecting to wireless network, you might need to change certain settings within them to be able to connect to the network. Please contact the appropriate vendor for help.

Check if Wireless Zero Configuration is disabled or not.

This service provides automatic configuration for the 802.11 adapters and should be set to Automatic Startup type.

Here’s how you may proceed to verify:

1)      Close Wireless Connection Properties window
2)      Click Start -> Run, Type: services.msc and click ok.
3)      Locate Wireless Zero Configuration.
4)      Right click Wireless Zero Configuration and select Properties.
5)      Make sure that the start-up type is set to Automatic. Change it to it if not.
6)      If Wireless Zero Configuration service is started or stopped, re-start or start service.
7)      Close Services windows (Wireless Network tab should now be activated)

If Wireless Network tab is present continue below:
 
1. Click on Wireless Network tab.
2. Make sure that there's a check mark on the "Use Windows to configure my wireless connection automatically".
3. In the Preferred Networks list, make sure that the box is empty, if not delete all access points found.
4. Just below the Preferred Networks list box, click on the Advanced button, a small window will appear.
          a)      Select the first option - Automatically connect to preferred networks
          b)      Put a check mark on the last line that says - Connect to non-preferred networks
          c)       Click close.
5.  Click Ok on the Wireless Network Properties
6. Now go to device manager by typing Device Manager button in the start search orb.
7. In Device Manager, click on the "+" beside Network Adapters to expand.
8. Right click and select un-install to remove the Wireless Network device and confirm removal on the warning window.
9. Restart your computer.

Upon startup Windows should automatically re-install the Wireless Network Device drivers.
Upon completion of installation, a pop-up window will appear stating "One or more wireless networks detected, click here for more options".

10. Click on the balloon to bring up the "Available Wireless Networks Windows".

You should now be able to see all available Wireless Networks in range.
11. Select desired Access Point to connect to then once connected, you should be able to browse the internet with no problems.

Notes: If after all these steps, you are still unable to connect or view any available networks, do the following with the Wireless Router.

1.       Reset router.
2.       If your router is 802.11g, make sure it it set to mixed mode (B+G) as almost all wireless card adapters and PPCs with built-in WiFi only work on B. If your router is "Enhanced G”,  you may need to disable the 'enhanced' mode as these also will not work with an 802.11b device.
3.       Try to connect to the wireless router with no security enabled, if successful and able to browse, you can then proceed to enable the security again

If you are able to detect Wireless Networks in range, you can safely assume that there's nothing wrong with the Wireless Network Card/Device installed on the computer.

If you’ve turned off your security software, by now you should enable them back.
