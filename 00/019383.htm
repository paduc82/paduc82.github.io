john - project say:

te5s3rakt,

I think Rod put it succinctly. Yes you could use VBA to do some formatting, but do you really want to do it? No, you don't. I've seen project plans that were virtual works of art with a profusion of font styles and colors. The problem is the amount of time to set up and maintain that dazzling plan is taking valuable time away from actually running the project.

If you want to color code the various phases of a plan, that's pretty simple and doesn't take a lot of effort but if you want to get down to special formatting for individual field cells, then I'd say you are going overboard. One other thing, you mention using a linked structure (master and subprojects). I feel obligated to warn you that a linked structure in Project is prone to corruption, particularly if more than a single user is working with the files. As a minimum, users must never rename, overwrite or move any of the files in the structure. It is also a good idea to keep all files in a single folder on a local drive, not operate it over a network.

Speaking from experience, as does Rod.

    If I tried to remove them from the earlier sections but then they also disappeared from the later numbered sections, They still appear to be connected in some way and yet I have disconnected the Header/Footer of the first numbered page, and also the Footer of the earlier section that first shows Roman-numeral page numbers.

    charles kenyon say:

    Here is a link to my Header/FooterSettings Recap. Some points from that recap:

    Changing the Even/Odd setting anywhere in a document will apply it throughout the document and it will apply to both headers and footers.

    Different first page is a section setting. It also applies to both headers and footers.

    If you looked at the underlying Word Object Model, it would seem as if even/odd is a section property as well, but that is not how it has ever worked, in Word.

    Every section has three headers and three footers, even if none of them are seen or used. A single page can have multiple sections!

    They are:

    • the Primary Header/Footer
    • the First Page Header/Footer
    • the Even Page Header/Footer

    These exist even if you never see them. Although a page can have many sections, it can only have one header and one footer. If you switch on different even/odd, the primary becomes the odd-page h/f.

    These three within a section are not linked or otherwise connected to each other within that section but may be linked to the previous section. Even if the one in your current section does not say it is linked to the previous, the corresponding header/footer in the next section may be linked to it. Each section after the first has six settings for "link to previous," one for each header and each footer.

    It is seldom that a document needs to change the headers/footers between sections. That is a myth propounded by poor training and books. Often the use of the three available and the StyleRef Field  will mean that most or all headers and footers in a document can be linked.

    When a header or footer is linked to the previous one, it means that Word treats this as a single header or footer regardless of how many sections use it. Changing it in one changes it in all to which it is linked.
