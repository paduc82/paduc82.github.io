4sccc say:

Computer goes to sleep while scanning

NOTE:  A Full Scan should over ride the sleep function, - and complete the scan, - before it goes to sleep...
kosh vorlon say:

Hi Ron,

 

Perhaps it should, but it doesn't.  If the computer goes to sleep, hybrid sleep, hibernation, or shutdown during the scan, the scan will stop and be cancelled.  That's how it works and cannot be changed.

 

The solution is to change the power saver settings from whatever they are to never (and I'd suggest if it is a laptop plugging in the AC power to make sure the battery doesn't run down and cause a shutdown or sleep mode or whatever before the scan finishes despite the power saver settings - since you don't want to set battery to never and have the system crash hard when the battery runs dry).  Then change them back after the scan.  Alternatively, create a new power saver plan called scan where it is set to never shut off or do anything but run continuously.  Then to switch all you need is to change the plan rather than reconfigure the settings each time you run the scan and then after it is completed (or just prior to and after a scheduled scan).

 

I personally run the scans in the background while I'm working and find they do not interfere much at all and then I can power down my system at night because no scans need to occur then.  If you run yours at night, there's no way to get the system to go to sleep or shutdown after the scan is complete - the option to set that does not exist - so it will stay on until you notice the scan is complete and shut it down yourself (or run all night until you awaken).  But how you do it is up to you.

 

I hope this helps.

 

Good luck!

Unfortunately, your brief question leaves it unclear as to what you actually want to do... There are at least 2 possibilities.

If you want the body text of the document to overprint the image, you can do as Robert suggests, but the image first must be formatted with None. Then you can right-click the image & select Arrange> Send Behind Text.

If you actually want to have separate text on the image as a label you need to create a Text Box [or other Shape] to hold the text. The Text Box then can be stacked on top of the image, but Text Wrapping will still have to be used. Which specific choice is dictated by whether the image is set as In Line with Text or has some form of
See Word Help on the various topics contained within Text Boxes, Shapes, and Pictures for more info.

One thing you cannot do in Word is actually edit the image to include text as an integral part of it.
