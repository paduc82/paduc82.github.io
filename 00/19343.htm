ossiemac say:

Hi Chris,


Try the following code and see if it is doing what you require.


Private Sub UserForm_Initialize()
  Me.txtBoxOther.Enabled = False
  Me.txtBoxOther.BackColor = &HC0C0C0
End Sub



Private Sub listTopics_Change()
   
    With Me.listTopics
        If .Selected(.ListCount - 1) Then
            If Me.txtBoxOther.Enabled = False Then  'Don't process if already enabled
                Me.txtBoxOther.Enabled = True
                Me.txtBoxOther.BackColor = &HFFFFFF
                Me.txtBoxOther.SetFocus
            End If
        Else
            Me.txtBoxOther.Value = ""   'If last option unselected
            Me.txtBoxOther.Enabled = False
            Me.txtBoxOther.BackColor = &HC0C0C0
        End If
    End With
End Sub
