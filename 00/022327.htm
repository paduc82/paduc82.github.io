Try this macro:


Sub CopyData()
    Dim wsh2 As Worksheet
    Dim wsh3 As Worksheet
    Dim r As Long
    Dim m As Long
    Dim cel As Range
    Application.ScreenUpdating = False
    Set wsh2 = Worksheets("99s")
    Set wsh3 = Worksheets("ALPHA1")
    m = wsh2.Range("B" & wsh2.Rows.Count).End(xlUp).Row
    For r = 1 To m
        If wsh2.Range("B" & r).Value <> "" And _
                wsh2.Range("B" & r).Value <> 99 Then
            Set cel = wsh3.Range("A:A").Find(What:=wsh2.Range("A" & r).Value, _
                LookAt:=xlWhole, MatchCase:=False)
            If Not cel Is Nothing Then
                cel.Offset(ColumnOffset:=1).Value = wsh2.Range("B" & r).Value
            End If
        End If
    Next r
    Application.ScreenUpdating = True
End Sub

hansv mvp say:

wsh2.Rows.Count is the number of rows in the 99s worksheet (65,536 rows if it is a .xls workbook, 1,048,576 if it is a .xlsm or .xlsb workbook).


"B" & wsh2.Rows.Count is either "B65536" or "B1048576" depending on the type of workbook.


wsh2.Range("B" & wsh2.Rows.Count) is the last (bottommost) cell in column B of the 99s worksheet.


wsh2.Range("B" & wsh2.Rows.Count).End(xlUp) moves upwards from that cell and stops at the first non-empty cell it encounters. In other words, it stops at the last filled cell in column B. For example, if B1 to B34 are filled and all cells from B35 down are empty, wsh2.Range("B" & wsh2.Rows.Count).End(xlUp) will be cell B34.


wsh2.Range("B" & wsh2.Rows.Count).End(xlUp).Row is the row number of that cell (34 in my example).
