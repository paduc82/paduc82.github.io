There is an easy way to convert DVD to MP4.

After lots of trial and error I found a method to convert a DVD into MP4 using VLC player (http://www.videolan.org/vlc/).

VLC player is a free all in one media player released under GPL, available for all most all platform.

Start the VLC player ->
1. Select "Open Disc" from the "File" menu (or do CTRL+D)
2. Select "DVD"
3. Click on "Stream/Save"
4. Click on "Settings" button
5. Click "File" from the "Outputs" list
6. Type or browse the file name to which you want to encode
7. Select "MP4" from the "Encapsulation Method"
8. Click "Video Codec" and Select "mp4v" from the drop down list
9. Select "2048" from the "Bitrate" drop down list
10. Click "Audio Codec" and Select "mp4a" from the drop down list
11. Select "128" from the "Bitrate" drop down list

12. Now in the "Target" (On the top of window) find scale=1, delete it and add width=640,height=480

13. Click OK
14. Click OK

Now it will start encoding the DVD in to mp4 file. After Completion just copy the file ( see step 5)to your N95 "My Videos" directory. you will now be able to view the video through the "Video Centre" menu item.

a. user say:

Hi Ahmadreza,


Error message "Element not found" and "DxDiag error" could show up if there is something wrong with the display card or its driver. Let's verify the same in order to troubleshoot the issue further. It can either be an incompatibility issue for the card with the operating system or the display setting needs a change with the operating system.

I see that there are several scenarios on Windows 10 Technical Preview wherein the display card involved is NVidia. Here's the list of them (although the issue is related to apps, we can still give it a try):

http://answers.microsoft.com/en-us/windows/forum/windows_tp-windows_store/windows-store-not-working/ed0702d5-db8e-41b0-8b90-8fcc6fa470df

http://answers.microsoft.com/en-us/windows/forum/windows_tp-performance/cannot-open-apps-after-update-explorerexe-error/bdbf9b11-e233-4919-87a8-669dc408ef49

To elaborate the steps taken on the links above, there can be couple of ways:

1) Change the display card's settings through the driver's Control Panel: Let's consider that you have NVidia card. What you need to do is open NVidia Control Panel from the desktop (Right‐click the Windows desktop, then click NVIDIA Control Panel from the context menu). In NVIDIA control panel, under "Manage 3d Settings", go to the "Global Settings" tab and select "Auto-select" under "preferred graphics processor".

See if it fixes the issue. If not, proceed to the next step.

2) Change the graphics card/processor in BIOS from NVidia to the integrated card. It's actually related to NVidia Optimus. What we need to do is change the graphics processor from NVidia Optimus to the integrated card (it could be any other graphics card like NVidia or Intel). To do this, you may refer the manual provided by the computer's manufacturer. Since you haven't mentioned the Make and Model of the system, here's an article for Lenovo computer that will help you understand the details about the settings and how to change them:

http://support.lenovo.com/en/documents/migr-76476

If the settings match the ones that are available under Display menu in BIOS on your computer, change the settings to Integrated and verify the results.

Please let us know the results.
