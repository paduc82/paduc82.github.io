Windows over time will accumulate data outside of the normal types of data we store. When running the troubleshooter, after "detecting problems" you will have a screen where you can select Next to continue or a link for Learn More. If you choose Learn More your browser should open to location http://answers.microsoft.com/en-us/windows/forum/windows_10-other_settings/settings-doesnt-launch-or-launches-the-store/ec439819-7ee4-4b4d-abdd-35d82e04c55f?auth=1

This page will indicate KB3081424 was released on 8/5/15 Cumulative Update for Windows 10: August 5, 2015

It describes KB3081424 as a fix that will prevent the problem from re-occuring, however it will not "fix" the problem if it already exist. Instead it implies one should run the troubleshooter first to correct the current issue (a work around, not a permanent fix), however once done when you get further Windows 10 updates, specifically KB3081424, the problem should not re-occur.
