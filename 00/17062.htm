Are you open to non-macro alternatives?

Try something like this (paste into H6):

=AVERAGE(OFFSET(C5, -(ROW(C6)-MATCH("sender",$F$1:$F5,1)-1),0,  (ROW(C5)-MATCH("sender",$F$1:$F5,1)),  1))

This only shows a result if the next row is a new sender; it looks to see what row the previous sender was on, and calculates the average of column C for everything in between
