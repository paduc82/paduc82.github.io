When a different antimalware protection product is installed, Windows Defender is disabled to make sure the two products do not interfere. In other words, McAfee tells Windows that McAfee is now protecting the system, and Windows therefore turns off Windows Defender.

There should be an uninstall entry for McAfee if you'd like to remove the trial. Go to Start | Settings | System | Apps & features to find McAfee's entry. Once you click on it, there should be an Uninstall button that'll kick off the uninstall steps.

A command prompt displaying during system startup doesn't always mean malware, it's often simply a standard Windows installation function.

In this case, the script involved appears to be installing the KB4406694 within the command line, which unfortunately wasn't easily found using an Internet search, so apparently it's not a well known module and more likely something like a driver or other package specific to the Sony PC it appears you're using.

As for the Taskeng.exe process that is displayed in the Command Prompt dialog box title header, that's the engine that runs the scheduled task which was being used to launch the installation itself.

"The Transient Control Process that is listed in the diagram is TASKENG.EXE.  TASKENG.EXE is the engine that runs in the task-defined user context and logs the event log messages.  It also generates the various processes that execute the actions defined in the task."

Task Scheduler Changes in Windows Vista and Windows Server 2008 ‚Äì Part Three

Most likely this was a one time process involved in an earlier Windows Update installation, which then created a temporary scheduled task to allow the item to be installed early in the startup process after the next system restart.

This is a little confusing. First, a Groove playlist can consist of music you own outright stored locally. It can also be the stuff from the online Groove catalog (or a mixture of both). You cannot burn any songs from the Groove catalog without buying them (and downloading them locally) first. You probably know about DRM restrictions on Groove (or any "rental" service) music and none of them allow burning. Some allow you to download to a certain number of devices (Groove does), and some are "streaming only". But no burning on any of them without actually buying the songs. A Groove playlist is for one of your authorized devices (PC or phone, basically) and is useless for burning.

As stated above, to burn a CD, you can use Windows Media Player. Assume you know how to do this. Zune had a "burn CD" feature. Xbox Music did not, and neither does Groove. Actually, WMP was always better (the Zune feature was/is extremely basic).

I don't know how to get a playlist you created in Groove into any other program (WMP---whatever). I think you have to create a new playlist with the same songs in WMP. It appears to me that playlists created in WMP show up in Groove....so I'm thinking you might save some time and just create your playlist there. However, no music with Groove DRM will appear in WMP. No streaming songs, no tracks you downloaded for offline listening from Groove (or Xbox, for that matter). The DRM changed with Xbox Music and WMP (and every other player including Zune) cannot use it.

If you've never burned a playlist in WMP, you want a "data CD" selected (top). When you see the files on the CD, you'll see one that is the playlist and others that are the tracks the playlist refers to. In a car stereo (like mine---pretty basic GM stereo) you pick the first track--the playlist--and play that one. My stereo indexes your CD into folders based on meta-data. The first folder is the playlist, in my case. Play that one and it plays the playlist correctly.
