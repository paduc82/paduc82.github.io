Import the Comcast address book into the Windows 10 People app - Microsoft Community

groversdad say:

Import the Comcast address book into the Windows 10 People app

A bit convoluted but...

  1. Export the Comcast address book into a CSV file.

  2. Move the email addresses in the CSV file from column ‚ÄúI‚Äù (‚ÄúHome E-mail Address‚Äù) to column ‚ÄúF‚Äù (‚ÄúE-mail Address‚Äù)

  3. Create (Log into) an account in Outlook.com (not the Mail app, but in a browser window) and go to the Outlook Mail screen.

  4. In the blue tic-tac-toe box choose People.

  5. Select Manage>Import Contacts and enter the name of the CSV file.

  6. Start the Windows 10 Mail app.

  7. Add the Outlook.com account from above to the app.

  8. The imported contacts should now be in the People app.

PLEASE TEST THIS WITH ONLY A COUPLE OF CONTACTS FIRST (edit the CSV file) in case I screwed up the above directions.
