There are two situations that will cause the value returns 0.

1.If a cell in a criteria range is empty, AVERAGEIFS treats it as a 0 value.

2.Cells in range that contain TRUE evaluate as 1; cells in range that contain FALSE evaluate as 0 (zero).

AVERAGEIFS(average_range, criteria_range1, criteria1, [criteria_range2, criteria2], ...), and criteria can be expressed as 32, "32", ">32", "apples", or B4, but data array is not suitable for being used as the criteria, so the numbers in brace are not an appropriate way to represent a criteria. Hi,

Make text subscript or superscript
I - Select the text that you want to format as subscript or superscript.

Do one of the following:

II- On the Home tab, in the Font group, click Subscript. Or press CTRL+=.

the subscript command in the font group

II(A) - On the Home tab, in the Font group, click Superscript. Or press CTRL+SHIFT+=.

the superscript command in the font group

Note: Keyboard shortcuts do not work if you are using Word Online.

To undo the formatting, click the Subscript or Superscript button again, or repeat the keyboard shortcut.
