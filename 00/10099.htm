hpholm say:

First, the device has a hard time locating my correct location when I enable 'my location' option.

I suggest focus on resolving this, since it is probably the culprit for the other issues.

- Has it been like this always?
- Did you add a protective cover, or do you keep the phone in a wallet?

The location services first get a rough estimate from nearby WiFi and triangulate from the location of cell towers, before it finds the accurate location from GPS satellites.
Anything (metallic) that blocks the antennas will affect the location accuracy.

Next, we need to consider a defect in the phone.
I suggest try the Hardware Tests app by developer Amaze Pictures from the Store and its GPS test. - It asks for permission to use location during install and on first use, make sure to allow that.

When running the GPS test in the app, notice the 'Position source' on the screen. It will first be WiFi or mobile, but it must change to Satellite after a while, and the accuracy must be within two-digit meters or less. Take the phone outdoors if you are in a building with metal foil in the wall and roof insulation.

--
Hans

You could do something like this to replace your first formula:

=IF(COUNTIF(GK16:GN16,GI14),"Y","N")

and something like this:

=IF(COUNTIF(GK16:IR16,GI14),"Y","N")

for your longer range. Note the implied >0 - anything greater than 0 will trigger "Y" to be returned.

Excel will only show a formula in the "Chart Data range" box if it is a range that can be expressed in a simple formula.

If the sources for series cannot be expressed as a simple range or a concatenation of simple ranges, Excel will leave the chart data range box empty.
