7. Replace the AAAA battery in the pen, and test on a known app. Please open and remove and replace with a new battery to ensure good connection. Follow the instructions for Change the battery in Surface Pen.

If the issue still persist after performing the steps, it would be better to contact our Surface Technical Support for further resolution.

cmdrkeene say:

After you login, you can change your password by pressing Ctrl-Alt-Delete. However, if you do not remember your "current password", this method will not work.  If this is the case, you can take these steps:

  1. While holding the Windows key, type R to open the Run box.
  2. Type control userpasswords2 and click OK.
  3. Click the Advanced button on the Advanced tab, which will open the Local Users Management Console.
  4. Open the Users folder, then right-click your account name and choose "Set Password".
Warning:  Resetting the password in this manner might cause irreversible loss of information.  For security, Windows protects certain information by making it impossible to access if the password is reset.  The data loss will occur next time you log off.  You should only use this command if you've forgotten the password and do not have a password reset disk. If you know the current password and want to change it, press Ctrl-Alt-Delete and choose Change Password instead.

The loss of information may affect you to different degrees depending on how you use your computer. If you are an administrator or able to take control of files, you will likely experience few side effects from this procedure.

omoikane1122 say:
Hello my name is Bradford and I'm an Independent Advisor for the Microsoft Community and I will try to resolve your problem with folder and file permissions.

You will need to run ‘Subinacl Tool’ to reset the permission to normal.

Then to reset system permissions, follow the steps:

1. Download subinacl.msi from the following link, and save it on the desktop.
https://www.microsoft.com/en-us/download/detail...
2. On the desktop, double-click subinacl.msi to install the tool.
3. Select C:\Windows\System32 as the destination folder.
**Note This step assumes that Windows is installed in C:\Windows. If Windows is installed elsewhere, select the appropriate path to .\System32.
4. Open "Notepad".
5. Copy the following commands and then paste them into the opened Notepad window. 
subinacl /subkeyreg HKEY_LOCAL_MACHINE /grant=administrators=f 
subinacl /subkeyreg HKEY_CURRENT_USER /grant=administrators=f 
subinacl /subkeyreg HKEY_CLASSES_ROOT /grant=administrators=f 
subinacl /subdirectories %SystemDrive% /grant=administrators=f 
subinacl /subkeyreg HKEY_LOCAL_MACHINE /grant=system=f 
subinacl /subkeyreg HKEY_CURRENT_USER /grant=system=f 
subinacl /subkeyreg HKEY_CLASSES_ROOT /grant=system=f 
subinacl /subdirectories %SystemDrive% /grant=system=f


6. In Notepad click File, Save As, and then type:
reset.cmd
7. In Notepad click Save as type, and then select All Files (*.*).
8. Save the "reset.cmd" file to your desktop, and close Notepad.
9. Double-click the "reset.cmd" file to reset the Windows Update permissions.
**Note This step may take several minutes, so please be patient. When the permissions have been reset, you will be prompted with "Finished, press any key to continue."
10. Press any key to complete the installation.

Thanks go to Amrita M. for developing this solution

In the meantime, after you completed the steps outlined above, please let me know if this fixes the issue!
