To clear up the confusion of this thread:

 

If the "SW update" app shows an update is available and states that you need a PC then it is an NSU (nokia software updater) exclusive update. This is where an update can only be done with a PC, not over the air.  You can confirm this checking for an over-the-air update by dialling *#0000# then press options, check for updates.  This will check using your wi-fi (when in range of your router) and will tell you that no updates are found.

 

This is because not all updates are released using both methods.  Some major updates require a PC connection whilst some do not.  

 

With updates that require a PC connection then that is exactly what you must do.  There isn't a workaround that will let you do it using wi-fi or any other method or operating system.  You must find a windows based computer that supports the nokia software updater, that computer must have a fast internet connection and you must connect the phone using a cable.  There are no workarounds that allow you to update via wi-fi if an over-the-air update is not available to you.

Since Word 2013, Word has had the ability to directly open pdf documents. However, some pdf documents (from scanners mostly) are really pictures of text rather than the text itself. Just as you can't look under the hood of a picture of a car, you can't edit a picture of a document.

In that case, you need to convert the picture to text. This is a process known as optical character recognition. This is built into Adobe Acrobat and is also in Office OneNote. Most scanner software comes with an OCR component as well. Once you have text, you can edit it with Word.

If you simply want to write on the document (but not in it) you can add a Text Box floating on top of the document layer, whether or not it has been put through the OCR process.

Word documents that have been saved as PDF will not need the OCR process, they retain their text, although not all of their Word structure and formatting.

Finally, documents converted from pdf (or really any other format) to Word can be tough to edit because the conversion process never has a one-to-one matching of how formatting is done. This means that a converted document will seldom be formatted in Word in a way that makes sense. An example is multiple section breaks to change margins, where in Word you would simply change the paragraph indent.

Difference between "Body Text First Indent" and "Body Text - Microsoft Community

wethotuwasatoad say:

Difference between "Body Text First Indent" and "Body Text Indent" styles in Word

Hello,

I've always been puzzled re the various Body Text styles in Word — particularly regarding the difference between "Body Text First Indent" and "Body Text Indent". Here are the Body Text styles as they appear in the Styles pane:

I've looked for an answer on several occasions and in several locations online but have never found a detailed description or explanation of how each differs from the others and when to use them. However, while writing this post, I finally discovered the answer (at least in part) so I'm posting it here for my own future reference and in case anyone else has the same question.

The Body Text styles can be broken into three groups as follows:

    [a] Body Text
    [b] Body Text 2
    [c] Body Text 3

    [d] Body Text First Indent
    [e] Body Text First Indent 2

    [f] Body Text Indent
    [g] Body Text Indent 2
    [h] Body Text Indent 3

The points to remember are that:

1. the two styles (d & e) which contain the word "First" indent only the first line of a paragraph,

2. the use of the number "2" in the middle group (e) means that the full paragraph is indented further than the previous style

3. the use of the number "2" in the first and last groups (b & g) indicates double-spacing

4. the use of the number "3" in the first and last groups (c & h) are simply for a smaller font size
