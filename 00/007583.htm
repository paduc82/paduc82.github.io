There is no such thing as the active cell on an inactive worksheet so you can't hyperlink to it.. The only active cell is the selected cell on the active (visible) worksheet.

 

To hyperlink to another worksheet, select the cell for the hyperlink and right click, click 'Hyperlink' . In the popup select 'Place in this document' and then select the worksheet, cell and a friendly name. OK out.

 

Alternatively, If you simply click the worksheet tab you want then that sheet will open and the selected cell will be where you left it.

Ensure that your AutoFilter is applied to all of the data because it is possible to apply it from the headers to the row before the first completely blank row if you just select one cell and then AutoFilter.

To ensure AutoFilter is applied to all of the data:

  1. Turn off AutoFilter
  2. Select all of the data.
  3. Apply AutoFilter.

  1. Now set all of the filters to blank which will result in only the fully blank rows being visible.
  2. Select all of the blank rows in the filtered range. (The row number on the left side will be blue to indicate the filtered range.)
  3. Click the Find & Select icon (Far right of Home ribbon)
  4. Select Goto Special
  5. Select Visible cells only and then OK.
  6. Right click over the highlighted area and select Delete Row.
  7. Clear the filters. (Easy way is to select Sort & filter -> Filter (Toggles AutoFilter off)
  8. Select a single cell in the header row and Select Sort & Filter -> Filter (Toggles AutoFilter back on)

Microsoft has redone their site to make Clip Art available from Bing.com & integrated it into Office 2013 for Windows. You can still access it from here.

However, the editing tools were dispensed with in Office 2008 for Mac & have not been restored, so you won't be able to Ungroup & edit as you could in Office 2004 & prior. You'll need to save the image & edit it using an image editing program.

The Clip Art Gallery has been replaced by the Clip Art Browser in the Office 2011 Media Browser.

I'm guessing that you had an earlier version of Mac Office installed but have since removed it... If the earlier version was in place when you installed Office 2011 the images in the old Gallery were retained but removing the earlier version may have crippled the operation of the old Gallery.
