You probably have an error and the code is ignoring it due to the On Error Resume Next. When you use a routine like this you should turn error trapping back on as soon as possible so that it will stop on any further errors  (or use an On Error routine that lets you know that an error has occurred).

    On Error Resume Next: Err.Clear: Dim wb As Workbook
