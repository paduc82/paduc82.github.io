Where did you find the two different Recycle Bins?  You should only have one Recycle Bin at the root of each of your drives.  If you have more than one on a single drive then I suspect that you may have a virus or other such pest, run a MalwareBytes scan on your machine.   If your Recycle Bins are at the root on different drives then that is how it should be.

Delete the Recycle Bins on all your drives then reboot the computer and Windows will recreate them.  Run the following command at an elevated (Run as administrator) command prompt:

rd /s /q c:\$Recycle.bin

Note that if you have more than one drive you must run the command against each drive, changing the drive letter accordingly, for example if you have a drive D: you also need to run the following:

rd /s /q d:\$Recycle.bin
