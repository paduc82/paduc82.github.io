carlaluulala say:

Interactive text box

I'm using powerpoint 2003. By using a text box from the Control Toolbar, is there any way that if a certain value is keyed in on the textbox during the slide show view, it will be directed to a specific slide in the presentation? Same thing with the checkboxes and command buttons, can it be directed to a different slide depending upon the selected answer? Thanks.

shyam pillai say:

Yes, assuming you've placed a textbox on the slide called textbox1. You can double click on the control and enter the visual basic editor. It will enter a empty event for textbox1_change. Delete that choose KeyDown event. In the key down event you can look for the Enter key. When user presses Enter. You will check the value entered in the text box and perform the desired action.

 

In the code below, I am checking the values are jumping to the appropriate slide in the slide show.


Private Sub TextBox1_KeyDown(ByVal KeyCode As MSForms.ReturnInteger, ByVal Shift As Integer)
If KeyCode = 13 Then
   
    Select Case LCase(TextBox1.Text)
    Case "cat"
        SlideShowWindows(1).View.GotoSlide 2
    Case "dog"
        SlideShowWindows(1).View.GotoSlide 3
    Case "rat"
        SlideShowWindows(1).View.GotoSlide 4
    Case Else
        MsgBox "Not valid"
    End Select
End If

End Sub

Creating a cross-reference to a heading in a different document - Microsoft Community

cheryl_ut say:

Creating a cross-reference to a heading in a different document

Does anyone know how to create a cross-reference to a heading in a different Word document. For example, in an Overview document, there is a list of bullets, such as Determining Value, Reporting Incidents and so forth. I want to create a cross-reference from the Determining Value bullet to the Determining Value heading in the Valuations document. Is there a way to do this? I just want a hyperlink to the Determining Value heading (I don't want to insert the text from the Determining Value section). Thanks for any direction you can provide.
jay freedman say:
Word's Insert Hyperlink dialog doesn't offer headings in other documents as targets, but it will offer bookmarks in those documents as targets.

The first step is to open the target document and insert a bookmark at each place you want to refer to. (The rules for bookmark names are that you can use letters, numbers, and underscores but no spaces or other characters.) Save the document (and optionally close it).

At the point where you want the reference, use the Insert Hyperlink dialog. Select the target document and then click the Bookmark button on the right. In the next dialog, select the desired bookmark and click OK. The address box will contain the file path and name, followed by the # character and the bookmark name. Modify the "Text to display" item if needed, for instance to repeat the target heading's text, and click OK.

ron coderre say:

No native Excel formula can reference a moving point in time (like TODAY()), and subsequently freeze it. That's because formulas can't change an actual value. They can only reference and calculate values.

You'd likely need a VBA solution to do what you want.
