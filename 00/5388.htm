a. user say:

This is a very old thread, but it's the top search result on Bing when searching for this error code. This is why I'm adding my findings here, which solved this particular error.

The problem was caused by the malware called Total Anti Malware Protection, which mimics MS Security Essentials. It introduces changes to the registry in order to launch itself instead of Security Essentials, and these changes prevents Security Essentials from being reinstalled with the above error code.

To fix this issue:

  1. Press windows key+R and type regedit
  2. Go to HKEY_LOCAL_MACHINE\Software\Microsoft\Windows NT\Current Version\Image File Execution Options
  3. Look at each subkey, they have the names of exe files. If the key contains the value Debugger, it means the particular .exe file is being redirected
  4. Delete the entire key for the .exe

You may expect to find msconfig.exe (or just msconfig) in the list, in addition to .exe files associated with Security Essentials.

Edit: this trick is evidently used by other malware programs as well.

I'm happy to see that this post has helped so many people :)

Are you interested in VBA code to do this for you? If so, the the code below. Not sure what expertise you have in VBA so here are instructions to install and run the code. (Suggest you use a new workbook to test and see if it does what you want.)

 

1. Open a workbook

2. To allow macros to run:

    (i) Select File -> Options -> Trust center (Left column)

    (ii) Click button "Trust center settings" (towards middle right of dialog box)

    (iii) Select Macro Setting (left column)

    (iv) Set option "Disable all macros with notification"

    (v) OK and OK again to exit.

 

3. Hold key and press F11 to open the VBA editor

4. In VBA editor, select menu item Insert -> Module

5. Copy the code below and paste into the large white area.

6. Edit the code as per the comments in green to suit your purposes

7. Position cursor anywhere within the code and press F5 to run

8. Hold Alt and press F11 will toggle back to worksheet

9. Save as Macro Enabled workbook

 

Edit the code accordingly to produce other series as required.

 

Sub RepeatSeries()
    Dim ws As Worksheet
    Dim i As Long
    Dim k As Long
    Dim lngStartRow As Long
    Dim lngLastRow As Long
    Dim lngSteps As Long
    Dim strCol As String
   
    Set ws = Worksheets("Sheet1")   'Edit "Sheet1" to your sheet name
    lngStartRow = 2                 'Edit 2 to the first row to start
    lngLastRow = 3000               'Edit 3000 to last required row
    lngSteps = 5                    'Edit 5 to number of repeats
    strCol = "A"                    'Edit "A" to column required
   
    With ws
        For i = lngStartRow To lngLastRow Step lngSteps
            k = k + 1
            .Range(.Cells(i, strCol), .Cells(i, strCol).Offset(4, 0)).Value = k
        Next i
    End With
End Sub
