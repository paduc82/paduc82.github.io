    In Windows 7 you cannot clear (and make it stick) the read only box on a folder.

    On a folder if you uncheck the box and apply it, it the box will change back to checked the next time you visit - that's the way it works.

    Something like this:

    Windows does not pay attention to the Read only attribute on folders - you can't make a folder read only but you can make the contents of the folder read only.

    That's why the message next to the box says "Only applies to files in folder".

    You cannot change the Read-only or System attributes of folders and you can read more about it here:

    https://support.microsoft.com/en-us/help/326549/you-cannot-view-or-change-the-read-only-or-the-system-attributes-of-fo

    So you can't make a folder read only but you can make the files in the folder read only if you apply that checkbox to the contents of the folder.
