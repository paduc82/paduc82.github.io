We did this so all memberships expire at the end of a month.

Is there a way to perform this calculation using the Expression Builder?

hansv mvp say:

Let's say the date the member paid their dues is in a field called PaidDate.

The expression for the expiration date is:

DateSerial(Year([PaidDate])+1,Month([PaidDate])+1,0)

scottgem say:

Hans has given the answer I would have given, but I should make the point that the expression he gave you should be used WHEREVER you need to display the expiration date. It should NOT be stored in a table. 

In a query use: ExpirationDate: DateSerial(Year([PaidDate])+1,Month([PaidDate])+1,0)

In a control use: =DateSerial(Year([PaidDate])+1,Month([PaidDate])+1,0)

Converting number & text to time. - Microsoft Community

rkirsch say:

Converting number & text to time.

I've got a spreadsheet with 1690 rows that I need to find the average time of, 3 times for 3 different columns. All 3 are times in the format 0h 0m. Is there anyway to convert this into time to find the average without manually changing them all to minutes?

ossiemac say:

Make a backup of your data in case the procedure does not perform as you expect.

I am assuming that your current data looks like the following screen snippet.

If my assumption is correct then the following procedure to convert to real times.

  1. Select the data (Do not include column headers but can select data in all appropriate columns.)
  2. On Home ribbon select "Find & Select" (Far right of ribbon)
  3. Enter the character h and a space in "Find what:" field
  4. Enter a colon character in "Replace with:" field
  5. Click Replace  All
  6. The h and space should now be replaced with a colon.
  7. Enter the character m in the "Find what:" field
  8. Clear the "Replace with": field so it is blank
  9. Click Replace  All
  10. The m should now be removed leaving real times like the following screen snippet

Use Average function to get the average time as per the following formula. It should return the average in hours and minutes format.

=AVERAGE(A2:A10)

rkirsch say: This solved part of my problem. What can I do if some of the times just show minutes. For example instead of 0h 4m it just shows 4m. This is throwing the entire average off as its just showing 4 instead of 0:04 once I find and replace. 

ossiemac say:

Assuming that your data looks like the example in column A of the screen snippet. (Column B is the modified data)

You indicated you have multiple columns so you can insert a column to right of the initial data and later delete the Initial data column so that column B takes its place.

  • Insert the following formula in cell B2.

        =IF(ISERROR(SEARCH("h",A2)),"0h "&A2,A2)

  • Copy the formula down,
  • Select column B and then Copy -> Paste Special -> Values (To remove the formulas and leave the values)
  • Now you should have all in the same format with 0h where no hours previously existed as in rows 5, 8 and 11 above.
  • You can now delete column A and column B will take its place. (Don't delete until after Copy and Paste Special above)
  • Repeat the above for the other columns of data
  • Follow the steps in my previous post to convert the text data to real times.

shane devenshire say:

In 2007:
1. Highlight all the cells or the rows you want formatted, lets assume B1
2. Choose Home, Conditional Formatting, Manage Rules, New Rule
3. Choose Use a formula to determine which cell to format
4. In the Format values where this formula is true enter the following
formula:

=A1="Yes"

5. Click the Format button and choose a format Green fill or font
6. Click OK twice
7. New Rule
8. Choose Use a formula to determine which cell to format
9. In the Format values where this formula is true enter the following
formula:

=A1="No" 

10. Click the Format button and choose a format Red

Repeat with A1=""  Yellow color
11. Click OK tree times.
