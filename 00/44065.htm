bossdweeb say:

"Is there anything to improve the lag but still keep the rest of the computer's performance high? "

- The best you can do, in addition to running the game on all low settings,  is not have any other software or browsers open when

running games. Games are system intensive so there is no 'keeping the rest of the performance high'.

You cannot multi-task while running games on low end machines.

- Laptops generally allocate around 2GB of the system RAM to graphics processing.

With Win10 this gets a bit confusing because of the terminology the choose to use, but mainly you want to verify the 'shared' RAM

is 2GB or more. There is also RAM that is shown as 'Dedicated GPU Memory'. 

You may be able to raise this amount, but if you are not trying to multi-task it should not matter as the game will still

use the available amount of RAM that the Intel HD graphics is capable of processing.

Look in -  Task Manager > Performance > Highlight , GPU0 (Intel HD Graphics)  for the amount of RAM 'dedicated' to the Intel HD graphics.

You can find out how to increase the amount of 'Dedicated GPU Memory'  in your user manual, or by contacting DELL.

With Desktop machines this is normally changed in BIOS, but laptops often have different methods.

- I'm afraid that with laptops, what you see is what you get. You cannot upgrade the graphics processor, and while Intel HD4400/5500

are much better than the old Intel GMA graphics, they are still absolute minimum spec for this game.

I see two Inspiron 15 3000 sub-models with different cpu's. One has Intel HD4400 score = 564   and another with Intel HD5500 score = 583

(The minimum dedicated gpu shown on the specs list score = 4127 , but you could obviously run a lower spec  DX11

dedicated graphics card as well ).

The cpu's on the Inspiron 15 3000 models are a bit low end for gaming. Dual core 2.0/2.1Ghz is a bit slow for many modern games,

and also comes in a bit under Fortnites minimum recommendation of 2.4Ghz.

- To get good gaming performance with anything other than games that have basic graphics, such as point and click adventure,

Jewel Quest type games, and a good number of old PC games on a laptop you need an expensive machine with a mid/high end

dedicated graphics card.

Even then there are problems controlling heat, and on a performance per dollar value, desktop machines win hands down.

- Part of the reason integrated graphics does not offer good gaming performance is it has no true dedicated video RAM (VRAM)

and has to 'share' the system RAM. The use of the term 'dedicated' RAM for integrated graphics is misleading.

VRAM is faster than system RAM, and games use both VRAM and system RAM (to hold and feed data to the VRAM),

when using a dedicated graphics card.
