What you're encountering is the new "simplified" account settings UI. There are now 3 ways to go about changing email account settings.

Option #1

From within Outlook
- Press CTRL+ALT+S (this brings up the Send/Receive Settings window)
- Select the group you want to edit and click on the <Edit> button
- Select the email account you want to modify --> click on <Account Properties>

Option #2

From within Outlook
- go to File --> Options --> Advanced --> click on the Send/Receive button
- click on the applicable <Account Group> (most people will only have one)
- click <Edit>
- select the <Email Account> --> click on <Account Properties

Option #3

With Outlook closed
- open the Control Panel <Mail> app
- select <Email Accounts>
- select the desired <email account> --> select <Change>

*** the Control Panel <Mail> app is not an option if the MS Office "Store" version is installed. If it is the store version, then the alternative in this instance is to click on Windows <Start> and <Run> the following command

Outlook.exe /manageprofiles

Note that I added a TRIM() function to my originally provided formula in order to avoid any possible lead, trailing or double spaces but the rules governing the addition of a space remain the same.

=IFERROR(TRIM(IF(EXACT(RIGHT(A2,1),UPPER(RIGHT(A2,1))),LEFT(A2,LEN(A2)-1)&CHAR(32)&RIGHT(A2,1),A2)),"")
