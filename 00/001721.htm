bernie deitrick say:

One way is to set up a table on another sheet "Variables" with a header row and column of labels - let's say, 10 headers in B1:K1 - for values that will be loaded into cell A1 of the sheet named "Calc", and 10 labels in A2:A11, for values that will be loaded into cell A2 of "Calc", with the final result appearing in cell Z100 of "Calc".

Then you could use a macro like this

Sub CreateTable()

    Dim r As Range

    Dim c As Range

    Dim shC As Worksheet

    Dim shV As Worksheet

    

    Set shC = Worksheets("Calc")

    Set shV = Worksheets("Variables")

    

    For Each r In shV.Range("B1:K1")

        shC.Range("A1").Value = r.Value

        For Each c In shV.Range("A2:A11")

            shC.Range("A2").Value = c.Value

            Application.CalculateFull

            Intersect(r.EntireColumn, c.EntireRow).Value = shC.Range("Z100").Value

        Next c

    Next r

    

End Sub

If you have trouble adapting this code to your actual layout, then describe what you have, what cells take what values, and where NPV is shown....
