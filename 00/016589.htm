I suspect that your intention is to array-enter the following (press ctrl+shift+Enter instead of just Enter):

{ =MEDIAN(IF((Table!A3:A840>=DATE(2015,1,1))*(Table!A3:A840<=DATE(2015,1,31)),Table!B3:B840)) }

Also note that I removed the superfluous "+", although it is not a problem.

{ =MEDIAN(IF(Table!A3:A840>=DATE(2015,1,1),IF(Table!A3:A840<=DATE(2015,1,31),Table!B3:B840))) }

mccbleue say: All modern Nokias "save messge for sending" - it is a feature which allows you to continue using the phone whilst it deals with the sending. Provided you have a good signal, the message will still be gone in a very small number of seconds, but if there is any network problem that temporarily prevents sending it doesn't freeze up the phone whilst it keeps trying.

The sending options are also a convenience feature, allowing you to choose:

(1) To send to any number that you have recently used (contacts or free-typed numbers) - it gives you quick access to your most recent and commonly used numbers without going through the entire contact list.
(2) Log items are your recently used numbers alone.
(3) The contacts list that you yourself prefer
(4) Groups within your contacts.

There is no way to shut this off from the shortcut, though what proportion of a person's calls are to saved contacts depends very much on that individual.

Option Explicit
Sub WordWrap()
'Wraps at W characters
Const W As Long = 17
Dim re As Object, mc As Object, M As Object
Dim str As String
Dim rSrc As Range, c As Range
Dim sTemp As String
'with offset as 1, split data will be adjacent to original data
'with offset = 0, split data will replace original data
Const lDestOffset As Long = 1

Set rSrc = Application.InputBox("Select cells to process:", Type:=8)
    If rSrc.Columns.Count <> 1 Then
        MsgBox ("You may only select" & vbLf & " Data in One (1) Row")
        Exit Sub
    End If
Set re = CreateObject("vbscript.regexp")
    re.Global = True
For Each c In rSrc
str = c.Value
'remove all line feeds and nbsp and space and replace with space
    re.Pattern = "[\xA0\r\n\s]"
    str = re.Replace(str, " ")
    re.Pattern = "\S.{0," & W - 1 & "}(?=\s|$)|\S{1," & W & "}"
If re.test(str) = True Then
    sTemp = ""
    Set mc = re.Execute(str)
    For Each M In mc
        sTemp = sTemp & vbLf & M
    Next M
        sTemp = Mid(sTemp, 2)
        With c.Offset(0, lDestOffset)
            .Value = sTemp
            .WrapText = False
            .EntireColumn.AutoFit
            .WrapText = True
            .EntireColumn.AutoFit
            .EntireRow.AutoFit
        End With
End If
Next c
Set re = Nothing
End Sub
