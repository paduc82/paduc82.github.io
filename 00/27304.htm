hansv mvp say:

Let's say you have a table of risk levels in Sheet1, with the levels 1, 2, ... in column A and various aspects of the risk levels in columns B, C, ... The entire range is A2:F20 (just as an example)

On Sheet2, you enter a risk level (1, 2, ...) in cell A2.

In cell B2, enter the formula =IFERORR(VLOOKUP(A2,Sheet1!$A$2:$F$50,2,FALSE),"")

In cell C2, enter the formula =IFERORR(VLOOKUP(A2,Sheet1!$A$2:$F$50,3,FALSE),"")

etc.

You can fill down these formulas as far as you want/need. If there is no valid risk level in column A yet, the formulas will return a blank.

ossiemac say:

AFAIK there is no vertical alignment option for a textbox on a userform. If text is too close to the top edge then your only options are to increase textbox height or set AutoSize to True.

Setting AutoSize to true also AutoSizes the width. I have in the past used code to in the AfterUpdate event where I first set to AutoSize and then reset the width to the required width so that the text will fit within the TextBox height and always retain the width of the TextBox.
