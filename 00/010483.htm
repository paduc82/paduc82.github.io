jim thomlinson say:

I would be inclined to use 2 other cells to hold the criteria dates and then use a sumproduct formula something like this...

  • put Sep 1 in D1
  • put Sep 15 in D2
  • Add this formula =SUMPRODUCT(--($A$1:$A$30>=$D$1), --($A$1:$A$30<=$D$2), $B$1:$B$30)

As you modify the dates in D1 and D2 you get the resulting profit calc... Note that this does not require the dates to be in any specific order and does not have an issue with missing dates.

Returns or sets a String that represents the text contained in the specified object. Read/write.

You will get an error if the layout doesn't have a footer placeholder though.

Try this:

Sub secfoot()
Dim osld As Slide
Dim oshp As Shape
Dim b_found As Boolean
If ActivePresentation.SectionProperties.Count > 0 Then
For Each osld In ActivePresentation.Slides
osld.HeadersFooters.Footer.Visible = True
For Each oshp In osld.Shapes
If oshp.Type = msoPlaceholder Then
If oshp.PlaceholderFormat.Type = ppPlaceholderFooter Then _
oshp.TextFrame.TextRange = ActivePresentation.SectionProperties.Name(osld.sectionIndex)
End If
Next oshp
Next osld
End If
End Sub
