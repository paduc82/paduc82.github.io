To use this as an Office keyboard shortcut, you must first turn off the Mac OS X keyboard shortcut for this key. On the Apple menu, click System Preferences. Under Keyboard click the Shortcuts tab, and under Keyboard clear the On check box for the key assignment that you want to turn off (^F5, in this case).  If it was not ON, make no change and go to the next step.

  1. Then in Word, go to the Tools menu and Customise Keyboard. In the "Type new shortcut" box, type Shift + F5. Do notpress Enter.  Immediately below the box you should now see "Currently assigned to: GoBack".
  2. If you do not, you need to re-assign it.
  3. In the Categories list, choose All Commands
  4. In the Commands list, scroll down to "GoBack" and select it.
  5. Ensure that Shift + F5 remains in the Type new shortcut box.
  6. Click Assign.
  7. OK you way out and quit Word.

When you restart, it should now work.  If it doesn't, come back, we need to carry out a few more steps.
