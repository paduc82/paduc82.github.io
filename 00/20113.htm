ossiemac say:

Provided that you know a cell reference that should be within the table range then the following code can be used to identify the table name and then assign the table to a List Object variable.

You can use any cell that should be within the table range including cells in the column headers. I have used Cell A1 which is the first cell in the table.

Sub IsCellWithinTable()

    Dim ws As Worksheet
    Dim lstObj As ListObject
    Dim strTblName As String
   
    Set ws = Worksheets("Sheet2")
   
    On Error Resume Next
    strTblName = ws.Range("A1").ListObject.Name
    On Error GoTo 0
   
    If strTblName <> "" Then
        Set lstObj = ws.ListObjects(strTblName)
        With lstObj
            MsgBox "Name of table is: " & lstObj.Name
        End With
    Else
        MsgBox "Table not found in range."
    End If
End Sub
