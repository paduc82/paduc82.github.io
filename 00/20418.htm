jennypurcell say:

Calendar on multiple worksheets (any year)

I am using the above worksheet and I have followed the instructions to use the note tab to input inforamtion in to my calendar which is great.  How do I get information into the notes cells that apear in each month.  I can see these cells have formulas I just don't know what to put in the date column on the notes tab so that it will feed into the individual months
rich007a say:

I take it you're talking about this:
http://office.microsoft.com/en-us/templates/calendar-on-multiple-worksheets-any-year-TC101923111.aspx

I think the "Notes:2 field is there for writing on when you've printed it out.
But if you want to set up a table with Jan, Feb, Mar, etc. in cells D2:D13 in the Notes sheet
and put your notes in cells E2:E13 (put "Month Notes" in E1), then you'll have to adjust each formula as follows:

Jan sheet, cell B6, paste:

=IF(B5=notes,IF(VLOOKUP(TEXT($B$2,"mmm"),Notes!$D$2:$E$13,2,FALSE)=0,"",VLOOKUP(TEXT($B$2,"mmm"),Notes!$D$2:$E$13,2,FALSE)),IF(ISERROR(VLOOKUP(B5, Notes!$A:$B,1, FALSE)), "", VLOOKUP(B5, Notes!$A:$B,2, FALSE)))

Copy B6.  Select B6:H6, B8:H8, B10:H10, B12:H12, B14:H14, B16:H16
Paste special, Formulas.

Repeat for each month sheet.

Now whatever is in Notes!E2 will appear in Jan!B6 and Jan!D16.  Note it will only appear in the cell beneath the cell containing "Notes:" and will not spread across the other empty cells in the Notes area.  These areas change shape with each month depending on what day of the week the month starts, and how many days in the month.

If you are happy to have one file per year, you could simply merge the notes cells (e.g. B6:G6 on the Jan sheet) and type your notes directly, or add the formula =Notes!E2 (for the Jan sheet, E3 for the Feb sheet, etc.), but you'd have to do it all again for a different year.

Hope that helps.

Cheers
Rich

[PS. the above long formula uses IFERROR, so will only work in Excel 2007]
