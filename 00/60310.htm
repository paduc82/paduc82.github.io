lemp say:
That's the location where System Restore points are stored.  This folder is "super hidden" for a reason: it's not good to manually delete things from it.  Doing so will probably break System Restore.

You can run Disk Cleanup (Start > Run > cleanmgr > OK) and click the "More Options" tab. Then use the button to "remove all but the most recent restore point."  See http://www.theeldergeek.com/disk_cleanup_utility.htm

Then configure System Restore to use no more than 1 GB of space: see http://bertk.mvps.org/html/diskspace.html

And you can disable the Indexing Service (which stores its indexes in the System Volume Information folder): http://lifehacker.com/031440/turn-off-indexing-and-speed-up-windows-xp
