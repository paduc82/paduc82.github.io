Input mask in Excel

I am entering data into an excel spreadsheet and would like to have the  / /  for my dates appear automatically when I type the corresponding numbers.  i.e. 01012016 would be entered and would appear as 01/01/2016.  Is there a formula or mask for that??
bernie deitrick say: No - the macro depends on your having pre-formatted the cells for text entry (to keep leading 0s) and entering a string that is 8 numeric characters long. You can enter values into as many cell as you want, using Ctrl-Enter, and the code will check them all.  Note that if you need to re-enter a value, it will need to be entered as a date or you need to format the cell for text prior to entering the new 8 digit date string. The code will work for as many columns as you format as text.

bernie deitrick say:

Format the cells where you want to enter your 8 digit dates as text, and use this change event macro:

         

1) Copy this code.
2) Right-Click the sheet tab of interest.
3) Select "View Code"
4) Paste the code into the window that appears.
5) Save the file as a macro-enabled .xlsm file.
6) Make changes as needed

Private Sub Worksheet_Change(ByVal Target As Range)
    Dim c As Range

    Application.EnableEvents = False

    For Each c In Intersect(Target, Me.UsedRange)

        If c.NumberFormat = "@" And Len(c.Text) = 8 Then
            strt = c.Text
            On Error GoTo NotNumber
            lngT = CLng(strt)
            On Error GoTo 0
            c.NumberFormat = "mm/dd/yyyy"
            c.Value = DateValue(Left(strt, 2) & "/" & Mid(strt, 3, 2) & "/" & Right(strt, 4))
NotNumber:
        End If
    Next c
    Application.EnableEvents = True
End Sub
