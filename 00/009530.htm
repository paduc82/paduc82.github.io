charliehlive say:

Determine office for mac version using pkgutil

-Charlie

john mcghie say:
Both of them are correct.

Office is a collection of multiple packages.  Some will update when Service Packs and Updates are applied, others will not.

Retrieve the version from Microsoft Word.app.  If you are fully up to date, you should see 12.3.0

If you retrieve the version from Microsoft Entourage.app, you will also get 12.3.0 unless you have Web Services Edition loaded.  If you have, you will see 13.1.0 if it's properly updated.

Note that there is no guarantee that the pkg files will exist on any given computer following an update.  They may be cleaned up by whoever did the installation.

12.1.0 is "Service Pack 1".  There's a "Service Pack 2" available also, which is 12.2.0.  Version 12.3.0 was issued as an "update", not a "Service Pack".  The difference being that an update requires the previous service pack as a prerequisite, while a Service Pack will update version 12.0.0.
