Private Sub Run_Prog_Bttn_Click()

Hi,

Points that can be responsible for slow and unstable Internet Explorer performance.

  • Defective Internet Explorer add-ons. .
  • Phishing filter is turned off.
  • Pop-up blocker is turned off.
  • Security settings are not set to recommended settings.
  • Internet Explore does not update cached pages or updates them inefficiently, causing launch to be slow.
  • Cache size is too small or too big, causing slow performance.
  • Concurrent server connections set too low or too high causing slow performance.
  • Pop-up blocker is turned off allowing pop screens.
  • Disabled Phishing Filter.
  • Disabled Data Execution Prevention for Internet Explorer.

You can manually check for the settings and make sure everything is set correct.
