operating system is not supported

Sounds like a User-Agent problem.  E.g. the host is not recognizing the IE11 User-Agent string and assuming the worst about it.   Are you really using W8 or W8.1?  In either case you could try using the Compatibility View Settings but that would force an emulation of IE7 Quirks which technically would not allow the host to infer that you were using either supported version of Windows.  (E.g. Windows 7 implies IE8 or greater and Windows 8 implies IE10,  IE11 if it has been upgraded to 8.1)

Alternatively, you may have to use the Developer Tools (F12) to override the User-Agent string with something that spoofs the site into thinking that you are using a supported OS.  Both IE10 and IE11 allow us to spoof any User-Agent string.  Unfortunately the UI is completely different in IE11.

I don't know what the User-Agent string for Firefox and Chrome look like;  however,  I wouldn't have assumed that they include information about the OS.  In any case, if the host supports W8 and you are actually running W8 then I can't imagine what the problem would be, so that may mean that you have an even bigger obstacle to overcome in making the host like your browser(s).

Another possibility is that the message is wrong and a red herring (e.g. should only be regarded as a sign that something didn't work, not what it says).  You could try using the Developer Tools, Network capture tool to try to get more clues on that tack.

Have you asked your University support group about your symptom?

    In cell K20 I have content like:

    21 september 2010   ab cx-y4z

    mike h.. say:

    Hi,

    If it wil *always* be 3 spaces after the date then

    Range("s20").NumberFormat = "d mmmm yyyy"
    Range("s20").Value = Left(Range("k20"), _
    InStr(1, Range("k20"), "   ", vbTextCompare))

    or as a worksheet formula, formatted to your requirement

    =LEFT(K20,FIND("   ",K20))

    If this post answers your question, please mark it as the Answer.
    Mike H

    mike h.. say:

    Hi,

    Here's another way

    Range("s20").NumberFormat = "d mmmm yyyy"
    v = Split(Range("k20"), " ")
    For x = 0 To 2
    newdate = newdate & v(x) & " "
    Next
    Range("s20") = newdate
