Hi Walter,

You have received excellent advise, and I expect that your problem is resolved.

I see one thing that has not been mentioned. While "DisableAntiSpyware"

is the predominant Policy  that is used by malware or "another" anti-virus

to disable Windows Defender, there is another possibility.

("DisableAntivirus")     You can determine if this is your case by examining

the Windows Defender "Policy" key in the Registry. To enter the Registry,

click on "start" and type "regedit". Then select "Run as administrator".

On the Registry Editor screen, navigate through the following branch.

Hkey_Local_Machine\Software\Policies\Microsoft and select the

Windows Defender folder.

If there is any Key present in this folder besides "Default", it is a Policy

change that has been made to Windows Defender. Expect to find

DisableAntivirus. If you do, you can disable this Policy by changing its

DWord value from 1 to 0.  Do that by right-clicking "Dword" and select

"modify". Type 0 to replace the highlighted 1.  Then click "OK"

If there are any other Keys in this folder, besides "Default", It may be

necessary to just delete the complete folder. If so, please return to this

thread and we will discuss it.

You should exit the Registry by retracing the steps taken to get here.

This keeps your Registry from getting cluttered. When you reach HKLM, 

select "File" and then "Exit"

Restart your PC and re-check Windows Defender. IF it is not enabled,

run the following command string in command prompt (admin).

CD\     <enter>

CD Program Files\Windows Defender     <enter>

MpCmdRun -wdenable     <enter>

Exit     <enter>

Restart your PC and check Windows Defender again.

Good luck,    Glen
