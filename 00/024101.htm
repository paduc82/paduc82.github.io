macropod say:

You could use a Letter merge or a Label merge to get the same result:

First, create a two-column, 1-row table for the required two columns.

Then, if you use a Letter merge, insert the required mergefields into each column and, if you want the second column to display a different record to the first, insert a 'NextRecord' field either at the end of the first cell or the start or the second cell. If you use a label merge, you need only complete the first column, then use the 'Update Labels' tool to do the second column and insert the 'NextRecord' field

Next, run the mailmerge, sending the output to a new document.

Finally, use Find/Replace to delete all the Section breaks created by the merge, using:

Find = ^12

Replace = nothing

You should now have a table with all your directory entries. You can print and/or save the output, as desired.

suzanne s. barnhill say:

Are you wanting the documents to have uniform formatting after they are combined or to retain their individual formatting? If the latter, then really the only viable solution is to convert them to PDF and combine them in Adobe Acrobat or equivalent.

From your reference to creating one master, however, it sounds as if you may want them to be uniform. For this purpose, your best bet is to choose the one document that has the format you want in the rest. You will then have to paste each additional document without section information (http://wordmvp.com/FAQs/Formatting/PasteWithoutSectionInfo.htm). That will take care of document/section-level formatting such as margins, header/footer, etc. The rest of the formatting will be determined by styles, which will be adjusted to match the target document. That is, a Normal or Body Text or Heading 1 paragraph in any of the pasted documents will take on the formatting of Normal or Body Text or Heading 1 in the target document. If styles have not been used systematically in the component documents, then direct formatting will probably be retained, and you'll need to define and apply styles consistently to make them uniform.
