dave peterson say:

Sometimes working with dates can be problematic.  I'd try:

Option Explicit
Sub testme()
    Dim res As Variant 'could be an error
    Dim myRng As Range
    Dim DestCell As Range
    Dim SalesDate As Range '????

    With ActiveSheet
       Set SalesDate = .Range("B1") 'where I put my date
       Set myRng = .Range("A:A")

       res = Application.Match(CLng(SalesDate.Value), myRng, 0)

       If IsError(res) Then
          Set DestCell = .Range("a2").End(xlDown).Offset(1, 0)
       Else
          Set DestCell = myRng(res)
       End If

       DestCell.Select

     End With
End Sub

I used cLng() because it seems to work better with =match() and dates.

But I used application.match() since it doesn't cause a run time error when
there is no match found. 
With Application.worksheetfunction.match() or simply worksheetfunction.match(),
you'll have to use some error handling to avoid that run time error if there is
no match.

  • Incoming (POP3) Server
