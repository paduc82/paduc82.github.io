If you are running Windows 8/8.1 then Windows Defender is full antivirus package and nothing else is required.

hansv mvp say:

You don't use a formula in B2 for that, but conditional formatting.

  • Select cell B2.
  • Click Conditional Formatting > New Rule... in the Home tab of the ribbon.
  • Click "Use a formula to determine which cells to format".
  • Enter the following formula in the box:

=B1="yes"

  • Click Format...
  • Activate the Fill tab.
  • Click the gray color that you prefer.
  • Click OK twice.

what should the path be for Microsoft security essentials set to start up from?
kosh vorlon say:

Hi peach,

 

The startup command is "C:\Program Files\Microsoft Security Client\msseces.exe" -hide -runkey

 

The registry location is HKLM\Software\Microsoft\Windows\CurrentVersion\Run

 

The target for the shortcut that opens MSE (which is in Start / All Programs / Microsoft Security Essentials) is "C:\Program Files\Microsoft Security Client\msseces.exe"

 

I hope this answers your question, but when MSE is installed all this should be setup already.  If you want a desktop icon to start it, just copy the shortcut from All Programs and paste it to the desktop.  The other entries should already be as stated in MSCONFIG.EXE and that item (Microsoft Security Client) should be checked to start at startup so it should rarely need to be started as you want it starting at startup and running while the computer is on - and that should be the case unless something goes wrong.

 

I hope this helps.

For those who have any interest in what I was doing wrong - here are the steps I ended up using, so maybe you don't do the same thing:

1. Open the merge template you designed.

2. Click on 'Mailings' on the tab.

3. Click on 'Select Recipients,' the second icon in the 'Start Mail Merge' area.  (This is where I was off.  I was going to 'Mailings/Start Mail Merge/Step by Step Mail Merge Wizard' and going to step 3 and trying to connect from there.  Once I got it RIGHT, it helped tremendously!)

4. On you click on 'Select Recipients,' the second icon in the 'Start Mail Merge' area, click on 'use existing list' and go select your data file.

5. The Mail Merge form should open and you're good to go! 

Just a note - it's so important to have the data file set up correctly.  I found that it may take a bit of effort to get things lined up right, but once you do - it's worth it!

Enjoy!
