bob jones aka: cybertaz say:

Word's Bookmarks are applied to the text to which they pertain. There are no 'pages' in the structure of a Word document, ergo, no specific places :-) Accordingly, they travel with the bookmarked text. [You just want to avoid inappropriately inserting text within an existing bookmarked passage. It helps to use Word> Preferences - View to put a check in the Bookmarks box so you better can see where they are.]

If the bookmarked text is deleted the bookmark applied to it will be deleted as well. You can delete Bookmarks, themselves, in the same dialog where you create them, but you can select only one at a time... IOW, you have to click the Delete button for each one you want to delete.

I'm not sure whether VBA could be used to locate & remove all Bookmarks, but it seems likely.

john korchok say:

Bookmarks are attached to specific blocks of text. Inserting text before will move the bookmark along with the text to which it is attached. Word will still be able to find the book mark.

If you delete or copy over the text with bookmark, the bookmark is destroyed.

You can delete existing bookmarks using the Insert>Bookmark dialog. Select the bookmark name from the list, then click on Delete. The bookmark is erased without affecting the underlying text.
