karaj22 say:

Hi, I've figured out a solution.

 

I've noticed that the phone connects to the router from the WLAN-wizard,but I got the "No gateway..." message to.

I tried all the things you said( powersaving disabled,hidden network login,etc.), did not work none of them.

I use at home(where the phone doesn't connect to net) a D-Link router with DHCP server enabled.Phone settings are all set to factory defaults (auto config in advanced wifi settings enabled).

I saw from the router's menu,that my phone didn't get any IP address from the DHCP server. So I had to disable the DHCP server. Then i add my phone MAC address to the router and typed a manual IP for my 5530.

In the phone i changed the IPv4 settings from auto->manual( in the address field i typed the IP i wrote in the router),but left the DNS settings in auto status(as well in the IPv6 settings).

Then i created an access point in the Destinations menu,and everything worked fine.

 

Short summary:

In the router: DHCP server disabled,manual IP to your phone(let's call it IP).

In the phone: Settings->Connectivity->Wireless Lan->highlight your network->Options->Settings->Options->Advanced settings->Automatic configuration: Enabled

Switch off the phone and on again.

Settings->Connectivity->Destinations->Internet->Options->New access point->No check->WLAN->WLAN network name: your SSID ->(your WEP/WPA code if yout network is protected)->highlight the new access point->Options->Edit->Options->Advanced settings->IPv4 settings->Phone IP address: IP ->Subnet mask: your subnet mask(see in the router config) -> Default gateway:your router's address(for example 192.168.1.1) -> DNS addresses:automatic

Then you are done.

If these don't work,try to change your DNS settings to your ISP's DNS addresses.

I hope it will help you.

Perform a clean boot in Windows 10 - Microsoft Community

trybird say:

Perform a clean boot in Windows 10


A clean boot is performed to start Windows by using a minimal set of drivers and startup programs. This helps eliminate software conflicts that occur when you install a program or an update or when you run a program in Windows 10. You may also troubleshoot or determine what conflict is causing the problem by performing a clean boot.


For performing a clean boot in Windows 10 with command line operation, first please know the following common knowledge about command line operation:


How to open a "Command Prompt (Admin)" window?
You could open a "Command Prompt (Admin)" window by one of the following three ways ('I. Keyboard' or 'II. Mouse' or 'III. Touch'):
I. Keyboard: Win+X, A, Alt+Y
II. Mouse: rightclick Start button, click "Command Prompt (Admin)", click "Yes" button
III. Touch: press-and-hold Start button, tap "Command Prompt (Admin)", tap "Yes" button


Common knowledge about command line operation:
In an opened command window - namely a "Command Prompt (Admin)" window or a PowerShell console window, the following are the conventional rules of command line operation:
[Type or paste the command] - Either type a command directly or paste a command text, both of their results effected are the same. At any time, you can directly type a command in the command window; or, you also can paste a command text in the command window. To copy a command text, please use "hold down and drag" to select the command text in here with your left mouse button, and then right-click, then there will pop up a shortcut menu, then click "Copy" item on the shortcut menu. To paste the command text, in the command window please right-click, then you will find that the command text already had been pasted in there.
[Hit or press the Enter key] - At any time, after finishing the "Type or paste the command" operation, you wanna execute the command, please hit or press the Enter key, then you will find that the command is running.
[Hit or press the Up key] - At any time, you wanna repeat the last command, please hit or press the Up key. Tip: you often wanna repeatedly use or edit a previous command, that can be done by using the Up or Down arrow keys to scroll through the history.
[Exit] - At any time, after finishing all the command line's operations, you wanna exit the command window, please click the Close button on the upper right corner of the command window; or you also could type "exit" command directly and then hit or press the Enter key to back to the top level window or close the window completely.


You could run the command below to view the startup items:
wmic startup
or
wmic startup > BootItemsList.txt


For getting the same results, also could execute the following command in the command line:
msinfo32
then navigate to:
Software Environment -> Startup Programs


Press "Win+R" key combinations, then type "regedit" or "regedt32" and click "OK", just start the "Registry Editor".
I suggest you'd better often check the following three hives where some malwares like to let themselves as startup apps put in:
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Run
HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Run
HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\Microsoft\Windows\CurrentVersion\Run
Also should check the following two hives:
HKEY_USERS\S-1-5-19\SOFTWARE\Microsoft\Windows\CurrentVersion\Run
HKEY_USERS\S-1-5-20\SOFTWARE\Microsoft\Windows\CurrentVersion\Run

To perform a clean boot namely to delete all the values in these startup keys, you could use Notepad to make a batch file that reads as follows and save it as "%UserProfile%\Documents\CleanBoot.bat":
reg delete HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Run /va /f
reg delete HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\Microsoft\Windows\CurrentVersion\Run /va /f
reg delete HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Run /va /f

for /f "delims=" %%u in ('reg query HKEY_USERS') do reg delete %%u\SOFTWARE\Microsoft\Windows\CurrentVersion\Run /va /f

Then you could open a "Command Prompt (Admin)" window by one of the following ways:
Keyboard: Win+X, A, Alt+Y
Mouse: rightclick Start button, click "Command Prompt (Admin)", click "Yes" button
Touch: press-and-hold Start button, tap "Command Prompt (Admin)", tap "Yes" button
Then in the "Command Prompt (Admin)" window, you could execute the following command to run the following batch file:
%UserProfile%\Documents\cleanboot
