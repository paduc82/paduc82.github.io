Disable Excel number format - Microsoft Community

Disable Excel number format

How do I disable auto format for numbers?  I try to enter numbers but Excel converts the numbers to dates.
joeu2004 say: Ezekiel wrote:
How do I disable auto format for numbers?  I try to enter numbers but Excel converts the numbers to dates.

We cannot disable Excel's interpretation of some forms of input as dates.  (The exact interpretation depends on your settings in the Regional and Language Options control panel.)

But I cannot think of a bona fide "number" that Excel would misinterpret as a date.

I suspect you are missing the "=" before an expression like 1/31.  If you are trying to divide (for example), that should be entered as =1/31.

Alternatively, I wonder if you are trying to enter ID "numbers" that include dashes or slashes; for example, 1-31-1969.  Such data should be entered as text since they are not bona fide numbers; that is, something that we might use in an arithmetic expression.

To enter such data as text, either format the cells as Text before entering the data.  Or enter the data with an apostrophe (single-quote) prefix; for example, '1-31-1969.  Even though you will see the apostrophe in the Formula Bar, it is not considered part of the value.  So for example, LEFT(A1,2) would be "1-" with my example.

graham mayor say:

This is mail merge - Mailmerge with Word 2007/2010

One record per document? http://www.gmayor.com/individual_merge_letters.htm and my two other mail merge add-ins will all do this apart from the addition of the hyperlinks. Without investigating too deeply, I suspect that the addition of the hyperlinks to the worksheet in the same process would be a step too far, at least as far as mail merge is concerned.

I would be inclined to create a separate process in Excel to create the hyperlink before running the merge and use that column to name the document. This would mean creating the filename and path(s) in Column B, derived from Column A with the addition of the path and extension. e.g.

=CONCATENATE("D:\My Documents\Merge\",A2,".docx"), and drag down to fill column B

then use the macro

Sub Add_Hlinks()
Dim lastRow As Long, i As Long
Dim xlSheet As Worksheet
Dim strPath As String
    Set xlSheet = ActiveSheet

    With xlSheet
        .Cells(1, 2) = "FILENAME"
        lastRow = .UsedRange.Rows(.UsedRange.Rows.Count).Row
        For i = 2 To lastRow
            strPath = .Cells(i, 2)
            xlSheet.Hyperlinks.Add .Cells(i, 2), strPath, , , strPath
        Next i
    End With
End Sub

to add the hyperlinks.

You can then merge with http://www.gmayor.com/individual_merge_letters.htm using the FILENAME field to name the documents. (Don't forget the check box to indicate the filename field includes the path(s)!)
