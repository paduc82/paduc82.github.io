jllatham say:

joe576 - Here is some code that I prepared as one example of how this could be done as a User Defined Function.  It is set up to work with numbers in column B, beginning at cell B2.  One routine adds up the numbers that are italicized in the cells, the other adds up the numbers that are not italicized.

 

You would use them just like a regular Excel worksheet function by putting formulas like this in cells on the sheet:

=ItalicizedTotal()

and

=NonItalicizedTotal()

 

To test the code, put it in a workbook into a Regular code module (see this page for 'how to')

http://www.contextures.com/xlvba01.html#videoreg

Then put some numbers into column B on a sheet, format some as italicized and leave some others alone.  Put the formulas onto the worksheet and watch the results.  It wouldn't necessarily have to be italicized and non italicized, it could be done based on whether a cell was shaded/colored or not.

 

Function ItalicizedTotal() As Variant
 Application.Volatile
'change these as needed
  Const numbersColumn = "B"
  Const firstNumberInCell = "B2"
'that's all you need to change
  Dim listRange As Range
  Dim anyEntry As Range

  Set listRange = Range(firstNumberInCell & ":" & _
   Range(numbersColumn & Rows.Count).End(xlUp).Address)
  For Each anyEntry In listRange
    If anyEntry.Font.Italic = True Then
      ItalicizedTotal = ItalicizedTotal + anyEntry
    End If
  Next
End Function


Function NonItalicizedTotal() As Variant
  Application.Volatile
'change these as needed
  Const numbersColumn = "B"
  Const firstNumberInCell = "B2"
'that's all you need to change
  Dim listRange As Range
  Dim anyEntry As Range

  Set listRange = Range(firstNumberInCell & ":" & _
   Range(numbersColumn & Rows.Count).End(xlUp).Address)
  For Each anyEntry In listRange
    If anyEntry.Font.Italic = False Then
      NonItalicizedTotal = NonItalicizedTotal + anyEntry
    End If
  Next
End Function

suzanne s. barnhill say:

Good question. There's an article about Paste Options at https://support.office.com/en-us/article/control-the-formatting-when-you-paste-text-20156a41-520e-48a6-8680-fb9ce15bf3d6 that doesn't mention Merge Formatting at all.

An old Answers thread cites another Paste Options article that says of Merge Formatting that "This option changes the formatting so that it matches the text that surrounds it." An Outlook question describes the action of Merge Formatting in such a way that it appears to be the same as Match Destination Formatting as defined in the first article I cited. What this suggests is that, if you have small amounts of direct font formatting (bold, italic), they would be retained by Merge Formatting but not by Keep Text Only.

diane poremsky mvp say:
if you don't use different formatting, they will be the same. Merge formatting changes the formatting so that it matches the text that surrounds it - it will keep bold and underline but will match the font families or colors with the surrounding text.

For example, you copy bold red text using comic sans from one document, merge formatting will keep the bold but not the red font.
