ruwim.b say:

Both with the new Microsoft Skype for Windows 10 (the 12.X version) and the classic Skype for Windows Desktop (the 7.X version), the chat history is saved locally on your computer and can be exported using either the build-in tool (available with the classic Skype) or using 3rd party applications.

 

With the new Microsoft Skype for Windows 10, the chat history is saved in the skype.db database file, which is located here:

 

%localappdata%\Packages\Microsoft.SkypeApp_kzf8qxf38zg5c\LocalState\<Skype Name>\skype.db

 

If you previously had the classic Skype installed on your computer, then you will find the chat history saved in the main.db file:

 

%appdata%\Skype\<Skype Name>\main.db

 

https://answers.microsoft.com/en-us/skype/forum/skype_windesk-skype_messms-skype_sms/sqlite-maindb-file/75258f84-4b2d-4284-82b0-958bed9f3b36

 

Both skype.db and main.db files are SQLite database files and can be opened using an SQLite browser/editor.

 

In the case of the main.db file (used by the classic version) you can use the free SkypeLogView application.

 

https://www.nirsoft.net/utils/skype_log_view.html

 

Using SkypeLogView it is possible to open the main.db file, read the content and export chat history to a text (.csv) or HTML file.

 

In the case of the skype.db file (Skype for Windows 10), you would need a standard SQLite browser to open this file and export the chat history.

 

A good example of such SQLite browser is the DB Browser for SQLite:application:

http://sqlitebrowser.org/

Insert a footnote and then select the footnote "number" in the text and press CTRL+SHIFT+s to display the Apply Styles dialog, which will have Footnote Reference in the Style Name control. 

Then click on the Modify button and in the Modify Style dialog, click on the U button to apply underlining to the characters used in that Style.

If you want the footnote reference character underlined in all documents created from the template that you are using, before clicking on OK in that dialog, select the radio button for "New documents based on this template"

stefan blom say:

The Footnote Reference style is applied, by default, to all footnote references (regardless of the selected numering format).

To modify a style, you can do the following: Make sure that the document that you want to modify is the active document in Word. Press Ctrl+Shift+S to display the Apply Styles pane. Type in the style name (Footnote Reference) and click Modify. Click the button for Underline and then click OK.
