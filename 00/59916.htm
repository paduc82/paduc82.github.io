wunders say:

There is a strong tendency to maintain the same IP address whenever possible -- although when you get right down to it, a lot depends on the particular DHCP server that assigns the address and Windows itself.

 

Yes.  Windows does remember its last IP address on a given network interface.  It is stored in the Registry (try searching the registry for your current address). When Windows first recognizes that it is connected to a network and that it needs an IP address assigned, it sends out a message that basically says "Hello DHCP server!  I need an IP address and, by the way, the last IP address I had was xxx.xxx.xxx.xxx".  The server can then respond with "Hello!  That IP address just happens to be currently available and you can use it starting now" ...  or it can just as easily respond "That IP address is not proper for this network at this time... would you consider address yyy.yyy.yyy.yyy instead?".   Also, some more sophisticated DHCP servers will remember the MAC (Ethernet) address of your network card from its last connection and attempt (at their end) to assign the last IP address that it remembered for you if it happens to be currently available.
