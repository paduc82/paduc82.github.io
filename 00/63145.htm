tryx3 say:

[Updated 11 June 2017 - para 7 added {potential solution to failures when making Recovery drives} and Note 3 shortened {I removed my invalid warning about making 32 bit installation media only on a 32 bit computer - I have now done so without any problems}]

Making and using a Windows 10 Recovery drive

Windows 10 has a better tool than Windows 7's System repair disk.  It is a

1  Right-click on the Start button, select Control panel from the list, [set it to View by - Small icons, if necessary],

Recovery, Create Recovery drive, set Include system files. 

2  Connect a 4GB USB [32 bit] or 8GB USB [64 bit].  You can use DVDs, I believe, but I find USBs to be more reliable and less breakable.

3  When you click Next, the system will spend an age looking for available drives and checking if they are big enough. So click Next and make a pot of tea.

4.1  Then it will tell you that it is ready to start and will show you a dropdown list of the drives that it will allow you to make the Recovery drive on.  Choose the correct drive then click Next.

4.2  Note that Windows does not know what your Bios can & cannot do and might list drives that your computer is not capable of booting from - in one case, it was happy for me to make the Recovery drive on an SD card that was in the computer even though that particular computer cannot boot from its SD card drive.

5.1  Then it will spend another age making the drive.

5.2  The progress bar it displays will not move for a long time and even after it starts to register progress, it will look as though it is not moving any more. You'll probably have finished your pot of tea by the time it has completed.  I think it took half an hour on one computer.

6.1  When the Recovery drive is ready, test it by booting from it. You do not want to have an emergency in the future and only then discover that the Recovery drive does not work. [I have no reason to suppose that they are unreliable, my caution stems from problems some of us found with Windows 7 System repair disks.]

6.2  To boot from the Recovery drive, connect it then restart the computer.  When your computer maker's logo appears, press the required key to get the Boot menu and then select the option to boot from the USB.  I cannot tell you which key to press as this varies between computer makers [it is not a Windows item as Windows has not loaded by that stage of starting the computer].  If it is not displayed onscreen alongside the computer maker's logo then check in your manual or on the computer maker's website.

6.3  When you boot, you will find many options for exiting without doing anything so you won't get into any difficulties by doing this test - you can, for example, select the option to Exit & continue into Windows 10.  But do take the opportunity to have a look around the menu system while you are in this "Windows recovery environment" [WRE] to see what the Recovery drive can do for you.

7  Many users have found that trying to make a Recovery drive with the Include system files option set keeps failing with the unhelpful message that it cannot make one.  I never found a solution to this in this forum so I stopped bothering [and returned to my previous practice of having an installation disk or System repair disk and then making frequent system images].  However, I have just found this proposed solution [but have not tested it or seen any reports from anybody who has] -

Denis

**  Notes

Note 1 - It is a reinstallation tool not an installation tool.  It can only be used to reinstall Windows 10 onto a computer that has previously had an activated copy of Windows 10 on it.

Note 2 - I understand that a Recovery drive cannot cope with reinstalling onto a disk with multiple partitions.  I have not seen that advice in any definitive form and have not had the chance to test this for myself yet. 

Note 3 - If you ever need to install or reinstall in other situations, see the procedural guide & links in Download Windows 10

tryx3 say:

As I wrote on 20th August, a Windows 10 Recovery drive is a reinstallation tool

A Windows 10 Recovery drive Windows recovery environment [WRE] is more capable than that of a System repair disk; a Windows 10 Recovery drive can do everything that a System repair disk can do & more.  Once you have a Windows 10 Recovery drive then there is no use for a System repair disk.

Boot up from your Windows 10 Recovery drive and you can have a look around its WRE perfectly safely [without actually doing anything].

If you ever need to restore a Windows 10 system image, boot up from the Recovery drive, connect the external drive.  Then in the WRE select Troubleshoot, Advanced options, System image recovery.

If Windows 10 is still working well enough, you could boot into its onboard WRE instead - click on the Windows button, click on Power, hold down the Shift key on the left hand side as you click on Restart, then let go of the Shift key.  This WRE does not have the full capabilities of the Recovery drive WRE but does include being able to restore from a system image.  You can also you can have a look around this WRE perfectly safely [without actually doing anything].

Will the repair CD(s) I created in Windows 7 (32 bit and 64Bit) work on a Windows 10 PC or do I need to create new ones? - these CDs know nothing about Windows 10 so could not be relied upon to conduct any repairs.  Since there are similarities between Windows 7 & Windows 10, it is possible that some of its repair functions might be reliable but you'd have no way of knowing. 

At the risk of telling you something you already know, it is useful to make new system images regularly.  I make one every week or fortnight.  The more recent the image that you restore, the less work you have to do afterwards [re-installing applications that were installed after the image was made and repeating changes to Windows or application settings made since the image was made].
