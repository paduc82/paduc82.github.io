In tblFiscal, I have a Fiscal Month field and a FiscalStart & FiscalEnd for the beginning and end dates of each month. I want the text box to look at the EventInHome on the current form and return the FiscalMonth. Right now, it's giving me a "#Name?" message.

Any help is greatly appreciated!

boblarson say:

A Dlookup has 3 parts. 

 

1. The field you want to return the value from in quotes

2. The table or query which contains the field you want the data from (in quotes)

3. The criteria as a string.

 

So,

DLookup("FieldNameHere", "TableNameHere", "[AnotherField]='Something'")

 

In your case you want dates so it would be:

 

=Nz(DLookup("[FiscalMonth]", "tblFiscal", "[EventInHome] Between [FiscalStart] And [FiscalEnd]")

amberaddy5 say:

Counting phone numbers as words

I am trying to set up a spreadsheet for our Classified Ads. I want to be able to type one ad in one cell and then in another cell count the number of words in the cell with the ad. I have found this formula  =IF(LEN(TRIM(A1))=0,0,LEN(TRIM(A1))-LEN(SUBSTITUTE(A1," ",""))+1).  And it works except it doesn't count the phone number as a word. So in my example I have
For   Sale-Diamond Ring. $400. (812)555-1234. 
However, it is only counting 5 words instead of 6.  Is there a way to make it count the phone number as a word?

jp ronse say:

Hi,

mike h.. say:

Hi,

The telephone number will be counted. I assume here that the 6 words are

For

Sale

Diamond

Ring

$400

(812)555-1234

The common way to count words is with a formula like this:-

=LEN(A1)-LEN(SUBSTITUTE(A1," ",""))

This simply compares the length of the string to the length of the string with all the spaces removed and the difference is the number of words. However, this will fail with your string because you have a hyphen between 'Sale-Diamond' so while I suspect that this; to you, is 2 words Excel counts it as one and returns 5 words in the string. We need to remove this first hyphen in our word counting formula like this:-

=LEN(SUBSTITUTE(A1,"-"," ",1))-LEN(SUBSTITUTE(SUBSTITUTE(A1,"-"," ",1)," ",""))

I have no idea how reliable this will be because it depends upon your writing style. If you sprinkle your Ad's with hyphens then it may not be reliable.

jeeped say:

You have a problem with the sample data that you supplied. It almost works, but only by coincidence and not for the reasons that are intended.

Specifically, you are attempting to count the words in that phrase by determining how many spaces are in the phrase and adding 1. This would normally give the number of words in any given sentence but yours is complicated by the fact that Sale-Diamond is separated by a hyphen, not a space. The problem is not that the phone number is not counted as a word; it is. The problem is that Sales-Diamond is counted as 1 word, not 2.

The next problem is that cannot simply add the count of hyphens to the count of spaces because there is a hyphen in the phone number and you want the phone number counted as 1 word, not 2.
