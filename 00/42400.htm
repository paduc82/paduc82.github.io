If you're seeing an error message when sending or receiving mail, please click on the error message to select it. Then use right-click to copy it and Ctrl-V to paste it into a reply here. The error code alone is not enough to allow for any sort of troubleshooting.

A messaging error cannot affect contacts - something else is happening. wlcomm.exe is the basic program for the Windows Live Communications Platform, that sees to keeping your contacts synchronized when you're signed in.

If you're signed in when WLMail starts, the first thing that happens is that your calendar and contacts are synchronized. This doesn't normally take long or use many resources, unless there's a problem with the databases. It looks as if you do have problems with your contacts databases.

> after signing out of Live mail it was still going 15mins later so I had to end proccess.

Do you mean that the program was still running after you had closed it? If so, that's quite normal, although it won't normally run for more than a few minutes while the databases are updated. However, these databases are pretty robust, so if you leave the processes running, they will most probably recover in due course.

If on the other hand you mean that wlcomm.exe was still running after you had signed out in WLMail using the right-most button on the Home tab of the ribbon, then again, this is probably quite normal. The process should shut down by itself along with wlmail.exe.

trekdozer say:
Can I use the key that was associated with the version that came on my notebook since it's now not in use since I bought a full version of Professional?

 

No, it's OEM.

 

Can You Move Windows 7 To A New Computer

If it's a retail Full or Upgrade license - yes. You can move it to a different computer as long as it's only installed on one computer at a time (and if it's a Windows 7 Upgrade version the new computer must have it's own qualifying XP/Vista license). The previous Windows 7 installation on the old computer must be formatted/deleted. You might have to call Microsoft and explain what happened to complete the activation. Activating it on the second computer will automatically in effect "deactivate" the license for the first computer. The key will work with both 32 and 64 bit, but only one can be installed at a time.

If it's an OEM license - no. OEM licenses, including Windows preinstalled on a computer before purchase and Windows bought separately, are tied to the first computer they are installed on and can not be transferred to a different computer. To install Windows on a different computer you will need to buy another copy.
