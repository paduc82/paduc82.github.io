If you are using the Data Analysis (Analysis ToolPak) Histogram tool, maybe your Input Range and/or Bin Range includes a non-numeric text label at the top of the column of numeric data, and maybe you did not check the Labels checkbox.

If you do check the Labels checkbox, as I recall, both the Input Range and Bin Range must have a label at the top of the ranges you specify.
