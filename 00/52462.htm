josé oliver-didier say:

It sounds that you may be following instructions to reset you Photo Gallery database as described in the support article: Windows Live Photo Gallery freezes or does not start

First, let me explain how Windows Photo Gallery (WPG) works. All metadata, thumbnails images are saved in a database either called Pictures.pd4", "Pictures.pd5" or "Pictures.pd6" for performance reasons, most metadata information (Captions, Star Ratings, Tags) is saved as well back to the image file, which is a good thing. In some cases it is not, for example if the folder is set to "read-only" or the image format does not support metadata.

In some rare cases Windows Photo Gallery's database may become corrupted. It has happened to me, with symptoms - blurry thumbnail or metadata not updating. This may happen due to a computer crash, a possible faulty hard drive (as it was in my case) or some other situation. If this is what happened to you, you could rename the files with any other name say "OLD_pictures.pd6" or "BACKUP_pictures.pd7" to back it up and re-launch Windows Photo Gallery. When WPG starts back up, it will not find the database file and create a new database file as if was just turned on for the first time. This will reset WPG back as if was new, so you may need to "Include folders" and depending on your photo collection it may take a while until all your photo files are read and metadata and thumbnails are visible again. I have about 200,000 photo files and it may take a few hours.

Keep in mind however, any information which was not saved back to the file would be lost as well as well certain settings will be reset. This is not bad however, as it may solve the problem you may be encountering.

If for some reason, this does not fix your problem, you can rollback the change by deleting the newly created Pictures database and renaming the backup file "OLD_pictures.pdx" back to the previous name.

Before performing the database reset procedure, try first to restart your machine. Make sure your PC has all the recent updates. If it is only on a single image file. Try opening it with another application such as "MS Paint", just to discard the possibility it is a single corrupted image file. Try removing and adding back the folder - from the Menu bar File | Include folder , selecting the folder and clicking the [Remove] button and adding it back again. If all fails, and does not resolve your situation, then you may need to reset the database as described in the support article.

As always, make sure you have your photos backed up on a seperate media, like an external hard drive. File corruption issues may be a sign your hard drive may be nearing its end.

Hope this helps and solves your situation.
