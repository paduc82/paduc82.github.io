a. user say:

Hi,

 

It is possible that the auto complete feature of your internet browser is activated. You may try deleting the AutoComplete information and enabling it again for later use. If you are using Internet Explorer as your internet browser, please follow these steps and check if this helps:

 

1. Open Internet Explorer by clicking the  Start button. In the search box, type  Internet Explorer, and then, in the list of results, click  Internet Explorer

2 . Click the  Tools button, and then click Internet Options

3. Click the Content tab

4. Under  AutoComplete, and then click Settings.

5. Select the option Delete AutoComplete History and select the all the options present and click Delete

6. Now in the AutoComplete settings window select the  Address bar,Forms, User names and passwords on forms, and ask me before saving passwords check boxes

7. Click  OK, and then click  OK again.

 

You may refer to this link for assistance on AutoComplete option. If you are using a different internet browser, please refer to your browser's help documentation.
