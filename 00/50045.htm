aashish g say:

Hi Dave,

 

Thank you for posting your question on Windows Vista Forums.

 

MDI is a proprietary file format of Microsoft for storing raster images of scanned documents together with optional annotations or metadata, and can only be produced or read by certain Microsoft software, such as Microsoft Office Document Imaging (MODI) module included in Microsoft Office 2003 and later.

 

However, when you upgrade to Windows Vista and Microsoft Office 2007, or bring the MDI files via USB flash drive or email to another computer with only Microsoft Office 2007, you will encounter an error of “Windows cannot open this file”.

 This problem occurs because “Microsoft Document Image Writer” feature is not installed by default when you run Microsoft Office 2007 setup.

 

In order to open and view or print the MDI files with Microsoft Office 2007 installed, try one of the following methods.

 

Method 1:

You can open the MDI file into OneNote page to view it instantly.

 

Method 2:

Try to Install Microsoft Document Imaging Writer on the machine.

1.       Click on Start and then click on Control Panel.

2.       Click on Program and features and then highlight Microsoft Office 2007.

3.       Select “Uninstall a program” link under Programs section.

4.       Click on “Change” located on the navigation link.

5.       Select “Add or Remove Features”, and then click “Continue”.

6.       Expand “Office Tools” section.

7.       Click on the drop down list for “Microsoft Office Document Imaging” and select “Run all from My Computer”.

8.       Click “Continue”.

9.       Click “Close” when installation done.

Unfortunately, no, but if the system came preinstalled with Windows 10 Pro OEM, you are entitled to the downgrade, but you will need to contact the manufacturer of your computer about obtaining downgrade media for Windows 7 Professional which will exempt the need for a key.

If all the systems are identical brands, you could get one Windows 7 Professional downgrade media then use it to install on all the systems you want to downgrade.

If your OEM has not injected your product key into the BIOS on your PC, follow these steps:

  1. Obtain genuine Windows 7 Professional or Windows Vista Business installation media and the corresponding product key. You may have to buy a full-package product copy of the Windows downgrade from a retailer.
  2. Insert the media for the downgrade version of Windows into the PC, and then follow the installation instructions.
  3. Type the product key when you are asked to do this. If the software was previously activated, you can't activate it online. In this case, the local Activation Support telephone number will be displayed. Call the number, and explain the circumstances. When it is determined that you have an eligible Windows license, the customer service representative will provide a single-use activation code to activate the software. Please be aware that Microsoft does not provide a full product key in this scenario.
