douglas_730 say:

Can't drop text on top of connector line.

I am trying to drop some text on top of a line and the line keeps moving away. I did not have this issue in Visio 2007. I've turned off "move other shapes away" and "connector splitting" but the line still moves away from the text. Any ideas? Thanks
barb way say:

I am guessing that you mean Connector lines in your diagram.  By default, the routing of Connectors in a template such as the FlowChart template is "Freely" or "As Needed" - which means that the connector will automatically re-route itself to avoid shapes.  Since text is just another shape, it will cause this behavior. 

You have two choices -

1) Change the connector routing style if you really need to 'drop' text shapes on it.  This is a bad idea - because then you will have to manually do all routing for the connector to avoid shapes that you do want to avoid.

2) Instead of adding text to a connector by dropping it, just select the connector and type your text, and it will become part of the connector line and route with it.

However, if you truly want to use method 1, and stop automatic routing for a line, you can use these steps for each line:

- Turn on Developer Mode (File tab / Options / Advanced / Run in Developer Mode

 (This is necessary to see the Developer Tab option for Behavior)

- Select a connector (or several connectors, but only connectors)

- From the Developer Tab, in the Shape Design group, select Behavior

- In the Behavior dialog, select the Connector tab, and select the Reroute drop down

- Choose "Never" to prevent any automatic line routing

Hope these ideas help!
