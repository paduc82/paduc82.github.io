Ruth,

My contact gave me the following for you to try (this is only for non US typecovers for  SP4). Please let me know if you try these and results either way:

Workaround #1

  1. Power+volume up to boot into UEFI settings.
  2. Choose Devices.
  3. Toggle Type Cover Port off and then on.

 

Workaround #2

  1. Check Settings > Ease of Access > Keyboard and ensure that Filter Keys is disabled.
  2. Check BIOS and make sure Type Cover is enabled in Devices.
  3. In Device Manager, click on the View menu > Show Hidden.
  4. Under HID (Human Interface Devices), scroll down and find Surface Pro 4 Typecover Integration > right-click - Uninstall. (chose option to delete driver/software.)
  5. In Device Manager, under Keyboard, if you do not see the type cover, no need to do anything else. I; if you do see it, right-click, uninstall (check to remove software).
  6. Shut down (hold shift for a clean shutdown).
  7. Hold power and volume up for 30 seconds. The device will flash to EFI and back until it eventually powers off fully.
  8. Power on the device. Now the keyboard should work, and all the drivers should be re-installed that we removed earlier.
