- I enter some text using the pen on my tablet

- I select the text using the lasso (pressing button on side of pen then circling text)

- I can move the selected text, but I can't see that there's an easy way to delete the selected text

tom versluys say:

I also searched for the solution and this is what I found.

Select the items with the lasso.

Press with your finger next to the move icon (circle with cross).

A box will appear and you can "cut, copy or paste or delete" your selection.
