jay freedman say:

The general idea is to set the font color of the header and footer to white, print the document, and then set the font color back to Automatic or whatever it was initially.

The simplest way to do that is to set ActiveDocument.Styles("Header").Font.ColorIndex = wdWhite and ActiveDocument.Styles("Footer").Font.ColorIndex = wdWhite at the start and then set them to wdAuto.

However, if your document has Different First Page turned on and then has different text in the main header/footer that should print, it's a bit harder. Instead of changing the styles (which affects the entire document), you have to change ActiveDocument.Sections(1).Headers(wdHeaderFooterFirstPage).Range.Font.ColorIndex and similarly for .Footers.
