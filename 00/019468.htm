doug robbins - mvp office apps & services (word) say:

When you create a document from a template, a reference is created from the document to that template so that items such as macros that are stored in the template are available for use with that document. 

If you subsequently save, close and then re-open the document on the machine that has the template, the reference to the template is re-created.  However, if the document is opened on a different computer on which that template is not available, then the reference to the template cannot be created. 

Therefore, if you need the macros to be available wherever the document is opened, they will need to be created in, or be imported into the document, which will need to then be saved as a macro-enabled document with a docm extension.

Note, even then, the macros may not be available if the document is opened on a machine that has the macro security set so that macros are disabled without notification.
