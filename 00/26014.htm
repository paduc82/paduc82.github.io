Function FindString(Str As String)
'// VBA:  Tools / References set to:
'// Microsoft VBScript Regular Expressions 5.5

    ' Word Boundary + 3 Letters + non-white space characters
    Const Pattern As String = "\b[A-Z]{3}\S*"
   
    Set RE = New RegExp

        .Pattern = Pattern
        .Global = True
        .IgnoreCase = True
        If .Test(Str) Then

charles kenyon say:
Word 2007 does not have a checkbox in the new format so you will have to use the legacy form controls and protect it for filling in forms. Unless you need formfields in an unprotected area, do not use any of the new controls in this form.

Do not use ActiveX controls unless nothing else works. These are different from legacy checkboxes. They are next to each other on the ribbon, you want the controls from the top line of the pop-up window, not any of the ones under Active-X.

An alternative is the macro / macrobutton checkbox but that is only for use if you are not protecting the document for forms. Add Interactive Toggle Objects to Simple Word Documents. Also available are Content Controls Custom Checkboxes. Both would need to be installed on the user's computer to work easily.

Note, you can password protect a document and not have it protected for forms.

I would recommend using the Add-In from Greg Maxey's site.
Add Classic FormField Controls to Ribbon
That gives you the lock button and makes it less likely that you will insert an Active-X checkbox by mistake.

Note that Greg Maxey's pages provide downloadable Add-Ins that will let you insert checkboxes without understanding a lot of code. The ones for Interactive Toggle Objects or Custom Content Controls, though, would have to be on the user's machine for the simplest implementation.

galahad72 say:

Cambria Math, Word's default math font, has both old style and lining numerals. It defaults to the numeral style you've chosen for the text around the math zones. If that style is old style, then Cambria Math will give old style numerals in the math zones. But it's normal to have lining numerals in math. I want to mix the elegance of old style in the text with the greater legibility of lining in the math. (e.g. for super- and subscripts). Is there a way to do this besides turning on the font's lining numerals every time an equation has numbers?

There doesn't seem to be a way to set a default for the Cambria Math font independently of the font used outside math zones; I hope I'm wrong.

jay freedman say:

There is no separate option for the font setting in math regions, but it is possible to fix that with a macro. Install the following macro in your Normal.dotm template or in a global template, and assign a Quick Access Toolbar button or keyboard shortcut to it. You can run the macro at any time and as often as needed, and it will change all the equations currently in the document.

Sub LiningNumeralsInEqs()
    Dim objEqn As OMath
    For Each objEqn In ActiveDocument.OMaths
        With objEqn.Range.Font
            .NumberForm = wdNumberFormLining
        End With
    Next objEqn
End Sub
