lars-Åke aspelin say:

I think that you will need a macro to do that. Try this one:

Sub remove_all_hyperlinks()
   Dim h As Hyperlink
   n = ActiveSheet.Hyperlinks.Count
   For Each h In ActiveSheet.Hyperlinks
     h.Delete
   Next h
   MsgBox n & " hyperlinks deleted!"
End Sub

Right click on the sheet tab and select "View Code", paste the code from above in the big text window, close the Visual Basic window, enter ALT+F8, select the macro remove_all_hyperlinks and click on Run.
