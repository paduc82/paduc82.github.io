How to update a custom document variable value? - Microsoft Community

johnny_dollar say:

How to update a custom document variable value?

I have an existing Word 2016 file that is a formal report.  I was tasked to create a new report of the same type, so I endeavored to edit and convert the existing file into a template (.dotx) file. 

After creating the template file, I came across something that puzzled me:  if I opened the template file, I saw all of my changes.  But if I created a new file based on the template, some of the edits disappeared and instead I saw the original, unedited text. 

Eventually, I determined that the places where the unedited text appeared were Word fields in the form of document variables.  One of them, for example, was called "officecode", which was clearly a custom variable not predefined in Word.  I haven't found any place to change the value of this variable.  For example, if I go to File-->Properties-->Advanced Properties-->Custom tab, the "name" list box does not show any "officecode" value to select.  Any thoughts on how I can change this variable value?

johnny_dollar say:

OK, I think I have it solved now.  Utilizing the information in the article from support.microsoft.com that Charles provided, I executed this statement for each variable Name and New Value:

ActiveDocument.Variables(Name).Value = NewValue

I also had to do a right-click and "Update field" on each instance of the document variable afterwards.

charleskenyon (ia) say:
If you are using macros anyway, try the following: 

Sub UpdateAllDocVariable()

'   Update all DocVariable fields in a document, even if in headers/footers or textboxes

'   Based on code at http://www.gmayor.com/installing_macro.htm
'   Charles Kenyon
'   18 October 2018
'
    Dim oStory As range
    Dim oField As Field
    '
    For Each oStory In ActiveDocument.StoryRanges
        For Each oField In oStory.Fields
            If oField.Type = wdFieldDocVariable Then oField.Update
        Next oField
        '
        If oStory.StoryType <> wdMainTextStory Then
            While Not (oStory.NextStoryRange Is Nothing)
            Set oStory = oStory.NextStoryRange
            Wend
        End If
        '
    Next oStory
    '
    Set oStory = Nothing
    Set oField = Nothing
End Sub
