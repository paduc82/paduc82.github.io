In order to do it for free you need a Pro product key for either 7 or 8.  Then you could clean install Build 10586 using its Media Creation Tool when that is put on the Microsoft website for it and use the retail key to activate.  The 10586 official .iso may not be available until the public release for obvious reasons (Gee, I hope I'm proven wrong on that one).  If you are also migrating from x86 to x64 you would have to make the media with the Media Creation Tool on a 64bit system.  

Build 10586 is officially "Windows 10 1511" and is called the "Fall update." It is not an insider build.  Insiders are simply getting the Fall Update ahead of the general public.  

Once you are on the Fall Update you can use the option in the Activation screen to upgrade from Home to Pro.  This cannot be done on any other builds except 10240 and 10586 because an insider build cannot be so upgraded.  The cost is $100 plus any local taxes and fees.

Leaving the program while on a public release build requires nothing more than changing the settings in Windows Update/Advanced Options and rebooting.  My experience is that stopping insider builds requires two attempts before the change sticks.  (On the other hand, leaving while on an insider build requires an upgrade-in-place back to the latest public release.)

ipellant say:

Immediately after a build version update there are a lot.. repeat "A LOT" of background clean up and configuration processes that begin running. I call some of them "barnacles"

Reduce the barnacle load - such as uncheck the "Allow files on this drive to have contents indexed in addition to file properties".... it often kicks off a massive re-index just after an OS replacement.

Use Task Manager to monitor the CPU load. It may be very high just after an OS update. Don't expect the computer to respond to you until the idle CPU load drops to under 16%.

I have Build 10061 running on an old HP Mini - 32bit Atom @ 1.6 GHz with 2GB RAM. A 2009 vintage netbook. It has updated all versions of Windows 10 TP so far with no problems (incredible). Just after an OS update, I leave it alone for an hour or more before expecting it to be responsive. Build 10061 began as a tortoise... now a few days later: it is a hare.
