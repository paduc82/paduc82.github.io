bernie deitrick say:

To do the average with substitution, use an array formula like this, entered using Ctrl-Shift-Enter:

=AVERAGE(IF(B1:B10<-50,-50,IF(B1:B10>50,50,B1:B10)))

But, it would be better if you deleted outliers completely rather than substituting values for them:

=AVERAGEIFS(B:B,B:B,">=-50",B:B,"<=50")

Or to get rid of, say, the 3 largest and 3 smallest values

=AVERAGEIFS(B:B,B:B,">" &SMALL(B:B,3),B:B,"<" & LARGE(B:B,3))

You might also want to consider the Median instead of the Average:

=MEDIAN(B:B)

tom ogilvy say:

If you don't believe in the data you should exclude it from the calculation and explain why you omitted the value.

Otherwise your calculations are a misrepresentation of the facts.   

If you want to engage in what might be perceived as a deceptive practice you can use a formula like. 

=SUM(IF(A1:A100=-150,-50,A1:A100))/COUNT(A1:A100)   entered with Ctrl+Shift+Enter since this is an array formula.

hansv mvp say:

Let's say your data are in A2:A13.

The following array formula confirmed with Ctrl+Shift+Enter will replace values below -50 with -50:

=AVERAGE(IF(A2:A13<-50,-50,A2:A13))

If you also want to replace values above 50 with 50, use (again confirmed with Ctrl+Shift+Enter):

=AVERAGE(IF(A2:A13>50,50,IF(A2:A13<-50,-50,A2:A13)))

I would use population statistics in some form. You could use a formula like this, entered using Ctrl-Shift-Enter (array-entered)

It will average any number within +/- 1.5 Std Deviations of the Median.  The 1.5 can be adjusted as you like....

=AVERAGEIFS($B$1:$B$500,$A$1:$A$500,A2,$B$1:$B$500,">" & MEDIAN(IF($A$1:$A$500=A2,$B$1:$B$500))-1.5*STDEV(IF($A$1:$A$500=A2,$B$1:$B$500)),$B$1:$B$500,"<" & MEDIAN(IF($A$1:$A$500=A2,$B$1:$B$500))+1.5*STDEV(IF($A$1:$A$500=A2,$B$1:$B$500)))
