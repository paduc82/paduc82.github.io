ossiemac say:

I am thinking that we may not be on the same wave length here. You initially asked how to return to the previous position and my first answer showed how to obtain the current position of the control, move it to a new location and return it to the original location. While my code gave the principles of doing this, if you really want to retain values for the next time code is run then the best way is to save them to a worksheet cell. I usually use a hidden worksheet for this purpose and then the values are saved until changed again even when the workbook has been saved and closed.

 

Your next post indicated that you simply wanted to position the control to a cell position based on the selection and if this is the case then there is no need to save the previous position because the top and left are re-allocated in the case statement based on the item selected.

 

If you want the top left corner to be offset from the top and left of the cell then you can add to these values. You mention having top left in center of cell so add half the width of the cell to the left value and add half the height of the cell to the top value as per the following.

 

    With WkSh
        Select Case MyIndex
            Case 1      'Move to N1
                dblNewTop = .Range("N1").Top + .Range("N1").Height / 2
                dblNewLeft = .Range("N1").Left + .Range("N1").Width / 2
               
            Case 2      'Move to A1
                dblNewTop = .Range("A1").Top + .Range("A1").Height / 2
                dblNewLeft = WkSh.Range("A1").Left + .Range("A1").Width / 2
               
            Case Else   'Move to V1
                dblNewTop = .Range("V1").Top + .Range("V1").Height / 2
                dblNewLeft = .Range("V1").Left + .Range("V1").Width / 2
               
        End Select
    End With

 

If this is not answering your question then try to explain step by step what you want it to do and what conditions need to be met for each step.
