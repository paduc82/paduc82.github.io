  3. Click the font that you want to remove. To select more than one font at a time, press and hold the CTRL key while you select each font.
  4. On the File menu, click Delete.
  5. When you receive the "Are you sure you want to delete these fonts?" prompt, click Yes.

To delete the fonts in HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Fonts:

  2. Type regedit, then hit Enter.
  3. Navigate the following folder HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Fonts.
  4. Click the font that you want to remove. To select more than one font at a time, press and hold the CTRL key while you select each font.
  5. Right-click on the highlighted fonts, then select Delete.
  6. When your receive the "Deleting certain registry values could cause system instability. Are you sure you want to permanently delete this value?" prompt, click Yes.

Once you're done with deleting all the fonts, please try to install them again.

Note: You can prevent a font from loading without removing it from the hard disk. To do this, move the font from the Windows\Fonts folder to another folder.
