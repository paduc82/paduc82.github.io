Select the document using Windows Explorer. Press F2 or right click and select Rename. Copy the full name including the extension and paste it into the document. Replace the path separators with double slashes


add quotes

Add IncludeText

{ IncludeText "c:\\path\\filename.docx" }

and press F9 to update

suzanne s. barnhill say:

A simpler solution, which will guarantee getting everything right, is to use Insert | Text | Object | Text from File | Insert as Link. This will insert an IncludeText field in the proper format.

If the desired text is just a portion of the source file, you can bookmark it and include the bookmark in the field by clicking on the Range... button in the Insert File dialog and typing in the bookmark name.
