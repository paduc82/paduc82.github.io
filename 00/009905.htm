hansv mvp say:

Use

=(C6-B6)+(C6<B6)+(E6-D6)+(E6<D6)+(G6-F6)+(G6<F6)+(I6-H6)+(I6<H6)+(K6-J6)+(K6<J6)+(M6-L6)+(M6<L6)+(O6-N6)+(O6<N6)

If the end time is earlier than the start time, it is actually on a later day than the start time. For this reason, we add 1 (i.e. 1 day = 24 hours) to the difference if the end time is earlier than the start time.

C6<B6 is TRUE = 1 if C6 is earlier than B6.

C6<B6 is FALSE = 0 if C6 is not earlier than B6.

Notes:

Have a look at the Name Manager for "MemberList". You will see the following "Refers to formula" which provides a dynamic list for the Defined Name and as you add names to the bottom, the "MemberList" automatically expands.

=OFFSET('2016 Members List'!$B$2,1,0,COUNTA('2016 Members List'!$B:$B)-2,1)

Explanation of the formula.(If you click in the Refers to formula field then immediately press F2 to place it in Edit mode otherwise when you move the cursor with arrow keys, Excel thinks you are selecting cells on the worksheet and enters them in the formula.)

Offset function has the following arguments:    OFFSET(reference, rows, cols, [height], [width])

Reference : Starts at cell B2 (Column1 Table header)   '2016 Members List'!$B$2

Rows:          Moves one row down to B3  '2016 Members List'!$B$2,1

Columns:    Moves zero rows across (remains in same column)  '2016 Members List'!$B$2,1,0

[Height]:    Firstly Counts the cells in column B that contain data.    COUNTA('2016 Members List'!$B:$B)

                  Secondly Subtracts 2 from the count (Cells B1 and B2)      COUNTA('2016 Members List'!$B:$B)-2

[Width]:     Total width of the range is 1     This is last parameter.

To see the VBA code, open the VBA editor and in the left column double click "ThisWorkbook" module in the Project Explorer (left column)

The code has been placed in this module so that you only need the one code sub for the entire workbook.

Workbook_SheetChange event is similar to Worksheet_Change event except that it passes the worksheet Id that was changed via Sh.

The code tests the first cell (A1 which is merged from A1:W1) and if it contains the header "AAA Skirmish Day Booking Form" then it will process for that sheet and this then excludes other worksheets in the workbook.

The code then tests that only one cell has been changed and exits if more than one. This will allow multiple deletes without causing a problem.

The code then tests to see if the changed cell is in the Range("C5:C64").(If you extend this range then you will need to edit the code to include the extended range).

If the user selects "Y" in column C then a validation DropDown is created for column B and the cell in column B is selected. If the users select "N" or clears the contents of cell in column C then everything is cleared from the adjacent cell in Column B.
