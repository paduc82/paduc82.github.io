a. user say:

Supported lifetime of the device is begging the question, and saying that it will be supported for the supported lifetime of the device is just circular.  We will support it as long as we support it.  The use of the term "device" alludes to hardware.  Hardware may become obsolete, and no longer supported by Windows, at some future date, but it's not predetermined.

Because Windows 10 is a rolling release distribution, it will continue to be updated to what would have been called Windows 11, 12, 13, but instead it will always be Windows 10, being updated from time to time.  The incremental updates won't typically be as dramatic as one might expect from a full shift from say Windows 7 to Windows 8, but it will be in bits and pieces instead of one giant change.

I don't think Microsoft can give a particular date to extend support to, because Windows 10 is not a static OS, so, as long as your computer is capable of running whatever Windows 10 becomes, it will be supported with security updates etc.  Will there come a time when your computer will no longer be compatible with what Windows 10 becomes?  Possibly - but we're not there yet, and I don't think that Microsoft knows in advance what hardware may at some point be deemed obsolete and unable to run what windows becomes.

For example, there are processor requirements for running Windows 7.  You can't run it on an 8088.  Suppose you have a core i7 processor.  In 20 years, it may be just as obsolete as the 8088 today.  There's very little chance that anyone would be using a 20 year old processor, so in reality few to no people would be harmed by discontinuing support for the i7 in 20 years, but Microsoft hasn't predetermined when a particular bit of hardware may become obsolete.
