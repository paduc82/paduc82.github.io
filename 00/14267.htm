Updating merged cross-reference document - Microsoft Community

janmwc say:

Updating merged cross-reference document

I am currently working in Word 2007.  We have primary (master) documents created in Word 2003 (which originally came from a WP environment!) which cross-reference paragraph numbers. When a primary document is merged, the cross-reference paragraph numbers show up in the merged document.  Recently, we noticed incorrect paragraph numbers and when we try to update them (update field), they are no longer "greyed" in (this would indicate to us that this was a cross-referenced paragraph).  Our office was all using 2003 and then late last year I was switched to 2007.  Everything still worked fine until last week when we noticed this cross-referencing problem.

doug robbins - mvp office apps & services (word) say:

You will need to unlink the cross reference fields before executing the merge as cross reference fields use hidden bookmarks and each bookmark in a document must be unique whereas the merge process would attempting to create a new set of the same bookmarks for each record in the data source.

Running a macro containing the following code on the main document before executing the merge will convert the cross references into ordinary text so that they no longer rely on the hidden bookmark to provide the data for them.

Dim aref As Field
With ActiveDocument
    For Each aref In .Fields
        If aref.Type = wdFieldRef Then
            aref.Unlink
        End If
    Next aref
End With

Either do not save the document after running the code, or run it on a copy of the document if you later on may need to edit the original.

-- Hope this helps.

vanessa oca say:

You may change the extension by following the steps provided below:

Step 1

  1. Click on File Explorer.
  2. On the menu tab, select View.
  3. Once you see the drop down, click Options located in the upper left pane.
  4. A small window will pop up, and on the menu tab; click View.
  5. Under Advanced settings, uncheck Hide Extensions for known file types.
  6. Then click OK.

Step 2

  1. Right click the Excel File.
  2. Select Rename.
  3. Change the extension of the file from .xlsx to .xls.
