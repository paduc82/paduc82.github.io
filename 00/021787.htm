Just in case: A Continuous Section Break will show as a double blue line with Section Break (Continuous) in the center.

If instead it says Section Break (Next Page), follow these steps:

  1. In Word, click to the left of the first paragraph mark on the page immediately after the break.

If it is already Continuous, use this technique to switch it to New Page, OK out, re-enter the dialog and change it back to Continuous on the off-chance that it will reset the behavior.

steve rindsberg say:
The features you'd need to manipulate the clipboard as you'd like to are simply not built into VB or VBA.  You'd need to call on a lot of Windows API stuff to get there.

As you mention, the Office clipboard isn't available under VBA (or any other way other than manually, as near as I can tell).  

If you're just looking to move text from one shape to another, this would do:

Sub CopyText(oSourceShape As Shape, oTargetShape As Shape)
    oTargetShape.TextFrame.TextRange.Text = _
    oSourceShape.TextFrame.TextRange.Text
End Sub

Sub Test()
' Put two shapes on an otherwise blank slide one
' Add text to the first of them
' Run this:
    With ActivePresentation.Slides(1)
        CopyText .Shapes(1), .Shapes(2)
    End With
End Sub

If you need to preserve whatever was on the clipboard, copy/pasting is out, unless you want to get into some tricky WinAPI stuff (and it may not really be possible at all).

On the other hand, if you're trying to preserve something that you put there from PPT in the first place and don't much care about blowing away things that other apps might have copied, it might be enough to create a temporary BLANK slide, paste the clipboard onto it, do whatever you need to do, then grab the first shape on the blank temp slide, copy it, then delete the slide.
