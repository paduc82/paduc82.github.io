It seems there's a problem in recent versions of Word (I'm on the Office 365 sub, version 1803). Even in safe mode it remains in memory, even if you don't actually open a document. This is playing havoc with scripts that run macros because the process never ends.

The problem is in clienttelemetry.dll. You can bypass this by setting the following DWORD registry key to 1:

HKCU\Software\Microsoft\Office\Common\ClientTelemetry\DisableTelemetry

Do try running in safe mode first though, in case it is a problem with an add-in or custom template.
