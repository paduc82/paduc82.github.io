Delete file in Word 2016 for Mac - Microsoft Community

david d'arcy say:

Delete file in Word 2016 for Mac

There's no right clicking on a Mac.  Typical Microsoft garbage; all counterintuitive!  

I'd like to hear from folks using 2016 Mac version, please.

bob jones aka: cybertaz say:

Version is irrelevant :-)

"Right-Click" is a method or producing a contextual menu which most definitely is supported on Macs provided the pointing device has a right button. If the device does not have a right button the contextual menu is evoked by holding a Control key while clicking.

However, you give no indication of what steps you're taking or where you are attempting to do so. Without any descriptive information it can only be assumed that perhaps you are attempting to delete a locally stored file while in a Save or Save As dialog window. Microsoft has nothing to do with that. It is a restriction imposed by Apple's operating system.

The more recent versions of macOS (at least as far back as 10.x.x) do permit deleting [Move to Trash] certain user files* while in the File> Open dialog. It can be done either by using the contextual menu methods described above or by holding the Command key while pressing delete key.

*Note that while using any given program only files recognized by that program can be deleted or duplicated. IOW, while using File> Open in Word you cannot delete an Excel file or other files of a different type.

In order to delete, rename, copy or move files otherwise you must use the macOS Finder to navigate to the location where the file is stored while the file is not in use. You then can either right-click or Control-click the file or simply drag it to the Trash.
