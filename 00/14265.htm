john korchok say:

Pages in Word are displayed on the fly, but don't exist as separate entities in the Word file. A Word page cannot exist without content to make it appear. So when you delete the text, the page disappears as well.

If the header and footer disappear on the first page, you aren't creating them correctly. Here's a Microsoft tutorial page to show you how: Add headers and footers and change page numbers in Word 2016

suzanne s. barnhill say:
Instead of using a section break, make use of Word's Different First Page setting to create your letterhead template. With this method you set up the template with both First Page Header (containing the letterhead) and primary Header (containing the continuation header); even though you delete the second page, Word remembers the header and will use it when a letter is longer than a page. I guess it depends how you are referencing them. Do the text boxes have specific names? Are they the only text boxes on the slide? If they are the only text boxes on the slide, you could do something like this. Note I'm typing this on my phone so it is untested and there are probably typos.

Sub CompareAll()

   Dim current as Long

Dim highest as Long

Dim oShp as Shape

Highest = 0

For Each oShp in ActivePresentation.SlideShowWindow.View.Slide

   If oShp.HasTextFrame Then

      current = oShp.TextFrame.TextRange.Text

     If current > highest Then

         highest = current

       End If

   EndIf

   Next oShp

   MsgBox highest

End Sub

You have a phone, email program, or email server that is accessing your account's server using pop3 protocol.  After the online Inbox emails are downloaded, a signal is sent telling the server to delete the downloaded messages.  Instead of deleting the emails, the outlook.com server is moving the emails to the POP folder. 

You need to go into the advanced settings on the phone, email program, or email server and select to Leave a copy of messages on the server after download.

stefan blom say:
The built-in caption labels are language-and system-specific, as you have noticed. If you make use of a custom label, the list of figures will work on all computers.

Note that all users must create the caption label, though, because labels are always stored locally, in your Normal template.

Windows Mobile can only sync with the default Outlook folders. When troubleshooting this can work to your advantage.

 

Find those three appointments refusing to sync, create anther Calendar folder in Outlook, and cut and paste those appointments there.
