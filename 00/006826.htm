john mcghie say:

Your default paper size is stored in your Normal template.

That can be literally "anywhere" on your system, and there may be several of them.  To find out which one Word thinks is the "real" one, start Word and go to Word>Preferences>File Locations...

Select the User Templates line, and click "Modify".

Don't make any changes, just drop down the folder name box at the top and write down the complete path to the folder.

Now, quit Word then navigate to the folder you wrote down.

Select the file named Normal.dotm and delete it.  If you find a file named "Normal" or "Normal.dot" delete them too: they are old versions from previous copies of Word: if you leave them there, their old settings will be picked up in the new file.

It pays to restart the system at this point.

Now, when you restart Word, it will create a new instance of Normal.dotm, and you will be able to set your preferred paper size.  You "may" need to delete the printer once more to get rid of the corrupted definition there.

Hope this helps

Chart with pounds and ounces - Microsoft Community

toosase say:

Chart with pounds and ounces

I need to make a chart that has the pounds and the ounces displayed, but not like 4.6 but in the format 4lbs 6ozs and have it chart correctly.
peter atherton say:

Excel can only chart numbers so you have to enter the weight in decimal format 7bl 8oz as 7.5 then you create a data label with a formula in the next column. if the weight is in A4 then B4 contains the formula

=INT(A4)&"lb "&(A4-INT(A4))*16& "oz"

copy this formula down

Create you your chart from the data in column A.

 

You now need a macro to add the data labels

 

Sub AddChtLabels()
  Dim seWeights As Series
  Dim pts As Points
  Dim pt As Point
  Dim rnglabels As Range
  Dim iPoint As Integer
 
  'Range containing labels
' edit this Range to suit data!!

  Set rnglabels = Range("B4:B12")
  'Get the  firstr series from chart
  Set seWeights = ActiveSheet.ChartObjects(1).Chart.SeriesCollection(1)
 
  seWeights.HasDataLabels = True
  Set pts = seWeights.Points
  For Each pt In pts
    iPoint = iPoint + 1
    pt.DataLabel.Text = rnglabels.Cells(iPoint).Text
    pt.DataLabel.Font.Bold = True
    pt.DataLabel.Position = xlLabelPositionAbove
   
  Next pt
End Sub

'==================

Press ALT + F11, Insert, Module and copy the above code. Press ALT Q to return to the worksheet.

Press ALT + F8 to activate the Macro Dialog, select the macro and click run.
