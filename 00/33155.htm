I was able to resolve this as follows:
1. Start in safe mode
2. Run registry editor
3. Search for repair.js
4. When found, delete it.
5. Restart PC in normal mode. Problem solved!

Detailed instructions below for performing the steps above:
First startup your PC in safemode.
A sure way to enable safe mode (for Win7 and Win8) is as follows. Run "msconfig", when msconfig is open, go to the boot tab and tick the checkbox that says "safe boot", apply changes then restart PC. It will start in safemode.
Note: to find "msconfig" just hit the windows key and type "msconfig" without quotes. Then click on it to run it.

Now once in safe mode, run "regedit". To find regedit, press windows key, type "regedit" without quotes, then click on it to run it. The Registry Editor will open.
When it is open, press ctrl+F to open the find window. Type "repair.js" then click on "find next" or just hit "Enter" key. It will begin searching the registry. When the search is complete it should come up with a list, one of them should be highlighted and will have the path,
C:\users\user\appdata\local\instant savings app\repair.js
Simply right click on it and click on delete. It will ask you if you are sure you want to delete it. Delete it with confidence as this is the source of your problem. When you delete it then search again for "repair.js" just in case it appears anywhere else in your registry. I doubt it will though. If there is however another file called "repair.js" in your registry, that is related to the instant savings app, delete it also.
Problem should be resolved, now just run your PC normally again. I.e. not in "safe mode".

To do this, run "msconfig" again by pressing windows key, typing "msconfig" then clicking on it. Go to the boot tab, uncheck the box saying "safe boot". Apply changes then restart your PC. It will start in regular mode and no more annoying message at start up.
