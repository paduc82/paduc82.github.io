Any thoughts?

Oscar

 

raju s das say:

Ensure you have not selected the option 'remove from server after' in Outlook.

Open Outlook> Tools> Account settings> Highlight the email account> Click on Change> More settings> Advanced> under ‘Leave a copy of messages on the server’ if you have check ‘Remove from server after’ then you may uncheck the box and verify the status.

oscarlerma say:

I found the cause of the emails being removed from the server.  When I went into Account Settings before I didn't notice that another account had been set up.  I believe the automated set up software from AT&T must have added the account since I did select Outlook as my default email client.  When I started Outlook the new account began downloading my existing emails and deleting them from the server since by default Outlook will do that with a new account.  But since the email address was the same they were all going into my regular inbox.

 

I removed that boggus email account and now everything is back to normal.

Oscar
