don cuthbert say:

$Windows.~WS is likely leftover from the upgrade via Media Creation Tool from Windows 7 or 8 to 10.

$Windows.~BT is used by Windows Update to upgrade Windows.  If you delete it the upgrade will fail although this folder will be recreated by the next Windows Update upgrade attempt.  The download is at least 3GB.

 

A file at $Windows.~BT\Sources\install.esd can be used to create an ISO, but this file is only there briefly, after Windows Update has begun preparing the upgrade and before a restart.  To make an ISO from it, a copy has to be made and the file needs to be converted to an ISO.  There are user scripts written that can do this using Microsoft software.  If you search for ESD Decrypter in these forums you will see discussions on this.

If your current Windows 10 is activated then upgrading to this new Insider Build should also be auto-activated once complete.
