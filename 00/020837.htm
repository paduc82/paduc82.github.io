In a query based on the table, add the fields that you want to display, plus a calculated column:

Clear the Show check box for this column, and enter the appropriate month number in the Criteria row, e.g. 1 for January, 2 for February etc.

Click in the Field: row in the first unused column.

Type

 

M: Month([Date of Birth])

 

and press Enter. Here, M is the "name" of the calculated column. Since we will use this column only to select records, not for display purposes, the name is not important, so I chose a short, meaningless one.
