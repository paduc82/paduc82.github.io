a. user say:

May i use SFC command in CMD for repairing the following CHKDSK error: unspecified error occurred

What are the advantages and limitations of this command? is it have any risk for stored data?

Thanks.

There isn't enough information. Give us more in detail.

1. SFC stands for system file chek. It looks for corrupted files and makes attempt to repair them. It does not ALWAYS succeed in repairing them. It will tell you if there are some it cannot repair.

2. CHKDSK stand for check disk . It is meant to repair the disk.

There is no risk whatsoever in running any of the above.

How to run system file check :
Start button > search box type cmd > look up, RIGHT click oncmd.exe > clickRun As Administrator > in that black and white cmd window, type at the flashing promptsfc /scannow > press ENTER key.
Note : there is a Space between "sfc" and "/".
Sit back and wait. It will take some time.
When done, exit cmd window.
Reboot ( re-start your computer )

How to run CHKDSK :

Start button > search box, type cmd > look up, right click oncmd.exe > Run As Administrator >
in the black and white window, at the flasing prompt, typechkdsk /f/r > press Enter key.
Note : there is a space between " chkdsk " and " / "
The screen will say something like can't do it now but do you want to run it on restart. Click " y " as in yes > press Enter > Exit cmd window.
Restart your computer. It will take quite some time. DO NOT shut down the machine. Just wait.
