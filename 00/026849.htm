jllatham say:

Make a copy of your workbook to test with.  Replace your current code with the following and see how it works for you:

 

Private Sub Worksheet_Change(ByVal Target As Range)
  Dim searchRange As Range
  Dim searchFor As Variant
  Dim foundCell As Range
 
  If Target.Column <> 11 Then
    Exit Sub ' in wrong column
  End If
  'for tests, make sure no leading/trailing whitespace
  'and that it is all uppercase.
  Select Case UCase(Trim(Target)) ' .Value is implied
    Case Is = "YES"
      Target.EntireRow.Copy _
       Destination:=Sheets("Blocked").Range("A" & Rows.Count).End(xlUp).Offset(1)
      Exit Sub ' all done
     
    Case Is = "NO"
      searchFor = Cells(Target.Row, 1) ' again, value is implied
      Set searchRange = Worksheets("Blocked").Range("A1:" & _
       Worksheets("Blocked").Range("A" & Rows.Count).End(xlUp).Address)
      Set foundCell = _
       searchRange.Find(searchFor, _
        LookIn:=xlValues, _
        lookat:=xlWhole, _
        MatchCase:=False)
      If Not foundCell Is Nothing Then
        'means we found it on the other sheet
        foundCell.EntireRow.Delete
      End If
     
    Case Else
      'do nothing, not "YES" and not "NO"
  End Select
End Sub

When you select Different Odd & Even Pages, Word starts displaying two different footers. One of those footers appears only on the even-numbered pages, and the other appears only on the odd-numbered pages. The two footers are completely independent of each other.

Therefore, to display an image only on the odd pages, and a page number on both odd and even pages, you must insert the page number (that is, a PAGE field) separately into

Yes, you need to convert the time display in Excel to a number display, so the mailmerge can parse the number and display it in the format you require. This isn't really a flaw in Word but a limitation of the data format you're trying to use; it simply isn't a time format recognised by either Excel or the converter that interprets the data. Even in Excel, if you were to look at the number displayed in the formula bar, you'd see a time rounded off to the nearest second.

It might be more work using the kind of field coding I've suggested, but the important thing is that is solves the problem. Hi Kamalpreet,

Turning on Track Changes gives you and your coworkers a way to make changes that are easy to spot. The changes are like suggestions that you can review, and then remove them or make them permanent.

We can use it on Word online, but there are a couple of important things:

  1. You can’t see tracked changes in Word Online but they’re still there. When you open a document in Word Online that has tracked changes, they’re preserved and any changes you make will also be tracked. You just won’t see them until you open the document in the Word desktop application.
  2. You can’t turn Track Changes on or off in Word Online; you can only do that the Word desktop application.
  3. You’ll know Track Changes is on when you see it in the status bar at the bottom of Word Online.

Using Co-authoring to enable multiple users to work on a document, at any time, without interfering with each other's changes. It can use on Office online normally.
