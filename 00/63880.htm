how to set up notepad to email

can somebody tell me how to set this up please ?
steve cochran say:

It is not having an account that is what you need, you need an actual program such as Outlook that comes with Microsoft Office or some other 3rd party email program.  The account is just where your mail is -- it is not the software how you access your email.  Wordpad cannot send email by itself.  It has to have a program that it accesses to send the email and you don't have one on your computer.  If you want to use Wordpad, you can compose the message in Wordpad and then use CTRL-A to select all the text and then use CTRL-C to copy it.  Then you can compose a new message in however you access your mail, and in the new message use CTRL-V to paste the copied text into the message.

 

steve
