    is gfc-0019 windows 7 oem or retail? - Microsoft Community

    beachcat say:

    is gfc-0019 windows 7 oem or retail?

    the label has GFC-00019 as the SKU, is this the retail version or oem version? it is Windows 7 Home Premium, but there are no stickers stating which version it is.
    andre da costa say:
    A GFC code symbolizes the product according to geographical location in most cases and in some case it has to do with the versions.
    The GFC-00019 is a retail version containing both 32 and 64 bit operating systems discs. TheGFC-00564 is a 32 bit OEM version. A retail version can be installed on any computer. But still only one at a time. But if you installed it on one computer (and then it got damaged beyond repair) and then bought a different one you could install it on the new computer.
    An OEM copy can only be installed on the one computer. If that computer got to the point of beyond repair you could not use it again like with the retail version. You can install the OEM as many times as you want on the original computer.
