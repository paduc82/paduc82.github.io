doug robbins - mvp office apps & services (word) say:

If you are using Legacy FormFields, you must protect the document for filling in forms.

You can do that either by using the Restrict Editing button on the Review tab of the ribbon or the button with the same name on the Developer tab.

Then in the Restrict Formatting and Editing pane that appears, check the box under 1. Editing restrictions and from the dropdown, select "Filling in forms" and the click on the "Yes, Start Enforcing Protection" button.

Alternatively, you might want to consider using Content Controls as they do not require that the document be protected along with the restrictions that it imposes of not being able to format text, spell check etc.
