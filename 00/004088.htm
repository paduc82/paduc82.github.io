doug robbins - mvp office apps & services (word) say:

Thanks, Alessio.

For others who may be interested, this is the code that was developed to populate a table identified by the bookmark BudgetTable with the required information:

Dim i As Long, j As Long, k As Long, l As Long
Dim budtable As Table
Dim H2Range As Range, H3Range As Range
With ActiveDocument
    Set budtable = .Bookmarks("BudgetTable").Range.Tables(1)
    With budtable
        If .Rows.count > 4 Then
            For i = .Rows.count - 3 To 2 Step -1
                .Rows(i).Delete
            Next i
        End If
    End With
    j = 2
    l = 1
    For i = 1 To .Paragraphs.count
        If .Paragraphs(i).Style = "Heading 2" Then
            Set H2Range = .Paragraphs(i).Range
            H2Range.End = H2Range.End - 1
Here:
            budtable.Rows.Add BeforeRow:=budtable.Rows(j)
            budtable.Rows(j).Cells(1).Range.Text = Chr(64 + l) & " " & H2Range.Text
            l = l + 1
            j = j + 1
            For k = i + 1 To .Paragraphs.count
                If .Paragraphs(k).Style = "Heading 3" Then
                    budtable.Rows.Add BeforeRow:=budtable.Rows(j)
                    Set H3Range = .Paragraphs(k).Range
                    H3Range.End = H3Range.End - 1
                    budtable.Rows(j).Cells(1).Range.Text = H3Range.Text
                    j = j + 1
                ElseIf .Paragraphs(k).Style = "Heading 2" Then
                    Set H2Range = .Paragraphs(k).Range
                    H2Range.End = H2Range.End - 1
                    Exit For
                    i = k + 1
                    GoTo Here
                End If
            Next k
        End If
    Next i
End With

r. mann say:

Low on storage space

I've got a HTC Touch with WM6 and I keep getting a "storage capacity low" message.  When I run explorer, I see that only a few KB's are free on the "hard drive".  I haven't put any music, photos, etc on the phone.  Only my outlook contacts (which is quite large), email (inbox only), etc.

 

I've gone through the sync settings and tried to eliminate as much as I could to free up space.  I have a storage card installed, but I don't know how to move the outlook files there, and if I do, I don' t think they will sync.

r. mann say:

I found the answer.  Download MemMaid from www.handango.com.  I did that and it found that I had a bunch of MB's eaten up by my IE cache.  The program cleaned up my problem in a matter of seconds.  No more worries about moving stuff to the card.  You can leave anything you want to sync on the main memory and still have plenty remaining.
