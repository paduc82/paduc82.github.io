a. user say:

"Active hours" means the period during which you are or may be actively using the computer. In Settings you can pre-set a schedule of up to 18 hours (perhaps longer in 1709) during which the computer will not restart to install an update. If the computer needs to re-start outside Active Hours and it detects that you are still using the computer then you are prompted and can delay the restart. You need to be present to see the prompt.

This is how Windows 10 is designed to work. There may be Enterprise versions that work differently, but these are generally not available to individuals.

A VM can run on just about any operating system (or no operating system.) If you need your VM to run 24/7 you're looking at a different operating system. Windows 10 can delay restarts for a long time, but eventually your computer needs to restart.
