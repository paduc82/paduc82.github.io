End Function

Hi BA-OK,

I'm sorry to hear of your trouble.  Unfortunately, the Word Web App only supports editing docx file.  Therefore doc files get converted to docx to support editing.  If your daughter made no changes, then you can probably just re-open the doc file you e-mailed her.  That said here are two possible ways for you to collaborate with your daughter:

  1. When you e-mail your daughter a doc file, instead of editing it through the Word Web App, she can download the file (click the word "download" below the file's name).  After downloading the file, she can open it in Microsoft Word on her desktop, make changes, save, and e-mail you back the new doc file.
  2. You could edit the docx file through the Word Web App directly in your browser and simply e-mail back and forth the link.  The Word Web App allows you to print your document by clicking "file" in the ribbon and selecting print.
  3. If you receive a docx file, you can use the Word Viewer to view it or even the Word Web App to view it online.  Better yet, depending on the version of Microsoft Word already on your computer, you could probably use the Microsoft Office Compatibility Pack to open and edit docx files.

steven. s say:

Hello everyone,

 

The N version of Windows 7 should already include Internet Explorer.  If you check the details of Windows 7 Home Premium N it makes note that some media elements aren’t provided, but makes no note of Internet Explorer being removed.

 

Early on news was that the E edition would not include Internet Explorer, but before launch the decision was made to include Internet Explorer in all releases of Windows 7. See: Update on E editions of Windows 7

 

If you don’t have Internet Explorer, paste this address into your address bar: %programfiles%\Internet Explorer. You should see an iexplore.exe, right click it and click Create Shortcut.

 

If you don’t have an Internet Explorer folder in your program files, you may need to enable Internet Explorer:

·         Click Start

·         Type Programs and Features into the search field

·         Click Turn Windows features on or off (on the left column)

·         Check the Internet Explorer 8 box

·         Click OK

 

Hope that helps, look forward to hearing back from you.
