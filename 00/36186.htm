folder permissions for XP home version - Microsoft Community

wild140sd say:

folder permissions for XP home version

how do I make a shared folder not read only ?

 

wunders say:

With Simple File Sharing (enforced on XP Home), there are three levels of sharing between users.  Level 3 is Read-only to other local users that physically log into the machine.  Level 4 is Read-only to local users and network users.  Level 5 is Read/Write to local and network users.  How to set up the level you desire is described in the following Microsoft Knowledge Base Article:

"How to use the Simple File Sharing feature to share files in Windows XP"
 < http://support.microsoft.com/kb/304040 >
