I believe the .MOD files from a JVC camera are encoded in MPEG-2 format. That requires a licensed MPEG-2 codec be present in order to edit. Assuming you are on Windows 8 or later, there is no included MPEG-2 codec so you can't edit MPEG-2 video using Movie Maker.

[SOLVED] Windows 8.1 — All Modern UI Apps with video - Microsoft Community

cw-web-print say:

[SOLVED] Windows 8.1 — All Modern UI Apps with video components (Xbox Video, Netflix, Vevo, Vimeo) crash without warnings or error messages upon video playback.

THE PROBLEM:

All Modern UI apps that feature any video playback components will fail upon video playback initialization.  Netflix reaches 99% and sits there for a few seconds before failing, Xbox Video fails upon any streaming initialization, and the same for Vevo and Vimeo.  The apps work, I can view titles, browse categories, but any video playback whatsoever results in a complete app crash.


THE SOLUTION:

The issue is with AMD Control Center, if "Morphological Filtering" is enabled in AMD Catalyst Control Center as the default game setting, any apps that utilize full screen video (Xbox Video, Netflix, Vimeo, Vevo, etc.) will crash to the Modern UI start screen.  Disable it, and your apps will work without any rebooting necessary.


To disable Morphological Filtering:

  • Windows Key + Q
  • Type "AMD"
  • Launch AMD Catalyst Control Center
  • Choose the tab that reads Gaming
  • Choose 3D Application Settings
  • Under System Settings, make sure that Morphological Filtering is set to "off".
  • Hit apply, close AMD Catalyst Control Center
  • Launch any apps you were having trouble with to see that your issue has been solved.

Thank you for posting your query in Microsoft Community.

I request you to download the drivers from manufacture's website for Windows 10, if there no drivers available for Windows 10, download the latest driver from the manufacturer’s website and try installing it in compatibility mode.

Compatibility mode runs the program using settings from a previous version of Windows.

Follow these steps to install the drivers in compatibility mode and check if that helps.

a. Download the driver from the manufacturer’s website.

b. Right-click on the driver setup file and click on ‘properties’.

c. Click on the ‘compatibility’ tab and check the box ‘Run this program in compatibility mode for’ and select operating system from the drop down menu and proceed with the installation.
