This is a Windows XP forum so I'm guessing you are running

When you right click a selected group of photos and choose...
Send to / Mail Recipient...you should see a dialog that allows
you to resize the photos.

If this is not happening...try the following steps, maybe it
will restore the option:

Go to...Start / Run and type
         (or copy/paste):

"REGSVR32 SHIMGVW.DLL"
(without the quotes)(Yes, there's
a space between 2 and S)

Click...OK.

A dialog box should report the following:

"DllRegisterServer in SHIMGVW.DLL succeeded"

Also there is a free Windows XP Powertoy that can be useful:

Go to the following link and download and
install:   ImageResizer.exe
http://www.microsoft.com/windowsxp/downloads/powertoys/xppowertoys.mspx
(will not run on Vista)

Then you can open any folder that contains
images and you will have an option to resize.
Just right click the selected image files and
choose Resize Pictures from the menu. You
can select one image or a whole group.

The program will place the resized copies in
the folder with your originals.
(Tip: *Never* overwrite your originals)

To insure that images in both portrait and
landscape orientation are resized to the
same dimensions...go to...Advanced...
