Hi,

 

I would use VBA. ALT+F11 to open VB editor, right click 'ThisWorkbook and insert module and paste the code below in and then close VB editor.

 

On the worksheet call with

 

=ConcRange(A1:A100)

 

or if you want a delimiter call with

 

=ConcRange(A1:A100,",")

 

The actual delimuter can be whatever you want

 

 

 

Function ConcRange(Rng As Range, Optional Delim As String = "")
Dim C As Range
For Each C In Rng.Cells
    ConcRange = ConcRange & C.Value & Delim
Next C
If Delim <> "" Then
    ConcRange = Left(ConcRange, Len(ConcRange) - Len(Delim))
End If
End Function
