jay freedman say:
Once you learn the linear "names" of the pieces you work with most often, the Math AutoCorrect works well. For the integral example, you would type

   \int_a^b<space>sin<space>x<space>dx

The spacebar (<space>) causes the autocorrect to take effect. You can also learn the correct format by using the ribbon buttons to construct an equation, then clicking the down arrow at the right end of the equation box and clicking Linear.

Hint: If you don't use set-theory notation often, it's helpful to delete the Math AutoCorrect entry named \in to prevent interference with the \int shortcut. To do that, while the cursor is in an equation, go to the Tools group at the left end of the Equation Tools ribbon; click the little arrow in the bottom right corner of the group to open the Equation Options dialog, and click the Math AutoCorrect button. Click on the \in entry and click the Delete button.

hansv mvp say:

You'd have to write code in the On Mouse Wheel event of the form. The code below uses SendKeys, which is reputed to be flaky (but it generally works OK for me):

 

Private Sub Form_MouseWheel(ByVal Page As Boolean, ByVal Count As Long)
    Dim i As Long
    Dim s As String
    If Me.ActiveControl.Name = "Notes" Then ' replace with name of text box
        If Count > 0 Then
            For i = 1 To Count
                s = s & "{DOWN}"
            Next i
        Else
            For i = 1 To -Count
                s = s & "{UP}"
            Next i
        End If
        SendKeys s
    End If
End Sub

varun shankar say:
You need to change the referencing to absolute referencing so that it would remain consistent throughout.
You may write the formula as
 =AVERAGE($B$2:$B$24)
 
You may refer the link below for more information.
http://office.microsoft.com/en-us/excel-help/switch-between-relative-absolute-and-mixed-references-HP010342940.aspx?CTT=1
Note: The above article also applies for Excel 2007.
