When an Excel file is linked to a Word documents, the path is hard-coded in the link. Then when you move the files to a new computer, all the paths are different and the links don't work. 

You can try right-clicking on the Excel item in Word. Choose Links..., then click on the Change Source button and re-link the file.

You may also be able to edit the links directly:

  1. In Word, close the linked file.
  2. Choose Word>Preferences>View.
  3. Check Field codes instead of values in the Show in document section.
  4. Reopen the Word file. In place of the Excel segments, you should see something like:
    { LINK Excel.Sheet.8 "Workbook1!Sheet1!R1C1:R2C4" "" \a \p }
  5. Edit the first quoted section to the actual path where the Excel file is stored.

Unfortunately, it's somewhat hit and miss trying to get field codes for linked objects to appear, so you may be stuck with the first method, right-clicking and choosing Links...

How do I not include certain rows? - Microsoft Community

phantombanker say:

How do I not include certain rows?

I'm downloading a weekly report with mortgage info. I'm looking for a formula that will count the number of rows that contain this data, as well as the sum of dollar amounts (column C). The quick and easy answer would be =COUNT('Sheet'!C:C) and then =SUM('Sheet'!C:C).

The problem I have is that the last row contains info I do not want included. How do I eliminate this row from the formula, given that there is an unknown number of rows each time I download the report?

chip pearson say:

You can use SUMPRODUCT to do this. Change the references to C1:C10 to C1:Cnnnn where nnnn is a row that is certain to be below all of your data.

 

=SUM(INDIRECT("C1:C"&MAX(ROW(C1:C10)*(C1:C10<>""))-1))

 

This is an array formula so you must enter it with CTRL SHIFT ENTER rather than just ENTER. Use CTRL SHIFT ENTER when you first enter the formula and whenever you edit it later. If you don't do this, you'll get a #REF error. If you properly enter the formula using CTRL SHIFT ENTER, Excel will display the formula in the formula bar enclosed in curly braces { }. You do not type in the braces; Excel puts them in automatically. See http://www.cpearson.com/Excel/ArrayFormulas.aspx for much more details and examples of array formulas.
