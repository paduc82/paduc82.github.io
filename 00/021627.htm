bob jones aka: cybertaz say:

AFAIK, John hit the nail on the head :-) Whether Paste & Match Formatting is available is dependent on what's on the Clipboard. That, in turn, is determined by the program being used when the copying is done as well as the nature of the content.

Some common examples:

  • Content copied from a spreadsheet does not enable the command
  • Selected text copied from PowerPoint will enable the command but a copied text box/placeholder will not
  • Plain Text copied from a text editor does not enable the command, but if the text in the same text editor document is formatted as Rich Text the command will be available in Word.
  • Content copied from a web page may/may not enable the command. It depends on the actual structure of the page & the operation of the browser being used

The direct answer to your question is that neither you nor Word can force the command to be active. Further, if the command is not available it makes no difference which keystroke is assigned to it... The keystroke will not work unless the command is active. In those cases you can try using Edit> Paste Special - Unformatted Text. However, there is no built-in means of assigning a keystroke to that specific choice.

john korchok say:

Here's the macro I use for the Paste as Unformatted Text command. If the clipboard doesn't hold text, it will do a regular Paste.

Sub PasteUnformattedText()
    On Error GoTo ClipboardNotText
        Selection.PasteSpecial DataType:=wdPasteText
        Exit Sub
ClipboardNotText:
    Selection.Paste
    Err.Clear
End Sub

charles kenyon say:

suzanne s. barnhill say:

You can add the FILENAME or FILENAME \p field to the footer of the template on which documents are based, but it is not a good idea to add this (or any other content) to the header or footer of the Normal template, as that has adverse affects on labels.

To insert this field in the footer of another template:

  1. Open the template for editing.
  2. Double-click in the header area to activate the header and footer.
  3. Scroll down to the footer and place the insertion point where you want the filename to appear.
  4. At this point there are several ways to insert the field.
  5. The simplest is just to type FILENAME, select it, and press Ctrl+F9 to make a field out of it, then F9 to toggle the display and update the field.
  6. Alternatively, you can press Ctrl+F9 first and then type FILENAME between the braces that appear. In either case, you can add the \p switch if you want the full path to appear (a sensible choice in this instance).
  7. If you are not confident of your ability to do it in one of these simple way, you can do it the hard way using Insert | Text | Quick Parts | Field. Scroll down to and select FileName, check the box for "Add path to filename," and click OK.
