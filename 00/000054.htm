Look in the same folder as the file is located in, for a file that has had the first 2 char of the problem file name replaced by ~$......DOC

Word uses this file to store the name of the person who currently has the file open. If you do not close Word properly, ie with the file open in word you turn off the power or Word abends, this file gets left behind. The next time you try to open it you get the message you are complaining about.

Delete the temp file, then you should be able to open the main file.
