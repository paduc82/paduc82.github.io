Hi

Any text that appears in a placeholder will be in the outline. You can add placeholders to a layout using the Insert Placeholder dropdown while in Master View (see here: http://echosvoice.com/2007/placeholders.htm).

The order they appear in the outline view is the order in which they were added to the layout, so if you want them in a different order, move them around on the layout in Master view. To see what I mean try creating a layout with 6 content placeholders, then in edit view add numbers 1 to 6 in the placeholders, then look at their order in outline view. You can then go back to Layout in Master view and move the placeholders around and see how that effects the look of the slide but not the outline order. bozbar say:

It's possible that you get two users running a transaction at the same time.

 

Use a single transaction for the for the SELECT and UPDATE statements in the SQL. COMMIT and then get the result.

 

You would use BEGIN TRANSACTION before the DECLARE statement in your SQL and add the COMMIT after the UPDATE statement.
