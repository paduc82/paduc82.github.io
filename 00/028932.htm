Hi,

Is there a way in which to conditional format a cell as a percentage, or currency based upon the input in a different cell?

Yes there is. This conditional format formula detects a cell format of currency
=CELL("format",A1)="C2"

and this percentage
=CELL("format",A1)="P2"

So if your cell is picking up data from any cell if, you apply those two CF's to a cell and apply the conditional format of currency to the first and % to the second the cell will format to whatever the format of a1 is.

To set to Bold:
Sub question_181_macro()
'
' question_181_macro Macro
'
'
    Selection.Font.Size = 10
    Selection.Font.Color = wdColorAutomatic
    Selection.Range.HighlightColorIndex = wdYellow

Sub question_181_macro()

' question_181_macro Macro

'
    Selection.Font.Size = 10

Sometimes when people are filling out my forms, they'll put a period out of habit at the end of a sentence. (Even though I tell them not to.) Then, when they perform the mailmerge, the contents merges into its field in the document, resulting in 2 periods. This is because my document has periods where periods are needed. Like: { MERGEFIELD "soandso" }.   If the contents of "soandso" from the form contains a period, then I'll get a sentence like: "The neighbor walks the dog in the morning.."   (Ending with 2 periods.)

Is it possible to fix this using mail merge code? Is there any way of testing a mergefield to see if the incoming content ends with a period?

Thanks,

Cason

macropod say:

While it's easy enough to test whether a merge string begins or ends with a particular character (e.g. a period), there is no practical means of testing whether it contains a particular character at a random location within the string. It's also impractical to test for all possible punctuation possibilities (.,!?:;) - at least 6 separate tests would be required.

To test whether a merge string ends with a period and conditionally add one if it's missing, you could use a field coded as:

{MERGEFIELD MyField}{IF{MERGEFIELD MyField}<> "*."  "."}

or:

«MyField»{IF«MyField»<> "*." "."}

where 'MyField' is your field name.

Note: The field brace pairs (i.e. '{ }') for the above examples are all created in the document itself, via Ctrl-F9 (Cmd-F9 on a Mac); you can't simply type them or copy & paste them from this message. Nor is it practical to add them via any of the standard Word dialogues. Likewise, you can't type or copy & paste the chevrons (i.e. '« »') - they're part of the actual mergefields, which you can insert from the 'Insert Merge Field' dropdown. The spaces represented in the field construction are all required.
