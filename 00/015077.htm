how to open .htm file using vba and replace a string with another? - Microsoft Community

davidfleischhauer say:

how to open .htm file using vba and replace a string with another?

Hey, I am writing a macro in Word 2011 for mac.  In the macro I am changing the name of some picture files which are referenced in .htm files.  What I am having trouble with is finding a way to open the .htm and replace all references of the old file name with the new one.  Any Help would be great.  Thanks!
j.e. mcgimpsey say: One way:

        Const sPath As String = "your path here"
        Dim fn As Long
        Dim sImport As String
        Dim sExport As String
        Dim sFname As String
        Dim sOldPicFName As String
        Dim sNewPicFName As String
        
        sFname = "your .htm file name"
        sOldPicFName = "your old picture filename"
        sNewPicFName = "your new picture filename"
        
        'Read the file in from disk
        fn = FreeFile
        Open sPath & sFname For Input As #fn
        sImport = Input(LOF(fn), #fn)
        Close #fn
        
        'change file names
        sExport = Replace(sImport, sOldPicFName, sNewPicFName)
        
        'Write the file back to disk
        fn = FreeFile
        Open sPath & sFname For Output As #fn
        Print #fn, sExport
        Close fn
