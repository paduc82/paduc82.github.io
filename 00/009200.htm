How to unblock someone whom I previously blocked?

How do I unblock someone whom I previously blocked?
julie_s1 say:

Hello

Here's how to unblock a contact:

In Skype for Windows Desktop

  1. In the main menu, select Tools > Options.
  2. In the Skype Options box, select Privacy > Blocked Contacts. A list of blocked contacts is displayed.
  3. Select the contact you'd like to unblock and click Unblock this person.

In Skype for Windows 10:

  1. Click your Profile.
  2. Go to Settings, scroll down and look for Contacts.
  3. Select Managed blocked contacts.

In Skype for Mac:

  1. In the main menu, go to Skype > Preferences....
  2. Click Privacy. In the Privacy panel, click Manage Blocked Users. A list of blocked users is displayed.
  3. Click the person you’d like to unblock, and then click Unblock.

In the new version of Skype for iOS and Android:

  1. In the Chats screen, tap your profile picture.
  2. Tap the settings button to view your profile.
  3. Tap Privacy, then tap Manage blocked users. You will see a full list of all the Skype contacts that you have blocked.
  4. Tap the Unblock button next to the contact you wish to unblock. 

If you have further questions, please do not hesitate to reply to this post. Thank you!

Something doesn't sound right :-) There seems to be some confusion of the 2 methods I mentioned. The contextual menu that appears when right-clicking an object should not contain the New Comment or Borders & Shading commands. Those commands don't pertain to pictures, shapes, etc. I only recall seeing them when right-clicking in a text area & there typically are more than just the 7 commands you mentioned. Let's start at the beginning...

To create Alt Text you first need to bring up the Format Picture Task Pane which displays on the right side of the document window. The pane can be produced in either of 4 ways;

  1. Double-click the object, or
  2. Right-click the object & select the Format [Picture or Shape or SmartArt, etc.] command in the contextual menu, or
  3. Select the object then go to Format in the main menu to select Picture or Shape or SmartArt, etc., or
  4. Select the object then use the Shift+Command+1 keyboard shortcut

Using either of those 4 methods will cause the Task Pane to appear. You can then click the Layout & Properties button as illustrated in this screenshot:

If you still cannot produce the Task Pane my bet is that the document is saved in the old .doc file format which does not support Alt Text. Does [Compatibility Mode] appear following the file name in the Title Bar? If so, use Save As to create a new copy of the document in the .docx file type. You then should be able to assign Alt Text.

joe_818 say:

I don't believe the answer given is correct.

The Top X function will collect and display the Top X records for the values in the column specified in the Top X dialog.

However, it does not sort them in order of the chosen value column which may add some confusion.

To correctly display a sorted, Top X list, do one of the following (clear any filters to start):

Option1.

Sort the relevant value column; this will give a correctly sorted data set. Now apply the Top X filter to the sorted column.

This is where things can go awry because its possible to apply the Top X filter to a different column from that used in the previous sort. This will result in an Top X list taken from the "wrong" column sorted by the "right" column.

Option 2.

Apply the Top X filter by specifying the target column in the Top X dialog. Now apply a sort to the column.

Option 2 is probably the safest option in a multi-column table since it forces the correct value column to be selected in the first step.


The confusion is compounded by the fact that there is no visibility of the exact nature of the current select/order by on the limited dataset.
