Pivot Table - Default to Classic Pivot Table Layout

I would like to know if it is possible to default EXCEL 2010 to display pivot tables in the classic layout.
roger govier [mvp] say:

Hi Martine

There is no inbuilt setting for this.
The following code will allow you to toggle between the Compact and the Outline (Classic) view.
If you add a shortcut to it say Ctrl+Shift+O and place the macro in your Personal.xlsm file, then you will alwys have it available, and give you a quick switch between views.

Sub SwitchOutlineCompact()
   Dim pt As PivotTable
   Set pt = ActiveSheet.PivotTables(1)
    If pt.LayoutRowDefault = xlCompactRow Then
       pt.LayoutRowDefault = xlOutlineRow
       pt.RowAxisLayout xlOutlineRow
   Else
       pt.LayoutRowDefault = xlCompactRow
       pt.RowAxisLayout xlCompactRow
   End If

End Sub
