rich007a say:

OK, I see what you mean now.

The answer is to not use merged cells.  They are a well known cause of issues for many things in Excel.

Alternatives are:
a) putting the values ABCD in the first row only (so in your file, cells C5, C7, C9 and C11 would contain A, B, C & D, but cells C6, C8, C10 and C12 would be blank),
or
b) putting the values ABCD in every row

If you do (a) above, you can use cell borders to make the cells appear merged.  You can do that automatically with Conditional Formatting:

Select cells C5:C12.  Unmerge them, then do Home tab > Conditional Formatting > New Rule > Format only cells that contain > Format only cells with: Blanks > Format > Borders tab... then apply only Left and Right borders using the buttons at the bottom of the large white box.  OK, OK.

Repeat the CF: Home tab > Conditional Formatting > New Rule > Format only cells that contain > Format only cells with: No Blanks > Format > Borders tab... then apply only Left and Right AND TOP borders.  OK, OK.

Apply borders to all sides of the other cells using All Borders in the Home tab > Font group > Borders.

Now do View tab > Show/Hide group > Gridlines (uncheck).

Hope that helps.
