a. user say:

Hi Robert,

 

The above issue might occur due to non-updated battery drivers installed on the computer.

 

Update the battery drivers manually from Device Manager by following the steps below and check if the issue is resolved.

 

  1. Press Windows logo + X keys on the keyboard and select Device Manager from the context menu.
  2. Expand Batteries option from the window, right click on the driver and click on Update this software from the context menu.
  3. Follow the on-screen instructions and wait for the update to complete.
  4. After the update is complete, close the window and check if the changes are effective.

 

Write to us with the status of limited power options available after installing Windows 10 update on the computer in the same post for further assistance. Your reply is most important for us to ensure we assist you accordingly.

 

Thank you.

Trash? Do you mean the Windows recycle bin, or a Deleted items folder either under an account or under Storage folders?

If you can't find the missing message files (with names like A0B1C2D3-00FF9876.eml) in the store folder, your best bet would be Previous versions:

 

  1. In Explorer, right-click on the store folder and select Restore previous versions. 
  2. If you can locate a previous version of your store folder that predates the folders' going missing, Open it and then locate the folders that are missing.
  3. Copy the folders into a temporary folder on your desktop.
  4. Use the File > Import messages function (Alt-F I) in Windows Live Mail to import the temporary folder.
