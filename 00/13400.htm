Unfortunately, you can't customize the indentation for the built-in list formats (applies equally to bullets and numbering).

Also, you can't fully intercept the Bullet (or Numbering) commands in Word 2013 with a macro. For example, you can intercept clicking the button, but not clicking the arrow on the button.

You can add a custom button (that applies a bulleted style) to the Quick Access Toolbar and/or attach a bulleted style it to a keyboard shortcut. In particular, note that List Bullet already has a built-in shortcut: Ctrl+Shift+L (the shortcut may not be the same on all language versions of Word, though).

No. Microsoft has made it impossible to do that.  Currently, it is only possible to merge (through manual transfer of email address, contacts, calendar, and onedrive from one account to another) a single hotmail.com/live.com address with outlook.com addresses.

If you are curious as to why, it is because Microsoft removed the "linking" function of the mail account service as phase 1, and later removed the "@hotmail.com" and "@live.com" from selection on alias creation page as phase 2.  Result was that it would be possible to merge "hotmail.com/live.com" address with "outlook.com" by deleting the "outlook.com" address, wait 30/60 days for the address to become available again, and add the "outlook.com" address as an alias to the "hotmail.com/live.com" address account.  (This is because only "outlook.com" can be selected as a domain when creating alias).

What this means to the users is that any "hotmail.com/live.com" account cannot be combined with another "hotmail.com/live.com" account.

FYI: Microsoft also dropped "live.com" address from being created.  Once you delete "live.com" address, there is no way of getting it back through regular methods.
