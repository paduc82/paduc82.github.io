The strike-through suggests you have an IMAP email account. IMAP email accounts (at least before Outlook 2010, and then it's only pretend) don't HAVE a "deleted Items" folder. Reason? an IMAP account mirrors the folders on the IMAP mail server, therefore a "Deleted Items" folder is superfluous.

suzanne s. barnhill say:

You can specify the default document folder for Word at File | Options | Save: Default local file location. When you first start Word, this will be the folder Word looks to when you use the Open or Save As command.

But if you open a document from a different folder (or subfolder) or save a document in a different folder, that folder becomes the new default for opening and saving files, with the exception that, if you use Save As to save an existing (already saved) file under a new name, Word will default to the folder where the existing file is saved.

macropod say:
Word's citation/bibliography tools have limited functionality and poor compliance with the relevant standards. What I do for a citation like Bailey, Hershbein, and Miller (2012)  is to copy the author names from the citation & paste them into the preceding text before editing the citation to show only the year. As for inserting multiple citations in the same parentheses, that is quite easily done by inserting the first citation, then clicking on it before inserting the second, and so on. That would give a combined citation like (Goldin & Katz, 2002; Bailey, Hershbein, and Miller, 2012)
