ch1234 say:

Thanks.  I did have that link, but wasn't sure if it would apply for Office 2010.  I today have another solution that is less drastic, in case anyone else experiences this same issue.  It is possible to remove after-the-fact the installation customizations that were set by using a network install that was created using the Office Installation Customization tool.  On the affected PC, close all Microsoft Office applications, then go into the registry and open HKEY_CURRENT_USER -> Software -> Microsoft -> Office.  Rename 14.0 to something else (like 14.0old), then open Microsoft Outlook.  Create a new meeting, and the Meeting Workspace button should no longer be greyed out.

Note: the above process will remove all customizations that were set by the original installation, so if your organization uses a particular installation configuration, you will need to re-configure settings in the Office programs on the afftected PC with the settings you had originally included in the network install.

This menu command is only used for saving attachments from messages. For everything else it will be unavailable.

If you wish to save a message to your desktop then you'll need to either copy/paste the contents into a text editor such as TextEdit or drag the message to your desktop to create a .eml file that Outlook can open later.
