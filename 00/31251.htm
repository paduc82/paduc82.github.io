¡firedog say:

     â€¢     I am using IMAP and an MSN.com email address

First question - why? The default configuration for msn.com accounts in Windows Live Mail is to use DeltaSync over HTTP. This gives the same results as IMAP, but much more efficiently.

Second, have you mapped the IMAP folders to Windows Live Mail's native folders? I suspect you haven't, which could be why you're having trouble deleting messages.

In Windows Live Mail, right-click on the account name in the folder pane and select Properties. On the IMAP tab, select Store special folders on IMAP server, then enter the server's names for the special folders in the boxes. Enter Sent into the box alongside Sent Items path, Junk alongside Junk email and Deleted alongside Deleted items.

Close the properties dialogue and press Ctrl-Shift-O for Options. On the Advanced tab, select Use the 'Deleted Items' folder for IMAP accounts.

Now log in to the account at mail.live.com. In the left-hand pane, expand the folders so you can see them all. If there are 'duplicates', move any messages that you want to keep from the folders with two-word names (Sent items, Junk email, Deleted items) into the equivalent one-word folder (Sent, Junk, Deleted), then right-click on the two-word folders and Delete them.

Back in Windows Live Mail, click on the account name in the folder pane and then press Alt-S F to open the Show/hide IMAP folders dialogue. Click Reset List to refresh the folder list, then use the buttons to show only the folders you want to see.

Right-click on the account name in the folder pane and select Download all folders. Once everything is in sync again, try deleting a message from the inbox.
