Hello,

after searching a lot for the solution...about the missing space...having about 100 Gb hidden somewhere...on the partition...  I found the true solution...I am using Windows 8.No fancy software used to solve this.

Disabling,deleting or resizing the System Protection value, helped a little bit...a drop in the ocean

Using WinDirStat helped to see which dir/folder how much space uses

BUT really... where's the difference of 100 Gb??? where's my space?

It was used by Volume Shadow Copy Service ........

If you want to know more about Volume Shadow Copy Service just Google about it.

Here's what I did to get back my 100 Gb...

I opened an elevated Command Prompt (type   Command Prompt       in the Start Menu's search box, right click on it and select Run as Administrator)...

ok...the black window (not widow :) ) opens

type this there:        VSSADMIN LIST SHADOWSTORAGE   and press Enter

you will see where's your stolen space ;) I had ONLY 100 GB !!!!!! Only ! (partition size was 232 Gb and I had about 42 Gb free...someone was constantly stealing space...but who? :D )

In my case was the C: partition, so I just typed these:

vssadmin resize shadowstorage /for=C: /on=C: /maxsize=10GB

forcing the system to resize the shadowstorage allocation to maximum of 10 Gb. Chose another value if you want.

Ok... but my space didn't came back yet... The last command tot get it back was the following (but be aware that you will delete all the shadowstorage.I didn't cared about it.Read about this on Google before type the command and blame me after! Read about Volume Shadow Copy Service if it helps you or not).

vssadmin delete shadows /all

and Shazammm.... my 100 Gb came back...
