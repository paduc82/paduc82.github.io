Does it really say 'delete some files' or does it say 'close some applications'? If it's the former then you may have filled your phone's storage memory (often referred to as C: drive) and need to free up some space from this drive which is different from your memory card.

If it's the latter then you've been trying to run an application or more than one and your phone has run out of RAM. This is the memory the phone uses for multitasking, running the OS and applications etc. If this is the case then switch the phone off and re-start it.

Your N73 has four types of memory.

Phone storage memory ( C: )
Memory card storage memory ( E: )
ROM - Read only memory, this is where the OS is stored and can only be changed with a firmware update ( Z: )
RAM - as mentioned above, dynamic memory used to run the OS and applications.
