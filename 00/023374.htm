kevin at cbb say:

Determine column number in VBA

Can anyone provide a VBA snippet for determining which column of a table your cursor is presently in?

graham mayor say:
While Greg's page linked by Suzanne will get you there, it is overkill for what you appear to have in mind.

The following will tell you which cell (column is the value of 'i') the cursor is in

Dim oTable As Table
Dim oRow As Row
Dim i As Long
Dim oRng As Range
Dim oCell As Range
Set oTable = Selection.Tables(1)
For Each oRow In oTable.Rows
    Set oRng = oRow.Range
    If Selection.InRange(oRng) Then
        For i = 1 To oTable.Columns.Count
            Set oCell = oTable.Cell(oRow.Index, i).Range
            If Selection.InRange(oCell) Then
                MsgBox "Cursor is in column " & i & " Row " & oRow.Index
            End If
        Next i
        Exit For
    End If
Next oRow

As for the second part

Dim oRng As Range
Set oRng = Selection.Cells(1).Range
If Selection.Start = oRng.Start Then
    MsgBox "Cursor is at the start of the cell"
Else
    MsgBox "Cursor is not at the start of the cell"
End If

will tell you if the cursor is at the start of the cell, but if you need the cursor at the start of the selected cell  it would be simpler just to set the range to the start of the cell the cursor is in

Dim oRng As Range
Set oRng = Selection.Cells(1).Range
oRng.End = oRng.Start
oRng.Select  ' or simply address the range with your macro
