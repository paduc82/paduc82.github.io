A little confused about what the actual question/issue is given on the one hand you state that you recently changed an account to IMAP but state that you have 4 IMAP and one EX account.........

In short, each IMAP and EX account will have it's own (ost) data file. The EX account will have a complete set of folders including contact/calender folders. Whether or not any of the IMAP folder trees contain contact/calendar folders depends on whether the associated ost file for the account has been set as the default data file for the profile (not recommended).

Going back to changing an account from POP to IMAP. <IF> you removed the POP email account from the profile, it would NOT have removed the PST file that appears in the Outlook Nav pane so ignoring all the other accounts for a moment, in this scenario you would have ended up with 2 folder trees in the Outlook Nav Pane if you didn't remove the PST file.

To narrow things down

#1 - Just to confirm again, you only have 5 email accounts configured at this point correct? (4 IMAP and 1 EX)
#2 - How many data files are currently configured in the profile?
Go to Outlook menu --> File --> Account Settings --> Data Files tab
#3 - Which of those data files is configured as the "default data file"? (this file will contain the default contact/calendar folders for the profile)

Lastly, with IMAP and EX accounts, you cannot merge everything into a single data file (Nav Pane folder tree).

A pop mail account uses a set of Personal folders and unless you modified the settings each pop account would have its own folder set including contacts/calendar.

The exchange mail account will have its own ost data folder set, usually named as per the mail address

If you added an IMAP account, that had previously been pop, then that too would have its own folder set again normally named as per the mail address, but its Calendar/Contacts/Tasks would still be contained in a pst data file

Within Outlook, File>Account Settings>Email Tab......... you can set the default mail a/c in this window

The IMAP folder set, depending on the provider, would have at least Inbox/Sent/Drafts/Trash
