fahad al-riyami say:

Yikes, nevermind. Found the solution. 

Just exclude the game folder in Windows Defender and it'll run.'

Steps:

  1. Open Windows Defender
  2. Click Virus & threat protection
  3. Click Virus & threat protection settings
  4. Under Exclusions, click Add or remove exclusions
  5. Click Add an exclusion, then click Folder
  6. Navigate to the main game folder, then click Select Folder

That's it. Game should now run.
