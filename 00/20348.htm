Function GetValue(rngMatch As Range, rngLookup As Range, rngReturn As Range) As Variant
    Dim cel As Range
    Dim c As Long
    Dim bolMatch As Boolean
   
    For Each cel In rngLookup.Columns(1).Cells
        If cel.Value = "" Then Exit For    'No more data in column to test
        bolMatch = False
        For c = 0 To rngMatch.Columns.Count - 1
            If cel.Offset(0, c).Value = rngMatch.Cells(1, c + 1).Value Then
                bolMatch = True
            Else
                'Cease testing the range once one fails match
                bolMatch = False
                Exit For
            End If
        Next c
        If bolMatch Then Exit For   'Match found for all cells in rngMatch
    Next cel
   
    If bolMatch = True And rngReturn.Cells(cel.Row, 1).Value <> "" Then
        GetValue = rngReturn.Cells(cel.Row, 1).Value
    Else
        GetValue = ""       'Return zero length string if no match or no data to return
    End If
End Function

Select worksheet "Full Payment Schedule"

In cell F2 enter the following formula (You can select the cell and then copy the formula from here and paste into the Formula editing bar)

=GetValue('Full Payment Schedule'!$B2:$D2,Batches!$A:$C,Batches!D:D)

You can copy the formula across into cells G2:I2 and then copy down the worksheet.

Explanation of the formula.

  • There are 3 ranges in the formula separated with commas.
  • First range  'Full Payment Schedule'!$B2:$D2   is the 3 cells that need to be found and matched in "Batch" worksheet. Note that the column Ids are absolute (with $ signs) but the rows are relative.
  • Second range Batches!$A:$C  is the 3 columns where the code searches for the matches. Note that it is fully absolute addressing (with the $ signs).
  • Third range   Batches!D:D  is the column from which to return the data if all matches are found. Note they are relative (No $ signs) so when the formula is copied to the columns to the right then the column to return from Batch also moves across.
  • If no match found or no data to return then the formula returns blank.

applied captions do not appear in TOC - Microsoft Community

jez johnson say:

applied captions do not appear in TOC

I am working in Word 2011 on Mac Lion.

I apply captions to tables  but they do not all appear in the TOC. When refreshing the TOC, some captions drop out.  If I move them to a random location, they will reappear in the TOC, but when they are placed above the table they again disappear. Any suggestions please?
john mcghie say:
I am working in Word 2011 on Mac Lion.

I apply captions to tables  but they do not all appear in the TOC. When refreshing the TOC, some captions drop out.  If I move them to a random location, they will reappear in the TOC, but when they are placed above the table they again disappear. Any suggestions please?

I would say you have some floating tables in there.  The captions for those have been created in floating text boxes.

Due to a bug in Word design, any field in a text box will not appear in the TOC generator.

The trick is to convert the Text Box into a "Frame".  Unfortunately, if you have applied Office for Mac SP 2 you will not be able to do that, because they have lost and broken the command you need.

You will need to find another version of Word (e.g. Word before 2011, or Word on the PC) and select the text box.  Use Format>TextBox>Convert to Frame.  The difference visually should be slight, but the Frame is a very old component that can place text as if it were floating while remaining in the Text Layer where the TOC generator can see it.

Failing access to a different version of Word, you will have to drag that caption out of the text box and place it on a paragraph of its how.  You can then use paragraph properties to place it and centre it above the table.

Just to be clear: The bug is that anything in a TextBox is invisible to the TOC generator.  The cure is "put the caption in anything except a TextBox.  One technique that will sometimes work is to create a new row as part of the table.  Merge the cells in that row and turn the borders off so it is invisible.  A caption placed in there will work :-)

Hope this helps
