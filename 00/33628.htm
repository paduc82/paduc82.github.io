andre da costa say:

If you bought Windows 8:

Go to the order summary page, there you will find your product key:

https://www.mswos.com/

 

If Windows 8 came preinstalled on a new computer you own:

With Windows 8, Microsoft has changed from stickers that have the product key that the user has to type in when installing the operating system to new BIOS embedded product keys. The idea is that by eliminating the sticker, you eliminate one of the easier ways for nefarious users to get a legitimate product key. Eliminating the product key sticker also removes any worry that the sticker might get damaged while at the same time eliminating the long and irritating process of typing in various letters and numbers when installing the operating system.

If the user has to reinstall the operating system on a machine that came with Windows 8, the installation process automatically grabs the software product key from the motherboard BIOS with no input from the user. This means that those familiar Windows product key stickers will no longer appear on the Windows 8 computers.

¡firedog say:

I reindexed WLM again hoping it was fix the problem ...

How did you do this? Indexing the Windows Live Mail message store won't affect your ability to conduct body searches using Search for a message; as I said, this uses the Windows Search index. 

Check that your store folder is being indexed and that mail message files are being indexed with properties and content. In Windows Live Mail, locate your store folder by opening the Options dialogue (Ctrl-Shift-O), opening the Advanced tab, clicking Maintenance and then Store folder. Write down the path.

 

Open Indexing Options in Control Panel (type Windows key  + R, type control to open Control Panel, then find and open the applet). If Windows has finished indexing your files, you will see Indexing complete below the item count. If you don't see this, this will explain why your searches are failing.

Click the Modify button, navigate to your store folder and ensure its box is ticked.

 

Click the Advanced button and open the File types tab. Check that the box beside extensions contents is selected at the foot of the dialogue for each one. 

 

If these settings are all as they should be and you didn't make any changes, try the troubleshooter at the bottom of the Indexing Options dialogue.

 

If after all this your searches still don't work, post back.
