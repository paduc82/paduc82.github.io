virtual memory is almost full

what should I remove to make more memory available?

jaimelobo say:

Virtual memory is hard drive space set aside for the PC to use in conjuction with the RAM while operating. Everytime you shut down the computer that memory is emptied. Generally, the virtural memory only becomes a problem if you are running a lot of applications at once or loading very large files (like a photo editing or CAD program). If the virtual memory is set too low, you may experience performance problems as you try to run more programs at once or open lots of files. The virtual memory is generally set at around twice what you have for RAM. There is a way to check and adjust that, if that is the problem.

OTOH, if you mean the hard drive is almost full, then you would need to look into archiving old files, removing or removing application you don't need, or getting rid of temporary and junk files. There also is a tool to clean up the drive and free some space, any of the following methods will start the tool:

  • Click Start, point to All Programs, point toAccessories, point to System Tools, and then clickDisk Cleanup.
  • In Windows Explorer or My Computer, right-click the disk in which you want to free up space, clickProperties, click the General tab, and then clickDisk Cleanup.
  • Click Start, and then click Run. In the Open box, type cleanmgr, and then click OK.

This will attempt to find and remove junk/temp files, empty the recycle bin, and offer to compress unsued files.

James Wolf MS-MVP [Media Center]
