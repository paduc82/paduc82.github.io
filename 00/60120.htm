Can I scan into Word from epson cs5900? - Microsoft Community

barheep say:

Can I scan into Word from epson cs5900?

since using windows 7 the scanning option on my printer has a different format ( windows)  I cannot scan  a document into word so that it can be edited. I could with XP  with  the same printer Epson cx5900.  What am I doing wrong??
ken blake say:
On Sun, 29 Aug 2010 06:58:02 +0000, barheep wrote:
 
> since using windows 7 the scanning option on
my printer has a different format ( windows)  I cannot scan  a
document into word so that it can be edited. I could with XP  with the
same printer Epson cx5900.  What am I doing wrong??
 
 
 
 
Your message is very confusing, but assuming that I understand you
correctly, here's the story.
 
First, you need a piece of hardware called a scanner, not a printer.
You perhaps have a combination unit that does both, but calling it a
printer rather than a scanner is very confusing.
 
Second, this has nothing to do with your printer (or even your
scanner) having a different format.
 
Third, there is no format called "Windows."
 
Fourth, you really couldn't scan a document into Word so that it can
be edited, not even with XP (read below for an explanation of why).
 
When you scan something, the result is a picture. If what you scanned
was text, what you get is a picture of text, not the text itself, and
Word can't edit a picture as if it were text.
 
So to edit the text in a scanned document (whether with Word, any
other word processor or text editor) you have to change that picture
to text. That has to be done with software, a piece of software called
an OCR (Optical Character Recognition) program. An OCR program looks
at the picture of text, sees and recognizes the text within it and
creates as output a text file to match it. You can then take that text
file and edit it in whatever word processor you use.
 
So clearly, in XP you had an OCR program (probably a lite version of
some such program) that came with your scanner. You don't have such a
program  available to you now in Windows 7.
 
You can certainly buy such a program (The two big names are OmniPage
and Abbeyy Fine Reader), but you should ask Epson whether they can
make available to you a free copy of the same one you had before, but
one that will run under Windows 7.
