Even simpler:

Sub Test()
Dim StrIn, StrOut As String, i As Integer
'Get the selected text, split by blanks into words
StrIn = Split(Selection)
'Construct the output string
For i = 0 To UBound(StrIn)
  StrOut = StrIn(i) & " " & StrOut
Next
'Replace the selected text
Selection = Trim(StrOut)
End Sub

Sub Test()
  Dim Text, Temp
  Dim i As Integer

hansv mvp say:

Select a cell containing the array formula.

Press F2 to edit the formula.

Select the entire formula, then press Ctrl+C to copy it.

Select the cells where you want to paste the formula.

Press F2.

Press Ctrl+V to paste the formula.

Edit the references as needed.

Confirm the formula with Ctrl+Shift+Enter to save it as an array formula.
