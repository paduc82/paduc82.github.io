ermahenderson say:

printer will not print documents from wordpad

my printer will print documents from my computer, but will not print documents from notepad or wordpad after i have typed the documnets.
lemp say:
I've seen a number of reports like this.  I assume that you CAN print from other programs, such as your Internet browser.

What is the make & model of the printer?

Does "will not print" mean that you are getting blank pages or no pages when you print from WordPad or Notepad?

If you can't print from ANY program, that's one thing, but if the only programs that aren't printing are WordPad and Notepad (and they print blank pages), the usual cause is that your printer (always a color inkjet of some type) will not print documents that are all black & white, but as long as there is some color in the page, the page prints.

The reason for that is something wrong with the black ink cartridge.  Have you recently put in a new black cartridge, and if so, is it not the ink recommended by the printer mfr?

Or has the printer been sitting unused for a long time?

Print a test page using the printer's front panel controls.  What happens?

Depending on the make and model of the printer, you might need to (a) run the printer's print head cleaning routine, (b) print a test page, perhaps several times, (c) manually clean the printer's printing head or related parts, (d) insert a new black ink cartridge of the recommended type.

I think I have solved the problem of  the disappearing system restore points (SRPs).

Before the time of Windows 8, the settings for the system restore points(SRP) were done by Windows automatically (5%), except when there was  an expert knowledge of the user.

Now with Windows 8, users  have to do the settings of the SRPs themselves. This is done when on Windows 8, you click on 'Configure'.

Before the time of Windows 8, Windows (as e. Windows Vista),set an SRP to 5% of the hard disk space.

A system restore point(SRP) does not save all the system files at the making of the SRP.

 A SRP does only save the changes of the system files  since the making of the SRP. Therefore an  SRP gets bigger (more MBs and more GBs) the longer it stays on the computer and the more programs are installed and uninstalled, including Microsoft updates, also the more desktop symbols are added. When you  use an SRP for restoring to a former state of the system files,  the changes of the system files are deleted and the state at the making of an SRP is restored.

Also on Windows 8, you can also see which programs are saved on one SRP.

It is important that SRPs do have enough hard disk space. Else all SRPs may disappear often. When the SRPs do not have enough hard disk space, the oldest SRPs disappeared at first.
When there is only one SRP which gets bigger and bigger and when it does not have enough space, it simply disappears and there is no SRP left.

 On Windows 8, you can easily choose the disk space for an SRP by clicking on 'Configure'.
