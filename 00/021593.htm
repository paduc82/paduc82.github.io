Are you clicking it once, or twice? And the other users?

By default, a MacroButton requires a double-click. There is an "Option" that can be set using VBA to make it a single click. Perhaps this has been activated on your machine, and not on others?

The other possible reason is that the macros in your document are being deactivated due to the users' settings in the "Trust Center". By default in Office 2007, macros are disabled without warning. But there are "trusted locations", where the user can copy the document so that macros will be trusted - without changing the "disable" settings.

The default locations are for templates, so the users will need to make some changes in the Trust Center. For example, they can set up a folder for your document, then go to "Trusted Locations" in the Word Options/Trust Center/Trust Center Settings and add that to their trusted locations.

Method 1

In many cases this would happen due any third party add on installed in the internet explorer 8

Check if any add-ons on the Internet explorer are causing the error. I would suggest you to try opening Internet explorer in the no add-ons mode.

a)   Click the Start button, click All Programs and click Accessories

b)   Click System Tools, and then click Internet Explorer (No Add-ons).

 

Refer the link below for more information.

http://windows.microsoft.com/en-us/Windows7/How-do-browser-add-ons-affect-my-computer

Method 2

If the above step fails then try resetting the internet explorer settings to default and check.

For more information refer the below link.

How to reset Internet Explorer settings

http://support.microsoft.com/kb/923737

Important: Reset Internet Explorer to its default configuration. This step will also disable any add-ons, plug-ins, or toolbars that are installed. Although this solution is quick, it also means that, if you want to use any of those add-ons in the future, they must be reinstalled.
