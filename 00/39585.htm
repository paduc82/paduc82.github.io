lost system restore point

ralf_g say:

about 2 months ago i created a system restore point and gave it a name, now i am looking to do a restore but that one is not listed in the restore points. can i recover it from somewhere?

why has it not been saved?

Two things that could pertain to this. First, older System Restore points do roll off the end of the list when the drive space allocated to system restore has been used up and new restore points are being created.

The second point is that system restore points are sometimes deleted when significant operating system updates include changes that are incompatible with files stored in those old restore point(s).

I'm presuming that you did tick the box in System Restore that says "show more restore points".

a. user say:

Restore points have a limited life. In Windows 7 that was 90 days. I understand it's shorter in Windows 10, but I don't know the length of time. In addition, restore points become invalid after a major update so they are deleted. You wouldn't want to restore a restore point that is two months old - it would make your system unstable.

Ever since System Restore was introduced, people have been confusing restore points with backups. They are not nearly the same. Once you understand what a restore point really is, you'll understand their limited usefulness.

The way to go back in time on a computer is to create a disk image aka 'system image.' That's a copy of your entire disk, compressed into a single file. When you restore an image, your computer goes back in time to the moment when you made the image. It's a fabulous way to backup. OEMs have been doing that for years - that's how a factory reset works.

Hi Matt,

Thank you for posting your query in Microsoft Community.

I would like to inform you that free upgrade is system specific and can be done on the same machine. Windows 10 is free for Windows 7(SP1), Windows 8.1 and Windows 10 preview.

If you use the ISO (Media Creation tool) and upgrade from within Windows, please make sure that you are selecting the correct edition for Windows 10.  For example:

        1. You are using Windows 8.1 Home.

        2. You uses the Media Creation Tool and selects Windows 10 Pro

        3. You tries to upgrade from Windows 8.1 with the media that was created in step #2

        4. End result:  They will be prompted for a product key.

Its important that you match the architecture and edition.

Please refer to the article How to Activate and resolve common Product key issues in Windows 10, by Mr. Andre Da Costa and check if it helps.
