Hi,

 

Well that's the best way but where people can get caught out is like this.

 

Going back to my previous example of storing the number (123) in a cell as text you can do this. Create another cell exactly the same so we have 123 stored in 2 cells as text, and let's say they are A4 and A5

 

Put this formula in a cell formatted as 'General' =A4*A5

 

Excel does the multiplication fine and returns 15129 which is 123*123 but the cell that was formatted as 'General' will now be formatted as 'Text'..

 

Now try another formula =SUM(A4,A5) and you will see this returns zero.

 

It's the mathematical operation of + - * or / that coerces a number stored as text into a number but this doesn't work for the =SUM formula.

 

You can spot these errors with error checking turned on:-

 

File \ Options \ Formulas and scroll down to error checking and ensure 'Enable background Error checking' is checked. These numbers stored as text are now identified by a coloured triangle in the corner of the cell and if you select the cell you get help on the error by clicking the dropdown that appears by the cell which also has an option to convert the text to a number.

    john korchok say:

    If the text doesn't display bullets, then when the user adds them, they will follow PowerPoint's defaults, not your formatting. For you to set the formatting, the bullets must be visible.

    You can create separate layouts that are identical except that one layout will have bullets for all levels and the other will have none. Or you can alternate, setting Level 1 without bullets, then Level 2 to be the same as Level 1, but with bullets added.

    Choose View>Slide Master and edit your bullets. If you edit them on the larger slide at the top (the Master), then the changes will ripple through all layouts. Or you can edit a layout, in which case your formatting will apply only to slides based on that layout.
