code numpty say:

Conditional format to apply if different formula is used

In certain places I might use formulas like this.

What I would like to do is use conditional formatting to highlight where I have used a formula different from the 1st example.

 

I can't work out if this is possible or not & would appreciate any pointers.

bill manville say:

You won't be able to use conditional formatting to do this.

 

Excel already provides a means of highlighting formulas that are inconsistent with others in the same region : File > Options > Formulas > Formulas inconsistent with other formulas in the same region.  If set, this will give a little green marker in the top left corner of the cells containing inconsistent formulas.

 

If this is not using the rules you would apply or is not a visible enough indication then you will need to use a macro to apply the desired formatting to the cells which satisfy your rules.

 

Note that if your workbook will not be used with versions of Excel prior to 2007 you could simplify your formula to

=ROUNDUP(D6695*IFERROR(INDEX(I_Section_Factors,MATCH(LEFT(A6695,2),Section_Code,0)),I_Base_Factor),2)

 

You could check for formulas of this general nature by
Sub HighlightNonStdFormulas()
  Dim C As Range
  For Each C In Selection.Cells
    If C.Formula Like "=ROUNDUP(*IFERROR(INDEX(I_Section_Factors,MATCH(LEFT(*,2),Section_Code,0)),I_Base_Factor),2)" Then
      C.Interior.ColorIndex = xlNone
    Else
      C.Interior.ColorIndex = 6
    End If
  Next
End Sub

bill manville say:

You will need to change the permissions.
When Excel saves a file it creates a new file with a temporary name (e.g. 123C345DE) in the folder containing the original file.
It then deletes the original file and renames the temporarily named file with the original name.
Thus to save an Excel file to a folder you do need delete access to the folder.

This process is good because it means you are less likely to lose the original file if there is a power or disc problem.
