Multiple Macro Running

Hi, 

Does anyone know how i can run two macros in one button : example macro 1 and 2

Once i press button once, macro 1 runs 

If i press again macro 2 runs, and this repeats 

PLS help Thanks

ossiemac say:

Need to alternate the name on the button each time code is called. The actual code depends on whether you are using ActiveX Command Button or a Forms Control Button. Code examples for both types below.

Note that you can initially insert a caption on the button but if you leave as default caption then the first time the code runs it will call Macro2 (which is the Else part) and then change the caption to Macro1 and after that they should be correct after each run.

Code for ActiveX Command Button. Assumes running Macro1 and Macro2. Example uses Macro names for the button caption but you may use whatever names you like for both the Macros and the Button Captions.

Private Sub CommandButton1_Click()

    'Can edit "Macro1" to any name you like
    If Me.CommandButton1.Caption = "Macro1" Then    'Edit CommandButton1 to your ActiveX Control Name
        Call Macro1
        Me.CommandButton1.Caption = "Macro2"    'Edit CommandButton1 to your ActiveX Control Name
    Else
        Call Macro2
        Me.CommandButton1.Caption = "Macro1"    'Edit CommandButton1 to your ActiveX Control Name
    End If
End Sub

Code for Forms Control button. As for ActiveX control Assumes running Macro1 and Macro2. Example uses Macro names for the button text but you may use whatever names you like.

Sub Button1_Click()
    Dim shp As Shape
   
    'Can edit "Macro1" to any name you like
   
    Set shp = ActiveSheet.Shapes("Button 1")        'Edit "Button 1" to your Forms Comntrol Name
   
    If shp.TextFrame.Characters.Text = "Macro1" Then
        Call Macro1
        shp.TextFrame.Characters.Text = "Macro2"
    Else
        Call Macro2
        shp.TextFrame.Characters.Text = "Macro1"
    End If
End Sub

Following example for called Macro code was used to test the above code.

Sub Macro1()
    MsgBox "Macro1 called"
End Sub

Sub Macro2()
    MsgBox "Macro2 called"
End Sub
