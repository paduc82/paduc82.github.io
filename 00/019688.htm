I've retyped the document and changed some bullet formatting and their indents, and it worked; 1) but is there a quick solution? 2) What was the main problem? I'm an EA so taking hours to reformat a whole financial document again (very possible) isn't in my best interest. 

Any advice is valuable!!!!!!!! Thanks in advance for your help!! 

suzanne s. barnhill say:

In each section, use Insert | Header & Footer | Page Number | Format Page Numbers... to change the number start to "Continue from previous section." You need to do this in each section of the document, not just the ones for which footers are displayed: if page numbering is restarted in a "landlocked" section in the middle of a page, it will affect the numbering in the following section.

Also, unless you're changing numbers of columns or page orientation, it may be that you have more section breaks than you actually need. Reducing those (and replacing them with manual page breaks if necessary) will make it easier to handle page numbering. It's also possible, looking at the numbering, that in the two sections with pages numbered 3 and 10 the page number has been typed manually instead of using a PAGE field. You can easily insert a PAGE field by pressing Alt+Shift+P.
