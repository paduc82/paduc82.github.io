travelor say:

Hello,

The Zune Playlist file has the 3 letter file extension of ZPL - If you search your computer for *.ZPL you'll find where the Playlists are stored. Ussually stored in ?:\User\Music\Playlist - Open that folder and copy the playlist and rename the 3 letter file extension to WPL - This will import the playlist into Windows Media Player - At which point you can open up Windows Media Player and view the Playlist - You can then use Windows Media Player to sync the playlist content to you non-zune MP3 player. 

Keep in mind that any items in the playlist are Zune Pass (DRM Protected) content they will not be sync'able and/or playable on the non-zune MP3 player.

This is a technical process and not recommended if you do not know what a 3 letter file extension is.

This also assumes that you can use Windows Media Player to "Sync" your non-zune MP3 player.
