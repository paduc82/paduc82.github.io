Read the Formula1 property from the validation, split it into an array by the seperator, modify the array as you like and store it back by joining the array with a comma.

Below is a simple example.

Andreas.

Sub Test()
  Dim Data
  With Range("A1")
    Data = Split(.Validation.Formula1, Application.International(xlListSeparator))
    ReDim Preserve Data(0 To UBound(Data) + 1)
    Data(UBound(Data)) = "New"
    .Validation.Delete
    .Validation.Add xlValidateList, Formula1:=Join(Data, ",")
  End With
End Sub
