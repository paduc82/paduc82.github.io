How can I compare two 8 digit binary numbers, from two separate cells, digit by digit, in a third cell? What I want to do is compare, say
11001011
and
11111111
and use an "and function, to say that, if both numbers in corresponding places are 1, a value of 1 is returned, but if there is a 0 in either number, in corresponding places, a value of 0 is returned. I'm trying to make a tool to analyse IP numbers with cidr notation, and this is for the step to determine the network address. Any help will be much appreciated. Thanks.
gary's student say:

If you assume that the binary numbers are actually strings (which functions like DEC2BIN() return), then try this little UDF:

 

Function lAnd(s1 As String, s2 As String) As String
lAnd = ""
If Len(s1) <> Len(s2) Then Exit Function
For i = 1 To Len(s1)
    If Mid(s1, i, 1) = "1" And Mid(s2, i, 1) = "1" Then
        lAnd = lAnd & "1"
    Else
        lAnd = lAnd & "0"
    End If
Next
End Function

 

This is not limited to 8 bit quantities.

There are no problems to create an update query with references to combos. Just choose fields needed to be update in Design query view and type smth like [Forms]![Form1]![Combo1] in "Update To:" section. It works well.

john p spencer say:

There are two things that normally cause this.

One: The form must be open - if it is closed, then the query cannot get values from the form.

Two: There is a typo in the reference to the control
The reference should be in the format [Forms]![NameOfForm]![NameOfControl] and if the form is being used as a subform the reference is different than that.

stefan blom say:

MVP Doug Robbins has created a macro which does what you want:

Sub NumberComments()

'Macro created by Doug Robbins, MVP, April 2017
Dim i as Long

Dim rngComment as Range

With ActiveDocument

    For i = 1 to .Comments.Count

        Set rngComment = .Comments(i).Range
        With rngComment
            If Left(.Text, 7) = "Comment" Then
                .Text = "Comment " & i & " " & Mid(.Text, Instr(.Text, "-"))

            Else

                .Text = "Comment " & i & " - " & .Text

            End If

        End With

    Next i

End With
End Sub
