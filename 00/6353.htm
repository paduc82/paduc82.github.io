cr_answers say:

You can change a Linked (paragraph and character) style (or a Character style, for that matter) back to a Paragraph style, unless it is built-in, without copying from another document or template. (For built-in, you must use a good Normal.dotx, as described in other replies, because you can't delete a built-in style.)

Here's how:

  1. Create a new style based upon the style that is Linked (paragraph and character), which  you want to be Paragraph again; leave the new style as Paragraph, set the "Style for following paragraph" to what it is for the original Linked (paragraph and character) style, and name it similarly to the original Linked (paragraph and character) style (perhaps with a trailing asterisk).
  2. Change all styles based upon the original Linked (paragraph and character) to be Linked (paragraph and character) style also set for the "Style for following paragraph", then change this setting for these styles to the new Paragraph style created in step1, as well.)
  3. Delete the original Linked (paragraph and character) style.
  4. Rename the new Paragraph style created in step 1 to the name of the original style that was Linked (paragraph and character).
