julie_s1 say:

Hello

I can see why you are concerned. In Skype for Windows 10, there is no option to configure the audio settings. Skype for Windows 10 will use the default audio device set up to your computer. If the device you want to use with Skype is not set up as the default audio device for your computer, Skype for Windows 10 will not detect it.

Your option is to either set up your preferred audio device as the default audio device for your computer, or install Skype for Windows Desktop(classic version) to be able to configure the audio settings.
