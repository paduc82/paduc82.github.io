arminatorx say:

[quote user="n00bsrus1423"]

"Sorry, Xbox Live profiles can't be downloaded right now. Try again later."

80072EFE

[/quote]

Errors starting with 8007 usually hint at network problems or problems that something can't be stored from the online service.

So please try the following:

First, do a connection test to Xbox Live, and double check that your console is able to completely connect to Live without problems.

Then on your storage device, make sure your profile still isn't there in the profile folder. If there is a profile showing as corrupted, delete it.

Then go back to the storage device selection, highlight any drive and press Y.

Choose to clear the system cache. Repeat this process three times.

Then back on the drive selection screen, press the following button combination:

LeftBumper, RightBumper, X, LeftBumper, RightBumper, X

Allow the console do the update maintenance, and if necessary, have the console restart and download the update.

After that, check on this website, if your security proofs (i.e. secondary email and cellphone number) are current and correct. Also check if you have two step verification set up. If so, you need to create an App Password further down the page!

And finally you're done.

On the console, bring up the guide menu again, and choose to download your profile.

If you use Two Step Verification, make sure you use your generated App Password instead of your "normal" password.
