Thanks!

P@

jay freedman say:

The only way you can create any document in Word with a page size of 11x17" is by installing a printer (or at least a printer driver) that supports pages that large. Most consumer printers/drivers don't support a page size wider than 8.5" in portrait orientation. When Word paginates a document, it relies on the currently selected printer driver for information about page size and character placement (http://www.word.mvps.org/FAQs/Formatting/TextReflow.htm).

One way your colleagues can see the same thing you see in Word is to install the printer driver software for your printer on their computers. They won't be able to print without the real hardware, but the screen layout will be the original.

If your colleagues don't need to edit the poster, you can save the document as a PDF file and send that. PDF readers aren't limited by printer drivers.

a. user say:

You may have got your answer somewhere else, but what will probably work for you here is

tell application "Microsoft Word"

activate

set myFind to find object of text object of active document

tell myFind

clear formatting

set content to ""

set italic of font object to true

tell replacement of myFind

clear formatting

set content to ""

set color index of font object to blue

end tell

execute find myFind replace replace all

end tell

beep

display dialog "Done!"

end tell
