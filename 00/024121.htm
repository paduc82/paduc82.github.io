mike h.. say:

Hi,

So all you want to do is infill the dates so they run in sequence with no gaps. Try this code which assumes the dates are sorted as they are in your example. Select any block of dates (just the dates) anywhere on the worksheet and run the code.

Sub filldates()
Dim x As Long, NumRows As Long
NumRows = Selection.Cells(Selection.rows.Count).Row
Dim RowsToInsert As Long
For x = NumRows To Selection.Cells(2).Row Step -1
RowsToInsert = (Cells(x, Selection.Column).Value - Cells(x - 1, Selection.Column).Value) - 1
If RowsToInsert > 0 Then
rows(x).Resize(RowsToInsert).Insert
rows(x - 1).Resize(RowsToInsert + 1).DataSeries Rowcol:=xlColumns, Type:=xlChronological, Date:= _
        xlDay, Step:=1, Trend:=False
End If
Next
End Sub

sandtones say: I finally did figure out what was going on, and thought I'd add a note in case someone else has the same problem.

In my file, I had a fair amount of conditional formatting - like all the cells in one column in a worksheet that had 20,000 rows.  And there were a few of those worksheets. In Excel 2010 (and 2007) conditional formatting seems to be saved as rules for ranges rather than on individual cells.  When converting to Excel 2010, it created a new formatting rule for each conditionally formatted cell, i.e. many thousands of conditional formatting rules.  When I deleted all the thousands of conditional rules and created one rule for each column, performance got back to what I would expect.

Pretty poor design of the converter, not to mention I couldn't find any documentation anywhere in Microsoft on the fact that this was a change one should be aware of.

Try the following.

Clear any existing Conditional Format. (Conditional format -> Clear -> Clear from entire sheet.)

 

Select ranges S26:BR31, S35:BR39 (Select the first range and then hold Ctrl key while selecting the second range.)

 

Select Conditional format -> New Rule

Select Use formula ..... etc

Enter the following formula. (Copy from here and paste)

 

=AND(S26<>"",COUNTIF($N$5:$BO$22,S26)>0)

 

Click the Format button and set the required format etc.
