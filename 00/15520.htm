danwhxu say: I found the answer myself by creating a NewMinutes field in the query with the expression of NewMinutes: IIf([Minutes] Mod 6=0,[Minutes],[Minutes]-([Minutes] Mod 6))

I then use the expression Pay Length: [NewMinutes]\60 & Format([NewMinutes] Mod 60,"\:00") to put it into the proper format.

marshall barton say:

The Format function always returns a text string so it is not particularly useful in a numeric calculation.  Use the DatePart or TimePart function instead.  In your case I think you could use:


   TimeCardEnd: [Actual End]-TimePart("n", [Minutes]) Mod 6

Or

   TimeCardEnd: TimeSerial(Hour(T), (Minute(T) \ 6) * 6, 0)


OTOH, if you really want the NEAREST 6 minute interval, use:

   TimeCardEnd: TimeSerial(Hour(T), ((Minute(T) + 3) \ 6) * 6, 0)

ken sheridan say: Or, if the values are always less than 24 fours, you can simply subtract the start time from the end time, round it down to the nearest 6 minute interval using the following function, and format the result:

Public Function RoundToZero(dblVal As Double, dblTo As Double) As Double

    Dim intUpDown As Integer
    Dim lngTestValue As Long
    Dim dblTestValue As Double
    Dim dblDenominator As Double
   
    If dblVal < 0 Then
        intUpDown = -1
    Else
        intUpDown = 1
    End If
   
    dblDenominator = intUpDown * dblTo
    dblTestValue = dblVal / dblDenominator
    lngTestValue = Int(dblTestValue)
    RoundToZero = intUpDown * lngTestValue * dblTo

End Function

You can see how it works in the debug window, firstly by simple subtraction without rounding down:

? Format(#2013-12-27 18:30# - #2013-12-27 14:15#,"hh:nn")
04:15

and with rounding down to #00:06#:

? Format(RoundToZero(#2013-12-27 18:30# - #2013-12-27 14:15#, #00:06#),"hh:nn")
04:12

For  time periods of 24 hours or longer there are other functions which can be brought into play, which you'll find illustrated in my little demo file Timesheet.zip in my public databases folder at:

https://skydrive.live.com/?cid=44CC60D7FEA42912&id=44CC60D7FEA42912!169

If you have difficulty opening the link copy its text (NB, not the link location) and paste it into your browser's address bar.

When doing date/time arithmetic like this it's important to only format the final presented values.  If you are summing a set of short durations over a long period, say, always sum the result of a simple subtraction of the values and then format the end result as required.  If each short duration is formatted as in the above examples and then an attempt to sum these values is made it won't work as the formatted values are string expressions not numeric values.

raystafarian say:

Sure, this is a pretty common request. You want to copy every nth row. You can use a formula like -

=OFFSET(Daily!$C$1,(ROW(A1)-1)*7,0)

The "*7" is what indicates the nth cell. You need to reference a cell in row 1 if your data begins in row 1, otherwise, the formula will need to be modified.

If you want to exclude your title cell, put in A2 instead of A1. If you need the data to start at a different point, you need to edit the second part of the formula -

     =OFFSET(Daily!$C$1,(ROW(A1)-1)*7)+2,0)

This will start at the 3rd cell in the column, rather than the first.

distributed text

I have tried control + shift + J to no effect. Apparently there is a symbol of two arrows pointing in opposite directions over a few lines, but I can't find it. Any ideas, please?

a. user say:

You can align a single line this way without using tabs or tables, by using the Justified alignment for a paragraph and ending the line with a hard line feed (shift-enter) at the end. But you have to ensure that there is only one space in the line (if you want left-right "distribution") and work a little harder if you want center-right "distribution.

Let's suppose you want to put "distributed text" at the left and "Word 14.3.9" at the right. Then
 a. start a new paragraph and format it as justified
 b. type "distributed text" but insert a non-breaking space (cmd-shift-space) instead of the space between "distributed" and "text".
 c. type a space
 d. type "Word 14.3.9", but with a non-breaking space between "Word" and "14"
 e. press shift-enter to insert a hard line feed.

If instead you want the "distributed text" centred and "Word 14.3.9" at the right, do the above, but then to the left of "distributed"
 e. insert an n-space (e.g. using Insert->Insert Symbol->Advanced Symbol...->Special characters) followed by an ordinary space.
