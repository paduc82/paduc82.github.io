This spreadsheet needs to be imported into an Access table called "tbl_NewCasesTempTable". which has two fields in particular "Ncp Last Name" and "Ncp First Name". I need to parse the spreasheet field "Ncp Name" into these two fields. I was thinking that I could import the "Ncp Name" data into the Access table field "Ncp Last Name" then run some kind of update query to move the first name over to the "Ncp First Name" field. boblarson say:

You don't put the Left and Mid stuff in criteria.  You use it in the Insert.  Like this:

 

INSERT INTO tbl_NewCasesTempTable2ndLevelOfImportingThenParsing ( Caseworker, Team, [Case Number], [Ncp Last Name], [Ncp First Name], [Ncp Participant Id], [Case Function] )
SELECT tbl_NewCasesTempTable1stLevelOfImporting.Caseworker, tbl_NewCasesTempTable1stLevelOfImporting.Team, tbl_NewCasesTempTable1stLevelOfImporting.[Case Number],

Left([Ncp Name],InStr(1,[Ncp Name],",")-1) As [Ncp LastName],

Mid([Ncp Name],InStr(1,[Ncp Name],",")+1) As [Ncp FirstName], tbl_NewCasesTempTable1stLevelOfImporting.[Ncp Participant Id], tbl_NewCasesTempTable1stLevelOfImporting.[Case Function]
FROM tbl_NewCasesTempTable1stLevelOfImporting
ORDER BY tbl_NewCasesTempTable1stLevelOfImporting.Caseworker, tbl_NewCasesTempTable1stLevelOfImporting.Team;
