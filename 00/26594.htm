I have several email accounts, all of which are POP except for 2 Exchange accounts. I'd like to share contacts and calendar between all accounts or have only one contacts list and one calendar. Both Exchange accounts are through Office 365's Outlook. I also sync my contacts and calendar with my Android phone using Mobisynapse. I'm using Outlook 2010 on Win7, Win8.1 and Win10 machines.


You are always going to have at least 3 contact folders in three different folder trees 

1) Each EX account will have a contact folder in the folder tree

2) There will be only one PST file with a contact folder for your POP accounts (unless you are using a PST file for one or more of the POP accounts that was previously set as the default for the profile). If all POP accounts use the same PST file, there will only be one contact folder

3) Next key item is that only one of those contact folders will be the <default> for the profile. <Default> contact/calendar folders reside in the default data file for the Outlook profile. From OL'2010 menu, go to 

File --> Account Settings --> Data Files tab --> select a data file associated with an EX account to be the default

If you select one of the EX accounts and copy all of you contacts to that contact folder, everything will be shared across all devices which have that account configured. All you need to do is configure you Android device to use that account using Exchange ActiveSync negating the need to manually sync via Mobisynapse
