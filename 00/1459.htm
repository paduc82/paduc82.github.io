The usual fix for issues like this is to not run the setup.exe, since the setup.exe will check OS version numbers and error out if the version is not what the setup.exe is expecting. 

If you have a folder with .inf and .sys files, that is all you need.  Manually install the drivers by going to device manager, right click the device in question, select "Update Driver Software", "Browse My Computer for driver software", "Let me pick from a list of device drivers on my computer".   Choose the device on the next screen and click "Have disk...".   Browse to the folder that contains the .inf file and click open. 

That should install the driver.  If this does not work, then the driver is not correct for the device.  I have installed many drivers this way, when the automatic methods did not work.
