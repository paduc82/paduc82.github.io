Print comments only in Word 2013 - Microsoft Community

malbr say:

Print comments only in Word 2013

I often work with long documents in Word 2013 and just want to pull out and print the comments in a separate list so that I can review them on their own.  In case others have the same problem, here is the solution I found:

View tab > Print Layout

Review tab > All Markup (in Tracking group)

Review tab > Show Markup > check Comments, uncheck next 3 items (Ink, Insertions and Deletions, Formatting)

Review tab > Show Markup > Balloons > Show Only Comments and Formatting in Balloons

Review tab > Specific People > make sure that the desired reviewers are checked

File tab > Print > Settings group, first dropdown box, select List of Markup

File tab > Print > Settings group, first dropdown box, check Print Markup

File tab > Print > Printer group, set the desired printer and printer properties

File tab > Print > click on Print.
