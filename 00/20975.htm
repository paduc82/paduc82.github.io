yang zheng msft support say: Hi UselessUser,

Modern authentication brings Active Directory Authentication Library (ADAL)-based sign-in to Office client apps across platforms. This enables sign-in features such as Multi-Factor Authentication (MFA), SAML-based third-party Identity Providers with Office client applications, smart card and certificate-based authentication, and it removes the need for Outlook to use the basic authentication protocol. The ADFS service is not required.

Active Directory Federation Services

suzanne s. barnhill say:
It sounds to me as if you need to create a parent document that contains specific chunks of text that can be bookmarked and then use IncludeText fields in the child documents. When you update the bookmarked content in the parent document, you should be able to then update it automatically in the child documents. For more, see http://word.mvps.org/FAQs/TblsFldsFms/includetextfields.htm
