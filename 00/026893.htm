Here is the macro that you wanted.  This solved the problem for me when nothing else would.

As you suggested, modify the Send/Receive groups so that there is only one account in the first group and put the others in the second group called "Others".  Disable the automated updates for the second group since this macro updates the second group when the first group is finished.  Add this to the ThisOutlookSession module:

Dim WithEvents mySync As Outlook.SyncObject
Private Sub UpdateOthers()
  Dim nsp As Outlook.NameSpace, sycs As Outlook.SyncObjects
  Set nsp = Application.GetNamespace("MAPI")
  Set sycs = nsp.SyncObjects
 
  Dim i As Integer, syc As Outlook.SyncObject
  For i = 1 To sycs.Count
    Set syc = sycs.Item(i)
    If syc.Name = "Others" Then syc.Start
  Next
End Sub
Sub UpdateFolders()
  mySync.Start
End Sub
Private Sub Application_Startup()
  Set mySync = Application.Session.SyncObjects.Item(1)
  UpdateFolders
End Sub
Private Sub mySync_SyncEnd()
  UpdateOthers
End Sub
