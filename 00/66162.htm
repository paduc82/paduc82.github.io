amosshapir say:

SOLVED!

I really tried every answer presented, including using regedit, and it still popped back to the same language (particularly annoying in case of Hebrew, because it makes a mirror image of the screen and all windows, even the little "x" is on top left).

Found out that this happens when the display language is set correctly, but the "override" option is on for a different language.  I have no idea who or what does this (or why there is such an option anyway).

The solution (at least in my case):

Settings -> Time & Language -> Region & language -> Additional time, region & language settings (on the top right) -> change input method

Choose the current display language; it is marked "enabled (override)"

Click "change override" -> set the language you want as the new override language, and click "Save" (at the bottom).

It should prompt you to sign out and back in.

Good luck & have fun,

Amos
