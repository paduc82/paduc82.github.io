Given the message, at some point you had a valid external drive connected that File History was saving file backups to.  Your system is trying to run the backup again to that drive according to a default schedule, and is not finding it.  So if the above is the case, reconnect the external drive, and File History will run another backup.

If this is not the case (if you're not using File History for backups, never had an external drive connected, or just want this to go away), you can kill it by turning off File History manually.  Go to Settings / Update & security / Backup and turn the "automatically back up my files" toggle under Backup / Back up using File HIstory to "off", and you shouldn't see that message again.

 

That said, it's wise to run backups, using File HIstory or the program of your choice, to an external drive.  File History can do this for you as long as it is able to find the backup drive, or you can find another (better) backup program to use.  Keeping a regular backup on an external drive will ensure that if anything goes sideways with your machine, you'll be able to easily  replace your personal files and won't lose anything important.
