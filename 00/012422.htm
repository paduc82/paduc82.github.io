suzanne s. barnhill say:

If you just want to print lines for your child to write on (by hand), then there are several methods you can use. When I want to print out a lot of lines for, say, a sign-up sheet, here's how I do it.

  1. In a blank document, press Enter so that you have two paragraphs.
  2. Select both paragraphs.
  3. On the Home tab, in the Paragraph group, click on the arrow next to the Borders button (if you haven't previously used it in a given Word session, its ScreenTip will say "Bottom Border") and select Inside Horizontal Border.
  4. Each time you press Enter, you'll get another line.
  5. You may find that Word's default spacing is adequate, but if you want the lines farther apart, you can change the Spacing Before/After. One quick way is to select all the lines and press Ctrl+0, which will add 12 points Spacing Before. To adjust spacing more precisely, you can select all the paragraphs, right-click, choose Paragraph..., and add more Spacing Before or After or remove the spacing and set the line spacing to an Exact amount (such as 24 or 36 points, the equivalent of double- or triple-spacing on a typewriter).

Workbook file formats that are supported in a browser window

Workbooks in any of the following file formats can be viewed in a browser window:

Excel workbook (.xlsx)
Excel binary workbook file (xlsb)
OpenDocument Spreadsheet file (.ods)
Excel macro-enabled workbook (.xlsm)
A workbook in this format can be opened but macros do not run in a browser window.
Excel 97- Excel 2003 Workbook (.xls) or Excel 97- Excel 2003 Template (.xlt)
If a workbook is in .xls or .xlt format, you can typically view but not edit a workbook in a browser window. When the file is edited, it is saved in .xlsx or .xltx format.
Workbooks in other formats, such as .csv format, are generally not supported for viewing in a browser window.

Best Practices for having outlook.com mirror outlook 365?

aravinda balakrishna say:

Hi,

 

I will help you to understand how Personal folder works and also help you to move the items from personal calendar to Outlook 365 calendar.

 

Personal folder is a pseudo .pst file created initially at the time of Outlook launch when no email account was setup with it. Since the calendar items does not sync as the Outlook 365 calendar, it signifies that this .pst is not connected or does not have the ability for that purpose. Hence you may either move all the items from that .pst file or the required items to Outlook 365 personal folder.

 

Let us copy all the calendar items from the personal calendar and then paste it to the Outlook 365 calendar.

 

You may also move/copy other items from the personal  folder to Outlook 365 folders.

Once you have moved/copied all the items that is needed to the Outlook 365 folder, ensure that the Outlook 365 data file is set as default. In order to check that, follow these steps:

  • Go to File, Info.
  • Click Account Settings.
  • Go to Data Files tab and ensure the Outlook 365 account data file is set as default. If not, select it and then click Set as default option at the top.

 

Let us know if this suggestion is helpful.
