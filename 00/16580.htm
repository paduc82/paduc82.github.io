The simplest fix for issues with anything under the WindowsSelfHost Registry Key is to simply delete the entire WindowsSelfHost Key. It will be rebuilt.

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\WindowsSelfHost

If that does not get you back on track, copy the following to notepad and create a file called update.txt:
net stop wuauserv
net stop cryptSvc
net stop bits
net stop msiserver
ren C:\Windows\SoftwareDistribution SoftwareDistribution.old
ren C:\Windows\System32\catroot2 Catroot2.old
net start wuauserv
net start cryptSvc
net start bits
net start msiserver

Now rename the file to update.bat, and run the batch file. Keep us posted on the results.

Option Explicit

Private Const GWL_STYLE = (-16)
Private Const WS_CAPTION = &HC00000
Private Declare Function FindWindow Lib "user32" Alias _
  "FindWindowA" (ByVal lpClassName As String, ByVal _
  lpWindowName As String) As Long
Private Declare Function GetWindowLong Lib "user32" Alias _
  "GetWindowLongA" (ByVal hwnd As Long, ByVal nIndex As Long) _
  As Long
Private Declare Function SetWindowLong Lib "user32" Alias _
  "SetWindowLongA" (ByVal hwnd As Long, ByVal nIndex As Long, _
  ByVal dwNewLong As Long) As Long
Private Declare Function DrawMenuBar Lib "user32" (ByVal hwnd _
  As Long) As Long

Private Sub UserForm_Activate()
  Dim Style As Long, hwnd As Long
  hwnd = FindWindow(vbNullString, Me.Caption)
  Style = GetWindowLong(hwnd, GWL_STYLE)
  Style = SetWindowLong(hwnd, GWL_STYLE, Style And Not _
    WS_CAPTION)
  DrawMenuBar hwnd
End Sub

martin v. - wp7 support say:

Hi,

I am not 100% sure what is your goal - but guessing that someone suggested you to export the favorites from the PC and organize them in OneNote. 

To export the list open the IE click Favorites on your PC - Click the arrow next to the Add to Favorites... to display additional options.
Click the Import and Export and follow the wizard for the export.
Open the exported file (bookmark.htm) and copy/paste the links to any editor (OneNote, Word etc.) if you need to make changes in the list.
Email the exported file or edited list of bookmarks to your phone so you can access it from the device. (OneNote document could be synced automatically with your Live account)

I think that the person is using the OneNote synchronized with Live account as a alternative list of bookmarks on his phone. Off course you can add the page opened from your list to the device Favorites list following the steps described here:
