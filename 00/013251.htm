If you want to delete the emails from server as soon as Outlook downloads them, you may follow the steps below.
·         In Outlook, go to File tab > Account Settings > Account Settings.
·         Highlight existing email account.
·         Click Change > More settings >Click Advanced tab.
·         In Delivery section uncheck box ‘Leave a copy of message on the server’.
·         Click OK > Next and Finish.

·         In the Advanced tab, check the option ‘Remove from server when deleted from ‘Deleted Items’.
Note: Emails will only be deleted once delete from Outlook and after emptying the ‘Deleted Item’ folder.

Word should still be able to autosave, so simply wait until the next autosave, then go to Task Manager - Processes and end winword.exe. This shuts down word in the 'non safe way'. doug robbins - mvp office apps & services (word) say:

If you can live with unlinking the INCLUDETEXT fields, you could run a macro containing the following code to do that and then update the remaining fields

Dim i As Long
With ActiveDocument
    For i = .Fields.count To 1 Step -1
        If .Fields(i).Type = wdFieldIncludeText Then
            .Fields(i).Unlink
        End If
    Next i
    .Range.Fields.Update
End With
