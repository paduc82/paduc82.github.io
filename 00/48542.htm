A print spooler is software that temporarily stores print jobs on the computer hard disk or in memory until the printer is ready to print them.

If you've been waiting an unusually long time for a document to print, or if you see an error message about the print spooler, you might need to restart the print spooler on your computer.

The simplest way to restart the spooler is to save your work and restart the computer. You can also try these steps:

To restart the Print Spooler service

    You must be logged on as an administrator to perform these steps.

  1. Click Start, type: services

  2. Hit Enter

  3. Right-click the Print Spooler service, and then click Properties.

  4. On the General tab, next to Startup type, make sure that Automatic is selected.

  5. If the service is not already running, under Service status, click Start, and then click OK. If you are prompted for an administrator password or confirmation, type the password or provide confirmation.

xb1r athena say:

Overview

Login Information

 

  • Gateway: 192.168.1.1
  • Username: admin
  • Password: admin

 


 

  

How To Find SSID (Network Name)

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter "admin" for the user name and "admin" for the password.
  4. Select "WLAN" from the menu at the top.
  5. Select "Basic Settings" from the sub menu.
  6. The network name will be labeled as "(Root) SSID".

 

 

How to Find Wireless Passphrase (WPA/WEP)

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter "admin" for the user name and "admin" for the password.
  4. Select "WLAN" from the menu at the top.
  5. Select "Security" from the sub menu.
  6. The passphrase will be labeled as "Pre-Shared Key".

 

 

Changing Wireless Security Type/ Disabling Wireless Security

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter "admin" for the user name and "admin" for the password.
  4. Select "WLAN" from the menu at the top.
  5. Select "Security" from the sub menu.
  6. Select the desired security type from the "Encryption" drop down menu. (To disable the wireless security, select "None".)
  7. Select Apply Changes.

 

Change Wireless Channel

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter "admin" for the user name and "admin" for the password.
  4. Select "WLAN" from the menu at the top.
  5. Select "Basic Settings" from the sub menu.
  6. Select the desired wireless channel from the "Channel Number" drop down menu. (To let the device select automatically, select "Auto".)
  7. Select Apply Changes.

 

Change Wireless Mode

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter "admin" for the user name and "admin" for the password.
  4. Select "WLAN" from the menu at the top.
  5. Select "Basic Settings" from the sub menu.
  6. Select the desired wireless mode from the "Send Rate" drop down menu.
  7. Select Apply Changes.

 

UPnP

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter "admin" for the user name and "admin" for the password.
  4. Select "Advance" from the menu at the top.
  5. Select "Other" from the sub menu.
  6. Select "UPNP" from the menu at the left.
  7. Select the radio button next to either "Disable" or "Enable" for the "UPnP" option, based on current configuration.
  8. Select Apply Changes.

 

DHCP Reservation

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter "admin" for the user name and "admin" for the password.
  4. Select "LAN" from the menu at the top.
  5. Select "DHCP Settings" from the sub menu.
  6. Select the "MAC-Base Assignment" button.
  7. Enter the MAC address of the Xbox in the "Host MAC Address (xx-xx-xx-xx-xx-xx)" box.
  8. Enter the desired IP address for the Xbox in the "Assigned IP Address (xxx.xxx.xxx.xxx)" box.
  9. Select Assign IP.

 

DMZ

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter "admin" for the user name and "admin" for the password.
  4. Select "Advance" from the menu at the top.
  5. Select "Virtual Server" from the sub menu, then "DMZ Settings" from the menu to the left.
  6. Check the "Enable DMZ" check box.
  7. Enter the IP address of the Xbox in the "DMZ Host IP Address" box.
  8. Select Apply Changes.

 

Disable DMZ

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter "admin" for the user name and "admin" for the password.
  4. Select "Advance" from the menu at the top.
  5. Select "Virtual Server" from the sub menu, then "DMZ Settings" from the menu to the left.
  6. Remove the check from the "Enable DMZ" check box.
  7. Select Apply Changes.

 

Port Forwarding

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter "admin" for the user name and "admin" for the password.
  4. Select "Advance" from the menu at the top.
  5. Select "Virtual Server" from the sub menu, then "Services" from the menu to the left.
  6. Select Add.
  7. Select the radio button next to "Custom Services" and enter "xbox".
  8. From the "Protocol" drop down menu, select "TCP/UDP".
  9. Enter 3074 into both the "WAN Port" and the "Server Host Port" boxes.
  10. Enter the IP address of the Xbox into the "Server IP Address" box.
  11. Select OK.

 

Disable Port Forwarding

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter "admin" for the user name and "admin" for the password.
  4. Select "Advance" from the menu at the top.
  5. Select "Virtual Server" from the sub menu, then "Services" from the menu to the left.
  6. Locate the port forwarding rule you would like to remove.
  7. Select the button to the right of the rule under "Delete".

 

MAC Filtering

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter "admin" for the user name and "admin" for the password.
  4. Select "Advance" from the menu at the top.
  5. Select "Firewall" from the sub menu, then "MAC Filter" from the menu to the left.
  6. Select "Delete All" to remove any rules that may be set up.
  7. Select the radio button next to "Allow" for both the "Outgoing" and the "Incoming" actions.
  8. Select Apply Changes.

 

Bridging Guide

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter "admin" for the user name and "admin" for the password.
  4. Select "Advance" from the menu at the top.
  5. Select "Other" from the sub menu, then select "IP PassThrough" from the menu at the left.
  6. Select the desired device (typically the router) from the "IP PassThrough" drop down box.
  7. Select Apply Changes.

 

Firewall

 

This feature is not available on this device.

 

Firmware Update

 

Due to the age of this device, the firmware is no longer available online.

 

Restore Factory Defaults

 

Through Firmware

 

  1. Plug the computer directly into the devices LAN port via an Ethernet cable.
  2. Open a web browser.
  3. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  4. When prompted, enter "admin" for the user name and "admin" for the password.
  5. Select "Admin" from the menu at the top.
  6. Select "Commit/Reboot" from the sub menu.
  7. Check the "reset to default settings" check box.
  8. select Reboot.

 

Through Hardware

 

  1. Plug the computer directly into the devices LAN port via an Ethernet cable.
  2. Locate the "Reset" button on the back of the device.
  3. Using a toothpick/pen/paper clip, press and hold the button for 10 seconds.

Thank you for clarifying the issue, David. If that’s the case, it is possible that you are unable to log in or switch user due to a hidden local user that is probably set up using your account. To confirm this information, we suggest that you enable Administrator account on your device to check what other accounts are currently set up on the computer. Proceed to the steps below:

Disclaimer: Before following the steps below, we suggest that you do a backup for your files. Saved on your OneDrive or use a storage device like a flash drive to do the backup. 

1. Click the Start menu and type Command Prompt.

2. Right-click on it and choose Run as administrator.

3. Type net user Administrator /active:yes and hit Enter.

4. Close the Command Prompt and restart the computer.

5. Sign into the Administrator account and then press the Windows key and R simultaneously on your keyboard to open the Run command.

6. Type netplwiz and click OK.

Once you confirmed, that there are other accounts available in the User Accounts window, we suggest that you remove all of them aside from the Administrator account. Doing so should enable us to add your accounts on the computer successfully. Here are the steps below:

  1. High-light each email account and click Remove.
  2. Once they are removed, click Apply and OK.
  3. Open the Run command again, by pressing the Windows key and R simultaneously on your keyboard and hit Enter.
  4. Type netplwiz and then click OK to get back the User Accounts window.
  5. Click Add button to add your account as MSA login and do the same for your wife account. Once they are added, click Apply and OK.
  6. Restart your computer and sign in to both accounts one by one to see if you are able to access them without any issues.  

Once the issue resolved after you performed the steps above, we recommend signing in either in your account or in your wife account to disable the Administrator account on your computer:

1. Once you signed into your account, click the Start menu and type Command Prompt.

2. Right-click on it and choose Run as administrator.

3. Type net user Administrator /active:no and hit Enter.

4. Close the Command Prompt and restart the computer.
