jay freedman say:
I haven't tried a layout like this, but I'm not surprised to hear that it has problems. However, there is another way.

Set the table on the left to have no text wrapping. Insert a Continuous section break before the table on the left, and another one just above the last text box. With the cursor somewhere between those two section breaks, but not in any table or text box, click Page Layout > Columns > More Columns. teylyn say:
Hello,

start with an empty XY chart.

Use the "Select Data" dialog and add the first series. Format the X axis and set it to plot "Values in reverse order". 

Use the "Select Data" dialog and add the second series. It will be plotted with x values in reverse order. Format that series it to be on the secondary axis. Now it will be plotted with x values in regular order.

Hi,

 

I'm a bit confused by this because for every S in Col C the value in Col B is the same but there we are.

 

Because you're using E2003 the formula is long, Put this ARRAY formula in a cell; see below on how to enter and ARRAY formula. ARRAY enter it and drag down for (say) 50 rows and it will return blanks cells. Put an S in K1 and it will return every value in Col B where there is an S in Col C

 

=IF(ISERROR(INDEX(B$1:B$50,SMALL(IF($C$1:$C$50=$K$1,ROW($A$1:$A$50)-ROW($A$1)+1),ROWS(B$1:B1)))),"",INDEX(B$1:B$50,SMALL(IF($C$1:$C$50=$K$1,ROW($A$1:$A$50)-ROW($A$1)+1),ROWS(B$1:B1))))

 

In case you need it if you drag one cell to the left it will return Col A and drag right it will return other columns

 

This is an array formula which must be entered by pressing CTRL+Shift+Enter

and not just Enter. If you do it correctly then Excel will put curly brackets
around the formula {}. You can't type these yourself. If you edit the formula
you must enter it again with CTRL+Shift+Enter.

john mcghie say:
Phillip wasn't wearing his glasses :-)

Your Normal template will be in your Documents folder, in the Microsoft User Data folder.  And it's named simply "Normal" with no extension (which is a bug, but Word won't load it if you give it the correct extension).

To fix this, don't bother with the template.  Use Format>Style and ensure "Normal" is highlighted.

Click "Modify" then "Format" and choose "Paragraph"

Set the spacing you require, and click OK.

Click "Add to Template" and click OK all the way out.

Now quit Word and re-start it.

By clicking "Add to Template" you force Word to write your change back to the template.  New documents created from now will be correct.  Old documents, you will need to change the Normal style as appropriate in each one.

If you do this and the change does not stick, your Normal template is corrupted, quit Word and delete it.  It will create a new one when it restarts.
