Create a Default Theme in PowerPoint 2010 for Windows

Technical Level : Basic

Summary
Learn how you can set up any Theme in PowerPoint 2010 to be the default for new presentations you create -- this can be very helpful indeed when you want to use your corporate, organization or company Theme as the default.
Details

Do you want to use a Theme other than the blank white design that PowerPoint 2010 provides by default? True, the basic white design is a great starting point -- but there are situations when you need to use the approved Theme of your company or organization for all your presentations. If that's true, follow these steps to create a default Theme:

  1. Start a new blank presentation. Access the Design tab of the Ribbon.
  2. Within the Themes group, click the More button to bring up the Themes drop-down gallery.
  3. Now, right-click any of the Themes to bring up a context menu. Within the context menu select the Set as Default Theme option.

  4. Hereafter, when you choose the File | New | Blank option, or even just launch PowerPoint -- then a presentation will be created based on the Theme you chose to be the default.

Note: What happens if you also have a default template in addition to a default Theme? To learn more about what's different between a default template and a default Theme -- and to learn how you can create a default template, look at the Change the Default Template or Theme in PowerPoint 2010 page (external link).

  3. Search for the Graphics Card device from the device list, right click on it and then select Uninstall. On the un-installation window, if you have an option: Delete the driver software for this device you may select that and then remove the corrupted drivers from the computer.

  4. Follow the on-screen instructions to complete it and the restart the computer if prompted.

Go to the Computer Manufacturers website and search for the Windows 10 compatible drivers for the graphics card and install it successfully on the computer.
