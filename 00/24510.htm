Word-speak does not call such usage a template because, in Word, a template is a special kind of document.

pamcaswell say: >Oh my, that makes things worse, I'm afraid.

Not really.  Think about this situation:   A user does not apply styles to text, that is, uses the normal paragraph style for regular text, and uses the bullet and number icons regularly.  Bulleted and numbered paragraphs in W2003 and before are in the normal paragraph style.  Later, if the user needs to change the regular text style to a different paragraph style (or even to reapply it) by selecting all instances of normal, the bullets and number paragraphs also change and the bullets and numbers are lost. 

Now, using W2010, the  icon-applied bullets and numbers are in the paragraph style called List Paragraph, which will not be selected when choosing "Select all X instances" of normal and so won't be changed.  I don't think many people notice this change (not many questions in this forum, for example). ---The people who benefit most, I think, from the new feature are those who clean up and reformat large documents. In many of my documents, I can select all instances of List Paragraph and change them to List Bullet. Just makes things go faster.

The sets of built-in paragraph styles like List Bullet and List Number work in W2007/10 as they did in W2003. Each of those paragraph styles is linked to a multilevel list. 

The List Paragraph (paragraph) style is not linked to a multilevel list-there's no mention of it in the setting description in the Modify Style dialog as there is for the List Bullet style, for example.  So it's automatic use with icon-applied bullets and numberings must be hard coded in the program. 
 
>Are these "list templates" taking the place of the list styles from Word 2003?

If by "list styles from Word 2003", you mean, say, the  List Bullet and List Number paragraph styles, then no. These styles are still in W2007/10.  List templates are multilevel list numbering schemes. The built-in ones are displayed under the List Library heading in the gallery of multilevel lists. They work

stefan blom say:

Note that the safest approach to setting up multilevel numbering (in any Word version from Word 97 and later) is to make use of styles.

In Word 2007, to set up a multilevel list from start, click the Home tab, click Multilevel List, and then click Define New Multilevel List. In the dialog box you should define all aspects of numbering, for all levels, and associate each level with a unique paragraph style. You would then apply numbering to text by applying the appropriate paragraph style. For assistance with the options of the dialog box, see http://www.shaunakelly.com/word/numbering/numbering20072010.html.

To edit an existing list, place the insertion point in the first level 1 item of the list (for heading numbering, this means the first Heading 1 paragraph of the document) and click Home tab | Multilevel List | Define New Multilevel List.

Your best bet is to define it as an equation and assign it an Autocorrect entry, given that Word does not interpret correctly Unicode's combining characters:



Select Insert > Equation and then type "A" and choose the "Accent" icon to select the vecto symbol:



Then go to Preferences > Autocorrect, enable the check marks and define an entry:




The "Formatted text" button is barely showing to the right, but that's what you have to select. Then, just type the sequence. As soon as Word detects it as a word, it replaces it. Any unique sequence will do (\va, va\, vectora, etc.).
