This problem occurs if the manifest file in the Office folder is corrupted or if it is not created.

Try the steps mentioned below and see if it helps.

Do you have another computer where Office is loaded?

If yes, then on the other computer search for a file by name powerpnt.exe.manifest in the location C:/Program Files/Microsoft Office/Office12or14 folder.

For 32 bit Windows 7 the location would be C:\Program Files\Microsoft Office\Office14

For 64 bit Windows 7 the location would be C:\Program Files x (86)\Microsoft Office\Office14

After pasting the file try opening PowerPoint and check if it opens up.
