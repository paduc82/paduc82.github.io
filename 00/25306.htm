_________________________________

To manually start Windows XP with a clean boot, follow these steps:

Step 1: Start the System Configuration Utility
  1. Click Start, click Run, type msconfig, and then clickOK.
  2. The System Configuration Utility dialog box is displayed.
Step 2: Configure selective startup options
  1. In the System Configuration Utility dialog box, click the General tab, and then click Selective Startup.
  2. Click to clear the Process SYSTEM.INI File check box.
  3. Click to clear the Process WIN.INI File check box.
  4. Click to clear the Load Startup Items check box. Verify thatLoad System Services and Use Original BOOT.INI are checked.
  5. Click the Services tab.
  6. Click to select the Hide All Microsoft Services check box.
  7. Click Disable All, and then click OK.
  8. When you are prompted, click Restart to restart the computer.
Step 3: Log on to Windows
  1. If you are prompted, log on to Windows.
  2. When you receive the following message, click to select the Don't show this message or launch the System Configuration Utility when Windows start check box, and then clickOK.

    Notes
    • You have used the System Configuration Utility to make changes to the way Windows starts.
    • The System Configuration Utility is currently in Diagnostic or Selective Startup mode, causing this message to be displayed and the utility to run every time Windows starts.
    • Choose the Normal Startup mode on the General tab to start Windows normally and undo the changes you made using the System Configuration Utility.

macropod say:

Without knowing what the macros are supposed to do or how they're coded, little specific advice can be given. Many macros work exactly the same in Word 2016 as in Word 2003; others don't.

Macros in existing templates and documents should work the same as before; macros that assume the use of the Normal template and default Style attributes (especially for the Normal Style) may need to be edited. Macros that open & save files may also need to be edited to work with the new file formats.

One particular issue you'll need to consider is that docx files cannot contain macros, though they can leverage macros in an attached template; to contain macros, documents native to Word 2016 need to be saved in the docm format.

stephanie sharp say:

Hi,

I'm Stephanie, and I'm a program manager on the Sway team. There's a few known issues with this feature that we have plans to fix in the future!

1. In the type of navigation you're using in the Sway you sent, we currently don't have a "subtle" version available. There are a couple bugs we plan to address. One being that we're not letting the user know that, and ultimately adding in a subtle type of animations for this navigation.

2. The "intense" version of the animations emphasis in ALL navigation types and styles causes images in titles and headings to have ambient animations while in your viewport.
