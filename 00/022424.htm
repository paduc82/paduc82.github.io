If - then X - Microsoft Community

susankw say:

If - then X

If the value of a cell is greater than zero put an X in the cell. If the value is zero leave cell blank.

How do I express this in an equation?

jeeped say:

Use custom number formatting to achieve your goal of having the same cell display an X or not.

Select the cells to be affected and tap Cltr+1. When the Format Cells dialog opens, choose the Number tab and select Custom from the list down the left. Supply one of the following for the Type:,

"X";"X";;

... if you want to avoid displaying any text that might be accidentally typed into the cells or,

"X";"X";;@

... if you want to show text that might be typed in.

Custom number formats use the syntax,

<greater than zero>;<less than zero>;<equals zero>;<text>

If you do not want negative numbers displayed either, remove the second "X" but retain the semi-colon.

j.dsouza say: Try this ...
Microsoft Excel 2007

1.   Click the Microsoft Office Button, and then click Excel Options.

2.   Click Advanced, and clear the Ignore other applications that use Dynamic Data Exchange (DDE) check box in the General area.

3.   Click OK.

 

Microsoft Excel 2003 or earlier versions of Excel

1.   Click Options, on the Tools menu.

2.   Click the General tab.

3.   Clear the Ignore other applications that use Dynamic Data Exchange (DDE) check box, and then click OK.

After you do this, you should be able to open workbooks by double-clicking them in Windows Explorer.

 

When you double-click an Excel workbook in Windows Explorer, a dynamic data exchange (DDE) message is sent to Excel, instructing it to open the workbook that you double-clicked.

If the Ignore other applications that use Dynamic Data Exchange (DDE) setting is selected, Excel ignores DDE messages sent to it by other programs. As a result, the DDE message sent to Excel by Windows Explorer is ignored, and Excel does not open the workbook that you double-clicked.
