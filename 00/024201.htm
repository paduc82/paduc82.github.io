scott mcdaniel say:

There may be other methods, but you could use the Switch method:

Year(BillDate) & " -" & SWITCH(Month(BillDate)<4, "1Q", Month(BillDate<7, "2Q", etc etc)

The Switch method examines the first set of items in the list, and if it is True, returns the second element of that set of items. If it is False, it moves to the next set of items, and so forth.

If it works, slowly add back programs testing the camera after every 4-5 that you add back to make sure it is working. If it stops then it should be easy to figure out what program is causing the issue.
