eli arnold say:

The Pocket MSN service will be shut down in Mar 2010, which may affect users ability to access the services offered through Pocket MSN.

 

Pocket MSN was a premium service (onetime fee) that offered access to MSN services (ie. Hotmail, Messenger) on Windows Mobile devices in 2005.

In 2008, Microsoft provided an e-mail message to Pocket MSN users  on how to upgrade access to Hotmail and Messenger.

 

A user may continue to have access to Hotmail and Messenger on Windows Mobile device by the following the instructions below:

1.       On the device use Internet Explorer to browse http://wl.windowsmobile.com

2.       The best language should be detected automatically (so you should land on the Korean page if your device is set to Korean language).

3.       For a non-touch phone like the Blackjack, you should be presented with the standard download.

4.       Download and install the cab.

 

If this option does not work, another option is to access Windows Live Browse for Windows Live mobile services. Access Windows Live Browse from your mobile internet browser:  http://mobile.live.com
