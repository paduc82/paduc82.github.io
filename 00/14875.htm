ericyang_msft6286 say:

Hi Mike,

Since the mechanism of notebooks sharing in Office 365 for Business is different from personal OneNote, it is not able to view the shared notebooks shared in this way. After sharing a notebook in this way, the user will receive a link to the notebook and the link need to be opened via a web browser, and view or edit the notebook in OneNote Online. In this case, a workaround for the users to access the notebook shared from Office 365 for Business on a mobile phone is to open the link in a web browser and access the notebooks via OneNote Online (you may need to run the browser under PC view).
