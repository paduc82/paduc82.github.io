scottgem say:

I don't think Tony's class module provides the tracking functions. You can easily create a log of when users open and close your apps with the instructions in my blog. Create a table like so:

tblAccessLog

AccessLogID (PK Autonumber)

AccessWhen

AccessBy

AccessTypeID (FK)

You can then use a sub like the following to record your activities:

Public Sub pActivityLog(lngUser As Long, lngActivty As Long)

Dim strSQL As String

strSQL = "INSERT INTO tblAccessLog (AccessBy, AccessWhen, AccessTypeID) " & _

               "VALUES(" & lngUser & ", #" & Now() & "#, " & lngActivity & ");"

CurrentDB.Execute strSQL, dbFailOnError

End Sub

That code would go in a global module. You would CALL it whenever you want to record an Activity. Your activities would go in a lookup table of Activity types. For example 1 would be Login, 2 would be Logout. 

So you would add a line in the login code just before you open the main form:

CALL pActivityLog(Forms!frmLogin!cboUser, 1)

That would add a record to log indicating when the user logged in. Put a similar line in the On Close event of frmLogin only use 2 instead of 1 for the type. You can use the same function with differing Activity Types to record other activities.

andreas killer say:

That context menu is build with code, more or less like the sample below.

Andreas.

Sub InstallKontextButtons()
  'Buttons zum Kopieren im Kontextmenü erstellen
  Dim CB As CommandBar
  Dim Ctrl As CommandBarButton, SubMenu As CommandBarPopup
  Dim Position As Integer, B

  On Error Resume Next
  For Each CB In Application.CommandBars
    For Each B In Array("List Range Popup", "Cell", "Row", "Column")
      If CB.Name = B Then
        With CB
          Position = .FindControl(ID:=19).Index
          Set Ctrl = .Controls.Add(Type:=msoControlButton, Temporary:=True)
          With Ctrl
            .Move Before:=Position + 1
            .Caption = "Formeln kopieren"
            .OnAction = "'" & ThisWorkbook.Name & "'!SelectionCopyFormula"
            .FaceId = 624
          End With

          Position = .FindControl(ID:=22).Index
          Set Ctrl = .Controls.Add(Type:=msoControlButton, Temporary:=True)
          With Ctrl
            .Move Before:=Position + 1
            .Caption = "Formeln einfügen"
            .OnAction = "'" & ThisWorkbook.Name & "'!SelectionPasteFormula"
            .FaceId = 625
          End With
        End With
        Exit For
      End If
    Next
  Next
End Sub
