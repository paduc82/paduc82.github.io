Lenovo desktop turns itself back on after upgrading to Windows 10

Hi

Thought I would share my experience after successfully upgrading my Lenovo desktop to Windows 10. I found that even though I was selecting the Shut Down option I would come back and find my desktop had restarted itself.

I did some searching around online and found out this is a common problem across Windows 10 and in my case my Wifi driver was set to automatically start my desktop if prompted by the wifi.

I disabled this option and hey presto no more desktop starting all on its own.

To resolve this I went to my Control Panel, then Network, then right clicked on the Wifi option to bring up the sub menu and selected Properties. I then selected the Configure button on the dialog box that appears and finally selected the Power Management tab and unticked the option that "Allow the device to wake the computer". If you use an Ethernet cable you may to to repeat these steps for that network option too.

Guess one for Microsoft to fix in a future patch as I never had this issue when the desktop ran Windows 8.1

anyway hope it helps anyone having similar problems.
