jim gordon mvp say:

Apply Name does not do anything to the cell you manually named. It changes cell references that already existed to that cell before you named it.

Here's an example of how it works (similar to your example).

1. Open a New workbook

2. Add a new worksheet (Sheet 2)

3. In Sheet 1, Cell A1 enter this formula =Sheet2!A1*Sheet2!B1

4. In cell A7 enter =A1

5. Select cell A1 and give it a name, for example MyCell

6. Select cell A7. Notice the formula still says =A1

7. While cell A7 is selected, use Insert > Name > Apply and click OK

8. Notice the cell formula in cell A7 changed from =A! to =MyCell

There's nothing wrong with Apply. You're just using it incorrectly.
