greg maxey say:

You can do it with a macro:

Sub ScratchMacro()
'A basic Word macro coded by Greg Maxey
Dim oCC As ContentControl
Dim oRng As Word.Range
Dim lngIndex As Long
'Assumes your dropdown is named DDList.
Set oCC = ActiveDocument.SelectContentControlsByTitle("DDList").Item(1)
Set oRng = Selection.Range
oCC.DropdownListEntries.Clear
'Assumes that you have a list selected in the document _
and each list entry starts a new paragraph.
For lngIndex = 1 To oRng.Paragraphs.Count
  oCC.DropdownListEntries.Add Left(oRng.Paragraphs(lngIndex).Range.Text, _
                         Len(oRng.Paragraphs(lngIndex).Range.Text) - 1)
Next lngIndex
End Sub

Difference between 3G and Dual Mode?

When I go through cellular network options, I get 3 options as GSM, 3G and Dual Mode. What is the difference between the Dual Mode and 3G, and what exactly is this Dual Mode? When should I choose that over 3G?

emceedjcd say:

Dual mode means that the phone will connect to either 2G (GSM) or 3G connections. It will try for a 3G connection if it's strong enough, otherwise, it will go to a 2G connection.

 

The difference between 3G and dual mode, is in 3G mode, the phone will only try to connect to a 3G connection.

 

In most cases, dual would be better. If you are in an area will really strong and constitent 3G, then it's probably better to be on 3G.

Hands-on with 10125 - Microsoft Community

a. user say:

Hands-on with 10125

So I was unhappy enough with 10122, that I figured the leak of 10125 couldn't possibly be any worse.  I'm not afraid to say it--10125 is the build that we should've gotten on the Fast ring instead of 10122.

Initial reactions from me:

The good:

  • the lock screen time format matches the user-set system time again
  • if you disable the requirement to sign in, you're not prompted for a password anyway like you were in 10122
  • the Aero blur effect is now present on the Notifications flyout (I don't recall seeing it in 10122)
  • New icons spread here and there
  • Another new sound effects package
  • Surface Pro 3 HD graphics drivers seem to have fewer problems and require fewer forced restarts
  • 10125 (and seemingly 10122 before it) are FAR more stable than 10074 was

The bad:

  • Project Spartan/Edge still can't seem to handle PDF markup properly (if I open a PDF and then add markup, I can't then return to the PDF and scroll down to a different part to do additional markup)
  • Aero blur is still missing from the title bars
  • Metro apps appear in small windows most of the time (part of a planned tweak to Tablet Mode, I can only assume this is still in development)
  • the restart cycle hangs sometimes on my Surface Pro 3
  • There seems to be some issues with default app assignments; out of the box, even an upgrade to 10125 sets Spartan as the default Web browser, while I prefer Firefox--yet, every time Firefox opens, when it asks if I want to use it as the default browser and I say yes, instead of assigning it to the default slot, Windows instead pops up a dialog box requiring that I go into Settings and change it manually

The odd:

  • I've noticed a new "cast to device" option in one of the context menus, this is something I'm curious about, particularly on my Surface
  • Maybe I got spoiled using Windows 8, but I really miss the options that it had for colors--I could pick a particular shade of purple, but then I could also adjust its brightness, its saturation, and so forth; Windows 10's own color selection is very limited, and I can't say I'm particularly fond of any of the colors, since the palette also seems to lack customization options
  • Following the above, after I inadvertently made a change to the color scheme of my Windows 10 desktop on one machine in 10122, that change propagated across my entire family of post-Windows 7 machines; my answer was to reset the theme in Windows 8 (which resides on my Surface), and that change did seem to sync to the Windows 10 install on the Surface...but it did not sync to the other computer.  I find that strange.

While still not quite perfect, for both the tablet and desktop PC experiences at least, 10125 represents a major step in the right direction.  Still some work to be done, but I still wanted to share some thoughts for anyone who was as frustrated with 10122 as I was.
