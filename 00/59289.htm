srfreeman say:

The inability to update is simply a symptom of the underlying system error. Addressing the individual update or the update process in general will do you no good at all. You must address the system error directly.

The 0x80073712 - ERROR_SXS_COMPONENT_STORE_CORRUPT error code is often thrown as a result of Windows system failure. Broken system entries are often a substantial danger to the well being of your pc.
 
There are several situations that will have resulted in a system file error. An unfinished install, a partial file deletion, improper erasure of utilities or components. It could also be triggered when a pc is infected by a pc virus or adware infection or through an improper shutdown of the computer. Any one of the previously mentioned activities could very well end up in the erasure or corruption of Windows system files. This corrupted system file will cause absent or incorrectly linked documents and files required for the proper working of the software.

It is well known that Windows will not update properly when this system error is present.

Simply inserting the error code as a key word in your favorite search engine will provide you with the causes of this error and how users before you have fixed this issue.

So much has been previously documented on the noted error, rehashing it in this forum would simply be a waste of typing.

Edit:

Given the nature of the error... If you do not want to search out the reason it has happened, an in-place upgrade may fix it. If an in-place upgrade does not do it for you, a clean install of Windows would certainly fix you up.

Be careful, if you do not find what caused the missing/damaged files, it is very likely that you will continue the same practices that caused the problem and it will happen again.

srfreeman say:

An in-place upgrade is simply upgrading the OS to the currently running version of the OS. Strictly a maintenance operation. With Windows 10 it easily done by using the 'Upgrade this PC' selection in the Media Creation Tool or running setup.exe in the media created by the same tool, from within a running copy of Windows 10. It refreshes the system files without affecting installed apps or client data.

A clean install of Windows can be more effective, since it removes any problems caused by installed apps or client data. Yes, failed installation or incomplete removal of applications can cause corruption in the SXS component. Applications and data can also exist in problematic areas of the drive that are not exposed without a clean install.

It is just a guess, but, the disk problems are probably the culprit in your case. Disk problems never go away, you have only masked them from the system through the software used. Corruption caused by such things just gets worse over time.

andre da costa say:

Windows 10 Activation reports your license will soon expire.

This can happen for a number of reasons:

  • You copied a Windows 10 Pro off the Internet (usually a volume license - organization - key) and changed on a Windows 10 Home installation. Windows 10 Pro is not free. If this is what you did, you will need to reinstall the previous edition or version you were running. You inadvertently installed the wrong edition.
    • The copy of Windows 10 you download and upgrade from will correspond with the edition of Windows you have installed, if it does not, this is why you might be experiencing problems activating:

  • Windows 7 Starter, Home Basic, Home Premium, Windows 8.0 Core, Windows 8.1 Core will install Windows 10 Home

  • Windows 7 Professional, Windows 7 Ultimate, Windows 8.0 Pro, Windows 8.1 Pro will install Windows 10 Pro

To find out if you have the right edition installed:

Click Start > Settings > Update & security > Activation > Edition.

If the edition of Windows 10 installed does not correspond with the edition of Windows 7 or Windows 8 you were previously running, then you need to reinstall the right edition.

Windows 10 came preinstalled on my computer, how do I find the product key?

Press Windows key + X

Click Command Prompt (admin)

Enter the following command:

wmic path SoftwareLicensingService get OA3xOriginalProductKey

Hit Enter

The product key will be revealed, copy the product key then enter it

Click Start > Settings > Update & security > Activation > Change the product key then enter the new product key then click Next to Activate

If you are unable to activate Windows 10 after all the above and you are certain you have the correct edition installed, then please attempt phone activation:

Activate Your Windows 10 License via Microsoft Chat Support

If this does not work, you need to contact the supplier.

a. user say:

Hi Malik,

Are you referring to the user credentials used to access shared resources (files or folders shared on other computers) within a home network or a workgroup?

If yes, then, you can uncheck the box that says “Remember my credential” box when you try to access a shared network for the first time as you will be asked to enter the User name and the password.

Otherwise, you can also disable the setting to make Windows Credential manager not to remember the credentials for the network authentication using Group Policy. Follow the steps:

  • Press Windows+R keys and type “gpedit.msc”. Hit Enter.
  • Navigate to the following location:
  • Computer Configuration\Windows Settings\Security Settings\Local Policies\Security Options
  • Double Click on the option “Network access: Do not allow storage of passwords and credentials for network authentication”.
  • Select the radio button which says “Enabled”. By default, disabled button will be selected which means Credential Manager will store passwords and credentials on this computer for later use for domain authentication.

For reference, you can refer to the link:

http://technet.microsoft.com/en-us/library/jj852185.aspx
