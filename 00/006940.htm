matthewmaude say:

shortcuts in x term (alias)

I've brought this up before but never really perfected it,
i've spent a bit of time on it now and its very useful and quite easy to do.

Firstly the shortcuts - 'alias' is similar to macro's in windows. Its using a letter, series of letters/numbers or a word to perform a command in the shell (X Terminal) for example to become root i just run 'r' or to run apt-get update i run 'ud'
you can customize these and add your own as you want, just don't use pre built commands like rm, mv and cp

There are 2 locations you need to save these
/home/user/.profile and /root/.bashrc (can be saved to /root/.profile also)
all the root commands need to be done as 'root' and not 'sudo gainroot' as sudo gainroot doesn't read .profile or .bashrc
 
you will need leafpad installed for these instructions

the first place if for anything that doesn't need root access
open xterm and run

leafpad .profile

copy/paste these commands to it and save it

alias r='root'
alias sr='sudo gainroot'
alias e='exit'
alias ls='ls -a'
alias u='cd ..'
alias h='cd /home/user/MyDocs'
alias lp='leafpad'
alias t='/usr/bin/osso-xterm ""'

then run

root
leafpad .bashrc

scroll down to the bottom of the file and copy/paste these commands to it and save it, then close xterm and when you re open it all the shortcuts will work

cpitchford say: Beware messing up your files!

If you screw up your .profile or .bashrc you run the risk of not being able to log in, or open a login session..

How to fix.. If you have ssh support installed, you could issue a command via SSH to your phone to edit the file:

ssh user@phone vi .bashrc
or if root
ssh root@phone vi .bashrc

Edit your bashrc or .profile and add this line:

[ -r /home/user/MyDocs/.aliases ] && . /home/users/MyDocs/aliases

(all on one line and be VERY careful with spaces, they've been exaggerate here to show you where they are)

Then edit MyDocs/.aliases

alias something=doesomestuff

Advantage, if you mess things up, whack your phone into a PC USB mode and delete the .aliases file! Things return to normal!

a. user say:

Hello RashmiRudrapatna:

 

Thank you for posting to the forum regarding the issue you are having syncing your Microsoft account to the phone.  Error code 8501001F tends to be related to either the account itself or your connection.  We have a couple things we would like you to try and some additional questions.

 

The first thing we would like you to do is go to accounts.live.com and sign into the account you are having issues syncing.  Once you are signed in, go to account settings.  Please go to personal info and make sure that the details listed are up to date and correct.  Once this is done, change the password to one that is less then 13 characters in length.  Once you have done this, go to the phone and tap settings > emails + accounts > tap the account > scroll to the bottom and tap password > enter the newly created password.  If you have any trouble performing these steps, please detail them for us in your next reply.

 

The next thing we would like you to do is connect to another network.  We would like you to try syncing the account in all three of the following ways:

  • Please connect the phone strictly via your cellular data connection.  To turn off Wi-Fi, go to settings > Wi-Fi > OFF.  To make sure the data connection is on, go to settings > cellular/mobile > ON.  Go to the emails + accounts and tap and hold the account then select sync.
  • Next, try connecting the phone via just Wi-Fi.  Do the opposite of above, turning Wi-Fi on and connecting to a listed network and then turning off cellular.
  • Finally, go to another location and connect to a different network.  Find a public hotspot or a friend/family members network.

Some additional information we would like to gather if the issue continues:

  • What is the domain associated with the primary account on your phone?
  • What country are you located in?
  • What is the error message that comes along with error code 8501001F?
  • Are you able to add other accounts to the phone, secondary Microsoft accounts, Facebook, Twitter, etc...?

Please let us know if you have any additional questions or concerns we can assist you with.

 

Mark W.
