bob jones aka: cybertaz say:

There's no explanation of how you went about it, but one likely reason for the result shown in your screen shot is that the cells actually are not formatted as Text. If they have General or Number formatting it isn't Excel that will not differentiate, it's the laws of mathematics :-) Trailing zeros in decimals are considered insignificant numbers, so when dealing with numerical values there is no distinction between "14.1" & "14.10"... or "14.100" or "14.100000000". As Rajesh C. suggested, the extraneous zeros merely are a matter of formatting, not an integral part of the value.

This screen shot illustrates that it is possible to do what you describe, but the numeric characters have to be (a) formatted as Text & (b) typed exactly as you want them to display.

These are the steps I took order to create what's shown after entering the data & formatting as Text. (I also changed the Alignment to Right & used the Error Alert button to Ignore Error).

  1. Select cell G1
  2. Apply Conditional Formatting> Highlight Cell Rules> Equal To
    1. Accept all defaults, with Cell Value - equal to - H1
    2. Click OK
  3. Go to Conditional Formatting> Manage Rules
    1. Delete the 2 $s in the Applies to column
    2. Click OK
  4. Copy G1
  5. Select G2:G7
  6. Go to Edit> Paste Special
    1. Select the Formats option
    2. Click OK
  7. Repeat for the cells in Column H but specify G1 in step 2.1 above

Note that the CF does not apply to the cells containing "16" .

    Using Alias with Outlook IMAP & Gmail - Microsoft Community

    michellegrove say:

    Using Alias with Outlook IMAP & Gmail

    I just moved all of somebody's email to Gmail and then accessed it in Outlook using IMAP. How can we set it up to send messages from the alias email in Outlook? 
    brian tillman [outlook mvp 2007-2019] say:
    In order to use an alias, set up another account in Outlook that uses the alias address as sender, but logs into the outgoing server using the main address.  Configure the send/receive group entry for the account so that it doesn't accept incoming mail, only sends outgoing mail.  You can use any value for the incoming server you'd like, since it doesn't have to be real and won't be used.  Also, uncheck the box that indicates the alias account should be included in the send/receive group.  Once this is done, you'll be able to choose that account with the From button to have the outgoing message come from the alias.

    To permanently set the zoom level

     

    You can permanently set the default zoom in Internet Explorer 10 to a custom level if you want to.
     
    To do this, follow these steps:
     
    1. Click Start, type regedit in the search box, and then click regedit in the Programs list.
    2. Locate and then click the following registry subkey:
     
    HKEY_CURRENT_USERS\SOFTWARE\Microsoft\Internet Explorer\Zoom
     
    3. In the details pane, double-click ZoomFactor.
    4. Click Decimal
    5. If you want to set the default zoom level to 150%, type 150000, and then click OK.
    6. Close Registry Editor.
    7. Log off of Windows and then log on again.
    8. Open Internet Explorer. The zoom level should now be set to 150%.
