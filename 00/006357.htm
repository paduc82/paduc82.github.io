Tom,

 

Thanks. The formula I posted won't ignore error cells in Col C; or any other column for that matter, for that we would need to use the formula below, now I'm no mathmatician but I have a problem with that.

If; say, A2 & B2 meet the formula criteria and C2 is an error value then we don't get a true standard deviation for all rows that meet the criteria, instead that row is ignored.

I would have no difficulty in using a formula that ignores error values in Col A & Col B because they don't meet the criteria but IMHO what the OP should do is arrange column C such that if an error value is returned it's trapped with IFERROR to make it return zero and we get a true SD for all rows that meet the criteria

 

As I said I'm no mathmatecian but that's how I'd do it.

 

=STDEV(IF(A2:A7="Red",IF(B2:B7>DATE(2011,1,1),IF(ISNUMBER(C2:C7),C2:C7))))
