Tools (cog icon) > Internet Options > Advanced tab > Reset...

After clearing all settings make sure to close IE and Outlook before checking.

If the problem comes back then, like us, your Group Policy might be causing the issue.  The setting that fixed if for us was:

Tools (cog icon) > Internet Options > Advanced tab > Settings

Under Security there was a check next to "Do not save encrypted pages to disk".

jay freedman say:

The Remove choice is grayed out when the button is contained in a template add-in (or possibly a COM add-in, but that's less common).

Close Word, and use Windows Explorer to go to the folder %appdata%\Microsoft\Word\Startup. If there are any *.dot, *.dotx, or *.dotm files there, move them to another folder. Restart Word, and the Quick Access Toolbar button should be gone.

Now you can start this process to pinpoint the one add-in that supplies the button: Move one template file back into the Startup folder and restart Word. The one template that causes the button to reappear is the culprit. You can decide whether that add-in is useful enough to keep or should be uninstalled. Then add back the rest of the add-in templates.
