Now Onenote notebooks get a special treatment to guarantee that only the Onenote client or the Web App synch these notebooks.

vanessa ouyang msft say:

Hi Wendy,

I do understand your experience. It is an expected behavior since subfolder belongs to Inbox folder. When we search something in Inbox folder, this detection will include subfolder no matter how long.

If you move emails from one subfolder to another subfolder, it will have a short while when you are using Outlook cache mode. Cached Outlook Mode optimizes the type and amount of data that is sent over a connection with the server. When emails which have been in your Outlook client are moved to another folder, there is a delay for about several seconds.

If it is inconvenient for you, you can turn off Cached mode referring to this article:

https://support.office.com/en-us/article/Turn-on-Cached-Exchange-Mode-7885af08-9a60-4ec3-850a-e221c1ed0c1c.

However, in this way, it will have a delay when you look over emails because all emails will be downloaded from server.

mike h.. say:

Hi,

STDEV will ignore blank cells but in this ARRAY format it wont so we need an additional check for blanks.

=STDEV(IF(NOT(ISNA(A1:A100)),IF(A1:A100<>"",A1:A100,"")))

This is an array formula which must be entered by pressing CTRL+Shift+Enter
and not just Enter. If you do it correctly then Excel will put curly brackets
around the formula {}. You can't type these yourself. If you edit the formula
you must enter it again with CTRL+Shift+Enter.
