Another flub from the hapless "Microsoft Forum Administrator."

Here are two methods you can try:

Method #1:

Find a recovery disc or a recovery USB flash drive (which I'll call recovery media for simplicity.) This can be any recovery media that came with one of your programs, e.g., your antivirus program. It doesn't even have to be from one of your own programs - it can be from a friend's program - and it doesn't even need to be so recent. What you're looking for is anything that can (1) boot your computer and (2) has a file manager similar to Windows' File Explorer.

Because you're booting from the recovery media, and not your computer, it doesn't matter how messed up your computer is. The recovery media bypasses your computer. As long as the recovery media's file manager can read an NTFS-formatted hard drive, and most can, it can 'see' your files.

Attach an external USB hard drive to your computer. Attach it to a USB port on the computer itself. You can do this before or after you've booted from the recovery media. Using the file manager in your recovery media, you can copy and paste your files from the toasted computer to the external USB drive. You're doing a total end run around toasty PC, so it doesn't matter how dark the toast it. Once you've copied everything - don't forget anything; perhaps prepare a list in advance - then you can do a clean install of Windows on toasty and restore your data later.

Tip (don't ask why I know this:) Make sure that you are copying actual files and folders and not instead creating shortcuts to files and folders! It's a good idea to attach the external USB drive to another computer to check that you can actually open your copied files and folders.

Method #2:

The idea here is basically the same as in Method #1 - to do an end run around the damaged computer. Remove the hard drive from the damaged computer and put it into another, known good computer as a secondary hard drive. Use Windows 10's File Explorer in the good computer to copy your files and folders to an external USB drive. Same method (and same caveat) as above.

Method #2A:

Okay, now we're starting to get fancy. Suppose your buddy isn't keen on the idea of installing your hard drive in their computer. You have to respect that. In that case you'll need (to probably buy) a SATA-to-USB adapter like this one, with which I am only too familiar. When you attach your hard drive to the SATA connector and attach the USB connector to your buddy's computer, it's just like attaching an external USB drive to the computer.

There are numerous options built into Windows to help you get out of this predicament:
  • Reboot the machine with your Windows Repair CD, then use System Restore to set Windows back to a point when your password still worked.
  • Log on under your spare admin account, then reset the password to your usual account (works only if you set up a spare admin account).
  • Log on as Administrator in Safe Mode, then reset the password to your usual account (the password is usually blank).
  • Insert your USB flash disk with the password keyfile (works only if you took the time to create such a keyfile).
  • Use the password hint (works only if you created a hint and if it is good enough to let you guess the password)
  • Check your paper records (works only if you jotted down the password somewhere)
  • Ask your friendly computer dealer to reset the password for you.

When finished, plan ahead a little and create, test and document a spare admin account, same as you have a spare house key. This will save you an enormous amount of trouble.

Q. If my customer asks me to upgrade a PC with new hardware components, when is a new operating system needed? When would the PC be considered "new"?
