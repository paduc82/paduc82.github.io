A small box comes up that says "Personalized Settings (Not Responding)"


Interesting.   The "black screen" part of your symptom could mean that  explorer.exe  has crashed and not been able to restart.   I don't know why you might then see this second symptom, except that it sounds like something which could be a part of explorer.exe recovery procedure, e.g. it reminds me of a procedure that came up automatically never to be seen again when W8 was first installed.

 

In any case, while you have a black screen you could try starting Task Manager and use it to start other things.   First though, it would be useful to differentiate about whether you have a "black" screen or a "blank" screen.   E.g. if there is a fringe of light around the edges and if it is just quite dark but not as dark as it would be if powered off, "blank" screen might be a more accurate description.   Then you might be able to see things if they were added to it, which would be less likely if it was black.   For example, can you move your mouse and see a pointer moving across the darkness?   Then I think that would be an example of a "blank" screen.

 

Then, to start Task Manager you would press Ctrl+Alt+Delete and use the Task Manager item in the menu that keyboard shortcut would bring up.   Then you could use its Prcesses tab to see if  Windows Explorer  appeared there.   If not you could use TM's File, Run... command to start it by entering there:  explorer.exe

 

However, another possibility that some people have been seeing is their primary screen inexplicably switched to a phantom second monitor.   Then pressing Win-P, CursorUp, and Enter  to try changing the Second Screen setting may be a simpler more effective procedure to try to get something showing on the monitor that you are looking at.

A ISO that is copy\pasted to a Flash Drive will not normally be bootable... The files on it can be used for a "Upgrade" running from within Windows, but not a Clean Install that requires booting...

If you still have the ISO you can put it on a Flash Drive that will be bootable with a free tool called RUFUS... There are multiple options to do this, so I included a screenshot as to what the settings should look like for a UEFI Bios machine...

Just put in the Flash Drive and Browse to the ISO... Select "GPT"... Note that this ISO is for a older version of Windows 10, but the process is the same...
