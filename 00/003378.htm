    seliaks say: If you have the issue of an old Skype account that is not linked to a Microsoft account, and another one that you have because it is associated with a newer Microsoft account, it is possible to merge the two accounts - log into Skype on the web using the credentials for the non-linked account/older account, go into 'account settings' and there you will see the option to connect a Microsoft account - by choosing this option you will merge the two Skype profiles together. The only downside is that the chat/history of the older Skype profile is retained, and any stuff in the newer account is lost - but at least you've got rid of the duplicate account. It may be possible to do the process from the Microsoft account linked profiles 'account settings' and retain the chat/history of that account/profile instead, but I didn't try that.
