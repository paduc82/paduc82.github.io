Hello,

Does anyone know where to put "Temporary Internet Files"  folder for best performance and speed? SSD? or HDD?

Mandeep

tc

jetta48 say:

Hi

When a system has both a SSD and HDD it is usual to have the Windows installation (C:) on the SSD. This is because the read/write speeds on the SSD are faster and all the system files that frequently get accessed are there.

When you first visit a web page it will normally be stored in your Temporary Internet Files. If you visit the same page again (and that page has not been updated on the web server), rather than downloading the page again over the internet your browser will instead access the page from your local drive because it is quicker to do so. The faster your local drive, the faster these things will potentially happen. Temporary Internet Files, by default are set to be stored on the Windows installation (C:) drive. Personally, I'd keep the Temporary Internet Files on this drive.

The capacity (GB) of SSDs is generally smaller than HDDs so make sure that you have ample free space on the drive at all times. Also note that performance can depend on lots of factors.
