bob jones aka: cybertaz say:

The color of Hyperlinks & Followed Hyperlinks is controlled by the Office Theme applied to the presentation. Accordingly, the easiest ways to use different colors is to select a different Theme or a different set of Theme Colors. If those options aren't satisfactory you'll need to create Custom Colors. I don't know of a way to remove the underlining, though.

One way to create a custom color scheme is to got to Format> Theme Colors..., then change the colors you want by clicking the corresponding swatches. You also can give the custom color scheme a name & save it to be applied in other presentations if you wish.

There is no specific Typewriter Animation. It can be simulated by using the Appear Entrance Effect & setting its Text Animations property to Animate Text: By Letter. Additionally, you can select Typewriter from the Sound: choices in the Effect Options properties. Adjust the Duration to your liking. [Note that the same settings can be used for animation effects other than Appear so you might want to experiment a bit.]
