vba code to move cell data based on criteria - Microsoft Community

richb330 say:

vba code to move cell data based on criteria

Hi,
I'm looking for a simple way to scan through a column and if data exists that starts with the numbers '07' then move this to the cell to the right but ignore everything else.

I there a simple way to do this?

Thanks
jllatham say:

Try this:

 

Sub MoveToRight()
'see this page for instructions on inserting this
'code into a workbook:
' http://www.contextures.com/xlvba01.html#Regular

  'change the values of these Const as required
  'name of the sheet to work with
  Const wsName = "Sheet1"
  'which column are we to examine?
  Const dataCol = "A"
  'what is the first row with data to examine?
  Const firstRow = 2
 
  Dim myWS As Worksheet
  Dim testRange As Range
  Dim testCell As Range
 
  Set myWS = ThisWorkbook.Worksheets(wsName)
  Set testRange = myWS.Range(dataCol & firstRow & ":" & _
   myWS.Range(dataCol & Rows.Count).End(xlUp).Address)
  Application.ScreenUpdating = False
  For Each testCell In testRange
    If Left(testCell.Text, 2) = "07" Then
      'copy to column to the right
      testCell.Offset(0, 1) = testCell
      'clear out of original cell
      testCell.ClearContents
    End If
  Next
  'cleanup
  Set testRange = Nothing
  Set myWS = Nothing
End Sub

Try this

 

Sub Move07()
  Dim c As Variant
  Dim str As String
 
  str = "07"
  For Each c In Selection
    If Mid(c, 1, 2) = str Then
      c.Offset(0, 1) = c
      c.Clear
    End If
  Next c
End Sub
