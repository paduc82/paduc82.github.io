- Press the keyboard combination :  Option+Command+p



The macro which will be executed is this one (if you're interested):

Sub pa()
Dim cell As Range
Dim tmp As String
Dim tmp2 As Integer, counter As Integer
For Each cell In Selection.Cells
    If cell.Value <> "" Then
    tmp = cell.Value
    If InStr(tmp, Chr(13)) <> 0 Then
        cell.Offset(0, 1).Value = Left(tmp, InStr(tmp, Chr(13)) - 1)
        tmp = Right(tmp, Len(tmp) - InStr(tmp, Chr(13)))
    End If
    counter = 0
    For tmp2 = Len(tmp) To 1 Step -1
        If Mid(tmp, tmp2, 1) = " " Then
            counter = counter + 1
            Select Case counter
                Case 1
                    cell.Offset(0, 4).Value = Right(tmp, Len(tmp) - tmp2)
                    tmp = Left(tmp, tmp2 - 1)
                Case 2
                    cell.Offset(0, 3).Value = Right(tmp, Len(tmp) - tmp2)
                    cell.Offset(0, 2).Value = Left(tmp, tmp2)
                    Exit For
            End Select
        End If
    Next
    End If

Get Current Archival storage usage report for all users - Microsoft Community

a. user say:

Get Current Archival storage usage report for all users

Hi all ,

Anyone know how to generate report to see all users current archival storage usage ? I know we can use portal to view one by one . But for archival storage , anyone know how to do it ?

lancecai_msft2205 say:

hi cs khor,

about the users current archival storage usage. i assume that you mean the archiving in eac.

to get all users archive mailbox storage usage, you need to involve some powershell scripts.the script below is used to find mailbox and online archive size. 

1.connect to exchange online using remote powershell via office 365 admin account.

2, run the following commands:

a. $mailboxes = get-mailbox -resultsize unlimited | select userprincipalname, identity, archivestatus

b. $mailboxsizes = @()

c. foreach ($mailbox in $mailboxes) {

$objproperties = new-object psobject

$mailboxstats = get-mailboxstatistics $mailbox.userprincipalname | select totalitemsize, itemcount

add-member -inputobject $objproperties -membertype noteproperty -name "userprincipalname" -value $mailbox.userprincipalname
add-member -inputobject $objproperties -membertype noteproperty -name "mailbox size" -value $mailboxstats.totalitemsize
add-member -inputobject $objproperties -membertype noteproperty -name "mailbox item count" -value $mailboxstats.itemcount

if ($mailbox.archivestatus -eq "active") {

$archivestats = get-mailboxstatistics $mailbox.userprincipalname -archive | select totalitemsize, itemcount

add-member -inputobject $objproperties -membertype noteproperty -name "archive size" -value $archivestats.totalitemsize
add-member -inputobject $objproperties -membertype noteproperty -name "archive item count" -value $archivestats.itemcount

}
else {

add-member -inputobject $objproperties -membertype noteproperty -name "archive size" -value "no archive"
add-member -inputobject $objproperties -membertype noteproperty -name "archive item count" -value "no archive"

}

$mailboxsizes += $objproperties

}

d. $mailboxsizes | out-gridview -title "mailbox and archive sizes"

then a window will pop out with the report of the archive storage usage.

if you want to export the report, replace command d to $mailboxsizes | export-csv c:\filename.csv

this command will export a csv file named filename.csv to the path you typed in. here is c drive for an example.

if you encounter any question about the script or the script does not meet your requirement, i recommend you contact our script center.

thanks for your effort and understanding,
