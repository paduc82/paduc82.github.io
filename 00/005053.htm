john sr wilson say:

I don't think there's a non code way to do this.

If you know how to add and run code this would produce a printable list which should help. Don't forget to change the target folder path.

Sub getProperties()
   Dim rayList() As String
   Dim strFolderPath As String
   Dim strFileSpec As String
   Dim strTemp As String
   Dim x As Long
   Dim ifile As Integer
   Dim strReport As String
   Dim strReportPath As String
   strReportPath = Environ("USERPROFILE") & "\Desktop\report.txt"
   ifile = FreeFile
    ' Note: MUST end in \ - change path to suit
   strFolderPath = "c:\users\optiplex\desktop\testme\"
   strFileSpec = "*.ppt*"      'ppt pptx pptm

   ReDim rayList(1 To 1) As String
   strTemp = Dir$(strFolderPath & strFileSpec)
   While strTemp <> ""
      rayList(UBound(rayList)) = strFolderPath & strTemp
      ReDim Preserve rayList(1 To UBound(rayList) + 1) As String
      strTemp = Dir
   Wend
   ReDim Preserve rayList(1 To UBound(rayList) - 1)
   If UBound(rayList) > 1 Then
      For x = 1 To UBound(rayList)
         strReport = strReport & getreport(rayList(x)) & vbCrLf
      Next x
   End If
   Open strReportPath For Output As ifile
   Print #ifile, "File Name" & vbTab & "Format"
    Print #ifile, "=========" & vbTab & "======"
   Print #ifile, strReport
   Close ifile
   Call Shell("NOTEPAD.EXE " & strReportPath, vbNormalFocus)
End Sub

Function getreport(myFile As String) As String
   Dim oPres As Presentation
   Set oPres = Presentations.Open(FileName:=myFile, WithWindow:=False)
   With oPres
      getreport = oPres.Name & vbTab & oPres.BuiltInDocumentProperties("Format")
   End With
   oPres.Close
End Function

pamcaswell say:

After document creation, templates don't contribute styles to the  unless you tell it toby going to Developer tab > Templates group > Document template and putting a check in the box before Automatically update template and clicking OK.  Immediately after you've done that, go back into the dialog and clear that check box.

The action will impose the template's style settings on the document's styles of the same name and insert new styles that are in the template but not in the document.  But it will not delete styles from the document.

Oh, yes, to get the Developer tab in the W2010 ribbon go to Word options > Customize Ribbon  > Main (or all) tabs section, and put a checkmark in the box before Customize.

HTH,

Pam

stefan blom say:

"Automatically update document styles" is a document-specific setting; you would have to enable it for each document where you want the styles to be updated by the template.

The setting may be unreliable with numbered styles, which is one reason why it isn't activated by default. More importantly, though, I suspect that users expect to change styles within a document, irrespective of the settings in the attached template.

Microsoft Word MVP
