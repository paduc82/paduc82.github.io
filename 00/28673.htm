Unable to install Jar Files on E71
Category: Uncategorized / Tags: no tag / Add Comment
You can definitely install jar files to the Nokia E71. However, if – you are like me – you keep on installing and uninstalling jar applications just to test them out, sooner or later you might encounter an error that won’t allow you to install jar apps anymore. Basically, what happens is, when you try installing a jar app, you will be able to select where to install, etc… then the installation bar would begin. However, when that bar reaches about 90%, your installation hangs, up then a warning sign will say “Unable to Install”, then the installation terminates.
Don’t panic like I did. You don’t have to reformat your phone or throw it to the wall. You can still install sis or sisx applications despitethe jar installation error.
All you need to do to fix this is to grab a copy of an application called Xplore.. and after installing said application, run it and choose tools>configuration from its menu. From there, you have to check the boxes that says “Show Hidden Files” and “Show system files/folders”, then click the “Back” soft key. This will bring you to your directories.
The thing that we need to do is to trace if any of the current installations on your phone has gone wrong. To do this, we need to see the registry on both your Phone Drive or your Memory Drive. This can be access by going to Drive:\System\Install\Registry. By going there, you will be able to see files that have a .reg ending. You need to go through each one of these types of file and see if there is something wrong with them.
The easiest way to do this is to select the .reg file, then click Menu>File>Hex Viewer. Once opened, you can see some binary inputson the middle. This would tell you that you have successfully viewed the insides of the .reg file. Focus your attention to the right side of the file/screen. You have to be able to see the name of the application within it (therefore, you have to be aware what applications you currently have installed on your device). If you cannot understand anything on the right side, try scrolling down. If still you cannot understand anything (if unreadable characters are present), then delete this particular .reg file.
Sometimes, when you try opening a .reg file for viewing, and it does not open on the Hex Viewer, this would also tell you that that is an error .reg file. So just delete it also.
Once you are done viewing all your .reg file, close XPlore and try installing a jar app. You should now be able to do so.
Please note that changing/deleting anything on your Registry might cause more damage to your phone or other applications. The Method I have narrated here worked for me, but it may not work foryou. So please be careful and do not blame me if the result is not what you expected.
