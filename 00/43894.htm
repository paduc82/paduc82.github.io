I also have had the same problem with my printer when I was trying to print to it from my laptop using the HomeGroup sharing of printers. I had the same symptoms: I could send the document to the printer, but it would not print until I restarted the computer  that the printer was physically connected to. The only way I was able to solve the problem was to  disable Bidirectional Support. Once I disabled that, I could print from my laptop to my printer that is connected to my desktop.

This is not the perfect solution because I will not get the notifications from my printer when the printer is out of paper or when a particular toner cartridge is low, etc. But at least I can share the printer with my laptop without having to physically unconnect the printer from the desktop and then connect  it to my laptop each time I want to print from my laptop.

No need to download Windows 10, use the built in recovery options that came with your Dell.

Consult the owners manual that came with your PC for instructions about how to reinstall Windows.

 

This is how the recovery partition is accessed for the most popular makes..

For Dell, press CTRL + F11 directly after powering up the computer

shinmila h say:

Hi je4n4,

 

The cause of this error is due to the fact that the version of wmp.dll in the registry is out of sync with what is installed.

 

The wrong wploc.dll version error is a common problem that usually happens when the original wmploc.dll file in C:\Windows\System32 folder is replaced. You definitely cannot download Windows Media Player 12 on a computer running Windows 7 except for N and KN editions of Windows 7.

 

To fix the incorrect wmploc.dll issue, follow the two Methods.

 

Method 1:

 

a.    Go to Start and type cmd.

b.    Right-Click on cmd and select Run as administrator.

c.     In the  Command Prompt type the following and press  Enter:

                       UNREGMP2.EXE /UPDATEWMPVERSION

             d.  Close the command prompt after running the above command. 

 

Run Windows Media Player.

 

Method 2: Re-register the wmp.dll file

 

Heres how:

 

a.    Open elevated command prompt.

b.    Type regsvr32 wmp.dllwithout the quotes in the command prompt and press Enter.

 

See if Windows Media Player works.

 

Method 3: Disable and re-enable the player

 

This should fix the issue.

 

a.    Go to Start and in the search bar type "Turn Windows features On or Off".

b.    Click on "Turn Windows features On or Off".

c.    Browse to Media Features and place a check mark in front of Windows Media Player.

d.    Restart the Computer.

 

That should be it.

 

For additional information, see: Turn Windows features on or off

barb bowman say:

The issue is most likely hardware. The only way to know for sure is to do a complete wipe and reimage from a downloadable recovery image https://support.microsoft.com/en-us/surfacerecoveryimage - if you have this issue after doing the reimage, it is definitely hardware. At that point, you can decide if you want an out of warranty exchange (running about $450 US) or if you want to keep using the USB adapter.

If it IS hardware and you are in the US, go to the closest physical retail Microsoft Store and discuss options with them.
