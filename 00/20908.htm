Changing equation editor default font size

Is there any hack for this?
jay freedman say:
You can define a style with a point size of 12, and add it to the Quick Style gallery in the ribbon or assign a keyboard shortcut to it. The fastest way to do that is to format any paragraph (even an empty paragraph mark) as 12 pt, select the whole paragraph, right-click it, and choose Styles > Save Selection as New Quick Style.

In the Quick Style gallery, right-click the new style and choose Modify. Set the "Style for following paragraph" to Normal (or other body text style), and click the option at the bottom for "New documents based on this template". Click OK.

To assign a shortcut, go to File > Options > Customize Ribbon > Keyboard Shortcuts Customize. Choose "Styles" near the bottom of the Categories list, choose your style in the Styles list, and enter a shortcut key combination. Click the Assign button before you click the Close button.

Now, when you're typing regular text and you hit Enter and Alt+= to write an equation, click the style in the Quick Style gallery or press the assigned shortcut key to apply the style.

If you type an equation inside a regular paragraph, select just the equation after you type it and apply the style to it. The rest of the paragraph will remain as it was.

jay freedman say:
When you insert an equation box, the point size of the equation text will be the same as the point size of the ordinary paragraph into which you inserted it. If you modify the Normal style (or whatever style you use for body text) to be 12 pt, then the equations will also default to 12 pt.

In theory you can change the default font, but so far Cambria Math is the only one that contains the full set of glyphs needed for the equation editor. This may change in the future -- but no guarantees. When the cursor is in an equation and the Equation Tools tab appears on the ribbon, click the little arrow in the bottom right corner of the Tools group to open the Equation Options dialog, and the first item there is "Default font for math regions"; but the only item in the dropdown is (so far) Cambria Math.
