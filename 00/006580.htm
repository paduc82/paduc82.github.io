karl timmermans [outlook mvp 2012-2018] say:

Assuming that you are referring to having installed the Outlook '2013 "click to run" version along side O'2003, then you will not be able to designate O'2003 as the email client referenced by your 3rd party application. Doesn't make a difference whether or not you have O'2003 open or not - the net result is that O'2013 will be invoked by your app and you have no control over that (leastwise not that I've been able to find that would over-ride the default behavior in this scenario). Not just an issue for your application but any app invoking Outlook programmatically.

 

Just to round out issues related to having 2 versions of Outlook on your system (O'2013 and any other version)

 

- each Outlook version can have a different default Outlook profile using different accounts/PST files etc

- If both versions use the same email accounts and data stores (i.e. PST files) - the last version of Outlook to be opened will fail since the data stores will already be in use

-40822, -40256, in the Days Pending field.

 

misplaced parenthesis: it should be

DateDiff("d", [Open Date], NZ([Closed Date], Date()))

The expression

NZ([Closed Date], Date()))

looks at the field [Closed Date]; if there is a date in that field, NZ returns it, if not, it will return the second argument of the function - here Date().

bernie deitrick say:

I'm sorry - It thought that the macro worked as you wanted but was just copying all the cells instead of the visible cells.

 

Anyway, here's a macro that I normally use to do that - assumes all the data starts in cell A1, and that you want to export based on column B:

 

Sub ExportSheetsFromDatabaseBasedOnColumnB()
'Based on the value in the the second column
Dim myCell As Range
Dim mySht As Worksheet
Dim myName As String
Dim myArea As Range

Set myArea = Range("A1").CurrentRegion.Columns(2).Offset(1, 0).Cells

Set myArea = myArea.Resize(myArea.Rows.Count - 1, 1)

For Each myCell In myArea
On Error GoTo NoSheet
myName = Worksheets(myCell.Value).Name
GoTo SheetExists:
NoSheet:
Set mySht = Worksheets.Add
mySht.Name = myCell.Value
With myCell.CurrentRegion
    .AutoFilter Field:=2, Criteria1:=myCell.Value
    .SpecialCells(xlCellTypeVisible).Copy _
        mySht.Range("A1")
    mySht.Cells.EntireColumn.AutoFit
    .AutoFilter
End With
Resume
SheetExists:
Next myCell
End Sub
