dennis zac say:

Hi Ravi_ 165,

 

Thank you for posting your query. Windows phones will require the recovery key when the reset protection is enabled on the device. When youve enabled it, you will be receiving a "Welcome to Reset Protection" confirmation sent to the email address associated with your Microsoft account. You can try to check your email and see if the recovery key is still available from there.

 

If in any case that the recovery key is not available from your email, then kindly follow these steps on how to get a recovery key:

 

  1. Go to account.microsoft.com/devices. (Kindly use your Microsoft account associated to your Windows phone.)
  2. Check the information under your Windows phone, then click Get recovery key.
  3. Tap the check box for Im ready to get a recovery key.
  4. Then, click Get key.

 

You can also get the recovery key by turning off the Reset Protection online. To do this, kindly follow these steps:

Disk Partitioning inconsistencies: Microsoft Docs vs. Clean - Microsoft Community

don cuthbert say:

Disk Partitioning inconsistencies: Microsoft Docs vs. Clean install results

As has been noted by others in these forums the default partition layout, after a clean install of Windows 10 from install media, does not follow Microsoft's recommended layout.  Below is the recommended layout shown in Microsoft Docs:

Here is what the documentation states, as to why to have the Recovery Tools at the end of the disk and rules for the size of the partition:

  • Recovery tools partition

This partition must be at least 300 MB.

This partition must have enough space for the Windows Recovery Environment tools image (winre.wim, typically between 250-300MB, depending on base language and customizations added), plus enough free space so that the partition can be captured by backup utilities:

  • If the partition is less than 500 MB, it must have at least 50 MB of free space.
  • If the partition is 500 MB or larger, it must have at least 320 MB of free space.
  • If the partition is larger than 1 GB, we recommend that it should have at least 1 GB free.
  • This partition must use the Type ID: DE94BBA4-06D1-4D40-A16A-BFD50179D6AC.
  • The recovery tools should be in a separate partition than the Windows partition to support automatic failover and to support booting partitions encrypted with Windows BitLocker Drive Encryption.

We recommend that you place this partition immediately after the Windows partition. This allows Windows to modify and recreate the partition later if future updates require a larger recovery image.

A clean install of Windows 10, instead places the Windows RE (Recovery) partition at the beginning of the disk. This poses potential issues for future upgrades, as the upgraded winre.wim file in the Recovery partition may be too large to fit and a new Recovery partition instead gets created at the end of the disk, rendering the original Recovery partition redundant.

Notice also, that the Recovery partition should be kept smaller than 500MB (450MB-499MB) as you would be required to leave 320MB of free space on a 500MB or larger Recovery partition.  The Windows 10 winre.wim file would not leave 320MB free space when placed in a 500MB partition.
