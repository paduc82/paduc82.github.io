    bob jones aka: cybertaz say:

    The ability to customize the Ribbon in Mac Office is limited to Office 365 subscribers. Go to Word> About Word in the program's main menu bar. If the License refers to anything other than subscription the UI provides the ability to customize the QAT only. Customizing the Ribbon would have to be done programmatically.

    While in the About Word window, note the version number, which should be 15.33 or later. If a lower version number is indicated, use Help> Check for Updates in order to bring Office to a current status.

    Ribbon customization was added by way of the 15.23 (160611) update, so even if you have an Office 365 subscription you won't have the feature if your present version precedes that.
