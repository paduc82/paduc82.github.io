"Segoe UI Light (TrueType)"="segoeuil.ttf"
"Segoe UI Semibold (TrueType)"="seguisbi.ttf"
"Segoe UI Symbol (TrueType)"="seguisym.ttf"

Located under: [HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Fonts]
