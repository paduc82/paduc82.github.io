cari [former ms-mvp] say:

First after install you need to go into the BIOS and check the BIOS sees the drives correctly.  Then you need to tell the BIOS which is the main hard drive that it will be booting from.

Once this is done, startup into Windows and provided you have the motherboard chipset drivers installed correctly in Windows, it should see and install the correct drivers for the hard drives.  They will not appear in My Computer at this point in time.  Right click on My Computer and select Manage.  On the following screen, click on Disk Management.  They should now show up on the right hand side but as unpartitioned and unformatted drives.  Click on one of them, create a partition and format the drive, and ensure Windows has assigned it a drive letter.  Do the same with the other one.

Now they should show up in Windows Explorer.
