john korchok say:

PowerPoint 2010 already does this by default when you're in Slide Show Mode (not Edit Mode). You may need an Office repair:

  1. Click on the Start orb and choose Control Panel.
  2. Choose Programs, then Programs and Features.
  3. Right-click on Microsoft Office 2010 and choose Change.
  4. Select the Repair option, then click on Continue.
  5. Allow the process to complete, restart your computer, then test hyperlinked text for the hand cursor.

If the presentation still does not display the hand cursor, please create a new blank presentation, insert a hyperlink, go to Slide Show and test. If the hand cursor appears, there is a problem with the original presentation.
