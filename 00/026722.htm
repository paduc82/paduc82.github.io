corninshit say:

Your printer may not be printing headers, where the Address on the envelope is located.

Please check your printer options through the Brother driver by right clicking the printer from the Printers and Devices tab.  It's one of the device options that may have "print header/footer" turned off.  Click on the box to print header/footer.  Select "do not print" for date and time otherwise you will get a date/time stamp in the lower right of everything you print.  

Then reopen Word and use the print envelope wizard, and try printing a sheet of 8.5 by 11 paper to make sure it prints. This is to make sure you Word is sending the envelope print information to the right section of the printer, and also to save envelopes.

Neither the return address nor the delivery address on an envelope is in the header, nor is there any option in Word to suppress header/footer printing. If there is such an option in the printer Properties, it would be for a header or footer generated by the printer, not created in Word. Again, a time/date stamp (unless explicitly added to the Normal template in Word) would be a printer option. The Brother MFC-7340 is a multifunction device, and there is no indication in the user's manual that it has any of these features.

I think the build 16215 issue might be that whatever Lock screen option you have cannot be changed.

Yours may already have been on Picture, but can you switch to Windows Spotlight or Slideshow?

You can always change the Lock screen picture shown by using the Photos App and right-click the image to make it the Lock screen. But if you do that, you cannot switch back to Windows Spotlight nor Slide show even though the Settings page shows Windows Spotlight. If you are already on Windows Spotlight, you cannot switch to Picture. If you select Picture, there is no browse button. If you are currently stuck on Windows Spotlight, you can try to switch to Slide show and can even select folders to use, but you cannot get Slideshow to actually remain selected and work.

That is how mine works after a clean install of 16215 Pro. Maybe upgraded installs work OK.
