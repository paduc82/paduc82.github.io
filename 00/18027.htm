stefan blom say:

Define a hanging indent for the Footnote Text paragraph style. Then you have to add the following macro to your Normal template (the macro will override the option to insert footnotes via the Insert Footnote button on the References tab):

Sub InsertFootnoteNow()
'Macro created by MVP Stefan Blom, December 2015
'Inspired by code created by Dave Rado, http://word.mvps.org

ActiveDocument.Footnotes.Add Range:=Selection.Range

With Selection
     .InsertAfter vbTab
     .Collapse wdCollapseEnd
End With

ActiveWindow.ScrollIntoView Selection.Range

End Sub

For installation instructions, see http://www.gmayor.com/installing_macro.htm.

Note that the code above fires when you click the Insert Footnote button on the References tab. It won't run if you click Insert Endnote or display the Footnote and Endnote dialog box.
