SELECT DISTINCT ID, Category FROM YourTableName;

https://www.thurrott.com/windows/windows-10/62435/microsoft-improves-edge-security-in-windows-10-1511

In addition I believe that these same security improvements in the Edge browser probably help to explain why there have been such a large number of basic browser hijack attacks against it.  Since there's really not much of a way to truly take over the browser itself and the Edge browser is the Windows 10 default, the most effective method available to malicious attackers is to socially engineer the PC user using whatever minor attack vectors remain.

In other words, though the Edge browser itself is actually much safer in real terms, it's also much more commonly used to display visually annoying pop-up type displays than other browsers which might be more susceptible to silent and so deadly attacks.  The problem for Microsoft as always is that the typical user isn't able to understand this, so they see the large number of visible attacks as a huge problem, since they often don't understand how to stop them or even that the pop-ups aren't "real" valid information.

So for the initial poster Ross in this thread, though you'll of course still need an effective antimalware program to protect against downloaded malware, it's probably not possible to have such other protections as Sandboxie with the Edge browser at present.  At least not until the extensions capability is added which should also allow those like Sandboxie to create their own additions.
