rohn007 say:
A "backup" and a "system restore" are fundamentally opposite.

A system restore is a Windows way of resetting some (not all) of the Windows system settings and some of the system files.  System restores use specialized backups called "checkpoints".

A "backup" can take many forms, but it is essentially a copy of all or part of your HD that can be restored at any time.

A "backup" can protect your installed software by making a copy of the instalaltion so that you can restore it after there is a problem. 

Making a system backup (even of a partly messed up system) BEFORE doing a system restore would give you a recovery point just in case the system restore makes things worse.

The most important part of your Office installation is the 25 character Product Key.  If you lose it, your copy of Office is completely gone.  Online download purchases set up an online account using your email address as the login ID. The account has your product key and a link to the installation file download.  To protect yourself, for example in case you lose access to the email address, you should either buy a copy of the "official" backup disk or burn the installation file to CD or USB drive(s) AND write down your Product Key somewhere that it won't get lost.
