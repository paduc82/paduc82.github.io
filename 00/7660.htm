In excel 2007 how do you add a value to a cells previous value. - Microsoft Community

blondie149 say:

In excel 2007 how do you add a value to a cells previous value.

Ex. Cell E5 is 3, then want to add 6 to the same cell so that it re-calculates it to read 9.
gary's student say: Include the following event macro in the worksheet code area:

Dim OldValue As Variant

Private Sub Worksheet_Change(ByVal Target As Range)
If Intersect(Range("E5"), Target) Is Nothing Then Exit Sub
Application.EnableEvents = False
If OldValue = "" Then
    OldValue = [E5]
Else
    [E5] = [E5] + OldValue
    OldValue = [E5]
End If
Application.EnableEvents = True
End Sub


Because it is worksheet code, it is very easy to install and automatic to use:

   1. right-click the tab name near the bottom of the Excel window
   2. select View Code - this brings up a VBE window
   3. paste the stuff in and close the VBE window

If you have any concerns, first try it on a trial worksheet.

If you save the workbook, the macro will be saved with it.


To remove the macro:

   1. bring up the VBE windows as above
   2. clear the code out
   3. close the VBE window

To learn more about macros in general, see:

http://www.mvps.org/dmcritchie/excel/getstarted.htm

To learn more about Event Macros (worksheet code), see:

http://www.mvps.org/dmcritchie/excel/event.htm

Macros must be enabled for this to work!

Running a macro containing the following code (on a copy of your document, in case you need to edit it further), will convert the endnotes to text, preceded by the Page #: for the page on which the text to which the end note refers.

    Dim aendnote As Endnote
    For Each aendnote In ActiveDocument.Endnotes
        ActiveDocument.Range.InsertAfter vbCr & "Page " & aendnote.Index & _
        aendnote.Reference.Information(wdActiveEndPageNumber) & ": " & aendnote.Range
        aendnote.Reference.InsertBefore "a" & aendnote.Index & "a"
    Next aendnote
    For Each aendnote In ActiveDocument.Endnotes
        aendnote.Reference.Delete
    Next aendnote
    Selection.Find.ClearFormatting
    Selection.Find.Replacement.ClearFormatting
    With Selection.Find
        .Text = "(a)([0-9]{1,})(a)"
        .Replacement.Text = "\2"
        .Forward = True
        .Wrap = wdFindContinue
        .Format = True
        .MatchWildcards = True
    End With
    Selection.Find.Execute Replace:=wdReplaceAll

Hi Jonathan,

Here's a different approach. Unlike Doug's, it retains the endnotes as endnotes and merely hides their references (the display of which you can toggle on/off). The page #s are also inserted as fields that reference a bookmark for each endnote, so that they remain valid if the document is subsequently reformatted. Your additional encoding has also been added.

Sub EndNotePageRefs()
Application.ScreenUpdating = False
Dim RngNote As Range, EndNt As Endnote, StrTmp As String, StrTxt As String
Dim l As Long, x As Long
With ActiveDocument
  For Each EndNt In .Endnotes
    .Bookmarks.Add Name:="_EndNt" & Format(EndNt.Index, "000"), Range:=EndNt.Reference
    Set RngNote = EndNt.Range
    If InStr(EndNt.Reference.Sentences.First.Text, " ") = 0 Then
      StrTmp = EndNt.Reference.Sentences.First.Previous.Sentences.Last.Text
    Else
      StrTmp = EndNt.Reference.Sentences.First.Text
    End If
    l = 3
    StrTxt = ""
    If UBound(Split(StrTmp, " ")) < l Then l = UBound(Split(StrTmp, " "))
    For x = 0 To l
      StrTxt = StrTxt & " " & Split(StrTmp, " ")(x)
    Next
    StrTxt = Trim(StrTxt)
    With RngNote
      .InsertBefore "></bold><tab><ital>" & StrTxt & ":</ital><space><"
      .Collapse wdCollapseStart
      .InsertCrossReference ReferenceType:="Bookmark", ReferenceKind:=wdPageNumber, _
        ReferenceItem:="_EndNt" & Format(EndNt.Index, "000"), InsertAsHyperlink:=True, _
        IncludePosition:=False, SeparateNumbers:=False, SeparatorString:=" "
      .InsertBefore "<bold>Page:<"
    End With
  Next
  .Styles("Endnote Reference").Font.Hidden = True
End With
Set RngNote = Nothing
Application.ScreenUpdating = True
End Sub
