try with this function:

Option Compare Database
Option Explicit

Public Function Suppress_multi_vbcrlf(Field_ As String) As String
Dim wChar As String
Dim wFirst As Boolean
Dim i As Long
  Suppress_multi_vbcrlf = ""
  wFirst = True
  For i = 1 To Len(Field_)
    wChar = Mid(Field_, i, 1)
    If wChar = Chr(13) Then
       i = i + 1
       If wFirst = True Then
          wFirst = False
          Suppress_multi_vbcrlf = Suppress_multi_vbcrlf & vbCrLf
       End If
    Else
       wFirst = True
       Suppress_multi_vbcrlf = Suppress_multi_vbcrlf & wChar
    End If
  Next
End Function

Let us try the following steps and check if it helps in resolving the issue.

Step 1:

Step 2:

Start Outlook in safe mode by following instructions mentioned below and verify whether issue re-occurs.

    Outlook /safe

Note: There is a space between Outlook and '/'.

a) Click on File menu, click Options, Add-ins, Go button besides Manage: Com-in Add.
b) Check if there are add-ins listed, clear the checkbox.
c) Close the Office program and restart it.
d) Disable each add-in one at a time, restart the Outlook, and repeat the above procedure. If the issue reappears again, it will help you to determine which add-in is causing this issue.
