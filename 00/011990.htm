ossiemac say:

If I am understanding the question correctly then the  code below should do what you want.

 

It will not allow the value in D2 to be deleted; only changed.

 

It then clears D4 if CCAN is selected or D3 if Contract is selected.

 

To install the code right click the sheet name tab and select View Code. Copy all of the code into the blank white area of the VBA editor. (DO NOT change the sub name or is it will not work.)

 

Close the VBA editor (Cross with red background top right.)

 

Save the workbook as Excel Macro Enabled workbook (.xlsm)

 

Ensure you have macros enabled. Select File -> Options -> Trust Center -> Macro Settings and then select the option "Disable all macros with notification". (You will then be asked if you want to allow macros when you open the workbook.)

 

 

Private Sub Worksheet_Change(ByVal Target As Range)
    On Error GoTo ReEnableEvents
    Application.EnableEvents = False
    If Target.Address = Range("SearchOption").Address Then

        Select Case UCase(Target.Value)
            Case ""
                Application.Undo
                GoTo ReEnableEvents

            Case UCase("CCAN")
                Range("Contract").ClearContents

            Case UCase("Contract")
                Range("CCAN").ClearContents

        End Select
    End If

ReEnableEvents:
    Application.EnableEvents = True
End Sub

You don't need PC Suite, you can transfer/sync all contacts via Bluetooth.

alexsun_2012 say:

Hi Martin,

Yes, it is feasible. As an Office 365 admin, you can run the eDiscovery search for the specific mailbox in Exchange admin center and export all the mailbox data into a .pst file. More detailed instructions can be found in the following articles:
https://technet.microsoft.com/en-us/library/dd298021%28v=exchg.150%29.aspx
https://technet.microsoft.com/en-us/library/dn440164%28v=exchg.160%29.aspx

If you want to keep the mailbox items deleted by the end user in the server side, you can consider placing a litigation hold or in-place hold on the mailbox and this will require an Exchange Online Plan 2 license.

Please let us know if you need further assistance.

Thanks,
Alex
