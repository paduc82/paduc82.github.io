Can I embed a Word file as an icon in another Word document? - Microsoft Community

nwcgrad say:

Can I embed a Word file as an icon in another Word document?

Is there a way to compress a Word 2010 file for insertion in another Word 2010 file as an icon that would open when double-clicked the same as is done with a Word file that's an attachment to an email?

jay freedman say:
Yes, you can do that.

On the Insert tab, click the Object button near the right end. In the dialog that opens, click the Create From File tab.

Click the Browse button and locate the document file to insert.

Check the box for Display As Icon, and click OK.

bharath-47 say:

We can Insert a Word file like a icon in any other Word file:

Assuming Doc1 as the container doc and Doc2 as the doc to be compressed as an icon.

 

1. Open Doc1, Insert Tab-> Under Text Group, Choose Object

 

2. In Object Dialog, Choose Object Type Microsoft Office Word Document

 

3. In Object Dialog,Create From File Tab, Browse For Doc2, Tick Display as Icon

 

4. Ok

 

It would insert the Doc2 object as an icon into Doc1 file. Which can then be opened by double clicking it, opens as Document in Document file.
