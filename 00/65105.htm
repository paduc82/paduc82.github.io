12.1807.264.0 is the currently latest version of the new Microsoft Skype for Windows 10 (Skype UWP) application.

 

If prior to installing this Skype version you have used the classic Skype for Windows Desktop and have used this version on the same computer also 3 years ago, then there is chance that all the old chat messages are still saved on you computer.

Go to Windows Start and in the Search/Run box type %appdata%\skype and then press Enter or click the OK button. The Windows File Explorer will pop up. Do you see there a folder named as your Skype Name?

Open this folder. You should see there a database file named main.db (might be shown as just main).

 

This main.db file should contain all the chat messages which you have received/sent using the previously installed classic Skype version.

 

To be able to open and read the content of this database file you would need an SQLite browser/editor. However, you can also make use of the free SkypeLogView application, which is especially designed for this purpose.
