Even though the maximum data entry is 9999:59:59, I have not found a practical limitation for the Custom [h]:mm:ss format.

For example, if A1 is 365000.123456 (1000+ years), formatting as Custom [h]:mm:ss correctly displays 8760002:57:47.

So =B4-(C4+D4) formatted as Custom [h]:mm:ss should work as long as B4, C4 and D4 are

Presumably B4 is text.  How did that happen?

If B4 is the result of copy-and-pasting or otherwise importing from another source, the following formula converts it to numeric time:

=INT(LEFT(B4,FIND(":"B4)-1)/24) + (MOD(LEFT(B4,FIND(":",B4)-1),24) & MID(B4,FIND(":",B4),6))

formatted as Custom [h]:mm:ss.

If B4 is a formula of the form =TEXT(SUM(A1:A1000),"[h]:mm:ss"), for example, try one of the following alternatives:

=IFERROR(--TEXT(SUM(A1:A1000),"[h]:mm:ss"),ROUND(SUM(A1:A1000)*86400,0)/86400)

or

=ROUND(SUM(A1:A1000)*86400,0)/86400)

formatted as Custom [h]:mm:ss.

Usually, --TEXT(...,"[h]:mm:ss") is the best way to ensure that calculated time is represented internally properly.  But that results in text instead of a number (if we omit "--") when SUM(A1:A1000) exceeds 9999:59:59.

Obviously, the latter alternative is simpler; and that is what many people would opt for.

But that might be infinitesimally different from --TEXT(SUM(A1:A1000),"[h]:mm:ss") in some cases, which defeats the purpose of explicitly rounding calculated time, namely:  so there are no surprises when comparing calculated time with time constants.  That might not be a concern for you.

'---
Sub AdjustMe()
 Dim dblChange As Double
 Dim objShp As Shape
 Set objShp = Me.Shapes(Application.Caller)
 If objShp.Height > 50 Then                        ' ADJUST SIZE
   dblChange = 0.5
 Else
   dblChange = 2
 End If
 objShp.Height = objShp.Height * dblChange
 objShp.Width = objShp.Width * dblChange
End Sub

industrypincer say:

I found a partial work around to the problem of having to disable balloons each time you review a document in Word 2016 (for Mac). It involves creating a keyboard shortcut:

1. <go to> Tools

2. Customize Keyboard...

3. Select "All Commands" in the Category panel

4. Search for "balloons" in the search box of the Commands panel

5. Select "NeverUseBalloons"

6. Specify a Keyboard Shortcut  (I use Command+Option+Control+B)

7. Click "Ok"

You still have to turn balloons off each time you work on a document.  But I find using one keyboard command to do that less frustrating than 3 or 4 mouse clicks.
