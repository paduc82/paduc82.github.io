4. At the top your country or region will most likely display France since that is what it is set to from the reset.
5. Click on the + sign below to Add Language.
6. in the search box type english
7. English will display a few different types of English, pick the English that best fits your needs.
8. Once you click the English you would like to use, you should then see French listed and English.
9. Click on English to highlight it , it should come up with options.
10. Now you will want to select as default.

zigzag3143 (ms -mvp) say:
ShinPark3

These are typically related to a HD issue.  Before you run CHKDSK I would remove express cache, and Avast (both are know to cause BSODS)

Avast  can be a  contributing cause of BSOD'S . 
Please remove and use the built in defender at least to test

http://files.avast.com/files/eng/aswclear5.exe



A- CHKDSK /R /F:

Chkdsk disclaimer: While performing chkdsk on the hard drive if any bad sectors are found  when chkdsk tries to repair that sector  any data available on that sector might be lost.


Run CHKDSK /R /F from an elevated (Run as administrator) Command Prompt.
Do this for each hard drive on your system.
When it tells you it can't do it right now - and asks you if you'd like to do it at the next reboot - answer Y (for Yes) and press Enter.

Then reboot and let the test run.
It may take a while for it to run, but keep an occasional eye on it to see if it generates any errors.
http://www.howtogeek.com/howto/windows-vista/guide-to-using-check-disk-in-windows-vista/


 See "CHKDSK LogFile" below in order to check the results of the test.


B- Elevated Command Prompt:
Go to Start and type in "cmd.exe" (without the quotes)
At the top of the Search Box, right click on Cmd.exe and select "Run as administrator"


C-CHKDSK LogFile:
Go to Start and type in "eventvwr.msc" (without the quotes) and press Enter
Expand the Windows logs heading, then select the Application log file entry.
Double click on the Source column header.
Scroll down the list until you find the Chkdsk entry (wininit for Win7) (winlogon for XP).
Copy/paste the results into your next post.

.Action file extension - Microsoft Community

rsolivero say:

.Action file extension

I'm having trouble opening a .action file extension from Square Trade.  How can I accomplish this?

Rick

a. user say:

Go to Start / Default programs / Associate a file or protocol with a program and search the list for the .action file extension (I've never heard of it or any that long before - they're usually only 3 digits - so be sure it's really the file extension and not actually part of the file name - Go to Start / Control Panel / Folder Options / View and uncheck the box to hide file extensions of known file types).  If you find it it should list a default program that is used to open such files.  If it is incorrect, click on Change and see if the correct option is listed.  If not, click on browse and navigate to the executable program for the program that is used to open such files (it must be installed on your system for the extension to show up and for you to open the file type).  If it is not listed then no program was installed on the system to add that extension to the list of file types and thier associations which is why it will not open.  You may have to download and install Square Trade (or something that can open their files) on your computer in order to open this file (check with Square Trade techical support for further assistance).

I hope this helps.

Good luck!
