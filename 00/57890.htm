a. user say:

Do you know when this problem began?  Try a System Restore to a point in timeBEFORE the problem began.  Here's the procedure: http://www.howtogeek.com/howto/windows-vista/using-windows-vista-system-restore/.  Be sure to check the box to show more than 5 days of restore points.  If the first attempt fails, then try an earlier point or two.  NOTE: You will have to re-install any software and updates you installed between now and the restore point, but you can use Windows Update for the updates.  Use the recovery disk if the system prompt doesn‚Äôt work. The recovery disk works a bit different from the above procedures but if you follow the prompts from the System Restore menu option with the above information you should be able to restore with no problems.

Go to Start / Control Panel / Mouse and check out the options listed for the pointer. If differs for different models or mice and computers, so I don't know what yours will say.  If there is an option to automatically move the pointer to the default button in a dialog box, then uncheck that option.  Also, if there is an option to select how clicking operates, there may be an option to choose where double-click or single-click or merely hovering will activate the option - change it to double-click (especially if it is on hover) and that should solve the problem.  You may also want to decrease the sensititivity of the mouse as the "automatic"clicks may just be a very light touch on the key that was uninintentional and even unnoticed.

If that doesn't work, you may be having troubles with your device drivers, firmware, and/or software.  Go to Device Manager by going to start / search box and type device manager and enter and then double-click on the program icon that appears.  Check each device for a red x, yellow ! or white ?.  These identify devices with problems (probably drivers, but also conflicts or something else).  Click on each for further details and troubleshooting tips.  If you need to get drivers (and you shouldget them for your mouse at least and any you have time to do - they may help in ways beyond just resolving this problem), do so from the computer vendor or device manufacturer (NOT from Microsoft Updates).  In fact, you should turn off automatic driver updates from Windows Update as follows: http://www.addictivetips.com/windows-tips/how-to-disable-automatic-driver-installation-in-windows-vista/. Proceed as follows to get the drivers: http://pcsupport.about.com/od/driverssupport/ht/driverdlmfgr.htm.  Once you have the drivers, you can install them through Device Manager as follows:http://www.vistax64.com/tutorials/193584-device-manager-install-driver.html. You may also want to try the Driver Verifier Manager to see if you can identify the driver or device responsible (assuming that is the cause):http://support.microsoft.com/kb/244617.  Once done, recheck the Mouse option in Control Panel as it may now have new or more features and see if any changes there as noted above will help.

If that still doesn't help, try using a different mouse and see if you have the same problem.  If not, try the old mouse on a different computer and see if that computer begins to have the problem.  This will  help us determine if the problem is the mouse itself or the computer.  If this test indicates it's the mouse, replace it.

I hope this helps.

Good luck.

2. Install 7-Zip

3. Find the directory where your files are backed up. Its name will be the name of the computer. I called my laptop VAIO so the directory is VAIO.

4. Open up the directory that contains the backed up files (they'll likely be stored in umpteen .ZIP files). You'll need administrator access on your current machine to access the files.

5. Select all the files

6. Right-click on the list and choose the 7-Zip submenu. From there choose Extract files... Make sure you choose and/or create a directory where you can save the extracted files.


PS Extracted 100 GB in 70 minutes :-) (using USB 2 on a slow laptop).


SLOW workaround (no knowledge needed):

1. Find the directory where your files are backed up. Its name will be the name of the computer. I called my laptop VAIO so the directory is VAIO.
