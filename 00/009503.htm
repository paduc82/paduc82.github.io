There are a lot of assumptions that go into this example code - if you shared your workbook.......

    Private Sub CommandButton1_Click()
    Dim D As Date
    Dim R As Long
    Dim C As Long
    Dim W As Worksheet

    R = Me.lbName.ListIndex + 3  'The row on the sheet is some offset from the listbox

    For D = CDate(Me.tbDateFrom.Text) To CDate(Me.tbDateTo.Text)
        Set W = Worksheets(Format(D, "mmmm"))  'Tab names are long-form month names
        C = Application.Match(Day(D), W.Range("3:3"), False)  'Find the column with the day number
        With W.Cells(R, C)  'Cell on the row and column
            .Interior.ColorIndex = Me.lbActivity.ListIndex + 3
            If Me.tbComment.Value <> "" Then
                .AddComment
                .Comment.Text Text:=Me.tbComment.Value
            End If
        End With
    Next D

End Sub

Generally that is what your Personal Macro Workbook is used for.

 

You place the macros in modules in Personal.xlsb, store it in your XLSTART folder and it runs whenever Excel is fired up.  It is set to be hidden and the macros are accessed through Developer>Macros or use shortcuts or add buttons to your menu.
