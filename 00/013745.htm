the usual checks and safeguards user data gets so as to not slow down processing unacceptably for rapid video display.  Consequently, it must be in an area of ram not visible to user programs because the usual checks and safeguards against user programs also
writing to that area are not present.  The ram is there but if it was available to your programs your monitor would soon display garbage.
