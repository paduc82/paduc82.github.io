Key: HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\Classes\CLSID\{DC6EFB56-9CFA-464D-8880-44885D7DC193}

Value type: REG_SZ

4.     Right click on the file and rename it to PDF.reg (Make sure the extension is .reg)
