andreas killer say:

As far as I can see the issue is only the formula in H20

If you want to change the weekend, you have to use NETWORKDAYS.INTL instead of NETWORKDAYS

H20:  =NETWORKDAYS.INTL(DATE(Calendar_Year,1,1),EDATE(DATE(Calendar_Year,1,1),12)-1,16)

The last argument 16 at the end of the formula indicates that only Friday is a weekend.
