rick rothstein, mvp - excel say:

You could create a macro to do both the inserting of the row and copying of the formula (add it to the QAT and make it a one-click operation). Here is the macro to do this...

Sub AddRowCopyFormulaInColumnG()
  ActiveCell.EntireRow.Insert
  Cells(ActiveCell.Row + 1, "G").Copy Cells(ActiveCell.Row, "G")
End Sub

If macros are a news thing to you, here is how to install it. Press Alt+F11 to go into VB editior, click Insert/Module from its menu bar, and copy/paste the above code into the code window that opened up. Now, select a cell on the row that you want to insert at, press Alt+F8 and select AddRowCopyFormulaInColumnG from the list and click the Run button. To install this on the QAT, click the down arrow to the right of the QAT bar, select More Commands from the popup menu that appears, select Macros from the "Choose commands from" drop-down and select AddRowCopyFormulaInColumnG from the list and click the Add button, then click OK.
