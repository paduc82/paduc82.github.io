You can use a formula like this to sum every other row:

 

Odd rows:

=SUMPRODUCT((A1:A100)*(MOD(ROW(A1:A100),2)<>0))

Even rows:

=SUMPRODUCT((A1:A100)*(MOD(ROW(A1:A100),2)=0))

suzanne s. barnhill say:

The first step is to convert each information block to a single paragraph. To do this, assuming (based on your description) that you pressed Enter twice between information blocks, you can do this as follows:

  1. Press Ctrl+H to open the Replace dialog.
  2. In the "Find what" box, type "^p" (without the quotation marks).
  3. In the "Replace with" box, type "^l" (without the quotation marks; the letter is a lowercase L).
  4. Replace All.
  5. Now repeat, replacing ^l^l with ^p. This gives you each block in a single paragraph with a line break at the end of each line except the last.
  6. You can now select all the text and use the Sort button to sort by the first letter(s) in the first line.

If you want to be able to sort by some other information, you can convert the text to a table. To do that, replace the line breaks (^l) with tab characters (^t). You can then use Insert | Tables | Table | Convert Text to Table to create a table, separating text at Tabs. This will give you a three-column table with one row for each website, and you can sort on any desired column. If you want to use this type of setup, it will be advisable to first convert the three-column document to a single column.

To add a new website, you can either add it at the end and resort (either paragraph text or a table), or you can insert a new paragraph/row and insert it in place.

In the above, I'm assuming you've used newspaper-style columns. If, as Rohn assumes, you've created the columns with tabs, then you'll probably need to start by converting the table to text, which will be more complex.

gord dibben say:

If you can use a macro try this one which "adds empty rows afterward"

Sub Insert_Alt_Blank_Rows()
    Application.ScreenUpdating = False
    Application.Calculation = xlCalculationManual
    Dim i  As Integer
    For i = Selection(Selection.Count).Row To _
        Selection(1).Row + 1 Step -1
        Rows(i).EntireRow.Insert
    Next i
    Application.Calculation = xlCalculationAutomatic
    Application.ScreenUpdating = True
End Sub

jp ronse say:

Hi,

An alternative formula that you can drag down as far as you need.

=IF(ISEVEN(ROW()),INDEX('Sheet2'!A:A,2+QUOTIENT(ROW(),2)),"")

Guessing that you report starts in row 2.

There is a small flaw in the formula, it returns 0 when it ran out of the data range in sheet2.

To avoid this, you can use:

=IFERROR(IF(ISEVEN(ROW()),INDEX(OFFSET(Sheet2!$A$3,0,0,COUNTA($A$3:$A$100000),1),QUOTIENT(ROW(),2)),""),"")

and drag it as far as you think you will need. This formula will generate an error when it ran out of the available data range but is intercepted by the IFERROR.
