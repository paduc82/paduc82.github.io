#1 potentially. MS does split the market up into various regions. US trade law restricts thing like encryption that can be sold to certain regions. Often, people will buy a copy of Office and install and activate it. But they forgot about the free trial. They run into activation problems. The fix is simple, you just have to uninstall one (usually the factory trial), and do an Office Repair of the remaining one before trying to do the activation again.

On Windows 8/8.1 when a third party antimalware program is installed the Windows Defender (WD) program is disabled.  Most issues involving WD failing to start are caused by incomplete removal of the third party antimalware program.

If you wish to use WD on W8 you must uninstall the third party antimalware program, run the McAfee Removal Tool, Norton Removal Tool, etc and reboot the computer.  Since you have already run the Norton Removal Tool I suggest you run it again and reboot the computer.  See List of anti-malware product removal tools'

WD should then be enabled though you might need to start the program (WD) in Security Services (and again) reboot the computer.

For further information see the following:

http://www.eightforums.com/tutorials/21962-windows-defender-turn-off-windows-8-a.html
