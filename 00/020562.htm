Thanks for the extra information. While the program interface doesn't include buttons to add bookmarks, they're easy to create manually. After creating a hyperlink, but before closing the dialog, add #yourbookmarkname (with yourbookmarkname being the actual bookmark name) to the end of the hyperlink.
