burrwalnut say:

To display the folder hierarchy, open Windows Explorer, navigate to the folder you wish to start at, hold down the Shift key, right-click on the folder name and choose Open command window here.

 

Type tree |clip and press Enter. Open your word processor program (or Notepad) and paste (Ctrl+V) the list to it. You can now print it, edit it or save it.

 

To display the files in each folder, type tree /f |clip and press Enter.
