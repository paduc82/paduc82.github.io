When you create a macro in a word 2010 form, can it be embedded in the - Microsoft Community

wrierson say:

When you create a macro in a word 2010 form, can it be embedded in the form so the macro travels with it when on the server and used on multiple machines?

I created a macro on a form created in Word 2010 which inserts a picture. I want our staff to be able to download this form from the cloud and have the macro work. When I load it to the cloud, the macro is not present. How can I get the macro to travel with the word doc so that it will work when downloaded on another machine?
jay freedman say:
There are two parts to getting this to work.

The first is that the code must actually be stored in the form (document or template, whichever you're using). In the macro editor, make sure the module containing the code appears under the form's project in the Project Explorer pane, not under the Normal project.

The second is that the file must be saved as a macro-enabled type (.docm for a document or .dotm for a template). If you save as a .docx or .dotx, Word will remove the code because these formats are explicitly not macro-enabled as a security measure.

Once the macro is actually present in the file as downloaded, there is still the question of whether the individual user's security settings in Word will permit the macro to run. There are macro enable/disable settings, trusted locations, and the possibility of digitally signing the macro project.

stevem-007 say:

Word Field Code calculation

Hi, I have a word form doc that has two check boxes and I want to assign a value if ticked (box 1 ticked would be 5 and box 2 if ticked 15). In a separate field I want to total the result of the check box selection. So the answer could either be 0, 5, 15 or 20. 

What's the easiest way to do this in word as it's driving me potty.

Thanks in advance. 

macropod say:

Word field codes cannot perform calculations based on checkbox values. Rather than using checkboxes, you might consider using dropdown formfields with Yes/No answers and their 'calculate on exit' property set in a document will 'filling in forms' protection applied. You could then do the calculation with a formula field coded like:

{={IF{REF Dropdown1}= "Yes" 5 0}+{IF{REF Dropdown1}= "Yes" 15 0} \# 0}

Note: The field brace pairs (i.e. '{ }') for the above example are all created in the document itself, via Ctrl-F9 (Cmd-F9 on a Mac); you can't simply type them or copy & paste them from this message. Nor is it practical to add them via any of the standard Word dialogues. The spaces represented in the field constructions are all required.
