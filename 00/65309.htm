daniel strommen [msft] say:

Ryan,

Our support for third-party browsers is limited.  As you know, browser makers (including us and Google) are always working to improve their browsers and sometimes this can make it challenging to keep up with all their new features.  Google Chrome uses the QUIC protocol between their browser and their own websites, which we are currently unable to filter.  There is a registry key that you can use to disable it using the Registry Editor.

[HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Google\Chrome]
"QuicAllowed"=dword:00000000

[Source: https://code.google.com/p/chromium/issues/detail?id=461739 ]

I realize that this is a technically advanced way of addressing your problem, and would need to be done manually on all your kids' machines.  As an alternative, it is also possible to block Google Chrome using App Restrictions.
