jmh3143 say:

For Outlook interface:

1. Go to your inbox
2. Select the desired email message you want to print
3. Above the message itself, you can see options like New, Reply, Delete, Junk, etc.
Please select the three dots ( ... ) near Categories
4. Select Print
5. A print preview of the message will appear, and also the print options. (be sure to click "all pages" for multi page emails.)
6. Just click on Print.
