The only way I can think of to do this is to actually print the report as a whole bunch of separate reports, one for each client.  That way, you can direct the first page of each of those individual reports to a separate printer tray.

You can drive the printing of a separate copy of the report using code that loops through a recordset of all the clients, and opens the report for each record, using the client ID from that record as a filter.  Code to do that might look something like this:

Dim rsClients As DAO.Recordset

Set rsClients = CurrentDB.OpenRecordset("tblClients")  ' or a query that returns the clients that will get the report

Do Until rsClients.EOF
