IF statement with multiple AND's - Microsoft Community

georgebean say:

IF statement with multiple AND's

=IF(AND(AND(AND(A2=A1,F1-E2<0,G2>E2,G2>=0))),E2,G2) returns the expected response in all column cells except in cell 2.  Row 1 has headers which causes me to get #VALUE! in cell 2.  How do I get rid of #VALULE! and still be able to use the same formula for all verticle cells?

tom ogilvy say:

=IFERROR(IF(AND(A2=A1,F1-E2<0,G2>E2,G2>=0),E2,G2),"")

Note that you don't need multiple AND functions.  A single AND will test that all the conditions listed are TRUE before it returns TRUE. 

IFERROR will suppress the error and display what is provided for the second argument.  In this case I have provide a null string "" to be displayed if an error is encountered.   You might want to replace that with E2 or G2 whichever you want returned for the first data row (row 2).

With some trial and error you will be able to figure out which value for "Exactly" is appropriate with the specific font.

With variable line spacing (Single, Double, Multiple etc.) the actual spacing is dependent on the program you are using (Word in this case) as well as the settings in the font itself. In a word processor, the printer driver that you are using may also affect the actual spacing.
