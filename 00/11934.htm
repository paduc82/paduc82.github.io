sfcsfcsfc say:

Script to resize a colum in a Word table with VBA

Hi all,

I very often work with documents that contain Word tables with six columns. I routinely need to work in column # 3 with large amounts of text. To do that, I manually resize (make as narrow as possible) columns 1, 2, 4, 5, and 6 so that column 3 is as wide as possible.

How can I do that with VBA?

Basically the script should go through each column in the selected table and make the column as narrow as possible (or resize to a predetermined width, as an alternative), except for column 3, which would then become as wide as possible as all the other columns have shrunk in width.

Thanks for your help!

Stef

graham mayor say:

The following will make the columns as narrow as possible with the reduction added to column 3

Sub SetTableWidths()
Dim oTable As Table
Dim iWidth As Long
Dim iNewWidth As Long
Dim i As Long
iNewWidth = 22        'The new default width to set the columns to
    iWidth = 0
    Set oTable = Selection.Tables(1)
    For i = 1 To oTable.Columns.Count
        If Not i = 3 Then
            iWidth = iWidth + oTable.Columns(i).Width
            oTable.Columns(i).Width = iNewWidth
            oTable.Columns(i).AutoFit
            iWidth = iWidth - oTable.Columns(i).Width
        End If
    Next i
    oTable.Columns(3).Width = oTable.Columns(3).Width + iWidth
lbl_Exit:
    Set oTable = Nothing
    Exit Sub
End Sub

andreas killer say:

IMHO we can use the AutoFit feature for the whole table.

Andreas.

Sub Test()
  Dim T As Table
  'Table selected?
  If Selection.Tables.Count = 0 Then Exit Sub
  Application.ScreenUpdating = False
  'For each table
  For Each T In Selection.Tables
    With T
      'Shrink the table to fit the content
      .AutoFitBehavior wdAutoFitContent
      'Enlarge the middle column
      .Columns(.Columns.Count / 2 + 0.1).Width = 999
      'Shrink the table to fit the window
      .AutoFitBehavior wdAutoFitWindow
    End With
  Next
  Application.ScreenUpdating = True
End Sub
