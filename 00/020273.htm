To synchronize your password, Azure AD Connect sync extracts your password hash from the on-premises Active Directory.

 

Based on the “Password expiration policy” in this article, if a user is in the scope of password synchronization, the cloud account password is set to "Never Expire". The on-premises password will always overwrite the password of the cloud password.

suzanne s. barnhill say:
Ctrl+PageDown executes the BrowseNext command. What this does depends on the "browse object" currently selected. When you first start Word, this is set to "Browse by Page," so the command executes Next Page. When you use the Find dialog, then BrowseNext becomes Next Find/Go To.
