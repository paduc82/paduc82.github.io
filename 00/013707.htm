Conditional formatting of cells based on the value in a cell

paulcordts say: You can use a formula to conditional format.
From the conditional Formatting Select the Use a formula to determine which cells to format.

For data like this
     A          B           C            D
1   25         x          Yes         32
2   37        35           y           76

Start in Cell A1 and apply this formula for you formatting
=COUNTIF($A1:$D1,"=x")<>0, pick the shading you want for x
In the applies to section, pick your entire data range (A1:D2) in this case

Again in cell A1 do the same for y, just change the formula to COUNTIF($A1:$D1,"=y")<>0

The key here is to set up the formula to only count in a single row of data.  Using $A1$D1 does this,  when the formatting is applied to the next row the 1 is read as 2, just as absolute and relative references when copying formulas.  When you set up the formula use the row that you are starting in as the row.  Adjust the columns to match your data range.

The Contextures site has a more detailed explanation and video

john mcghie say:

This is dead easy, but you need to know two things:

1)  The spelling engine uses the language marked on the text. It will change language word-by-word depending on what language you have marked the text with.

2)  The spelling language is set in the DOCUMENT, not in Word. 

Look in the Word help for the topic "Check spelling and grammar in a different language". Study all  the topics in that article.

Disregard the part about changing the spelling default: that affects only documents you create in the future.  The documents you have now have their spelling language already set: you need to change this in each document.

Because you have a book, you will have formatted the document using styles.  You can set the spelling language in the styles.  Usually, if you set the language in the Normal style, this will change the whole document.  However, you need to check that the other styles pick up the change.  It is possible for each style to have its own language set.

To begin, go to Format>Style>Modify>Format>Language and set the language you want to use.

If someone has market the language using direct formatting, then you will need to set the language for the whole document.

* Select all the text in the document (Command + a) then use Tools>Language... to set the language you want to use.

If you set the language using direct formatting, that overrides all the styles in that document. Normally I do not recommend using direct formatting for anything, especially in a book: it stuffs up the formatting in the whole document.

Hope this helps

buck jeppson say:

How can I make the header unprotected and leave the rest of the document protected by password?

I have a Word document in which I have protected some formulas in a table but everything else can be edited. However, when the doc is protected I can't edit the header. How can I add the header to the unprotected area?
jay freedman say:

Buck Jeppson wrote:

I have a Word document in which I have protected some formulas in a
table but everything else can be edited. However, when the doc is
protected I can't edit the header. How can I add the header to the
unprotected area?

If you and everyone else who edits the document has Word 2007 or later, there is a way. If anyone with Word 2003 tries to edit the document, though, they'll find the whole document to be locked.

First remove whatever protection (I'm guessing it's "protected for forms") is in the document now.

Click the Protect Document button (on either the Review tab or the Developer tab) and choose Restrict Formatting and Editing.

Under Editing Restrictions, check the box for "Allow only this type of editing" and set the dropdown to "No changes (Read only)".

Select the part of the document before your formulas, and check the box next to Everyone in the Exceptions section of the pane. Select the part after the formulas, and again check the Everyone box. If there are editable areas between the formulas, do the same for each one. Finally, open the header pane, select its contents, and check the Everyone box.

Click the "Yes, Start Enforcing Protection" button in the pane, enter your password, and click OK.
