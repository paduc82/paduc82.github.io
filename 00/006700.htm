javed a say: In reply to deleted message

Assume you need automation to create IF formula for lookup purpose and Value/Result pair will change.

Below Function will create a String of IF formula when a table is given as input. You need to value paste and it will work like actual Excel IF Formula

Say you have a table A1:B4 with value/result and your test value is in D1, then in E1  type  =Myif(A1:B4,D1)

Function MyIf(rTable As Range, rSource As Range) As String
    
    Dim strFormula As String, strAdd As String
    Dim i  As Long
    
    strAdd = rSource.Address(0, 0, xlA1, 0)
       
    For i = rTable.Rows.Count To 1 Step -1
        strFormula = strFormula & "if(" & strAdd & "=" _
        & rTable.Range("a" & i).Address & "," & rTable.Range("b" & i).Address & ","
    Next i
    
    strFormula = "=" & strFormula & "0" & String(rTable.Rows.Count, ")")
    
    MyIf = strFormula

End Function
