No - it won't work. For a start there is no need to repeat all that code when only a single digit changes, and you have also duplicated labels and several exit points.

The following will be closer - however on initialising the form, unless you are going to store the last used value to be recalled next time, only one option button will be selected and that will be the one you choose as default, so the rest is superfluous.

You will have to call the code whenever you change the option button, so it would be better to set the values as a function and call the function from the change events of the buttons. Even so there will be a lot of opening and closing of the source document which may not be a good thing.

When it gets this complex, it is better to manipulate the form in the calling macro. You can then leave the data source open throughout and call upon it as required.

Rather than use option buttons, take a look at cascading list boxes as should in Greg's page.

Option Explicit
Private Sub UserForm_Initialize()
Dim arrData() As String
Dim sourcedoc As Document
Dim i As Long
Dim j As Long
Dim myitem As Range
Dim m As Long
Dim n As Long
Dim iTableNo As Long

Select Case True
    Case Option1.Value
        iTableNo = 1
    Case Option2.Value
        iTableNo = 2
    Case Option3.Value
        iTableNo = 3
End Select

Set sourcedoc1 = Documents.Open(FileName:="D:\Data Stores\sourceWord.doc", Visible:=False)
'Get the number of list members (i.e., table rows - 1 if header row is used)
i = sourcedoc.Tables(iTableNo).Rows.Count - 1
'Get the number of list member attritbutes (i.e., table columns)
j = sourcedoc.Tables(iTableNo).Columns.Count
'Set the number of columns in the Listbox
ListBox1.ColumnCount = j
'Load list members into an array
ReDim arrData(i - 1, j - 1)
For n = 0 To j - 1
    For m = 0 To i - 1
        Set myitem = sourcedoc.Tables(iTableNo).Cell(m + 2, n + 1).Range
        myitem.End = myitem.End - 1
        arrData(m, n) = myitem.Text
    Next m
Next n
'Use the .List property to populate the listbox with the array data
ListBox1.Clear
ListBox1.list = arrData
'Close the source file
sourcedoc.Close SaveChanges:=wdDoNotSaveChanges
End Sub
