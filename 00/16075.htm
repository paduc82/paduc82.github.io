terryjl01 say:

A VPN is a "Virtual Private Network".  When attached to a VPN, a virtual "tunnel" is created between your computer and a private network.  Your computer is then assigned an IP address, masking your network's IP address. When browsing the internet, your computer's internet activity is hidden from your actual IP address.

There are a number of VPN providers.  One provider that has received great reviews is NordVPN.  I've been testing and using them in all my insider builds.  The process is simple to setup. Software is downloaded to your computer.  All you have to do is run the application and connect.  NOTE:  due to the anonymity provided, some sites will not connect properly. This is especially true for sites that utilize security (e.g. banks) protocols that verify your IP address.  It's also true for some streaming services.

jeeped say:

Depending upon the consistency of the data you bring into an empty worksheet, the data records could be transposed into columns.

If it can be reasonably assumed that the data is coming into column A with each record owning three rows, the formula in C2 would be,

=OFFSET($A$1,(ROW(1:1)-1)*3+COLUMN(A:A)-1,0)

This can be filled or copied both right and down to split off each record into appropriate columns. Blank rows will not affect the parse beyond putting in zeroes that can be removed with a Find & Replace later but records that are three rows in some places and four rows in others will foul up the sequence.
