trusted file - Microsoft Community

cynthia holland say:

trusted file

How do I convert an old Word 2007 trusted file to be able to read it on Word 2013?

[Moved from: Office / Word / Windows 10 / Unknown/other]

jay freedman say:

You should not have to "convert" anything from Word 2007, as Word 2013 can open older documents. I'm not sure what "trusted" has to do with it.

If you try to open a document -- whether from an older Word version or not -- and it displays garbage characters or error messages, the document may be corrupted. Is that what you're seeing? Or does the document open in Protected View, or not open at all?

There is one thing to check out: In Word 2013, go to File > Options > Trust Center > Trust Center Settings > File Block Settings. If either of the lines that mention Word 2007 have check marks in the Open column, click to remove those checks. Those boxes aren't checked by default, so clicking the Restore Defaults button would also do.

The concepts of trusted documents and trusted locations are concerned with whether "active content" (macros, ActiveX controls, and some other things) is allowed to run or is blocked. You can trust individual documents, or you can store otherwise-untrusted documents in a trusted location; in either case, Word's security procedures won't be applied to those documents. That treatment should be reserved for documents that you create yourself, and for documents whose creators you know and trust (and you're sure they haven't been modified by malware).

How do you turn on and or off the email address autofill in Outlook?

Tools>Options>E-0mail Options>Advanced E-mail Options. Check or uncheck the box labeled "Suggest names while completing To, Cc, or Bcc fields"

Is there a way to view the addresses which are saved? I dont mean the autofill of my addresses, but the autofill of email addresses which were used in the past.

There is only one "autofill" and that's the autocompletion cache that remembers the last 2,000 addresses to which you've sent mail. There are tools available that can export the file (like this one: http://nk2.info/ , which is free) or others that allow you to edit the file (like this one, which is not free: http://www.ingressor.com/).
