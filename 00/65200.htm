1.Unzip the sdutil.zip file and rename it from sdutil.ex_ to sdutil.exe
2.Copy the sdutil.exe to the root of the driver which contains the backup files
3.Open the folder Catalogs located bellow the folder BackupSet “aaaammdd” “xxxxxx”
4.Copy the file GlobalCatalog.wbcat to the root of this HD
5.Go to Start -> Execute and type CMD.exe
6.At this “DOS” window type: sdutil mediaid GlobalCatalog.wbcat [HD letter (as an example D:\)]
6(2) At this “DOS” window type the HDD letter of your backup disk (as an example F:\).  This will change the following command to execute directly on the backup drive rather than addressing that drive with a command on C:\  type sdutil mediaid GlobalCatalog.wbcat [HD letter (as an example F:\)]
7.A new MediaID.bin is created
8.Try to restore the backup as usual using the advanced restore option and next the option restore an older backup created in this machine
9.If done: it’s over!
10.If it fails: Map a network driver at your LAN
11.Copy the MediaId.bin file to the root of the previous mapped LAN driver
12.Copy the whole folder BackupSet “aaaammdd” “xxxxxx” to this same driver
13.Try to restore the backup as usual using the advanced restore option and next the option restore backup created in another machine
14.At the next menu, point the backup folder at your LAN mapped driver and start the restore process
15.Now you shall have your backup proper restored at the right machine
16.Good Luck!

Questions about any version of Microsoft Money should be asked in https://social.microsoft.com/Forums/en-US/home?forum=money

Archiving Money data is not recommended. Money is designed to work with a single data file.

If you choose to create an archive, it can be done by creating a Money data file containing only the data you want to archive. To do this the currently used data file must be copied and renamed.

If you want to retrieve data into another data file you will have to export the data as qif and import it into the new data file. This is tedious. Money data files cannot be merged and data cannot be copied and pasted between them.

Dedicated Video RAM

I bought a game called Just Cause 3. My computer is able to run it but there is a problem with the dedicated Video RAM. i have 1536 mb but it needs 2048 mb. Is there any way i can increase my amount of Video RAM Mb's? P.S i have a windows 10 pc. 

bossdweeb say:

Short answer... No.

Long answer.  You cannot add more VRAM to a dedicated graphics card.

Machines that use Integrated graphics commonly have an option to allocate up to 2GB system RAM to games,

but this is not actually VRAM.  Integrated graphics run via 'shared' system RAM and have no VRAM.

Dedicated graphics cards also make use of system RAM as a temporary storage for game data which can be called

on when required by the gpu (loading new areas or levels), but when dealing with the graphics settings in a

games setup only VRAM is taken into consideration, as this is what is in use in the part of the game you see on screen.
