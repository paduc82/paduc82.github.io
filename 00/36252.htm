frederik long say:

Having a single admin account is as risky as having a single house key. If something goes wrong with it then you're in serious trouble. The following commands let you create a spare admin account and delete the unwanted account (assuming that it is a local rather than a Microsoft account).

  1. Log on under your admin account.
  2. Create a desktop shortcut for cmd.exe.
  3. Right-click this shortcut, then left-click "Run as administrator".
  4. Accept the challenge.
  5. Type these commands and press Enter after each:
    net users
    (to get a list of all account names)
    net user "John Doe"  /del
    (to delete the "John Doe" account)
    net user "Jane Dew" 12345678 /add
    (to create an account for Jane with a password of 12345678)
    net localgroup administrators "Jane Dew" /add
    (to promote Jane's account to admin level)
    net accounts /MaxPwAge:unlimited
    (to prevent the passwords from expiring)
  6. Log on under Jane's account to test it, then make a written not of its details.
