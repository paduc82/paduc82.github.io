    afzal taher say:

    Hi Mostafa Hijazi,

    This issue occurs because keyboards have to discriminate between the keys that are pressed. Keyboards do this by scanning the keys that are electrically arranged in a set
    of rows and columns. When you press multiple keys at the same time, it creates the most demanding scenario for the keyboard to uniquely detect keys. There are many combinations
    of keys for which it is impossible to detect all the keys that are pressed. When you can press a set of keys with one finger, such as "2," "W," "S," and "X," the keys generally belong to the
    same column. Usually, you cannot combine these keys as a valid multi-key chord. Keyboards are more likely to accept keys that you typically press with different fingers.

    Some keys are typically used with other keys as alternative keyboard shortcuts. These keys include SHIFT, CTRL, ALT, and the Windows logo key. You can generally use the following
    key combinations with most keys to obtain a valid three-key combination:

    • CTRL+ALT
    • CTRL+Windows logo key
    • CTRL+SHIFT
    • ALT+Windows logo key
    • ALT+SHIFT
    • SHIFT+Windows logo key

    http://support.microsoft.com/kb/909528
