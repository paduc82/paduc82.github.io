Hi RGOOD33,

 

Yes, they are related there are several causes of Code 39 errors:

·        A required device driver is missing.

·        A required binary file is corrupted.

·        There is a problem with the file I/O (Input output) process.

·        A driver that is referencing an entry point in another binary file is unable to load.

Step 1: Uninstall and reinstall the network adapter drivers

You must be logged on as an administrator to perform these steps.

a)   Open Device Manager by clicking the  Start button, clicking Control Panel, clicking System and Maintenance, and then clicking Device Manager.‌if you are prompted for an administrator password or confirmation, type the password or provide confirmation.

b)   In the Device Manager, locate network adapter and then double-click the device name.

c)   Click the Driver tab, and then click uninstall Driver  and follow the instructions.

Download (from a different computer) and install the latest network adapter drivers from the computer or device manufacturer’s website.
