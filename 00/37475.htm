 "What's the deal with Windows 8, do I need an optimizing software such as CC++ or does the Windows 8 software come with all the tools?"

 

My reply:

 Windows 8 comes with new improved "Optimization" tools built into the operating system. which can be assessed/modified via Action Centre, then expand Maintenance. (should be set to automatic)

Also if you click on the folder on your task bar (from your desktop) then select Computer and select your drive, you will see the Drive Tools and Manage options, if you click on this you will see the Optimize feature as well as other tools to improve your system. I recommend you analyse before just clicking as you might not need to perform any tasks, as Windows 8 should be doing all the hard work for you. In short you should not need any other third party software to optimize. My opinion is that they never really work and cause more harm than good anyway. "How about DirectX, do I need to download it or is it integrated? "

 

My reply:

Direct x 11 is integrated with Windows 8. You do not need to download it separately.
