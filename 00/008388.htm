How to manage "my templates" in Office 2011 for Mac? - Microsoft Community

antoon naus say:

How to manage "my templates" in Office 2011 for Mac?

I can't figure out how to delete templates I've created in Word 2011 for Mac.
Does anyone have a clue?
Regards,
Antoon Naus
ed zollars say: Microsoft and Apple have conspired to make this a bit more difficult than you might expect.  There's no easy way in Word to simply say "manage my templates" or even figure out where they are located.

Apple, starting I believe with Lion, decided to up the ante in the "where are my templates?" game by deciding the Library folder was something users really shouldn't see.

The templates are located in:

/Users/your user name/Library/Application Support/Microsoft/Office/User Templates/My Templates

The easiest way to get there is to open Finder and pull down the "Go" menu.  With that menu displaying, hold down the Option key and you'll see Library added to the list of places you can go.  Select that option and it will open your user library folder.

Now go to Microsoft, then Office, then User Templates, and then to My Templates.  

You have now found where Office has stashed your user templates.  You can delete any you want to from there and Office will no longer show them to you.

Microsoft plays a similar (and actually a bit more involved) game of hiding user templates in the Windows version of Office.  

jim gordon mvp say:

I think managing your templates is a bit easier than Antoon described. Although you can use his method, there is an easier alternative.


Go to Word Preferences. Click on File Locations to display the file location preferences. Double-click on User Templates. Viola! The user templates folder displays in Finder.

Hi,


Have a go with this. It takes each row and inserts n rows below that and copies the data down into those rows.


Note at the start of the code there are 3 parameters you can set:-


The column to work down, currently A

No of rows to insert, currently 3

Start row, currently 2



Sub insertrow()
Dim MyColumn As String, x As Long
Dim RowsToInsert As Long
Dim StartRow As Long
'Change this to any column you want
MyColumn = "A"
'This is the rows to insert
RowsToInsert = 3
''This the topmost row of the data
StartRow = 2
For x = Cells(Rows.Count, MyColumn).End(xlUp).Row To StartRow Step -1
    Rows(x + 1).Resize(RowsToInsert).Insert
    Rows(x).Copy Rows(x + 1).Resize(RowsToInsert)
Next x
End Sub
