3. Select the folder "A team" in the library, click ... next to the folder name > SHARE > Shared with > ADVANCED > Stop Inheriting Permissions > Grant Permissions > type the group name of A team > click SHOW OPTIONS > choose the
4. Go to each of the files in the folder "A team", click ... next to the file name > SHARE > Shared with > ADVANCED > Stop Inheriting Permissions > select the group name of A team > Edit User Permissions > choose
