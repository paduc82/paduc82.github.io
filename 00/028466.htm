I suspect you meant to say you used Insert > ACTION SETTING > Run Program ... etc.

192.168.1.1 is IP address used by several router manufactureres as default address for the router itself.

This address is mostly used by Linksys routers as well open-source linux-based firmwares that are compatible with Linlsys routers. As a consequence 192.168.1.1 is likely to be set as default gateway on computers connected to the router.

http://192.168.1.1 also usually contains web-interface that controls router's configuration. If you are unable to access this address it's likely that you have some kind of network configuration problem.
