    1. Select Insert> Break> Section Break (Next Page)
    2. Go to Insert> Footnote
      1. Confirm that Endnote is the selected option
      2. Choose End of Section
      3. Choose Continuous numbering (optional)
      4. Click Apply

    1. Check the Suppress Endnotes box
    2. Confirm that Apply to is set to Whole Document
    3. Click OK
  3. Place the insertion point in the last paragraph of body text
    1. Go back to Format> Document – Layout
    2. Clear the Suppress Endnotes check box
    3. Confirm that Apply to is set to This Section
    4. Click OK
  4. Go to what now is the end of the document & insert your Appendix
