Excel does convert entries such as 31-10 to a date. 31-10 is interpreted as 31 October in the current year. amit tandon say:

To clarify, the above formulas also return UNIQUE no. of Companies besides accounting for "Yes" in Either of the 2 columns B / C. So if column A mentions "Microsoft" twice the above formulas will count this as one.

In case you only want to account for "Yes" in Either of the 2 columns B / C and there is NO possibility of duplicate companies in column A then you may use either of the shorter formulas:

1. Array formula (Ctrl+Shift+Enter):

=SUM(IF(FREQUENCY(IF((B2:C5="Yes"),ROW(A2:A5)),ROW(A2:A5)-ROW(A2)+1),1))

OR

2. Array formula (Ctrl+Shift+Enter):

=SUM(--(MMULT(--(B2:C5="yes"),TRANSPOSE(COLUMN(B2:C5))^0)>0))

This would be similar to the formula already mentioned in this thread wherein the manual "{1;1}" is replaced by "TRANSPOSE(COLUMN(B2:C5))^0" which obviates the need to manually alter the formula if say the number of columns increases to 3 (from B:C to B:D).

cpitchford say:

How Maemo works with folders

 

Maemo / The N900 uses magic folder names..

 

If you open the file manager on your phone, you'll see a list of folders. There are some magic ones:

 

Audio Clips

Documents

 

are two examples..

 

These folders aren't really called this.

 

"Audio Clips" is called ".sounds"

"Documents" is called ".documents"

 

One reason it does this magic renaming is locale.. If you're phone is set to use the english language, you'll see them as above.. However, if you change them to a different language, the names of these folders changes.

 

In English, I can see "Documents"

In German I can see "Dokumente"

 

In both cases, this magic label points to ".documents"

 

 

What does this mean for you?

 

Strangely, in the world of Unix files and folders whose name starts with a . (ie .documents) are usually treated as "hidden". You're Blaupunkt may be honoring this tradition

 

Unlike your N900, it doesn't know that .sounds is not supposed to be hidden, and contains your music files so it's skipping over it

 

Why does it work with the N97?

 

The N97 does a very similar thing to the N900.. It gives the folder containing the music a different, fake name:

 

My N97 shows a folder called "Sound Files" in the file manager app, but when I connect it to my PC using mass storage mode, I can see the folder is actually called "Sounds"

 

So regardless of what the folder looks like on the phone's file manager application, the real physical location is different:

 

On the N97, the music is stored on e:\Sounds

On the N900, it is stored in e:\.sounds

 

the .sounds folder is treated as hidden, the Sounds folder is not..

 

What can you do?

 

Plug your phone into a PC using Mass-Storage mode..

Copy a single MP3 onto your phone and put it in the root folder of your N900 ( e.g e:\ )

 

If you then try your phone on your Blaupunkt, does it see the single MP3 file as you would expect?

 

If yes, then I've probably identified the problem and we can look at a possible solution (its tricky)

If not, then its something else and I'll shut up :)

No, thankfully.. the SD card is a but more "freestyle!"

 

There is a config file (don't edit it.. seriously.. if you change only this file it will break other things.. seriously)

 

$ cat .config/user-dirs.dirs
XDG_DOCUMENTS_DIR="$HOME/MyDocs/.documents"
XDG_MUSIC_DIR="$HOME/MyDocs/.sounds"
XDG_PICTURES_DIR="$HOME/MyDocs/.images"
XDG_VIDEOS_DIR="$HOME/MyDocs/.videos"
NOKIA_CAMERA_DIR="$HOME/MyDocs/DCIM"
NOKIA_MMC_CAMERA_DIR="/media/mmc1/DCIM"

The red items refer to magic folders on the internal flash.. The blue item is the only magic item on the SD card..

 

So... if you put all your music on an SD card (maybe in a folder called Music) it may be seen by the Blaupunkt..

 

However...

 

There is no guarantee the Blaupunkt supports TWO flash drives! Remember, if you plug your phone in using USB storage mode, it appears as a flash card reader with two slots.. the first is the internal flash storage, the second is the SD card.. The Blaupunkt may only use the first device.. it may not even see the SD card

albert d. kallal say:

As mentioned you can create any new "custom" function in access. These functions can be used anywhere inside of a query, forms expressons, and even in addtional functions and code you write.

 

All you do is create a standard code module, and then inside of that code module you can add as many new "custom" functions. Those new functions can be a combination of existing functions cobbled together to build something that doesn't exist.

 

For example, let's say that we needed a function that returns the first Friday as a date value of current month. Inside of a code module, you would use the following code:

 

 

 

Public Function FirstFriday() As Date

 

  Dim dtFirst     As Date        ' first day of month

 

  dtFirst = DateSerial(Year(Date), Month(Date), 1)

  

  ' dtfirst is now the first day of the month.

  ' Use day of week 6 (friday)

  ' less the starting day of week.

  ' to offset this value to first friday

  

  FirstFriday = dtFirst + (6 - Weekday(dtFirst))

  

  

End Function

 

Now, in code, or even in a query, we could go:

 

Select firstName, LastName, firstFriday() as Friday from tblcustomers

bob jones aka: cybertaz say:

Initially, Word inherits Paper Size from what is specified in System Preferences as the Default paper size for the Default Printer. Make certain that it is set correctly... IOW, you only should have to change the default in Word if it differs from what you prefer as the global default for all programs.

In order to set the default Paper Size in  Word are you following the procedure described in Word Help?

Change paper size - Word for Mac - Office Support - Office 365

If you continue to have a problem please explain exactly what steps you're taking & specifically what happens - right or wrong. Be sure to specify your version of OS X as well as your present update level of Office 2016 (go to Word> About Word for the Version & License info)... and check the printer mfr's support site to see if there is an updated driver for your model.
