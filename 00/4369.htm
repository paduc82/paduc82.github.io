Hello,

Thank you for posting your query on Microsoft Community Forums.

If you like to delete browsing history using Internet Explorer, you can do that following the steps given below.

  1. Launch Internet Explorer.
  2. Click on Tools icon on the Internet Explorer.
  3. In General tab, find Browsing History.
  4. Under Browsing History, Check the Box  Delete Browsing history on Exit.
  5. Click on Apply, and OK.

Now try to launch Internet Explorer and check.

However, this feature is not available if you are using Microsoft Edge.
