For I =1 to 3

doug robbins - mvp office apps & services (word) say:

Use

Dim acontrol As Control

Dim i as Long

For i = 1 To 3
    For Each acontrol In Me.Controls
        If acontrol.Name = "CheckBox" & i Then
            If acontrol.Value = True Then
                ActiveDocument.FormFields("Check" & i & "Y").CheckBox.Value = True
                ActiveDocument.FormFields("Check" & i & "N").CheckBox.Value = False
            Else
                ActiveDocument.FormFields("Check" & i & "Y").CheckBox.Value = False
                ActiveDocument.FormFields("Check" & i & "N").CheckBox.Value = True

        End If
    Next acontrol
Next i

YES

Yes, you can "transfer" that unused one to a new computer.  That is actually the main "beauty" of the 365 system. 

What you can do is go into the MyAccount page:  http://office.microsoft.com/MyAccount

On the new computer, login to Myaccount with your email address, click on the "insert" button (yes, it is counter-intuitive).  Find the computer you want to take it off of and click on the "Deactivate Install" link beside that name. You have now freed up an install count. You can now click on the "install" button to get it on this "new" computer.

The beauty of this approach is that you don't actually have to uninstall from the old computer. On the Old computer Office still there, just in "limited function" mode, so you can still read and print the files.  If you need to edit using the "old" computer, you can simply deactivate one of your other installations and activate on this computer again. Instant, no install delay.
