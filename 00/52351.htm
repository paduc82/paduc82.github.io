If all else fails, try reimaging from a downloadable recovery image.

(with a different Windows computer if yours is not working properly) 
and download a recovery image for your Surface. You will need a 16GB USB stick
formatted FAT32 (use a Windows computer). Use FULL format and not quick format.
Give the volume a name like surfboot.
Unzip/extract the recovery zip to the formatted USB stick and then-
 
1. Make sure your Surface is turned off and plugged in, and then insert the USB
recovery drive into the USB port.
2. Press and hold the volume-down button while you press and release the power
button.
3. When the Microsoft or Surface logo appears, release the volume-down button.
4. When prompted, select the language and keyboard layout you want.
5. Select Troubleshoot , and then select Recover from a drive . If prompted for

Note:
Be sure to select the Troubleshoot option on this screen. Don’t select the Use a
device option. If you select the Use a device option, your Surface will reboot
from the USB recovery drive again and the reset process will start over.
6. Choose either Just remove my files or Fully clean the drive.
The option to clean the drive is more secure but takes much longer. For example,
if you’re recycling your Surface, you should clean the drive. If you’re keeping
your Surface, you just need to remove your files.
7. Select Recover.
Surface restarts and displays the Surface logo while the reset process
continues. This can take several minutes.

ralf_g say:

I ran into this same issue and it turned out to be a problem of the Users group not having permission to access the Recycle bin folder on that single partition.

The solution was to simply give the Users group those permissions.  

Open This PC or File Explorer.

Click View in the menu and select Options- Change folder and search options.

On the View tab unselect Hide Protected operating system files and change the Hidden Files and folders option to "show ...". Then click Apply. This will allow File explorer to display the hidden recycle bin folders on your drives.

Explore the drive having the recycle bin problem and double click on the recycle bin folder. It should normally open to show the contents. If instead of opening the folder you get a "do not have permission" error you'll need to change that folder's permissions to allow the Users group to have access to the Recycle bin. Right click on the recycle bin and select Properties, then go to the Security tab. Click the link to give yourself access to the Security settings.

Once that is done you can use the Advanced security settings to add "Users" to the Permissions list with all Basic permissions enabled.
