ossiemac say:

Hello Anne Troy,

This solution is somewhat different than what you asked for but I thought that you might be interested.

It adds a worksheet to the workbook and lists the full address of the errors (workbook name, worksheet name and cell address) and then puts a hyperlink beside it so you can jump to the error.

The added worksheet is always the last worksheet and you only have to re-select the sheet and select the next hyperlink.

You will always know where you are up because the cursor remains on the last used hyperlink and the used hyperlinks change color so you can see which ones you have done.

When finished just delete the worksheet.

 

Sub FindNA()
    Dim rngToSearch As Range
    Dim rngFound As Range
    Dim strFirst As String
    Dim wb As Workbook
    Dim ws As Worksheet
    Dim arrErrors()
    Dim i
    Dim strHyperLink As String
    Dim cel As Range
   
    Set wb = ThisWorkbook      'Can set to another open workbook
    
    For Each ws In wb.Sheets
        strFirst = ""
        Set rngToSearch = ws.Cells
        Set rngFound = rngToSearch.Find(What:="#N/A", _
                        LookIn:=xlValues, _
                        LookAt:=xlPart, _
                        SearchOrder:=xlByRows, _
                        SearchDirection:=xlNext, _
                        MatchCase:=False)
        
         If Not rngFound Is Nothing Then
             strFirst = rngFound.Address
             Do
                i = i + 1
                ReDim Preserve arrErrors(1 To i)
               
                arrErrors(i) = rngFound.Address(External:=True)
                
                Set rngFound = rngToSearch.FindNext(rngFound)
               
             Loop While Not rngFound Is Nothing And _
                    rngFound.Address <> strFirst
         End If
    Next ws
   
    If i > 0 Then       'If i equals zero then no errors found
        Set ws = wb.Sheets.Add(After:=Sheets(Sheets.Count))
       
        ws.Cells(1, 1).Resize(UBound(arrErrors), 1) _
                    = WorksheetFunction.Transpose(arrErrors())
                   
        Set ws = ActiveSheet
       
        For Each cel In ws.Columns(1).Cells
            If cel.Value = "" Then Exit For     'Past end of data
       
            strHyperLink = "=HYPERLINK(" & Chr(34) & Left(cel, InStr(1, cel.Value, "]")) _
                            & Chr(39) & Mid(cel, InStr(1, cel.Value, "]") + 1) & Chr(34) _
                            & "," & Chr(34) & "Click to jump to address" & Chr(34) & ")"
                           
            cel.Offset(0, 1) = strHyperLink
        Next cel
       
        ws.Columns("A:B").AutoFit
    Else
        MsgBox "No errors found."
    End If
   

 End Sub

john p spencer say:

I would make two changes to your query.  One add a column to specify if the amount is a credit or a debit and second do a UNION ALL instead of a UNION.  Union All will not attempt to consolidate the rows in an attempt to eliminate duplicates and will be faster than UNION.

SELECT  [AccountType],[Account Category],[SumofDebitAccount], "Debits" as CRDR

FROM [qy-PLreportEXp]

UNION  ALL

SELECT [AccountType],[Account Category],[SumofCreditAmount], "Credits" as CRDR

FROM [qy-PLReportSales];

Then for your Sum expression you could use something like the following.

SUM([SumOfDebitAccount] * IIF(CRDR="Credits",1,-1))

john mcghie say:
When working with text boxes, you need to be aware of where they are anchored.

Each text box exists as a "picture" in the drawing layer of the document.  And each one is anchored to a paragraph in the main text.

If you select a text box, an anchor symbol appears in the margin to the left of the paragraph to which the text box is anchored.  That paragraph can be anywhere.  However, if that paragraph gets moved off the page, or deleted, the text box is indeed "gone".

Hope this helps
