"Cost Accrue At" in MS Project - Microsoft Community

prasadnk say:

"Cost Accrue At" in MS Project

There are three cost accrual in MS project -:

1. prorated
2. start
3. end

Suppose employee  worked for  Monday, Tuesday and then on friday.

So, prorated will calculate cost for 3 days.

Now, I learnt from somewhere that start calculates cost in the beginning. But how ?  

I mean, how can you predict when is employee going to come to office or when is not ?


ismet kocaman say: Hello prasadnk,

>> If we set cost accural as START , then we will simply enter the value .. Say 250 $ ..no matter
>> how much time that task takes.. we will pay 250$ only. ( hence, MS project does not calculate
>> value like in the case of PRORATED)

I believe it will be all clear after reading the articles listed above.

  • We do not enter cost values, instead we just enter Standard Rate, Overtime Rate, Per Use Costs for the resources and specify the cost of actual work plus any fixed costs for the task to date.
             Actual Cost = (Actual Work * Standard Rate) + (Actual Overtime Work * Overtime Rate) + Resource Per Use Costs + Task Fixed Cost

  • In the above formula, only the fixed cost for a task remains constant regardless of the task duration or the work performed by a resource. So how much time that task takes affects the costs. Rates are defined as "per time unit", e.g. $50/hour
  •  How and when actual costs are automatically calculated depend on the Standard Rate, Overtime Rate, Per Use Cost, and Cost accrual settings for the resources.
  • Suppose the accrual method selected is START. You will see the total cost in Cost (scheduled cost) field and zero in the Actual Cost field until you update the schedule with actuals. As soon as you enter any piece of actual information, the Actual Cost field will display the total actual cost. In this case, we can say all the costs will incur (in Project) as soon as resources start to work but it is always your decision whether to pay money to people.   

Hope the above explanation helps.
