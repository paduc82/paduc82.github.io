Hi Rodney,

The winmail.dat file is used to preserve Rich Text formatting. Outlook uses it when sending a Rich Text-formatted message. During transport, the content of the message may be changed, preventing the receiving client from being able to read the formatting instructions. In other cases, the receiving client does not use or recognize the winmail.dat file. To resolve this issue, send your messages in either HTML or plain text format.

Change the message format to HTML, Rich Text Format, or plain text.
