charles kenyon say:
Source formatting is easy to understand.
Text only actually pastes the text from the source into the paragraph style of the selection point.

From Paste Options


 Keep Source Formatting    This option preserves the look of the original text.
Keep Text Only    This option removes all the original formatting from the text.
Link & Keep Source Formatting    This option preserves the look of the original text, and it maintains a link to the source file and updates the pasted text with any changes that are made to the source file.
Link & Use Destination Styles    This option formats the text to match the style that’s applied where the text is pasted. It also maintains a link to the source file and updates the pasted text with any changes that are made to the source file.
Merge Formatting    This option changes the formatting so that it matches the text that surrounds it.
Picture    This option inserts the text as an image.
Use Destination Styles    This option formats the text to match the style that’s applied where the text is pasted.
Use Destination Theme    This option formats the text to match the theme that’s applied to the document where the text is pasted.

jmcnay63 say:

There's one thing I've learned about Excel 2013 that can mess up your formatting, especially when opening a document created in an earlier version of Excel - and that's your "Display Setting".  If you go to:

"Control Panel\Display"

you will see the following page.

If your display is set to anything other than the default (Smaller - 100%), formatting of Cell Heights to an existing Excel file will be changed.  If you try to change the "Cell Height" back to the existing documents "Cell Height" - you'll notice the "Cell Height" options have also changed, making it impossible to reformat your new document to match your old document.

For example, say you have an existing Excel file, created from an earlier version of Excel, and in this document all the "Cell Heights" are set to "11.25".    If you open this same document in Excel 2013, while having your Display Setting set to "Medium - 125%" - you will notice that all your "Cell Heights" have been changed.  If you try to readjust the "Cell Height" back to "11.25", you'll notice that option size is no longer available to you.  Instead you will get options like "11.00 & 11.75", but no option of "11.25".   

Hope this is making sense.

The good news is, if you change your "Display Setting" back to the "Smaller - 100%" default, all your documents original formatting goes back to normal.
