pesach shelnitz say:

Hi,

Try  the following macro. Note that the field codes in the macro, including all spaces, must match your field codes exactly. In particular, I have included spaces at the beginning and end of the field codes and the required space before the backslash.

Sub DeleteEmptyFields()
    Dim i As Integer

    With ActiveDocument
        For i = .Fields.Count To 1 Step -1
            If .Fields(i).Code.Text = " DOI1 \* Charformat " _
              Or .Fields(i).Code.Text = " DOI2 \* Charformat " _
              Or .Fields(i).Code.Text = " DOI3 \* Charformat " Then
                If .Fields(i).Result = "" Then
                    .Fields(i).Delete
                End If
            End If
        Next
    End With
End Sub

The cells are changing because your references are relative (eg. C10). If you don't want your cells to change when you copy and paste the formula, use absolute reference (eg. ($C$10).

=SUM(C10*$C$23*$C$27,$D$23*$D$27,$E$23*$E$27,$F$23*$F$27,$G$23*$G$27,$H$23*$H$27,$I$23*$I$27,$J$23*$J$27,$K$23*$K$27,$L$23*$L$27,$M$23*$M$27,$N$23*$N$27)

amit tandon say:

As already mentioned you may use absolute reference with the $ sign so that copying the formula will not change cell references. Else if you copy to the right or down then column & row reference changes respectively.

You may use a shorter formula to reference simultaneous column ranges instead of adding each cell repeatedly. For your above formula, using below will give the same result:

=SUM($C$10*$C$23*$C$27,INDEX(OFFSET($D$23,,,,11)*OFFSET($D$27,,,,11),,))

This returns the SUM of (i) $C$10*$C$23*$C$27 and (ii) "INDEX(OFFSET($D$23,,,,11)*OFFSET($D$27,,,,11),,)" which are the 11 columns from D to N ie. D23*D27 till N23*N27.

Putting that SID there (which should be the SID for your own account) should clear the other Administrator message and remove the Take Control button.  Since you created another Admin account you might have to sort out which SID is actually yours (but it will probably be the one with number S-1-5-21, like mine).
