    • unable to type an email due to lag - waiting for the text on the screen to catch up to the point that it was unusable.

    amanda zhang msft say:

    Hi Yukonn,

    About your question, there are links between Microsoft Teams Planner and O365 Planner.

    First,create a plan and then it creates a group – create a group and then it creates a plan. It is the same as teams – create a team and it creates a group – which creates a plan.

    You can add a new Planner tab in teams. Adding the Planner tab to Microsoft Teams means we can create new tasks in Planner (tasks.office.com) through Microsoft Teams. The planner tab and Planner aren’t the same . In other words, adding the Planner tab means creating a new plan in Planner. I’d like to explain how this works as below:

    1. When user A creates a new team in Microsoft Teams, it will create a plan in the All plans section in user A’s Planner.

    2. If user A creates a new task in the Planner tab and assign it to himself, this task will also be created in user A’s Planner > My tasks. If the task has been assigned to another user such as user B in the team, it will appear in user B’s Planner.

    3. If user A creates a new task in the Planner tab, it won’t be created in user A’s Planner unless he is assigned, because it isn’t a plan or a task of himself but a plan of the group. However, tasks created or modified in the new bucket in A’s Planner > My tasks will be synced to the Planner tab.

    For more detailed information, you can refer to this article: 

    Microsoft Teams and Microsoft Planner–what to expect.

    Amanda
