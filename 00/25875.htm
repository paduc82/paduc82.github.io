steve rindsberg say:

I'm guessing that you have a very good idea how to use Masters and layouts already, Jonathan.  It certainly sounds that way from your explanation with just a soupcon of (entirely justified) rant for spice.

Fact of the matter is that MS made a mess when they changed the way footers/headers work going from 2003 to 2007 and onward.  Put otherwise:  It's not you, it's PowerPoint that's at fault here.

The quick fix will be along these lines:

Go to the headers/footers dialog while in normal view, remove all headers/footers/etceterators and Apply to All.  Do the same in Master view.

Check to verify that there are no stray bits of headerfooter text on your slides or masters; delete them if so.

Then in master view, add a text box where you want, say, a slide number.  Type any text you want into the text box then on the Insert tab, choose Slide Number (or Date/Time if you'd rather a date time footer).  Format to taste. 

If you want this to appear on every slide, put it on the topmost master (the Slide Master itself).  If you'd rather have it appear on some but not all slides, put it on one of the layouts then copy it to the others where you want the header/footer to appear.

Now they'll act more like the ones you're used to friom 2003.  Controllable but not selectable from the slides themselves.

There are several types of password protection, all of which can involve passwords & can be used in virtually any combination. You haven't indicated which type of password protection you want to apply:

  • Password to Open (File Level)
  • Password to Modify [Read Only] (File Level)
  • Protect Worksheet (cells)
  • Protect Workbook

I'd suggest that you go to Help> Excel Help in the main menu & search using the term protection. You'll find complete & accurate descriptive information on protection for Worksheets & Workbooks. However, the article on File Level protection (which is what Karl referred you to) accurately describes what that type of protection does the steps are for a Windows version. They do not pertain to Mac Excel... to begin with, step 1 is "Select File> Info" which doesn't even exist in the Mac version of the program. Reply here with complete descriptive details if you have any additional questions or need further clarification.

For File Level password protection click the Options button in the Save or Save As dialog then supply & confirm either or both. Include the check for Read-only (recommended) if you want a prompt to advise users when the file is opened. It gives them the choice of supplying the password in order to be able to modify the file or simply to open it in read only mode.

NOTE: Excel password protection cannot be bypassed so if you forget the password you will bot be able to access the file. Keep a record fo the passwords you use.

It is not feasible to configure Exchange Online to do that. Theres no such setting from Exchange Online side. Sorry for any inconvenience this may cause.

However, I have re-considered your requirement, and I suppose you might try a not-official-recommended way to see if it helps. Generally, we suggest Office 365 users set one MX record only. If your Exchange Server 1 and Server 2 can be distinguished by using different MX records, you may try to set two MX records with different priority levels.
