A visual check is very difficult to do: the only reliable way to isolate these problems is to take a number that you know is giving problems and use the LATEST version of Search (so normally that's NOT the version that came with the phone but one you've downloaded and installed in the last couple of months) to search for PARTS of the number.

The reason you have to search for parts of the number is to handle number formatting differences. If you had:

"0208 123 4567"
"020 8123 4567"
"02081234567"

in your phone this would cause a problem, however searching for "0208 123 4567" would NOT find either of the other two while in this case, searching for "4567" would find all 3.

You would probably have to try a number of different parts of the number depending on how many different formats you use.
