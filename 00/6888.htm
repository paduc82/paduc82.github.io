daniel g. grau say:

This is a hard one, because it's affecting all Office applications. Possible causes I can think of:

  • Bad Office preferences
    • Quit all Office applications (I know, I know, but it's part of the recipe)
    • Go to the Finder
    • Press the Option key and select Go > Library
    • Locate the Preferences folder
    • Move all files named "com.Microsoft … " to another folder (yes, let's be radical). You can create a new folder in the Desktop.
    • Test the Office applications
      • If they work and you don't want to loose any good preference files, place them back judiciously. For instance, don't put back com.microsoft.office.plist. 
  • Bad permissions.
    • Fire up Disk Utility in Applications > Utilities and repair permissions in your hard drive.

  • Duplicate fonts
    • Office is duplicate-intolerant. Open Font Book in Applications and make it look for duplicates. Alto tell it to validate fonts, just in case.
    • If any are problems are found, disable those fonts. That may not be enough to coerce Office into proper behavior and duplicate deletion may be required
  • When you last updated, did the update complete as planned? If not…
    • Try to reapply the update. For this, you'll need the .dmg file. If you no longer have it, download it here (find a list at the end of the page).
    • If that fails to fix Office, download Office and run the installer.
    • If this fails as well, you'll need to uninstall (there are official MS instructions for this) and install again.
  • Have you recently installed any software that could be considered system-wide (i.e., usable from more than one application)? If so, it may be affecting Office.
    • To check this, restart your Mac while pressing the Shift key. Release it when you see the gray thermometer. (This is called Safe mode and will load a reduced set of fonts, add-ins, etc.)
    • Test the Office applications.
      • If they work, you'll have to locate a newer version of the problem-causing software (and I don't mean Office).
    • Another way to test for this possibility is to create a new user account and test Office there. If it works, then something is affecting only your regular account. However, if there is any system-wide problem software installed, it will also affect the new user.
