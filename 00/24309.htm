Thank you,

Eli

bob jones aka: cybertaz say:

There is no facility in Word to save only a portion of a document as a new Word file, but there are at least 3 alternatives available:

  • Copy what you want to save separately, then paste to a new blank document & save that as a new file.
  • Alternatively, use Save As to create a new copy of the document & delete from it the part you don't want to retain.
  • If the requirement isn't for an editable Word document the page range can be exported as a PDF using the Print function.
