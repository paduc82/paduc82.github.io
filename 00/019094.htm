CloudNet is a dangerous trojan. First off, what I would recommend doing is to reboot in Safe Mode. To do that, press the Shift key as you are pressing the Restart button.   • Press Windows key and type Task Manager.
  • End any suspicious tasks. (This can damage your system if you are not careful.)
  • Press Windows key and R at the same time. Type msconfig and press enter.
  • Remove any unknown file from the startup tab. If you don't know what the application is and/or its developer is Unknown, remove it from startup.
  • Press Windows key and R again. Type Regedit and press enter.
  • Press control and F at the same time and type in "Cloudnet."
  • Right-click and remove any file with "Cloudnet" in the name.
  • Remove all browsing data from the browser. If you don't know how to do this, you can search how to do it on that browser.

each of the other worksheets might or might not have the number in (Column A)but if they do I want it to sum each of the quantities in (column B)on the following sheets onto the summary sheet

jeeped say:
Column A       Column B
1060801        100
1060802        200
1060805        300
1060809        600
1060810        900


With ten possible worksheets to collect data from, your solution may best be solved with a strictly 'brick-by-brick' approach of stacking successive SUMIF() functions. Assuming a named Summary worksheet together with ten individual worksheets (Sheet2, Sheet2 ... Sheet11) and Summary!A2 contains 1060801 (as above)  then try this in Summary!B2,

=SUMIF(Sheet2!A:A,Summary!A2,Sheet2!B:B)+SUMIF(Sheet3!A:A,Summary!A2,Sheet3!B:B)+SUMIF(Sheet4!A:A,Summary!A2,Sheet4!B:B)+SUMIF(Sheet5!A:A,Summary!A2,Sheet5!B:B)+SUMIF(Sheet6!A:A,Summary!A2,Sheet6!B:B)+SUMIF(Sheet7!A:A,Summary!A2,Sheet7!B:B)+SUMIF(Sheet8!A:A,Summary!A2,Sheet8!B:B)+SUMIF(Sheet9!A:A,Summary!A2,Sheet9!B:B)+SUMIF(Sheet10!A:A,Summary!A2,Sheet10!B:B)+SUMIF(Sheet11!A:A,Summary!A2,Sheet11!B:B)

That formula may seem long-winded but it is actually quite readable. A VBA solution to SUMIF() across multiple worksheets is also possible but it has some drawbacks, most notably dynamic updating.

Where are the PDF files? ﻿Having PDF files on the Windows - Microsoft Community

a. user say:

Where are the PDF files? ﻿Having PDF files on the Windows Phone﻿.

Answering questions like:

Where the Downloaded PDF Files are stored?

Why are my files disappearing?  Where do the downloads go? How do I minimize a file?

1. USING SD CARD

There is a limitation of 50 PDF files per folder when you add a big quantity of PDFs on the SD card.

It means that you can't read the 51st file on the same folder using the PDF reader app, so you will need to sort the files into folders containing 50 files each.

7 people found this helpful

a. user say:

2. The Folder Documents

If the PDF's are on the folder Documents, they will be found by the office/phone/documents, but read by the Adobe reader / PDF reader (the system will prompt a message asking on what reader you wish to open it). You can have as many files as you want there. No need to separate in folders of 50 files.

If the files are shown on the Office documents, the Adobe reader / PDF reader won't show those files on its Library. 

a. user say:

3.  Phone's internal memory

Any file saved into sub-folders of Documents will not show up on the Office Mobile document's page on the Phone nor on the Adobe reader's documents.

a. user say:

4. Downloading and sharing a PDF

* You can't explore downloaded PDFs when connecting with the Windows Explorer. If needed share with yourself by email then organize in folders on your SD card or add to the documents folders.

If you decide to download a PDF from the Internet, you will find it by opening the App's Library (Adobe Reader or another PDF reader).

a. user say:

5. Deleting a PDF

If you wish to delete a PDF from the phone, you can either Tap and hold the file to open the option Delete File or choose the list button on the very bottom of the page then check the box beside each file which wishes to delete, then tap the Bin icon.

a. user say:

5. Transferring via Bluetooth

Similar to a download made from the Internet Explorer the file transferred via Bluetooth will be found on the PDF reader App installed on the phone.

Hi John

You mean the column width, the row height, or both?

In Table Layout/Table/Properties you can set an exact row height in the Row tab.

Columns are a bit trickier, unless you've used VBA to create that table and explicitly set it to a default behavior that pre-dates the "Internet behavior" that was introduced in Word 2000 and later. But...

That should stabilize things.
