Hi Ahmad,

Theoretically, Word/Excel will delete all of its temporary files when you exit the program. However, these temporary files might be left if Word hangs, if you exit the program abnormally, or if you left a large number of items in the Clipboard when you exited the program.

Would you follow these steps to see if there is any improvement?
1. Restart your computer. The temporary files are supposed to be deleted once the program is completely closed.
2. If the temp files still exist, please open the folders where you normally save documents. In File Explorer, unselect "Don't show hidden files and folders". Now when you look under those saved folders, you will see the hidden ".tmp" files. Delete the .tmp files.
3. In File Explorer > search bar, type ".tmp" to get a list of all these file types. Delete all the .tmp files.

If the issue persists, please let me know

1. Your Windows edition.

2. Your Office version number. (Click File > Account > next to Update Options, capture a screenshot of the version number )

Pick a meeting Time / Date - Microsoft Community

reagan ewing say:

Pick a meeting Time / Date

Need to schedule MANY appointments. Want people to pick their own times/days. How can I send invitations to 100+ individuals to choose their own appointment times in Outlook Group Calendar?
vanessa ouyang msft say:

Hi Reagan,

 

If you want to send multiple meeting requests to someone so they can pick time they are convenience, please follow the steps below:

  1. Create an appointment:
    1. Include the Location, Date and Time of the meeting in the Subject(important).
    2. Invite yourself and mandatory attendees(important).
    3. Send the appoint to yourself.
  2. Go to your calendar and double click appointment and save as .ics file to your local PC.
  3. Repeat loop for other times appointment in the series.
  4. Create a new email message.
  5. Attach all appointment .ics files including every time to the message.
  6. Send the message to group.
  7. Let recipients choose only one meeting by double clicking on the appropriate attachment and accept it.

 But the workaround has some limitations:

a. The meeting can reoccur. However, the time must be set as a specific period in advance. We cannot force invitee to select unique time period and just accept one appointment. 

b. The appointments cannot show in Group calendar.

Thanks,

Vanessa
