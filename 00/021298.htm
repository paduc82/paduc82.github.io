lz. say:

Hi Andrei

Passing dynamic value/parameter to a Query has been answered n times here & there... Assuming initial URL is http://ABCD/DEF/IJK

  1. Follow Justin's instructions to build the initial query and enter the above URL (http://ABCD/DEF/IJK)
    if you look at the M code in the Power Query Advanced Editor, somewhere at the beginning you will see something like Web.Contents("http://ABCD/DEF/IJK") - hard coded URL between quotes
  2. Load the result into your XL workbook

Done. Hope this helps

guessed say:

The cross reference is always to the caption rather than to the image which is why we see this behaviour.

The easy fix is to position the caption above the figure.

A harder waymight work along the following lines...

When you reach a point in the document preparation that the caption numbers have stopped changing, lock each of the cross-reference fields and then cut each of the graphics (including the paragraph mark) and paste each one into the start of the caption. This will put the graphic inside the hidden bookmark that is used by the cross-reference to provide the link anchor and text content. If the cross-references were then updated they would each include the graphic which is why you need to lock them before you do this step.

This could be automated with a macro very easily.

You could also post-process the PDF to relocate the links to the preferred location.

transfering Office 2010 to a new computer

rohn007 say:

The process is simple enough. The question is exactly which Office license you have.

If you bought Office 2010 installed and activated at the factory, you cannot transfer it.

If you bought an Office 2010 "Key Card" with the product key on it, you cannot transfer it.

If you bought Office 2010 in the "big ugly plastic ecologically insensitive CD holder" box then you have the right to transfer it to new computers. You also have the right to install Home and Student on up to 3 computers at the same time, and Home and Business or Professional have the right to install on up to 2 computers at the same time.

If you have a version that allows transfer and/or install on multiple devices you first have to figure out if you used up your install count.  If you have used up the install count, then you have to "free up" an install count. You can do that by uninstalling from one computer, or confirming that a machine it was formerly installed on has been completely removed from service (destroyed) or has had Office uninstalled or re-installed Windows.

Now that you have an install count freed up, just install and activate it using the 25 character Product Key

As long as you have the key, you can download the installation file from this page

https://products.office.com/en-us/download-back-up-restore-microsoft-office-products   365 / 2016 / 2013 / 2010 (you may have to scroll down)

This page has full install instructions:

https://support.office.com/en-us/article/install-office-2010-1b8f3c9b-bdd2-4a4f-8c88-aa756546529d?ocmsassetID=1b8f3c9b-bdd2-4a4f-8c88-aa756546529d&ui=en-US&rs=en-US&ad=US

Note: if you don't have the product key, we can provide tools to "extract" the key from your current installation. (2010 is the last version we can do that with)

HINT:

Download and save the install file. Make sure to save a copy of it in a (or more than one!) safe place for future use. Office 2010 will no longer be supported after Oct 2020, so you will no longer have access to the install file after that time.
