When a report area is wider than the page it could be due to a horizontal line exceeding the page width. For example, if you use a page 8.5 inches wide with left and right margins set to 0.5 inches the report area should not be more that 7.5 inches wide. When you are working with Powerpoint and embedding media on it, the media is an external file. Although the code is present on the slides and the media is visible on your computer, it does not get played on the other computer as the linking is relative to your computer.

You need to create a package file where all the media, fonts and anything extra that you have used are present and linked properly and can be viewed the same way as you view it on your computer.

Powerpoint helps you create that package by using the Package for CD option found is the Publishing options. 

When you use this option you are given 2 options. Copy to CD and copy to folder. Choose copy to folder and choose where you want to save the package.

Next, you will be asked if you want to include all the linked files. Click on yes. It will then create a folder for you where all of the files are included. 

A PPT viewer is also included which should be used to view your presentation. If the ppt file is directly viewed it may take the default options of the other PC and may not link the files correctly. Start the presentation through PPT View which is included in the folder.
