Outlook cannot find its data if the path leading to the identity changes.

You need to correct it back to reflect what it was before.

The path is different in Outlook 2011 and 2016.

In 2011 it's ~/Documents/Microsoft User Data/Office 2011 Identities

In 2016 it's ~/Library/Group Containers/UBF8T346G9.Office/Outlook/Outlook 15 Profiles/Main Profile

(~/ represents your user account).

Double-check all parent folders leading to the identity to make sure it's OK.

Be careful though since when you relaunched Outlook after making that change, it probably re-created a new empty identity in the correct path. Check the folder sizes and creation dates to be sure.

Safe mode starts Windows with a limited set of files and drivers. Startup programs don't run in safe mode, and only the basic drivers needed to start Windows are installed.

 

1. Click on the Start button.
2. Type msconfig (without any quotes) in the start search box and press Enter.
3. System Configuration windows will be open.
4. Click on the Boot tab located at the top of the System Configuration window.
5. Check the checkbox the left of Safe boot.
6. Click on OK.

 

You will then be prompted to either Restart, which will restart your computer immediately, or Exit without restart, which will close the window and allow you to continue to use your computer, in which case you'll need to restart manually.

 

After restarting, Windows will automatically boot to Safe Mode with networking.


Important: Windows will continue to start in Safe Mode automatically until System Configuration is configured to again boot normally, which we'll do over the next several steps.


When your work in Safe Mode is complete, again start System Configuration as you did in Steps 1 and above.


1. Click on the Boot tab located at the top of the System Configuration window.
2. Uncheck the checkbox the left of Safe boot.
3. Then Choose the Normal startup radio button (on theGeneral tab) and then tap or click on OK.
4. You'll again be prompted with the samerestart your computer question as in Step 6. Choose one option, most likely Restart.

 

Your computer will restart and Windows will start normally... and will continue to do so.

Example

 cells 1,2,3 are blank, cells 4,5,6 have data, cells 7,9,11 are blank, cells 8, 10, and 12 have data, cells 13 and on are blank. The count should come up with 9 cells. 6 with data, and 3 without data in between the cells with data.

Thanks

ronrosenfeld say:

The column of the first cell is given by the array-entered (enter by holding down <ctrl+shift> while hitting <enter>)  formula:

=MATCH(TRUE,1:1<>"",0)

The last occupied cell column is given by the formula:

=LOOKUP(2,1/ISNUMBER(1/($1:$1<>"")),COLUMN($1:$1))

So the total cells from one to the next is given by the **array-entered** formula:

=LOOKUP(2,1/ISNUMBER(1/(1:1<>"")),COLUMN(1:1))-MATCH(TRUE,1:1<>"",0)+1
