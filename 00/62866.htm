xircal say:

Hi zanfran,

I had this problem myself updating Nvidia drivers on my own system.

It's caused by the Add New Hardware Wizard which unlike previous versions of Windows, runs silently in the background in 8.1 and installs a generic driver.

What you have to do to prevent that is the following.

  1. Hit Windows key + Pause/Break to take you to "System" and then click the link to "Advanced system settings" top left.
  2. In the next menu, click the "Hardware" tab and then click the "Device Installation Settings" button.
  3. Next, select: "No, let me choose what to do" and "Never install driver software from Windows Update". You can leave the bottom one checkmarked.
  4. Next, go back to the "System" menu and then click the link bottom left to "Windows Updates".
  5. Next, click the link to "Change Settings" top left.
  6. In the following menu, change whatever setting you have to "Never check for updates". This is only temporary, so don't worry.
  7. Close Windows Updates and then right click the Start button and go to Device Manager.
  8. Expand the tree to your graphics card and then uninstall the generic driver.
  9. Next, reboot and go to the location where you saved the driver update.

You should now be able to install the new drivers without a problem. But do make sure the drivers are WHQL-certified otherwise you'll experience stability issues and possibly BSODs. 

After you successfully update the drivers, reset Windows Updates to whatever it was set to previously.
