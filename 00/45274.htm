They are genuine Microsoft product keys, they are actually retail licenses, but it is intended for a particular product channel either the Microsoft Software Developer Network (MSDN) or TechNet for IT Professionals who pay a subscription fee. andre da costa say:
Windows 7 retail version or retail license is available in both upgrade and full versions. Retail versions differ from other distributions of Windows 7 in the sense that they usually come in a shrink wrapped box, the license can be transferred to another computer if it is removed from the current computer.

As noted, there are two types of retail Windows 7 licenses:

Full Version, also known as full packaged product - This license can be used to upgrade an existing edition as well as do a custom clean install without a previous qualifying license installed such as Windows XP or Vista.

Upgrade Version - this license known as a upgrade product requires that the user has a previous qualifying version of Windows installed on the computer before they can qualify to use it, such as Windows Vista or XP. 

Both can be transferred to another computer and reactivated without problems. A key difference between the two is, one is more expensive that the other, full versions cost more than upgrade versions. Also, retail versions of Windows 7 require product activation.
--------------

Pre-activated Windows 7 - Also known as OEM SLP or OEM OA (SLP stands for Software License Protection and OA stands for OEM Activation.

These normally come preloaded on new computers from name brand manufacturers such as DELL, HP, Lenovo etc. They use special instructions that check the motherboard and bypass the need to do product activation.

OEM versions of Windows 7 are identical to Full License Retail versions except for the following:

- OEM versions do not offer any free Microsoft direct support from Microsoft support personnel

- OEM licenses are tied to the very first computer you install and activate it on

- OEM versions allow all hardware upgrades except for an upgrade to a different model motherboard

- OEM versions cannot be used to directly upgrade from an older Windows operating system
