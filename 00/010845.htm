mike h.. say:

Hi,

 

I've only did this for 20 rows to demonstrate the method so you will need to increase the range to match your requirements.  These are ARRAY formula, see below on how to enter them.

 

 

This returns the smallest number excluding zero. You would change to last 1 to a 2 for the second smallest and to a 3 for the third

 

=SMALL(IF(A1:A20<>0,A1:A20),1)

 

 

An likewise for the largest, once again changing the last 1 to a 2 and 3

 

=LARGE(IF(A1:A20<>0,A1:A20),1)

 

 

This is an array formula which must be entered by pressing CTRL+Shift+Enter
and not just Enter. If you do it correctly then Excel will put curly brackets
around the formula {}. You can't type these yourself. If you edit the formula
you must enter it again with CTRL+Shift+Enter.

hansv mvp say:

So the 'unused' dates contain zero values, not blanks. That makes it more complicated, since none of the COUNTIF, SUMIF etc. group of functions work across multiple sheets, and a "3D" reference such as 'Aug-1:Aug-22'!H6 is not valid when used by itself.

One possible solution:

- In another cell on each worksheet, say H66, enter the formula =--(H6>0)

- This formula returns 1 if H6>0, and 0 if H6=0.

- You can now use the formula

=SUM('Aug-1:Aug-12'!H6)/SUM('Aug-1:Aug-12'!H66)

Another possibility is to create a custom AVERAGEIF3D function in VBA, but that would probably be slow.
