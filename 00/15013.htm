talking moose say:
Rules are triggered on messages two different ways:
  1. They are triggered and act on new incoming mail messages.
  2. They are manually run on selected mail messages.
Rules do not run automatically on mail messages you've already received. Unfortunately, this cannot be scheduled to occur automatically.

You can manually run your rule to move messages over XX days according to your rule but you must have those messages selected first. While viewing your messages select one of them and then select Edit menu --> Select All to select all messages within the current mail folder. Select Message menu --> Rules --> Apply --> Apply All. Messages that meet your rule's criteria will move.
