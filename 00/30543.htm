The cause for the error "Unable to reset your PC. A required drive partition is missing" is normally a corrupt MBR (Master Boot Record) or BCD (Boot Configuration Data) files which contains boot configuration parameters and controls operating system. To resolve this issue, follow below steps.

1. Restart your PC and boot into Advanced Boot Options. From the Windows Recovery Environment, select

    Troubleshoot > Advanced options > Command Prompt.

2. Run check disk using the command line chkdsk /r command.

    (This command locates bad sectors and recovers readable information)

Ex: chkdsk /r E:
