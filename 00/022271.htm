1. Assume the text is in cell B3

3. In cell C3, enter

 

IFERROR(MID(B3,LOOKUP(1E+100,SEARCH(F7:F11,B3)),IFERROR(SEARCH(" ",B3,LOOKUP(1E+100,SEARCH(F7:F11,B3))+1),50)-LOOKUP(1E+100,SEARCH(F7:F11,B3))),"")

 

If there are multiple matches in the keywords range, then this formula will return the first match only.

exclusions to word count - Microsoft Community

jeremy_967 say:

exclusions to word count

How can one exclude caption fields from word counts in Word version 15-31 for Mac
macropod say:

Or you could get a Caption Style word count via a macro that you could assign to a keyboard shortcut:

Sub CaptionWordCount()
Dim c As Long
With ActiveDocument.Range
  With .Find
    .ClearFormatting
    .Replacement.ClearFormatting
    .Text = ""
    .Replacement.Text = ""
    .Forward = True
    .Wrap = wdFindStop
    .Format = True
    .Style = "Caption"
    .MatchCase = False
    .MatchWholeWord = False
    .MatchWildcards = False
    .MatchSoundsLike = False
    .MatchAllWordForms = False
    .Execute
  End With
  Do While .Find.Found
    c = c + .ComputeStatistics(wdStatisticWords)
    .Collapse wdCollapseEnd
    .Find.Execute
  Loop
End With
MsgBox "There are " & c & " words in Captions in this document."
End Sub
