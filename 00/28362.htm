greginmich say:

Lets not lose sight of whats really important here, because lots of people have been hurt by this just because they didnt know how to force-close their browser. With the garden variety Tech Support Scam page, there really isnt any malware to hunt down because your PC isnt really infected with anything at all. The only malware involved is the script thats running in the scam page and preventing you from closing the browser. So you dont need to start downloading malware-removal tools when one of these scam pages locks up your browser because the only "malware-removal" tool that youll normally need is only a couple of mouse clicks away. Right-click on your taskbar and choose Task Manager; and then select the browser app and click on the End task button. Then relaunch the browser and go directly to your home page and Voil, the malware will normally be gone for good (except in Edge).

http://windows.microsoft.com/en-us/windows/exit-program-not-responding#1TC=windows-7

So ending the browser process isnt something thats optional in getting rid of this type of malware, its actually a necessary and sufficient fix in most cases, as the people at Malwarebytes have always insisted:

https://blog.malwarebytes.org/fraud-scam/2014/11/psa-tech-support-scams-pop-ups-on-the-rise/

https://blog.malwarebytes.org/fraud-scam/2015/09/avoid-this-bsod-tech-support-scam/

Now, Edge is a special case, because its developers apparently werent aware of how these scam pages actually work, even though theyve been a major problem for years (the Tech Support Scam pages are the direct descendants of browser-locking ransomware). Edge always forces a last-session recovery that brings the Tech Support Scam page right back up again when you relaunch it normally after force-closing it. So what you have to do is open Edge indirectly by typing a short search term in the Search box and hitting the enter key. Then just close and reopen Edge in the normal way or right-click on the tab of the Search result page and select Close other tabs. Its also possible to install a registry switch that will allow you to turn off Edges AutoRecover feature and once thats turned off, you can escape from these scam pages by just force-closing them in Task Manager:

https://blog.malwarebytes.org/online-security/2015/09/regaining-control-over-edge/

Note that a few of these Tech Support Scam pages actually are the handiwork of malware (PUPs or malicious browser add-ons), but those would normally be the full screen mock-ups, or pages that pop up again after being force-closed. If you run into something like this, then you really will need to start scanning with the tools in the Malwaretips removal guide.

https://blog.malwarebytes.org/fraud-scam/2015/07/techsupportscams-and-the-blue-screen-of-death/

https://malwaretips.com/blogs/remove-tech-support-scam-popups/

jim gordon mvp say:

~$ files are temporary files used by Word. They are not Word document files. They are not AutoRecovery files.

Description of how Word creates temporary files

AutoRecovery comes into play if you first save the file to a local directory (not a network drive, OneDrive, DropBox, etc). If you save to OneDrive, there's a different version recovery system. It appears the file was saved somewhere, but you didn't say exactly where.

You suggest that the template may be in an old file format. Using File > Convert would have converted it to a current file format.

Regarding corruption - please go to the Word menu and choose About Word. You should have version 16.14.1 or 16.15. If you have any other version, install updates to fight against possible file corruption.

Make sure your Mac has a working internet connection before checking for updates. Install Mac OS X updates followed by Office updates.

Install Mac OS X updates first:

To get Mac OS updates, go to the App store on your dock, and then click the Updates button to reveal available updates.

Then Install Office Updates

To get the update:

1. Open any Office application

2. Go to the Help menu and choose Check for Updates

3. Let AutoUpdate check for and install updates

4. Restart your Mac

If you have any add-ins such as WebEx, Mendelay, EndNote, Zotero, TypeIt4me, etc it is essential that you install the latest updates for your add-ins.

Every once in a while you should start your Mac up in Safe Mode by holding the SHIFT key down as your Mac starts up from Restart or Shut Down. It takes your Mac a long time to boot to safe mode. A lot of maintenance utilities run in the background and performance can be improved. After you get to the desktop in Safe Mode, do a regular restart.
