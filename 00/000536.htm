doug robbins - mvp office apps & services (word) say:

There are lots of ways of doing it.  Just which is best for you may depend on how you want the document to appear.

One possibility would be to use a one column by three row table, with the first row height set to an exact figure to suit the pictures, the second row also set to an exact height to suit the caption and the third row allowed to expand to provide space for the description.

You could then simplify the creation of the document by using a document with a table set up as described as a mail merge main document which you merged with a data source in one field of which you insert the path and filename of the pictures using / as the path separator instead of the normal \, in another field, you would have the caption and in a third field, you would have the description.
