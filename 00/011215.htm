mike h.. say:

Hi,

 

Try this, it adds a suffix number and groups sheets by client No.

 

Sub somesub()
Dim Val As String, x As Long, num As Long
Dim marker As Long
num = 1
Val = Sheets("Data Entry").Range("A2").Value
For x = 1 To Worksheets.Count
    If Left(Sheets(x).Name, Len(Val)) = Val And Len(Sheets(x).Name) - 4 = Len(Val) Then
        num = num + 1
        marker = x
    End If
Next
If marker = 0 Then
Sheets("Data Entry").Copy after:=Sheets("Definitions")
Else
Sheets("Data Entry").Copy after:=Sheets(marker)
End If
ActiveSheet.Name = Val & " - " & num
End Sub

ricw-nw say:

Hi Ron - Yes, it's a simple enough process... The main pitfall with copying files is deleting them before making sure they have copied... Always check the copies are there and working before deleting the originals!

1. Open 'Computer' (This PC) - you should see the USB device listed.

2. Open the USB device to see the files/folders on it.

3. Highlight the Files/Folders you wish to copy - right-click and select 'Copy'.

4. Open your Music library folder (NOT WMP) - right click in any space and click 'Paste' and you will see the items transfer to the folder. Check they are all present and will play with your Media Player. If satisfied you can now delete the files from the USB drive. 

Note if these music files are not organized into folders it may be best to do this on the USB stick before copying them... get back if you want more on this point. - Cheers, Ric.
