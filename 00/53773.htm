I just got this multi-function machine working on Windows 10. Here's what I did:

  1. Turn off the printer, and disconnect it from your computer if it is currently connected via USB.
  2. If you have installed any drivers or supporting software, or attempted to do so, make sure all of those things are uninstalled. This may require multiple reboots.
  3. If you have not already done so in step 2, reboot.
  4. Download the Windows 8.1 driver from Canon's Web site (64 bit or 32 bit, as appropriate to your system).
  5. Temporarily disable your firewall, whether that is the Windows Firewall or in some other software package.
  6. Right click on the downloaded file, and manually set compatibility mode to Windows 8.
  7. Right click on the downloaded file, and Run as Administrator.
  8. The driver installation will give you two choices: USB or network. Choose network.
  9. The network installation gives you two choices, if I recall correctly. You want the one that talks about installing the printer for the first time.
  10. When prompted by the installer, connect the pr
  11. The installer will first search for the printer on your local network, but will not find it. It will then search for your wireless access point.
  12. Choose your access point from the list, and enter its password.
  13. The installer will tell you the printer is installed, and then proceed to test the performance of the connection. This takes a couple of minutes, and there's no way to skip it, so hang tight.
  14. You're done! Open up Notepad, type "Test", and hit CTRL-P to print.

This method will also work for installing the drivers in USB mode, if that's how you use the printer. The scanning function requires the Canon MP Navigator software. Same thing--download the Windows 8.1 version from Canon, set the compatibility mode, and Run as Administrator.

I didn't do anything with the faxing capability, but I believe that this method would likely be successful there. Just be sure to download current drivers and software from Canon. Don't rely on anything on a CD--much too old!

User accounts - Recommended Setup (Vista and
You absolutely do not want to have only one user account. Like XP and all other modern operating systems, Vista and Windows 7 are multi-user operating systems with built-in system accounts such as Administrator, Default, and Guest. These accounts should be left alone as they are part of the operating system structure.

You particularly don't want only one user account with administrative privileges on Vista and Windows 7 because the built-in Administrator account (normally only used in emergencies) is disabled by default. If you're running as Administrator for your daily work and that account gets corrupted, things will be Difficult. It isn't impossible to activate the built-in Administrator to rescue things, but it may require more work than you want to do. Best not to get into a bad situation to begin with.

The user account that is for your daily work should be a Standard user, with the extra administrative user (call it something like "CompAdmin" or "Tech" or the like) only there for elevation purposes. Running as a Standard user is best practice for security purposes and will help protect your computer from infection. After you create "CompAdmin", log into it and change your regular user account to Standard. Then log back into your regular account.

If you want to go directly to the Desktop and skip the Welcome Screen with the icons of user accounts, you can do this:
