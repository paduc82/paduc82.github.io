Microsoft Bingo General FAQ - Updated 1/12/2015

Have a question about Microsoft Bingo? Check here first to see if we've got an answer!

How do I get Microsoft Bingo? How much is it?

  • The good news- Microsoft Bingo is ABSOLUTELY FREE!  

Bingo is telling me I need to switch to a Microsoft account/add a Microsoft account/access & update my Xbox Live info. Why?

  • Imagine for a second that the unthinkable happens- your computer/tablet breaks entirely and you can't use it anymore. When this happened in the past, it meant all your saved games and progress died with your device. NOT ANYMORE!  Because Microsoft Bingo accesses your Microsoft account/Xbox Live account, it saves all your progress and achievements to the "cloud"- which means all your player game data is safe even if your computer isn't. 
  • If you're wondering "why is it asking to access & update my Xbox Live info- I don't have a Xbox!" this is because you don't need a Xbox to have a Xbox Live account. Now for some good news- your Microsoft Account IS your Xbox Live account! So you don't need to create a new account somewhere or anything like that- it automagically happens for you.  All you have to worry about is getting those Bingos. :)

I tried to play Microsoft Bingo today but there was a problem with it! Help!

  • Technology is great but sometimes it doesn't always work the way it should because it can do so many things now. In fact, sometimes your apps & games won't work correctly because of something unrelated- like Windows having optional and/or waiting updates that your game or app needs to run.
  • DON'T PANIC. :)  Try these steps first:

(On Windows 8.1)

  1. Press the Windows key
  2. Type in Windows Update
  3. Under the search bar you see on the right, choose Check for Updates
  4. In the Windows Update screen, press the Check Now button
  5. Even if it doesn't seem to find any updates, click on View Details
  6. Install any updates you see that haven't been installed - including optional ones
  7. Go to your Shutdown menu and choose 'Restart' from the options. You must choose restart- turning your PC/tablet off will not work!
  8. Sometimes, it can happen that Windows updates are stuck. If you would like to make sure that this is not the case or if you receive an error while manually trying to update, please run the Windows Update troubleshooter, which you can find here: http://windows.microsoft.com/en-us/windows7/open-the-windows-update-troubleshooter
     

Still having problems? Try these steps once you've tried the ones above. Remember, you WON'T lose your progress by uninstalling the app because it's nice & safe in the cloud under your Xbox account.

  1. Uninstall Microsoft Bingo (to uninstall, right-click on the app icon and choose Uninstall from the bottom menu)
  2. Press Windows Key and R (or go to Start and search for Run)
  3. In the box that pops up, type wsreset.exe  and press Enter or OK
  4. The Windows Store will launch for you (with its cache nice and clean!)
  5. Hover over the right-hand side to bring up the charm bar and go to Settings
  6. In the Settings Menu, go to App Updates
  7. In the App Updates menu, press both Check for Updates AND Sync Licenses
  8. Press the arrow key at the top left to go back
  9. In the search bar at the right, type in Microsoft Bingo
  10. Reinstall Microsoft Bingo

Tried all of the above and it still doesn't work? Feel free to either go to the Xbox Forums (http://forums.xbox.com/gfw/xbox_on_windows/l-o/f/4505.aspx
