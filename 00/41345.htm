DMG is just another disc image format - one closely associated with macintosh.
  
On a Windows PC the dmg2img program will convert a .DMG file to a standard (hfsplus) image disk file.
 
OS X just knows to mount the image and use it like the CD/DVD/other media type it represents.  Nothing like that is present natively in Windows XP.
 
You can get third party programs (by the droves) to do the same thing with various formats.  ISO is one of the more common image types.  Around that one you could use these tools:

a.m.user say:

Hi BACCbirds. It sounds like either you have selected to import a painting, in which case only .fppx files will be seen, or Fresh Paint is looking in a different picture folder to the one you want.  I cannot see or replicate what Luke describes, or be sure exactly which options you have tried, but I think and hope one or more of these four suggestions may help you.

 

  1. There will be multiple picture folders on your computer. There is a picture folder for each user account, a picture folder for the user called “public”, and also a library called “pictures”. Perhaps the easiest approach is to go to your desktop, open file explorer and find the folder or folders holding any images you want to work with. Right click on the folder, select “Include in Library” and then select “Pictures.” At least then within Fresh Paint you will be able to select “Libraries”, then “Pictures” and then see whatever folders you want to work with, and know you are in the right spot.

  2. On the first page in Fresh Paint find the blue “NEW from image” button in the left edge of the screen. Note the icon of a rectangle with an arrow descending in from above.
    Select “NEW from image” -> Libraries -> Pictures and then hopefully you will now see the picture you want. Select it then be patient until Fresh Paint is ready for you to work with it. It may take 5 minutes or more. You might see two small squares at the top of the outside edges of your picture. Clicking on either of them should bring up the palette.

  3. On the first page swipe in and select “Import paintings”. Only .pffx files will be available.

  4. Create a new painting. The palette should be open. Select the import image icon in the top left (the rectangle with the arrow coming down from above), then libraries then pictures, then the folder and image you want. jpg, png and fppx should all be available.
