Fix for 'End time / Start time' issue when trying to - Microsoft Community

spydor say:

Fix for 'End time / Start time' issue when trying to disable 'Events from Email'

Issue

I was experiencing a problem trying to turn off the Events from Email feature for the Calendar options where it would not save, instead showing me a misplaced message stating:

"The end time that you entered occurs before the start time.  The start time must occur before the end time."

Solution

I eventually found the cause of this problem in the Calendar appearance section of the Calendar options.  For some strange reason my working hours were set to start at 16:00 and end at 01:00 (despite me never having used these options before).  Outlook does not allow you to set your working hours to traverse midnight, so it will assume an earlier end time occurs on the same day, before the start time and therefor refuse to save it.

Simply setting your start and end times to be one after the other on the same day will sort the issue and allow you to go back and disable Events from Email.

Hopefully this will help anyone else whose working times are inexplicably screwed up in the Outlook calendar options and are unable to disable unwanted features like this, until Microsoft alter the code, as enabling/disabling this option should not be dependant on the working hours.
