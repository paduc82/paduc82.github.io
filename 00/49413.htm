Hi,

Thank you for writing to Microsoft Community Forums.

Nonpaged pool is kernel memory which can't be paged out into the page file when Windows runs out of free physical memory. It is used by drivers to allocate memory which they need. For us to isolate the issue, we'd like to know more information on this.

1. What is the make and the model number of the computer?

2. Which build version of Windows 10 are you using?

3. Are you aware of any changes made on the computer prior to this issue?

We can try the below steps and check if we can fix this.

1. Press Windows key + R together to open Run.

2. Type regedit.exe and press Enter.

3. Navigate to the following location:

    HKEY_LOCAL_MACHINE\SYSTEM\ControlSet001\Control\SessionManager\MemoryManagement

4. On the right hand side, right click on NonPagedPoolSize and click on Modify.

5. Make a check on Decimal.

6. Change the Value data to 192.

7. Click on OK.
