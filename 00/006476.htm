Paste Link - Excel 2010 - Microsoft Community

jgreen620 say:

Paste Link - Excel 2010

Good morning, MS Experts!

I'm working in Excel 2010, using Windows 7.  Two questions: 

Q1.)  Is it possible to paste/link one master worksheet to multiple worksheets in the same document while keeping all the formatting in each worksheet as the master? 

I'm creating a quarterly report using Excel 2010.  The master document has all the information that needs to be reviewed each quarter.  Last quarter I created a worksheet that linked to the Master and all formatting appears to be exact (highlighted areas, different color fonts for particular items, bolded text in certain cells, etc.) as I was asked to do. 

However, the next quarter has arrived and I need to link the master worksheet to another new worksheet.  No matter what I try, it's copying the information to the new worksheet using basic text in each cell showing a "0" in blank cells and all the formatting is gone.  Even the font style and size have changed. There are thousands of rows and 12 columns.  If I have to go line-by-line and fix it, this will be quite time consuming.  The information does link so that's a positive.  Is MS Excel limited when pasting a link in multiple locations?  Is there a workaround?  If I just copy/paste, the formatting is correct.  Something is up when I paste/link.

Q2.) Is there a way to save the worksheet I linked last quarter so no further updates are made to it?  As I said above, the links are working correctly on the updated worksheet, but it is also updating the worksheet I created last quarter.  I'm worried that won't be acceptable to my supervisor.

Again, thank you to everyone that reaches out to help me with this.  Have a wonderful day.

08-18-16 - UPDATE!   I figured out what to do.  I feel so good about this.  Anyway, for anyone else that has this problem, I'm going to give you the solution that worked for me. 

Start in the master worksheet and highlight whatever part (or all) of the worksheet you want to link, copy the material you want linked, open the blank worksheet and do a regular, old paste function.  The information copies to the new sheet.  Now while the entire worksheet is still highlighted, either go to the tool bar and choose the drop-down arrow under the Paste icon and look for the Link icon or if you are more comfortable, right click and under "Paste Option" choose the link icon (sorry I cannot show it here) or if that's not showing, choose "Paste Special" and it will be a choice under that selection.  You can check to make sure it links by making a change to your master.  But you will know it links when you see all the "0's" in your blank cells.  Those you can remove by going into File, Options, Advanced and unclicking the box that says "show a zero in cells that have a zero value" under "Display options for this worksheet."

I hope this helps.  Thank you to everyone that provided suggestions.  It means a lot to me.

Best Practices for Termed Users - Microsoft Community

derekgillespie- [o365] say:

Best Practices for Termed Users

Hi all.  I wanted to get everyone's take on what the best practices are for administering terminated employees in Office 365.  We currently have E3 licenses for all users.  The goal is to keep the integrity of the mailbox and user profile at the smallest cost possible.

Here are the different scenarios:

  1.  User is terminated and the mailbox doesn't need to be preserved.  It is highly unlikely that the user will return to work for us again.
  2. User is terminated and the mailbox needs to be preserved for manager/replacement access.
  3. User goes on year-long sabbatical.  Monitoring of the mailbox by other employees is needed.

Thanks everyone!  If there is anything to add to above scenarios, please do!

- Derek

garyzhu_msft3613 say:

hi derek,

if you want to keep the integrity of the mailboxes and user profile at the smallest cost in the three scenarios, please see below:

for the first scenario, if the mailboxes donâ€™t need to be preserved, as vasil mentioned above, you can change those mailboxes to inactive mailboxes (enable litigation hold/in place hold before deleting the mailboxes in the active users.)

for the second scenario, you can convert the usersâ€™ mailboxes to shared mailboxes (licenses are not needed) and then remove their licenses. if the mailboxes need to be preserved for manager access, just grant the manager full access permission  for those shared mailboxes so that the manager can access them.
 

for the third one, like the second scenario, i suggest you convert the mailboxes that go on year-long sabbatical to shared mailboxes, grant full access, and then you can remove the exchange online (plan2) (or other office 365 enterprise e3 licenses). once the users come back from sabbatical, you can convert  shared mailboxes to regular mailboxes and reassign the exchange online (plan2) licenses to those accounts.
 

thanks,
gary
