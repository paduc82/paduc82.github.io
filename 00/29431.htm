Hi Ghaleb,

 

Currently LifeCam HD-3000 does not have driver update compatible with Windows 10 operating system. However, you can install the drivers of Windows 7 of the device and run the same in Compatibility mode on the computer.

 

You can download LifeCam HD-3000 drivers for Windows 7 OS from the link below:

http://www.microsoft.com/hardware/en-in/d/lifecam-hd-3000

 

There are two ways to install the device drivers in Compatibility mode. Follow the steps below to install the device drivers in Compatibility mode.

 

Step 1: Compatibility mode runs the program in the current version of Windows using the settings from previous version of Windows.

 

  1. Locate and right-click on the driver setup file on your PC and click on Properties.
  2. Click on the compatibility tab, check the box Run this program in compatibility mode for, select Windows 8.1/7 operating system from the drop down menu and proceed with the installation. Once this is done, restart the computer for the effective changes to take place.

 

Step 2: You can also follow the steps below and check if the issue is resolved.

 

  1. Press Windows logo key on the keyboard and type Run programs made for previous versions of Windows.
  2. Select the top most search result and follow from step 2 from the article Make older programs compatible with this version of Windows (Also applies to Windows 10).
  3. Check if the changes are effective.

 

Write to us with the status of LifeCam HD-3000 query on the PC in the same post for further assistance. Your reply is most important for us to ensure we assist you accordingly.

 

Thank you.

Porting in of number to Skype is not yet available. If you need to be contacted thru Skype by a mobile and landlines, you will need to purchase Skype number. To explain more about this product, Skype number is a Skype subscription that will allow you to receive calls from mobiles and landlines.
