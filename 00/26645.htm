formtext datatype word automation - Microsoft Community

wily100 say:

formtext datatype word automation

I would like to use Word automation (with Delphi) to change the datatype of a formfield from 'number' to regular text, but I haven't been able to find the correct property to set.  Is it even possible to do with COM?
graham mayor say:
I don't know anything about Delphi, but if you were programming this in Word vba, the following would do the trick
where "Text1" is the name of the field in question.

Dim ofld As Word.FormField
Dim oDoc As Word.Document
Dim strText As String
Dim bProtected As Boolean
Set oDoc = ActiveDocument
For Each ofld In oDoc.FormFields
    If ofld.Name = "Text1" Then
        strText = ofld.Result
        If oDoc.ProtectionType <> wdNoProtection Then
            bProtected = True
            oDoc.Unprotect Password:=""
        End If
        ofld.TextInput.EditType wdRegularText
        If bProtected = True Then
            oDoc.Protect _
                    Type:=wdAllowOnlyFormFields, _
                    NoReset:=True, _
                    Password:=""
        End If
        ofld.Result = strText
        Exit For
    End If
Next ofld

smittypro1 say:

You could do a simple loop. This assumes the value to increment is in C1 (so you don't have to adjust the code all the time):

Sub foo()
    Dim x As Long
    Dim c As Range
   
    x = Range("C1").Value
   
    For Each c In Range("A1:A6")
        c.Value = c.Value + x
    Next c
