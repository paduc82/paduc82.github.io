FrederickHebert -

 

Thank you for posting on the Xbox Music Community.

 

If you want to get all of your music on your PC and have it easily accessable, you can definitely use Xbox Music on Windows 8 to do this.  By default, the Xbox Music app looks at the Music Library folder on the PC.  If you haven't changed that, it should be under:
C:\Users\YourUsername\My Music\

 

When you download music from the marketplace, it will create a folder in there called Xbox Music, and then another folder for the artist name in there, with yet another sub-folder for the album name.  For example:
C:\Users\YourUsername\My Music\Xbox Music\Daft Punk\Random Access Memories

 

If you rip CDs onto your computer, I definitely recommend following the same organization style to make everything easier to keep track of.  You don't need to put it into an Xbox Music subfolder.  Personally, I have a "Ripped Albums" folder in my My Music folder at home, so I can keep track of what came from where.

 

Once you have all your music in that folder, just launch the Xbox Music app and it'll look in there and add everything to your collection.

    ossiemac say:

    See if the following 2 examples help. Note that by dimensioning shp as a shape. When you use

    With Shp

    then immediately you place the dot at the stat of each folowing line the intellisense kicks in and helps with the coding.


    The best way of ensuring that you have the correct shape to delete is to name it when creating it and then it is deleted by name as per the first example. However, the second example also shows how to delete it by the TopLeftCell position.

     

    Private Sub CommandButton1_Click()
        Dim shp As Shape
        Dim s As String
       
        'Delete by Shape Name (If Known)  _
         On Error required in case shape name does not exist
        On Error Resume Next
        Set shp = ActiveSheet.Shapes("MyTempShape")
        On Error GoTo 0
       
        If Not shp Is Nothing Then shp.Delete       'Not nothing then is something
       
        Set shp = Me.Shapes.AddTextbox(msoTextOrientationHorizontal, 100, 170, 214, 118)
       
        s = "My Test Text"
       
        With shp
            .Name = "MyTempShape"
            .TextFrame2.AutoSize = msoAutoSizeShapeToFitText
            .TextFrame2.TextRange.Characters.Text = s
            .Left = Range("C20").Left + Range("C20").Width / 2
            .Top = Range("C20").Top + Range("C20").Height / 2
            .TextFrame2.AutoSize = msoAutoSizeTextToFitShape
        End With
       
    End Sub


     

    Private Sub CommandButton2_Click()
        Dim shp As Shape
        Dim s As String
       
        'Delete by position of TopLeftCell of shape
        For Each shp In Me.Shapes
            If shp.TopLeftCell.Address = Range("C20").Address Then shp.Delete
        Next shp
       
        Set shp = Me.Shapes.AddTextbox(msoTextOrientationHorizontal, 100, 170, 214, 118)
       
        s = "My Test Text"
       
        With shp
            .Name = "MyTempShape"
            '.LockAspectRatio = msoCTrue    'Lookup  LockAspectRatio in help for more info
            .TextFrame2.AutoSize = msoAutoSizeShapeToFitText
            .TextFrame2.TextRange.Characters.Text = s
            .Left = Range("C20").Left + Range("C20").Width / 2
            .Top = Range("C20").Top + Range("C20").Height / 2
            .TextFrame2.AutoSize = msoAutoSizeTextToFitShape
        End With

       
    End Sub


     

    Added with Edit after initial post:

     

    I inadvertantly left in "msoAutoSizeShapeToFitText" and "msoAutoSizeTextToFitShape".  You would not use both of these with the same shape. The last one will be acted upon.
