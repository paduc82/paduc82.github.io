john visio mvp say:

There is also the Geometry NoShow cell. Setting it's value to a boolean based on the value of a custom property is how the multi-shape works.

You define a geometry for each shape (ie a circle, square, rectangle and hexagon) and set the value of the corresponding NoShow cells to be true if it is not that shape. This is how the multisahpes in the flowchart stencil work.

As Paul pointed out, you will have to use the HideText as well, but in this case, use the corresponding NoShow cell to determine the value for HideText.

Followup:

The VBA code?

'shp.Cells("Geometry1.NoShow") = True

'shp.Cells("Geometry1.NoLine") = True

As mentioned, the NoShow takes care of the NoLine, but I just included it for completeness.
each shape may have one or more Geometry sections so you will need a NoShow for each Geometry section.

Also, Paul's comment is also true, so you would also need 

'shp.Cells("HideText").ResultIU = True

luckily you only do it once for a shape.
