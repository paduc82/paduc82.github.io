Normally the Banner page is configured on the print server.  If you are the administrator on the print server you can change this in the driver settings.  If you are not the admin, you cannot change this.

I use secure print on a Xerox 5655 from a Windows 7 client.  The hard drive needs to be installed on the printer.  If this is not working you should verify the hard drive is installed.
