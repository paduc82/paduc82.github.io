a. user say:

 

Based on the information, you are trying to find out when trying to move favorites from one computer to another, whether the already existing favorites in the second computer will be removed.

Usually when you import Favorites to another computer, it will not replace the already existing favorites on the second computer.

However to be on a safer side, you may take a backup of the Favorites on the second computer to a different folder or on a different location.  Try to Import favorites from the first computer to the second computer, check if you are able to access the existing favorites along with the imported one.

If not, try to add the favorites from the backup location to the imported favorites and check if it helps.

 

You may also copy the favorites from the second computer to the first computer and save it to a new folder. Try creating another folder containing the favorites from both the first and second computer and then import the favorites from this folder to the second computer.

For further information, refer to the article:

Add to, view, and organize favorites

Do this:

Delete everything except your save files off your HDD that has to do with Halo 4.

Put Disc 2 in. Boot it up. The game will ask to install the cache. Let it. Then take that disc out.

Put Disc 1 in. Install it off the dashboard, then play the game normally. The disc will spin down within a few minutes of boot up.

One of the discs needs to create a cache on your HDD, and the other can be installed via the dashboard. Installing both via the dashboard won't create the cache, and the game will think there's nothing on your HDD and read off the disc instead.
