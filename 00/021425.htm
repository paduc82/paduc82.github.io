Two Formulas in one cell based on a dropdown list - Microsoft Community

bryana1987 say:

Two Formulas in one cell based on a dropdown list

Can I use a COUNTIF and SUMIF Formula in the same cell that changes based on the Word selected from a dropdown list? IE- If the Dropdown list says "Daily", then the cell would be COUNTIF. If the drop down list is "Hourly", then the formula would be SUMIF.
chip pearson say:

If you have a dropdown in a cell, that cell can contain only one value from the drop down list. You cannot have both a value and a formula in a cell. You can have one, but not both. You could write a formula in another cell that reads the value of the cell with the dropdown and executes the appropriate formula. For example, if the drop down is in A1, use the following in another cell:

 

=IF(A1="Daily",COUNTIF(...),IF(A1="Hourly",SUMIF(...), value_if_neither_Daily_or_Hourly))

cristin yan say:
Anyone?  I can't be the first to need to edit a data connection using VBA (although I haven't managed to find any worthwhile information by Googling the subject)!


1. Please check the following subroutine which contains two parameters that passed by reference, not by values.

 

Private Const strPath As String = "G:\"

 

Sub MyDemo(ByRef strConn As String, ByRef strCmdText As String)
    If Dir(strPath, vbDirectory) <> "" Then
        strConn = "connection string 1"
        strCmdText = "command text 1"
    Else
        strConn = "connection string 2"
        strCmdText = "command text 2"
    End If
End Sub

 


2. Please go ahead to run the following macro "TestMyDemo" which I wrote for you:

 

Sub TestMyDemo()
    Dim strConn$, strCmdText$
   
    MyDemo strConn, strCmdText
   
    Debug.Print "Connection String is ", strConn
    Debug.Print "Command Text is ", strCmdText
End Sub

 

Hope it helps.

a. user say:

Hi Biggy,

 

1. Did you make any changes to the computer/Internet Explorer prior to the issue?

2. Is the issue specific to certain websites?

 

Here are a few steps which you can follow:

a. Navigate to the website that isn't displaying properly.

 

b. In Internet Explorer (desktop), tap and hold on the bar at the top (or right click).

 

c. Tap on "Menu bar" so it's highlighted and there is a check mark by it. A menu with File, Edit, View, Favorites, Tools and Help should appear.

 

d. Go to tools and open "Compatibility View Settings". "Add this website" should have the domain of the site you've visited already filled, so click "Add".

 

e. Close the windows then refresh the website that wasn't displaying properly and check.
