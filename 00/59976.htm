a. user say:

Hello Ronak,

Thank you for contacting Microsoft Community.

Regret the inconvenience you have experienced.

Do you mean unable to Pin to Start or taskbar?

Suggest you to check update for the Mail & Calendar app in Windows Store. Launch Windows Store. Click on your Profile on top right of Windows Store, and select Download and Updates. Now click Check for updates, to see if any update available for Mail & Calendar app. Once update is installed, open Mail app to see if problem still persist.

If incase problem still persist, then try to uninstall and reinstall the Mail & Calendar app to see if it fix the issue.

* Press Windows key + X and select Command prompt (Admin).

* Copy and paste the below command lines one by one and hit Enter key.

Powershell

Get-AppxPackage *windowscommunicationsapps* | Remove-AppxPackage

* Now go back to Store and try to install the apps again.

When you open the store to install the mail and calendar app, if it says the product is already installed, then press Windows key + R, type WSReset.exe and hit Enter. Now check if it allows you install again. If the problem still persist, try to restart your PC, post complete shutdown and then check the issue.

Keep us posted if you require further assistance.

I see you telling that the problem started after automatic Windows Update. If that is the case, try to uninstall that particular Windows Update and see if it fix the issue. You might need to uninstall more than one update if you don’t know which one is causing the error.

  3. Select Uninstall.

  4. If it prompt for confirmation, just confirm the same. Once it complete uninstalling, reboot your PC and see if it makes any difference.
