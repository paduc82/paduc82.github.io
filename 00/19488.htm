suzanne s. barnhill say:
This can happen if you have applied direct number formatting to the page number instead of selecting the desired format in the Page Number Format dialog.

sub212 say:

That is correct.

for anybody trying to make the same mistake:

go to the individual page, double click on the footer, then right click on the page number and change format. This, apparently, changes the numbers, but not how they are listed in the TOC.

for anybody NOT wanting to repeat the mistake:

go to anywhere in the section in which the numbers should be changed to Greek (or different than the standard numbering in other sections, then to Insert tab, Header & Footer tab, Page Number icon - Page Number Format. This looks identical to if you right click on the page number, but here, you will see, the format is still the "standard format (in my case arabic).

this needs to be done in each section.

Now, the TOC will display the page numbers in the format that they are in on the pages, too.

sofspra say:

I cannot get my computer to recognize my scanner

This is a question that is being asked more frequently. 

You are a new member to this site.  This site is for testing beta releases of W10 prior to release to the public.  We are now testing 18358.  Are you running an Insider edition  or public build?  You can run "winver" to find out.  If you are running a public build, it should be in the 17763.xxx range. 

This site does not have Microsoft support.  We are all volunteers here.

When I come across a PC that has stopped recognizing a scanner, I check my cables.  If wireless, I check for a wireless connection.  If that does not do it, I re-install the software readily available from the printer manufacturer.  Also, I always check my list of printers.  A printer with scanner will often list a 2nd printer.  A HP printer will usually show a printer with Twain in the name. 

When you simply want to print a page, the program at hand should select a default printer. Make sure it is defaulting to the proper printer.  A program that is scanning should also default to the "Scanning" printer. If these defaults change, you just need to select the proper printer.

A wireless printer should just install out of the box.  If this fails, an USB cable will usually pick up the wireless settings.  I installed a new Brother wireless printer last week.  I downloaded the software from Brother and it failed on the wireless install.  I used the USB cable and at a point I was told to remove the USB cable and I got a good install.

You do not provide a lot of detail as to whether this is a new install or a prior install that is now failing.  A re-install of software usually clears most issues.  However, the scan facility will not work without a full software install.  The print facility usually works out of the box.

You can go to Control Panel and under Devices and Printers see your line up of Printers.  The default printer should show. You should see a Printer that is devoted to scans.  Also, a wireless printer should show a Network while an USB connected printed show show Local.

Hope this helps.  Report back if more questions but also provide more detail as to just what you have.

If you are using a public build, then Microsoft support should be available to you.  However, most printer issues should likely be directed to the printer manufacturer.

suzanne s. barnhill say:

In order to add text to a page number such that it can be included in a TOC, you must use the "Include chapter number" feature in the Page Number Format dialog. The "chapter number" must be the paragraph number of an autonumbered paragraph in one of Word's built-in heading styles.

For more information on using chapter numbering, see http://word.mvps.org/FAQs/Numbering/ChapterNumber.htm.

For more information on numbering appendixes, see http://www.ShaunaKelly.com/word/numbering/NumberingAppendixes.html.

Note that, to be included in the numbering "Appen" would have to be part of the paragraph number. This may not be practical, but you can certainly use "Appendix" as part of the numbering. Your appendix title pages can use a lower-level heading style with numbering set up as Appendix A, Appendix B, etc., or Annex A, Annex B, etc. For examples of headings with such text included, look at the ones using Article, Section, and Chapter in the Multilevel List Numbering gallery.

You can certainly have a separate TOC for appendixes and one for annexes. Just create a TOC that includes only the relevant heading style. You may actually find it easier to use a Table of Figures, which allows you to more efficiently specify a single style to be included. For more on customizing TOCs, see http://sbarnhill.mvps.org/WordFAQs/TOCTips.htm.

Note that, in order to have more than one TOC, you must override Word's eagerness to replace the existing TOC with any additional one you may want to insert. You'll probably find working with all TOCs more straightforward if you insert them all using References | Table of Contents | Insert Table of Contents... instead of selecting one of the Automatic Tables from the gallery.
