  • corrupt hardware: damaged hard disk, video card problem, RAM corruption, problems with the processor (this is the most common cause)
  • driver compatibility issues (rare)
  • heat related problems, e.g. overclocking the system
  • a corrupt Windows registry (rare, unless you've been mucking around)
  • corrupt or removed Windows-specific system files (rare, unless you've been mucking around)

Fix #1: Disable overclocking from BIOS

Make sure the Overclocking feature from your computer’s BIOS is disabled. Some computers might not have this, so don't worry if you can't find it.

To do so, follow these methods to open BIOS:

Method #1: Complete shutdown

  1. Hold the SHIFT key while also pressing the SHUTDOWN button
  2. Select Restart

Method #2: Open Charm Bar

  1. Press Windows+C (or swipe from the right of the screen) to open the Charm Bar
  2. Click Settings
  3. Click Change PC Settings
  4. Click General
  5. Click on Advanced Startup and then click Restart Now
  6. Go to Troubleshoot, then Advanced Options and then UEFI Firmware Settings
  7. Click Restart

Method #3: Automatic Repair

  1. Trigger your PC's Automatic Repair function. This can be done by turning your computer on and off three times in quick succession by holding down the power button.
  2. Once the Automatic Repair fails (and it will, since there's nothing wrong), click Troubleshoot
  3. Click Advanced Options
  4. Click UEFI Firmware Settings

After your computer restarts, it should open into the BIOS menu automatically:

  1. Go to Advanced tab
  2. Go to Performance
  3. Look for a Overclocking option.
  4. If it's there, make sure it’s disabled
  5. If it’s enabled, disable it
  6. Save the settings and exit BIOS by hitting the  key. Choose Yes if you’re asked to exit with saving the changes

Fix #2: Check drivers

If you recently installed a new driver, make sure it’s compatible with the Windows version you currently have on your computer. 

To do so, follow these steps:

  1. Start your computer
  2. Press the  and  keys to open Charm Bar
  3. Type Device Manager in the search box
  4. Determine if there’s any corrupt drivers installed (any errors reported) from the list of drivers installed.

    If there are no errors reported in Device Manager, update the software of your most recently installed driver.

    To do so, right-click on the drive and click Update Driver Software

Fix #3: Run chkdsk or Windows Memory Diagnostic

To troubleshoot if the 0x0000124 (WHEA_UNCORRECTABLE_ERROR) error is caused by a corrupt or failing hard drive or memory, you can run the following tools:

  • chkdsk
  • Windows Memory Diagnostic
  • Easy Recovery Essentials

  1. Press the  and  keys to open Charm Bar
  2. Type cmd in the search box
  3. Right-click on Command Prompt from the results list

  4. Click Run as Administrator
  5. Once Command Prompt has loaded, type: 
    chkdsk C: /f

    Replace C: with the letter of the drive where your Windows is installed.

  1. Press the  and  keys to open the search box from Charm Bar
  2. Type Windows Memory Diagnostic in the search box
  3. Click on Windows Memory Diagnostic from the search results list
  4. Continue with the Windows Memory Diagnostic wizard to complete the process and wait for the results
