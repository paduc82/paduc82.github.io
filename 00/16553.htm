hansv mvp say:

You don't need a crosstab query for this, but an 'ordinary' Totals query.

I'd start from scratch:

  • On the Create tab of the ribbon, click Query Design.
  • Add the Customers table, then close the Show Table dialog.
  • Add the City and CustID fields.
  • Click to turn on Totals in the Show/Hide group of the Design tab of the ribbon.
  • Leave the Total: option for Customer set to the default Group By.
  • Change the Total: option for CityID to Count.

The X is not in a Red box until you have your mouse pointer on it. Otherwise it is currently the background color for the application (blue for Word, Green for Excel in colorful theme).. When you have your pointer over it, it will say "Close" which is half-true. If there is more than one document/workbook open, it will close the current one.
