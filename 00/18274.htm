If i make a cell add up the two cells to the left of it, and if i expand it downwards each cell below will add up
the two cells towards its left. I am looking for away to lock 1 cell, so that they all add that cell to the one on
their left, without me having to put the fomula into each cell anew.
Thx
ron coderre say:

Put a dollar sign ($) in front of the part of the reference you want to freeze.



Examples:
=$B4...will always reference column B, but the row reference will change when the cell is copied up or down
=B$4...will always reference row 4, but the column reference will change whent the cell is copied left or right
=$B$4...will always reference cell B4

Does that help?

Put a $ sign in front of the part you want to "lock". For example, using A1 as an example, to freeze the Row only, you would write A$1... to freeze the column only, you would write $A1... and to freeze both the row and column, you would write $A$1.
