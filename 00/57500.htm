Method 1: Launch the elevated Command Prompt and check the issue.

 

To open the elevated command prompt, you may follow these steps:

b)       Select “Command Prompt (Admin)” from that menu and then try running the commands on it and let’s know the result.

http://windows.microsoft.com/en-US/windows-8/create-user-account

 

After creating a new local account, you may set administrative privilege for the new account. Follow these steps for the same.

 

Step 2: Change your Account Type from Standard to Administrator:

 

a)       Login to the computer using an administrator account.

b)       Press “Windows Logo” + “W” keys on the keyboard to open the “Settings” search bar.

c)       Type “User Accounts” in the search box and click on “User Accounts” icon.

d)       From the “User Accounts” window, click on “Manage another account”.

e)       Click on the new user account which you have just created and then click on “Change the account type”.

f)        You can change the account type from “Standard” to “Administrator” from there.

g)       Click on “Change Account Type” button.

trekdozer say:

If you receive the error message "The EPSON printer driver dealing with CD printing is not installed" when you start EPSON Print CD in Windows, then either the printer driver or EPSON EasyPrintModule is not installed on your system.

EPSON EasyPrintModule collects data from any EPSON printer driver installed on your system. This data is needed by applications like EPSON Creativity Suite or EPSON Print CD to show the correct printing options for your printer(s).

If EPSON EasyPrintModule is not installed then this data cannot be collected. As a result, the application cannot find the printer driver and reports that it is not installed on the system.

To resolve the issue, ensure that the following software is installed:

  • The EPSON printer driver.
  • The latest version of EPSON EasyPrintModule.
