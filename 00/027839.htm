gabriel bratton say:

For those still seeing missing attachment issues I wanted to update the latest I know.  Most recently we had two main bugs with attachments.

The first one is an issue with sending plain text emails with Outlook 2016, https://support.microsoft.com/en-us/kb/3132071.  It was fixed in the January update.  More details are listed in the KB.

The second issue causes attachments to go missing if there are inline images or if the email has a signature with an image.  You can confirm this issue by sending the email with attachments without the image.  This issue is scheduled for a fix in Outlook 2013 in the February update.  The fix for Outlook 2016 has been checked in but we do not yet know what release it will go in.  I'll update when I know more about the 2016 release.
