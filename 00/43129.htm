One of the easiest ways to transfer files is using the Windows Easy Transfer.

You can transfer files and settings using a network share, or transfer files and settings using removable media such as CD/DVD, or an external hard drive. Windows Easy Transfer does not migrate installed applications.

Using Windows Easy Transfer, you can migrate most files and program settings. Below is a table describing the types of data and settings that can be moved.

 


Data Type
Specifics

 Files and Folders Any files found in My Documents, My Pictures, and Shared Documents folder will be selected to be moved automatically. You can also add other documents that are in other locations. E-mail messages and Settings E-mail messages, contacts, account settings, and address books from Outlook Express, Outlook, Windows Mail, and other programs. Transfer Manually:
Log on with the new user account that you may have created. Make sure that this new user account has Administrator privilege.

Open Explorer > Organize > Folder and Search Options > View tab > check Show hidden files and folders box > OK to save settings.

Now navigate to your C:\Users\(Old Username) folder and copy all the files and folders in the C:\Users\(Old Username) folder to the C:\Users\(New Username) folder.

¡firedog say:
Did this coincide with a Windows update?


Probably. Windows update KB2670838 (26 February) has caused problems like those you describe, as well as many other types of problem with other applications and with Windows itself.

You can find out whether you have it installed at Control Panel > Windows Update > Installed Updates. If you have, then please check whether uninstalling it using the Uninstall button on that page and rebooting the computer restores the proper display in Windows Live Mail. Note that because KB2670838 is essential for IE10, uninstalling it will also uninstall IE10.

You might also want to investigate whether updated display drivers are available for your graphics card, and whether you have third-party graphics handlers (e.g. picture viewers) that maybe designed to work with older versions of the Windows components that this update changes.
