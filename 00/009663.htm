bob jones aka: cybertaz say:

In Word, Language is a character level attribute. Whatever is specified as the Default Language applies only to new documents generated from that point forward... it has no bearing at all on existing documents.

In order to change the Language of content in existing documents the text must be selected before changing the Language specification. [IOW, selecting a different Language while just having the insertion point in the text affects only text typed at that point.] If the Language of the entire document needs to be changed, Select All, then specify the Language... Or (preferably) the Styles used in the document can be modified to include the preferred Language attribute..

Split multiline cell into separate rows - Microsoft Community

ali khafaji say:

Split multiline cell into separate rows

I have hundreds of tables in a document with each containing long multiline cells. I have to layout the document in indesign and the problem is that indesign oversets tables with multiline cells larger than one page.

Is there a way to split multiline cells into separate rows automatically?

doug robbins - mvp office apps & services (word) say:

Use:

Dim i As Long, j As Long
Dim docsource As Document, doctarget As Document
Dim tblsource As Table, tbltarget As Table
Dim rowtarget As Row
Set docsource = ActiveDocument
Set doctarget = Documents.Add
For j = 1 To docsource.Tables.Count
    Set tblsource = docsource.Tables(1)
    Set tbltarget = doctarget.Tables.Add(doctarget.Range, 1, 3)
    tbltarget.Columns(1).Width = InchesToPoints(2.5)
    tbltarget.Columns(2).Width = InchesToPoints(1)
    tbltarget.Columns(3).Width = InchesToPoints(2.5)
    With tblsource
        For i = 1 To .Cell(1, 1).Range.Paragraphs.Count
            Set rowtarget = tbltarget.Rows.Add
            rowtarget.Cells(1).Range.Text = Replace(.Cell(i, 1).Range.Paragraphs(i).Range.Text, vbCr, "")
            rowtarget.Cells(3).Range.Text = Replace(.Cell(i, 3).Range.Paragraphs(i).Range.Text, vbCr, "")
        Next i
    End With
    With tbltarget
        .Rows(1).Delete
        .Columns(1).Width = InchesToPoints(3)
        .Columns(2).Width = InchesToPoints(1)
        .Columns(3).Width = InchesToPoints(3)
        With .Range.Font
            .Name = "Arial"
            .Size = 9
        End With
    End With
Next j
