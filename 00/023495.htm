jay freedman say:

Don't use AUTONUMLGL (or AUTONUM or LISTNUM) field for this, because you can't set the number of digits they use. Instead, use SEQ fields, with a numeric picture format:

{ SEQ drawing \# "TSN 0000" }

The word 'drawing' can be anything you choose, as long as it's the same in all the SEQ fields used for drawings (and don't use 'figure' because that's what the Insert Caption command uses for its fields).
