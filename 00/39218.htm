HOW DO I MAKE MYSELF THE ADMIN WITH NO OTHER ADMIN ON USER ACCOUNTS
malkeleah say: You would never want to do this in Vista (or Windows 7). All regular users - including you - should be Standard users for their daily work with an administrative account (which no one ever needs to log into) set up for elevation purposes.

General Recommendations For Setting Up Users In Vista:

You absolutely do not want to have only one user account. Like XP and all other modern operating systems, Vista is a multi-user operating system with built-in system accounts such as Administrator, Default, All Users, and Guest. These accounts should be left alone as they are part of the operating system structure.

You particularly don't want only one user account with administrative privileges on Vista because the built-in Administrator account (normally only used in emergencies) is disabled by default. If you're running as Administrator for your daily work and that account gets corrupted, things will be Difficult. It isn't impossible to activate the built-in Administrator to rescue things, but it will require third-party tools and working outside the operating system.

The user account that is for your daily work should be a Standard user, with the extra administrative user (call it something like "CompAdmin" or "Tech" or the like) only there for elevation purposes. After you create "CompAdmin", log into it and change your regular user account to Standard. Then log back into your regular account.

In order to print the documents, the computer must have some software that can "understand" the documents. 

If you are permitted to connect a USB flash drive to your work computer (and some businesses not only don't permit this but have taken steps to disable this ability), the first thing to do is to connect the USB drive and double-click on one of the files.  If it opens so that you can read it, you can print it (generally by clicking File > Print in the program that is used to view it).

If you get a message that Windows does not recognize the file or lacks a program to open it, then if you are permitted to install software on your work computer (and you very well may not be allowed to do so) --

If the documents were created in Word 2003 or earlier (the file names end .doc), you can download and install the free Word Viewer --> http://www.microsoft.com/en-us/download/details.aspx?id=4

If the documents were created in Word 2007 or newer (file names end .docx), you have to install the Word Viewer AND then install the free Office Compatibility Pack --> http://www.microsoft.com/en-us/download/details.aspx?id=3

The Word Viewer is also supposed to be able to view (and thus print) the following:
  • Word Macro-Enabled Document (.docm)
  • Rich Text Format (.rtf)
  • Text (.txt)
  • Web Page formats (.htm, .html, .mht, .mhtml)
  • WordPerfect 5.x (.wpd)
  • WordPerfect 6.x (.doc, .wpd)
  • Works 6.0 (.wps)
  • Works 7.0 (.wps)
  • XML (.xml)
