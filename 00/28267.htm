reference a cell in a worksheet in a formula with a cell in a - Microsoft Community

javablood say:

reference a cell in a worksheet in a formula with a cell in a different worksheet

I have a formula in worksheet "SheetA" in which a want to reference two cells that will be used to identify a range in another worksheet.  Here is the formula:

 

=IFERROR(INDEX(Actuals!$R$3:$R$81,MATCH($A$4:$A$85,Actuals!$A$3:$A$81,0)),0)

 

Within "Actuals"  are multiple columns that represent years so I want to be able to insert the column reference (in this case "R") from "SheetA"

so I do not have to change the formulas in 50 rows.  So let's say I am working in "SheetA" and I put an "R" in L2, now what.  I tried many

versions with &s and quotes with no luck.  Any help is appreciated.

chip pearson say:

Use the INDIRECT function. For example,

 

=IFERROR(INDEX(INDIRECT("Actuals!$"&L1&"$3:"&L2&"$81"),MATCH($A$4:$A$85,Actuals!$A$3:$A$81,0)),0)

So, if L1 on sheet1 contains the letter R, you'll get column R from Actuals. The INDIRECT function takes text (which can be built up with the various string functions) as its input and resolves that text to a real range that can be used by other functions.

 

A very simple example of INDIRECT is

 

=INDIRECT(A1&"10")

 

This will return the value in row 10 in the column identified within A1. So, if A1 is "C" (no quotes), the formula is the same as =C10 .

Be careful with INDIRECT and pay close attention to the quoted and unquoted pieces of the text string that is built up and passed to INDIRECT. It is easy to get lost in the quotes and parentheses.

The only way to get Word to "skip over" words is to tell it not to check spelling and grammar. You can do this with a No Proofing character style applied to those words. I don't think there's any way to exempt on the basis of font.
