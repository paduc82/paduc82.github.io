suzanne s. barnhill say:

You can save it as a building block (Quick Part), either as an AutoText entry or specifically in the Footers gallery.

  1. Create the desired footer in a document.
  2. Select the footer content.
  3. In the Header & Footer group on the contextual Header & Footer Tools | Design tab, click the Footer button.
  4. On the menu, choose Save Selection to Footer Gallery...
  5. In the Create New Building Block dialog, enter a name for the footer and click OK to save it with the default settings.

By default, it you save it as a building block in the Footers gallery, it will be saved in Building Blocks.dotx. If you save it as an AutoText entry (select the text and press Alt+F3 to get the same dialog with different settings), it will be saved in the AutoText gallery in Normal.dotm. If some template (for example a shared one) is attached to the document, you will also have the option to share the building block or AutoText entry in that template. Templates that are loaded as add-ins (from the Word Startup folder) are also options. If you have a shared Startup folder, an add-in template would be a good choice.

This facilitates inserting the footer. Unfortunately building blocks (including AutoText entries) cannot be edited: to change one, you have to create the revised version and save it in the same place with the same name. If you want it to be updatable in existing documents, you can insert an AutoText entry as an AutoText field; when updated along with other fields, it will update to the current content of the AutoText entry.

bernd p. say:

I see 2 workarounds:

  1. Because the online mode works: Navigate on Onedrive to your notebook (don't open it). Then use Folder Actions to download your notebook (maybe more than one notebook gets downloaded), as a ZIP file. Unpack the (right) notebook (the folder with the name of your notebook, this IS your notebook if it's stored in the Windows file system) into the folder Onenote Notebooks under Documents. Change the notebooks/folders name !
    Close the old notebook to prevent the crash. Then open the new now local copy with Onenote. If no error message appears use the Share function to move it to Onedrive. If this works as well then use it.
  2. As an alternative: Stop your Internet connection to prevent Onenote synching. Open Onenote. Create a new local notebook. Now copy the content from the not synching notebook to the new one. Close the not working notebook. Reconnect Internet. Now share the new notebook to Onedrive as in workaround #1.

If any of these workarounds solves your problem after some days of hope use online mode to delete the problem notebook.

jnail7 say:

unable to access format autoshape for textbox or save selection to watermark gallery

This is a critical issue as many of our Graduate Students make use of Landscape pages for extended figures/tables.  For previous versions of Word we had students do the following to create appropriate page numbers for such pages: http://www.youtube.com/watch?v=SYZi8T_bVpU

 

Here are the step by step instructions:

" Use the following steps to create landscaped pages and have the page numbers display correctly (Note that landscaped pages will need to be rotated in the final .pdf to match the portrait pages):

1.      Isolate the landscaped pages from the rest of the document by going into the header and footer and removing the Link to previous in both the section containing the landscaped pages AND the section immediately after the landscaped pages (This will keep any changes to the header/footer from affecting the entire document.).  You will need to click in the footer to remove the Link, and then click in the header to remove the Link there.

2.      Once isolated, go back to the footer in the landscaped section and delete the current page number and set the margins to the following: Top 1.25 Left 1.2 (this will guarantee that there is nothing overlapping the page number), Right 1.25" and Bottom 1

3.      While in the footer of the landscaped section, insert a new page number by clicking on the Page Number option on the ribbon, then move your mouse over Page Margins and select Large, Left in the plain number section.

4.      Click on the new page number and then highlight all of the text inside this text box.

5.      Go to the home tab and change the font of the page number to be the same font and size used in the document (usually Times New Roman 12pt)

6.      With the text still highlighted, right-click and select Paragraph.  Make sure the text is set for single-spaced and that spacing before and after are set to 0.

7.      Now click the edge of the textbox so that just the border is highlighted (you should see the little control circles for altering the text boxs size).  Right click and select format autoshape.

8.      In the Format Autoshape window, click on the Text Box tab and set all internal margins to 0.  Also remove the check from Wordwrap text in autoshape and put a check in Resize autoshape to fit text. (this removes excess space that can misalign centering later)

9.      Now click on the Text box tools Tab (in yellow above the tool ribbon) and click text direction (located in the upper left of the tool ribbon).

10.  Now click on Position (near the right of the tool ribbon) and select more layout options

11.  In the layout options window, set the following:
            Horizontal: Absolute Position 1 to the right of Page
            Horizontal: Absolute Position 1 to the right of Page
            Vertical: Alignment Centered relative to Margin.

            Vertical: Alignment Centered relative to Margin.

12.  Note: that the centering can be out of alignment if the header and/or footer is larger than the page margins.  To check for this, reduce the header and footer until the page number stops moving vertically.

The Landscaped page number is now done.   For multiple separate landscaped sections, you only have to do the above steps once.  After creating the landscaped page number, you can then select the page numbers text box and simply use copy/paste to add it to new section footers.  Just make certain that all landscaped sections are isolated from the rest of the document by doing step 1 above for each section.

Advanced option: saving the new page number style in the normal.dotx template for future use:

1.      Go into the header/footer of a landscaped section and highlight the text box of the page number.

2.      In the Page Layout tab, select Watermark and click Save selection to watermark gallery

3.      In the Create New Building Block window enter the following
            Name:  Landscaped Page Number
            Name:  Landscaped Page Number
            Gallery:  Page Numbers (Margin)
            Gallery:  Page Numbers (Margin)
            Category:  Plain Number
            Category:  Plain Number
            Description:  Page number for landscaped pages.              Description:  Page number for landscaped pages.  1 from bottom and centered by margins.
                        Save in:  Building Blocks.dotx
                        Save in:  Building Blocks.dotx
                        Options:  Insert content only

                        Options:  Insert content only

The style is now saved to the base template file that Word uses.  It will now be available for future use when new files are opened.  The newly saved page number will be found just like any other page number (page number->Page margins-> Plain number ->Landscaped Page number)"
