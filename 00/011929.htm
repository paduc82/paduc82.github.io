Also, check under Settings: System/Storage. It will show you all of your available memory. This is what mine shows:

In theory, if you are getting that message it is because your internal memory is low. Consider moving all Music, pictures etc to the memory card. I generally install my programs to Internal Memory but games and programs I rarely use I install on the memory card.

Personally, I would not move programs to the Memory card. I would uninstall, reboot my device then reinstall to the Memory card. Don't forget to change where it is installed to before you reinstall.

Also, keep in mind that even though you install to Memory Card some of it has to be placed in Main Memory

Lastly, it is easy to replace a memory card. If you get a new SD card, there should be a slot where it is located that allows you to easily slide it out and replace it. 

To remove your memory card first do this:

Click on Settings/Storage

Double tap on your SD card and it will open

Scroll to the bottom and click on REMOVE

Then you will be able to remove it and insert the new card. I generally power off my device before inserting a new card then power it back on. It should detect your new card.
