Function Concat(strtcell As Range, Optional delim As String) As String
Dim LastRow As Long
Dim c As Range
LastRow = Cells(Cells.Rows.Count, strtcell.Column).End(xlUp).Row
For Each c In Range(Cells(strtcell.Row, strtcell.Column), Cells(LastRow, strtcell.Column))
    Concat = Concat & delim & c.Value
Next
    Concat = Mid(Concat, Len(delim) + 1)
End Function

suzanne s. barnhill say:
  1. Display nonprinting characters and table gridlines so that you can see what you're doing.
  2. Make sure that the insertion point is in the header before you insert the table.
  3. Make sure that the table wrapping is set to None rather than Around and that there is not an empty paragraph above the table.
  4. There will be an empty paragraph below the table; this is unavoidable but not really a problem since you want space between the header and the document body anyway. If for some reason you want the header flush to the top of the document, you can format the empty paragraph as 1 point or Hidden.
  5. A header that expands beyond the default depth (if you have the default 1" top margin and 0.5" header margin, this will be half an inch), will push the document body down. This can be used to advantage if the header is intended for only some pages, but if the header is to appear on every page, it would be better to modify the top and header margins to allow space for the header.
  6. When you have the header adjusted to your satisfaction, you can save the document as a template. If it is just the header you want to save for use in documents based on other templates, you can save it as a header building block.

Password saving is on by default in Internet Explorer, but here's how to turn off password saving:

  1. Open Internet Explorer by clicking the Internet Explorer icon on the taskbar.

  2. Click the Tools button , and then tap or click Internet options.

  3. On the Content tab, under AutoComplete, tap or click Settings.

  4. Clear the User names and passwords on forms check box, and then tap or click OK.

AutoComplete saves you time by remembering passwords and other info you enter into forms online. Since this info is securely stored on whatever PC you're using at the time, you should be careful about using AutoComplete on public or shared computers. When you're using a public or shared PC, make sure AutoComplete is turned off. You can also remove AutoComplete history from that PC if you accidentally enter info that you don't want to share—like the billing address for a credit card.

To delete AutoComplete history on a PC

  1. Open Internet Explorer by clicking the Internet Explorer icon on the taskbar.

  2. Click the Tools button , and then click Internet options.

  3. On the Content tab, under AutoComplete, click Settings, and then click Delete AutoComplete history.

  4. In the Delete Browsing History dialog box, select Form data and Passwords. To clear web address entries, you must select the History check box, which also deletes your browsing history.

  5. Click Delete to clear your AutoComplete history. Click OK, and then click OK again.

Even if AutoComplete is turned off, Internet Explorer will still ask if you want to save passwords. You should turn off password saving if you're concerned about your privacy.
