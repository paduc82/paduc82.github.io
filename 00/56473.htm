The PF on an SSD would be slightly quicker than on an HDD. Very slightly.
With an HDD  superfetch should be on.
I haven't studied Intel Rapid Storage, but an SSD is a nice home for the OS.

Reinstall windows 10 without losing data

my laptop currently windows 10 won't boot into desktop or in login screen after I did some force shutdown by holding down power button.

    2. Bootrec /fixboot

    4. Chkdsk c: /f /r /x

in step 4, I've got an error in step 2 then i close the cmd in advanced boot options.

now my question is, what other method should I need to do without losing my very important data inside my drive? 

is system repairing using a bootable disk can be use? withput losing data?

mark ole say:

Hi Chris,

It is important to know what was the result of running chkdsk, as this will help us determine if this is a possible hard drive failure. Can you tell us what was the result of chkdsk?

However, if the result did not find any errors, then you can check for corruption in the system files that is preventing your computer from booting up. We suggest that you run the System File Checker tool to address this. Use the installation media to run the tool.

  1. Connect the installation media to your computer and then turn it on.
  2. On the initial setup screen, enter your language and other preferences, and then select Next. If you're not seeing the setup screen, your PC might not be set up to boot from a drive. Check your PC manufacturer's website for info on how to change your PC's boot order, and then try again.
  3. Select Repair your computer. On the Choose an option screen, select Troubleshoot > Command prompt.
  4. Type sfc /scannow and press Enter.
  5. Once done, restart the computer.

If still your computer is unable to boot up, then we recommend you refresh your PC. You're not going to lose any files as long as you choose this option. However, we cannot guarantee that you're not going to lose any programs. To do this , follow these steps:

  1. Connect the installation media to your computer and then turn it on.
  2. Select Repair your computer. On the Choose an option screen, select Troubleshoot >Refresh your PC.

Let us know how it goes after performing the suggested steps.
