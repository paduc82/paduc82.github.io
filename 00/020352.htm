doug robbins - mvp office apps & services (word) say:

With the cursor located at the end of the third page, on the Page Layout tab of the Ribbon, click on the Breaks dropdown in the Page Setup section of the ribbon and then insert a Next Page Section Break (removing any unwanted blank page between the Section Break and the start of the text on the fourth page.  Then access the Headers and Footers and uncheck the Link to Previous, (also uncheck the different first page).  Then, in the Header & Footer section of the Header & Footer Tools>Design tab of the ribbon, expand the Page Number item and select Format Page Numbers ...

In the Page Number Format dialog that opens, set the Page Numbering to Start at 1.

 

With the Header and Footer unlinked from that in the previous Section of the document, you can insert a header in the Section that starts on page 4, independent of any (or no) header on the first three pages that are in first Section of the document.

    With Worksheets("Datasheet")
        For RwNr = 2 To .Range("U:U").Cells.SpecialCells(xlCellTypeConstants).Count
            .Cells(RwNr, 23).Value = Evaluate("=WEEKNUM(" & .Cells(i, 21).Value & ";2)")

This keeps coming up with a #VALUE! error. When I enter the function directly into excel, it does work.

mike h.. say:

Hi,

 

The variable i in your code isn't initialised or incremented so that's one source of an error. The other is the use of .cells( when using evaluate. The formula is expecting an address. Try it like this.

 

Dim RwNr As Long
With Worksheets("Datasheet")
         For RwNr = 2 To .Range("U:U").Cells.SpecialCells(xlCellTypeConstants).Count
             .Cells(RwNr, 23).Value = Evaluate("=WEEKNUM(" & .Cells(RwNr, 21).Address & ",2)")
         Next RwNr
End With
