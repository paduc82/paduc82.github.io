myronh say:

Normally you can change ownership on a folder through the admin group account...yours. Sometimes, however, Windows will balk. At this point it's usually time to break out the big guns...the hidden administrator account.

You activate this account through the command prompt.

Click the windows orb and type cmd in the search block
Right click the cmd.exe above the search block and run as administrator
in the command window type net user administrator /active:yes
Switch users
log onto the administrator account
Make your ownership changes...if you want to change ownership to this account, complete your deletion here as well.
log off the administrator account and switch back into your normal account
If you didn't log off, the command window is still active (if you did log off, reopen the command prompt as administrator)
in the command window type net user administrator /active:no*
Exit the command prompt

* If you would like to keep the administrator account active (not recommended, but your choice) you don't need to disabe, but you should include a strong password. This can be done in the command prompt or through the control panel user accounts section. If you forget the password you can always change it with your regular admin account.

smartypants_92 say: For all HP amd 7690m xt switchable graphics users, please find the drivers at. Please note that these are drivers for 64 bit Windows 8.

Even if you don't own an HP laptop, you should still be able to use this driver. Follow these instructions and report back if you face any problems.

Before we begin, make sure that you have catalyst control manager installed (without the display driver)
1. Go to device manager and navigate to the display driver section
2. Right click on intel hd 3000 and select update drivers -> Browse my computer for drivers
3. Provide the path to the folder W86A_INF present in the directory where the driver specified above is saved.
For example: C:\SP58815\Packages\Drivers\Display\W86A_INF

The device manager should install the drivers. If not try the same steps for the amd card listed under device manager.
