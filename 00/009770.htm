A couple of things....

It would be both a upgrade AND a clean install...

You would be upgrading to fulfil the requirement, but during the course of the install you would select to "Keep Nothing"... which is not technically a "Clean" install, but basically the same thing...

The difference is that with a true Clean Install, you would boot from the installation media, do a CUSTOM Install, and Reformat your OS drive or Partition....Or start with a blank HDD.

With the "Keep Nothing" option you're pre-existing files would be moved to a WINDOWS.OLD folder and isolated from the upgrade... Once the installation is finished the Windows.old folder can be deleted with a DISK CLEANUP of SYSTEM FILES.

Once this is done, you can start over and actually do a "Clean" install, probably by using your Win 7 license... You might also get lucky... Check out "Secondly" below.

About your Insider Preview...

You don't have to click on the box to GET PREVIEWS to have an activated and valid preview build on your machine...

That is something that started with builds after 10130 and only applies to getting more builds... It doesn't concern the actual "Activation" status of the build you are using...

All that is necessary is to right click on This PC and select Properties...

Down near the bottom, it should say "Windows Is Activated" in the area where the "Product ID" is listed...
