john korchok say:

Text boxes can be formatted

  1. with a theme,
  2. with a non-theme default font, or
  3. with local formatting.

A theme-formatted text box is one created using the steps I posted previously, where you set the slide placeholders and default text box to the font theme.

The examples you mentioned have the default text box set, but to a non-theme font.

A locally-formatted text box is one that the user has created, then set the font manually using Home>Font. Locally formatted textboxes do not use the theme any more, so pasting them using the destination does nothing. PowerPoint will always honor local formatting over themed or styled text.

All of these will behave a little differently when pasted.

john korchok say:

You're not seeing the Unformatted text option because you're pasting a text box. You would only see Unformatted text if you had selected the text inside the box and tried to paste that.

Compatibility with the Mac is not likely an issue, but in your first message you said this was a new template you created in 2010.

  1. In PowerPoint, select a slide where you see the problem.
  2. Choose View>Slide Master. The slide layout used in the problem slide is highlighted.
  3. Choose Slide Master>Edit Theme>Fonts. What font theme displays?
  4. Click on the title and text placeholders on the layout. On the Home tab, what font displays when those placeholders are selected?

Column A = Address

Column B = City

Column C = State/Province

 

Using a macro in Excel I want to call Google Maps passing the info from columns A, B and C to have the corresponding map displayed within IE.  I would like to know the correct call or http statement to call from within a macro to display something like 725 5th Avenue, New York, NY

 

Has anyone tried this with success?

hansv mvp say:

Try this (modify as needed):

 

Sub GetGoogleMaps()
    Dim r As Long
    Dim strURL As String
    r = 2 ' change in your code
    strURL = "http://maps.google.com/maps?q="
    strURL = strURL & Replace(Range("A" & r), " ", "+")
    strURL = strURL & "+" & Replace(Range("B" & r), " ", "+")
    strURL = strURL & "+" & Replace(Range("C" & r), " ", "+")
    ActiveWorkbook.FollowHyperlink strURL
End Sub
