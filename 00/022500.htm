jay freedman say:

These two steps should solve your problem with printing markups:

  • Go to File > Options > Trust Center > Trust Center Settings > Privacy Options. Under the "Document-specific settings" heading, uncheck the box for "Make hidden markup visible when opening or saving". (Despite the heading, this option applies to all documents you open.)
  • On the Review tab, be sure the top right dropdown in the Tracking group is set to "No Markup" and save the document. This will need to be done for each existing document that you use as a "template" (by the way, that's a bad practice; you should use actual templates to create new documents) so the new ones you create will inherit that setting.

When I make these two settings, the Print Markup setting on the Print page is automatically not checked, even though the document contains large numbers of changes and comments.

While the SEQ field Doug describes will certainly do the trick, if you want numbering that updates entirely automatically, then apply numbering using the Numbering button on the Home tab. You could use a SEQ field and create and autocorrect entry that contained the word Recommendation and that field to facilitate it use.

 

Press Control+F9 to insert a pair of field delimiters inside of which you type

{ SEQ rec }

Then in front of it type Recommendation #

Recommendation #1{ SEQ rec }

Then select that and then from the File tab of the ribbon, select Options and then click on Proofing and then AutoCorrect Options and go to the Autocorrect tab and in the replace text as you type section, you should see

Recommendation #1 under the With:

Enter something to the left of it which will be replaced by Recommendation # and a sequential number each time it is used (such as rec3) then click on the  Add button and press OK then OK again

Back in your document, now use Alt + F9 to toggle of the display of field codes and thereafter, whenever you type rec3 and then press the space bar, it will be replaced by Recommendation #n where n is a sequential number.

If you delete one or insert one out of sequence, using Ctrl+A and then F9 to update fields will renumber the recommendations in sequential order.

    hansv mvp say:

    Select the one-by-two table, then run this macro:

    Sub SplitVertically()
      Dim strParts() As String
      Dim i As Long
      Dim rng As Range
      Set rng = Selection.Cells(1)
      strParts = Split(rng.Value, vbLf)
      For i = 0 To UBound(strParts)
        rng.Offset(i, 0) = strParts(i)
      Next i
      Set rng = Selection.Cells(2)
      strParts = Split(rng.Value, vbLf & vbLf)
      For i = 0 To UBound(strParts)
        rng.Offset(i, 0) = strParts(i)
      Next i
    End Sub
