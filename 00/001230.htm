bob jones aka: cybertaz say:

I appreciate how you might believe as you do, but Word really has no way of knowing whether its output is being directed to 2 sides of a sheet or only to one... nor does it care :-) In fact, Word doesn't even know what size paper is being used [despite the specified document dimensions], the number of pages per sheet, etc.  It simply hands off the data including the page layout specs for margins & orientation. IOW, Duplex is not a layout specification of the document. It's a print function handled by the printer & its driver software.

The most common reason for a printer to not perform as it's expected to is that its driver software is corrupt or out of date. In such cases, it isn't unusual for certain features to "work just fine" with some programs but not with others. There have been numerous reports of similar situations related to Apple's rapid-fire release of ne operating system software... developers no sooner get something right when Apple goes & pushes out a whole new set of underpinnings which require different coding.

Anyway... The first steps to resolution of the issue are:

  • Make certain that your version of OS X is fully updated.
  • Office 2016 should be at v15.27 or later
  • Equally important, check the mfr's web site to see if there is a firmware and/or driver update for your model. You might also see if others are reporting a comparable issue.

        Dim Item As Object

            Set SubFolder = Nothing
            Set Inbox = Nothing
            Set ns = Nothing

            Set wsh = CreateObject("WScript.Shell")
            Set fs = CreateObject("Scripting.FileSystemObject")

            DestFolder = MyDocPath & "\" & Format(Now, "dd-mmm-yyyy hh-mm-ss")

            DestFolder = DestFolder & "\"

                If LCase(Right(Atmt.FileName, Len(ExtString))) = LCase(ExtString) Then
                    FileName = DestFolder & Item.SenderName & " " & Atmt.FileName

                End If
            Next Atmt

            MsgBox "No attached files in your mail.", vbInformation, "Finished!"
        End If

        ' Clear memory
    ThisMacro_exit:

        Set ns = Nothing
        Set fs = Nothing
