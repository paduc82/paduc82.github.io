Hello Mike

Some background info on how things work in general regardless of which version of Outlook is being used - whether that's Outlook '2003 or '365

Any desktop <email client> (of which Outlook, any version, is just one of many) "connects" to an <email account>. What gets "sync'd" (if anything) is completely controlled by an "industry standard" protocol (POP3, Imap or Exchange) applicable to any <email client>. In that respect, there is nothing special in Outlook '365 ('2013/'2016/'2019) in terms of POP3 accounts (or IMAP/Exch for that matter).

So in terms of protocols

POP3 - is a one-way one-time download of the <Inbox> data on the device where the account is configured. Other email folders or data types such as contacts/calendar are not sync'd with the server.

IMAP - is a bi-directional sync of email folders between server and device. Contact/Calendar data is not sync'd with the server. Every device connected to the account using IMAP will see the same email data.

Exchange - is a bi-directional sync all data types between server and device which includes Contact/Calendar data. Like IMAP, every device connected to an Exchange account will see the same email/contact/calendar data.

In reference to "having things available when offline" - that applies regardless of protocol used. Imap and Exchange accounts store the offline data in an .ost file (Offline Storage) which is what is used when the account is disconnected. Conceptually no different then a PST file used by a POP3 account.

In terms of the "Store version of Office" (codenamed Centennial), the way to resolve that problem is to (1) uninstall the store version and then (2) download/install the standard C2R (click-to-run) version of Office. The Store version of Office stores the Outlook profiles differently which is likely the crux of whatever issue you may be encountering with a 3rd party sync program.

Also in terms of sharing files between devices, that's done by using OneDrive, something that can be done regardless of which version of Office is used (if any at all). What Office '365 does is make it much easier and seamless when working/saving docs in the various applications.
