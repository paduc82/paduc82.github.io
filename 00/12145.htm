hansv mvp say:

The color functions on Chip Pearson's web page are for directly applied colors; they don't work with conditional formatting colors.

If a row is grey if column N contains 0, you can use column N in your SUMIF formula:

 

=SUMIF($N$2:$N$100, 0, E$2:E$100)

 

sums the cells in E2:E100 where the corresponding cells in N2:N100 are 0, and

 

=SUMIF($N$2:$N$100, "<>0", E$2:E$100)

 

sums the cells in E2:E100 where the corresponding cells in N2:N100 are unequal to 0. You can fill these formulas to the right to sum F2:F100 etc.
