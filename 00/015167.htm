cjlim say: The most common causes of caller ID problems are:

- duplicate entries in your contacts, including entries stored on your sim card. This is a problem becuase the phone does not know which name to display and so it doesn't display any. Duplication also includes the same number in different formats. Try searching for a PART of the number as if for example you search for +353 123 456 789, it won't find +353123456789, whereas if you search for 6789 it would.

Having different formats is a problem for caller ID.

- using your international dial code instead of "+", eg: using 00353123456789 (wrong) instead or +353123456789 (right).

- spaces in numbers (some phones, some firmware versions only), eg: +353 123 456 789 (wrong) instead or +353123456789 (right).

- MS Outlook putting "(0)" in number - if you define a country for a contact in MS Outlook it will automatically add a country code and insert a "(0)" in the number in front of the area code, eg: +353123456789 becomes +353 (0) 123456789. You need to remove the "(0)"s

- there has been reports of more than 7 digit numbers causing problems. Some phones may not differentiate between +353123456789 and +353993456789.

suzanne s. barnhill say:

Doug's reply is the correct approach if you really want or need a section break. If you don't, but just want to omit the header on the last page, there is another approach:

  1. After entering the content you want in the header on pages 2-6, select it and press Ctrl+X to copy it to the Clipboard.
  2. In the header, press Ctrl+F9 to insert a field.
  3. In the field (between the braces), type IF and a space.
  4. After the space press Alt+Shift+P to insert a { PAGE } field.
  5. Type a space and then < and another space.
  6. Press Ctrl+F9 to insert another field.
  7. In that field, type NUMPAGES and then place the insertion point outside the { NUMPAGES } field but still in the IF field.
  8. Type two quotation marks ("").
  9. Between the quotation marks paste the content from the Clipboard.
  10. Your resulting field will look like this:
    { IF { PAGE } < { NUMPAGES } "Content you want in the header" }
  11. Press F9 to update the field. You should now see the header on every page but the first and last.

The syntax of the IF field tells Word to print the header content if the page number is less than the number of pages.

In Excel how do I set default properties? - Microsoft Community

donald o'connor say:

In Excel how do I set default properties?

In Excel how do I set default properties such as cell wrap?

hansv mvp say:

For a single workbook: right-click the Normal style in the list of styles.

Click Modify...

Set up the style the way you want, then click OK.

For all new workbooks:

Set up a workbook the way you want; in particular, modify the Normal style.

Save the workbook as an Excel template named Book.xltx (this name is required) in the folder %appdata%\Microsoft\Excel\XLSTART.

Excel will use this template:

- When Excel starts

- When you click New on the Quick Access Toolbar

- When you press the keyboard shortcut Ctrl+N

It won't use the template when you select File > New and select Blank Workbook.

Also create a version with just one worksheet and save it as an Excel template named Sheet.xltx in the same folder. Excel will use this template when you insert a new worksheet.

douglastryan say:

LIVE ID / Skype ID

How do I determine what my LIVE ID is when I am using Windows 10?
karen_[01] say:

Hello DouglasTRyan,

Welcome to Skype Community forum!

There's no option to view your live ID in Skype for Windows 10 (UWP). However, you can view it through Skype website, here's how:

  • In the Skype for Windows 10 (UWP), click your profile icon > select Manage account.
  • It will route you to Skype website, scroll down and click Purchase history. At the top corner of the Purchase history page, you will see your live ID.

I hope this information helps but if you have questions and clarifications, please do not hesitate to reply back.

julie_s1 say:

Hello

Please follow the troubleshooting steps below, then update us on what will happen.

To delete the shared.xml file:
1. On your keyboard, press the Windows key and R at the same time. The Run window opens.
2. Type %appdata%\skype and click OK.
3. Locate and delete the shared.xml file.
4. Open your Skype Name folder and look for config.xml. Delete this file as well.

To Re-name Skype Folder:
1. On your keyboard, press the Windows key and R at the same time. The Run window opens.
2. Type %appdata% and click OK.
3. Locate Skype and Rename it to SkypeOld
