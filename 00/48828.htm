simonorj say:
These are the steps I used:
  1. First, Go to:Start>Control Panel>Clock, Language, and Region>Change Keyboard or Input Methods>Change Keyboards.  (what you mentioned above)
  2. On the windows that pops up, click "Add..." button.
  3. Then scroll down to "Korean (Korea)" section and extract it. Also, extract "Keyboards".
  4. DON'T PUT A CHECKMARK ON THE CHECKBOX "KOREAN". If you already did, uncheck it.
  5. Put a checkmark on "Microsoft IME" box. Then press "OK". Note: If you don't see "Microsoft IME", click "Show more" and scroll down to "Microsoft IME" and check it. Then press "OK". No need to preview.
  6. Change the default input language and make it "Korean (Korea) - Microsoft IME".
  7. Click "OK" to finish.
Now you should be able to change between English and Korean by pressing LEFT "Alt" button. If it doesn't work, try rebooting your computer. Good luck!
