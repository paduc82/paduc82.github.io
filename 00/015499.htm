JD

vijay a. verma say:
Open Excel
File > Options > Proofing > AutoCorrect Options
Type ACN in Replace: box
Now CAN will automatically appear in With: box
You may be required to select this in down box if not automatically selected
Now press Delete

davem121 say:
Ho John, this is one of the most commonly mis-spelled words, so it is in the Autocorrect in Excel

In Excel, go to File - Options

Go to Proofing - Autocorrect Options

Scroll down to find acn, select and delete that entry to stop this behavior

That's It
