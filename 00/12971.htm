jay freedman say:

The macro already does what's necessary. Setting CC.LockContentControl = False unchecks the box in the Properties for "Content control cannot be deleted", which is the only thing that prevents deleting the control. The setting of "Contents cannot be edited" has no effect on CC.Delete.

What may be confusing the issue is that if the content control contains anything (that is, if CC.ShowingPlaceholderText = False), then executing CC.Delete removes the content control but leaves the content in place as plain text. If you need to get rid of all traces of the content control, you need to execute CC.LockContents = False to unlock the content and  CC.Range.Delete to remove the content before you delete the control.

Your macro has a lot of unnecessary code (unless you're only showing part of the full macro). This version is more efficient:

Sub Finish_DeleteCCs()
    Dim oRng As Range
    Dim CC   As ContentControl
    Dim LC   As Integer
'Remove all content controls
    Set oRng = ActiveDocument.Content
    For LC = oRng.ContentControls.Count To 1 Step -1
        Set CC = oRng.ContentControls(LC)
        CC.LockContentControl = False
        CC.LockContents = False
        If CC.ShowingPlaceholderText = True Then
            CC.Range.Delete
        End If
        CC.Delete
    Next
End Sub

dean bailes say:
The start time works when using IE, but does not work if you are using FireFox or Chrome.

You can get this working in Chrome or Firefox if you don't mind using a user script to do it (added through extensions on the browser).  I use the following script on Chrome to default my Outlook.com Calendar start time to 8am:

// ==UserScript==
// @name         Outlook.com Calendar Customiser
// @author       Dean A Bailes
// @description Scroll calendar time by default to the required time
// @include https://*calendar.live.com*
// @version 1.0
// ==/UserScript==

function scrollToTime() {
var domParent = document.getElementsByClassName('timeTagContainer')[0];
if (typeof domParent == 'undefined') return false;

var domScrollTime = domParent.getElementsByClassName('timeTagItem')[8]; // Change 8 to whichever hour you require (in 24 hour)
var domDayView = document.getElementsByClassName('dayViewContainer')[0];
domDayView.scrollTop = domScrollTime.offsetTop;

return true;
}

window.addEventListener("load",scrollToTime,false);
