How do I create a loop to display worksheets like a slideshow? - Microsoft Community

nitro_450 say:

How do I create a loop to display worksheets like a slideshow?

I have a workbook with 5 worksheets that I want to display in succession, for 5 minutes each, then loop back to the first worksheet and start again. Loop would be broken by hitting esc, for example.
mike h.. say:

Hi,

 

Alt +F11 to open VB editor, right click 'ThisWorkbook' and insert module and paste the code below in. Close VB editor.

 

Run the code. To stop it hold doen the CTRL key and tap the 'Break' key. Note that while the code is running you will only be able to view the sheets.

 

 

Sub Loop_Sheets()
Dim NumSheets As Long
NumSheets = Worksheets.Count
Do
Application.Wait Now + TimeValue("00:05:00")
If ActiveSheet.Index = NumSheets Then
    Sheets(1).Activate
Else
    Sheets(ActiveSheet.Index + 1).Activate
End If
Loop
End Sub
