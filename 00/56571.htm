AutoSave and AutoRecover are some of the recovery features available on selected Office application. At the moment, these features are not applicable in WordPad thus, any unsaved changes will be lost if the app crashed or closed unexpectedly. Alternatively, we can check and restore the file if a previous version of the it is available prior the shutdown. To do this, follow the steps below:

  1. Locate and right-click on the folder where the WordPad document was saved.
  2. Select Properties.
  3. Select the Previous Versions tab.
  4. Under File versions, select the most recently modified file, then click the Open button.
  5. Now check if the changes made prior the shutdown appears. If yes, you can save a copy of the file.
