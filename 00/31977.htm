Replacing Vista in your computer with Windows 7 should not be done using the process known as "Upgrade".   Experts will confirm that the only way that they would ever install a new operating system is by using what we call the Clean install process.  It is referred to as "Custom" in the actual Windows 7 install program.  The Custom install ERASES EVERYTHING, so you must have backed up your data first.
