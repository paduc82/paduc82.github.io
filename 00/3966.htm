See if it is related to multi-monitor support anyway.  For example, make sure that the Display mode is Duplicate (or the appropriate mode for the monitor being viewed) using Win-P.  Also, play with the Taskbar Properties Taskbar settings, especially the ones for Multiple displays and Taskbar buttons to see if doing that can change your symptom.

If the Taskbar button that you are looking at is pinned I would unpin it to clarify the symptom description.  E.g. if the button goes away then you would know that it doesn't have anything to do with your open windows.  Then I would consider how the window was being launched.  E.g. if it is being launched from a shortcut make sure that the shortcut does not (somehow) have the ability to make the window (incompletely?) "hidden" this way.

You can also fix it by not using the areas of the passed range, but passing the multiple ranges explicitly:

Fro example, change

Public Function RangeID(rngArg As Range) As Variant
to

Public Function RangeID(rngArg1 As Range, rngArg2 As Range) As Variant

and change your code to not step through areas.

If you don't always have the same number of ranges, make one or more optional, and include code to check for their argument's existence.

Sub InSpec()
'
Sheets(X).Activate
'
i = 2
'
Do While Cells(i, 1) <> ""
    Rows(i & ":" & i + 11).Select
    Selection.Insert Shift:=xlDown
    '
    For j = 0 To 11
        Cells(i + j, 5) = Cells(1, 5 + j)
        Cells(i + j, 6) = Cells(i + 12, 5 + j)

    '
    For j = 0 To 3
        Cells(i, 1 + j) = Cells(i + 12, 1 + j)
    Next j

    Rows(i + 12 & ":" & i + 12).Select
    Selection.Delete Shift:=xlUp

LB_Series and TransperancyInput are form field inputs.

andy pope say:

I came it with this code for a similar problem on another forum

http://www.mrexcel.com/forum/showthread.php?t=464942

Sub x()
 Dim cht As Chart
 Dim Ser As Series
 Dim lngIndex As Long
 Dim lngChartType As XlChartType

 Set cht = ActiveSheet.ChartObjects(1).Chart
 Set Ser = cht.SeriesCollection(1)
 lngChartType = Ser.ChartType
 Ser.ChartType = xlColumnClustered
 For lngIndex = 1 To 2
  With Ser.Format.Fill
   .Solid
   .Visible = True
   .ForeColor.RGB = RGB(255, 0, 0)
   .Transparency = 0.5
  End With
  Ser.ChartType = lngChartType
 Next
End Sub
