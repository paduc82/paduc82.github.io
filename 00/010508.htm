I have looked at your file, unfortunately I have to say that this file is near death.
And there is no simple way to fix the issue.

There are 2 problems and the good news is both are inside the sheet "Invest Details":

a) The sheet contains 68214 conditional formatting's. That is really huge!
b) You use a lot of OFFSET formulas e.g. in here:
B6: =A6<>OFFSET(A6,1,0)
C6: =MONTH(A6)<>MONTH(OFFSET(A6,1,0))

OFFSET is a volatile function, means it is recalculated whenever a value changes everywhere. I can understand you, it's easier to use OFFSET if you want to insert rows and not to fix the cell references manually every time, but it slows down your file.

Okay, the first step to repair the file:

Make a copy of the file
Open the file
Create a new sheet
Go into sheet "Invest Details"
Select cell A1
Press the CTRL and the SHIFT key and hold it down
Press the END key (This selects all used cells in the sheet)
Release the CTRL and the SHIFT key
Press CTRL-C
Go into the new sheet
Right-Click cell A1
And paste formulas (only) !!!

Now re-create the cell formats step by step manually.
DO NOT USE THE FORMAT PAINTER TO GET THE FORMATS FROM "Invest Details".
Sorry for shouting, but if you do so, you get all the conditional formatting back and at the end your work was senseless.
Do not work out all the details, just enough that you have an overview.

Now go into your other sheets and change the charts (and maybe formulas) so they get the data from the new sheet.
Finally delete the sheet "Invest Details".
Save the file.
Close the file.

If you compare the size of the old and the new file you'll see the size is reduces around 40%, all that was just formatting's.
