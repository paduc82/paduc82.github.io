Hi tjsxox,

 

File restore in Windows 7 is not designed to restore Programs. it restores only user data and 'program settings'. So you need to reinstall the programs and those programs will have the old settings you had before (as 'program settings' are restored from backup)

 

 

Also, if you are using an email client like outlook or windows live mail install the application and you should be able to find the emails in the application itself.

 

In case files under C:\Program Files etc. are chosen for restoring, we block restoring files from backup to such locations as it might cause the applications to not function. If this is what you would like to do, we allow restoring these files to an alternate location and then moving them to the location you would like.

 

In case you need to restore applications, currently only taking a system-image backup & system-image restore from that backup is the supported solution.

 

You don’t have to do a format installation again.

 

If you’re receiving access denied error message, then I would suggest you to take the ownership of the files.

 

Steps on how to take ownership and set permissions to files and folders

a.    Locate the file or folder on which you want to take ownership in windows explorer

b.    Right click on file or folder and select “Properties” from Context Menu

c.    Click on Security tab

d.    Click on “Advance”

e.    Now click on Owner tab in Advance Security Settings for User windows

f.     Click on Edit Button and select user from given Change Owner to list if user or group is not in given list. Then click on “other users or groups”.

h.    Now select User/group and click apply and ok. (Check “Replace owner on subcontainers and objects” if you have files and folder within selected folder)

i.     Click ok when Windows Security Prompt is displayed

j.     Now Owner name must have changed.

k.    Now click Ok to exist from Properties windows

 

Once you have taken the ownership of file or folder next part comes is Granting Permissions to that file/folder or object.

 

l.     Locate the file or folder on which you want to take ownership in windows explorer

m.  Right click on file or folder and select “Properties” from Context Menu

n.    Click on Edit button in Properties windows Click ok to confirm UAC elevation request.

o.    Select user/group from permission windows or click add to add other user or group.

q.    Click Ok for changes to take effect and click ok final ok to exit from Properties window.
