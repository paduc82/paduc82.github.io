gary's student say:

The following event macro will take the value that is placed in I8 thru the DV pull-down.  It will place this value in N8 as the default value.  The user still has the option to over-ride the N8 value if they wish:

 

Private Sub Worksheet_Change(ByVal Target As Range)
Dim I8 As Range, N8 As Range
Set I8 = Range("I8")
Set N8 = Range("N8")
If Intersect(Target, I8) Is Nothing Then Exit Sub
Application.EnableEvents = False
N8 = I8
Application.EnableEvents = True
End Sub

Because it is worksheet code, it is very easy to install and automatic to use:

   1. right-click the tab name near the bottom of the Excel window
   2. select View Code - this brings up a VBE window
   3. paste the stuff in and close the VBE window

If you have any concerns, first try it on a trial worksheet.

If you save the workbook, the macro will be saved with it.


To remove the macro:

   1. bring up the VBE windows as above
   2. clear the code out
   3. close the VBE window

I don't have the same problem with a simple example using a ListBox.  If you can't get your code to work, here is an alternative you could try, which stores all the selected indices in an array and then uses the array to do the rest of the processing.  The interesting thing (to me) is that I don't see where you ever use the selected index that you store in cell C3 in the rest of your code...

Hope this helps,

Eric

Sub Cycle_Sequences()

    Dim i as Long

    Dim nSel as Long

    Dim theArr() as Long

    nSel = 0

    With Sheets("Data Storage").ListBox_Sequences

        For i = 1 To .ListCount

            If (.Selected(i - 1)) Then

                ReDim Preserve theArr(1 to i) ' Store the selected items from the ListBox

                nSel = nSel + 1

                theArr(i) = i

            End If

        Next i

    End With

    For i = 1 To nSel ' Go through each of the selected items

        Sheets("Data Storage").Range("C3").Value = theArr(i)

        Call Iteration

    Next i
