In order for search feature to work, the data file type should be *.pst (Personal Data File) or *.ost (Offline Data File).
iCloud integration with Outlook does not create neither PST or OST file format, due to which Outlook can index the file format for the search feature to work with the iCloud Contacts and Calendar.

 

If you synchronize the Contacts and Calendars through iTunes, it will sync with the default data file. This will allow you to use the search function, however all the Outlook contacts and iPhone contacts will be merged.

 

You may contact Apple support and check if there is an alternate way of synchronization.

 

If you have issues with Office suite and Office related products, reply and we will be happy to assist you.
