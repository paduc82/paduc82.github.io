a. user say:

Getting into the BIOS depends on your motherboard...

When you power up or restart your system, you will usually see on the screen a black background with white text as the motherboard goes through the Power On Self Test (aka POST) procedure.

If you look on the screen, sometimes near the bottom, it usually tells you what key to press to get into the BIOS or System Configuration. 

Looking on the Internet (where everything you read is true), I think on yours, it will be the 'Del' key and that is not the same as the 'Delete' key.  The 'Del' key is in the numeric key section.

There may be a key to press to get into a boot menu.  On my system, that is the F11 key so if I press F11 during the POST, the BIOS will offer me a list of devices and I can choose what I want to boot on and one of those choices is the CD drive.

If you miss it, you need to wait for things to settle down and try again.

Using the Hiren's boot CD mini XP mode, you can see a desktop that you are familiar with (My Computer, etc.) and you can open your C drive, Properties, Tools and run the disk Error-Checking from there (choosing both correction boxes) and wait a while for it to finish.  You can also easily edit/fix your potentially afflicted c:\boot.ini file.

If you need help with your boot.ini file, here is an example of a general purpose one:

[boot loader]
timeout=5
default=multi(0)disk(0)rdisk(0)partition(1)\WINDOWS
[operating systems]
multi(0)disk(0)rdisk(0)partition(1)\WINDOWS="Microsoft Windows XP Professional" /noexecute=optin  /fastdetect

how to install windows 8 64 bit on the new iMac(without cd - Microsoft Community

alasdairgordon say:

how to install windows 8 64 bit on the new iMac(without cd drive) without have windows prior on the boot camp partition

I have just purchased the new 2012 iMac(without the CD drive) and I want to install windows 8 64bit on my bootcamp. How can I do this without spending 80$ on the external drive and which version of windows do I need because I do not have a windows OS on bootcamp prior.
andre da costa say:

First thing:

Go to a working computer running a 64 bit version of Windows (whether that is Windows Vista, Windows 7 or Windows 8), create a .ISO file and copy it to a thumb drive then copy it to your Mac.

https://techingiteasy.wordpress.com/2013/02/14/how-to-obtain-64-bit-installer-and-create-install-media-for-windows-8/

 

Second thing you need to do is boot into OS X, launch Boot Camp (Finder\Application\Utilities) use the Boot Camp wizard to download the Windows drivers. Please note, you should upgrade to OS 10.8.3 in order to get native Windows 8 drivers.

 

This will allow you to use the Boot Camp wizard to create a bootable copy of Windows 8 with the Boot Camp drivers on your thumb drive.

 

I honestly would not recommend you delete OS X, but if you must, at a minimum, create a Time Machine backup.

 

After you have used the Boot Camp wizard to create a bootable copy of Windows 8 with drivers on a USB thumb drive. boot from the thumb drive:

 

Boot from the media you created to install on another partition

: http://windows.microsoft.com/en-US/windows-8/boot-install-partition-media-usb-dvd

 

Click Install Now

Accept License Agreement

When the option is displayed to select an installation type, click (Custom Advanced)

Click Drive Options

Select the drive/s click Delete

Click New

Click Apply

Click OK

Click Format and click Next to proceed with the installation
