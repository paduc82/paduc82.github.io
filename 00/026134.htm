Create a new standard module by selecting Insert | Module in the Visual Basic Editor.

Copy the following code into it:

 

Public Declare Function ShellExecute Lib "shell32.dll" Alias "ShellExecuteA" ( _
    ByVal hwnd As Long, _
    ByVal lpOperation As String, _
    ByVal lpFile As String, _
    ByVal lpParameters As String, _
    ByVal lpDirectory As String, _
    ByVal nShowCmd As Long) As Long

Public Const SW_SHOWMAXIMIZED As Long = 3
Public Const SW_SHOWNORMAL As Long = 1

 

Use it like this in the On Click event of a command button on a form:

 

Private Sub cmdOpenPDF_Click()
    Dim strFile As String
    ' Modify path and filename as needed
    strFile= Me.txtFilename
    If ShellExecute(Application.hWndAccessApp, "Open", _
            strFile, 0&, 0&, SW_SHOWNORMAL) < 33 Then
        MsgBox "Couldn't open PDF file.", vbInformation
    End If
End Sub

ehart811 say:

How to make new pages (copies of empty recipe page) appear in table of contents?

My table of contents only updates for information on the pages that came with the template I downloaded. Obviously I have more than 5 recipes! How can I connect all of my pages to the table of contents like the original pages are?
teylyn say:

Hello,

the table of contents is not designed to update automatically. You need to create a new recipe entry manually.  I find that these steps work:

1. Copy an empty recipe tab

2. Rename the recipe tab, for example call it "Bangers and Mash". Put the same name into cell E3 and select a category in cell G3.

3. Go to the "Recipe Contents" tab and select the cell of column B in the next empty row below the table. That means if the last recipe is listed in row 9, click cell B9. Edit: typo - click cell B10, not B9.

4. Hold down the Ctrl key and press "d". This will copy the formula from the cell above. (Ctrl-D is the keyboard shortcut for "Copy down").

5. With cell B10 selected, hit the F2 key to edit the formula. In the downloaded workbook, cell B9 has this formula:

='Empty Recipe 5'!RecipeName

Change it to 

='Bangers and Mash'!RecipeName

Now you can see the category filled in automatically. If you fill in the other sections on the "Bangers and Mash" sheet, the contents will update automatically for that sheet.
