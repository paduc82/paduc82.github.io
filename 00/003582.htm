jasles say:

Those rows are not really what is in your workbook. That is the total amount of available rows. These will not affect the file size. The only way it would affect your file size is if you entered a value in the last row.

When you enter values in the workbook, Excel tracks what cells have values, and if you enter something in the last row, it could make the file size larger, but if you are only using the first 1,200 rows, that's what is being saved.

There could be a variety of external factors making the saving and downloading take longer - such as your network, whether your are accessing the file over the internet, etc.
