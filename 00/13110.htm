This is a symptom of sending your messages in Rich Text format even though you believe you're sending them in HTML.  Only another Outlook user can receive Rich Text messages properly.

The global setting for this is on File>Options>Mail, at the very top.  What does the "Compose messages in this format" drop-down say?

david j parker say:

Did you use the Boundary or Space shape from the Resources stencil for your Floor areas?

Both of these shapes have a User.visFMArea row, which stores the shapes area in square inches.

Both shapes have a User.visFMAreaDisplay row, which converts this value to the display units for the page using the formula:

=IF(ThePage!User.visSpaceLabel=0,"",FORMATEX(User.visFMArea/(DropOnPageScale^2),User.visFMAreaPrec,"sq in",User.visSqUnits))

=IF(ThePage!User.visSpaceLabel=0,"",FORMATEX(User.visFMArea,User.visFMAreaPrec,"sq in",User.visSqUnits))

The Space shape also has Prop.VisArea row which displays the string from User.visFMAreaDisplay.

 

Therefore, if you want to export the areas from these shapes, then you need to decide whether you want the base square inches or a converted value, or both.
