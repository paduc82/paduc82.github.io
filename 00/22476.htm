Hi

Both IE11 and Edge are built-in to Windows 10.

Here are the basic differences:

  • Edge cannot use add-ons (except the built-in Flash Player) so it will not work with sites that need Silverlight, Java, etc. You also won't be able to install 3rd party toolbars.
  • Edge only has a favorites bar built-in - there is no Menu bar, Command bar, or Status bar.
  • Organising favorites in Edge is a pain at the moment.
  • Edge is, in my experience, quicker than IE11.
  • Edge is, in my experience so far, more stable than IE has been for me in the past. This may be at least in part due to the lack of add-ons which can be incompatible, corrupted, etc.
  • Edge currently has fewer options and settings compared to IE11.
  • Lack of add-ons may help with security. See also: http://blogs.windows.com/msedgedev/2015/05/11/microsoft-edge-building-a-safer-browser/ .
  • Edge will be regularly updated so the days of upgrading from a major IE version to the next will hopefully not be such an issue - see: http://blogs.windows.com/msedgedev/2015/09/21/understanding-versions-in-an-evergreen-browser/

If you use Edge, you can select the 3 dots ... near the top right corner then Open with Internet Explorer if you find a site won't work with it.

Personally, I think that Edge needs more polish particularly with regard to organising favorites and opening tab before I'll use it as my main browser. It does look very promising to me though. The choice is yours.

dianeoforegon say: Get Started Lesson: Learn to Setup Rules

Each type of account has it's own rules. Click on the correct tab to create your rules. POP, IMAP, Exchange, Outgoing


RULE BASICS

There are 3 different kind of rules:

Mailing List Manager (MLM)
Junk Mail Filter
Rules (custom rules you create)

They run in the order listed above. The custom rules also run in the order they are listed. (top runs first)

1) POP Rules act on a message ONLY when it first arrives in the Inbox (On My Computer), NOT when it arrives in any other folder. Once it's moved to another folder it won't be seen by rules.

2) IMAP Rules are meant to act on a message ONLY when it first arrives in the IMAP INBOX, not when moved to another IMAP folder.

3) Exchange Rules act on a message ONLY when it first arrives in the Exchange Inbox.

  8. Provide the date you want.
  9. Click Save.

3.) Uncheck the box "Use User Account Control (UAC) to help protect your computer" and click "OK".
4.) Restart computer then run the installs and they should work.


If you are in Classic View:

1.) Go to Start; Control Panel; Security Center
2.) In the "Windows Security Center" screen, go to the last item "User Account Control" and click the button to turn off.

After the installs have been updated, make sure you go back in and turn the UAC back "on" then restart your computer so that unauthorized updates do not occur.   Always try turning this off if installs don't work.

How do I take a scanned document saved in my documents library and be - Microsoft Community

sandrakmarshall say:

How do I take a scanned document saved in my documents library and be able to type onto the document?

I've scanned an application and saved in my documents library and now I'm unable to access or open it to enable to fill it out.  I really need help today as I need a job yesterday.
suzanne s. barnhill say:

Presumably you scanned the application as a picture, so these are your options:

  1. Don't try to fill it out in Word. If it was given to you as hard copy, the intention was probably for you to fill it out by hand or with a typewriter. Some employers judge your handwriting as well as the content of your entries.
  2. You can convert the scanned picture to editable text using an OCR application. Some such was probably supplied with your scanner, but you may need to rescan the document, specifying that it be scanned as text, which will perform the OCR before creating the file. The results, however, are unlikely to be usable, since forms are notoriously difficult to OCR accurately.
  3. You can insert the picture in your document (cropping the white space around the edge to make it fit properly within your margins) and then add text boxes (No Line, No Fill) where you need to add text.
  4. You can insert the picture as a background to a Word document and then type on it. The best approach is to open the header and insert the picture; format it as Behind Text and resize it to fill the page. Then use "Click and type" to insert text in the desired locations. This last option is the most likely to be satisfactory if you don't use option 1.
