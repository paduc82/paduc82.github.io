    Because the computer has already had Windows 10 installed from the upgrade program it has a Digital Entitlement License issued. This allows you to upgrade or install Windows 10 throughout the life of the computer. For Microsoft Licensing puposes the motherboard is considered as the "computer". (you cannot do a "triggers broom" on it)

    You do not have to do an upgrade as a clean install will call on that DEL. Download the Windows 10 media from here:

    https://support.microsoft.com/en-us/help/15088/windows-create-installation-media

    The easiest option is to use the USB option to create a bootable key. When the HP logo shows press the appropriate key for the one time boot menu (I think it is F11) and select the key which needs to be in the computer when switched on.

    Run through setup clearing partitions (the recovery partition probably wont work anyway and you don't need it with this clean boot option available) and create a new one. Windows setup will create extra ones but they will be small.

    When it gets to the point of asking for a key use the "I don't have a key" option.

    Once setup completes it will pick up the DEL.
