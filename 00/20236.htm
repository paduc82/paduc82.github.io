myall_blues say:

VLOOKUP doesn't work 'backwards' - it can only look up a value in the first column and return values in subsequent columns.

Try =INDEX(A2:A100,MATCH(MAX(E2:E100),E2:E100,0))

You will need to put a bounding limit on the bottom of the ranges  i.e. E2:E100, not E2:E

david a matson say: Greetings papillon70!  Welcome to the Windows Mobile Total Access forums!

There are two general problems associated to CEUTIL.DLL, and those are:

1.) There are too many copies of this file on your device.  Just the presence of duplicates can throw some programs into turmoil.  Best to keep one in the Windows directory and rename (not delete) the others that you find on your device.   If the program you are using still complains, try leaving the one *it* may install alone and rename all the others and see what happens.
2.) The only copy you have on your system is corrupted.   If this is the case, your only hope is to perform a Hard Reset to restore the original file.   Then hopefully a subsequent installation of your 3rd party software will work.
