  1. Go to your inbox.
  2. Click on the gear icon at the top right.
  3. Choose "More mail settings".
  4. Click "Email forwarding".
  5. Check the settings.

Some folders under the "C:\" root folder could move - Microsoft Community

trybird say:

Some folders under the "C:\" root folder could move them into the "C:\Windows" system root folder or other place

Some folders under the "C:\" root folder, like "C:\Config.Msi", "C:\EFI", "C:\MSOCache", "C:\PerfLogs", "C:\Recovery", "C:\RecoveryImage", could move them into the "C:\Windows" system root folder or other place.

If C:\Config.Msi folder already exists, and I don't like its situation which is putted under the C:\ root folder directly, then I can move it into the C:\Windows folder by the following commands:
robocopy "C:\Config.Msi" "C:\Windows\Config.Msi" /E /COPYALL /XJ
takeown /F "C:\Config.Msi" /R /A /D Y
icacls "C:\Config.Msi" /grant Administrators:F /T /C
rd /s /q "C:\Config.Msi"
mklink /J "C:\Config.Msi" "C:\Windows\Config.Msi"


Or, if C:\Config.Msi folder does not exist, and I hope it hereafter won't be putted under the "C:\" root folder directly, then I can preset its folder into the C:\Windows folder by the following commands:
md C:\Windows\Config.Msi
mklink /J "C:\Config.Msi" "C:\Windows\Config.Msi"

If C:\EFI folder already exists, and I don't like its situation which is putted under the C:\ root folder directly, then I can move it into the C:\Windows folder by the following commands:
robocopy "C:\EFI" "C:\Windows\EFI" /E /COPYALL /XJ
takeown /F "C:\EFI" /R /A /D Y
icacls "C:\EFI" /grant Administrators:F /T /C

If I installed Microsoft Office products in my Windows system, certainly I can move the C:\MSOCache folder into the system root folder, but first I must replace all the "C:\MSOCache" strings with the "C:\Windows\MSOCache" strings in the registry, more details about this please refer to my previous thread:
