If I remember correctly from other posts on similar topics, Acrobat for Mac doesn't include the Adobe PDFMaker add-in that's automatically installed with Windows Acrobat versions.

When you print to the Adobe PDF driver or use the Mac's built-in PDF feature, it does just that:  prints.  Just as it would to any other printer. 

The PDFMaker add-in that's part of Windows Acrobat does a bunch of extra work before creating the PDF, including some bits that preserve hyperlinks in the PDF.  So does the built-in Save As PDF in Windows PPT 2007 and onward.

If you have access to a Windows computer with Office, you could try saving as PDF from there.  Otherwise, I'm pretty sure your only option is adding the hyperlinks manually in Acrobat after creating the PDF.
