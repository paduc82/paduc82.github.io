a. user say:

The NTFS file system for windows 7 & 10 are the same.
This issue you have is that by default windows 10 turns on - "fast startup" in power settings.
This means the operating system does not fully shutdown and makes the PC boot faster but affects other drives

causing the drive errors.

You need to turn off "fast Startup" and the other hard drives will be ok.
Right-click start, Power options
Click - Require a password on wakeup
Click - Change settings that are currently unavailable
Untick - [ ] Turn on fast startup
Click - [Save changes]

Sub AddSheetsTest()
    Dim ws As Worksheet
    Dim lngNextSht As Long
    Dim i As Long
    Dim j As Long
   
    For i = 1 To Worksheets.Count
        Worksheets(i).Name = "ExistSht" & i
    Next i
    lngNextSht = Worksheets.Count + 1
   
    For i = lngNextSht To lngNextSht + 3
        Set ws = Sheets.Add(After:=Sheets(Sheets.Count))
        ws.Name = "MyNewSht" & i
    Next i
   
    For Each ws In Worksheets
        MsgBox "Sheet tab name = " & ws.Name & vbCrLf _
                & "Sheet Code Name = " & ws.CodeName
    Next ws
    Sheets(1).Select
   
End Sub
