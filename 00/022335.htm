In the query below the ID is the autonumber field of your table to break ties --

SELECT Person.ID, Person.PRODUCT, Person.MARKS, (SELECT Count(*) FROM Person AS [XX] WHERE [XX].PRODUCT = Person.PRODUCT AND  [xx].id <=  Person.ID) AS Rank2

marshall barton say:

To automatically create the incrementing number for new records, use the form's BeforeUpdate event with a line of code like:

   Me.[your table's counter field] = Nz(DMax("[your table's counter field]", "table name", "Person = """ & Me.person & """ "), 0) + 1

If you have a lot of existing records, you can use an Update query to put the numbers in the existing records:

UPDATE [table name]
SET [your table's counter field] = DCount("*", "table name", "person = """ & person & """ And category <= """ & category & """ ")
