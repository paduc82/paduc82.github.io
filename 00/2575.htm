 You are doing the right things. Excel should prompt you when closing Excel to verify that you want to save changes to personal.xlsb. If not, I would start trying to identify why not.

 

Another possibility is that you are working in a virtual machine environment (Citrix, etc) where the environment is actually locked down for security- you can do lots of things (install add-ins, write macros, etc.) but every time you access the Citrix desktop server from scratch, you get a 'clean' instance of the environment. In those situations, some directory pointers in the citrix machine may be pointed to your local fileshare or local machine (such as where to save program files) but that environment might not have 'local' versions of things like personal.xlsb... in which case, every time you (or anyone) loads the citrix client desktop, you get the fresh/plain version of all system files including personal.xlsb.

 

To determine if this is the problem (regardless of whether it is Citrix or a similar product), when you were upgraded to 2007 and then 2010, did they come and physically install the new versions on your PC? macropod say:

Try something based on:

Sub FormatImage()
With Selection.ShapeRange(1)
  .WrapFormat.Type = wdWrapSquare
  With .Line
    .Visible = True
    .Style = msoLineSingle
    .ForeColor.RGB = RGB(0, 0, 0)
    .Weight = 2.25
  End With
  With .Shadow
    .Style = msoShadowStyleOuterShadow
    .Size = 100
    .Type = msoShadow4
    .Transparency = 0.5
    .ForeColor.RGB = RGB(127, 127, 127)
  End With
End With
End Sub

The weight represents the thickness in points. RGB(0, 0, 0) defines the colour (in this case, black) - RGB(127, 127, 127) is a middle grey.

You will need to experiment with the various parameters to get the result you want. There are 20 shadow types, for example.
