doug robbins - mvp office apps & services (word) say:

The error message indicates you probably have an older version of the Avery Dennison Label Wizard add-in for Word that is not compatible with Word 2010. The solution is to uninstall the add-in.

If the error message continues to appear after you uninstall the Avery wizard, you have to manually delete the template that contains the faulty macro. Shut down Word and start up the File Explorer. In the address bar of the Explorer, type or paste

%appdata%\Microsoft\Word\Startup

and press Enter. Select the file AVWiz12s.dotm (the number may be different, depending on the version of the wizard) and delete it.
