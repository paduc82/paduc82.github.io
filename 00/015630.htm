If you're using Excel 2003 then you can't use entire columns as range references in some functions in array formulas.

You'll have to use specific range addresses in these functions:

IF('All Staff (data)'!H:H

ROW('All Staff (data)'!E:E

I didn't read the whole thread but it looks like you want to return TEXT values.

Here's the array formula** (without the sheet name, you can add it yourself) with an error trap for TEXT.

=LOOKUP("zzzzz",CHOOSE({1,2},"",INDEX(E:E,SMALL(IF(H$2:H$10=$B$2,ROW(E$2:E$10)),ROWS(A$1:A1)))))

** array formulas need to be entered using the key combination of CTRL,SHIFT,ENTER (not just ENTER). Hold down both the CTRL key and the SHIFT key then hit ENTER.

Moving a folder "up" the tree is a bit tricky... since the pre-Anniversary Update version of Edge uses a "bread crumb" view rather than a full "tree view".  So, to move something "up", you drop the item on the word "FAVORITES" (not an icon) at the top of the favorites pane.

Yeah, the Anniversary Update does make moving between folders a bit easier...
