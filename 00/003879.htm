Suggestion

Start Outlook in safe mode and disable add-ins

Sometimes add-ins can conflict with Outlook. Starting Outlook in safe mode, which starts Outlook without add-ins loaded, can tell you if an add-in is the problem.

  1. Start Outlook in safe mode    
    1. Choose Start > Run.
    2. Type Outlook /safe, and choose OK.
    3. In the Choose Profiles dialog box, accept the default setting of Outlook and choose OK.
    4. If prompted, enter your password and choose Accept.
  2. Disable add-ins
    1. In the upper-left corner of Outlook, choose File > Options > Add-ins.
    2. At the bottom of the View and manage Office Add-ins, make sure the Manage box shows COM Add-ins, and then choose Go.
    3. As a precaution, when the COM Add-ins listing of your current add-ins opens, do one of the following:
    4. After you've captured the titles of the selected add-ins, clear all selected check boxes, and then choose OK.
    5. Choose File > Exit.
  3. Restart
    1. Choose Start > Run, and in the Open box, type Outlook.
    2. In the upper-left corner of Outlook, choose File > Options > Add-ins.
    3. Select the check box next to an add-in you want to re-enable, and then choose OK.
    4. Repeat all steps until you've re-enabled all of the original add-ins and revealed the source of the error.

Time Card Formulas

Hi! This is what I am trying to do.

joeu2004 say: Susy wrote:

C4 6:48 Time In

C7 17:00 Time Out

E7 10.12 Total Hours For the day

There is no cell with a formula for 30 minute lunch. ( I put it into E5 to try your formula)

Expected results with lunch and rounded should be 9.75 ( but with the formula is comes out as -10.25)

=ROUND((C4-C7+(C4>C7)-TIME(0,30,0)*(E5="y"))*96,0)/4


You are repeatedly subtracting TimeOut (C7) from TimeIn (C4).  It should be the other way around, as in all of my examples.  The formula should be:

=ROUND((C7-C4+(C4>C7)-TIME(0,30,0)*(E5="y"))*96,0)/4

You did write (C4>C7) correctly.  It effectively adds 24 hours if TimeIn > TimeOut, written as 24-hour time.  For example, if TimeIn is 17:00 and TimeOut is 1:00 (8-hour shift from 5pm to 1am), C7-C4 would be -16:00.  By adding 24:00, we get 8:00, which is the correct elapsed time.

However, I suspect the corrected formula above will give you 10.25 instead of 9.75.  That is because you are putting ":30" into E5 instead of "y", which I specified in my original example.

I did suggest, alternatively, that you can put the time 0:30 into E5, not the text ":30".  And that can be the actual time for lunch, e.g. 0:45 or 1:00.  If you put the time in E5 instead of "y", you can simplify the formula as follows:

=ROUND((C7-C4+(C4>C7)-E5)*96,0)/4
