suzanne s. barnhill say:

In the Define New Multilevel List dialog, when you look at the numbers in the "Enter formatting for number" box for Level 3, do all three numbers have gray shading behind them? Probably not. This means they've been entered as static text rather than number fields.

If that's the case, the easiest way to correct the problem is to start over.

  1. With the insertion point in the first Level 1 paragraph in the document, click the arrow beside Multilevel List and choose Define New Multilevel List.
  2. Select Level 3 and delete all the numbers in the "Enter formatting for number" box (clear the box).
  3. Click the down arrow for "Include level number from" and select Level 1. That will insert a 1 with gray shading.
  4. Type a period after this shaded number.
  5. Use "Include level number from" again to insert Level 2 and type a period after it.
  6. In the "Number style for this level" box, select 1, 2, 3.
  7. Click OK to close.

If any paragraphs are still numbered incorrectly, press Ctrl+Q to sort them out.
