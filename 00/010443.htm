suzanne s. barnhill say:

Doug and Charles have both given you good advice, but the reason you're experiencing the problem is that you are inserting Header and Page Number building blocks, and either will replace the other. There are Header building blocks that include page numbers or positions for page numbers or other text, but you don't need to use a building block at all; you can type directly in the header after accessing the header pane by double-clicking in the header area.

When you access the header area directly, the style applied is the Header style, which by default includes a center-aligned tab stop centered between the margins and a right-aligned tab stop at the right margin. These are alignment tabs, so they will adjust automatically if you change the margins.
