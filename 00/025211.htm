Private Sub Worksheet_Change(ByVal Target As Range)

zorvek say:
You can do this without VBA. To accumulate or iterate a value on a worksheet without VBA set the iteration setting on and the number of iterations to 1 (Calculation or Formula options). This allows a static accumulator can be created. For example, to create a summing calculator that sums all values entered into cell A1, enter the following formula into cell A2:

   =IF(OR(ISNUMBER(A1),A1<>0),A1+A2,0)

Choose the menu command File->Options, navigate to the Formulas section, set Iteration on and enter 1 in the text box "Maximum iterations". Any value entered into cell A1 will be added to the value in A2. A2 will continue accumulating values entered into A1 until A1 is cleared or set to a non-numeric value.

To maintain the maximum value the formula in B1 would be:

    =MAX(A1,B1)
