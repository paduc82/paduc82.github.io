I think it is best to use the Sum function in a text box (named txtTotal) in the subform's Header or Footer section:.

   =Sum(IIf(Medication Like "*EPO*",Dose, 0)

to make sure the Sum is only over the records in the form's possibly filtered recordset.  The text box and/or the form header/footer can be invisible.

 

Then, to display that total in the main form text box,Text45, set its exprssion to something like:

   =[subform control name].Form.txtTotal
