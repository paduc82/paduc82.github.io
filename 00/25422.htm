felix tao msft say:

Hi Duke,

 

To troubleshoot this issue, you may try the following steps:

1.Check if the issue is related to the specific page, you can create a new page and check the outcome. To do that:

In the SharePoint Online site, click the gear on the upper right corner>Select Add a page to create a new page>In the new page, click App Part under INSERT>Choose the Calendar list and click Add>Click Save to check the outcome.


If the events display in the new page, you can set the new page as the Homepage. To do it:

In the new page, click PAGE tab>Click Make Homepage to set it as the homepage>Click Edit Properties to change the page name.

2.To see if the issue is related to the network environment, please try another network environment(out of your organization) to check the outcome.
3.You may open the site in another browser or start in private browsing to check whether the issue is related to the browser.
