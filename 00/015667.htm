Merging several worksheets from different excel files into one excel - Microsoft Community

riafirm say:

Merging several worksheets from different excel files into one excel file

How do I create a new file taking several worksheets from different files and merging/consolidating them into one file.  This new file would have all these worksheets as separate tabs (worksheets).  I want to do this monthly so it would be nice to set up some kind of macro or automated function so I don't have to recreate it each time.
hansv mvp say:

The following macro assumes that you have placed the workbooks to be merged in a folder that doesn't contain any other workbooks. The code will prompt to select the folder.

Sub MergeWorkbooks()
  Dim wbkCur As Workbook
  Dim wbkAdd As Workbook
  Dim strPath As String
  Dim strFile As String
  Set wbkCur = ActiveWorkbook
  With Application.FileDialog(msoFileDialogFolderPicker)
    If .Show Then
      strPath = .SelectedItems(1)
    Else
      MsgBox "You didn't select a folder!", vbExclamation
      Exit Sub
    End If
  End With
  Application.ScreenUpdating = False
  If Right(strPath, 1) <> "\" Then
    strPath = strPath & "\"
  End If
  strFile = Dir(strPath & "*.xls*")
  Do While strFile <> ""
    Set wbkAdd = Workbooks.Open(strPath & strFile)
    wbkAdd.Worksheets.Copy After:=wbkCur.Worksheets(wbkCur.Worksheets.Count)
    wbkAdd.Close SaveChanges:=False
    strFile = Dir
  Loop
  Application.ScreenUpdating = True
End Sub

What I would do is add a parameters table. A parameters table is a ONE record table that contains configuration info about the file. Another method is is to use an external INI file.

Dim strFilename As String

To add to John's EXCELLENT advice.

The proper method to copy a file to a USB Thumb Drive or USB Stick is:

  1. Save file to Computer First.
  2. Then insert USB stick and wait until it completely mounted. 
    (many better sticks have an activity light that flashes on and of when its reading/writing.
    If it has such wait until activity light quits).
  3. Locate the File and and click on the option key ( âŒ¥) and and Drag the icon of the file to the thumb drive/USB stick.
  4. Wait until activity light either goes out completely of or remains lit constantly (no Flashing different brands set up activity light differently.)
  5. Now Dismount the drive/stick and again wait for no activity showing in activity light.
  6. Remove the Drive/stick.

If the device is removed while any activity is going on, then you will corrupt any files that are being written or read. Or may corrupt the entire Stick.
