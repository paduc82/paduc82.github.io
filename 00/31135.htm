andre da costa say:

Depends, if the Windows Vista license came preloaded with the computer, its not transferable to another machine.

If its a full version retail Windows Vista license, and the Windows 7 is an upgrade license, the Windows Vista license is still not transferable because Windows 7 is using it as a qualifying base license.

If its a full version retail Windows Vista and Windows 7 license then the Vista license is transferable.

Here is how you copy CD\DVD and burn them: (the following steps will be helpful if your CD\DVD is capable of burning CD\DVD)

a. Right-click on CD\DVD drive icon and select Explore.

b. Select all the files and right-click on them. Now select Copy.

c. Now go to the location where you want to paste it. Right-click on the free space and select Paste.

d. Now insert a brand new DVD.

e. Now open the DVD in Windows explorer.

f.  Copy all the files which you want to burn and right-click on free space on DVD Windows and select Paste.

g. You must click on the "Write these files to CD\DVD" link located in the upper left corner of the Window.

h. The CD\DVD Writing Wizard will appear and ask you to name the CD\DVD. Insert a name or leave the default and then click on Next.

i. Wait until the burning process is finished.
