joe's answer say: How to synchronize OneNote and SkyDrive

   1. On your PC, open OneNote 2010.
   2. Go to the Notebook you want to enable sync, click on File > Share
   3. Choose your Notebook and then click on the Sign In button and enter your Live ID(aka MSN/Hotmail/XBox ID)
   4. Once you sign in, click on the Share Notebook button in the lower right. It will upload your notebook to your Skydrive
   5. Now notebooks will be automatically synced among different OneNote clients if they are online.

brian tillman [outlook mvp 2007-2019] say:

Click File>Account Settings>Account Settings.  Select the Data FIles tab.  Which of the data files is marked as the default?  That data file is, by definition, the data file containing your default folders.  That's the data file containing the main Calendar folder and where you should be storing events about which you want Outlook to inform you.  If you want to copy the contents of a folder named "Calendar" in another data file to the default Calendar folder, open the source Calendar folder, change the view to a table view like "List", select all the items in the Calendar folder by pressing Ctrl+A, then hold Ctrl as you click on and drag the selection to the default Calendar.  When you release the mouse button, the items will be copied to your default Calendar folder.

If you want the events in a separate folder, what I'd do is create the new folder with a new name describing its purpose, making sure you select "Calendar Items" ad the type of item it contains, then populate it with the contents of the course Calendar folder as I've already described.

You can remove a non-default data file on the Data FIles tab you've already visited or you can right-click its root in the Navigation Pane and choose Close.
