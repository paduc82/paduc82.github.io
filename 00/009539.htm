table of contents style change - Microsoft Community

rmosgoode say:

table of contents style change

URGENT please help.

 I have gone to many mac websites for help and even called technical suport but they could not help me. I have created a table of contents and obviously used different heading styles. One style is a default one listed called "intense emphasis" whiich i renamed. I cannot insert this style into my table of contents when I go to "options" from the table of contents menu, then i select a template for my table of contents ("classic") and then  i click options again. it says "available styles" which i can use but none of them are the intense emphasis that i used.. how can i add this??

a. user say:

That's probably because "intense emphasis" is a character style, and the TOC field wants to work with paragraph styles. In that case, you should either re-use an existing paragraph style name (generally preferred) or create a new paragraph style, and apply it to the paragraphs that you want to have in the TOC.

If you are trying to insert the text from part of a paragraph in the TOC, using a character style won't work - instead, you can create TC fields and include them in your TOC via the checkbox in the TOC's options pane.
I think you have to use Insert->Field to insert TC fields in Mac Word (or use cmd-F9 to insert field code braces manually and take it from there).
A TC field looks like this
{ TC "The text you want in the TOC" }
and you can specify some behaviour using switches, e.g. 
{ TC "The text you want in the TOC" \f x \l 3 \n }
The \f x lets you define TC entries for different TOCs, e.g. you can use \f x to define the entries for one TOC and \f y to define the entries for another.
The \l lets you define the outline level that the TOC will use
The \n causes page numbers to be suppressed in the TOC.

Now when a computer starts, it has a whole bunch of things it needs to do during boot and during startup.  It has to start programs, enable devices, start sevices, start the scheduled start-up programs, connect to the networks and lots of other things.  Most of these are contained in MSCONFIG.  Depending on what you have starting, how much is starting, the power of your computer and its devices, how well maintained the system is, how clean the system is (not just in terms of viruses but excessive and unecessary files or defragmentation or how many temp files are on the system or quite a number of different things), and the order things start in (which is often not under your control), things can start slower or faster, and some things can start before other things.  Somewhere in there MSE starts.  If everything else needed for MSE to run hasn't started at that point, it will show red because it isn't yet ready to start.  On some systems this can take less than 30 seconds, and on others it can take a few minutes or more.  If MSE eventually starts after a few minutes, then chances are good that your system has too much running at startup and/or is not adequately powered and/or is so fragmented and filled with unnecessary files that it takes much to long to load them all (or it could be other things too including malware infections).
