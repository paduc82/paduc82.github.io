My first idea to work around this was to have an auxiliary workbook (.xlsx) that is used only for these settings.  The workbook could be opened when the add-in loads and saved and closed when Excel exits.  But it changes the way Excel behaves in a far-too-noticeable way:

Without the add-in With the add-in
1. On startup If Excel is loaded without a file, a new blank workbook is created. Hidden auxiliary workbook is opened, but empty Excel window is visible instead of a new blank workbook.
2. On exiting When the last open workbook is closed with the Excel window "x" button, Excel also exits.  (If it is closed with the document "x" button, Excel remains open with a blank window.) When the last visible workbook is closed (regardless of which "x" button is used) Excel remains open with a blank window.

I think I could work around issue 1 by checking to see if there are other workbooks open when my add-in loads (i.e. did they open Excel using a file name or not) and if not, add a new workbook.

Is there a way to use a helper workbook and have Excel behave the same way as before?

Is there a better way to give the user access to the stored settings, without using an auxiliary workbook?

One idea I have is to use a temporary auxiliary workbook.  In this case, the settings would be stored in the add-in, but would be copied to a new workbook by the user's request, and copied back to the add-in also at the user's request.  When copied back, the temporary workbook would be closed.

andreas killer say:

Worksheets within an add-in can't be made visible.

That is not correct, set the IsAddIn property of the workbook to false.

I have an Add-In with a toggle button in the ribbon that executes the sub below.

Andreas.

Sub OnActionTglButton_639963A2_35E6_4C00_964D_B12AB121430D(control As IRibbonControl, pressed As Boolean)
  Select Case control.ID
    Case "tgbSetup"
      If ThisWorkbook.ReadOnly And ThisWorkbook.IsAddin Then
        MsgBox "Datei ist schreibgeschützt, Änderungen können nicht gespeichert werden." & vbCrLf & vbCrLf & ThisWorkbook.FullName, vbInformation, ThisWorkbook.Name
      End If
      ThisWorkbook.IsAddin = Not ThisWorkbook.IsAddin
      pressed = Not ThisWorkbook.IsAddin
      If Not ThisWorkbook.Saved Then
        If Not ThisWorkbook.ReadOnly And ThisWorkbook.IsAddin Then
          ThisWorkbook.Save
        End If
      End If
  End Select
End Sub
