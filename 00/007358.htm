That windows update option is left over from when Office was installed the right way, locally, using MSI installation files (sorry, I let a little (very little!) of my anti CTR feeling slip out there) you could use Windows update to also automatically update Office, and co-incidentally any other MS applications (Money etc) you may have installed. Windows Update is a "Pull" process. We run Windows update, it PULLS updates from MS then we had the option to decide which ones we would not install, if we didn't want them. Which was good when MS released an update that caused problems.

Office 365 and Office 2016 are installed in the CTR, click-to-run (or not) virtual environment.  CTR installations, exactly like Windows 10 now is, are designed to be provided with updates through a "Push" process. MS releases the Office (and Windows) updates and then automagically PUSHES them out to us (whether we want them or not), whenever MS decides to. Bottom Line

There is nothing you have to do, or can do, to get Office 365 updates now. You get them when MS gives them to you.

Re: "While trying to figure out how to move Outlook's .ost file to where it is more readily backed up"

Why exactly are you backing up .ost files? An OST file is simply a mirror image of what is contained on the server for the email account to which the OST file applies. It can only be used by the same Outlook profile/email account that originally created it. To be even more specific, a new OST file is created any time the email account is configured. A new OST file will also be automatically created if Outlook can't find it when it opens.

If you are using IMAP accounts in OL'2013/'2016 then what you need to be concerned with is any data found in any IMAP folder with the words "This Computer Only" in the folder description (i.e. contacts/calendar). That

Tech Support Scamming through unsolicited phone calls, browser pop-ups and emails (aka fake "alert messages" has become an increasing common scam tactic over the past several years.   • Fake Your computer is locked screenlocker...contact a Microsoft technician

  • Microsoft Help Desk Tech Support Scam

  • Microsoft Office Activation Wizard Tech Support Scam

  • Windows has been blocked due to suspicious activity Tech Support Scam

  • Your Windows has been Banned Scam

  • 1-844-879-7840 Suspicious Activity Tech Support Scam


The scammer may claim to be affiliated with Microsoft or Windows Support. Microsoft does not contact users via web page messages, phone or email and instruct them to call tech support to fix your computer.
