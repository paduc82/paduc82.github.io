How to let a Standard User run a desktop application in Administrator - Microsoft Community

spath21 say:

How to let a Standard User run a desktop application in Administrator mode?

Hi there,


I have installed a Desktop Application on a Windows 8.1 tablet which asks for administrator password to run. Is there a way to let a standard user run this application in administrator mode, without having to fill out an administrator password?


I have tried to change the Security properties on the .exe file to allow full control for all users, but when opening the application it still prompts for a password. Also, in the compatibility properties for the .exe the checkbox "Run this program as an administrator" is not selected.


Please advise.


Thank you for your time.

anannya podder say:

Hi Spath,

 

If you want to allow a standard user account to run an application as administrator without a UAC or password prompt, you can create a shortcut that uses the run as command with the /savecred switch, which saves the password.

 

Step 1:

 

Log -in using an admin account.

 

Step 2:

 

Now we’ll create a new shortcut that launches the application with Administrator privileges.

 

To so refer to these steps:

 

a. Right-click the desktop (or elsewhere), point to New, and select Shortcut.

 

b. Enter a command based on the following one into the box that appears:

runas /user:ComputerName\Administrator /savecred “C:\Path\To\Program.exe“

c. Replace ComputerName with the name of your computer and C:\Path\To\Program.exe with the full path of the program you want to run. For example, if your computer’s name was Laptop and you wanted to run XYZ, you’d enter the following path:

runas /user:Laptop\Administrator /savecred “C:\Program Files\XYZ\XYZ.exe”

 

d. Enter a name for the shortcut.

 

e. To select an icon for your new shortcut, right-click it and select Properties.

 

f. Click the Change Icon button in the Properties window.

 

g. Select an icon for your shortcut.

 

h. The first time you double-click your shortcut, you’ll be prompted to enter the Administrator account’s password.

 

i. This password will be saved – the next time you double-click the shortcut, the application will launch as Administrator without asking you for a password.

 

j. As we mentioned above, the standard user account now has the ability to run any application as Administrator without entering a password.

 

The Administrator password is saved in the Windows Credential Manager – if you want to remove the saved password, you can do it from there.

 

Please get back with the issue status.
