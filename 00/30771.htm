Random freezes are usually caused by one of four things:

1. Faulty Memory - Click on the Start orb and type Windows Memory Diagnostic in the search window.

2. Overheating due to dust buildup or fan failures on the CPU and/or Video card heat sinks .  - Blow out excessive dust inside the PC with compressed air.

3. Defective power supply. Swap with a spare if you have one.

I'm assuming you are not overclocking the CPU wich can also cause these problems.

mariajessika say: I got the same problem and this is what I did on my Win7 PC: 
(Forgive me if my English is not good)

1. Checking my Hosted Network in my driver. I run Command Prompt by Administrator, and type netsh wlan start hostednetwork 
if you get this message:
The hosted network couldn't be started. The group or resource is not in the correct state to perform the requested operation.

Then you have the same problem with me. This is the next step.

2. Checking my driver. Type on command Prompt: netsh wlan show driver
Then I saw that my driver support hosted Network. If your driver doesn't support hosted network, I suggest you to update your driver or reinstall it.

How to fnd your driver and update it: Start - Control Panel - System and Security - Device Manager (under the system) - Network Adapters

3. If your driver support hosted network but still cannot start it, then the problem might be you didn't have Microsoft Virtual WiFi Miniport Adapter (The same problem happened to me). You can search and download it. I've tried and install the new adapter.

Now I have Microsoft Virtual WiFi Miniport Adapter under Network Adapters,in Device Manager.

4. I tried step 1 and it was working! So I can run my Connectify and that error message never showed again.

Hopefully it works for you too.
