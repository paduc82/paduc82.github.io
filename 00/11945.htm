from Google

Too many simultaneous connections

If you have multiple users frequently accessing the same account or more than one IMAP client accessing Gmail at the same time, you may reach a connection threshold and receive the error message, 'Too many simultaneous connections.'

Gmail currently has a limit of 15 simultaneous IMAP connections per account.

Please note that your mail client may actually open multiple connections in the background. This means it's possible to reach a connection threshold simply by using only two mail clients to access the same account at the same time.

If you often use multiple mail clients or mobile devices with the same Gmail account, we recommend closing and/or signing out of any mail clients you're not actively using.

myall_blues say:

Hi

You can't use INDIRECT as charts have to have a fixed address, but you can can do it using named ranges.

Assume your computed entry is in M428, then define named ranges for the x range and y range using Formulas-> Define Name (or Name Manager) as follows:

xrange: =OFFSET(Table!$G$428,0,0,Table!$M$428)

yrange: =OFFSET(Table!$K$428,0,0,Table!$M$428)

Make the scope of the ranges 'Workbook'.

Then edit the series in your graph to include the named ranges rather than specific ranges. Note that you must include the workbook name:

=SERIES("Series 1",Book1!xrange,Book1!yrange,8)

Search "Dynamic chart range" for lots of examples.
