cmdrkeene say:
Hi José,
Your mouse cursor is stored in your user profile, so while rebooting it's normal for it to show the default cursor when it has logged out of your profile and is in the default system account.

All the technical details aside, this usually means the mouse properties in Control Panel is having trouble saving to the registry.

This can be caused by several things, and the recommend solution is to create a new user account on your computer.  However if you are feeling brave, here's how to dig into the registry.

! Important - Before you go further, type 'restore' into your Start menu's search box and choose "Create a restore point", so you have a backup if any registry changes cause problems.  I don't anticipate a problem, and these are simple steps, but a backup is very important to have.  Microsoft will also point out that you should back up your entire system.

Okay, do this:
  1. Press Windows-R to open the Run box, type regedit, and press OK.  Registry Editor will open (you may be asked for an administrator confirmation).


  2. In the folder list to the left (they're actually called keys in the registry), open up this path:
    HKEY_CURRENT_USER\Control Panel\Cursors


  3. In the right, you'll see the cursor file names being used for various cursors.  Edit the Arrow cursor file name to modify the default cursor style.


  4. Reboot to take effect.

Hi paulgibsonTF,

The problem is that the Windows designers are (I think) trying to be too clever -- if your computer does not have an HDMI cable plugged in to it, and you don't have an HDMI device at the other end of the cable (like a TV or surround sound amplifier / speaker system), then, Windows will just ignore that the Windows software driver for the HDMI is loaded.

And even when you DO have an HDMI device plugged in and turned on, sometimes, Windows doesn't see that it's there. So, it won't bother letting you set that device's parameters.

Now... to start looking. Go down on the system tray (that bar across the bottom edge of the screen), and find the Sound icon -- looks like a little speaker. Click on it, and click on Playback Devices. On my computer, I get a SOUND window, and it only shows the one device (Speakers / Headphones).



I don't have an HDMI device plugged in right now. But I do have the device drivers loaded.

Just recently I tried to plug this same laptop in to my HDMI TV, and... of course, Windows won't see that the HDMI devices are there and plugged in. Again.
WHY? I don't know. But what I had to go through two years ago (same laptop) was about four cycles of uninstall the HDMI device driver, reinstall it, turn things off, unplug them, turn them on, plug them in,.... and suddenly it started working.

    ken blake - microsoft mvp say:
    How can i install canoscan N650u in my windows7 laptop to work effectively?


     

    Whenever you change to a new version of Windows, you need to install drivers (software that provides the interface between the hardware and Windows) for printers, scanners, video cards, etc., that have been written for that new version of Windows.  And also note that you need different drivers for a 32-bit version of Windows and a 64-bit version.

    It's up to the manufacturer of the hardware to write and make available for download those drivers. Especially if your hardware is older, it may  well be that the manufacturer has decided that it's not worth his time and effort to develop drivers for hardware that he is no longer selling.

    Check with the manufacturer or on their web site to see if a driver exists. If it doesn't, you're out of luck. Also check their web sites in other countries, if there are any; sometimes you can find the driver you need that way.

    So in the future, before you do such an upgrade, be sure that you've done the appropriate research to find out what drivers don't exist, and therefore what hardware you will have to replace if you upgrade.
