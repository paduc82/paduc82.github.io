You can't move it out of the trash or delete it. Close outlook and restart it using the /resetfolders switch. It will create new 'this computer only' folders. (How to use switches: https://www.slipstick.com/outlook/command-lines...)

It should have synced down to the icloud's calendar if the phone added it to the icloud calendar (and not a different calendar that was set as default).

If you accept the event in outlook, it would be added to the trash calendar - but if accepted on the phone, it should sync to outlook via the icloud. If nothing is syncing up or down, you need to confirm the addin is enabled. https://www.slipstick.com/outlook/icloud-sync-f...

how can i remove duplicates from excel with 2010 starter edition, I - Microsoft Community

c32ttu say:

how can i remove duplicates from excel with 2010 starter edition, I have no data bar showing
teylyn say:
Hello,

you need a multi-step approach. In a helper column, use a formula to identify duplicates, then filter the data on the identified rows and delete these. For example: with your data in column A, use this formula in column B, starting in B2

=IF(COUNTIF($A$1:A2,A2)>1,"duplicate","")

Now sort the data by column B. All the duplicates will sit at the end of the list. Select the rows and delete them.
