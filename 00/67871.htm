pauli o say:

On small home network typical subnet mask is 255.255.255.0 and basically it's same as netmask and mostly both names are used to refer same mask.

On large networks subnet mask can be used to divide large IP-address space for smaller groups (sub networks), but that's not necessary for private home networks.

With subnet (or netmask) 255.255.255.0 three first digit of IP-address represent network address and last digit represent devices on that network.

DNS address tell PC where it should look corresponding IP-address for given network / computer name and on small home networks that is in most cases same as gateway address. This case DNS query is directed to ISP operator network.
