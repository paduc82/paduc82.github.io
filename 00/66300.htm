    Hi,

     

    To begin with, I would like to inform that you may not be able to recover the password/reset the password for the Standard User Accounts. However, I suggest that you try the steps below and check if it helps

    1. If you have another Local Admin Account in your PC or a Microsoft Account, you can use the account and try to reset the password.

    2. Right click on the Start Menu and select Command Prompt (Admin).

    3. Here copy/type the command below and hit Enter the command

    net user username *

    Note: username here is the username of the account that needs a password reset 

    4. You will receive a prompt to enter the new password for the account. Hit enter after entering the first password option. 

    5. Once the command runs successfully, reboot your system and check.

    Hope this helps

    13. Change password.

    joxala say:

    I have still a question about this:

    notice the onscreen warning about this also resetting stored network & internet passwords

    Which kind of passwords does it mean? Microsoft account password? Others?

    tryx3 say:

    This refers to the passwords stored by your browser that log you in to network & internet sites automatically. 

    • It does not reset the passwords themselves, it just affects their use for automatically logging in. 
    • So the user would have to log in to all websites manually at the next visit just as if it was a new computer.
    • The user will then see the message at the bottom of the screen asking, for example, Do you want IE to remember this password. 
    • Then it is back to normal and automatic logins resume.

    The user would also lose access to encrypted files & email messages unless the relevant security key had already been backed up.

    You can see a list of them for the currently logged in user at Control Panel, Credential Manager.  I do not know of any method for viewing the credentials of other user accounts.

    I am not an expert in managing credentials or file/email encryption.  I only learnt enough to know that I should only reset a user account's password from another user account if it was unavoidable.  If you still have a Windows 7 computer, the subject is explained very briefly in the on-board Help section What are the risks of resetting a password?     I have never managed to find an equivalent page for Windows 10.

    Denis
