Hyperlink endnotes vrs regular footnotes/endnotes - Microsoft Community

ubcdbaker say:

Hyperlink endnotes vrs regular footnotes/endnotes

I am working on a manuscript in which, for some reason, the endnotes for the first chapter are all hyperlinks.  In revising that chapter, I tried to add an endnote, but that endnote was treated as totally separate from the other endnotes. It was given the number 1 and placed after all the other endnotes!  I'm using MS Word for Mac 2011. Is there anyway to either get all the endnotes to convert to being regular endnotes, or to make the new endnote also a hyperlink endnote so that it sit among the other endnotes where it belongs?  (When clicked the hyperlinked endnotes read "sdfootnote#sym, with # the number for the endnote). Any help any of you can give me with this would be greatly appreciated. 

john mcghie say:

Sadly, some clever person has produced endnotes that are not endnotes :-)  Those are the hyperlinked ones.

You will need to insert new hyperlinks in their place.

  1. Copy the text of the endnote
  2. Look for the note reference character in the text.
  3. Immediately beside it, insert an endnote.  It will get an automatic number, which will come right when you have done them all.
  4. Paste the copied text into the endnote.
  5. Delete the old endnote reference mark.
  6. Delete the old endnote.
  7. Rinse and repeat for each of the non-endnote endnotes.

The key to this is to understand that a Word "endnote" is an entry in a generated table within Word.  The actual text of the endnote is stored in the endnote reference mark, which is a little container that stores the text, the formatting, the numbering, and the positioning.

People who do things such as manually insert endnotes, or save the document in one of the freeware Word look-alikes, convert the endnotes to ordinary typed text.  When that happens, the whole endnote structure won't work.  When you get a document like that, about the only thing you can do is re-insert the endnotes as real endnotes, as I described.

Hope this helps

windows mobile admin say:

Setting IE Mobile browser favorites

To add Web sites to your Favorites list:

1. From the Web page you want to add, select 'Menu' > 'Add Favorite'.

2. To store the Favorite in a new folder, select 'Add Folder', enter a folder name and click 'Add Favorite'.

3. To store the Favorite in an existing folder, select that folder and click 'Add Favorite'.

4. If you don't want to store the new Favorite in a folder, click 'Add Favorite'.

 

To visit a site stored in your Favorites list:

1. Press 'Start' and select 'Internet Explorer'.

2. Select the Favorite you want and click to open the link.
