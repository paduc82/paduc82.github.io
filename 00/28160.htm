For the syncing process to work properly, we need to make sure that everything is set up correctly. Please follow the steps below:

Add your Outlook.com account to the Microsoft Outlook app:

1. Open the Microsoft Outlook app and select Settings > Add Account > Add Email Account.
2. Enter your email address and select Add Account.
3. Enter your password and select Sign in.
4. If you've enabled two-step verification for your account, approve the sign-in request.
