If you want to create a form where users should only add pieces of text, there is a number of options for that in Word. For example, you can make use of form fields (these are backward compatible with older Word versions) or you can use the newer content controls.

As I already mentioned, it is possible to restrict the use of formatting, so that users/editors can focus on content. Click Review | Restrict Editing; in the task pane, select "Limit formatting to a selection of styles" and then click "Settings" to specify which styles can be used in the document. None of these options would require "locking" Word itself.
