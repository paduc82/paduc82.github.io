However, if you use the Cash Flow Report as the visual report, you will get both timescaled Cost and timescaled Fixed Cost exported to the Excel Pivot table.

You could also simply use the Task Usage view in Project and add the Cost field and Fixed Cost field to the timescaled data. Adjust the timescale values to whatever you need and print your report from there.

Hope this helps.
