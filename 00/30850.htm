Hi

169.254.xxx.xxx is a default IP that Windows installs when it cannot connect to a DHCP server.

Your computer A is configured correctly.

Computer B has No Default Gateway (which in your case it should be the IP of the Router 192.168.1.1).

No Default gateway means that the computer is not aware of the existence of the Router.

Try to update the Network card Drivers.
