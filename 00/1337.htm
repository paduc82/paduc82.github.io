jimwilson312 say:

karl timmermans [outlook mvp 2012-2018] say:

How do I import my contacts and emails from 2003 to Outlook in Office 365?

No need to "import" anything but what steps you need to follow would depend on the type (POP3, Imap etc) account that is involved.

If POP - copy the PST file associated with the account and make sure you specify it when configuring the account in OL'2013/'2016

If Imap - you'll need to copy the PST file designated as the default data file which contains the default contact/calendar folders, open it in Outlook and make it the default in OL'2013/'2016 after you've configured the Imap account.

If Outlook.com - nothing to do - just configure the account.

Suggest you run ScanPST on any PST files you plan to copy on your old machine <before> you copy them (also make sure Outlook is closed before you copy the file(s).

Also suggest that if your existing PST files have "97-2002" in the PST file format description to export the file first (after running ScanPST on it) to create a <Unicode> based PST file and copy the newly created PST file for use in OL'2013/'2016.
