I manage to solve it!

This is what I described as the problem: As you can see format is "Date" and cells values are shown as it.

This is how I solved the problem:

  1. Clicked drop-down format menu
  2. Chose "More Number Format..." ("Date" format is shown as selected)
  3. On "number" tab, on "Category" list I clicked "Custom"
  4. A specific format shown selected (in my case it was "[$-F800]dddd, mmmm dd, yyyy")

One thing you can try is to try disabling it via MSCONFIG.

Windows key + R and then run 'msconfig'.

Look for the Words calendar under the Startup tab and/or under the Services tab and disable.
