gord dibben say:

Yes you did state how many columns..............in your subject line.

 

My bad for reading just the description only.

 

This macro snakes your 7 columns to 14 columns.

 

Public Sub Snake7to14()
Dim myRange As Range
Dim colsize As Long
Dim maxrow As Long
Const numgroup As Integer = 2
Const NumCols As Integer = 14
    On Error GoTo fileerror
    colsize = Int((ActiveSheet.UsedRange.Rows.Count + _
            ((NumCols - 1)) / NumCols)) / numgroup
     Range("A1").Select
    With ActiveCell.Parent.UsedRange
        maxrow = .Cells(.Cells.Count).Row + 1
    End With
    ActiveCell.Parent.Cells(maxrow, ActiveCell.Column) _
            .End(xlUp).Offset(1, 0).Select
    Set myRange = Range(ActiveCell.Address & ":" _
            & ActiveCell.Offset(-colsize, (NumCols - 1)).Address)
    myRange.Cut Destination:=ActiveSheet.Range("H1")
    Application.CutCopyMode = False
    Range("A1").Select
fileerror:
End Sub

Solution:

1. Delete Outlook signature

2. Exit Outlook

3. Delete "NormalEmail.dotm" (%AppData%\Microsoft\Templates)

4. Start Outlook and re-create the signature.

There are at least 3 ways to toggle the display of non-printing characters On/Off:

  • Click the ¶ button in the  Paragraph group on the Home tab,
  • Use the keyboard shortcut Command+8,
  • Go to Word> Preferences> View to clear the check for All in the Non-Printing Characters section.
