suzanne s. barnhill say:

It sounds as if the levels have just gotten mixed up in the "Number format" box. The easiest way to see what the problem is is to temporarily set the starting number of each level to its level number. That is, set Level 2 to start at 2, Level 3 to start at 3, and so on. Then when you look at the number format for Level 3, you will see 1.2.3. If instead you see 1.3.2 or something else, you'll know the levels have gotten mixed.

 

When that happens, clear the Number format box entirely and start over. As an example, for Level 3, click the arrow beside "Previous level number" and choose Level 1. This will give you a 1 in the box. Type a period. Then click again and choose Level 2. This will add a 2 in the box. Type a period. Then choose 1, 2, 3 for the "Number style." This will add a 3 in the box. Type a period if desired. Repeat for any other levels that are messed up.

 

When you have all the levels straightened out, go back and reset the starting numbers to 1.

 

An easier approach,  however, if you want this number series, is to select the select the pane in the Outline Numbering gallery that shows 1 Heading 1, 1.1 Heading 2, 1.1.1 Heading 3. This should be the second from the left on the bottom row. If that pane does not show that formatting, click Reset. That formatting has the levels already linked to the built-in heading styles, but, whatever format you use, do make sure that all the headings are linked. If you want a final period on the numbering of this format, you can add it at each level.

russ valentine say:

Let's hope you copied the correct file. The file you need is your Personal Folders file (*.pst) It's where all the mail, calendar, contacts etc are stored. You'll need to pay particular attention as to whether you want or need to upgrade to the new PST file format that was used after Outlook 2002 since you are skipping multiple versions with this upgrade.

Take a look at these pages for info on Outlook data backup or transfer:
http://www.slipstick.com/config/move_outlook.asp
http://www.howto-outlook.com/Howto/backupandrestore.htm
http://office.microsoft.com/en-us/assistance/HA010771141033.aspx

Note that some of the information on the Microsoft page is outdated and contains errors, particularly when applied to Outlook 2003, 2007 or 2010. Specifically:
1. Ignore any advice to copy your PST file to Outlook's default location to connect it to your profile. Doing so will often corrupt your Outlook profile. Copy it anywhere BUT the default location and then open it within your Outlook profile.
2. Ignore any advice that tells you to use export or import to transfer an entire Outlook data file. That has never been sound advice, but the process has become too deeply flawed to be trusted.
