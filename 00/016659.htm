print button - Microsoft Community

wolfiessam say:

print button

By my own  admission  I am a techno dinosaur, but cansomeone please tell me where the print button  is on word

suzanne s. barnhill say:

By default Word 2013 does not have a Print button the way older (menu) versions did. There are several alternatives:

  1. What Microsoft expects you to do is click File and select the Print tab in the "backstage," which displays a print preview as well as the printing options.
  2. You can also use the keyboard shortcut Ctrl+P to accomplish the same thing, opening the Print tab directly.
  3. There are also several buttons you can add to the Quick Access Toolbar. The Print Preview and Print button does the same thing as (1) and (2) above; that is, it runs the FilePrint command. The Quick Print button runs the FilePrintDefault command (the same command activated by the Print button on the Standard toolbar in Word 2003 and earlier). This command automatically prints the entire current document using the current print settings.

To add a button to the Quick Access Toolbar (if that's your choice), right-click on the QAT or on the Ribbon and choose Customize Quick Access Toolbar... If you haven't customized the QAT before (and it sounds like you haven't), it will still have only a small handful of buttons and will be above the Ribbon. If you plan to add more buttons, you'll probably want to use the Show Quick Access Toolbar Below the Ribbon button (on the same menu) to move it.

In the Customize the Quick Access Toolbar dialog, you can select commands from a variety of categories and click Add>> to add them to the QAT. Both Print Preview and Print and Quick Print are in the Popular Commands category. Other categories represent the Ribbon tabs; commands that can't be found anywhere on the Ribbon are listed in Commands Not in the Ribbon, and All Commands has a complete list. In addition, many Ribbon buttons can be added to the QAT by just right-clicking on them and choosing Add to Quick Access Toolbar.

My clock won't keep time while computer is running. - Microsoft Community

onegreenday say:

My clock won't keep time while computer is running.

My clock won't keep time while computer is running. Windows  Home XP Sp3

Is MS Security Essentials keeping the computer clock from working by blocking continuous synchronization? Thanks
kosh vorlon say:

Hi onegreenday,

 

Be assured that MSE is not the culprit here.

 

The clock, even when set to synchronize with an Internet Time Server, does not do so "continuously", but at periodic intervals (unless you're using some type of 3rd party software).

 

99% of the time, a problem with the clock keeping time correctly is due to a weak or nearly fully rundown CMOS battery on the motherboard.  Try replacing that battery and see if things return to normal.

 

If not, then you can still rest assured that MSE is not responsible for the problem.  It would most certainly not block the clock from synchronizing and again, the problem here isn't that synchornization isn't working but that the clock is malfunctioning (because as I said synchonization is periodic and optional - the clock should function almost as well as the time servers on its own unless there is a problem - and that problem is what must be found and resolved).  Furthermore, if the clock is going bad that quickly, it has nothing to do with synchronization because the clock should actually run perfectly fine without needing to be synchronized at all (that's something of an option though most people use it and it does protect against a slightly weak battery that is only losing very little time and the periodic synchronizations can fix that with no apparent harm).

 

If changing the battery does not resolve the problem, then I recommend you repost here for assistance with what may be a more complicated problem: http://answers.microsoft.com/en-us/windows/forum/windows_xp-performance?page=1&tab=all&tm=1314262670034.

 

I hope this helps.

 

Good luck!
