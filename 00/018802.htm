suzanne s. barnhill say:
To clarify what macropod is saying, if you enter text in the "Find what" box and leave the "Replace with" box blank and unformatted, the found text will be deleted. But if you define a format for the "Replace with" box, that format will be applied to the found text whether you leave the box blank or use ^& (the code for "Find what text"). So if you were to search for "document" in the example Hans gave but pressed Ctrl+I in the "Replace with" box, you would see Format: Font: Italic under the box, and Replace All, instead of deleting the found text, would format it with the designated formatting.
