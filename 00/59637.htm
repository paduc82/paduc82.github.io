Login Information

  • Gateway - 192.168.1.1 or 192.168.1.254
  • Username - user | admin | root     | user   |   root    | root
  • Password - pass | admin | 12345 | 12345 | central | ZOOM-DSL

Port Forwarding

  1. Open a web browser.
  2. Enter http://192.168.1.1 or http://192.168.1.254into the address bar. (Or click the link) When prompted enter the admin for the username and admin for the password if the gateway asks.
  3. From the menu on the left, select Advanced Setup.
  4. Select NAT from the submenu on the left.
  5. Select Virtual Servers on the next submenu.
  6. Select the Add button.
  7. Select Custom Service and input the name "Xbox".
  8. Enter the IP Address of the Xbox in "Server IP Address".
  9. Input 3074 for the External Port Start/End and Internal Port Start/End.
  10. Select TCP/UDP for the Protocol.
  11. Select Apply/Save.

 

Disabling Port Forwarding

  1. Open a web browser.
  2. Enter http://192.168.1.1 or http://192.168.1.254 into the address bar. (Or click the link) When prompted enter the admin for the username and admin for the password if the gateway asks.
  3. From the menu on the left, select Advanced Setup.
  4. Select NAT from the sub-menu on the left.
  5. Select Virtual Servers on the next sub-menu.
  6. At the far right side of the table place checks in the fields you wish to remove.
  7. Select the Remove button.
