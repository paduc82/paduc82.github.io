norman david jones say:

Hi Robert,

Try something like this:

  • Right-click the worksheet tab
  • Select the View Code option from the resultant context menu
  • Paste the following event code: 
'=========>>
Option Explicit

'--------->>
Private Sub Worksheet_Calculate()
    Dim Rng As Range, rCell As Range
    Const myColumn As String = "G:G"  '// Col 7
    Const myRows As String = "10:250"

    Set Rng = Intersect(Me.Rows(myRows), Me.Columns(myColumn))

    On Error GoTo XIT
    With Application
        .ScreenUpdating = False
        .EnableEvents = False
    End With
    For Each rCell In Rng.Cells
        With rCell
            .EntireRow.Hidden = .Value = True
        End With
    Next rCell
XIT:
    With Application
        .ScreenUpdating = True
        .EnableEvents = True
    End With
End Sub
'<<=========

  • Alt+Q to close the VBA editor and return to Excel
  • Save the file as a macro-enabled workbook with an xlsm extension

You can download my test file Robert20160207.xlsm at:

https://www.dropbox.com/s/0ht5428ar3ncbsr/Robert20160207.xlsm?dl=0

For the purposes of my test I have made the cells G10:G250 on Sheet2 dependent on the corresponding column A cells on Sheet1. More specifically, the cell G10 on Sheet2 contains the formula 

        =MOD(Sheet1!A10,2)=0

and this formula has been dragged down as far as G250. In this way. rows on Sheet 2 will be hidden or unhidden according to whether the corresponding column A cells on Sheet1 are even or odd.

anne troy say:

1. Stop hitting Enter. Let Word wrap your sentences around for you.

2. You can turn the feature off. Tools-->Options, Autocorrect, Autoformat as you type. Uncheck it.

suzanne s. barnhill say:

As Anne suggests, Word will not capitalize the first letter of a line unless it is also the first letter of a paragraph (which Word assumes is the beginning of a sentence). If for some reason you do not want to use word wrap on your text (for example, if these are lines of a poem), you can use Shift+Enter to insert a line break instead of a paragraph break (Enter). Since Word will not see your new line as the beginning of a paragraph/sentence, it will not capitalize the first letter.

 

If you're typing something like dictionary entries (which are separate paragraphs but should not be capitalized), you can temporarily disable the option to capitalize the first letter of a sentence at Tools | AutoCorrect Options | AutoCorrect: Capitalize first letter of sentences.
