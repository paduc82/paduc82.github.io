babu v say:

Hi Benjamin,

Thank you for posting your query in Microsoft Community.

Words that are added to the dictionary are saved in the default.dic file. It is located in the following location.

"C:\Users\<user name>\AppData\Roaming\Microsoft\Spelling\<language>\default.dic"

Here user name is your current login name, example: John

  1. Press Windows Key + E, it opens File Explorer.
  2. Double click Local Disk (C:).
  3. Double click User name
  4. Browse through the following location "\AppData\Roaming\Microsoft\Spelling".
  5. You will see the folders based on the language you are using. For U.S. English, this is "en-US". Open that folder.
  6. Open the file "default.dic". It should open in Notepad.
  7. Delete the words that have added, leaving no blank, and save the file.
  8. Now go back to Mail app and check if you see the words.

Hope the information provided is helpful. Do reply, if you need further help.

Method A:
Instead of allowing the automatic installation to run from the CD I viewed the files on the CD and right clicked on "start.exe" then selected "Run as administrator". The installation proceeded perfectly.

When I ran Quicken 2004, of course it wanted me to enter the "Installation Key Code" to renew the licence. I did that but got a message saying that I had installed it on a different computer and asking me to contact Intuit to renew the licence. However I then uninstalled Quicken 2004 and tried a slightly different method:

Method B:
I created a new folder on my hard drive (say "Quicken 2004") and copied all the files (and folders) from the installation CD to that new folder on my hard drive.

I then found the "start.exe" file in the hard drive folder and right clicked, selected 'Properties/Compatibility and ticked the "Run this program in compatibility mode for Windows XP Service Pack 3"

I then right clicked on "start.exe" again and selected "Run as administrator".

The installation again proceeded perfectly and this time my Installation Product Key was accepted without any hassle.

I think the important thing is that Quicken 2004 installation has to be "run as administrator" rather than as normal.  There have been many posts on various forums about copying certain files and manually registering various files but IMO that is needless if you run the installation as adminstrator.
