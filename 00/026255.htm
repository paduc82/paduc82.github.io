genevieve cri say:

Hi,

Merging all of the .pst files is a great idea for you to get all important emails and folders into one. To perform this, you need to import all of the data files first in Outlook from different locations. Once it‚Äôs finished, export it, create the file name, and save to the preferred storage location. To do this, just follow these steps:

  1. Open Outlook.
  2. On the File menu, click Open & Export and select Import/Export.
  3. Click Import from another program or file and click Next.
  4. Select Outlook Data File (.pst) and click Next.
  5. Click Browse to locate your data files, select Do not import duplicates, then click Next. This will take some time because it will depend on the .pst file size.
  6. Once it‚Äôs done, click Finish.
  7. Repeat step 2 to 6 for other data files that you want to merge.
  8. Manage all emails and folders and check if all of them are there.
  9. On the File menu, click Open & Export and select Import/Export.
  10. Click Export to a file and click Next.
  11. Select Outlook Data File (.pst) and click Next.
  12. Highlight the email account that you want to merge into one .pst file.
  13. Check the box for Include subfolders and click Next.
  14. Click Browse to locate the place where you want to save this .pst file.
  15. Create the file name and click OK.
  16. Select Finish and click OK.

Let us know if you‚Äôre able to merge your .pst files by following the steps above.
