andreepin say:

Hi,

Since you aren't able to log-in with your Microsoft account, what I can suggest is for you to have access using your Administrator account. This Administrator account is for the security of your computer. To have your computer fixed, kindly follow the methods below:

Step 1: Enable Built in Admin account:

  1. Insert Windows 10 installation media and restart the computer.
  2. Swipe F12 key (typically this is F12, but it can differ between computer manufacturers) and choose the drive that you inserted the installation media into.
  3. Once the Windows Setup appears, follow these steps:
  4. Swipe Next and select Repair your computer.
  5.  Then you will see a blue screen and an option to choose.
  6. Choose the option Troubleshoot and select Advanced options.
  7. Select Command prompt
  8. Type: net user administrator /active: yes

Once you reach the login screen screen, click on the Administrator account at the login screen. From Built in administrator account, create a new user account and transfer all the user profile data of the old account (The account which you are not able to log in).

Note: Please follow the same steps and type net user administrator /active: no to disable admin account after the troubleshooting is done.

Step 2: Create a user account. Create a new user account from inbuilt admin account:

  1. Press the Windows key + R on the keyboard.

  2. Type netplwiz and press Enter.

  3. Click on Add button and select Sign in without a Microsoft account.

  4. Now click on Local account and follow the on screen to create the local account.

Step 3: Change new account into administrator:

  1. Press Window key + X and select Control Panel.

  2. Click on User accounts.

  3. Click on Manage another account.

  4. Click on the new account that you have just created.

  5. Click on Change account type settings.

  6. Click on Change account type.

  7. Click on Administrator.

  8. Click on change account type.

  9. Restart the computer and log in through the new administrator account.

This should help in getting your query resolved.

Let us know how it goes.

Hi Pallav Kumar,

You can remove write protection from your usb flash drive using the below method:

a. Open start menu, in the search bar type regedit and press enter. This will open the registry editor.

b. Navigate to the following path:

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\StorageDevicePolicies

Note:
If the registry key StorageDevicePolicies does not exist, you will need to create it manually.
c. Highlight StorageDevicePolicies, and then create a New DWORD (32-bit) Value named as WriteProtect.
d. Double click the key WriteProtect in the right window and set the value to 0 in the Value
Data Box and press OK button
e. Restart your computer and try copying files into your USB drives.

mohdrafi say: How to turn on windows firewall ?

  1. Open Windows Firewall by clicking the Start button , clicking Control Panel , clicking Security , and then clickingWindows Firewall .

  2. Click Turn Windows Firewall on or off .  If you are prompted for an administrator password or confirmation, type the password or provide confirmation.

  3. Click On (recommended) , and then click OK .

If you want the firewall to block everything, including the programs selected on theExceptions tab, select theBlock all incoming connections check box.

Warning : Running more than one firewall program at the same time could cause conflicts. It's best to just use one firewall program.
