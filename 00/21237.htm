You can also try to use third party software to read older data (like SkypeLogViewer).  If you made any backups of your main.db file, you could try restoring an earlier copy of it.

    Finding a string of text in a column of text strings. - Microsoft Community

    emseasquaredsdad say:

    Finding a string of text in a column of text strings.

    I have a list of text strings in a column (A). I know that each of these strings are contained within a cell in another column (B; the text I seek is only part of a string in B, but the arrangement of the text is exactly the same). Example: if I Ctrl-F "Example number one" in column B, I find "This is Example number one". What formula can I use to find each text string of column A in column B?
    ron coderre say:

    If at least one of the Col_B cells  will *always* contain each Col_A value...

    This formula returns the position in the Col_B list that the A1 value is found:
    C1: =MATCH("*"&A1&"*",$B$1:$B$10,0)

    If some of the Col_A values may not be found in Col_B, try this:
    C1: =IF(COUNTIF($B$1:$B$10,"*"&A1&"*"),MATCH("*"&A1&"*",$B$1:$B$10,0),"Not found")

    Is that something you can work with?
