john - project say:

Bennett,

In addition to the options outlined by Dale, you could also get close to what you need by using the Resource or Task Usage view and display Cumulative Work. Then it is a simple manual calculation to subtract the cumulative value from the first day in your date range from last day in your date range. If for some reason you want/need to capture that value, you could write it into an extra field (e.g. Text1). You could do this manually or with VBA.

Update: This macro will ask the user for two dates and then write the total work hours into the Task Text1 field. The field will be customized by writing the date range into the field name.

Sub DateRngWork()
Dim t As Task
Dim TWV As TimeScaleValues
Dim TW As TimeScaleValue
Dim SDate As Date, EDate As Date
Dim Hrs As Single, CumWrk As Single
SDate = InputBox("Enter Start Date of Range as mm/dd/yy", "Date Range Work")
EDate = InputBox("Enter End Date of Range as mm/dd/yy", "Date Range Work")

For Each t In ActiveProject.Tasks
    If Not t Is Nothing Then
    t.Text1 = ""
    CumWrk = 0
        Set TWV = t.TimeScaleData(StartDate:=SDate, EndDate:=EDate, _
            Type:=pjTaskTimescaledWork, timescaleunit:=pjTimescaleHours)
        For Each TW In TWV
            If TW = "" Then
                Hrs = 0
            Else
                Hrs = TW.Value
            End If
            CumWrk = CumWrk + Hrs
        Next TW
        t.Text1 = CStr(CumWrk / 60) & " Hrs"
    End If
Next t
CustomFieldRename FieldID:=pjCustomTaskText1, newname:="Work Hrs " _
    & Format(SDate, "m/d/yy") & " - " & Format(EDate, "m/d/yy")
End Sub

If you're looking for information on how to find your Office 365, 2016, or 2013 product key, please read the Find your Office product key article.

  • In most cases, you won't receive (and don't need) a product key to install Office 365, 2016 or 2013 if you purchase it online. Instead, you can go to https://office.com/myaccount, sign in with the Microsoft account you associated with the product and choose Install.
