hansv mvp say:

You must use Date(). If you want to delete records BEFORE a certain date, use < instead of >=.

 

DELETE *
FROM [Purchase Card Monthly File Raw]
WHERE [Post Date]<DateSerial(Year(Date()),10,1)

 

This would delete all records with a Post Date before October 1 of the current year.

The only way to skip 0 values is to replace them with #NA.  You need to follow following steps -

1. CTRL+H

2.  In Find What: put 0

3. In Replace With: put =NA()

4. Press Advance and Tick the Box - Match Entire Cell Contents

You need to plot your graph on this new table.
