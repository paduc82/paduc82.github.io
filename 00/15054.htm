Try this which tests the cells for a date and returns the highest row No of the latest date

 

Sub somesub()
Dim MyRange As Range, R As Long
Dim c As Range, TheRow As Long
On Error Resume Next
R = 0
Set MyRange = Range("A700:A7000")
For Each c In MyRange
    If IsDate(c.Value) Then
       If Int(c.Value) >= Int(R) Then
        TheRow = c.Row
        R = Int(c.Value)
        End If
    End If
 Next
On Error GoTo 0
End Sub

jamesoborne say: I need to search within the access database though. I can calculate distance fine. thats not what I want. I want to see how many entries are within a radius of a selected entry or an inputted decimal degree coordinate.

I need help with using the query function in Access to do this.

tom van stiphout say:

The basic idea is that in the query you call the GreatCircleDistance function. Something like:

select GreatCircleDistance(...) as theDistance from theTable

The arguments of the function are two fields of your table with Lat and Long, and two parameters picking up the new project's coordinates (presumably using Forms!theForm!theLatControl etc.), and a few constants.

Then you set a where-clause to this column in the query of "<1" if you wanted only the results of less than a mile.
