Feedback on Multi-Desktops (Task View)

I'm glad the Windows Team has added the Task View. Power users make good use of it through 3rd party apps (Dexpot is a great example).

A couple of feedback points on Task View:

1. There should be an option to switch between desktops with user-specified keyboard shortcuts. Or at least a keyboard shortcut to pop Task View and use arrows to then select desktops->app.

2. Users will benefit more from this feature if it created separate virtual desktops rather than bins to place running apps. Think of it as separate sessions of Windows 10 within one user space. Currently, all running apps in the different task views appear on the taskbar which defeats the purpose of having separate desktops.

           2.B - User should be able to move apps from one virtual desktop to another.

           2.C - I'm not sure if this is feasible but would it be possible to put apps on the idle desktops in sleep mode and save on RAM?

Ali

Give this macro a try...

Sub ColorSubconstractors()
  Dim X As Long, LastCol As Long, LastRow As Long, NewCol As Long, NewRange As Range
  Const FirstRow As Long = 11
  Const FirstCol As Long = 4
  Const SubcontractorRow As Long = 9
  LastCol = Cells(SubcontractorRow, Columns.Count).End(xlToLeft).Column

                     Interior.ColorIndex = Cells(SubcontractorRow, X).Interior.ColorIndex

  NewRange.Delete
