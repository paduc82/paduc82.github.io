How do I exclude a heading from the document map in Word 2011 for - Microsoft Community

rhestet1 say:

How do I exclude a heading from the document map in Word 2011 for Mac?

I am trying to use the document map feature in word and cannot figure out how to exclude a certain style from showing in the document map. I want to save specific text formatting but want it to function like the "Normal" style. I have seen people on the web point to a webpage discussing how to do this that no longer exists. Any help would be greatly appreciated.
bob jones aka: cybertaz say:

What's displayed in the Navigation Pane [aka Document Map] is determined by the Outline Level attribute of the Style applied to the heading. IMO, the only sensible way to have a heading formatted the same as another but not be listed is to create a new Style identical to the other except that the Outline Level would be set as Body Text rather than assigning Level 1-9.

Alternatively, Headings in Tables & Text Boxes will not display in the Document Map even if there is an Outline Level assigned to the Style applied to them. In my experience, though, that can lead to numerous other limitations such as not being able to include those Headings in a TOC. It also means adding complexity to the structure of the document.

For non-Exchange-based mail systems, Office 365 only support IMAP migration. Here is a workaround:

  1. Login your POP3 account in Outlook client and Export or backup email, contacts, and calendar to an Outlook .pst file .
  2. Login your Office 365 account in Outlook client and Import email, contacts, and calendar from an Outlook .pst file .
  3. After doing that, those emails will be synchronized to Exchange online.
