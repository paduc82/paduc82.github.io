If you only want to count for the month of any year...

=SUMPRODUCT(--(MONTH(A1:A4)=7))

Note that if counting for month January an empty cell will evaluate as month January. In those cases you'd have to include a test that the cell is not empty or in fact contains a date.

=SUMPRODUCT(--(A1:A4<>""),--(MONTH(A1:A4)=1))

=SUMPRODUCT(--(ISNUMBER(A1:A4)),--(MONTH(A1:A4)=1))

hi jarod,

onedrive for business 2016(groove.exe) is installed within the office suite, we cannot uninstall separately. to remove it, we need to uninstall the whole office suite and then reinstall office without the onedrive for business sync client.

to reinstall office without onedrive for business, could you following steps using office deployment tool?

1. we can create an odt folder to download and install the office deployment tool (office2016 version) via the article.
2. open the configuration file with notepad in the odt folder and delete all information on it.
3. open office 365 proplus configuration xml editor and select the product you would like to install as below. note: to exclude onedrive for business, please select groove in exclude office programs option.

4. click add and copy the information on the right to the configuration file in the odt folder as below.

5. download office 2016 source file to the odt folder via >setup.exe /download <path of configuration file> in this article.
note: it usually takes about ten minute depending on your network to finish the process.

6. install office 2016 via >setup.exe /configure <path of configuration.xml file>.

7. activate office with your work/school account.

if anything is misunderstood, please let us know and we will continue to discuss the situation based on the update.
