considerate_guy say:

The key is you need software for burning a video DVD.  I don't think that was made clear in the replies before.  There is lots of software for burning data files on to a DVD; in fact Windows itself can do that.  But a video file burned onto a DVD disc is not a playable video DVD.

 

The DVD standard is pretty particular and requires various folders, with .IFO, .BUP and .VOB files.  The video must be encoded in MPEG-2 format into the .VOB files and if you have a menu structure, that must also be included.

 

In Windows Vista and Windows 7, there was an application called Windows DVD Maker that could be used to make video DVDs.  In fact, if you run Movie Maker on those versions of Windows, and it finds Windows DVD Maker installed, you'll see an option directly in Movie Maker to save your movie and Burn a DVD.

 

In Windows 8, you'll have to find your own solution.  Many PC manufacturers will include video DVD software from Nero, or other 3rd parties.  Alternatively, you can look into freeware options like DVD Flick or DVD Styler.

 

So in short, you need software for burning a video DVD in order for it to be playable in a standard DVD player.

4. Select the 'Display my Out of Office information to contacts in my Friends and Family, Workgroup, and Colleagues privacy relationships' check box to sync your SFB and Outlook accounts.
