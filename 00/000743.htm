Double-click the header. On the Header & Footer Tools Design tab, select "Different First Page." This creates a separate header for the first page.

If the document currently contains just one page and you want to modify the header (and/or footer) for the following pages, double-click the main body of the document and temporarily insert a page break so that you can see the following pages. Go to the main header and make the desired changes. When you are done, you can remove the page break (the content of invisible headers/footers will be preserved).

chrisrennick say: I think you nailed the issue with the section break.

The essential problem with the "Different First Page" option is that the Headers that are not first page headers still have the same height as the first page header even though the content may be different.

When people ask on these forums to have a header on the cover or on one page only they are often also asking how to avoid huge blank areas at the top of subsequent pages once they figure out they can check the "Different First Page" option.

It is extremely frustrating to think you've got a "Different First Page" header and then discover that when you changed the vertical margins in the regular headers you then messed up the First Page header.  The first page header isn't that different after all!

So the answer is that when you insert the section break at the bottom of your first page you can now break the chain and format the next headers however you want including thin, blank headers.  You can even have a first page per section subject to the above constraint on margins/rulers.  You can add as many section breaks as needed to handle chapters, appendices, etc. (although you may be able to avoid special sections with some field codes in the header/footer).

Pragmatically, the section break also allows you to make the first page look exactly like you want with essentially no header/footer and then save the repetitive stuff for the back matter.

So, thank you Mike!

doug robbins - mvp office apps & services (word) say:

The Section Break is NOT necessary for the purposes that you maintain.


The size of the First Page Header will have no influence on the distance of the text on subsequent pages from the top of the page, which is solely dictated by the top margin setting or the height of the primary header (the one on the subsequent pages)

We do have some steps, when it comes to cancelling the Skype Number. To cancel your Skype Number:
