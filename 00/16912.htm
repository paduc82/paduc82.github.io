ossiemac say:

The conditional format must be removed from the cell and then set the cell to required color.

To install the code:

  1. Right click the worksheet tab name to open VBA editor.
  2. Select View code
  3. Copy the code below and paste into the VBA Editor.
  4. Close the VBA editor (Cross far top right of editor window)
  5. Save the workbook as "Macro enabled"
  6. ensure macros are enabled. See Help for how to do this. (Option to "Disable all macros with notification" should be OK)

Private Sub Worksheet_SelectionChange(ByVal Target As Range)
    If Target.Column = 1 And Target.Cells.Count = 1 Then
        Target.FormatConditions.Delete
       'Edit to required color (RGB or color numbers can be used in lieu of vbGreen)
        Target.Interior.Color = vbGreen
    End If
End Sub
