jeeped say:
That code should work. It is missing a closing ) but otherwise it should run. Try it like this,

    Lrow = Cells(Rows.Count, 1).End(xlUp).Row 

Personally, I like to define where Cell(...) resides. Something like,

  With Sheets("Sheet1")
    Lrow = .Cells(Rows.Count, 1).End(xlUp).Row 
  End With

Note the . that acts as a prefix in .Cells(...). That means that it is referring to the Cells within Sheet1. The With ... End With wrapper can be expanded to reference many other functions like this,

  With Sheets("Sheet1")
    Lrow = .Cells(Rows.Count, 1).End(xlUp).Row 
    .Range("B2:B" & Lrow).Formula = "=A1+A2"
  End With

It is always better to definitively identify your cell and range references rather than relying upon the default ActiveSheet. If you post more of the code, I can help nest them into the With ... End With statement.
