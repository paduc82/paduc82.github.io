how do I make Word on a Mac use A5 but printing on A4 paper? - Microsoft Community

williampruitt06 say:

how do I make Word on a Mac use A5 but printing on A4 paper?

how do I make Word on a Mac use A5 but printing on A4 paper?

bob jones aka: cybertaz say:

The document dimensions are determined in File> Page Setup. Open the Paper Size list & choose A5 if the selected printer supports it.

If it isn't listed, open the Format for: dropdown in that same dialog & select Any Printer. [Only sizes supported by installed printers are explicitly listed... If A5 still does not appear in the Paper Size list you'll need to define a Custom Paper Size.]

When you print;

  1. Open the Copies & Pages dropdown,
  2. Select Paper Handling,
  3. Check the box for Scale to fit paper size
  4. Choose A4 from the Destination Paper Size
  5. Check the box for Scale down only if appropriate
