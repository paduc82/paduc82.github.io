peter atherton say:

Hi Rob,

 

Right-click on your sheet tab and choose View Code. Insert the two subs into the sheet module.

Press ALT + Q to return to your sheet

 

Enter a new value in column A and the formatting takes place.

 

Option Explicit

Private Sub Worksheet_Change(ByVal Target As Range)
  Set Target = Range("A:A")
  FormatRows
End Sub

 

Sub FormatRows()
  Dim rng As Range
  Dim lLastRow As Long, iLastCol As Integer
  Dim i As Long
 
  Set rng = Range("A1").CurrentRegion
  lLastRow = rng.Rows.Count
  iLastCol = rng.Columns.Count
 
  For i = 1 To lLastRow
    If CStr(Cells(i, 1)) = CStr("Scheduled") Then
      With Range(Cells(i, 1), Cells(i, iLastCol)).Interior
        .Pattern = xlSolid
        .PatternColorIndex = xlAutomatic
        .ThemeColor = xlThemeColorLight2
        .TintAndShade = 0.799981688894314
      End With
    ElseIf CStr(Cells(i, 1)) = CStr("Submitted") Then
      With Range(Cells(i, 1), Cells(i, iLastCol)).Interior
        .Pattern = xlSolid
        .PatternColorIndex = xlAutomatic
        .ThemeColor = xlThemeColorAccent3
        .TintAndShade = 0.799981688894314
      End With
    ElseIf CStr(Cells(i, 1)) = CStr("Pending") Then
      With Range(Cells(i, 1), Cells(i, iLastCol)).Interior
        .Pattern = xlSolid
        .PatternColorIndex = xlAutomatic
        .Color = 255
      End With
    End If
        
  Next i
End Sub

Note:

The only change that you need to make in both procedures is if the statements are not in column A as you indicated. If they are in say Column C then

 

in the Worksheet_Change procedure change the line

Set Target = Range("A:A")

 

to; Set Target = Range("C:C")

 

In the FormatRows change

 

    If CStr(Cells(i, 1)) = CStr("Scheduled") Then
      With Range(Cells(i, 1), Cells(i, iLastCol)).Interior

to;     If CStr(Cells(i, 3)) = CStr("Scheduled") Then
      With Range(Cells(i, 1), Cells(i, iLastCol)).Interior

You will need to do this 3 times.

 

HTH

Peter

To use the annotation pen during a slide show:

  1. Open the presentation in Slide Show view.

Example

Financial Year End is March 31st each year.

=IF(MONTH(C1018)<=3,MONTH(C1018)+9,MONTH(C1018)-3), where C1018 contains the Transaction Date, so for a Transaction Date in January, Month returns 1, then add 9 gives Period 10 for the Financial year.

=DATE(YEAR(A1)+(A1>DATE(YEAR(A1),3,31)),3,31) 

or, if you just want the year:

=YEAR(DATE(YEAR(A1)+(A1>DATE(YEAR(A1),3,31)),3,31))
