ken sheridan say: The RowSource property returns the values for the drop down list of a combo box.  If you are using an unbound control in a form or report then you should use a text box and set the text box's ControlSource property to:

=DateDiff("d",[Date Invoice Received],Date())

In a query enter the following in the 'field' row of a blank column in query design view:

Age:DateDiff("d",[Date Invoice Received],Date())

This will then translate to  DateDiff("d",[Date Invoice Received],Date()) AS Age in the SELECT clause of the query's SQL statement.

As you have found out, what you cannot do is use an expression which calls the Date function to create a calculated field in the table itself.

Select cell F1.  Then Select column F.  Cell F1 should now be the Active cell.

Set up the conditional formatting using this formula:

=LEN(F1)>40

As long as there are no $s in your formula, the formula should be applied intelligently to the other cells in column F.
