hairyfool say:

Absolutely agree with Bill, do NOT delete your recovery partitions. The two smaller ones are a function of UEFI systems and although your layout could have been done better messing around with them could be damaging. The larger is where the actual recovery image is and as it ends the disk is not relevant to your problem

We would frequently see users trying to repair/reload their computer and have been unable to do so because of the misguided idea of removing a key option.

If you are looking to do this because of a lack of space there are other options. The 4 main Library folders have a Location function which would allow them to be moved to another partition.

Other software that is in the Program Files directories often have custom setup modes that allow you to define the install directory.

Xbox One Slow To Load Game Xbox One Game Disks

hordie say:

Hi vtBADGER, 

Sorry you're having some slowness loading games on your console. There are a few items that come in to factor with the original load times. First is game size. The larger the game, the longer it takes to install. Second, the update size. Almost all games have a considerable sized update that must be required to be downloaded and then patched. Depending on your ISP speed, it could take a long time. Last, digital vs disc. If you have a digital game you're downloading, you must download the patch and the game itself, which again if you have slower internet, it can take a long time. That said, after you install games to your hard drive (disc and digital), your performance once the game starts is going to be MUCH better than if you're using a disc. So it's a slower initial install vs a slower overall experience. If you would like a feature changed that is by design like this, I would suggest going to http://feedback.xbox.com and submitting your ideas there. That said, this is likely something that will not change with the Xbox One as it improves the overall experience.

Installing drivers through Windows Update

Windows Update is the premier starting point to resolve most common driver issues you might experience immediately after installing Windows 10.

Simply launch it from Start > Settings > Update and Security and click Check for Updates

or

Press Windows key + R

Type: wuapp

Hit Enter

Click Check for updates (make sure you have an active Internet connection).

Immediately, Windows Update will detect your system configuration and download all the appropriate drivers for your system.
