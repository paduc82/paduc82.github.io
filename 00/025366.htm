joeu2004 say:

ewelle wrote:

I am multiplying one cell (no Formula) by another cell (formulated) and receiving a dollar number that is not correct.

978,193.00 x 99.62% = 974,475.87 (correct answer) - By Calculator

978,193.00 x 99.62% = 974,436.00 (incorrect answer) - By Excel format

978,193.00 - no formula

99.62% - derived from a formula

Your mistake is looking at the displayed values instead of the actual values.

First, if you multiply 978,193 by 0.9962 (99.62%) on a calculator, you will see 974,475.8666 if you display up to 4 decimal places.

And you will see the same thing in Excel, if you enter those values manually into cells and format the result of the multiplication to display up to 4 decimal places.

Your result, 974,436.00, is either due to how the cell is formatted, or it is because what appears to be 99.62% is not "exactly" 99.62%.  Format the latter as Percentage with 13 decimal places.

If you have any further doubts, I suggest that you upload an example Excel file to a file-sharing website and post the "shared" URL in a response here.

[EDIT] That said, if you want the result to be accurate to 2 decimal places, it would be prudent to explicitly round the result 2 decimal places instead of relying just on formatting.  For example, =ROUND(A1*A2,2).
