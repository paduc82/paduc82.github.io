Try to follow these methods, mentioned below: 

Method 1:

 Disconnect all the external devices (printers, scanner, USB (universal serial bus) drives, etc...) except mouse and the keyboard and then boot up.

If the issue still persists, follow the Method 2.

Method 2:

1. Press Windows Key + R, type devmgmt.msc.
2. Check the setting in the device manager for Keyboard and mouse and make sure the following has a checkmark "Allow this device to wake the computer".

If the issue still persists, follow the Method 3.

Method 3:

Step 1:

Uninstall and reinstall the video card drivers
1. Press Windows Key + R, type devmgmt.msc.
2. Expand the graphic card, click on the drivers.
3. Click on properties, click on uninstall tab.
4. Click OK.
5. Download and install the latest drivers from the manufacturer’s website and check.

Step 2:

If the issue still persists, I suggest you to install the latest above mentioned drivers in compatibility mode and check, if it works.
Follow the steps to install the above mentioned drivers in compatibility mode.
 1. Download the latest above mentioned driver from the manufacturer’s website.
 2. Right-click on the setup file and click on "Properties".
 3. Click on the ‘compatibility’ tab and check the box "Run this program in compatibility mode” and select previous Operating System from the drop down.
 4. Click on ‘Apply’ and click ‘OK’ and run the file to install it.

If the issue still persists, follow the Method 4.

Method 4:

I would suggest you to create a new power plan and check the issue status. Creating a new power might help you to diagnose the issue.
