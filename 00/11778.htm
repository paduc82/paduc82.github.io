scottgem say:
Ok, First, data changes and form changes are two different things. And you don't want them together.

The problem with your code is the flow doesn't track. In the Form's On Current event you set mSaved to False. then in the form's Before Update event you test to see if its false which it will be because, until you press the Save Record it will remain false.

On a bound form, as soon as the record loses focus the data is saved to the table. So all you really need is the Before Update event: 

If MsgBox("Do you want to save this record",vbYesNo) = vbNo Then
     Cancel = True
     Me.Undo
End If


Before the record is saved you will be prompted to save it. If you reply No, the save will be cancelled.

Saving the form should only be done during development when you make changes to the design of the form.

ken sheridan say: This could be done in a rather convoluted query, using subqueries.  Here's an example using Northwind data:

SELECT [Order Date], Employee, TotalSales,
    (SELECT COUNT(*)
     FROM
         (SELECT Employees.ID, [Order Date],    
          SUM([Quantity]*[Unit Price]) AS TotalSales
          FROM (Employees INNER JOIN Orders ON Employees.ID = Orders.[Employee ID])
          INNER JOIN [Order Details] ON Orders.[Order ID] = [Order Details].[Order ID]
          GROUP BY Employees.ID, [Order Date]) AS QTS1
       WHERE QTS1.ID = Q_TotalSales.ID
       AND QTS1.[Order Date] <= Q_TotalSales.[Order Date]) AS SalesNo,
    (SELECT MAX(TotalSales)
     FROM
         (SELECT Employees.ID,     
          SUM([Quantity]*[Unit Price]) AS TotalSales
          FROM (Employees INNER JOIN Orders ON Employees.ID = Orders.[Employee ID])
          INNER JOIN [Order Details] ON Orders.[Order ID] = [Order Details].[Order ID]
          GROUP BY Employees.ID,[Order Date]) AS QTS2
       WHERE QTS2.ID = Q_TotalSales.ID) AS MaxSales
FROM
    (SELECT Employees.ID, [First Name] & " " & [Last Name] AS Employee, [Order Date],     
      SUM([Quantity]*[Unit Price]) AS TotalSales
      FROM (Employees INNER JOIN Orders ON Employees.ID = Orders.[Employee ID])
      INNER JOIN [Order Details] ON Orders.[Order ID] = [Order Details].[Order ID]
      GROUP BY Employees.ID, [First Name] & " " & [Last Name], [Order Date]) AS Q_TotalSales;

Once the contacts are imported to your account, sync the account on the phone by going to settings > emails + accounts > tap and hold the account you added the contacts to > select sync.  If you have yet to set up the phone, add this account to emails + accounts and those contacts should be synced to the phone with the account.
