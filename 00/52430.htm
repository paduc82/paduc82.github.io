It depends on how Windows 10 was originally configured on your computer.

If its a full retail store bought license online or offline, it is transferable to a new computer or motherboard.

If its the free upgrade from a retail store bought Windows 7 or Windows 8 license, it is transferable to a new computer or motherboard.

If its the free upgrade from a OEM preinstalled Windows 7 or Windows 8 license, it is NOT transferable to a new computer or motherboard.

place all three patches in the main fallout directory (leave them rared (zipped))

unrar (unzip) the Fallout v.1.3.5 patch by TeamX (unofficial) and it creates a folder in the directory called fallup13. copy the contents into the fallout directory and run fallout, then exit fallout.

run this file

click DONE then exit the game.

restart fallout

at this point you can delete the rar (zip) files and the corresponding folders they have created from the fallout directory if you wish (not necessary though).
