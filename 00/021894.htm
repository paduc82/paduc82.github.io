  9. A final question should be (paraphrasing) "Some error was found with the directory lock_maps. Convert folder to file?" - Answer "yes" to this as well
  10. After it's done, go the the E:\cities fodler again - lock_maps should now be a regular file

  12. Restart Ovi Maps and enjoy.

karl timmermans (outlook mvp 2012-2018) say:
>>
I had to reload my Outlook 365 but lost my contacts, and calendar data. How do I recover it?
<<

If the email account to which your contact/calendar data belonged was configured as an IMAP account (contact/calendar folders would have had the added words "This Computer Only" in the folder description), then unfortunately that data is lost unless you explicitly made a manual backup of those folders by exporting them to a PST file.

To mitigate this from occurring in the future, suggest one of the following two approaches

Option #1

#1 - Create a new PST file in the profile
(go to Home ribbon --> <new items> --> <more items> --> <Outlook Data file>
#2 - Go to File --> Account Settings --> Data Files tab
#3 - Select the PST file just added --> Set it as the default data file

Once the above is done, the default contact/calendar folders will be stored/maintained in the PST file which doesn't get deleted if Outlook (or the email account) is ever re-installed

Option #2

#1 - Create a free Outlook.com email account
#2 - Add the email account in Outlook
#3 - Go to File --> Account Settings --> Data Files tab --> select the OST file for the Outlook.com account --> set it as the default data file for the profile

This approach enables sharing contact/calendar data across multiple devices where the account has been configured.
