some missing footers - Microsoft Community

richardtomkins say:

some missing footers

I am creating a Word 2010 document with multiple sections.

I have mirrored pages, with different odd and even footers.

I am using section and page numbering as this makes it fairly easy to navigate the technical document.

I have observed the following behavious and wonder what I am missing?

 

After an odd page section break, my footer for an odd page shows, the text flows onto the next page which is the even page and the appropriate footer shows and the text flows onto the thrid page and there is no footer.

 

So, what have I not set or configured to have the odd page footer show up on the second odd page in the section, which is the thrid page in the section?

 

 

suzanne s. barnhill say:

When you disabled "Different first page," you lost the First Page Footer, which is what was appearing on the first (odd) page of each section. Since you have no text in the Odd Page Footer, you are now not getting a footer on any odd pages. This is completely logical. You need to add the text you had in the First Page Footer to the Odd Page Footer.

With regard to linking, page numbering, etc., I think you would benefit by reading these articles:

"Making the most of headers and footers"
http://sbarnhill.mvps.org/WordFAQs/HeaderFooter.htm

"How to set up a document with front matter numbered separately"
http://www.word.mvps.org/FAQs/Formatting/front_matter_2007.htm

"How to control the page numbering in a Word document"
http://word.mvps.org/FAQs/Numbering/PageNumbering.htm

suzanne s. barnhill say:

Unlinking doesn't unlink sections; it unlinks specific headers and footers, as described in http://sbarnhill.mvps.org/WordFAQs/HeaderFooter.htm. If you unlink the Odd Page Header, you're unlinking it from the Odd Page Header in the previous section. You are not affecting any First Page Header or Footer, any Even Page Header or Footer, or the Odd Page Footer.

The "Different odd and even" setting is applied to the entire document; you can't enable it for specific sections. You can enable "Different first page" for specific sections, but when you create a new section, by inserting a section break, the new section inherits the properties of the previous one, including "Different first page" (if enabled), restarting numbering, etc.

The First Page Footer appears only on the first page of a section (whether it is even or odd). The Odd Page Footer appears on odd pages that are not the first page of the section (if "Different first page" is enabled).
