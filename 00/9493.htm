TTY/TDD is telephonic support for the deaf. You can find out its general purpose by doing a general Bing search.

charles kenyon say:

I'm sorry. I think we have a failure to communicate. (Or perhaps you are simply telling me that you get it.)

Personalizations end up in all sorts of places.
Word 2007 & Later Key Data File Locations

You can store many of them in a template and set it up as a global template or Add-In.

Read the link: Global Template

This shows what such a template is and how you make a template into a global template.

You put ribbon changes into a template through XML modifications. I gave three links to give you a start on that.

You put QAT modifications into a template rather into the special QAT file. This is relatively easy. You simply tell Word to store it in the open template. This can be done without any programming through the Word Interface. Modifying the QAT and Ribbon in Microsoft Word

You can store macros in a template rather than in the normal template. The template in which you want to store the macro must be the attached template for the open document.

You can store keyboard shortcuts in a template rather than in the normal template. The template in which you want to store the keyboard shortcut must be open for editing or the attached template for the open document.

You can store Building Blocks and AutoText in a custom template rather than in the default templates. Where can Building Blocks be stored?

The custom template can be one or more templates. If you are storing macros in a template, you want any keyboard shortcuts or QAT modifications using those macros stored in the same template.

You can move or shuffle many of these things around if you want them someplace other than where you originally stored them.

Moving (Sharing) Customizations in Microsoft Word

Below is a screenshot showing some of my personalizations I have stored in global templates (a list of the templates). All but two of those are ones I created.

A number of them are available for download from my downloads page if you want to take a look and see how they are set up. It is not hard to create a global template.

Downloads - Microsoft Word - Add-Ins - Tutorials - Letterhead - Templates

The key that can hold many customizations by default is the normal template. You could store many of your customizations there. The normal template is an uber-global template. However customizations stored there may not transfer as easily or survive microsoft updates that well. Normal Template in Microsoft Word - How to Find or Open the Normal Template
