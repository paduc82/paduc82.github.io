short circuit. say: Sometimes Windows Update gets stuck and needs to be refreshed.
I have a batch file that I use when Windows Update get into a loop where half of it decides the update has already been done
and the other half insists that it hasn't received the update. It's a simple Script to Reset / Clear Windows Update.
Copy the following, open Notepad and past it into Notepad. Save this as WindowsUpdateReset.bat




@ECHO OFF
echo Simple Script to Reset / Clear Windows Update
echo.
PAUSE
echo.
attrib -h -r -s %windir%\system32\catroot2
attrib -h -r -s %windir%\system32\catroot2\*.*
net stop wuauserv
net stop CryptSvc
net stop BITS
ren %windir%\system32\catroot2 catroot2.old
ren %windir%\SoftwareDistribution sold.old
ren “%ALLUSERSPROFILE%\application data\Microsoft\Network\downloader” downloader.old
net Start BITS
net start CryptSvc
net start wuauserv
echo.
echo Task completed successfully…
echo.
PAUSE




Right click on the .bat file you've created and select "Run as Administrator".
Once completed, reboot the system and try your update again.

greg carmack - mvp 2010-2020 say:

Is the hard drive detected in BIOS setup or UEFI Firmware?  You can access this by powering on from the tiny Novo button next to power button, or by pressing the F1 or F2 key as you power up the PC.

In addition to being listed under Storage devices, then drive should be listed in the Boot Priority order.  For Legacy BIOS it should be set first to boot, for the more modern UEFI the Windows Boot Manager should come first. 

If the drive is not listed in BIOS or UEFI firmware then you'll need to try reinstalling it to get it detected, checking cables and connections.  You can try plugging it into another PC to test it.  If it cannot be detected replace it.  An SSD is the best replacement, and the best Upgrade one can have - lightning fast, faster than a new higher end PC without one. 

Then do a Clean Install Windows 10.

If the hard drive is detected but still will not boot, follow the steps for Troubleshooting Windows 8 and 10 Failure to Start - Microsoft Community which includes everything possible to get it started - including a hard drive test to be surest about it's condition.

When does this error occur?

Google searches for avesuduc.exe or tddvihtb result in no hits, which suggests that this likely is malware.  The error indicates that your antivirus or antimalware software has removed the actual malware executable file (avesuduc.exe) but has left behind the instructions that cause Windows to attempt to run that file.

Because taskeng.exe is the Windows Task Scheduler, it's likely that whatever this is created a scheduled task that is attempting to run avesuduc.exe. 

Because the Task Scheduler does not have a search function, download (or run from the web) Autoruns:  https://docs.microsoft.com/en-us/sysinternals/downloads/autoruns

You can either select the "Scheduled Tasks" tab and look in the "Image Path" column for avesuduc.exe or simply  select the "Everything" tab and then press CTRL+F to search for avesuduc.exe.  Assuming that you find at least one line that includes avesuduc.exe, either uncheck the box at the left of the line to disable the task from running or -- if you're certain that the task really is malware, delete the line.  If Autoruns complains that it can't make a change, close it and restart it by right-clicking and selecting "Run as administrator."
