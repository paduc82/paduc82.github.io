doug robbins - mvp office apps & services (word) say:

Sub splitter()

'

' splitter Macro

' Macro created 16-08-98 by Doug Robbins to save each page of a document

' as a separate file with the name Page#.DOC

'

Dim Counter As Long, Source As Document, Target As Document

Set Source = ActiveDocument

Selection.HomeKey Unit:=wdStory

Pages = Source.BuiltInDocumentProperties(wdPropertyPages)

Counter = 0

While Counter < Pages

    Counter = Counter + 1

    DocName = "Page" & Format(Counter)

    Source.Bookmarks("\Page").Range.Cut

    Set Target = Documents.Add

    Target.Range.Paste

    Target.SaveAs FileName:=DocName

    Target.Close

Wend

End Sub

doug robbins - mvp office apps & services (word) say:

The Files will be saved in the folder set for Documents under File>Options>Advanced>General>File Locations.

If they were saved to the Desktop, that is where you must have that option set.

The following modified code will save the documents in the same folder as the original

Dim Counter As Long, Source As Document, Target As Document, strPath as String

Set Source = ActiveDocument

strPath = ActiveDocument.Path & "\"

Selection.HomeKey Unit:=wdStory

Pages = Source.BuiltInDocumentProperties(wdPropertyPages)

Counter = 0

While Counter < Pages

    Counter = Counter + 1

    DocName = "Page" & Format(Counter)

    Source.Bookmarks("\Page").Range.Cut

    Set Target = Documents.Add

    Target.Range.Paste

    Target.SaveAs FileName:=strPath & DocName

    Target.Close

Wend

End Sub

graham mayor say:

This is going to be difficult - because there are no 'pages' in a Word document. Word is not a page layout application. Doug's suggestion works by using Word's internal 'page' bookmarks to determine where to make the split.

Beyond that you are going to have to identify fixed markers to determine where to make the breaks. These could be section breaks or physical markers, but without such delineation your task requirement seems hopeless.
