suzanne s. barnhill say:

Two ways to include text that will not print are:

  1. Format the font color as white.
  2. Format the text as Hidden.

If you format a heading as Hidden, it will not print, will not take up space, will not be displayed unless you display Hidden text...and unfortunately will not show up in the Navigation pane.

But if you format it as white, it will not print, but it will show up in the Navigation pane. It does take up space, but you can format the font size as 1 point to reduce this (the size in the Navigation pane is unchanged). Unfortunately, you won't be able to see it in your document--only in the Navigation pane.
