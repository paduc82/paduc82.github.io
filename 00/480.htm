david marcovitz say: If I understand your code, John, I think you interpreted the question differently than I did. I think Em wants to hide other slides when the check box is checked, not the current slide. That would involve something more like:

Private Sub CheckBox1_Click()
With ActivePresentation
    .Slides(7).SlideShowTransition.Hidden = Me.CheckBox1.Value
    .Slides(14).SlideShowTransition.Hidden = Me.CheckBox1.Value
    .Slides(21).SlideShowTransition.Hidden = Me.CheckBox1.Value
    .Slides(28).SlideShowTransition.Hidden = Me.CheckBox1.Value
End With
End Sub

In this case 7, 14, 21, 28 simply represent the slides that you want to hide when the box is checked. Change those numbers (and add or subtract lines) if you want to hide a different set of slides.

Of course, if you are talking about hiding the same slide the checkbox is on, ignore this and go with what John said.

steve rindsberg say:
For example, like making modules (several slides worth) visible or invisible.  Tell me I'll need a "Mr. Tags" t-shirt if I keep it up but ... I'd use tags.

When you click, say, the "APPLES" checkbox, the code does something like:

Call SetVisibility("APPLES", ShowOrNot)
' ShowOrNot is a boolean, True to show the slide, False to hide it
' It's = the value of the Checkbox in question

Sub SetVisibility(sTag as String, ShowOrNot as Boolean)
  Dim oSl as Slide
  For Each oSl in ActivePresentation.Slides
    If oSl.Tags("Module") = sTag Then
      .SlideShowTransition.Hidden = ShowOrNot
    End If
  Next
End Sub

To make it more entertaining, you might want to tag each slide with multiple module names:

"APPLE|PEACH"

and then substitute:

If Instr(oSl.Tags("Module"),"sTag) > 0 

So it finds the slide when you click either the APPLES or PEACH but not PEARS checkboxes

Sub AKTest()
  Dim Source As Range
  Dim R As Range
  Dim Wb As Workbook
  
  Set Wb = ActiveWorkbook
  'If the main file is the current file
  If ThisWorkbook.Name = Wb.Name Then
    MsgBox "Switch to the data workbook and try again"
    Exit Sub
  End If
  
  Application.EnableEvents = False
  Application.ScreenUpdating = False
  Application.DisplayAlerts = False
  
  'Refer to the data rows in the data file
  Set Source = Range("A4", Range("A" & Rows.Count).End(xlUp))
  'Visit each row
  For Each R In Source
    'Expand to the end
    Set R = Range(R, R.Offset(, Columns.Count - R.Column).End(xlToLeft))
    R.Copy
    'Paste into the main file into the column
    ThisWorkbook.Sheets("Details").Range("D1").PasteSpecial xlPasteValues, Transpose:=True
    'Copy off!
    Application.CutCopyMode = False
    'Switch to the main file and export
    ThisWorkbook.Activate
    ThisWorkbook.Sheets(Array("Template (option 2)", "Quick Guide")).Select
    ActiveSheet.ExportAsFixedFormat Type:=xlTypePDF, _
      Filename:=ThisWorkbook.Path & "\MM Scorecard Individual " & Range("G2").Value & " " & Range("B2").Value & " - " & Range("B1").Value, _
      Quality:=xlQualityStandard, IncludeDocProperties:=True, IgnorePrintAreas:=False, OpenAfterPublish:=False
    'Switch back to the data file
    Wb.Activate
  Next
  'Events on
  Application.EnableEvents = True
End Sub
