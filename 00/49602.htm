lori mey say: I would be able to do it via a batch file 

frederik long say:
I would be able to do it via a batch file 

OK, here is what you could do.

  1. Create the batch file C:\Tools\Shut.bat.
  2. Create the script file C:\Tools\Toggle.vbs
  3. Modify the scheduled restart task so that it invokes C:\Tools\Shut.bat
  4. Create a desktop shortcut to invoke C:\Tools\Toggle.vbs.

Double-clicking the new shortcut will create a file that will block the next restart.

Double-clicking it again will remove the block file.

A pop-up informs the user about the result of his double-click.

Contents of C:\Tools\Shut.bat:

@echo off

goto Start
---------------------------------------------------
This batch file is triggered by the Task Scheduler.
It will conditionally restart Windows, depending
on the existence of the file C:\Tools\Block.txt.
The file C:\Tools\Block.txt is created or deleted
by the user through C:\Tools\Toggle.vbs.
26.7.2018 FNL
---------------------------------------------------
:Start
if exist c:\Tools\Block.txt (
  del c:\tools\Block.txt
) else (
  shutdown /f  /r  /t  300
)

Contents of C:\Tools\Toggle.vbs:

'--------------------------------------------------------
'Create or delete the file C:\Tools\Block.txt.
'Its existence determines if the script C:\Tools\Shut.bat
'will allow or defer the scheduled 9 pm restart.
'26.7.2018 FNL
'--------------------------------------------------------
sBlock = "C:\Tools\Block.txt"
Set oFSO = CreateObject("Scripting.Filesystemobject")
If Not oFSO.FolderExists("C:\Tools") Then oFSO.CreateFolder("C:\Tools")
If oFSO.FileExists(sBlock) Then
oFSO.DeleteFile(sBlock)
sMsg = "The system will restart at 9pm as scheduled."
Else
    oFSO.CreateTextFile(sBlock)
    sMsg = "The scheduled restart has been deferred by 24 hours."
End If
MsgBox sMsg, 64, "Shut-down blocker"
