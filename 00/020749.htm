If you began with a VM and activated Windows 7 or 8, that VM could have been upgraded to activated Windows 10. However, the Windows 10 would no longer have any unique Product Key. It would instead use a generic Product Key and would be activated with a digital license.

Once you have done the above, then that VM could be maintained as a perpetually activated VM, as long as you never delete its files.  If you delete the VM files (other than the virtual hard disk file) you have deleted the device image that is recognized by the activation servers.  Creating a new VM, even on the same host PC, will not result in an activated Windows 10.

I have some VMs that were activated back in early 2015 with Windows 10 when it could be activated by an Insider Product Key. I have kept a copy of the small VM files (except the hard disk file) since then. I can still install Windows 10 Pro and it will auto-activate Windows 10 Pro once online.  I have never tried using the VM files on another PC, but I doubt that they would activate on another PC.

If you can get Windows 7 or 8 to activate on the VM using the keys, by transferring the Windows 7 or 8 license to the VM, I would say it might be possible to then upgrade to Windows 10 and activate, but I don't know if it would work. Sometimes Microsoft Support staff will help with specific Activation scenarios, since the License Terms do say that standalone licenses and upgraded licenses can be transferred to another device. As has been said already, this is not possible with OEM Windows licenses.

excel and word comparison - Microsoft Community

apollon17 say:

excel and word comparison

 how comparable is excel  to Word, and what about the look of excel

hansv mvp say:

The programs serve different purposes.

Word is intended to create documents that consist mostly of text. It has extensive text formatting capabilities - not just for formatting characters, but also for formatting entire paragraphs. On the other hand, it has limited and rather primitive support for formulas and calculations.

Excel is intended for performing calculations. It has extensive capabilities for formulas and functions. Its text formatting features are limited compared to Word.

Workbooks containing macros can be saved in 3 formats using File > Save As:

.xlsm (Macro enabled format)

.xlst (Macro enabled template)

.xlsb (binary format)

Make sure your file name has no special characters - especially slashes and asterisks.

davem121 say:
Conventional wisdom says this problem is caused by these 2 dll's if you have them on your system:

C:\Program Files\Common Files\Apple\Internet Services\ShellStreams.dll
C:\Program Files(x86)\Common Files\Apple\Internet Services\ShellStreams.dll

Rename them to ShellStreams1.dll temporarily, restart your PC and try to open your workbook. If your workbook opens well and good, if it does not, you should revert these files to their original name . . .

Failing that, have you tried to repair the workbook?
If not, in Excel choose File - Open, select the file and instead of clicking 'Open', click the small arrow to the right of the 'Open' button and select 'Open and Repair'

You say 'if I try to save it in a different format in excel it deletes the macros', a spreadsheet with macros must be saved as an xlsm file or xltm in the case of a Template with macros. Otherwise the macros will be deleted in order for you to be able to open the file again . . .
