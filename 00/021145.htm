chip pearson say:

An error 9, "subscript out of range," occurs when you attempt to access a member of a collection that does not exist, or when you attempt to get a value from an array using an index number that is larger or smaller than the array. For example,

 

Dim WS As Worksheet
Set WS = Worksheets("Sheet4")

This will cause an error 9 if there is no worksheet named "Sheet4". You can also get an error 9 if you attempt to go outside the bounds of an array:

 

Dim Arr(1 To 3) As Variant
Dim V As Variant
V = Arr(4)

This will raise an error 9 because you are attempting to access element 4 and there are only 3 elements in the array.,
