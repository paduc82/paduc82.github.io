You can use a macro to clear the content of all the headers:

Sub ScratchMaco()
Dim oSec As Section
Dim i As Long
For Each oSec In ActiveDocument.Range.Sections
For i = 1 To 3
oSec.Headers(i).Range.Delete
Next i
Next oSec
End Sub

daniel pineault say:

If you open your excel workbook as an instance within Access then you can reference it a execue the Macro(s) no problem.  Try using a function such as the one I quickly pieced together for you.

Function RunXLSMacro(sFile As String, sMacroName As String) As String
'Requires a reference to the Microsoft Excel xx.0 Object Library
On Error GoTo Error_Handler
    Dim xlApp       As Object
    Dim xlWb        As Object
    Dim sFileName   As String
   
    Set xlApp = CreateObject("Excel.Application") 'Create an Excel instance
    Set xlWb = xlApp.Workbooks.Open(sFile, True)  'Open the specified workbook
    xlApp.Visible = True                          'Control whether or not to show Excel to your user
   
    sFileName = Right(sFile, Len(sFile) - InStrRev(sFile, "\"))

    xlApp.Run sFileName & "!" & sMacroName         'Execute the specified macro

Error_Handler_Exit:
    On Error Resume Next
    xlWb.Close (True)                             'Save the excel workbook
    xlApp.Quit                                    'Close/Quit Excel
    Set xlWb = Nothing
    Set xlApp = Nothing
    Exit Function

Error_Handler:
    MsgBox "MS Access has generated the following error" & vbCrLf & vbCrLf & "Error Number: " & _
    Err.Number & vbCrLf & "Error Source: RunXLSMacro" & vbCrLf & "Error Description: " & _
    Err.Description, vbCritical, "An Error has Occured!"
    Resume Error_Handler_Exit
End Function

I think the original posts explain this but,

If you don't use the Outlook app for email, have more than one hotmail account and both use ActiveSync on your mobile device (mine is an android tablet) you might get an email suggesting that you should either download the Outlook app or 'click here' to continue using an alternative app.

If you choose 'click here' and all's fine I assume no problem. If like me you start receiving emails for the account that didn't get the Sync email to the account that did; so seeing the emails twice in your In Box, ditto for diary reminders.

You have three options.

Delete the account you're seeing twice from your email app and let ActiveSync forward the emails to your second account (I didn't want to do that)

Delete the account that did get the email, add it again with ActiveSync and ignore the email, not sure what would happen if you did this, the email says your account won't sync properly (I didn't want to do that)

Delete the account that did get the email and add it again manually using IMAP instead of ActiveSync (I did this and don't have any problems.

There's a fourth option, which is to switch to the Outlook app. I'm not sure what would happen if you did that.

To be sure this is the problem, check that the accounts aren't linked as explained in the above responses, if yes unlink them if you don't want them linked.

Also get someone to send you an email to the account that didn't get the Sync email from Microsoft. If you see it twice, delete one of them. If they both disappear from your In Box the I'm pretty sure this is your problem
