multiple conditions in merge document

doug robbins - mvp office apps & services (word) say:

You have not said what you want if the conditions are not true, but if nothing should be inserted in that case then use

{ IF { MERGEFIELD 1 } = "" { IF { MERGEFIELD 2 } = "cv" "text" "" } "" }

You must use CTRL+F9 to insert each pair of field delimiters and you use ALT+F9 to toggle off their display.
