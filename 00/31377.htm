david. f say: You will not lose any functionality by using the compatibility options.
Since the game is supported on Windows 7, it is very unlikely to fix the problem you are having by using compatibility mode.

As mentioned, you can create a new post on the Zoo Tycoon forums that Arya linked to for assistance with the game not working.

To try and explain more on the differences between xp mode and the compatibility options:

XP Mode is a virtualized stand-alone version of Windows XP - you can think of it as a separate computer all together that runs through a program on Windows 7.

The compatibility options make adjustments in the background so that the program has a better chance to run natively on Windows 7.

It is better to use compatibility mode if you can as performance will be better - while it will not work for every program, it generally does a very good job.
XP Mode will run any program that runs on Windows XP as long as it does not require hardware acceleration, meaning most 3D games will not work.
