hansv mvp say:

Excel interprets the arguments to trigonometric functions such as SIN, COS and TAN as radians, not as degrees. 360 degrees = 2*pi radians where pi = 3.141592...

 

You can use the RADIANS function for this:

 

=SIN(RADIANS(45))

john sr wilson say:

As Steve says - bad smell!

This vba is not totally foolproof but will usually tell you what is missing

Sub chexFont()
Dim wdapp As Object
Dim fonts() As String
Dim L As Long
Dim strFontName As String
Dim strReport As String
Set wdapp = CreateObject("Word.Application")
ReDim fonts(1 To wdapp.fontnames.Count)
For L = 1 To wdapp.fontnames.Count
fonts(L) = wdapp.fontnames(L)
Next L
Set wdapp = Nothing
For L = 1 To ActivePresentation.fonts.Count
strFontName = ActivePresentation.fonts(L).Name
If Not isMatch(strFontName, fonts) Then strReport = strReport & strFontName & " is missing (probably)." & vbCrLf
Next L

If strReport = "" Then strReport = "All OK"

MsgBox strReport
End Sub

Function isMatch(strFontName As String, fonts As Variant) As Boolean
Dim L As Long
For L = 1 To UBound(fonts)
If UCase(fonts(L)) = UCase(strFontName) Then
isMatch = True
Exit Function
End If
Next L
End Function
