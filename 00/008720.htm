jay freedman say:
It would theoretically be possible to move the Program Files (x86)\Microsoft Office\Office14 folder with all its contents to another drive. But then you'd have to hunt down every reference in the registry to anything in that folder and its subfolders and change the drive letter (and any other part of the path that changed). Also, I couldn't guarantee that there isn't something in a binary-encoded entry somewhere. You might also have to go through what's in the ProgramData folder and the AppData folder of your profile. If you got to the end of that multi-hour exercise and Office didn't work, you'd have wasted your time.

It would be much easier and more likely to succeed if you completely uninstall Office, reboot, and reinstall with a Custom installation (one of the choices on the first page of the wizard) to specify a non-default location.

While an uninstall/reinstall should not touch any of your data files, it's a good idea to do a full backup before starting (and make it an image backup if you have that ability). I don't think Office will insist on re-activating, but if it does then it should do so over the Internet without complaining.

Save to cloud other than Onedrive - Microsoft Community

jeffrey999 say:

Save to cloud other than Onedrive

How do I save a word document (word 2013/win 10) to another cloud beside OneDrive, eg, Acer, Amazon S3, or Dropbox? 
jay freedman say:

To clarify one point: When you install the Dropbox app, it creates an actual folder on your local drive. You can specify where you want that folder to be -- I keep mine on the D: drive. The existence of that folder is what makes it appear in File Explorer and Office's Save As dialog box.

The app watches for any additions or changes in that folder, and uploads them to the Dropbox server "in the cloud". Similarly, changes on the server are synced to the local folder on any other device that's logged into the same account.

I expect that most cloud-based sharing services work similarly.
